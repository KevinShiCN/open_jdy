### 功能简介
- 接口适用于精斗云云进销存。
- 阐述API调用的流程，并以示例的方式介绍API如何调用。

### 前置条件
- [创建应用](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=eb5f1404d9dd11ed8e36016f0b230280&noside=true)
- [申请沙箱环境](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=94258fafd9de11edbfb2c7da3836a943&noside=true)

### 获取授权信息
- 从[获取授权信息](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=9d5e184dd9de11ed8e3697d4973ff91a&noside=true)步骤中，获取以下信息。

| 参数名称  | 说明                       |
| -------- | -------------------------- |
| clientId| 应用ID，用于获取access_token |
| app_key| 授权key，用于获取access_token |
| app_secret| 授权密钥，用于获取access_token |
| accountId| 产品账套ID，用于调用API |
| domain| 环境所属IDC，用于调用API |

### 获取access_token
- 通过client_id、app_key、app_secret获取[access_token](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=9bd40f8fda9d11edbfb2bf125f1b3e41&noside=true)。

### 调用API
- 以客户列表接口为示例
#### params参数

| 参数名称 | 类型   | 必选 | 说明                       |
| -------- | ------ | -------- | -------------------------- |
| acees_token | String | true | 金蝶云账号token |
|dbId|String|true |账套ID，【获取授权信息】时返回的accountId|

#### header参数
- **<font color="#dd0000">header参数为必填</font>**

| 参数名称   | 类型   | 必选  |  说明 |
|-----------|----------|---------|--------|
|X-GW-Router-Addr|string|true|IDC域名，【获取授权信息】时返回的domain|

<font color="#dd0000">**特别注意**</font>：X-GW-Router-Addr是变动的，当生产环境租户被拉入灰度环境时接口无法调用，接口返回502，则需要[主动获取授权](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=801d558dda9d11edbfb2d572de04f9ce&noside=true)信息，以获取最新X-GW-Router-Addr。

#### 请求示例
```txt
curl --location --request POST 'https://api.kingdee.com/jdy/customer/list?access_token={{access_token}}&dbId=7878787889' \
--header 'Content-Type: application/json' \
--header 'X-GW-Router-Addr: http://vip1-gd.jdy.com'
--data-raw '{
}'
```

#### 响应示例
```json
{
    msg: 成功,
    totalsize: 21,
    code: 0,
    records: 21,
    totalPages: 1,
    page: 1,
    items: [   { }  ]
}
```
