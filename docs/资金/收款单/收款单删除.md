### 基本信息
- 用途说明：删除收款单
- 请求方式：POST
- 请求地址：https://api.kingdee.com/jdyscm/receipt/delete
### 请求参数
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)
#### params参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID

#### body参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|items |*true*	| array|单据编号列表

#### 请求示例
```
POST 
- https://api.kingdee.com/jdyscm/receipt/delete?access_token=xx&amp;dbId=xx

HTTP 1.1
```
```
BODY
{
    items:[
        SK20190318003,
		SK20190318004
    ]
}
```

### 响应参数
返回参数 	|字段类型 	|字段说明 
:---   |     :---     | :---
code	|string	| 返回码（0成功，-2失败，-3部分失败）
msg	|string 	|结果或错误描述
totalsize|number|记录数
items|array|详细结果
--number|number|单据编码
--code|number|返回码(单张单据)
--msg|string|描述(单张单据)

#### 返回码
返回值 	|说明
:---   |     :---     
-3	| 部分成功
0	|成功
500	|内部服务器错误
40001|无API访问权限
46001|数据验证不通过
46002|无查询记录
	
其他返回值参照产品概述章节列明的通用返回值说明。
