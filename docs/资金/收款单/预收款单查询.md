### []()基本信息

- 用途说明：查询预收款单列表

- 请求方式：POST

- 请求地址：https://api.kingdee.com/jdyscm/newest/preReceipt/query

### []()请求参数

#### []()headers参数

Content-Type: application/x-www-form-urlencoded

[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)

#### []()params参数

| 参数名称 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| access_token | true | string | 云平台访问令牌 |
| dbId | true | number | 账套ID |

#### []()body参数

| 参数名称 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| buIds | false | string | 客户ids(多个逗号隔开) |
| beginDate | true | date | 单据日期-起始（yyyy-MM-dd ） |
| endDate | true | date | 单据日期-结束（yyyy-MM-dd ） |
| rows | false | int | 分页参数-每页记录数（默认10，最多500） |
| page | false | int | 分页参数-当前页码（默认1） |

#### []()请求示例

```
POST 
https://api.kingdee.com/jdyscm/receipt/preReceiptList?access_token=xxx&amp;dbId=xxx

Content-Type: application/x-www-form-urlencoded

HTTP 1.1
```

```

}
