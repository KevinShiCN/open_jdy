### 基本信息
- 用途说明：查询收款单列表
- 请求方式：POST
- 请求地址：https://api.kingdee.com/jdyscm/receipt/list
### 请求参数
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)
#### params参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID

#### body参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|filter |*false*	| object|过滤条件|
|-- number	|false	|string	|单据编码|
|--checkStatus|false|number|审核状态(0:未审核; 1:已审核; 2:所有; 默认查所有) |
|-- beginDate|false|date|单据日期-起始（yyyy-MM-dd ）|
|-- endDate|false|date|单据日期-结束（yyyy-MM-dd ）|
|-- updTimeBegin	|false	|datetime	|修改时间-起始（yyyy-MM-dd HH:mm:ss）|
|-- updTimeEnd	|false	|datetime	|修改时间-结束（yyyy-MM-dd HH:mm:ss）|
|-- employeeNumber|false|string|收款人编码|
|-- buIds|false|string|客户id(支持多个，用逗号隔开)|
|-- billDesc|false|string|单据备注|
|-- originNo|false|string|来源单号|
|--pageSize	|false	|int	|分页参数-每页记录数（默认10，最多500）|
|--page	|false	|int	|分页参数-当前页码（默认1）|
#### 请求示例
```
POST 
https://api.kingdee.com/jdyscm/receipt/list?access_token=xxx&amp;dbId=xxx

HTTP 1.1
```
```
BODY
{
	filter:
	{
		pageSize:10,
		page:1,
		originNo:10086
	}
}
```
