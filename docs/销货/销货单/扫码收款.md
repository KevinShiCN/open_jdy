### 基本信息
- 用途说明：现金收款
- 请求方式：POST
- 请求地址：https://api.kingdee.com/jdyscm/onlinePayOrder/scanqrPay

### 请求参数  
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)
#### params参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID

#### body参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
| authCode|*true*|string|授权码，支付宝/微信付款码
| amount|*true*|number|收款金额
| payId|*true*|number|支付通道id
| billId|*true*|number|单据id
| disAmount|*false*|number|优惠金额
| remark|*false*|string|收款备注
| createIp|*false*|string|ip

#### 请求示例
```
POST 
https://api.kingdee.com/jdyscm/onlinePayOrder/cashPay?access_token=1510557056dbfff1a4ab8bcf8a686679&amp;dbId=37436958539

HTTP 1.1
```
```
BODY
{
    authCode: 123*****************321,
    amount: 10.01,
    disAmount": 0.5,
    createIp: 0.0.0.0,
    payId": 123***456,
    billId: 321***412,
    remark: 鸭脖
}
```

--disAmount| number|优惠金额
--remark| string|收款备注
--receiptId| number|收款单id
