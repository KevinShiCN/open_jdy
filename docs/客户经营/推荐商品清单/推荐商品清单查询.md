```
#### params参数

| 参数名称 | 必选  | 类型  | 说明  |
| --- | --- | --- | --- |
| access_token | *true* | string | 云平台访问令牌 |
| dbId | *true* | number | 账套ID |

#### body参数

| 参数名称 | 必选  | 类型  | 说明  |

| relatedInvSkus| false| Array | 关联商品id  |
| page| false | string |页码 |
| rows| false | string | 页size|

```
