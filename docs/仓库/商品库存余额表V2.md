### 基本信息
- 用途说明：商品库存余额表，查询普通商品、辅助属性商品库存
- 请求方式：GET
- 请求地址：https://api.kingdee.com/jdyscm/v2/inventory/list

### 请求参数
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)

#### body参数

|参数名称 |  是否必选 |参数类型|参数说明  |
| :---    | :---   |     :---     | :---
|access_token	|true  |string	|云平台访问令牌
|dbId	|true	 |number	|账套ID
|number	|false	 |string	|商品编号(查辅助属性时必填，支持多编号查询，用逗号隔开,如001,002,003)
|location	|false	|string	|仓库编号
|skuNumber	|false	|string	|辅助属性编号
|showZero	|false	|number	|是否显示0库存商品(1：显示，0：不显示，默认0)
|isChecked	|false	|number	|是否显示无发生额的商品(1：显示，0：不显示，默认0)
|pageSize	|false	|int	|分页参数-每页记录数（默认10，500）
|page	|false|int	|分页参数-当前页码（默认1）
|isShowSku      |false  |number |是否显示辅助属性商品（1：显示，默认不显示）
|updTimeBegin      |false  |dateTime |更新时间-起始（yyyy-MM-dd HH:mm:ss）
|updTimeEnd      |false  |dateTime |更新时间-结束（yyyy-MM-dd HH:mm:ss）
#### 请求示例
```json
GET 
https://api.kingdee.com/jdyscm/inventory/list?access_token=xx&amp;dbId=xx&amp;number=003&amp;location=002&amp;showZero=1&amp;pageSize=5&amp;page=1&amp;skuNumber=001-2

HTTP 1.1 
```

### []()基本信息

- 用途说明：商品库存余额表，查询普通商品、辅助属性商品库存

- 请求方式：GET

- 请求地址：https://api.kingdee.com/jdyscm/v2/inventory/list

### []()请求参数

#### []()headers参数

[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)

#### []()body参数

| 参数名称 | 是否必选 | 参数类型 | 参数说明 |
| --- | --- | --- | --- |
| access_token | true | string | 云平台访问令牌 |
| dbId | true | number | 账套ID |
| number | false | string | 商品编号(查辅助属性时必填，支持多编号查询，用逗号隔开,如001,002,003) |
| location | false | string | 仓库编号 |
| skuNumber | false | string | 辅助属性编号 |
| showZero | false | number | 是否显示0库存商品(1：显示，0：不显示，默认0) |
| isChecked | false | number | 是否显示无发生额的商品(1：显示，0：不显示，默认0) |
| pageSize | false | int | 分页参数-每页记录数（默认10，500） |
| page | false | int | 分页参数-当前页码（默认1） |
| isShowSku | false | number | 是否显示辅助属性商品（1：显示，默认不显示） |
| updTimeBegin | false | dateTime | 更新时间-起始（yyyy-MM-dd HH:mm:ss） |
| updTimeEnd | false | dateTime | 更新时间-结束（yyyy-MM-dd HH:mm:ss） |

HTTP 1.1
```
