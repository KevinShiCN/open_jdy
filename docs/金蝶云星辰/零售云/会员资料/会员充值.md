| source | integer | true | 来源 ISV都填写9 |
| src_bill_id | string | true | 商城单据内码 |
| bill_date | string | false | 充值时间(不填以系统时间) |
| rule_detail_id | string | false | 优惠规则明细ID |
| rule_id | string | false | 优惠规则ID |
| settle_entries | List <SettleEntries> | false | 结算明细分录 |
| store_id | string | false | 门店ID |

**SettleEntries**

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| pay_type | string | true | 1、微信、2、支付宝、 90、会员卡(储值) |
| settle_amt | string | true | 收款金额 |
