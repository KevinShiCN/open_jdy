| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| rows | List <MaterialStockRow> | 数据行 |

**MaterialStockRow**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| inv_base_unit | string | 基本单位名称 |
| inv_im_qty | string | 即时库存数量 |
| inv_qty | string | 可用库存数量 |
| inv_stock | string | 仓库名称 |
| inv_stock_id | string | 仓库id |
| inv_stock_position | string | 仓位名称 |
