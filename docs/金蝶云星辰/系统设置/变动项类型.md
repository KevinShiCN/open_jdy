#### 变动后的使用部门对象 txtdept 

| 参数名称           | 类型      | 必选                    |说明                     |
| ------------------ | --------- | ------------------------ |------------------------ |
| dept_id | string|true| 变动后的部门id |
| percent | string|true|  部门占比（%）；范围(0,100]， 所有percent合计为100|

#### 变动后的原值对象 originalvalue 

| 参数名称           | 类型 | 必选     | 说明                     |
| ------------------ | --------- | ------------------------ |------------------------ |
| currency_id | string| false| 币别id，本位币可以不传 |
| exchange_rate | string|false|  币别汇率（数值），不赋值默认为1|
| amount_for | string|false|  原币金额（数值），不赋值默认为amount值，不会检测汇率和原币、本位币金额是否一致|
| amount | string| true| 本位币金额（数值，按系统本位币精度） |

#### 变动后的多科目对象 acctList （适用于txtexpenseacct、txtoppositeacct） 

| 参数名称           | 类型 | 必选     | 说明                     |
| ------------------ | --------- | ------------------------ |------------------------ |
| dept_id | string| true| 部门id；如果没有变动部门，取原部门id；否则取变动后的部门id|
| account_id| string|true| 折旧分配科目id|
| assist |  array<data.assist>  | false|  科目对应的核算项目，科目启用了必录核算项目是必录|
| percent | string|true|  同一部门的分配比例，范围(0,100]， 同一部门占比合计为100|

####  核算项目列表对象 （assist）
| 参数名称   | 类型   | 必选  | 说明        |
| ------- | ------ | ----- | ------- |
| type    |string| true   | 辅助核算类别(客户-bd_customer，供应商-bd_supplier， 商品-bd_material，部门-bd_department，职员-bd_employee，仓库-bd_store)， 辅助资料-bd_auxinfo |
| bd_aux_info_type_id| string| false | 辅助资料类别id，**当type为bd_auxinfo时必填**|
| id      | string| true | type类别下的基础资料id，例如type=“bd_customer”，id值为客户id |
