                    导航目录**    
### []()基本信息

- 用途说明：获取商品价格策略

- 请求方式：GET

- 请求地址：https://api.kingdee.com/jdy/v2/bd/material_price

### []()请求参数

#### []()headers参数

| 参数名称 | 参数示例 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| Content-Type | application/json | true | 固定传：application/json |
| X-Api-ClientID | 205022 | true | 应用id，创建指引 |
| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | 参考加密规则 |
| app-token | xxx | true | 产品账套级别的token，获取指引 |
| X-GW-Router-Addr | https://tf.jdy.com | true | IDC域名，获取appKey返回消息体内的domain字段 |

#### []()params参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| bill_date | string | true | 单据日期 |
| bill_type_id | string | true | 单据类型，参考<基础资料及业务单据类型> |
| currency_id | string | true | 币别id |
| exchange_rate | string | true | 汇率，应大于零，币别为本币时，汇率取1,汇率小于零时，修正为1 |
| material_id | string | true | 商品ID |
| qty | array | true | 商品数量 |
| unit_id | array | true | 单位ID |
| aux_prop_id | array | false | 商品辅助属性id（辅助商品必填） |
| customer_id | string | false | 客户ID（销售类单据必填） |
| supplier_id | string | false | 供应商ID（采购类单据必填） |

#### []()请求示例

```
curl --location --request  get 'https://api.kingdee.com/jdy/v2/bd/material_price?bill_date=2019-0-01&bill_type_id=sal_bill_order&currency_id=1&exchange_rate=1&material_id=1&qty=[]&unit_id=[]&aux_prop_id=[]&customer_id=1&supplier_id=1' \
--header 'Content-Type: application/json' \
--header 'X-Api-ClientID: 205022' \
--header 'X-Api-Auth-Version: 2.0' \
--header 'X-Api-TimeStamp: 1655775240000' \
--header 'X-Api-Nonce: 1655775240000' \
--header 'X-Api-SignHeaders: X-Api-TimeStamp,X-Api-Nonce' \
--header 'X-Api-Signature: xxx' \
--header 'app-token: xxx' \
--header 'X-GW-Router-Addr: https://tf.jdy.com' \
```

### []()响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| rows | List <PriceReply> | 数据行 |

**PriceReply**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| aux_prop_id | string | 辅助属性id |
| barcode | string | 条码 |
| dis_amount | number | 折扣额 |
| dis_count | number | 折扣 |
| dis_rate | number | 折扣率（预留） |
| price | number | 价格 |
| qty | string | 数量 |
| scheme_id | string | 价格方案id |
| unit_id | string | 单位id |

#### []()响应示例

```
{
	"data":{
		"rows":[
			{
				"aux_prop_id":"1",
				"barcode":"001",
				"dis_amount":1,

				"price":1,
				"qty":"1",
				"scheme_id":"1",
				"unit_id":"1"
			}
		]
	},
	"description":"success",
	"errcode":0
}
```
