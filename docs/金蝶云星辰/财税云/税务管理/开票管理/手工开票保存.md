### []()响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| data | string | 返回流水号 |
| error_code | string | 错误码 |
| message | string | 返回消息 |
| success | boolean | 是否成功(true/false) |
