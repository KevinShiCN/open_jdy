- 请求方式：POST
- 请求地址：https://api.kingdee.com/jdy/v2/fi/assistdetail_report

#### body参数
| 参数名称              | 参数类型              | 是否必填  | 参数说明                           |
| --- | --- | --- | --- |
| assist_list       | List <AssistList> | true  | 辅助核算参数列表                       
| end_period        | string            | true  | 结束年度期间（例：“202405”）             
| start_period      | string            | true  | 开始年度期间（例：“202405”）             
| account_id        | List <string>     | false | 科目id，与科目编码同时有值，以id为准           
| account_number    | List <string>     | false | 科目编码，与科目id同时有值,以id为准           
| currency_id       | string            | false | 币别id(0：综合本位币，-1：所有币别，默认0)      
| explanation       | string            | false | 摘要                             
| hide_balance_zero | string            | false | 余额为0不显示（1：是，0：否，默认0）           
| hide_no_happen    | integer           | false | 无发生额且余额为0不显示（1：是，0：否，默认0）      
| hide_total        | string            | false | 发生额为0不显示本期合计、本年累计（1：是，0：否，默认1） 
| page              | string            | false | 当前页（默认1，如page页超出最大页，则取最后一页）    
| page_size         | string            | false | 每页显示条数，大于0，不传该字段表示不分页          
| price_scale       | string            | false | 单价精度（默认4位）                     
| qty_scale         | string            | false | 数量精度（默认2位）                     

**AssistList**
| 参数名称   | 参数类型   | 是否必填 | 参数说明                                         |
| --- | --- | --- | --- |
| id     | string | true | 过滤具体的辅助核算id，与number同时有值，以id为准；2）与number二选一必录 
| number | string | true | 过滤具体的辅助核算编码：1）与id同时有值，以id为准；2）与id二选一必录       
| type   | string | true | 辅助核算类别id，通过【辅助核算类别列表】接口assist_type_list获取
