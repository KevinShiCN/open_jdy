| 参数名称 | 参数类型 | 参数说明 |

| 参数名称 | 参数类型 | 参数说明 |

| creator_number | string | 制单人编码 |
| entry | List <Entry> | 资产分录 |
| id | string | 变动单id |
| number | string | 单据编码 |
| period | string | 期间 |
| voucher_id | string | 凭证id，id值=0标识没有生成凭证，生成凭证不能修改和删除 |

| 参数名称 | 参数类型 | 参数说明 |

| asset_number | string | 资产编码 |
| mode_id | string | 变动方式 |
| mode_name | string | 变动方式名称 |
| mode_number | string | 变动方式编码 |
| seq | integer | 序号 |
| sub_entry | List <SubEntry> | 变动分录 |

**SubEntry**

| 参数名称 | 参数类型 | 参数说明 |

| new_val | string | 变动后值 |
