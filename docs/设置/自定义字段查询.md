### 基本信息
- 用途说明：查询单据自定义字段
- 请求方式：POST
- 请求地址：https://api.kingdee.com/jdyscm/udf/list

### 请求参数
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)

#### params参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID

#### body参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|filter |*false*	| object|过滤条件
|--code	|*true*	|string	|单据类型（sale：销货单（销货订单），saleback：销货退货单，pur：购货单（购货订单），purback：购货订单）

#### 请求示例
```
POST 
https://api.kingdee.com/jdyscm/udf/list?access_token=xxx&amp;dbId=xxx

HTTP 1.1
```
```
BODY
{
	filter:
	{
		code:sale
	}
}
```

### 返回参数
返回参数 	|字段类型 	|字段说明 
:---   |     :---     | :---
code	|string	| 返回码
msg	|string 	|结果或错误描述
items|array|列表
--index | number |顺序号（【1~5】，1指示第一个自定义字段）
--name | string |名称
--type | string |类型（text：文本，radio：单选，date：日期）
--value | string |单选值
page|number|当前页码
totalsize|number|当前返回总记录数
records|number|总记录数
totalPages|number|总页数

#### 返回码
返回值 	|说明
:---   |     :---     
0	|成功
500	|内部服务器错误
40001|无API访问权限
46001|数据验证不通过
46002|无查询记录
	
其他返回值参照产品概述章节列明的通用返回值说明。

                3,

            ]

            index: 4,

        }
    ]
}
```
```
