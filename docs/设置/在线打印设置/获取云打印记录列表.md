```
## **获取云打印记录列表**

### 基本信息

- 用途说明：查询云打印记录信息
- 请求地址：https://api.kingdee.com/jdyscm/v2/onlinePrint/getPrintLogList
- 请求方式：POST

### 请求参数

#### headers参数

[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)

#### params参数

| 参数名称 | 必选  | 类型  | 说明  |
| --- | --- | --- | --- |
| access_token | *true* | string | 云平台访问令牌 |
| dbId | *true* | number | 账套ID |

#### body参数

| 参数名称 | 必选  | 类型  | 说明  |
| --- | --- | --- | --- |
| page | *false* | number | 页码  |
| rows | *false* | number | 每页行数 |
| search | *false* | string | 模糊搜索（支持搜索 任务编号、打印机编号、打印机名称、单据编号、打印人） |
| beginDate | *false* | array | 开始日期 |
| endDate | *false* | array | 截止日期 |
| statusList | *false* | array | 打印记录状态列表（0 正在打印 1 打印超时 2 打印成功 3 打印失败） |
| businessCodeList | *false* | boolean | 业务编码列表（详见末尾附表） |

```
POST

### []()基本信息

- 用途说明：查询云打印记录信息

- 请求地址：https://api.kingdee.com/jdyscm/v2/onlinePrint/getPrintLogList

- 请求方式：POST

### []()请求参数

#### []()headers参数

[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)

#### []()params参数

| 参数名称 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| access_token | true | string | 云平台访问令牌 |
| dbId | true | number | 账套ID |

#### []()body参数

| 参数名称 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| page | false | number | 页码 |
