## **获取模板列表**

### 基本信息

- 用途说明：查询在线打印模板
- 请求地址：https://api.kingdee.com/jdyscm/v2/onlinePrint/queryTemplates
- 请求方式：GET

### 请求参数

#### headers参数

[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)

#### params参数

| 参数名称 | 必选  | 类型  | 说明  |
| --- | --- | --- | --- |
| access_token | *true* | string | 云平台访问令牌 |
| dbId | *true* | number | 账套ID |
| businessCode | *false* | string | 业务编码 |
| templateType | *false* | number | 模板类型 1-单据 2-小票 3-标签 |
| templateSource | *false* | number | 模板来源 0-在线打印（默认） 1-在线套打 |

#### 请求示例

```
GET
https://api.kingdee.com/jdyscm/v2/onlinePrint/queryTemplates?access_token=1510557056dbfff1a4ab8bcf8a686679&dbId=37436958539&businessCode=SALES&templateType=1&templateSource=0

HTTP 1.1
```

| 返回参数 | 字段类型 | 字段说明 |
| --- | --- | --- |
| code | string | 返回码 |
| msg | string | 结果或错误描述 |
| data | array | 模板列表 |
| –id | string | 模板id |
| –businessCode | string | 业务编码 |
| –name | string | 模板名称 |
| –isDefault | boolean | 是否默认模板 |
| –templateType | number | 模板类型 1-单据 2-小票 3-标签 |
| –templateSource | number | 模板来源 0-在线打印（默认） 1-在线套打 |
