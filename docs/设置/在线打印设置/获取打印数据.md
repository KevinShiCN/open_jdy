```
## **获取打印数据**

### 基本信息

- 用途说明：获取打印数据（PDF、指令、图片）
- 请求地址：https://api.kingdee.com/jdyscm/v2/onlinePrint/getPrintContent
- 请求方式：POST

### 请求参数

#### headers参数

[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)

#### params参数

| 参数名称 | 必选  | 类型  | 说明  |
| --- | --- | --- | --- |
| access_token | *true* | string | 云平台访问令牌 |
| dbId | *true* | number | 账套ID |

#### body参数

| 参数名称 | 必选  | 类型  | 说明  |
| --- | --- | --- | --- |
| businessCode | *true* | string | 业务编码 |
| templateId | *true* | number | 打印模板id |
| templateSource | *false* | number | 打印模板来源（0-在线打印 1-在线专业套打  不传默认在线打印） |
| ids | *true* | array | 单据ID列表 |
| preferenceDataType | *false* | string | 打印数据类型偏好（单据：PDF、ORDER；小票：仅支持ECS/POS指令，此参数无效；标签：PICTURE、ZPL、TSPL） |

#### 请求示例

```
POST
https://api.kingdee.com/jdyscm/v2/onlinePrint/getPrintContent?access_token=1510557056dbfff1a4ab8bcf8a686679&dbId=37436958539

HTTP 1.1
```

```
BODY
{
    "businessCode": "SALES",
    "templateId": "1",
    "templateSource": 0,
    "ids": [795796059110788],
    "preferenceDataType": "PDF"
}
```

| 返回参数 | 字段类型 | 字段说明 |
| --- | --- | --- |
| code | string | 返回码 |
| msg | string | 结果或错误描述 |
| data | object | 成功时返回 “创建云打印任务成功” |
| - data | string | 打印数据 |
| - dataType | string | 数据类型 |

#### 响应示例

```
{
    "code": 0,
    "msg": "成功",
    "data": {
        "data": "https://fdfs-dev-1307755952.cos.ap-beijing.myqcloud.com/printtmp/797669169171/cac8624a6ee0431fa5dd915a00f156d0.pdf",
        "dataType": "PDF"
    }
}
```

#### 返回码

| 返回值 | 说明  |
| --- | --- |
| 0   | 成功  |
| 500 | 内部服务器错误 |
| 40001 | 无API访问权限 |
| 46001 | 数据验证不通过 |
| 46002 | 无查询记录 |
| 48010 | 无相关权限 |

其他返回值参照产品概述章节列明的通用返回值说明。
```
