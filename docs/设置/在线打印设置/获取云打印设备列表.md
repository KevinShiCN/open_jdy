## **获取云打印设备列表**

### 基本信息

- 用途说明：查询云打印设备信息
- 请求地址：https://api.kingdee.com/jdyscm/v2/onlinePrint/getPrinterList
- 请求方式：POST

### 请求参数

#### headers参数

[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)

#### params参数

| 参数名称 | 必选  | 类型  | 说明  |
| --- | --- | --- | --- |
| access_token | *true* | string | 云平台访问令牌 |
| dbId | *true* | number | 账套ID |

#### body参数

| 参数名称 | 必选  | 类型  | 说明  |
| --- | --- | --- | --- |
| page | *false* | number | 页码  |
| rows | *false* | number | 每页行数 |
| printerNumber | *false* | string | 打印设备编号 |
| companyList | *false* | array | 厂家编号列表（0-得实，1-链科） |
| locationIds | *false* | array | 绑定仓库ID列表 |
| statusList | *false* | array | 设备状态列表（0 正常 1 缺纸 2 离线 3 未连接）该参数不为空时默认查询设备实时状态，此时withStatus参数无效 |
| withStatus | *false* | boolean | 是否查询设备实时状态（查询有耗时） |

#### 请求示例

```
POST
https://api.kingdee.com/jdyscm/v2/onlinePrint/getPrinterList?access_token=1510557056dbfff1a4ab8bcf8a686679&dbId=37436958539

HTTP 1.1
```

    "page": 1,
    "rows": 10,

    "statusList": [0],
