### 基本信息
- 用途说明：获取精斗云云进销存系统的系统参数设置
- 请求方式：POST
- 请求地址：https://api.kingdee.com/jdyscm/profile/query

### 请求参数
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)

#### params参数
| 参数名称         | 必选   | 类型     | 说明      |
| ------------ | ---- | ------ | ------- |
| access_token | true | string | 云平台访问令牌 |
| dbId         | true | number | 账套ID    |

#### 请求示例

```
GET
https://api.kingdee.com/jdyscm/profile/query?access_token=1510557056dbfff1a4ab8bcf8a686679&amp;dbId=37436958539
HTTP 1.1
```

### 返回参数

| 返回参数                 | 字段类型    | 字段说明               |
| -------------------- | ------- | ------------------ |
| code                 | string  | 返回码                |
| msg                  | string  | 结果或错误描述            |
| data                 | object  | 参数数据对象             |
| -companyName         | string  | 公司名称               |
| -companyAddr         | string  | 公司地址               |
| -started             | boolean | 是否启用               |
| -startDate           | date    | 启用日期               |
| -naturalCurrency     | string  | 本位币编码              |
| -qtyPrecision        | number  | 数量小数位              |
| -pricePrecision      | number  | 价格小数位              |
| -storeCheckEnable    | boolean | 是否启用负库存检查          |
| -billApprovalEnabled | boolean | 是否启用单据审核           |
| -invPriceIsTax       | boolean | 商品价格是否含税           |
| -isAlowAlterBill     | boolean | 是否启用销售类单据不允许修改单据日期 |

#### 返回码

| 返回值   | 说明       |
| ----- | -------- |
| 0     | 成功       |
| 500   | 内部服务器错误  |
| 40001 | 无API访问权限 |
| 46001 | 数据验证不通过  |

其他返回值参照产品概述章节列明的通用返回值说明。
