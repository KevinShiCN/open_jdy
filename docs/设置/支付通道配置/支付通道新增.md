### 基本信息

- 请求地址：https://api.kingdee.com/jdyscm/onlinePayInfo/add

### 请求参数  
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)
#### params参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID

#### body参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|items |*true*	| object|
|-- payAcctNumber|*true*|string|结算账户编号
|-- payMethodName|*true*|string|结算方式名称
|-- name|*true*|string|账户标题
|-- order|*true*|number|排序值，min:1，max:1000
|-- type|*true*|number|支付类型，1：现金，2：聚合支付，3：二维码付款
|-- isDelete|*true*|number|启用/禁用标记；0：启用、1：禁用
|-- payType|*false*|array|支付渠道；1：通联支付,99：ISV聚合支付
|-- customNo|*false*|string|商户号
|-- appId|*false*|string|APPID
|-- privateKey|*false*|string|商户秘钥
|-- publicKey|*false*|string|通道公钥
|-- url|*false*|string|服务URL
|-- chnlCode|*false*|string|通道代码

| 返回参数 | 字段类型 | 字段说明 |
| --- | --- | --- |
| code | string | 返回码 |
| msg | string | 结果或错误描述 |
| data | array | 单据列表 |
| –fid | number | ID |
| –name | string | 账户标题 |
| –payType | numer | 支付渠道；1：通联支付 |
| –payMethodName | string | 结算方式名称 |
| –payAcctName | string | 结算账户名称 |
| –isDelete | number | 启用/禁用标记；0：启用、1：禁用 |

| –order | number | 排序值 |
| –privateKey | string | 商户公钥 |
| –publickey | string | 商户私钥 |
| –appid | number | 商户号 |
| –url | string | 服务URL |
| –chnlCode | string | 通道代码 |
