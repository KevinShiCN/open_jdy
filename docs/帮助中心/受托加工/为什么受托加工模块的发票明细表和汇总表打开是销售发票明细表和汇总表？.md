# 为什么受托加工模块的发票明细表和汇总表打开是销售发票明细表和汇总表？

**1、问题描述**

为什么受托加工模块的发票明细表和汇总表打开是销售发票明细表和汇总表？

2、场景介绍

受托加工业务实际上也是一种销售业务，向委托方销售加工服务，并收取加工费，所以将受托加工发票和销售发票放在同一张报表中进行统计，便于企业财务人员和管理者查询和分析。
