# 受托加工订单未出库,为什么显示全部执行了?

【问题描述】

受托加工订单未出库,为什么显示全部执行了?

![](https://vip.kingdee.com/download/0109310f16a6882442a79258a2a8811170c1.png)

【解释说明】

前提:系统设置-系统参数-进销存参数中,库存更新方式设置为审核时更新库存。

![](https://vip.kingdee.com/download/01095ebead0ce48a42ae9c551f505733fc15.png)

当受托加工订单生成了受托出库单后,如果该出库单尚未审核,那么受托加工订单的状态将变为“已执行、未出库”。具体操作步骤和状态变化可参考下方的动图演示。
