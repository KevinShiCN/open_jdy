# 客供材料汇总表

> 产品：金蝶云·星辰 | 分类：受托加工 | 类型：功能介绍, 新特性, 快速入门
> 作者：Lisa798 | 阅读：526 | 有用：14

客供材料汇总表

![上传图片](https://vip.kingdee.com/download/010049ca3b9111b14edca7e91608e196d633.png)

**应用场景及价值**：

仓管员可通过不同的汇总依据，多维度掌握客供材料入库及退货汇总情况。

**取数说明：**

客供材料汇总表是按所选的时间段按所选的汇总依据来统计过滤期间内所有的客供材料入库/退货、生产领料/退料、委外发料/退料汇总情况。

**权限说明：**该报表受数据权限控制，即若用户无某物料、客户、部门、仓库、职员的数据权限，则对应的客供材料入库和退货数据在该报表中无法查询

**参数说明：**

1) 显示辅助属性：勾选该参数后汇总表中方可显示辅助属性相关字段；且汇总依据的维度细化为【物料编码+辅助属性】维度

**其他说明**：双击客供材料汇总表的任意数据行，可跳转至对应的客供材料明细数据
