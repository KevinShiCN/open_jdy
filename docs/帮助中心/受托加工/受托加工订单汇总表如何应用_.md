# 受托加工订单汇总表如何应用?

> 产品：金蝶云·星辰 | 分类：受托加工 | 类型：常见问题
> 作者：云社区用户13pG7392 | 阅读：97 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/594331750126113280)

【业务背景】

业务员或管理员可通过不同的汇总依据,多维度掌握订单的总体情况。

【解释说明】

取数说明:

受托加工订单汇总表是按所选的时间段按所选的汇总依据来统计过滤期间内所有的受托加工订单汇总情况。

权限说明:

该报表受数据权限控制,即若用户无某物料、客户、部门、仓库、职员的数据权限,则对应的受托加工订单数据在该报表中无法查询

参数说明:

显示辅助属性:勾选该参数后汇总表中方可显示辅助属性相关字段;且汇总依据的维度细化为【商品编码+辅助属性】维度
