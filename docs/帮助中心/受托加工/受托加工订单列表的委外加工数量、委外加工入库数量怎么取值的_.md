# 受托加工订单列表的委外加工数量、委外加工入库数量怎么取值的?

> 产品：金蝶云·星辰 | 分类：受托加工 | 类型：常见问题
> 作者：Schatz | 阅读：97 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/634198446756098048)

【问题描述】

受托加工订单列表的委外加工数量、委外加工入库数量怎么取值的?

![](https://vip.kingdee.com/download/0109255523af3ca7488494c780f076bd7651.png)

【解释说明】

委外加工数量:取自关联了本张受托加工订单为源单的委外加工单的数量;

委外加工入库数量:取自关联了本张受托加工订单为源单的委外加工单的已执行入库数量。

![](https://vip.kingdee.com/download/0109fbe5a16753f34160a76893ed13ece81d.png)

【温馨提示】

委外加工单生成了委外入库单,无论入库单是否审核,其数量都会反写到委外加工单的已执行入库数量中。

如下图,委外加工单生成了委外入库单,数量1,单据未审核。

![](https://vip.kingdee.com/download/01090df39bccc5484d068b000c71da908222.png)

委外加工单上的已执行入库数量反写为1。

![](https://vip.kingdee.com/download/010948277c569a0447d7b6bf4532281e9066.png)
