**1 产品概述**

### **1.1 产品介绍**

企业受托加工业务中，相关人员需要通过移动端快捷新增、审核、查询受托加工业务相关单据，实现受托加工业务在移动端与 Web 端的数据实时同步。点击对应功能图标可查看单据列表和详情，进行新增单据、审核、下推下游单据、编辑、删除、打印等一系列操作，满足受托加工业务移动办公需求。

### **1.2 适用版本**

金蝶云·星辰旗舰版

### **1.3 业务场景**

1）场景一：客户将用于加工的材料送至企业，仓库人员在现场验收时，不便使用电脑登录系统，希望通过移动端快速录入客供材料的种类、数量等信息，生成客供材料入库单；仓库管理人员在巡查仓库时，可通过移动端查看客供材料入库单，了解客供材料的入库情况；客供材料入库单需要部门负责人审核，负责人不在办公室时，可通过移动端便捷完成审核操作。

2）场景二：若客供材料存在质量问题或数量不符等情况，需要将材料退回给客户，仓库人员在现场处理时，可通过移动端录入相关信息，生成客供材料退货单，准确记录退货的材料信息

3）场景三：企业完成受托加工产品的生产后，需要将产品交付给客户，车间人员或仓库人员在发货现场，可通过移动端扫描产品条码等方式，快速生成受托加工出库单，记录出库产品的信息；运输人员在出发前，通过移动端查看受托加工出库单，核对所运产品是否准确；受托加工出库单需要业务人员审核，业务人员在外与客户沟通时，可通过移动端及时审核，确保产品顺利出库。​

4）场景四：客户收到受托加工产品后，发现产品存在质量问题或其他不符合约定的情况，需要将产品退回企业，企业接收人员在现场可通过移动端录入退货产品的信息，生成受托加工退货单；车间人员需要了解退货产品的情况，在生产现场通过移动端查看受托加工退货单，以便安排返工等后续处理；受托加工退货单需要生产主管审核，主管在车间巡查时可通过移动端完成审核，及时推进退货产品的处理工作。
