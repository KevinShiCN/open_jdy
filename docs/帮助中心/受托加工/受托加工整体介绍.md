1 产品概述

### 1.1 产品介绍

受托加工模块用于处理企业的受托加工业务。受托加工结合生产管理实现全流程数字管理，提供准确的数据和成本核算，解决企业来料加工管理难题，助力企业提升竞争力。

### 1.2 适用版本

星辰旗舰版

4  功能清单

（主要的功能清单，可链接到具体的用户手册内容）

| 生产管理 | 二级菜单 | 三级菜单 | 功能说明 |
| --- | --- | --- | --- |
| 生产单据 | 受托加工订单 | 购销双方共同签署的、确认购销活动的标志，是详细记录企业物资的循环流动轨迹、累积企业管理决策所需要的经营运作信息的关键 |  |
| 客供材料入库单 | 记录客户提供的加工所需材料信息 |  |  |
| 客供材料退货单 | 记录退回给客户的客供材料信息 |  |  |
| 受托加工出库单 | 主要应用于受托加工订单下单后发出货物的场景 |  |  |
| 受托加工退货单 | 记录受托加工的产品出库后被退回的信息 |  |  |
| 受托加工发票 | 当企业确定了实际的加工金额或退货金额后，向委托方开出发票或收据时，需要在系统中录入受托加工发票 |  |  |
| 受托加工报表 | 客供材料明细表 | 仓管员可通过客供材料明细表掌握不同客户、仓库、部门、物料等的客供材料入库及退货明细的情况 |  |
| 客供材料汇总表 | 仓管员可通过不同的汇总依据，多维度掌握客供材料入库及退货汇总情况 |  |  |
| 受托加工订单跟踪表 | 实时跟踪一段时期内所有受托加工订单的详细信息 |  |  |
| 受托加工订单汇总表 | 业务员或管理员可通过不同的汇总依据，多维度掌握受托加工订单的总体情况 |  |  |
| 受托加工出库明细表 | 业务员或管理者可通过受托加工出库明细表掌握受托加工物资的出库明细情况 |  |  |
| 受托加工出库汇总表 | 务员或管理员可通过不同的汇总依据，多维度掌握出库和退货的总体情况 |  |  |
| 销售发票明细表 | 统计一段时期内商品的销售发票及销售退货发票的详细情况 |  |  |
| 销售发票汇总表 | 统计一段时期内商品的销售发票及销售退货发票的汇总情况 |  |  |

注：备注了（旗舰版）字样的为仅旗舰版支持的单据。
