# 受托加工退货单的“出库成本”怎么计算?

> 产品：金蝶云·星辰 | 分类：受托加工 | 类型：常见问题
> 作者：Schatz | 阅读：114 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/626584419297541120)

【问题描述】

受托加工退货单的“出库成本”怎么计算?

【解释说明】

受托加工退货单的“出库成本”是由库存管理-出库成本核算计算后返填回单据的。

![](https://vip.kingdee.com/download/0109508c721efe90413892bfff89b6b0216d.png)

计算规则:

**有关联源单**:若受托加工退货单是关联受托加工出库单生成的,则成本取源单的出库成本。

**未关联源单**:若受托加工退货单无源单,则根据系统设置-系统参数-存货核算参数中的“无源单的销售退货单成本来源”来取出库成本。

![](https://vip.kingdee.com/download/0109d4c838433f854e7bb1477297eb57a067.png)
