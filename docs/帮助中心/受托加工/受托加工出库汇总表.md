# 受托加工出库汇总表

> 产品：金蝶云·星辰 | 分类：受托加工 | 类型：快速入门, 功能介绍, 新特性
> 作者：蒋章龙 | 阅读：385 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/466666415240749312)

应用场景及价值：

业务员或管理员可通过不同的汇总依据，多维度掌握出库和退货的总体情况。

**适用版本**

- 金蝶云·星辰旗舰版

![](https://vip.kingdee.com/download/010063c6c4e058444e3cb61c382c39a3f344.png)

取数说明：

受托加工出库汇总表是按所选的时间段按所选的汇总依据来统计过滤期间内所有受托加工出库和退货的汇总情况。

权限说明：

该报表受数据权限控制，即若用户无某物料、客户、部门、仓库、职员的数据权限，则对应的受托加工出库和退货数据在该报表中无法查询。

参数说明：

显示商品明细：勾选该参数后汇总表中方可显示商品明细相关字段；且汇总依据的维度按商品进行细化。例如：勾选前按客户汇总，勾选后按客户+商品编码进行汇总；

显示辅助属性：勾选该参数后汇总表中进一步显示辅助属性相关字段；且汇总依据的维度进一步细化为【商品编码+辅助属性】维度。
