# 赠品如何开票？

> 产品：金蝶云·星辰 | 分类：税务管理 | 类型：功能介绍, 进阶技巧, 精华汇总

> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/326755094346121984)

问题描述：

解决方案：

销售出库单上有赠品，客户希望可以把赠品在单据上体现出来，要怎么设置？

1、赠品在销售出库时没有录入单价，无法直接开具。如果要开具赠品，必须要在出库单上把赠品做100%折扣（即：出库单上，赠品的单价按开票单价填写，并设置100%折扣，实际出库金额为0）。

2、开票参数去掉【以销售单据折扣后金额开票】，就可以给赠品开票了
