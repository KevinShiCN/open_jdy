# 在销售开票的时候提示:“商品(或费用项目)金额不能为0!”如何处理?

> 产品：金蝶云·星辰 | 分类：税务管理 | 类型：常见问题
> 作者：_Leo | 阅读：971 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/396106049570270464)

【问题描述】

在销售开票界面,对销售出库单开票提示“商品(或费用项目)金额不能为0!”,如图:

![](https://vip.kingdee.com/download/01097502e35c8591475ea9ea7fdf4ef003fd.png)

【解释说明】

主要原因是在开票的时候,将开票参数-“以销售单据折扣后金额开票”勾选了,导致无法正常开票。因为税盘要求单价不能为0,价税合计的金额可以为0,因此关闭此功能键即可正常开票。
