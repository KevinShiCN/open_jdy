# 发票云客户端收票操作指引（数电客户版本）

> 作者：Daisy_ | 阅读：4187 | 有用：9

一、系统参数选择：

   系统设置-税务参数-取票工具选择：发票云

![上传图片](https://vip.kingdee.com/download/0100ec24f06baaa046acbdfd89e909588c27.png)

二、企业认证

   系统设置-税务参数-点击企业认证，输入客户登录电子税务局的账号及密码进行企业认证

![上传图片](https://vip.kingdee.com/download/0100f799145dbf9a4bc19c13c0bce232fab0.png)

![上传图片](https://vip.kingdee.com/download/0100a611bf13c8f549c2b559e5675ed2eb4a.png)

![上传图片](https://vip.kingdee.com/download/0100194420089fdf4597903acb612fd29320.png)

√ 认证成功会显示已认证的状态

![上传图片](https://vip.kingdee.com/download/0100660ece24482c4b32947cc91e41bd6d08.png)

**三、数电服务开通及数电取票账号维护**

       系统设置-系统参数-税务参数-数电管理，启用【开通数电服务】按钮，点击【升级数电】按钮，维护【账号管理】有收票权限人员的账号信息并进行保存（身份为办税员、财务负责人、法定代表人均为收票权限，需要和电子税局收票人员保持一致）；

**四、取票操作**

       1、依次点击税务管理-发票管理-智能取票下拉箭头，进入取票设置界面

![上传图片](https://vip.kingdee.com/download/0100d2d658711b88481a93a07db84976bbeb.png)

![上传图片](https://vip.kingdee.com/download/01006d6bf4ad950f477eaa8f1e54da3c207d.png)

       2、选择无盘取票

       取票方式选择：无盘取票，点击【确定】按钮，下次取票时即可保存云化取票方式

![上传图片](https://vip.kingdee.com/download/01008eb67e28d2334dd78ff09ef8145725e2.png)

       3、点击智能取票，选择需要采集的发票日期及类型等，点击智能取票按钮，系统会校验电子税局账号是否登录，如未登录则弹出电子税局登录弹窗，用户填写短信验证码登录，登录完毕后再次点击智能取票即可完成发票采集

![上传图片](https://vip.kingdee.com/download/0100dbd503356d3442978c00c1828e0118d7.png)

![上传图片](https://vip.kingdee.com/download/01009c6637cf1e3b491daccb1bed39556f5a.png)
