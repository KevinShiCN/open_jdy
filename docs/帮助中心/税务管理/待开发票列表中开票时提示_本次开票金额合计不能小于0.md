# 待开发票列表中开票时提示:本次开票金额合计不能小于0

> 产品：金蝶云·星辰 | 分类：税务管理 | 类型：常见问题
> 作者：酸梅汤啦啦 | 阅读：103 | 有用：0

【问题描述】

待开发票列表中开票时提示:本次开票金额合计不能小于0

【解决方案】

**税务管理系统中发票开具的金额合计只能大于0。**

如果有退货单需要开票,需和销售出库单一起合并开票即可,或者有赠品需要开票可按折扣录入后再开票,具体可参考如下知识操作:

[销售出库单和销售退货单一出一入不需要开票的,可以设置不显示在待开发票列表这里吗](https://vip.kingdee.com/knowledge/585553181543543296?productLineId=35&isKnowledge=2&lang=zh-CN)
