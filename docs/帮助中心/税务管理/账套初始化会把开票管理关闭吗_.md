# 账套初始化会把开票管理关闭吗?

> 产品：金蝶云·星辰 | 分类：税务管理 | 类型：常见问题
> 作者：云社区用户8x80607 | 阅读：56 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/776251470844275200)

【问题描述】

账套初始化会把开票管理关闭吗?

【解释说明】

初始化账套开票管理按钮不影响,还是打开状态可以正常使用。如当前账套由于业务原因需要初始化更换企业公司做账,可将该账套初始化重新创建即可,开票管理可在当前账套继续使用。

点击右上角账套名称,可查看开票管理状态
