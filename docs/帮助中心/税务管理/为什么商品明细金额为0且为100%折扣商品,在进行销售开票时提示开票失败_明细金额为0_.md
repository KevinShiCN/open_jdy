# 为什么商品明细金额为0且为100%折扣商品,在进行销售开票时提示开票失败:明细金额为0?

> 产品：金蝶云·星辰 | 分类：税务管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：305 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/537436485507781376)

【问题描述】

为什么商品明细金额为0且为100%折扣商品,在进行销售开票时提示开票失败:明细金额为0?

【解决方案】

点击【税务管理】-【待开发票列表】-【参数】,取消勾选[以单据折扣后金额开票]参数,确定。
