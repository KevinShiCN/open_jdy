# 经计算,单据编号为xxxx的商品二极管数量<0,不能开票;建议与为正数的相同商品行进行合并开票。

> 产品：金蝶云·星辰 | 分类：税务管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：20 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/804154676412001536)

【问题描述】

销售出库与销售退货合并开票提示:经计算,单据编号为XXXX的商品二极管数量<0,不能开票;建议与为正数的相同商品行进行合并开票。

![](https://vip.kingdee.com/download/01091c0382dec25c451684d730e1da6a05f0.png)

【解释说明】

销售出库与销售退货合并开票时,需要勾选:单据的纳税人名称、税号、地址电话、开户行及账号一致时合并、商品名称、单位、税率。即可合并开票。

![](https://vip.kingdee.com/download/01094f05c99a4c6343b3a1cbe126b4b26f5c.png)开具效果:
请核对开票内容:第 1 张 / 共 2 张,一张发票金额为0,一张正常的发票。

![](https://vip.kingdee.com/download/0109998ceb2d60e448acbfd17caf72f53c96.png)
