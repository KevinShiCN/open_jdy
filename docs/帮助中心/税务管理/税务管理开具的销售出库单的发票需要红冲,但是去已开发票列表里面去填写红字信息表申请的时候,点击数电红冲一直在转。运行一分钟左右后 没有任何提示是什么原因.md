# 税务管理开具的销售出库单的发票需要红冲,但是去已开发票列表里面去填写红字信息表申请的时候,点击数电红冲一直在转。运行一分钟左右后 没有任何提示是什么原因

> 作者：云社区用户6w624200 | 阅读：56 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/680948846654057472)

【问题描述】

税务管理开具的销售出库单的发票需要红冲,但是去已开发票列表里面去填写红字信息表申请的时候,点击数电红冲一直在转。运行一分钟左右后 没有任何提示是什么原因

![](https://vip.kingdee.com/download/0109bbc2ad63b6214426858f2ab908c0a277.png)

【解释说明】

数电红冲一直转是由于,是发票云发票云接口返回内容为空。这种一般是等一等,

会正常获取红字信息表,如果还不行提单后台处理下。
