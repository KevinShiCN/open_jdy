# 什么场景下需要维护“票据类型”？

> 产品：金蝶云·星辰 | 分类：税务管理 | 类型：常见问题, 进阶技巧
> 作者：简天 | 阅读：470 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/23538)

问题描述：

解决方案：

1、票据能参与增值税抵扣：除增值税发票抵扣税金外，目前越来越多的票据能参与抵扣，比如：员工的火车票、飞机票等，每种票据抵扣税额的计算方法和税率有一定的差异。这时就需要定义该票据抵扣税率及计算方式。

2、需要在发票管理中生成凭证的票据。
