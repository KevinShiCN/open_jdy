# 海关专用缴款书功能如何使用?

【问题描述】

海关专用缴款书功能如何使用?

![](https://vip.kingdee.com/download/0109213b6af1ad244b6a916975bac150dae9.png)
【解释说明】

**海关专用缴款书是指海关出具给进口人缴纳关税、进口环节增值税、消费税等税费的一种凭证**。这种缴款书在国际贸易中具有重要作用,是进口人完成进口货物税费缴纳的重要证明。

在星辰系统中,通过智能取票可以把海关专用缴款书取到系统中,在【税务管理】-【增值税管理】-【海关专用缴款书】界面可以查询到

支持按填发日期、认证状态、是否生成凭证查询

![](https://vip.kingdee.com/download/0109d8718b1f8cc140bfad591fe9fa3fc03c.png)
支持删除、标记认证、标记进项转出、补充生成凭证用的辅助核算

![](https://vip.kingdee.com/download/01091df45334df534c548d75e8a0407b643f.png)
