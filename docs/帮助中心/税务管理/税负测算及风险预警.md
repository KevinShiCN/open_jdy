# 税负测算及风险预警

> 产品：金蝶云·星辰 | 分类：税务管理 | 类型：快速入门, 功能介绍
> 作者：ylim | 阅读：1612 | 有用：10
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/630437443266955264)

通过系统中的发票数据可以进行税负测算、增值税风险提示等。

| 步骤 | 具体说明 | 学习链接 |
| --- | --- | --- |
| 1.认证方案 | 按预期税负率和波动范围,推荐适合的发票认证并计算税负。   | 点击学习 |
| 2.税负测算 | 按增值税销项发票,及认证的进项发票、其他可抵扣发票测算月度的增值税税负率。除发票数据外还包含一些调整项,比如:不开票收入的税项、加计抵扣的税项、以及各种减免税等。   | 点击学习 |
| 3.增值税风险报告 | 以财务、业务、发票、核算、报表、申报等数据为依据,综合评估纳税人的增值税税务风险,为企业的税务筹划提供参考。   | 点击学习 |
| 4.进销项对比表（仅货物） | 进销项对比表可从发票角度查看企业货物的销售与采购对比情况,快速查询开具的销项发票是否有对应的进项发票,以控制税务风险。   | 点击学习 |
| 5.增值税预申报表 | 增值税一般纳税人申报表编制,为企业提供申报参考；为一键报税提供数据准备   | 点击学习 |
