# 为什么维护了开票员或者办税员的电子税局账号，在开票或者取票界面仍不显示账号？

> 产品：金蝶云·星辰 | 分类：税务管理 | 类型：进阶技巧, 常见问题, 新特性
> 作者：Daisy_ | 阅读：342 | 有用：5
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/657151189569811712)

【问题】

在税务参数-账号管理里维护了开票员或者办税员的电子税局账号，但是在开票或者取票界面仍不显示开票/取票账号是什么原因？

【原因】

维护税务的账号管理时，可以看到有不同的身份类型，不同的身份类型对应的权限不同（和电子税局一致），其中：

- 若您需要在星辰开票，需维护：开票员/财务负责人/法定代表人（三者至少维护一者）
- 若您需要在星辰取票，需维护：办税员/财务负责人/法定代表人（三者至少维护一者）

所以有时需要开票，却维护成办税员了，再或者需要取票，却维护成开票员，会导致在开票/取票界面看不到对应的账号
