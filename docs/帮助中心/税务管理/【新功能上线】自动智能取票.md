# 【新功能上线】自动智能取票

# **1 简介  **

## **1.1 功能介绍**

智能取票支持自动采集发票，是指进入账套时，登录电子税务局取票账号，后台自动采集完整发票数据。

**适用场景**：发票量大，每次做智能取票都要同步在界面等待很久；或者发票采集的字段有缺失，发票源文件采集不完整。

**功能价值**：

1）无需等待：告别手动点击与在线同步的漫长等待，可直接查询发票；

2）发票数据完整：自动获取结构化发票数据与源文件，解决发票字段缺少，认证状态不及时的问题；

3）不错不漏：保证账号在线，后台能够每天持续增量更新发票，重要发票不错过。

## **1.2 适用版本**

星辰4.0及以上标准版/专业版/旗舰版

# **2 操作指南**

**取票流程**：设置自动取票参数—维护电子税局取票账号—每次登录星辰账套登录税局取票账号—保持登录状态后台凌晨会自动执行发票采集任务。

## **2.1 针对老用户第1种情况**

老用户进入星辰首页，会看到智能取票的气泡页，点击【立即体验】即使用该功能，系统会执行自动取票初始化采集任务（即自动采集近一年的全量发票，后续每天增量采集发票）。

![上传图片](https://vip.kingdee.com/download/01000f6d95004bc947dab07bcb2a5470e01d.png)

## **2.2针对老用户第2种情况**

老用户没有点击气泡页的立即体验，可以在发票管理-取票设置-勾选【进入星辰时登录账号，后台自动采集发票（推荐）】，再到系统参数-税务参数-账号管理，维护一遍取票账号，即可执行自动取票初始化采集任务。

![上传图片](https://vip.kingdee.com/download/01009fc6a0f801db46489f9d8f082e9a55b1.png)

![上传图片](https://vip.kingdee.com/download/010023458fccfaba46f38a05da04f29d64fc.png)

## **2.3针对新用户**

新用户建账后，在系统参数-税务参数-账号管理-新增取票账号后，系统会自动执行自动取票初始化采集任务。
