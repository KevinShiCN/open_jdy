# 销售出库单开票成功后，因购方需要退货,在星辰里对发票进行红冲/作废，该张销售出库单及销售退货单还在销售开票状态里应该怎么核销？

> 作者：Daisy_ | 阅读：2278 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/327530272391044864)

问题描述：

销售出库单开票成功后，因购方需要退货,在星辰里对发票进行红冲/作废，该张销售出库单及销售退货单还在销售开票状态里应该怎么核销？

解决方案：

遇到开票后需退货的场景，可进行如下操作：

1.对已开销项发票进行红冲/作废成功后，对应销售出库单状态为未开票；

2.在销售开票列表把销售出库单和销售退货单同时勾选，进行合并开票；

![](https://vip.kingdee.com/download/010928b24ed7ce1c42adad75eddad6bf6eaf.png)

![](https://vip.kingdee.com/download/0109624c07fe59f045a3a9e3333a9edce653.png)

3.返回已开发票列表查看金额为0的销退合并发票，两张业务单据即核销成功

![](https://vip.kingdee.com/download/0109eabbd159455d414185d3a2fe082eacba.png)
