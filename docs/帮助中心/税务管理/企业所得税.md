# 根据自身企业情况进行所得税及减免税申报事项设置，可以出具所得税风险报告、生成所得税预缴申报表（A类）。

| 步骤 | 具体说明 | 学习链接 |
| --- | --- | --- |
| 1.所得税设置 | 企业所得税支持按季度/月度填报计算所得税申报表   | 点击学习 |
| 2.减免税申报事项设置 | 系统预置了税务总局已发布的当期还在有效期的所得税优惠政策,用户可根据自身的实际情况选择设置。   | 点击学习 |
| 3.所得税风险报告 | 财务主管或税务会计在年度业务完成后,查看企业经营的总体情况,查看有哪些潜在风险并及时改进。   | 点击学习 |
| 4.所得税预缴申报表(A类) | 企业所得税预缴申报表(A类)包括一张主表 A200000、 两张副表 A201020、A202000,除企业从业人数外,其他数据系统已预置取数逻辑,可以从系统中自动取值计算。   | 点击学习 |
