# 若有需申报的不开票收入、服务不动产扣除额、进项抵扣和销项税等调整,在哪里录入?录入后会重新测算吗?

> 作者：winnie_wen | 阅读：111 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352531014233088)

【问题描述】

若有需申报的不开票收入、服务不动产扣除额、进项抵扣和销项税等调整,在哪里录入?录入后会重新测算吗?

【解决方案】

点击【税务管理】-【税负测算】

![](https://vip.kingdee.com/download/0109537db5ac577d40a0b8339e9afe20ca90.png)

在销项列表、进项对应填入数据,填入后右上角点击重新测算会出现新的税额数据。

![](https://vip.kingdee.com/download/0109dac3a3e2e9c24457947c0eda75077a6a.png)
