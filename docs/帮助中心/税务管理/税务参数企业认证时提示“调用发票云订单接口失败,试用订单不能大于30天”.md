# 税务参数企业认证时提示“调用发票云订单接口失败,试用订单不能大于30天”

【业务场景】

税务参数企业认证时提示“调用发票云订单接口失败,试用订单不能大于30天”

![](https://vip.kingdee.com/download/0109b59c782c11b146d0a17b0df4525cb0a3.png)

【解释说明】

税务模块可以申请试用,但超过30天后不能再申请,是从试用账套申请日开始30天内有效。

星辰体验和试用账套说明可参考以下链接:[星辰体验及试用账套](https://vip.kingdee.com/link/s/lR42t)
