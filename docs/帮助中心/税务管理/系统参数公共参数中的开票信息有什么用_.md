# 系统参数公共参数中的开票信息有什么用?

> 作者：影子游戏 | 阅读：75 | 有用：0

【问题描述】

系统参数公共参数中的开票信息有什么用?

![](https://vip.kingdee.com/download/0109ad61ecc7b9ca45bf9c5e94c7f63384de.png)

【解释说明】

该信息是同步到税务参数中开票信息的,购买开票管理时,在系统中开具销售发票默认带出的销售方信息

![](https://vip.kingdee.com/download/01091ce06b539921445ca976f15943394d18.png)

购买纳税管理时,进行企业认证带出校验的信息

![](https://vip.kingdee.com/download/01098508fac454c04bdb9483d0ba8b0d529e.png)
