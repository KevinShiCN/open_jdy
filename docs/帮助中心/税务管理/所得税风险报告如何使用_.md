# 所得税风险报告如何使用?

> 作者：ylim | 阅读：270 | 有用：2

【问题描述】

所得税风险报告如何使用?

![](https://vip.kingdee.com/download/0109fb6a9e0f71a84ee2a8bb841a6da01ea3.png)

【业务场景】

财务主管或税务会计在年度业务完成后,查看企业经营的总体情况,查看有哪些潜在风险并及时改进。

【解释说明】

1)年度业务完成后,可评估全年的经营情况和税务风险;

2)从企业日常经营情况分析潜在的所得税风险,提出应对举措和改进建议。
