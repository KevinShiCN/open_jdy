# 智能取票支持取到数电票（铁路电子客票）、数电票（航空运输电子客票行程单）

> 产品：金蝶云·星辰 | 分类：税务管理 | 类型：功能介绍, 新特性, 进阶技巧
> 作者：Daisy_ | 阅读：227 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/656860106616776448)

使用场景：

2024年11月1日起，我国铁路客运领域将全面推广使用电子发票-铁路电子客票（数电铁路客票）。这一变革标志着纸质火车票即将成为历史。

2024年12月1日起，国家税务总局、财政部、中国民用航空局11月6日联合对外发布公告，在我国民航旅客运输服务领域推广使用全面数字化的电子发票-航空运输电子客票行程单。

操作路径：

进入金蝶云星辰-税务管理-发票管理，点击取票设置-无盘取票
