# 电子税局与星辰系统是否能同时登录？

> 产品：金蝶云·星辰 | 分类：税务管理 | 类型：常见问题, 快速入门
> 作者：票税云莫颖娟 | 阅读：131 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/809476299370116352)

# **1 问题描述**

数电开票时，用户是否能同时登录电子税局与星辰系统

# 2 适用版本

星辰标准版/专业版/旗舰版

# **3 解决方案**

电子税局与星辰系统不能同时登录。数电开票时，用户如果中间登录过网页版的电子税务局网站，会导致开票失败，需重新登录电子税局。再次点击开票，星辰系统会给出登录弹框让用户重新登录。
