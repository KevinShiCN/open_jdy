# 发票处理

> 产品：金蝶云·星辰 | 分类：税务管理 | 类型：快速入门, 功能介绍
> 作者：ylim | 阅读：3605 | 有用：10
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/627489176304271872)

| 步骤 | 具体说明 | 学习链接 |
| --- | --- | --- |
| 1.待开发票列表 | 显示系统中未开票的销售出库单、销售退货单、委托代销结算单、委托结算退货单；可以直接通过这些单据进行开票同步到税局系统。 | 点击学习 |
| 2.外部单据开票列表 | 未购买星辰的进销存模块，无法使用星辰的销售单据开具发票，但有自研或已使用行业的业务系统，希望可以通过自身业务系统的单据直接开具发票。支持生态伙伴业务系统链接到星辰的开票管理，第三方业务单据直接在星辰开具发票，并同步发票信息。 | 点击学习 |
| 3.手工开票 | 手工开票指直接手工新增，手工录入开票信息如购买方信息、开票商品等；清单开票、支持批量引入、选销售订单开票。 | 点击学习 |
| 4.已开发票列表 | 在系统中通过单据开出的或者是手工开票开出的发票，会显示在已开票发票列表中；如果开票失败，在已开发票列表中可以查看失败详情。已开票发票列表还支持对开票的发票进行红冲、下载、交付等。 | 点击学习 |
| 5.开票统计 | 开票统计中按年份筛选查询，按月份汇总统计开票的分数、金额。 | 点击学习 |
| 6.发票管理 | 通过发票管理可手工录入发票或从税控盘批量读取销项发票和进项专用发票。 | 点击学习 |
| 7.海关专用缴款书 | 统计显示通过发票管理取票取到的海关缴款书；可以删除、做认证标记、做进项转出。 | 点击学习 |
