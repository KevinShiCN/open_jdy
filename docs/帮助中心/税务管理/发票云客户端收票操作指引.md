# 发票云客户端收票操作指引

【说明】

金蝶云星辰发票云客户端收票操作流程

【环境准备】

1.安装税局开票软件,保证可以登陆进去;

2.安装税务证书应用客户端:[点击下载](https://vip.kingdee.com/tolink?target=https%3A%2F%2Finv-veri.chinatax.gov.cn%2Fdownload%2Fzhfwpt%2F%25E7%25A8%258E%25E5%258A%25A1%25E8%25AF%2581%25E4%25B9%25A6%25E5%25BA%2594%25E7%2594%25A8%25E5%25AE%25A2%25E6%2588%25B7%25E7%25AB%25AFV1.4.zip&productLineId=35)

3.安装发票智慧管家客户端:[点击下载](https://vip.kingdee.com/tolink?target=http%3A%2F%2Fapi.piaozone.com%2Fm28%2Fpublic%2Fclient%2Fdownload%3FoperatingSystemType%3Dwindows%26release%3D10&productLineId=35)

注意:点击下载进入下载页面,提示点击继续访问时没有反应,可复制链接在新建窗口粘贴进入下载

![](https://vip.kingdee.com/download/01099a377b66dc05423db0965af580ca4ae7.png)

![](https://vip.kingdee.com/download/0109eee47448a9bb434e826a7e14cc1f9549.png)

【安装发票云客户端】

1.发票管理-智能取票中,点击链接,下载发票云取票客户端

![](https://vip.kingdee.com/download/01097ee11189bf994e5eb1486f2b2580c177.png)

2.在Windows的开票电脑上安装【发票智慧管家】客户端

![](https://vip.kingdee.com/download/0109919b26b8145442068caafc72c6ef5ca1.png)

3.打开客户端后,需要先绑定微信。打开微信扫一扫功能,扫码客户端二维码进行绑定。(微信是谁的无限制,可勾选30天免登录)

![](https://vip.kingdee.com/download/01090a73382c0200455cb2b454c12e280d91.png)

4.在微信端按照指引操作,点【预约服务】-【升级全电】,填写公司税务信息,提交后邮箱会收到激活码

![](https://vip.kingdee.com/download/010909834606285549e6aed2821cfd640a11.png)

5.在客户端输入在激活邮件中的激活码(激活码有效期24小时,如已过期,请再次点击激活链接获取新激活码)

![](https://vip.kingdee.com/download/0109857eba882b7347f289bfc69fdd5f8577.png)

6.激活成功后,如下图展示:

![](https://vip.kingdee.com/download/01093b7c461a62514d42852612a62c341a26.png)

【配置客户端企业信息】

1.全电客户:点击【企业信息配置】,点击企业名称进入信息栏,点击【新增账号】,选择地区,填写登陆电子税务局的开票员账号密码

![](https://vip.kingdee.com/download/010975477da834ae442d908854b90d3c440b.png)

![](https://vip.kingdee.com/download/0109e62caf703c994f788a96da366f3c2363.png)

![](https://vip.kingdee.com/download/01091445a89ce5fd45c5897ed022536eb2e4.png)

![](https://vip.kingdee.com/download/0109f45f7c7a02d9427293ef49d51562a452.png)

配置成功如图所示:

![](https://vip.kingdee.com/download/01099f4e70a5f6cf40c198ab9e3b3cea4f5f.png)

2.税盘客户:插入税盘或UKEY,点击【税盘管理】,录入密码保存并验证登录

![](https://vip.kingdee.com/download/0109c6a3708a603d402ab8d8514f5edd1d75.png)

三个密码分别是:登录开票软件的证书密码、登录增值税综合服务平台的密码、进项勾选确认的密码。

注意:如果提示密码错误一定不要反复重试,先登录开票软件和增值税综合服务平台确认好密码后,再来重试,以防税盘锁死。税盘锁死后只能到税局解锁

3.全电+税盘客户:上述 1)、2)均需要做配置

【取销项发票】

全电客户在右上角选择【全电模式】,税盘/UKEY客户在右上角选择【税控模式】,在【发票查询统计】中选择【开具发票】,过滤开票日期、发票类型后,点击【下载发票】。间隔10分钟后,再去星辰发票管理中【智能取票】。

![](https://vip.kingdee.com/download/01098779d3e26267481f93ea2c3afec99ab9.png)

 【取进项发票】

方法一:

全电客户在右上角选择【全电模式】,税盘/UKEY客户在右上角选择【税控模式】,在【发票查询统计】中选择【取得发票】,过滤开票日期、发票类型后,点击【下载发票】。间隔10分钟后,再去星辰发票管理中【智能取票】。

这种方式可以取到所有的进项全电发票、专用发票和普通发票,但是无法获取认证状态和认证期间。

![](https://vip.kingdee.com/download/0109c47ffc6ecdd8441e9a5c3163abb25276.png)

方法二:

在【发票勾选确认】中按抵扣状态、开票日期、发票类型过滤,点击【下载发票】。间隔10分钟后,再去星辰发票管理中【智能取票】。这种方式可以取到所有可抵扣的进项发票,包括认证状态和认证期间。

![](https://vip.kingdee.com/download/01094613c115d19d447894fd3b2810b10095.png)

方法一和方法二可以结合使用,数据不会重复。
