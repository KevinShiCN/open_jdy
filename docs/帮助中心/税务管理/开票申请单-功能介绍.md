# 开票申请单-功能介绍

## **1 简介  **

### **1.1 功能介绍**

本功能旨在支持业务员在进行销售开单后，业务员提出开票申请，财务开具发票，由于业务员清楚实际客户的开票诉求，业务员可以维护具体开票的信息（如：发票抬头、开票数量、开票金额、发票备注等），所以希望能够支持业务员提交开票申请，财务收到申请后进行开票。

。

### **1.2 适用版本**

星辰4.0标准版/专业版/旗舰版

### **1.3 系统路径**

参数启用：

1）系统设置-系统参数-税务参数-开票流程-【启用开票申请】

![](https://vip-admin.kingdee.com/download/01093b9cf1e780324c838cb015ba30c0c132.png)

2）关联参数：

【单据开票前必须申请】：若启用了单据开票前必须申请，则必须通过业务员提交开票申请单开票，若无开票申请单，开票员无法在待开发票直接勾选单据开票

提醒：一经选定开票流程，不建议更改

![](https://vip-admin.kingdee.com/download/01091f404e8ac5bd4354a5e78a7f48a8fdb8.png)

## **2 操作指引**

### **2.1 **操作流程

1）业务员新增开票申请单

税务管理-开票管理-新增开票申请单

![](https://vip-admin.kingdee.com/download/01095bc4250402b44a658d8afa7cf5e2532e.png)

开票信息栏-选择源单，支持选择：销售出库单（标准版/专业版）/销售发票（旗舰版）、委托代销结算单、受托加工发票

![](https://vip-admin.kingdee.com/download/0109505f87c9529e4cf08ac6d9f694b87a79.png)

支持维护发票抬头、发票折扣，选择合并开票方式

![](https://vip-admin.kingdee.com/download/010918617c76058c4d2ababd1fcd630ab348.png)

支持维护本次申请开票数量、本次申请开票金额

**![](https://vip-admin.kingdee.com/download/01095099605eb9664c9ebc37e3f7e8a6b914.png)**

【保存】开票申请单，即提交开票申请

![](https://vip-admin.kingdee.com/download/0109bda1f6236c474fdcb0aac0aba6172f27.png)

2）开票员根据开票申请单/销售类单据开具发票

路径：税务管理-开票管理-待开发票列表

**① 参数仅勾选：【启用开票申请】**

该参数支持业务员提交开票申请单，开票员可以根据开票申请单开票，也可以勾选未开票的销售单据开票

待开发票列表支持选择：销售类单据列表、开票申请单列表开票

![](https://vip-admin.kingdee.com/download/010981ad6d61c63a47cf810fc953311542f2.png)

**② 参数同时勾选：【启用开票申请】、【单据开票前必须申请】**

该参数支持业务员提交开票申请单，且开票员必须通过开票申请单开具发票，销售单据不做开票申请不允许开票

待开发票列表仅支持选择：开票申请单列表开票

![](https://vip-admin.kingdee.com/download/0109a20a0d36ff0249f6998641fd8540a158.png)

通过开票申请单开具发票，不允许部分开票，需要按照业务员提交的申请数量、金额全部开票。
