# 销售开票,发票备注如何设置?

> 产品：金蝶云·星辰 | 分类：税务管理 | 类型：常见问题
> 作者：Simone邓 | 阅读：1858 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/525115319988119552)

【问题描述】

销售开票,发票备注如何设置?

【解释说明】

税务系统开票时,系统支持在参数中设置发票备注取值;

**发票备注支持取的字段有**:单据编号、客户编号、单据日期、单据备注、关联销售订单编号、关联客户订单号、购买方银行信息(仅对数电票生效)、销售方银行信息(仅对数电票生效);

同时支持文本框中输入固定文本信息,销售开票备注模块最多支持166个字符,超过会被系统拦截。

【操作指引】

依次点击【税务管理】>【待开发票列表】

![](https://vip.kingdee.com/download/01095ed21ba552f443b3b78104a54166ae2c.png)

打开后点击右上角<参数>设置“发票备注取值”,可勾选需要显示的字段,

![](https://vip.kingdee.com/download/0109d0c94b7d23e041e9b477724f28c59a58.png)

如希望发票预览界面默认显示银行信息,不用每点开一张票勾选一次,请勾选参数“销售方银行信息”即可。

![](https://vip.kingdee.com/download/010941a16e3c3da042da9fa563274db1d9d4.png)

系统不支持勾选的字段,建议录入在备注栏,手工输入固定的备注内容。

![](https://vip.kingdee.com/download/0109ca70936b6af44519aecb78a71eb39c4d.png)
