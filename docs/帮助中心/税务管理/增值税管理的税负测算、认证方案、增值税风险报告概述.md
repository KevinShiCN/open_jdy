# 增值税管理的税负测算、认证方案、增值税风险报告概述

【问题描述】

增值税管理的税负测算、认证方案、增值税风险报告概述

【名词解释】

税负测算:

按增值税销项发票,及认证的进项发票、其他可抵扣发票测算月度的增值税税负率。除发票数据外还包含一些调整项,比如:不开票收入的税项、加计抵扣的税项、以及各种减免税等。

在测算前手工录入调整金额,系统会自动计算税负率及应交增值税、应交增值税附加等。

在测算税负的同时,也会对当月的增值税进行一个整体的评估,并提示可能的风险项及操作建议,用户可按需选用。

认证方案:

按预期税负率和波动范围,推荐适合的发票认证并计算税负。

提供4种认证方案可选:

a、  税负优先方案:按最优税负率推荐可认证的发票组合

b、  发票时间优先方案:按发票时间顺序推荐可认证的发票组合。

c、  综合优先方案:按发票时间和税负率综合评估,推荐可认证的发票组合。

d、  自选:自选发票,自动测算。

增值税风险报告:

以财务、业务、发票、核算、报表、申报等数据为依据,综合评估纳税人的增值税税务风险,为企业的税务筹划提供参考。

该风险评价指标借鉴了税务机关的涉税风险评价体系,辅以企业内部的经营管理及财务核算评估得来,将风险评估划为“合规性”、“一致性”、“时效性”、“指标类”四类,每一大类下设若干指标,并把指标安全性量化分解为“安全”“有风险”“风险较高”“风险极高”“无法评价”等几个等级。风险较大的指标项,系统给出针对性的建议,同时在线对接财税专家,提供一对一专业服务。
