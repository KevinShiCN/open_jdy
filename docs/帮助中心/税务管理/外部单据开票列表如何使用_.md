# 外部单据开票列表如何使用?

> 产品：金蝶云·星辰 | 分类：税务管理 | 类型：常见问题
> 作者：ylim | 阅读：560 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/639268260093086208)

【问题描述】

外部单据开票列表如何使用?

【业务场景】

未购买星辰的进销存模块,无法使用星辰的销售单据开具发票,但有自研或已使用行业的业务系统,希望可以通过自身业务系统的单据直接开具发票。支持生态伙伴业务系统链接到星辰的开票管理,第三方业务单据直接在星辰开具发票,并同步发票信息。

【操作指引】

点击【税务管理】-【外部单据开票列表】

![](https://vip.kingdee.com/download/0109729f088bc4dd42669d0007409689a679.png)
在外部单据开票列表上可以勾选已同步的外部单据点击【开票】

![](https://vip.kingdee.com/download/01099876206800fb43cb9ef5fd729ea2bd3b.png)
点击右上角【参数】可设置是否按折后金额开票、商品超八行是否自动拆分发票、默认携带的开票备注信息

![](https://vip.kingdee.com/download/01096f93fbbdc98740179a62413a7a44e5e6.png)
【温馨提示】

外部单据开票需对接星辰标准API接口,由公司开发者研发,可参考操作文档:[API文档](https://open.jdy.com/#/files/api/detail?index=2&categrayId=1f51c576013945e2af68ef15d4245a48&id=525e704824d24b178ab466530456c037)
