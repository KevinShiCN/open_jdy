# 在发票管理中，发票的入账日期及入账状态是指什么？

> 作者：票税云莫颖娟 | 阅读：74 | 有用：1

# **1 问题描述**

在发票管理中，发票的入账日期及入账状态是指什么？

# 2 适用版本

星辰3.0及以上标准版/专业版/旗舰版/所有版本

# **3 解决方案**

1）发票入账功能介绍

发票入账是指通过电子发票平台对纳税人取得的全部发票（含海关缴款书）及代扣代缴完税凭证数据进行归集。在使用数电票过程中，避免开票方误开红字发票、受票方电子发票重复入账归档。

2）如何进行发票入账

纳税人可以通过登录新电子税局，点击【我要办税】－【税务数字账户】－【发票入账标识】模块进行发票入账操作。

![上传图片](https://vip.kingdee.com/download/01003ea8fcb8e5f647af92cb67357eef9221.png)

3）纳税人可以根据需要在发票、海关缴款书、代扣代缴完税凭证三种标签类型下输入或选择查询条件进行查询，系统将在查询结果以列表形式展示符合条件的发票信息：

![上传图片](https://vip.kingdee.com/download/01006716640942d049b29025c969907b0938.png)

4）通过星辰系统中发票管理-智能取票，可以获取到电子税局中的数电发票，同步入账时间及入账状态。

![上传图片](https://vip.kingdee.com/download/0100d400ae8ee003441da5d5a99c1d3709f6.png)
