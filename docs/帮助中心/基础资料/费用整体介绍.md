# **1 费用概述**

### **1.1 含义及应用单据**

在供应链业务中，不同类型的业务在开展的过程中都会发生一些商品之外的费用，例如运费、卸货费、人工费等等，这部分费用会对成本产生影响。星辰不同的单据类型有不同的费用类别，可以灵活满足不同的业务需求。

| 费用名称 | 适用单据 | 适用版本 | 含义 | 对成本的影响 |
| --- | --- | --- | --- | --- |
| 预计销售费用 | 销售订单 | 所有版本 | 预估的费用，表示费用未发生，定义详见销售费用含义。 | 不影响成本 |
| 销售费用 | 销售出库单 | 所有版本 | 为促进产品销售发生的各项支出，含广告宣传费、销售渠道搭建费、展会参展费等 | 不影响成本，影响销售毛利 |
| 预计客户承担费用 | 销售订单 | 所有版本 | 预估的费用，表示费用未发生，定义详见客户承担费用含义。 | 不影响成本 |
| 销售出库单 | 旗舰版 |  |  |  |
| 客户承担费用 | 销售出库单 | 标准版/专业版 | 销售过程中，销售方先帮客户垫付的费用（如运费、搬运费等），后续客户需要支付给销售方 | 不影响成本，影响应收 |
| 销售发票 | 旗舰版 |  |  |  |
| 预计采购费用 | 采购订单 | 所有版本 | 预估的费用，表示费用未发生，定义详见采购费用含义。 | 不影响成本 |
| 采购发票 | 旗舰版 |  |  |  |
| 采购费用 | 采购入库 | 所有版本 | 采购货物过程里，除货款外的相关支出，如采购过程发生的运输费、装卸费、保险费等 | 分摊后影响成本，摊入采购入库成本 |
| 代销费用 | 委托结算 | 专业版/旗舰版 | 委托他人代销商品时，委托方支付给代销方的报酬及相关支出。具体包括但不限于：代销方的佣金为代销提供的场地费宣传推广费等 | 分摊后影响成本 |
| 受托结算 | 专业版/旗舰版 | 主要是指在接受委托代销商品的过程中，受托方为完成代销业务所发生的各项成本及应获取的报酬，具体包括但不限于受托方的仓储费、保管费为推广代销商品发生的宣传费、广告费、促销费等销售人员的薪酬/提成代销佣金运输代销商品过程中产生的运输费、装卸费（若由受托方承担）等 | 分摊后影响成本 |  |
| 退货费用 | 销售退货采购退货 | 所有版本 | 退货过程产生的诸如包装费、运费、装卸费等等 | 分摊后影响成本 |
| 调拨费用 | 调拨入库（异价调拨） | 所有版本 | 调拨过程产生的人工费、运费等等 | 分摊后影响成本。 |
| 组装费用 | 组装单 | 所有版本 | 组装过程产生的包装费、材料费、人工费、运费等等 | 分摊后影响成本。 |
| 拆卸费用 | 拆卸单 | 所有版本 | 拆卸过程产生的包装费、材料费、人工费、运费等等 | 分摊后影响成本。 |

## **2 应用场景**

### 2.1 同单费用

**含义**：即录业务单据的时候，费用已经确认了，可以在录单的时候一并给录入，并在业务单据上完成分摊。

**适用场景**：开单的时候费用就已经明确的，可以选择开单时就分摊到商品中，也可以不分摊。

**操作流程：**请查看详细知识——[销售费用处理方案](https://vip.kingdee.com/link/s/ZS2WL)、[采购过程的费用处理](https://vip.kingdee.com/link/s/ZS51D)

### 2.2 事后费用/费用分摊

**含义**：即录业务单据的时候，费用还未确认，需要在月末或者跨月的时候才能确认。

**适用场景**：事后才确认的费用，跟业务的发生不同时，往往会滞后，影响的是结存成本，不影响已录单据上的成本。

**操作流程：**请查看详细知识——[费用分摊单功能说明与应用场景](https://vip.kingdee.com/link/s/ZJhur)

### 2.3 审核后补录费用

**含义**：即录业务单据的时候，费用还未确认，但是单据已经先审核了，在当期结账前，需要补录这笔费用到单据上，并分摊到单据上。

**适用场景**：业务单据审核后，结账前补录到单据上的费用。补录后不分摊则不影响单据上商品的成本，补录后点分摊后则影响商品成本。

**操作流程：**请查看详细知识——[允许单据审核后修改费用功能说明](https://vip.kingdee.com/link/s/ZS5By)
