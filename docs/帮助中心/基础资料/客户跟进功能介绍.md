# 1 应用场景及价值

当销售人员在外拜访/线上联系客户或下游经销渠道时，可以使用客户跟进功能规划拜访的路线、记录拜访/联系的内容和对象、以及拜访的地理位置等详情，方便销售经理查看员工的外出拜访情况以及各门店的现状。

# **3 功能特性**

- 跟进任务：允许一个客户同一天建立多个任务，支持按线路和客户分类建立任务。APP和PC端均可编辑
- 跟进记录：当跟进任务提交，或临时拜访添加记录后，可汇总查看所有的记录。APP和PC端均支持查看编辑。
- 拜访轨迹：可查看当天业务员的拜访正式客户整个路径轨迹
- 跟进日报：领导可查看整个部门的跟进情况，业务员可查看自己的跟进情况
- 考勤相关参数设置：可按公司要求设定水印、打卡距离等设置
- 进行中任务检查：客户添加任务或者记录时，回去检查该业务该客户当天有无进行中的任务，有的话跳出弹框，可直接跳转至进行中的任务
- 指派：部门负责人需将任务指派给另一个业务员

# 5 应用流程

## 5.1 正式客户拜访/联系

- 针对老客户，销售员可制定周期性计划，定期执行后，在系统内填写跟进记录进行存档
- 计划预警能及时提醒销售员完成跟进任务，保障每一个客户都能及时跟进，防止客户流失
- 销售员通过定期巡店，可维护下游经销商或代理商
- 巡店过程中，可上报订单、产品销量、库存情况，进行现场销售出库等

（1）设置跟进任务。在“客户经营>跟进任务”中新建跟进（拜访/联系）任务，输入计划跟进时间，计划跟进客户，如有周期设置，可编辑周期重复逻辑，输入完成后，保存

（2）跟进客户及填写跟进记录。选中任务跳转跟进界面，到店作业（打卡、图片采集、拜访总结、销售开单、销售出库、销售退货、本品上报、竞品上报、收款）。拜访任务支持快速“预约下次跟进”

## 5.2 潜客跟进/联系

- 针对众多网络平台（如百度表单、抖音、钉钉及企微、订货商城等）老板/销售部负责人可批量导入第三方潜客数据，并完成潜客的资源分配、回收，销售员的跟进管控
- 销售员进行陌拜时，可将拓展的潜在客户导入星辰系统，并完成后续的潜客跟进、开单转化
- 单击选中任务，跳转跟进界面，到店作业（到离店卡、图片采集、拜访总结）
