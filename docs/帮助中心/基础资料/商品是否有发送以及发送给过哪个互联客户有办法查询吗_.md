# 商品是否有发送以及发送给过哪个互联客户有办法查询吗?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：Schatz | 阅读：51 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/662099324351850496)

【问题描述】

商品是否有发送以及发送给过哪个互联客户有办法查询吗?

【解释说明】

进入【基础资料】-【商品】页面,点击右上角的齿轮图标,打开列设置。在列设置中找到“发送状态”字段,并将其开启显示。

![](https://vip.kingdee.com/download/0109fe9e70863fb54bd0ba5f1e3a8a89db5a.png)

随后,在商品列表中就可以看到“发送状态”列。通过这一列,可以确认商品是否已发送。对于已经发送的商品,点击其“已发送”状态,即可打开<商品共享日志>界面,在该界面中,可以查看该商品已经发送给了哪些互联客户。

![](https://vip.kingdee.com/download/0109cae33fa0e7a34abb97d7d16b93c23579.png)
