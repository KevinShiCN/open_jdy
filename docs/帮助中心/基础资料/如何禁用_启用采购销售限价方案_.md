# 如何禁用/启用采购销售限价方案?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：Simone邓 | 阅读：195 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/653045469186312192)

【问题描述】

如何禁用/启用采购销售限价方案?

【业务场景】

若需取消或暂停限价方案的执行,可在采购销售限价列表中直接**禁用该方案**。

【操作步骤】

【基础资料】-【采购销售限价】列表,选中对应的限价方案<禁用>

![](https://vip.kingdee.com/download/0109da7104571d8a4c3c9d5d35c078469cce.png)
若后续还需要继续使用可查询出禁用的方案再启用

【限价方案列表】,筛选条件【状态】选择【禁用】查询出被禁用的方案,选中之后点<启用>即可。

![](https://vip.kingdee.com/download/01097224d7dfd5c34ac6932395be1e87f743.png)
