# 如何导入客户初始数据？

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：功能介绍, 快速入门
> 作者：陈娇智 | 阅读：930 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/23267)

问题描述：

解决方案：

客户初始数据支持引入引出。系统预置了引入引出标准模板，可以先引出模板，在模板中录入数据后，再引入到系统中。如果客户初始数据较多，建议使用引入功能。操作步骤如下：

1、单击【引出】-【按模板引出】，选择模板，单击【确定】按钮，如下图所示：

![](https://vip.kingdee.com/download/01003b4ec3fcea7c429285b6a4f447dc8056.png)

2、在模板中录入客户初始数据。

3、单击【引入】，选择引入文件，单击【开始引入】，引入完成后，系统会显示引入结果。也可以通过【引入详情】查看近7天的引入记录。引入界面及结果如下列各图所示：

![](https://vip.kingdee.com/download/01005c4059e5a89a4d43a8fc8386e54b40a1.png)

![](https://vip.kingdee.com/download/0100549e288469ab4a38a742e5055997c49f.png)
