# 商品资料中的保质期商品到期日计算方式、修改到期日功能说明

# **1 简介  **

## **1.1 功能介绍**

商品资料中，保质期管理有“到期日计算方式”、“修改到期日”两个参数，可以灵活满足不同行业的的保质期管理要求，更贴合实际业务情况。

## **1.2 适用版本**

星辰3.1以上所有版本

# **2 操作指南**

## **2.1 应用场景**

### **场景一：****需要根据到期日、生产日期、保质期反算的**

• **计算规则1：到期日=生产日期+保质期；**

如：2022-08-01生产，保质期30天，到期日为2022-08-31

• **计算规则2：到期日=生产日期+保质期-1天；**

如：2022-08-01生产，保质期30天，到期日为2022-08-30

### **场景二：****不需要根据到期日、生产日期、保质期反算，如实反馈实际业务需求的**

如：2022-08-01生产，保质期30天，到期日为2022-09-15

## 2.2 **应用流程**

1）在系统参数>基础资料参数>商品参数中开启“启用保质期”参数

2）新增商品，开启保质期管理

![上传图片](https://vip.kingdee.com/download/01001754bf58db7f4972aee339f05e3119b3.png)

3）根据实际业务需求填写保质期单位/保质期/预警天数

4）设置到期日计算方式。系统提供2种规则

| 规则计算逻辑 | 举例 |
| --- | --- |
| 规则1：到期日=生产日期+保质期 | 2022-08-01生产，保质期30天，到期日为2022-08-31 |
| 规则2：到期日=生产日期+保质期-1天 | 2022-08-01生产，保质期30天，到期日为2022-08-30 |

5）设置“修改到期日”后的关联操作

| 保质期单位 | 修改到期日——重新计算保质期 | 修改到期日——重新计算“生产日期” | 修改到期日——不重算 |
| --- | --- | --- | --- |
| 天 | 本规则仅适用于保质期单位为“天”。示例：例如生产日期为2022-06-02，保质期为10天，原到期日为2022-06-12，改为2022-06-18后，保质期自动变成16。 | / | 修改商品到期日，其他值均不变。示例： |
| 月/年 | / | 本规则仅适用于保质期单位为“月/年”。示例：例如生产日期为2022-06-02，保质期为1年，原到期日为2023-06-02，改为2023-06-28后，生产日期自动变成2022-06-28。 |  |

具体的设置规则可以看“修改到期日”右边的小问号说明。
