# 凭证模板

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：功能介绍, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：2473 | 有用：1

# **1 简介  **

### **1.1 应用场景**

凭证模板主要用于记录日常常用凭证业务，可以在录入凭证时调用实现凭证快速录入。

### **1.2 适用版本**

星辰所有版本

### **1.3 系统路径**

基础资料>财务资料>凭证模板

# 2 操作指南

单击基础资料>财务资料>凭证模板，即可进入凭证模板界面，如下图所示：![](https://vip.kingdee.com/download/0100d0f045962f7d4054be49ed47783a3c44.png)

### 2.1 预设凭证模板

系统根据会计制度，预设了日常会计核算中较常用的凭证模板。

### 2.2 凭证模板类别管理

• 凭证模板类别支持新增、修改和删除。

• 单击左边凭证模板分类树后方的【编辑】功能，出现“完成编辑”后，在下方单击【![](https://vip.kingdee.com/download/0100fa38b8a9e9b04d759e928574116f5bf4.png)】增加功能按钮，可以新增凭证模板类别，如下图所示：

![](https://vip.kingdee.com/download/01003c993ecdb2be4e63afa3379de091fe9f.png)

提示：凭证模板类别下有凭证模板的，删除时会提示：“当前凭证模板类别下已有凭证模板，删除时会同时删掉该类别下的凭证模板，是否继续？”，请谨慎操作。

### 2.3 新增凭证模板

单击【新增】按钮，可以直接新增凭证模板。如下图所示：

![](https://vip.kingdee.com/download/01005aad32aba08f403a8e7355cfda69557b.png)

### 2.4 新增凭证模板相关字段功能说明

| 字段名称 | 字段说明 | 是否必录 |
| --- | --- | --- |
| 凭证模板类别 | 支持下拉选择，默认携带新增时定位的类别，若定位在“全部”，则当前页面凭证模板类别显示为“常用模板”。 | 是 |
| 凭证模板名称 | 手工录入，凭证模板名称唯一 | 是 |
| 金额来源-手工录入 | 凭证模板分录行可直接手工录入金额，不按百分比判断是否平衡 | 否 |
| 金额来源-金额百分百 | 凭证模板分录行的金额，会根据录入的借贷方金额和百分比判断是否平衡 | 否 |
| 金额来源-发票税项 | 选择此项后，分录行的金额来源可选择税额、不含税金额、价税合计等字段。用于附件生成凭证时直接取识别到的税额、不含税金额、价税合计。 |  |
| 摘要 | 手工录入，系统会记忆保存过的凭证模板所有摘要（与录凭证摘要打通），手工录入时可进行模糊匹配 | 是 |
| 借/贷方向 | 下拉选择：借、贷，默认借 | 是 |
| 科目 | 从会计科目表的明细科目中选择，支持模糊匹配。带核算项目的科目可选到具体核算项目，但核算项目非必录 | 是 |
| 数量/单价 | 模板中存在科目设置了辅助核算才显示 | 否 |
| 币别/原币/汇率 | 模板中存在科目设置了外币核算才显示 | 否 |
| 金额 | 支持手工录入，保留本位币小数位 | 否 |
| 金额比例 | 鼠标移动到【？】图标会有对应功能提示：用于凭证附件生成凭证时借、贷方科目金额占总金额的比例默认100%，可手工修改 | 否 |
| 金额取值 | 当金额来源选择“发票税项”时才会出现。字段来源可选择税额、不含税金额、价税合计等字段。用于附件生成凭证时直接取识别到的税额、不含税金额、价税合计。 | 否 |

提示：凭证模板还可以在录入凭证时保存为凭证模板，详见凭证录入功能说明。
