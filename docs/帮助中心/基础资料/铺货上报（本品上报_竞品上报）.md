# 铺货上报（本品上报/竞品上报）

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：快速入门, 功能介绍
> 作者：Freya_peng | 阅读：2527 | 有用：9
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/350235980059458816)

# **1 应用场景及价值**

销售人员在外拜访客户或下游经销渠道时，可以使用到店后上报该店铺我司商品的数量信息进行上报，以及竞争对手的商品的数量信息，便于销售经理记录门店的供货现状，并可通过上报的数量信息及时去补货

# **2 操作路径入口**

(1) 移动端：访销>巡店任务>跟进任务>跟进>铺货上报

(2) 电脑端：客户经营>本品上报/竞品上报（仅查看）

# **3 名词解释**

(1) 本品上报：上报该客户持有我司商品的数量

(2) 竞品上报：上报该客户持有我司竞争对手商品的数量

# **4 应用流程**

**场景：巡店/拜访任务**

a. 针对老客户，销售员可制定周期性计划.定期执行后，在系统内铺货上报>本品上报填写客户的我司商品在货架的商品数量，在系统内铺货上报>竞品上报填写竞争公司的商品在货架的商品数量进行存档

b. 巡店过程中，在铺货上报>本品上报填写客户的我司商品库存进行存档，在系统内铺货上报>竞品上报填写竞争公司的商品进行存档

**操作流程：**

步骤一：打开巡店任务菜单

![](https://vip.kingdee.com/download/01008f52e2721f8b425f9672ba5143412fd7.png)

步骤二：单击跟进任务，开始跟进任务。首先进店打卡

![](https://vip.kingdee.com/download/01004702e032b0434fe6a52209ef9cc2ac13.png)

步骤三：点击铺货上报，进入本品上报。可通过选商品的方式跳转商品列表选择商品数量；也可通过扫商品条码，识别到对应商品后输入商品数量。通过这种方式可上报该客户持有我司商品的数量。还可以通过历史查看商品历销

![](https://vip.kingdee.com/download/010083cf5e82b3c84f80ac099a555b8fada2.png)

步骤四：点击铺货上报，切换标签页进入竞品上报。可通过扫条码的方式识别商品。只能识别国际标准商品码。扫码后识别出对应商品，录入数量。通过这种方式可上报该客户持有竞品的数量

![](https://vip.kingdee.com/download/0100b38b7171a4634cd4a0f569a6e02e027d.png)

步骤五：电脑端可查看铺货上报的结果

![](https://vip.kingdee.com/download/0100880ddbe8dcac483e8f7664a474ed3429.png)
