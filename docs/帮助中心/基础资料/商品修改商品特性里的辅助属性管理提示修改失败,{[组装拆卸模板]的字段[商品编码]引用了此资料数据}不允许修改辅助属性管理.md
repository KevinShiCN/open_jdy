# 商品修改商品特性里的辅助属性管理提示修改失败,{[组装拆卸模板]的字段[商品编码]引用了此资料数据}不允许修改辅助属性管理

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：ylim | 阅读：414 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/571138782745753856)

【问题描述】

商品修改商品特性里的辅助属性管理提示修改失败,{[组装拆卸模板]的字段[商品编码]引用了此资料数据}不允许修改辅助属性管理

![](https://vip.kingdee.com/download/0109c79d233c30454cc89e678471db87770f.png)

【解释说明】

说明商品被使用过了,比如提示被组装拆卸模板的字段引用了,就要检查组装拆卸模板

点击【库存管理】,鼠标放【组装拆卸模板】上点击【查询】按钮,

![](https://vip.kingdee.com/download/01096acd74099ff54036977a679797763132.png)到模板列表上之后筛选商品信息看是否有做模板

![](https://vip.kingdee.com/download/0109b1c1278212514208b550b9accd9ffd09.png)查询到之后勾选模板删除,如果显示已审核就先反审核再删除

![](https://vip.kingdee.com/download/01093cd87bda74384b18849726841646ed18.png)
再去修改商品即可保存成功

![](https://vip.kingdee.com/download/0109bbfc24d1fa2e483b8a2fa0469b8ac2d7.png)
