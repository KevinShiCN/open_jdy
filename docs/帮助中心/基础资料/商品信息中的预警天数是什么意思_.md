# 商品信息中的预警天数是什么意思?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：shaoling | 阅读：180 | 有用：0

【问题描述】

商品信息中的预警天数是什么意思?

【解释说明】

商品信息里的预警天数,是开启保质期管理商品特有的字段,对于有保质期管理的商品,**预警天数**是一个关键的安全参数,它专门用于监控商品的有效期限,防止商品过期造成损失。

预警天数可根据业务需要手工设置,在保质期还有多少天前提前提示预警,也可以根据有效期的百分比进行设置,比如一些短保商品效期过了三分之一就提示预警
