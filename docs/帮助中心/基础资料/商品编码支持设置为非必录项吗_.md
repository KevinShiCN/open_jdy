# 商品编码支持设置为非必录项吗?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：shaoling | 阅读：67 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/690364766606615808)

【问题描述】

商品编码支持设置为非必录项吗?

【解释说明】

1、商品编码不支持设置为非必录的,系统商品编码作为唯一识别商品的字段。不管是在新增商品的时候,还是在新增单据的时候都是必须录入或选择到对应商品编码。

2、在新增商品的时候,商品编码可以按照需要手工录入或者根据商品编码规则生成。[如何设置商品编码规则?](https://vip.kingdee.com/knowledge/627671427285565952?get_from=article-id&productLineId=35&isKnowledge=2&lang=zh-CN)

3、在新增单据的时候,商品编码作为选择商品的字段,也是必录的。

单据录入的时候在商品编码字段可以根据商品名称、规格型号等模糊搜索选择对应商品信息
![](https://vip.kingdee.com/download/0109a0fd99c6d60948229e527eb28055fc1b.png)
所以在录入单据的时候,不能把商品编码字段设置隐藏不显示,避免后续无法成功开单
