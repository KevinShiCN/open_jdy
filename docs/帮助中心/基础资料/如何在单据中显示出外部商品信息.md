# 如何在单据中显示出外部商品信息

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：云社区用户xBO70916 | 阅读：233 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/604841057771152384)

【问题描述】

在销售出库单、采购入库单上是否可以直观的看到外部商品信息
【操作说明】

首先需要在【基础资料】-【外部商品库】维护好客户和供应商的商品编码和名称

详情可参考:[外部商品库的应用场景](https://vip.kingdee.com/link/s/lsxpx)

维护好了外部商品库开单时选择客户或者供应商,再选择对应的商品即可带出外部商品信息

已销售出库单为例,销售出库单列设置可以打开字段客户商品编码、客户商品名称、客户商品规格型号字段(这些字段的内容就是代表外部商品的信息)

![](https://vip.kingdee.com/download/0109442df33aa23148819ea511a9966e80bf.png)

采购入库单同理,对应显示的是供应商商品编码、供应商商品名称、供应商商品规格型号

![](https://vip.kingdee.com/download/0109356a3fd3922248e1803be230d06f3ff0.png)
