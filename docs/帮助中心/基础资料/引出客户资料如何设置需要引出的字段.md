# 引出客户资料如何设置需要引出的字段

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：shaoling | 阅读：234 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/550119963646570496)

【问题描述】

引出客户资料如何设置需要引出的字段

【业务场景】
客户信息引出想要按自己需要的字段自定义设置模板引出
【解决方案】

客户资料引出可以选择按引入模板引出,引入模板是可以根据自己需要自定义设置操作。操作流程如下:

从【基础资料】——【客户】进入客户列表

![](https://vip.kingdee.com/download/0109fa5a2f1f30a947138fb05113fb555efb.png)

在客户列表右上角点击【引入】——【模板管理】

![](https://vip.kingdee.com/download/0109def78ca7ad0748c98e7e555c8f59d84f.png)

进入模板管理后即可【新增模板】操作

![](https://vip.kingdee.com/download/0109103966cf501f4795b1d2b5eb8566fb43.png)

新增模板可根据自己需要设置开启或关闭哪些字段要引入引出

![](https://vip.kingdee.com/download/010935725d2ab13946e6a40c294b79d4be34.png)

![](https://vip.kingdee.com/download/0109ae513934b0894eda968726e801c396ee.png)

引入模板设置成功后再到【引出】——【按引入模板引出】里操作

![](https://vip.kingdee.com/download/01094f824c2b36e740ce9e6f66bbd04066f0.png)

选择之前自定义设置的模板

![](https://vip.kingdee.com/download/0109705c74a2698f448c8b6bc194f180329b.png)

选择对应模板后即可按设置的自定义模板下载导出客户资料的数据

![](https://vip.kingdee.com/download/010903156747aefa4b448f6340971d77ffea.png)
