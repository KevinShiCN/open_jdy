# 如何导入会计科目?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：ylim | 阅读：1211 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/575849724033616640)

【问题描述】

如何导入会计科目?

【解决方案】

点击【基础资料】-【财务资料】-【科目】

![](https://vip.kingdee.com/download/01094c5acf00f36c4f829ca5e3a90701538e.png)
点击【引入】-【模板管理】,选择要用的引入模板,点击【下载】
![](https://vip.kingdee.com/download/01092ef4ac4db12441c6be0efe85c8c61d01.png)
在下载的模板上录入好科目信息,红色字体为必录项,录好后保存模板文件
![](https://vip.kingdee.com/download/0109326282c7790c48aa940bfc3edfc2af2c.png)
系统里点击【引入】,上传录好的模板文件,上传成功后点击【开始引入】
![](https://vip.kingdee.com/download/0109dc37351e7b7149c591474028b7f1a2d9.png)
如果不知道模板该怎么填写,可以系统上直接新增一些科目后,点击<引出>

![](https://vip.kingdee.com/download/010909877d54efb84c91ac0331040cb02506.png)
可以参考引出模板如何在引入模板中录入
![](https://vip.kingdee.com/download/0109637028f4eaa04493943b75c0d77331b6.png)
引入成功后科目默认是启用状态,若是需要引入禁用的科目,需要在模版中选择启用状态为“禁用”

![](https://vip.kingdee.com/download/0109eb25940d83e34c0db49470db22d6df8e.png)

【温馨提示】

若引入失败,可点击“查看详情”下载错误提示,或通过“引入结果查询”中下载“失败清单”,按系统提示调整模板数据后再重新导入。

![](https://vip.kingdee.com/download/01099034a15ffdb04d2d8cafd20da33bac5c.png)

或者通过引入结果查询,下载失败清单,查看失败原因:

![](https://vip.kingdee.com/download/0109ab465c528cb64a11a836aa72e363467c.png)

按excel表里提示的错误原因,修改引入文件后,重新引入。

![](https://vip.kingdee.com/download/0109cfc3dce3f4424be19a6d9277a5e749e8.png)
