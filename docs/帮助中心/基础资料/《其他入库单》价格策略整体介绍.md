# 《其他入库单》价格策略整体介绍

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：功能介绍, 快速入门
> 作者：赵光芬 | 阅读：1651 | 有用：15
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/61553284460148992)

在价格取数优先级中，可以对其他入库单设置取价优先级，目前支持以下几种取价策略：

![上传图片](https://vip.kingdee.com/download/0100b8afeb78a2824af8acc8d5eace29b907.png)

1、  支持取商品价格资料中的采购价作为其他入库单的入库单位成本；

2、  支持取商品价格资料中的参考成本作为其他入库单的入库单位成本；

3、  支持取已审核的最近的采购订单/采购入库单中的采购单价作为其他入库单的入库单位成本。

![](https://vip.kingdee.com/download/01002147e71a5532427299f211ba684615ca.png)

以上三种价格来源，根据优先级来取价，同时，建议将不参与取价的规则禁用，以提高单据取价时的速度。
