# 如何新增职员?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：杨桂平 | 阅读：6307 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/129339770097650368)

【问题描述】

如何新增职员?

【操作流程】

在基础资料职员中新增职员,可单个新增,也可以批量新增或批量引入。

单击【基础资料】-【业务资料】-【职员】进入到职员列表,职员列表左侧为部门列表。
![](https://vip.kingdee.com/download/0109d670d1c6636b4662882654dfe28a2937.png)
1、单个新增职员

点击右上角【新增】进入职员新增界面,录入相关信息点击【保存】或【保存并新增】即可。
![](https://vip.kingdee.com/download/01099ab398722866470eb2968052d1e4de67.png)
2、批量新增

选择左侧部门后,点击右上角【批量新增】按钮,输入职员信息后点击保存即可

![](https://vip.kingdee.com/download/01095711fae0e9d14257849d3ef0e49c090c.png)

3、引入职员

点击右上角【引入】可通过Excel引入职员,引入时需要下载系统预置模板,在模版中录入职员信息后再引入。

3.1 下载引入模板

点击右上角【引入】-【模版管理】下载引入模板。

![](https://vip.kingdee.com/download/010906a69b8e15844e06bacfe9197c28666b.png)

3.2 填写模板

打开下载的模板并录入职员信息,保存表格。
![](https://vip.kingdee.com/download/01099f6c4b4f2af247f684792cb8953f70e8.png)
3.3 导入职员

引入模板录入完成后在系统中点击【引入】-【上传文件】-【开始引入】等待引入完成即可。

![](https://vip.kingdee.com/download/010913954ed62a3948e09411c5107a2911e7.png)

![](https://vip.kingdee.com/download/0109700e1a7e31f0421b8daf09d72d7c89ba.png)

3.4 引入说明

- 1)引入的字段信息可通过建立新的模板或是修改现有模板设置需要引入的字段,但*号字段职员编码、职员名称、性别为必须引入字段;
- 2)所属部门不填写,则引入后归属为待分配部门人员;
- 3)其他字段不填写,则引入后默认为空 ;

【注意事项】

职员被使用后不允许删除。
