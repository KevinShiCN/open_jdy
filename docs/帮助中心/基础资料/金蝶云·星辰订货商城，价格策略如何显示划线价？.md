# 金蝶云·星辰订货商城，价格策略如何显示划线价？

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：应用案例, 进阶技巧
> 作者：Lizzy | 阅读：721 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/271559407124210176)

当您建立客户价格策略是，以折扣/折扣率的方式设置方案。

订货商城将以销售单价作为划线价，即折前单价，折扣计算后的价格展示在购买单价，并商品显示“显示特价”标签。如图所示。

![](https://vip.kingdee.com/download/010951981876deeb41eabf29dd1f2da9947f.png)

![](https://vip.kingdee.com/download/0109b1168380831d440bb704c07a69db31d1.png)

客户价格策略设置打折

![](https://vip.kingdee.com/download/0109dc77198b4f024d5fbffdfa42a5b9c505.png)

商城端效果

商城用户对该类商品下单，客户价格策略的销售单价将记录在商品分录的“批发单价”、折扣金额记录“特价优惠金额”。

![](https://vip.kingdee.com/download/010983ae6ab4df9f475eb45e9330724335cc.png)

商城订单详情

商城订单审核，下推到销售订单上，商城订单的“批发单价”对应填充销售订单的“含税单价”，“特价优惠金额”+“促销优惠金额”+“优惠券优惠金额”合计对应填充“含税折扣额”

![](https://vip.kingdee.com/download/0109632aee42a13641788907850ed1bcb2d3.png)
