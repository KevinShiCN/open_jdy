# 删除辅助属性商品的属性组合时提示:属性组合[xx]行删除失败-[即时库存实体]的字段[辅助属性编码]引用了此资料数据

> 作者：若有灵兮山之阿 | 阅读：605 | 有用：2

【问题描述】

删除辅助属性商品的属性组合时提示:属性组合[xx]行删除失败-[即时库存实体]的字段[辅助属性编码]引用了此资料数据,怎么处理?

【业务场景】

部分商品的辅助属性不需要使用了,且未录入过对应的单据,需要对其辅助属性进行删除,但是删除时提示:属性组合[xx]行删除失败-[即时库存实体]的字段[辅助属性编码]引用了此资料数据,需要如何操作?

**若希望了解如何删除辅助属性,请点击:[辅助分类属性如何删除?](https://vip.kingdee.com/knowledge/713195518271972352?productLineId=35&isKnowledge=2&lang=zh-CN)**

【操作说明】

出现以上提示一般是由于系统缓存导致的,可以通过校对库存的方式更新库存缓存数据。

点击【库存管理】>【库存查询表】>【校对库存】,操作后,再删除辅助属性商品的属性组合即可
