# 现金流量初始数据

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：快速入门, 精华汇总, 功能介绍

# **1 简介**

## **1.1 应用场景**

总账非第1期启用时，需进行初始化数据维护，录入初始化年初到初始化当期的本年累计流量。录入界面以报表的形式展示。

## **1.2 操作路径**

基础资料>期初录入>现金流量初始数据

## **1.3 适用版本**

星辰3.1以上所有版本

# 2 操作指南

## 3.1 录入

打开基础资料>期初录入>现金流量初始数据，进入现金流量初始数据录入界面，直接根据本年累计现金流量即可。如下图所示：

录入到“本年累计”后，单击“保存”即可。注意，只能录入未结账期间的数据。

![](https://vip.kingdee.com/download/010003b22ffd2042479e93ec429590c91203.png)

## 3.2 平衡检查

单击【平衡检查】按钮，系统会进行平衡检查。如下图所示：
