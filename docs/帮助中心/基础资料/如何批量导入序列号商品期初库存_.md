# 如何批量导入序列号商品期初库存?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：杨桂平 | 阅读：402 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/537798988985141760)

【问题描述】

如何批量导入序列号商品期初库存?

【操作流程】

a、打开[基础资料]-[商品期初],点击右上角[引入]

![](https://vip.kingdee.com/download/0109793d185feeea4ff7b2e90a7c286bf122.png)

b、点开引入数据界面,点击[模板管理]-[新增模板]

![](https://vip.kingdee.com/download/0109c79aa25fdb1744a6a757e6a7f41c7538.png)

![](https://vip.kingdee.com/download/01096a6bc16af22d4f179e4d4503f8a3df6c.png)

c、将[序列号]字段启用必录和引入(单位成本与期初总价需要导入也可以启用),启用后录入模板编码与名称,点击[保存]

![](https://vip.kingdee.com/download/01094f6440d2025047a9ae332504200efd13.png)

d、保存新增模板后,关闭返回模板管理界面,勾选新增模板,点击[下载]

![](https://vip.kingdee.com/download/0109a9334966ff3c411aabaabcd3737b42cc.png)

e、打开模板录入序列号商品期初数据

![](https://vip.kingdee.com/download/0109d2d34c3e8e7842a98e8f9603c50a75ea.png)

f、单击[引入],选择引入文件,单击[开始引入],引入完成后,系统会显示引入结果。也可以通过[引入结果查询]查看近7天的引入记录。引入界面及结果如下列各图所示:

![](https://vip.kingdee.com/download/0109026d430a61f14f0ab22f2802eb1d89bd.png)

![](https://vip.kingdee.com/download/0109e07b653d3c6242518a7b911f5a1a633e.png)
