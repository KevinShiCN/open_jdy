# 系统都有哪些采购价格策略

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：214 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/585547394327154688)

【业务场景】

【基础资料】-【价格取数优先级】中,可以对采购单据设置取价优先级,目前支持以下几种取价策略:

![](https://vip.kingdee.com/download/0109853d457180ac4de09ce3f01c6eb2ff79.png)

一是取“最近采购单价”,来源于已审核的最近的采购订单/采购入库单中的采购单价,按照供应商匹配。适用于商品价格变动频繁的场景。

二是取“商品价格资料中的采购价”,来源于商品价格资料,适用于商品采购价相对固定的的场景。

![](https://vip.kingdee.com/download/0109a4e94d09b74f4364989d9b0d509db2d3.png)

三是取【基础资料】-【供应商价格资料】,可以在供应商价格资料中维护同一商品,不同供应商的采购价。

![](https://vip.kingdee.com/download/010945de47f08934411bba95156daddc5d5a.png)

![](https://vip.kingdee.com/download/0109454a46ada6484f9e81d455d4e9dd211e.png)

系统支持商品多单位、多条码设置不同或相同的价格,采购类单据(采购申请单、采购订单、采购入库单、采购退货单)在取价时,按照商品编码+辅助属性+单位+条码来匹配对应的价格。
