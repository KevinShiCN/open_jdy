# 如何查询不同会计制度下的科目信息?如:对应的编码、科目类别、余额方向等

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：ylim | 阅读：2093 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/606652690109799168)

【业务场景】

如何查询不同会计制度下的科目信息?如:对应的编码、科目类别、余额方向等

【操作步骤】

1,可通过官网的【资源与服务】-【免费工具】-【会计科目汇编】进行查询科目相关信息;也可以直接点击:[会计科目汇编](https://www.jdy.com/new-tools/1530109464324186113.html) 跳转
![01090186a25ede1646d89026cb84be63ba8b.png](https://vip.kingdee.com/download/01092ded1d49a33543b193487bba2ac93913.png)

左侧选择对应的会计制度,可以看到所选会计制度对应的科目有哪些,编码余额方向等是什么

![](https://vip.kingdee.com/download/01093b0150b9a4c045338fe13414ecadb943.png)

也可以在右上角搜索栏输入科目名字,点击搜索即可查询科目相关信息,如图:合同结算

![0109933538705612459088dd1d74ad598229.png](https://vip.kingdee.com/download/010981b4bd7becdc4688a7d8c97e027478fb.png)

另外,还可以下参考方式查询:

2、**通过财政部官方发布的会计科目表**:根据企业会计准则的相关规定,企业可以结合自身实际情况自行确定会计科目编码。财政部网站提供了相关的科目编码信息,企业可以参照这些信息来设置自己的会计科目编码。

3、**使用专业的会计科目网站**:例如中国会计网提供了最新的会计科目表,包括各个科目的编码、名称以及适用范围。
