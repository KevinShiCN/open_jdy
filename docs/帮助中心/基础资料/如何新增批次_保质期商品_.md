# 如何新增批次/保质期商品?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题, 快速入门

【问题描述】

如何新增批次/保质期商品?

【相关视频】

[如何启用批次/保质期管理商品](https://vip.kingdee.com/school/189684414404758528?productLineId=35&lang=zh-CN&cosmic_type=video)

【名词解释】

批次、批号保质期是针对食品药品等行业,需要进行保质期、批次管理的功能;系统支持对每一批次的商品出入库及库存进行跟踪,并提供保质期预警;

说明:商品可以分别启用批次或者保质期单独使用,也可以同时都启用结合使用,结合使用时需要在商品特性中同时勾选“批次管理”和“保质期管理”,并且开单时都需要录入,不可以只录其中一种特性。

【操作流程】:

(1)在【系统设置】-【系统参数】-【基础资料参数】中,勾选“启用批次管理”或是“启用保质期”,也可以同时勾选两项;

![](https://vip.kingdee.com/download/01096e181cea0a34464fbef2e048dfed1200.png)

(2)在【基础资料】-【商品信息】中,新增商品,在商品编辑页面中,【商品特性】勾选“批次管理”或是“保质期管理”,录入其他信息后,点击右上角的<保存>

**tips:启用保质期管理后,需要额外设置“保质期单位”、“保质期”、“预警天数”、“到期日计算方式”和“修改到期日”**
![](https://vip.kingdee.com/download/0109e76bf176b15a46f2a801af9d589272a5.png)

(3)在单据开单的界面,选择批次/保质期的商品,在批次/生产日期的字段下输入对应的批次和保质期信息即可

![](https://vip.kingdee.com/download/0109374040c431124182aac9b2c954100123.png)

【温馨提示】

**1.如果已经在新增单据界面选择了商品之后才修改商品启用商品特性,单据上需要重新选择商品或者关闭单据新增界面重新打开;**

**2.保质期商品的生产日期和到期日的格式必须是年、月、日的形式,暂不支持其他格式。**
