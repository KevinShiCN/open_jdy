#  客户经营权限控制说明

> 作者：Freya_peng | 阅读：1033 | 有用：0

# 1 原客户经营权限背景

潜客、跟进任务、跟进记录、跟进汇总表的权限为：系统自带权限（职员只能看自己的，部门负责人可以看整个部门的）与系统设置的数据授权取交集。

# 2 客户经营全新权限

潜客、跟进任务、跟进记录、跟进汇总表、客户分布地图功能，仅受系统设置的数据权限限制。未设置数据授权时，可以看到所有的客户经营相关数据。具体数据权限控制见下表

| 功能 | 数据权限控制 |
| --- | --- |
| 潜客 | 部门、业务员 |
| 跟进任务 | 客户、部门、业务员 |
| 跟进记录 | 客户、部门、业务员 |
| 跟进汇总表 | 客户、部门、业务员 |

前提：所有用户都必须设置关联的业务员

# 3 操作步骤

系统路径：用户管理<授权<数据授权

![](https://vip.kingdee.com/download/0100e2b565f453874a5b9680b34ad7d74b02.png)

如需控制业务员仅能查看自己的数据，请设置职员的数据授权。如上图所示，如需设置彭裕只能看到自己的数据，对彭裕设置数据授权，职员名称选择彭裕。
