# 外币科目设置时需要同时勾选人民币吗?

【问题描述】
外币科目设置时需要同时勾选人民币吗?

【解释说明】

外币科目是否勾选人民币取决于该科目是否会发生人民币业务,企业按照实际需求勾选即可
