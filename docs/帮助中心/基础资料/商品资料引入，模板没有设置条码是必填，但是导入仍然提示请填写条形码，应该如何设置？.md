# 商品资料引入，模板没有设置条码是必填，但是导入仍然提示请填写条形码，应该如何设置？

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：快速入门, 常见问题, 新特性
> 作者：进销存赵嘉丽 | 阅读：1018 | 有用：7
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/338332097506962176)

【问题概述】

商品资料引入，模板没有设置条码是必填，但是导入仍然提示请填写条形码，应该如何设置？

![](https://vip.kingdee.com/download/01007af70934e6e64e58b1e604beea8b48c3.png)

![](https://vip.kingdee.com/download/01006730024de8ba490da054262e6cf2a713.png)

【原因说明】

出现这个提示一般是因为单据字段配置中的商品资料开启了条形码必录

![上传图片](https://vip.kingdee.com/download/0100f9f25201d3024a7e9296d16a0ad33b11.png)

【解决方案】

关闭系统设置>个性化设置>单据字段配置>基础资料>商品信息的“条形码”必录后即可引入。
