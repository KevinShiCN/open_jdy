# 销售发票期初（旗舰版）

## 1 简介

### 1.1 功能介绍

比如机械设备行业，商品的销售和采购周期都比较长，销售时，为了提前收款，会提前给客户开票，存在已开票还未出库的数据，在软件上线启用账套时，需要记录这部分数据。

### 1.2 适用版本

星辰3.1旗舰版

### 1.3 系统路径

基础资料-期初录入-销售发票期初![](https://vip-admin.kingdee.com/download/010048840b6071a34123a4f2b6a2e51c1e50.png)

### 1.4 功能特性

• 用于记录启用账套前已开票还未出库的业务数据

• 支持《销售发票期初》下推生成销售出库/销售退货单

• 支持《销售发票期初》与销售出库/销售退货单进行单票勾稽

## 2 操作指南

1）销售发票期初，支持的业务类型包括出库、退货，用于登记期初已开票未出库/退货的数据。请注意此单据不影响应收余额，客户期初应收余额请到《客户初始数据》中录入。

![](https://vip.kingdee.com/download/0100e16288d0fede4b60883f637b4959cfed.png)

2）销售发票期初，支持手工勾稽，将销售发票期初与销售出库/退货单关联起来，同时更新销售发票期初中的出库状态、出库数量、出库金额，销售出库单上的开票状态、开票数量、开票金额。

![](https://vip.kingdee.com/download/010044e31c86c5114dcb88dc19495e858e7b.png)
