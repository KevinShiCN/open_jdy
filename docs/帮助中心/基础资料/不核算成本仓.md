# 不核算成本仓

# 1 简介

## 1.1 应用场景

- **赠品管理场景：**采购员采购时，供应商可能会附送一些赠品。这部分赠品没有成本，可以正常销售，销售金额算进收入，但不需要进行成本核算，后续还要进行仓内业务
- **受托加工场景：**客供件材料入库后，需要对材料库存进行盘点，满足受托加工业务对客供材料的数量管控需求

## 1.2 适用版本

星辰3.1及以上专业版和旗舰版

## 1.3 功能介绍

仓库类型新增“不核算成本仓”，该类仓库仅核算库存数量，不核算成本。销售、采购、库存类单据均支持选择到该类仓库，实现无成本商品的全流程业务管理

![](https://vip.kingdee.com/download/0100992f30ebef624a20b6e06f5613bc2f8f.png)

# 2 功能介绍

## 2.1 基础设置

（1）新增“仓库类型”为“不核算成本仓”的仓库。已发生过业务的仓库，不允许更换仓库类型

![](https://vip.kingdee.com/download/01004ac34d94b838478e9a1c7822b9665cb4.png)

（2） 除序列号商品外，其他所有商品都可设置“不核算成本仓”为“默认仓库”。序列号商品在开单/盘点时也不支持选择该类仓库

![](https://vip.kingdee.com/download/010067075e28b500450991bb1d793f5588dc.png)

### 2.2.2 核心业务规则

“不核算成本仓”在各类单据中的规则说明如下：

**采购类单据**：

- **采购订单、收料通知单、采购入库单、采购退货单**：选择“不核算成本仓”后，系统会自动清空金额相关字段并置灰不可编辑，且该仓库的商品行不参与采购费用分摊
- **采购入库单、采购退货单**：“不核算成本仓”的商品行不支持对等核销、勾稽操作，也无法下推生成采购发票
- **采购发票：**商品行不支持选择“不核算成本仓”

**销售类单据**：

- **销售出库单、销售退货单、委托代销单、委托代销退货单：**选择“不核算成本仓”后，成本类字段会清空，后续不参与成本核算，也不会反写商品成本
- **销售发票**：商品行支持选择“不核算成本仓”，但仅能与销售出库单上同类型仓库的商品行勾稽

**库存类单据**：

- **其他入库单、调拨出库单（异价调拨的调入仓库）、调拨入库单、盘盈单、组装单（子件）、拆卸单（组件）**：选择“不核算成本仓”后，金额字段会清空并置灰不可编辑，且不参与费用分摊
- **其他出库单、移仓单、调拨入库单、盘亏单、组装单（子件）、拆卸单（组件）**：选择“不核算成本仓”后，成本类字段为空，不参与成本核算
- **移仓单、同价调拨单：**调出仓与调入仓的仓库类型必须一致（同为“不核算成本仓”或“普通仓”）。
- **异价调拨单**：调出仓和调入仓不能同时为“不核算成本仓”
- **组装拆卸单：**组件仓库为“不核算成本仓”时，子件仓库必须同为“不核算成本仓”；组件仓库为“普通仓”时，子件仓库不能全为“不核算成本仓”（至少有一个子件仓库为普通仓）
- **盘点单：**支持选择“不核算成本仓”，但无法选择序列号管理的商品

### 2.2.3 报表

所有含“仓库”字段筛选的报表均可下拉选择“不核算成本仓”，快速过滤该类仓库业务数据

![](https://vip.kingdee.com/download/0100ff2a5bda89434a0288ab4313dc71448e.png)

### 2.2.4 业务结账检查

- **无成本单据检查，不包含以下单据中“不核算成本仓”的商品行：**销售出库单、销售退货单、 委托代销单 、委托代销退货单 、其他出库单 、组装单 、拆卸单、 盘亏单 、移仓单 、调拨出库单（调出仓）、 生产领料单、 生产退料单
- **单价为0的单据检查，不包含****以下单据中“不核算成本仓”的商品行：**采购入库单、采购退货单、其他入库单、盘盈单

### 2.2.5 出库成本核算

- **分仓核算时，**不包含“不核算成本仓”
- **总仓核算时，**不包含“不核算成本仓”对应的库存类单据的商品行
- **出库成本核算，**不包含“不核算成本仓”的商品行
- **无源单的销售退货单、负结存出库单据，**即使勾选了“成本来源”，“不核算成本仓”的商品行也不会反写成本
- **核算单据查询中，**不显示“不核算成本仓”的商品行
