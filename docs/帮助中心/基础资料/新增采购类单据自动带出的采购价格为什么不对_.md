# 新增采购类单据自动带出的采购价格为什么不对?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：杨桂平 | 阅读：181 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/651852295646431232)

【业务场景】

新增采购类单据是带出的单价为什么不对呢?

【解决方案】

新增采购类单据时采购单价是根据价格取数优先级的设置自动带出,需要根据【基础资料】-【价格取数优先级】-【采购类单据取价】的设置逐一判断。

1、检查是否开启取价规则

  1.1 未开启:新增采购类单据不会带出单价。

  1.2 开启:根据优先级带出对应价格,数值越小优先级越高。
![](https://vip.kingdee.com/download/01095fe3d1771f7945cab28efe86f4d58ef5.png)
2、采购价来源分析

  2.1 最近采购价:可通过采购明细表及采购订单跟踪表,查询对应供应商、对应商品的最近的已审核单据的采购价。
![](https://vip.kingdee.com/download/0109af70f87e48fa471c9fdaa2626869e4e9.png)
  2.2 商品价格资料中的采购价:检查【商品价格资料】中对应商品的“采购价”是多少。
![](https://vip.kingdee.com/download/010945fc8b756d7f405cb637955cf78e9a11.png)
  2.3 最近采购成交价:可通过采购明细表及采购订单跟踪表查询最近已审核的采购订单/采购入库单的采购单价和折扣率,按照供应商匹配。
![](https://vip.kingdee.com/download/01097bf4febb15de432b9e707e75470732f0.png)
  2.4 供应商价格资料:检查【基础资料】--【供应商价格资料】列表的<优先级设置>检查优先级设置,点击<价格查询>功能检查这个商品是不是同时设置了多个方案,若同时存在于多个方案时系统会自动取最新方案里面的价格资料。
![](https://vip.kingdee.com/download/010974cceb9a3155413d817aa19d5a4e8283.png)
3、检查商品价格为含税单价参数

在【系统参数】--【基础资料参数】中检查【商品价格为含税单价】参数是否勾选:[商品价格为含税单价参数是否勾选对价格的影响?](https://vip.kingdee.com/article/634919583408733952?productLineId=35&isKnowledge=2&lang=zh-CN)
