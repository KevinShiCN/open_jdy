# 采购销售如何按外部商品信息开单?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：shaoling | 阅读：84 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/703411740108731392)

【问题描述】

采购销售如何按外部商品信息开单?

【解释说明】

做单据的时候,系统是可以按外部商品信息(**供应商商品编码/客户商品编码**)选择商品进行开单操作的;并且可以在单据编辑界面直接新增外部商品库。

【操作流程】

在做单据时,列设置可以设置显示供应商商品编码/客户商品编码。录入外部商品信息(如外部商品名称),选择到对应商品进行开单操作。

以销售出库单为例,可在“列设置”里设置显示客户商品编码、客户商品名称等外部商品信息字段

![](https://vip.kingdee.com/download/010955da1a91c1cc425b9425c1fed10756e4.png)
**1、按客户商品编码选择商品**

在客户商品编码里可录入外部商品编码、外部商品名称、外部规格型号、外部条形码等外部商品信息选择商品

![](https://vip.kingdee.com/download/0109a3524907cb394d48bf543e92ebfc2dd1.png)
选择到外部商品信息后,系统会根据外部商品库的设置带出对应的商品信息

![](https://vip.kingdee.com/download/0109a42c6d6063fa4943863217ca89fd6dba.png)
单据商品信息录入后保存/审核单据操作即可开单成功

![](https://vip.kingdee.com/download/0109c6e1c000bd00460ea225d3336c7281c8.png)

**2、在单据编辑界面新增外币商品编码**

在单据编辑界面选择客户商品编码时,点击“新增”按钮可以打开外部商品库新增编辑界面直接新增。

![](https://vip.kingdee.com/download/010965e4d2fe17044e8c908b0f07239baaa8.png)
