# 删除商品提示“商品标签关联关系”的字段“商品id”引用了此数据,不能被删除

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：Siwennn | 阅读：52 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/784948240700925952)

【问题描述】

删除商品提示“商品标签关联关系”的字段“商品id”引用了此数据,不能被删除

![](https://vip.kingdee.com/download/0109f8ea3835db2c4d3a843f2b56eda10804.png)

【解决方案】

在商品资料的列设置中显示【商品标签】字段,确认商品是否已存在标签信息。

![](https://vip.kingdee.com/download/0109086ac7ce15ad48729570e6f808232323.png)

若存在标签,则勾选商品,通过【批量修改】功能,选中【商品标签】项但不选择具体内容,直接确定。

![](https://vip.kingdee.com/download/0109df6ca666fe3e44239677cb5a526afdaa.png)

在弹出的确认窗口中选择【确定】,即可清空其标签信息。

![](https://vip.kingdee.com/download/01099040c01a2e72468da4e0e146072a103b.png)
完成标签清空后,即可重新勾选并删除商品资料。
