# 怎么禁用/启用科目?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：姜宁8 | 阅读：3506 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352000837454080)

【问题描述】

怎么禁用/启用科目?

【解释说明】

不需要使用的科目可以禁用,等需要使用时再启用;已使用/已结账期间的科目支持禁用/启用。

【温馨提示】

1**、科目禁用后,在录入凭证的时候,无法选择和使用到这个科目。报表仍可以查询到此科目的历史数据;**

**2、多级科目不支持只禁用上级科目,上级科目被禁用,则该科目下的所有明细科目也会自动同步禁用;**

**3、多级科目禁用了明细科目之后也不能直接用上级科目做账,只能用最明细科目做账;**

**4、若科目禁用前还有余额可以直接禁用,也可做凭证转出余额之后再禁用,可查看知识:[科目有余额,能否禁用?](https://vip.kingdee.com/link/s/Z26Mz)**

【操作指引】

菜单栏点击【基础资料】>【科目】

![](https://vip.kingdee.com/download/010960453b12ebe54ee1b9d8aa3f1fa1f7e4.png)

在科目列表界面勾选科目,点击禁用,弹窗提示点击【确定】,支持勾选多个科目批量禁用

![](https://vip.kingdee.com/download/0109027a610591c248a398d69092970580ae.png)

已经禁用的科目,先把禁用科目筛选出来,点击【展开过滤】,按“禁用”状态筛选出禁用的科目,勾选科目,点击<启用>

![](https://vip.kingdee.com/download/0109b32b51cafad24f5c9b9ecd89f65b1c93.png)
