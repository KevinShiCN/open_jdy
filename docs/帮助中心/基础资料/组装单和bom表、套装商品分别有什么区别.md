# 组装单和bom表、套装商品分别有什么区别

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题

【问题描述】

组装单和BOM表、套装商品分别有什么区别

【解释说明】

**组装单:**用于记录将多个子件或组件(如A和B)组装成一个新的成品(如C)的过程, 采购原材料或配件后,将多种原材料或配件通过加工或组装的方式,变为一种新的产品销售,可用组装单处理。

**BOM表:**即物料清单,也叫产品结构或配方,指物料(通常是产成品或半成品、部件)的组成情况,包括该物料由哪些下级物料组成,每一下级物料的用量是多少,其对应的属性等。

**套装商品:**用于处理套装或者原金蝶产品中所指的虚拟件,是指由一组具体物料(实件)组成的、以虚拟形式存在的成套件,所以套装不是一个具体物料,不进行成本核算。在销售出库时以套装子件的形式进行出库、扣减库存、计算收入。
