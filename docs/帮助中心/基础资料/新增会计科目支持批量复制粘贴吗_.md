# 新增会计科目支持批量复制粘贴吗?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：徐某某A | 阅读：51 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/717905854976221696)

【问题描述】

新增会计科目支持批量复制粘贴吗?

【概述】

科目支持批量复制,比如1001下面有多个明细科目100101、100102、100103,可以支持直接复制1001科目然后把1001的明细科目全部复制自动生成新的科目代码。

批量复制不支持指定复制到已有科目下

【操作步骤】

点击【基础资料】>【科目】功能

![](https://vip.kingdee.com/download/0109bab5ce7fafae47339cca8a6a8c78ca54.png)
鼠标选中需要批量复制的科目,之后点击工具栏的<批量复制>功能即可复制生成新的科目代码。
