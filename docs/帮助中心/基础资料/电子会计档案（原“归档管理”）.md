# 电子会计档案（原“归档管理”）

# **1 简介  **

## **1.1 应用场景**

电子会计档案，可以让用户对已经全部处理完成且已结账的年度的账务信息（如：凭证、会计账簿、法定报表银行对账单）进行数据归档，方便后续查询和追溯。

## **1.2 适用版本**

星辰所有版本

## **1.3 系统路径**

基础资料>财务资料>电子会计档案

# 2 操作指南

## 2.1 新增

点击"新建档案"，则进入数据归档的页面，按页面提示录入并操作即可。

![上传图片](https://vip.kingdee.com/download/01009c92e12215424e798d7e8d28c301e5e6.png)

## 2.2 导出

点击右上角的“审计导出服务”可以将其进行导出。选择到对应的账套,可以指定年度导出,导出的文件格式是ais的文件格式

![上传图片](https://vip.kingdee.com/download/0100e1cd418554584b93bf2e1a88ceb7ab11.png)

温馨提示：系统目前仅支出导出AIS格式的审计文件。
