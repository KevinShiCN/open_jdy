# 汇率小数位如何设置?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：Schatz | 阅读：518 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/617164183708364544)

【问题描述】

汇率小数位如何设置?

【操作步骤】

点击【基础资料】>【币别】

![](https://vip.kingdee.com/download/010982af544fdcfd486cb7359860c9ea813f.png)
要设置哪个币别的汇率小数位,点击其“币别编码”

![](https://vip.kingdee.com/download/0109713feabe93ab426b9f990d6848da6d5b.png)

汇率小数位最多设置10位小数位,无法再增加小数位。10位以内需要多少就填写多少,比如填写7.6,就只会显示7.6。

![](https://vip.kingdee.com/download/01092138234cded643cab8c7c31ea0c07959.png)
