# 在制品初始数据（购买了星辰生产管理才会有）

# **1 简介**

## **1.1 应用场景**

在制品初始数据用于记录在启用生产系统前，已在车间下达生产任务，已完成领料，但尚未入库的这部分在制品，包括其数量、已消耗的直接材料成本、人工费、水电费等。

在制品初始数据后续可进行产品入库，与生产任务单一起，参与到后续的完工与在制品归集、费用分配以及生产成本核算中。

## **1.2 操作路径**

基础资料>期初录入>在制品初始数据

## **1.3 适用版本**

星辰3.1以上专业版/旗舰版（购买了星辰生产管理才会有）

# 2 操作指南

打开础资料>期初录入>在制品初始数据，单击右上角的新增，进入在制品初始数据录入页面，入必录字段后，保存即可。

# 3 常见问题

## 1） **在制品初始数据是否可以维护的判定依据是什么？**

通过2个条件进行判断：

• 判断系统中最早的生产任务单所在期间，是否小于当前业务期间，如果是，则不允许维护在制品初始数据，反之则允许；

• 判断系统中已存在的在制品初始数据的制单期间，是否处于当前业务期间，如果是，则允许维护在制品初始数据，反之则不允许。

例如：

①账套启用期间为2023年1期，在2023年3期首次录入了生产任务单，单据日期为2023-03-01，若结账到2023年4期或以后期间，则不允许再维护在制品初始数据；

②账套启用期间为2023年1期，在2023年3期录入了在制品初始数据，若结账到以后期间或以前期间，均不允许录入在制品初始数据，即在制品初始数据只允许在一个业务期间录入。

## 2） **在制品初始数据的制单期间是什么？**

指录入在制品初始数据时，当前所处的业务期间。例如当前系统日期为2023年5月4日，当前业务期间为2023年3期，则录入在制品初始数据时，制单期间为202303。
