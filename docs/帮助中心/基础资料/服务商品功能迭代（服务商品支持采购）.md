# **1 简介  **

## **1.1 功能介绍**

支持如外包服务、技术支持等“不考虑库存、不核算成本、只计入往来”的服务商品进行采购业务开单，实现服务商品采购流程闭环。

## **1.2 功能特性**

• 服务商品支持多单位（不支持浮动换算）及其他更多商品信息

• 服务商品支持维护采购类价格

• 服务商品支持采购流程

## **1.3 适用版本**

星辰3.0以上所有版本

### 2.3.2 服务商品与普通商品在采购单据上的差异

①采购入库单的服务商品行可不维护仓库。

②采购入库审核后不增加服务商品的库存，也不支持服务商品的库存查询。

③可在单据上查看服务商品的入库成本，但服务商品不参与出库成本核算。

![](https://vip.kingdee.com/download/010070976ff3060f4d66b45a70939f5bf78e.png)

④服务商品不参与采购费用的分摊，包括同单采购费用分摊、费用分摊单分摊。

⑤库存查询表、预计可用库存查询表、库存预警表、商品收发明细表、商品收发汇总表等库存类报表均不支持查询服务商品的库存及收发明细。

### 2.3.3 其他服务商品相关的更新

①销售订单上若包含可采购的服务商品，支持以销订购流程；

②销售单据上若包含可采购的服务商品，支持直接复制为采购类单据；

③销售单据复制为采购类单据时，若商品不可采购，系统支持复制成功并剔除不可采购商品后给出提示；同理，采购单据复制为销售类单据时，若商品不可销售，系统支持复制成功并剔除不可销售商品后给出提示。

④包含服务商品的销售单据或采购单据复制为库存类单据时，系统支持复制成功并剔除服务商品后给出提示。

⑤包含服务商品的单据进行发送给互联供应商、发送给互联客户时，单据能否发送成功，取决于接收方目标单据上是否支持服务商品。例如，采购订单（含服务商品A）发送给供应商（供应商接收后生成销售订单）时，若服务商品A在供应商账套内不可销售，则该采购订单发送失败。
