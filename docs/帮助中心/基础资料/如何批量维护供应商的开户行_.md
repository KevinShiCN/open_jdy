# 如何批量维护供应商的开户行?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：云社区用户8x80607 | 阅读：259 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/619699136203157248)

【问题描述】

如何批量维护供应商的开户行?

【操作步骤】

1、批量新增供应商信息并维护开户行。

【基础资料】-【供应商列表】,点击右上角“引入”按钮,模板管理-点击系统预置模板进入模板编辑页面,先复制一个模版修改,或者直接新增一个模版

![](https://vip.kingdee.com/download/0109224db6c269184108b65180cf30464b44.png)把需要引入的字段,如开户银行字段打开【是否允许引入引出】,维护模板编码、模板名称,点击保存
![](https://vip.kingdee.com/download/0109d40fb2a2a5f144c8a4d04329c07a5643.png)
点击引入,勾选刚才保存的模板,点击下载
![](https://vip.kingdee.com/download/0109a0f1b7da17e945a5b603cd4f499347f4.png)
模板里填写好供应商信息与开户行号,保存模板。进行引入该模板即可![](https://vip.kingdee.com/download/01091dc5d626b0e24374b4bc14894717c4ba.png)
2、批量覆盖维护开户行。根据步骤一,新增好开户行字段打开的模板。勾选供应商,点击【按引入模板引出】,选择开户行模板,点击确定。如下图所示![](https://vip.kingdee.com/download/01096cea8dcc5a734d7b94325bb8bc5c924c.png)
模板里批量维护好开户行号,保存模板![](https://vip.kingdee.com/download/0109ce8fecdec4b741c6b3307dc50c58f962.png)
引入方式选择【覆盖新增】导入完成即可更新开户行字段![](https://vip.kingdee.com/download/01092ab7a1fb1a5c488990e28cd42b605610.png)
