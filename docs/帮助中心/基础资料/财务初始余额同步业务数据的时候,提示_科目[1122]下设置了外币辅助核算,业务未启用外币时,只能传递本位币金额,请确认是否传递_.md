# 财务初始余额同步业务数据的时候,提示:科目[1122]下设置了外币辅助核算,业务未启用外币时,只能传递本位币金额,请确认是否传递?

> 作者：胡大大 | 阅读：133 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/703054791819286528)

【问题描述】

财务初始余额同步业务数据的时候,提示:科目[1122]下设置了外币辅助核算,业务未启用外币时,只能传递本位币金额,请确认是否传递?

![](https://vip.kingdee.com/download/0109f363717b392445f7ae14171c360847a7.png)
【原因】

因为1122科目启用了外币核算,但是进销存系统参数没有启用外币。如进销存不需要启用外币管理可以直接点击确定引入数据即可,如进销存需要启用外币管理则在【系统设置】-【系统参数】-【进销存参数】中勾选"启用外币"即可
