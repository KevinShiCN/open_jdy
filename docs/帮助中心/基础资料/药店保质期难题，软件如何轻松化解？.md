# 药店保质期难题，软件如何轻松化解？

> 作者：毛毛酱 | 阅读：66 | 有用：0

![上传图片](https://vip.kingdee.com/download/0100c1b2bd90ec0f45108079caab4785e059.png)

（药店里，顾客进进出出，收银台的扫码声和店员的答疑声此起彼伏，小K和小D在仓库整理药品）

小K对小D说：“小D，上次前台说顾客急着买那款止咳糖浆，系统显示还有20盒库存，可仓库里就找到5盒，还是临期的，保质期还都对不上，后面客户直接去别家买了，唉，你说这事儿办的。”

小D在电脑前快速操作着库存管理软件，回应道：“啊，那肯定是批次出岔子了。咱们这软件有带保质期的批次管理功能，每批止咳糖浆都可以记录详细的‘电子履历’呢，是不是咱们没用上。”

小K：“还有这么厉害的功能？快教教我”

小D便打开电脑操作了起来：“先打开【系统设置】-【系统参数】-【基础资料参数】中,勾选“启用批次管理”和“启用保质期管理”,保存。

![上传图片](https://vip.kingdee.com/download/0100642663c1e5ab49968e6ff41c3768ccca.png)

![上传图片](https://vip.kingdee.com/download/01003444aa6b17074812a729dcfbef8d979d.png)然后你去【基础资料】-【商品信息】新增要体现批次和保质期的商品，比如我们刚刚说的止咳糖浆

![上传图片](https://vip.kingdee.com/download/0100d613291ceb674f2fb77571c5f399617f.png)

![上传图片](https://vip.kingdee.com/download/0100ff59aa1ea22a45ac9dac213de47ffbd8.png)

新增的时候记得要在【商品特性】里开启“批次管理”和“保质期管理”，保质期这需要额外设置“保质期单位”、“保质期”、“预警天数”、“到期日计算方式”和“修改到期日”，再进行保存。

![上传图片](https://vip.kingdee.com/download/0100977b59ad8bab4c97ae84804d44f9634e.png)

在完成新增后，我们可以进行开单操作，入库时就可以录入具体的批次和保质期信息。”

![上传图片](https://vip.kingdee.com/download/0100a8cc0122123f42af979999e4c1273e05.png)

小K又问到：“那新增好后我去哪里看哪些临期呢？系统能提醒我吗？”

“当然可以了，单据保存审核好后就能去【库存管理】-【保质期预警查询表】里面查询具体商品的批次、生产日期、到期日、剩余到期天数以及预警天数。左上角的【登录时首页预警提醒】勾选上，还能去软件首页看到预警消息，再也不怕商品过保质期了“。

小K竖起大拇指，赞叹道：“这功能太实用了，以后咱们药店管理药品就全靠它保驾护航了。有了这软件，咱们既能避免缺货丢生意，又能保证顾客用药安全，简直一举两得！”

小D自信满满地说：“没错，以后咱们药店的经营肯定会越来越顺，顾客也会越来越信任咱们！
