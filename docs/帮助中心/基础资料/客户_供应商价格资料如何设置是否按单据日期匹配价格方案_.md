# 客户/供应商价格资料如何设置是否按单据日期匹配价格方案?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：Simone邓 | 阅读：29 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/799443878100610816)

【问题描述】

客户/供应商价格资料如何设置是否按单据日期匹配价格方案?

【解释说明】

生鲜行业,企业A向某食堂供应蔬菜,商品价格变化快,每周都有价格波动;需提前一周确定商品的价格并提前录单,开单时按单据日期取价。

在客户价格资料、供应商价格资料的“优先级设置”中,新增参数“按单据日期匹配价格方案”。若勾选该参数,开单时按单据日期匹配价格方案;若不勾选,则按单据的创建时间匹配价格方案

**注意:该功能仅专业版支持,且目前处于灰度测试阶段。如需提前体验,请复制工作台的服务 ID 到售后在线服务界面发送给人工客服,申请加入灰度测试。**

【操作步骤】

在客户/供应商价格资料列表,打开【优先级设置】-【参数】,可勾选“按单据日期匹配价格方案”,截图以客户价格资料为例,供应商价格资料同理。

![](https://vip.kingdee.com/download/01092f7eec59f56b4b2b95668e77f7de6536.png)
**勾选“按单据日期匹配价格方案”:**

- 1、开单取价时,按单据日期匹配价格方案中的“开始时间”和“结束时间”,获取有效的价格与限价。由于单据日期不含时分秒,匹配时仅区分年月日。
- 2、参与匹配的价格方案须为非禁用状态。
- **示例:**当前系统日期为2025-12-23,在客户价格资料新增方案1的开始时间和结束时间为2025-11-01~2025-12-01,新增方案2的开始时间和结束时间为2025-12-02~2025-12-31,此时,在客户价格资料列表中看到方案1的执行状态为“已结束”的状态,但只要该方案未被禁用,按单据日期取价时,只要单据日期落在了方案1的有效期内,就会取到这个价格方案。(执行状态是用当前系统日期与方案有效期比较的,对于2025-12-23来说,方案1已于12月1日结束,所以显示为已结束的状态,但该方案仍然有效。)
- ![](https://vip.kingdee.com/download/0109927c7a895c4544bc8cf4a7d42a8e8b46.png)新增销售订单时,默认单据日期为2025-12-23,此时取方案2的价格;若修改单据日期为2025-11-06,系统将弹出提示,询问是否按新日期重新取价,由用户确认是否重新取价,点击确定,将按新的日期重新匹配价格方案进行取价,获取到方案1的价格,点击取消,则不重新取价。
- ![](https://vip.kingdee.com/download/01093abbed3c275d48cdb7fba291ce114c98.png)**不****勾选****“按单据日期匹配价格方案”:**
-
- 1、开单取价时,按单据的制单时间匹配价格方案中的“开始时间”和“结束时间”,以获取有效的价格与限价。由于制单时间精确到年月日时分秒,匹配时亦区分价格方案有效期中的年月日时分秒。
- 2、参与匹配的价格方案须为非禁用状态。由于制单时间是获取的系统时间,用户无法修改,所以在价格资料列表根据系统时间与价格有效期比较得到的执行状态“未开始、执行中、已结束”就能准确的标识出来这个方案是否可以在开单时被读取,未开始、已结束的方案在开单时按制单时间是无法匹配到的,只能匹配到执行中的价格方案。
- ![](https://vip.kingdee.com/download/01091df174f13ad242dca6fb97906d53389a.png)3、开单时修改单据日期,系统不会提示重新取价。
