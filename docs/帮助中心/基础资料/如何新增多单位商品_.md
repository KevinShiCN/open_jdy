# 如何新增多单位商品?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：winnie_wen | 阅读：1448 | 有用：1

【问题描述】

如何新增多单位商品?

【相关视频】

[商品启用多单位管理和使用](https://vip.kingdee.com/school/211509161518128128?productLineId=35&lang=zh-CN&cosmic_type=video)

【操作流程】

多计量单位商品录入说明:

a、在【基础资料】-【商品信息】中,新增商品;

![](https://vip.kingdee.com/download/0109d36a1dd0f9464b80bb35018cb790e242.png)

b、计量单位选择最小单位,并勾选【启用多单位】;

![](https://vip.kingdee.com/download/0109ef872ecc49e24f74a8a50c43d830c594.png)

c、在【单位设置】中,设置多个计量单位之间的换算关系(比如1kg=1000g,1000即为换算率)

**tips:目前支持设置4个单位,一个基本单位,3个副单位;支持启用一个浮动单位**

d、若商品存在浮动换算关系,在【单位设置】中第一行单位后勾选【浮动换算】,并在第一行单位处维护浮动单位与基本单位之间的换算关系,则第一行单位为浮动单位。其他计量单位可与基本单位和浮动单位进行换算。

![](https://vip.kingdee.com/download/01099ebc9f164a3947d18509e086dab97d3a.png)
在录入换算率时,是否可以输入小数位,根据基本信息中的计量单位小数位决定,[计量单位的小数位](https://vip.kingdee.com/link/s/ZQ9Fj)设置的是几位,换算率就可以录入几位小数

![](https://vip.kingdee.com/download/0109b002f80ba7624bc4bf1334b4df7a6972.png)

e、补充录入其他商品信息保存即可。

f、多单位商品支持设置默认开单单位,参考:[多单位商品如何设置开单默认单位?](https://vip.kingdee.com/link/s/Zj5D0)

【温馨提示】

商品启用辅助属性管理,各辅助属性**不支持**分别设置多单位功能。
