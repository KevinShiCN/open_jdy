# 如何删除部门?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：杨桂平 | 阅读：650 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/510828869155325952)

【问题描述】

如何删除部门?

【解释说明】

未使用过的部门可直接删除;**已被使用的部门不允许直接删除**,需先删除关联该部门资料的对应单据,方可执行部门资料删除操作。

若部门后续不再使用但需保留历史业务数据,**建议优先操作禁用**;如确需删除部门,需同步清理相关业务记录,且**务必在删除单据前完成数据备份**,避免数据丢失。

【操作步骤】

1、在【基础资料】-【业务资料】中,找到【部门】

![](https://vip.kingdee.com/download/0109946ec912772a423088f66f6625a5acea.png)

2、打开【部门】页签,选择需要删除的部门,点击【删除】即可。

![](https://vip.kingdee.com/download/010979972f4ded144c04a03964a133678ac3.png)
