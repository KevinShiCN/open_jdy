# 查询基础资料时为什么找不到对应的数据?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：shaoling | 阅读：102 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/715370694711617024)

【问题描述】

查询基础资料时为什么找不到对应的数据?

【解释说明】

若查询基础资料时找不到对应数据,管理员可检查子账号是否已获得基础资料查询的数据授权,或确认基础资料是否已被禁用且查询时未勾选显示禁用状态。

【操作流程】

1、管理员账号检查是否对子账号的基础资料查询设置了数据授权

管理员账号登录后,从【系统设置】>【用户管理】进入用户管理列表页面

![](https://vip.kingdee.com/download/0109683d8bc102b2485abab3325a856ac92c.png)

在用户管理列表页面,选择对应的用户点击<授权>按钮

![](https://vip.kingdee.com/download/01097350122f9a0a4b1fbe9b163fec98dc57.png)

在【用户授权】>【数据授权】页面,找到相关数据授权项,以客户数据授权为例,检查对应的 “客户” 和 “客户分类” 是否已进行数据授权(若未设置表示有所有客户数据权限,需检查是否是禁用了)

![](https://vip.kingdee.com/download/0109fd7efe0699f345aeb745cff7eb0083bd.png)



2、检查基础资料是否已禁用,基础资料查询时,可取消“启用”状态过滤条件

以客户基础资料为例,从【基础资料】>【客户】进入客户列表页面

![](https://vip.kingdee.com/download/0109a7e7409e9a0a474d9093a8962572bbfc.png)
在客户列表页面,取消勾选 “可用” 状态过滤条件,查看是否显示对应客户的基础资料信息
![](https://vip.kingdee.com/download/0109fa9e0fe8576b48c88fc264146e2902d5.png)
