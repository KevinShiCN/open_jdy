# 如何新增科目？

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题, 快速入门
> 作者：杨桂平 | 阅读：1946 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/360751232878646272)

【操作流程】

1、点击基础资料-科目,点击[新增]新增一级科目,根据科目编码结构输入科目代码,输入科目名称,选择科目的其他属性,保存;

2、点击科目名称前方的[+]号即可新增该科目的明细科目,在二级科目前方点[+]号,可新增三级科目,以此类推
