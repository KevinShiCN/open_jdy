# **1 简介**

## 1 应用场景

当财务科目定义完成后，需要根据企业当前的财务状况设置一些科目的初始余额，以便后续进行账务处理。 财务初始余额包括初始余额维护 、 试算平衡检查 和引入引出 。

期初录入数据根据会计启用期间不同有所不同，如果是第1期启用，则只需要科目期初余额（即年初余额），如果是年度中间启用，还需要录入累计的发生额。

期初录入可以和启用期的日常业务处理同时进行，只需要在结账前完成初始余额录入工作即可。系统进行结账时，同时结束期初录入工作。

## **1.2 适用版本**

星辰3.1以上所有版本

## **1.3 操作路径**

基础资料>期初录入>财务初始余额

# 2 操作指南

### 2.1 新增

在上方有显示对应的科目类别，单击对应类别可进行对应类别下科目初始余额录入。

![](https://vip.kingdee.com/download/0100b261b378a9804b6db8fa6e740a7e2b4d.png)

根据科目属性不同，财务初始余额录入会有所不同（仅以第1期启用为例说明）

    1）普通科目：直接在上图界面录入年初余额即可

    2）外币科目：科目下设外币核算时，需要单击【币别】下方的下拉框，选择对应的币别录入原币和本位币对应的年初余额。

![](https://vip.kingdee.com/download/01005ac65dd7a245409b80d4f8da06477979.png)

    3）数量金额科目：科目下设数量金额辅助核算时，对应的科目需要录入数量和金额的年初余额。

    4）辅助核算科目：科目下设辅助核算时，科目前方会有一个【+】号，单击会打开辅助核算明细录入新页签。

### 2.2 试算平衡

根据记账基本原则，财务初始余额录入必须借贷平衡。单击右上角的“试算平衡”，系统会进行试算平衡检查。
