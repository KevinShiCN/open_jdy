# 标准工时小数位如何设置?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：76 | 有用：0

【问题描述】

标准工时小数位如何设置?

【解释说明】

商品资料中标准工时录入时,**小数位默认是2位,**无法增加和减少。

另外,在执行商品资料引入操作时,该字段同样受到严格管控,引入的表格中的标准工时不能超过2位小数,否则无法引入成功。
