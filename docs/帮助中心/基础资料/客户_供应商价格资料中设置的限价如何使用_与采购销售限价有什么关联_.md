# 客户/供应商价格资料中设置的限价如何使用?与采购销售限价有什么关联?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：Simone邓 | 阅读：6 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/819375708957650688)

【问题描述】

客户/供应商价格资料中设置的限价如何使用?与采购销售限价有什么关联?

【解释说明】

在设置客户 / 供应商价格资料时,可维护**限价**字段,并在价格资料列表中进行**限价单据设置**与**优先级设置**,开单时系统将按照价格资料配置执行校验。

若同时在基础资料新增了**采购 / 销售限价方案**,将**优先执行**客户 / 供应商价格资料中**优先级最高**的限价;若该客户 / 供应商价格资料未设置限价,则再读取采购限价方案中的限价。

【操作流程】

1、客户 / 供应商价格资料时录入**限价,**以供应商价格资料为例:

![](https://vip.kingdee.com/download/01092f19cfac2d364c9db41c85ff7b6ed6ab.png)
2、在客户 / 供应商价格资料列表操作**限价单据设置**

![](https://vip.kingdee.com/download/010935ea63933fcd4e7f81fb512dd277dd10.png)
若有多个不同的价格资料方案可设置优先级进行匹配

![](https://vip.kingdee.com/download/01093830568cba52421ab34e087c6cde7f70.png)
3、【基础资料】-【价格取数优先级】,开启供应商价格且上移到最上面

![](https://vip.kingdee.com/download/01094b8454fbd36a4f9f9da556145c36771a.png)

4、若同时在基础资料新增了**采购 / 销售限价方案**,可参考:[采购销售限价如何设置或取消?](https://vip.kingdee.com/link/s/ZVHCu)

![](https://vip.kingdee.com/download/0109d8ff90e614174308af4dd496730fc271.png)

开单时优先执行供应商价格资料中优先级最高的限价 ,若供应商价格资料未设置限价,再读取采购限价中的限价方案。
