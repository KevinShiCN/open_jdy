# 采购销售限价方案中的限价对象、限价依据是取的不含税价格还是含税价格?

【问题描述】

采购销售限价方案中的限价对象、限价依据是取的不含税价格还是含税价格?

【解释说明】

**1、限价对象**

采购的限价对象可以选择“实际含税单价”、“实际不含税单价”、“含税单价”、“单价”,销售的限价对象可以选择“实际含税单价”、“实际不含税单价”、“含税单价”、“单价”、“销售折扣”。

是否为含税价格,根据所选对象决定

![](https://vip.kingdee.com/download/0109decb0739dcd84215a32da6dc28468178.png)![](https://vip.kingdee.com/download/0109bea3b09770a54fc1b1c6a3d49ed09ad5.png)

**2、限价依据**

采购限价依据包括:采购价、最近销售单价、批发价、零售价、最低销售价、最近销售含税单价、最高采购含税单价、历史最低采购价、最低采购含税单价、历史最高销售单价、最高销售含税单价、历史最低销售价、最低销售含税单价

销售限价依据包括:最高采购价、批发价、零售价、价格等级(20个)、历史最高采购价、最高采购含税单价、最低采购含税单价、历史最高销售单价、最高销售含税单价、最低销售含税单价
