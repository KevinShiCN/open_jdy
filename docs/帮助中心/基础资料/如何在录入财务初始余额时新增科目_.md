# 如何在录入财务初始余额时新增科目?

【问题描述】

如何在录入财务初始余额时新增科目?

【解释说明】

录入财务初始余额界面**不支持**直接新增科目,可先在科目列表新增科目再去录入财务初始余额。

在【基础资料】-【科目】操作新增科目,新增科目具体操作参考:[如何新增科目?](https://vip.kingdee.com/knowledge/360751232878646272?productLineId=35&isKnowledge=2&lang=zh-CN)
