# 如何设置登录用户关联职员？

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题, 快速入门
> 作者：移动生产白建凡 | 阅读：2994 | 有用：7

# 1 问题描述

如何设置当前用户关联的职员信息？

# 2 业务场景

- 巡店访销、客户拜访、进店打卡等功能涉及到不同部门、不同职员间的数据权限管控，使用这类功能，必须将用户与职员及部门关联起来
- 拣验货场景下，需将当前用户与职员进行关联，关联后“拣货员”字段会自动带出当前用户对应的职员信息

# 3 解决方案

（1）先在“基础资料<业务资料<职员”中设置好职员及所属部门

![上传图片](https://vip.kingdee.com/download/0100256915abec4c402e9a79b00de7cd53c1.png)

![上传图片](https://vip.kingdee.com/download/01005a12f09f3ff6448b84aad2717281ab6a.png)

（2）在“系统设置<基础设置<用户管理”中设置用户关联职员

- 点击登录名进入用户设置页面

![上传图片](https://vip.kingdee.com/download/010042c5f277b89f4057ba723113be9dd48c.png)

- 在“关联职员”中选择该用户关联的职员后保存，即完成用户与职员的关联

![上传图片](https://vip.kingdee.com/download/010006c0d288acb4471f8b887efc61ce6b2d.png)

![上传图片](https://vip.kingdee.com/download/0100ab366447097c4027b71b64d5a080ff59.png)
