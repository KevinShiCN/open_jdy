# 有外贸业务,是否所有科目都要启用外币核算?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：云社区用户8x80607 | 阅读：480 | 有用：1

【问题描述】

有外贸业务,是否所有科目都要启用外币核算?

【解释说明】

没有必须设置,主要取决于具体的会计处理和业务情况。如果涉及有外贸业务相关的科目需要做外币业务,那么就需要启用。
