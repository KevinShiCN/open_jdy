# 一个商品是否可以同时启用多种商品特性（如批次+保质期+序列号+辅助属性）？如何启用？

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题, 快速入门
> 作者：进销存赵嘉丽 | 阅读：1341 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/163293963595065600)

【问题概述】

一个商品是否可以同时启用多种商品特性（如批次+保质期+序列号+辅助属性）？如何启用？

【解决方案】

答：支持。应用流程如下：打开基础资料>商品资料>商品信息，新增一个商品，可同时启用多种商品特性，勾选完毕，单击“保存”即可。

![上传图片](https://vip.kingdee.com/download/010048803ca69b8d41b5a83d414406ef8cdb.png)

【适用版本】

星辰3.1以上所有版本
