# 商品价格资料界面如何显示商品图片?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：68 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/639630439467812096)

【问题描述】

商品价格资料界面如何显示商品图片?

【解决方案】

在基础资料-商品价格资料,列设置开启显示图片即可。

![](https://vip.kingdee.com/download/01094e20e3906d3241c09e3a017053e229af.png)
鼠标放到图片上,图片会自动放大查看。
