# 如何查询/导出辅助资料的明细?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：云社区用户55os622 | 阅读：183 | 有用：0

【问题描述】
如何查询/导出辅助资料的明细?

【操作指引】

菜单栏点击【基础资料】>【辅助资料】,进入辅助资料明细列表界面![](https://vip.kingdee.com/download/0109ac6c5668d5694317845578f4f31bd4da.png)
左侧选择需要的辅助资料大类进行查看

![](https://vip.kingdee.com/download/010915d2544a45914958bdb3aa46d8dbde97.png)
展开右上角引出按钮,支持按3种方式引出辅助资料明细
![](https://vip.kingdee.com/download/01099fa8363835284f779e8dbb4ec7392037.png)
