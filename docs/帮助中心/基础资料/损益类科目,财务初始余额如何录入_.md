# 损益类科目,财务初始余额如何录入?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：1030 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/690002192765437440)

【问题描述】

损益类科目,财务初始余额如何录入?

【操作指引】

**一、年初启用账套****(即1月启用的账套)**

1.如参数勾选了“期末结账前必须结转损益”和“年结前必须结转损益”
![](https://vip.kingdee.com/download/0109eb7578fb595b49b6a4fca0e7d336b28a.png)
2.则损益类科目“期初余额”与“年初余额”不允许有值,无需录入损益类科目的期初余额**,**录入数据保存,系统会出现提示

![](https://vip.kingdee.com/download/01096f80f9afa4e5429b91cdeedb0048b8fb.png)

3.若参数未勾选“期末结账前必须结转损益”和“年结前必须结转损益”,年初余额允许录入数据保存

![](https://vip.kingdee.com/download/0109518076e1b5a743388b0fb15c52f3628a.png)

**二、年中启用账套(2-12月份启用)**

若是年中启用账套,则需要填写损益类科目的财务初始余额**。**

**损益类科目财务初始余额录入规则:“本年累计借方”=“本年累计借方”=“本年累计损益发生额”**

如账套启用时间是10月,录入财务初始余额损益科目时,本年累计借贷,按1-10月科目余额表中的本年累计借贷录入。“本年累计损益发生额”和本年累计借贷的金额保持一致即可

![](https://vip.kingdee.com/download/01097558313ead5948359ac9a1dc2c279678.png)
