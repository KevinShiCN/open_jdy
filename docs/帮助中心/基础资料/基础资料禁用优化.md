# 基础资料禁用优化

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：功能介绍, 新特性, 快速入门
> 作者：进销存熊诗琴 | 阅读：584 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/682972366674121216)

## 1 简介

### 1.1 功能介绍

在操作单据时，需要对禁用的基础资料进行校验，避免已禁用的商品等信息还在开单。

### 1.2 适用版本

星辰3.1以上所有版本

## 2 功能特性

### 2.1 禁用基础资料余额校验

1、系统参数>基础资料参数>基础资料禁用参数中，增加商品、客户、供应商、仓库、账户基础资料禁用时的余额校验，可选择不控制、仅提示、严格控制：

![](https://vip-admin.kingdee.com/download/0109206e42ab4de349d2ab737dc4012af7bb.png)

| 控制强度 | 禁用时校验逻辑 |
| --- | --- |
| 不控制 | 不判断余额，可直接禁用 |
| 仅提示 | 判断余额，并弹窗提示，可选择继续禁用 |
| 严格控制 | 判断余额，有余额时不允许禁用 |

### 2.2 操作单据时禁用基础资料校验

1、系统参数优化

系统参数>基础资料参数>基础资料禁用参数中，增加单据操作时对禁用基础资料的校验强度控制（默认严格控制）：

![](https://vip-admin.kingdee.com/download/01091d29ff4b2628441683af855ab36ceb4d.png)

| 控制强度 | 禁用时校验逻辑 |
| --- | --- |
| 不控制 | 不判断基础资料的禁用状态 |
| 仅提示 | 判断单据中基础资料的禁用状态，可选择继续操作单据 |
| 严格控制 | 存在禁用状态的基础资料时，单据不允许操作 |

- 禁用基础资料的校验包括：

商品、辅助属性、SKU编码、供应商/结算供应商、客户/结算客户、供应商商品编码、客户商品编码、部门、业务员、仓库、仓位、币别、计量单位、结算期限、物流公司及其他辅助资料、结算账户、结算方式、退货原因（辅助资料）、费用单据体中的供应商/支出类别/部门/业务员/付款账户/付款方式等。



2、单据操作优化

（1）单据详情中操作保存、提交、审核时，若存在已禁用或未审核的商品，会进行弹窗提示（单据列表批量操作时，不支持一键删除商品行）：

![](https://vip-admin.kingdee.com/download/010923814c4c96cc4377a32a701a7e82ec6d.png)

支持在弹窗中一键删除单据中已禁用或未审核的商品。

注意：单据启用工作流后，审核单据时不支持批量删除商品的弹窗；

禁用基础资料校验系统参数为仅提示时审核自动通过，为严格控制时会提示异常商品，如

![](https://vip-admin.kingdee.com/download/01090f85af9581324ca08acc729d283fe6d3.png)

（2）若单据表头的客户/结算客户、供应商/结算供应商已禁用，则单据下推、选源单时会进行提示：

- 系统参数为仅提示：

 ![](https://vip-admin.kingdee.com/download/01096a43f358a0b94efbbc9475e3e7cf57a5.png)

- 系统参数为严格控制：

 ![](https://vip-admin.kingdee.com/download/0109790e9aa736cd4c3982f091e40229f5da.png)

### 2.3 报表支持筛选禁用基础资料

1、商品筛选

报表的筛选条件中，商品筛选器中增加“显示禁用”、“显示未审核”的勾选项，按用户记住上一次的勾选项：

![](https://vip-admin.kingdee.com/download/0109ff4e678a926d4c5287fe43e332d595d4.png)

2、其他基础资料筛选

客户、供应商、业务员、仓库、仓位筛选器中，增加“显示禁用”的勾选项，按用户记住上一次的勾选项（以仓库为例）：

![](https://vip-admin.kingdee.com/download/010957be5c2d2ac8464e8ccc477261f137b4.png)

3、关于未审核、已禁用的基础资料，只能在筛选器弹窗中进行筛选，下拉选择模糊搜索时，还是只能搜索到已审核、未禁用的数据。
