# 辅助属性组合上限是多少

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：财税李炳全 | 阅读：694 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/568601879312241152)

【问题描述】

 辅助属性组合上限是多少?可以再增加吗

【解释说明】

**单个商品的辅助属性组合**上限是1000个,单个辅助属性商品中所有属性组可组成的属性组合总数,1000个不区分启用/禁用状态,即使禁用了也会算进去。

组合数量上限是经过性能测试设计的,再多会影响系统运行,不会再放开上限。![](https://vip.kingdee.com/download/0109a4873818de7b414c864152f38d8b84bc.png)
