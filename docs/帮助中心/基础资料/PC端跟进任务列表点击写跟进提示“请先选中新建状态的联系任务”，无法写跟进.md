# PC端跟进任务列表点击写跟进提示“请先选中新建状态的联系任务”，无法写跟进

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题, 进阶技巧, 精华汇总
> 作者：Freya_peng | 阅读：702 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/393781273380207616)

# **1 问题描述**

电脑端跟进任务列表点击写跟进提示“请先选中新建状态的联系任务”，无法写跟进

![上传图片](https://vip.kingdee.com/download/0100a9bf9963fa6a4bf6bf8a67598708b380.png)

# **2 原因分析**

电脑端在跟进任务中选中任务后写跟进，只有“跟进类型”为“联系”且“跟进状态”为“新建”的任务可以直接写跟进

# **3 解决方案**

1）判断选中的任务，“跟进状态”是否为“新建”，只有新建状态的任务直接写跟进

2）判断选中的任务，“跟进类型”是否为“联系”，只有联系支持直接写跟进

![上传图片](https://vip.kingdee.com/download/0100d36025f6b74e452da173ebe0622ea7ab.png)

备注：由于电脑端不支持打卡，所以暂时不支持跟进类型为客户拜访的在电脑端直接写跟进
