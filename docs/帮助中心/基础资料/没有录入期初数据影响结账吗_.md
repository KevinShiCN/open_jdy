# 没有录入期初数据影响结账吗?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题

【问题描述】

没有录入期初数据影响结账吗?

【解释说明】

**是否录入期初数据(****包含财务模块和业务模块****)均不影响结账。**

**若未录入初始数据,结账时系统会温馨提示,可忽略提示继续结账。也支持后续再补录****期初,反结账到启用期间再录入即可。**
