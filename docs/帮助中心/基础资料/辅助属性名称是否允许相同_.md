# 辅助属性名称是否允许相同?

> 作者：云社区用户0531183 | 阅读：82 | 有用：0

【问题描述】

辅助属性名称是否允许相同?

【解释说明】

辅助属性名称不允许相同,新增的辅助属性名称与已有的相同时会提示:辅助属性名称:”值“XX” 已存在,请输入其他值。
