# 打印多排条码时,如何设置条码间的间距?

【问题描述】

打印多排条码时,如何设置条码间的间距?

【解释说明】

通过【系统设置】—【打印模板】,找到需要调整的打印模板
![](https://vip.kingdee.com/download/0109f18e5a2ebcc34fa491a921e096051241.png)

找到需要修改的模板

![](https://vip.kingdee.com/download/0109338ea379be044016b1bf3204f90230e5.png)

模板修改前打印预览的效果如下:

![](https://vip.kingdee.com/download/010918d523245f1349f6983cf77edb72249f.png)

给模板添加空白文本框

![](https://vip.kingdee.com/download/0109ebb6a6da417648e7b5fff2004b8b45b8.png)

修改后,保存模板,打印预览看看效果(相比修改前,多排条码间距明显变宽)

![](https://vip.kingdee.com/download/01097acf1ed196db4ece86cf380f6e4e3df7.png)
