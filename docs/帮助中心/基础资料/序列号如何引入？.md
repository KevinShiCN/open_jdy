# 序列号如何引入？

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：642 | 有用：5
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/371691636558022400)

【问题概述】

序列号如何引入？

【解决方案】

进销存云中，商品期初资料、出入库单据均支持引入序列号，采购申请单 、销售订单、采购订单**不支持**。系统有预置的引入模板。

![上传图片](https://vip.kingdee.com/download/0100b2b5333f2f9442b8b32cfc0382bf8992.png)

- **期初的引入**

在引入引出模板列表下载引入模板后，录入序列号到"序列号字段，然后再引入即可。

![上传图片](https://vip.kingdee.com/download/010049ea703d647e4f6c99c96fa68d019d73.png)

>
**注意：星辰商品期初引入不支持引入同商品不同单位成本的序列号期初。**

- **单据的引入**

在引入引出模板列表下载引入模板，单据的引入模板上，序列号对应的字段是“序列号组合”。同一个商品，序列号以分号隔开再导入，数量请填写序列号的个数。

例如：SN001;SN002;SN003，数量对应就是3，可以参考下图。

![上传图片](https://vip.kingdee.com/download/0100a1714f45c6584b0197f38ffa23f82158.png)

【适用版本】

星辰3.1以上所有版本
