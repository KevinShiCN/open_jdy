# 如何查询禁用商品?商品如何禁用/启用?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：杨桂平 | 阅读：2067 | 有用：0

【问题描述】

如何查询禁用商品?商品如何禁用/启用?

【名词解释】

**禁用商品:**当商品暂时停用并设置为 “禁用” 状态后,后续录入单据或凭证时,该商品不会显示在可选列表中,无法被选中使用。若已有订单包含该禁用商品,后续将无法对该订单执行下推、物料需求计算等相关操作。

**商品禁用后有什么影响:**

1、**以前的数据仍可以保留查询,只是禁用后不可在新录单据界面选择到该商品,若商品还有数量、成本,还会体现在报表里的。建议禁用商品前先将该商品的数量、成本通过其他出库单转出;**

**2、商品禁用之后商品编码和商品名称仍然存在,新增商品不允许录入重复的商品编码,参数允许商品名称的情况下允许新增相同名称的商品,[新增/引入商品资料时是否允许重复?](https://vip.kingdee.com/link/s/Zmi8z);**

**3、禁用的商品不能进行物料需求计算;**

**4、目前不支持自动禁用商品,商品禁用需手工操作,禁用后支持继续启用;**

**5、禁用的商品若需要修改,需要先启用商品后,再进行编辑修改;若商品启用了审核,则需要反审核之后再修改,修改完成后,重新审核即可。**

【操作流程】

菜单栏点击【基础资料】>【商品信息】

![](https://vip.kingdee.com/download/010950e6640b3a824cc697ed9f689ef70b57.png)

在商品列表页面,勾选商品后点击<禁用>按钮,即可将商品禁用,支持选择多个商品,批量点击<禁用>

![](https://vip.kingdee.com/download/0109a087eeb550194378ac9cfb7268d8c363.png)

若需查询禁用商品或重新启用禁用商品,可点击<展开过滤>

![](https://vip.kingdee.com/download/010981f0d92b3ce74028a9771e83ca13d599.png)
“状态”栏选择“禁用”即可显示出禁用商品,勾选需要重启的商品,点击<启用>按钮即可启用

![](https://vip.kingdee.com/download/0109f1dd5155fef34e63a6a42c599c93a5e5.png)
