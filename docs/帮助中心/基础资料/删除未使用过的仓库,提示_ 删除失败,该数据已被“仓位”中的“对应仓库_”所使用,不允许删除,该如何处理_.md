# 删除未使用过的仓库,提示: 删除失败,该数据已被“仓位”中的“对应仓库:”所使用,不允许删除,该如何处理?

删除未使用过的仓库,提示: 删除失败,该数据已被“仓位”中的“对应仓库:”所使用,不允许删除

![](https://vip.kingdee.com/download/0109ba02b55a75c544d49fa87a5fff456915.png)

【解决方案】

a、点开仓库查询仓位信息

![](https://vip.kingdee.com/download/0109f80371dcf9a04446aa9af4b8798078d5.png)

b、点开仓位编码,重新选择对应仓库

![](https://vip.kingdee.com/download/0109c7d61e105f7541c2aca50841cca832db.png)

c、清空仓位信息后,重新删除仓库

![](https://vip.kingdee.com/download/0109864f90af5a944832b34234cbe3f50c41.png)
