# 销售出库期初录入时是否需要录入成本?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：嘿唬哈 | 阅读：257 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/617164173927215872)

【问题描述】

销售出库期初录入时是否需要录入成本

【概述】

当商品是服务商品时可以不录入,当商品是普通商品时需要录入以便下推发票能有成本。

【适用版本】

星辰旗舰版
