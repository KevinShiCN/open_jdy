# 供应商/客户引入模板有结算供应商/客户,但是直接新增为什么看不到?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：Schatz | 阅读：50 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/660287091825292544)

【问题描述】

供应商/客户引入模板有结算供应商/客户,但是直接新增为什么看不到?

![](https://vip.kingdee.com/download/0109d8d7a355a68345f1a9206369376a75ff.png)

【解释说明】

1、先核查供应商/客户界面的列设置,确认是否已启用结算供应商/客户的显示。

![](https://vip.kingdee.com/download/01094cdde5081f6149b0b7c8bc9187b60fbd.png)

2、若列设置中未找到结算供应商/客户的字段,则需前往【系统设置】-【系统参数】下的进销存参数配置,检查是否已启用“第三方结算”功能。

对于未启用第三方结算的账套,系统将不包含结算供应商/客户的设置项。但是在引入模板中未隐藏该字段,不过引入设置了结算供应商/客户也不起作用。

![](https://vip.kingdee.com/download/010991bdf7ade7bc42f5b98d121b5d1c7687.png)

3、在启用第三方结算的情况下,若子账户在供应商/客户资料中无法看到结算供应商/客户的字段,可能是因为该字段的授权被限制。建议使用管理员账户登录系统,前往【系统设置】-【用户管理】界面检查字段授权设置,如果发现有限制,可以进行调整以允许子账户查看相关字段。

![](https://vip.kingdee.com/download/01091c26434e1288421191f31862ba03ba03.png)
