# 已使用过的科目可以启用外币核算吗?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：云社区用户5j6u671 | 阅读：492 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/609551761375557632)

【问题描述】

已使用过的科目可以启用外币核算吗?

【解释说明】

已使用的科目支持中途启用外币核算。

【解决方案】

1、币别里面先新增外币

操作路径:【系统设置】-【币别】-新增

![](https://vip.kingdee.com/download/010911016ffaba424ca681ff18a81899384b.png)
2、科目启用外币核算。

操作路径:【基础资料】>【科目】

![](https://vip.kingdee.com/download/0109c6bbcc7ac59641a89bc7ad35be81fc58.png)

勾选已使用科目,点科目编码进去,启用外币核算。

![](https://vip.kingdee.com/download/010928b71764418145d0b2dee20eb70433a5.png)

之前没有启用外币之前的凭证会按本位币展示币别,汇率=1,原币金额信息。

![](https://vip.kingdee.com/download/010937e6c19b5dc4454cb29dbef6b0fb4d3e.png)

新增凭证就可以选外币的币别,填写汇率、原币信息

![](https://vip.kingdee.com/download/0109afd1d67f1e1e407587588f1d39737936.png)
