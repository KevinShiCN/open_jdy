# 导入客户价格策略失败提示:销量区间(到)必须大于销量区间(从),或者存在重复的商品行

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：春日负暄 | 阅读：112 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/650144081892098048)

【问题描述】导入客户价格策略失败提示:销量区间(到)必须大于销量区间(从),或者存在重复的商品行

![](https://vip.kingdee.com/download/01090a73822497fb436a9d3a43d8adf11f68.png)

【解释说明】

在引入价格资料时,如果需要按照销量设置不同的价格区间,需要在【销售(从)】、【销售(到)】字段分别录入数值。比如销售数量为【0,50)时,价格为20元,【50,100)时,价格为18元。

【解决方案】

在客户价格资料策略的引入文件中,录入了两行相同商品分录,需要按照销量区间设置价格,但是在【销售(到)】字段没有录入数值,会提示销量区间(到)必须大于销量区间(从)

![](https://vip.kingdee.com/download/010984d12a9db82a4e37a2cc54c7026d2440.png)

此时需要去录入【销售(到)】字段的值

![](https://vip.kingdee.com/download/0109e389042b77284e149b24fe97e9adbb3d.png)
