# 商品如何进行分类管理(如原材料/成品)?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：大lz | 阅读：3376 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/534899794523831040)

【问题描述】

商品如何进行分类管理(如原材料/成品)?

【解释说明】

成品、材料、设备等各类实物资产,均统一在**商品信息**中新增并管理;可依据商品属性或使用用途划分商品类别(如原材料、半成品、库存商品等),实现库存的精细化高效管理。

各类业务报表(库存类、销售类、采购类等)均支持按**商品类别**汇总查询数据,便于快速统计各类商品的业务数据。

【操作步骤】

在商品列表中可以新增商品类别,支持设置多级商品类别。

1、新增商品类别

依次点击【基础资料】-【商品信息】打开商品列表界面,可以直接点击右上角【类别管理】中新增,或点击左上角的商品类别的“编辑”功能添加。

![](https://vip.kingdee.com/download/010933c02915f86c46dba477bc94d28dd3ca.png)

1.1 在商品列表列表中新增

点击右上角【类别管理】打开【商品类别列表】界面,点击右上角【新增】输入“类别编码”、“类别名称”,根据需要完善其他字段信息后点击<保存>即可。

![](https://vip.kingdee.com/download/0109112321373e504d5c87f2c1214ab4d987.png)

商品类别支持通过Excel表格引入,具体可以参考知识【[商品类别是否可以引入引出?如何操作?](https://vip.kingdee.com/article/524752849259849984?productLineId=35&isKnowledge=2&lang=zh-CN)】操作,

1.2  在商品列表界面新增

在商品列表界面点击左侧商品类别【编辑】,再点击“+”打开新增商品类别界面,输入类别信息后点击<保存>即可。

![](https://vip.kingdee.com/download/0109e9e147101ea7493f8d7023a493f82822.png)
2、设置下级类别
新增商品类别时“上级类别”属性为空表示为一级类别,新增时选择对应的“上级类别”即为明细类别。
![](https://vip.kingdee.com/download/01094d69795a88c84728b14044265eab3eaa.png)
3、新增类别不选上级类别即是新增一级商品类别

4、商品类别最多支持10级

![](https://vip.kingdee.com/download/0109227b7beb66dd44268e25ab2f977d62df.png)

5、库存、采购、销售类报表可根据商品分类进行筛选统计

如,库存查询表可统计某个类别的商品库存

![](https://vip.kingdee.com/download/0109f8197fec23ed439e80957b4c34cad304.png)
如,销售汇总表可统计某个类别的商品销售数据

![](https://vip.kingdee.com/download/01092ee0f3873afd4f8fb766d5351a601c34.png)
【温馨提示】

暂不支持在单据中直接新增商品类别;如需添加,可在单据的 “新增商品” 页面,通过类别选择入口完成新增操作。

![](https://vip.kingdee.com/download/0109c0afb8f673fc42dc957566223e1f032d.png)
