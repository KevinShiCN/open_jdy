# 供应商价格资料商品的采购单价是否可以批量调整?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：云社区用户8x80607 | 阅读：213 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/624772183130232064)

【问题描述】

供应商价格资料商品的采购单价是否可以批量调整?

【操作步骤】

方法一:**采购单价批量填充调整**

1)基础资料-供应商价格资料,点击供应商价格方案编码进入

![](https://vip.kingdee.com/download/010974c9f1e77a624c5b93cde8da39ecb51e.png)
2)如下图所示,采购单价列表头,选择商品对应价格,使用运算方法:加减乘除填充
![](https://vip.kingdee.com/download/01094ed24ddc7b7d4fffb2c305dfce252187.png)
3)比如:选择商品采购价+20,点击确定
![](https://vip.kingdee.com/download/01091d25cb8fc14741488c69c10594923673.png)
4)填充效果如下图所示![](https://vip.kingdee.com/download/0109bb0cb656ed604f03affef4cf6c92ce25.png)
方法二:引入模板覆盖新增方法批量调整

1)基础资料-供应商价格资料,选择需要修改方案,按引入模板引出,如下图所示。需注意:不同价格类型的方案需选择不同模板引出![](https://vip.kingdee.com/download/0109ff3555c23f524af68e6d14aa5ff32aba.png)
2)excel表格编辑修改后的采购单价,商品单位及商品。修改完成,保存模板

![](https://vip.kingdee.com/download/0109b19fee6dac0045788e41c068db38c7ce.png)
3)基础资料-供应商价格资料。点击引入,引入方式选择【覆盖新增】即可![](https://vip.kingdee.com/download/0109bc4f5c0d35a74d0eab9df2525ff83a49.png)
