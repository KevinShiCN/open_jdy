# 供应商价格资料查询表

# **1 简介**

## **1.1 功能介绍**

实现按商品、按供应商、按价格类型查询价格的功能，展示在供应商价格资料中维护的价格资料。

## **1.2 适用版本**

星辰3.1以上所有版本

## **1.3 应用场景**

查询供应商价格资料（以前只能进入到每一个供应商价格方案详情中查看价格资料，现在可通过报表查询所有的价格方案）。

# **2 应用流程**

1） 在供应商价格资料列表中，单击右上角的“价格查询”，可跳转到供应商价格资料查询表。

2） 供应商价格资料查询表中，数据展示参照单据取价优先级的规则。

- 按商品查询时，输入过滤条件“商品”，查询结果展示该商品所涉及的所有价格资料，包括按商品设置的价格资料，以及按该商品所属的商品类别、商品品牌设置的价格资料。
- 按供应商查询时，输入过滤条件“供应商”，查询结果展示该供应商所涉及的所有价格资料。
- 按价格类型查询时，则必须录入过滤条件“价格类型”，只展示该价格类型下的所有价格方案。
