# 如何删除辅助核算明细?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：130 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/778802366996493056)

【问题描述】

如何删除辅助核算明细?

【业务场景】

比如科目“2202应付账款”启用了“供应商”的辅助核算,现在要删除某一个供应商明细如何操作(其他科目的辅助核算明细操作方式一样)。

【解释说明】

未使用的辅助核算明细可以直接删除,已使用过的辅助核算明细不能直接删除,需要删除对应业务后才能删除,若不想删除关联单据等数据,可以选择将明细禁用。下面以删除供应商辅助核算为例说明,删除其他辅助核算明细同理操作。

如需取消科目辅助核算,详细可以点击:[已经使用过的科目是否可以添加或者取消辅助核算?](https://vip.kingdee.com/link/s/ZchaF)

【操作指引】

如需删除供应商的辅助核算明细,菜单栏点击【基础资料】>【供应商】

![](https://vip.kingdee.com/download/0109f60ed96bd8a14842a1ef36b3de6f7bb6.png)

供应商界面,选择对应供应商删除即可

![](https://vip.kingdee.com/download/01098752beedf6e7467d9a9df81521dfb8e9.png)

若供应商已被使用,不能直接删除,需要删除对应业务后才能删除,若不想删除关联单据等数据,建议点击<禁用>,禁用后不影响已经录入的数据,后期若需要启用,请点击:[如何查询禁用/启用供应商?](https://vip.kingdee.com/knowledge/391352046303709440?productLineId=35&isKnowledge=2&lang=zh-CN)

![](https://vip.kingdee.com/download/010987769500dfbe4057a106cdd2837f413c.png)

如需删除其他辅助核算的明细数据,可以点击到对应的基础资料中删除

| 辅助核算维度 | 操作步骤 |
| --- | --- |
| 客户 | 可点击:如何删除客户? |
| 部门 | 可点击:如何删除部门? |
| 职员 | 可点击:如何删除职员信息? |
| 商品 | 可点击:删除商品,如何操作? |
| 仓库 | 可点击:如何删除仓库信息? |
| 品牌 | 可点击:如何删除商品品牌? |
| 收入类别 | 可点击:如何删除收入类别? |
| 支出类别 | 可点击:如何删除支出类别? |
| 账户 | 可点击:怎么删除账户管理中的账户? |
| 自定义 | 可点击:如何删除辅助资料? |
