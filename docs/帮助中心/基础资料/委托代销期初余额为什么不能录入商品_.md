# 委托代销期初余额为什么不能录入商品?

【解释说明】

因为业务当前期间不是启用期间,所以不能录入期初数据。需要将业务期间反结账的到启用期间。

1:业务反结账到启用期间。

![](https://vip.kingdee.com/download/010967203cfd462b4bf6b305419440a6953f.png)
![](https://vip.kingdee.com/download/010991b5a3e1b54b4b3fbc9f177a293cfdf7.png)

2、页面刷新以后,重新打开委托代销期初余额,就可以重新录入了。也可以单据列表新增。

![](https://vip.kingdee.com/download/0109dd2cbe6982ba40498f9aec112355df23.png)
