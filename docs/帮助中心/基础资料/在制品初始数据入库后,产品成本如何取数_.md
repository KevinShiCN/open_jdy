# 在制品初始数据入库后,产品成本如何取数?

【业务场景】

在制品初始数据入库后,产品成本如何取数?

![](https://vip.kingdee.com/download/0109cf8714e97d53448fa620db42e2f043a1.png)

【解释说明】

在制品初始数据入库后,是产成品直接入库,生成产品入库单,产品总成本取在制品初始数据的【总成本】。

![](https://vip.kingdee.com/download/01099da8f79acbe14156997573fcc6795157.png)

注意:在制品初始数据下推【入库】后,需要进行【生成成本核算】产品入库单才会核算出对应的入库成本;并且生成相关的生产成本计算表,查看核算明细。

![](https://vip.kingdee.com/download/01092a0a2a209bbc4bf69d38d643a9ed6d61.png)
