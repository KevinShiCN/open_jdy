# 部门资料如何禁用启用?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：杨桂平 | 阅读：302 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/570413815100353792)

【问题描述】

部门资料如何禁用启用?

【解决方案】

**一、禁用部门**

点击【基础资料】>【部门】,勾选要禁用的部门资料点击<禁用>按钮

![](https://vip.kingdee.com/download/0109f84a1acf95fb47dc8499852dac7dea47.png)

**二、启用已禁用的部门资料**

1、首先在【基础资料】-【部门】里面,筛选出所有或者单独筛选出禁用状态的部门(默认只显示启用的部门),才能看到禁用的部门信息 ;

![](https://vip.kingdee.com/download/0109778e0763408d483488c7b8cd0d9053f7.png)

2、选择需要启用的部门,点击上面的【启用】按钮,即可重新启用部门。
![](https://vip.kingdee.com/download/01090f2e0e6764e442eaa08c099723ba90a0.png)
【温馨提示】

**1.禁用后的部门无法在新增单据的时候继续被选择到,不影响已经录入单据的数据;**

**2.已经发生业务的部门无法删除,只能禁用,除非将相关单据删除后,才可以删除部门。**
