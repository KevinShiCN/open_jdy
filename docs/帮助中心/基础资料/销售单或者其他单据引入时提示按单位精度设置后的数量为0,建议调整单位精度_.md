# 销售单或者其他单据引入时提示按单位精度设置后的数量为0,建议调整单位精度?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：杨桂平 | 阅读：306 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/534899775666329856)

【问题描述】

引入销售出库单提示“按单位精度设置后的数量为0,建议调整单位精度”

![](https://vip.kingdee.com/download/0109efe427e3471240cc97b147c9071830f7.png)

【解决方案】

出入库单据的商品数量不允许为 0;若导入单据时触发该提示,原因是导入文件中存在多行列的商品数量为 0.00,需将商品数量设置为大于 0 的数值,方可完成正常导入。

![](https://vip.kingdee.com/download/0109c600870a3e694c16915306b2e30f6f56.png)
