# 总账模块的数据可以同步到业务模块吗?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：云社区用户8x80607 | 阅读：160 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/617524552888671488)

【问题描述】

总账模块的数据可以同步到业务模块吗?

【操作步骤】

总账的凭证不支持同步到业务模块生成业务单据。

如果只在总账录入凭证,业务数据不会同步更新,需要另外到业务模块录入对应的单据同步更新进销存的数据。

比如收到一笔货款,财务只在总账做了一张冲应收的凭证,只会减少财务应收账款科目的余额,进销存对应客户的应收余额不会减少,需要到进销存做一张收款单选择销售出库单核销进销存的应收余额。(其他单据同理)

综上,建议购买了进销存和财务模块的客户先在进销存录入业务单据,然后通过业务单据生成凭证入到总账,提高工作效率。

[如何生成凭证](javascript:void(0);)

目前支持业务的账户初始数据同步总账科目余额。

1、如下图所示,点击基础资料-账户初始余额,点击右上角的**【从科目引入数据】**![](https://vip.kingdee.com/download/010970ad5ea24d3f4e8f9077f556b835bedc.png)
2、点击之后,选择需要引入科目的期间,以及需要引入的数据,有两种引入方式:引入科目余额及发生与只引入余额或发生额不为零的科目。选择完成之后,点击确定,即可通过当期科目余额引入到账户年初余额![](https://vip.kingdee.com/download/0109860f4be01066450f8b51c4308fc5f3d1.png)
3、需注意,**引入之前账户与科目需要进行科目关联设置**。财务处理-科目关联设置,点击账户管理进入,之后点击设置科目,选择需关联的科目设置完成之后即可操作引入科目数据。![](https://vip.kingdee.com/download/0109aacd656116014aabaa93002ad3a58cb9.png)
【温馨提示】一个科目只能对应一个账户,如果存在对个账户对应同一个科目,则无法匹配金额,则出现如下图提示。如果业务有外币账户,那么建议总账单独新增一个科目去启用外币,不在原科目启用外币。
