# 客户基础资料里边维护了首要联系人,但是下单的时候带不出来,只能带出客户的第一个联系人,应该如何处理?

> 作者：杨桂平 | 阅读：11 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/761030763172116992)

【问题描述】

客户基础资料里边维护了首要联系人,但是下单的时候带不出来,只能带出客户的第一个联系人,应该如何处理?

![](https://vip.kingdee.com/download/0109217e5975d5f646cb92595a5c93c2dd6a.png)

![](https://vip.kingdee.com/download/0109851516eb1c0a4c37aab396e78839039e.png)

【解释说明】

检查一下单据字段配置中是否开启了支持模糊搜索,开启了支持模糊搜索时,默认带出客户的第一个联系人,同时再开启模糊搜索时显示时,选择客户的时候可以选择联系人,如果客户需要选择客户时默认带出首要联系人,同时关闭支持模糊搜索和模糊搜索时显示即可。

![](https://vip.kingdee.com/download/010974b1a75c8c2a402c86f28beb9b7c0042.png)

1.支持模糊搜索和模糊搜索时显示都关闭时,开单默认带出客户的首要联系人

2.开启支持模糊搜索,关闭模糊搜索时显示时,开单默认带出客户的第一个联系人

3.支持模糊搜索和模糊搜索时显示都开启时,效果如下:
