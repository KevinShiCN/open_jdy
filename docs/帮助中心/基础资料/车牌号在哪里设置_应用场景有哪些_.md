# 车牌号在哪里设置?应用场景有哪些?

【操作流程】

**1.“车牌号”新增路径:**

点击【基础资料】【辅助资料】进入到辅助资料列表界面。

![](https://vip.kingdee.com/download/0109fc08a9bdca514d82b1d5773ba119c102.png)

在辅助资料列表界面中,左侧点击“车牌号”后,再点击右上方“新增”按键,即可增加对应的车牌信息。

![](https://vip.kingdee.com/download/01093e6b0a776179480c83683bf090df3c8c.png)
若不再使用,可勾选点击<禁用>或<删除>,已使用的不支持删除,可以禁用

![](https://vip.kingdee.com/download/0109381bbdd5907f4a01b2775c2aa3b60de6.png)
点击辅助资料编辑可对车牌信息进行修改,若已使用,修改后历史单据中的信息将一起更新

![](https://vip.kingdee.com/download/0109f99dcaa0a55b4d8397358db9b691432c.png)

**2.“车牌号”的应用场景:**

“车牌号”属于辅助资料,使用场景主要有以下两个。

①作为核算项目使用:可以将辅助资料或者自定义的辅助资料加入到核算项目或者用户自定义的核算项目中,设置用户自己的核算项目。

![](https://vip.kingdee.com/download/0109c5da9750377a456eae999cc01d2378bf.png)

②作为单据上的辅助信息使用:可以通过单据自定义字段功能,将自定义的辅助资料加入到单据中,用于记录单据所对应的车辆信息。

![](https://vip.kingdee.com/download/0109dbd569054f554828b0bfcd92caa41a48.png)
