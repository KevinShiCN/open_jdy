# 如何设置商品图片?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：杨桂平 | 阅读：2317 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/537798953249653248)

【问题描述】

如何设置商品图片?

【操作流程】

每个商品可以上传不超过10张格式为.jpg\.jepg\.png\.gif的图片,包括每个商品的辅助属性图片,单张图片不超过**5M****。**默认第一张为主图,可设置主图(商品列表中商品图片、单据中商品行的商品图片均只展示主图)。可通过拖拽修改图片顺序。 商品图片支持批量上传,支持批量引入。

1)批量上传图片:

适用于单个商品上传图片。在[基础资料]→[商品资料]→[商品信息]界面,点击商品列表中每行商品前的“+”;也可在商品新增界面的商品图片,点击“+”,多选图片后即可为商品批量添加图片。

![](https://vip.kingdee.com/download/0109170a139c89e945a0ade1106126e0942b.png)

2)批量引入图片 :

[基础资料]→[商品资料]→[商品信息]→引入→引入商品图片,根据图片命名格式修改图片的名称,当图片名称符合要求时系统会自动为商品匹配图片。

![](https://vip.kingdee.com/download/01093e0ab2182a6849a2a87ff2ec67e59819.png)
