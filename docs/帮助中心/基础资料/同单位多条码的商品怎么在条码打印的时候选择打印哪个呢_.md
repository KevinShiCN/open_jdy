# 同单位多条码的商品怎么在条码打印的时候选择打印哪个呢?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：Schatz | 阅读：131 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/655575732432523520)

【问题描述】

同单位多条码的商品怎么在条码打印的时候选择打印哪个呢?

![](https://vip.kingdee.com/download/0109d07f0df0b7434ce29c10f9a192dde32a.png)

【解释说明】

1、基础资料直接打印条码标签

在【基础资料】-【条码标签打印界面】,直接在左上角输入或扫描条码,系统将自动显示对应的商品信息及条码。随后,设定所需打印的数量后点击打印即可完成;

![](https://vip.kingdee.com/download/010929805f38189944e3b9ca9b6c421c5ffd.png)

从右上角选择特定商品,此时会列出该商品的所有可用条码选项。只需勾选想要打印的条码,并设置好打印份数后进行打印。

![](https://vip.kingdee.com/download/010909df7a2220014955b5e901b9516533be.png)

若点击右上角的“选单”,则只能选择采购入库单,且仅能打印采购入库单上商品的条码。

![](https://vip.kingdee.com/download/01097136d38efc694939b229db792063a9ed.png)

2、单据上打印条码标签

会根据单据上的信息自动提取相应的条码数据进行打印。

![](https://vip.kingdee.com/download/010959fae1bc39d148228035ae5bff0617e2.png)

【温馨提示】

对于同一单位拥有多个条码的商品,在制作单据时,必须通过扫码来带出对应条码(可手工在扫码位置录入条码或者扫码枪扫码带出)。

![](https://vip.kingdee.com/download/0109ef6afc53996e42f8900cd35e15693a00.png)

在商品编码维护界面中,使用条码模糊搜索功能时,系统仅会显示商品的主条码(如果未维护主条码,则会显示多单位条码中的第一个),并且无法手工修改成其他条码。详细情况可见下方动图:

![](https://vip.kingdee.com/download/0109e9810d5066644e38918818eae59e19a3.gif)
