# 如何禁用/启用支出类别(支出名称)?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：shaoling | 阅读：192 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/624410176761552640)

【问题描述】

如何禁用/启用支出类别(支出名称)?

【业务场景】

做其他支出单时某些支出类别(即支出名称)可能之后不会再使用,做单据的时候不想要再显示,需要把对应的支出类别隐藏禁用操作;若后续需要重新使用可以再次启用。

【操作说明】

依次点击【基础资料】—【支出类别】打开支出类别列表界面。

![](https://vip.kingdee.com/download/0109e03ea7476a97492ca8015d7f4d6a639f.png)

**1、禁用**

勾选需要禁用的支出类别(支持批量选择多个),再点击“禁用”即可。

![](https://vip.kingdee.com/download/0109bdc4727efaa54f6c943b9186830b6f66.png)

**2、查询及启用**

在支出类别列表界面展开过滤条件,筛选条件中状态选择全部即可查到已禁用的信息,勾选需要启用的支出类别后点击“启用”即可。

![](https://vip.kingdee.com/download/0109f9a6bd2298d64cf9b40c619789ff1275.png)
