# 设置了商品价格为什么做单带出来的价格不对?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：杨桂平 | 阅读：283 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/568239403986946304)

【业务场景】

设置了商品价格为什么做单带出来的价格不对?

【解释说明】

需要检查【基础资料】-【价格取数优先级】中的设置,设置的是商品价格资料,优先级取数又没有开启或者是没有优先到第一级,就会取不到设置好的价格,修改一下优先级取数即可。
