# 批发促销功能

# **1 简介  **

## **1.1 功能介绍**

运营制定促销策略和方案，业务员开单时执行促销，自动识别和添加赠品到单据

## **1.2 适用版本**

星辰3.1以上所有版本

## **1.3 应用场景**

线下批发业务中，存在买赠等促销方案，运营提前制定促销方案，业务员开单时想要通过执行促销、选择促销方案来自动添加赠品。

注：目前批发促销只支持商品买*件赠送赠品*件（即单品满件数赠）的规则，更多促销规则待后续迭代，若有需求可通售后客服提工单反馈给我们。

## **1.4 系统路径**

基础资料>价格设置>批发促销

## 1.5 适用单据

销售订单、销售出库单

# **2 操作指南**

## 2.1 字段说明

**适用渠道**：订货商城（购买了订货则可用）、线下批发（指的是进销存的销售类单据）

**适用商品**：即促销方案所应用的商品范围，可以是全部，也可以是指定的商品。

**优惠规则**：目前只有“同一商品满件数”这一规则。可以选择满赠，即达到一定数量后，赠送商品。也可以选择倍增，如规则设置满2件赠1件，满5件赠2件，满10件赠5件只有消费满10件时才会倍增赠10件，消费不满10件时不会倍增。

**适用对象**：即促销方案所适用的客户，可以是客户，也可以是指定的客户。

## 2.2 具体应用

1）打开基础资料>价格设置>批发促销，点击“新增”

![上传图片](https://vip.kingdee.com/download/010021cde88dddaa474ca84fc478ea1f5c3c.png)

提前制定促销方案。目前只支持“单品促销”，更多促销规则待后续迭代，若有需求可通售后客服提工单反馈给我们。

![上传图片](https://vip.kingdee.com/download/010014b07e92ccc248628cd3f9d7df181da4.png)

2）在方案中录入必录项，然后保存即可。

**示例1：开启赠送其他参数**

如购买某茶叶，买4袋送1袋糕点。对应的促销方案设置如下，

![上传图片](https://vip.kingdee.com/download/0100f096b966f9d144df80fb13edfb007487.png)

**示例2：开允许倍增参数**

如：购买某酱牛肉，买5包送1包，多买多包。对应的促销方案设置如下，

![上传图片](https://vip.kingdee.com/download/0100ee9952600f4c43069d30047d4c8e1f49.png)

3）业务员开单，点击执行促销，选择促销方案，执行成功后，可以在商品行上看到对应执行的促销方案名称。

- 【茶叶满4送1，赠送其他】开单效果：

![上传图片](https://vip.kingdee.com/download/01006605f5a777c444babeb303373a31e98b.png)

点击确定后，可以看到单据上出现了一行赠品，赠品是绿豆糕，这就是赠送其他的效果。

![上传图片](https://vip.kingdee.com/download/01008f80f9bbbfb94b3ea437fd3f709ccaa4.png)

- 【买某酱牛肉，支持倍增】开单效果：

![上传图片](https://vip.kingdee.com/download/010005bde03cfb08404bb9b9d957a4b935c2.png)

点击确定后，可以看到单据上出现了一行赠品，数量是2，因为满5赠1，满10赠2，所以开单数量11时，赠品数量倍增了，赠送2，这就是倍增的效果。
