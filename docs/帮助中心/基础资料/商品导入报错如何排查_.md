# 商品导入报错如何排查?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：杨桂平 | 阅读：460 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/479454744855873536)

【问题描述】

商品导入报错如何排查?

【解决方案】

导入报错以后,点查看详情,可以查看报错的表格,即可查看报错行的原因。

![](https://vip.kingdee.com/download/0109f970d44a7bd84b05a6c76085d4098f6c.png)

1)如果数据不多的话,会有报错表格自动弹出显示在左下角。

![](https://vip.kingdee.com/download/0109d365f83175eb454eb50ad24e01ea386f.png)2)如果报错的数据较多,系统会提示在引入结果查询中下载表格后再进行查看。

![](https://vip.kingdee.com/download/0109982b0a3a22804aaa827b657b7f053a17.png)
