# 报表辅助单位如何设置?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：咚咚 | 阅读：634 | 有用：0

【问题描述】

报表辅助单位如何设置?

【解释说明】

多单位商品,当单位数量超过两个,可通过**报表辅助单位**功能,让库存报表显示**基本单位 + 指定副单位**;报表将按所选辅助单位展示对应数量。

报表辅助单位只能填写一个,而且不能是基本单位,只能是副单位。

路径:【基础资料】>【商品信息】

假设商品设置报表辅助单位是包,库存查询表显示辅助单位就是包。
