# 切换成其他语言版本之后，部分系统字段没有值，显示为空

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：快速入门, 常见问题, 新特性
> 作者：进销存熊诗琴 | 阅读：1422 | 有用：11
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/674923874886114560)

# **1 问题描述**

- 从多语言版本切换成中文版后，同一张单据的客户信息为空，变成“-”符号；
- 切换成英文版之后，部分系统字段没有值，显示为空；

![](https://vip.kingdee.com/download/01091990027fdc3641798be33cc116d98cf2.png)

# 2 适用版本

星辰3.1以上所有版本，需要先购买多语言模块

# **3 原因解析**

客户信息、商品信息、供应商、部门、职员、仓库、辅助属性、单据备注等有多语言组件也就是小地球图标的字段都需要维护对应语言的值，切换版本的时候才能正常显示。

![上传图片](https://vip.kingdee.com/download/01000627165a4971404eb05b1e276a108f88.png)

# **4 解决方案**

多语言组件维护方式：

1）手动维护多语言组件中三种语言对应的内容；

2）修改某一语言下的信息，可以切换对应语言版本将信息比如商品资料批量引出，然后在excel里面维护之后再覆盖引入；

![](https://vip.kingdee.com/download/010982c2325d74cb4176b386f8f9aa86061b.png)

>
具体操作请参考知识：如何批量维护商品的英文名称？： [https://vip.kingdee.com/link/s/ZYBhZ](https://vip.kingdee.com/link/s/ZYBhZ)

3）使用多语言初始化功能，一键同步，根据用户配置显示的语言，多语言初始化同步这里就可以勾选哪些语言同步，比如将简体中文的值同步到英语和印尼语。

![上传图片](https://vip.kingdee.com/download/0100c1ba3b7de43948c68e159569b4156ec8.png)

注意：

- 使用多语言初始化功能同步信息后不可以撤销，因此在同步之前会做一次数据备份，如有需要可以恢复账套备份数据进行还原；
- 同步之后多语言组件其他多语言文本框里显示的内容和数据源语言里的是一样的。
