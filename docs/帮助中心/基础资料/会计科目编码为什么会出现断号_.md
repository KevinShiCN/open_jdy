# 会计科目编码为什么会出现断号?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题

【问题描述】

会计科目编码为什么会出现断号?

【业务背景】

会计科目编号是按会计科目表中科目分类排列次序编列的,一般要求是: 号码编列要科学合理,既标出大类、小类之别又要显示排列次序,层次分明,相互关联。编号要简单明了,便于记忆,使用方便。编号位数不宜过多,要具有伸缩性便于增补科目。会计科目的编号是以数字、文字、字母和横线以及圈点等方式进行的。

【概述】

1、一级科目(总账科目)断号,如截图没有1003,这是因为此会计制度系统没有预设的1003科目

![](https://vip.kingdee.com/download/0109f91b26676f3840c6849fad2766b2251d.png)
2、二级科目(明细科目)断号

如下截图:有110103直接到了110110;此时需要考虑新增明细科目时最后一个明细科目科目代码手工改成了110110;或者新增的连续的科目代码;把中间的科目代码删除了。

![](https://vip.kingdee.com/download/01090ac97ab7cdaf45eca758121ccf21558b.png)
