# 打印实践案例——珠宝标签适配与应用

# 1 应用场景

部分行业由于其标签纸非常小，如果条码非常复杂且长度超过10位的话，那么打印的标签的扫码效果会打折扣。例如**珠宝标签**打印受制于其打印范围面积之小（通常实际打印区域都小于3*3cm，其中还包括了防伪信息、商标信息等logo），对打印设备的要求非常高，对模板设计也有特殊的要求。

星辰3.1以上所有版本

# 3 设计要点

**珠宝标签**打印受制于其打印范围面积之小，对打印设备的要求非常高，一般是有专门的标签打印机及标签设计器，而影响珠宝类标签的条码打印清晰度及扫码识别的因素主要有以下几点：

- **条码大小与标签纸大小**--没有标准参数建议，需要手动调整及测试。要保证条码能够放置在实际打印区域中，且边距要留一定空白
- **编码方式**（如不是使用国家标准码或者通过条码规则生成的的13位条码，建议用code128）
- **碳带打印速度**--速度太快，打印头没来得及散热，会影响打印效果（清晰度）
- **标签是否转向**-会影响打印头的实际打印效果
- **打印浓度**--要看所使用的的标签打印机是否支持调整，星辰打印模板不支持调整浓度
