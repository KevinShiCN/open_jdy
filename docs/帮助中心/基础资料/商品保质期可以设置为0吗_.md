# 商品保质期可以设置为0吗?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：胡大大 | 阅读：62 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/713562949251409920)

【问题描述】

商品保质期可以设置为0吗?

【解释说明】

启用保质期管理的**商品保质期不支持设置为0**.若不想录入保质期建议不要启用保质期管理,但是又想单独体现一个生产日期时可以建议在单据中自定义一个“生产日期”的字段,具体可用参考知识【[如何新增单据自定义字段?](https://vip.kingdee.com/knowledge/566427477040599552?productLineId=35&isKnowledge=2&lang=zh-CN)】操作。

商品启用保质期管理后,商品的保质期必须录入,否则保存时系统会提示“启用保质期管理,保质期必须大于0。”

![](https://vip.kingdee.com/download/01097749b924bc594d73a3fd74ea3f84fa60.png)

新增单据时单据的保质期也不能修改为0。
![](https://vip.kingdee.com/download/0109f95fadbdeeff4ee295012a3212a7c120.png)
