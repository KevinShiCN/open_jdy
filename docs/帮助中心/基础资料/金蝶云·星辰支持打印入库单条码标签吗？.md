# 金蝶云·星辰支持打印入库单条码标签吗？

问题描述：金蝶云·星辰支持打印入库单条码标签吗？

解决方案：

金蝶云·星辰支持在采购入库时，根据入库单信息比如供应商、商品的批次、保质期、商品入库的仓库、仓位等打印成标签贴在商品上。
