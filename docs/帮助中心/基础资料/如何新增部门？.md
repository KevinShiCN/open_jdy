# 如何新增部门？

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题, 快速入门
> 作者：小微微 | 阅读：980 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/23951)

问题描述：

解决方案：

单击【基础资料】→【业务资料】→【部门】进入到部门列表

1、新增部门：点击右上角新增进入部门新增界面，录入部门名称，选择上级部门保存即可。上级部门若选择公司账套名称则为一级部门，若选择一级部门则为二级部门，以此类推。

![](https://vip.kingdee.com/download/010093d3df78a65942afa21139fe2062893f.png)

2、引入说明：

- 引入的字段信息可通过建立新的模板或是修改现有模板设置需要引入的字段，但*号字段部门编码、部门名称、上级部门为必须引入字段
- 其他字段不填写，则引入后默认为空

**注意事项**：部门被使用后不允许删除。
