# 给老板的私密汇报：结算期限+账龄管理=应收账款风险防范

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：快速入门, 知识软文
> 作者：Simone邓 | 阅读：1196 | 有用：8
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/321960302932170240)

小斗一进办公室就看到老板在骂小精。

       只见老板拿着一份客户对账单，冲着小精吼道：这个客户的货款逾期4个月了，6万块钱到现在一分钱都还没回款，难道你没发现吗？你是怎么做客户回款管理的？

       小精今年刚从学校毕业，进了公司一直勤勤恳恳、任劳任怨；经常主动加班，老板对自己的表现也挺满意。今天老板突然发这么大的火，她一时间有点手足无措，只能杵在原地，低着头，也不敢回话。“该怎么样做客户的回款管理，包括客户的回款周期，货款逾期的预警，这是你的本分工作！下班前想好方案跟我汇报！”老板带着怒气回了自己办公室，不过他经过小斗工位的时候，给小斗使了个眼色。

      小斗立刻理解到老板的用意，老板是刀子嘴豆腐心，新人本分工作没做好要给一点压力，让小精能吸取教训，想要她这个老员工带一下。

      小斗看了看还杵在原地的小精，不禁想起自己刚毕业步入职场的情景，赶紧走到小精身边，把她拉回工位安慰她道：姐跟你说，我以前工作做错了，给老板骂得更惨。新人总会有个成长的过程，你的表现大家都看在眼里，我们还是很认可你的。你只是对我们系统的功能不熟悉。其实我们系统可以设置结算期限，设置好就能看到客户的应收账龄，就能做回款管理了，还能预警提示，我来教你：

首先到【基础资料】--【业务资料】--【结算期限】设置对应的结算期限

![image.png](https://vip.kingdee.com/download/01003742ce648b644f1680e7a3bc2a6e41f3.png)

![image.png](https://vip.kingdee.com/download/0100458059fb1cef47b192bc6e0d9f689fc4.png)

增加客户时，进入客户资料可以选择到对应的结算期限，如下图：

![image.png](https://vip.kingdee.com/download/0100427c3c1ba2a243dfa8d467bf2f944ec0.png)

![image.png](https://vip.kingdee.com/download/0100bbe88892f05f4aa2ace99701d1be7729.png)

【系统设置】--【系统参数】-【应收应付参数】设置【账龄管理参数】--根据公司的实际情况设置应收账款，到期前几天会进行预警提示

![](https://vip.kingdee.com/download/0109fd21005425e04a0dade55a931fdc259c.png)

做销售出库时，选择客户后，会带出该客户对应的结算期限

![image.png](https://vip.kingdee.com/download/0100aa66db2fa31042448a80bd4b5ef4651a.png)

这里值得注意的是，销售单据必须挂应收账款，根据单据的结算日期与查询报表时当前日期比对计算，同时根据系统参数中设置的应收账款到期预警天数，在设置的预警天数范围内或超过结算日期未收款才显示，进入应收账款预警表可以查询到相关数据，如图：

![image.png](https://vip.kingdee.com/download/010048a694c9cc0f4e698b6242b5aca9d5b7.png)

勾选参数，即可在系统首页右侧查看预警信息

![image.png](https://vip.kingdee.com/download/0100ab39e7ac31df4965bf794c43b5095b6d.png)

 小斗说，这样在登录系统的时候，系统右侧会进行预警提示，就能方便的对客户的应收做一个管控了，所有数据一目了然，应收账款预警还可以点击右上角的导出，给老板查看就可以了呢！

小精一脸羡慕的看向小斗：斗姐，你也太厉害了吧。这么复杂的功能你也会用！小斗说，刚开始接触系统，大家都是小白，其实并没你想象中那么难，主要你肯花时间去学，你也可以的!斗姐都是在金蝶云社区学习(kingdee.com)，里面有很多的知识点，输入关键字即可带出相关知识点，可以根据需要去学习！
