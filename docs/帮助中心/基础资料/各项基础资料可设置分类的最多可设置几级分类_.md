# 各项基础资料可设置分类的最多可设置几级分类?

【问题描述】

各项基础资料可设置分类的最多可设置几级分类?

【解释说明】

在基础资料中可设置分类的有商品信息、商品品牌、辅助属性、客户、供应商、仓库、收入类别、支出类别、凭证模板,各自可设置几级分类,请查看下方列表:

| 基础资料名称 | 可设置的最大分类级次 | 图例 |
| --- | --- | --- |
| 商品信息 | 最大可设置10级分类 |  |
| 商品品牌 | 只能设置一级分类 |  |
| 辅助属性 | 最大可设置二级分类 |  |
| 客户 | 最大可设置10级分类 |  |
| 供应商 | 最大可设置10级分类 |  |
| 仓库 | 最大可设置10级分类 |  |
| 收入类别 | 最大可设置10级分类 |  |
| 支出类别 | 最大可设置10级分类 |  |
| 凭证模板 | 只能设置一级分类 |  |
