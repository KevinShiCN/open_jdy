# 删除客户失败提示被销售订单中的客户使用

【问题描述】

删除客户失败提示被销售订单中的客户使用

![](https://vip.kingdee.com/download/010984b13239ee2342e2b4782d72fbc00257.png)

【解释说明】

①要删除客户资料,必须先删除与该客户相关的销售订单。

**注意:单据删除后无法恢复,所以删除前建议进行数据备份。**[如何进行数据备份](javascript:void(0);)

②若不希望删除业务单据,可选择禁用客户资料。一旦禁用,后续录单时将无法选择该客户。

![](https://vip.kingdee.com/download/0109d578e462912f4265a45affc5775906d5.png)
