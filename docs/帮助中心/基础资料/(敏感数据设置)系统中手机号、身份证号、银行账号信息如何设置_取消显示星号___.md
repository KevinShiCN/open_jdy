# (敏感数据设置)系统中手机号、身份证号、银行账号信息如何设置/取消显示星号**?

【问题描述】

(敏感数据设置)系统中手机号、身份证号、银行账号信息如何设置/取消显示星号**?

如图所示,想要设置客户手机号加密显示或是取消加密:

![](https://vip.kingdee.com/download/0109ab0953af96fe4e41bdcbf32de342a675.png)

【解释说明】

系统中手机号、身份证号、银行账号等敏感信息的显示完整与否,由系统参数 “界面中脱敏显示” 控制。

【操作指引】

菜单栏点击【系统设置】>【系统参数】

![](https://vip.kingdee.com/download/01090a90c3c1745a44c48e8128882bac4da4.png)

在公共参数界面,敏感数据设置中“界面中脱敏显示”参数:

①**勾选后,系统中的手机号、身份证号、银行账号都将脱敏显示**

②**不勾选就完整的显示出来**

![](https://vip.kingdee.com/download/0109ef6549b5246846c59be280e8c68bb3ed.png)

取消脱敏之后还显示*号,通常是因为引入的数据本身是脱敏显示的,导致引入后,数据库就会按脱敏的数据存储,造成显示异常,详情请查看链接:[为什么客户/供应商里的银行账号、手机号,点了取消脱敏,还是显示星号?](https://vip.kingdee.com/link/s/l9e8v)
