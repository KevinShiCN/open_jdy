# 怎么查看客户是被谁禁用的?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：云社区用户55os622 | 阅读：142 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/642896038075575808)

【问题描述】
怎么查看客户是被谁禁用的?

【解释说明】

操作路径:【基础资料】-【客户】,进入客户列表![](https://vip.kingdee.com/download/01093d7a0ef0098c4bb899eac757461b9f33.png)

点击【展开过滤】,状态条件选择【禁用】状态,可以筛选出已被禁用客户

![](https://vip.kingdee.com/download/010990512a15ae76416287bf87a200ad76b2.png)

点击右上角列设置显示【禁用人】字段

![](https://vip.kingdee.com/download/01096abef345ddcf46dbb8c50415dfa7dea5.png)

即可查看客户的禁用人

![](https://vip.kingdee.com/download/01099c03bff3474a42e495b6e0fbf3c405a0.png)
