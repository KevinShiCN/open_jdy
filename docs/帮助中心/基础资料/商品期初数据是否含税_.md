# 商品期初数据是否含税?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：_Leo | 阅读：722 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/481991027834890240)

【问题描述】

请问商品期初数据录入,是录入含税的金额还是不含税金额?

【解释说明】

商品期初,是指商品期初库存成本金额,根据规范的公司业务管理来说,成本默认是不含税金额。

当然,若贵司管理方面要求成本含税,可以计算出含税成本后录入商品期初成本,那么后期商品出库也将按含税成本计算出库成本。

公司可以根据实际情况需选择是否录入含税成本,系统不作要求。
