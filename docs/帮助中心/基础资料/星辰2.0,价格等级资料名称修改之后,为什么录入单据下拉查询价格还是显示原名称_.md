# 星辰2.0,价格等级资料名称修改之后,为什么录入单据下拉查询价格还是显示原名称?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：新用户000 | 阅读：273 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/446477019556164864)

【故障现象】

   修改了价格等级名称保存成功,查询商品价格资料名称是更新的,但是做单据下拉查看单价情况的时候,价格等级的名称还是原名称,没有更新。

![](https://vip.kingdee.com/download/0109b1814065d4fd49a3871722d5da2d930b.png)![](https://vip.kingdee.com/download/01093a64e17fb64c43d5a4a769e346594ffa.png)![](https://vip.kingdee.com/download/0109f3fa614b967748509438da31212dd752.png)

【概述】

  一般出现这个情况是出现在第一次修改价格等级名称的时候,需要把商品的价格设置恢复一下默认列设置,名称即可同步。

【解决方案】

    打开【基础资料】-【商品信息】界面,任意打开一个普通商品的资料,打开价格设置的列设置界面,点击恢复默认,在保存列设置即可。

![](https://vip.kingdee.com/download/0109f5010813e09f42b589dcffd45ca10ecf.png)![](https://vip.kingdee.com/download/0109a6ae89639039424e87883e943b752f36.png)
