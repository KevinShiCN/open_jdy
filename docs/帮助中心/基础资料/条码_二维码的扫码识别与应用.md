# 条码/二维码的扫码识别与应用

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：精华汇总, 快速入门, 功能介绍
> 作者：进销存赵嘉丽 | 阅读：5416 | 有用：15

# 1 **适用版本**

星辰3.1以上所有版本

# 2 应用场景

主要应用于开单/期初录入扫码，或者是出入库/盘点业务的扫码，提高录单效率，降低人为错误。

| 场景 | 应用方式 | 条码示例 | 补充说明 |
| --- | --- | --- | --- |
| 商品标签 | 即记录商品基础信息的标签，通常包含商品条形码（必须）、商品编码、商品名称、单位、品牌、规格型号等信息，具体以实际业务需求为准，可以自定义选择展示哪些商品基础信息。 |  | 如果商品是辅助属性商品（如服装、五金配件、食品行业等），使用了SKU编码管理的话，也是支持按SKU编码进行打印的。注：如果商品资料中的SKU编码没有勾选“是否启用”，则在条码标签打印中会选不到商品，无法打印。打印效果如下（即墨绿色不允许打印）：       |
| 商品期初标签 | 系统启用后，在创建好商品基础资料后，可以打印商品期初的标签 |  | 参考商品标签的设计即可 |
| 出入库条码标签 | 支持在开单后，直接打印单据上对应的商品标签，方便后续的管理，同时也支持采购入库/销售出库时，根据入库单信息，如供应商、商品的批次、保质期、序列号、商品入库的仓库、仓位等打印成标签贴在商品上，方便后续归类上架及扫码查询、盘点管理。 |  | 具体标签上要放什么字段，取决于实际业务的需求 |

# 3 功能特性

- 支持商品标签、出入库标签，功能详情：[条码标签打印功能](https://vip.kingdee.com/link/s/lOKKv)
- 支持条形码，也支持二维码，关于两种不同的编码方式差异查看：[条码设计基础](https://vip.kingdee.com/link/s/lO7QM)
- 支持打印商品特性字段：如辅助属性。也支持打印单据分录上的特性字段：如批次、保质期、序列号
- **不支持**打印箱码（例如扫这个码，可以看到这个箱里包含的所有商品及数量信息）
- **不支持**打印仓位码（例如扫这个码，可以看到这个仓位里包含的所有商品及库存信息）

# **4 设备支持**

|  | 扫码设备/技术 | 备注 |
| --- | --- | --- |
| 支持 | USB扫码枪无线扫码枪USB扫描平台 | 星辰没有指定任何扫码设备或者品牌，只要是市面上的通用品牌一般都能兼容。验证方法：插上电脑后，打开一个记事本，扫描条码可以带出对应的条码信息，且解析信息准确即证明扫码枪正常运行。 |
| 不支持 | 无线射频识别（RFID） |  |

# 5 字段

## 5.1 支持绑定的字段

- 商品资料中的**商品编码**、**条形码**、**SKU编码**
- 出入库单据中商品的**批次、序列号、保质期等等**

**注意，条码上可以绑定的字段是很多的，这里不一一列举，具体以打印模板中的数据源为准，但是可以绑定不代表扫码时都可以解析到单据上，目前只有移动端支持了扫码解析，具体可见下面第八大点。**

# 6 扫码前检查（确保识别率）

- 条码足够清晰（无挤压、无重叠、无毛边、无波浪纹/锯齿纹、浓度适中）
- 编码方式合适（匹配条码规则及实际纸张的长度，具体可以参考：[条码标签打印模板说明及其设计](https://vip.kingdee.com/link/s/lv78w) 中的“编码方式”）
- 标签边距要留白
- 时刻保持英文输入法！（为了扫码解析准确，使用扫码枪扫码时务必保证电脑上输入法是英文，而不是中文，因目前市面上大多数扫码设备在中文输入法下解析都会不兼容，导致解析错误）
- 要扫码前提是商品已经在星辰中维护好了，并且条形码字段有值

# 7 PC扫码应用

## 7.1 PC扫码入口及说明

| 入口位置 | 效果 | 产品演示 |
| --- | --- | --- |
| 商品分录的商品编码单元格 | 在商品分录中直接扫码，扫完后光标会切换到下个字 | 留意扫码后，光标会切换到“仓库”字段 |
| 商品编码右侧的【扫码】 | 商品编码右侧的【扫码】中扫码录入，如果连续扫描同一个条码，则会累加当前商品的数量 | 留意“数量”字段 |

## **7.4 开单/出入库/盘点场景扫码**

星辰支持在开订单、出入库单以及盘点时通过扫码完成录入，可避免人工录错，提高录单效率。
