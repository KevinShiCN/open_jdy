# 金蝶云星辰商品条行码的应用场景

> 作者：小橙子和小柠檬 | 阅读：431 | 有用：0

金蝶云星辰商品条行码的应用场景：

1、打印出来为商品标签贴在商品上，可以通过扫码枪进行扫码快速出入库；

2、可以移动端扫码，可以带出这个商品信息，和有做了哪些单据；
