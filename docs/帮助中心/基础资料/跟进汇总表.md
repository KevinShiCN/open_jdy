# 跟进汇总表

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：功能介绍, 新特性, 快速入门

# **1 功能介绍**

**支持按业务员、客户、潜客三个维度汇总跟进数据**

1）按业务员+跟进类型汇总：业务员或管理层可查看各个业务员跟进的客户数，以及在跟进过程中做单数量及金额的情况，精准把握跟进的结果

2）按客户+跟进类型汇总：业务员或管理层可查看各个客户跟进的频率，以及跟进过程中这个客户达成交易的情况，精准把握客户动向

3）按潜客+跟进类型汇总：业务员或管理层可查看各个潜客跟进的频率，以及跟进过程中这个潜客的交易需求的变动，为潜客的下一步转化调整策略

**跟进作业可统计的数据有如下：**

| 汇总依据 | 相关数据 |
| --- | --- |
| 业务员+跟进类型 | 客户数、潜客数、跟进总数、签到数、报价单数、报价单金额、订单数、订单金额、出库单数、出库单金额、退货数、退货金额、收款单数、收款金额、换货申请数、换货申请金额、本品上报数、竞品上报数 |
| 客户+跟进类型 | 客户方分层、客户分类、跟进总数、签到数、报价单数、报价单金额、订单数、订单金额、出库单数、出库单金额、退货数、退货金额、收款单数、收款金额、换货申请数、换货申请金额、本品上报数、竞品上报数 |
| 潜客+跟进类型 | 跟进总数、签到数 |

# **2 适用版本**

- 不区分版本（所有版本适用）

# **3 应用场景**

跟进汇总表可以让经营者了解相关销售员拜访后结果，可以辅助经营者做相关决策，并让经营者更好了解销售员跟进任务执行情况
