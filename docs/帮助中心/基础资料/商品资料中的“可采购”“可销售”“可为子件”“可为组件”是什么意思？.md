# 商品资料中的“可采购”“可销售”“可为子件”“可为组件”是什么意思？

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题, 快速入门
> 作者：赵光芬 | 阅读：2474 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/23590)

问题描述：

解决方案：

**适用场景：**

1、商品不再采购或者不再允许销售，或者仅内部管理使用不参与采购销售的商品，可通过“可采购”“可销售”进行管理。

2、存在组装、拆卸业务时，可通过“可为子件”“可为组件”管理子件商品、组件商品。只有“可为子件”“可为组件”商品才可进行组装、拆卸。



**设置路径：**

【基础资料】-【商品资料】-【商品信息】-【商品】-【商品特性】中开启/关闭：
