# 如何删除科目

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题, 快速入门
> 作者：杨桂平 | 阅读：1124 | 有用：30
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/360807498762431488)

【操作流程】

登录系统后依次点击【基础资料】-【财务资料】-【科目】,可以选中对应的科目,点击【删除】按钮。

![上传图片](https://vip.kingdee.com/download/0100c5205a1930064b6aa71a354571f2aa1e.png)

【温馨提示】

已增加的科目若想删除,需要把涉及过该科目的凭证、期初数都删空,在什么都没有数据的情况下,才能删除。如果是从未使用过的科目,可以直接删除。请确认之前有没有录过凭证或期初数。

如果确认已经删除完了,但是还是提示被科目余额表占用,可尝试以下操作:

【操作流程】

1、这个科目随便录入一张凭证,然后删除这张凭证;

2、退出系统,重新登录,然后再去删除科目。

3、或者建议您新增新的科目,然后做凭证,将旧科目的余额转入到新科目里面,然后禁用旧科目,这样旧科目就无法使用了,以后用新科目录凭证。
