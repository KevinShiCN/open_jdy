# 引入采购入库单提示:某一行商品数量与序列号数量不一致?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：杨桂平 | 阅读：780 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391351712671738880)

【问题描述】

引入采购入库单提示:某一行商品数量与序列号数量不一致?

【业务场景】

引入采购入库单提示:第4行商品数量与序列号数量不一致?但是表格中实际的数量和序列号数量是一致的。

![](https://vip.kingdee.com/download/010933c4efbd40bf45cb930af62088f120a5.png)

【解释说明】

序列号组合中录入多个序列号时需要用英文状态下的分号[ ; ]隔开,不能用其他的符号。

【解决方案】

修改导入的表格,将序列号用英文状态下的分号隔开即可。

![](https://vip.kingdee.com/download/01090a5011fd990846688d6253346683a1d6.png)
