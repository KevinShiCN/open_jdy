# 基础资料中的字段支持调整前后显示顺序吗?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：ZOELI | 阅读：59 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/747984899029575936)

【问题描述】

基础资料中的字段支持调整前后显示顺序吗?

【业务场景】

基础资料中的商品、客户、供应商等资料里面的字段移动位置。

【解决方案】

基础资料中的字段支持做位置移动,比如以客户资料为例,点击列表控制,勾选想要移动的字段,可以点击“上移”“下移”“移到”来控制字段显示位置。

![](https://vip.kingdee.com/download/010993764a5529f449a294ecc44b69993672.png)

还可以通过客户列表鼠标定位到需要移动的字段,拖动字段到想要显示的位置,如图演示:

![](https://vip.kingdee.com/download/0109c4bbcdff0e164a39815e0a6589b51331.gif)
