# 如何批量修改基础资料？

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题, 快速入门
> 作者：小微微 | 阅读：1360 | 有用：27
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/61095892706487552)

问题描述：如何批量修改基础资料？

解决方案：批量修改基础资料的方法有以下两种方式：

1、使用系统提供的批量修改功能。

系统为商品信息、客户、供应商提供了批量修改功能。以商品信息举例，在商品信息列表，选中需要批量修改的商品点击批量修改，勾选要修改的项，维护修改的内容点击确认即可。

![上传图片](https://vip.kingdee.com/download/01007cfac9a4fab24228bfa4130a3769a408.png)

2、使用引入模板覆盖引入。

系统为基础资料提供了覆盖并新增的引入模式，可以将需要修改的资料按引入模板引出后，直接在引出的EXCEL中编辑要修改的项目，然后再引入到系统中即可。注意：不可将编码进行修改；引入模板需要包含要修改的字段信息；引入时选择“覆盖新增”的方式引入。

![](https://vip.kingdee.com/download/010035d24e53d68f4daa86760e2a720b8b9d.png)

![](https://vip.kingdee.com/download/010075304687488843c9af2501be0de6bfac.png)

在引出的EXCEL中修改后，使用覆盖新增的方式引入。
