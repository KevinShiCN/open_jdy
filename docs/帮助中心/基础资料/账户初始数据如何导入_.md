# 账户初始数据如何导入?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：370 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/645066646565794048)

【问题描述】

账户初始数据如何导入?

【解释说明】

账户初始数据录入仅支持在启用期间录入,如果已经结账需要反结账。

【操作流程】

在【基础资料】-【账户初始数据】点击右上角引入

![](https://vip.kingdee.com/download/010962fa49ed90214b16bdc7e552cd274f70.png)

下载引入模板

![](https://vip.kingdee.com/download/0109bc209a1b1fd742cd86dd31b8621acc4a.png)在下载的导入模板录入好账户初始数据

![](https://vip.kingdee.com/download/010958f3e7a5f27247788cd85ee40ca47f27.png)

然后上传录入好的文件,引入即可
