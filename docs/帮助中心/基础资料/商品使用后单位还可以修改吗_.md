# 商品使用后单位还可以修改吗?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：ylim | 阅读：1545 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/566065295182519552)

【问题描述】

商品使用后单位还可以修改吗?

【解释说明】

没有使用过的商品单位可以直接修改,使用过的商品见以下说明:

**1、单单位商品**

使用过的商品基本单位(即商品信息中的计量单位字段)不能修改,可以开启多单位添加辅助单位;

**2、多单位商品**

2.1、固定换算多单位商品

使用过的商品基本单位不能修改,辅助单位可以修改,并且换算率可以修改,可以关闭多单位;

2.2、浮动换算的多单位商品

使用过的商品基本单位不能修改,浮动换算的辅助单位不可以修改、固定换算的辅助单位可以修改,可以修改换算率,不能关闭多单位。

![](https://vip.kingdee.com/download/01095fa323644cc94249991e2bb7e70104a9.png)

【温馨提示】

1、对于可以修改的单位如果需要修改,建议先备份再操作

如何备份,可以点击:[账套备份与恢复如何使用](https://vip.kingdee.com/link/s/ljH5a) 查看流程步骤

2、对于无法修改单位的商品,可采用禁用旧商品编码,新建商品编码的方式变通处理。

对于旧商品剩余库存数量处理,可把旧商品的库存数量通过其他出库单出清后,禁用旧商品编码,同时把对应数量的新商品通过其他入库单以入库。

![](https://vip.kingdee.com/download/01093ccedaf7a39b4e8d9ff0f4f4d320414a.png)
其他入库单使用说明:[其他入库](https://vip.kingdee.com/article/522827747409011456?productLineId=35&isKnowledge=2&lang=zh-CN)

其他出库单使用说明:[其他出库](https://vip.kingdee.com/article/522826358809206528?specialId=194821463944026112&productLineId=35&isKnowledge=2&lang=zh-CN)

3、若商品相关单据不多,也可以删除商品所有相关单据后,再修改商品的基本单位。

4、商品使用后,多计量单位暂不支持调整为单计量单位。
