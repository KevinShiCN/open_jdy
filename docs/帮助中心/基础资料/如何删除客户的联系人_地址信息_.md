# 如何删除客户的联系人/地址信息?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：ylim | 阅读：1065 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/571138761489021184)

【问题描述】

如何删除客户的联系人/地址信息?

【业务场景】

维护的客户联系人/地址信息,单据中新增的客户地址同步至客户的联系人/地址信息如何删除?

【解决方案】

菜单栏点击【基础资料】>【客户】

![](https://vip.kingdee.com/download/010964cc354fba954127bda827c81c686081.png)
点击需要删除联系人/地址信息的客户编码
![](https://vip.kingdee.com/download/0109f57799f7b0c449ab9740189c52fdbec5.png)
鼠标放需要删除的联系信息行上,前面会出现![](https://vip.kingdee.com/download/01098a83227ea6494860afa37ac72ddfc7a0.png),点击![](https://vip.kingdee.com/download/01099dc9cf0f8adc4774b2d79762bb19fa06.png)即可删除对应联系人信息行,删除后点击有上角<保存>按钮
![](https://vip.kingdee.com/download/01090e065b6a622f4c049cba44e94ba7e4ec.png)
