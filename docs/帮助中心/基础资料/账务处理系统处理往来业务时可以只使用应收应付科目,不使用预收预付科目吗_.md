# 账务处理系统处理往来业务时可以只使用应收应付科目,不使用预收预付科目吗?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：shaoling | 阅读：58 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/710658719914377984)

【问题描述】

账务处理系统处理往来业务时可以只使用应收应付科目,不使用预收预付科目吗?

【业务场景】

小公司的业务量少且交易很简单,基本以赊销赊购为主,预收预付款项业务也很少,业务单据生成凭证过程中,偶尔有预收预付业务,不想用预收预付科目,需要将预收记应收账款贷方,预付记应付账款借方处理

【解释说明】

账务处理系统处理往来业务时,企业可按需在科目关联设置里,仅关联应收、应付科目,将预收记应收账款贷方,预付记应付账款借方。

若您想要了解科目关联设置如何设置科目问题,请点击:[如何通过科目关联设置,实现业务资料与会计科目的匹配?](https://vip.kingdee.com/knowledge/279985772801801984?productLineId=35&isKnowledge=2&lang=zh-CN)
