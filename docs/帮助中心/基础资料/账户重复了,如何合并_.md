# 账户重复了,如何合并?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题

【问题描述】

账户重复了,如何合并?

【操作步骤】

如果账户A和账户B实际是一个账户,可以将一个账户的余额转入另一个账户,转移后禁用余额为0的账户。

业务上可以把账户A的金额做资金转账单转到B,然后A账户没有金额后,把A账户禁用掉,再以后就用B。参考:**[资金转账单的作用是什么,如何录入?](https://vip.kingdee.com/link/s/Z7tux)**
