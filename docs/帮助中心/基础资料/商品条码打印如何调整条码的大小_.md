# 商品条码打印如何调整条码的大小?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：4385 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/640355121464709632)

【问题描述】

商品条码打印如何调整条码的大小?

【解决方案】

第一步:在【系统设置】-【打印模板】-商品条码,系统自带的打印模板无法修改,复制个新的模板。

![](https://vip.kingdee.com/download/010911556e781a4643b39b62cfb9419b2c95.png)

第二步:在复制的新模板里面调整条码大小。

![](https://vip.kingdee.com/download/01096cf7fa20528e47679a5c589a21535074.png)

打印模板里面选中条码直接拖动来调整条码的位置,调整后之后保存模板即可。

![](https://vip.kingdee.com/download/0109eac81e9b0c8c41aa9126c55e755088f8.png)

第三步:商品条码打印,选择编辑好的模板打印即可。

![](https://vip.kingdee.com/download/01091c21709701734f76bcf149a4c1e803ef.png)
