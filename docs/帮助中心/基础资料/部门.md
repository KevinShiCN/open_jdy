# **1 简介  **

## **1.1 功能介绍**

用于反馈制单人所在的组织/业务单位类型，例如最常见的销售部、采购部、财务部。或者单个的门店、业务部等等，具体视实际业务场景而定。进销存的报表大多支持按部门维度进行数据的统计与查询。

## **1.2 适用版本**

星辰所有版本

## **1.3 系统路径**

基础资料>业务资料>部门

# **2 操作指南**

## **2.1 新增**

点击右上角新增进入部门新增界面，录入部门名称，选择上级部门保存即可。上级部门若选择公司账套名称则为一级部门，若选择一级部门则为二级部门，以此类推。

![](https://vip.kingdee.com/download/0100c916ca4144b94c088bcf3b589bc0e6ba.png)

## **2.2 删除**

部门被使用后不允许删除

## 2.3 **引入说明**

• 引入的字段信息可通过建立新的模板或是修改现有模板设置需要引入的字段，但*号字段部门编码、部门名称、上级部门为必须引入字段

• 其他字段不填写，则引入后默认为空
