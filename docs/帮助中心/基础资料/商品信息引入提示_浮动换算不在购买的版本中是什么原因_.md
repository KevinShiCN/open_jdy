# 商品信息引入提示:浮动换算不在购买的版本中是什么原因?

【问题描述】

商品信息引入提示:浮动换算不在购买的版本中是什么原因?

![](https://vip.kingdee.com/download/0109500ae0df6f704feba6b32a17410ac822.png)

【解决方案】

金蝶云星辰标准版不支持多单位浮动换算,重新设置引入模板,将“浮动换算”字段关闭是否允许引入引出按钮,重新下载引入模板复制粘贴信息后引入即可
