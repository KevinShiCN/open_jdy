# 系统的辅助属性、批次、序列号如何使用?

【**问题描述**】

系统中的辅助属性、序列号和批次各自有什么作用,如何操作?

![](https://vip.kingdee.com/download/0109f61d087cbe434fc3a1263d959712b6d8.png)

【**应用场景**】

**辅助属性:**商品的一些属性,如衣服的颜色、大小这些需要进行库存管理,企业在商品管理就可以对其进行辅助属性管理。一般情况下,辅助属性适合成本或进价相同、售价相同的商品,如果商品属性不同,成本售价差异很大且希望按属性来核算成本和利润,那么我们建议你单独作为一个商品管理,而不使用辅助属性。

首先,在【系统设置】-【系统参数】-【基础资料参数】中勾选“启用辅助属性”,如图:

![](https://vip.kingdee.com/download/01092b70140e55f5447caf7a97a704a40db6.png)

然后,点击【基础资料】-【商品资料】-【辅助属性】,进入到辅助属性列表界面,可维护辅助属性类别、属性明细,如图:![](https://vip.kingdee.com/download/01099bb091b7f50a47dbbd0ea328b5cfb356.png)
(详情可参考 辅助属性管理:[辅助属性(应用、新增、管理)](https://vip.kingdee.com/link/s/lBlyf))
然后,可在新增商品界面,启用商品辅助属性并生成组合,如图:

![](https://vip.kingdee.com/download/0109470b89bd0ecb4ff1af0edc5ce6681e65.png)
最后,辅助属性商品出入库制单,输入商品后可选择属性录单,如图:

![](https://vip.kingdee.com/download/0109d4282ecde4df48e6863cc01e471192b1.png)

**序列号管理:**产品序列号管理软件广泛应用于手机、数码产品等3C行业,每个产品都有一个唯一的序列号编码,用于设备标识,跟踪记录产品的生产、销售、流通、售后等记录。

(详情可参考,序列号商品设置使用流程:[如何新增序列号商品?](https://vip.kingdee.com/link/s/lBljp))

**批次管理**:即批次保质期管理,主要适用于那些对商品质量和安全性有较高要求的行业。如食品行业、医药行业、化妆品行业,因这些行业的产品质量和安全直接关系到消费者健康和生命安全。通过对产品的生产日期、保质期等信息的精确记录和管理,可以确保商品在有效期内销售和使用,从而保障消费者的健康。

(详情可参考,批次保质期商品设置使用流程:[如何新增批次/保质期商品?](https://vip.kingdee.com/link/s/lBlRB))
