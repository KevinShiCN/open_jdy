# 初始化设置

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：快速入门, 整体介绍
> 作者：Cat_Chow | 阅读：8884 | 有用：13
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/627283563821449728)

财务初始化设置包括财务参数和基础资料（财务业务资料和财务期初余额数据）维护。一般情况下财务初始化设置的主要流程如下图：

![](https://vip.kingdee.com/download/010968eee132fd3a45fe81599c315bd916ed.png)

温馨提示：具体流程可能因企业实际情况不同而有所差异，建议根据实际情况进行调整和优化。

| 步骤 | 具体说明 | 学习链接 |
| --- | --- | --- |
| 1.财务参数 |  | —— |
| 1.1基础参数 | (1)启用期间：即建账时设置的系统启用期间，未结账系统支持修改。每个模块的启用期间是独立的，修改财务启用期间不影响资产、出纳、进销存云等模块。(2)会计制度：存在以下情况，会计制度不支持修改。①已录入财务初始数据，②已录入现金流量初始数据，③已录入资产卡片，④出纳日记账选择了对方科目，⑤已录入凭证。 | 点击学习 |
| 1.2财务参数设置 | 设置账薄展示方向，凭证责任划分控制，结账条件设置等。 | 点击学习 |
| 2.财务资料 | 辅助资料维护，财务资料维护，如币别、科目、凭证模板，财务初始余额维护等 | —— |
| 2.1币别 | 在系统启用时，已经选定好记账本位币。系统预置外币信息，企业可根据实际情况增加其他的币别，并维护其汇率。币别与单价、金额小数位有关，不同的币别可以分别设置小数位数；但注意：单价小数位、金额小数位一经改大无法改小，请一定谨慎修改。 | 点击学习 |
| 2.2科目 | (1)根据选择会计制度，系统预先设定配套的科目；在这个基础上，可根据企业自身的记账需要对科目进行维护，包括手工新增或批量导入。 | 点击学习 |
| (2)支持辅助核算科目可以设置12个核算项目，核算项目可以设置必录和非必录。 | 点击学习 |  |
| (3)支持数量核算 科目启用数量核算,录凭证的时候录入单价和数量，然后在数量金额账薄体现。 | 点击学习 |  |
| (4)支持外币核算 期末调汇金额=外币科目的原币余额*调整汇率-外币科目的本位币余额   | 点击学习 |  |
| (5)科目参数设置科目级次、科目编码长度设置。注意： 科目级次和编码长度调大后,不能再调小(即不能再次恢复到调整前的级次） | 点击学习 |  |
| 2.3凭证字 | 系统预设了【记】字凭证字，可根据实际业务需要新增【收】、【转】、【付】等 | 点击学习 |
| 2.4凭证模板 | 凭证模板主要记录日常常用凭证业务，可以在录入凭证时调用，实现凭证快速录入。 | 点击学习 |
| 2.5电子会计档案 | (1)支持用户对已经全部处理完成且已结账的年度的账务信息（如：凭证、会计账簿、法定报表银行对账单）进行数据归档，方便后续查询和追溯。  (2)导出审计文件， 可指定年度导出，目前导出的文件格式是ais的文件格式   | 点击学习 |
| 2.6账户管理 | 出纳模块及进销存云模块的收入支出账户，可在账户管理中新增维护，支持外币账户。 | 点击学习 |
| 2.7辅助资料 | 系统预置辅助核算类型：客户、供应商、部门、职员、商品、仓库和品牌，可以在商品信息和业务资料维护辅助核算明细。 | 点击学习 |
| 系统自定义增加辅助核算类别，通过【辅助资料】自定义辅助核算类别，并录入相关辅助核算明细。 | 点击学习 |  |
| 3.财务期初录入 |  | —— |
| 3.1财务初始余额 | 期初录入数据根据会计启用期间不同有所不同【年初启用】如果是第1期启用，则只需要科目期初余额（即年初余额）【年中启用】(启用期间除1月份外的其他月份)，需录入年初余额，本年累计借方和本年累计贷方，系统会倒数期初余额，在资产负债表里可体现。【注意】损益类科目需要输入实际损益发生额，以保证利润表取数正确，（本年累计借方=本年累计贷方=实际损益发生额）。【示例】如4月建账，启用期间就选择2021年第4期，年初余额就要输入2020年12月底的各科目的余额，还有1-3月的各科目的发生额，按3月份的科目余额表录入。 | 点击学习 |
| 3.2现金流量初始数据 | 年中启用的账套，还需录入初始化年初到初始化当期的本年累计流量，从而保证现金流量表的数据完整（年初启用不需要录入)。【示例】如4月建账，录入1-3的本年累计流量。 | 点击学习 |
| 3.3账户初始数据 | 若需要使用出纳管理系统，并且有现金、银行余额，则可以录入账户初始数据。 | 点击学习 |
