# 收入和支出类别设置-基础资料-金蝶云·星辰

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：功能介绍, 快速入门
> 作者： | 阅读：919 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/6197)

金蝶云星空是支持多组织的，对于存货重点突出了两个概念：所有权归属方及存储地。所有权归属方即货主（组织结构勾选业务组织即为货主），存储地即仓库，仓库所属组织即“库存组织”（组织机构勾选业务组织，还得勾选“库存组织”职能）。从组织机构的设置来看，业务组织一定可以作为货主，但不一定是库存组织，但库存组织一定是业务组织。
      了解这两个概念后，我们再说下库存组织关账与存货核算关账的区别：
      库存组织的作用是管理所有存货库存异动，是从存货存储地的角度进行的，他管理的存货既有企业本身各工厂的，也有供应商或客户的，因此库存组织关账是针对存储在这个库存组织下所有仓库中的存货在关账日之前不能再有变动，并计算库存结存。
      存货核算关账，存货核算是按货主进行核算，是从存货所用者的角度进行关账的。由于多组织的特性以及组织间的协同等等，同一货主的存货不一定存储在自身的仓库中，甚至该货主都没有仓库，因此存货核算关账时，必须将各仓库中属于自身的那部分存货进行关账，关账日之前不能有异动，而不能影响相同仓库中别人家的存货。
       综上，库存组织关账，关闭的是存储在该库存组织下所有货主的存货账，而存货核算关账，关闭的是该货主存储在所有库存组织下的存货账，两者角度不一样。
