# 如何查询客户信用信息,有没有次数限制?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：408 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352041220189184)

【问题描述】

如何查询客户信用信息,有没有次数限制?

【解决方案】

基础资料-客户列表,公开信息列可链接查询客户的信用信息,免费查询次数是200次/3个月)

![](https://vip.kingdee.com/download/010900b3fceda1af474b933e4634826e9d45.png)
【温馨提示】

查询结果来源于企业公开工商等信息,可为企业评估合作方的商业风险提供多角度分析,信息更新日为每月1日;免费使用次数限制为200次/3个月;同一企业的公开信用报告查询次数不得超过5次/月。
