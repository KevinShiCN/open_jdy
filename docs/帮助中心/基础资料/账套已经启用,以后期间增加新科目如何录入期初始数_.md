# 账套已经启用,以后期间增加新科目如何录入期初始数?

【问题描述】

账套已经启用,以后期间增加新科目如何录入期初始数?

【解释说明】

星辰账套具有连续性,财务初始余额指的是启用系统前的最近一期的科目余额数据。若后续需新增科目,只能通过凭证录入来调整科目余额。
