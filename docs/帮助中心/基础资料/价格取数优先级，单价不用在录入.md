# 价格取数优先级，单价不用在录入

价格取数优先级，单价不用在录入

![上传图片](https://vip.kingdee.com/download/010096eae743d84747c0b3b5e7534b8e766c.png)

      小黑是个勤快人，公司大大小小货物进出，价格这块都由小黑严格把关。不忙的时候还能应付，对着价格表一个个查过去。真要是忙起来饭都顾不上吃，别说核对价格了。小黑心想可不能在这样了，既然用了系统，就要为我所用啊。随即他看了下系统，不点不知道，一点发现新大陆了，“价格取数优先级”吸引了他的注意。小黑究竟看到了什么，我们一起来看看吧！

     首先，价格取数优先级支持的单据有：销售类单据取价，采购类单据取价，其他入库单取价，盘盈单取价，异价调拨单取价。这些单据设置完价格取数优先级后，录单时系统就会自动带出单价。![](https://vip.kingdee.com/download/01001285dbd1218e4fb6bc518a7987b3ab8b.png)

      其次，价格取数优先级使用方法：选中某类单据取价，将需要的取数来源开关启用即可。如果该类单据取价有多个取数来源，则按优先级高低依次进行取数，直到取到数为止。如果取数来源都没有数，则单据里的单价就是空。备注：优先级数值越小表示优先级越高。

![](https://vip.kingdee.com/download/0100b555c6a66b4449cca772e5be256f357a.png)

      最后，价格取数优先级的取数来源：选中各类单据，查看对应的“应用说明”。应用说明很详细地介绍了这些来源取数来源取的哪里。![](https://vip.kingdee.com/download/0100e641011cc71643d48bd713f62e94a7e1.png)

比方说销售类单据取价主要有三大来源：

      1、如果启用了商品价格资料中的零售价/批发价/客户等级价/配送价。则是取商品价格资料里面对应价格。![](https://vip.kingdee.com/download/010061d5a62fef0143bf9abc0acdd07e31e9.png)

![](https://vip.kingdee.com/download/0100710ab4d5892748afaa510d5c313a71b7.png)

举例：商品零售价是5，优先级最高，则销售单单价取5。

![](https://vip.kingdee.com/download/01001b30ae9198df430db2570d66cd4d4fd9.png)

      2、如果启用了最近销售单价/最近销售成交价，则是取自最近已审核的销售订单/销售出库单/委托代销结算单的销售单价。区别：最近销售单价不考虑折扣率，而最近销售成交价则除了带出最近销售单价，还会带出之前单据里的折扣率。

![](https://vip.kingdee.com/download/0100b9dabd6e383d4bcf86e0d6df530483d3.png)

举例：商品价格资料显示的最近销售单价为10，上一单设置了折扣率10%。

![](https://vip.kingdee.com/download/010098769e9b9cc34336985ac492f7196715.png)

![](https://vip.kingdee.com/download/0100edc3a052ac6e4b409032696a5eb4dba1.png)

价格取数优先级分别取最近销售价与最近销售成交价，销售单的单价（为了方便对比放一张单，实际操作是两个单）如下：![](https://vip.kingdee.com/download/01002aceaa591ddf42139d9bdf42680e7d21.png)

      3、如果启用了最近采购入库成本，则是取最近已审核的采购入库单的入库单位成本，按照商品+单位匹配。

举例：商品价格资料显示的最近入库单位成本是5，则销售单的单价取5。

![](https://vip.kingdee.com/download/0100dc60ed54ff18477993ed84840bcd70fb.png)

![](https://vip.kingdee.com/download/0100e7410f88a7774b53b6ffa652dae760ff.png)

      4、客户价格资料，取自客户价格资料，多个客户价格资料内部也可以设置优先级。

![](https://vip.kingdee.com/download/0100afe776f14f134190b7039c1cafdf1567.png)

![](https://vip.kingdee.com/download/010004b8b4ca452b4909aff8e88c1c977b9f.png)

举例：设置“客户+商品”的优先级高于“商品+客户”，在分别设置“客户+商品”的销售单价是30，“商品+客户”的销售单价是20。则销售单的单价取30。

![](https://vip.kingdee.com/download/0100f7184884b8334973b4ca0b80b9f36cfc.png)

![](https://vip.kingdee.com/download/0100f2e9e145e0e94424bcfcda2242ad84f9.png)

![](https://vip.kingdee.com/download/0100bb31cb04a6ff48f08ed4baf51275cc8d.png)

![](https://vip.kingdee.com/download/0100b3c3d62f6698423293c6f0856f5ea091.png)
