# 电子会计档案如何使用?财务数据如何归档?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：若有灵兮山之阿 | 阅读：1136 | 有用：1

【问题描述】

电子会计档案如何使用?财务数据如何归档?

【业务场景】

电子会计档案,可以在年度业务全部完成后,对财务数据统一打印、归档、保存,方便日后查阅。财务资料归集电子档案提供在线归档保存、审核、下载保存等。前置条件:只有已全部结账的年度档案才可以归集

支持导出AIS格式审计文件,导出审计文件参考:**[如何导出数据进行审计?支持导出什么格式的文件?](https://vip.kingdee.com/link/s/ZQWqO)**

财务数据归档管理步骤如下:

【操作步骤】

**1、归案的内容**

会计凭证、法定会计报表、会计账簿(总账、明细账、余额表、数量账等)、银行对账单、余额调节表、固定资产折旧等,以及系统外的资料,如:审计报告、合同等需要归案的重要资料。

**2、操作步骤**

依次点击【基础资料】-【财务资料】-【电子会计档案】,打开后点击“新建档案”。

![](https://vip.kingdee.com/download/010965c3f611467547d58c82b419fe62d58c.png)

选择年度结账已全部完成的年度及需要归档的内容后点击【下一步】。
![](https://vip.kingdee.com/download/0109c872e0ad20b5419b9296578cba7528ce.png)
系统会自动显示上一步中已经勾选的需要汇总的清单项,并可以根据需要上传附件信息,确认无误后,点击<下一步>。

![](https://vip.kingdee.com/download/0109eba038e8517e4b88a5fe3c3aec1733eb.png)确认归档后系统会提示“您已完成归档设置”,若归档涉及的资料较多,归档过程需要一定的时间,点击【完成】后可以隔一段时间再查看结果。

![](https://vip.kingdee.com/download/01097409d544009f4b7ba86f0ce98ef7d125.png)

归档完成的数据会在电子会计档案列表中进行显示,可以对归档资料进行审批、下载、补充资料操作

![](https://vip.kingdee.com/download/0109b3ae3d37f7944b3bb3ca417a9bba94e0.png)下载后的文件是以zip压缩包的格式保存的,保存的文件格式是PDF格式,如果需要导出excel表格的电子形式,可以点击进入到各个模块中的右上角进行导出相关表格
![](https://vip.kingdee.com/download/01098f44e58c4ee44b1d8fbffa047e7c5f6b.png)

【温馨提示】

1、一个会计年度只能有一份电子档案,事后可以补充完善档案资料。

2、若无法在基础资料中查看到,请检查是否菜单被隐藏或者没要被授权
