# 如何引出科目?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：杨桂平 | 阅读：416 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/723705007043941888)

【问题描述】

如何引出科目?

【解决方案】

在基础资料科目列表中可以导出科目。支持导出部分科目,也支持导出所有科目;导出方式支持按引入模板格式引出,也支持直接导出科目列表。

依次点击【基础资料】-【科目】打开科目列表界面,点击右上角【引出】即可将科目导出为Excel。

![](https://vip.kingdee.com/download/0109fd20cee3d4e44eabb9d2543633d1cf5c.png)

1、按需要导出科目

1.1 导出部分科目:选择需要引出的科目后再点击右上角引出即可。

![](https://vip.kingdee.com/download/0109e910f34701aa49628ecf37a5ba2697d8.png)

1.2 导出所有科目:点击左上角“全部”后再点击引出即可。

![](https://vip.kingdee.com/download/0109db961936aef04da59e2e8c39a96a1a41.png)

2、导出方式

2.1 按引入模版格式引出:直接点击【引出】按钮就是按引入模版引出,点击引出后系统会提示让选择引入模版,选择模板后点击【确定】即可导出成功。

![](https://vip.kingdee.com/download/0109da670755dd8a4e8aaab56d2e7fef8256.png)

2.2 按列表引出:选择引出的科目后点击引出下拉框的“按列表引出”即可。

![](https://vip.kingdee.com/download/0109c51d2ae0b9274adda55c4db27094ecd3.png)

【温馨提示】

1、**按引入模版格式引出的科目可以直接再导入系统,**并且可以自定义引入模版设置引出的内容,具体可以参考知识:[引入引出模板如何使用?](https://vip.kingdee.com/knowledge/534899739829966336?get_from=knowledge-id&productLineId=35&isKnowledge=2&lang=zh-CN)

2、按列表引出科目时可以通过右上列设置自定义引出的字段内容:[引出单据或报表时如何设置引出的字段信息?](https://vip.kingdee.com/knowledge/471481773592047104?productLineId=35&isKnowledge=2&lang=zh-CN)
