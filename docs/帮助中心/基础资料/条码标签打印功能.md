# 条码标签打印功能

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：功能介绍, 快速入门, 精华汇总

## 1 应用场景

- 对于企业内部管理使用的条码，在商品入库或者针对某个商品/条码需要打印出来；
- 在制作商品价签时需要将商品信息以及商品相关的价格信息打印出来。

有了条码标签后，可以应用在诸如查询商品、扫码开单、扫码出入库、盘点、期初录入等等场景，通过扫码解析，降低手工录入的时间及出错概率，提高日常经营管理的效率。

## 3 应用流程概述

1）先设计条码标签的模板，也可以使用系统自带的预置模板进行调整

2）选单：将采购入库单上有条码的商品行携带到条码标签打印界面，采购入库单上该条码的数量即为打印数量，可修改。

3）选商品：将所选商品的全部条码携带到条码标签打印界面，打印数量默认1，可修改。

4）扫条码：通过扫码枪直接扫描条码录入，找到该商品，打印所扫条码，打印数量默认1，可修改。

5）打印：在下拉列表的打印设置里设置好模板后，可以直接点击打印。

#### **4.2.2 操作流程**

1）单击【基础资料】→【商品资料】→【条码标签打印】，进入条码标签打印界面

2）点击 **选商品/选单/扫条码 **进行选择需要打印的条码。

3）录入打印数量

4）点击打印即可（注意打印-打印设置里一定要提前选择好打印模板，见下图），如果是打印一维码，“打印方式”建议使用“PDF”打印。

![](https://vip.kingdee.com/download/0109992a88d7b8704e60994d0d4e09504079.png)

### 4.3 单据条码标签打印（只适用于专业版、旗舰版）

#### 4.3.1 操作路径

如果是打印单据上的条码，则在单据列表或者单据详情页的打印里选择对应的条码打印即可：

- **单据列表**

![](https://vip-admin.kingdee.com/download/0109fe5f328588a34a86a891df81069b40a3.png)

- **单据详情页**

![](https://vip-admin.kingdee.com/download/01092df5499a99014ec3b7031f128edfb907.png)

#### **4.3.2 标签打印数量**

同时，如果是单据打印条码，条码打印设置里还支持 **按单据商品数量/****每行商品只打印一个标签/打印手工指定** 三种不同的打印方式：

![](https://vip-admin.kingdee.com/download/010993c22ff8c2b34721a7f7e40f2c6d25dc.png)

**标签打印数量参数说明：**

| 参数名称 | 含义 |
| --- | --- |
| 按商品数量打印 | 例如单据中A商品数量10，B商品数量20，则打印的A商品的标签数为10个，B商品的标签为20个。 |
| 每行商品只打印一个标签 | 不管单据中A、B商品数量多少，每行商品都只打印1个标签出来。若模板中有序列号信息，则只打印第一个序列号。 |
| 打印前手工指定 | 不管单据中A、B商品数量多少，每行商品都只打印1个标签出来。若模板中有序列号信息，则只打印第一个序列号。 |

## 6 打印的硬件及纸张配置

在进行正式打印前，需要先配置默认打印机及纸张的规格，通过测试打印页，检测出纸、墨水/碳带、打印效果等是否正常，并需要根据异常结果进行调整。**如果是跟打印机相关的，建议阁下咨询您所使用的的设备厂家售后**。

### 6.1 打印机设置

星辰没有指定任何打印设备，只要是市面上的通用品牌都能兼容。请务必确保使用星辰打印功能时，所使用的打印机是系统默认打印机（可以在本地打印机设置中检查）。

### 6.2 纸张设置

确保条码标签的打印模板大小跟默认打印机实际支持的纸张大小保持一致，且与打印首选项纸张规格是一致的。
