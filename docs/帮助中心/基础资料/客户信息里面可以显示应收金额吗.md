# 客户信息里面可以显示应收金额吗

> 作者：杨桂平 | 阅读：119 | 有用：0

【问题描述】

客户信息里面可以显示应收金额吗

【解释说明】

1、客户信息里暂时无法查看应收余额

![](https://vip.kingdee.com/download/010936b21d8071ed4c3d91fba42f16d60bcb.png)
2、销售类单据选择客户之后会带出应收余额

![](https://vip.kingdee.com/download/01094256cc653f97467fa52e72b7f644e978.png)

3、可以在【应收应付】模块,通过应收账款明细表和应收账款汇总表查看客户的应收余额。![](https://vip.kingdee.com/download/0109f8b6c3c903484a44a5c357eef39967ff.png)
