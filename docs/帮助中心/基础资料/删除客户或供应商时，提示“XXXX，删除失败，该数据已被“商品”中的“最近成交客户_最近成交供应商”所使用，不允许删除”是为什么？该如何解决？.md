# 删除客户或供应商时，提示“XXXX，删除失败，该数据已被“商品”中的“最近成交客户/最近成交供应商”所使用，不允许删除”是为什么？该如何解决？

**【问题概述】**

删除客户或供应商时，系统提示“XXXX，删除失败，该数据已被商品中的“最近成交客户/最近成交供应商”字段引用，不允许删除”是什么原因？该如何解决？

![](https://vip.kingdee.com/download/0100498176d153d24711af79f9752cf9f4cb.png)

**【原因说明】**

最近成交客户/最近成交供应商”是商品价格资料中的字段，是取单据上的值：

- **最近成交供应商**：取最近已审核的采购订单或采购入库单
- **最近成交客户**：取最近已审核的销售订单或销售出库单

通常当单据被反审核或者删除后，一般都会自动清除。

如果删除客户或供应商时，引用过该客户或供应商的单据确实都已经反审核或者已经删除了，但还是提示上述提示语，一般可能是缓存或其他异常原因导致的。

**【解决方案】**

打开商品价格资料列表，找到“最近成交客户/最近成交供应商”还有值的商品，选中，并点击“更新最近价”，出现“更新最近价成功”后，点击“刷新”，操作完毕后再到对应的客户/供应商列表删除对应的基础资料即可。

![上传图片](https://vip.kingdee.com/download/010031100d442f0e48de896794d8ce97ded8.png)

**【适用版本】**

星辰3.1以上所有版本
