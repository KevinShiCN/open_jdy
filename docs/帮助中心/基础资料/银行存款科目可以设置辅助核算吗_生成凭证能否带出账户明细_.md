# 银行存款科目可以设置辅助核算吗?生成凭证能否带出账户明细?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：shaoling | 阅读：1092 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/609551778153014784)

【问题描述】

银行存款科目可以设置辅助核算吗?生成凭证能否带出账户明细?

【解释说明】

1、银行存款科目支持设置辅助核算,业务单据生成凭证时可指定核算维度,并自动带出对应账户的明细信息。

2、**录入银行银行存款凭证时暂不支持录入结算方式**,若企业一定要录入结算方式时建议设置自定义辅助核算的方式,具体操作可以参考[哪些基础资料可以作为辅助核算项?如何自定义的辅助核算?](https://vip.kingdee.com/knowledge/129339770097650293?productLineId=35&isKnowledge=2&lang=zh-CN)

【操作步骤】

点击【基础资料】>【科目】
![](https://vip.kingdee.com/download/0109ab094e5d02874e159054c511d2f2c2e9.png)
点击要设置的科目编码,开启“辅助核算”按钮,选择“账户”保存科目(为了保证账户金额准确性,建议开启必录)

![](https://vip.kingdee.com/download/0109324f5f04b4e647e5873f955076a9cb95.png)
在录入凭证时,即可选择到账户管理中的账户明细
![](https://vip.kingdee.com/download/0109aa24a35a54ce43f0abb0eff83f9c2784.png)
在设置业务科目关联设置时,也可以设置通用科目

![](https://vip.kingdee.com/download/01096963a19d674d401585cbf22dfc3df72c.png)
业务单据凭证模板中,账户管理对应科目可设置核算维度选择账户在单据中的字段取值

![](https://vip.kingdee.com/download/01094a9c0e3e52b4412c9dd0d76a186947dd.png)

业务单据生成凭证时即可带出银行存款科目的明细

![](https://vip.kingdee.com/download/01093512d923b7984dc0a7a2dd2330726a00.png)

【温馨提示】

**1.若需要设置其他辅助核算类别,可以通过自定义辅助核算的方式,进行关联,详情请点击:[哪些基础资料可以作为辅助核算项?如何自定义的辅助核算?](https://vip.kingdee.com/knowledge/129339770097650293?productLineId=35&isKnowledge=2&lang=zh-CN);**

**2.若需要显示账户的账号信息,可以在账户名称中录入账号信息。**
