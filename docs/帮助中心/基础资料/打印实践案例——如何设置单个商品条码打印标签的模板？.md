# 打印实践案例——如何设置单个商品条码打印标签的模板？

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：快速入门, 精华汇总, 应用案例
> 作者：进销存赵嘉丽 | 阅读：1790 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/458581611433293312)

# 1 应用场景

打印商品标签，方便扫码识别商品信息。

# 2 适用版本

星辰3.1以上所有版本

# **3 操作流程**

**下面以打印50mm*30mm的商品条码标签为例，讲解如何设置商品条码标签打印。**

1） 将模板纸张设置为“自定义”，宽度50mm，高度30mm，方向默认为纵向即可，边距都设为1；

>
注意：条码标签的宽高设置保持跟标签纸实际尺寸一致即可，即不粘胶的部分。

![上传图片](https://vip.kingdee.com/download/01008e1fc090eb1547e9b419265b07e116e2.png)

2） 将条形码控件拖至模板上，拖拽调整尺寸，让条码跟纸张尺寸适配，并绑定字段为条形码（点击…选择字段），根据实际需求选择编码方式，本例子中设为code128，勾选“显示标签”，意思就是标签上要显示条码；

![上传图片](https://vip.kingdee.com/download/01002fb7eda4da49465a9cc17a92b4e9dfd1.png)

根据需要还可以拖商品名称等字段到模板上

>
注意：如果不需要包含标题，注意去掉数据源左下角的“包含标题”

![上传图片](https://vip.kingdee.com/download/0100c3bfceb5710c4a649bf4c9711ef1404a.png)

3） 添加完所有需要的元素后，打印预览看一下，结合预览效果，进行微调，完成后保持模板即可。

注意条形码控件与边距要有一定的空白，以保证可以生成完整的条码并且支持识别。
