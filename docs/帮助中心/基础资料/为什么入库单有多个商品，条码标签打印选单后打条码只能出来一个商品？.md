# 为什么入库单有多个商品，条码标签打印选单后打条码只能出来一个商品？  

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题, 快速入门
> 作者：进销存赵嘉丽 | 阅读：609 | 有用：24
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/126687709179280640)

**【问题描述】**

为什么入库单有多个商品，条码标签打印选单后打条码只能出来一个商品？

**【原因解析】**

![](https://vip.kingdee.com/download/01009cca1a54d8cf44e6ba50b842bc6042de.png)

通过流程图可以看到，条码标签打印的**前提**是商品有条码，**没有条码则无法打印**。如果商品未维护条码，在条码标签打印时都会有对应提示：

| 操作入口 | 提示语 |
| --- | --- |
| 选商品 | 没有可以打印的条码信息。 |
| 选单 | 选中单据上没有录入条码信息。 |
| 扫条码 | 系统中未找到该条码信息或者该条码的商品已被禁用。 |

如果一张单包含多条分录，其中一行分录没有维护条形码，

![上传图片](https://vip.kingdee.com/download/0100ad9b8f7c9ea4407497c294315ce256c4.png)

那么最终只能带出维护了条形码的商品行。

![上传图片](https://vip.kingdee.com/download/01007e3a4f46001d4aa7889b8a635f41267f.png)

**【解决方案】**

需先维护商品条码信息。维护路径参考此回答开头的【条码标签打印】流程图的**第一步**。

**【适用版本】**

星辰3.1以上所有版本
