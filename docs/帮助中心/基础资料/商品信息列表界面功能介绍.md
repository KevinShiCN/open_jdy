# 商品信息列表界面功能介绍

【问题描述】

商品信息列表界面功能介绍

【解释说明】

点击【基础资料】-【商品信息】即可打开商品列表,默认展示【可用】状态的商品信息。

![](https://vip.kingdee.com/download/010995f6a44a89644f34b1930f255a3e13be.png)

1、查询方式

![](https://vip.kingdee.com/download/010921c81cb35da5497f9ff9675526d2b415.png)2、商品类别管理:左侧为商品类别,可直接新增商品类别;或是点击右上角【商品类别】管理进行新增商品列表左侧,新增商品类别编辑,增加商品类别。

![](https://vip.kingdee.com/download/0109e601b806f105404089ca648e14a016f3.png)

具体操作步骤可以参考以下链接:[如何新增商品类别?](javascript:void(0);)

3、商品列表常用功能:启用、禁用、删除、复制、批量修改、批量生成条码等。

![](https://vip.kingdee.com/download/0109a0a0b016c993434783a4c8a45e0ee349.png)

启用:禁用的商品可以启用在后续重新使用。

禁用:不再使用的商品可以直接禁用,禁用后做单选择禁用的商品信息。

删除:没有做过任何单据的商品,不使用可以删掉。

复制:相同的商品名称可以直接用复制,复制一个新的商品信息。

批量修改:可以批量修改商品的特性,商品类别等等。

批量生成条码:维护了条码生成规则的前提下,在商品列表选中多个商品也可以批量生成条码。

智能赋码:新增商品信息根据输入的商品名称智能匹配税收分类编码和税率。

发送:设置了企业互联,可以发送商品。发送成功的商品资料,将在发送方的账套里,同步创建《外部商品库》,建立客户商品编码与本账套商品编码的绑定关系。

更新辅助属性:辅助属性名称发生了变化,则可在商品列表勾选需要更新的商品后,点击“更新辅助属性”。若不手动更新,则商品及历史单据中,仍显示修改前的辅助属性名称。

4、商品信息基本功能:新增、引入、引出

新增:新增新的商品:

引入:可下载导入模板导入批量导入商品。

引出:可引出商品信息。

![](https://vip.kingdee.com/download/0109edf82d9cabda420c928b4f271ee4c358.png)

5、商品信息列设置:显示/隐藏部分字段信息,列设置可以上下移动

![](https://vip.kingdee.com/download/010976dde378d6574a6e9801699b3f6e2749.png)
