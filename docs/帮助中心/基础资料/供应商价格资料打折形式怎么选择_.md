# 供应商价格资料打折形式怎么选择?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题

【问题描述】

供应商价格资料打折形式怎么选择?

【操作步骤】

基础资料-供应商价格资料,新增方案,此时可选择**打折形式,主要有两种:折扣率(%)和折扣(折)**![](https://vip.kingdee.com/download/0109ba12559d862d4515bdd17f71374700ee.png)
**1、当折扣形式为折扣率时**

1)以供应商+商品价格类型为例,如下图所示,设置供应商GYS00001**打折形式为折扣率(%)**![](https://vip.kingdee.com/download/0109effcde15777c42768c289a6b32faf4d4.png)
2)设置SP00102商品采购单价900,折扣率为30%

![](https://vip.kingdee.com/download/010983a2fd8ba8e8408e8390135d76d1bb02.png)
3)录入采购入库单,效果如下图所示:**商品SP00102,折扣率30%,折扣7折,现价630**

折扣额=原价*折扣率=900*30%=270

![](https://vip.kingdee.com/download/010976b52298d4b44a7fb1ebff02a50370be.png)
2、**当折扣形式为折扣(折)时**

1)以供应商+商品价格类型为例,如下图所示,设置供应商GYS00001**打折形式为折扣(折)**![](https://vip.kingdee.com/download/0109263e31a515614e42b12713e845412a6a.png)
2)设置SP00102商品采购单价900,折扣为3(折)![](https://vip.kingdee.com/download/0109c21ebcdcb6f34628849c94ed370b2f9f.png)
3)录入采购入库单,效果如下图所示:**商品SP00102。折扣率70%,折扣3折,现价270**

折扣额=原价*折扣率=900*0.7=630;折扣率=(原价-现价)/原价=(900-270)/900=0.7![](https://vip.kingdee.com/download/01094d4510cc727f4b8d95a5bc8cc6a6cce4.png)

温馨提示:可根据业务需要,选择不同的折扣形式
【名词解释】
**折扣**:是商品买卖中的让利、减价的优惠。例如一件商品原价100元,现在打九折,那么折扣就是10元。这种表述方式直接告诉消费者或买家,他们将获得多少金额的直接减免。

**折扣率**:是销售企业让利给购货商,商业折扣和现金折扣。折扣率更多地用于计算和比较不同商品或服务之间的优惠程度,它提供了一个相对的比例,有助于消费者或买家比较不同选项的优惠幅度。

简言之,折扣是便宜的钱(具体数值);折扣率是便宜的钱的比值(相对指标)。比如打8折就是原价的80%,如果折扣率是8的话就只有原价的20%
