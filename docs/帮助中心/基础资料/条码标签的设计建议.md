# 设计建议

**1、售前需求场景确认与调试验证**

客户如果有条码使用的场景，在正式购买产品前，强烈建议先咨询清楚打印和扫码的场景，了解清楚是否有限制，同时建议先进行调试和验证，特别是特小尺寸（如2*3cm，3*4cm等等）的条码打印，一定要控制条码的复杂度（长度不能太长，尽量用纯数字），售前一定要做打印和扫描测试，避免购买后发现无法满足打印场景的情况。

**2、模板设计关键节点检查**

条码扫码受到诸多因素的影响，关键节点的设计会影响到最终打印及识别的效果，的确有一定的操作难度，后期星辰会持续优化并简化设计流程，改善用户体验，现阶段建议用户在设计时尽量使用系统预置模板，特殊行业的标签在设计自定义模板时，也建议严格按照上述的指引进行设计，从而减少调试试错的时间。

**3、珠宝行业参考模板：**

本专题提供了一个星辰设计的**珠宝类标签的预置模板**。

珠宝类标签模板（金蝶设计样式）（勿动-原文件引入）.zip

用户可根据实际标签纸大小、业务字段来进行调整及测试。由于星辰打印模板不支持直接转向，**故所提供的打印模板，在进行打印时，都需要先设置打印机的出纸方向**。请以实际设备的调整结果为准。
