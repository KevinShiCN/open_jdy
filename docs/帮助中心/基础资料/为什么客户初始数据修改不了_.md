# 为什么客户初始数据修改不了?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：Schatz | 阅读：382 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/637456328046079488)

【问题描述】

为什么客户初始数据修改不了?

【解释说明】

客户初始数据仅在业务模块的启用期间可进行新增、修改或删除操作。若处于非启用期间,相关操作选项将呈现灰色显示状态,无法执行任何调整。

![](https://vip.kingdee.com/download/010984c3aa70cd63407095afe5d5a0154148.png)

【操作步骤】

在库存管理模块-业务期末结账界面,

![](https://vip.kingdee.com/download/01095e4cc83ee1cd47e0a6e1e29c36409cac.png)

点击“反结账”按钮,并选择将期间反结至启用期间;

![](https://vip.kingdee.com/download/010996b75da4e7274aea848cfca33e37b8c2.png)

便可以对客户初始数据进行修改。

![](https://vip.kingdee.com/download/0109e941a6f1378b4b14a0f90cd59d4ba35d.png)

【注意事项】

如果期初数据已被核销,需要先删除相关的核销单据,之后才能进行反审核并修改数据。

![](https://vip.kingdee.com/download/0109fb400d7860744499ba9e0f296111abae.png)
