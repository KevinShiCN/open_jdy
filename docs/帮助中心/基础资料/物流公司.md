# 物流公司

# 1 简介

## 1.1 应用场景

销售发货时大多采用物流发货的方式，作为物流发货的基础资料。系统提供维护物流公司信息，后续在销售出库单、调拨出库单上选择交货方式为物流发货时，可选择对应物流发货的物流公司。

## 1.2 操作路径

基础资料>业务资料>物流公司

## 1.3 适用版本

星辰3.1以上所有版本

# 2 维护指南

## 2.1 系统预置的物流公司

系统预置了多个物流公司，用户可根据实际需要新增物流公司。

## 2.2 新增/修改

点击新增，录入物流公司编码、名称即可。对于物流联系人、联系信息等信息字段信息可选填。

# 3 开单录入

## 3.1 适用单据

| 支持录入交货方式的单据 | 采购订单、销售报价、销售订单、销售换货申请、销售退货申请 |
| --- | --- |
| 支持录入交货方式及明细的单据 | 采购入库、采购退货、销售出库、销售退货、委托代销、其他出库、调拨出库 |

## 3.2 开单维护

1）打开适用物流公司的单据，开单维护单据头和商品分录信息

2）在交货信息单据体中选择“物流发货”

3）选择物流公司、录入物流单号、发货时间、物流备注

4）保存并审核单据

## 3.3 批量维护

如果有多张业务单据都需要维护交货信息，打开单据列表后，选择多张目标单据，找到批量更新按钮，在下拉中选择“修改交货信息”

![上传图片](https://vip.kingdee.com/download/010015470f251c8f418788f64ed55d288a37.png)

再录入交货明细，点击“确定”即可。

![上传图片](https://vip.kingdee.com/download/010078a4b521dd494023b791343974596bb8.png)

如果单据已经审核了，需要审核后允许修改，需要在系统设置>单据字段配置中，找到对应单据的“交货方式”，开启“审核后允许修改”并保存。

![上传图片](https://vip.kingdee.com/download/0100183689d77bf0459ba92be23ee50d0d80.png)

# 4 查询

## 4.1 单据联查

录入了交货明细的单据，在单据详情或者单据列表，点击“物流单号”，会跳转到快递100官网联查物流进度，联查无需收费。

>
注意：星辰只是提供了快捷跳转的入口，至于在快递100能否查询出具体的信息，取决于第三方物流公司跟快递100之间是否有合作，或者说当查询过于高频时，查询不出来，属于快递100官方的限流，建议稍等一会儿再重新查询，如果依然有问题，建议咨询快递100官方售后。

## 4.2 报表

采购/销售明细表支持展示交货方式及明细。
