# 如何录入外币科目财务初始余额?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：116 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/774076810941677824)

【问题描述】

如何录入外币科目财务初始余额?

【解释说明】

科目启用外币核算后([如何设置外币科目?如何进行期末调汇?](https://vip.kingdee.com/link/s/ZgRgW)),进入财务初始余额录入界面,可通过 “币别” 栏的下拉菜单选择其他币别,进而录入对应币别的财务初始余额。

所有币别初始余额录入完成后,在币别选择栏勾选 “综合本位币”,系统将自动按本位币折算所有币别金额,并统一展示折算后的结果。

【操作指引】

**1、新增外币**

菜单栏点击【基础资料】>【币别】

![](https://vip.kingdee.com/download/0109b5e176e15a3c40b1ba85803370a48174.png)
如新增美元,详细步骤可以点击:[币别在哪里设置,应用的地方在哪里?](https://vip.kingdee.com/link/s/ZgRXm)

![](https://vip.kingdee.com/download/01097c512d05109c427889ddbd3447f09875.png)

**2、科目启用外币核算**

菜单栏点击【基础资料】>【科目】

![](https://vip.kingdee.com/download/01097565baca52954ab1bfd405f4f9d3da7c.png)
点击科目编码,进入到科目编辑界面,已使用的科目同样支持再启用外币核算

![](https://vip.kingdee.com/download/010963522b47a3ea4d4bbd4afa50443987af.png)

**3、录入外币科目初始余额**

菜单栏点击【基础资料】>【财务初始余额】

![](https://vip.kingdee.com/download/01093056604c6cbb40959292681a4ce32af2.png)
在财务初始余额录入界面,下拉币别可以选择对应的币别录入初始余额数据

![](https://vip.kingdee.com/download/0109d137121676f34c0f8f9a609e48a6f836.png)
如选择美元,录入对应的原币金额100美元,系统会按新增币别填写的汇率换算本位币的金额为100*0.75=750人民币

注:财务初始余额界面不支持直接录入汇率,如需修改汇率,详细可以点击:[币别的汇率如何录入及修改?](https://vip.kingdee.com/link/s/ZgoU1)

![](https://vip.kingdee.com/download/0109672930132d2a4004803a417ed0796a0c.png)
如银行再选择“港币”,录入原币金额100元,汇算本位币80元

![](https://vip.kingdee.com/download/01094a9d99ccf9224e2e8959e0d820ddc5f2.png)

所有币别的初始余额录入完成后,可以筛选“综合本位币”,系统将自动按本位币折算所有币别金额,即为750元+80元=830元

![](https://vip.kingdee.com/download/0109ec69f83ad5544a46922ad08e42757fb2.png)
