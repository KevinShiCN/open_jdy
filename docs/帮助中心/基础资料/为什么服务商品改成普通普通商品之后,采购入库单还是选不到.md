# 为什么服务商品改成普通普通商品之后,采购入库单还是选不到

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：8ear6iu6iu | 阅读：24 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/784590473968435456)

【问题描述】

为什么服务商品的商品类型已经修改成普通商品,采购单据还是搜索不到

![](https://vip.kingdee.com/download/0109cde5d281a8e54428a2abd1e5a11d1dd3.png)

【解释说明】

1.这是因为在新增商品时如果选择了服务商品,商品特性是默认不允许修改的

![](https://vip.kingdee.com/download/0109a173b9652eda4855b7ddb8b9b5c50702.png)

2.此时如果把商品类型修改成普通商品,商品特性会显示出来,但是商品特性里面的“可采购”是没有勾选的,因为之前是服务商品时,服务商品不支持采购

![](https://vip.kingdee.com/download/0109132fb0fe1f3a4489b276bf59ee52b88a.png)

3.所以在变更商品类型之后要检查商品特性对应的参数是否勾选,勾选上可采购之后,再到采购单据里面可以正常显示

![](https://vip.kingdee.com/download/0109e8c1ee60eec44b1f82c5cf4b16ea5d1b.png)![](https://vip.kingdee.com/download/010973a3ae70d2e84c2eae2b1eab0bce0776.png)
