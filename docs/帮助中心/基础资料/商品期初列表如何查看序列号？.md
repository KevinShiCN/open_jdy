# 商品期初列表如何查看序列号？

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：快速入门, 精华汇总, 常见问题
> 作者：进销存赵嘉丽 | 阅读：832 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/371692039043434240)

【问题概述】

商品期初列表如何查看序列号？

【解决方案】

由于一个商品可能对应有多个序列号，如果全部展示出来，期初列表的列宽有限，全部展示也不便于查看，所以这里没有直接展示出来，需要通过点击序列号控件来进行查看：

| 版本 | 演示动图 |
| --- | --- |
| 星辰2.0 | 展示点击序列号空白处，即可弹出序列号弹框 |
| 星辰3.1以上版本 | 双击SN，即可弹出序列号弹框 |
