# 设置供应商价格资料基本单位为什么自动选择“否”?

【问题描述】

设置供应商价格资料基本单位为什么自动选择“否”?

【业务场景】

在设置供应商价格资料时,选择对应的商品后,基本单位显示“否”这是什么原因导致的

![](https://vip.kingdee.com/download/0109a66ecf0208a24ee191a77dc246527b68.png)

【解释说明】

设置供应商价格资料时基本单位自动显示 “否” 的原因及处理方式:

目前该问题是由于网络环境波动导致的,暂时无需特殊处理,研发处理完成将自动恢复正常。

价格资料保存后将返回正常值,步骤操作:

1、先为商品设置对应价格,点击【保存】

![](https://vip.kingdee.com/download/010979cb51b4c69a465c927b2a7a35d99690.png)
2、保存后基本单位会自动恢复为正常显示 “是”
