# 单据自定义字段是否支持公式设置？

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：快速入门, 常见问题
> 作者：赵光芬 | 阅读：2467 | 有用：18
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/115009358143980032)

问题描述：

解决方案：

       支持。新增单据的自定义字段，选择数值型，就可以设置公式了。

![](https://vip.kingdee.com/download/010043cf24eb0bd946319fc18fe94a402953.png)

       在对表头的数值型字段进行公式设置时，若选择了表体的字段，系统自动对表体的数值型字段进行求和后参与公式计算。

      自定义公式还支持设置单据本身的自定义数值型字段参与计算。

![](https://vip.kingdee.com/download/0100f5c779e321834740b0e97864ada8525f.png)
