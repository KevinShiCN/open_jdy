# 在制品初始数据如何查询/修改/删除?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：嘿唬哈 | 阅读：51 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/722980974459271680)

【问题描述】

在制品初始数据如何查询/修改/删除?

【概述】

需先反结账到进销存启用期间,若在以后期间,则初始数据是无法修改的。修改、审核、删除按钮都是灰显的。如果初始数据已经入库需先删除入库单。

![](https://vip.kingdee.com/download/010963850f3469f448d286d918c77f8bf3f9.png)
【操作步骤】

点击【库存管理】>【业务期末结账】>【反结账】;

![](https://vip.kingdee.com/download/0109fc14154759b247ca8f9eeb3235296236.png)

![](https://vip.kingdee.com/download/010977b6caafec9c4a68a867c66bfe6c94ae.png)
点击【基础资料】>【在制品初始数据】可以查看已录入的在制品初始数据记录

![](https://vip.kingdee.com/download/01099cb5e61122494204b8c97dfe2fe11766.png)
反结账到期间启用后,可以查看到在制品初始数据界面,修改、审核、删除按钮均可以点击,但是如果有下游单据,反审核时会提示‘已有以下关联单据,不能反审核,关联单据编号:XXXXXX’,此时我们点击<全流程跟踪>,即可查看到对应的产品入库单,将产品入库单反审核后删除即可。

![](https://vip.kingdee.com/download/0109b4ba27d16d104ce7ba49529c77b6ac26.png)
![](https://vip.kingdee.com/download/0109ba3db34beb0d45fbbf71e332b869810b.png)
![](https://vip.kingdee.com/download/01093bf0ae18bd2e46d9b97539eadcbcb072.png)
删除产品入库单后点击<反审核>,反审核成功后可点击<修改>或者<删除>进行处理。

![](https://vip.kingdee.com/download/010970c81e5e08ad4175b402e502dd107c99.png)
![](https://vip.kingdee.com/download/0109bf2228a766f84f2f8bfe6b87dac7ab38.png)
