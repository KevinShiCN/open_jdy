# 如何新增和管理企业资金账户?

【问题描述】

如何新增和管理企业资金账户?

【相关视频】

[如何新增和管理企业资金账户](https://vip.kingdee.com/school/211507667104390144?productLineId=35&lang=zh-CN&cosmic_type=video)

【解释说明】

企业在采购销售活动中,往往会涉及支付,记录不同的结算账户,可以使得业务结算、查账对账更清晰。

在系统建账时已经预置了现金、银行账户,客户可根据实际情况把公司所使用的具体账户明细新增添加进来(如:工商银行、中国银行、微信、支付宝等),以便后续收付款时选择对应的账户信息记录收支情况。

**各账户之间是平行的关系,不能新增上下级账户。**

【操作流程】

菜单栏点击【基础资料】>【账户管理】,即可进入账户管理界面,

![](https://vip.kingdee.com/download/01098ff1cb633f354a6f9ea0681f344c6820.png)

单击右上角<新增>按钮,在弹出的窗口,维护“账户编码、账户名称、银行账号、开户名、开户行、备注,选择币别”,可设置是否默认账号,最后保存即可。

tips:**若是需设置外币账户,可在币别栏选择对应外币币别,若没有可用币别,可先新增币别再设置账户**,详细可点击:[如何新增币别?](https://vip.kingdee.com/link/s/ZgRXm)

![](https://vip.kingdee.com/download/0109431c509b73d541df82cc17a0c4026e75.png)

可在账户列表对已增加的账户进行【删除】、【禁用】等操作

![](https://vip.kingdee.com/download/0109d63b55c0737b4ee1a897729d8d0ef271.png)如果账户信息过多也可以在excel表格里面填好再导入

路径:账户管理列表右上角点击<引入>按钮,下载默认模版,或者到模版管理里面自定义新增模版,填写资料后导入

![](https://vip.kingdee.com/download/01097625d8f02a0d49e297dfe8f063420611.png)

【温馨提示】

**1.结算方式和账户管理的主要区别是:**

**结算方式主要是用来计入收款的方式,比如现金、转账、支付保等;**

**账户管理主要是资金进入到哪个账户下,比如招行,建行等。**

**2.结算方式新增时,不允许名称一致;账户管理新增时,不允许编码一致,名称可以一致。**
