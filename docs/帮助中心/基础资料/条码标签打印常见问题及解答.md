# 常见问题及解决方案

以下星辰所收到的一些客户使用标签打印过程中遇到的一些常见问题及其处理方案汇集：

>
说明：下文中的图片点击可以放大查看。

| 问题类型 | 问题描述 | 问题定位及处理方案 |
| --- | --- | --- |
| 打印模板设置 | 使用PDF打印，星辰2.0打印模板不能旋转，要怎么转向打印？ | PDF的纸张方向是在打印机里设置，不是打印预览里面设置，具体设置是找到客户用的“默认打印机”设备，然后找“打印首选项”，调整好出纸方向，并保证纸张尺寸是跟打印模板的尺寸是一致的。设置好默认打印机的纸张方向后，直接点击PDF打印时，就会按设置的出纸方向出纸，不需要再手动选择转向如果打印模板里需要支持转向，建议升级星辰3.1 |
| 打印模板设置 | 设计器内是否可以调整条码的深浅？打印不清晰怎么办？     | 星辰不支持调整浓度，只能去打印机设置里调整浓度（或深度）与速度，把打印机速度调低，绝对浓度调大可以改善条码清晰度。如果经过调整后打印效果依然不佳，建议使用分辨率更高的设备。 |
| 打印模板设置 | 可以单独调大条码下方的数字吗？ | 不支持，除非条形码的控件属性中关闭“显示标签”，然后单独设置文本字段，绑定条形码后调整字体大小即可 |
| 打印模板设置 | 珠宝标签打印出来跟纸张方向和大小不匹配，该如何设计模板？ | 1）纸张设置是否有考虑标签纸空白的部分，模板大小与实际纸张是否匹配；2）标签上的信息及条码位置的摆放是否正确；3）出纸方向是否正确。 |
| 打印效果 | 条码出现了断线是什么原因？                | 一般是打印机碳带有问题或者打印头有脏污，需要更换耗材或者联系打印机售后寻求帮助 |
| 打印效果 | 打印预览正常，但是条码打印出来出现锯齿/波浪纹是什么原因？ | 如果打印预览是正常的，按以下顺序检查和调整：1）检查打印出来的竖线、边框等是否有波浪纹，如果有，需重装插件或者重启打印机；2）如果打印方式为插件打印，尝试改用PDF打印。 |
| 打印效果 | 条码标签不清晰 | 建议使用分辨率（DPI）大于300以上的打印设备注：DPI一般可在【打印机属性】中获取（如下图），如果没有查询到也可以向打印机厂家或售后咨询。 |
| 打印效果 | 为什么打印预览时，条形码不显示？ | 通常是因为条形码中包含了中文导致的。条形码的编码规则并不支持中文，因此如果需要扫码解析后支持中文，请使用二维码控件。 |
| 扫码 | 条形码扫不出来或者扫出来不对是什么原因？ | 具体问题具体分析：1）检查编码方式是否跟条码一致如果使用的是13位的69码，编码方式只能是ean-13，那么使用code128就会无法识别如果是自定义规则的条码，建议使用code128条形码编码本身有误，使用了ean-13的格式，但是校验码实际是错误的，检查条形码的正确性2）检查是否条码密度已经超过标签纸能容纳的范围了，如果条码长度很长，很复杂（同时包含字母数字符号等信息），建议调整条码样式，改成用二维码，或者使用更大的标签纸3）检查PC端的输入法，务必确保是英文输入法4）如果是特小标签（打印区域3*2cm）检查：①是否打印速度太快；②是否打印浓度太高；③是否打印转向；④是否编码方式不对；⑤是否条码太复杂了 |
| 扫码 | 用ean-13的条码，打印出来后，有的商品能识别，有的不能识别 | 详见知识：ean-13的条码，打印出来后，有的商品能识别，有的不能识别，是什么原因？ |
| 扫码 | 序列号弹窗录入序列号时，扫码后没有自动换行是什么原因？ | 是因为中文输入法不兼容扫码换行的功能实现。将中文输入法切换成英文输入法即可，可见下图效果演示。 |
| 扫码 | 单据上扫码序列号时，为什么扫码后没反应，需要手动敲回车，或者提示扫码后提示商品不存在？ | 出现上述情况都是序列号中包含了英文字母，因为中文输入法不兼容含英文字母的条码解析，只需将中文输入法切换成英文输入法即可。扫序列号后没反应的解决办法：扫序列号后提示商品不存在的解决办法： |
| 打印出纸 | 为什么点击了打印，但是打印不出纸？ | 通常打印不出纸是由于打印模板中的纸张设置与打印机的首选项中的纸张尺寸不一致引起的，需要检查是否一致，并进行调整，具体的操作办法详见下方的知识。普通纸张及标签纸打印均适用，思路是一样的。相关知识：可以打印预览，但是点击打印后不出纸的原因及解决方法 |
