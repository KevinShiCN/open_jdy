# 引出基础资料时,会有一部分字段显示乱码或带出语言信息

【问题描述】

引出基础资料时,会有一部分字段显示乱码或带出语言信息
![](https://vip.kingdee.com/download/0109ed297497446b4d638dc5f833c4044be0.png)
【解释说明】

此问题预计8月12日(周二晚修复),可8月13日(周三)上午重新操作验证,目前可通过以下变通方案解决

【解决方案】

在列表选择500条/页,然后全选后,去掉一条,**选择499行再引出,这样引出不会有这个问题**
