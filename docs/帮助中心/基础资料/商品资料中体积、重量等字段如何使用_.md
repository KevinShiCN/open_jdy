# 商品资料中体积、重量等字段如何使用?

【问题描述】

商品资料中体积、重量等字段如何使用?

【业务场景】
- 对于食品/粮油/大米批发类客户,需要根据商品的重量来计算司机的提成
- 对于五金建材行业、家装行业(板材、 陶瓷、墙纸等),需要根据客户订单的面积来确认价格
- 对于海鲜、电器行业按照商品的体积来规划装箱或装车的空间
- 【解释说明】
- 在商品资料中提前维护好商品的毛重、净重及长 × 宽 × 高信息后,后续进行业务开单时,系统会自动带出该商品的净重、毛重,以及按开单数量计算的总净重、总毛重、单个体积和总体积数据,无需手动录入。
-
- •总毛重=商品基本单位数量*毛重(商品资料上的毛重)
- •总净重=商品基本单位数量*净重(商品资料上的净重)
- •总体积=商品基本单位数量*体积(商品资料上的体积)
- •体积 =(商品资料中的)长*宽*高
- 【操作指引】
- 维护好商品信息-其他信息中的相关字段信息,维护重量和长宽高需要维护对应的体积单位和重量单位(必录)
- ![](https://vip.kingdee.com/download/0109db12bfd60ca447ba8f4a8e2c5cab9511.png)
- 若新增商品界面没有显示对应的字段信息,可以点击右上角设置按钮,显示对应的重量和长宽高字段等信息
- ![](https://vip.kingdee.com/download/0109c84c152bdae44a10af7c2a55589184c8.png)
- 在业务开单界面,可以在列设置显示对应的字段信息
- ![](https://vip.kingdee.com/download/010985722e45fb6c410e914867b7ba8d3ccd.png)
- 选择商品,录入数量后,系统会自动带出相关字段信息
- ![](https://vip.kingdee.com/download/0109004f2678211b41a9a37d3b9c494a50d7.png)
