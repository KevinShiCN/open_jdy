# 会计科目列表的删除功能可以隐藏吗?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：Schatz | 阅读：97 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/646878375423913984)

【问题描述】

会计科目列表的删除功能可以隐藏吗?

【解释说明】

删除按钮无法隐藏,但可以设置“删除全部科目”选项不直接显示。在科目列表界面,点击右上角的“更多”,然后取消勾选“删除全部科目”,这样该选项就不会直接显示出来。

![](https://vip.kingdee.com/download/010922af1989abb54a3896aa09f1796b6314.png)

如果不希望子账户拥有删除科目的权限,管理员可以在系统设置的用户管理界面进行操作授权,取消勾选子账户的科目删除权限。

![](https://vip.kingdee.com/download/01098f26749cef6b4ed68840cb66d609f53a.png)

这样一来,当子账户点击“删除”或“删除全部科目”时,系统会报错提示无权限,从而阻止其进行删除操作。

![](https://vip.kingdee.com/download/010910f1a6d2086d4dce93e10ca9fd212927.png)
