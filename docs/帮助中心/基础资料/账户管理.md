# **1 简介  **

## **1.1 **应用场景

企业在采购销售活动中，往往会涉及支付，记录不同的结算账户，可以使得业务结算、查账对账更清晰。在系统建账时已经预置了现金。

## **1.2 适用版本**

星辰所有版本

## **1.3 系统路径**

基础资料>财务资料>账户管理

# 2 操作指南

## 2.1 新增

单击【新增】按钮，在弹出的【账户管理】窗口，录入账户编码、账户名称、银行账号、开户名、开户行、备注，选择币别，设置是否默认账号，再单击【保存】即可完成账户录入返回主界面。如果需要连续新增多个账户，在录入完上述信息后，单击【保存并新增】，系统会自动保存录入的账户，同时继续保持当前【账户管理】新增窗口。

![](https://vip.kingdee.com/download/01005011c4df7d2e4e1585cda3ea168ff4cd.png)

## 2.2 删除、禁用

如果需要对已增加的账户管理进行删除、禁用等操作，单击账户列表左边的复选框，上方会显示对应的操作按钮，如下图所示：

![](https://vip.kingdee.com/download/01003ec0fcaa1113417c86be23e5367a321f.png)

## 2.3 修改

如果需要修改对应的账户，直接双击对应账户所在行，或单击账户编码即可。

## 2.4 银企互联

为了提升企业资金管理效率，金蝶云星辰上线了银企互联功能，申请银企互联后，可直接在星辰查询企业最新的交易动态及余额，无需登录银行系统。

目前对接的银行有：交通银行、招商银行、平安银行、广发银行、民生银行、浦发银行，星辰正在快马加鞭陆续对接更多的银行，后续将为大家提供全方位的银企互联服务，请大家持续关注哦~~
