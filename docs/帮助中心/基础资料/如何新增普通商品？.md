# 如何新增普通商品？

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：快速入门, 常见问题
> 作者：小微微 | 阅读：854 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/23958)

问题描述：

解决方案：

单击【基础资料】→【商品资料】→【商品信息】→“新增”，进入商品资料新增界面，可按下面的方式录入商品信息：

1）扫码录入

对于国家商品库中存有条码信息的商品，可点击左上角的“扫码”，通过扫描条码或手工录入条码，可自动带出该商品的相关信息，包括商品条码、商品名称、品牌、规格型号、计量单位、产地；

![](https://vip.kingdee.com/download/01003577d8cd8b85495b8984db861c31eb50.png)

2）手工录入

必录字段包括： ①商品名称 ②计量单位 ③成本计算方法：默认为“移动平均法”，可根据企业需要修改。
