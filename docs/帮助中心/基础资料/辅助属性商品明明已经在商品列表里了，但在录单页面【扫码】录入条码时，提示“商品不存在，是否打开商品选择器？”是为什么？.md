# 辅助属性商品明明已经在商品列表里了，但在录单页面【扫码】录入条码时，提示“商品不存在，是否打开商品选择器？”是为什么？

# **【问题现象】**

辅助属性商品明明已经在商品列表里了，但在录单页面【扫码】录入条码时，提示“商品不存在，是否打开商品选择器？”是为什么？

![上传图片](https://vip.kingdee.com/download/01000d9b593cb7854d38a1a430226b2e3d4a.png)

![上传图片](https://vip.kingdee.com/download/01001991953c6ec6445cae3fa9cfd1fd6d29.png)

# 【原因定位】

通常出现这个原因都是因为辅助属性条码里含有空格，导致解析失败造成的。

![上传图片](https://vip.kingdee.com/download/01005f3cf524bedf4447b0c8075c970096af.png)

一般正常系统按条码规则自动生成的条码是不会有空格的，但如果条码生成规则是拿“商品编码”来生成，且“商品编码”里本身首位就含有空格，那么就容易导致这种情况。

![上传图片](https://vip.kingdee.com/download/0100a4062882248c47149526caf35eaa10b3.png)

# 【解决办法】

先把商品编码前面的空格去掉（如果商品编码前面没有空格则忽略这一步），然后把辅助属性条码全部选中，点击“清空”后重新生成，保存即可。
