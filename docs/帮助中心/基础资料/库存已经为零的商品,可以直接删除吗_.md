# 库存已经为零的商品,可以直接删除吗?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：云社区用户55os622 | 阅读：351 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/637456374015568384)

【问题描述】
库存已经为零的商品,可以直接删除吗?

【解释说明】

商品资料需要在没有使用过的情况才允许删除**,若商品已使用过(发生过业务单据),即使库存已为零,也是不允许删除的**。

如果该商品确认库存为零且未来一段时间内都不需使用,可做商品禁用操作,这样录入单据就不会再弹出不需要的商品。

禁用商品依次点击【基础资料】-【商品资料】
![](https://vip.kingdee.com/download/0109ec032bd9e5864f5fbd667e7412cf39f1.png)

选择对应的商品,点击【禁用】即可

![](https://vip.kingdee.com/download/0109ca93e9a440014b33902d3e2e1b6c99c9.png)
