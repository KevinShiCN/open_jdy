# 怎么新增商品计量单位?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：杨桂平 | 阅读：3208 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/129339770080873005)

【问题描述】

怎么新增商品计量单位?

【解释说明】

商品计量单位是对商品的数量、重量、体积、长度、面积等属性进行计量核算所采用的标准量度单位,是商品采购、销售、库存管理、账务核算等业务环节中,统一商品计量口径、准确统计商品数量的基础依据,也是财务核算、成本归集的重要计量标准。每个商品信息都需要维护对应的计量单价。

【操作流程】

1、手工录入:

【基础资料】-【计量单位】,点右上角【新增】,录入单位名称,设置小数位数并预览效果,最后点【保存】即新增成功(※为必录字段)。

![](https://vip.kingdee.com/download/0109b4af71e6e5de416bb2f868cd01d61aaf.png)2、通过excel导入:依次点击【基础资料】-【计量单位】,点右上角【引入】下载系统的引入模板,编辑后再引入。

![](https://vip.kingdee.com/download/01094b87fc8ca34947f0aecabb77c1ba1a0e.png)

3、如果需要对已增加的计量单位进行删除、禁用等操作,单击计量单位列表左边的复选框,上方会显示对应的操作按钮;

![](https://vip.kingdee.com/download/0109dd723416546c4251af3935082ce67621.png)**注意:**未被商品引用的计量单位可以删除,已被商品引用的计量单位只支持禁用。

4、如果需要修改对应的计量单位,直接双击对应计量单位所在行,或单击计量单位名称即可。

![](https://vip.kingdee.com/download/01098c6eb525420d433ebc62283acd14aa0a.png)

**注意**: 未被商品引用的计量单位可以修改数量小数位(改大改小),已被商品引用的计量单位数量小数位仅支持改大,不支持改小,数量小数位一经改大无法改小,请一定谨慎修改,修改前建议操作备份账套,以防操作失误

5、系统的计量单位支持引出:依次点击【基础资料】-【计量单位】,点右上角【引出】即可。
