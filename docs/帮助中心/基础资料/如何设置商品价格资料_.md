# 如何设置商品价格资料?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：杨桂平 | 阅读：3522 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/129339770097650197)

【问题描述】

如何设置商品价格资料?

【解释说明】

企业A,商品向固定的供应商拿货,采购价格波动不大。向客户销售时,所有的客户售价一致。这种情况下,企业A只需要维护商品的采购价格和销售价格等价格即可。

即商品的采购价格及销售价格较为固定,或是同一等级的客户同一商品为相同的销售价格。可以通过商品价格资料维护采购价及销售价格。新增单据时,选择供应商、客户时即可带出对应的价格。

商品价格为非必录项,若前期未维护相关价格信息,可暂不录入,后续业务开展过程中支持随时补录。

【操作步骤】

**1、设置商品价格资料:商品价格资料在商品信息或商品价格资料中维护**

1.1 进入【基础资料】-【商品资料】-【商品信息】

![](https://vip.kingdee.com/download/01092055148eb11f4e7390d2810fef543f79.png)

打开商品详情可对商品进行价格设置,如下图:

![](https://vip.kingdee.com/download/0109281df7b6e9334d58ac60faebc0a61911.png)

1.2 在【基础资料】-【价格设置】-【商品价格资料】,修改/批量修改商品价格。

![](https://vip.kingdee.com/download/010994b7c06aa7c84165bff8bb82c01b4bc0.png)商品价格资料列表的更多操作,可以参考以下链接:[商品价格资料](https://vip.kingdee.com/link/s/lnUSg)

**2、价格等级设置:系统已有预置的价格等级,企业也可以根据实际情况自定义新增;价格等级最多支持10个。**

在【基础资料】-【辅助资料】中,选择【价格等级】进行新增。每新增一个价格等级,商品价格资料可多维护一个对应的等级价格。

![](https://vip.kingdee.com/download/01095f684bacba8342d3b5417ca6e23f8d39.png)

价格等级设置更多详细说明可参考:[客户资料中的价格等级是什么,如何设置使用?](https://vip.kingdee.com/link/s/lnUSE)

**3、****价格取数优先级设置:****设置开单自动带出对应等级价格**

![](https://vip.kingdee.com/download/0109ad8c02ac9b45426c94e66401e77fa0eb.png)

价格取数优先级更多详细说明可参考:[如何设置价格取数优先级?](https://vip.kingdee.com/link/s/ZFCZQ)

**4、采购开单或是销售出库时,即可带出对应价格。**

![](https://vip.kingdee.com/download/0109dbdc6a0d8e8b4a4c9ac0ab2561383ee7.png)

**5、商品价格资料支持批量导入**

①商品编码、单位必录,根据商品编码和单位确定要更新的价格资料;

②条形码非必录,若同单位多个条码,未录入条码的情况下,默认更新第一行的价格;若条码有值,则根据条码值进行匹配;若条码值有值但匹配不成功,则导入失败;

③启用了辅助属性的商品,SKU编码非必录,在未录入的情况下,更新主商品的价格;若SKU编码有录入,则更新或增加对应辅助属性的价格资料;若SKU编码有值但匹配不成功,则导入失败;

④若未匹配到系统中的商品编码和单位时,导入不成功;

⑤启用了多单位的商品,导入时按单位导入,即导入商品A件的价格为8元,箱的价格不会被覆盖,原来是0还是0,原来有值,还是原来的值。
