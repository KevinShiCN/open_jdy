# 导入商品提示“数值字段长/宽/高格式异常”或“体积单位不能为空”

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：杨桂平 | 阅读：61 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/743281618068033536)

【问题描述】

导入商品失败提示“数值字段长/宽/高格式异常”或“体积单位不能为空”如何处理?
![](https://vip.kingdee.com/download/0109ec0d2e021d8a4a3daa6f8533de6a5b96.png)
 【解决方案】

导入商品时提示数值字段格式异常,表示该字段需录入数字格式才能导入,不能输入文字或其他字符;提示体积单位不能为空,表示商品信息录入长、宽、高后必须录入体积单位,否则导入失败。

![](https://vip.kingdee.com/download/0109e5bf3be608db4d1ca8ab8b0b20911c10.png)

修改导入模板,录入正确的数据后重新导入:

上述案例中在导入模板中找到长 / 宽 / 高字段,录入数字后再导入。若需要录入长、宽、高的信息,则必须选择体积单位。
![](https://vip.kingdee.com/download/010945f22d0e0d8a4ba28b1f4bfd847529b5.png)
