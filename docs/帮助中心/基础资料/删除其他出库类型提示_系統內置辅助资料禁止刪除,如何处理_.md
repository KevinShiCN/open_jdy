# 删除其他出库类型提示:系統內置辅助资料禁止刪除,如何处理?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：69 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/635281640763412736)

【问题描述】

删除其他出库类型提示:系統內置辅助资料禁止刪除,如何处理?

![](https://vip.kingdee.com/download/0109d003a53d9c1c402db8a3e206c96a93e4.png)

【解决方案】

系统预设的其他出库类型**不支持删除,**

可以修改其他出库类型的名称后续使用。

![](https://vip.kingdee.com/download/01095ddd6e848bac4efda4dec80fc4d3c9cb.png)
如后续不需要使用,可以禁用掉。

![](https://vip.kingdee.com/download/010994c5b571573f428b8a86ffdcdab82c96.png)

录单无法选择禁用掉的其他出库类型。

![](https://vip.kingdee.com/download/0109c021b2c36a4046bfba9c3619bf184489.png)
