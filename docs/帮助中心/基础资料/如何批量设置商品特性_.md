# 如何批量设置商品特性?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：BBAABABA | 阅读：323 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/550119998442875904)

【问题描述】

如何批量设置商品特性?

【业务场景】

希望批量修改商品的特性,如何操作?

【操作路径】

点击基础资料--商品信息,勾选要批量修改商品特性的商品点击【批量修改】

![](https://vip.kingdee.com/download/0109d751c6fcf32e4a218392e802aa047ee7.png)

已经做过单据的商品,不能修改“序列号”、“批次”、“保质期”管理,若需要修改的话,需要把相关的单据都先删除才可修改。详情请参考:**[没有启用批次/保质期管理的商品还能启用批次/保质期管理吗?](https://vip.kingdee.com/knowledge/565702841348951808?productLineId=35&isKnowledge=2&lang=zh-CN)**

![](https://vip.kingdee.com/download/010993913d5b75fe4fd7b248e9ec6c263fbd.png)
支持批量开启或关闭的商品特性如下截图,**辅助属性的商品特性不支持直接在系统中批量启用或关闭。**
![](https://vip.kingdee.com/download/0109319fdf6e38404424a8055ea7c3e6965a.png)
辅助属性管理如需批量开启或关闭,请通过按模板导出商品,模板中修改后,再覆盖新增导入。

![](https://vip.kingdee.com/download/0109c2aa952966094561b0f6afcce59115d8.png)
模板中辅助属性管理一项选择是或者否,保存模板

![](https://vip.kingdee.com/download/01097d37f4d6733240f8aa1c6085dc956580.png)
辅助属性分录补录或者是删除后保存模板。

![](https://vip.kingdee.com/download/0109c6f349958cd24b7aacbab45e61557e16.png)
到系统中点击【引入】,选择“覆盖新增”,上传修改后模板,然后点击【开始引入】,引入成功后就会根据修改结果更新;如果是**使用过的不支持修改关闭或者启用**

![](https://vip.kingdee.com/download/0109ee1f5656cbdc46429986cb19664df796.png)
