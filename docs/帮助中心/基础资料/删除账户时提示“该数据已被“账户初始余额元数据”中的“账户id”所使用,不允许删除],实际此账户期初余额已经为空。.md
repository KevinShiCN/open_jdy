# 删除账户时提示“该数据已被“账户初始余额元数据”中的“账户id”所使用,不允许删除],实际此账户期初余额已经为空。

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：姜宁8 | 阅读：453 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391351997279965696)

【故障现象】

删除账户时提示“该数据已被“账户初始余额元数据”中的“账户id”所使用,不允许删除],实际此账户期初余额已经为空

![](https://vip.kingdee.com/download/010947fa4ed4e8ec47dd967347931bce93ef.png)![](https://vip.kingdee.com/download/01091597694632724da9a059ea531523ae69.png)

【解决方案】

如“删除测试”账户期初余额仅删除数值,账户余额仍是被占用,需要把整行记录完全删除后,才可能成功删除账户。

![](https://vip.kingdee.com/download/01098b8f7500a2ad4d40ba8659f74cf374cf.png)![](https://vip.kingdee.com/download/010957f460aba0d84a529a747bb0fc94970e.png)

【温馨提示】

如果有做客户对账单,银行对账单相关数据未删除,删除后即可再来删除这个账户。
