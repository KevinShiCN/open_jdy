# 币别的折算方式可以修改吗?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：岸边把酒 | 阅读：250 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/632383190791075328)

【问题描述】

币别的折算方式分为原币 * 汇率 = 本位币和原币 / 汇率 = 本位币,当设置好币别以后,想要修改折算方式,这个是可以的吗?

【解释说明】

币别的折算方式是可以修改的

1、如果该币别没有被使用过,即没有录入过该币别的单据,直接可以通过基础资料-财务资料-币别进行修改。

(1)首先进入币别列表

![](https://vip.kingdee.com/download/010916da6c8488a54052a92de4bdb91940d2.png)
(2)然后在币别列表界面点击对应币别编码,修改折算方式以后,保存即可

![](https://vip.kingdee.com/download/0109bfd08474123e459eb6d3a199b15f7080.png)

2、存在修改折算方式时,提示“币别已被引用,无法修改折算方式”。这个是因为有录入用该币别核算的单据,导致无法修改。

![](https://vip.kingdee.com/download/010972bed310b8df4e88a33decbba29601d3.png)

(1)首先找到所有使用该币别核算的单据

![](https://vip.kingdee.com/download/01096eca219af6bd45659b211482f01dac73.png)

(2)然后反审核删除这些单据

![](https://vip.kingdee.com/download/0109e8a91b4200414920bc891547dc330200.png)

(3)再返回币别列表,修改折算方式即可

![](https://vip.kingdee.com/download/01091a0eb8f988bc4dc7b6babcce79196a66.png)
