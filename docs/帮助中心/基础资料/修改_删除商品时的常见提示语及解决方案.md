# 修改/删除商品时的常见提示语及解决方案

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题, 快速入门
> 作者：进销存赵嘉丽 | 阅读：1779 | 有用：7
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/193743927366143232)

# 【问题概述】

修改/删除基础资料时，已删除相关的单据，但是系统出现以下提示是什么原因？如何解决？

提示1：属性组合[XXXX]行删除失败-[即时库存临时表实体]的字段[辅助属性编码]引[用了此资料数据

提示2：该数据已被‘即时库存临时表实体’中的‘XXX’所使用，不允许修改/删除”

提示3：商品修改失败-{XXX的字段“商品编码”引用了此资料数据}不允许修改基本单位

# 【原因说明】

要修改或删除的基础资料如果之前发生过相关业务数据，那么库存查询表、预计可用库存表通常就会引用到这部分的数据，并存储该字段的记录保存在后台数据库中。要怎么查询这部分的数据呢？请继续往下看。

# 【解决方案】

## **方法1：查询库存查询表**

在库存查询表中勾选"显示零库存商品"，检查是否存在库存字段都已经为0的商品行。如果存在，说明即时库存已经是0，但是数据库中存数了。这时候，就算相关的单据已经删除了，数据库的记录还在，系统就会判断基础资料被引用了，不允许修改或删除。

![](https://vip.kingdee.com/download/0100c4b07da67716485694520c88b596f0da.png)

**解决办法：**

1）如果存在上述的情况，那么打开库存查询表后，勾选“显示零库存商品”，找到库存为0的商品对应的记录

2）单击右上角的“校对库存”后可以清理

3）刷新库存查询表后，重新回到要修改或删除的资料界面进行修改或删除即可。

![](https://vip.kingdee.com/download/010064aed08d339e4b3b83dee536da03ac03.png)

## **方法2：查询预计可用库存表**

如果在库存查询表中勾选“显示零库存商品”后，没有查询到任何记录，那么通常会是还有预计可用库存，可以通过预计可用库存表来查看商品是否还有一些预计入库项。

例如：可用库存公式里没有勾选“采购在途”，但是还有已审核未下推的采购订单，此时在库存查询表就查不到任何的库存，但是预计可用库存表里是有采购在途的数量的。

**解决办法：**

1）如果存在上述的情况，可以在预计可用库存表里，点击对应的数字（一般蓝色字体的表示可以联查），联查引用了该商品的单据。

![上传图片](https://vip.kingdee.com/download/01001d420354062040f294a4f50e1959cb87.png)

2）删除相关联的单据

3）再到库存查询表重新校对库存后，再删除/修改商品即可。

>
注：如果按上述两种方法都操作过后，还是不能删除/修改，请联系售后在线技术支持提单排查。

# 【适用版本】

星辰3.1以上所有版本
