# 如何合并仓位?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：shaoling | 阅读：84 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/710296145100105216)

【问题描述】

如何合并仓位?

【业务场景】

仓库的货架更换更大的,更换后实际仓位重新整理了,整理后仓位已经没有那么多,想要把系统设置的仓位也整理下,如何合并系统原有的仓位

公司的仓库重新规划了仓位,仓位重新规划后,需要把仓位合并处理下

【解释说明】

目前系统暂**不支持**仓位合并功能。如需停用仓位,建议先将仓位内的商品转移至其他仓位,再进行禁用操作。

将仓位内的商品转移至其他仓位,操作详情可查看:[如何将商品库存从a仓库/仓位转移到b仓库/仓位中?](https://vip.kingdee.com/knowledge/502284805602587392?productLineId=35&isKnowledge=2&lang=zh-CN)

如何禁用仓位,操作详情可查看:[如何查询启用/禁用仓位?](https://vip.kingdee.com/knowledge/682754810675509504?channel_level=%E9%87%91%E8%9D%B6%E4%BA%91%E7%A4%BE%E5%8C%BA%7C%E6%90%9C%E7%B4%A2%7C%E5%AE%98%E6%96%B9%E7%9F%A5%E8%AF%86&productLineId=35&isKnowledge=2&lang=zh-CN)
