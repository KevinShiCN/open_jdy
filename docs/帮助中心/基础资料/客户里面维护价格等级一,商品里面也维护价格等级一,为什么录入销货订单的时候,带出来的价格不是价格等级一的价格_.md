# 客户里面维护价格等级一,商品里面也维护价格等级一,为什么录入销货订单的时候,带出来的价格不是价格等级一的价格?

> 作者：杨桂平 | 阅读：445 | 有用：0

【问题描述】

客户里面维护价格等级一,商品里面也维护价格等级一,为什么录入销货订单的时候,带出来的价格不是价格等级一的价格?

【解决方案】

【基础资料】-【价格设置】-【价格取数优先级】,检查“商品价格资料中的客户等级价”为优先级是否是最高的,需要上移到1
