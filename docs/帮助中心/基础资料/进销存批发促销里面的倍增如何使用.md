# 进销存批发促销里面的倍增如何使用

【问题描述】

进销存批发促销里面的倍增如何使用?

![](https://vip.kingdee.com/download/01093d39da60da4b450cb013b77c3faebd1c.png)

【解释说明】

如购买B啤酒,买5件送1件,买10件送2件,多买多送;

【操作流程】

1)打开【基础资料】-【价格设置】-【批发促销】,点击“新增”

![](https://vip.kingdee.com/download/0109b93ac6c24485439eb4d1c2087fc5aa50.png)
设置【啤酒赠送方案】勾选允许倍增。

![](https://vip.kingdee.com/download/0109263d36675b934c24a877e10d3a9a5105.png)

【啤酒赠送方案】勾选允许倍增的开单效果。
