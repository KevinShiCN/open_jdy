解决方案：

排查思路---此类问题一般为套打模板中存在超出当前页面范围的布局表格等空间影响。

可打开套打模板，点击页面设置适当放大当前页面，寻找超出范围的控件，如下图，在模板右下方存在超出范围的布局表格，在右方导航树选中后，点击键盘 Delete键删除即可
