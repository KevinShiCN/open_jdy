# 年中启用账套财务初始余额的年初余额取数原理?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：大家好才是真的好 | 阅读：15 | 有用：0

【问题描述】

年中启用账套财务初始余额的年初余额取数原理?

【解释说明】

贷方科目:**科目期初余额** + **累计借方发生额** - **累计贷方发生额**

借方科目:**科目期初余额** + **累计贷方发生额** - **累计借方发生额**

【示例】

贷方科目:启用月科目期初余额 + 本年1月至启用期间上月的累计借方发生额-本年1月至启用期间上月的累计贷方发生额

借方科目年初余额:80=180+200-300

![](https://vip.kingdee.com/download/0109057c85feb216491292809864370b40f6.png)

借方科目:启用月科目期初余额 + 本年1月至启用期间上月的累计贷方发生额-本年1月至启用期间上月的累计借方发生额

借方科目年初余额:280=180+300-200
