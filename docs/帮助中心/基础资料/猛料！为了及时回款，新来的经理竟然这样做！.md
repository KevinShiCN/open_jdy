# 猛料！为了及时回款，新来的经理竟然这样做！

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：进阶技巧, 知识软文
> 作者：Simone邓 | 阅读：855 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/323068128525307904)

部门经理退休，就在小星以为凭借自己的能力和资历能当上部门经理的时候，总公司空降了一个经理。所以，小星心里很不服气。

      新领导刚上任的第一件事情就要求小星整理公司客户的应收账款，第二天九点上班就要查看。这天一大早小星就已经整理好数据，将文件放在了经理的办公桌上。

     小星心想，自己在公司做了这么久的销售主管，对于这些数据的整理都是小菜一碟，并且自己对ERP系统熟练，经理刚来，对公司业务以及系统都不熟悉，应该只是装装样子，新官上任三把火罢了。

      过了一会儿，经理将小星喊了过去温和的说道：小星，你知道一个公司想要长久的存活最重要的是什么吗？

     小星想都没有想，不以为意的说道：当然最重要的是团结啦！

      经理说道：团结很重要，但是最重要的公司的资金周转，资金周转就像人的血液一样，需要循环，才能存活，公司的应收账款我刚才看了一眼，情况非常槽糕，作为一位销售主管，你有责任和义务去监督每一个客户的应收账款回款情况，一旦超过就要考虑该客户是否还能继续销售，这样，既能减少坏账的发生，又能解决公司的资金周转困难，提高公司资金流通速度，也提高了公司资金使用效率，公司才能长久存活。那么如何做到呢？我们在使用的这个系统就有信用额度的功能，可以用来控制和监督应收账款回款情况，你看我操作给你看看：

首先【系统设置】--【系统参数】--【应收应付参数】根据公司的实际情况勾选参数

![](https://vip.kingdee.com/download/0109cad2a5ca79e74d7bbfb82fe59afbce16.png)

值得注意这里可以设置“销售信用额度控制方式”，有三种方式可以选择：“不控制”、“提示，允许继续保存”、“提示，不允许继续保存”，当选择"不控制”或“提示，允许继续保存”可以继续保存单据，根据公司实际需要进行设置。

【基础资料】---【业务资料】--【客户】进入进行对相关客户设置信用额度。

![image.png](https://vip.kingdee.com/download/0100935134dd4bb041d688fd445e70973ce4.png)

进入【销售管理】--【销售订单】新增该客户的销售订单，即可会进行提示“超过信用额度是否继续保存”，点击是即可继续保存

![image.png](https://vip.kingdee.com/download/01004392ce09b2604297b56e7cffcbab2b41.png)

这里值得注意的是信用额度系统检测的是这个客户的整体的应收账款，并非只是这张单据。

进入应收账款预警表，即可查看客户信用情况。

![image.png](https://vip.kingdee.com/download/01002899c887691e436591766cd30cf2bc6c.png)

小星会心的点点头，然后佩服的说道，经理果然厉害，懂的东西比我们多得多，和优秀的人一起共事是多么骄傲的事！！
