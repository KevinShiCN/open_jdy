# 怎么录入商品的毛重?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：云社区用户13pG7392 | 阅读：490 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/571500951719395072)

【业务背景】
- 对于食品/粮油/大米批发类客户,需要根据商品的重量来计算司机的提成
- 【操作流程】
- 1、单击【基础资料】→【商品资料】→【商品信息】-【其他信息】页签录入毛重即可。
- ![](https://vip.kingdee.com/download/01092bdee1c494fe41309cb89bffcd461812.png)
- 2、如果再【其他信息】页签没有显示毛重,请点击右上角小齿轮-筛选【毛重】字段,开启显示列;此时就可以在【其他信息】页签显示【毛重】字段。
- ![](https://vip.kingdee.com/download/0109b80045dbdf3c4e6791d8db333b5760ea.png)商品信息维护好毛重之后开单选择商品会自动带出毛重
- ![](https://vip.kingdee.com/download/0109dfbd12f8841b4926b1e5ae7a97b37c8d.png)
