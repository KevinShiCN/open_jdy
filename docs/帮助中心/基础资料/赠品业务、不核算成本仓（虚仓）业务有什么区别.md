# **1 问题描述**

赠品业务、不核算成本仓（虚仓）业务有什么区别

# **3 适用场景**

## 3.1 赠品业务

- 供应商的赠品，无入库成本，参与库存核算，影响库存成本
- 送给客户的赠品，无单价，但按照出库成本核算记录成本

## 3.2 不核算成本仓（虚仓业务）

- 针对受托加工客户提供的客供料，公司作为代管，仅需管理库存，无成本
- 生产过程中的实验品，不合格品等的管理
- 针对供应商赠送的赠品，不需要参与成本核算，不要在成本核算提示异常，不影响后续的库存成本
- 将供应商的赠品送给客户，这部分赠送客户的赠品有单价也可能0单价，但是0成本

# **4 特别说明**

标准版用户，如有只核算数量不核算成本的业务，请参考该帖处理：[怎么设置赠品不参与成本核算](https://vip.kingdee.com/link/s/Z2tlH)
