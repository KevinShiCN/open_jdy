# 客户催货？物流跟踪让货物不再逗留！

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：知识软文, 进阶技巧
> 作者：Simone邓 | 阅读：1987 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/323105622365528064)

一大早进公司门，就看到小林一脸凝重的盯着电脑。

       怎么了？一大早就这么凝重？王哥问道。

       做了单据，给客户发货，客户说货物还没到，年底了大家都来催，我也只能一个一个快递单号在物流官网上查，感觉查询的比较麻烦，头疼…

        这个你问我呀！王哥得意地说道。以前我专门给客户发货的时候，使用系统的这个功能来跟踪的，可方便了！

这个功能可以跟进某一笔业务的物流情况，比如:销售员或仓管员在销售发货出库后，需要了解物流状态及进展，异地调拨物流发货出库后，需要了解物流状态及进展这样子的，这个功能涉及单据有销售订单、销售出库单、调拨出库单、调拨入库单，在单据里面或单据列表上就可以进行物流追踪。

        王哥，您快操作我看一下，这个可把我头疼死了，小林说道。
        马上操作给你看，说着，王哥走到小林的电脑边上，演示起来：

首先，维护物流公司，在【基础资料】-【业务资料】-【物流公司】可直接使用系统预设置的物流信息，也可以根据企业合作的物流公司新增，如下操作：

![image.png](https://vip.kingdee.com/download/01006eba601b891e4b8ca5c6fcea611bfd97.png)

![image.png](https://vip.kingdee.com/download/0100ac4653e6eef646d1b910c7a53de58597.png)

这里以销售单为例：
 客户下订单，销售订单选交货方式，交货方式选择物流发货时，出库可填写物流信息

![image.png](https://vip.kingdee.com/download/01001bd7989a525846fda080d258d2a7d2c3.png)

![image.png](https://vip.kingdee.com/download/0100a7905d76e25648d6b67f378c2f017042.png)

填写对应物流信息，直接点击物流单号即可进入查看物流状态

![image.png](https://vip.kingdee.com/download/0100ecb1ba72298c49559015309f9df9c5ef.png)

![image.png](https://vip.kingdee.com/download/0100c7116b8bbf1a460f9077decc9d1aa2f0.png)

如要在单据列表查看物流信息，需要先将物流单号字段放出，如下图

![image.png](https://vip.kingdee.com/download/0100b0f29a0cb4994aeeb9c5d437d20871f0.png)

直接在销售单据列表，点击物流单号即可直接进入查看物流状态

![image.png](https://vip.kingdee.com/download/0100a56ac044fe794c3ab6c202d1c486a678.png)

调拨出库物流追踪流程同销售出库操作，【调拨出库】—【录入物流信息】如下图：

![image.png](https://vip.kingdee.com/download/0100e3c77874b350428a9a8c72cb71a58ab7.png)

![image.png](https://vip.kingdee.com/download/010043aab03a4fc84766ae0dc6d6f76d2d66.png)

单据列表放出【物流单号】字段，点击物流单号进入即可查看物流状态

![image.png](https://vip.kingdee.com/download/010033ba8e7cbb1649d0812e1ddf344fb116.png)

       需要注意的是目前暂不支持中通快递查询 ，其他的快递正常查询, 由于快递100接口服务器原因，目前暂时查询不了韵达快递以及百世快递，等待恢复即可查询。

       哇！王哥，你太棒了！这样不仅可以随时查询物流状态，还可以直接联系对应物流公司催货，让我们客户的货物不再逗留！
