# 仓库分类有什么用,如何新增?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：_Leo | 阅读：566 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/470032280095010560)

【问题描述】

仓库分类有什么用,如何新增?

【解释说明】

仓库分类是根据仓库的不同特性、用途、保管条件以及建筑结构等进行的分类,后续查询报表时,可以按仓库分类筛选查看。

仓库都是平级的,不支持直接新增下级明细仓库,若需对仓库进行分级,建议将仓库启用仓位功能,详细点击:[仓位如何新增使用?和仓库有什么区别?](https://vip.kingdee.com/link/s/ZFYdG)

【操作指引】

首先,在【基础资料】-【业务资料】-【仓库】界面,点击左上角的【编辑】按钮,进行新增。

![](https://vip.kingdee.com/download/0109c34a95c6848e4fdca6323357bf7b09e4.png)然后,点击“全部”后面的“+”号,输入类别编码、类别名称、选择上级分类(没有可不选),保存即可,如图:

![](https://vip.kingdee.com/download/0109533c169de622493eac3b55b980ecdc0e.png)
仓库分类旁边有三角符号表示还有下级分类,可以点击三角符号展开查询下级类别的仓库信息

![](https://vip.kingdee.com/download/01093c57d3380f3941f8b71b0f1bc6f7f13b.png)
后续查询库存报表数据时,支持按仓库分类进行筛选查看

![](https://vip.kingdee.com/download/0109e88562967b234310b4190be356f582c3.png)
