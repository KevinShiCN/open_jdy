# 可以设置部分仓库/仓位的库存无法使用吗?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：云社区用户6w624200 | 阅读：43 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/759586559615141888)

【问题描述】

可以设置部分仓库/仓位的库存无法使用吗?

【解释说明】

系统暂不支持直接针对仓库 / 仓位的特定库存设置 “禁止入库” 或 “禁止出库” ;若需限制对应库存的收发存操作,可通过**禁用仓库 / 仓位**的方式灵活处理:

仓库 / 仓位禁用后,新增单据时该仓库 / 仓位将不再显示于可选列表中,从而避免后续业务误选并限制其收发存操作;

禁用不影响历史数据:此前期间已使用该仓库 / 仓位的单据记录保持有效,对应库存数据也会正常显示在相关报表中,确保数据追溯完整性。

【操作步骤】

点击【基础资料】-【业务资料】-【仓库】或【仓位】![](https://vip.kingdee.com/download/0109267cb54aee324baf97724f907173f04d.png)

勾选不使用的仓库或仓位点击<禁用>![](https://vip.kingdee.com/download/0109660c5ab9cde84d21ad7ca885c6723b9e.png)

禁用的仓库/仓位做单据的时候不显示。比如下图中【CS仓123】已被禁用。

![](https://vip.kingdee.com/download/010915aef00cec994268874979683e3ac622.png)

但是商品里这个仓库的库存会显示。

![](https://vip.kingdee.com/download/01094cb25702f2d84222849dfb322f51128e.png)

仓库禁用后,之前的做单这个仓库有库存,在报表里会显示。![](https://vip.kingdee.com/download/01096f89e66d4ac0407f902b5e14db45487a.png)
