# 如何新增支出类别?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题
> 作者：杨桂平 | 阅读：477 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/129339770097650394)

【问题描述】

如何新增支出类别?

【解释说明】

支出类别支持单个新增,也支持批量新增。

【操作流程】

菜单栏点击【基础资料】>【支出类别】

![](https://vip.kingdee.com/download/0109038c3cc3d29c43a9a9e828cb5285257c.png)

点击右上角新增进入支出类别新增界面,录入相关信息保存即可。

![](https://vip.kingdee.com/download/01094aea9b9c94ea4bba9451e2df1cccc7b4.png)

可以点击右上角<批量新增>,将Excel表格中的支出名称批量复制过来

![](https://vip.kingdee.com/download/01092fbba187aefe4f1a8fe1560e0b8e5e79.png)

点击粘贴,系统会将数据批量粘贴过来,支出编码为空时,保存系统后会根据编码规则自动填充

![](https://vip.kingdee.com/download/01091033455b5ff84877a6a7fbc7f3786d64.png)

另外支持通过引入的方式,批量引入,详细可以点击: [如何引入收入/支出类别?](https://vip.kingdee.com/link/s/ZPmXW)
