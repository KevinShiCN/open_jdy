# 辅助资料里面的文章分类有什么作用?

【问题描述】

  辅助资料里面的文章分类有什么作用?

【解释说明】

  辅助资料里面的文章分类是用于订货商城模块的文章列表可以选择文章分类。![](https://vip.kingdee.com/download/0109a13d85507c8947f39138ad3bc9afdc19.png)![](https://vip.kingdee.com/download/010981ca9dd48a434374ba14322680e92227.png)商城首页方案设置中,设置链接时,支持设置跳转到对商品/商城的介绍或者推广文章。![](https://vip.kingdee.com/download/01098918dcedd4934801b6befe738337f35d.png)
![](https://vip.kingdee.com/download/0109b19e03937a54482293330ac5e3964921.png)
  辅助资料的文章分类可以作为单据上的辅助信息使用:可以通过单据自定义功能,将自定义的辅助资料加入到单据中,构造用户自己的单据。![](https://vip.kingdee.com/download/01097d88f1170d1e41c5bacba8c676abd883.png)
