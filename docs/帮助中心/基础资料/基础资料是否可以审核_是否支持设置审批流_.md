# 基础资料是否可以审核?是否支持设置审批流?

> 产品：金蝶云·星辰 | 分类：基础资料 | 类型：常见问题

【问题描述】

基础资料是否可以审核?是否支持设置审批流?

![](https://vip.kingdee.com/download/0109649ff45367ac45d3a1a4dd51291001c8.jpg)[视频观看](https://vip.kingdee.com/school/detail/727232989129588736?productLineId=35&lang=zh-CN&cosmic_type=video)

【业务场景】

生产制造等行业,商品信息需要由多个人员共同创建,且需要经过审核后商品才可使用;客户价格资料、供应商价格资料也需要审核后才能生效,因此需要对商品、价格资料增加审核功能。

【注意事项】

1、系统目前仅**支持商品信息、客户价格资料、供应商价格资料启用审核功能**;其他基础资料(客户/供应商/科目等)暂不支持;

2、系统暂**不支持**基础资料设置审批流程;

3、若商品禁用后,**不支持**操作审核或反审核,需启用后再操作。

【操作指引】
菜单栏点击【系统设置】>【系统参数】
![](https://vip.kingdee.com/download/01095270bdefdbdd43bebf828cb015792dd1.png)

在基础资料参数的其他参数中,可以选择勾选商品、价格资料启用审核的参数

**tips:两项参数仅管理员可操作,非管理员只可查看,不可修改参数**
![](https://vip.kingdee.com/download/010967f38528a95e4243b3e78dae6ac6348c.png)
启用参数后,点击保存时,会对历史的数据进行处理,建议避开业务繁忙的时段再开启参数

![](https://vip.kingdee.com/download/01096bc16cb6cdcf49e5ad15095e3b2ddeff.png)

参数启用后历史商品的“审核状态”将更新为“已审核”

![](https://vip.kingdee.com/download/010941bad42ff4d4435586bb6573b77698e6.png)

新增的商品,审核状态为“未审核”,需要审核后才能正常使用

![](https://vip.kingdee.com/download/010967379cbe65c54299adb08a806f53b17d.png)

已审核的商品,不允许编辑,如需设置审核后允许编辑,可在单据字段配置中设置商品信息审核后允许修改

路径:【系统设置】>【单据字段配置】,进入到单据字段配置界面

![](https://vip.kingdee.com/download/01093e05026e1ca94415be024f7a9c9b60ad.png)

点击【基础资料】>【商品信息】,可以设置“审核后允许修改”的字段信息

![](https://vip.kingdee.com/download/0109de122b1eb8294bd4a2d2062ae397c822.png)

客户/供应商价格资料同理,需要审核后生效,已审核的价格资料,不允许编辑、不允许覆盖引入。如需修改,则需要反审核价格资料

![](https://vip.kingdee.com/download/01095ff4a71cb6414d0eac6672f995e29e3f.png)
