# 用户管理实践案例——业务员离职了，来了新的业务员，如何变更历史单据上的业务员信息？

# 职员与业务员取值及更新逻辑说明

1、单据上的“业务员”以及用户管理中的“关联职员”，都是从基础资料>业务资料>职员列表中获取的

2、如果更新了职员姓名，那么历史单据上对应的业务员也是会更新

换句话说，一旦更新修改职员名称，所有单据（无论新增还和历史存在的单）上的该“业务员”都会取到最新的职员名字

# 应用场景及解决方案

## **场景一**：离职的业务员手机号是公司的，不是个人的，手机号所绑定的用户没有变，只是需要把业务员名字变了（如由李四改成张三），需要怎么处理？

解决方案：

打开基础资料>业务资料>职员列表，把职员名称从“李四”修改“张三”后点击保存即可。

![上传图片](https://vip.kingdee.com/download/0100ca1cb8db2cb84d8fa24545ba020766c0.png)

修改后，所有历史单据上业务员是“李四”的，都会变成“张三”。

![上传图片](https://vip.kingdee.com/download/0100f8a104bad98a45db93c5f2eadef8d07c.png)

## **场景二**：新入职的员工与离职员工手机号不同，需要把新的业务员“张三”添加到用户里，同时需要把离职业务员“李四”做的历史单据转给现在新来的业务员“张三”，最后对业务员“李四”禁用或删除。

解决方案：

1、新增用户，录入张三手机、姓名

![上传图片](https://vip.kingdee.com/download/01001d9d7226093249b9a2399fec26ad8060.png)

2、在用户管理找到“李四”，并取消关联职员关系（清空即可）

![上传图片](https://vip.kingdee.com/download/0100c750fa1538554a9095d5d77e93b56640.png)

3、在职员列表，把“李四”改成新职员“张三”即可

**注意，这一步更新后，历史单据上的对应的业务员“李四”也会更新为“张三”。**

![上传图片](https://vip.kingdee.com/download/0100bc464892364142de860cbf59d4263be1.png)

4、在用户管理里，把新邀请的用户“张三”（也就是新添加的业务员）的“关联职员”选择“张三”并点击保存

![上传图片](https://vip.kingdee.com/download/0100795ba5143597460e8a98127bdbf551db.png)

5、在用户管理里，禁用/删除离职员工“李四”即可完成
