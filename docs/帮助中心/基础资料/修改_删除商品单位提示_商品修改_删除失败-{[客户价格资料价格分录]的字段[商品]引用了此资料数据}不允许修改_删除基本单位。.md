# 修改/删除商品单位提示:商品修改/删除失败-{[客户价格资料价格分录]的字段[商品]引用了此资料数据}不允许修改/删除基本单位。

> 作者：杨桂平 | 阅读：399 | 有用：0

【问题描述】

修改/删除商品单位提示:商品修改/删除失败-{[客户价格资料价格分录]的字段[商品]引用了此资料数据}不允许修改/删除基本单位。

【解释说明】

下面以修改商品单位举例说明:

修改商品单位提示如下:

![](https://vip.kingdee.com/download/01092f5b41bc8cd84e6e84cca65c0f2a2d63.png)

按照提示检查客户价格资料是否引用该商品,删除引用的记录就可以修改/删除商品单位;由于客户价格资料方案较多,商品不能快速定位,建议通过右上角【价格查询】按钮进行搜索。

![](https://vip.kingdee.com/download/0109c0559ac9e4824913b3dee4e96906c467.png)

可以查询到引用的方案

![](https://vip.kingdee.com/download/010975c859252d074664a28485761a3e0c9d.png)

如果客户价格方案未引用,可以删除方案,就可以修改/删除商品基本单位;

如果客户价格方案较多,都引用过,开过单据,说明该商品多处都引用,需要一个一个删除引用记录,建议直接新增一个替代的商品。

[商品使用后单位还可以修改吗](https://vip.kingdee.com/article/566065295165742336?smartSearchResId=634064352961432320&productLineId=35&isKnowledge=2&lang=zh-CN)
