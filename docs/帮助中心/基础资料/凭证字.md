# **1 简介  **

## **1.1 应用场景**

凭证字是凭证录入时必须填写的基础信息，系统预设了记字凭证字，可以根据实际业务需要进行新增。

## **1.2 适用版本**

星辰所有版本

## **1.3 系统路径**

基础资料>财务资料>凭证字

# 2 操作指南

单击基础资料>财务资料>凭证字，即可进入凭证字管理界面，如下图所示：

![](https://vip.kingdee.com/download/01009269b9acf4a945f4a401e215df1a77b4.png)

## 2.1 新增

单击【新增】按钮，在弹出的【凭证字】窗口，录入凭证字，设置是否默认，再单击【保存】即可完成凭证字录入返回主界面。如果需要连续新增多个凭证字，在录入完上述信息后，单击【保存并新增】，系统会自动保存录入的凭证字，同时继续保持当前【凭证字】新增窗口。

![](https://vip.kingdee.com/download/0100a19e34134afe42498e5ee6b6d51848c3.png)

## 2.2 删除、修改

如果需要对已增加的凭证字进行删除、修改等操作，单击列表左边的复选框，操作工具栏对应的操作按钮即可
