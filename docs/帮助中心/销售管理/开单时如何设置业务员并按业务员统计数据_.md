# 开单时如何设置业务员并按业务员统计数据?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：ZXYzxc | 阅读：59 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/788935161349433344)

【问题描述】

开单时如何设置业务员并按业务员统计数据?

【解释说明】

①业务开单时,可直接在单据中录入业务员信息;若已在客户 / 供应商基础资料中维护对应业务员信息,开单选择客户或供应商后,系统将自动带出默认业务员,且支持手工修改。

②业务单据指定业务员后,后续查询相关报表时,可按业务员维度进行筛选与统计。

【操作步骤】

以销售出库单为例,在新增销售出库单后,在表头的业务员字段,点击键盘空格,根据显示的业务员内容,选择对应的业务员。

![](https://vip.kingdee.com/download/010934883de404a44039ace9c2dfc2d96c7a.png)或者点击业务员字段最后的"...",到【职员】界面进行选择对应的业务员。

![](https://vip.kingdee.com/download/01096bce49c5ca354e8490def85be0108242.png)
若在客户资料中维护了业务人员,开单选择客户后,会自动带出业务员信息,带出后再支持手工修改

![](https://vip.kingdee.com/download/0109d67cd9730a854910bc4fffd2b1704c06.png)

后续查询销售类报表时支持按业务员筛选统计查看

![](https://vip.kingdee.com/download/0109c5ed5cb58b434f76883c0736b17f736b.png)
