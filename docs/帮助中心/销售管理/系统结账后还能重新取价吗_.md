# 系统结账后还能重新取价吗?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：98 | 有用：0

【问题描述】

系统结账后还能重新取价吗?

【解释说明】

结账后**不可以**重新取价。已结账的单据操作重新取价系统会提示:不允许操作以前期间的单据,请先反结账。
