# 做销售出库单时表体的基本单位成本价格不显示

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：进销存赵嘉丽 | 阅读：1018 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/457348654886552064)

**【问题概述】**

 销售出库单上的单位成本、基本单位成本、出库成本为什么录入商品后没有值?

是如何计算的? 是否可以开单后直接算出成本?

![](https://vip.kingdee.com/download/01090a6a48a1586a45aabfe3f108e61b6f34.png)

**【解决方案】**

 1、出库类单据上的“出库成本”都是要通过【库存管理】-【出库成本核算】才能算出来的,不支持开单实时带出成本(实时计算会有性能问题);

![](https://vip.kingdee.com/download/0109a8fe29365fe241a3bc1d913cde20df51.png)2、出库成本的具体计算过程可以查看“出库成本核算结果查询”中的“成本计算报告”

![](https://vip.kingdee.com/download/0109b42e25407ff84ed9a6b656a81c6fdad7.png)

![](https://vip.kingdee.com/download/01096faeba1821194e41a9fad3f3a10a2f60.png)3、计算逻辑:

单位成本=出库成本/数量

基本单位成本=出库成本/基本数量

注,如果刚好“单位”=“基本单位”,那么单位成本=基本出库成本
