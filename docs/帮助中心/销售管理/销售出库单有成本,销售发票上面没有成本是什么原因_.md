# 销售出库单有成本,销售发票上面没有成本是什么原因?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题

【问题描述】
销售出库单有成本,销售发票上面没有成本是什么原因?![](https://vip.kingdee.com/download/0109d02b448a37eb475f9858c5a418e07523.png)
![](https://vip.kingdee.com/download/0109733aa9802774468483feaea33391dd0f.png)

【解释说明】
**适用版本:金蝶云星辰旗舰版**
出现该情况的原因为:销售发票与销售出库单尚未进行勾稽,因此销售发票上无法同步销售出库单的出库成本。需要进行销售单票勾稽,以给销售发票赋予对应的出库成本,使收入与成本匹配,以便根据销售发票核算毛利。

依次点击【销售管理】-【销售发票】查询,进入销售发票列表![](https://vip.kingdee.com/download/0109b066eb4123f1465794169e64014c88d7.png)

选择对应的发票,点击上方<自动勾稽>

![](https://vip.kingdee.com/download/01099683d3aac2a047d49fed23db42e8368c.png)
完成勾稽后,销售发票上将更新出库成本

![](https://vip.kingdee.com/download/010990680120702f4cc1a48225ef98b657fc.png)

【温馨提示】

销售退货单和销售退货发票规则一样,若销售退货单有成本,销售退货发票没有成本时需要进行单票勾稽,勾稽成功后销售退货单的成本会反写到销售退货发票中。

![](https://vip.kingdee.com/download/0109a86f016fb79c428da35395fe28534f92.png)
