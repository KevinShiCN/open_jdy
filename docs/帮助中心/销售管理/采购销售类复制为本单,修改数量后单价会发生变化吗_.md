# 采购销售类复制为本单,修改数量后单价会发生变化吗?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：大家好才是真的好 | 阅读：21 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/805242135707685376)

【问题描述】

采购销售类复制为本单,修改数量后单价会发生变化吗?

【解释说明】

采购、销售类单据复制为本单并修改数量后,系统将按照**价格取数优先级规则重新取价**:

①若价格取数优先级首位为「最近采购单价 / 最近销售单价」,则采购 / 销售单据复制后,单价优先取对应最近单价,修改数量后仍按此规则取价,**单价保持不变**;

②若价格取数优先级为首的是其他规则,单据复制并修改数量后,系统将**按优先级规则重新计算并取值**。

【操作指引】

如图:销售订单数量为1,单价100,含税单价113。

![](https://vip.kingdee.com/download/0109500d7ee25aa54d9aac8a84d37365c0a5.png)
销售类单据价格取数优先取客户等级价

![](https://vip.kingdee.com/download/0109319913d4bccb48ba96a87b28766cec19.png)
复制为本单时,将数量修改为6,系统会按价格取数优先级客户等级价取值,单价变为客户等级价80,含税单价为90.4元

![](https://vip.kingdee.com/download/010907c1e037f2034880be98ff5d0b9c2a72.png)
若将价格取数优先级中“最近销售单价”优先级调整为第一

![](https://vip.kingdee.com/download/0109c96d3fd440f64cd29444b0e8cf01c316.png)

则复制销售订单时,单价优先取对应最近单价,修改数量后仍按此规则取价,**单价保持不变**

![](https://vip.kingdee.com/download/0109dab4cf2368d1464f919b7e5d8bc7b392.png)

【温馨提示】

通过“复制本单”功能生成的新单据,若内容未经修改,其中的商品价格不会自动更新,仍保留原单据的价格。
