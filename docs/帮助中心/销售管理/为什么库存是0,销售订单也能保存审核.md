# 为什么库存是0,销售订单也能保存审核

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：ylim | 阅读：415 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/576936721250423552)

【问题描述】

为什么库存是0,销售订单也能保存审核

【解释说明】

销售订单本身只是预订的单据,不影响库存数量和往来金额,如果需要销售订单也影响库存数,需要设置参数。

【操作流程】

点击【系统设置】-【系统参数】

![](https://vip.kingdee.com/download/01095ed30c6e04824cb78d4585775b574d2c.png)
在【进销存参数】-【可用库存参数】下,勾选“已审核销售订单”这个选项,保存参数,这样库存不足时销售订单就无法审核。

![](https://vip.kingdee.com/download/01097ab13c6cdc864351a313c7adf834ecc5.png)
如图,西瓜商品在这个仓库的可用库存为0,审核销售订单时就提示了库存不足无法审核成功。

![](https://vip.kingdee.com/download/0109c524908d8ea04c27a35dc0233264b2dd.png)
【温馨提示】

如果所选仓库本身允许负库存,那么以上设置不生效,库存为0时一样可以审核单据
