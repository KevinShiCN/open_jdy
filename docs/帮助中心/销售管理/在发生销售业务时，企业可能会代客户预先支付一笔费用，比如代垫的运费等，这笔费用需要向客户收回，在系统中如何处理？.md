# 在发生销售业务时，企业可能会代客户预先支付一笔费用，比如代垫的运费等，这笔费用需要向客户收回，在系统中如何处理？

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：应用案例, 快速入门
> 作者：xiaoli1234 | 阅读：1582 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/111125379506842624)

问题描述：

解决方案：

**可以在销售出库单中填写“客户承担费用”**

**功能特性：**

1，  在录入销售出库单时，可同步记录帮客户垫付的费用

2，  垫付费用的币别可以与销售业务的币别不相同

3，  支持同时垫付多笔不同类型的费用

4，  不同企业的对账方式不同，可选择代垫的费用是计入其他应收款，还是合并计入本次销售业务的应收账款。

**应用流程：**

**1， ****在系统参数中，选择客户承担费用的计入方式**

**选择计入应收账款，客户承担费用的金额会和销售出库单的成交金额相加，成为本张销售出库单的应收账款**

**选择计入其他应收款，销售出库单审核后，客户承担费用会单独生成一张其他收入单。**

![](https://vip.kingdee.com/download/010053bbd8b806944e1b9f6d758f9486725b.png)



**2， ****录入销售出库单时，同步填写客户承担费用**

![](https://vip.kingdee.com/download/0100b82685af49634b339f138c23536dc39c.png)

如下图，选择客户承担费用计入应收账款后，本次应收账款为成交金额与客户承担费用之和

![](https://vip.kingdee.com/download/010038a784fbaffa44309dcbc4fd9706f83c.png)

如下图，选择客户承担费用计入其他应收款后，本次应收账款为成交金额，不包含客户承担费用。



![](https://vip.kingdee.com/download/0100b328b449d5c24294979ee49b1f9f944a.png)
