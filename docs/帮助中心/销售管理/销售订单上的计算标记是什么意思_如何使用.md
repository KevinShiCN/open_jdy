# 销售订单上的计算标记是什么意思?如何使用

【问题描述】

销售订单上的计算标记是什么意思?如何使用

【业务场景】

查看销售订单时,计算标记字段一直显示是空的,是不是销售订单还没有计算,如何让销售订单计算处理下

![](https://vip.kingdee.com/download/0109e79d9e2555d1454589d50485354dc82a.png)

【解释说明】

销售订单上的计算标记是记录该订单该商品是否已执行物料需求建议的标记。若为空说明没有做过物料需求建议,物料需求建议是生产功能,没有购买使用生产模块,可忽略该字段显示。
