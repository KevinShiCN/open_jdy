# 销售订单有多行商品,其中部分商品行已经执行完毕了,这种情况下无法对整个单据直接点击锁库,如果想对整个单据未出库部分进行锁库应该如何处理?

> 作者：_Leo | 阅读：345 | 有用：0

#### 【问题描述】

销售订单有多行商品,其中部分商品行已经执行完毕了,这种情况下无法对整个单据直接点击锁库,如果想对整个单据未出库部分进行锁库应该如何处理?

#### 【解决方案】

#### 关闭整单按钮,如图:

![](https://vip.kingdee.com/download/0109d904ef4e68b345419ef9f40562e9af2e.png)

#### 并在列设置中启用“同一单据拆分显示”,如图:

![](https://vip.kingdee.com/download/01096b33257ad097474cb24eabab41b2a62a.png)

#### 勾选未出库完毕的商品行进行锁库即可。如图:

![](https://vip.kingdee.com/download/01097d86d1ba3ef040b58157462dc8f91673.png)
