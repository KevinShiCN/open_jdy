# 销售订单会影响库存数量吗?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：lykk | 阅读：589 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/654125778266830080)

【问题描述】

销售订单会影响库存数量吗?

【业务说明】

销售订单可跨期往前或往后录入,未审核的销售订单不影响任何库存;

已审核的销售订单不影响即时库存,但可能影响可用库存、预计可用库存

1、当【系统设置】-【系统参数】-【进销存参数】中的可用库存参数、预计可用库存参数中未勾选“已审核销售订单”时,则不影响可用库存、预计可用库存

![](https://vip.kingdee.com/download/0109b6f8d4793be44041817474ca43837fd8.png)

![](https://vip.kingdee.com/download/0109dae4339a526149ea9a5bde9a6bbae9e8.png)

2、当可用库存参数,勾选“已审核的销售订单”时,销售订单审核后会减少可用库存;

![](https://vip.kingdee.com/download/01099c856d33da9e4fa08b4b3d6711146d00.png)

![](https://vip.kingdee.com/download/0109e6f172d477a548dabb93b3c3131f0a9c.png)

3、当预计可用库存参数,勾选“已审核的销售订单”时,销售订单审核后会减少预计可用库存;

![](https://vip.kingdee.com/download/0109e8ba5dd265d5401c84b0e987077a283b.png)

![](https://vip.kingdee.com/download/01095158051803b348ec9a7d47567447dce5.png)

备注:①针对批次、保质期、序列号、辅助属性商品订单不包含商品批次,保质期、序列号明细、辅助属性信息,可用库存不会扣减/增加此部分商品的订单数量,若需要查看请到预计可用库存表查询;

![](https://vip.kingdee.com/download/0109e899a5ab70e244a18468e22eaa55f3d8.png)

销售订单审核后,不影响可用库存;

![](https://vip.kingdee.com/download/010963c1de4d99f441a5ad8e2f3082953c09.png)

![](https://vip.kingdee.com/download/0109b8ed53b7c07844e0ad9dffc357498995.png)

预计可用库存会影响库存数量

![](https://vip.kingdee.com/download/0109f4cb6ca7df0945949b88fd3c73f49a8e.png)

4、订单商品行未选择仓库的情况下不会影响可用库存、预计可用计算

![](https://vip.kingdee.com/download/01094e0ac7265a80442b952f0a7f64a733a3.png)

5、销售订单预留后,可用库存、即时可用库存都会进行扣减;

![](https://vip.kingdee.com/download/010985104ad4f0294f1a990cd8d6088473d3.png)
【温馨提示】

若设置已审核销售订单影响库可用库存、预计可用库存,当未执行的销售订单反审核或删除时,将减少可用库存、预计可用库存
