# 发货通知单支持拣货验货

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：功能介绍, 新特性, 快速入门
> 作者：Freya_peng | 阅读：593 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/637012230725989888)

1.1 应用场景

业务人员先下订单，临近发货时间与客户确认后，提交发货申请，仓库主管依据发货通知单在按照仓库维度建立拣货任务派发，仓管员按任务拣货验货，一次或多次核对出入库的商品和商品数量及外观，减少出错货导致客户经营损失的情况。

1.2 功能特性

• 拣验货单的源单增加发货通知单

1.3 适用版本

星辰3.1旗舰版

注意：发货通知单仅旗舰版支持，拣验货单需要购买后才能使用；

1.4 功能详情

电脑端

1）在拣验货配置中新增发货通知单审核后自动下推拣验货单的参数选项：

![](https://vip.kingdee.com/download/0109a70bf2c6c7d1479db14d799702917d5c.png)

设置单据审核完成自动生成拣验货单，仓管员直接在待拣清单查看自己需拣任务，一键拣货

2）在发货通知单列表和详情页增加下推拣验货单的按钮；

![](https://vip.kingdee.com/download/0100f11d0007350849179784923f69df1438.png)

![](https://vip.kingdee.com/download/0100c557d52bc6c64035b2bbd51c3d3eac94.png)

• 点击去拣验：自动生成拣验货单，且将生成的拣验货单在工作台打开，支持选择多个源单一键拣货

• 点击生成拣验货单：按照选择的源单一单生成一个拣验货单，选择多张源单生成多个拣验货单

• 点击合并生成拣验货单：按照选择的源单合并生成一张拣验货单

3）在拣验货单选源单页面支持选择发货通知单，订单一旦下推了发货通知单后就不可下推拣验货单

![](https://vip.kingdee.com/download/0100dff6053ada3946a88380d832b5a42d4f.png)

![](https://vip.kingdee.com/download/01003ea031603ee949cba242b2f801d94693.png)

温馨提醒：为了更好的管控业务流程，可以在业务流程设置中，设置拣验货单必须从发货通知下推，不能直接创建。业务流程设置[功能详情请点击](https://vip.kingdee.com/link/s/lOQ9a normalLink \tdfe -10 \tdlt text \tdsub normalLink \tdkey lkz8z5)

![](https://vip.kingdee.com/download/0100272630ba23fb45b3983f62e1f8eabd4b.png)

温馨提醒：拣验货单选择源单后，会将源单仓库、仓位、批次、保质期等字段全部携带至拣验货单，拣货时可按照对应批次保质期拣货。

3）点击去拣验，跳转至拣验工作台扫码/手动搜索或输入数量拣验货。

![](https://vip.kingdee.com/download/0100c9a7f34af0ba4433a18906eaf6ff6d4c.png)

温馨提醒：拣验货单源单为发货通知单时，点击拣验完成不允许超额拣验，增加了强控制。

4）当商品拣完，点击拣验完成，跳转至拣验货单，支持一键出库

![](https://vip.kingdee.com/download/010094b6046dec0c42738df01c6e3fb191ca.png)

温馨提醒：点击销售出库，会将拣验货单本次拣验数量、仓库、仓位、批次、保质期等字段全部携带至销售出库单，按照拣货数量进行出库。
