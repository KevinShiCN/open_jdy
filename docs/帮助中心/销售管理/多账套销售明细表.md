# 1 应用场景

随着业务规模的扩大，小微企业逐渐发展成为有多个独立核算的实体公司如多个销售公司或者供产销分离的多个公司。多个销售公司可能经营相同的业务，也可能按地区、品类、品类等划分经营，公司之间存在调拨等销售协同；产供销分离的多个公司由上游公司统一生产或采购后向下游公司供货，下游公司或者上下游多公司分别对外销售。

为了掌握企业整体经营情况，管理者需要统一分析各公司的销售情况，但是各公司的业务数据分散在多个账套，给业务的全局洞察增加不便。**此次增加“多账套销售明细表”，支持在同一个服务ID下购买多个账套后，在一个账套汇总查看多个账套的销售明细并剔除账套间的内部交易。**

# 2 适用版本

星辰3.0以上专业版/旗舰版

## 3.1 多账套销售明细表的启用与设置

| 步骤 | 角色 | 步骤说明及图片示例 | 预期结果及图片示例 |  |  |
| --- | --- | --- | --- | --- | --- |
| 1 | 服务管理员/账套管理员 | 启用合并报表 |  | 菜单增加组织机构设置 |  |
| 2 | 服务管理员 | 设置账套为母公司 |  | 母公司的账套增加多账套销售明细表 |  |
| 3 | 服务管理员/账套管理员 | 设置账套参与销售汇总 |  | 多账套销售报表取到相关账套的销售数据 |  |

### 3.1.2 详细步骤

#### 步骤1：启用合并报表

一个服务下的多个账套共用一份组织机构设置，账套管理员与服务管理员的《组织机构设置》列表的可见性与可操作事项不同

1）服务管理员查看《组织机构设置》，可以看到购买的该服务下所有账套，对任意账套进行后续操作

![上传图片](https://vip.kingdee.com/download/0100eb976589de714d0cb258d0d7c1549a07.png)

2）账套管理员查看《组织机构设置》，仅可查看到本账套以及对本账套进行设置

![](https://vip.kingdee.com/download/01005ceebc09cc3d48b6b18ba913753fe747.png)

#### 步骤2：设置母公司

1、概述：集团购买多个账套，其中有一个账套为母公司在使用，希望在该母公司账套直接查看到多个账套的销售明细

2、权限：**仅服务管理员可以设置一个账套为母公司**，通过账套列表的“是否母公司”可判断设置情况

- 服务管理员设置母公司

![上传图片](https://vip.kingdee.com/download/010090b6006e21ab46ae8620d2126fc206d5.png)

- 账套管理员设置母公司提示“不允许操作”

![](https://vip.kingdee.com/download/01001f1a385e3b014b83b9a4597fac55fde3.png)

3、变更：已存在母公司，若切换其他账套为母公司，财务合并报表的相关设置将会删除。

**财务合并报表： **[https://vip.kingdee.com/link/s/Z2hnI](https://vip.kingdee.com/link/s/Z2hnI)

![](https://vip.kingdee.com/download/01004a4aecfc34534e00a0c0fcdcfa97a73a.png)

4、影响：

1）设置母公司后，**仅母公司对应的账套**增加《多账套销售明细表》，其他账套不可见

2）设置母公司后，**服务管理员与账套管理员**才可以进行组织机构设置的后续操作

![](https://vip.kingdee.com/download/01006a86267c3bca401a94425c901a336885.png)

#### 步骤3：设置账套参与销售汇总

1、概述：集团购买多个账套，一些账套为销售公司使用，发生真正的对外交易，整体经营洞察时需要汇总这些账套的销售数据；相反，部分账套对应为完全负责统一对外采购后仅向内部其他公司销售，或者负责生产后向内部其他公司供货，则无需汇总分析这些账套的销售数据

2、权限：①服务管理员可以设置所有账套参与汇总，②账套管理员仅可以设置本账套参与汇总

3、影响：控制《多账套销售明细表》汇总哪些账套的销售类单据，具体指

①专业版：销售出库单、销售退货单、委托代销结算、委托结算退货

②旗舰版：销售出库单期初、销售出库单、销售退货单、委托代销单、委托代销退货

4、操作步骤：

- 数据合并设置-参与销售汇总：点击账套名称，在组织机构信息弹窗中勾选“参与销售汇总”参数

![](https://vip.kingdee.com/download/010092ad744054854dea82311d02a0cc3d10.png)

- 勾选后，组织机构列表可见“参与销售汇总”的开启情况：

![](https://vip.kingdee.com/download/0100885f67b53cc443649915e65360ff46e6.png)

#### 步骤4：设置账套的内部客户

1、概述：集团下的多个公司之间存在业务如互相调货、生产型公司向销售型公司供货，供应方在账套做销售出库单，需求方在账套做采购入库单。如果这一部分内部往来于集团整体不产生销售收入，希望汇总查看多个账套的销售明细时剔除内部往来，就需要设置账套的内部客户

2、权限：①服务管理员可以设置所有账套的内部客户，②账套管理员仅可以设置本账套的内部客户

3、影响：设置内部客户后，多账套销售明细表“包含内部交易”参数默认未勾选时，不显示各账套与内部客户的销售明细

![上传图片](https://vip.kingdee.com/download/010019f07910eed54ad8a2fc7f9edfc6cb93.png)

4、设置：

1）前提：账套“母公司”添加客户“内部客户A”

![上传图片](https://vip.kingdee.com/download/010042604b4fd01f4bc8a11f829c9cfba20c.png)

2）场景：账套“母公司”的客户 “内部客户A”等同于账套 “子公司”

即现实中发生一笔交易，增加账套“母公司”的收入，同时增加账套“子公司”的支出。

3）操作步骤：《组织机构设置》点击账套“母公司”，设置该账套的内部客户

• 设置某账套的内部客户，内部客商设置的账套列表展示的是除该账套之外的其他账套；

• 设置某账套的内部客户，内部客商设置的内部客户列表来源于该账套的客户列表

![](https://vip.kingdee.com/download/01001bd11e3ddcb04fec94d86246c7e86e0c.png)

4）说明：

①使用多账套销售明细表，无需设置内部供应商。财务合并报表涉及使用“内部供应商”；

②内部客商的唯一性控制：一个账套只能对应为本账套的一个客户。

### 3.1.3 注意事项

1）母公司设置后，不能取消，如没有做过任何财务合并相关的操作，可直接切换。

2）如已存在报表模板、合并报表初始数据、科目映射、个别报表、抵销分录、合并工作底稿，但还未生成合并报表，则不允许修改母公司和修改财务上级组织、股权比例、是否参与财务合并。

3）如已生成合并报表，可修改组织机构设置。但修改后，已使用过的报表模板、已生成的合并报表初始数据/科目映射/个别报表/抵销分录/合并工作底稿/合并报表将会打上“历史”标记，无法再修改、删除。

4）切换母公司或母公司账套重新初始化后，组织机构的所有设置(包括其他账套的设置)也将会被全部清空。

5）母公司账套不支持删除，如需删除，需要先切换母公司。

6）母公司账套重新初始化后，所有子公司的组织机构的设置也将会全部被清空。

## 3.2 多账套销售明细表的使用

### 3.2.1 报表位置

- 设为母公司的账套，销售管理>销售报表>多账套销售明细表

![上传图片](https://vip.kingdee.com/download/0100a107b0e623ee422b87c2241c3e4e2dc3.png)

### 3.2.2 报表权限

- 报表支持功能授权、字段授权，**无数据授权**

**无数据授权是指：**

①母公司账套的用户非子公司的用户，在多账套销售明细中仍可查看到子公司账套的销售明细；

②母公司账套的用户也是子公司的用户，在子公司账套的销售明细表中不可查看的数据在多账套销售明细表中可查看；

③母公司账套的用户在销售明细表不可查看的销售明细，在多账套销售明细表中可查看。

### 3.2.3 报表功能

- 报表支持查看、引出、打印、报表列设置

1、查看明细时，默认剔除内部交易（见第一部分 设置账套的内部客户说明），勾选“包含内部交易”参数后，可查看

• 未勾选“包含内部交易”

![上传图片](https://vip.kingdee.com/download/01000bd61b45fb8943bf85f69f3326f88555.png)

• 勾选“包含内部交易”

![上传图片](https://vip.kingdee.com/download/0100468f9420bb8140ae9d29f531e831a782.png)

2、报表排序

报表默认的排序规则是按所属账套（升序）＋单据日期（升序）＋单据编号（升序），支持报表列设置对多个字段设置排序依据，且被存储在列设置中。

![上传图片](https://vip.kingdee.com/download/0100ef0be80f4a4241faa1931b3692438699.png)

3、 报表支持增加公式字段

![上传图片](https://vip.kingdee.com/download/01004d0c7cf93df34733986897394d05a63e.png)
