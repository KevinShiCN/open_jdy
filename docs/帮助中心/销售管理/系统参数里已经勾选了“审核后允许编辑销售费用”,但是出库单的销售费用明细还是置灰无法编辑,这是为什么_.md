# 系统参数里已经勾选了“审核后允许编辑销售费用”,但是出库单的销售费用明细还是置灰无法编辑,这是为什么?

> 作者：进销存熊诗琴 | 阅读：14 | 有用：0

【问题描述】

系统参数里已经勾选了“审核后允许编辑销售费用”,但是出库单的销售费用明细还是置灰无法编辑,这是为什么?

【适用版本】

星辰3.1以上所有版本

【原因解析】

1、全流程跟踪检查销售出库单是否生成了凭证;旗舰版需要检查销售出库单是否生成发票/单票勾稽;

若已经生成凭证或者有关联下游单据,则需要删除凭证或者删除下游单据后再进行修改。

![](https://vip.kingdee.com/download/0109da8fb465504945a4a8673a12385f2339.png)

2、检查进销存的启用期间和当前期间,已结账期间的销售出库单无法再修改/新增销售费用,若需要修改/新增销售费用,则需要反结账到对应的期间进行操作

若需要了解如何操作反结账,请参考:**[业务系统期末结账与反结账的流程?](https://vip.kingdee.com/knowledge/129339770097650234?productLineId=35&isKnowledge=2&lang=zh-CN)**
