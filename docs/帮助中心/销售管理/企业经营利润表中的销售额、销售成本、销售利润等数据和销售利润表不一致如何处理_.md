# 企业经营利润表中的销售额、销售成本、销售利润等数据和销售利润表不一致如何处理?

> 作者：嘿唬哈 | 阅读：186 | 有用：0

【问题描述】

企业经营利润表中的销售额、销售成本、销售利润等数据和销售利润表不一致如何处理?

【概述】

企业经营利润表和销售利润表取值逻辑不同

企业经营利润表销售额:销售汇总表中的“折前金额”合计

销售利润表销售额:销售单价税合计或实际不含税金额(根据企业内部要求,一般指实际不含税金额)

企业经营利润表利润:业务经营利润+其他收支及费用

销售利润表毛利:销售收入 - 销售成本 -抹零金额 -销售总费用
