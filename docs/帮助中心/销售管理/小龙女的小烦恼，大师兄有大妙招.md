# 小龙女的小烦恼，大师兄有大妙招

      近日，每到傍晚时分，东海上空就会飘起一朵小乌云，小雨淅沥淅沥的下着，甚是怪异。大师兄好几日都没有看到紫霞仙子的真容了，于是就想着去东海一探究竟。

      刚到东海上空，就看到一名绝美女子坐在一只蓝鲸上掩面而泣，鱼虾在其身侧穿行，也有飞鸟在背上停。大师兄飘然落下，原来是有“东海泉眼”之称的小龙女在独自神伤。见到大师兄，小龙女哭的更厉害了，顿时狂风大作，波涛汹涌，雷雨交加。大师兄赶忙劝到：“别哭别哭，有啥事跟俺老孙说，我最见不得女娃子在我面前哭了”小龙女抽了抽鼻子，娓娓道来······

      前文说到，东海龙王靠水吃水，开了一家东海水产贸易有限公司，由于东海水质好，蟹肥虾甜，各种海产品应有尽有，加之老龙王经营得当，生意遍布大江南北，在全国各个城市都设立了转运仓库，只要客户下单，就会在最近的仓库中发货，快速到达客户手中。

      小龙女甚是孝顺，为了给父亲分忧，主动提出给父亲帮忙在云星辰软件中录入原始单据。奈何父亲“家大业大”，单据属实是忒多了，还有同一个客户很多相同的商品需要在不同的仓库出库，录入到单据中，这可把小龙女累坏了。

      大师兄笑道：“不就是复制单据内相同的多行分录并粘贴在单据内嘛，俺老孙有妙招，你且记好了”

      以销售订单为例（其他销售类单据、采购类单据也是同样的操作方法）：假设职员A录制了一张销售订单001，由于该客户在深圳和上海均有仓库，故做单给同一个客户时会从不同的仓库出库，但是商品相同，职员A录完了深圳仓的4条分录后，为节约录单时间，希望直接复制粘贴4条相同的分录，再修改仓库为上海仓即可。

![](https://vip.kingdee.com/download/01002dfa2beb4aa54e6dbcb60e7619184d9f.png)

      具体操作：

      1、 在商品编码列按住鼠标下拉选中要复制的商品编码

      2、 新增分录

      3、 在【商品编码】列的新增分录行单元格内输入“商品编码”四个字，敲空格（一定要输入空格！）

      4、 粘贴刚刚复制的商品编码，复制的内容即可自动带出，按回车键完成。

      说明：上面是步骤拆解，下方的动图则是对应的操作演示。

![](https://vip.kingdee.com/download/01001731232efd034adf83afae28a6da5b9e.GIF)

      值得注意的是：

      1、单据内直接复制粘贴只适用于编码类型的信息（条形码、SKU编码、商品编码、客户商品编码等），不支持【商品名称】的复制粘贴；

      2、粘贴的单元格一定是在【商品编码】那一列中输入；

      3、粘贴前一定要输入所复制的信息的列名，这样系统才能识别出所属的字段列。

      例：复制商品编码的话，粘贴时要先输入“商品编码”四个字，并敲空格（目的是与编码间隔开，方便系统识别读取，不能省略！），复制条形码的话，粘贴时要先输入“条形码”四个字，并敲空格。

      4、批次、序列号商品也支持单据内的复制粘贴；

      5、辅助属性如果要复制粘贴，务必确保辅助属性有启用条形码，且条形码唯一，如无启用条形码，则多组辅助属性的商品编码都一样，系统则无法识别粘贴；

      6、复制粘贴必须至少大于4行分录才能成功。
