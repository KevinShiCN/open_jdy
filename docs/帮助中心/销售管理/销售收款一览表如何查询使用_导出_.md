# 销售收款一览表如何查询使用/导出?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：进销存赵嘉丽 | 阅读：447 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/633107509174929920)

【问题描述】

销售收款一览表如何查询使用/导出?

【适用版本】

星辰3.1以上所有版本

【使用说明】

销售收款一览表的展示信息和查询与【**系统设置**】**>【系统参数****】>【应收应付****参数****】中的款项核销方式**有关。

![](https://vip.kingdee.com/download/010904723ddac7734664bee9b31aaedde32b.png)

![](https://vip.kingdee.com/download/01093f4d7d865dd447f882c128d360ed6f0a.png)
下面就以专业版为例(其他版本也是相同的逻辑),详细说明三种不同参数下对应的销售收款一览表展示方式的差异:

| 系统参数>应收应付>基础参数 | 参数应用场景 | 不同参数下销售收款一览表页面展示效果(图片点击可放大) | 差异说明 | 收款相关的字段(图片点击可放大) |
| --- | --- | --- | --- | --- |
| 按单据核销 | 收/付款项时,需要知道收/付的是哪笔交易的钱,中等明细程度的记录 |  | 按单据核销时,筛选条件中会显示“收款日期”,同时筛选项中会显示“显示商品明细”、“显示收款明细”、“已收款金额为0不显示”三个参数。 | 已收款/未收款/回款率% 等等 |
| 按分录核销 | 收/付款项时,需要知道收/付的是哪笔交易哪笔分录的钱,最明细的记录 |  | 按分录核销时,筛选条件中不显示“收款日期”,同时筛选项中会隐藏“显示商品明细”、“显示收款明细”、“已收款金额为0不显示”三个参数。原因:“收款日期”取的是单据头的字段,“显示商品明细”、“显示收款明细”、“已收款金额为0不显示”也是基于整单(也就单据头维度)来展示的。而按分录核销时,显示的数据都是单据体的,二者不是一个维度的,因此相关的参数都会自动隐藏。 | 商品已收款/商品未收款/商品回款率%简单来讲,按分录核销时、已收款/未收款/回款率% 前面都会增加“商品”二字,表示是对应到商品分录的收款情况。 |
| 按余额核销 | 收/付款项时,系统自动根据录入的收/付款金额和设置的核销单据顺序进行核销,核销单、退款单不支持 |  | 按余额核销时,筛选条件中会显示“收款日期”,同时筛选项中会显示“显示商品明细”、“显示收款明细”、“已收款金额为0不显示”三个参数。跟“按单据核销”的效果是一样的。 | 已收款/未收款/回款率% 等等 |

根据需要设置查询条件,如单据日期、收款日期、审核状态、计算毛利等查询

![](https://vip.kingdee.com/download/0109362f44bd2c7542c6982c53963c5b6f49.png)
查询后可点击右上角<引出>按钮导出查询结果
