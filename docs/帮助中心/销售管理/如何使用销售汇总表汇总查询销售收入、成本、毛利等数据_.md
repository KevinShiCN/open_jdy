# 如何使用销售汇总表汇总查询销售收入、成本、毛利等数据?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：1066 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352609899882752)

【问题描述】

如何使用销售汇总表汇总查询销售收入、成本、毛利等数据?

【业务场景】

需要汇总查询销售的收入、成本、毛利、数量,按客户、按商品、按业务员等不同维度汇总查询

建议先进行[出库成本核算](https://vip.kingdee.com/link/s/Z7HKK),当单据无成本时,报表中不显示成本或成本不准确

【操作流程】

在【销售管理】-【销售报表】-【销售汇总表】界面

![](https://vip.kingdee.com/download/0109d62b4f511bbd4d49b309a383724a31e8.png)

**a,统计范围:**

销售汇总表中的商品及数量、金额数据来源于以下单据的商品行,“单据类型”默认为全部,不同版本的单据详见下表(本报表不统计未结存出库的零售单数据!):

| 版本 | 单据类型 |
| --- | --- |
| 旗舰版 | 销售出库期初销售出库单销售退货单委托代销单委托代销退货单零售出库(购买了零售云才有)零售退货(购买了零售云才有) |
| 专业版 | 销售出库单销售退货单委托代销单委托代销退货单零售出库(购买了零售云才有)零售退货(购买了零售云才有) |
| 标准版 | 销售出库单销售退货单零售出库(购买了零售云才有)零售退货(购买了零售云才有) |

**b,支持按不同汇总依据查询(**[销售汇总表的汇总依据可以增加/修改/删除吗?](https://vip.kingdee.com/link/s/ZPO32)**)**

按不同的汇总依据查询时,报表有不同的展现形式,当选择汇总依据为客户,商品类别,仓库时等除商品以外的汇总依据,还可以通过勾选“显示商品明细”来实现多维度的汇总统计,如果勾选了“显示商品明细”还可以勾选“显示辅助属性”来展示统计。

当统计到商品维度时,若系统启用了辅助属性管理,可以通过勾选“显示辅助属性”来统计到商品的属性维度。

![](https://vip.kingdee.com/download/01090df0219ea0ab4606be7784518a282500.png)

当单据类型为空时,默认为全部单据类型,相同字段的数据由销售数据汇总减去退货数据汇总得出;

**c,查询条件及界面:**

①日期范围可以任意调整;默认为本月的第一天到当天;单据类型默认为全部,可选择仅查询销售出库单或销售退货单;点击<更多>可显示出更多过滤条件

![](https://vip.kingdee.com/download/0109e6154701e4714b729ad86d6dacda57d0.png)

②客户、商品、仓库等过滤条件如果空表示查找所有客户、商品、仓库等,可分别打开各选择框进行勾选指定具体客户、商品、业务员、部门、仓库等;

审核状态、币别、是否赠品等均可指定条件

![](https://vip.kingdee.com/download/0109dc627bcf19054b2e8d68670b8f881a22.png)

 ③退货原因仅用于查询销售退货单时有效。

![](https://vip.kingdee.com/download/01091747ff07de174e30b4921d5235570038.png)

④当需要**查看销售**相关的**收入、成本、毛利、毛利率**时,可以勾选“计算毛利”后查询,可选择是否将“销售总费用”从毛利中扣减;(收入为**实际不含税金额**、不勾选计算毛利也可查看)

![](https://vip.kingdee.com/download/0109d90b8bc7993c43bcbc5a91a8de4e76fe.png)

点击<筛选设置>可对表头过滤条件进行显示隐藏

![](https://vip.kingdee.com/download/01092c9e5374921445f7af9b838eba5a5211.png)
⑤若有些字段未显示出来或需要隐藏不查看,可点击报表右上角<列设置>进行字段显示和隐藏的调整(列设置中“增加公式”的应用请参考:[进销存部分采购/销售类报表支持添加公式字段](https://vip.kingdee.com/link/s/ZPRYH))

![](https://vip.kingdee.com/download/01095480e0e5716d4125a4ae2215711a6193.png)
列设置的“选项”中可以控制数值列的尾零是否显示(开启即为显示,关闭即为不显示)

![](https://vip.kingdee.com/download/010934d222e6a3194251a8868d9bcdc290c0.png)
