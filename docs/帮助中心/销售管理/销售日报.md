# 销售日报

> 作者：进销存赵嘉丽 | 阅读：1020 | 有用：6

# **1 简介  **

### **1.1 功能介绍**

通过销售日报，可以查询指定日期或者月份的销售概况，例如：销售订单总数、金额、销售出库单总数、金额、销售退货单总数、金额，还可以查看销售毛利及毛利率。

## 1.2 **应用特性**

1、支持按指定日期查询每一日的销售概况明细，也支持按年份查询每个月的销售概况明细；

2、可自定义是否要将未审核的单据统计进去

## **1.3 适用版本**

星辰3.1以上所有版本

## **1.4 系统路径**

销售管理>经营分析>销售日报

# **2 应用流程**

1、打开销售管理>经营分析>销售日报，进入日报界面：

2、可在“筛选”中设置查询的期间和数据的明细程度（按日或者按月），以及是否要将未审核的单据统计进去，选择好后单据“查询”即可。
