# 采购/销售等库存单据中的同单费用优化

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：功能介绍, 新特性, 快速入门
> 作者：进销存熊诗琴 | 阅读：1143 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/667788074822427648)

## **1 简介**

### **1.1 功能介绍**

在采购销售业务中产生的费用会同步登记在单据中，根据不同企业对费用管理过程中提出的需求，进行了一些优化。

### **1.2 适用版本**

星辰3.1以上所有版本

## **2 功能特性**

### 2.1 在同单费用录入时计算规则优化

当税率发生变化时，可选择应付金额不变重算不含税金额，还是不含税金额不变重算应付金额。

![](https://vip.kingdee.com/download/0109569f10efd72b425886ddb380d086f7d8.png)

### 2.2 增加同单费用分摊规则

| 增加的分摊规则 | 涉及单据 |
| --- | --- |
| 总净重，总毛重，总体积 | 采购入库，销售出库，销售退货，委托代销结算，异价调拨出库，异价调拨入库 |
| 数量，辅助数量，基本数量 | 采购入库，销售出库，销售退货，委托代销结算，异价调拨出库，异价调拨入库，组装单，拆卸单 |
| 调出金额 | 异价调拨出库 |
| 调入金额 | 异价调拨入库 |

### 2.3 系统参数增加费用相关参数

![](https://vip.kingdee.com/download/010910444b235feb413a9363a875a7131eef.png)

1. 费用可设置强制分摊：系统参数中，可对不同的单据选择是否需要强制分摊同单费用，开启后，本张单据在录入同单费用时，则隐藏“保存”按钮，仅显示“保存并分摊”按钮。

支持强制分摊同单费用的单据：采购入库单、销售出库单、销售退货单、委托代销结算单（代销费用）、委托代销结算单（销售费用）、组装单、拆卸单。

![](https://vip.kingdee.com/download/010900d7553815b94489a987057a732c3a1f.png)

2. 系统参数中，可对不同的单据选择是否需要同步删除费用生成的单据。

支持同步删除同单费用生成的单据：采购入库单、采购退货单、销售出库单、销售退货单、委托代销结算单、销售发票。

![](https://vip.kingdee.com/download/0109780a77cda7054c70907978d40d148b5b.png)

**举例说明：**

销售出库单中的销售费用，生成了其他支出单；

（1）若这笔费用已分摊到销售出库商品行，则在其他支出单列表不允许将该单删除；

（2）若没有开启销售出库单和费用单据同步删除，可先删除销售出库单，再删除其他支出单，或者在销售出库单费用弹窗中，打开这张其他支出单进行删除，并手工删除对应的费用行；

![上传图片](https://vip.kingdee.com/download/0100418f1b56bc0f40e3a4f32f7905ae879c.png)

（3）若开启了销售出库和费用单据同步删除，在删除销售出库单时，会同步删除关联的其他支出单；若其他支出单已审核无法删除，则本张销售出库单也不能删除。

![](https://vip.kingdee.com/download/01096f0fb2b78a6645db97b85b1940f34e10.png)

![](https://vip.kingdee.com/download/010962512c3e17d843f1a42ac58730eb1ed3.png)

3. 系统参数中，针对“调拨出库，调拨入库，组装单，拆卸单”，可选择在录入同单费用时是否默认待支出。

### 2.4 订单下推时费用处理优化

1. 系统参数中，可选择采购/销售订单下推采购入库/销售出库时，是否携带费用：

![上传图片](https://vip.kingdee.com/download/0100d6e6ccbd50614bfeaa52230ad61891fa.png)

2. 采购/销售订单中录入了预计采购/销售费用，当同一客户/供应商的多张订单合并生成一张采购入库/销售出库时，可将订单中的预计费用也逐行携带到下游单据中；

注意：仅当订单整单执行状态为“未执行”时才会携带预计费用；若订单是部分执行状态，不会携带费用到下游单据；

![上传图片](https://vip.kingdee.com/download/010063df1990d2774b3bb186383583e57d19.png)

3. 在订单合并下推且携带费用时，若开启了参数“强制分摊同单费用”，在执行下推的结果中已执行了费用分摊，不需要再手工点击分摊按钮；

4. 在采购入库/销售出库费用弹窗中，若勾选了“按源单分摊”，在费用分摊时，会将来源于同一张订单的费用，仅分摊到该订单对应的商品行中，不会被其他订单的商品行占用；

注意：此时若手工补录了新的费用行，就会被分摊到手工新增的商品行中，若需要新增费用也参与到所有商品行的分摊，请不要勾选“按源单分摊”；

### 2.6 采购退货单中支持录入退货费用

采购退货单中，可支持录入退货过程中产生的费用，录入方式与其他同单费用相同，退货单审核后也会生成其他支出单；但是，采购退货费用不支持分摊
