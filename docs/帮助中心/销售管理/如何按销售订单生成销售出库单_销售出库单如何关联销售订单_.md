# 如何按销售订单生成销售出库单?销售出库单如何关联销售订单?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：3934 | 有用：13
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391351743222458624)

【问题描述】

如何按销售订单生成销售出库单?销售出库单如何关联销售订单?

【解释说明】

a)销售订单出库可以是单笔订单出库也可以是多笔订单合并出库

b)已手工录入保存审核的销售出库单,需[删除单据](https://vip.kingdee.com/link/s/ZSHLb)后重新根据销售订单下推;或反审核出库单点击选源单进行关联

c)通过销售订单下推销售出库单后,有手工在销售出库单增加行,增加新的商品出库的,手工增加的商品行不支持关联销售订单作为源单,由订单下推的商品行才会源单单号显示

d)通过订单下推或销售出库单上选源单关联步骤如下

【操作流程】

**一、单笔销售订单生成销售出库单**

1、点击【销售管理】-【销售订单】-【查询】打开销售订单列表

![](https://vip.kingdee.com/download/010908ef997161c34c20a2817c5f883e6c5f.png)

2、在销售订单列表中,勾选需要出库的销售订单(必须是已审核状态),点击右上角【销售出库】直接生成销售出库单,系统会将未出库的销售订单数量下推到销售出库单,在销售出库单界面可按实际出库修改出库数量;

ps:直接新增销售出库单时,可在**进销存参数**中设置单据默认携带数量为 0 或 1;由**销售订单下推生成销售出库单时,单据将自动携带订单的未出库数量,该数量取值规则不支持修改。**

![](https://vip.kingdee.com/download/01094adb2076d44445ba8e9f6b49c156b424.png)

3、或是打开单据编辑界面,点击右上角【销售出库】即可生成销售出库单

![](https://vip.kingdee.com/download/01090470276995164c84b66cbec7b9cf5e84.png)

**二、多笔销售订单合并生成销售出库单**

1、在销售订单列表中,同时勾选多笔销售订单点击【销售合并出库】

![](https://vip.kingdee.com/download/0109489a256039e44558adb98ce007fdf94f.png)

**三、按销售订单的商品分录生成销售出库单**

在销售订单列表,把订单按商品分录拆分

![](https://vip.kingdee.com/download/010986b3ac84614f434bbd88d231ec727bf9.png)

再分别勾选商品单独出库或是合并出库

![](https://vip.kingdee.com/download/0109bb671f5c0ed0492b8aa893c7cca527be.png)

**四、未审核的销售出库单选源单管理**

1、直接新增销售出库单,选择客户,然后点击右上角【选源单】,可以选择该客户名下的一笔或是多笔销售订单出库。

![](https://vip.kingdee.com/download/0109b6e59e0196014441a0b1055b68008bfb.png)

![](https://vip.kingdee.com/download/010952cffd1bb8414e9f81d47b7ac52ff151.png)

注意:

1、当订单全部出库后,订单状态自动变为“已关闭”;

2、订单生成出库后不会修改源订单数量的,源订单数量不会变;

3、暂不支持设置销售订单保存或者审核后自动生成销售出库单;

4、在销货订单编辑界面,可以设置销售订单生成销售出库单携带数据,如销售订单日期、业务员、客户。

![](https://vip.kingdee.com/download/0109ca792e3ffd034121b4b71e1af31c9889.png)
