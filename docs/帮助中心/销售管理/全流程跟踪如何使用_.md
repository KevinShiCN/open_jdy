# 全流程跟踪如何使用?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：shaoling | 阅读：517 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/683480221760336128)

【问题描述】

全流程跟踪如何使用?

【业务场景】

如:需要查询与某一个销售出库单相关联的上下游单据,包括销售订单、预收单、核销单、收款单、销售退货单、收款退款单等相关联的单据,可通过“全流程跟踪”功能查询。

【解释说明】

**系统提供了两种方法进行全流程跟踪查询单据,支持“全流程跟踪”查询;或者通过“上查”、“下查”进行查询**

![](https://vip.kingdee.com/download/010967445a9b816f4b8a9dfb153930b56452.png)

**全流程跟踪:指用于联查与当前单据所有相关联的单据信息。**

**上查:指查看单据上游的单据,只能查询到直接关联的一张上游单据**

**下查:指查看单据下游的单据,只能查询到直接关联的一张下游单据**

【操作流程】

以销售出库单为例,依次点击【销售管理】-【销售出库】-【查询】打开销售出库单列表界面。

![](https://vip.kingdee.com/download/010950272578b2a94dc7b9109bf139016ec9.png)

**方法一:**

在单据列表中选中某一个需要查看关联单据或单据中,直接点击“全流程跟踪”按钮,即可看到与本单关联的所有单据

![](https://vip.kingdee.com/download/0109a0b07da1f54d4618980ed1217427a974.png)

**方法二:**

在单据列表界面勾选需查询的单据,点击“全流程跟踪”按钮,的下拉框再点击“上查”或“下查”

![](https://vip.kingdee.com/download/0109e69c55aab5b746e899d9fc1a90c831a8.png)

可以查询到与本单据相关的上游或下游单据![](https://vip.kingdee.com/download/010932663a95f8fc41f793ec41c8563571e9.png)

上查、下查只能关联查询一张上下游单据
