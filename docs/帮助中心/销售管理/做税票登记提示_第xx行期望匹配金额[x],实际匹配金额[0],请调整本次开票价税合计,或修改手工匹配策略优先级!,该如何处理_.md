# 做税票登记提示:第xx行期望匹配金额[x],实际匹配金额[0],请调整本次开票价税合计,或修改手工匹配策略优先级!,该如何处理?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：116 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/703773403802654720)

【问题描述】

做税票登记提示:第XX行期望匹配金额[X],实际匹配金额[0],请调整本次开票价税合计,或修改手工匹配策略优先级!,该如何处理?

![](https://vip.kingdee.com/download/01097805de96f74d43ae9233710f7ffd8556.png)

【解决方案】

在税票登记界面,打开【手工匹配规则】把规则【单据商品行与税票商品行按顺序匹配】开启并且“上移”到第一个置顶,重新选单做税票登记即可。

![](https://vip.kingdee.com/download/01099a93e4eca5bc4e8783813442bc4de1ea.png)
