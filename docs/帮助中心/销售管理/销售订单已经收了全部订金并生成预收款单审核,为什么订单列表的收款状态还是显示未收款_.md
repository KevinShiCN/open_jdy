# 销售订单已经收了全部订金并生成预收款单审核,为什么订单列表的收款状态还是显示未收款?

> 作者：海鸥乔纳森 | 阅读：661 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391351766274654976)

【问题描述】

销售订单已经收了全部订金并生成预收款单审核,为什么订单列表的收款状态还是显示未收款?

【解释说明】

订单上收到的订金记录在本次订金和订金比例及累计预收字段,销售订单列表的收款状态是根据订单下推生成销售单之后是否收款来显示的,关联的销售单收款了则订单列表的收款状态就会显示部分收款或者全部收款。
