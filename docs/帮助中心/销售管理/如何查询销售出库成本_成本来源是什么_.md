# 如何查询销售出库成本?成本来源是什么?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：2485 | 有用：3

【问题描述】

如何查询销售出库成本?成本来源是什么?

![](https://vip.kingdee.com/download/0109dc0f681acda74b73ae86ba8c74f9c77f.png) [视频观看](https://vip.kingdee.com/school/detail/770663518172165632?productLineId=35&lang=zh-CN&cosmic_type=video)

【解释说明】

出库成本是根据商品的入库成本计算出来的,做完销售出库单之后需要在**库存管理**进行**出库成本核算**才能计算出成本,然后再到销售出库单或者销售报表中查看成本金额。

出库成本计算规则可查看知识:[移动平均法,加权平均法,先进先出法三种成本计算方法是如何计算数据的?](https://vip.kingdee.com/article/442853165864643328?productLineId=35&isKnowledge=2&lang=zh-CN)

【操作步骤】

**1、进行出库成本核算**

审核销售出库单后需要在【库存管理】-【出库成本核算】中进行出库成本核算

相关知识:[出库成本核算如何操作?](https://vip.kingdee.com/article/391350577223119872?productLineId=35&isKnowledge=2&lang=zh-CN)

![](https://vip.kingdee.com/download/0109a408a0367f454465a0b785a6b731866f.png)

**2、成本****查询**

**2.1、在销售出库单和销售单列表查询销售成本**

①销售出库单详情界面打开表体列设置,将成本字段开启显示即可查询成本

![](https://vip.kingdee.com/download/010963f07101770947f3b5502101203742bb.png)

②销售出库单列表也是先在列设置中将成本字段显示出来查询成本,其中**出库成本本位币**在列表支持显示合计,可通过该字段在单据列表查询成本合计金额。

![](https://vip.kingdee.com/download/01099d93013b02474975b1d3a19b1ac66b70.png)

**2.2、在销售报表中查询成本**

销售明细表中的商品及数量、金额、成本来源于已审核的销售出库单、销售退货单、委托代销结算单和委托结算退货单等单据的**商品行******

**只当有已审核的单据做了出库成本核算后,单据上才会有成本,这样销售明细表也就能取到数了,如果单据上本身成本就没有值,那么明细表上也会查不到。******

销售利润表、销售明细表、销售汇总表、销售排行表、销售收款一览表都支持查询销售成本

![](https://vip.kingdee.com/download/0109ce2898c8b54947a9acdaa747d673e3d6.png)

销售明细表计算毛利可查询每行分录的成本

![](https://vip.kingdee.com/download/01090dfb9ddce0a04b84bf28498faa4535b9.png)

销售利润表计算毛利可查询整单的成本

![](https://vip.kingdee.com/download/01096997e06d505840939c8f05c51edd964c.png)

销售汇总表计算毛利可按不同的汇总依据查询汇总成本

![](https://vip.kingdee.com/download/01096dbf9957912d48a19152dfd19f661ccb.png)

销售排行表计算毛利也可以查询对应的成本金额

![](https://vip.kingdee.com/download/01092304bfa61a6942a3890cd7e03ea15d8d.png)**2.3、商品收发明细表和商品收发汇总库存报表中查询销售成本**

![](https://vip.kingdee.com/download/0109d7f36b9606e04ac1af34bbdc2da566e5.png)商品收发明细表可查询每笔销售对应的成本

![](https://vip.kingdee.com/download/0109718dcc2b923d412199fd74ad7dcfd363.png)

商品收发汇总表可根据不同的汇总依据查询汇总成本

![](https://vip.kingdee.com/download/01091991a4c6a88147f0b0c4e57186e09304.png)

【注意事项】

如果勾选了计算毛利,也做了出库成本核算,但是报表上看不到成本、毛利字段,请检查:
1)检查报表的“列设置”,看一下成本、销售毛利是否有开启显示

2)是否设置了授权不允许查看,可查看知识:[如何设置子账号能否查看成本,单价,金额字段的权限?](https://vip.kingdee.com/link/s/Z03Jo)
