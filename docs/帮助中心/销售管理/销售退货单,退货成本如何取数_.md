# 销售退货单,退货成本如何取数?

【业务场景】

销售退货单,退货成本如何取数?

![](https://vip.kingdee.com/download/01091cc1c17f074743ae8137be901140208b.jpg)[视频观看](https://vip.kingdee.com/school/detail/727233849448448000?productLineId=35&lang=zh-CN&cosmic_type=video)

【解释说明】

如果按以下方式计算后没有成本,可根据:**[出库/退货类单据为什么没有成本?](https://vip.kingdee.com/link/s/ZmUZq)** 排查

销售退货成本有两种取数方式:

1、按源销售单生销售退货单,取源销售单的出库成本做为退货单的成本。

如图,源销售出库单位成本2400元,按源单生成采购退货,出库单位成本也是2400元

![](https://vip.kingdee.com/download/0109c93219540ecc42d49e7f033245331908.png)

![](https://vip.kingdee.com/download/0109be9e4e550eff43f89ba9293003630592.png)

2、直接新增销售退货单,退货成本可自定义取数规则(路径:左下角【系统设置】-【系统参数】-【存货核算参数】-【无源单的销售退货单成本来源】,设置的取数规则,取销售退货成本。)

如图:取数规则优先取“上次出库成本”

![](https://vip.kingdee.com/download/010914c6505f5ddf45e692c3680707904089.png)

该商品确实有上次出库成本,核算成本时,则取上次出库成本做为本次无源单销售退货单的成本。

![](https://vip.kingdee.com/download/01099a9ef750fc544603b0b9ec9dd9343a99.png)

注意:如该商品没有“上次出库成本”,则按系统设置的取数规则依次取:期初平均成本、上次入库成本、商品参考成本。

3、若没有启用【无源单的销售退货单成本来源】参数,则销售退货单无法取到退货成本,出库成本为0。进行成本核算时,提示“单价(无源单),根据系统参数,未成功获取成本来源:单价=0,数量*0=0 (无成本单价)”。

如图:

![](https://vip.kingdee.com/download/0109bc168b950b7a44f89a4bee4fd6198dfa.png)

注意:核算成本前,需要把当期的所有单据审核,并且有修改系统参数成本取数规则,必须重算成本。

4、系统也支持手工录入单据成本,请参考知识:[核算单据查询功能如何使用?](https://vip.kingdee.com/link/s/ZgJNx)

【库存管理】>【核算单据查询】

![](https://vip.kingdee.com/download/01095a8b7ec4ace4495b949255da69560b15.png)

【温馨提示】

若要重算无源单的退货成本,请在进行【库存管理】>【出库成本核算】时,勾选【无源单销售退货单重新核算成本】参数,如图:

![](https://vip.kingdee.com/download/0109df10263acbb04b12a8f673ee07e7af7d.png)

【温馨提示】

1、销售退货单的成本由系统自动核算,不能直接在单据中录入,也不能通过Excel导入,支持通过“核算单据查询”功能中录入;

2、如您是生产的产品退库单无法带出成本,可检查产品退库成本核算流程是否有正常计算:[产品退库单如何使用,成本费用等如何计算录入?](https://vip.kingdee.com/link/s/ZLeEv)

3、采购退货成本取值,详细可点击:[采购退货单,退货成本如何取数?](https://vip.kingdee.com/knowledge/579473470099109120?productLineId=35&isKnowledge=2&lang=zh-CN)
