# 销售利润表

# **1、简介  **

## **1.1功能介绍**

用户在开展完销售业务以后，除了在销售明细表和销售汇总表查看销售毛利外，还可以在单独的利润表中查看当前销售的利润，便于统计和经营决策。按指定的日期查询统计，能很清楚的看到当天的营业额，成本，费用支出和利润，以及收款状态该等，做到心中有数。

## **1.2适用版本**

星辰3.1以上所有版本

## **1.3系统路径**

销售管理>销售报表.销售利润表

## **1.4字段/按钮说明**

**必录字段：**

- **查看类型**：查看整单，默认只展示整单查询条件，默认按“查看整单”查询；当选择查看商品明细时，所有查询条件均显示。即按照单据分录展示，每个分录商品单独展示一行，按照明细表查看表单展示到最细维度。
- **单据日期**：默认查询当月1号到查询当天的日期（假如今天是2022年12月20日，那么默认查询日期就是2022-12-01-2022-12-20）

**非必录字段：**

- 提供单据类型、审核状态、单据编号、客户、部门、业务员、商品（商品编码、商品品牌、商品类别、规格型号等维度）等字段，按查看整单和查看商品明细时的筛选条件不一样，不指定条件时则代表全选。
- 单据类型：销售出库、销售退货、委托代销结算、委托结算退货，默认全选。
- 审核状态：默认只取“已审核”状态的单据
- 最后收（退）款时间：默认为空，点击查询时统计所有时间段的收款汇总；
- 筛选设置：跟“查看类型”有关，不同的类型下，筛选项会有部分差异，详见下方表格说明。

| 查看类型 | 对应的筛选项 | 筛选项含义 |
| --- | --- | --- |
| 查看整单 | 计算毛利 | 不论是查看类型是“查看整单”还是“查看商品明细”都会显示。默认勾选，不得去掉，只有勾选了才能显示利润 |
| 显示未开票销售出库/退货单 | 仅适用于星辰旗舰版。不论是查看类型是“查看整单”还是“查看商品明细”都会显示。 |  |
| 查看商品明细 | 显示批次和保质期 | 默认不勾选，不显示商品批次和保质期信息，列表中批次、保质期列也不显示，勾选后，列表中自动显示批次、保质期信息列 |
| 显示辅助属性 | 默认不勾选，不显示辅助信息，列表中辅助属性列也不显示，勾选后，列表中自动显示辅助属性 |  |

# **2、主要操作流程**

## **2.1查询说明**

1） 单据排序方式：按单据日期、单据编号、商品行号（就是开单时的商品顺序）依次排升序

2） 当点击“单据编号”时，能够在新的页面中，打开相应的单据详情，并跳转到单据详情页面

![上传图片](https://vip.kingdee.com/download/01000a7efc1263424e5ea7fcd2e155a19b02.png)

3） 默认查询所有币别，当系统启用外币时，查询结果中与金额相关的字段对应的是单据上的币别，如需查询本位币的字段，需要在列设置中把含“本位币”相关的字段开启显示。

![上传图片](https://vip.kingdee.com/download/0100d21199ba48d74bdd92e468e782fbb616.png)

![上传图片](https://vip.kingdee.com/download/010069648eb16e2a40f28dc412a8287ac4e7.png)

>
注意：不带“本位币”的金额字段，如价税合计、实际不含税金额、成本，都是直接从单据上取值的，而这些字段对应的本位币字段，都是根据原币字段跟汇率计算后得出的。

## **2.2 应用流程**

1）打开销售管理-销售报表-销售利润表；

2）由于星辰的出库成本不是实时计算的，为保证成本准确性，查询利润前，请先进行出库成本核算（直接点击即可跳转到出库成本核算页面），核算完毕再**刷新**销售利润表；

![上传图片](https://vip.kingdee.com/download/0100e0b2a0713e6248658d5a695c63135690.png)

3）按照需要查询的条件，输入筛选条件后，点击“查询”即可；

4）支持引出，查询，打印、列设置、组合排序等通用报表的功能。
