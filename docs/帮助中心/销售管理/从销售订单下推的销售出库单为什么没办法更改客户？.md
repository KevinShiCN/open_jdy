# 从销售订单下推的销售出库单为什么没办法更改客户？

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：快速入门, 常见问题
> 作者：进销存赵嘉丽 | 阅读：419 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/113682407848663552)

# 【问题概述】

从销售订单下推的销售出库单为什么没办法更改客户？

# 【原因说明】

订单下推的销售出库单，为保证与销售订单一致，系统不允许修改客户，否则会导致数据不一致。

# 【适用版本】

星辰3.1以上所有版本
