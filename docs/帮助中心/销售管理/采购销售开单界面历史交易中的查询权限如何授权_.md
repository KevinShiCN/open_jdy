# 采购销售开单界面历史交易中的查询权限如何授权?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：云社区用户55os622 | 阅读：151 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/665722922878454528)

【问题描述】
销售出库单录入界面历史交易的查看权限在哪里控制?![](https://vip.kingdee.com/download/010963db4972b6bb4aa485f68c70101489cd.png)

【解释说明】
单据历史交易中记录的取值对应关系为:

| 采购记录 | 采购明细表 |
| --- | --- |
| 销售记录、委托代销记录 | 销售明细表 |
| 库存数量 | 库存查询表 |

故要限制或打开历史交易中的数据查询,需管理员登录,进入【系统设置】-【用户管理】,找到的对应的职员,打开对应的功能授权![](https://vip.kingdee.com/download/010966076631f9024dcd87fbf0e18420f7a9.png)
