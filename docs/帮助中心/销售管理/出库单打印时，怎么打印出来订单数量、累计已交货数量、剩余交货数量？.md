# 出库单打印时，怎么打印出来订单数量、累计已交货数量、剩余交货数量？

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：进阶技巧, 精华汇总, 应用案例
> 作者：小微微 | 阅读：34 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/798968734273376256)

# **1 业务场景**

客户要货数量比较大，订单分批次交货给客户，在用销售出库单打印发货清单给客户时，想要在发货清单上体现出客户要货数量，累计已交货的数量，待交货数量。

# **2 解决方案**

针对上述场景，若商品为单单位商品，或者销售开单以基本单位来开单，可以通过设计星辰的出库单打印模板来实现。

# **3 适用版本**

星辰3.0以上标准版/专业版/旗舰版/所有版本。星辰旗舰版仅适用于用订单下推出库的场景。

# **4 应用流程**

使用前提：商品为单单位商品，或者销售开单以基本单位来开单。

1）模板设计：在使用的销售出库单打印模板上分别添加订单数量、本次交货数量、累计已交货数量、待交货数量列。

![上传图片](https://vip.kingdee.com/download/010061472f8fbf29479dae3cee83c006b5da.png)

其中订单数量取值为“订单基本数量”，本次交货数量即出库单上的数量，累计交货数量取值为公式“数量+订单已执行数量”，剩余交付数量取值为公式“订单基本数量-数量-订单已执行数量”。

![上传图片](https://vip.kingdee.com/download/0100d3216fdf860f483cb234e247d7169a60.png)

![上传图片](https://vip.kingdee.com/download/01000056bfb9875c405abb9263ccd9305347.png)

合计行，本次交货数量直接对数量累加求和，累计交付数量取值为公式sum(数量+订单已执行数量)，剩余交货数量取值为公式sum(订单基本数量)-sum(数量+订单已执行数量)

![上传图片](https://vip.kingdee.com/download/01009e584f2c3d114b1aacf1806e02b95614.png)

![上传图片](https://vip.kingdee.com/download/0100b21f2c3334b944c584d288d7cd8ce224.png)
