# 2.0在哪里登记单据发票信息?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：云社区用户8x80607 | 阅读：151 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/621511497428049408)

【问题描述】

2.0在哪里登记单据发票信息?

【操作步骤】

以销售出库单登记发票信息为例。

1、销售出库-销售出库单列表,右上角-【录入发票信息】

![](https://vip.kingdee.com/download/01094fc7383009594950987c9427ac8a996f.png)
2、即可录入该单据的发票,支持录入的字段:*开票日期、开票税号、*开票名称

*发票类型、*发票号码、发票代码、未开票金额、*开票金额、备注。“*”号为必填项

![](https://vip.kingdee.com/download/01097a67923703094e45b66debf4aed3650f.png)
3、列设置,打开发票相关字段查看:开票状态、开票日期、已开票金额、未开票金额、商品开票状态、需要开票

![](https://vip.kingdee.com/download/0109f5920e4188e64ee9b7ef8a580aec23d5.png)
4、可点击已开票金额数字,查看该单据的发票信息。
![](https://vip.kingdee.com/download/0109f6c449151e5c42f79673503a13ffa1cd.png)
5、如果没有【录入发票信息】功能,登录管理员账号-系统设置-用户管理-点击功能授权。如下图所示![](https://vip.kingdee.com/download/0109b222abf347384da4bc0280c55f353704.png)
