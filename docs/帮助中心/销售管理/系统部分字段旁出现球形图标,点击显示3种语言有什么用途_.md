# 系统部分字段旁出现球形图标,点击显示3种语言有什么用途?

> 作者：倒也不扶墙 | 阅读：18 | 有用：0

【问题描述】

系统部分字段旁出现球形图标,点击显示3种语言有什么用途?

【解释说明】

在基础资料或是单据录入时,部分字段旁有个小地球![](https://vip.kingdee.com/download/01090d3adc3bc6ca48dfa1f8c9f7b3cc2742.png)的图标,点开后可以针对同一个字段,分别录入简体中文、繁体中文和英文的数据。这个主要用于购买了星辰星辰多语言模块,切换不同语言环境可查看到相应语言录入的数据。

![](https://vip.kingdee.com/download/0109a1412086bfee46c68b684bee71873ab0.png)
