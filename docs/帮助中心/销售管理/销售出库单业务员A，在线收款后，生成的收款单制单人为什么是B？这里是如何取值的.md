# 销售出库单业务员A，在线收款后，生成的收款单制单人为什么是B？这里是如何取值的

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题, 进阶技巧
> 作者：echoquelqjisf | 阅读：56 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/789441715161032704)

# **1 问题描述**

销售出库单业务员A，在线收款后，生成的收款单为什么是B？

收款单的制单人和业务员是如何取值的？

# 2 适用版本

星辰3.0以上标准版/专业版

# **3 原因解析**

**以下5个场景中**

员工如有收款单新增 审核权限，则取员工作为制单人

| 不同场景 | 收款单制单人 | 收款单业务员 |
| --- | --- | --- |
| 员工A 在PC端使用扫码枪收款 | 员工A | 与收款单中的业务员一致 |
| 员工B在金蝶云星辰APP上使用摄像头收款 | 员工B |  |
| 员工C在金蝶云星辰APP上出示收款码，客户扫码付款 | 员工C |  |
| 员工D分享销售出库单后，客户在手机上通过微信或浏览器打开分享单据进行付款 | 员工D |  |
| 员工E创建销售出库单打印模板，将二维码打印出来张贴在货物上，客户收货后，用手机微信扫码打开单据进行付款 | 员工E |  |

如无，则取服务管理员权限，确保收款流程不受阻碍（此逻辑为2025/11/27更新）
