# 如何查看商品的或者销售单据的整单毛利?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：进销存赵嘉丽 | 阅读：1606 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/634919529000053248)

【问题描述】

如何查看商品的或者销售单据的整单毛利?

【操作指引】

星辰里,可以查询销售毛利的地方有以下几张销售类报表:

| 版本 | 报表 |
| --- | --- |
| 所有版本 | 销售明细表 |
| 所有版本 | 销售汇总表 |
| 所有版本 | 销售利润表 |
| 所有版本 | 销售排行表 |
| 所有版本 | 销售收款一览表 |
| 旗舰版 | 销售发票明细表 |
| 旗舰版 | 销售发票汇总表 |

下面我们以旗舰版的销售利润表为例,讲解如何查询销售毛利,其他版本查询也是同理:

1)先打开销售利润表,如果太久没结账,报表里也可以看到对应的提示语说明,此时建议先结账,结账可以保证成本的准确性,:

![](https://vip.kingdee.com/download/01093e536722370a439bac2f915f61748422.png)

点击提示语右侧的“出库成本核算”,会自动跳转到【出库成本结算】的页面:

![](https://vip.kingdee.com/download/010972f2cae472414705873de0bc3b6b4afe.png)
2)选择好核算参数,点击“开始核算”:

如果核算时提示“核算日期范围已经超过3个月。请您先进行【业务期末结账】操作,逐期计算成本。”(如下图),那么建议先点击【业务期末结账】去进行结账操作,如您没有权限,请联系账套管理员进行操作,待业务结账完成后再回到出库成本核算页面重新点击“开始核算”。

![](https://vip.kingdee.com/download/01097f19ec1f6af74860861769e4911477e9.png)

如果核算日期范围没有超过三个月,则可以正常核算,核算完成会提示“出库成本核算,计算完毕”,然后点击“完成”即可关闭当前页面。

![](https://vip.kingdee.com/download/0109afeaf99c12434a4e8875960a23e89d47.png)
**出库成本核算的结果是,算出核算期间内影响应收的业务单据上的“出库成本”,并最终****反写到单据上(如下图)。**同理,如果您的单据上的出库成本为空,建议先审核业务单据,然后做完出库成本核算后,再重新打开单据即可。

![](https://vip.kingdee.com/download/0109658b80e654fd473ba66c7fd7757111fb.png)

**关于成本的具体计算过程可以查看库存管理>存货核算>出库核算结果查询,点击“成本计算报告”可以查看计算明细。**如果您对计算的过程或结果有疑问的,请直接联系售后技术支持进行咨询。

![](https://vip.kingdee.com/download/010988b96ca977d44dacbf53b61c32526a13.png)

3)完成出库成本核算后,回到销售利润表中,点击“刷新”

![](https://vip.kingdee.com/download/01097e83634b824d404387924ddce901122d.png)

4)筛选设置中勾选“计算毛利”,销售利润表的此参数是默认勾选的,无需勾选,如果是其他报表,如销售明细表、销售收款一览表,**则需要用户自己先勾选**,此外,**用户可以自定义是否要在销售毛利中剔除销售类单据的抹零金额、销售总费用,如果需要剔除,那就勾上,不勾则不扣减**:

按整单查询的计算毛利公式:

![](https://vip.kingdee.com/download/01098836f1ecafa6429492612d0316071a11.png)
按查看商品明细的计算毛利公式,**没有“抹零金额”这个扣减项(因为这个字段是单据头字段,只有在查看整单条件下才能剔除,商品分录的维度不支持)**:

![](https://vip.kingdee.com/download/0109adcee4646b0a4f6aa37de80911e05c41.png)

5)最后点击“查询”,就能看到对应的销售毛利的数据了。
![](https://vip.kingdee.com/download/010908404b47b50c46128951cb4b34eb4f36.png)
- 销售毛利=实际不含税金额-成本(取自单据上的出库成本)
- 销售毛利本位币=实际不含税金额本位币-成本本位币(取自单据上的出库成本)
- 毛利率=销售毛利/实际不含税金额
- 毛利率=销售毛利本位币/实际不含税金额本位币

6)如果想查询商品行的毛利,查看类型选择“查看商品明细”后,点击“查询”即可

![](https://vip.kingdee.com/download/0109a125ea9b88cc4cd5a502c270ccd2ac2a.png)
