# 销售类单据列表重新取价功能

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：进阶技巧, 精华汇总, 功能介绍
> 作者：进销存赵嘉丽 | 阅读：3641 | 有用：9
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/139420269231570688)

问题描述：

**问题描述**

销售订单生成后，在出库前，商品价格发生了变化，能否对订单重新取价?

解决方案：

**回答**

销售类单据支持重新取价。但是有一定的条件限制，详见下面说明。

**应用场景及价值**：

当商品的销售价格变化后，未执行完成的销售单据需要按照新的定价更新价格，以更好地反映实际交易金额。

**应用流程**：

1、首先要在【系统设置】-【系统参数】-【进销存参数】-【销售参数】中开启允许重新取价的功能（如下图）

![](https://vip.kingdee.com/download/010063040f1cf68d410e8dccf1f3309b5e6d.png)

- **注意！：销售订单已关闭、已出库； 销售出库单已生成凭证、已收款、已开票、有退货； 销售退货单已生成凭证、已退款； 销售换货申请单已有出入库记录，则不允许重新取价。**

2、打开【销售订单列表】，选中要改价的单据，点击【更多】-【重新取价】即可重新取价。

![](https://vip.kingdee.com/download/0100ca6aa5eb8c584741877f01aa750dded1.png)

注意：重新取价的依据来自【基础资料】-【价格取数优先级】-【销售类单据取价】中的优先级顺序：

![](https://vip.kingdee.com/download/01001322190329644e58a5b9c7d03344c686.png)

- **注意！销售订单下推生成销售出库单时是携带源单上的价格，修改销售出库单上的数量也不重新取价，但是点击“重新取价**”后，不管是不是源单下推，都会更新当前单据的价格。
