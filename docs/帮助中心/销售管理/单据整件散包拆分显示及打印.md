# 单据整件散包拆分显示及打印

# 1 简介

## **1.1 应用场景及价值**

对于商贸企业，商品启用了多单位（比如1件=20包），订单拣货出库时，需要快速按总件数和总包数去拣货和对数，提高准确性，提升效率。

## **1.2 功能特性**

支持按单位显示整件数和散包数，并支持数量的合计，支持打印

## 1.3 适用版本

星辰3.1以上所有版本

# **2 应用流程**

（1）单据的列设置里可以展示整件数或散包数，根据实际情况选择是否显示。

（2）系统已经预置了整件散包的打印模板

![上传图片](https://vip.kingdee.com/download/010015171baf3fd6450a9b4357618222ba36.png)

![上传图片](https://vip.kingdee.com/download/0100b78a3e63b886436a9ac7f9f89d1ad9c9.png)

用户也可以在自定义模板上设置字段和合计。

可在左侧数据源搜索“整”以及“散”的关键字，单据即可添加到模板上。

![上传图片](https://vip.kingdee.com/download/0100beca21e4043b476697819f066ae7f606.png)

设置合计行。在合计行设置好合计类型后，绑定对应的整件散包字段即可。最后再设置一下字段的格式（选择格式为“数值”，可以设置尾零不显示、小数位等等）
