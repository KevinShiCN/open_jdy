# 采购销售汇总表如何使用?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题

【问题描述】

采购销售汇总表如何使用?

![](https://vip.kingdee.com/download/0109a7f0cac0c18b4512bfe8138e912e0510.png)

【解释说明】

采购销售汇总表的数据来源包含**已审核**和**未审核**的**采购入库单,采购退货单,销售出库单,销售退货单**。

统计了一段时间内商品采购和销售的数量、金额、进销差价、期末数量成本等数据。

支持按日期、仓库、商品类别、商品、辅助属性等条件查询

![](https://vip.kingdee.com/download/0109550f26b415bf4142bd236fe46ed85c0e.png)
支持设置显示的辅助单位

![](https://vip.kingdee.com/download/0109e2a190ee082b44c798d5383825b22d20.png)
支持设置按辅助属性统计、分仓库统计、无采购销售不显示、期末库存为0不显示、计算毛利扣减销售总费用

![](https://vip.kingdee.com/download/01092c9ce12073b94eeaaba8541d53b6b5b6.png)
查询结果支持引出、打印、设置列显示字段、组合排序

![](https://vip.kingdee.com/download/0109d5812e6afe444a85b4b5627333c95bf5.png)
