# 直运销售如何抵扣预收订金?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：Simone邓 | 阅读：33 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/797631666461618176)

【问题描述】

直运销售如何抵扣预收订金?

【解释说明】

直运销售支持定向抵扣预收账款。

**注意:该功能仅专业版支持,且目前处于灰度测试阶段。如需提前体验,请复制工作台的服务 ID 到售后在线服务界面发送给人工客服,申请加入灰度测试。**

销售订单录入预收订金时,勾选**定向抵扣**选项;订单审核后会自动生成预收款单,需完成该预收款单的审核操作。

后续由直运采购入库单生成直运销售出库单时,系统将自动从源销售订单的订金中完成抵扣。

![](https://vip.kingdee.com/download/010982aa853a2da24781b3e81af8453beab1.png)
销售订单审核后生成未审核的预收款单,点击单据编号打开预收款单进行审核
![](https://vip.kingdee.com/download/0109f839daf024db4f3f909791012fbee28d.png)
满足上述条件采购入库单生成直运销售时会自动抵扣销售订单上预收的订金

![](https://vip.kingdee.com/download/0109377895e229194583be6beba1ce829b43.png)
