# 销售出库单和发票勾稽后,为什么销售订单上的开票状态没有更新?

![](https://vip.kingdee.com/download/010930fadc81dc244e25919ee1a7da00544d.png)【解释说明】
**适用版本:金蝶云星辰旗舰版**
销售单票勾稽不会更新销售订单的开票状态及出库状态。

若想在订单列表看到订单的全流程状态,需要销售出库单和销售发票都以订单为源头单据进行下推,例如:

① 销售订单→销售出库单→销售发票
② 或销售订单→销售出库单;销售订单→销售发票,销售出库单与销售发票再自动勾稽
