# 销售订单全流程跟踪的下游单据显示顺序是什么?

【问题描述】

销售订单全流程跟踪的下游单据显示顺序是什么?

【解释说明】

单据全流程跟踪下游单据的显示顺序无规则,有关联性的单据随机排列显示。不根据下游单据的制单/审核时候进行排列。
