# 销售管理,企业经营利润表如何使用?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题, 进阶技巧
> 作者：杨桂平 | 阅读：2319 | 有用：3

【问题描述】

销售管理,企业经营利润表如何使用?

【业务场景】

是按照业务维度出具的统计企业主营业务、其他业务的收入、成本、利润情况的综合报表,不同于财务云系统的利润表,用于统计进销存云系统的业务数据。

【操作流程】

打开【销售管理】-【企业经营利润表】查看企业的经营利润。

![](https://vip.kingdee.com/download/010977a9425a0c994a3abd30966293ffb536.png)
![](https://vip.kingdee.com/download/010948cd7bb0b1754e28b4e36e969d516bb9.png)

**企业经营利润表筛选条件设置:**

- 支持按客户、部门、业务员筛选,为空表示所有的客户、部门、业务员;
- 报表数据所包含的单据类型,账套版本不同有所区别,详情点击:**[企业经营利润表上数据来源于哪些单据?](https://vip.kingdee.com/article/621873527737807872?productLineId=35&isKnowledge=2&lang=zh-CN)** 查看
- 包括未审核:勾选包括未审核时将可以查询筛选时间范围内的未审核单据。
- 收支类别发生额为0不显示:勾选后收支类别为的数据不显示在报表中。

【温馨提示】
