# 进销存单据、列表、报表部分字段增加Tip说明

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：快速入门, 功能介绍, 新特性
> 作者：进销存熊诗琴 | 阅读：329 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/667836623152736512)

## 1 简介

### 1.1 功能介绍

单据列表、单据详情、报表优化，部分字段右侧增加Tip说明，增强引导。

### 1.2 字段说明

- 销售订单列表优化，列表列标题右侧增加tip说明

| 版本 | 列表名称 | 字段 | Tip文案 |
| --- | --- | --- | --- |
| 所有版本 | 销售订单列表 | 出库状态 | 反映销售订单生成的销售出库单是否减少了库存。根据系统参数，分为保存时更新库存和审核时更新库存；商品行中可查看本行的已执行未出库数量和已执行已出库数量，查看详情。 |
| 标准版/专业版 | 销售订单列表 | 执行状态 | 指销售订单生成的销售出库单的情况，查看详情。 |
| 旗舰版 | 指销售订单生成的销售出库单或发货通知单的情况，查看详情。 |  |  |

- 单据详情优化，字段右侧增加tip说明：

| 版本 | 涉及单据/报表 | 字段 | Tip文案 |
| --- | --- | --- | --- |
| 专业版/标准版 | 销售出库单 | 抹零金额 | 只有“收取货款”不为零时，才能使用收款抹零功能。 |
| 旗舰版 | 销售发票 | 抹零金额 | 只有“收取货款”不为零时，才能使用收款抹零功能。 |
| 所有版本 | 预警设置-商品预警 | 滞销商品未销售预警 | 关于滞销商品的取数说明及滞销标签的生效时间，查看详情。 |
| 所有版本 | 组装单 | 成本分摊比例 | 取决于系统参数-仓存参数-组装单参数中“成本分摊比例计算依据”的设置 |
| 所有版本 | 拆卸单 | 成本分摊比例 | 取决于系统参数-仓存参数-拆卸单参数中“成本分摊比例计算依据”的设置 |
| 所有版本 | 销售出库单 | 销售费用分摊 | 不增加出库成本；通过销售报表查询毛利时可以选择扣减费用，查看详情。 |
| 税率 | 税率优先取商品信息上的销项税率，其次取客户资料里的增值税税率，最后取系统参数上的默认增值税税率，查看详情。本字段支持键盘F6键向下填充。 |  |  |
| 产地 | 取最近一次已审核的入库类单据或商品期初上的记录，注册证号、生产许可证同理，查看详情。本字段支持键盘F6键向下填充。 |  |  |
| 销售订单 | 行已执行未出库数量 | 行已执行未出库数量=行已执行数量-行已执行已出库数量，查看详情。 |  |

- 报表优化，字段右侧增加tip说明：

| 版本 | 涉及单据/报表 | 字段 | Tip文案 |
| --- | --- | --- | --- |
| 所有版本 | 销售收款一览表 | 筛选条件“收款日期” | 此字段仅适用于系统参数-应收应付参数-基础参数为“按单据核销”或“按余额核销”。当核销方式为“按分录核销”时，此查询条件会隐藏，无法查询商品分录的具体收款日期，查看详情。 |
| 所有版本 | 商品收发明细表 | 仓库 | 期初显示的总的所有仓库的期初,如果要查具体仓库的期初库存,直接筛选过滤条件中的“仓库”即可，查看详情。 |
| 批次 | 如果要查具体批次的期初库存,直接筛选过滤条件中的“批次”即可。 |  |  |
| 所有版本 | 库存预警查询表 | 仓库名称 | 只有在基础资料-商品信息-库存预警中，开启了“分仓管理库存”，仓库才会有值，查看详情。 |
| 所有版本 | 销售利润表 | 最后收(退)款日期 | 即单据上存储的"最后收款日期/最后退款日期"，查看详情。 |
| 销售费用分摊 | 来源于销售单据商品分录上的销售费用分摊 |  |  |
| 关联销售费用 | 来源于销售单据商品分录上的关联销售费用，此字段是由费用分摊单反写到销售单据上的。 |  |  |
| 销售总费用 | 销售总费用=销售费用分摊+关联销售费用。其中，“关联销售费用”只有专业版/旗舰版支持。 |  |  |
| 成本 | 取自销售单据商品分录上的出库成本，出库成本通过出库成本核算来计算得出。 |  |  |
| 所有版本 | 库存查询表 | 参考零售价 | 取自商品资料-价格设置中的基本单位的零售价 |
| 参考零售总金额 | 零售总金额=零售参考价*即时库存（基本） |  |  |
| 参考采购价 | 取自商品资料-价格设置中的基本单位的采购价 |  |  |
| 参考采购总金额 | 采购总金额=采购参考价*即时库存（基本） |  |  |
| 参考批发价 | 取自商品资料-价格设置中的基本单位的批发价 |  |  |
| 参考批发总金额 | 批发总金额=批发参考价*即时库存（基本） |  |  |
