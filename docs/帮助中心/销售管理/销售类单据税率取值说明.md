# 销售类单据税率取值说明

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：进销存赵嘉丽 | 阅读：2502 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/632747508942485248)

【问题描述】

销售类单据税率取值说明

【解释说明】

销售类单据上的“增值税税率”取值来源及优先级如下:

| 优先级 | 取值来源 | 产品截图说明 |
| --- | --- | --- |
| 1 | 商品信息上的销项税率 |  |
| 2 | 客户资料里的增值税税率 |  |
| 3 | 系统参数上的默认增值税税率 |  |

1)当最高优先级没有取到则默认按下一优先级取,直到取完所有优先级。

2)销售开单的税率优先级是默认的,不允许调整优先级的取数顺序。

【特别说明】

如果商品上开启了“免税”,则开单时税率是不会带出来的。
