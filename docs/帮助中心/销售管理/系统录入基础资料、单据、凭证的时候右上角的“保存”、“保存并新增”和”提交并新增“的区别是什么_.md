# 系统录入基础资料、单据、凭证的时候右上角的“保存”、“保存并新增”和”提交并新增“的区别是什么?

> 作者：杨桂平 | 阅读：449 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/537798910350365440)

【问题描述】

系统录入基础资料、单据、凭证的时候右上角的“保存”、“保存并新增”和”提交并新增“的区别是什么?

【名词解释】

1、【保存】,单据并停在原来界面,仅仅是保存该单据;

2、【保存并新增】,点击后保存原单据同时打开新单据录入页面。

![](https://vip.kingdee.com/download/010941d46e83547f4595b85164bcb24c26ca.png)
3、【提交并新增】,启用审批流的单据需提交审批,单据界面设有“提交”与“提交并新增”按钮。若单据未录完,可点“保存”暂存信息或“保存并新增”转至新增界面;若已录完,直接点“提交并新增”,可同步提交审批并开启新单据录入

![](https://vip.kingdee.com/download/0109bc06f36c3c904ad495e97523452b8535.png)
