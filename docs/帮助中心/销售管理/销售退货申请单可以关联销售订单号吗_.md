# 销售退货申请单可以关联销售订单号吗?

【问题描述】

销售退货申请单可以关联销售订单号吗?

【操作步骤】

1、销售退货申请单可以关联销售订单号,若销售出库单是由销售订单下推而成。此时销售管理-销售退货申请单,列设置-【销售订单号】字段显示打开![](https://vip.kingdee.com/download/010933c536b0481d4d098982f584bcd66fe9.png)
2、可查看关联销售订单号![](https://vip.kingdee.com/download/01090ff453f4ae5f4259a3cddf45a164e50b.png)
