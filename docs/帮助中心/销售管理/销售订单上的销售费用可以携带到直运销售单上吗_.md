# 销售订单上的销售费用可以携带到直运销售单上吗?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：Simone邓 | 阅读：29 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/797631694009784064)

【问题描述】

销售订单上的销售费用可以携带到直运销售单上吗?

【操作步骤】

直运销售支持将销售订单上录入的预计销售费用携带到直运销售出库单上。

**注意:该功能仅专业版支持,且目前处于灰度测试阶段。如需提前体验,请复制工作台的服务 ID 到售后在线服务界面发送给人工客服,申请加入灰度测试。**

**携带规则如下:**

1、若是一对一下推,会自动携带;

2、若是多对一下推,则根据系统参数“销售类单据合并下推携带销售费用”是否勾选来决定是否合并携带;

3、若是多对多或一对多下推则仅在第一次下推时携带。

操作步骤:

【系统设置】-【系统参数】-【进销存参数】-【销售参数】,勾选“销售订单合并出库携带销售费用”
