# 订单上的整件散包、整件数、散包数是什么意思,是如何携带到单据上的?应用场景是什么?

【问题描述】

订单上的整件散包、整件数、散包数是什么意思,是如何携带到单据上的?应用场景是什么?

![](https://vip.kingdee.com/download/0109a32f3bc648e5419a8f5e73bb147f5492.png)

【场景介绍】

整件散包其实是属于多单位商品的应用,适用于如批发零售行业中,同时存在按整箱/整件、以及按拆零散装销售及发货的业务场景,按整件散包下单,也可以方便仓管人员拣货。例如:食品行业,客户下单食用油13桶,假设1件=4桶,那么仓库拣货就可以按3件1桶来拣货,提高拣货效率。

【名词解释】

整件数3:在同时启用四个单位时的最大单位

整件数2:在同时启用四个单位时的第二大单位,或者同时启用三个单位时的最大单位

整件数1:在同时启用四个单位时的第二大单位,或者同时启用三个单位时的第二大单位,或同时启用两个单位时的最大单位

散包数:多单位中的最小单位

整件散包:即整件数和散包数的汇总

辅助单位:就是除商品基本单位以外的单位(见下图说明)

![](https://vip.kingdee.com/download/0109e2f9707fe1794606a332df787d1019ca.png)

【操作流程】

录单取数规则:单据上整件数、散包数携带是跟商品启用的单位数量有关的:

a)假设启用了四个单位:

![](https://vip.kingdee.com/download/0109325b3a9079b6430bb790ce179070de19.png)

整件数3对应最大的单位,

整件数2对应第二大的单位,

整件数1对应第三大的单位,

散包数对应最小的单位。

![](https://vip.kingdee.com/download/0109503f9f5397c649f38fc76486fb51f1f2.png)

b)假设启用了三个单位:

![](https://vip.kingdee.com/download/010994e05ec7f2524a1cb11bec48fc956598.png)

整件数3为空,

整件数2对应最大的单位,

整件数1对应第二大的单位,

散包数对应最小的单位。

![](https://vip.kingdee.com/download/0109911195d6a0874c9a8b07fce98bc3570f.png)

c)假设启用了两个单位:

![](https://vip.kingdee.com/download/0109a6be720cb2e84de49d4f33a3895262f1.png)

整件数3、整件数2为空,

整件数1对应最大的单位,也就是基本单位,

散包数对应最小的单位。

![](https://vip.kingdee.com/download/01093c1f5f9869b1471698bdaf583f64af07.png)
