# 委托代销退货单选不到源单是什么原因?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：shaoling | 阅读：24 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/725889514467709440)

【问题描述】

委托代销退货单选不到源单是什么原因?

【业务场景】

做了委托代销单后需要退货,委托代销退货单选源单没有单据可选是什么原因?

![](https://vip.kingdee.com/download/0109fd81601373634db0a61b718f514a331a.png)

【分析原因】

1、没有查询源单单据的权限

2、要退货的源单未审核

3、委托代销退货单所选币别和源单币别不一致

4、委托代销退货单客户选择错误

【解决方案】

1、如选源单委托代销单选不到,可先打开单据列表检查看下

若是无法进入单据列表提示没有权限操作,应该是没有该单据查询权限

可联系管理员授权单据查询权限,详情可参考:[如何设置子账号是否能查询单据的权限](https://vip.kingdee.com/knowledge/621511426527687424?channel_level=%E9%87%91%E8%9D%B6%E4%BA%91%E7%A4%BE%E5%8C%BA%7C%E6%90%9C%E7%B4%A2%7C%E5%AE%98%E6%96%B9%E7%9F%A5%E8%AF%86&productLineId=35&isKnowledge=2&lang=zh-CN)?

2、若单据列表里可显示单据,检查单据的审核状态

![](https://vip.kingdee.com/download/0109c5a4a15ff20843ba899ce5c0c87fe0d7.png)

若单据未审核,先点击<审核>操作

![](https://vip.kingdee.com/download/01093a6dda4c9f1a41e5ac47eeb50c16b8cd.png)
3、检查委托代销单的币别跟委托代销退货单的币别选择是否一致

![](https://vip.kingdee.com/download/0109bc46b4abdf4a42adaa842d13e86b86fd.png)

委托代销退货单的币别选择,跟委托代销单的币别选择需要一致

![](https://vip.kingdee.com/download/01091250c8069cc54446b6106e3b6fa5fa66.png)
若币别不一致,切换成一致的币别,再去操作选择源单

![](https://vip.kingdee.com/download/01099e807bede38d4d7c922a78b9f5026428.png)
4、检查客户是否存在同名,单据选错对应客户,可在客户列表搜索客户名称查看

![](https://vip.kingdee.com/download/01094d0e42ca8a3c4b64821b10a7c9b5e354.png)

若有多个同名客户,可重新按客户编码选择客户后再去选择源单操作

![](https://vip.kingdee.com/download/0109d8b617617a444f69999fd87bbe31f955.png)
