# 销售出库单在线收款

# 1 简介

## 1.1 功能介绍

金蝶云·星辰，面向批发业务支持销售出库单在线收款，自动生成收款单，并更新销售出库单收款状态

- 电脑网页端使用扫码枪进行扫码（客户出示付款码，商家扫码）
- 金蝶云APP面对面扫码 （商家出示收款码，客户扫码；客户出示付款码，商家扫码）
- 客户在微信中打开分享单据 / 通过微信扫单据打印的二维码（支持在线支付）

## 1.2 适用版本

金蝶云·星辰3.1标准版/专业版（暂不支持旗舰版）

## 1.3 功能特性

（1）支持的单据：销售出库单，支持整单全额付。暂不支持折扣、抵扣、优惠金额及抹零收款（特殊情况会计入优惠金额）

（2）支持的支付渠道：通联支付（支付宝/微信）、金蝶支付（支付宝/微信）、微信直连

（3）支持的币别：人民币

（4）支持的金额精度：因目前国内支付渠道仅支持2位小数，若本次应收超过2位小数，差额将计入收款单的优惠金额。如图：

## **2.3 销售出库单在线收款**

- 销售出库单开单时勾选“收款二维码”并选择支付渠道。针对已勾选“收款二维码”的单据，为避免重复生成收款单，已做如下控制： 当前销售出库单不支持手工下推收款单 ，收款单选源单时过滤掉当前销售出库单
- 销售出库单在线收款权限优化

**以下5个场景中，**员工如有收款单新增/审核权限，则取员工对应的权限；如无，则取服务管理员权限；确保收款流程不受阻碍

- 员工在PC端使用扫码枪收款
- 员工在金蝶云星辰APP上使用摄像头收款
- 员工在金蝶云星辰APP上出示收款码，客户扫码付款
- 员工分享销售出库单后，客户在手机上通过微信或浏览器打开分享单据进行付款
- 员工创建销售出库单打印模板，将二维码打印出来张贴在货物上，客户收货后，用手机微信扫码打开单据进行付款

![上传图片](https://vip.kingdee.com/download/010047047ebfe12041d68658605433cad0c2.png)



（2）收款：商家审核销售出库单后，点击<在线收款>按钮，使用扫码枪扫码收款

![上传图片](https://vip.kingdee.com/download/01009c7b414f96bb43a292aba7f2ccc89697.png) ![上传图片](https://vip.kingdee.com/download/01000fb1e04b817942ff8dc962a90761cc38.png)

#### **2.3.1.2 移动端收款**

（1）开单：商家在金蝶云APP新增销售出库单时，勾选“收款二维码”，并选择“支付渠道”

（2）收款：销售出库单审核后，在销售出库单详情页点击“在线收款”，商家扫客户付款码收款，或是客户扫商家收款码付款
