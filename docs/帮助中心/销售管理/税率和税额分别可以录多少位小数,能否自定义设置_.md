# 税率和税额分别可以录多少位小数,能否自定义设置?

> 作者：Schatz | 阅读：715 | 有用：0

【问题描述】

税率和税额分别可以录多少位小数,能否自定义设置?

【解释说明】

增值税税率最多录入4位小数,税额最多2位小数,并且都无法自定义设置小数位。如下图,在输入小数位数据是无法输入的:
