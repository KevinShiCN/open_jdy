# 星辰2.0,多账户收款如何把多个账户信息打印到单据上?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：winnie_wen | 阅读：206 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/568239383904777984)

【问题描述】

多账户收款如何把多个账户信息打印到单据上?

【解决方案】

打印模板上取“付款信息”字段,如图:

![](https://vip.kingdee.com/download/0109967f86ecdcfb4a58bed711e6c7d776ee.png)

支持的单据:采购订单,采购入库单,采购退货单;销售订单,销售出库单,销售退货单

打印效果,如图:

![](https://vip.kingdee.com/download/010983b382e6479e4aa2b4693e2660a0ae5e.png)

注意:数据源需取“单据头”,销售类单据绑定字段也为“付款信息”

【适用版本】

星辰2.0标准版/专业版
