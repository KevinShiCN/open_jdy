# 通过销售开票关联单据进行开票的记录不允许删除。该如何解决？

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：快速入门, 精华汇总, 常见问题
> 作者：进销存赵嘉丽 | 阅读：164 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/801823646163100928)

# **1 问题描述**

销售出库单反审核失败，原因是该单据已登记税票，需先删除税票登记记录。但在删除税票登记记录时，系统提示：通过销售开票关联单据进行开票的记录不允许删除。该如何解决？

# 2 适用版本

星辰3.0及以上所有版本

# **3 解决方案**

自**2025-04-28起，**销售开票已支持解除发票与业务单据的关联关系。可前往“税务管理>已开发票列表”进行单据解绑，解除单据与发票的关联关系，再删除税票登记，即可反审核单据。

![上传图片](https://vip.kingdee.com/download/010053eb101b2aca46438585e22cf1e638e1.png)

具体操作方法：

打开“已开发票列表”，勾选蓝字、红字发票，点击“单据解绑”，即可解除发票与单据的关联关系，并将销售税票登记表的登记方式更新为手工登记。

![](https://vip.kingdee.com/download/01004793b71b399f4b21a1f5c45a841b0498.png)如需修改或删除业务单据，可前往“销售管理>销售税票登记表”删除税票登记记录，然后反审核并修改业务单据。
