# 销售报价单中商品都已经下推销售了,但是销售报价单没有关闭,该怎么关闭,关闭后还能否启用?

> 作者：杨桂平 | 阅读：535 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/487064448205446144)

【问题描述】

销售报价单中商品都已经下推销售了,但是销售报价单没有关闭,该怎么关闭,关闭后还能否启用?

![](https://vip.kingdee.com/download/010911fad7b949d5444d839d7973f4b0b549.png)【解释说明】

销售报价单生成销售订单后不会自动关闭,其单据状态必须手动操作关闭;未关闭的销售报价单可以一直作为报价依据生成销售订单,已关闭状态的销售报价单不允许再次生成销售订单。![](https://vip.kingdee.com/download/0109d35ea059d6914b19a05b124579d3e927.png)![](https://vip.kingdee.com/download/010909903e8865574bc9b7457c0b39542743.png)
对手动关闭的销售报价单,后续若需再次启用,还可以勾选单据点<启用>按钮启用,启用成功后可继续下推销售订单。

![](https://vip.kingdee.com/download/0109c2ec391267d54260a8087dce9972c734.png)
