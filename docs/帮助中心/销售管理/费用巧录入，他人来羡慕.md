# 费用巧录入，他人来羡慕

![上传图片](https://vip.kingdee.com/download/01005e1bb225486347259f749ba906b24f5a.png)

      大张负责采购，小张负责销售。他俩除了对库存，一般没有太多交集。最近大张因为月末要对账加了几天班。但他发现每次下班，小张都在。出于好奇，大张跑去问小张最近忙啥，小张说：我们给客户发货，有的运费我们承担，有的运费客户承担，我在一笔笔核对这些运费呢，头都炸了。大张心想：销售费用跟采购费用应该也类似。于是他看了下小张做的其他支出单，不看不知道，一看惊呆了。小张都是手动做的其他支出单，然后在手动备注从哪一笔销售出库单来，难怪这么累。

      于是他对小张说：兄弟，你看，如果是销售出库单产生的运费，运费要是我们出，可以在销售出库单里录入销售费用，销售单审核后会自动生成关联的其他支出单。通过其他支出单给物流商那边付运费。

![](https://vip.kingdee.com/download/0100b26e4923882a45ab869698c24f32a327.png)

而且其他支出单的单据备注会显示“由源单:xxxxxx 生成”的字样。

![](https://vip.kingdee.com/download/0100c5159f3e7efb432682caff3956fd5830.png)

销售出库单列表也可以全流程跟踪查到对应的其他支出单。

![](https://vip.kingdee.com/download/01001b167a01c5b6491fa95fa8afdae4e620.png)

![](https://vip.kingdee.com/download/0100752f4d665e824beb9e165d298aef30fa.png)

      反过来，其他支出单列表也可以显示单据备注，如果显示的是“由源单:xxxxxx 生成”，就表示这个其他支出单是销售出库单产生的。

![](https://vip.kingdee.com/download/01001a80b2a684aa41dc96ed819e262f6318.png)

补充：销售费用点分摊会影响销售毛利，不点分摊不影响销售毛利，不影响成本，点不点分摊都会生成其他支出单。

![](https://vip.kingdee.com/download/01009a62433d1cff45d287002354401e76ae.png)

![](https://vip.kingdee.com/download/010082dd2ad50c554e7c938bc335b3960e10.png)

      小张好像看到了希望的曙光，赶紧接着问，那如果是客户承担运费要怎么录入呢？

大张在销售出库单看到了客户承担费用，于是转头告诉小张：如果是客户付运费，表示我们帮客户垫付，未来要从客户那里收回，可以直接录在销售出库单的客户承担费用里。

诶，奇怪！怎么本次应收款没有增加呢？

![](https://vip.kingdee.com/download/0100a905d89dc552463c8dea52dce2071b92.png)大张赶紧找了下系统参数，原来参数设置的是客户承担费用计入其他收入单。难改不在销售单上体现本次应收，而是在其他收入单那边体现其他应收了。

![](https://vip.kingdee.com/download/010008cab85285a34bddb13d920822a1baac.png)

![](https://vip.kingdee.com/download/010040e2105c92cf4d7da9ef9d587c50d833.png)

![](https://vip.kingdee.com/download/01007011234c62b746158f2d102d2671bd8f.png)

如果参数选客户承担费用计入本次应收款，那客户承担费用就会增加销售出库单的本次应收款了。这就对了。

![](https://vip.kingdee.com/download/01000d5b310e7b9a407c9923a6512253989f.png)

![](https://vip.kingdee.com/download/0100f3f650c4ebc34a8182d92819a310016d.png)

      小张看着大张操作，听着大张讲解，瞬间茅塞顿开，思路也清晰了起来。对于大张的感谢之情直接夜宵代替吧！小张使劲拉着大张往外走：兄弟，今晚我请客，想吃啥尽管说。

补充：采购费用类似，采购入库单录入采购费用以后点分摊，影响入库成本；不点分摊不影响入库成本。点不点分摊都会生成其他支出单。

![](https://vip.kingdee.com/download/01005511eed80b5c410787ae08b784d19786.png)
