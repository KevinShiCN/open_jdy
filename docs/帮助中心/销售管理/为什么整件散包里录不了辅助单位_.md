# 为什么整件散包里录不了辅助单位?

![](https://vip.kingdee.com/download/0109060abe91d5884f51bbda24b3b9e29e0d.png)

【解释说明】

**上面问题的原因在于,整件散包只能与固定单位做换算,不支持按浮动单位进行换算,包含了三种情况:**

**①****已开启“浮动换算”的单位,不参与整件散包的换算**

**②****与浮动单位进行换算的单位,不参与整件散包的换算**

**③****若开启了“浮动换算”,且其他多单位都与浮动单位进行换算,则以商品的“基本单位”参与整件散包的换算**
