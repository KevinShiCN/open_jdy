# 其他出库单和销售出库单有什么区别?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题

【问题描述】

其他出库单和销售出库单有什么区别?

【解释说明】
**其他出库单:处理非销售**类型的出库单据,不会产生应收款,比如销售附赠物料、报损、委托加工等不参与销售管理的出库类业务。

**销售出库单:**是销售商品给客户,产生收入,可以形成应收款的单据。

若销售出库时开启赠品按钮,则不影响客户应收款,与录入其他出库单无本质差异,都只影响商品的库存。
