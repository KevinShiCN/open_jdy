# 如何录入、查询、修改、删除单据发票信息?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：6911 | 有用：7
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/393930781493914368)

【问题描述】

如何录入、查询、修改、删除单据发票信息?

【业务场景】

星辰系统中支持对采购入库单、采购退货单、采购发票(旗舰)、销售出库单、销售退货单、委托代销结算单、委托结算退货单、销售发票(旗舰版)、其他支出单、其他收入单等单据进行发票登记、修改、删除

![](https://vip.kingdee.com/download/010941e7143fc7824d72a1fdc7fc98d24e4f.png)
![](https://vip.kingdee.com/download/01095273a1f4f19f415c9f36bc91e99b5c5c.png)
若是有购买开票管理需要通过销售单直接开具税务发票, 可参考:[税务管理如何开具销售发票?](https://vip.kingdee.com/link/s/ZPtnE)

**以下操作说明采购入库单、销售出库单一致,到对应模块操作即可**

**适用于星辰3.1标准版、专业版、旗舰版**

【操作步骤】

1、录入发票信息

1.1  依次点击点击【销售管理】-【销售出库】-【查询】打开销售出库单列表

![](https://vip.kingdee.com/download/0109055572086d984bff9100fc00d3aef869.png)

1.2  勾选已审核未全部开票的单据,点击【销售税票登记】。

![](https://vip.kingdee.com/download/01095a2416b0f68a484989d02d282f83fbad.png)

或者直接进行销售税票登记

![](https://vip.kingdee.com/download/010950da8221e8014102a5ec340a9e23332d.png)
![](https://vip.kingdee.com/download/01094f604200719c4623b518a6dc9032989a.png)
新增税票登记表时,需选择**登记方式、结算供应商 / 客户、币别**;选定币别后,下一步仅会带出**对应币别**的源单。

![](https://vip.kingdee.com/download/01095ffd5f7923ef4c0bb206762fa9d92665.png)

录入好发票信息点击<登记>(基本信息的客户信息、“币别”,税票信息中的“未开票价税合计”来自单据信息中所选单据的相关信息,不支持手工录入修改)

![](https://vip.kingdee.com/download/0109ed948b8cc1e646c8bc87ceeaa2a85e51.png)

会提示登记成功,点击<完成>

![](https://vip.kingdee.com/download/010998104bdf9f2c48ab9f2c6bcb115dc33f.png)

1.3  刷新单据列表即可看到单据上的发票字段信息同步更新

![](https://vip.kingdee.com/download/01093c63e72a08964a3aa2bc32754f15d1e5.png)

**2、查询发票**

2.1  在销售出库单列表,可打开“开票状态”列,查看筛选哪些单据已开票与未开票数据,如单据列表没有对应的字段可查看筛选,可点开列设置检查是否未显示字段:[如何设置单据/报表字段的显示或隐藏?](https://vip.kingdee.com/link/s/ZAvsE)

![](https://vip.kingdee.com/download/0109e7e75c953eb44d72aa8fadfea22e344a.png)

2.2 或者直接点击【销售管理】-【销售税票登记表】进行查看。

![](https://vip.kingdee.com/download/0109bea58cfa3353461c858563bfcf4d3462.png)

**3、修改/删除发票信息:**

**仅手工登记和手工匹配的记录支持修改;暂估调整(采购)和自动匹配不支持修改**

![](https://vip.kingdee.com/download/0109030606370a9b4e699f5272cb65618869.png)

3.1、在单据列表点击已开税票价税合计,可以打开到销售税票登记列表界面;

![](https://vip.kingdee.com/download/010995f8e520c0424ccea404c7607e17e8ba.png)

在打开的销售税票登记列表界面,可以勾选联查到的发票,操作修改删除

![](https://vip.kingdee.com/download/0109d059fdbe320c4f7aba4befabc1894a96.png)

3.2、销售管理-“销售税票登记列表”界面

![](https://vip.kingdee.com/download/0109a4901e1fa75a4c10b6c3c71f177e74f8.png)
选中发票记录,操作修改删除

![](https://vip.kingdee.com/download/0109bf044684a59049e59cfcf6c9d9e97150.png)

3.3、开启了暂估的采购发票修改删除略有不同,需要对暂估调整单进行修改,如需删除发票信息也是删除关联的暂估调整单。

![](https://vip.kingdee.com/download/0109222701ea8b1d4089b01e244b25183940.png)
3.4、自动匹配的发票记录不能修改,可直接删除,重新取票之后再和单据自动匹配,生成税票登记

![](https://vip.kingdee.com/download/010914e3d4b3f1f54dcf98320a4b39db0868.png)

![](https://vip.kingdee.com/download/0109dd36794f8b964406b61ce239ff230c27.png)

**旗舰版**在采购发票和销售发票上进行税票登记:[旗舰版,是否支持对采购/销售发票进行税票登记?](https://vip.kingdee.com/link/s/ZQfjk)

登记后可在销售税票登记表、采购税票登记表进行查询、修改、删除

![](https://vip.kingdee.com/download/0109d473bd79fc6c4677aa8d2d0e60a79151.png)![](https://vip.kingdee.com/download/0109410a13c7ace54f71b7a66e36c49323dc.png)
![](https://vip.kingdee.com/download/01095f10a5653a0e4f5d9c239abef26d294d.png)
旗舰版录的“销售发票”单据需要删除时,在销售发票单据的列表上找到对应单据删除即可(若有审核需先反审核)

![](https://vip.kingdee.com/download/0109de8c7189aa46473692b5f69ba56ca31f.png)

【注意事项】

①业务单据必须审核才能登记税票信息

②一张单据可以多次录入发票信息,但是不能超过总的单据金额

③业务单据是否生成凭证不影响税票登记

④业务模块是否结账不影响税票登记
