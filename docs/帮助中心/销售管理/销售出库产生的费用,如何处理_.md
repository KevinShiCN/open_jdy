# 销售出库产生的费用,如何处理?

【问题描述】

销售出库产生的费用,如何处理?

![](https://vip.kingdee.com/download/0109a11a338b82de4d7a834dec85e9c6147c.png)[视频观看](https://vip.kingdee.com/school/detail/738070012141028096?productLineId=35&lang=zh-CN&cosmic_type=video)

【业务场景】

销售出库过程产生的费用,如运费、搬运费等,如何处理?

销售业务后期或者跨期产生的费用如何处理?

【操作步骤】

**方式一、销售类单据上录入(适用业务发生的同时能确认费用的场景)**

1、如果有销售订单环节,支持在订单上录入预计销售销售费用,销售出库时带入销售出库

![](https://vip.kingdee.com/download/0109179ed02f6ce24656be40e74ed9102d6e.png)

2、本次销售出库产生的费用,在销售出库单的【销售费用】中直接录入。

点击【添加费用】,选择对应供应商、支出类别、金额等。

①支持一次录入多种类型费用。

②部门、业务员自动带出销售出库单表头部门、业务员。

③币别可与销售出库单币别不一致。

![](https://vip.kingdee.com/download/0109ac88dcd36057453790f6aebf80a21f8c.png)

④如费用部分支付或是全部支付,可点击【付款信息】录入实际支付账户和金额信息,生成已支付/部分付款的其他支出单;未录付款信息也会生成其他支出单。未支付的费用默认挂其他应付账款,后续通过【资金】-【付款单】核销。

![](https://vip.kingdee.com/download/0109bfc2f6e63bb74e1c81044944937e2902.png)

3、销售费用分摊。

①录入费用后,可仅【保存】则仅保存记录该笔费用的支出,不参与销售出库单的毛利核算。

②录入费用后,点击【保存并分摊】,保存记录该笔费用的同时,系统会将费用按分摊规则分摊至销售出库单,后续可参与销售毛利核算。

![](https://vip.kingdee.com/download/010904fa5753f8b14d04800549aecfa68553.png)

③用户也可以在商品分录行中,手工修改每个商品分摊的销售费用,但总合计与销售费费用总金额一致。(若该费用已支付或是部分支付不允许手工修改)

![](https://vip.kingdee.com/download/0109d3fa885f83904bdfbfc0b0e9ec535963.png)

④销售费用【保存并分摊】在销售利润表计算毛利时,可以参与销售毛利的计算。如图,勾选”销售总费用“销售毛利减去销售出库单的销售费用,计算最终的毛利。若仅保存未分摊,销售费用无法参考销售毛利计算。

![](https://vip.kingdee.com/download/01093e70a05ddb4342d69b63694c1d3bc145.png)

⑤【分摊规则】是指将销售费用分摊入每条商品行的依据。目前支持两种:价税合计和数量,用户根据需要选择。当本次销售的多种商品,计量单位差异较大,比如有的商品单位是件,有的商品单位是斤,此时不适合选择“数量”的分摊方式,建议选择“价税合计”的方式分摊。

**方式二、通过费用分摊单分摊到对应的业务单据(适用业务发生后才能确认费用场景)**

**【适用版本】星辰专业版、星辰旗舰版**

1、业务发生所属期间已关账 / 结账后,无法直接在原业务单据上追加费用记录;若需核算该笔费用,可在当前期间通过录入 “其他支出单” 的方式单独记录,相关知识:[如何新增其他支出单?](https://vip.kingdee.com/link/s/ZzH0w)

【应收应付】-【其他支出单】

![](https://vip.kingdee.com/download/01092d913c7db5de49a7ac24d15904c90370.png)
2、需要分摊到销售单上时做费用分摊单,具体操作步骤可参考:[费用分摊单的应用场景、功能特性及操作方法](https://vip.kingdee.com/article/568964215587837184?smartSearchResId=581402022201603072&productLineId=35&isKnowledge=2&lang=zh-CN)

说明:专业版直接分摊到销售出库单,旗舰版则是分摊到销售发票

【库存管理】-【费用分摊单】

![](https://vip.kingdee.com/download/0109fe2a8380e3e4487ea3b6de339c6bb497.png)

【其他说明】

1、运费、人工费、停车费等支出类别在【基础资料-支出类别】中新增

2、若未关账未结账的状态下,已审核的销售类单据是否可以修改或是增加销售费用,可在【系统设置】-【系统参数】-【进销存参数】中勾选“允许修改销售费用、客户承担费用、退还客户承担费用、代销费用(已审核销售类单据)”即可。
