# 为什么单据上有“物流信息”，但是找不到填“交货方式”的入口？如何解决？

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题, 快速入门
> 作者：进销存赵嘉丽 | 阅读：862 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/208632800764057856)

# 【问题概述】

为什么单据上有“交货信息”，但是找不到填“交货方式”的入口？如何解决？

![上传图片](https://vip.kingdee.com/download/0100047c0ed5e3dc440e82ab3e98212dfe38.png)

# 【原因说明】

出现这种情况是由于“交货方式”没有开启“显示列”导致的。

# 【解决方案】

打开单据表头的列设置，找到“交货方式”，开启“显示列”，单击“确定”即可。

![上传图片](https://vip.kingdee.com/download/010043520a72435d40ee80bc41688811a613.png)

开启后即可正常显示：

![上传图片](https://vip.kingdee.com/download/0100badfa70a09d948018a3cf65b05c81aeb.png)

# 【适用版本】

星辰3.1以上所有版本
