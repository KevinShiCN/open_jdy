# 出入库开单时如何选择商品?

【问题描述】

出入库开单时如何选择商品?

【解决方案】

方法一:开单可以直接按商品编码/商品名称/规格型号等模糊搜索选择开单。

![](https://vip.kingdee.com/download/0109b0988218fa88446299eaa8ff5a40f696.png)
方法二:录单打开更多商品信息界面

![](https://vip.kingdee.com/download/0109e14667bf880b459fb897996b28c26f0d.png)
跳转到商品信息列表可以按商品类别/按条件搜索筛选商品录单。
![](https://vip.kingdee.com/download/0109dff0d3a346974d5db1616937ea74a7e7.png)
