# 单据上的税率怎么取? 不想启用税率怎么设置?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：3096 | 有用：11
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/129339770097650400)

【问题描述】

单据上的税率怎么取? 不想启用税率怎么设置?

【解释说明】

**开单时税率的取值顺序依次是**:

| 优先级 | 取值来源 |
| --- | --- |
| 1 | 商品资料里设置的进项和销项税率 |
| 2 | 客户资料(销售类单据)或供应商资料(采购类单据)里设置的税率 |
| 3 | 系统参数—基础资料参数里设置的默认税率 |

采购类单据除以上优先级外,支持在基础资料参数中单独设置税率取值的优先级。

如果不想启用税率,可以将【系统设置】-【系统参数】-【基础资料参数】“默认增值税税率为”里的税率设置为空,同时在新增商品、客户/供应商时也不要录入税率。

【操作流程】

1、开单时税率优先取【商品信息】里设置的进项税率(采购类单据)/销项税率(销售类单据);

路径:【基础资料】-【商品信息】列表,【批量修改】或是打开商品编辑界面录入进项或者销项税率。

![](https://vip.kingdee.com/download/010987ff71feee794da492ee1b8516cf4ef8.png)若商品设置了免税,则销售开单带出税率为空

![](https://vip.kingdee.com/download/01095b0b9ae1b6db4c41bd050bf73dece4c8.png)

2、商品未设置税率,则取客户资料中设置的税率(销售类单据)或供应商资料(采购类单据)里设置的税率;

路径:【基础资料】-【客户|列表】/【供应商|列表】,点击【批量修改】设置税率;
![](https://vip.kingdee.com/download/01092920931b68744ec38e2f147713613eee.png)
3、商品资料和客户/供应商资料中均未设置税率,则取【系统参数】中设置的默认增值税税率;

路径:【系统设置】>【系统参数>【基础资料参数】

![](https://vip.kingdee.com/download/0109d94e8f5f2ecd41a0ac0ed6933f186fe0.png)
采购类单据除以上优先级外,还需考虑参数中是否有设置采购类单据税率取值优先级,若有设置,按参数设置中的优先级设置取值

![](https://vip.kingdee.com/download/01097cc207fc8b974b3eb0b3aa8be486204f.png)
