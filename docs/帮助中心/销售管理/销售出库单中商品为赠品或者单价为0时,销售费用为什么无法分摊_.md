# 销售出库单中商品为赠品或者单价为0时,销售费用为什么无法分摊?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：ylim | 阅读：103 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/654851895521065984)

【问题描述】

销售出库单中商品为赠品或者单价为0时,销售费用为什么无法分摊?

【解释说明】

若商品为赠品或者价格为0时销售费用无法分摊,一般是因为分摊规则设置的按价税合计分摊,需检查并修改销售费用的分摊规则。

【操作指引】

点击销售出库单销售费用明细旁的<编辑费用>按钮

![](https://vip.kingdee.com/download/010998f00ba1b87c48ea9be2e34c5eced0ba.png)
费用分摊规则选择“数量”,点击<保存并分摊>按钮

![](https://vip.kingdee.com/download/01094c6a293abc7c491b98db47d8cb9000cb.png)
销售费用即可被分摊上

![](https://vip.kingdee.com/download/01098e7b3b72d11347efab99d74fcb745771.png)
