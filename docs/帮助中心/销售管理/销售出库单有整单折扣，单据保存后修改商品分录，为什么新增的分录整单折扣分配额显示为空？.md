# 销售出库单有整单折扣，单据保存后修改商品分录，为什么新增的分录整单折扣分配额显示为空？

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：快速入门, 常见问题
> 作者：进销存赵嘉丽 | 阅读：952 | 有用：4

# 【问题概述】

销售出库单有整单折扣，单据保存后修改商品分录，为什么新增的分录整单折扣分配额显示为空？

![上传图片](https://vip.kingdee.com/download/0100bcf987b7485341d0b11f766f3eff5ceb.png)

# 【原因说明】

录销售出库单时，当把所有分录录入好之后，填写整单折扣，系统会自动分摊一次，然后单据保存后，数据就存库了。

# 【解决方案】

若单据**已保存后**新增分录，那么新增的分录行的整单折扣分配额**不会自动分配，而是显示为空**，需要手动点击“分摊”重新分摊到所有分录中。

![上传图片](https://vip.kingdee.com/download/01002f6429651abd447a947156272e3f0c24.png)

需要注意的是，如果新增分录后，手工修改了“整单折扣额”，此时系统会自动重新分摊到所有分录的“整单折扣分配额”，因为源头已经发生变化，所以分录上的值也会自动更新。

![上传图片](https://vip.kingdee.com/download/0100b550013f6b0d4f3faad269a4b6cfacfb.png)

# 【适用版本】

星辰3.1以上所有版本
