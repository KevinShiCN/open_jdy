# **1 简介 **

## **1.1 功能介绍**

在对销售出库单开单时，系统提供金额计算参数，会根据用户当前账套的系统参数，展示当前单价是否为含税单价，同步系统参数的设置显示，同时提供了若干反算的参数，灵活满足不同客户的计算规则。

## 1.2 历史优化说明

2025年3月27后，星辰整合了销售出库单原“选项”中有关金额类字段相关的参数。优化前的背景与现状：

在开单时，若用户对系统计算出来的税额、价税合计等字段存在疑义，通常需要先检查相关的设置，包括：

① 系统参数中的“商品价格为含税单价”

![](https://vip.kingdee.com/download/01005b1fb56dd09843d9bb44c3ba849ccf0e.png)

② 单据“选项”中的“修改价税合计后反算单价”、“修改税额重算价税合计”；

![](https://vip.kingdee.com/download/01002ecaf7ccf65945ee8d20aa95617606fa.png)

③ 由于系统计算是根据存储的单价金额计算的，不是显示的单价精度，计算结果与期望结果可能存在差异。

基于优化前的问题，为了降低用户歧义，改善开单体验，产品侧对原有选项中的参数进行了整合。

## **1.3 适用版本**

星辰3.1以所有版本

## **1.5 字段/按钮说明**

- 该参数是按账套级别生效的，也就是说，同一个账套内，A用户设置后，会影响BCD...等用户的操作。

# **2 功能说明**

- **设置入口**

历史销售出库单“选项”中的"修改价税合计后反算单价"、"修改税额重算价税合计"整合后的参数在销售出库单商品信息栏“金额计算参数”中：

![](https://vip.kingdee.com/download/010011a90ef032eb4374b4e0db98636fd14c.png)

- **示例说明**

点击“示例说明”，可查看开启不同参数，对金额计算的影响：

![](https://vip.kingdee.com/download/0100684d391fedde4431aa1f08c1ce14eb30.png)

- **偏好设置**

因为录入含税价和不含税价，计算出来的价税合计可能出现精度差异，建议开启录入偏好设置。开启后，含税类字段和不含税费字段，只可录入其中一种，另一种不允许编辑以保持数据计算的一致性。

![](https://vip.kingdee.com/download/010059f8d50eaa0e49ffb1c657caadfce6ce.png)
