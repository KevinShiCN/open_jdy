# 委托代销功能如何使用?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：云社区用户8x80607 | 阅读：2026 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/617524527253353984)

**【问题描述】**

委托代销功能如何使用?

**【解释说明】**

‌**‌**‌委托代销是指‌受托方按照委托方的要求销售委托方的货物,并收取手续费的经营活动。在这种模式下,受托方作为一个代理商,不拥有货物的所有权,而是代表委托方进行销售。

| 功能模块 | 功能特性 | 课程链接 |
| --- | --- | --- |
| 委托代销期初余额 | 在使用星辰系统前,已经将商品发出,库存已经减少,后续若需要在星辰系统中处理结算/退货业务,需要将以前未结算完的委托出库信息登记在系统中;不会再次减少库存。 | 点击学习 |
| 委托代销 | 某公司把自己的商品委托给下游经销商或门店销售,货权没有转移,下游经销商或门店根据实际销售的数量和金额与公司结算。 | 点击学习 |
| 委托代销结算 | 登记下游反馈的实际销售数据,用于跟下游经销商结算应收回的货款,及计算还剩余多少未销售完 | 点击学习 |
| 委托代销退货 | 下游经销商将商品发还到公司仓库 | 点击学习 |
| 委托结算退货 | 与经销商已结算完成后,若终端客户因质量问题退货,损失由本公司承担时,可对委托结算单进行退货,减少应收账款 | 点击学习 |
| 委托商品跟踪表 | 查询本公司发给下游经销商的每种商品数量,已结算数量,已退货数量,未结算数量 | 点击学习 |
| 委托库存余额表 | 查询本公司发给下游经销商代销的商品还有多少没有结算完 | 点击学习 |

温馨提示:注意:委托结算单,委托代销退货单,不能手工新增,只能由委托代销单或委托代销期初余额关联产生。
