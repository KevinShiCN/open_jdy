# 星辰2.0,销售订单/销售出库单中整件散包如何拆分显示和打印?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：1000 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/129339770080872963)

【问题描述】

销售订单/销售出库单中整件散包如何拆分显示和打印?

【操作流程】

1、销售订单/销售出库单中将整件数、散包数显示出来,打开单据编辑界面,点右上角列设置,整件数、散包数选择显示,可根据实际情况选择是否显示。

![](https://vip.kingdee.com/download/0109dddd1ccede08467983ad37e81295a449.png)

2、套打模板中设置好对应列值取数,并设置好合计数;

![](https://vip.kingdee.com/download/01092c9ac381d933496782894e8136cc6971.png)

![](https://vip.kingdee.com/download/010943fed43e2f9141cb8f4f3d3e61900d92.png)

3、打印模板设置后可打印已保存好的单据,打印预览效果如图:

![](https://vip.kingdee.com/download/0109151e31c7a7a34009800104e014a311c7.png)
