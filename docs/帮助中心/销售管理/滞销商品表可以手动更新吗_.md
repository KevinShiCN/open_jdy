# 滞销商品表可以手动更新吗?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题

【问题描述】

滞销商品表可以手动更新吗?

【解释说明】

**不支持手动更新。**报表数据非实时更新,默认每晚执行定时任务自动更新。
