# 销售报价与模拟报价的区别

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：春日负暄 | 阅读：4 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/820100051434089984)

【问题描述】

销售报价与模拟报价的区别

【解释说明】

1、【销售报价单】属于销售管理模块,用于对外客户报价,按价格优先级取数,取数来源较单一,且仅可针对销售的成品进行取价。可查看历史报价并直接下推生成销售订单。

【销售报价单】具体说明可参考:[销售报价单:](https://vip.kingdee.com/link/s/ZROfK)

2、【模拟报价】属于生产管理模块,基于 BOM、材料成本与委外费用进行内部成本测算、预估定价,取价来源更多样,可算明细成本,可关联【销售订单】 / 【销售预测单】取数,可选销售订单 / 预测单源单,可下推销售报价单

【模拟报价单】价格取数具体可参考:[模拟报价如何使用:](https://vip.kingdee.com/link/s/Zo6bd)![](https://vip.kingdee.com/download/010961cedac875a447218927313ebd060b33.png)
关联:【模拟报价单】可下推生成【销售报价单】。
