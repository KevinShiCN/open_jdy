# 单据打印预览动态走纸显示0页,打印缩小或不完整是什么原因?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：红笺小字 | 阅读：46 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/787485138782958592)

【问题描述】

单据打印预览动态走纸显示0页,打印预览设置为适合可打印区域就靠左缩小显示;

![](https://vip.kingdee.com/download/0109e4b1d7b00c244148b9422373472a49d3.png)

取消适合可打印区域的勾选,打印预览显示商品信息不完整,如图所示

![](https://vip.kingdee.com/download/01091d25630f8ef64c4f9b418e0bfa6ce354.png)

【解释说明】

该情况属于设备问题。打印机有最大纸张的限制,是打印机最大纸张长度不够,限制了动态走纸可打印的最大长度。导致打印预览不全或缩小显示。

预览界面动态走纸字体显示为红色时就是不支持动态走纸打印。

若要用动态走纸,建议把单据分多张录入。
