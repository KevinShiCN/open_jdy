**1、录单保存提示： 第x行分录：辅助单位【x】是固定单位，与商品资料上的单位类型不一致，请重新录入是什么意思？**

【故障现象】

录单保存提示：“ 第X行分录：辅助单位【X】是固定单位，与商品资料上的单位类型不一致，请重新录入”是什么意思？

【原因】

商品一开始是固定多单位商品

![](https://vip.kingdee.com/download/010050fdffbb200746a7a7fa359aacfdb696.png)

然后录单了，选择了这个商品，但是单据还没保存

![](https://vip.kingdee.com/download/0100f88e4a2c35694fa8942a8a2682c64e86.png)

在单据还没保存时，又去修改了商品资料，勾选了浮动换算，并且保存成功。

![](https://vip.kingdee.com/download/01008d0d828fe86f4ef998f938101660a263.png)

那么这个时候去保存单据时就会提示“第X行分录：辅助单位【X】是固定单位，与商品资料上的单位类型不一致，请重新录入”，表示校验失败，不允许保存。

![](https://vip.kingdee.com/download/01003f2d494b206444f2a0e42c8636c2a220.png)

【解决方案】

先确认好商品到底是要浮动换算还是固定换算后，删除单据上已经录入的商品行，重新录入商品，再重新保存单据即可。

![](https://vip.kingdee.com/download/01005d7f81fc64574d6284f441953995de9e.png)

**2、销售出库单可以显示销售出库成本吗？**

【业务场景】

可以在销售出库单中显示销售出库成本吗？

【解释说明】

销售出库成本是需要通过出库成本核算后计算出的，在库存管理进行出库成本核算后才能查看销售出库单中的成本信息。

1、进行出库成本核算

审核销售出库单后需要在【库存管理】-【出库成本核算】中进行出库成本核算。

2、销售出库单显示成本字段

在销售出库单列设置中显示成本字段即可查询销售出库成本。

![上传图片](https://vip.kingdee.com/download/01004cc9ec6b0edf4861954be0f3ffb3bdde.png)

![上传图片](https://vip.kingdee.com/download/010040896fde2cdb42a299f8aed73cf44c85.png)

3、打印销售出库成本

若打印销售出库单时需要打印销售出库成本时可以在打印模板中添加销售成本字段。

![上传图片](https://vip.kingdee.com/download/0100218fb9bee2ea4bb0bc44e23dafb291f0.png)

**3、销售订单如何只选择部分商品行进行出库？**

【业务场景】

销售订单如何只选择部分商品行进行出库？如：一个销售订单有10个商品分录，如何只选择其中2行商品生成销售出库单？

![上传图片](https://vip.kingdee.com/download/0100fefc31c2d9f747fd8c1a05c700bf9f3e.png)

【解决方案】

单据列设置中启用“同一单据拆分显示”功能即可。

在销售订单列表界面，点击左上角列设置，启用“同一单据拆分显示”，点击确定后同一个单据的商品分录拆分显示，选择需要生成销售出库单的商品行点击“销售出库”即可。

**4、销售管理经营日报中的流失预警客户怎么计算的？**

【解决方案】

经营日报中的流失预警客户根据系统参数中【客户大于*天没有回购，加入到客户回购异常风险预警名单】参数统计。

【操作流程】

依次点击【系统设置】-【预警设置】-【客户预警】中的【客户流失预警】参数控制，如将其设置为60天，则60天没有销售记录的客户会计入流失预警客户名单中。

**6、滞销商品表的应用？**

【说明】

方法1：通过【销售管理】-【经营分析】-【经营日报】中的“库存情况”点击查看“滞销商品”的概况，点击对应的数字可联查到【滞销商品表】；

方法2：直接通过【销售管理】-【经营分析】-【滞销商品表】查看。

【应用价值和场景】

1、通过滞销商品表，可快速了解商品的周转速度和库存积压情况，包括无销售天数，最近销售日期、入库日期、库存数量、金额等等，以便及时调整商品的销售策略，如通过促销活动加速库存周转和消耗；

2、如食品批发零售、加工、餐饮等行业，还可以结合保质期预警查询表一起使用，以便了解原材料或者半成品的积压情况及效期剩余时间，以便合理安排先进先出以及库存清理的工作，避免库存积压过多，效期变短带来的库存货值损失。

3、通过滞销商品表，结合销售分析表，可以对商品的整体销售效果有预判，通过筛选了解各商品类别、品牌下滞销商品的分布情况。

应用流程：

1、在【系统参数-预警设置-商品预警】中设置滞销商品的自定义天数：

![上传图片](https://vip.kingdee.com/download/01009511ab38fb8d403a8b1aed3d2e0492b9.png)

注：关于滞销商品的统计范围：

1）统计范围：当前库存余额大于0的商品（不包含已禁用商品）

2）在所选时间范围内增加过库存的商品不统计在内（以下单据除外：销售退货，调拨入库，移仓入库）

在【销售管理】-【经营分析】-【滞销商品表】查看具体的滞销商品汇总情况

**7、为什么商品上维护了产地，注册证号、生产许可证号，但是销售订单/销售出库单的时候，自动带出信息跟商品资料上的不一样？**

【解释说明】

商品上维护了产地，注册证号、生产许可证号，但是销售订单/销售出库单的时候，自动带出信息跟商品资料上的不一样问题与产地、注册证号、生产许可证的自动携带规则有关。

目前，销售订单/销售出库单默认携带最近一次已审核采购入库单时商品对应的产地，注册证号、生产许可证号信息。

【应用场景】

医疗器械行业、食品行业，采购入库时需要记录来货商品的产地、生产许可证、注册证号等信息，并希望后续开单时，默认携带最近一次入库上的信息，以减少人工录入的工作，提高开单效率。

【携带逻辑】

一、新增销售订单/销售出库单携带说明：

• 如果有已审核的采购入库单，取最近已审核的采购入库单上的信息，如果最近已审核入库单上的产地信息为空，则携带更早一次采购入库单上的信息，允许手工修改；

• 如果没有任何已审核的采购入库单，携带商品资料上的（如果为空，则不携带），允许手工修改；

• 如果单据是复制的，则携带被复制的单据上的信息。

二、 功能特性：

（1）产地，注册证号、生产许可证号其实类似于商品价格资料里的最近采购价、最新销售价，是取最近一次已审核的入库单的，因此一旦新增或删除已审核的采购入库单，这个存储的值都会变。

（2）产地，注册证号、生产许可证号三个字段是一组整体的信息来存数据库的，只要单据上其中一个字段有更新，一整组数据都会更新，如果一整组都是空，则不更新，系统还是存上一组有值的数据。

举例：

某商品库存查询表显示的产地：深圳、注册证号：SZ001、生产许可证号：SZ001

9月1号做采购入库，产地为：深圳、注册证号：SZ001、生产许可证号：SP00062

（3） 批次商品

 销售出库时，针对批次商品，携带上一次采购入库单时批次信息上对应的产地，注册证号、生产许可证号，也就是会按批次维度携带这些信息。

批次信息弹框里，增加产地产地，注册证号、生产许可证号的信息显示，方便用户查看。
