# 如何设置销售订单是否支持超额出库?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：1490 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/129339770080872965)

【问题描述】

如何设置销售订单是否支持超额出库?

【解释说明】

销售订单生成销售出库单、销售发票(旗舰版)是否允许数量大于订单数量,是根据系统参数中”允许出库数量大于订单数量““允许出库数量及开票数量大于订单数量”(旗舰版)的参数来控制的;且**支持设置按比例控制超发。**

【解决方案】

依次点击【系统设置】-【系统参数】

![](https://vip.kingdee.com/download/010950231e1aaebe475e91b1c5ebf0c99d4d.png)

【进销存参数】-【销售参数】中不勾选“允许出库数量大于订单数量”“允许出库数量及开票数量大于订单数量”表示销售出库数量不能超过销售订单数;

若允许销售出库单的数量大于销售订单的数量出库时,则需要勾选系统参数,勾选后可以设置“按比例控制超发”

**专业版和标准版参数截图**

![](https://vip.kingdee.com/download/0109e764ac1b1bf74ebdb58b408550954cd7.png)

**旗舰版参数截图**

![](https://vip.kingdee.com/download/010988b1989efbc04143a6590e647e25a328.png)

1、不勾选参数,销售出库单、销售发票(**旗舰版**)大于订单数量时:

销售出库单会提示不允许超额出库

![](https://vip.kingdee.com/download/010941be2df32e41468b907733aa778e4347.png)
销售发票会提示不允许超额开票

![](https://vip.kingdee.com/download/0109e7d37d33a94b429fb8dac64019edae5c.png)以上操作针对“未关闭”和“未完成”的销售订单,“已关闭”和“已完成”的订单不支持再次出库。

2、勾选允许出库数量大于订单数且勾选了“按比例控制超发”,可在商品信息的其他信息中设置销售订单超发比例(若未显示,点击商品右上角的列设置按钮开启字段显示)

![](https://vip.kingdee.com/download/0109b7d919a9ac8544ad8ac30f90fb7f8573.png)
在录订单时可在商品分录行设置开启“控制发货上限”、“超发比例(%)”、“发货上限”字段的显示

![](https://vip.kingdee.com/download/0109e38ae1a8c46f42cc9f8600cc9adbee8a.png)

录入商品后,开启商品行的“控制发货上限”,维护“超发比例(%)”,系统会自动算出“发货上限”,(超发比例按商品中设置的带出,也可直接在单据上修改)其中:

发货上限=数量*(1+超发比例)

发货上限(基本):表示基本单位对应的发货上限数量。

![](https://vip.kingdee.com/download/0109ffbbf4af3a584b628ad8f96545dbe92b.png)

当订单下推出库,出库数量大于订单上限时,单据保存失败,并提示超过发货上限

![](https://vip.kingdee.com/download/0109410666fbe6514a09a058752892681ff1.png)
