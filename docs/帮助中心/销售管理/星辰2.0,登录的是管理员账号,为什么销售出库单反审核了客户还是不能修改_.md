# 星辰2.0,登录的是管理员账号,为什么销售出库单反审核了客户还是不能修改?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：云社区用户5Uv15796 | 阅读：187 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/534899806972199424)

【业务场景】

金蝶云星辰,登录的是管理员账号,为什么销售出库单反审核了客户还是不能修改?

【故障原因】

单据有抵扣预收款,所以存在关联单据,如图:

![](https://vip.kingdee.com/download/0109e84042c848a94843a61d6662a838990d.png)

【解决方案】

先删除抵扣预收款,然后再修改,就可以修改客户了

![](https://vip.kingdee.com/download/0109aa9595a7383e4879961b55f2bd23d7b1.png)
如图,删除后修改,是可以选择到客户

![](https://vip.kingdee.com/download/010909cea827e63b41099e66325fb9769d31.png)
