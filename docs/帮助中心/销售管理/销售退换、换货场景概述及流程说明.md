# 销售退换、换货场景概述及流程说明

# **1 简介**

## **1.1 应用场景**

- **退货**：客户的实际需求取消，商品不符合验收标准等其他客户需退货的场景。
- **换货**：客户的实际需求发生改变（如规格型号改变），或产品有质量问题、效期问题等其他需换货原因。换货支持三种典型场景：

**1） 先退回再换出**

客户换货需要提前申请，先将原商品退回再发出新的商品，类似淘宝换货。例如：五金行业，定制商品。

**2） 先换出后退回**

仓库重新发货，客户收到后再将需更换的货物退回仓库，适用于**长期合作的客户**换货**，**支持先发出新品，再退回旧品。

**3） 直接换货**

适用于**买卖双方现场直接交货**的场景。

## **1.2 功能特性**

结合系统参数在审核销售换货申请单时，自动生成销售出库单、销售退货单，可以实现销售换货申请单审核后，自动生成销售出库单、销售退货单。

## **1.3 适用版本**

星辰3.1以上所有版本
