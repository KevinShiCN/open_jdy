# 在列表选中出库单做销售税票登记的自动匹配提示“暂无可匹配的税票”，但手动匹配税票是正常的可以选中这笔发票的，这是为什么呢？

# **1 问题描述**

销售出库单的商品行税票状态显示“未开票”，直接在列表选中出库单做“自动匹配”，提示“暂无可匹配的税票”，但手动匹配税票却可以选中这笔发票，这是为什么呢？

# 2 适用版本

星辰3.0及以上所有版本

# **3 原因解析**

通常是因为存在不符合自动匹配规则的情况。例如，单据商品行未开税票价税合计≠税票商品行未核销金额，此时只能选择手工匹配税票。

上述问题描述中，销售出库单的价税合计是900，如果税票商品行未核销金额不是900，这种情况下就无法实现自动匹配。

# **4 解决方案**

在不满足自动匹配条件的前提下，只能采用手工匹配方式登记税票。
