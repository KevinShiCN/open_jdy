# 采购 / 销售订单选定币别后,生成的下游单据能否修改币别?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：Siwennn | 阅读：197 | 有用：0

【问题描述】

采购 / 销售订单选定币别后,生成的下游单据能否修改币别?

【解释说明】

下游单据的币别无法修改。

由订单下推生成的入库单、出库单,会自动同步源订单的币别信息;若需调整币别,需先修改订单的币别参数,再重新下推生成下游单据。
