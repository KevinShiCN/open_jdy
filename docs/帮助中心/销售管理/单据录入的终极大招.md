# 单据录入的终极大招

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：知识软文, 快速入门
> 作者：杨桂平 | 阅读：1046 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/333242391995988224)

![](https://vip.kingdee.com/download/01090838362cd5af40a0867cadf0a0fdaab6.png)

书接上文，高翠兰手工录入销售出库单，由于单据商品较多、分录数量冗长，不慎将单据录入错误，导致发错货物给到客户，老员外恨铁不成钢，痛批了女儿一顿，恰逢大师兄与小白龙一道前来和八戒小聚，便出手解围教了翠兰一招准确无误且快速录入的小妙招。

高府一座雅致的凉亭内，八戒师兄弟三人及高翠兰一同落席，侍女送来了美味佳肴，几次推杯换盏后八戒开口问道：“大师兄，对于单据录入，你还有什么别的妙招没有？”

大师兄悠悠的放下了酒杯：“不知足的呆子，脾气秉性一点都没改，也不怕一口气吃成了个胖子。”

高翠兰笑骂道：“他可不就是个胖子，哈哈，大师兄您教教我吧”

众人皆是啼笑皆非，随即大师兄便给高翠兰讲了一个单据录入的终极大招——单据引入

1、 以销售出库单为例，在【销售管理】-【销售出库单】鼠标点击【查询】按钮，进入到【销售出库单列表】；点击右上方【引入】按钮，弹出引入数据的对话框，再点击对话框下方【模板管理】，选择一个引入的模板点击下载，把引入的模板下载下来

![](https://vip.kingdee.com/download/0100832a9034f8584f798ec5de9e56fda1e5.png)

![](https://vip.kingdee.com/download/01006a22de0ea2674c998a34c0326fa63966.png)

2、 打开下载好的模板，到模板中填写相关数据。这里要注意的是，引入文件必须符合模板格式，下载好的模板不要删减改变它任何的格式，只需将相关数据填写或复制粘贴原有数据到模板里，具体的填写还可以点击下载【引入填写示例】，依葫芦画瓢

![](https://vip.kingdee.com/download/0100720113d124484c29acbef0f4e5517824.png)

3、 填写好模板后，回到软件销售出库单列表的引入数据对话框，可点击【上传文件】或【点击或拖拽上传】将刚才处理好的模板上传到系统，点击【开始引入】等待引入完成即可；

![](https://vip.kingdee.com/download/0100d99d61b445f44da0bd952b5319419753.png)

听完如此简单又神奇高效的单据录入方法，八戒连忙给大师兄斟满了酒，与大师兄碰杯后一饮而尽：“感情深一口闷，老猪我不太会说话，都在酒里了”

大师兄也不和他计较，又补充道：“具体的操作，我这边还有操作视频讲解，有空看看学习一下”
