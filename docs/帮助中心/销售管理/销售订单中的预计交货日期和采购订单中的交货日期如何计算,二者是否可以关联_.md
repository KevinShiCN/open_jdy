# 销售订单中的预计交货日期和采购订单中的交货日期如何计算,二者是否可以关联?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：3475 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/398279495624647424)

【问题描述】

销售订单中的预计交货日期和采购订单中的交货日期如何计算,二者是否可以关联?

【解释说明】

**销售订单**中的商品行**预计交货日期**是根据单据日期加上商品信息中的“**交货周期(天)”**计算得出;**采购订单**中的商品行**交货日期**是根据单据日期加商品信息中的**“备货天数”**计算得出的。

商品信息里没有设置“交货周期(天)”和备货天数”时,销售订单的预计交货日期和采购订单的交货日期默认等于单据日期,**可手工修改,不能早于单据日期**。

二者没有关联关系,不能互相反写。

【操作指引】

1、在【商品信息】中,将“交货周期(天)”和备货天数”字段显示出来(若有启用审核且商品已审核需先反审核)

![](https://vip.kingdee.com/download/0109fc084cf1e92b4c1cbfffc7c8dce114be.png)

2、录入好该商品的“交货周期(天)”和备货天数”保存(若有启用审核需审核)

![](https://vip.kingdee.com/download/010954b2a66e5d6d4ee7a5b84078ac5b331d.png)

3、在销售订单中将单据日期填写好之后,输入商品行信息后,预计交货日期会自动计算,当然也可以根据需要手动进行修改
![](https://vip.kingdee.com/download/010913a922ee8c7f4de783ac9da70da41141.png)
采购订单的交货日期同理根据单据日期+备货天数计算,可手工修改

![](https://vip.kingdee.com/download/01092a60a043d76b4951aeaeaa3cf812251e.png)
