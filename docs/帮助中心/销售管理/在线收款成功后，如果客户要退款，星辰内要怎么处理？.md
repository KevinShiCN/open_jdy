# 在线收款成功后，如果客户要退款，星辰内要怎么处理？

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题, 进阶技巧
> 作者：echoquelqjisf | 阅读：48 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/789449628705384448)

# **1 问题描述**

在线收款成功后，如果客户要退款，星辰内要怎么处理？

目前星辰暂时还没有对接支付平台的退款流程，收款退款单也无法关联源单，要怎么操作呢？

# 2 适用版本

星辰3.0以上标准版/专业版

# **3 原因解析**

上述问题是因为星辰还没有对接支付平台退款功能而引起的，星辰内建议使用【其他收入退款单】；后续星辰随着迭代增强在线收款功能后，将会提供更流畅的体验

# **4 解决方案**

1）第一步，前往对账平台，操作账户退款

【金蝶支付】

金蝶支付对账平台网址：[https://www.kingdeepay.com/merchant/](https://www.kingdeepay.com/merchant/)

【通联支付】

- 通联商户对账平台网址：[https://www.vsp.allinpay.com](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fwww.vsp.allinpay.com)
- 通联公众号：微信搜索“通联收银宝”
- 微信公众号或微信小程序后台登录地址：[https://mp.weixin.qq.com](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fmp.weixin.qq.com)

2）第二步，在星辰内新增【其他收入退款单】，手工完成登记
