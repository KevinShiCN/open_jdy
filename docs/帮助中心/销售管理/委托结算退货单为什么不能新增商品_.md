# 委托结算退货单为什么不能新增商品?

> 作者：咚咚 | 阅读：46 | 有用：1

【问题描述】

委托结算退货单为什么不能新增商品?

![](https://vip.kingdee.com/download/010908ebd6f5f36c4fc598300b67cc9f1564.png)

【解释说明】

委托结算退货单不能手动新增,只能通过委托代销结算单“结算退货”或者委托结算退货单新增界面选源单。
