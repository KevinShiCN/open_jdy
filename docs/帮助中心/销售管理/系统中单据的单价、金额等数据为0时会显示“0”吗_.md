# 系统中单据的单价、金额等数据为0时会显示“0”吗?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：小朱呀 | 阅读：79 | 有用：1

【问题描述】

系统中单据的单价、金额等数据为0时会显示“0”吗?

【解释说明】

系统后台统一控制显示规则是业务单据字段内容为0时会显示为空,目前不支持设置显示为0
