# 采购销售汇总表

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：快速入门, 精华汇总, 功能介绍
> 作者：进销存赵嘉丽 | 阅读：500 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/616648790786628096)

1 简介

### 1.1 功能介绍

统计一段时期内商品的采购和销售的总体情况，可选择全部商品、仓库进行统计，也可挑选部分商品、仓库进行统计，数据来源包含已审核和未审核的采购入库单，采购退货单，销售出库单，销售退货单。

### 1.2适用版本

金蝶云·星辰所有版本

### 1.3系统路径

销售管理-销售单据-采购销售汇总表

### 1.4 字段/按钮说明

| 字段类型 | 取数逻辑 |
| --- | --- |
| 数量类 | 基本数量（采购）: 采购入库单上商品按基本单位换算的数量基本数量（销售）: 销售出库单上商品按基本单位换算的数量退货基本数量（采购）：采购退货单上商品按基本单位换算的数量退货基本数量（销售）：销售退货单上商品按基本单位换算的数量基本数量（期末）=基本数量（期初）+基本数量（采购）-退货基本数量（采购）-基本数量（销售）+退货基本数量（销售） |
| 金额类 | 采购金额：采购入库单上的商品的价税合计退货金额（采购）：采购退货单上商品的价税合计销售金额：销售出库单上的商品的价税合计退货金额（销售）：销售退货单上商品的价税合计 |
| 成本类 | 销售成本：销售出库单上总的出库成本-销售退货单上总的出库成本期末库存成本=期初成本+采购金额-退货金额（采购）-销售成本 |
| 单价类 | 进货均价（基本）=采购金额/基本数量（采购）销售均价（销售）=销售金额/基本数量（销售）进销存差价=销售均价（销售）-进货均价（基本）库存均价=期末库存成本/基本数量（期末） |

注意：报表中展示的期初库存与期末库存为查询起止时间的实际库存。

## 2 操作指南

### 2.1 单据来源

已保存/已审核的销售出库单、采购入库单、销售/采购退货单

### 2.2 查询说明

1）查询条件及界面

①查询条件

如果有选择器按钮（三个点号）或者下拉，可以直接选，如果不选择，为空表示查找所有的条件，例如：客户、商品、仓库如果空表示查找所有客户、商品、仓库，可分别打开各选择框进行勾选；

②单据日期

日期范围可以任意调整；默认为本月的第一天到当天；

**③筛选设置**

- **按辅助属性统计**：当不勾选“按辅助属性统计时”，以主商品纬度统计数据；当勾选“按辅助属性统计”时，启用辅助辅助属性管理的商品，会具体到某属性组合的纬度统计数据；
- **分仓库统计**：当不勾选“分仓库统计”时，查询结果中不显示仓库，汇总了所有仓库的采购销售及库存数据；当勾选了“分仓库统计”时，查询结果按仓库排序，显示每个仓库中每个商品的采购销售及库存数据；
- **无采购销售不显示：**勾选则表示在查询范围内，若某商品没有采购和销售记录，则不显示该商品的信息；
- **期末库存为0不显示**：勾选则表示在查询截止日期时，某商品的库存数量为0，则不显示该商品信息。
- **销售毛利 = 销售收入 - 销售成本-![](https://vip.kingdee.com/download/0109d39feb24a4f2497f82abfcb89e39bf96.png)销售总费用**：如果销售毛利希望剔除销售总费用，则勾选“销售总费用”，默认不勾选。

## 3 常见问题

### 1）采购销售汇总表中的销售均价（基本）与销售价格分析表中的平均价格有什么区别？

答：

**计算逻辑：**

两个字段的计算维度是不一样的，最大的差异是，第一个公式是考虑了销售数量的，第二个公式是不考虑销售数量的。

- 销售均价（基本)= 销售金额 / 基本数量（销售）
- 平均价格 = 每次交易单价的合计/ 交易记录数（如果同一张单中有多行，会按行数统计）

**适用场景**：

- 销售均价（基本）更适合用于长期的价格趋势分析，或者在比较不同时间段内产品价格稳定性时使用。
- 平均价格 更适合用于分析促销活动或特殊交易条件对价格的影响，以及在评估市场接受度和价格敏感性时使用，

**应用意义**：

- 销售均价（基本）更适用于库存管理和成本控制，因为它关注的是单位产品的销售价格。
- 平均价格更适用于销售分析和定价策略，因为它关注的是交易层面的价格。

总的来说，选择使用哪个指标取决于分析的目的和需要解决的具体问题。在实际应用中，这两个指标可以提供不同的视角来理解和分析销售数据。
