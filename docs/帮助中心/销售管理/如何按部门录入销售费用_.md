# 如何按部门录入销售费用?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：岸边把酒 | 阅读：214 | 有用：5
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/660649245866476288)

【问题描述】

如果销售费用需要分部门进行填写,应该如何操作?

【解释说明】

1、首先在销售费用录入界面,点击右上方“齿轮”图标,进入费用列表列设置界面。

![](https://vip.kingdee.com/download/0109a78e0bf4e5c644288618c24649ead177.png)

2、然后找到“部门”字段,勾选显示列,并点击确定。

![](https://vip.kingdee.com/download/0109e2185ff48e6242f6b0445473af85cd5a.png)

3、然后就可以在费用录入界面,按部门录入费用信息了。

![](https://vip.kingdee.com/download/01094980f49c076145349d21a3334b59e7b1.png)
