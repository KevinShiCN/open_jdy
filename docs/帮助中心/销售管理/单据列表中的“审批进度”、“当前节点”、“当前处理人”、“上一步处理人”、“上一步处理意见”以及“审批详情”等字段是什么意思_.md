# 单据列表中的“审批进度”、“当前节点”、“当前处理人”、“上一步处理人”、“上一步处理意见”以及“审批详情”等字段是什么意思?

【问题描述】

单据列表中的“审批进度”、“当前节点”、“当前处理人”、“上一步处理人”、“上一步处理意见”以及“审批详情”等字段是什么意思?

【解释说明】

启用了工作流的单据可以在单据列表显示单据各个审核环节,以便关注单据的审核情况;

以销售订单列表为例,其他单据相同:

1、审批进度指单据当前所处的状态,包括未提交、处理中、处理完成、已驳回、挂起。注意:如果单据显示挂起可以去系统设置-审核日志中查看异常提示,查询是什么原因导致审批流程无法继续进行。

单据挂起处理方式,推荐参考链接:[单据审批在审批环节中提示异常挂起怎么处理?](https://vip.kingdee.com/knowledge/534899751742087680?productLineId=35&isKnowledge=2&lang=zh-CN)

![](https://vip.kingdee.com/download/01097634571474fd4d8094d2f5a9fa53e35d.png)

2、当前节点指单据目前所处的审核级数,需要查看单据审批节点设置可以点击系统设置-流程设计。

![](https://vip.kingdee.com/download/0109b22a943fb0fb47fd9267766025a2f88a.png)
点击具体单据进入之后,可以查看单据目前设置的是几级审批。

![](https://vip.kingdee.com/download/0109a66c4deb53fe49fbb838376535d18a67.png)
如果单据目前只是保存未提交状态,未到具体的审批节点,当前节点为空。

![](https://vip.kingdee.com/download/0109d15c5d9e26fc44a6b66a73c0a164026f.png)
以上述销售订单流程设计为例,如果一级审批已经通过,那么当前节点就是二级审批。
![](https://vip.kingdee.com/download/0109795a693291c44d399af1ef5f8b6f09b4.png)
3、当前处理人指目前所处的审批节点,拥有审批权限的用户。

![](https://vip.kingdee.com/download/010911f308eb44fb4ea69f90afec584ec5af.png)
以下图为例,流程设计中拥有二级审批权限的用户是这两个账号。

![](https://vip.kingdee.com/download/01095d59d228cbbb452fb7dd634176dc35ae.png)

4、上一步处理人指当前审批节点的前一级审批处理人

![](https://vip.kingdee.com/download/0109bc2f203fa417433da55db02282b83331.png)

5、上一步处理意见指当前审批节点上一步处理人选择的审核状态,以及审核意见内容。

![](https://vip.kingdee.com/download/0109ac59db00ba804f8c81dfaa52b5bab463.png)
具体显示详情如下图所示:
![](https://vip.kingdee.com/download/01091a4c66387bbc4afda27d72f93981413a.png)

6、审批详情指单据从提交到当前审批节点的审批流程

![](https://vip.kingdee.com/download/0109cc2beb75f1914d70a62c4b57c753b1e3.png)
