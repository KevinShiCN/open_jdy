# 销售预测单为什么无法生成销售订单?

【问题描述】

销售预测单为什么无法生成销售订单?

【解释说明】

此功能适用于星辰旗舰版。销售预测单下推销售订单时,销售预测单需符合的条件:“已审核”、“未行关闭”。
