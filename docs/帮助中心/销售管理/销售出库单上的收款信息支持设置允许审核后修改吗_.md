# 销售出库单上的收款信息支持设置允许审核后修改吗?

> 作者：Schatz | 阅读：99 | 有用：0

【问题描述】

销售出库单上的收款信息支持设置允许审核后修改吗?

【解释说明】

销售出库单的收款信息在审核后无法直接修改。但可以使用金额变更功能来变通操作,调整收款信息。具体操作步骤请参考以下动图:
