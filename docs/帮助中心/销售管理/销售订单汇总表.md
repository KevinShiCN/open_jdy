# 销售订单汇总表

# 1 简介

## 1.1 功能介绍

主要应用于统计一段时期内商品的销售订单总体情况，支持九总汇总维度进行统计。

## 1.2 适用版本

星辰3.1以上所有版本

## 1.3 系统路径

销售管理>销售报表>销售订单汇总表

## 1.5 筛选条件/筛选项说明

**汇总依据**：支持9种不同的维度----按客户、按客户分类、按商品、按商品类别、按商品品牌、按部门、按业务员、按仓库、按业务员+客户

**类别汇总级次**：当使用类别字段（如商品类别、客户类别）进行汇总时，可自定义展示的类别级次，默认1级。

举例：假设“商品类别”有两个级别，那么选2后可以分别展示出来对应的类别，如下图：

![](https://vip.kingdee.com/download/01002e861f81c3e641d79bb4b5156d61a45e.png)

**审核状态**：默认已审核

**显示商品明细**：不勾选时则按汇总依据的字段维度进行汇总，勾选了则按最细的商品维度进行展开。

注意，汇总依据为“商品”时，默认勾选“显示商品明细”，不允许取消。

![](https://vip.kingdee.com/download/0100f7fe971c181947f1b0f17a8d9c30401e.png)

**显示辅助属性**：必须先勾选“显示商品明细”才能勾选，不勾选时则按汇总依据的字段维度进行汇总，勾选了则按辅助属性维度进行展开

**显示小计**：必须先勾选“显示商品明细”才能勾选，勾选后报表不支持排序

# 2 操作演示

1）用户输入指定的时间、汇总依据及其他相关字段后，点击“查询”即可。

2）报表支持打印、引出、列设置、刷新、组合排序等通用报表功能
