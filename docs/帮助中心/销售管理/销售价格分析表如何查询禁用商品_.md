# 销售价格分析表如何查询禁用商品?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：云社区用户8x80607 | 阅读：81 | 有用：0

【问题描述】

销售价格分析表如何查询禁用商品?

【操作步骤】

1、销售分析-销售价格分析表,筛选条件勾选【显示禁用商品】,点击查询,如下图所示![](https://vip.kingdee.com/download/01091ed921c12d1c49bdb94fe1d02cc4a793.png)
2、如商品SP00116为禁用状态,勾选该参数之后,销售价格分析表可查询出禁用商品。效果如下图所示

![](https://vip.kingdee.com/download/01096ae966e4c8494c06b036e268792cfbd4.png)
