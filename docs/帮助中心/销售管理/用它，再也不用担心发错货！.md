# 用它，再也不用担心发错货！

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：进阶技巧, 知识软文
> 作者：七戒大师兄 | 阅读：1087 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/378842424396607744)

![](https://vip.kingdee.com/download/01096ffe0a9ff7584e6a85fd3345314f1e63.png)

      “刀下留人！”

      东海龙王循声望去，只见一道刺眼的强光冲进龙宫，溅起阵阵涟漪，待水雾散去，只见一只美猴王飒爽站定。

      “大圣，是什么风把您给吹来了？”龙王心有戚戚然地问到。要知道，当年这泼猴可以把东海翻了个底朝天呀

      大师兄告知：“放心，这回不是来你这龙宫找宝贝的，相反我要送你个宝贝”

      老龙王有些哭笑不得，但还是恭回道：“大圣就别开老头玩笑啦”

      大师兄咧嘴一笑：“我听说龟丞相前些日把你们公司客户的订单发错货了，此时正拉去问斩？你这个老泥鳅也太不讲情面了吧，他跟了你这么多年，没有功劳也有苦劳，就为了这点小事不至于哈，给你看样好东西，有它以后再也不用担心发错货了”

      大师兄随即掏出了一部手机，里面安装了精斗云APP，“看我来演示给你看，对了，把龟丞相也叫过来一起学吧”

      1、新增销售订单——审核销售订单——选择PDA验货——选择商品的仓库——开始扫码

      注意：1、精斗云APP（金蝶云·星辰）支持PDA验货，防止错发多发。但目前只适用于从销售订单直接下推销售出库单才可PDA验货；

                2、PDA验货后，直接下推出库单，所以验货时仓库为必填项，必须先选择销售订单里商品对应的仓库后再扫码

![](https://vip.kingdee.com/download/0100112247d7bbc9428b9b86bb1f0fd09bf8.png)

      2、PDA扫码完成后，即可查看验货情况，是正常还是超出还是未扫还是部分出库，防止错发多发漏发。根据实际情况，下推为销售出库单。

![](https://vip.kingdee.com/download/0100cae6d93b65ed4eac8cfff596d3065852.png)

      龟丞相看完后小声嘀咕了一句：“要是早些知道这个功能就好了，唉~”

      龙王看后大喜，笑骂龟丞相：“还不赶快谢过大圣！”

      龟丞相作势要跪下去，大师兄摆了摆手：“不必了不必了，猴子猴孙都跟我说了，之前我跟随师父西行，龟丞相对我花果山关照不少，再说这也是我该做的，记住以后细心点就好了”

      从此之后，学会PDA验货操作方法的龟丞相再也没有发错过货了
