# 销售订单列表如何显示预计销售费用明细?

> 作者：lykk | 阅读：180 | 有用：0

【问题描述】

销售订单列表如何显示预计销售费用明细?

【解决方案】

销售订单列设置中,将“预计销售费用”、“预计销售费用明细”字段显示
