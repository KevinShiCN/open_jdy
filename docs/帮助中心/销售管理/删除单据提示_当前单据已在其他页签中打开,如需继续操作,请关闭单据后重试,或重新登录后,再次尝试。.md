# 删除单据提示:当前单据已在其他页签中打开,如需继续操作,请关闭单据后重试,或重新登录后,再次尝试。

> 作者：Siwennn | 阅读：228 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/634557385997879808)

【问题描述】

删除单据提示:当前单据已在其他页签中打开,如需继续操作,请关闭单据后重试,或重新登录后,再次尝试。

【解释说明】

由于删除单据时,该单据详情界面已经在页签中打开,需要先关闭单据页签后再进行操作。
