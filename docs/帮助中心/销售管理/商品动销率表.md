# 商品动销率表

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：快速入门, 功能介绍
> 作者：进销存赵嘉丽 | 阅读：1857 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/170258454139978496)

# **1 简介  **

## **1.1 功能介绍**

主要用于当天查询节点为止，当前账套各仓库的库存商品总数，及本年、本月整体动销品种数和动销率指标，帮助企业了解商品的动销情况，以评估销售的效果，即库存商品中实际产生销售的商品种类及占比，从而制定有针对性的销售策略或指导补货。

## **1.2 适用版本**

星辰3.1以上所有版本

## **1.3 系统路径**

销售管理>经营分析>商品动销率表

# **2 名词释义**

**1）计算公式**

| 名称 | 释义 |
| --- | --- |
| 动销 | 即有库存的商品所拉动的销售 |
| 库存商品总数 | 即该仓库目前在库且库存>0的商品种数（包含已禁用商品，不包含“不可销售”商品） |
| 本月动销品种 | 即本月库存商品总数中有销售的商品种数 |
| 本年动销品种 | 即本年库存商品总数中有销售的商品种数 |
| 本月动销率 | =本月有销售的商品种数/截止到当前有库存的商品种数（包含已禁用商品，不包含“不可销售”商品） |
| 本年动销率 | =本年有销售的商品种数/截止到当前有库存的商品种数（包含已禁用商品，不包含“不可销售”商品） |

**2）有销售的商品种数**：在已审核的销售订单、销售出库单中存在记录。

**3）查询时间**：商品动销率表目前只支持查询当天的数据，不支持查询历史数据。

**4）指标意义**：如动销率过低，说明可能存在较多滞销商品；动销率大于100%，说明可能存在需要补货的商品，具体动销率的高低标准以各企业或行业的标准为主。

# **3 应用流程**

打开销售管理>经营分析>商品动销率表可查看，支持引出及打印。
