# 委托商品跟踪表如何显示已禁用商品?

> 作者：云社区用户8x80607 | 阅读：73 | 有用：0

【问题描述】

委托商品跟踪表如何显示已禁用商品?

【操作步骤】

销售管理-委托商品跟踪表,查询条件勾选【显示已禁用商品】,如下图所示,SP00125商品已禁用,需勾选参数才可显示

![](https://vip.kingdee.com/download/01098b131a02358e44a097270f3677cd5494.png)
