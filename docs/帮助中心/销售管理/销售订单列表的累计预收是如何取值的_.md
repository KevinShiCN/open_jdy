# 销售订单列表的累计预收是如何取值的?

【问题描述】

销售订单列表的累计预收是如何取值的?

![](https://vip.kingdee.com/download/01094c7ed4bea8d44c45b10437c95a007814.png)

【业务场景】

金蝶云星辰销售订单列表的累计预收数据来源是什么的呢,做了销售订单后,在做预收款单的时候选该订单生成,销售订单里面的累计预收金额是不是会累计增加

若您想要了解查看销售订单预收款问题,请点击:[如何查询销售订单累计预收金额?](https://vip.kingdee.com/knowledge/690733691697612288?productLineId=35&isKnowledge=2&lang=zh-CN)

【解释说明】

销售订单列表的累计预收是根据与该**销售订单相关联的预收款**来计算的。

如销售订单录入时录入的本次预收款生成预收款单,或录入预收单时,通过选源单(销售订单)关联销售订单,系统会将关联该销售订单的预收单金额统计到对应销售订单的累计预收字段中。

![](https://vip.kingdee.com/download/01098ffad399b9cf4f62a384c1c232ed7ff4.png)若存在多张预款收单与同一销售订单关联,系统也会将这些预收单的金额进行汇总,然后显示在销售订单列表的累计预收字段中,以反映该销售订单截至当前已收到的预收款项总额
