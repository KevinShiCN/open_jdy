# 销售报价单如何设置审批流?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：云社区用户8x80607 | 阅读：594 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/628760619298262784)

【问题描述】

销售报价单如何设置审批流?

【操作步骤】

1、系统设置-流程设计-【销售管理】,点击【新增】,选择销售报价单,如下图所示![](https://vip.kingdee.com/download/010988e1a0e92c314c97a6939022e97366cc.png)
2、选择【审核级数】、【会审设置】,选择是否设置会审。选择完成点击【保存】

若设会审节点,会审节点上参与人必须全部同意才可通过;若不设会审节点,审核节点上任一人同意即可通过。

![](https://vip.kingdee.com/download/0109c085dbf4a58141f2868e26fbba4c0274.png)
3、点击审批流程类型名称进入设计器。在【流程区域】点击选中审核节点可以针对该节点进行编辑操作,见下图

![](https://vip.kingdee.com/download/01096ee0981c81ce46b89708b6f26a87837d.png)4、选择特定人员审批:参与人点击【新增】可以从人员中去选择(仅支持启用状态的职员)如下图:![](https://vip.kingdee.com/download/01099465a36dae2749ea98e2e9c83b53d470.png)
5、审批人添加完成效果如下图,有两种审批方式选择:所有参与人都会收到任务、当满足条件时分配对应的参与人,可根据需要选择
![](https://vip.kingdee.com/download/0109567b8fcd89a44f409bbdda907c1902b1.png)
6、选择:当满足条件时分配对应的参与人时,可通过如果/否则条件设置审批人,

```
1)【如果】逻辑:按照设定的条件选择对应的参与人。
```

```
2)【否则】逻辑:【如果】规则不满足后选择的参与人。
```

```
3)【空】逻辑:不配置条件,始终都会选择的参与人
```

![](https://vip.kingdee.com/download/0109f1d29d6e745d4a92bcd966c33374f368.png)

7、会审设置基本信息设置:单据是否可修改,选择允许修改字段![](https://vip.kingdee.com/download/0109df56616f1127494c8527d2145395905d.png)
8、设置节点跳过,是否允许上传流程附件、当前节点处理人在前续节点已经同意时自动审批。设置完成,点击【保存】/【发布】更多流程设计知识可点击:[工作流操作手册](https://vip.kingdee.com/questions/266867384562816256/answers/266867384562816256?productLineId=35&isKnowledge=2&lang=zh-CN)

![](https://vip.kingdee.com/download/0109dacdef94b7664634bf3d4993ff1d6093.png)
9、流程发布成功后,单据可点击“提交”和“提交并新增”按钮,该单据即可命中【流程管理】中处于启用状态的流程,审批时,可以点击审核处理,如图:![](https://vip.kingdee.com/download/0109cdc4a48087594262a1d781544b3edc99.png)
10、参与人审批,可以到系统设置-任务中心处理,也可以到单据列表处理

单据列表会新增三个工作流相关字段:审批进度,当前节点和上一步处理人,如下图所示![](https://vip.kingdee.com/download/010982e868452bc5461b90863e403557075f.png)
