# 如何限制导出销售出单的价格字段?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：57 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/646878292209020416)

【解决方案】

如何限制导出销售出单的价格字段?

【解决方案】

**不给子账号查看销售出库单价格字段的权限,子账号引出销售出库单无法显示价格字段。**

在【系统设置】-【用户管理】勾选子账号做授权。

![](https://vip.kingdee.com/download/01097a107b29d6e645d8ab9d67abefd9abc1.png)
销售出库单选择金额字段,不给子账号查看的权限。
![](https://vip.kingdee.com/download/0109f2e2afcbf03b42ec9b04fdd9f3f576a9.png)
子账号无法再系统查看到单价金额字段,引出销售单也无法查看到金额字段。
