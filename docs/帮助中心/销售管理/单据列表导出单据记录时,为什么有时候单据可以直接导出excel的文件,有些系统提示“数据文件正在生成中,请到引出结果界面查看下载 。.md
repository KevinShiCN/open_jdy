# 单据列表导出单据记录时,为什么有时候单据可以直接导出excel的文件,有些系统提示“数据文件正在生成中,请到引出结果界面查看下载 。

【问题描述】

单据列表导出单据记录时,为什么有时候单据可以直接导出excel的文件,有些系统提示“数据文件正在生成中,请到引出结果界面查看下载 。

【解释说明】

如果导出的文件,表体和分录数据没有超过300条记录,系统在执行导出的时候,会直接导出Excel的文件格式,如果表体和分录数据超过300条记录,会提示到引出结果界面查看下载 。
