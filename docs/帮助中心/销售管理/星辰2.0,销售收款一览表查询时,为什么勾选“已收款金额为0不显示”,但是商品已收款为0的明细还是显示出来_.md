# 星辰2.0,销售收款一览表查询时,为什么勾选“已收款金额为0不显示”,但是商品已收款为0的明细还是显示出来?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：winnie_wen | 阅读：249 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/568601861310118144)

【问题描述】

销售收款一览表,查询时,为什么勾选“已收款金额为0不显示”,但是商品已收款为0的明细还是显示出来?

【解释说明】

因为报表中的参数“已收款金额为0不显示”针对的是整单的收款,当应收应付参数选择“按分录核销”时,是到商品分录的,对应的收款字段是“商品已收款”,不是“已收款”,所以此时勾选了是不生效的:

![](https://vip.kingdee.com/download/01097831be8798664ab29573d86abe683dfe.png)

![](https://vip.kingdee.com/download/010951a9bb2b534d445a8c48caa4ba984a0b.png)

只有应收应付参数选择“按单据核销”或“按余额核销”时,也就是按整单的收款时,

![](https://vip.kingdee.com/download/0109d247a6c602b14a3292bac5acf390cbe1.png)

这个筛选项才能用,且对应的字段是“已收款”:

![](https://vip.kingdee.com/download/0109ebb4dc91f11245c6a499cf6dcecaa32c.png)
【适用版本】

星辰2.0
