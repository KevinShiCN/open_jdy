# 销售出库单看操作日志时什么情况下单据审核时间会早于最后修改时间?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：winnie_wen | 阅读：514 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352510243966464)

【解释说明】

如果单据有设置自定义字段,自定义字段设置【审核后允许修改】,会出现修改时间晚于审核时间 ;

![](https://vip.kingdee.com/download/010983dfc0059de84e01bcf225bf70cefdeb.png)

如果单据审核后,后面进行的收付款单核销,核销会更新单据的已收金额和未收金额, 也会出现修改时间晚于审核时间 ;

单据本身的某些字段(如 采购费用、销售费用、客户承担费用、允许重新取价等)设置了允许审核后修改,也会出现修改时间晚于审核时间。
