# 销售退货单和产品退库单有什么区别?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：岸边把酒 | 阅读：349 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/660649248417105408)

【问题描述】

销售退货单和产品退库单有什么区别?

【解释说明】

销售退货单和产品退库单在主要适用业务范围,录入单据信息等方面还是存在一定区别,需要根据自身业务情况合理选择正确的单据进行录入。

1、对商品库存数量的影响方向不同

(1)销售退货单主要是减少销售出库的数量,减少出库,即增加库存。

(2)产品退库单主要是减少产品入库的数量,减少入库,即减少库存。

![](https://vip.kingdee.com/download/01091a7d7dcb787f469d9fb90df342622caf.png)

2、适用业务场景不同

(1)销售退货单主要适用于日常销货过程中,记录客户退货数量及退货金额的业务。

(2)产品退库单主要适用于生产过程中,记录对于已入库的不合格产品进行退库的业务。

3、关联单据不同

(1)销售退货单可以选择销售出库单和销售退货申请单为源单。

![](https://vip.kingdee.com/download/0109ea0539f2a7ed4ac9899eb4dc90054725.png)
(2)产品退库单可以选择产品入库单为源单。

![](https://vip.kingdee.com/download/010989f664f46c064715a7c74ad3f42cfb35.png)

4、单据可录入关键信息不同

(1)销售退货单可以在表头录入客户名称、结算期限和应收款余额等和客户有关联的信息。

![](https://vip.kingdee.com/download/0109600bf0bda9c04a249cac36295f651594.png)

在表尾可录入退货金额等金额信息

![](https://vip.kingdee.com/download/010926686b50a3924679be7142eb034df470.png)
(2)产品退库单可以录入仓管员,收货人等和生产相关信息。

![](https://vip.kingdee.com/download/010949b229ebbbe5491099bab040fdca1c30.png)
