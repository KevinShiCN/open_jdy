# 录入单据使用快速粘贴功能粘贴客户商品编码/供应商商品编码时需要注意什么?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：171 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/716457178680530432)

【问题描述】

录入单据使用快速粘贴功能粘贴客户商品编码/供应商商品编码时需要注意什么?

【业务场景】

[录入单据时支持使用快速粘贴功能实现快速录单](https://vip.kingdee.com/knowledge/391351756862874368?productLineId=35&isKnowledge=2&lang=zh-CN),若系统设置了[外部商品库](https://vip.kingdee.com/knowledge/609189599868446976?productLineId=35&isKnowledge=2&lang=zh-CN),也支持在录入采购/销售类单据时使用供应商商品编码/客户商品编码操作快速粘贴。

【解决方案】

在录入采购类/销售类单据时使用供应商商品编码/客户商品编码操作快速粘贴功能**必须先选择单据的供应商/客户,再粘贴数据**。
1、设置外部商品库存
在【基础资料】-【外部商品库】中设置客户/供应商对应外部商品信息,具体可以参考知识【[外部商品库如何使用?](https://vip.kingdee.com/knowledge/609189599868446976?productLineId=35&isKnowledge=2&lang=zh-CN)】操作。

![](https://vip.kingdee.com/download/010914cd60bddb4e4e13b77a8ad4168e295f.png)

2、维护Excel表格

在Excel表格中维护客户商品编码/供应商商品编码及单据的其他信息,再复制表格内容。

![](https://vip.kingdee.com/download/010969ea206f5d93426280c46dd3c2980dd4.png)
3、粘贴数据

在新增采购/销售类单据时先选择供应商/客户,再粘贴数据。以销售出库单为例,新增销售出库单时先选择外部商品库中已经维护好的客户信息,如“A客户”,然后再把鼠标放到商品编码的表格中,按ctrl+V把上述表格中的数据粘贴即可。

![](https://vip.kingdee.com/download/01098e1770c544294bb4b9c7923336e25889.png)
【温馨提示】

若没有选择供应商/客户,直接将供应商商品编码/客户商品编码及相关数据粘贴过来时操作失败,系统会提示:第1行数据无法识别。 请检查商品名称 或 商品编码 或 条形码 或 客户/供应商商品编码是否正确 或 商品是否已审核。 如果为辅助属性商品,请检查SKU编码、辅助属性是否正确或是否被禁用。 如果为序列号商品,请检查序列号是否正确。 如果为服务或套装商品,请检查商品类型是否正确。

![](https://vip.kingdee.com/download/0109cb0634db7755444d9ed1c6e5706045fc.png)
