# 销售管理概述

## 1、适用版本

## 2、产品概述

### 2.1产品介绍

销售业务是企业经营活动中最重要的一环。销售业务主要包括报价、与客户签订合同，按合同约定的商品数量、价格、发货时间、收款方式进行发货并收回款项；销售业务还需处理因各种原因导致客户换货、退货时，收回退货商品和给客户退回货款。此外还有一部分销售业务是先出库，后面再跟客户做结算，这部分属于委托代销业务。还有一部分客户，是通过发票来确认收入的，这部分是销售发票业务。

### 2.2业务场景

目前星辰支持以下销售场景：

- 销售报价---用于销售正式开单前的询价环节，报价确认后可直接转化为销售订单。
- 销售订单---用于记录合同信息，订单一旦确定不能随意修改。订单不影响实际库存和资金。
- 销售出库---用于将货物从仓库发出，减少商品库存，同时收到资金或增加应收账款。
- 销售退货---用于将客户退回的商品入库，并退回货款或减少应收账款。
- 销售发票---用户通过发票确认应收的场景。【仅适用于星辰旗舰版】
- 销售核销---用户处理，原销售出库单的未开票数量无需继续开票，且销售退货单也无需开红冲发票，可将未开票数量相等的销售出库单与销售退货单进行对等核销的场景。【仅适用于星辰旗舰版】
- 销售退货申请---用于客户需要退货，销售人员发起退货申请【仅适用于星辰专业版/旗舰版】
- 销售换货申请---用于客户需要换货，销售人员发起换货申请【仅适用于星辰专业版/旗舰版】
- 委托代销---处理某公司把自己的商品委托给下游经销商或门店销售，货权没有转移，下游经销商或门店根据实际销售的数量和金额与公司结算。【仅适用于星辰专业版/旗舰版】
