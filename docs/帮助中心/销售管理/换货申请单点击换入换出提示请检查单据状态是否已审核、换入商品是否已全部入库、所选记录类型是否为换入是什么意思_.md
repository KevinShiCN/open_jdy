# 换货申请单点击换入换出提示请检查单据状态是否已审核、换入商品是否已全部入库、所选记录类型是否为换入是什么意思?

【问题描述】

换货申请单点击换入换出提示请检查单据状态是否已审核、换入商品是否已全部入库、所选记录类型是否为换入/换出是什么意思?

![](https://vip.kingdee.com/download/01099ae4bacdbbb74a1d9bc515bcdeed39ad.png)

【解释说明】

换货申请单点击换入或者换出时出现以上提示请检查该单据的出入库状态:

**全部出入库、全部出库:**则表示已经生成了销售出库单和销售退货单,完成了换货操作,不需要再换入换出

**部分入库、部分出库、未入库、未出库:**才可以操作换入换出。
