# 按库存开单优化

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：快速入门, 功能介绍, 新特性
> 作者：进销存熊诗琴 | 阅读：878 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/668024866804620544)

## 1 简介

### 1.1 功能介绍

业务开单时，支持查询库存后，直接选择商品进行开单，并携带出商品对应的仓库、仓位、批次、保质期、库存数量等信息。

### 1.2 适用版本

星辰3.1以上所有版本

### 1.3 按钮说明

按库存开单：按商品或仓库等筛选条件查询库存后，直接选择商品按照库存数量进行开单。

![](https://vip.kingdee.com/download/0109fa5d676f6c494693be4fb3ffce12ab3a.png)

## 2 功能特性

### 2.1 支持按库存开单的单据

| 采购类 | 销售类 | 库存类 |
| --- | --- | --- |
| 采购申请单 | 销售报价单 | 其他入库单 |
| 采购订单 | 销售订单 | 其他出库单 |
| 采购入库单 | 发货通知单 | 调拨出库单 |
| 采购退货单 | 销售出库单 | 盘盈单 |
| 采购发票 | 销售发票 | 盘亏单 |
|  | 委托代销单 | 预留单 |

- 采购发票、销售发票说明：仅业务类型选择“入库”“出库”时才显示“按库存开单”的按钮，业务类型选择“退货”时不显示该按钮。

### 2.2 操作说明

第一步：单据详情中，增加“按库存开单”的功能按钮，点击可打开“按库存开单”的弹窗；

![](https://vip.kingdee.com/download/010908e655a9e2534d81a19cc1ab98d6e3ef.png)

按库存开单弹窗：

![](https://vip.kingdee.com/download/01095e6e5a3560b74054adc7507fc918d342.png)

第二步：弹窗中输入商品或仓库等筛选条件后，可查询对应数据的库存，此时可按仓位、批次、保质期维度展开库存；

![上传图片](https://vip.kingdee.com/download/0100cce3a5c64ecf44b3a7dd00696eebdf7f.jpg)

第三步：弹窗中勾选数据后，可以将查询出来的数据携带到单据中。若需要按库存数量开单，则可勾选弹窗中的【按“开单数量（基本）”开单】参数；

![上传图片](https://vip.kingdee.com/download/0100bcc423f07c5b4c9c902abf24cc0d14c9.jpg)

- 相关功能点说明：

| 功能点 | 说明 |
| --- | --- |
| 查询数据上限 | 若查询出来的库存数量超过3000条，则需要输入筛选条件，缩小数据范围后再查询 |
| 按“开单数量（基本）”开单 | 1、按用户记住上一次的选项，不区分单据。2、  “开单数量（基本）”默认等于“可出库数量（基本）”，可手工修改。3、  勾选该参数后，最终以修改后的数量携带到单据的“数量”字段中，同时会将商品的“基本单位”携带到单据的“单位”字段中 |
| 勾选数据上限 | 目前最多支持勾选100条数据携带到单据中 |
| 携带数据 | 按查询出的库存数据维度携带相关信息。例如，查询的是商品+仓库维度的库存数据，则携带商品、仓库到单据中。若查询出的是商品+仓库+仓位+批次+保质期维度的库存，则携带商品、仓库、仓位、批次、保质期信息到单据中 |
