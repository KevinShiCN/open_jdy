# 明明做了其他支出单,为何企业经营利润表不统计这笔支出?

【问题描述】

明明做了其他支出单,为何企业经营利润表不统计这笔支出?

![](https://vip.kingdee.com/download/01091f9a9e7ec34544bb9cc647b5181eb269.png)
![](https://vip.kingdee.com/download/01096a8799dd13cb46e5b9b933f4b81d3a45.png)
【解释说明】

出现这种情况的原因是,在基础资料—支出类别中,没有勾选计入企业利润

![](https://vip.kingdee.com/download/010910e0e9c6e4a14db883be03b833c6821c.png)
勾选之后再去重新查询企业经营利润表即可正常显示

![](https://vip.kingdee.com/download/01094ce01d30bd6e42bb86850ab16e1813e8.png)
