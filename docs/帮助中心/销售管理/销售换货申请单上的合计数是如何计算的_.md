# 销售换货申请单上的合计数是如何计算的?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：shaoling | 阅读：31 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/722619663304968448)

【问题描述】

销售换货申请单上的合计数是如何计算的?

【解释说明】

销售换货申请单的合计行,是换入与换出商品数量或金额的累计总和

![](https://vip.kingdee.com/download/01096ce5b41f00e24b5a81fa274c8dd01c39.png)

若需查看换入换出的金额差额,可直接查看本次应收账款金额

例如:当换入商品金额为 100.00 元、换出商品金额为 200.00 元时,价税合计金额按累计规则显示为 300.00 元(100.00+200.00);本次应收账款金额为 100.00 元(200.00-100.00),即客户需补付的差额款项。

![](https://vip.kingdee.com/download/0109924ff17d8020498380e981c5e8972c6f.png)
