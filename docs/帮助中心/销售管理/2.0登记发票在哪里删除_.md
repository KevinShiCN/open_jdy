# 2.0登记发票在哪里删除?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：云社区用户8x80607 | 阅读：202 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/621511502830312960)

【问题描述】

2.0登记发票在哪里删除?

【操作步骤】

以销售出库单登记发票为例

1、销售管理-销售出库单列表,列设置-【已开票金额】字段打开

![](https://vip.kingdee.com/download/0109804c0a8807804b92b7d4e43bd90b427e.png)
2、点击已开票金额,数字进入发票信息。点击【删除】即可,如下图所示![](https://vip.kingdee.com/download/01099e3f549a04e2451aba40d767e7d6ce84.png)

3、如果没有【发票删除】功能,登录管理员账号-系统设置-用户管理-点击功能授权。如下图所示![](https://vip.kingdee.com/download/0109ce215477d13648d6acd3a378454dfc60.png)
