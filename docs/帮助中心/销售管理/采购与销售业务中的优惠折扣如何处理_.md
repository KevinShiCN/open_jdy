# 采购与销售业务中的优惠折扣如何处理?

【业务场景】

以销售出库单为例(采购业务同理)

在商品销售过程中,为提升客户满意度,我们提供以下灵活的价格优惠方式:

1、商品行折扣:针对单个商品给予特定折扣

2、整单折扣:对整张销售单据给予统一折扣

3、收款抹零优惠:在销售出库单"收取货款"金额不为零时,可对尾数进行抹零处理

4、收款单优惠:在收款单录入优惠金额减免部分金额

【操作流程】

**方法一:商品行折扣**

在商品行对应的折扣率和折扣额字段输入数值

**折扣率**:输入百分比数值(如10%表示打9折)

**折扣额**:直接输入优惠金额(如10元表示减免10元)

**折扣(折):**直接输入折扣数字(如9表示打9折),折扣(折)为10表示不打折

![](https://vip.kingdee.com/download/0109aa31660a2e814bbaac78393280eaf170.png)**方法二:单据整单折扣**

在单据表尾金额信息部分的整单折扣额和整单折扣率字段输入折扣数值,针对整张单据所有的商品设置统一折扣

![](https://vip.kingdee.com/download/010960b94d5483fd4ccfaed3e6f310bbd4d2.png)

**录入整单折扣额后**,系统将自动按商品金额比例将折扣金额分摊至各分录行的整单折扣分配额字段,也会更新价税合计和成交金额,以整单折扣后的金额确认本次应收金额(不影响单价)

![](https://vip.kingdee.com/download/0109930640aba6f5453b9d228fef0d7e5717.png)若在录入整单折扣后又操作了新增商品分录、修改商品数量、调整商品单价,此时需手动点击【分摊】按钮重新计算分配

![](https://vip.kingdee.com/download/010956a28a0e70b74c6b85f8633e7dce9bc7.png)

【整单折扣注意事项】

1、整单折扣自动分配规则:是按商品价税合计的金额比例分摊到对应的商品分录中。

2、整单折扣的分配金额可以在商品分录中手工修改,但需要保证商品行中整单折扣分配额合计跟表尾录入的整单折扣额相等

3、整单折扣额默认是自动分摊的,不能关闭,如果不想分摊折扣建议使用方法四,做收付款单时进行优惠

**方法三:单据收款抹零**

在销售出库单上填写收款信息时,针对交易金额的小额尾数可以”收款抹零“

示例如下:**应收金额**120元,**实收金额**110元,**收款抹零**10元,累计收款显示全部收款

![](https://vip.kingdee.com/download/01094453a49566e44a489702c1d5c9202468.png)
此方法必须是“收取货款”不为零时才可以使用,比如使用预收款抵扣了部分货款,剩余的小金额尾款就不支持用**收款抹零**核销
![](https://vip.kingdee.com/download/01091c3399a67e4540629afbd55bf255ee27.png)

**方法四:收款单的优惠金额**

当销售出库单未设置任何折扣或优惠,后续需给予价格让步,可在客户支付货款时通过收款单进行优惠

![](https://vip.kingdee.com/download/01090c007aa9238442678e014c8ea319e8c0.png)
示例如下:**应收金额**120元,**账户实收金额**115元,**优惠金额**5元![](https://vip.kingdee.com/download/010982db2348450e41a390db73cdacd65d90.png)
