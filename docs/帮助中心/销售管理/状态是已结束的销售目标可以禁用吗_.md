# 状态是已结束的销售目标可以禁用吗?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题

【问题描述】

状态是已结束的销售目标可以禁用吗?

【解释说明】

**状态为已结束的销售目标不可以禁用。**

禁用会提示:目标[XX年销售目标]已结束,不允许禁用;
