# 系统里面的单据名称支持自定义修改吗?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：97 | 有用：0

【问题描述】

系统里面的单据名称支持自定义修改吗?

【解释说明】

**系统里面的单据名称是固定的不支持自定义修改。**如销售出库单/销售退货单/采购入库单等。
