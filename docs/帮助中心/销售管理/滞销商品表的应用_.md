# 滞销商品表的应用?

【问题描述】

滞销商品表如何应用?

【业务场景】

1、通过滞销商品表,可快速了解商品的周转速度和库存积压情况,包括无销售天数,最近销售日期、入库日期、库存数量、金额等等,以便及时调整商品的销售策略,如通过促销活动加速库存周转和消耗;

2、如食品批发零售、加工、餐饮等行业,还可以结合保质期预警查询表一起使用,以便了解原材料或者半成品的积压情况及效期剩余时间,以便合理安排先进先出以及库存清理的工作,避免库存积压过多,效期变短带来的库存货值损失。

3、通过滞销商品表,结合销售分析表,可以对商品的整体销售效果有预判,通过筛选了解各商品类别、品牌下滞销商品的分布情况。

【查询路径】

方法1:通过【销售管理】-【经营分析】-【经营日报】中的“库存情况”点击查看“滞销商品”的概况,点击对应的数字可联查到【滞销商品表】;
