# 单据分享的二维码/链接分享是否有时间限制?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：云社区用户CFA6487 | 阅读：38 | 有用：0

【问题描述】

单据分享的二维码/链接分享是否有时间限制?

【解释说明】

单据的分享功能,二维码/链接分享默认是永久的,没有时间限制,也没有办法设置失效时间。
