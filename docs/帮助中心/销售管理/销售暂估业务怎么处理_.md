# 销售暂估业务怎么处理?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：Schatz | 阅读：1690 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/637094109412502016)

【问题描述】

销售暂估业务怎么处理?

【解决方案】

1、星辰标准版和专业版不具备暂估功能,但可以先按照预估金额制作销售出库单。待发票开具后,可使用原销售出库单进行退货处理,并依据发票重新做销售出库单。

2、在星辰旗舰版中,实现了单票分离。对于暂估的情况,可以先制作销售出库单进行出库操作。等开完发票,便可通过下推销售发票来确认收入。

**若您想了解旗舰版业务,请查看:**[星辰旗舰版(进销存)业务应用与常见问题](https://vip.kingdee.com/article/602143212584657664?productLineId=35&isKnowledge=2&lang=zh-CN)

【操作步骤】

**星辰标准版和专业版**

①先按暂估的价格做销售出库单;

![](https://vip.kingdee.com/download/0109b5edb53faaa64d029b1c2e750f871972.png)

②发票开具之后,先把销售出库单下推销售退货单;

![](https://vip.kingdee.com/download/01092b58010b4c7548fea3c3d8e9751c433c.png)

③重新按发票金额做销售出库单。

![](https://vip.kingdee.com/download/0109df32009899744fa69c6d645fca3dbcc2.png)

④后期收到货款,做收款单可以把暂估的销售出库单以及退货单,还有发票金额的销售出库单一起选择为源单来核销。

![](https://vip.kingdee.com/download/0109d5916a039a5741098834df47610c1f4b.png)

**星辰旗舰版**

①先按暂估金额做销售出库单;

![](https://vip.kingdee.com/download/01099a853d14638c4ae4b52620c71105edc2.png)

②发票开具之后,销售出库单下推销售发票;

![](https://vip.kingdee.com/download/010936eba49ebbf64c6fa2e4bad8c3c389c6.png)

③按发票金额修改销售发票的金额,确认收入。

![](https://vip.kingdee.com/download/01097e3b52322c7645f698d98f62747398f6.png)

④后期收款的时候,做收款单,选择销售发票为源单收款。

![](https://vip.kingdee.com/download/0109d1175aa81ecd43f792ecee87e24ef6e2.png)

【温馨提示】

销售发票会生成应收凭证,收款单可生成确认收入凭证,若实际收入与销售发票有差异,可在收款单做优惠金额处理,生成收款单凭证时一般生成到优惠相关的科目(如财务费用-优惠等)实现账务平衡。
