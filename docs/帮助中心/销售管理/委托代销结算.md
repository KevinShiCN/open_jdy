# 委托代销结算

1 简介

### 1.1 功能介绍

1）登记下游经销商反馈的实际销售数据，用于跟下游经销商结算应收回的货款，及计算还剩余多少未销售完；

2）委托代销结算时，可以不收款，部分收款或者全部收款。

3）若已提前向客户收取了定金或预收款，可以在委托结算时直接抵扣预收款。

4）可以直接点击委托代销单列表的“委托结算”生成委托代销结算单，也可以在委托代销结算单新增界面，点击“选源单”，选择对应的委托代销单进行结算。结算数量不能大于委托代销数量。不能无源单结算。

5）企业指定了商品的最低销售限价，低于该价格时给予提示。

6）若支付给代销方的佣金是另外结算的，可在“费用及收款”的代销结算单中填写代销费用。

### 1.2 适用版本

星辰专业版/旗舰版

### 1.3 系统路径

销售管理>委托代销>委托代销结算

## 2 应用说明

### 2.1 委托代销单下推结算单

选中已审核的委托代销单，在单据详情或者单据列表可以操作“委托结算”，会下推生成保存状态的委托代销结算单，填写必录字段后审核即可。

![](https://vip-admin.kingdee.com/download/0109041ffb69337b4e56b41b917071e69862.png)

### 2.2 委托代销结算单列表自动匹配源单

#### 应用场景：

委托代销业务中，代理商将货物卖出后，与公司结算时，为了获取出库成本，核算利润，不能直接新增委托代销结算单，必须选择源委托代销单/委托代销期初单，下推结算单结算；当供货次数较多，统一结算时，人工寻找源单效率较低。

通过“待结算清单”列表，汇总显示客户的可结算商品，录入本次结算数量后，由系统自动匹配源单，获取成本，核算利润，提高结算效率。

#### 应用流程：

1）打开委托代销结算单列表，点击“总量结算”功能，进入待结算清单列表

![](https://vip.kingdee.com/download/0109f270e5ab14f9453c9b252219d8c6b6ce.png)

2）汇总显示客户的可结算商品，录入本次结算数量后，点击“生成委托代销结算单”，由系统自动匹配源单

![](https://vip.kingdee.com/download/0109e95d80d9209c4a55a0750f31cc3db6dd.png)

## 3 补充说明

1）委托代销结算单与销售出库单有什么区别？

答：①对库存的影响：委托代销结算单不会减少库存，不会发货，不能登记物流信息。

②报表统计：在库存类报表中，比如商品收发明细表，商品收发汇总表等，不统计委托代销结算单

2）委托代销结算单与销售出库单有什么相同点？

答：二者都会增加应收账款，可以计算毛利；在销售类报表和资金类报表中，比如销售明细表，销售汇总表，销售利润表，应收账款明细表等，统计委托代销结算单；
