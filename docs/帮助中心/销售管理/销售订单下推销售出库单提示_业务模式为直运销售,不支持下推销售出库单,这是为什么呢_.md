# 销售订单下推销售出库单提示:业务模式为直运销售,不支持下推销售出库单,这是为什么呢?

> 作者：大家好才是真的好 | 阅读：105 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/771905965221418240)

【问题描述】

销售订单下推销售出库单提示:业务模式为直运销售,不支持下推销售出库单,这是为什么呢?

【解释说明】

直运销售模式下,商品直接从客户发起采购,因此系统设计上不支持直接下推生成销售出库单。直运销售订单仅能下推直运采购订单。

![](https://vip.kingdee.com/download/0109075c9ec80d184a1d926c4333eb99e8e2.png)

【变通处理】

1、销售订单的业务模式修改为“普通销售”,审核后再生成销售出库单。

![](https://vip.kingdee.com/download/01097719f65ac1594147b6cb2f2022822c98.png)

2、通过直运销售业务流程处理,直运销售订单→直运采购订单→直运采购入库单→直运销售出库单。
