# 销售订单投放之后哪里可以看到哪些物料已经投放生产,哪些是委外加工?哪些是采购的?

> 产品：金蝶云·星辰 | 分类：销售管理 | 类型：常见问题
> 作者：_Leo | 阅读：347 | 有用：0

【问题描述】

销售订单投放后想看到哪些物料已经投放生产,哪些是委外加工,哪些是采购的,想要快捷看到。

【操作流程】

在【销售订单列表】,点击右上角的“齿轮”列设置按钮,将对应的字段显示出来即可看到,目前已增加相应的字段。如图:
