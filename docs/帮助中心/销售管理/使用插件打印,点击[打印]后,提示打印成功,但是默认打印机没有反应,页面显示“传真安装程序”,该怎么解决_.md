# 使用插件打印,点击[打印]后,提示打印成功,但是默认打印机没有反应,页面显示“传真安装程序”,该怎么解决?

> 作者：杨桂平 | 阅读：1781 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/534899758654195712)

【问题描述】

  使用插件打印,点击[打印]后,提示打印成功,但是默认打印机没有反应,页面显示“传真安装程序”,该怎么解决?

![](https://vip.kingdee.com/download/010940528fc270a141ec8207b9e744f33f42.png)

【解释说明】

 一般这种提示是因为[打印预览]里的打印机不是电脑的默认打印机或者重装插件后默认打印机没有被调用引起的

【解决方案】

第一步,重新在打印设置里的"打印方式"选择插件打印

![](https://vip.kingdee.com/download/0109655036adb24b4717af672b50ec552f3b.png)

  第二步,单击“打印预览”

![](https://vip.kingdee.com/download/010956806430f7fb4539ab488201505362c1.png)

第三步,勾选预览页的打印机下拉列表,切换其他打印机后再重新选择电脑所绑定的默认打印机即可。



![](https://vip.kingdee.com/download/01092aeeb5e7dd534fa58399519918dc7abb.png)
