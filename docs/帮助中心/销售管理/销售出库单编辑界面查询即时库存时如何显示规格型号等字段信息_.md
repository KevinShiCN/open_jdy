# 销售出库单编辑界面查询即时库存时如何显示规格型号等字段信息?

> 作者：云社区用户8x80607 | 阅读：108 | 有用：0

【问题描述】

销售出库单编辑界面查询即时库存时如何显示规格型号等字段信息?

【操作步骤】

1、【销售管理】-【销售出库单】,如下图所示,销售出库录入编辑页面,点击即时库存数字进入即时库存查看页面

![](https://vip.kingdee.com/download/0109efbdccdde0b34f8e9103bf06e0465843.png)
2、列设置-规格型号,显示打开

![](https://vip.kingdee.com/download/01099c572ff3505740feb32940b28c104e60.png)
3、打开效果如下图所示,销售出库单编辑界面查询即时库存即可显示规格型号字段![](https://vip.kingdee.com/download/01097d5117ed0f1b4a2ea253c3cca9dc0511.png)
