# 一、 功能说明

微信公众平台开放了模板消息功能。通过接入模板消息接口，微信公众号可向关注其账号的用户发送预设模板的消息，以便向用户及时发送重要的服务通知，如信用卡刷卡通知、商品购买成功通知等。但不支持广告等营销类消息以及其它所有可能对用户造成骚扰的消息。

# 五、操作说明

第一步：开通商家自己的微信服务号

点击[https://mp.weixin.qq.com/](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fmp.weixin.qq.com%2F)，进入微信公众平台，点击【立即注册】申请开通商家自己的微信服务号，按照微信官方引导填写相关信息，注册微信服务号。这个是微信的功能，请商家自行完成。

![上传图片](https://vip.kingdee.com/download/0100a0300e25086949a88fd38ca476fda847.png)

第二步：完成微信认证

微信公众号注册成功后，点击右上角【认证详情】按照微信官方引导填写相关信息，完成微信认证。（注意：请务必完成微信认证，否则可能会影响微信卡券、微信模板消息等功能的正常使用）。

![](https://vip.kingdee.com/download/010069639e6af70d42e892af1c5927ea97eb.png)

第三步：开通微信支付

进入微信公众平台，前往【设置与开发-开发-接口权限】，下拉页面找到“功能服务-微信支付，点击【申请】进入微信支付界面，或者点击【广告与服务-更多能力-微信支付】进入微信支付界面，点击【发起申请】或【关联更多商户号】进入微信支付商户后台，按照微信官方引导填写相关信息，申请开通微信支付并绑定商户号。这个是微信的功能，请商家自行完成。(注意：商家要使用微信会员在线充值功能，必须开通微信支付。）

微信官方审核通过后，回到微信支付界面，确认待关联的商户号，并点击【关联更多商户号】将申请通过的微信支付商户号完成关联绑定即可。

第四步：开通卡券功能

进入微信公众平台，前往【广告与服务-服务-卡券】，按照微信官方引导填写相关信息申请开通卡券功能。这个是微信的功能，请商家自行完成。

第五步：开通并添加微信模板消息

1. 业务场景及价值说明

## **零售客户进行会员营销时，需要根据客户的消费及订单状况、权益信息、会员业务向会员推送消息，以增强会员用户体验，促进增购转化复购。**

2. 操作指引

## **️ 注意：由于微信限制，本功能依赖于微信会员卡，同时需要会员关注您的微信服务号方可收到消息提醒。如还未在星辰后台配置微信会员卡，可参考进行配置后再操作。**

[如何对接微信会员卡 (kingdee.com)](https://vip.kingdee.com/school/detail/312170767582868992?productLineId=31&lang=zh-CN)

###

**（1） 服务类目配置** 绑定公众号配置完成微信会员卡后，前往公众号后台，在「设置与开发-公众号设置」中将「**生活服务 > 百货/超市/便利店**」加入服务类目。

![](https://vip.kingdee.com/download/0100e5186a7e15104667af6e6d79f5c1fabe.png)

**（2）模板页面介绍** 前往星辰后台「会员管理-公众号消息提醒」，可查看并启用消息提醒模板。

![](https://vip.kingdee.com/download/0100f5fb31e810014e25802f388787aa4aa4.png)

历史模板：微信已不支持新增历史模板，适用于先前创建过历史模板的客户。若客户先前在微信公众号后台创建过历史模板，且与系统内置模板相符，则显示存在的历史模板。

️ 注意：**由于微信政策调整，历史模板删除后无法创建，请慎重操作。**

**类目模版：**类目模板为微信新推出的公众号消息模板，用以替代历史模板。类目模板需要先在公众号后台添加「生活服务 > 百货/超市/便利店」作为类目之一，方可启用。

| 退款通知 | 储值成功通知 | 引荐会员注册通知 | 订单发货通知 |
| / | 充次成功通知 | 推广奖励发放通知 | 订单已签收通知 |
| / | 扣次成功提醒 | / | / |
| / | 商品寄存成功通知 | / | / |
| / | 寄存商品提取成功通知 | / | / |

其他模板：可展示非星辰系统创建的微信消息模板，仅可查看和进行删除操作，无法进行消息推送。如启用模板数超过微信数量限制，您可在此处快速删除未使用的模板。

**（3）模板启用方法**

您可通过开启一键启用，由系统批量为您开通全部类目消息模板，启用前请确保剩余模板数量充足（如数量不足可在页面下方「其他模板」处删除部分不再使用的模板，或前往微信公众号手动操作）。您也可通过启用或关闭单个类目模版实现更精准的配置。

**（4） 测试模板消息** 如您希望测试模板推送效果，可选中一个您想要推送的模板，点击「发送测试」，选择已领取微信会员卡的会员点击发送，即可测试模板消息效果。

第六步：完成微信授权

1) 进入星辰后台，前往【会员管理-服务-微信授权】，点击【立即绑定】按步骤完成授权绑定即可。

![](https://vip.kingdee.com/download/010023cd057804c2467ca4e7695978925159.png)

2) 授权绑定成功后，如下图，下图红框内信息均会自动带过来。（注意：1.开发者appserect非必填；2.二维码在完成微信会员卡设置后会自动显示）。若需要重新授权，可点击右上角【重新授权】按引导完成操作。

![](https://vip.kingdee.com/download/010065a06c62258d40c7a3daf6515822f8bf.png)

第七步：完成微信会员卡设置

进入星辰后台，前往【会员管理-服务-微信会员卡设置】，完成以下内容配置即可。以下内容有必填或选填，建议商家按照自身业务诉求，全部都配置一下。配置完成后，微信授权界面的二维码将会自动显示出来。（注意：点击保存若提示“提交微信异常api unauthorized hint：[hjN40a00228128]”，请登录微信公众平台，检查是否完成微信认证并开通卡券功能，需要完成微信认证并开通卡券功能，再到后台设置微信会员卡）。若按照操作手册操作依然配置失败，可联系星辰官方。

![](https://vip.kingdee.com/download/01003e4db0a0a2f54c81964e30ad65a24232.png)

七、注意事项说明

1) 注册微信服务号后，请务必完成微信认证，否则可能影响微信卡券、微信模板消息等功能的正常使用。

2) 商家在使用微信会员在线充值功能前，务必前往微信公众平台开通微信支付，否则无法进行在线充值。

3) 开通微信模板消息提醒前，请先开通微信支付功能和卡券功能，否则无法正常使用微信模板消息。

4) 开通微信模板消息功能时所选的行业，建议尽量选择“IT科技/互联网|电子商务”，与星辰官方所选择的行业保持一致，否则可能存在无法推送模板消息的情况。

5) 若要正常使用微信卡券、微信模板消息等功能，请前往星辰后台完成微信授权和微信会员卡配置。（注意：1.微信授权界面的开发者appserect非必填；2.二维码在完成微信会员卡设置后会自动显示）。

6) 进行微信会员卡设置时，点击保存若提示“提交微信异常api unauthorized hint：[hjN40a00228128]”，请登录微信公众平台，检查是否完成微信认证并开通卡券功能，需要完成微信认证并开通卡券功能，再到后台设置微信会员卡才能配置成功。

7) 用户要正常接收在商家的各种消费的微信消息提醒，需关注商家的微信公众号，且不取关。
