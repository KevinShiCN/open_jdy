# 金蝶云星辰零售会员、会员业务、积分、储值、绑定微信公众号、赊账Q&A及常见问题解决办法

问题描述：

金蝶云星辰零售会员、会员业务、积分、储值、绑定微信公众号、赊账Q&A及常见问题解决办法

解决方案：

**会员储值**

**1、Q：可以在后台给会员充值吗？**

A：不支持。目前只能在后台配置充值服务商品和充值赠送规则，在POS给会员充值。

**2、Q：后台设置了多个充值赠送方案，POS进行会员充值时会同时触发多个方案吗？**

A：不会，只会触发最新创建的那个方案，如果需要设置不同条件的赠送规则，需要在一个方案中设置多个规则。

**3、Q：在后台设置了两个充值赠送规则方案，都启用后在POS上进行充值，只生效其中一个，是什么原因，规则是怎样的？**

A：默认取最近更新时间的充值赠送规则。

**4、Q：POS会员充值可以在后台抵消吗？**

A：不支持。

**5、Q：充值赠送优惠券，赠送的数量录入错误，修改后，原已赠送到会员账户上的优惠券会自动增加或减少吗？**

A：不会自动增加或减少。若需要补给会员优惠券可以直接到优惠券方案中帮会员领取，若是优惠券赠送多了就无法减少。

**6、Q：怎么导入会员初始余额？**

A：前往POS【店务-会员批量充值】操作，充值前需要修改服务商品单位。

**会员积分**

**1、Q：支持给会员手动调整积分吗？**

A：支持。前往后台【会员管理-会员资料-会员信息】操作，勾选会员，点击【调整积分】。

**2、Q：会员积分可以设置某个时间，达到该时间后自动清空积分吗？**

A：目前暂不支持。积分清零开发中，预计 10 月上线。

**3、Q：积分规则可以设置按件数积分吗？**

A：不支持。消费得积分只与消费金额有关，与购买件数无关。

**会员赊账**

**1、Q：赊账还款时能不能指定还哪一笔，我怎么知道我还的是哪一笔？**

A：不能指定还哪一笔。如果是一次性还款，则本次还款即全部还款；如果是分期还款，则默认还的最近一笔到期的赊账，退款则相反，默认退的是最远一笔到期的赊账。比如A赊账300元，分三期还完，每笔100元，分别于3月1日、4月1日、5月1日到期，A在2月28日还款一笔150元，则默认还的是3月1日到期的，剩余50元是4月1日到期的，若发生退款80元，则默认退的是4月1日到期的，然后再是3月1日到期的。

**2、Q：赊账还款时能不能优惠？**

A：不能。目前只支持赊账时给消费者优惠，还款不能优惠。若应还100，实际还了50，则剩余50不是优惠掉了，是没有还完。

**3、Q：非会员可以在POS端赊账吗？**

A： 不可以，赊账是会员业务且需关联客户。

**4、Q：客户原来有一个账套，产生了一些赊账记录，购买了一个新账套，如何导入期初赊账数据？**

A：不支持导入。

**会员资料、业务及权益管理**

**1、Q：会员卡如果购买了付费会员，过期后会员卡的余额会自动清零吗？**

A：不会。余额还在，只是不再享有权益。

**2、Q：后台/POS新增会员，入会方式支持新增吗？**

A：不支持。仅支持营业员发展/会员推荐两种方式。

**3、Q：同一个手机号支持在多个门店同时开通会员吗？**

A：不支持。星辰会员是基于商户的，不是基于门店的。

**4、Q：在会员列表批量选择了10个会员进行授信，但是授信记录只有3条，是什么原因呢？**

A：先检查所选会员是否全都关联了客户，若没有关联客户，是无法进行授信的。若全都关联了客户，则再反馈研发这边排查是否bug。

**5、Q：POS会员绑定的关联客客户有什么作用，可以通过这个客户查询零售信息吗 ？**

A：会员关联客户是为了会员赊账可以在应收账款明细中根据客户区分统计。

**6、Q：客户买了AB两个账套，A和B账套下的门店，它们的会员数据是共用的吗？**

A：不是，是完全独立的，A账套下门店的会员与B账套下门店的会员互相不可使用。

**7、Q：后台会员信息列表的可用积分跟累计积分不一致，是什么原因？**

A：会员信息通过导入的方式导入了可用积分，未导入累计积分；累计积分为0，只会统计后续在POS开单获取的积分。

**8、Q：会员可以控制在某个门店使用，其他门店不能使用吗？**

A：会员都是全门店共用的，目前只能设置会员开卡门店。

**9、Q：会员级别折扣设置多个方案，同一个商品在A方案中设置了排除，在B方案中设置不排除，以哪个为准？**

A：按照不排除方案生效会员级别折扣。

**10、Q：在门店零售价给某商品设置了会员价，然后再到会员管理设置会员级别折扣方案，设置会员折扣优先会员价，排除该商品，POS开单时这个商品究竟是否打折呢？**

A：此时会按照后台设置的会员价进行优惠开单。

**11、Q：后台可以删除会员数据吗？**

A：会员未发生任何业务数据（储值业务或零售开单业务）时可以删除，若发生过储值或零售开单，产生了业务数据，则不支持删除会员。如果客户强行要删除，可以参考以下方法：方法1—登录后台前往【系统设置-基础设置-重新初始化】选择初始化范围为“全部资料”，可以删除全部会员数据；方法2—走数据提单，让开发提单给运维，在数据库删除数据。注意：方法1>方法2

**12.Q：POS会员模块的会员修改密码与收银员修改密码有什么区别？**

A：相同点—都是收银员给会员修改密码。不同点—收银员修改密码无需会员验证，一般应用在商家与顾客相互信任的场景；会员修改密码需要会员验证，保证密码安全。

**13、Q：可以自定义设置会员卡号吗？**

A：系统自动生成的会员卡号不支持自定义，可以另外设置自定义卡号，如果客户自己有实体会员卡，可以将实体会员卡号维护到自定义卡号中。

**14、Q：POS端会员关联实体IC卡功能，一个会员支持关联多张IC卡吗？**

A：不支持。一个会员只能关联一张IC卡，关联后再次关联时，原来已关联的IC卡会失效，以新关联的IC卡为准。

**15、Q：POS端会员关联实体IC卡后，零售开单时刷卡没有任何反应，是什么原因呢？**

A：刷卡时需要打开选择会员弹层，点击【识别IC卡】即可刷卡识别会员。

**微信公众号（服务号）绑定、微信会员卡及微信公众号消息提醒**

**1、Q：我完成微信授权了，为什么没有微信会员卡二维码？**

A：完成微信授权后，需要再完成微信会员卡设置，微信会员卡二维码就会自动生成。

**2、Q：我在设置微信会员卡时，点击保存提示“提交微信异常api unauthorized hint：[hjN40a00228128]”或“提交微信异常function category of API need be confirmed by componet hint：[n15sha0344r283]”是什么意思，怎么处理？**

A：登录微信公众平台，检查是否完成微信认证并开通卡券功能，需要完成微信认证并开通卡券功能，再到后台设置微信会员卡。

**3、Q：微信会员卡二维码支持下载吗？**

A：不支持。建议方案—鼠标放到二维码图片上，单击右键，选择“图片另存为”保存到本地时，图片文件名称后面加一个.jpg，保存即可。

**4、Q：微信会员授权后，会员卡界面的开发者appsecret需要填吗？**

A：选填。

**5、Q：商家开通了微信模板消息，顾客消费时收不到微信模板消息提醒，是什么原因？**

A：先确认顾客是否有关注商家的微信公众号，只有关注公众号才能收到消费的模板消息。若顾客已关注，则前往【微信公众平台-广告与服务-模板消息】点击【模板库】检查所选行业是否“IT科技/互联网|电子商务”，若非此行业，可能存在推送失败的情况，看客户是否可以换成“IT科技/互联网|电子商务”，但更换行业后原行业的模板消息将会被删除，需要重新添加。

**6、Q：会员卡绑定微信公众号时是否需要设置微信支付？**

A：微信公众号上的微信支付不是必须的，开通微信公众号微信支付的作用如下：1）微信支付（商户功能），是公众平台向有出售物品需求的公众号提供推广销售、支付收款、经营分析的整套解决方案；2）商户通过自定义菜单、关键字回复等方式向订阅用户推送商品消息，用户可在微信公众号中完成选购支付的流程；3）商户也可以把商品网页生成二维码，张贴在线下的场景，如车站和广告海报。用户扫描后可打开商品详情，在微信中直接购买。

**7、Q：会员在门店消费后，为什么没有收到消费的微信模板消息？**

A：登录微信公众平台，检查是否完成微信认证并开通模板消息，需要完成微信认证并开通模板消息功能，同时引导会员关注商家的微信公众号。

**8、Q：如果客户没有绑定微信公众平台，可以给会员发优惠券信息和会员消费信息吗？**

A：不可以。客户一定要绑定公众号，完成微信认证，开通模板消息。

**9、Q：目前星辰零售支持会员消费后给会员发送微信模板消息与短信提醒吗？**

A：支持。前者需要在微信公众平台完成微信认证、开通模板消息，且会员要关注商家的微信公众平台；后者请前往后台【营销管理-短信】购买短信服务。
