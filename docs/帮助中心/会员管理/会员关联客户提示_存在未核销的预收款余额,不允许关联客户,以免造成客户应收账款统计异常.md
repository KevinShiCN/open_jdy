# 会员关联客户提示:存在未核销的预收款余额,不允许关联客户,以免造成客户应收账款统计异常

> 作者：shaoling | 阅读：58 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/722619557323522816)

【问题描述】

会员关联客户提示:存在未核销的预收款余额,不允许关联客户,以免造成客户应收账款统计异常

【解释说明】

会员关联客户时出现 “存在未核销的预收款余额,不允许关联客户,以免造成客户应收账款统计异常” 的提示,是因为被关联客户存在未核销的预收款余额,需先处理该款项。

![](https://vip.kingdee.com/download/010974e256bc9e39422184582b0e24e568bd.png)

若要关联客户,建议先完成预收款核销再操作,操作详情可查看:[预收款有哪些核销方式?](https://vip.kingdee.com/knowledge/721537571075163136?productLineId=35&isKnowledge=2&lang=zh-CN)

若您想要了解查询预付或预收款的核销情况问题,请点击:[如何查询预付款单/预收款单的核销情况?](https://vip.kingdee.com/knowledge/618974029239744000?productLineId=35&isKnowledge=2&lang=zh-CN)
