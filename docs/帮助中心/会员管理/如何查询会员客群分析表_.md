# 如何查询会员客群分析表?

> 产品：金蝶云·星辰 | 分类：会员管理 | 类型：常见问题
> 作者：lykk | 阅读：87 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/652314274894462976)

【问题描述】

如何查询会员客群分析表?

【业务场景】

根据对会员历史消费情况,对会员入会门店、消费门店、入会渠道、会员级别等进行统计分析,提升门店的营业能力

【解决方案】

路径:【会员管理】-【会员客群分析表】

![](https://vip.kingdee.com/download/010972fcd5702f2b46ddbbc1f3628168a1a7.png)

报表中包含以下字段:

| 字段名称 | 解释说明 |
| --- | --- |
| 消费门店 | 会员实际消费的门店 |
| 会员累计消费次数 | 会员业务中业务类型为“零售单销售”单据的累计数量 |
| 会员累计消费金额 | 会员业务中业务类型为“零售单销售”单据的总金额 |
| 会员累计消费商品数量 | 会员业务中业务类型为“零售单销售”单据的商品总数量 |
| 会员客单价(消费次数维度) | 会员累计消费金额/会员累计消费次数 |
| 会员客单价(会员数量维度) | 会员累计消费金额/会员数量 |
| 会员件单价 | 会员累计消费金额/会员累计消费商品数量 |
| 会员客单量 | 会员累计消费商品数量/会员累计消费次数 |
| 会员品类连带率 | 会员累计消费商品品类数量/会员累计消费次数 |
| 会员贡献率 | 会员累计消费金额/门店经营查询开始日期的会员数量 |
| 会员复购频次 | 会员平均消费次数-1;会员平均消费次数=会员累计消费次数/发生交易会员数 |
| 会员复购率 | 门店经营查询期间内有二次消费记录的会员数量/门店经营查询开始日期的会员数量 |
| 会员储值率 | 发生储值会员数量/会员数量 |
| 会员动销率 | 会员累计消费商品数量/当前时点商品上架数量 |
| 会员促销触达率 | 含有零售促销活动的会员消费次数/会员累计消费次数 |
| 发生交易会员数 | 有消费记录的会员数量 |
| 无交易会员数 | 无有消费记录的会员数量 |
| 发生交易会员比例 | 发生交易会员数/(发生交易会员数+无交易会员数) |
| 无交易会员比例 | 无交易会员数/(发生交易会员数+无交易会员数) |
| 会员增长率 | 查询期间内会员增长数/门店经营查询开始日期的会员数量 |
| 会员流失率 | 查询期间内无购买记录会员数量/门店经营查询开始日期的会员数量 |
| 新会员数量 | 门店经营查询结束日期的会员数量-门店经营查询开始日期的会员数量 |
| 新会员复购率 | 查询期间内有二次消费记录的新会员数量/新会员数量 |
