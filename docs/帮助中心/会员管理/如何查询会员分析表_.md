# 如何查询会员分析表?

> 产品：金蝶云·星辰 | 分类：会员管理 | 类型：常见问题
> 作者：lykk | 阅读：179 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/652314284927238144)

【问题描述】

如何查询会员分析表?

【业务场景】

根据对会员历史消费情况,对会员个人进行统计分析,可直接将结果下推至精准营销,持续挖掘会员价值

【解决方案】

路径:【会员管理】-【会员分析表】

![](https://vip.kingdee.com/download/010993258eeb62bd4a648991b4f066c6dd61.png)

![](https://vip.kingdee.com/download/01098860d25c5d1348f88b6ffce34956fffc.png)

报表中字段如下:

| 字段名称 | 解释说明 |
| --- | --- |
| 会员名称 | 会员信息设置的名称 |
| 会员手机号 | 会员信息设置的手机号 |
| 开卡门店 | 创建绑定会员卡的门店 |
| 入会时间 | 创建绑定会员卡的时间 |
| 会员级别 | 会员等级 |
| 性别 | 会员信息设置的性别 |
| 标签 | 会员信息所贴的标签 |
| 关联客户 | 会员绑定的后台客户 |
| 消费门店 | 实际购买消费的门店 |
| 累计消费次数 | 业务类型为“零售单销售”单据的累计数量 |
| 累计消费金额 | 业务类型为“零售单销售”单据的总金额 |
| 累计消费商品数量 | 业务类型为“零售单销售”单据的商品总数量 |
| 累计储值金额 | 会员累计储值金额 |
| 累计积分数量 | 会员累计积分数量 |
| 累计积分兑换商品数量 | 会员累计通过积分兑换商品的数量 |
| 平均消费商品数量(次) | 累计消费商品数量/累计消费次数 |
| 平均消费金额(次) | 累计消费金额/累计消费次数;等同于会员自身客单价 |
| 消费频次(日) | 累计消费次数/消费日期区间内的天数 |
| 平均消费商品数量(日) | 累计消费商品数量/消费日期区间内的天数 |
| 平均消费金额(日) | 累计消费金额/消费日期区间内的天数 |
| 促销消费率 | 含有零售促销活动的消费次数/累计消费次数 |
| 储值消费率 | 含有储值核销的消费次数/累计消费次数 |
| 积分消费率 | 含有积分抵现的消费次数/累计消费次数 |
| 累计退货单量 | 同一张零售单多次退货记一退货单量,无源单退货记一退货单量 |
| 累计退货次数 | 业务单据类型为“零售单退货”单据的累计数量 |
| 累次退货金额 | 业务单据类型为“零售单退货”单据的总金额 |
| 累计退货商品数量 | 业务单据类型为“零售单退货”单据的商品总数量 |
| 退货率(单量维度) | 累计退货单量/(累计消费次数+累计退货单量) |
| 退货率(次数维度) | 累计退货次数/(累计消费次数+累计退货次数) |
| 退货率(金额维度) | 累计退货金额/(累计消费金额+累计退货金额) |
| 最近消费日期 | 同查询消费日期无关联 |
| 距今N天无消费记录 | 系统当前日期-最近消费日期(同查询消费日期无关联) |
| 偏好商品 | 累计购买数量前三名的商品(同查询消费日期无关联) |
