# 导入固定资产提示年初累计折旧+本年累计折旧调整+本年计提折旧不等于累计折旧,请重新录入!怎么处理?

【问题描述】

如下图所示,导入固定资产提示年初累计折旧+本年累计折旧调整+本年计提折旧不等于累计折旧,请重新录入!怎么处理?

![](https://vip.kingdee.com/download/01097acb0c14c70640d1bc4d7f9a846da2c9.png)
【解释说明】

引入资产,需注意年初值的计算:

1)年初原值=原值(本位币金额)-本年原值调整;

2)年初数量=数量-本年数量调整;

3)年初减值准备=减值准备-本年减值准备调整;

4)年初累计折旧=累计折旧-本年计提折旧(或本年累计折旧调整)

【解决方案】

如下图所示,导入模板年初原值=原值(本位币金额)-本年原值调整;其他字段同理。
