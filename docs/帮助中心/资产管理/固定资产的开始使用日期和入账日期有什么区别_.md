# 固定资产的开始使用日期和入账日期有什么区别?

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：常见问题
> 作者：姜宁8 | 阅读：2814 | 有用：9
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352128411380736)

【问题描述】

固定资产卡片中,开始使用日期与入账日期有什么区别?

【解释说明】

**开始使用日期:**指固定资产实际开始使用的日期,填了后,已折旧期间数可自动计算出来。可填以前期间的日期。

**入账日期:**指入星辰账的日期,即卡片录入系统后,第一次在星辰系统记账的日期。

**入账期间:**按入账日期计算带出。
