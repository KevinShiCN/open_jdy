# 资产列表中选择已清理的资产的时候显示原值资产原值为0,点击资产详情查看又有原值,是什么原因?

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：常见问题
> 作者：_Leo | 阅读：380 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/396106046365822208)

#### 【问题描述】

在资产列表中筛选有原值的资产的时候,显示的是0,实际卡片是有原值的。如图:

![](https://vip.kingdee.com/download/010960b22d317a834a35b7251064388d9e88.png)

【解释说明】

目前系统规则是:如果资产已经清理了,不会在资产列表中显示原值信息,防止统计的时候在列表统计进去。

【温馨提示】

如果需要批量进行查看,可在资产清理列表中,可以看到对应的原值。如图:
![](https://vip.kingdee.com/download/010928d53ec47793481d9d7ce43fa9131a41.png)
