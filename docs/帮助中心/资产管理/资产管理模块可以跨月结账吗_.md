# 资产管理模块可以跨月结账吗?

【问题描述】

资产管理模块可以跨月结账吗?

【解释说明】

资产管理每期需要先计提折旧才能结账,计提折旧也只能计提当期,所以资产模块不支持跨月多期一起结账。
