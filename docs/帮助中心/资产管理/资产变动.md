# 资产变动

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：整体介绍, 快速入门
> 作者：Zyl | 阅读：3354 | 有用：11
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/23552)

1、对已经入账的资产，需要对原值、折旧、部门、折旧方法等进行调整，可进行资产变动，整卡片信息。

2、资产变动单有三个入口：

入口一：【资产管理】→【资产变动】，可打开一张新的资产变动单界面，如下图：

![](https://vip.kingdee.com/download/01004b24fdfcfa904ea890540840816d511d.png)



入口二：【资产列表】选择资产后点击“变动”，可携带所选资产到一张新的资产变动单界面。

入口三：【资产变动单列表】点击新增，可打开一张新的资产变动单界面

3、通过【变动项】，选择需要变动的内容，通过【资产编码】选择需要变动的资产。所选择的变动项，变动后和变动前的值不能相同。

4、资产变动单需要选择变动项目和变动的资产。可批量变动资产，也可单个变动。批量变动时，所选资产的变动项必须相同。

5、当期新增的资产，当期不能变动。

6、 一张卡片一期只能变动一次，如需再次变动，可删除变动单后，重新新增变动单，或打开变动单修改变动记录。

7、 卡片变动后，不调整以前期间累计折旧，按净值和剩余使用期间来计提后续折旧。

8、 卡片当期变动后，不允许再做清理和拆分。

9、 变动单生成凭证后，必须要删除凭证，才能编辑和删除变动单。

10、资产变动单列表，显示所有的变动单记录，可对变动单进行删除、引出的操作
