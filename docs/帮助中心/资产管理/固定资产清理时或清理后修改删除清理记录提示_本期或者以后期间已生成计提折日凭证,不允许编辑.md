# 固定资产清理时或清理后修改删除清理记录提示:本期或者以后期间已生成计提折日凭证,不允许编辑

> 作者：云社区用户u1tk1853 | 阅读：242 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/633107465185545216)

【问题描述】

固定资产清理时或清理后修改删除清理记录提示:本期或者以后期间已生成计提折日凭证,不允许编辑

【解释说明】

1、若当期或以后期间已经生成了计提折旧的凭证,固定资产不允许清理,需删除折旧凭证或结账到未生成计提折旧凭证的期间再操作清理

![](https://vip.kingdee.com/download/0109709714b2cdcc4a6d8a613b2df0fb411b.png)

2、固定资产当期清理后计提了折旧,清理记录无法直接删除/修改,需删除当期的折旧凭证后才可修改或删除清理记录,修改或删除后重新计提折旧

![](https://vip.kingdee.com/download/0109df010436eba54658b025856a86f001bb.png)

删除折旧凭证参考:[怎样删除系统计提折旧凭证?](https://vip.kingdee.com/article/585551162791770112?productLineId=35&isKnowledge=2&lang=zh-CN)
