# 如何在资产列表查询或者上传附件?

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：常见问题
> 作者：云社区用户0531183 | 阅读：62 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/690733624269662720)

【问题描述】

如何在资产列表查询或者上传附件?

【操作步骤】

1.点击【资产管理】>【资产新增】>【查询】打开资产列表。

![](https://vip.kingdee.com/download/0109aeccc5d687de408c803e6bfe4f22c058.png)
2.找到“附件及附件数”列。点击“+”号,点击“本地导入”

![](https://vip.kingdee.com/download/01096f0f672e51094c12a549a9efd6de98e0.png)
3.上传文件并导入

![](https://vip.kingdee.com/download/01094ad1f9beb2db403a994f83f7b6a6732c.png)
4. 已上传附件的资产会显示附件数,点击可以查看,并且可以继续添加附件。

![](https://vip.kingdee.com/download/010936eb31bfbf034b649e0a26713c537fef.png)
