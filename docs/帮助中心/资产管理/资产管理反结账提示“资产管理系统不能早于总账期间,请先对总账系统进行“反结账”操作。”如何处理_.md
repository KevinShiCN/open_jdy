# 资产管理反结账提示“资产管理系统不能早于总账期间,请先对总账系统进行“反结账”操作。”如何处理?

> 作者：shaoling | 阅读：65 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/703054779706296832)

【问题描述】

资产管理反结账提示“资产管理系统不能早于总账期间,请先对总账系统进行“反结账”操作。”如何处理?

![](https://vip.kingdee.com/download/01097b4a5bba68c9430d8e67c998fdaa1bdb.png)

【解释说明】

当资产模块反结账提示“资产管理系统不能早于总账期间,请先对总账系统进行“反结账”操作。”,可先把总账模块反结账操作后,再对资产模块进行反结账

这是由于在【系统设置】>【系统参数】>【资产参数】>【资产管理参数】里选择“与总账连用”且未选择“与总账一起结账”的情况下,

总账在结账前会要求资产必须先结账,资产反结账也同样要求总账先反结账操作
![](https://vip.kingdee.com/download/010995d87ee062c9405f9c38ff06985db784.png)
【操作步骤】

1、总账反结账

单击【账务处理】>【期末结账】,进入“期末结账”页面。如下图所示:

![](https://vip.kingdee.com/download/01091db77896e623431492682c71be25c093.png)
在期末结账页面点击<反结账>
![](https://vip.kingdee.com/download/01090529f6388830417d8b976d2e488163fc.png)
选择对应期间进行反结账
![](https://vip.kingdee.com/download/01099b12b902a3bf46cdbd77a66b097eaa2a.png)
提示反结账到对应的期间点击<确定>

![](https://vip.kingdee.com/download/01098087bcfce8de4b71bd70759072809067.png)
总账反结账成功

![](https://vip.kingdee.com/download/0109ab1199ee20fe4a578d305a3e521b96fd.png)

2、资产反结账

单击【资产管理】>【结账】,进入“结账”页面。如下图所示:

![](https://vip.kingdee.com/download/01098b0c627e4656447fb5bae59f807c6632.png)
在资产结账页面点击<反结账>
![](https://vip.kingdee.com/download/0109912543305ea343238d91871044c1811c.png)
选择对应期间进行反结账
![](https://vip.kingdee.com/download/0109dfcfca902f5943daa7d4098fd36f859b.png)

提示反结账到对应的期间点击<确定>

![](https://vip.kingdee.com/download/0109ea1aceee31e04e908c078ec77d4a9007.png)资产反结账成功

![](https://vip.kingdee.com/download/01093ab6a1b26b2f4105bc06ae430fa1d12f.png)
