# 如何修改资产类别编码?

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：常见问题
> 作者：云社区用户5j6u671 | 阅读：320 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/611726439414298880)

【问题描述】

如何修改资产类别编码?

【解决方案】

操作路径:【资产管理】-【资产类别】

![](https://vip.kingdee.com/download/01091e3cd1682c804374aa5ddea93b3f8065.png)

点资产编码进来就可以修改资产类别编码,修改完以后点保存即可。

![](https://vip.kingdee.com/download/01091957c45fe43145b1bd09fbb02c5bc6db.png)
如果点资产编码进来是灰色,需要检查资产类别的编码规则。

![](https://vip.kingdee.com/download/010903a5490483b142f4aabbb6f8e7605f77.png)
路径:【系统设置】-【编码规则】,应用规则选:新增显示且允许修改即可。

![](https://vip.kingdee.com/download/0109d9317ef64c464e629e4bad9b78e3cb6d.png)
