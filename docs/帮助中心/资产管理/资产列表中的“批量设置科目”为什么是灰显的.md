# 资产列表中的“批量设置科目”为什么是灰显的

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：进阶技巧, 常见问题
> 作者：Zyl | 阅读：1115 | 有用：12
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/65100185054855424)

问题描述：

资产列表中的“批量设置科目”为什么是灰显的

解决方案：

此功能用于处理之前不与总账连用，录入卡片后，修改系统参数为“与总账连用”，保存参数或计提折旧时，提示需要补充录入历史卡片的相关科目时才可使用。
