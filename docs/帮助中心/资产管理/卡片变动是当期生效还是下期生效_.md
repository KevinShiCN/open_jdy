# 卡片变动是当期生效还是下期生效?

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：1643 | 有用：7
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/129339770080873013)

【问题描述】

卡片变动是当期生效还是下期生效?

【解释说明】

固定资产发生变动后,将影响当期及后续期间的折旧数据,变动前已生成的折旧凭证数据不受影响。

1、如果卡片变动的是与折旧金额无关的项目,例如:折旧费用分配科目,那么变动当期就会生效。

例如下面这张卡片,新增的时候,折旧费用分配科目选的是“管理费用”,但是本期变动为“销售费用”,那么在折旧生成凭证的时候,借方科目在当期就会变成销售费用,如下图所示:

![](https://vip.kingdee.com/download/01096f656ad9cf864ddabd85d556d73d952b.png)

折旧费用分配科目的变动,本期立即生效:

![](https://vip.kingdee.com/download/0109b036b36ee483408faae698fe613b0993.png)

2、如果资产变动的项目是会影响折旧金额计算的字段,例如:原值、预计折旧期数、净残值、累计折旧(按净值计提的情况下),则会根据资产类别中的折旧政策来控制,

情形一:

如果选择"**新增当期计提,清理当期不计提**",则此类资产变动是**当期**造成影响;

情形二:

如果选择"**新增当期不计提,清理当期计提**",则此类资产变动是**下期**造成影响。

![](https://vip.kingdee.com/download/0109710e65d8f035470c84130eb556743980.png)

【温馨提示】

卡片变动前当期不能计提折旧,若已经计提折旧,需要将折旧凭证删除,卡片变动完成后,再重新进行计提。
