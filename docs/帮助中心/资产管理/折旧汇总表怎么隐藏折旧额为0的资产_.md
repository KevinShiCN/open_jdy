# 折旧汇总表怎么隐藏折旧额为0的资产?

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：常见问题
> 作者：云社区用户55os622 | 阅读：69 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/627671593094890752)

【问题描述】
折旧汇总表怎么隐藏折旧额为0的资产?

【解释说明】

依次点击资产管理-折旧汇总表,进入折旧汇总表查询界面

![](https://vip.kingdee.com/download/010906eabb190f2a47879b196e6b3249c751.png)

取消勾选:包括本期折旧为0的资产 参数, 再点击查询,即可隐藏/过滤掉本期折旧额为0的资产

![](https://vip.kingdee.com/download/0109f9807c771ad04358946774d020233e6f.png)
