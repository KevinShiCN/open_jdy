# 已清理的固定资产支持修改吗?

> 作者：杨桂平 | 阅读：147 | 有用：0

【问题描述】

已清理的固定资产支持修改吗?

【解释说明】

已清理的固定资产不支持修改,卡片当期清理后,不能再变动、拆分且清理卡片原值默认为0
