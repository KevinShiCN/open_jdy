# 资产管理系统的主要业务流程是什么?

【问题描述】

资产管理系统的主要业务流程是什么?

【操作步骤】
资产管理的业务流程图如下:

![](https://vip.kingdee.com/download/01096339bede2f594138ba8ae69bfc270b4a.png)
**一、资产日常处理**

**1、资产新增**

两种卡片新增方式:
   1)启用账套前已经购入的资产卡片,需要开启【初始化资产】按钮
   2)启用账套后购入的资产卡片,直接录入卡片信息即可。**
**

![](https://vip.kingdee.com/download/0109f73f0e8c7a9545a4b4b948a8156df6e3.png)

【温馨提示】

[固定资产卡片如何生成凭证?](https://vip.kingdee.com/knowledge/129339770097650215?productLineId=35&isKnowledge=2&lang=zh-CN)

**2、资产拆分**
已入账的资产进行部分变动或清理时,可先把需要变动或清理的部分数量拆分出来,再进行后续操作。
   1)当期新增的卡片,当期不能做拆分。
   2)在【资产编码】中选择资产进行拆分,输入拆分数量,拆分数量输入多少,就会拆分成多少张新卡片。拆分数量必须是大于1的整数。卡片拆分后,原卡片原值、累计折旧、减值准备、净残值、净值全部自动清零。
   3)拆分后,可通过【资产列表】,选择拆分前卡片,取消拆分。
   4)当期做过变动/清理的卡片,不能再次拆分;当期拆分后的资产,可以变动/清理。

![](https://vip.kingdee.com/download/0109189c831977ac41b9827ee2e3a6a67d85.png)
**3、资产变动**

对已经入账的资产,需要对原值、折旧、部门、折旧方法等进行调整,可进行资产变动,整卡片信息。
   1)可批量变动资产,也可单个变动;批量变动时,所选资产的变动项必须相同。
   2)当期新增的资产,当期不能变动。
   3)一张卡片一期只能变动一次,如需再次变动,可删除变动单后,重新新增变动单,或打开变动单修改变动记录。
   4)卡片变动后,不调整以前期间累计折旧,按净值和剩余使用期间来计提后续折旧。
   5)卡片当期变动后,不允许再做清理和拆分。
   6)变动单生成凭证后,必须要删除凭证,才能编辑和删除变动单。
   7)资产变动单列表,显示所有的变动单记录,可对变动单进行删除、引出的操作。**
**

![](https://vip.kingdee.com/download/010937a75e2375de424db1b5c8ad3b8c087a.png)
**4、资产清理
**已入账的资产,出现丢失、损坏或出售,则可进行资产清理。
   1)本期新增的卡片,本期不能清理。
   2)可批量清理,也可清理单一卡片;卡片可部分清理,也可全部清理
   3) 一张卡片一期只能清理一次,如需再次清理,可删除清理单后,重新新增清理单,或打开清理单修改。
   4)卡片当期清理后,不能再变动、拆分。
   5) 清理单生成凭证后,必须要删除凭证,才能编辑和删除清理单。
![](https://vip.kingdee.com/download/010945342a7416b644fb87c77797acacc9da.png)
**二、折旧管理、查表**

**1、折旧摊销 **

系统提供四种折旧方法:年限平均法、年数总和法、双倍余额递减法、工作量法,另外,还支持单张卡片不折旧,和整个系统不折旧。**
**

![](https://vip.kingdee.com/download/0109e63630a9713c4c4dbba223218dbce22c.png)![](https://vip.kingdee.com/download/0109334cceb3096d4e5bab3e1600c3b25bb6.png)
**2、资产报表**

折旧明细表:可按期查询所有入账的资产折旧情况。
折旧汇总表:可按期查询所有入账的资产折旧情况。
折旧分配表:可以查询折旧分配的情况。

![](https://vip.kingdee.com/download/0109ec0037ba276c4d059030c469c542bbbb.png)
**三、资产结账**

期末结账处理:资产不与总账连用或资产与总账连用,但不与总账一起结账时,资产管理模块提供期末结账的功能。
