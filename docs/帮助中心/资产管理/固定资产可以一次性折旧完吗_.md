# 固定资产可以一次性折旧完吗?

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：608 | 有用：0

【业务场景】

固定资产可以一次性折旧完吗?

【解决方案】

固定资产卡片的“预计折旧期间数”录“1”即可一次计提完所有的折旧。
