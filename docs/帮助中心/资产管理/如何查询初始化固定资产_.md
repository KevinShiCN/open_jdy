# 如何查询初始化固定资产?

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：常见问题
> 作者：shaoling | 阅读：90 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/703773509733921280)

【问题描述】

如何查询初始化固定资产?

【操作流程】

从【资产管理】>【资产新增】进入资产列表页面

![](https://vip.kingdee.com/download/010984d78646b1a84e4895a6bc44acf2ae52.png)
在资产列表里<展开过滤>

![](https://vip.kingdee.com/download/0109a3dc7516e58f4465ad970bbfa7506ddb.png)“初始化资产”筛选“是”进行查询

![](https://vip.kingdee.com/download/0109beba18fda9034587a620e9f1cb9f1c8e.png)
筛选初始化资产查询后即可查询到初始化固定资产
![](https://vip.kingdee.com/download/010924e47fbf77234a738f9e4f1570da5bfe.png)
