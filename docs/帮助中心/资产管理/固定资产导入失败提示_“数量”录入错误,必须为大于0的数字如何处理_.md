# 固定资产导入失败提示:“数量”录入错误,必须为大于0的数字如何处理?

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：常见问题
> 作者：happy忆霖 | 阅读：41 | 有用：1

【问题描述】

固定资产导入失败提示:“数量”录入错误,必须为大于0的数字如何处理?

【业务场景】

导入固定资产卡片的时候,提示“数量”必须大于0,是什么原因?

![](https://vip.kingdee.com/download/0109dd1339ce0a274cdf9dca8dbccc4af5b9.png)

**若需要了解如何导入固定资产,请点击:[](https://vip.kingdee.com/knowledge/581648086968067840?productLineId=35&isKnowledge=2&lang=zh-CN)[资产卡片如何导入?](https://vip.kingdee.com/knowledge/581648086968067840?productLineId=35&isKnowledge=2&lang=zh-CN)**

【解释说明】

**固定资产导入时,“数量”字段为必录字段,且数值必须为正整数,不支持负数、0、小数。**

**因此提示“数量”必须大于0的主要原因是由于导入的时候模板中的“数量”字段的数据输入成负数了,需要修改为正整数后,重新进行导入即可。**
