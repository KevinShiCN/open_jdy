# 资产卡片能新增自定义字段吗?

【问题描述】

资产卡片能新增自定义字段吗?

【解释说明】

资产卡片不支持添加自定义字段。如有额外内容需要标注,请填写在资产卡片的备注栏或其他无需使用的字段中。
