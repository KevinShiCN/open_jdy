# 资产管理

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：快速入门, 功能介绍
> 作者：Simone邓 | 阅读：7979 | 有用：8
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/632657481763457792)

资产管理模块涉及到固定资产的购置、使用、维护、折旧和处置等多个方面业务处理。业务流程如下：

![](https://vip.kingdee.com/download/0109c4cf1bcee25041b1923869ee53dbae1f.png)

| 步骤 | 具体说明 | 学习链接 |
| --- | --- | --- |
| 1.参数设置 |  | —— |
| 1.1资产参数 | 可以对资产管理系统的业务处理规则和处理惯例进行统一的设置和管理。分为基础参数和资产管理参数。 | —— |
| 1.1.1基础参数 | 启用期间：即建账时设置的系统启用期间,刚启用的账套,如果没有录入任何数据,可以修改启用期间,。若已录入数据如需要修改,系统会给出对应提示,根据更改到的期间进行业务数据的判断。当前期间：当前资产管理所在期间,仅显示,不允许修改,由系统自动判断显示。 | 点击学习 |
| 1.1.2资产管理参数 | (1)与总账连用用于控制资产管理系统与总账的关联，勾选此参数，则资产管理系统与总账相关联，资产管理系统卡片及折旧等信息均需生成凭证传递到总账 |  点击学习 |
| (2)与总账一起结账仅在与总账连用时可设置，当与总账连用时，资产管理系统即可单独结账，也可勾选此参数，与总账一起结账，则在账务处理中进行期末结账操作时，会同时进行资产管理系统结账处理 |    点击学习   |  |
| (3)不折旧（整个系统）用于控制资产管理系统中录入的资产是否需要进行计提折旧，如果勾选，则资产管理系统所有资产均不进行折旧计提，录入资产信息时，也不需要录入折旧相关信息 |    点击学习   |  |
| (4)结账自动计提折旧用于控制资产管理系是手动操作生成折旧凭证还是月末结账时自动生成折旧凭证 |  点击学习 |  |
| 1.2资产资料 |  | —— |
| 1.2.1资产类别 | 根据资产作用性质等不同因素进行进行归类划分，系统有预置的资产类别，也支持自定义新增。 | 点击学习 |
| 1.2.2变动方式 | 固定资产发生新增、变动或减少的方式 |   点击学习  |
| 1.2.3存放地点 | 资产存放的位置 |  点击学习  |
| 1.2.4使用状态 | 根据不同的分类标准和目的对资产进行状态划分 | 点击学习 |
| 2.资产日常处理 | 登记、变动、清理、拆分 |  点击学习   |
| 2.1资产新增 | 资产新增通常指的是企业购买新的资产需要登记入账 | 点击学习 |
| 2.2资产变动 | 资产变动通常指的是个人或机构持有的资产价值、数量或类型的变化 |   点击学习   |
| 2.3资产清理 | 资产清理通常是指对公司持有的资产进行出售、报废或其他形式的处理 |     点击学习     |
| 2.4资产拆分 | 资产拆分通常指的是出于各种原因需要将一个较大的资产分解成几个较小的部分分别管理 |  点击学习 |
| 3.折旧管理 |  | —— |
| 3.1计提折旧 | 折旧是资产在其使用期间价值逐渐减少的过程，通常通过计提折旧来反映 | 点击学习 |
| 3.2报表查询 | 折旧明细表、折旧汇总表和折旧分配表是记录和分析资产折旧的重要工具 | —— |
| 3.2.1折旧明细表 | 记录每项固定资产折旧情况的详细报表 |     点击学习     |
| 3.2.2折旧汇总表 | 展示在一定期间内所有固定资产的总体折旧情况 |     点击学习     |
| 3.2.3折旧分配表 | 展示折旧费用分配到不同的成本中心或部门的报表 |   点击学习   |
| 4.结账 |  | —— |
| 4.1资产结账 | 资产结账是会计周期结束时进行的一项重要活动，其目的是确保所有资产的账面价值准确反映在财务报表中，并且符合会计准则和税务规定 | 点击学习 |
