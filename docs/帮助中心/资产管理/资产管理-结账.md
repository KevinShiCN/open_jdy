# 资产管理-结账

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：整体介绍, 快速入门
> 作者：Zyl | 阅读：2032 | 有用：7
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/23560)

1、资产不与总账连用或资产与总账连用，但不与总账一起结账时，资产管理模块提供期末结账的功能

2、单击【资产管理】→【期末处理】→【结账】，进入“期末结账”页面。如下图所示：

![](https://vip.kingdee.com/download/010063d9d760ec75408cb223ae99e47131e3.png)

3、在进行期末结账前，系统提供“结账前检查”功能，可以对期末结账前相关业务进行检查。检查项包括：初始化卡片是否录入、是否计提折旧、是否生成凭证。

1）初始化卡片是否录入：只预警和提醒，不强制要求

2）是否计提折旧：当存在需要计提折旧的卡片时，则需要判断是否计提折旧。如果没有计提折旧，则不允许进行期末结账

3）是否生成凭证：当资产与总账连用时，则需要判断卡片、变动单、清理单、折旧是否生成凭证。未生成凭证不允许进行期末结账。

4、单击检查项目，可以进入异常处理界面，进行相关业务处理。

5、单击“结账”，则进行期末结账。

6、系统提供反结账功能，操作与结账一致。
