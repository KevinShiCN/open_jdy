# 固定资产变动预计折旧期数,需要变动折旧依据吗?

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：常见问题
> 作者：Siwennn | 阅读：321 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/616075383577963520)

【问题描述】

固定资产变动预计折旧期数,需要变动折旧依据吗?

【解释说明】

当变动项影响折旧时,系统建议将折旧依据调整为按净值折旧,但用户也可选择不进行调整,系统不会强制要求此操作。

![](https://vip.kingdee.com/download/0109de51d21e896a4d22a667fd6292ddb259.png)

①如果不改变折旧方式,还是选择“按原值计算折旧”时,**变动后的月折旧=(原值-净值)/变动后的预计折旧期数。**

![](https://vip.kingdee.com/download/0109779dc73875ec4dda802d590d697db45f.png)

在最后期间提折旧时,系统将自动补提之前各期未足额提取的折旧额。

![](https://vip.kingdee.com/download/0109bb911e15711d4b11971bb35563494ed3.png)
②如果改变折旧方式,选择“按净值计算折旧”时,**变动后的月折旧额=(原值-累计折旧-净值)/(预计折旧期间数-已折旧期间数)**。

![](https://vip.kingdee.com/download/0109a41910d2792349c1bf6eb9f3af8127f9.png)
