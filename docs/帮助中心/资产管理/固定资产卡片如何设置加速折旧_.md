# 固定资产卡片如何设置加速折旧?

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：常见问题
> 作者：happy忆霖 | 阅读：105 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/717905971678548736)

【问题描述】

固定资产卡片如何设置加速折旧?

【业务场景】

新增的卡片/已经使用的卡片设置折旧方式的时候,是否支持设置加速折旧法?

【解释说明】

**目前系统支持新增卡片/变动卡片时,选择双倍余额递减减法、年数总和法的加速折旧方法:**

**1.双倍余额递减法(按原值)**

在该资产预计折旧年限到期的两年前:

月折旧率 = 2 ÷(预计折旧年限 * 一年中的会计期间数)*100%

月折旧额 =(年初原值-年初累计折旧)*月折旧率

**折旧年限到期前两年内(含两年):**

月折旧额 =(原值 - 累计折旧 - 减值准备 - 预计净残值)/ 剩余使用期间

**2.双倍余额递减法(按净值)**

在该资产预计折旧年限到期的两年前:

月折旧率 = 2 ÷(预计折旧年限 * 一年中的会计期间数)*100%

月折旧额 =(变动后年初原值 -变动后年初减值准备 -年初累计折旧)*月折旧率

**折旧年限到期前两年内(含两年):**

月折旧额 =(变动后原值-变动后累计折旧-变动后预计净残值-变动后减值准备)/剩余折旧期间数

**3.年数总和法(按原值)**

年折旧率 =(预计折旧年限-已折旧年限)/[预计折旧年限*(预计折旧年限+1)/2] *100%

月折旧率=年折旧率/一年中的会计期间数

年折旧额=(原值-预计净残值)*年折旧率

月折旧额 = 年折旧额 /一年中的会计期间数

**当按整年计算后,折旧年限不够一年时:**

月折旧额 =(原值 - 累计折旧 - 减值准备 - 预计净残值)/ 剩余可折旧期间

月折旧率 = 月折旧额 / 原值 * 100%

**4.年数总和法(按净值)**

年折旧率 =(预计折旧年限 - 已折旧年限)/ [预计折旧年限 *(预计折旧年限+1)/2] *100%

月折旧率 = 年折旧率 / 一年中的会计期间数

年折旧额 =(变动后原值-变动后减值准备-变动后预计净残值)*年折旧率

月折旧额 = 年折旧额 / 一年中的会计期间数

**当按整年计算后,折旧年限不够一年时:**

月折旧额 =(原值 - 累计折旧 - 减值准备 - 预计净残值)/ 剩余可折旧期间

月折旧率 = 月折旧额 / 原值 * 100%

【操作步骤】

**情况一:新增卡片时选择加速折旧**

在资产新增界面,点击“折旧方法”下的![](https://vip.kingdee.com/download/01097f76e65793344ee1bfa0043fe737d3e3.png)按钮,选择“双倍余额递减法”或者“年数总和法”,录入完其他信息后,点击右上角的<保存>

若需要了解如何新增卡片,请点击:[如何新增录入资产?](https://vip.kingdee.com/knowledge/566427467762433280?productLineId=35&isKnowledge=2&lang=zh-CN)

![](https://vip.kingdee.com/download/0109201ca680904a4970b48df83bb6754c1e.png)

**情况二:变动卡片时选择加速折旧**

在资产变动界面,点击变动项右侧的![](https://vip.kingdee.com/download/01099cdc7b5bc6a14314b01494210fe8e176.png)按钮,选择“折旧方法”,选择需要变动的资产和变动方式,选择“折旧方法”字段列下变动后的折旧方式,操作完成后,点击右上角的<保存>

若需要了解如何变动资产卡片,请点击:**[如何对资产进行修改变动?](https://vip.kingdee.com/knowledge/566065203545381120?productLineId=35&isKnowledge=2&lang=zh-CN)**
