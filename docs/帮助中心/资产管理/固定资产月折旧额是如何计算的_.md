# 固定资产月折旧额是如何计算的?

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：5465 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/632382964147621888)

【问题描述】

固定资产月折旧额是如何计算的?

【解释说明】

资产详情中的折旧方法,上方的小问号里面解释系统不同折旧方法对应不同的月折旧额计算方式。

![](https://vip.kingdee.com/download/01097850550f620f484ba5d25eacf137cb99.png)

1、年限平均法(按原值)
月折旧额 =(原值 - 预计净残值)÷ 预计折旧期间数
月折旧率 = 月折旧额 / 原值 * 100%

2、年限平均法(按净值)
月折旧额 =(变动后原值-变动后累计折旧-变动后减值准备-变
动后预计净残值)÷变动后剩余折旧期间数
月折旧率 = 月折旧额 / 原值 * 100%

3、双倍余额递减法(按原值)
在该资产预计折旧年限到期的两年前:
月折旧率 = 2 ÷(预计折旧年限 * 一年中的
会计期间数)*100%
月折旧额 =(年初原值-年初累计折旧)*月折旧率
折旧年限到期前两年内(含两年):
月折旧额 =(原值 - 累计折旧 - 减值准备 - 预计
净残值)/ 剩余使用期间

4、双倍余额递减法(按净值)
在该资产预计折旧年限到期的两年前:
月折旧率 = 2 ÷(预计折旧年限 * 一年中的
会计期间数)*100%
月折旧额 =(变动后年初原值 -变动后年初减值准备 -年初累
计折旧)*月折旧率
折旧年限到期前两年内(含两年):
月折旧额 =(变动后原值-变动后累计折旧-变动后预计净残值-
-变动后减值准备)/剩余折旧期间数

5、年数总和法(按原值)
年折旧率 =(预计折旧年限-已折旧年限)/[预计折旧年限*(
预计折旧年限+1)/2] *100%
月折旧率=年折旧率/一年中的会计期间数
年折旧额=(原值-预计净残值)*年折旧率
月折旧额 = 年折旧额 /一年中的会计期间数
当按整年计算后,折旧年限不够一年时:
月折旧额 =(原值 - 累计折旧 - 减值准备 - 预计净
残值)/ 剩余可折旧期间
月折旧率 = 月折旧额 / 原值 * 100%

6、年数总和法(按净值)
年折旧率 =(预计折旧年限 - 已折旧年限)
/ [预计折旧年限 *(预计折旧年限+1)/2] *100%
月折旧率 = 年折旧率 / 一年中的会计期间数
年折旧额 =(变动后原值-变动后减值准备-变动后预计净残值)
*年折旧率
月折旧额 = 年折旧额 / 一年中的会计期间数
当按整年计算后,折旧年限不够一年时:
月折旧额 =(原值 - 累计折旧 - 减值准备 - 预计净
残值)/ 剩余可折旧期间
月折旧率 = 月折旧额 / 原值 * 100%

7、工作量法(按原值)
单位折旧=(原值-净残值)/工作总量
月折旧额 = (原值-净残值)/工作总量×本月工作量

8、工作量法(按净值)
单位折旧=(原值-累计折旧-减值准备 - 入账净残值)/(入
账工作总量 - 入账已使用工作量)
月折旧额 = (调整后原值-调整后的累计折旧-调整后的减值准
备-调整后的净残值)/调整后剩余工作总量×本月工作量
