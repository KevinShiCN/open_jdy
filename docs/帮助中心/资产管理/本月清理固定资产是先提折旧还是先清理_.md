# 本月清理固定资产是先提折旧还是先清理?

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：常见问题
> 作者：云社区用户5j6u671 | 阅读：1552 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/607015036887923456)

【问题描述】

本月清理固定资产是先提折旧还是先清理?

【解决方案】

如果本期先折旧,在清理,清理单据保存的时候会提示:“本期或者以后期间已经生成计提折旧凭证,不能编辑”。

![](https://vip.kingdee.com/download/01095e4e7b7eb2764cd4a208684e7dc5d33e.png)
所以需要先清理,在生成折旧凭证。

上述情况,需要删除本期的折旧凭证。在保存清理的单据即可。

![](https://vip.kingdee.com/download/0109d8c7a8f4be64419c99a531a800fd5bad.png)
