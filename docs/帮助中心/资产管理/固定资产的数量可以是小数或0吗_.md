# 固定资产的数量可以是小数或0吗?

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：常见问题
> 作者：Schatz | 阅读：340 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/621511458404097536)

【问题描述】

固定资产的数量可以是小数或0吗?

【解释说明】

固定资产数量必须是**大于0的整数**。若需记录小数,请在资产卡片的备注栏中注明,并将数量进行四舍五入处理后填写;

若是0相当于没有了,无需录入;或后期没有对应资产了就做[清理](https://vip.kingdee.com/link/s/ZdVpg)
