# 计提折旧凭证,提示:本期或以后期间折旧凭证已审核/复核/冲销,请先删除凭证再重新计提折旧! 应该怎么处理

> 作者：杨桂平 | 阅读：1148 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/537799021331558400)

【问题描述】

计提折旧凭证,提示:本期或以后期间折旧凭证已审核/复核/冲销,请先删除凭证再重新计提折旧! 应该怎么处理

【操作步骤】

在【资产管理】-【折旧管理】里点计提折旧,会提示本期或以后期间折旧凭证已审核/复核/冲销,请先删除凭证再重新计提折旧!

![](https://vip.kingdee.com/download/0109b1d4d05673124ec0bc829cb45d976ff6.png)

【故障原因】
本期或以后期间已经生成了计提折旧的凭证,不允许再计提。

【解决方案】
本期或者以后期间已经生成了折旧凭证的固定资产不允许再计提折旧,需要先删除计提折旧的凭证,可到【账务处理】-【查凭证】中展开过滤条件,

①选中“资产管理折旧”过滤项,凭证期间与凭证日期选择不限,点击查询

②选中当前及以后期间的折旧凭证,点击删除

![](https://vip.kingdee.com/download/01093bc11205c71c459a87ca24278b3a28af.png)

【说明】

删除折旧凭证,卡片上的折旧金额不会被删除。重新计提折旧,会覆盖本月上一次的折旧金额。
