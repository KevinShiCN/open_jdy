# 折旧分配表

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：快速入门, 整体介绍
> 作者：Zyl | 阅读：1086 | 有用：9
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/23559)

1、折旧分配表可按期查询所有入账的资产折旧情况。

2、单击【资产管理】→【报表】→【折旧分配表】，进入“折旧分配表”页面。如下图所示：

![](https://vip.kingdee.com/download/0100bcffeb726df643cda82dc4e7e671afcd.png)

3、折旧分配表提供四种汇总方式可选择：按资产类别、按使用部门、按变动方式、按使用状态。如果不选择小计依据，则按卡片单独显示。

3、资产与总账连用时，则显示参数“显示科目辅助核算”。勾选后，报表显示辅助核算列
