# 多张固定资产卡片如何合并成一张卡片?

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：常见问题
> 作者：shaoling | 阅读：97 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/726273406194226432)

【问题描述】

多张固定资产卡片如何合并成一张卡片?

【业务场景】

有20台老旧电脑,都已经折旧完毕,但都还在继续使用,想把这一批电脑整合在一张卡片体现

【解释说明】

固定资产卡片不支持直接合并,建议整理数据后新建一张资产卡片,将合并信息录入其中,并对之前的多张卡片进行清理处理
