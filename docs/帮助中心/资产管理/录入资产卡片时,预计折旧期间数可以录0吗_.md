# 录入资产卡片时,预计折旧期间数可以录0吗?

> 产品：金蝶云·星辰 | 分类：资产管理 | 类型：常见问题
> 作者：Schatz | 阅读：215 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/614262946151048704)

【问题描述】

录入资产卡片时,预计折旧期间数可以录0吗?

【解释说明】

预计折旧期间数不能等于0,必须大于0。

![](https://vip.kingdee.com/download/0109b9a4a6927de44bd38661f973b01ce903.png)

【温馨提示】

如果不需要折旧,折旧方法选不折旧就好;

![](https://vip.kingdee.com/download/010926807bd4d81e4e9d8fef53e7e63c4049.png)

如果已经折旧完,已折旧期间数录入等于预计折旧期间数即可。

![](https://vip.kingdee.com/download/0109c5bed129c0b3419bb589c4800745ff49.png)
