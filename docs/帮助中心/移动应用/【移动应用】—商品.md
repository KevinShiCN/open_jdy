# 【移动应用】—商品

### 移动端-商品

### 1、功能介绍

手机商品列表功能通常包括查看商品详情、筛选商品、比较价格等功能，可以根据个人需求选择合适的商品。

### 2、应用场景

商品展示：商品列表会展示商品名、型号、即时库存、可用库存，自定义显示字段内容。

搜索功能：手机商品列表通常提供搜索功能，用户可以通过输入关键词或筛选条件，快速找到自己需要的商品。

排序功能：用户可以根据价格、销量等指标对手机进行排序，以便更方便地找到适合自己的商品。

详细信息：点击感兴趣的手机，可以查看更详细的商品信息，包括配置、价格等。

### 3、操作步骤

系统路径：金蝶云app-应用-仓库管理-商品

1） 进入商品列表中，默认显示所有的商品

筛选项：仓库

排序：编码、名称、库存数量

分享：可以对每个商品进行分享到微信、qq、钉钉等

![](https://vip.kingdee.com/download/01094f2253a999c04199afad1f213ed42949.png)

2） 快速搜索框

扫码：切换到对应的类型搜索，可以更快速搜索，特别是扫码，能有更好的体验

筛选：可以同时选择多个条件进行搜索

![](https://vip.kingdee.com/download/01094b3416011a10468bb3d752e10612148b.png)

3） 商品详情

点击商品，跳转到详情中

库存信息：可以显示不同仓库、仓位的库存信息；显示库存预警设置：最低库存、最高库存、安全库存数量。

价格信息：商品资料中维护的各种价格，采购价、最高采购价、历史最高采购价、参考成本价、配送价、批发价、零售价等。

基本信息：商品的其他基础信息，如：产品、品牌、注册证好等。

销售信息：根据时间段统计各个仓库的销售数量。

![](https://vip.kingdee.com/download/0109a38d6502e9b0495f8d2037de336e45c4.png)

4） 列设置

可以对商品列表自定义设置显示的字段内容。（商品详情中不支持设置，表头、表体的字段设置是对开单选商品的显示）

![](https://vip.kingdee.com/download/01099c1761ec311f4bbbb45ee66831ff8886.png)

5） 其他说明

支持新增商品、删除商品、编辑、禁用。
