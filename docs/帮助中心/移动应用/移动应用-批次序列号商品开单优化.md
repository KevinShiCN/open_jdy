# 移动应用-批次序列号商品开单优化

> 产品：金蝶云·星辰 | 分类：移动应用 | 类型：快速入门, 应用案例, 新特性
> 作者：echoquelqjisf | 阅读：858 | 有用：2

# **1 业务场景**

1.  高效入库操作

•  场景描述：仓库管理员在采购入库时，需要快速准确地记录批次号和序列号，确保库存数据的准确性和可追溯性。

•  用户痛点：当前的扫码和搜索组件布局不合理，导致操作效率低下，容易出错。

2.  便捷出库管理

•  场景描述：销售人员在销售出库时，需要快速选择正确的批次号和序列号，确保出库商品的准确性和可追溯性。

•  用户痛点：当前的出库方式不明确，用户在选择批次和序列号时容易混淆，影响出库效率。

# **2 解决方案**

1.  优化组件布局

•  功能描述：将扫码组件和搜索框布局在一起，减少用户操作的移动距离，提高操作效率。

•  具体实现：将扫码按钮和搜索框并排放置，用户可以在不移动手指的情况下快速切换使用。

2.  明确功能触发动作

•  功能描述：通过图标和文字提示区分扫码新增和扫码搜索的功能，减少用户混淆。

•  具体实现：扫码新增使用加号图标，扫码搜索并在操作前弹出提示框，明确告知用户即将执行的操作。

3.  简化已选列表设计

•  功能描述：仅通过勾选状态区分已选项，减少用户的认知负担。

•  具体实现：通过不同列表区分已选和未选的批次号或序列号，减少已选列表的冗余设计。

4.  统一数量计数逻辑

•  功能描述：确保勾选批次号或序列号后的数量计数一致，提高数据准确性。

•  具体实现：在页面顶部或操作区域显示当前选中的数量，并在每次操作后实时更新，确保用户能够清晰看到操作结果。

5.  支持PDA设备快捷操作

•  功能描述：通过PDA设备的快捷键扫码，自动获取批次号，提高操作效率。

•  具体实现：在PDA设备上设置专用的快捷键，用户按一下快捷键即可完成扫码操作，并自动填充批次号，减少手动输入的步骤。

6.  明确出库方式选择

•  功能描述：提供明确的出库方式选择提示，确保用户能够正确操作。

•  具体实现：通过弹出提示框或操作指南，明确告知用户“扫批次携带对应序列号”和“扫序列号携带批次”两种方式的具体操作步骤，确保用户能够正确选择和操作。

通过这些功能特性，可以显著提升用户在使用批次序列号管理功能时的体验，减少操作错误，提高工作效率。
