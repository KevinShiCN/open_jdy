# 金蝶云app如何查询批次保质期库存?

【问题描述】

金蝶云APP如何查询批次保质期库存?

【操作步骤】

操作路径:【应用】-【仓库管理】-【库存查询】-【筛选】-【参数选择】选择【显示批次保质期】,确定保存。如下图:
