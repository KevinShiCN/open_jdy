# 【移动应用】产品入库单功能简介

> 产品：金蝶云·星辰 | 分类：移动应用 | 类型：功能介绍, 快速入门
> 作者：L_Meng | 阅读：1215 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/373113032379356672)

# **1 业务场景**

- 场景一： 车间在完成生产任务后，需要将完工产品入库，仓库人员需要能快速扫码产品条码入库，生成产品入库单
- 场景二：车间人员在生产现场不便用电脑登录系统查看产品完工入库情况，希望通过移动端快速查看产品完工入库情况
- 场景三：产品入库单需要仓管人员审核，仓管人员经常在各个仓库工作，不便于在电脑上审核产品入库单，希望可以在手机上审核产品入库单

# **2 解决方案**

 产品入库单，适用于车间在完成生产任务后，需要将完工产品入到仓库中，可通过产品入库单记录入库的数量及仓库信息，便于管理库存，支持扫描物料二维码快速选择商品生成产品入库单，支持在移动端查看产品入库情况，并进行审核、反审核、删除产品入库单等操作。

# **3 操作步骤+效果呈现**

系统路径：生产管理-产品入库单

1）新增产品入库单，支持录入交货人、仓库员、交货部门等字段，这些字段都是非必录字段，可以按照需要进行录入。

![](https://vip.kingdee.com/download/01094170fdabe62c4102b29bccc0a655493d.png)

2）产品入库支持手工选择商品，也支持扫条码选择商品，且可以点击历入，选择历史完工入库的商品快速选择商品进行录入。

![](https://vip.kingdee.com/download/010974bf5947afbb486c92ab14c172cf3b7e.png)

3）支持查看产品入库单列表，并可以按照单号、交货部门、商品编码、商品名称和商品规格型号搜索对应的产品入库单据。

![](https://vip.kingdee.com/download/010940b52d1f3cc543af928c0c53e14026df.png)

4）通过单据列表选择单据查看单据详情界面，支持查看商品信息，单据信息，也可以通过联查信息查看产品入库单的上下游单据以及对应的单据编号。

![](https://vip.kingdee.com/download/01095e85fcd3555a4f33be17f227e6e046cc.png)

5）产品入库单支持单据列设置，用户可依据单据的表头，表体，列表设置展示或隐藏，系统内置了关键必录字段不可设置，比如商品名称、单据编号等信息必须展示，不系统内置必须展示的字段，用户可自行设置展示或隐藏。设置隐藏就在开单和详情界面不展示，移动端列设置只对移动端单据生效，与PC端列设置是分开的，互不影响。

![](https://vip.kingdee.com/download/010994d57f95bea942d9aa308995cf972e74.png)

6）新增产品入库单，表头没有默认显示仓库字段，需要使用表头仓库，可以通过列设置显示出表头仓库字段，支持使用表头仓库批量修改表体物料的仓库，如下图所示：

![](https://vip.kingdee.com/download/0109130a8507bb37401ebc5aa39d84e57b1e.png)

7）产品入库单支持小票打印，小票打印详细介绍参见：[https://vip.kingdee.com/article/236142129310144000](https://vip.kingdee.com/article/236142129310144000)

![](https://vip.kingdee.com/download/010950f04d5dc23a4836be6cde9cf13103cd.png)
