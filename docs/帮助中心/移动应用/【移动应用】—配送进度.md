# 【移动应用】—配送进度

### 配送进度

1、功能介绍

在生成销售出库单时，物流&配送选择【车辆配送】这个交货方式。其他配送信息选填。配送进度可在“配送进度“模块查看

2、操作步骤

系统路径：销售管理-配送进度

![](https://vip.kingdee.com/download/01093f90a71f57794c5aae9f32598d823fb9.png)

1） 在销售出库单里，物流&配送选择了【车辆配送】这个交货方式。其他配送信息选填。如图：

![](https://vip.kingdee.com/download/01095e4218818c9241878000fe367367d3e3.png)

2） 单据审核完成以后，在配送进度的【待配送】页面可以直接查到该订单信息，注意：此处配送进度的时间是根据制单时间安排而不是填写的配送时间

![](https://vip.kingdee.com/download/01098fab390e5a794a1c8adcefef215d24ad.png)

3）点击【配送】，在配送进度的【已配送】页面可以直接查到该订单信息

![](https://vip.kingdee.com/download/0109c7295b91f3374883b47e1de4338cf642.png)

4）用户权限设置：

1、在星辰PC端用户管理，设置用户关联职员

2、在功能授权里，设置用户权限，比如销售出库单的查询、审核、配送等权限。在数据授权里，可设置客户权限，限定用户可见客户信息范围。

![](https://vip.kingdee.com/download/0109d2582283c37c4df3a05b226ee89f7546.png)
