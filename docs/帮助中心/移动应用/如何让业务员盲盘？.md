# 如何让业务员盲盘？

> 产品：金蝶云·星辰 | 分类：移动应用 | 类型：进阶技巧, 常见问题
> 作者：移动生产白建凡 | 阅读：1010 | 有用：12
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/241230706922540800)

问题描述：如何让业务员盲盘？

解决方案：业务员盲盘，即业务员看不到盘点单里的账存数，可以通过星辰PC端【盘点单-账存数】字段授权控制。

![](https://vip.kingdee.com/download/0100f736abe7207b4ce4be3d2477f7823622.png)

**操作流程**：系统设置→用户管理→用户授权→功能授权【盘点单】→字段授权【账存数】，启用禁止查看，则此业务员看不到账存数，可以实现盲盘。

![](https://vip.kingdee.com/download/01006425d8379bc94527b504f80dc0cac575.png)
