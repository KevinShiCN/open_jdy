# 销售出库单详情页面的“直接退货”按钮可以设置隐藏吗？

> 产品：金蝶云·星辰 | 分类：移动应用 | 类型：常见问题, 快速入门, 精华汇总
> 作者：移动生产白建凡 | 阅读：20 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/797486127233588480)

# **1 问题描述**

销售出库单详情页面的“直接退货”按钮可以设置隐藏吗？

![上传图片](https://vip.kingdee.com/download/0100c8531fde489449cfb05b7934a00096bc.png)

# 2 适用版本

星辰3.0以上所有版本

# **3 解决方案**

不支持直接隐藏按钮，如果希望不显示，可以从权限上控制，不给销售退货单的新增权限

![上传图片](https://vip.kingdee.com/download/010026b572c319f540d193d4f5663ab1ec4c.png)

这时移动端销售出库单详情页面的“直接退货”按钮就会隐藏

![上传图片](https://vip.kingdee.com/download/0100924a1fb05fd24247a3144cccdbf78677.png)
