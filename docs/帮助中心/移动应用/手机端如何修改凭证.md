# 手机端如何修改凭证

> 产品：金蝶云·星辰 | 分类：移动应用 | 类型：常见问题
> 作者：ylim | 阅读：410 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/570776245781411584)

【问题描述】

手机端如何修改凭证

【解释说明】

金蝶云APP支持修改未结账期间的凭证,可修改的内容有:凭证字号、日期、附件、摘要、科目、金额、借贷方向、增加或删除分录。

【解决方案】

手机端登录金蝶云app后点击【应用】-【财务管理】-【凭证】,点击需要修改的凭证

![](https://vip.kingdee.com/download/0109f0494075affa4dfd8e7d5857d8e7ee7e.png)
对于已经审核的凭证需要先点击【取消审核】,然后点击【编辑】按钮,编辑修改凭证后点击【保存】按钮

![](https://vip.kingdee.com/download/010956ad0e8ce7e44f718f32a907347cf8d8.png)
对于保存成功的凭证,如需直接审核了,可以点击【返回查看】,回到凭证详情界面,点击【审核】把凭证审核了。

![](https://vip.kingdee.com/download/01099d05bfabeb444473a4242ea463a44db5.png)
