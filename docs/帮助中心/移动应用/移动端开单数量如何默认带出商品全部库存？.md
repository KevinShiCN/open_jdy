# 移动端开单数量如何默认带出商品全部库存？

> 产品：金蝶云·星辰 | 分类：移动应用 | 类型：常见问题, 快速入门
> 作者：毛豆豆_x | 阅读：15 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/779361709998825216)

# **1 问题描述**

移动端开单数量如何默认带出商品全部库存？

# 2 适用版本

星辰3.0以上专业版/旗舰版

# **3 使用场景**

业务单据通过选商品开单时，按照基本单位进行开单，并且每次需要将商品全部出库。

# **4 解决方案**

以销售订单为例

1、操作路径：PC端统一列设置>>移动端列设置>>选择业务单据>>表体-商品行分录>>数量/辅助数量字段，设置公式。

![上传图片](https://vip.kingdee.com/download/0100589693b5b6e444fc83bc5422b932757d.png)

2、将数量字段默认数值配置为即时库存或可用库存。

![上传图片](https://vip.kingdee.com/download/01002b14e76da24c4f2280bba5f1b15af895.png)

3、移动端销售订单>>选商品开单，点击开单数量组件时则会根据设置的公式自动填充商品的即时库存

>
注意：设置公式是直接取库存数量的数值进行填充，商品涉及多单位时并不会根据换算率进行数值换算并填充，如果需要按商品库存全部出库，建议您开单时使用基本单位。

关于如何设置开单时仅展示销售单位，请查看：[（食品行业）移动端开单选商品/拣验货优化](https://vip.kingdee.com/knowledge/748926799668429568?get_from=knowledge-id&productLineId=35&isKnowledge=2&lang=zh-CN)
