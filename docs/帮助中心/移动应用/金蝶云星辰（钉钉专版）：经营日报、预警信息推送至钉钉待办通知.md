# 金蝶云星辰（钉钉专版）：经营日报、预警信息推送至钉钉待办通知

## 1 经营日报、预警信息推送至钉钉待办通知

### 1.1 业务场景

经营日报、预警消息等自动推送钉钉消息给管理者审批，让业务能及时高效的流转，从而提升整体工作效率。

### 1.2 功能价值

企业管理者可以设置定时推送应付账款预警、应收账款预警、库存预警、客户流失预警等预警信息，及时把握公司业务瓶颈

企业管理者可以设置定时推送经营日报、经营周报、经营月报等公司经营信息，足不出户掌握公司经营动态

### 1.3 操作流程

1、 打开消息中心，设置经营日报推送、预警消息推送；操作路径：系统设置-消息设置-预警消息。

2、 设置推送时间、推送对象、推送内容、推送方式等配置

3、 点击启用，启用推送流程
