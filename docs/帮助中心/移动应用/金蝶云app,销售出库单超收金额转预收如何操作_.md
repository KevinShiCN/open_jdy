# 金蝶云app,销售出库单超收金额转预收如何操作?

【问题描述】

金蝶云APP,销售出库单超收金额转预收如何操作?

【业务场景】

当销售出库单中账户收款合计,大于本单应收(本单应收合计-抵扣预收-优惠),用户如果是填错金额了,可以返回修改;如果是真实收款则会自动转预收款

【解决方案】

销售出库单,收款金额合计-本次应收>0,则产生预收款,下单的时候会弹框收款提示。

确认转预收款后,下单成功,则生成销售出库单并关联预收款单

【温馨提示】

1、销售出库单,修改了客户、账户、商品、金额相关的字段重新保存后,则会自动删除关联生成的预收款单,并重新生成新的预收款单。

2、通过销售出库单超收生成的预收款单不允许修改和删除,销售出库单删除后,会同步删除关联的预收款单。

3、关联的预收款单审核后,则不能删除销售出库单,需要反审核预收款单才能删除销售出库单
