# 金蝶云app录入异价调拨单如何录入调拨费用?

> 作者：Schatz | 阅读：79 | 有用：0

【问题描述】

金蝶云APP录入异价调拨单如何录入调拨费用?

【解释说明】

金蝶云app上操作异价调拨单,**暂不支持录入调拨费用**。调拨费用只能在PC端录入。
