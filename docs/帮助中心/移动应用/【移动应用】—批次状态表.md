# 【移动应用】—批次状态表

### 批次状态表

### 1、功能介绍

移动端批次状态表是一个方便用户随时随地查看产品批次信息的工具。它能够实时更新库存状态，让用户了解每个批次的库存数量、生产日期、保质期等关键信息。

### 2、应用场景

库存管理：企业需要对产品库存进行实时管理，了解每个批次的库存数量、保质期等信息。通过使用移动端批次状态表，库存管理人员可以随时随地查看库存状态，及时进行库存调整和补货，避免库存积压或缺货现象的发生。

生产管理：移动端批次状态表，生产管理人员可以随时查看每个批次的状态，及时跟进生产进度，确保产品质量和生产计划的顺利进行。

销售管理：销售人员需要了解每个批次产品的销售情况，以便制定销售策略和推广方案。通过使用移动端批次状态表，销售人员可以随时查看产品销售情况和库存状态，及时调整销售策略，提高销售业绩。

物流管理：在物流过程中，使用移动端批次状态表，物流管理人员可以随时查看每个批次的状态，及时处理物流问题，确保产品按时到达目的地。

质量追溯：当产品质量出现问题时，使用移动端批次状态表，质量管理人员可以随时查看每个批次的生产日期、原料来源等信息，及时进行质量追溯和问题解决。

### 3、操作步骤

系统路径：金蝶云app-应用-仓库管理-批次状态表

1） 进入列表中，显示所有批次商品的商品名称、批次号，所属仓库、库存数量、保质期天数

2） 点击具体的批次弹框显示更多信息，如：编码、条形码、批次编码、生产日期、到期日、剩余天数等。

快速筛选框：商品(名称、编号、条码)、批次号、扫描条码

筛选项：商品类别、仓库

排序：商品名、库存数量
