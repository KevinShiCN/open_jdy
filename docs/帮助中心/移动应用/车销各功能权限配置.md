# 车销各功能权限配置

> 产品：金蝶云·星辰 | 分类：移动应用 | 类型：快速入门, 精华汇总, 常见问题
> 作者：移动生产白建凡 | 阅读：144 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/738058259918915840)

## **1 问题描述**

- 如何设置不让子账号看到司机管理模块？
- 车销管理中的各个功能分别对应什么权限，什么单据类型？

## 2 适用版本

星辰3.0以上所有版本

## **3 解决方案**

金蝶云APP车销模块中各个功能对应的PC端的功能如下：

| 移动端功能 | 对应的PC端功能 |
| --- | --- |
| 车仓管理 | 仓库 |
| 司机管理 | 职员 |
| 提货装车 | 移仓单 |
| 订单上报 | 销售订单 |
| 车辆开单 | 销售出库单 |
| 综合开单 | 销售出库单、销售退货单 |
| 车销回库 | 移仓单 |
| 收支日结 | 账户收支明细表 |
|  |  |

如需控制子账号不能查看/编辑相关功能，在PC端对子账号设置对应功能的查看/编辑权限即可
