# 移动应用-委外管理整体介绍

1 产品概述

### 1.1 产品介绍

在企业的生产业务流程中，车间人员需要能够在移动端通过扫码的方式，快速新增或审核委外单据，并下推下游单据。同时在移动办公的情况下，能方便快捷查看委外单据的相关信息，实现委外业务在移动端与 Web 端的数据互通。为此，金蝶云APP 在委外模块新增委外单据相关功能，点击对应的功能图标，即可查看单据列表及详情，还能对单据进行编辑、删除、审核、反审核、下推下游单据、完工、打印等操作，也可手工新增委外单据。

### 1.2 适用版本

星辰3.0以上专业版/旗舰版

### 1.3 业务场景

**1）场景一**：委外加工单通常由生产研发主管、工艺管理人员或车间人员负责制定，但这些人员经常不在工位，很多业务都需在手机上处理。希望能够直接在手机上选择商品或者扫描商品条码来录入委外加工单。

车间人员在生产现场时，不方便使用电脑登录系统查看委外加工单列表。因此，希望可以通过移动端快速查看委外加工单的加工、发料以及[入库](https://www.kingdee.com/product/stellar_inventory.html?utm_source=shequ)等情况。

委外加工单常常急需车间组长、上级领导或研发主管审核，但这些人员可能在车间现场或身处异地，不方便通过电脑审核单据，需要能够在移动端便捷地完成对委外加工单的审核工作。

**2）场景二：**仓管员需要在移动端查询这段时间一共发出了多少材料给到加工商。希望能够查询历史单据，可以根据不同的条件过滤到要查看的单据，且可以统一查看单据信息、状态等。

车间人员／仓管员在车间／仓库现场不便用电脑登录系统，希望通过移动端快速发料，跟进委外加工的发料状态。

委外发料单急需车间／仓库组长审核，车间／仓库组长在车间／仓库现场不便在电脑端审核单据，需要能移动端便捷审核领料单。

**3）场景三：**仓管员需要在移动端查询这段时间加工商一共退回了多少材料。希望能够查询历史单据，可以根据不同的条件过滤到要查看的单据，且可以统一查看单据信息、状态等。

车间人员／仓管员在车间／仓库现场不便用电脑登录系统，希望通过移动端快速新建退料单，跟进委外加工的退料状态。

委外退料单急需车间／仓库组长审核，车间／仓库组长在车间／仓库现场不便在电脑端审核单据，需要能移动端便捷审核退料单。

**4）场景四：**仓管员很少在办公室办公，需要在车间快速进行委外产品入库。

仓管员需要在移动端委外产品入库单列表界面清楚地了解各个产品入库的明细。希望能够查询历史单据，可以根据不同的条件过滤到要查看的单据，统一查看单据信息、状态，查看单据详情并对单据进行审核、反审核、编辑、打印等操作。

委外供应商完成委外任务进行交货时，仓管员需要快速记录委外产品完工入库情况，并操作审核、反审核、打印等功能，希望能实现快捷扫码入库。

**5）场景五：**仓管员很少在办公室办公，需要在车间快速进行委外产品退货。痛点分析：需在PC端建单，不方便移动快速建单入库。

仓管员需要在移动端委外产品退货单列表界面清楚地了解各个产品退货的明细。希望能够查询历史单据，可以根据不同的条件过滤到要查看的单据，统一查看单据信息、状态，查看单据详情并对单据进行审核、反审核、编辑、打印等操作。

委外供应商完成委外任务进行交货时，仓管员需要检查需退货的商品，快速记录委外产品完工入库退货情况，并操作审核、反审核、打印等功能，希望能实现快捷扫码退货。
