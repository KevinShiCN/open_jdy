# 收款退款单、预收退款单、付款退款单、预付退款单

> 产品：金蝶云·星辰 | 分类：移动应用 | 类型：快速入门, 功能介绍, 新特性
> 作者：Freya_peng | 阅读：2449 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/489106306175672576)

# 1 功能介绍

收款退款单、预收退款单、付款退款单、预付退款单，适用于因退货或缺货等因素需要退款，金蝶云APP支持新增各类退款单，使得现有的应收应付流程的业务场景覆盖更全面

# 2 应用场景

业务员遇到客户退货需要退款时，可以直接在手机端做退款单

# 3 系统路径

资金管理<收款退款/预收退款/付款退款/预付退款

# 4 操作步骤（以收款退款为例）

1） 新增收款退款单。录入结算客户后，选择源单进行退款，支持选择收款单、销售发票、其他收入退款、零售结存单为源单。选择源单后录入退款金额方式等信息

![](https://vip.kingdee.com/download/0109a01c7f8bc06a4c918367ae86c38ffe29.png)



2） 跳转选源单页面，支持编辑已选源单的本次核销金额，也可以删除后重新选择新的源单

![](https://vip.kingdee.com/download/010904e7a3922dd7475db6e6283595d98043.png)

3）收款退款单支持全流程跟踪。在收款退款单详情界面支持关联单据信息联查，保存后的收款退款单可以在付款记录中查看

![](https://vip.kingdee.com/download/01093d88918e6a924e06bc79ad7e331a7e11.png)

4） 收款退款单支持单据列设置，用户可根据个人需求，自行配置单据表头、表体及列表字段的展示或隐藏。移动端列设置只对移动端单据生效，与PC端列设置相互独立，互不影响。

![](https://vip.kingdee.com/download/01091a373b67f7144fe48278f282cace1680.png)

5）其他说明

收款退款单还支持单据字段配置，单据操作控制，自定义字段等
