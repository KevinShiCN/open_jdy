# 手机app商品模糊搜索可以搜索哪些字段?

【问题描述】

手机app商品模糊搜索可以搜索哪些字段?

【解释说明】

手机端搜索商品不支持设置哪些字段可以模糊搜索,默认按商品名称、编码、规格、条码综合搜索,开单选商品时,还支持指定按名称、编码、规格、条码中的一个进行搜索
