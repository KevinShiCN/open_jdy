# 【移动应用】BOM单查询及审批

# 1 业务场景

- 场景一：销售人员在外面处理业务时，需要关注产品使用到的原材料是什么，通过BOM查询了解客户想要使用的材料对应的名称和规格型号。和客户沟通好之后，有需要将零部件发给客户看时，销售人员可以快速的反馈给仓库，让仓库安排发货。
- 场景二：客户老板有在手机上查看BOM的需求，电脑经常不在身边且携带起来不方便，因此需要在手机上看研发人员做的BOM
- 场景三：BOM只有一个人有审核权限，当该职员不在办公室时，又急需审批BOM，希望可以在手机上审批BOM

# 2 解决方案

BOM 单（物料清单）主要用于详细定义产品结构，列出生产一个成品所需的所有原材料、零部件及其数量等信息。支持在移动端查看BOM单，支持审核、禁用、反审核、删除BOM单

# 3 操作步骤+效果呈现

系统路径：生产管理/委外管理-BOM单查询

1）支持查询BOM单列表，支持按照制单时间、BOM分类、BOM是否禁用等过滤BOM单

2）BOM单详情支持查询产品信息、联副产品信息（旗舰版启用联副产品参数）、子件信息，当子件存在BOM时，可以在下级BOM处查询到子件已审核的最新版本的BOM

![上传图片](https://vip.kingdee.com/download/01002a2ff506ae19495ebe157eba40f58393.png)

3）移动端支持使用审批流审批BOM单，支持接收审批消息，支持在消息通知处查看BOM单据消息
