# 【移动应用】—委托代销单

**委托代销**

### 1、功能介绍
通过委托代销单公司把自己的商品委托给下游经销商或门店销售，货权没有转移，下游经销商或门店根据实际销售的数量和金额与公司结算。
### 2、应用场景

场景一：某公司把自己的商品委托给下游经销商或门店销售，委托代销单审核后，公司仓库的库存减少，但是并没有发生实际的销售业务，因此不产生应收账款和收入，不能收款。

场景二：仓库收到订单可以发货的信息后，仓管员可以直接点击销售订单列表的“委托代销”生成委托代销单，也可以在委托代销单新增界面，点击“选源单”，选择对应的订单出库。

### 3、操作步骤

系统路径：销售管理-委托代销

![](https://vip.kingdee.com/download/0109302c79259a904d1fa47ddae14b6ce60e.png)

### 订单列表状态字段说明：

审核状态：根据是否审核分为未审核、已审核

结算状态：根据结算情况分为待结算、未结算、部分结算、全部结算

![](https://vip.kingdee.com/download/0100983bd745f20a48c09e513ab1a2af236e.png)

### 订单列表筛选条件说明：

委托代销查询支持按常用时间/自定义时间、单据状态、部门查询订单；可按来源、客户、仓库、业务员等筛选展示委托代销单。

来源：可以筛选三个订单的来源，分别是APP、订货和PC端创建的订单。

列表显示设置：可显示单据信息或者商品信息即分录信息，商品信息最多展示5条

![](https://vip.kingdee.com/download/0100cd6d976831304c37bd5ef1f6e5e8cd13.png)

1） 新增委托代销有两种方式，方法一：手工新增委托代销单，点击右下角“新增”，新增生成的委托代销单可以以‘选源单’方式关联销售订单；方法二：从销售订单里打开已审核完成未关闭的订单，点击委托代销关联生成委托代销单

 ![](https://vip.kingdee.com/download/010967efbddbe9644ce5816c408c4f89ae44.png)

2） 选择客户（若无预先录入客户信息，可在客户字段选择“新增客户”录入）

![](https://vip.kingdee.com/download/010069a6e91ed8d7402ba6fd895dc49f231f.png)

3） 选择商品信息

2种录入商品信息方式：1、选择商品库中的商品；2、直接扫码录入商品

  ![](https://vip.kingdee.com/download/0109ed8e6e06f0624ec4b7ab6d81692b7fea.png)

注意：该页面扫条码仅支持扫条码录入了基础商品信息的商品，如需扫码新增商品需要在选商品页面新增商品信息

![](https://vip.kingdee.com/download/0109712b7942c23c4c3289aae1bd4caa85b0.png)

选择销售商品时支持送给客户的赠品，需要先在移动端的统一列设置打开【是否赠品】字段：

![](https://vip.kingdee.com/download/01099c75f8ddf7064c41a62c470912eb163c.png)

4）选择好销售商品后，有两种结算呈现方式，下拉或者切换页签查看进行结算：核对客户信息、金额等其他信息

![](https://vip.kingdee.com/download/0109680474a15e2e4e459d40fd8d76bd0f10.png)

5） 点击保存提交审批或直接下单（注：审批单据将自动流转到审批人处，审批完成后即可下单）



6） 结算部分字段说明：

1、委托代销单审核后，公司仓库的库存减少，但是并没有发生实际的销售业务，因此不产生应收账款和收入，不能收款

2、    如需其他字段，如，交货方式等，请在辅助资料中提前录入并设置销售出库单字段

7）委托代销支持全流程跟踪，在委托代销单详情界面支持关联单据信息联查：目前移动应用只支持委托代销和委托代销退货单，所以联查单据只有委托代销退货单可以跳转，其他单据只能在电脑端操作，支持联查信息展示但无法跳转

![](https://vip.kingdee.com/download/01093e5b5fad727b48c7b05b7201cb2a8d87.png)

9）由销售订单下推生成的委托代销单中，商品的实际出库数量若要大于订单中的数量，需要在“系统参数-进销存参数-销售参数”中勾选“允许出库数量大于订单数量”

10）  委托代销单支持单据列设置，用户可依据单据的表头，表体，列表设置展示或隐藏，系统内置了关键必录字段不可设置，比如客户名称、商品名称、单据编号等信息必须展示，不系统内置必须展示的字段，用户可自行设置展示或隐藏。设置隐藏就在开单和详情界面不展示，移动端列设置只对移动端单据生效，与PC端列设置是分开的，互不影响。目前列表、表头、表体分别支持开启15、30、35个字段

![](https://vip.kingdee.com/download/0109995c4dbe22674e969a21218112388fbf.png)
