# 移动端，商品开启序列号和批次，销售出库时扫描序列号可以带出批次信息吗？

> 产品：金蝶云·星辰 | 分类：移动应用 | 类型：快速入门, 精华汇总, 常见问题
> 作者：移动生产白建凡 | 阅读：71 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/791984693205296896)

# **1 问题描述**

移动端，商品开启序列号和批次，销售出库时扫描序列号可以自动带出批次信息吗？

# 2 适用版本

星辰3.0以上所有版本

# **3 解决方案**

开单选商品后，录入序列号批次的页面，点击右上角设置按钮，选择“直接录入SN”的方式，扫SN码后即自动带出批次号信息，如下图所示
