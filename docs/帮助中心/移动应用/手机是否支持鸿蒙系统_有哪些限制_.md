# 手机是否支持鸿蒙系统?有哪些限制?

【问题描述】

手机是否支持鸿蒙系统?有哪些限制?

【解释说明】

![](https://vip.kingdee.com/download/0109dcabed83c9104015a42bf4c14a594655.png)

华为鸿蒙next使用金蝶云APP,目前不支持首页卡片(待办、审批、销售看板、采购看板、商城看板等)、访销地图定位等功能。

纯血鸿蒙系统(HarmonyOS 5.0版本系统版本及以上)支持APP消息推送、工作审批、蓝牙打印、定位打卡
