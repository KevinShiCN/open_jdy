# 手机端做生产领料单选源单时,如何按物料搜索任务单?

> 作者：Schatz | 阅读：49 | 有用：0

【问题描述】

手机端做生产领料单选源单时,如何按物料搜索任务单?

【解释说明】

在手机端进行生产领料单操作时,选择源单界面可以展示物料信息。但是,目前系统尚未支持根据物料信息来搜索任务单的功能。
