# 手机扫码入库如何跟采购订单或者收料通知单关联?

【问题描述】

  手机扫码入库如何跟采购订单或者收料通知单关联?

【操作步骤】

  手机新增采购入库单,选择供应商后点击<选源单>

![](https://vip.kingdee.com/download/010914ecadd5204e46bcb84700c4182a1ae6.png)

选择源单类型后点击扫码框扫商品条码,带出对应单据后选择点击<确定>

![](https://vip.kingdee.com/download/01096b6b2f7389c4436b8979aa8c235fec30.png)
确认入库信息无误后保存审核单据

![](https://vip.kingdee.com/download/01098c8e5861acf542db82a5006d1d017759.png)
