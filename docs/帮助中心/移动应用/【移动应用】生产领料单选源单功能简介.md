# 1、简介

## 1.1功能介绍

在生产领料单新增界面里，新增选源单功能按钮，点击选源单进入可选源单列表。

## 1.3应用场景

此功能主要应用于以下场景：

场景一：企业以销定产的特征明显，生产领料单是由企业的生产主管/工艺管理人员/车间人员/仓库人员制定，但本人经常不在工位，到处活动，很多业务都是在手机上处理，希望能在手机上通过选生产任务单为源单快速新增领料单。

场景二：企业以销定产的特征明显，且生产任务单多数存在倒冲领料为领料方式的物料，在移动端选生产任务单为源单新增领料单时，希望自动过滤倒冲领料为领料方式的物料，不允许领此种物料。

此功能可帮助经常离开工位的生产/仓库人员在手机快速通过选源单新增领料单，数据与Web端实现同步。

# 2、主要操作流程

(1) 点击选源单，进入可选生产任务单列表。可以上下拨动寻找要选的生产任务单，也可在搜索框中输入生产任务单号、生产车间、商品（编号/名称/规格型号）进行搜索。

![](https://vip.kingdee.com/download/010905544fa2a5a9425fbfe94d8afe068c50.png)![](https://vip.kingdee.com/download/0109a22473977d00462ba985e1ebfa9fa56a.png)

(2）确定所选源单后，进入核对和编辑所领物料的界面，支持编辑商品的数量以及其他可修改字段，如仓库、物料行备注等。源单已录信息默认自动携带到生产领料单。

![](https://vip.kingdee.com/download/01099b8f71e19bf3421d8366a31167644b1e.gif)

（3）保存后即可成功开单。
