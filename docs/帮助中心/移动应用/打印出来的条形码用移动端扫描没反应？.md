# 打印出来的条形码用移动端扫描没反应？

> 产品：金蝶云·星辰 | 分类：移动应用 | 类型：快速入门, 精华汇总, 常见问题
> 作者：移动生产白建凡 | 阅读：70 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/792423569153158144)

# **1 问题描述**

打印出来的条形码，用金蝶云星辰APP扫描没反应？

# 2 适用版本

星辰3.0以上所有版本

# **3 原因解析**

条形码对标签纸的大小及条码长度有要求，长度越长，越复杂，扫码识别难度会增加

# **4 解决方案**

建议使用包含信息更多更易被识别的二维码，或者使用条码设备生成标准长度的条形码。在同等打印区域下，二维码比一维条形码更好识别，容错率更高。更多关于二维码、条形码的使用说明参见**条码设计基础**
