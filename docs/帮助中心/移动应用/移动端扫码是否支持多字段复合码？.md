# 移动端扫码是否支持多字段复合码？

> 产品：金蝶云·星辰 | 分类：移动应用 | 类型：常见问题, 快速入门

# 【问题描述】

移动端扫码是否支持多字段复合码？

# 【解释说明】

当前功能：APP不能直接识别多字段的复合码，只能识别**批次和批次复合**、**序列号和序列号复合**这种简单模式

以批次号、生产日期、保质期三个字段的复合码为例，虽然星辰暂时不支持直接解析，但是出库时可以根据已有批次信息携带出生产日期、保质期

如果需要复杂的多字段复合，目前还是建议购买ISV产品
