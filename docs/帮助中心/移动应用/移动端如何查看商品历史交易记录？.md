# 移动端如何查看商品历史交易记录？

# **1 问题描述**

怎么在移动端查看同PC端一样的商品历史交易记录？

移动端开单时怎么查看商品的价格资料？

# 2 适用版本

星辰3.0所有版本

# **4 解决方案**

1、入口及信息展示

业务单据开单时，点击商品唤起商品开单详情页后，点击右上角“库存报价参考”即可查看该商品的历史交易记录情况。

该页面包括该商品的价格资料，销售记录，采购记录，和库存信息。其中，商品采销记录默认显示已审核单据，支持开关配置展示未审核单据信息；也可通过开关配置查看当前开单客户/供应商的交易记录情况。

![上传图片](https://vip.kingdee.com/download/0100ef1136e328444306a528ead4f42226ab.png) ![上传图片](https://vip.kingdee.com/download/01006d6cc2887ffb417eafe8145b11361c39.png) ![上传图片](https://vip.kingdee.com/download/010061306b5fef234cb3a213f7aaae5ea932.png)

2、权限控制

1）商品价格中价格信息展示，受商品信息价格相关的字段权限控制；

2）销售记录：来源于销售出库单、销售退货单，功能权限受销售明细表查询权限控制，同时支持字段授权；

3）采购记录：来源于采购入库单、采购退货单，功能权限受采购明细表查询权限控制，同时支持字段授权、数据授权

4）库存信息：来源于采购入库单、采购退货单，功能权限受采购明细表查询权限控制，同时支持字段授权、数据授权
