# 经营分析概述

> 产品：金蝶云·星辰 | 分类：移动应用 | 类型：快速入门, 整体介绍
> 作者：毛豆豆_x | 阅读：298 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/660219082326493696)

## **1 整体介绍**

经营分析覆盖采购、销售、资金等关键环节，从商品、客户、收支情况等多方面统计数据并呈现，提供全面的业务情况。支持数据和趋势图多形式呈现数据，支持按条件筛选查看相关数据进行查看。

![上传图片](https://vip.kingdee.com/download/0100f8bafc4737eb450f810ae7526432d24a.png)

## **2 适用版本**

星辰3.0以上所有版本

## **3 ****数据来源及权限设置**

- **模块内各应用权限：**模块内各应用是否可见，由对应功能的查询权限控制。如不想要用户看到采购分析功能，就不给采购汇总表的查询权限（采购分析的数据来源为采购汇总表），用户在经营分析模块内就看不到“采购分析”功能。各应用对应的功能权限如下

| 应用 | 功能权限 |
| --- | --- |
| 销售排行 | 销售管理-销售排行表 |
| 销售汇总表 | 销售管理-销售汇总表 |
| 英雄榜 | 销售管理-销售排行表 |
| 商品销售排行 | 销售管理-销售排行表 |
| 客户销售排行 | 销售管理-销售排行表 |
| 部门销售排行 | 销售管理-销售排行表 |
| 采购分析 | 采购管理-采购汇总表 |
| 采购汇总表 | 采购管理-采购汇总表 |
| 零售分析 | 门店管理-门店销售汇总表 |
| 商品分析 | 库存管理-库存查询表/销售排行表 |
| 客户分析 | 基础资料-客户 |
| 收支分析 | 应收应付-企业收支明细表 |
| 其他收支分析 | 应收应付-其他收支明细表 |
| 应收分析 | 应收应付-应收账款汇总表 |
| 应付分析 | 应收应付-应付账款汇总表 |
| 销售分析 | 销售管理-销售汇总表 |
| 销售毛利 | 销售管理-销售利润表 |
| 资金分析 | 应收应付-账户余额表 |

- **应用内的数据来源：**经营分析模块内的各个应用，可以通过点击“？”查看具体数据来源

![上传图片](https://vip.kingdee.com/download/0100d5538dc2df834ae6a4e30a44eeb64948.png)

- **字段权限设置**：在星辰PC端系统设置>基础设置>用户管理里，勾选需要授权的用户后点击“授权”设置对应数据的字段权限。找到对应的数据来源字段，设置“查询”功能权限。用户授权的具体操作参见：[用户管理与授权](https://vip.kingdee.com/link/s/Zbo4N)
