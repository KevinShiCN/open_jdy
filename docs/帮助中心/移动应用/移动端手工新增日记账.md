# 移动端手工新增日记账

1、简介

1.1功能介绍

移动端支持手动新增日记账

1.2适用版本

• 金蝶云·星辰专业版

1.3应用场景

移动端手工新增日记账功能主要应用于出纳人员移动办公有银行收支业务发生时，需要及时入账，但却不方便使用电脑的场景

2、主要操作流程

1）点击【日记账】，进入日记账查询界面：

①按照日期倒序排列。

![](https://vip.kingdee.com/download/01095b0ba13f10924ec4ab1b5390eb23ff1d.jpg)

“时间”支持下拉框选择常用时间和自定义设置时间，默认查询“本周”日记账

![](https://vip.kingdee.com/download/0109d6138f09d26f43658650d8d8be239309.jpg)

“类型”默认为全部类型，支持单独选择收入或支出

![](https://vip.kingdee.com/download/0109c7bcd302b17f485abd730f5283ffa2d1.jpg)

“账户”默认显示网页端基础资料-账户中的默认账户，支持下拉框选择其他账户

![](https://vip.kingdee.com/download/010954feabbdec3444ee9d79ca44ea384d9a.jpg)

⑤“币别”默认取网页端基础资料-账户中选择的第一个币别，支持下拉框选择基础资料-币别中的其他币别

![](https://vip.kingdee.com/download/010989c7170bdcb742e5a00af66c2fe73731.jpg)

2）在日记账查询界面，点击，进入日记账新增界面：

![](https://vip.kingdee.com/download/0109de0ef03a3a0748059bd87385edf3cf9f.jpg)

   默认带出日记账查询界面的账户、币别。

   支持录入“可用”状态账户的日记账。

当整个系统只有一个币别“本位币”时，新增界面不显示币别、汇率字段。

收支类别默认为收入类别，支持修改成支出类别

摘要支持手工录入，也可以通过摘要库选择，还可以选择快捷摘要

点击“展开更多”，支持录入业务日期、关联账户、备注等字段。此三个字段新增时默认为空。

![](https://vip.kingdee.com/download/0109062dbffa20464b88a5ef657cfe5c101c.jpg)

  点击“保存”，会保存日记账，并关闭新增界面，回到日记账查询界面；点击“保存并新增”，则保存此日记账，并同时打开一张新增日记账的界面。
