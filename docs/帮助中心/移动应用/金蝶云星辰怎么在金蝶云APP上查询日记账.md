# 金蝶云星辰怎么在金蝶云APP上查询日记账

**1、问题描述**

- 使用金蝶云星辰，在金蝶云APP上怎么查询出纳日记账

2、场景介绍

移动办公需要查询下账户余额

3、解决方案

登录金蝶云APP-【财务管理】-财税看板-出纳资金分析，即可查看出纳日记账及余额
