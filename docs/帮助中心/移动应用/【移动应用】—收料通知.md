# 【移动应用】—收料通知

> 产品：金蝶云·星辰 | 分类：移动应用 | 类型：功能介绍, 新特性, 快速入门
> 作者：Freya_peng | 阅读：448 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/598545623289294592)

# 1、 功能介绍

星辰旗舰版增加《收料通知单》功能菜单，作为采购订单到采购入库单以及委外加工单到委外产品入库单的中间流程，用于通知仓库收货。

# 2、 应用场景

- 收料通知承接采购订单和采购入库，当订单分批到货时需要分批通知仓管入库
- 收料通知承接采购订单和采购质检，需要通知质检员本次到货信息，哪些物料需要检验，检验完成后检验结果如何
- 收料通知承接委外加工单和委外产品入库单，当委外加工物资分批到货时需要分批通知仓管入库
- 收料通知承接委外加工单和委外检验，需要通知质检员本次到货信息，哪些物料需要检验，检验完成后检验结果如何
- 检验不合格时，收料通知需要下推退料通知，通知仓管退料给供应商。对于紧急采购或小批量采购等场景，无需做采购订单，可直接做收料通知再进行入库

![](https://vip.kingdee.com/download/01097da7ba12f25b48779f8715c1244a8cde.png)

# 3、 适用版本

• 星辰3.1旗舰版

# 4、 操作步骤

系统路径：采购管理-收料通知/委外管理-收料通知（如看不到应用，可在应用管理处添加应用）

![](https://vip.kingdee.com/download/01004c0df4875151409c83069aea78926e0b.png)

1） 新增收料通知单，录入供应商后，用户可手工新增收料通知单，如图：

![](https://vip.kingdee.com/download/0100503d1b8919a74538b840551ad8a93546.png)

2） 跳转选源单页面，支持按整单选择源单，也支持按商品行选择源单

![](https://vip.kingdee.com/download/0100b34989a1fe5d4ac3a505bb019c985601.png)

收料通知单支持录入单价、金额，选采购订单下推生成时，也会携带订单中的单价、折扣等信息，以便处理分批收货的金额携带场景。

注意：

①仓库基础资料改动说明：基础资料-业务资料-仓库-增加了【仓库类型】，仓库类型增加了【待检仓】。【待检仓】仅收料通知单、退料通知单可选择，且收料通知单中需要检验的商品行才能选择【待检仓】

②收料通知单承接【采购】和【委外】两种业务，根据源单的类型自动写入收料通知单的【业务类型】。

③当商品基础资料开启了【采购检验】，则收料通知单中的【是否检验】默认为【是】，支持修改。

④收料通知单为最终确认物料是否需要检验的单据，若选择是，则收料通知单必须进行质检后方可继续后续业务流程。

⑤若商品基础资料开启了【采购检验】，则采购订单对应的商品行不允许直接入库，必须先下推收料通知单，执行质检流程后才可入库；若直接入库则会提示：商品需要质检，请先生成收料通知单。

3） 已审核、未关闭的收料通知单，可一对一下推采购入库单和退料单，下推后支持单据联查，上查采购订单，下查入库单/退料单

![](https://vip.kingdee.com/download/01002601fe113b9f4561a1678e54d88627b2.png)

注意：收料通知单下推质检单后，质检完成后反写行检验状态、检验数量、合格数量、让步接收数量、不合格数量、判退数量、报废数量等至收料通知单，收料通知单对合格、让步接收、报废的数量进行入库，判退的数量退回供应商

4） 收料通知单支持单据列设置，用户可依据单据的表头，表体，列表设置展示或隐藏，系统内置了关键必录字段不可设置，不系统内置必须展示的字段，用户可自行设置展示或隐藏。设置隐藏就在开单和详情界面不展示，移动端列设置只对移动端单据生效，与PC端列设置是分开的，互不影响。

![](https://vip.kingdee.com/download/0100fb7bafaffcac425587eefe169f6903e4.png)

5）其他说明

• 收料通知单还支持单据字段配置，单据操作控制，自定义字段、工作流、消息通知单等。

• 当在星辰旗舰版的当前账套购买了云蝶轻智造管理系统的“质量管理”模块“或”为谷的“质检管理”，在系统参数-进销存参数可自行配置显示生态的单据还是星辰自有的收料通知单，当已购买生态对应模块，默认不显示星辰的收料通知单
