# 【移动应用】—自定义专题

> 产品：金蝶云·星辰 | 分类：移动应用 | 类型：快速入门, 功能介绍
> 作者：卷不动的二可 | 阅读：496 | 有用：3

**移动端自定义专题**

**自定义字段****——PC****端配置，移动端展示**

1）用户在PC端【系统设置-单据新增设置】配置了自定义字段、自定义计算，手机端可以展示自定义字段，同时可以执行自定义计算逻辑。需要先在单据设置里打开对应的自定义字段。**表头**字段对应手机端结算信息页面自定义字段；**表体**字段对应手机端商品页自定义字段。**适用单据可同时多选。**

![](https://vip.kingdee.com/download/0109e1a99032827a437b9922f0a935f082ef.png)



自定义字段类型目前支持6种：

①文本

②数值（自定义计算公式）

③日期

④引用辅助资料，

⑤引用基础资料

⑥带出已有基础资料属性



![](https://vip.kingdee.com/download/01093a1136abf75d4b95b62906e7fc6176d3.png)



**2****）自定义计算公式的计算规则**

l  公式中只包含单据体相关字段（含单据体自定义字段），可以计算

l  公式中只包含单据体相关字段 + 单据头相关字段 ，不可计算

l  公式中只包含单据体相关字段 + 基础资料相关字段 ，可计算

l  公式中只包含单据头字段 ，不可计算

l  公式中只包含单据头字段 + 基础资料字段，不可计算

l  公式中只包含基础资料字段，不可计算![](https://vip.kingdee.com/download/0109e4af08b5742e4d109b442ef4f57a3cde.png)



3）辅助属性商品有自定义字段时，这个商品辅助属性有下划线，字体颜色为蓝色，客户可点击，点击后可展示该商品的自定义字段信息，自定义字段有计算公式的可直接查看到公式值。

![](https://vip.kingdee.com/download/010991bfa9f4303244fea3f1bea7542f8ef8.png)
