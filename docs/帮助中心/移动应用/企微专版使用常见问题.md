# 1 试用/购买/续费/退款相关

### **1.1 企微平台是否可以代客下单？**

可以，请联系运营发送代下单原因以及产品、规格、用户数、年限。

### **1.2 过了续费期，显示新购，如何处理？**

账套数据自动保留1个月，新购相同规格会自动续上。

### **1.3 转续费到官网如何操作？**

（1）进销存、云会计

①官网下单

②子用户注册登录官网，并且重新授权

③去旧账套备份数据，导出

④去新账套导入，恢复数据

（2）云星辰

①官网购买服务

②子用户注册登录官网，并且重新授权

③多账套的请先把子账套创建好

④如果没有数据，直接使用即可。数据较少的，到旧账套引出数据，新账套中引入数据。数据很多的，则需要研发迁移数据。迁移数据请提单处理，提供客户旧服务ID+名称，新服务ID+名称

### **1.4 如何退款？**

客户端主动退款。路径：企微管理后台-应用管理-第三方-查看购买订单-选择需要退款的订单-订单服务-退款。7天内可退款，超过7天无法退款，企微需客户主动申请退款，申请退款后5个工作日会自动退款

### **1.5 续费续错规格，想退款重新购买，如何处理？**

先在云管家进行退款处理，处理完成后到企微退款，再重新续费（注意，切勿先退款企微，会造成原订单状态失效）

## 2 授权、管理员移交相关

### **2.1 如何新增用户，对用户授权？**

云星辰企微专版新增用户/用户授权操作说明：[点击查看](https://vip.kingdee.com/link/s/MUrlA )

### **2.2 钉钉和企微中云星辰的服务管理员怎么移交？**

通过登录官网www.jdy.com做移交。[操作步骤点击查看](https://vip.kingdee.com/link/s/ZvOvG)

### **2.3 无法进入应用，提示无购买接口需联系供应商？**

请先核对客户应用可见范围设置人数是否与购买用户数一致。若一致请提供客户企业名称、客户的企业ID、客户下单时间、购买产品规格及时长发送给刘武韬核实情况，若同时购买多款金蝶产品，请一并反馈
