# 【移动应用】生产任务单功能简介

## 1、简介

## 1.1功能介绍

企业生产业务中，车间人员需要移动端扫码快捷新增/审核生产任务单，移动办公的情况下支持查看生产任务单信息，实现生产业务移动端和Web端数据互通。金蝶云APP将在生产模块中新增生产任务单功能，点击此功能图标可查看生产任务单列表和详情、手工新增生产任务单以及对单据进行审核、下推下游单据、完工、反审核、编辑、删除、打印等一系列操作。

## 1.2适用版本

- 金蝶云·星辰专业版

   注意：本次产品更新仅在最新版本的星辰专业版V3.1更新，如需抢鲜体验本次更新的功能，请联系我们升级加入灰度验证。

## 1.3应用场景

场景一：生产任务单是由生产主管/工艺管理人员/车间人员制定，但本人经常不在工位，到处活动，很多业务都是在手机上处理，希望能在手机上选择商品/扫描商品条码录入生产任务单。

场景二：车间人员在生产现场不便用电脑登录系统查看生产任务单列表，希望通过移动端快速查看生产任务单的加工，领料和入库情况；

场景三：生产任务单急需车间组长审核，车间组长在车间现场不便审核单据，需要能移动端便捷审核生产任务单。

## 2、主要操作流程

(1) 生产任务单列表

点击生产任务单功能图标进入生产任务单列表，并可以按照单号、生产车间、计划员、商品（编码、名称、规格型号）搜索对应的生产任务单据。单据的状态与web端的状态一致，包括：

- 审核状态： --未审核 --已审核
- 领料状态： --未领料 --部分领料 --全部领料 --超额领料
- 入库状态： --未入库 --部分入库 --全部入库
- 完工状态 --未完工 --已完工 --手动完工

已审核状态的单据一共有三种：

- 在已审核、没有任何下游单据且未关闭时，仅显示已审核状态。
- 在已审核、有下游单据且未关闭时会隐藏已审核状态和未完工状态，仅显示入库状态和领料状态。
- 在已审核且完工（包括手动完工和自动完工）时，仅显示完工状态。

![](https://vip.kingdee.com/download/0109ca20c2724d2e486b8d1822fe428a1396.png)

列表界面支持按时间、单据状态、部门、制单人、业务类型以及计划员进行筛选。

![](https://vip.kingdee.com/download/0109763077c9da444008b92f17703b91fbcb.png)![](https://vip.kingdee.com/download/0109085177ea68be42ffa583dd43e29bce79.png)

(2）单据详情

通过单据列表选择单据查看单据详情界面，支持查看商品和材料信息，单据信息，也可以通过联查信息查看生产任务单的上下游单据以及对应的单据编号。

![](https://vip.kingdee.com/download/01092f04693633644327a480d318605c6ce6.png)

单据详情界面下方，支持对单据进行审核、下推下游单据（入库和领料）、完工、反审核、编辑、删除、打印等一系列操作。不同单据状态下的单据会显示不同的按钮：

1）未审核状态的单据

| 操作按钮 | 说明 |
| --- | --- |
| 审核 | 可审核单据，审核成功后单据状态切换为【已审核】。 |
| 编辑 | 进入编辑单据的界面，可重新选择商品、编辑商品的仓库、单位、数量等可编辑的字段。 |
| 打印 | 可打印单据 |
| 更多 | 包括打印设置和删除按钮。打印设置：可选择打印方式，配置打印模板，设置打印份数等。删除：删除单据 |

![](https://vip.kingdee.com/download/01092e238e82239145f7a844b4956fb48056.png)

2）已审核状态下的单据

| 操作按钮 | 说明 |
| --- | --- |
| 完工 | 可将此单据手动完工。 |
| 领料 | 可关联此生产任务单进行生产领料。 |
| 入库 | 可关联此生产任务单进行产品入库。 |
| 更多 | 包括反审核、编辑、打印、打印设置按钮。反审核：可反审核单据，需注意的是，若有下游单据，无法反审核成功。编辑：可修改单据的审核后可编辑的字段。字段可在web端-基础设置-单据字段配置中设置。打印：可打印单据打印设置：可选择打印方式，配置打印模板，设置打印份数等。 |

![](https://vip.kingdee.com/download/0109c47016fb57074964b7dd1bb7837a35cd.png)

3）完工状态的单据

| 操作按钮 | 说明 |
| --- | --- |
| 编辑 | 可修改单据的审核后可编辑的字段。此类字段可在web端-基础设置-单据字段配置中设置。 |
| 打印 | 可打印单据 |
| 打印设置 | 可选择打印方式，配置打印模板，设置打印份数等。 |
| 重新开工 | 可对此单据重新开工。此按钮只会在手动完工状态的单据中显示，自动完工状态的单据中不显示。 |

![](https://vip.kingdee.com/download/01093ad2f5b2b1e14870b56f7a095da49800.png)

（3）新增生产任务单

在生产任务单列表下方，点击新增按钮可新增生产任务单，支持录入业务类型、计划员、生产车间、单据标签等字段，其中业务类型和生产车间是必录字段，必须录入。

![](https://vip.kingdee.com/download/01099d674f2a25e641a6aeaa54a1893a5b8a.png)![](https://vip.kingdee.com/download/010953da022b48074eb2a3f8c4574170674a.png)

生产任务单支持手工选择商品，也支持扫条码选择商品，且可以点击历史，选择历史生产的商品快速选择商品进行录入。新增商品时，点击想要生产的商品，需录入商品的信息，如BOM编号，数量、计划开工日期和计划完工日期等，如下图。其中BOM编号自动携带该商品的最新版本已审核启用的BOM，计划开工日期和计划完工日期默认为开单当天系统日期。

![](https://vip.kingdee.com/download/0109668168d120b44eeeb5138e201055ff98.png)

维护好商品信息后点击确定进入下一步，可查看生产该商品的材料信息，重新修改商品信息。需注意的是，目前的材料信息暂不可添加并编辑，系统自动计算并携带商品维护的BOM单里的材料信息，后续第二次迭代将会放开材料信息可添加并编辑，敬请期待。

![](https://vip.kingdee.com/download/0109b558c52f6243469bbf267e7e8cf37bbd.png)

确认商品信息无误后，点击保存即可成功开单。

![](https://vip.kingdee.com/download/01099a6a2b1fe8f14822a9a12cf0c32d85d3.png)

（4）生产任务单单据列设置

用户可依据单据的表头，表体，列表设置展示或隐藏，系统内置了关键必录字段不可设置，比如商品名称、单据编号等信息必须展示，不系统内置必须展示的字段，用户可自行设置展示或隐藏。设置隐藏就在开单和详情界面不展示，移动端列设置只对移动端单据生效，与PC端列设置是分开的，互不影响。  新增生产任务单，表头没有默认显示单据标签字段，需要录入该字段的信息，可以通过列设置显示出单据标签字段。

![](https://vip.kingdee.com/download/0109cc16a9a237bc423b8baf7d213e5210e4.png)
