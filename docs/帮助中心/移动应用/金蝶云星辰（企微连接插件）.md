# 金蝶云星辰（企微连接插件）

> 产品：金蝶云·星辰 | 分类：移动应用 | 类型：功能介绍, 新特性, 快速入门
> 作者：移动生产白建凡 | 阅读：10680 | 有用：8
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/537667026417104896)

## **1 功能说明**

解决金蝶云星辰需要接入到企微中使用的问题。已经购买了金蝶云·星辰的客户，有同时使用企业微信作为内部交流或者跟客户沟通的OA工具，希望金蝶云·星辰和企业微信打通系统的业务数据，提升内外部协同

## 2 **功能优势**

- 消息协同：金蝶云·星辰的单据消息通过企业微信通知到用户，达到快速审核单据、预警、经营决策的目的（基础服务）
- 边聊边开单：融合企业微信开单助手，实现一键报价、一键开单、一键对账收款（互通服务）
- 企微客户同步：能把公司在企业微信管理的客户一键同步到金蝶云·星辰系统中（互通服务）

特别说明：企业微信的平台审批不支持接入到金蝶云·星辰

## **3 适用版本**

金蝶云·星辰标准版、专业版、旗舰版

## 4 购买说明

### **4.1 价格公示**

开通连接插件后，由于企业微信拦截无法进入应用，需购买企业微信的接口许可方能正常使用。许可类型、价格、功能公示如下：

| 许可类型 | 价格 | 功能 |
| --- | --- | --- |
| 企微基础服务 | 10元/年/用户 | 业务单据、预警消息、经营消息推送企业微信 |
| 企微互通服务 | 100元/年/用户 | 业务单据、预警消息、经营消息推送企业微信企业微信客户同步星辰客户企业微信侧边栏助手 |
| 其他： 两种账号可以组合购买，总购买用户数小于星辰用户数即可 |  |  |

### **4.2 如何购买**

** **请联系产品服务商或者销售通过云管家下单

![](https://vip.kingdee.com/download/010913b5a403b8ea4d6fa0de0f3df24eac00.gif)

**购买说明：购买金蝶云·星辰，且绑定了企业微信后才能购买开通企微通服务**

需要创建企微组织的管理员购买金蝶云·星辰连接插件，该管理员也必须是金蝶云·星辰工作台的管理员（建议手机号保持一致）

![上传图片](https://vip.kingdee.com/download/010046d80f677cd8403f83b3c5de1d4b301e.png)

不要求企微主体和购买金蝶云·星辰的主体一致

### **4.3 如何确定购买用户数**

购买金蝶云·星辰时，购买PC端的用户数会同时赠送相同的移动端的用户数。有多少用户需要在企业微信中使用金蝶云·星辰，就购买多少个企微通的用户数，总体不能超过星辰的用户数。绑定一个用户就扣减企微通购买的用户数。授权操作见本文 5.2（4）的内容

金蝶云·星辰PC端的用户绑定企业微信后可登陆PC端和移动端的企业微信，金蝶云·星辰仅授权移动端的用户绑定企业微信后仅能登陆移动端的企业微信

若金蝶云·星辰PC端购买了10个用户，移动端另外购买了2个用户（即移动端共计12个用户）：

（1）这12个用户都需要在手机端企业微信使用金蝶云·星辰，那么就需要购买12个企微通的用户数

（2）这12个用户中只有5个需要在手机端企业微信使用金蝶云·星辰，那么就购买5个企微通的用户数

## **5 安装绑定**

### **5.1**** 企业微信扫码安装应用**

（1）使用管理员账号，打开企业微信扫描下方二维码安装【金蝶云星辰-连接插件】应用

注意：该管理员必须同时是企业微信以及精斗云工作台管理员

![上传图片](https://vip.kingdee.com/download/010019af436b2a53423bbeac5e43dbc77186.png)

（2）添加员工到应用的可见范围（可以使用应用的员工）。

![上传图片](https://vip.kingdee.com/download/01007fd3a680780b4096adbe4d366af57ebe.jpg)![上传图片](https://vip.kingdee.com/download/010070a3c6fb9cdc4103816ee8ea9e932c00.jpg)![上传图片](https://vip.kingdee.com/download/0100a8121c0231fa410d8c62ea401b80215f.jpg)

### 5.2 管理员登录并绑定

**（1）管理员登录**

**管理员：**同时是企业微信管理员账号、金蝶公司管理员账号

![](https://vip.kingdee.com/download/0109070a53d6516d4be6b09222cf952b8bb3.png)

![](https://vip.kingdee.com/download/0109864c7b710a0c4429ad4e2195e41bab84.png)

默认验证码登录，可切换扫码登录，需**管理员账号**操作

![](https://vip.kingdee.com/download/0109acd3424bde624ee9831d1ea7950b8a76.png)

**（2）选择绑定公司**

- 如右边的金蝶公司多个，选择要绑定的一个公司（购买了星辰的公司），点击下一步

**（3）选择绑定公司**

- 企微客户列，选择可见范围用户，绑定金蝶账号
- 选择用户后，点击确认并且绑定

**![](https://vip.kingdee.com/download/01090d5bad8da9b5404d9c23f08330a26ba8.gif)**

**（4）用户管理**

**场景：**此前仅授权2个企微账号，现在需要授权更多企微用户使用

**步骤一：**在企业微信应用中设置可见范围

操作路径： 企业微信后台管理>应用管理>金蝶云星辰连接插件>编辑可见范围

![](https://vip.kingdee.com/download/0109dd14b44f04b94c989df389dcb5b29f23.png)

**步骤二：**在金蝶云·星辰连接插件中关联可见范围职员与星辰用户
