# 手机端资金看板的数据取值是什么?是否自定义调整?

【问题描述】

手机端资金看板的数据取值是什么?是否自定义调整?

【业务场景】

手机端的资金看板中的数据取数来源是什么,如何通过授权进行控制?

【解释说明】

**资金看板中的字段数据来源于星辰后台电脑端中对应报表的数据,字段的取数由系统默认设置,暂不支持手动进行修改取数来源和增加新的取数字段。以下为各个字段的取数来源和功能权限:**

**资金余额**

**数据来源:账户余额表**

**资金余额:账户余额本位币******

**功能权限:账户余额表查询权限**

**应收余额**

**数据来源:应收账款汇总表**

**应收余额:应收账款汇总表的余额本位币**

**功能权限:应收账款汇总表查询**

**应付余额**

**数据来源:应付账款汇总表**

**应付余额:应付账款汇总表的余额本位币**

**功能权限:应付账款汇总表查询**

**收付款金额**

**数据来源:账户收支明细表**

**收款金额:账户收支明细表的收入本位币合计**

**付款金额:账户收支明细表的支出本位币合计**

**功能权限控制:账户收支明细表查询**
