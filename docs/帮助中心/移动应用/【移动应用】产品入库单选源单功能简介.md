# 1、简介

## 1.1功能介绍

在产品入库单新增界面里，新增选源单功能按钮，点击选源单进入可选源单列表。

## 1.2适用版本

- 金蝶云·星辰专业版

   注意：本次产品更新仅在苍穹5.0平台更新，如需抢鲜体验本次更新的功能，请联系我们升级加入灰度验证。

## 1.3应用场景

企业以销定产的特征明显，生产领料单是由企业的生产主管/工艺管理人员/车间人员/仓库人员制定，但本人经常不在工位，到处活动，很多业务都是在手机上处理，希望能在手机上通过选生产任务单为源单快速新增产品入库单。

此功能可帮助经常离开工位的生产/仓库人员在手机快速通过选源单新增产品入库单，数据与Web端实现同步。

## 1.4APP路径

金蝶云APP-生产管理-产品入库单-新增-选源单

![](https://vip.kingdee.com/download/0109d646171c73f74549bbc99838b6f146f3.gif)

# 2、主要操作流程

(1) 点击选源单，进入可选生产任务单列表。可以上下拨动寻找要选的生产任务单，也可在搜索框中输入生产任务单号、生产车间、商品（编号/名称/规格型号）进行搜索。

![](https://vip.kingdee.com/download/0109a0af71d19b044db3880bcd615f863ec9.png)![](https://vip.kingdee.com/download/0109e989b050db10467586bc63dc875d9534.png)

(2）确定所选源单后，进入核对和编辑入库商品的界面，支持编辑商品的数量以及其他可修改字段，如仓库、商品行备注等。

![](https://vip.kingdee.com/download/0109a66f6b22c7f744d199f4b6f5e29b834d.gif)

（3）保存后即可成功开单。
