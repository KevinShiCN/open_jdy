## 1 功能介绍

使用手机金蝶云星辰APP，打印采购、销售、仓库、资金相关的单据，可以开单后即时打印，也可以对历史单据查询后打印。手机端打印能够满足各种业务场景的打印需求。

- 金蝶云APP支持的打印类型：云打印、热敏小票、针式蓝牙、条码标签打印
- PC端支持的打印类型：常规、蓝牙小票、蓝牙针式、蓝牙标签

PC端和金蝶云APP的打印类型匹配：

| --- | --- | --- |
| 常规 | 云打印 | 移动应用-云打印 |

| 蓝牙针式 | 针式蓝牙 | 移动应用-针式蓝牙打印 |
| 蓝牙标签 | 热敏小票（仅打印商品条码标签） | 移动应用-条码标签打印 |

## 2 应用场景

| 打印类型 | 应用场景 |
| --- | --- |
| 云打印 | 销售员现场开单，仓库打印机立即打印相关单据，通过云打印实现销售发货同步 |
| 仓管在仓库使用手机连续拣货发货，因为货物是分散的，使用手机捡完货直接打单，帮助仓管员快速连续拣货 |  |
| 热敏小票 | 业务员现场给客户开订单，使用手机端打印销售订单小票凭证给客户，客户以小票作为购买凭证 |
| 针式蓝牙 | 业务员销售出库，打印多联，分别给客户、仓库和财务，作为销售凭证分别核对做账 |
| 条码标签打印 | 手持蓝牙设备，商品入库时打印商品标签，一边入库一边贴标 |

## 3 适用版本

适用星辰3.1以上所有版本

## 4 打印模板介绍

本地模板（金蝶云APP创建）

- 金蝶云APP，公共应用>打印设置>选择单据>默认打印模板>新增
- 金蝶云APP，单据详情>更多>打印设置>默认打印模板>新增

云模板（PC端创建）

PC端，系统设置>个性化设置>打印模板

- 云打印：选择对应模块下对应单据，新增“常规”打印类型的打印模板
- 蓝牙打印（条码标签打印）：期初/条码打印>商品条码/采购入库条码/产品入库单条码/其他入库单条码，新增打印模板，模板类型为“蓝牙标签”

![](https://vip.kingdee.com/download/01093da01739d5294055965f6cb702a97784.png)

其中：

- 商品条码适用于金蝶云APP中，公共应用>条码标签打印里的打印
- 采购入库条码/产品入库单条码/其他入库单条码适用于金蝶云APP中，从对应模块进入对应单据的单据详情页面的条码打印

![](https://vip.kingdee.com/download/0109dc717d10f7304e3192897db13a821429.png)

- 其他蓝牙打印：选择对应模块下对应单据，新增“蓝牙小票”、“蓝牙针式”打印类型的打印模板

![](https://vip.kingdee.com/download/0109ba41c7e753c346a493cf8182f120ebc9.png)

打印模板内容配置及相关控件说明可参考社区知识：[星辰3.1打印手册](https://vip.kingdee.com/link/s/ZtVkP)
