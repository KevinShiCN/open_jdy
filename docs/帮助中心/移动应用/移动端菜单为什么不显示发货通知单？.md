# 移动端菜单为什么不显示发货通知单？

> 产品：金蝶云·星辰 | 分类：移动应用 | 类型：常见问题, 快速入门

# **1 问题描述**

移动端菜单为什么不显示发货通知单？

# 2 场景及解决方案

1、场景1：购买了星辰旗舰版在移动端登录使用，在应用>>销售管理中没有看到发货通知单入口。

解决方案：切换到应用界面>>点击右上角应用管理>>进入应用添加页面>>找到销售管理中发货通知单，田间菜单即可。

![上传图片](https://vip.kingdee.com/download/0100af1c15b5552a4be6a6266bbdfe2fb229.png) ![上传图片](https://vip.kingdee.com/download/01007e69f129083b4defa545c277ef9063da.png) ![上传图片](https://vip.kingdee.com/download/0100f8f73b4bd8284777a863573f16332fc0.png)

2、场景2：**购买了云蝶/为谷销售管理**，在应用管理中也没有看到“发货通知单”，无法添加。

解决方案：请前往PC端>>系统参数>>进销存参数，最后面的【单据显示控制】中，勾选“**显示【销售管理】下的发货通知单功能**”即可。

注意：库存管理中生产领料单/生产退料单/产品入库单/产品退库单，有相同的情况，也请勾选对应的显示参数。
