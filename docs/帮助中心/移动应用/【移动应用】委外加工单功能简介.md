# 【移动应用】委外加工单功能简介

# 1 简介

## 1.1 功能介绍

在企业的生产业务流程中，车间人员需要能够在移动端通过扫码的方式，快速新增或审核委外加工单，并下推下游单据。同时在移动办公的情况下，能方便快捷查看委外加工单相关信息，实现委外业务在移动端与 Web 端的数据互通。为此，金蝶云APP 在委外模块新增委外加工单功能，点击该功能图标，即可查看委外加工单列表及详情，还能对单据进行编辑、删除、审核、反审核、下推下游单据、完工、打印等操作，也可手工新增委外加工单。

## 1.2 适用版本

金蝶云·星辰专业版/旗舰版V3.1及以上版本

## 1.3 应用场景

场景一：委外加工单通常由生产研发主管、工艺管理人员或车间人员负责制定，但这些人员经常不在工位，而是在各处活动，很多业务都需在手机上处理。希望能够直接在手机上选择商品或者扫描商品条码来录入委外加工单。

场景二：车间人员在生产现场时，不方便使用电脑登录系统查看委外加工单列表。因此，希望可以通过移动端快速查看委外加工单的加工、发料以及入库等情况。

场景三：委外加工单常常急需车间组长、上级领导或研发主管审核，但这些人员可能在车间现场或身处异地，不方便通过电脑审核单据，需要能够在移动端便捷地完成对委外加工单的审核工作。

# 2 主要功能

## 2.1 单据列表

点击委外加工单功能图标进入委外加工单列表，并可以按照单号、供应商、商品（编码、名称、规格型号）以及单据备注搜索对应的委外加工单据。单据的状态与Web端保持一致，包括：

- 审核状态：分为未审核、已审核两种。
- 付款状态：包括未付款、部分付款、全部付款三种。
- 关闭状态：有未关闭、已关闭、手动关闭三种情况。![](https://vip-admin.kingdee.com/download/010950a27a3b37fb4ff497c3c7737d6fc2a4.png)

已审核状态的单据默认不显示审核状态，只显示付款状态和关闭状态。若单据未关闭，默认不显示不关闭状态。

![](https://vip.kingdee.com/download/0109144f2f072d564423b53af491d30e4750.png)

列表界面支持按时间、单据状态、部门、业务类型、供应商、结算供应商（系统参数开启第三方结算后才显示）以及业务员进行筛选。

![](https://vip.kingdee.com/download/010906d4f140979c4cbd91a20f194eb0abed.png)

## 2.2 单据详情

从单据列表中选择相应单据，即可查看单据详情界面。在此界面，可查看商品和材料信息、单据信息，也能通过联查信息查看委外加工单的上下游单据及对应的单据编号。

![](https://vip.kingdee.com/download/010977b2a39abd924d419e9b4e49ef9cd65b.png)

在单据详情界面下方，可对单据进行审核、下推下游单据（如入库和发料）、关闭、反审核、编辑、删除、打印等一系列操作。不同状态下的单据会显示不同的按钮：

1）未审核状态的单据

| 操作按钮 | 说明 |
| --- | --- |
| 审核 | 可审核单据 |
| 编辑 | 进入编辑单据的界面，可重新选择商品、编辑商品的仓库、单位、数量等字段 |
| 删除 | 可删除单据，该按钮只有在单据未审核时才会显示 |

![](https://vip.kingdee.com/download/010959e3092fe9f847fb8c8d1352316208d0.png)

2）已审核状态下的单据

| 操作按钮 | 说明 |
| --- | --- |
| 发料 | 可关联此委外加工单进行委外发料 |
| 入库 | 可关联此委外加工单进行委外产品入库 |
| 编辑 | 可修改单据审核后可编辑的字段。字段可在Web端-基础设置-单据字段配置中设置 |
| 更多 | 包括反审核、关闭按钮反审核：可反审核单据。若有下游单据，则无法反审核关闭：可将此单据手动关闭 |

![](https://vip.kingdee.com/download/0109bbe465fe7316474594086497d378689d.png)

3）关闭状态的单据

| 操作按钮 | 说明 |
| --- | --- |
| 编辑 | 可修改单据审核后可编辑的字段。此类字段可在Web端-基础设置-单据字段配置中设置 |
| 启用 | 可对此单据重新启用。此按钮只会在手动关闭状态的单据中显示，自动关闭状态的单据不显示 |

![](https://vip.kingdee.com/download/01096f4b42ae96b14c7faec331c5742397d0.png)

## 2.3 新增委外加工单

在委外加工单列表下方，点击“新增”按钮可新增委外加工单，支持录入供应商、结算供应商（需在系统参数里开启第三方结算才显示）、业务类型、业务员、仓库、币别、汇率、单据标签、供应商发货地址、收货地址等字段，其中供应商、结算供应商、业务类型是必录字段，必须填写。

![](https://vip.kingdee.com/download/0109d669b86917c84709b88d9f99fa1a5ea0.png)

委外加工单支持多种选择商品的方式，如手工选择、选源单选择以及扫条码选择。新增商品时，点击想要委外的商品后，需录入商品相关信息，如单位、加工费单价、BOM 编号、数量、计划开工日期和计划完工日期等。若商品已维护BOM，则自动携带该商品已审核的最新版本BOM，计划开工日期和计划完工日期默认为开单当天的系统日期。加工费单价支持按价格取数优先级取价

![](https://vip.kingdee.com/download/0109e06eac82777c4980b0ff60e3f5ea555e.png)

维护好商品信息后点击“确定”进入下一步，点击“编辑”可重新修改商品信息。若已维护好商品BOM，系统会自动计算并携带BOM单上的材料，可点击“查看材料”查看该商品的材料信息。若未维护BOM，可点击“新增材料”按钮添加材料。新增材料后，目前仅支持修改材料的“计划投料数”，材料的其分录字段暂不支持编辑。

![](https://vip.kingdee.com/download/010939e4d2e7f4774040b1a16a9a1311526e.png)

确认商品信息和材料信息无误后，点击保存即可成功开单。

![](https://vip-admin.kingdee.com/download/01099a6a2b1fe8f14822a9a12cf0c32d85d3.png)

## 2.4 单据列设置

用户可根据需要设置单据的表头、表体、列表相关字段的展示或隐藏状态。系统内置的一些关键必录字段不可设置，如商品名称、单据编号等信息。其他字段用户可自行决定展示或隐藏。设置为隐藏后，在开单和详情界面都不会展示。注意：移动端列设置只对移动端单据生效，与 PC 端列设置相互独立，互不影响。

![](https://vip.kingdee.com/download/01097a95085d015145d0ac794a5c7d44ab0b.png)
