# 移动应用-生产管理整体介绍

1 产品概述

### 1.1 产品介绍

企业生产业务中，车间人员需要移动端扫码快捷新增/审核生产管理相关单据，移动办公的情况下支持查看生产单据信息，实现生产业务移动端和Web端数据互通。金蝶云APP在生产管理模块支持生产相关单据的功能，点击对应的功能图标可查看单据列表和详情、新增单据以及对单据进行审核、下推下游单据、完工、反审核、编辑、删除、打印等一系列操作。

### 1.2 适用版本

星辰3.0以上专业版/旗舰版

### 1.3 业务场景

1）场景一：生产任务单是由生产主管/工艺管理人员/车间人员制定，但本人经常不在工位，到处活动，很多业务都是在手机上处理，希望能在手机上选择商品/扫描商品条码录入生产任务单。 车间人员在生产现场不便用电脑登录系统查看生产任务单列表，希望通过移动端快速查看生产任务单的加工，领料和[入库](https://www.kingdee.com/product/stellar_inventory.html?utm_source=shequ)情况； 生产任务单急需车间组长审核，车间组长在车间现场不便审核单据，需要能移动端便捷审核生产任务单。

2）场景二：车间人员需要领用材料进行生产时，前往仓库进行领料，由仓管员进行材料的分拣，分拣过程需要能够通过扫描物料条码快速准确地完成，生成对应的领料单；车间人员在生产现场不便用电脑登录系统查看领料单，希望通过移动端快速查看物料领用情况；生产领料单急需车间组长审核，车间组长在车间现场不便审核单据，需要能移动端便捷审核领料单。

3）场景三：车间人员领完料后，生产剩余需要退回仓库，或者是发现所领物料为不良品，也需要退回仓库，需要在移动端通过生产领料单生成生产退料单。[仓库管理](https://www.kingdee.com/product/stellar_inventory.html?utm_source=shequ)人员在盘点过程中不便用电脑登录系统查看生产退料单，希望通过移动端快速查看物料退回情况；车间人员希望能通过移动端快速扫码退料，生成生产退料单。

4） 场景四： 车间在完成生产任务后，需要将完工产品[入库](https://www.kingdee.com/product/stellar_inventory.html?utm_source=shequ)，仓库人员需要能快速扫码产品条码入库，生成产品入库单；

车间人员在生产现场不便用电脑登录系统查看产品完工入库情况，希望通过移动端快速查看产品完工入库情况；产品入库单需要仓管人员审核，仓管人员经常在各个仓库工作，不便于在电脑上审核产品入库单，希望可以在手机上审核产品入库单。

5）场景五： 产品[入库](https://www.kingdee.com/product/stellar_inventory.html?utm_source=shequ)后，仓管人员发现入库数量有误，需要退回部分产品，需要能快速扫码产品条码退库，生成产品退库单；产品入库后，部分质检不合格需要返工的产品需要退库，需要能在移动端之间操作产品退库；车间人员在生产现场不便用电脑登录系统查看产品退库情况，需要通过移动端快速查看产品退库情况。
