# 按客户、按线路批量新建跟进计划

> 产品：金蝶云·星辰 | 分类：移动应用 | 类型：常见问题
> 作者：杨桂平 | 阅读：602 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391353541874978304)

**【**应用场景及价值】

当销售人员在外拜访/线上联系客户或下游经销渠道时,可以使用按规划拜访的路线、或批量选择客户建立自己一天/周期性的拜访,让用户可以快速创建客户跟进任务

**【**整体流程】

1、 功能特性:

• 允许一个客户同一天建立多个任务,支持按线路和客户建立任务。APP和PC端均可编辑

• 当选中了客户建立任务保存后,按客户数生成任务条数 ,例如:建任务时选择了3个客户,则会生成三条任务

• 跟进任务状态为新建时,联系人和业务员、计划跟进内容仍可修改,其他字段不可编辑

• PC端支持删除状态为新建的跟进任务

2、 入口介绍

• APP访销-巡店任务-新建

• APP客户管理-跟进任务-新建

• WEB端跟进任务-新建

**【**应用流程】

1、 场景:新建正式客户拜访/联系任务

• 针对老客户,销售员可制定周期性计划,定期执行后,在系统内填写跟进记录进行存档

• 计划预警能及时提醒销售员完成跟进任务,保障每一个客户都能及时跟进,防止客户流失

• 销售员通过定期巡店,可维护下游经销商或代理商

2、 APP入口

步骤1:新建跟进任务:

【客户经营-跟进任务/访销-巡店】菜单中点击新建跟进(拜访/联系)任务

![](https://vip.kingdee.com/download/01090cf3098ef13e41b1bd59395b3aba9963.png)

步骤2:设置跟进任务

1) 输入计划跟进时间,如有周期设置,可编辑周期重复逻辑,

2)选择客户(可从线路选择(支持选择该用户权限内可见线路),带出线路上的客户,也可从客户列表批量选择客户),选择完成后,点击保存

---------新建任务------------从线路添加客户-------------从客户列表添加-------------填写所有内容后保存

![](https://vip.kingdee.com/download/010900fb0365ada244b491f5fd8c51bb2a24.png)

步骤三:查看任务

![](https://vip.kingdee.com/download/0109b1977d0cba1f436c8e537e96c9c34f20.png)
