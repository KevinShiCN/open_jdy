# 费用报销在手机端如何预览图片附件?

【问题描述】

费用报销在手机端如何预览图片附件?

【适用版本】

金蝶云app

【业务场景】

希望在手机端查询费用报销的图片附件信息以及将图片放大后查询,如何操作?

若希望了解金蝶云app上如何切换账套,请点击:[金蝶云app上如何切换账套?](https://vip.kingdee.com/search?searchText=%E9%87%91%E8%9D%B6%E4%BA%91app%E4%B8%8A%E5%A6%82%E4%BD%95%E5%88%87%E6%8D%A2%E8%B4%A6%E5%A5%97%3F&tabName=knowledge&source=DIRECT_SEARCH&productLineId=35&lang=zh-CN&productId=9)

若希望了解如何新增报销单,请点击:[如何新增报销单?](https://vip.kingdee.com/knowledge/612451107738206208?productLineId=35&isKnowledge=2&lang=zh-CN)

若希望了解如何新增付款申请单,请点击:[费用报销模块,付款申请单应用说明](https://vip.kingdee.com/knowledge/627671413058536192?productLineId=35&isKnowledge=2&lang=zh-CN)

【解决方案】

在金蝶云app费用报销的账套下,点击【应用】>【我的】>【我的单据】,进入到“我的单据”列表中

![](https://vip.kingdee.com/download/0109be79697c8c8b4b1fb19e213720f6166c.png)

选择需要查询的单据类型(报销单、借款单、付款申请单)。以报销单为例子,点击需要查询的单据,进入到单据详情中

![](https://vip.kingdee.com/download/01095946f08c943b4c94ae0b6450a4885fcd.png)

在“费用明细”栏下面,点击需要查阅的费用明细,进入到费用明细详情中

![](https://vip.kingdee.com/download/0109eb6aad53b3634d758ef6832be4eb30bb.png)

在“附件”栏中可以看到上传的图片信息

![](https://vip.kingdee.com/download/0109bcef79d2d5cb4a5aa94321bc2f10c977.png)

点击“图片”可以放大进行查询

![](https://vip.kingdee.com/download/01096327adffead14d8ea4a0608911c05811.png)
【温馨提示】

**目前暂不支持在“我的单据”中查询附件图片信息**

![](https://vip.kingdee.com/download/0109cc08ee350fef4b528c86a1099b658d43.png)
