# 【移动应用】产品退库单功能简介

> 产品：金蝶云·星辰 | 分类：移动应用 | 类型：快速入门, 新特性
> 作者：L_Meng | 阅读：978 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/413736616818244096)

# **1 业务场景**

- 场景一： 产品入库后，仓管人员发现入库数量有误，需要退回部分产品，需要能快速扫码产品条码退库，生成产品退库单
- 场景二：产品入库后，部分质检不合格需要返工的产品需要退库，需要能在移动端之间操作产品退库
- 场景三：车间人员在生产现场不便用电脑登录系统查看产品退库情况，需要通过移动端快速查看产品退库情况

# **2 解决方案**

产品退库单，适用于仓管人员，可通过产品退库单记录退库的数量及仓库信息，便于管理退货信息，支持扫描物料二维码快速选择商品生成产品退库单，支持在移动端查看产品退库情况，并进行审核、反审核、删除产品退库单等操作。

# **3 操作步骤+效果呈现**

系统路径：生产管理/仓存管理-产品退库单

1）新增产品退库单，支持录入收货人、仓库员、收货部门等字段，这些字段都是非必录字段，按照需要录入即可，点击上方时间，可修改单据日期。

![](https://vip.kingdee.com/download/0109115b0d95b46d4c6dbffe5427ee5ef2cd.png)

2）产品退库支持手工选择商品，也支持扫条码选择商品，且可以点击历退，快速选择历史产品退库的商品进行录入。

![](https://vip.kingdee.com/download/0109e91050c03f1e41888d63ddbe75a00e60.png)

3）支持通过产品入库单直接生成关联的产品退库单。

![](https://vip.kingdee.com/download/010938b79e70fb01438fae04ed3b5ba50907.png)

4）支持查看产品退库单列表，并可以按照单号、收货部门、商品编码、商品名称和商品规格型号搜索对应的产品退库单据。

![](https://vip.kingdee.com/download/0109019e5385ad5a473987fea5edc456fae0.png)

5）通过单据列表选择单据查看单据详情界面，支持查看商品信息，单据信息，也可以通过联查信息查看产品退库单的上下游单据以及对应的单据编号。

![](https://vip.kingdee.com/download/01096754a9783add43118c4cf0c66489b8df.png)![](https://vip.kingdee.com/download/0109678f4fabd5f24b30ab70c5067131ba47.png)![](https://vip.kingdee.com/download/01096d83d5f0e0d248ac9019c74b43e71394.png)

6）产品退库单支持单据列设置，用户可依据单据的表头，表体，列表设置展示或隐藏，系统内置了关键必录字段不可设置，比如商品名称、单据编号等信息必须展示，不系统内置必须展示的字段，用户可自行设置展示或隐藏。设置隐藏就在开单和详情界面不展示，移动端列设置只对移动端单据生效，与PC端列设置是分开的，互不影响。

![](https://vip.kingdee.com/download/010921cd093270704db292d1e6764ffa221a.png)
