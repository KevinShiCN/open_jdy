# 移动端app首页不显示工作审批?

> 作者：云社区用户8x80607 | 阅读：114 | 有用：0

【问题描述】

移动端APP首页不显示工作审批?

![](https://vip.kingdee.com/download/01095aba637a2aa84d93aec5455633b84dac.png)

【解决方案】

APP首页拉到最下面,有设置按钮,点击【设置】、找到工作审批,点击前面的+号![](https://vip.kingdee.com/download/01099754f05dc3474e48a8cf348397ee6886.png)
添加完成,点击保存。显示如下图所示

![](https://vip.kingdee.com/download/01096a04baa254b247f38ba1be587528463d.png)
