# 【移动应用】生产退料单功能简介

> 产品：金蝶云·星辰 | 分类：移动应用 | 类型：功能介绍, 新特性, 快速入门
> 作者：L_Meng | 阅读：578 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/415897984887680256)

# **1 业务场景**

- 场景一：车间人员领完料后，生产剩余需要退回仓库，或者是发现所领物料为不良品，也需要退回仓库，需要在移动端通过生产领料单生成生产退料单
- 场景二：仓库管理人员在盘点过程中不便用电脑登录系统查看生产退料单，希望通过移动端快速查看物料退回情况
- 场景三：车间人员希望能通过移动端快速扫码退料，生成生产退料单

# **2 解决方案**

       生产退料单，是领料人员将不能使用或者不需要使用的材料退回的单据凭证，支持扫描物料二维码快速选择物料生成退料单据。支持在移动端查看生产领=退料单列表，确认退料信息，并操作审核、反审核、删除生产退料单。

# **3 操作步骤+效果呈现**

系统路径：生产管理/仓存管理-生产退料单

1）新增生产退料单，可以录入退料人、仓管员和退料部门，这些都是非必录字段，没有需要也可以不录入。

![](https://vip.kingdee.com/download/010948aa159f372344f8aa5d24d4fa4a0027.png)

2）新增生产退料单支持手工选择商品，也支持扫条码选择商品，还可以通过历史退料快速选择历史已经退过的商品。

![](https://vip.kingdee.com/download/0109855fc957f31e40b78581aac67cbb0905.png)

3）支持通过生产领料单点击【直接退料】生成生产退料单。

![](https://vip.kingdee.com/download/01090edebc3119a14e9cb3d4ae71162180c4.png)

4）支持查看生产退料单列表，列表界面支持按照单号、退料部门、物料编码、物料名称、物料规格型号进行搜索。

![](https://vip.kingdee.com/download/01094c9e259a7b5443d6b9074b5482912ce0.png)

5）从生产退料单列表点击单据可以查看单据详情，单据详情界面支持显示商品信息、单据信息，且支持在联查信息界面查看生产退料单的上下游单据和对应的单据编号。

![](https://vip.kingdee.com/download/0109ca3775e882f74c9bb44fe348303935d6.png)

6）生产退料单支持单据列设置，用户可依据单据的表头，表体，列表设置展示或隐藏，系统内置了关键必录字段不可设置，比如商品名称、单据编号等信息必须展示，不系统内置必须展示的字段，用户可自行设置展示或隐藏。设置隐藏就在开单和详情界面不展示，移动端列设置只对移动端单据生效，与PC端列设置是分开的，互不影响。

![](https://vip.kingdee.com/download/0109780f49db234c45559ee35a48de7e3f53.png)
