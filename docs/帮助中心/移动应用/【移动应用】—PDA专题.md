# 【移动应用】—PDA专题

### 1、功能介绍

金蝶云星辰的审批功能通过定义多级审批流程和节点,实现业务单据的标准化、智能化的在线审批管理。

支持版本：专业版、旗舰版

### 2、应用场景

货物收发管理：使用移动PDA扫描货物收据和货物外包装的标签来收集信息，确认货物的正确性。

拣货发货：在拣选货物时，只需要进入系统即可即时找到货物的存放位置和货架号，提高拣选的工作效率。

库存盘点：在盘点过程中，工作人员使用PDA设备扫描货物的条码或二维码，快速准确地采集货物信息，并将数据传输到计算机系统中。这种方法可以大大提高盘点效率和准确性，减少人工操作错误，提高库存管理水平。

订单处理：仓库工作人员可以使用移动PDA接收、处理和跟踪订单。通过扫描订单上的条形码或二维码，可以快速准确地确认订单信息，并执行相应的拣选和发货操作。

货物追溯：移动PDA可以用于记录货物的详细信息，包括生产日期、批次号、保质期等。在需要追溯货物来源或质量时，可以通过移动PDA快速查询相关信息，提高追溯效率。
