# 如何导入工资单?

> 产品：金蝶云·星辰 | 分类：工资管理 | 类型：常见问题
> 作者：Cat_Chow | 阅读：1005 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/571138757966093056)

【业务场景】

如何导入工资单?

【操作步骤】

**三种方式快速导入或生成工资单:**

**方法一、下载引入模板,按引入格式要求编辑工资内容和数据,再批量引入**

![](https://vip.kingdee.com/download/010972e26e48d1544f00a97a8eed6b70cedf.png)

注意:

①红色为必录项

②黑色为非必录项,不涉及到可以为空

③不要修改或是删除引入模板表头

![](https://vip.kingdee.com/download/0109eb84e78d6f844806a678efde7f530d35.png)

④当前期间可以引入多个工资表,如可以按不同的员工,部门等分别引入当期工资表

![](https://vip.kingdee.com/download/0109dd5aa28ed40c4c40a8e0a16318ae5d3e.png)

⑤工资单上的项目,都来源于【基础设置】-【工资项目】。工资单导入模板,不受工资项目中的公式设置的影响,可以直接在模板上手工录入数值,直接导入。

⑥支持自定义引入模板,除了必录项,其他字段可根据企业实际情况进行调整:

打开系统默认的模板,点击【复制】生成一个自定义模板,在这个基础上调整需要引入的字段,

![](https://vip.kingdee.com/download/01096121f589814c4db4b447c7f51913ca89.png)

![](https://vip.kingdee.com/download/010962faeaf8c5904d00a258d49722981624.png)

![](https://vip.kingdee.com/download/01090f83a186dd4d445288bbc2b5b02d0e2a.png)

⑦如有自定义工资项,自定义模板也会自动增加,根据企业情况,选择是否引入模板增加该字段:

![](https://vip.kingdee.com/download/01094196356aee8d43b7b872cedafdd88708.png)

**方法二:将上个月的工资表,按引入模板格式引出,然后按引出模板的格式编辑或是调整数据后,再引入当前期间**

![](https://vip.kingdee.com/download/0109ee450a98f563425da1030911d82a783e.png)

**方法三:直接复制上期工资单,如果本月职员、工资项目、工资数据和上月一致,可以采用“复制上期”的功能,复制上月工资单。**

如图,当前期间是2024-4期【复制上期】直接带出2024-3期工资单

![](https://vip.kingdee.com/download/01097b9da131bd0e49469ae07cc14e5c1ecd.png)
