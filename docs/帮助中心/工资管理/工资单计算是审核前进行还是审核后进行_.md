# 工资单计算是审核前进行还是审核后进行?

> 产品：金蝶云·星辰 | 分类：工资管理 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：97 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/642167152832847104)

【问题描述】

工资单计算是审核前进行还是审核后进行?

【解决方案】

**工资单计算是审核前计算**。审核后计算系统会提示:当前及以后期间存在已审核的工资单,请将其“反审核”后再进行工资计算。
