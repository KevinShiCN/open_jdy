# 工资单打印的 “模板陷阱”

工资单打印的“模板陷阱”

![上传图片](https://vip.kingdee.com/download/0100f1f0b66df4e44477a5a88cb6bda6c432.jpg)

每月最后一个工作日，财务小周处理完薪资核对和个税申报，在金蝶系统打印工资单时，预览界面字段却显示空白；无论她如何反复刷新页面或更改所选月份，预览界面的字段始终显示为空白。

上个月操作一切正常，本月她只因觉得系统预置模板字段太简单，便直接新增空白模板做了个明细版，加了对公部分、绩效工资等明细字段。发现取不到数据后，小周先排查数据源，按指引配置并绑定了自定义数据源，可打印界面依旧无数据。

她花了半小时尝试解决问题，包括检查文件路径、核实字段设置以及重启电脑，但问题依旧存在。急忙致电 IT 老陈：“陈哥，工资单打印取不到数据，我自建了新模板，也配了自定义数据源，连接成功就是没数据，这咋回事啊？”

老陈赶来后先看模板管理，一眼发现问题，笑着问：“你这新模板是直接建的空白版吧？没基于系统预置模板改？” 小周点头称是，老陈当即解释：“系统预置模板内置专属数据关联接口，只有基于它修改的模板，才能和数据源建立有效调取链路。你直接建的空白模板，只做了表面字段布局，没触发底层关联规则，配好数据源也没法调取数据。”

老陈开始现场演示操作流程，以下是他的具体步骤：

首先，到【系统设置】→【打印模板】→【工资管理】→【工资单】，找到系统预置的工资单打印模板，点击“复制”，复制生成一个新的模板；

![上传图片](https://vip.kingdee.com/download/01005a22521729914059b190062f82ff307c.png)

![上传图片](https://vip.kingdee.com/download/0100f8e00af448254b58934be062500029b4.png)

之后，在打印模板界面点击【数据源】→【自定义数据源】，再点击系统预置的数据源，新增空白行；

![上传图片](https://vip.kingdee.com/download/0100148bcf0e84104c9c966c5b2c095727e4.png)

在【工资管理】→【基础设置】→【工资项目】中，查看新增的自定义项目编码和名称，并将其填入打印模板的自定义数据源。

![上传图片](https://vip.kingdee.com/download/0100b57f80a0602c40bdad982120ac1add4e.png)

![上传图片](https://vip.kingdee.com/download/010015d89cb36b0d4554bd788f7d26b6eb41.png)

之后，找到系统预置的数据表格，删除不需要的内容；在选中的数据行单元格内点击【内容】，选中需要的字段后点击【确认】，将单元格绑定字段；

![上传图片](https://vip.kingdee.com/download/01008ae31941c5d0434ab364657cbf131c5a.png)

打印模板设置完成后，点击【保存】。

![上传图片](https://vip.kingdee.com/download/010051d95d2afd8e42dcb756627a073376ec.png)

最后，到工资单界面，在【打印】→【打印设置】中选择新建的模板并进行打印。

![上传图片](https://vip.kingdee.com/download/01002230b1e71e094161af2cb5548c1acfa9.png)

![上传图片](https://vip.kingdee.com/download/0100bc02271c0ba549c8b493e34ea50ba835.png)

![上传图片](https://vip.kingdee.com/download/01002d9c66240bad4d03a40975604afb0dc7.png)

按照老陈指导的步骤正确配置模板后，屏幕上立刻显示了清晰的工资数据，包括姓名、部门和各项薪资明细，这些信息整齐排列，与小周之前核对的薪资表完全一致。

小周看着屏幕上的正常数据恍然大悟，拍着脑门说：“原来问题不在数据源，是我建模板的方式错了，以为空白模板更灵活，反倒踩了坑。”

老陈一边帮她把新模板设为默认，一边补充：“预置模板不只是样式，更是数据调取的‘桥梁’，基础字段都和底层数据库绑定了。空白模板没搭起这道桥，再对的数据源也没用，以后改模板在预置版上删改就行，既保关联又合需求。”

小周按方法重操作一遍，打印机顺利打出清晰工资单。她把教训记在工作笔记上：工资单模板切勿直接新增空白版，需基于预置模板修改，否则无法触发数据关联。

此后，小周在调整模板时都严格遵循这个规则，再也没有遇到过数据问题。她也深刻理解到，在职场中遵循系统规则远比盲目创新更为高效。
