# 工资管理

> 产品：金蝶云·星辰 | 分类：工资管理 | 类型：功能介绍, 快速入门
> 作者：海鸥乔纳森 | 阅读：4838 | 有用：7
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/626006896587166976)

星辰工资模块主要处理工资薪金的计算及统计，适用版本专业版和旗舰版，目前标准版不支持。

![](https://vip.kingdee.com/download/01096bfb4d6af2854d2ca1d29081f824c1c6.png)

| 步骤 | 具体说明 | 学习链接 |
| --- | --- | --- |
| 1.基础设置 | 使用工资管理模块前，需进行工资基础设置：新增职员、部门，工资项目中新增并设置好计算公式，社保统一设置、专项附加扣除先录入数据，方便后续录入工资单及进行工资计算   | —— |
| 1.1职员 | 新增工资单中的职员信息需要在提前在基础资料-职员中维护   | 点击学习 |
| 1.2工资项目 | 工资模块中的工资项目是指构成员工薪酬的各个组成部分 | 点击学习 |
| 1.3社保统一设置 | 工资模块中的社保统一设置通常是指在企业的工资管理系统中，对社会保险的各种参数进行集中配置的过程 | 点击学习 |
| 1.4专项附加扣除 | 专项附加扣除是指在计算个人所得税应纳税所得额时，除了基本减除费用标准之外，纳税人还可以依法享受的额外扣除。这些扣除项目通常与个人或家庭的实际支出有关，旨在减轻纳税人的税收负担，鼓励和支持某些社会行为或生活需求 | 点击学习 |
| 2.工资管理 |  | —— |
| 2.1工资单 | 工资单用于给企业员工每月计算工资   | 点击学习 |
| 2.2个税计算 | 每月需给企业员工计算代扣代缴工资薪金个税,可录入完工资单后,进行个税计算   | 点击学习 |
| 3.薪资及统计 |  | —— |
| 3.1工资统计表 | 用于统计企业每月的工资发放情况   | 点击学习 |
| 3.2社保变更记录表 | 存在社保变更的情况，可通过社保变更记录表查下历史变更记录况   | 点击学习 |
