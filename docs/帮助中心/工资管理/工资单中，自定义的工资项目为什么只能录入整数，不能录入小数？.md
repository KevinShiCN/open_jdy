# 工资单中，自定义的工资项目为什么只能录入整数，不能录入小数？

> 作者：Zyl | 阅读：855 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/360358227986179840)

****

1、问题描述

- 工资单中，自定义的工资项目为什么只能录入整数，不能录入小数？

2、场景介绍

工资项目中新增了“出勤天数”的自定义项目，工资单中只能录入整数，不能录入小数，实际上是要录入21.75录入不了，

3、解决方案

1）工资管理-工资项目-打开自定义的项目-在“小数位数”中录入“2”即可。

![](https://vip.kingdee.com/download/01094ec0af5845284ca19271f3f474ab4fbd.png)

2）工资项目的小数位数规则是：内置的工资项目，默认取本位币币别中的金额精度；新增的数值型工资项目，小数位数可以自定义，支持录入0-10位精度。
