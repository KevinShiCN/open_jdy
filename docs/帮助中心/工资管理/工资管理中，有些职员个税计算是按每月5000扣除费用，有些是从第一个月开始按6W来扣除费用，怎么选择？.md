# 工资管理中，有些职员个税计算是按每月5000扣除费用，有些是从第一个月开始按6W来扣除费用，怎么选择？

> 作者：Zyl | 阅读：795 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/360362517434015488)

****

**1、问题描述**

- 工资管理中，有些职员个税计算是按每月5000扣除费用，有些是从第一个月开始按6W来扣除费用，怎么选择？

2、场景介绍

有新员工、老员工。很多老员工是要按从今年1月起直接按6W来扣除费用，有些新入职的员工每月按5000来扣除费用。需要区分不同的员工来设置。

3、解决方案

1）基础资料-职员-工资专项设置中可以选择是按5000还是按6W扣除

2）新增的职员和使用工资模块前已有的历史职员，默认选择的是按5000扣除，允许修改。
