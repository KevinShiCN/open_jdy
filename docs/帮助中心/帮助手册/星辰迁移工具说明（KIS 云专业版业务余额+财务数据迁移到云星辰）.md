# 星辰迁移工具说明（KIS 云专业版业务余额+财务数据迁移到云星辰）

> 产品：金蝶云·星辰 | 分类：帮助手册 | 类型：快速入门, 精华汇总, 功能介绍

## 一、支持迁移的专业版版本：16.0，历史版本需要先在16.0执行升级操作。

1、KIS专业版下载地址：[https://cloud.189.cn/web/share?code=JV7Jva6jYfeu](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fcloud.189.cn%2Fweb%2Fshare%3Fcode%3DJV7Jva6jYfeu)（密码：sky6）

2、KIS云专业版低版本如何升级到16.0：

    1）安装好KIS云专业版16.0的私有云

    2）在金蝶KIS私有云服务器上，通过【开始】→【所有程序】→【金蝶KIS云·专业版】→【金蝶KIS云·专业版-系统管理】→【账套管理】界面，使用【恢复账套】功能，恢复低版本的账套，点击该账套的【更多】→【升级】

![image.png](https://vip.kingdee.com/download/01096d4700753a0b4b689ceeeb3f35613017.png)

## 二、KIS云专业版不支持迁移到金蝶云·星辰的情况

KIS云专业版存在下列情况的账套，整个账套不支持迁移到星辰：

| 会计期间设置为 13 期的账套 |
| --- |
| 业务和财务均未结束初始化的账套 |
| 物料中设置了辅助属性的账套 |
| 科目级次超过 9 级，或增加了表外科目的账套科目体系为小企业会计制度或小企业会计准则，但增加了共同类科目的账套 |

## 三、其他注意事项

1）KIS 云专业版固定资产卡片未录入原值，对应的卡片不迁移。

2）KIS 云专业版中的用户名无法迁移。凭证、卡片、日记账迁移后，制单人、审核人会自动获取当前操作迁移的用户名。

3）如果压缩后 KIS 云专业版数据量文件过大，无法上传账套文件，我们建议您先通过金蝶 KIS 云专业版【系统工具】-【辅助工具】-【账套结转工具】，先结转新账套后再迁移星辰，原 KIS 云专业版数据可以继续在专业版中查询。

4）未完结的业务单据，如销售出库未开票、采购发票未收款等业务需要在星辰继续处理，建议您可以用Excel 导入星辰，但需要同步修正商品、客户、供应商初始数据。

5）财务三大报表不支持升级（资产负债表、利润表、现金流量表）

## 四、迁移步骤

1、账套备份

1）安装版：通过KIS云专业版【系统管理】-【手动备份账套】，得到后缀为bak、dbb的两个文件

2）KIS云专业版：通过账套列表，备份文件账套后，再通过账套备份列表，下载备份文件。

![image.png](https://vip.kingdee.com/download/010953a0c08df8064fe6b8d932994cde17f4.png)

![image.png](https://vip.kingdee.com/download/0109cd7632195ba449318394f71e79757137.png)

下载下来是ADF后缀的备份。![image.png](https://vip.kingdee.com/download/010971a5cf2b3e9942a9afac71ed7c60141e.png)

需要把.ADF后缀改成.zip：![image.png](https://vip.kingdee.com/download/01099452162b73b0432b8802aff08668000c.png)

解压zip后，得到后缀为bak、dbb的两个文件

2、请使用压缩工具将bak文件压缩成zip格式文件

![image.png](https://vip.kingdee.com/download/01098f1cb51225ef4255afad8110d156fd55.png)

这里需要注意：

1）不要直接改bak的后缀为zip，需要使用压缩工具压缩成格式文件；

2）bak 的文件压缩前文件名不要使用特殊符号或空格，升级可能会识别不到文件。

3、进入金蝶云星辰新建账套界面，切换到“第三方导入”页签，选择【从KIS云专业版导入】的迁移方案

![image.png](https://vip.kingdee.com/download/0109dabc1a24192f4cfc8d4b4b13f495a1f9.png)

4、进入上传压缩文件界面，选择由bak直接压缩成的zip文件上传

![image.png](https://vip.kingdee.com/download/0109e5551157117e4491861927c1e9ccc11c.png)

5、账套上传后，自动预检并恢复数据到迁移服务器，检测数据是否属于BAK的压缩文件，是否满足迁移条件。

![image.png](https://vip.kingdee.com/download/0109807d0922236446b0b4132ef0fd1cf6c5.png)

6、预检成功后，我们建议您点击【检查日志】查看是否有不能迁移的数据。如果有，请在其他数据迁移完成后，再进行数据调整。

![image.png](https://vip.kingdee.com/download/01097842c2c32742479580e8ad5312dcc644.png)

7、预检成功后，点击“开始升级”，自动匹配当前初始化状态的账套进行升级

![image.png](https://vip.kingdee.com/download/01096ad5afcbcea545c0af475300a2d27431.png)

8、迁移完成后，我们建议您点击【导入日志】查看迁移数据明细。如果有数据未迁移成功，会在日志列表显示，请根据日志提示进行数据修复后再迁移

![image.png](https://vip.kingdee.com/download/01099332ff800908476ba8a18173df763f1f.png)

9、迁移成功后，可直接进入账套。如升级成功后，未出现进入账套按钮，请先退出星辰，重新登录账套即可。

## 五、升级对应关系如下表

| 序号 | KIS   专业版数据 | 金蝶云星辰数据 |
| --- | --- | --- |
| 1 | 系统参数 | 有对应的参数对应迁移星辰有专业版没有的，按星辰默认设置专业版有星辰没有的，不迁移 |
| 2 | 核算项目及基础资料 | 对应基础资料 |
| 3 | 交货方式 | 辅助资料-交货方式 |
| 4 | 自定义核算项目 | 辅助资料-新增对应类别 |
| 5 | 现金流量项目 | 被科目引用时迁移到辅助资料 |
| 6 | 科目初始数据余额 | 财务初始余额 |
| 7 | 物料期末余额 | 商品期初 |
| 8 | 客户往来余额 | 客户初始数据 |
| 9 | 供应商往来余额 | 供应商初始数据 |
| 10 | 出纳初始数据 | 账户、出纳初始余额 |
| 11 | 固定资产初始数据 | 资产卡片（初始化资产） |
| 12 |  凭证 |  凭证 |
| 13 | 出纳日记账 | 现金银行流水 |
| 14 | 固定资产卡片 | 资产卡片 |
| 15 | 固定资产变动记录 | 资产变动记录 |
