# 实践案例---F6批量填充

## 1 简介

### 1.1 应用场景

一般单据体的列名有批量填充的按钮时（下图绿色按钮），都代表支持批量填充。

![](https://vip.kingdee.com/download/0100eaeaf5d5ee374f6cb276f7e71f15ef59.png)

但有的字段，即使列名没有这个批量填充的按钮，但是也是支持F6批量填充的（除了系统的标准字段，单据体的自定义字段也支持），而这极易被忽视，但是对于开单来说，是效率神器。

### 1.2 适用版本

星辰所有版本

## 2 应用流程

除了在单据上，例如在一些功能列表，如以销定购、智能补货，F6快捷键也是生效的。使用的方法是，鼠标单据要填充的单元格，光标是蓝色（表示全部选中），然后按键盘F6键，即可向下填充。
