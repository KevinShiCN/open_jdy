# 二、迁移规则及数据说明

**1、单据升级规则：**

1）升级后，已审核采购入库（退货）单会由系统下推一张**已审核的采购发票**。未审核采购入库（退货）单升级后不变。

2）升级后，已审核销售出库（退货）单会由系统下推一张**已审核的销售发票**。未审核销售出库（退货）单升级后不变。

3）费用分摊单关联的出库单，会升级成费用分摊单关联销售发票。

4）资金关联的入库，出库单据，会升级成关联系统下推的发票单据

**2、升级前后数据对比：（下面截图以专业版为例，标准版同理）**

![上传图片](https://vip.kingdee.com/download/010025273b69409d4a208e29d62a39c3d0fb.png)

**3、****特别说明：**

**旗舰版中的销售发票与专业版/标准版中的发票登记状态没有关系**。若专业版/标准版中销售出库单开票状态是未开票或部分开票，只要销售出库单已审核，升级后销售出库单都会按全部数量**自动生成销售发票**。

# 二、 操作说明

1、 新购一套星辰3.1旗舰版，在启用服务后新建账套页面选择《星辰系列迁移》，请仔细阅读迁移前后、以及迁移中的注意事项；

![](https://vip.kingdee.com/download/0100eea1731bbb6d45b3a923fa615d49f70a.png)

![](https://vip.kingdee.com/download/010960f46973c0a249d6aa36a8f2695bf66e.png)

2、 输入账号后选择需要迁移的星辰3.1专业版、标准版的账套；

![](https://vip.kingdee.com/download/0100a4d1870d69d642bcb36e75df302eef63.png)

![](https://vip.kingdee.com/download/01004b600b0cf9db47a18caa63db5d6e56cb.png)

3、 迁移前会进行账套预检，若存在中断项则不允许继续升级，不存在中断项的情况下可以继续升级，但请您留意预警项内容；

![上传图片](https://vip.kingdee.com/download/010023b65885c67f46709f17209c483dd0d5.png)

4、 迁移过程中您可以退出页面，我们会记住当前迁移的进度（重新登录后输入相同账号、选择相同账套）

5、 迁移完成后请重新登陆
