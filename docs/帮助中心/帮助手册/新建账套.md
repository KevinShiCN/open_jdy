# 1 建账方式

在工作台启用星辰的新购服务后，点击进入产品，

![上传图片](https://vip.kingdee.com/download/0100346654729db34547b376a7eb9ffc4ab6.png)

会出现建账页面，星辰新建账套有3种方式：

- 创建空白账套，然后通过引入的方式完成资料导入和初始化的配置
- 通过迁移工具进行第三方的迁移。
- 星辰系列迁移。目前只适用于星辰旗舰版，例如从标准版/专业版迁移到星辰旗舰版。

# **2 新建账套**

启用服务成功，系统自动跳转到新建账套界面。如下图所示：

![](https://vip.kingdee.com/download/0100fde79881558443dbb0d3f09b8086308a.png)

## 2.1 账套基础信息设置

1、公司名称：录入公司的名称即可，同时也是账套名称；

2、系统起始期间：账套开始使用时间，即从哪个期间开始录入数据；

3、期间设置（点击系统起始期间右侧的日历控件按钮可以看到）：

![](https://vip.kingdee.com/download/01001f4580779ccf4abb92328c8a7a4f3edb.png)

系统默认为自然月份会计期间，即一年12个期间，每个期间的1日为期间开始日，也可以设置自定义会计期间；

![](https://vip.kingdee.com/download/010022e55cfb2e2244269f8d18beeb59b789.png)

>
注意：建账后将按上表规律生成启用年度前5年至后15年的会计期间

4、本位币：默认人民币，可下拉选择其他币别为本位币；

5、会计制度：账套使用的会计科目体系，可下拉选择切换，单击后方的科目表、利润表和资产负债表可以预览对应制度的科目及报表结构；支持的会计制度有：小企业会计准则（2013年颁）、新企业会计准则（2006颁）、新企业会计准则(2018未执行新金融和新收入准则)、新企业会计准则(2019执行新收入新金融新租赁准则)、民间非营利组织会计制度、农民专业合作社会计制度、基层工会会计制度。

![](https://vip.kingdee.com/download/01000d7b9574410344119278611eb91bc7d6.png)

## 2.2 所属行业选择

录入完基础信息后，单击【下一步】，进入的所属行业选择界面，根据企业实际所处行业选择即可。

![](https://vip.kingdee.com/download/01003c0951ad19dd49d8ac76eed19f48c04d.png)

上述信息确认无误后，单击【开始建账】，系统开始进行建账。建账完成后自动进入软件操作界面，然后按照初始化流程（插入链接）来进行系统信息的配置即可，如果基础资料和信息比较多，建议通过excel引入的方式来配置。引入相关的内容可以查看下方的帖子：

## 2.3 注意事项

1、系统起始期间默认按自然年度期间，若您未购买财务模块，建议您使用默认值即可；若您购买了财务模块，这里具体是选择自然年度期间还是自定义期间，请您咨询您公司财务。中国大陆的公司一般都使用的是自然年度期间。

2、支持自定义期间吗？

答：支持自定义期间。在新建账套时，鼠标放置在系统起始期间上，后方会出现日历控件按钮。

![上传图片](https://vip.kingdee.com/download/010070300f09ef1342b399810425993b55eb.png)

点击后出现期间页面,

![上传图片](https://vip.kingdee.com/download/01003b5beac572714a2b99aa2b59c1c8ebf2.png)

单击“期间设置”后，可以选择系统期间为自然年度期间还是自定义期间，默认为自然年度期间。

![上传图片](https://vip.kingdee.com/download/0100dd597e812b8e4a27b67d50fb8c455672.png)

## 2.4 自定义期间设置

在新建账套时点击系统起始期间后方的日历logo后，再点击“期间设置”，在期间设置弹窗中选中自定义期间，然后设置好起始年度和开始日期，点击保存，回到新建账套界面选择起始期间，然后建账即可。

举例说明：某公司采用自定义期间，每月21号到下月的20号为一个期间，12月21日到1月20日为每年第一期。且系统是从2021年10月21日开始启用的，那么在新建账套时期间设置应该这样设置：

![](https://vip.kingdee.com/download/010085a54abe0fc844369c727c712df9862d.png)

## 2.5 本位币设置

本位币中预置了常用的币别，若您使用的本位币确实不在范围内，可以任意选择一个币别进行建账，进入账套后第一时间在币别列表中将现有的本位币名称修改为实际需要的币别名称即可。

详见知识：[新建账套时,如何添加系统预置列表里没有的其他币别作为本位币?](https://vip.kingdee.com/link/s/ZvRZ5)

# **3 迁移工具**

如果在启用星辰前，企业本身有使用其他云ERP产品，可以通过迁移工具来进行数据的迁移。

![](https://vip.kingdee.com/download/0100d9274ff3424041de91bdaef9e9b969af.png)

星辰目前支持精斗云、KIS及部分第三方产品的迁移，具体支持的产品及升级内容、注意事项，可以点击专题

[【星辰迁移工具】](https://vip.kingdee.com/link/s/lH0e2)进行查看。
