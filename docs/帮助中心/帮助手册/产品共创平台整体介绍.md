# 产品共创平台整体介绍

# 1 整体概述​

## 1.1 介绍​

产品共创平台是面向真实用户的建议反馈平台，通过点赞机制筛选出优质需求，高赞建议将由研发团队进行评估，为产品创新提供精准、贴合场景的参考依据。

![上传图片](https://vip.kingdee.com/download/0100e555749383ec4bce9deea54a2829b7e6.png)

## 1.2 适用版本​

星辰所有版本​

## 1.3 业务场景​

用户的核心关注点在于如何快速反馈需求、如何让想法被重视，研发团队的关注点在于如何发现广泛存在、真实的痛点。产品共创平台打通“需求提出 - 筛选 - 评估 - 落地” 的全链路，让用户的想法转化为产品创新依据。

![上传图片](https://vip.kingdee.com/download/0100678ba0fb869c45539da87818ceeb78bb.png)

# 2 功能亮点​

## 2.1 智能匹配相似需求

通过背景、问题、方案建议等信息，快速生成摘要，并匹配相似需求，可以对相似度高的需求进行点赞，确保高价值需求脱颖而出

![上传图片](https://vip.kingdee.com/download/0100ea4e37bd3aef466b8fc78b27716f5246.png)

## 2.2 公平筛选机制​

以点赞数为核心筛选维度，让真正有价值、广泛存在的需求凸显，确保需求筛选的公平性与高效性，​实时展示点赞数情况，清晰呈现热点需求，帮助研发团队快速聚焦共性需求。​

![上传图片](https://vip.kingdee.com/download/0100a6d73290ad20435a8b16fc3288b574a2.png)

## 2.3 直达研发评估通道​

产品经理将对高赞需求进行专业评估，评估结果将通过平台反馈给提交人，形成闭环管理

![上传图片](https://vip.kingdee.com/download/010008c50f6f858d4092a08f90fb76b18789.png)

​注：产品共创平台部分功能将陆续上线，后续功能更新动态将通过金蝶云社区公告同步，敬请关注。
