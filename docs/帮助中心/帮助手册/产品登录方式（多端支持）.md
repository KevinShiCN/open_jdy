# 4 钉钉专版/链接器

## **钉钉专版**

是基于金蝶云星辰专业版并融合钉钉的内外部协同能力、仅在钉钉平台销售的专版产品。

产品与钉钉深度融合，包含钉钉费用报销单审批后自动记账，经营数据推送，金蝶云星辰业务单据同步钉钉智能审批等能力。

星辰官网介绍页：[https://www.jdy.com/dd/](https://www.jdy.com/dd/)

## **钉钉链接器**

解决金蝶云星辰需要接入到钉钉平台中使用问题。很多购买了云星辰的客户，同时使用钉钉作为内部交流或者跟客户沟通的OA工具，希望云星辰和钉钉平台打通系统的业务数据，提升内外部的协同。

具体的安装和操作请查看：[金蝶云星辰（钉钉专版/钉钉连接器）功能介绍](https://vip.kingdee.com/link/s/ZSAkm)

更多关于钉钉专版/链接器的内容，请移步到专题查看：[钉钉连接器/钉钉专版](https://vip.kingdee.com/link/s/ZSA83)

# 5 企微专版/**企微连接插件**

## **企微专版**

是基于金蝶云·星辰并融合企业微信的内外部协同能力、仅在企业微信平台销售的专版产品。旨在助力品牌代理经销商企业管理销售外勤和客户拜访、线上拓客，提升企业销售效率和销售业绩，同时可以帮助企业实现采购、销售、库存和往来管理，提升内部协同效率。

星辰官网介绍页：[https://www.jdy.com/kxzs/](https://www.jdy.com/kxzs/)  （内含体验及下载的入口）

## **企微连接插件**

是解决金蝶云星辰需要接入到企微中使用的问题。已经购买了金蝶云·星辰的客户，有同时使用企业微信作为内部交流或者跟客户沟通的OA工具，希望金蝶云·星辰和企业微信打通系统的业务数据，提升内外部协同

具体的安装和操作请查看：[金蝶云星辰（企微连接插件）](https://vip.kingdee.com/link/s/ZS0hQ)

更多关于企微专版/企微连接插件的内容，请移步到专题查看：[企微通/企微专版](https://vip.kingdee.com/link/s/ZS0fJ)
