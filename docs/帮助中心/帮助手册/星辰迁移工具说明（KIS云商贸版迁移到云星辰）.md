# 星辰迁移工具说明（KIS云商贸版迁移到云星辰）

> 产品：金蝶云·星辰 | 分类：帮助手册 | 类型：功能介绍, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：9401 | 有用：29

## 一、支持迁移的商贸版版本：8.0、9.0，其他低版本请先升级到这两个版本。

1、KIS云商贸版下载地址：[https://cloud.189.cn/web/share?code=JV7Jva6jYfeu](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fcloud.189.cn%2Fweb%2Fshare%3Fcode%3DJV7Jva6jYfeu) （密码：sky6）

2、KIS云商贸版低版本如何升级到8.0、9.0

1）通过金蝶KIS云·商贸版V8.0、V9.0账套管理中的升级功能升级。具体操作路径：在金蝶KIS商贸版私有云服务器上，通过【开始】→【所有程序】→【金蝶KIS云·商贸版】→【金蝶KIS云·商贸版-系统管理】→【账套管理】界面，使用【恢复账套】功能，恢复低版本的账套，点击该账套的【更多】→【升级】。

![上传图片](https://vip.kingdee.com/download/0100bbedb6394de3420cb3c408e44fe5bcba.png)

2）如果是KIS商贸版的基础版或标准版，则需要通过【开始】→【所有程序】→【金蝶KIS云·商贸版】→【系统工具】→【辅助工具】→【KIS商贸版迁移工具】中选择低版本账套来迁移。

![image.png](https://vip.kingdee.com/download/0109636d47540525402984d9b7da25fd3492.png)

## 二、商贸版不支持迁移到星辰的情况

KIS允商贸版存在下列情况的账套，整个账套不支持迁移到星辰：

| 不支持启用S/N号管理的账套迁移 |
| --- |
| 不支持启用辅助属性的账套迁移 |
| 不支持有委托代销业务的账套迁移 |
| 不支持有返利业务的账套迁移 |
| 不支持有估价入库业务的账套迁移 |

## 三、其他注意事项

1）如果账套过大无法上传（超过200M），请先使用附件“清表指南”中的脚本进行清理；清理后如果还无法上传，请联系金蝶总部协助处理。

2）升级失败，请打开错误日志，复制到Excel方便后续查询原因。

3）KIS商贸版本启用了浮动计量单位，只支持升级到星辰专业版，不支持升级到星辰标准版。

4) KIS商贸版中的采购发票记录、销售发票记录还不支持升级到星辰。

5）KIS商贸版有新增固定资产卡片，升级后，将会自动创建一个默认部门，用于固定资产卡片上对应的部门字段。

6）KIS 商贸版如果有组装拆卸费用时，升级后，将会自动创建一个默认的支出类别“费用”。

7）KIS 商贸版如果采购费用没有关联生成付款单，升级后，将会自动创建支出类别“费用”，供应商“费用专用”。

8）KIS商贸版中的用户名无法迁移。凭证、单据升迁移后，制单人、审核人会自动获取当前操作迁移的用户名。

9）商品自定义字段不支持升级，其他数据可以升级。

10）单据自定义字段不支持升级，其他数据可以升级。

11)  费用单不支持升级

12）三大报表数据不支持升级（资产负债表、利润表、现金流量表）

13）如果数据量过大，或只需要升级余额，请先通过商贸版-系统工具-辅助工具-账套结转工具，先结转新账套后再升级。或者使用金蝶云·星辰中的升级业务余额的方式来升级。

## 四、升级步骤

1、备份账套：

1）安装版：通过KIS商贸版系统管理-手动备份账套

![image.png](https://vip.kingdee.com/download/01097e36da22080d44f6a3795a778a14ff62.png)

2）KIS云商贸版：通过账套备份列表，下载备份文件

![image.png](https://vip.kingdee.com/download/01090f1843b7487f4959a9b9ee92d4f53e12.png)

生成的备份文件后缀是.ADF：

![上传图片](https://vip.kingdee.com/download/0100628651016ef84973980d67b4b91a88b0.png)

2、把.ADF后缀改成.zip：

![上传图片](https://vip.kingdee.com/download/01009561904525f54f9ebf96743dba2066af.png)

3、解压zip后，得到后缀为bak、dbb的两个文件

4、请使用压缩工具将bak文件压缩成zip格式文件

![image.png](https://vip.kingdee.com/download/0109f3a7ae6efd9941b4b1b858c8e682a7d7.png)

     这里需要注意：

    1）不要直接改bak的后缀为zip，需要使用压缩工具压缩成格式文件；

    2）bak 的文件压缩前文件名不要使用特殊符号或空格，升级可能会识别不到文件。

5、进入金蝶云·星辰新建账套界面，切换到“第三方导入”页签，选择KIS商贸版的迁移方案

![image.png](https://vip.kingdee.com/download/01093cbfe6cbb54f43ae80ad0047468e1030.png)

6、进入上传压缩文件界面，选择由bak直接压缩成的zip文件上传

![image.png](https://vip.kingdee.com/download/0109492b051d0b4a474dbde56f9690d18a7b.png)

7、账套上传后，自动预检，检测数据是否属于BAK的压缩文件，是否满足KIS商贸版迁移条件。

![image.png](https://vip.kingdee.com/download/0109fbd476e43ea648e59e5cad4324566785.png)

8、预检成功后，点击“开始升级”，自动匹配当前初始化状态的账套进行升级

![image.png](https://vip.kingdee.com/download/0109146aa48c38b043bf8457a745edec9670.png)

![image.png](https://vip.kingdee.com/download/01090fc6ebfaade5430685e5a97df66ed9a3.png)

9、迁移成功后，可直接进入账套。如升级成功后，未出现进入账套按钮，请先退出星辰，重新登录账套即可。

![image.png](https://vip.kingdee.com/download/0109ee91961fa976429cafdb8a9355fd100a.png)
