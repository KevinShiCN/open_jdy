# **2 试用账套**

## **2.1 进入的入口**

方式一：通过体验账套左上角快速进入到试用账套中。

方式二：在官网首页直接注册试用账套

下面就针对第二种方式做详细的讲解：

## 2.2 创建方式

### **2.1.1 登录**

打开星辰官网 [jdy.com](https://www.jdy.com/) ，点击导航栏的“免费试用”

![上传图片](https://vip.kingdee.com/download/01000e6378c1a3f14d47acb76a720fa4e853.png)

需使用未注册过的手机号进行注册，已经注册并开通过试用账套的手机号不允许再次试用。

![上传图片](https://vip.kingdee.com/download/01009d91e3c0e5d84283a343d609bf4f48a1.png)

![上传图片](https://vip.kingdee.com/download/0100abb0d4435a0c43f5842d13cedbed07a2.png)

## **2.4 数据迁移（试用转正式）**

**支持标准版升级专业版、同一版本内增减模块迁移试用数据。** （注：试用专业版，购买标准版时，无法迁移试用数据）

举例：

- 同版本：试用星辰标准版进销存模块，购买标准版进销存模块，可迁移数据
- 同版本增模块：试用星辰标准版进销存，购买标准版进销存模块+财务模块，可迁移数据
- 同版本减模块：试用星辰标准版进销存+财务模块，购买标准版进销存模块，可迁移数据
- 升版本：试用星辰标准版进销存，购买专业版进销存模块，可迁移数据
- 升版本增模块：试用星辰标准版进销存，购买专业版进销存模块+财务模块，可迁移数据
- 升版本减模块：试用星辰标准版进销存+财务模块，购买专业版进销存模块，可迁移数据
- 降版本：只能低版本合并到高版本，不支持高版本迁移到低版本。例如试用的是专业版，正式账套是标准版，那么试用转正式时，不支持把专业版的数据合并到标准版，不会出现是否合并试用数据的提示。

**情形1：****企业管理员通过试用账套内顶部提示，点击购买；或   企业管理员在工作台，点击转付费**

自动迁移，试用服务直接转为正式服务

**情形2：金蝶工作人员帮企业进行代客下单**

企业管理员启用正式服务时，满足不同条件，产生的效果不同，具体说明如下：

同时满足以下条件时：

1、仅1个试用服务

2、试用服务处于有效期内（从创建账套当天的日期起算，共30天）

3、试用服务与正式服务 版本一致

产生的效果：无需管理员手动确认，系统自动迁移数据

同时满足以下条件时：

1、仅1个试用服务

2、试用服务处于有效期内（从创建账套当天的日期起算，共30天）

3、试用标准版 购买专业版

产生的效果：系统弹窗提醒是否保留试用数据

如无需保留数据，企业管理员点击【否，直接开启】

如需保留数据，企业管理员点击【是，保留并开启】

![上传图片](https://vip.kingdee.com/download/010027478a83fbbc49e799551b9a929b0ed9.png)



同时满足以下条件时：

1、多个试用服务

2、多个试用服务处于有效期内（从创建账套当天的日期起算，共30天）

3、试用标准版 购买专业版  或 试用与购买版本一致

产生的效果：系统弹窗提醒需先选择对应的试用账套数据

![上传图片](https://vip.kingdee.com/download/0100fa7c14c2d179464f95f564f4f5fd45a8.png)

特殊说明：目前1个企业仅支持服务管理员开通1个星辰试用账套，但存在携带账套的情况，该操作会导致当前企业下存在多个试用账套。

举例：a公司管理员向b公司管理员(此时b公司只有唯一一个人)发送邀请，选择允许携带帐套；b公司管理员登陆工作台接受邀请，选择携带帐套。

>
**试用转正式完成后，可以从工作台进入付费账套内，正常使用即可。**

## **2.5 试用时长说明**

| 期间 | 时长说明 |
| --- | --- |
| 试用期 | 15天（含创建账套的当天）转付费可保留数据 |
| 数据只读期 | 第16-30天（也就是从试用服务过期的次日起算，共15天）工作台服务可见，进入产品弹窗提示“您试用的金蝶云•星辰已过期，请您尽快购买。”并提供转付费入口 产品内控制不能新增数据，只能查询和引出。转付费可保留数据 |
| 数据清理期 | 30天以上对账套数据进行回收清理。不可再进入星辰产品内。注意，数据回收是指数据库执行删除，删除后不再支持找回数据。 |

>
温馨提示：如需保留试用数据，建议您在有效期内及时转付费。

# **3 常见问题Q&A**

**Q1：试用账套和正式账套有什么差别？**

答：可以简单认为 试用账套≈只有15天使用时长的正式账套，功能方面几乎一样

**Q2：先在官网注册了试用的星辰标准版，然后付费开通了正式星辰标准版，现在就是有两个产品在工作台，怎么把试用的数据迁移到正式开通的产品里？**

答：企业管理员在工作台启用正式服务时，如试用服务仍处于有效期内（30天），将有弹窗提示数据合并

**Q3：企业管理员在工作台启用正式服务时，弹窗提示合并时选择不合并。后续如果想再迁移试用数据怎么办？**

答：

方式1：如数据量不多，建议通过手工引入引出进行处理；

方式2：如果数据量很多，不想手工引入，只能通过售后，反馈至技术支持部门，经技术支持部评估，如符合数据迁移要求，将提交数据迁移提单（注意由于是跨服务id的数据恢复，此项属于收费服务，具体可咨询售后客服）

**Q4：请问下列情况正常吗？试用了全部模块，正式购买只买了部分模块，进入账套提示“购买模块已经过期”**

情形：试用时选择了全模块，只购买了进销存和委外模块；工作台合并时弹窗提示是否合并试用数据时，选择了合并。

进入星辰产品后，门店、订货商城、财务这些模块都有，但是却使用不了，提示试用模块已过期。

![上传图片](https://vip.kingdee.com/download/01004c1b3bcd72334e96ad5184e0cca3a3be.png)

答：上述提示属于正常。

因选择了合并试用数据，所以全模块菜单都保留；因只购买了进销存和委外模块，故其他模块不可使用，购买后即可正常使用其他模块。如无需购买可将其他模块的菜单进行隐藏。

解决方案建议：

1、手动勾选不展示生产模块的菜单

2、重新初始化（但是如果有数据的话不建议）

本情形下，如客户非常介意未购买模块菜单，建议不要合并试用数据。
