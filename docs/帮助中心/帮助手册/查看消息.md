# **1 简介**

## **1.1 功能介绍**

查看消息功能主要用于产品共创平台内，用户实时追踪已发布需求的互动反馈、跟进需求进展、接收关键节点通知等，包括他人对需求的点赞、评论、回复、需求采纳通知、需求状态变更提醒等内容。

![上传图片](https://vip.kingdee.com/download/0100ac82f444c616449eb6c010cc64417385.png)

## **1.2 适用版本**

星辰所有版本

# **2 操作指南**

您可以直接点击消息，查看最新需求状态变化；也可以点击“更多”，查看全部消息。
