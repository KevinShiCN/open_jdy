# 发布需求

# **1 简介**

## **1.1 功能介绍**

发布需求功能主要应用于产品共创平台内发起产品优化建议，可以解决需求信息分散、对接低效、反馈无闭环的问题，从而帮助用户快速触达研发团队，高效传递需求核心诉求与落地期望。

![上传图片](https://vip.kingdee.com/download/0100deae20c85ef441a7ab593f909d0416e9.png)

## **1.2 适用版本**

星辰所有版本。

## **1.3 系统路径**

产品共创平台（导航栏）→发布需求。

![上传图片](https://vip.kingdee.com/download/010037d5f784ea30474f96c8a8698a99d0d9.png)

## **1.4 字段说明**

**模块**：即当前所反馈需求所处的模块，如财务云、进销存云、生产云等，需求将会根据所选模块反馈至对应的研发团队。

**角色：**即您当前在企业内所处的角色，如财务、采购、销售等。

![上传图片](https://vip.kingdee.com/download/01002e8152d7d98c4d908718c17caef93691.png)

# **2 操作指南**

- 第一步，根据需求问题，选择好模块，并选择好您的角色。

>
注意：当模块选择错误时，将会影响需求评估的反馈速度。

- 第二步，结合您的问题，分别填写业务背景、遇到问题、方案建议，在描述**业务背景**时，需要您从业务价值角度描述需求的背景信息，常见的业务背景有：部门架构、经营决策、法律合规、工作效率等方面产生的需求；**遇到问题**，指的是对您造成的实际困难或不满，常见的问题有：重复操作、数据不准确、用户体验不佳等，在描述问题时，您可以采用“程度”+“范围”的方式，例如“单次处理需要2小时，每月需要处理3-4次”；**方案建议**，指的是您希望可以采取的产品优化措施，您的建议将会作为研发人员产品优化的重要参考来源。
- 第三步，生成摘要，当您填写完业务背景、遇到问题、方案建议后，将会智能生成需求摘要作为标题，并为您匹配相似的需求，在提交需求前，您可以根据相似度，查看相似需求，如果他人的描述与您的想法一致，您可以为该需求点赞，从而避免重复提交。
- 第四步，当您确认需要提交需求后，可以选择补充附件（截图、pdf、xlxs文件等），之后点击提交，确认提交成功。
