# POS密码修改

# 一、星辰后台重置密码

使用场景：如收银员忘记密码，可在后台进行重置密码。

操作路径：【零售资料】——【收银员管理】——【重置密码】

![上传图片](https://vip.kingdee.com/download/010088d8e6a11669473c86f9ac023b016766.png)

# 二、POS端修改密码

使用场景：收银员密码被其他人知道，为防止个人信息安全，需要修改密码。如忘记密码，请联系账套管理员在星辰后台收银员管理处重置密码。

操作路径：POS——【设置】——【改修密码】
