# 操作指南

>
配置钱箱需要将钱箱的连接线接入到小票打印机，如收银机内置打印机的，则需要在有线连接的状态才能使用自动弹出钱箱的功能。

## 一、自动开钱箱设置

在打印机设置中勾选“开钱箱”,当结算收款选择"现金收款"时，钱箱则自动弹出。

![上传图片](https://vip.kingdee.com/download/01001c074be637304d9489a31a676db52243.png)

在收银员的权限设置中设置好弹出钱箱的密码。

![上传图片](https://vip.kingdee.com/download/01002e1597ed155f4090a7e6eddd379d365c.png)

## 二、手工打开钱箱

在开单界面，点击“...”，选择“开钱箱”，则钱箱弹出。

![上传图片](https://vip.kingdee.com/download/01001a165c0ec93d40b0b62c276432366191.png)
