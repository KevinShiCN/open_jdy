# 操作指南

开单时选择“自提配送”，结算完成后零售单流转到【配送台】，在配送台进行拣货发货，完成流程单据自动结存。若手动设置过默认仓库或开单手动选择过仓库，则在新选择的仓库对应的POS配送台进行拣货发货。
