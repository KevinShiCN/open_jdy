# 操作指南

登录金蝶云星辰后台【零售资料】>【基础设置】>【门店POS图片管理】，可以设置POS机上的显示图片，包含单屏收银机、双屏收银机和自助收银机设置。

![上传图片](https://vip.kingdee.com/download/010061ebf05bdbc44c17b35982e124b0cd1b.png)

具体图片规格按系统说明为准:

-
PC POS主屏LOGO：支持.png，.jpg，.jpeg的图片类型，最多上传1张，建议单张图片不超过5M；

-
PC POS副屏推广轮播图支持.png，.jpg，.jpeg，.gif的图片类型，最多上传10张，建议单张图片不超过20M

-
大屏自助POS主屏背景图：支持.png，.jpg，.jpeg，.gif的图片类型，最多上传1张，建议单张图片不超过20M，尺寸不超过1080乘1920

-
支持上传POS副屏推广视频，仅支持.mp4视频类型，最大50M

设置完门店POS图片方案之后，可以选择适用门店，全部设置完成检查无误后点击保存、启用。
