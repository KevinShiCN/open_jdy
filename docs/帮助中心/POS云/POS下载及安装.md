# POS下载及安装

> 产品：金蝶云·星辰 | 分类：POS云 | 类型：功能介绍, 快速入门
> 作者：Weixinxin | 阅读：488 | 有用：1

# 一、POS下载

使用管理员账号登录进金蝶云·星辰官网，进入【零售资料】-【基础设置】-【POS安装包下载】点击POS安装包下载，选择版本，点击下载即可。

![上传图片](https://vip.kingdee.com/download/01006b65503924774369bd5676b6fafafa84.png)

![上传图片](https://vip.kingdee.com/download/01001f2ba153b4a448659d5c17f74e166484.png)

>
**注意：**

**如果安装版本较高，请通过下面链接进行下载：**

**安卓高版本**（超过安卓12及以上）：安装包链接 [https://cdn.jdy.com/third/retail_android502.apk](https://cdn.jdy.com/third/retail_android502.apk)

**IOS高版本**：星辰3.1用户可以直接在APPStore上搜索【金蝶云星辰POS】下载安装即可

# 二、POS安装

双击安装包，点击下一步，选择安装目录，可以默认安装，然后点击安装即可。
