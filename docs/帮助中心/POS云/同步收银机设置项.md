# 同步收银机设置项

> 产品：金蝶云·星辰 | 分类：POS云 | 类型：功能介绍, 快速入门
> 作者：Weixinxin | 阅读：228 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/748857129896749312)

# 一、应用场景

当在一台收银机上配置好参数，可一键将配置同步到其他门店和收银机，适用于远距离多门店的设置。

# 二、操作路径

POS>设置>同步设置项

# 三、操作指南

选择对应需要同步的设置，点击“同步到其他收银机”

![上传图片](https://vip.kingdee.com/download/0100b211946cf3134c3cb7837c00448d0998.png)

选择门店或者直接选择需要同步的收银机进行同步

![上传图片](https://vip.kingdee.com/download/0100b3d13b90bdd04bdea97d84ead7964181.png)
