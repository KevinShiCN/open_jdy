# POS使用整体介绍

## **1 产品概述**

### **1.1 产品介绍**

本专题围绕金蝶云・星辰 POS 系统的实际操作展开，旨在为零售门店收银员、管理人员提供标准化的业务操作指南。系统衔接星辰后台数据，覆盖从开单收银到交班核对、发票管理的全业务链条，支持权限管控、数据同步与个性化配置，确保门店日常收银业务规范高效，同时为后台数据统计、业绩核算与库存管理提供准确数据支撑。

## **二、业务场景**

### 1.2 适用版本

星辰3.0以上标准版/专业版/旗舰版

### **1.3 业务场景**

-
**日常收银开单场景**：门店接待顾客时，收银员需快速完成商品选择、会员绑定、优惠应用、支付收款等操作，部分场景需进行改价、挂单 / 取单或促销查询，权限不足时需发起现场 / 远程授权。

-
**售后退货场景**：顾客因商品问题申请退货时，可通过源单（关联原零售单）或无源单（无原单）方式处理，需按权限完成退货操作并扣减对应库存。

-
**收银员交接班场景**：当班收银员结束工作后，需核对收款数据、经营概况、现金明细，完成现金上缴并确认交班，管理人员在后台审核缴款数据并查看交班记录。

-
**顾客开票场景**：顾客购物后需开具发票时，通过 POS 打印含开票二维码的小票，顾客扫码自助开票，管理人员可在后台查询开票记录。
