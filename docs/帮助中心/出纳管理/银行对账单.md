# 银行对账单

> 产品：金蝶云·星辰 | 分类：出纳管理 | 类型：快速入门, 整体介绍
> 作者：Zyl | 阅读：3555 | 有用：8
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/159613988622870784)

1、业务场景：

企业发生的业务资金通过银行进行结转，银行都会客观地逐笔记录下来。企业需要把银行对账单与企业内部出纳根据日常业务登记的日记账在同一个系统里自动进行核对，故系统提供银行对账单的录入和导入功能。

2、应用指南：

单击【出纳管理】→【现金银行】→【银行对账单】，进入列表。
