# 银行存款对账是否支持跨币别对账?

【问题描述】

银行存款对账是否支持跨币别对账?

【解释说明】

**银行存款对账不支持跨币别对账**。业务场景:每月月末,企业的出纳人员将企业的银行存款日记账与开户银行发来的当月银行存款对账单进行逐笔核对,勾对已达账项,找出未达账项,并编制每月银行存款余额调节表。
