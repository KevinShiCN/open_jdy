# 如何查询/导出/打印序时账?

> 产品：金蝶云·星辰 | 分类：出纳管理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：24532 | 有用：9
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/395742741558258944)

【问题描述】

如何查询/导出/打印序时账?

【名称解释】

**序时账:**是按照经济业务发生的时间先后顺序逐日逐笔登记的账簿。

建议在系统中可以导出凭证明细或者是导出日记账记录。

【操作步骤】

点击【财务处理】>【查凭证】,进入到查询凭证列表的界面

![](https://vip.kingdee.com/download/010921bdb13f94ad4308ad22676d66815238.png)

点击<展开过滤条件>,根据需要进行筛选查询凭证的期间、凭证状态、制单人等信息

![](https://vip.kingdee.com/download/010958442892be83454086f76d768d7dc040.png)

也可以点击<常用过滤条件>-<新增方案>

![](https://vip.kingdee.com/download/010920066cd4fb954408aac0b489b9f72d45.png)
可以在新增方案的界面,根据需要添加条件,并录入方案名称,点击保存即可查询到需要查询到的信息

![](https://vip.kingdee.com/download/0109ee850981e095484badda4080c179606b.png)

查询出来的结果就是序时账,点击右上角的<引出>即可。如图:

![](https://vip.kingdee.com/download/0109ea89155fa89c45c88ea2c26bdc9f2911.png)

导出后效果:

![](https://vip.kingdee.com/download/0109a412747051b44b7ebd9a1e0438ade761.png)
若需按某个字段排序引出,可在系统点击字段名称排序后再引出,如按日期排序引出

![](https://vip.kingdee.com/download/0109ca2c4e0477ca421c84121ad61b380375.png)![](https://vip.kingdee.com/download/01098cdd457c870b4b048cb56df12421786b.png)

(注意:若导出日记账对应的凭证,可在筛选条件中过滤库存现金、银行存款等科目后查询导出)

**凭证序时账暂不支持打印,建议先引出为Excel后,再通过Excel打印凭证序时账**。

**若想了解如何打印凭证,请查看:[如何设置凭证打印模板打印凭证?](https://vip.kingdee.com/knowledge/626946834358869248?productLineId=35&isKnowledge=2&lang=zh-CN)**

其次,若出纳岗位已在日记账录入现金银行流水信息,想直接导出日记账,可打开【出纳管理】-【日记账】,选择需要查询期限,左边选择需要查看导出的账户,点击右上角引出按钮即可引出Excel表格,如图:

![](https://vip.kingdee.com/download/01091c67a127b31745828ce01a38bac1b71e.png)点击引出下的<引出全部>,可以引出所查全部账户的日记账流水

![](https://vip.kingdee.com/download/010994b643386a9048b1a111587aca37b723.png)
也可以在日记账界面点击<序时簿>

![](https://vip.kingdee.com/download/0109ae19ea8902b746efa414332a302e1143.png)
根据需要筛选查询后点击右上角<引出>按钮

![](https://vip.kingdee.com/download/010937f8e9d0c8c3449b94b2512746e45a37.png)

**若想了解如何打印日记账,请查看:[如何打印日记账?](https://vip.kingdee.com/knowledge/585551599754262528?productLineId=35&isKnowledge=2&lang=zh-CN)**

【温馨提示】

可以将序时账导出后自行调整再打印。
