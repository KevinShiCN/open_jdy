# 出纳结账

> 产品：金蝶云·星辰 | 分类：出纳管理 | 类型：快速入门, 整体介绍
> 作者：Zyl | 阅读：4073 | 有用：19
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/159617024677627392)

1、 【系统设置】-【系统参数】-【出纳参数】中未勾选【与总账一起结账】时，出纳提供期末结账的功能。

![](https://vip.kingdee.com/download/0100434370073c8b43a693212c822eb15c1e.png)

2、 单击【出纳管理】→【期末结账】→【出纳结账】，进入“出纳结账”页面。如下图所示：

![](https://vip.kingdee.com/download/0100085be742fcff41219302b80d48427fe6.png)

![](https://vip.kingdee.com/download/01007d5418db4913472db5a6d50c8695a48d.png)

3、 在进行期末结账前，系统提供“结账前检查”功能，可以对期末结账前相关业务进行检查。

![](https://vip.kingdee.com/download/010021e5084c2d9d45528f0eac775ea3aacc.png)

检查项包括：账户初始余额是否录入、业务单据是否生成了日记账、日记账是否生成了凭证、出纳与总账对账是否平衡

1）账户初始余额是否录入：第一期只预警提醒，不强制要求；后续期间只显示“正常”，不再提示；

2）业务单据是否生成了日记账：当系统参数中选择的是单据登账时，结账前检查会检查业务单据是否生成了日记账。如存在未生成的，则会预警提醒。 并可关联到“业务单据生成日记账”列表；

3）日记账是否生成了凭证：如存在未生成凭证的日记账，结账前检查时会给出预警提示，并可关联到业务单据生成凭证界面。如果是单据生成的日记账，日记账不需生成凭证；

4）出纳与总账对账是否平衡：如果不平衡，也会预警提醒，可打开对账详情界面查看。

4、单击检查项目，可以进入异常处理界面，进行相关业务处理。

![](https://vip.kingdee.com/download/010033eb4ecfdc7b4ab19596b16c0a76308a.png)

5、单击“结账”，则进行期末结账。

6、系统提供反结账功能，操作与结账一致。
