# 银行流水文件搜集贴

> 产品：金蝶云·星辰 | 分类：出纳管理 | 类型：常见问题, 精华汇总, 进阶技巧
> 作者：Daisy_ | 阅读：1044 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/657174138485778176)

【问题】

您是否在星辰的出纳管理-日记账-流水引入时，出现了引入结果：成功 0 ，失败xxx 的情况？

【方案】

- 根据失败详情检查，是否有必录项为空的情况；
- 核对银行流水是否为系统已支持的银行，目前星辰已支持：建设银行、招商银行、工商银行、平安银行、兴业银行、交通银行、农业银行、邮政储蓄、光大银行、中国银行、华夏银行、徽商银行、泰隆银行、浦发银行、稠州银行 、广发银行、杭州联合银行、宁波银行、贵州银行、浙商银行、民生银行、微信、支付宝等，格式可以参考：[https://vip.kingdee.com/link/s/lHwbl](https://vip.kingdee.com/link/s/lHwbl)[](https://vip.kingdee.com/link/s/lHwbl)若格式和支持的格式不一致，可以调整银行流水字段于支持的字段一致，即可导入
- 如果您的银行，目前星辰仍不支持，请在评论区标注银行名称，下载流水的路径，上传文件格式，我们会从评论区搜集客户的银行流水格式予以持续支持。（重要提示：请您在上传流水文件前，把需要保密的文本/数值删除，切勿造成隐私泄露！！！）
