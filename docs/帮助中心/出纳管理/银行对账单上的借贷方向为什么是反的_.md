# 银行对账单上的借贷方向为什么是反的?

> 产品：金蝶云·星辰 | 分类：出纳管理 | 类型：常见问题
> 作者：若有灵兮山之阿 | 阅读：2000 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/617164202951645696)

【问题描述】

银行对账单上的借贷方向为什么是反的?
【情景说明】

银行对账单去核对时,看起来像是反的

![](https://vip.kingdee.com/download/010936879eee1c2d4ff1828a78728ddd055b.png)【解释说明】

银行对账单上的借贷方向之所以可能看起来是反的,主要是因为会计原则和银行对账单记录方式的差异。以下是一些可能的原因:

1. **会计原则**:在会计中,借方和贷方的记录是根据双重记账法进行的。例如,当银行账户收到存款时,在会计账簿上会记录为借方增加,表示资产的增加。而在银行对账单上,这笔存款可能会被标记为贷方,因为银行是从其角度记录的,即负债(客户存款)的增加。

2. **银行视角**:银行对账单是从银行的角度来记录交易的。因此,当客户存入资金时,银行的负债增加,这在对账单上表现为贷方;当客户支出或提款时,银行的负债减少,这在对账单上表现为借方。

3. **客户视角**:客户在查看自己的银行账户时,是从自己的视角出发。存入资金时,客户的资产增加,这在客户的账簿上记录为借方;支出时,客户的资产减少,记录为贷方。

4. **对账过程**:在对账过程中,企业需要将银行对账单上的交易与自己的会计记录进行匹配。这可能需要将银行对账单上的借贷方向转换为会计账簿上的借贷方向,以确保双方记录的一致性。

5. **误解或错误**:有时候,借贷方向的“反”可能是由于误解或记录错误造成的。在这种情况下,需要仔细检查交易记录,确保所有条目都正确无误。

6. **不同的会计系统**:不同的会计系统可能有不同的记录习惯,这也可能导致借贷方向看起来不一致。

了解这些差异有助于正确解读银行对账单,并确保会计记录的准确性。如果有疑问,最好咨询会计专业人士或银行以获得更详细的解释。
