# 银行流水引入日记账，通过往来账户直接带到日记账的客户/供应商字段

> 产品：金蝶云·星辰 | 分类：出纳管理 | 类型：新特性, 进阶技巧, 常见问题
> 作者：Daisy_ | 阅读：659 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/661868843186369792)

问题：通过银行流水引入的日记账，我能否控制往来账户智能自动携带到日记账的客户/供应商字段中？

解决方案：

1.您可以在系统设置-系统参数-出纳参数，勾选参数：【银行流水引入日记账时，根据往来账户自动匹配客户/供应商】，如果不需要可以取消勾选。

![上传图片](https://vip.kingdee.com/download/0100614d4c672f1a4deb9ced5e3704d912db.png)

2.银行流水中涉及的往来账户，会根据多个维度：收支方向/基础资料是否存在该客户/供应商去智能填写您的日记账客户/供应商字段。
