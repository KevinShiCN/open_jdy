# 你以为批量导入很高效？我来告诉你自动生成才是王道！

> 产品：金蝶云·星辰 | 分类：出纳管理 | 类型：知识软文, 快速入门
> 作者：Simone邓 | 阅读：1085 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/321959158406598400)

阿芝是深圳南山高新科技员一家it公司的业务员，因为公司还在初创期，很多小伙伴都是身兼数职，阿芝也不例外。

     今天上午上班高峰期，阿龙在电梯见到满脸倦容的阿芝，关切的问：芝姐，你昨晚又加班了？工作再忙也要注意身体啊。

      阿芝满脸惆怅的说：阿龙，你也知道公司现在人少，我不但要录进销存的单据，还要根据进销存的收支明细到财务模块登记日记账，开始的时候单据少，工作量也不大。现在每天业务单据这么多，每一笔涉及的支出和收入我都要再登记一次，不加班根本做不完，都使用导入的办法进行导入数据了，但是业务量真的太多了，还要将数据录入在导入模板里面，昨晚加班到10点，又刚好下雨，等到11点才打到车，回到家都12点了。。。。。。

      没等阿芝说完，阿龙就说：呀，你不早说，我刚好知道这个功能，系统可以设置，录入单据之后自动登记日记账的。到了办公室，阿龙在阿芝的电脑上操作起来：

      首先进入【系统设置】—【系统参数】-【出纳参数】勾选业务单据审核后自动登账参数，之后进销存的单据自动登记日记账，需要注意的是开启参数后，操作员必须同时拥有新增日记账的权限，审核的业务单据才会自动生成日记账。

![](https://vip.kingdee.com/download/0109782b536d8b3843759e486899ece68b6b.png)

如销售一件商品200元，现销现收款，现金账户进行收款，如果是其他的方式收款，选择其他的账户，如下图：

![图片.png](https://vip.kingdee.com/download/010073eab60f60cb47b3b43e9f39e839517c.png)

之后进入出纳管理-日记账查询，查询日记账如下图：

![](https://vip.kingdee.com/download/0109b4c2beeb14934f4585d0b443708449f0.png)

筛选对应期间的单据，将需要的单据筛选出来，如下图，销售单据收取的货款自动登记日记账

![图片.png](https://vip.kingdee.com/download/01000f8830dfbeb44e859cc337a91b34190b.png)

       阿芝说，不错呀，阿龙，你可减轻了我一半的工作了！阿龙说，不是我减轻你一半工作，我只是恰好发现这个功能而已，是erp的自动化帮你减轻了工作压力，发现和这个功能之后，再也不用担心登记日记账了！系统还要很多新功能等待我们去发现，会更加减轻我们的工作呢！我们一起研究吧！
