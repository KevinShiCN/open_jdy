# 星辰2.0,业务单据生成日记账提示“无日记账新增权限”请联系管理员,如何处理?

> 产品：金蝶云·星辰 | 分类：出纳管理 | 类型：常见问题
> 作者：_Leo | 阅读：254 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391353470789292544)

【问题描述】

在收款单界面,点击【生成日记账】按钮,出现提示:“无日记账新增权限”。

如图:![](https://vip.kingdee.com/download/01099bdd9a54f15248b99c5cc9ac50fa80e8.png)

【操作流程】

在【用户管理】-【功能授权】 ,除了分配【业务单据生成日记账】的查询权限外,还要分配【现金银行流水】新增的权限,子账号才可以按业务生成日记账。

如图:

![](https://vip.kingdee.com/download/010907d4e64f58cc443e82efad0d8c998947.png)
