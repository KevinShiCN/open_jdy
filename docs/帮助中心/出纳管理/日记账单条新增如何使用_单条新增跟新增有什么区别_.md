# 日记账单条新增如何使用?单条新增跟新增有什么区别?

【问题描述】

日记账单条新增如何使用?单条新增跟新增有什么区别?

【解释说明】

单条新增日记账,新增界面保持与日记账详情页面一致,一次仅支持录入一条日记账信息,同时支持新增日记账时上传附件。

日记账单条新增该功能目前正处于验证阶段,若您想体验此新功能,可填写问卷报名申请。(每个新功能/专题特性诚邀10家客户尝鲜验证,名额有限,赶紧报名,请填写问卷[https://survey.kingdee.com/vm/Pk2Pfhz.aspx](https://survey.kingdee.com/vm/Pk2Pfhz.aspx))
