# 业务单据自动生成日记账后,若业务单据生成凭证,生成的凭证是否支持与该日记账关联?

> 产品：金蝶云·星辰 | 分类：出纳管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：600 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/568964241843971072)

【问题描述】

业务单据自动生成日记账后,若业务单据生成凭证,生成的凭证是否支持与该日记账关联?

【业务场景】

出纳的登账方式是“单据登账与手工登账”,业务单据登账生成日记账后对单据生成会计凭证,那么日记账怎么生成凭证呢?

如:在应收应付系统中新增一笔预收款单,并且登账生成了现金日记账,预收款单生成凭证后,现金日记账会自动关联到这笔凭证吗?

【解决方案】

**业务单据登账生成日记账后,对应的业务单据生成了会计凭证,则日记账中会自动关联到对应的凭证字号。**

例如:在应收应付系统中新增一笔预收款单YSKD-20230918-00001,并且登账生成了现金日记账RJZ-20230918-00001,预收款单生成凭证后,现金日记账会自动关联到这笔凭证。

![](https://vip.kingdee.com/download/0109ca9bad9fbd354b61bea9615919a983b8.png)

【温馨提示】

**业务单据生成的日记账不能生成凭证,在【****生成凭证****】>【****日记账****】界面无法查看,只需要对业务单据生成记账凭证即可。**
