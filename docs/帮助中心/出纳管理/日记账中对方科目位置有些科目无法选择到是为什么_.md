# 日记账中对方科目位置有些科目无法选择到是为什么?

> 产品：金蝶云·星辰 | 分类：出纳管理 | 类型：常见问题
> 作者：Schatz | 阅读：186 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/660286733010718976)

【问题描述】

日记账中对方科目位置有些科目无法选择到是为什么?

![](https://vip.kingdee.com/download/0109e55e9d3ae652471fa0f6e88526b2fee7.png)

【解释说明】

在日记账中,如果某些科目无法被选择为对方科目,可以从以下两个方面进行排查:

**1、科目被禁用**

进入【基础资料】-【科目】界面,清除“启用状态”的筛选条件(或者设置为同时查询启用和禁用的科目),检查筛选结果,确认科目是否已被禁用。

![](https://vip.kingdee.com/download/0109f3ef134904a74c58bbaff9cc7547f0c9.png)

**2、无权限查看科目**

如果即使在不设置任何条件的情况下仍然无法找到所需科目,则可能是由于权限限制。需要使用管理员账户登录账套,前往【系统设置】-【用户管理】界面,选择相关子账户并点击授权,在数据授权部分检查科目的权限设置情况。

![](https://vip.kingdee.com/download/0109317711f9552d449ab82fa69db7b96939.png)
