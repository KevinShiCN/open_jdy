# 为什么已经设置了业务单据审核后自动登账，但是收款单中的手续费没有同步到日记账中？

> 产品：金蝶云·星辰 | 分类：出纳管理 | 类型：常见问题, 进阶技巧
> 作者：Daisy_ | 阅读：1433 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/249906370739605504)

问题描述：

为什么已经设置了业务单据审核后自动登账，但是收款单中的手续费没有同步到日记账中？

解决方案：

第一步：【系统设置】-【出纳参数】-登账方式选择：单据登账与手工登账，并勾选业务单据审核后自动登账参数；

![](https://vip.kingdee.com/download/010028d80ba8d69f4ac2892d29fc9c3d82ba.png)第二步：将收款单涉及到的手续费生成其他支出单并审核；

![](https://vip.kingdee.com/download/0100abc89580fcbd4f54b87e5681bd8262fa.png)第三步：查询日记账。

![](https://vip.kingdee.com/download/01002e1ab967699f405eae7da4892ce78123.png)
