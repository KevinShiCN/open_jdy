# 凭证登账下，出现了凭证未生成日记账的情况应该如何处理？

> 产品：金蝶云·星辰 | 分类：出纳管理 | 类型：常见问题, 进阶技巧
> 作者：Daisy_ | 阅读：2408 | 有用：13
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/243682299370226944)

问题描述：

系统参数中选择了登账方式为：凭证登账与手工登账，但是偶尔会出现某张凭证未生成日记账的情况，应该如何处理？

解决方案：

在录凭证过程中，可能会出现网络延迟等特殊原因造成个别凭证未能成功生成日记账的情况，星辰对这种情况是做了补偿机制的，可通过【出纳管理】-【日记账】-【引入】-【总账引入】的方式，引入未成功生成的日记账，保持日记账与总账一致。
