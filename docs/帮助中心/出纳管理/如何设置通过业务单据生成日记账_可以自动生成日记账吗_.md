# 如何设置通过业务单据生成日记账?可以自动生成日记账吗?

> 产品：金蝶云·星辰 | 分类：出纳管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：2243 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/585552454704546816)

【问题描述】

如何设置通过业务单据生成日记账?可以自动生成日记账吗?

【解释说明】

出纳参数登账方式选择“**单据登账与手工登账**”,业务单据审核后,支持将业务单据生成日记账,**如参数同时选择“业务单据审核后自动登账”,则单据审核后会自动生成日记账。**

业务单据不是必须生成日记账,也可手工录入日记账,是手工录入还是通过单据生成,可根据日常需要选择,在通过手工录入日记账时,注意不要再通过单据重复生成,确保数据正确即可。

【操作指引】

1.菜单栏点击【系统设置】>【系统参数】。

![](https://vip.kingdee.com/download/01090aa2be62d3e8465abc25458b7247dd93.png)

2.出纳参数中,登账方式选择“单据登账与手工登账”,业务单据中录入了收付款金额,单据审核后,可选择业务单据生成凭证。

![](https://vip.kingdee.com/download/0109a31cb3a9dae4420b8bdf5da9f109716d.png)

3.菜单栏点击【出纳管理】>【业务单据生成日记账】。

![](https://vip.kingdee.com/download/0109b91d58b1cbef4a8cb8059f88988fcf9b.png)

勾选对应的业务单据,生成日记账,如勾选多张,每一张单据会生成一条日记账记录

①**日记账日期**:可选择单据日期、审核日期和指定日期

②**日记账摘要**:可自定义日记账摘要取值字段

![](https://vip.kingdee.com/download/0109671b30be5f8a48caa67ffd7859e6abd2.png)

例如预收单生成日记账,勾选单据后点击<生成日记账>,选择日期与摘要取值来源后点击<生成>即可。

![](https://vip.kingdee.com/download/0109e966e21023ee4144906ca4e3b5447844.png)

4.若参数中勾选了“业务单据审核后自动登账”。

![](https://vip.kingdee.com/download/010975d95ea9006d4c62a8a28b5547e8ef8b.png)

则业务单据审核后会自动生成一条日记账记录,例如在预收账款列表,找到"收款信息"点击"日记账编号"可查看关联的日记账信息。

**若想了解如何显示单据字段,请参考:[如何设置单据/报表字段的显示或隐藏?](https://vip.kingdee.com/knowledge/547945651133299200?productLineId=35&isKnowledge=2&lang=zh-CN)**

![](https://vip.kingdee.com/download/0109adec0107b5644b569062ac658aa89061.png)

业务单据生成日记账后,如需要反审核业务单据,需要先删除日记账记录才能反审核。
![](https://vip.kingdee.com/download/0109a93e45503997447bbad7ed57a03839d5.png)
