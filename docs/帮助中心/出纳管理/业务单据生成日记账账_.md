# 业务单据生成日记账账


> 产品：金蝶云·星辰 | 分类：出纳管理 | 类型：整体介绍, 快速入门
> 作者：Zyl | 阅读：5421 | 有用：11
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/159613177108350464)

1、业务场景：

同时购买了进销存云+财务云的客户，业务单据中录入了收付款金额，单据审核后，可通过单据界面的“生成出纳日记账”按钮或通过【出纳管理】→【业务单据生成日记账】功能生成日记账，无需重复手工新增日记账。涉及到的单据类型包括以下15种：采购入库/退货单、销售出库/退货单、收款单、预收/退款单、付款单、预付/退款单、其他收入/退款单、其他支出/退款单、资金转账单中，有收付款的情况。

2、应用指南：

单击【出纳管理】→【业务单据生成日记账】进入列表，查询出已经审核过的涉及到收付款业务的单据生成日记账。

![](https://vip.kingdee.com/download/010962411171b5c64a68af59c63275b3a8d3.png)

![](https://vip.kingdee.com/download/01002a1175cd33fe438c897879070e6d35be.png)

生成日记账时，支持按单据日期、单据审核日期或者手工指定日记账日期来生成。

单据日期：业务单据上的单据日期

审核日期：业务单据的审核日期

手工指定：手工选择一个日期，不能为出纳已结账期间的日期。

**当系统参数选择“单据审核后自动生成日记账”，自动生成日记账的日期默认按单据日期。可以手动选择审核日期后，后续将会默认按审核日期生成。手工自定日期只当次生效。**

**报销单按付款日期生成日记账，请选择“审核日期”生成。**

![](https://vip.kingdee.com/download/0109f23672d761b44742b17d5733ff199c3e.png)



业务单据生成日记账后，如需要反审核业务单据，则需要先删除日记账记录才能反审核。
