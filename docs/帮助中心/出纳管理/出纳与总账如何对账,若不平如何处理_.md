# 出纳与总账如何对账,若不平如何处理?

【问题描述】

出纳与总账如何对账,若不平如何处理?

【解释说明】

可通过【出纳管理】--【结账】--【结账前检查】或【账务处理】--【财务业务对账】--【总账与出纳对账】来检查出纳与总账是否平衡一致

![](https://vip.kingdee.com/download/01093174b46535f14213867232b1bf67d140.png)![](https://vip.kingdee.com/download/010906abb9e1b895435da531adb8abb32f26.png)

若是提示日记账与总账对账不平?账户余额和科目余额不一致,或对账结果显示不平时可通过以下解决方案排查处理

![](https://vip.kingdee.com/download/010915895392c0fe4a668773a88c48f1353f.png)![](https://vip.kingdee.com/download/01097a1a77dca9f04863ad206cefda019e22.png)
【解决方案】

财务与出纳对账不平,提示“日记账与总账对账不平”表示出纳的账户余额和总账现金/银行科目的余额不一致,检查提示不平可以继续结账,若需要检查并调整平衡再结账可以参考如下步骤操作:

1、检查账户关联的科目设置是否正确

依次点击【账务处理】--【科目关联设置】--【账户管理】打开账户关联科目设置界面。

![](https://vip.kingdee.com/download/0109a3fe0632db5f4b699118aab78c91786c.png)
检查出纳账户是否关联了对应的会计科目,及关联的会计科目是否正确;若设置错误可以点击设置科目关联正确的会计科目后再重新核对。

![](https://vip.kingdee.com/download/0109ce7beff23c2a48df8c7606b2cc9fd57f.png)

2、检查具体是从哪一期开始对账不平

可以通过【账务处理】--【财务业务对账】--【总账与出纳对账】界面左上角选择期间,再点击右上角“开始核对”,检查具体是从哪一期开始数据不一致。

![](https://vip.kingdee.com/download/01090b85652ead23400da7c1086c44beae83.png)
2.1  账套启用的第一期数据不一样

若账套启用的第一期数据不一样,先检查初始化数据录入是否正确,依次点击【基础资料】--【财务初始余额】及【账户初始数据】,分别检查现金/银行科目的初始化余额与账户的期初余额是否一样,若存在初始化数据不一样的情况则需要先调整成一致后再重新核对。

![](https://vip.kingdee.com/download/0109d5a0363fcdd34756a1f609794444635e.png)

![](https://vip.kingdee.com/download/0109342ca52cba0248eea3bcf898c7b8e788.png)

2.2  系统启用后某个期间的数据不一样

若发现是从账套启用后中间某一期开始数据不一样了,就检查这一期的相关业务数据(可以用明细账和日记账核对明细):

①是否有录入了业务凭证,但是没有录入日记账,则需要登账生成日记账或者补充日记账。

②是否有日记账,但是没有录入凭证,则需要生成凭证或补充凭证。

③若日记账和凭证是分开录入的,检查是否存在两个数据录的不一样,则需要修改成一致。

④单据/日记账生成凭证,是否有修改了凭证,建议重新生成凭证。

【温馨提示】

出纳管理期末结账的结账前检查可以快速检查“账户初始余额录入”、“业务单据生成日记账”、“日记账生成凭证”的情况。

![](https://vip.kingdee.com/download/01096769a0f0b4c2439290fb122b6773dd5f.png)

3、原币一致,本位币不一致

一般出现这种情况说明出纳日记账与凭证的外币汇率不一致,需要做期末调汇处理。出纳期末调汇如何处理,请查看:[出纳模块期末如何调汇?](https://vip.kingdee.com/knowledge/633107398965358592?productLineId=35&isKnowledge=2&lang=zh-CN)
