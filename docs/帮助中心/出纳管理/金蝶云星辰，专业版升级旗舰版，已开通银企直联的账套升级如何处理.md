# 金蝶云星辰，专业版升级旗舰版，已开通银企直联的账套升级如何处理

> 产品：金蝶云·星辰 | 分类：出纳管理 | 类型：快速入门, 常见问题
> 作者：鸟人 | 阅读：543 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/506474811346963968)

****

**1、问题描述**

金蝶云星辰，专业版升级旗舰版，已绑定银企直联的账套升级如何处理，是否会影响使用

2、场景介绍

使用升级工具进行专业版到旗舰版的升级，原账套开通了银企直联

3、解决方案

如专业版的账套已开通了银企直联，使用升级工具进行升级后，系统会自动将原有的银企互联账号绑定到新的账套id，用户无需再手工处理，新账套可正常使用，原有账套的银企将不能再使用
