# 怎么删除银行余额调节表内数据?

> 产品：金蝶云·星辰 | 分类：出纳管理 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：274 | 有用：0

【问题描述】

怎么删除银行余额调节表内数据?

【解决方案】

银行余额调节表内的数据是根据录入的日记账及新增或导入的对账单显示的,无法直接删除,如需删除需要删除日记账和对账单。
