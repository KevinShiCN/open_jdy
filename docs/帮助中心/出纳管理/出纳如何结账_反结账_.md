# 出纳如何结账/反结账?

> 产品：金蝶云·星辰 | 分类：出纳管理 | 类型：常见问题
> 作者：云社区用户5j6u671 | 阅读：1939 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/578026198274279424)

【问题描述】

出纳如何结账/反结账?

【解决方案】

1.检查出纳参数设置的结账方式是什么

点击【系统设置】-【系统参数】

![](https://vip.kingdee.com/download/0109aee6969f82d54ab3b2322227c72df191.png)
看【出纳参数】中“与总账一起结账”的参数是否有勾选

![](https://vip.kingdee.com/download/01092fb8e510b1f6404e8f7dcb023341e503.png)

2.未勾选的情况下,出纳单独结账/反结账

点击【出纳管理】-【结账】

![](https://vip.kingdee.com/download/0109696ec6d6a0f84f418899bdbfda96782a.png)
选择要结账的期间,点击【结账】按钮,确定即可结账

![](https://vip.kingdee.com/download/010954fab401f52146d9b0e16eaf2f72b360.gif)
如需反结账则点击【反结账】按钮选择期间操作反结账

![](https://vip.kingdee.com/download/0109c023027553d44a5fb5db33cfce1bcaa5.png)
3.若【出纳参数】中勾选了“与总账一起结账”的参数

![](https://vip.kingdee.com/download/0109c8e500a5c8f840ee9b8f4b846bb0742b.png)
则出纳无法单独结账,需通过【账务处理】-【期末结账】进行结账,财务结账后出纳会跟着一起结,同样反结账也是在【账务处理】-【期末结账】中操作。

![](https://vip.kingdee.com/download/0109d489dab11f004a1bb727d8cd74094361.png)
【温馨提示】

若需修改出纳结账方式,点击【系统设置】-【系统参数】-【出纳参数】,通过勾选或者取消勾选“与总账一起结账”的参数进行修改

![](https://vip.kingdee.com/download/01096e2312daee384901ad6ce4718707085f.png)
需注意,如果原本没有勾选一起结账,出纳是单独结账的,现需要修改为和总账一起结账,则出纳的期间需与总账期间保持一致,否则无法勾选“与总账一起结账”参数

![](https://vip.kingdee.com/download/010992540fb7cef747babac65819e498cc31.png)
