# 出纳日记账如何设置对方科目生成凭证?

【问题描述】

出纳日记账如何设置对方科目生成凭证?

【解释说明】

**新增日记账的时候需要指定好对方科目,以便日记账生成凭证时可以取到对方科目。(需有[科目的权限](https://vip.kingdee.com/link/s/ZAazv))**

因为系统无法自动判断日记账是什么业务产生的,所以需要人工根据实际情况选择好对方科目。

一笔日记账只能有一个对方科目,若需要一笔账对应多个对方科目,需分多笔录入日记账或生成凭证后手工修改凭证。**
**

**举例说明什么是对方科目:**

1、当日记账为收到款项时,账户余额增加,生成凭证借方为现金银行科目,贷方就是被核销的应收业务对应科目,比如收到客户的货款对方科目就应选择应收账款冲抵应收余额;

2、相反支出类型的日记账就是账户余额减少,生成凭证贷方为现金银行科目,要选择一个对方科目作为借方科目,比如支付供应商货款对方科目就选择应付账款科目冲应付余额。

【操作流程】

**方式一、新增日记账时手工选择对方科目**

![](https://vip.kingdee.com/download/01099fecd0a084cf40bfb2f1f7d305ec65e9.png)
对方科目需要选择最明细科目,如果科目启用了辅助核算,则先选择科目,再选择辅助核算明细,如客户、供应商、部门、业务员等。

![](https://vip.kingdee.com/download/0109c6d3260d99874d5dabf570db94e603a4.png)

如果是自定义添加的辅助项目就在日记账上添加自定义字段,再进行选择

[如何新增单据自定义字段?](https://vip.kingdee.com/link/s/lED4y)、[科目怎么设置自定义的辅助核算?](https://vip.kingdee.com/link/s/lEDXW)
