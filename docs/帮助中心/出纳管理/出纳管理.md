# 出纳管理

> 产品：金蝶云·星辰 | 分类：出纳管理 | 类型：快速入门, 整体介绍
> 作者：海鸥乔纳森 | 阅读：6859 | 有用：7
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/635854977625243392)

企业日常经营发生资金收付业务，出纳需将收付款信息登记生成现金银行日记账流水。适用版本专业版和旗舰版，目前标准版不支持。

![](https://vip.kingdee.com/download/01095ab6ea9988b443068779be28c0383ce0.png)

| 步骤 | 具体说明 | 学习链接 |
| --- | --- | --- |
| 1.出纳参数设置 |  | —— |
| 1.1启用期间 | 启用期间：即建账时设置的系统启用期间，刚启用的账套，如果没有录入任何数据，可以修改启用期间，录入数据如果需要再修改，系统会给出对应提示，根据更改到的期间进行业务数据的判断。 | 点击学习 |
| 1.2登账方式 | 登账方式支持2种登账方式：(1)单据登账+手工登录：此方式需同时购买财务云与进销存云，可通过业务单据生成日记账。(2)凭证登账+手工登录：可选择凭证审核/复核/保存时，自动生成日记账，登账方式支持更换。 | 点击学习 |
| 2.出纳基础资料 |  | —— |
| 2.1账户管理 | 企业在采购销售活动中，往往会涉及支付，记录不同的结算账户，可以使得业务结算、查账对账更清晰 | 点击学习 |
| 2.2账户初始数据 | (1)手工录入账户初始数据，引入账户初始数据，科目引入账户初始数据； (2)当出纳与进销存的启用期间不同时，账户初始数据录入规则说明 | 点击学习 |
| 3.登记现金银行流水 |  | —— |
| 3.1手工登账 | 支持手工直接新增日记账，下载引入模板批量引入，支持部分银行流水引入 | 点击学习 |
| 3.2业务单据生成日记账 | 单据登账+手工登账：支持勾选业务单据审核后自动生成日记账 | 点击学习 |
| 3.3凭证生成日记账 | 凭证登账+手工登录，可选择凭证审核/复核/保存时，自动生成日记账 | 点击学习 |
| 3.4银企互联 | 开通银企互联,从网银直接下载银行流水明细，目前支持招商银行、平安银行、浦发银行、民生银行、交通银行，广发银行 | 点击学习 |
| 4.银行对账 |  | —— |
| 4.1银行对账单 | 企业发生的业务资金通过银行进行结转,银行都会客观地逐笔记录下来。企业需要把银行对账单与企业内部出纳根据日常业务登记的日记账在同一个系统里自动进行核对,故系统提供银行对账单的录入和导入功能。 | 点击学习 |
| 4.2银行存款对账 | 每月月末，企业的出纳人员将企业的银行存款日记账与开户银行发来的当月银行存款对账单进行逐笔核对，勾对已达账项，找出未达账项，并编制每月银行存款余额调节表 | 点击学习 |
| 5.出纳报表 |  | —— |
| 5.1银行余额调节表 | 在银行对账单余额与企业账面余额的基础上，各自加上对方已收、本单位未收账项数额，减去对方已付、本单位未付账项数额，以调整双方余额使其一致。 | 点击学习 |
| 5.2出纳余额表 | 可查询所有账户的发生额和余额情况 | 点击学习 |
| 6.出纳结账 | 出纳模块支持与总账一起结账，或是出纳单独结账 | 点击学习 |
