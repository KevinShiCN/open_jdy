# 费用报销,我的待办功能介绍

> 产品：金蝶云·星辰 | 分类：费用报销 | 类型：常见问题

【业务场景】

费用报销,我的待办功能介绍

【名词解释】

我的待办包含以下模块:待我审批、待我支付、待我确认

![](https://vip.kingdee.com/download/0109973f666935b14848a2bab482e1af4a7a.png)

1、待我审批:

此列表是用于展示待我审批的、以及我已审批的单据列表,包含必要的信息如单据事由、提交人、单据金额等。用户可通过查看【待审批】列表,快速了解待办事项,并执行批准或拒绝等操作,同时也可通过查看【已审批】列表,查看过去的审批记录,追溯历史决策。

默认按创建时间排序,可点击字段名称自定义排序(不支持保存,重新打开还是默认排序的顺序)

![](https://vip.kingdee.com/download/01098b72d322a2ae488396ad353e2b63e442.png)

2、待我支付:

此列表是用于展示待我支付的、以及我已支付的单据列表,包含必要的信息如单据事由、提交人、单据金额等。用户可通过查看【待支付】列表,清晰地查看应支付金额、费用事由以及相关的付款详情等信息,同时也可通过查看【已支付】列表,清晰地查看已支付的金额、费用事由等信息。

3、待我确认:

用户可通过查看【待我确认】列表,快速查看待确认的还款或被驳回的还款,同时也可通过查看【已确认】列表,查看过去的已处理记录。其中:
①待【待确认列表】,用于展示以下两类情况的还款单:
a. 待确认收款的还款单。此类单据仅支持可操作确认收款的用户可见。 可操作确认收款的用户设置路径:在【系统设置-全局设置-全局系统参数-单据支付设置-借款单】中设置。
b. 被驳回的还款单。此类单据仅支持还款人可见。还款人可对被驳回的还款单进行编辑、重新提交或删除。
②【已确认列表】,用于展示用户已操作确认收款、或已操作驳回的还款单。
