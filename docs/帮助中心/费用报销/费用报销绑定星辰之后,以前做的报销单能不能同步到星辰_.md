# 费用报销绑定星辰之后,以前做的报销单能不能同步到星辰?

> 产品：金蝶云·星辰 | 分类：费用报销 | 类型：常见问题
> 作者：Schatz | 阅读：191 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/652314277494931456)

【问题描述】

费用报销绑定星辰之后,以前做的报销单能不能同步到星辰?

【解释说明】

在费用报销账套与星辰系统关联之前,已经存在的报销单据不会自动同步至星辰系统。

如下图所示,名为“A公司”的星辰账套目前尚未与云报销系统进行绑定。

![](https://vip.kingdee.com/download/0109519e0b7e30e940f0887843695f5ff3f4.png)

然而,在云报销系统中已经有录入了报销单据。

![](https://vip.kingdee.com/download/0109cd34b55872db40138acf65f31e300051.png)

现在在费用报销账套的系统设置中的承担公司选项里,将“A公司”这个承担公司与星辰账套“A公司”进行绑定。

![](https://vip.kingdee.com/download/01092c4c3ae9c4d34b76ab933683af886daf.png)

完成绑定后,到星辰账套的费用报销-单据同步界面检查,是没有自动同步过来以前的报销单据的。

![](https://vip.kingdee.com/download/01090a61a1f097904d04ad1b0a988f43acfd.png)

如要同步这些单据,在单据同步界面点击右上角的同步管理-同步单据进行手工操作。只有“A公司”这个承担公司下已支付的报销单据可以同步过来。

![](https://vip.kingdee.com/download/0109ddd452501a94423fab8f52d87d86cb26.png)
