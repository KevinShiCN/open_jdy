# 报销单支付后发现附件错了可以修改吗?

> 产品：金蝶云·星辰 | 分类：费用报销 | 类型：常见问题
> 作者：Mosh555555 | 阅读：116 | 有用：1

【问题描述】

报销单支付后发现附件错了可以修改吗?

![](https://vip.kingdee.com/download/0109eccaae599e3d4a6cba899e42a3445b9f.png)

【解释说明】

已支付的单据不可修改,报销单支付详情界面的附件也是不能修改的。可以变通处理:

1.作废这张单据后重新创建新的报销单据,上传正确的附件重新提交审批支付

2.出纳撤销支付,并做支付驳回,修改后重新提交审批支付

(撤销支付是新功能,若账套中没有请提供账套[服务ID](https://vip.kingdee.com/link/s/ZzcJn)联系售后拉灰度处理)
