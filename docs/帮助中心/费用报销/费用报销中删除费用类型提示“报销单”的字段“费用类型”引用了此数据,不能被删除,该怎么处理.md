# 费用报销中删除费用类型提示“报销单”的字段“费用类型”引用了此数据,不能被删除,该怎么处理

> 产品：金蝶云·星辰 | 分类：费用报销 | 类型：常见问题
> 作者：云社区用户55os622 | 阅读：155 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/627671346553602560)

【问题描述】
删除费用类型时提示“报销单”的字段“费用类型”引用了此数据,不能被删除。该怎么处理

【解释说明】

当费用类型已被使用(被报销单等单据引用过),则不允许删除该费用类型。如后续确认不需要再使用该费用类型,可以点击禁用按钮将费用类型禁用。

【操作步骤】

基础资料-费用类型,进入费用类型列表
![](https://vip.kingdee.com/download/0109e8340147091e4eb9886830190507a3b6.png)

选中需要禁用的类型,点击禁用
![](https://vip.kingdee.com/download/0109775064287ed440a1aa6e44424fd7ae23.png)
