# 3.1云报销生成星辰日记账指南

> 产品：金蝶云·星辰 | 分类：费用报销 | 类型：常见问题
> 作者：财税李炳全 | 阅读：1184 | 有用：5

【问题描述】

3.1云报销生成星辰日记账指南

【业务场景】

报销单据同步到星辰后,就可以到 【出纳管理】-【业务单据生成日记账】去生成日记账。只有单据匹配状态正常的才可以去生成日记账。

【操作指引】

菜单栏依次点击【出纳管理】-【业务单据生成日记账】

![](https://vip.kingdee.com/download/0109a10dabb28d7c438d9f7efbaecda4f0ea.png)

生成日记账:单据日期是报销单据的申请日期,审核日期是报销单据支付的日期,所以一般报销单据去生成日记账时,选择审核日期生成。

![](https://vip.kingdee.com/download/01095857533706244aee93733899c0a43599.png)
