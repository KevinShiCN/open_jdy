# 借款单点击还款提示还款中怎么处理?

> 产品：金蝶云·星辰 | 分类：费用报销 | 类型：常见问题

【问题描述】

借款单点击还款提示还款中怎么处理?
![](https://vip.kingdee.com/download/0109b035a96cbbbd4d3885517d2b8105c0c0.png)
【解释说明】

**还款中金额**:指已提交但流程未结束的【还款单或核销单】的还款/冲抵金额,

注:此处流程未结束指,还款单已提交但未确认收款 或 核销的单据已提交但未完成支付。

【操作指引】

出纳账号登录,依次点击【我的待办】-【待我确认】将借款单提交还款进行确认收款
![](https://vip.kingdee.com/download/01098ade5cae6f8449bdb87f369fbf76315a.png)
