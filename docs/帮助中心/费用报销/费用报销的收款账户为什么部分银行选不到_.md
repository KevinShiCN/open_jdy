# 费用报销的收款账户为什么部分银行选不到?

> 产品：金蝶云·星辰 | 分类：费用报销 | 类型：常见问题
> 作者：云社区用户5j6u671 | 阅读：424 | 有用：1

【问题描述】

费用报销的收款账户为什么部分银行选不到?

![](https://vip.kingdee.com/download/0109e4f11d29c8ef4d87b34e6b5111dae5b4.png)

【解释说明】

云报销3.1的收款账户,在选择银行名称的时候,像南京银行,北京银行,长沙银行这类的城市银行是在“城市商业银行”的分类下

所以城市商业银行、农村合作银行、农村商业银行、农村信用联社,在银行名称里面先录入大类,才能在开户银行处选择到具体的银行。

【解决方案】

先在“银行名称”处选择“城市商业银行”,再到“开户银行”处理选择具体的开户银行。
