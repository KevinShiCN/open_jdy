# 报销管理如何操作?

> 产品：金蝶云·星辰 | 分类：费用报销 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：613 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/637094318959958272)

【问题描述】

报销管理如何操作?

【解释说明】

报销管理是用于展示当前用户查看权限范围中的报销单,便于用户快速获取、跟踪、查询组织、部门内的报销单据

【操作指引】

1、菜单栏点击【单据管理】-【报销管理】

![](https://vip.kingdee.com/download/0109d9c5a0df49e549b28d8584a249a2dada.png)

2、单据查询范围与当前用户角色设置单据查看范围有关

①若设置【所有数据】,该角色将允许查看系统内所有报销单据;

②若设置【与我相关】,当前用户为部门负责人时,此用户可以查看自己以及该部门所有职员的报销单,若当前用户为非部门负责人时,此用户在报销管理仅可查看自己提交的单据。

![](https://vip.kingdee.com/download/0109366acab4f58e4f52a7b8798487fde5b9.png)

**注意:自己新增的及别人新增的未提交的报销单不会在【报销管理】体现;可通过【我的单据】-【报销单查询】查看所有的报销单(包括自己未提交的)**。

别人未提交的单据,即使是管理员有查看所有单据权限也看不到。

3、报销单列表界面支持作废单据、打印和引出数据

![](https://vip.kingdee.com/download/0109d06497c26342455b80dd96744d5c85ad.png)

4、右上角可以点击设置按钮进行列设置

![](https://vip.kingdee.com/download/0109a7e5fbf26854407e84dd3a665dbd8c8b.png)

【温馨提示】

单据管理中借款管理和付款申请管理同理!
