# 点击添加职员,为什么提示:根节点不能设置权限用户?

> 产品：金蝶云·星辰 | 分类：费用报销 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：132 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/624772219436286720)

【业务场景】

点击添加职员,为什么提示:根节点不能设置权限用户?

![](https://vip.kingdee.com/download/01098a25f493f020408f81673b8e77229b72.png)

【解释说明】

角色权限界面的添加职员,主要是对已经存在的职员信息添加对应的角色,可以点击到左边对应的角色上,比如【会计】,再点击右上角【添加职员】,授予对应职员【会计】的角色;

如鼠标停留在【全部】,没有选择对应的角色,点击右上角添加职员,就会提示:根节点不能设置权限用户,左侧需要选择到对应的角色即可

【操作指引】

比如添加张三为会计的角色

路径:点击到【会计】角色,点击右上角【添加职员】,选择张三即可

![](https://vip.kingdee.com/download/01098e075935017141a09133801cd6696e5e.png)

添加完成如图显示,即张三拥有会计角色的权限

![](https://vip.kingdee.com/download/0109b5db5c8ba7074a81a8a0e0146cf70068.png)【温馨提示】

如添加新的职员信息,需要在组织架构中添加对应的职员
