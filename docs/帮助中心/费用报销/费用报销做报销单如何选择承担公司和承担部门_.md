# 费用报销做报销单如何选择承担公司和承担部门?

> 作者：一见茹顾 | 阅读：112 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/738200109971681536)

【问题描述】

  费用报销做报销单如何选择承担公司和承担部门?

【解释说明】

  录单时,可以自行选择承担公司,选择好之后,承担部门自动根据承担公司带出。![](https://vip.kingdee.com/download/01099be9a2926e774cbea7f5628f44a2551b.png)
  如果先选择承担部门,承担公司需要再选择录入。![](https://vip.kingdee.com/download/01099e07fd3e042340a0b5600309ffefac5f.png)
如果想要默认带出承担公司和部门,可以参考[费用报销如何设置新增单据默认带出的承担公司?](https://vip.kingdee.com/link/s/ZSyuV)
