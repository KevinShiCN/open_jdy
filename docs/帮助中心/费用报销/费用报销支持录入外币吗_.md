# 费用报销支持录入外币吗?

> 产品：金蝶云·星辰 | 分类：费用报销 | 类型：常见问题

【问题描述】

费用报销支持录入外币吗?

【解释说明】

费用报销暂不支持设置币别,当报销单据同步到星辰中时默认为本位币。如有外币报销需求,建议将金额换算为本位币后再提交单据。
