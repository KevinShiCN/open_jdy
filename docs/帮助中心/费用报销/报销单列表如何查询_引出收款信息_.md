# 报销单列表如何查询/引出收款信息?

> 产品：金蝶云·星辰 | 分类：费用报销 | 类型：常见问题
> 作者：Schatz | 阅读：137 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/649777742405384704)

【问题描述】

报销单列表如何查询/引出收款信息?

【解释说明】

在报销单列表界面,默认展示的是费用明细的分录数据。由于该界面无法一次性显示多种不同分录信息,若需查询单据的收款详情,必须调整查询范围为查询收款信息。

【操作步骤】

**1、查询**

依次点击【我的单据】-【报销单-查询】,打开报销单列表界面,点击左上角“展开过滤”,查询范围选择“基本信息 / 收款信息”即可查询到收款信息相关字段。

![](https://vip.kingdee.com/download/01090b31f895cdf344a4bfd2bcb90a358fcb.png)

如果查询范围选择“基本信息 / 收款信息”还是看不到收款信息相关字段,,请点击右上角的齿轮按钮进入列设置界面,并开启相关字段的显示。

![](https://vip.kingdee.com/download/0109d45dc9c5b86c4290bf81d5d531abbd0e.png)
**2、引出**

查询到收款信息后点击右上角“引出”即可将数据导出为Excel。

![](https://vip.kingdee.com/download/01091b7356065f14483696f49155e7d3f95b.png)

【温馨提示】

1、在单据管理中查询报销管理列表,查询收款信息的操作步骤相同。

2、若未将查询范围设定为“收款信息”,即使开启了字段显示,相关信息也不会被展示。

![](https://vip.kingdee.com/download/0109a608c25534d442d0a46291327b9258c2.png)
