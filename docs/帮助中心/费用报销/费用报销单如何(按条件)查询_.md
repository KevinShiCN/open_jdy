# 费用报销单如何(按条件)查询?

> 产品：金蝶云·星辰 | 分类：费用报销 | 类型：常见问题
> 作者：云社区用户8x80607 | 阅读：365 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/622235915737509120)

【问题描述】

费用报销单如何(按条件)查询?

【操作步骤】

进入费用报销,菜单栏点击【我的单据】>【报销单-查询】

![](https://vip.kingdee.com/download/0109f11d3df695fd46398c2d1a29faf81ae4.png)

在报销单列表界面,可以展开筛选条件,按费用明细范围或者是按收款信息范围进行查询

![](https://vip.kingdee.com/download/0109ce6dd740b91e4ab7a672d6b3550af3aa.png)
左上角搜索栏可以按单据编号、事由、承担公司等搜索,可以点击<列表搜索设置>条件可搜索的条件,最多可启用5个查询条件

![](https://vip.kingdee.com/download/0109ce54ec38a3554bc292f3294719f58e8e.png)
表头默认可以按申请日期、业务类型、单据状态等条件查询

![](https://vip.kingdee.com/download/010929d524b6902247ee9cd37445560078a4.png)
在列字段名称上,鼠标放上若有小漏斗显示,则表示可通过该字段过滤查询

![](https://vip.kingdee.com/download/0109a4b6b5109e7140768f73bb02093ffd17.png)
也可点击<新增方案>

![](https://vip.kingdee.com/download/010944972075bce84f669112da8ef4657cee.png)
添加条件设置查询的条件范围,如按业务类型、单据状态、账户名称等

![](https://vip.kingdee.com/download/0109055f9e4a805a4f1d973c3bbf36414a90.png)

或者是在报销管理界面查看权限范围内的报销单

![](https://vip.kingdee.com/download/010948bfb78b4ff247be9a9d769b3580ab69.png)
