# 单据同步数据如何引出?

> 产品：金蝶云·星辰 | 分类：费用报销 | 类型：常见问题
> 作者：云社区用户8x80607 | 阅读：160 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/625134638859718400)

【问题描述】

单据同步数据如何引出?

【操作步骤】

费用报销-单据同步,系统支持两种引出方式:按列表引出、按引入模板引出![](https://vip.kingdee.com/download/010948a3941e854c4b7b80cb5626128a1cc8.png)

1、按列表引出,效果如下图所示![](https://vip.kingdee.com/download/01096c3b437fe0184f589d04849cb2d983ba.png)
2、按引入模板引出-新增模板,

1)第一次引出需要新增模板。如下图所示
![](https://vip.kingdee.com/download/0109850ca5e6b37346daaff65f5dd5075908.png)
2)根据业务需要,把需要的字段【是否允许引入引出】打开显示,可选择引出属性,名称/编码。![](https://vip.kingdee.com/download/01091a61cc673866429c859623fb615c2c2e.png)

3)选择完成,维护模板编码,名称。可选择是否为模板。如下图所示,设置完成,点击保存
![](https://vip.kingdee.com/download/010904b749bd92c64aaf81e60722ef0cbe86.png)
4)引出-【按引入模板引出】-选择模板-确定![](https://vip.kingdee.com/download/010918ce0437370f40129d33d809a15ecfd0.png)
5)引出效果如下图所示![](https://vip.kingdee.com/download/010991a1ddfa6a4b4d1dac47874d7590cdc1.png)
