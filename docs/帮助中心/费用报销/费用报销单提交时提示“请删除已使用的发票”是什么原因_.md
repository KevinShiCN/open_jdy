# 费用报销单提交时提示“请删除已使用的发票”是什么原因?

> 产品：金蝶云·星辰 | 分类：费用报销 | 类型：常见问题
> 作者：云社区用户0531183 | 阅读：83 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/703054806633671680)

【问题描述】

费用报销单提交时提示“请删除已使用的发票”是什么原因?

【解释说明】

费用报销系统中开启了发票重复校验,上传过的发票不能重复上传。

【解决方案】

1.打开【系统设置】-【全局设置】

![](https://vip.kingdee.com/download/010921dc2f712cf84ace8646c5c965e6efef.png)

2. 检查发票设置是否开启了发票校验,并且勾选了发票重复校验。启用该选项后,上传发票时,会根据发票号码、发票代码等校验重复发票,删除重复发票才可进行提交。![](https://vip.kingdee.com/download/0109cc9da74818d84cd9ab65c74c78410092.png)

3. 如允许重复上传发票可以关闭该参数重新提交(不建议关闭参数允许重复上传,以免出现重复报销的情况),或者删除重复发票才可进行提交。

![](https://vip.kingdee.com/download/01092420ca4df07f41ad9eeeb28ceba9b985.png)
