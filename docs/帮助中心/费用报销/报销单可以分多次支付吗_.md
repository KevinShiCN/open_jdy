# 报销单可以分多次支付吗?

> 产品：金蝶云·星辰 | 分类：费用报销 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：276 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/655575717299660800)

【问题描述】

报销单可以分多次支付吗?

【解释说明】

**报销单不支持分多次支付。**

比如金额是200,支付的时候付款金额录100,系统会提示:付款总金额需要等于支付金额。

![](https://vip.kingdee.com/download/01093502d07d03cf4d688439a9da6a074176.png)
【温馨提示】

如果是多账户付款,在支付界面付款账户点击前面的“+”号,添加账户付款即可。

![](https://vip.kingdee.com/download/0109f7c194a5b20843f692ceaa1198d0b8cb.png)
