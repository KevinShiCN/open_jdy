# 费用报销整体介绍

1 产品概述

### 1.1 产品介绍

金蝶云星辰费用报销是通过星辰集成云报销形成的星辰费用报销整体解决方案。为小微企业提供高效，便捷，安全的数据入账，让费用更好的统计，更快的入账，更多的节流，构建起小微企业的业财税费一体的数字化管理体系。云报销让费用更清晰。

### 1.2 适用版本

金蝶云·星辰专业版、金蝶云·星辰旗舰版

### 1.3 业务场景

1）员工报销；
2）外部付款；
3）员工借还款；

4）费用报销单一键生成凭证入账；

5）费用报销单一键生成日记账；

6）应收应付资金报表统计费用报销单；
