# 云报销子账号看不见承担公司如何处理?

> 产品：金蝶云·星辰 | 分类：费用报销 | 类型：常见问题
> 作者：云社区用户6Rw2283 | 阅读：15 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/794370653167118848)

【问题描述】

云报销子账号看不见承担公司如何处理?

【操作步骤】

1、菜单栏点击【系统设置】>【组织架构】,

![](https://vip.kingdee.com/download/0109794122a4713646caa2eac40265c729f2.png)

检查承担公司可用范围,是否设置部分可用。部分可用可按照部门与职员设置,检查该子账号是否在该部门下/在所选职员下。
![](https://vip.kingdee.com/download/01096bfdc8315a81495d82cf294e2223296d.png)
【系统设置】-【组织架构】可查看子账号所属部门,以及状态是否为启用

![](https://vip.kingdee.com/download/0109be6bd7c184df4258909f8a5698e49ea9.png)
2、已排查步骤一,【系统设置】-【组织架构】,查看该子账号角色

![](https://vip.kingdee.com/download/0109dd6ccdc78b1d4f9bbaf0262ee349108d.png)
【系统设置】-【角色权限】,查看对应角色权限,是否有查看承担公司功能 ![](https://vip.kingdee.com/download/0109784314e08d664876a1e6b42d5b3e2a62.png)
