# 费用报销的账户类型如何设置?

> 产品：金蝶云·星辰 | 分类：费用报销 | 类型：常见问题
> 作者：shaoling | 阅读：16 | 有用：1

【问题描述】

费用报销的账户类型如何设置?

【解释说明】

**账户类型**指的是根据账户属性对账户进行的分类。

费用报销新增收款、付款的账户类型为系统默认设置,暂时**无法自定义新增和修改**,目前支持的账户类型有银行账户、微信、支付宝、现金账户、其他账户。
