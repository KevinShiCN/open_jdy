# 费用报销升级到新版后,星辰里的[费用报销]-[报销设置]绑定的服务编码是旧费用报销的服务编码,这是否影响新的费用报销的使用?

> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/660649068598768896)

【问题描述】

费用报销升级到新版后,星辰里的【费用报销】-【报销设置】绑定的服务编码是旧费用报销的服务编码,这是否影响新的费用报销的使用?

![](https://vip.kingdee.com/download/0109fb3c51fc0f4b442284b63a6651f2b1cc.png)

![](https://vip.kingdee.com/download/01092cb9b89c08594e59aefba8ec6b4f4f5a.png)

【解释说明】

不影响新版的费用报销的使用,旧版到时候到期后,新版还是一样的使用,之类的服务编码不影响。绑定新的不依赖旧的解绑。
