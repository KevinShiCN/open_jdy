# 潜在客户如何使用?

> 产品：金蝶云·星辰 | 分类：客户经营 | 类型：常见问题
> 作者：ylim | 阅读：339 | 有用：0

【问题描述】

潜在客户如何使用?

【业务场景】

销售员进行陌拜时,可将拓展的潜在客户录入星辰系统,并完成后续跟进及转化(PC及移动端均支持);针对众多网络平台上的客户资源,老板/销售部负责人可批量导入第三方潜客数据,并完成潜客的资源分配及销售员跟进管控。

**提升销售员效率:**潜客管理中一方面销售员可根据实际情况录入潜客信息,另一方面老板/销售部负责人可手动进行潜客分配,以此提升销售员的效率;
**降低客户流失:**对企业而言,客户资料是重要的资源。通过手工录入或批量导入第三方数据的方式,将原来多渠道的客户资料及跟进记录统一规整,通通留存在企业内,降低了由于员工离职带来的客户流失;
**降低重复工作:**销售员在录入潜客信息前,可提前进行查重,避免撞单产生无意义的重复工作;
**保障资源归属:**潜客管理中不同销售员间无法查看彼此的客户数据,不仅保障了销售员的资源归属,也提升了企业客户资料的安全性。

【操作指引】

点击【客户经营】-【潜在客户】

![](https://vip.kingdee.com/download/0109f3dc9bb6b475403d87292b9751216aa8.png)
可以看到潜在客户列表分为三部分:潜客列表、潜客公海、回收站;一般还未分配的潜客资源放在潜客公海中,分配或者领取后员工在潜客列表中查看自己的潜客信息,对于已流失的潜客不想显示出来可以删除,删除后在回收站中显示,已到回收站的潜客可以恢复到潜客公海,也可以删除(在回收站里删除即是彻底删除)

![](https://vip.kingdee.com/download/01090aa081e0b12b4cc6a5e8fe2cd8cb095f.png)
潜客列表中支持对潜客进行分配、移入公海、转为正式客户、写跟进记录、添加报价单、批量修改、删除、新增、引入、引出、查重

![](https://vip.kingdee.com/download/01092f9e25878e164331b16284ddb373048a.png)
潜客公海中支持对潜客进行分配、转为正式客户、批量修改、领取、删除、新增、引入、引出、查重

![](https://vip.kingdee.com/download/0109f99df6078b414f27a6f07975810167ba.png)
回收站中支持对已删除的潜客进行恢复、删除、引入、引出、查重,支持新增潜客
