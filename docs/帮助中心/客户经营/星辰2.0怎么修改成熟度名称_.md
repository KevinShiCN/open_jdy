# 星辰2.0怎么修改成熟度名称?

> 产品：金蝶云·星辰 | 分类：客户经营 | 类型：常见问题
> 作者：杨桂平 | 阅读：165 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352781296668160)

【问题描述】

怎么修改成熟度名称?

【解决方案】

1、点击【客户经营】-【跟进记录】-成熟度列表

![](https://vip.kingdee.com/download/0109962a49f63f0a466a94c84781ea0e8013.png)

2、点击要修改的成熟度名称,进入名称编辑界面,修改完成后保存

![](https://vip.kingdee.com/download/01097d9745741b804c3ba989ae49c1fc55b4.png)

注意:允许多个成熟度使用一样的名称,并同时启用。
