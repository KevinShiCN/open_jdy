# 跟进任务

> 产品：金蝶云·星辰 | 分类：客户经营 | 类型：快速入门, 功能介绍, 新特性
> 作者：Freya_peng | 阅读：2809 | 有用：7
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/354931154522340608)

# **1 应用场景**

- 销售员可制定周期性计划（按规划拜访路线、或批量选择客户建立自己一天/周期性的拜访），定期执行后在系统内填写跟进记录进行存档
- 计划预警能及时提醒销售员完成跟进任务，保障每一个客户都能及时跟进，防止客户流失
- 销售员通过定期巡店，可维护下游经销商或代理商

# **2 功能特性**

- 允许一个客户同一天建立多个任务，支持按线路和客户建立任务。APP和PC端均可编辑
- 选中客户建立任务保存后，按客户数生成任务条数 。例如：建任务时选择了3个客户，则会生成3条任务
- 跟进任务状态为新建时，联系人和业务员、计划跟进内容仍可修改，其他字段不可编辑
- PC端支持删除状态为新建的跟进任务

# **3 应用流程**

## 3.1 移动端

步骤一：新建跟进任务

客户管理>跟进任务，或访销>巡店任务中点击新建按钮，新建跟进（拜访/联系）任务

![](https://vip.kingdee.com/download/01000966d59fa88147ec961fa222895c7f43.jpeg)

步骤2：设置跟进任务

1）输入计划跟进时间，如有周期设置，可编辑周期重复逻辑

2）选择客户。可从线路选择带出线路上的客户（支持选择该用户权限内可见线路），也可从客户列表批量选择客户），选择完成后保存

![](https://vip.kingdee.com/download/01095b8adaa4bb514ee981e6313ed4a35622.png)

步骤三：查看任务

![](https://vip.kingdee.com/download/0109ef5b27f222204319ae1db194cd642074.png)
