# 客户分析表

> 产品：金蝶云·星辰 | 分类：客户经营 | 类型：功能介绍, 精华汇总, 快速入门
> 作者：进销存赵嘉丽 | 阅读：2499 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/169922371741666560)

## 1 简介

### 1.1功能介绍

通过客户分析表，可快速了解客户的业务往来情况，包括交易的金额、首日和最近交易的日期、累计成交单数、成交金额、欠款金额等等，以便及时调整对应的营销策略，如根据客户的成交金额量、回款及时性来评估客户的交易潜力、信用情况，进而有针对性地展开客户管理工作，最终实现经营效益的提升。

### 1.2适用版本

星辰所有版本

### 1.3系统路径

客户经营>客户分析>客户分析表

### 1.4功能特性

- 可通过指定新增客户天数、无交易客户天数、未跟进客户天数来查询对应的客户数据
- 可查询首次/上次/交易日期、交易金额、未交易天数、累计成交单数、成交金额、欠款金额等等

## 2 应用流程

1）在系统参数>消息和预警>客户预警中，勾选设置客户未联系/未拜访天数、客户流失预警、客户生日关怀，保存。

| 参数 | 使用说明 |
| --- | --- |
| 客户未联系预警 | 未联系表示“未打电话或微信等方式，未在系统记录联系内容“，超出设置范围后会加入客户逾期未联系风险预警名单 |
| 客户未拜访预警 | 未拜访表示“未线下拜访、打卡签到记录拜访内容“，超出设置范围后会加入客户逾期未拜访风险预警名单 |
| 客户流失预警 | 按期限内是否复购预警：设置最近X天未复购，将会根据设置的天数期限内，查询是否有已审核的销售出库单/委托代销结算单/零售结存单（含退货），若期限内无相关下单记录即将预警提醒 |
| 客户生日关怀 | 取客户资料中联系人生日，若未维护生日可前往“基础资料/客户”维护。 |

2）设置好后，在销售管理>经营分析>经营日报中“客户情况”卡片中查看“流失预警”和欠款客户数，点击可以进入客户分析表

3）在客户分析表中，可以看到每个客户的所有交易汇总情况，也可以在“筛选”中自定义查询条件
