# 客户经营

> 产品：金蝶云·星辰 | 分类：客户经营 | 类型：快速入门, 整体介绍, 精华汇总
> 作者：_Leo | 阅读：3038 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/627211645567253504)

客户经营是企业CRM系统重要组成部分，销售员陌拜时，可将潜在客户录入系统并完成后续跟进、转化；负责人可批量导入第三方潜客数据，并完成潜客的资源分配及销售员跟进管控。帮助企业全面提升客户关系管理水平，优化业务流程，实现业务增长，提高市场竞争力。

业务流程如下：

| 1.客户经营参数 | 可以对客户经营管理的业务处理规则和处理惯例进行统一的设置和管理。分为客户参数、外勤参数 | —— |

| 2.2潜在客户 | 销售员拜访陌生客户时，可将潜在客户录入星辰系统，并支持将潜客转为正式客户 | 点击学习 |
| 2.3正式客户 | 即[基础资料]中的正式[客户]资料 | 点击学习 |
| 3.客户跟进 |  | —— |
| 3.1线路管理 | 通过线路来分配客户,并指定负责人来维护客情关系,以便后续外勤人员按线路拜访客户 | 点击学习 |
| 3.2跟进任务 | 维护客户关系时，建立线上联系和线下拜访的跟进任务帮助业务员追踪后续的行动计划和承诺，确保及时回复客户的需求和问题 | 点击学习 |
| 3.3跟进记录 | 业务员跟进任务后，填写客户跟进情况，以便后续进一步管理客情关系 | 点击学习 |
| 3.4本品上报 | 业务员拜访客户或下游经销渠道时,可到店后将该店铺中我司商品的数量信息进行上报,便于销售经理记录门店本品供货现状 | 点击学习 |
| 3.5竞品上报 | 业务员拜访客户或下游经销渠道时,可到店后将该店铺中竞争对手的商品数量信息进行上报,以便销售经理记录门店竞品供货现状 | 点击学习 |
| 4.客户分析 |  | —— |

| 4.3跟进汇总表 | 跟进汇总表用来查询潜客及正式客户的跟进汇总记录   | 点击学习 |
