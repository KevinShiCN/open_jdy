# 跟进任务的周期设置如何使用?

> 产品：金蝶云·星辰 | 分类：客户经营 | 类型：常见问题
> 作者：云社区用户8x80607 | 阅读：208 | 有用：0

【问题描述】

跟进任务的周期设置如何使用?

【操作步骤】

客户经营-跟进任务,点击新增。跟进任务周期设置支持两种:按天设置、按周设置![](https://vip.kingdee.com/download/0109ccb43c92f26c4550b259f580a5e6c0c0.png)
1、**周期选择按天设置**,维护:跟进间隔天数、跟进跳过设置、周期截止时间(*号为必填项),注意:周期设置只能限定在6个月内,设置完成,点击【保存】![](https://vip.kingdee.com/download/01091b665e1ce6e2452488f7d1bb36fdf255.png)
2、此时跟进任务,会根据设置的周期及起始日期自动生成对应的跟进计划。效果如下图所示

![](https://vip.kingdee.com/download/0109ed2b2be415c64adf964a6efa63d31308.png)
3、**周期选择按周设置**,维护:跟进间隔天数、跟进跳过设置、周期截止时间(*号为必填项),注意:周期设置只能限定在6个月内,设置完成,点击【保存】![](https://vip.kingdee.com/download/0109ef83fa5271494ff2bc5b33c81e42faa5.png)
4、此时跟进任务,会根据设置的周期及起始日期自动生成对应的跟进计划。效果如下图所示
![](https://vip.kingdee.com/download/010974a587ecbb914db1a3bee1ca62f04ad7.png)
