# 销售漏斗分析表

> 产品：金蝶云·星辰 | 分类：客户经营 | 类型：快速入门, 精华汇总
> 作者：卷不动的二可 | 阅读：406 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/647833613848690432)

# **1 业务场景**

业务员可以根据潜客的不同阶段查看和跟进潜客的转化情况

# **2 解决方案**

针对上述场景，可以通过星辰的销售漏斗表来实现

# **3 适用版本**

星辰3.0以上所有版本

# **4 应用流程**

## 4.1 操作路径

移动端：客户管理>销售漏斗分析

电脑端：客户经营>客户分析>销售漏斗分析表

## 4.2 使用说明

- 销售漏斗分析表，如果不筛选条件，默认显示全部潜客
- 左侧表格显示潜客、当前业务员、销售阶段、创建时间。目前这四个字段不支持隐藏或改变顺序，也不支持新增其他字段。右侧显示漏斗形状图表。从上到下依次显示初步洽谈、需求确定、成交、流失商机四个销售阶段
- 鼠标放到某个阶段上，会显示处于这个销售阶段的潜客数量和占总潜客数的比例。漏斗形图表不支持隐藏或修改几个销售阶段的上下顺序
