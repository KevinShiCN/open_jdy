# 移动端客户分布地图

> 产品：金蝶云·星辰 | 分类：客户经营 | 类型：功能介绍, 新特性, 快速入门
> 作者：Freya_peng | 阅读：5003 | 有用：12
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/446267469242462208)

# 1 应用场景

当公司希望业务员打开客户列表就可以显示或搜索附近客户，方便进行拜访或是开单

# 2 适用版本

星辰标准版/星辰专业版&旗舰版（专业版&旗舰版按客户分层分布）

# 3 功能介绍

标准版：按正式客户和潜客的维度展示的客户和潜客在地图上分布的情况

专业版/旗舰版：按各个客户分层（五层模式：核心客户、重要客户、一般客户、休眠客户、潜在客户）展示的各类客户和潜客在地图上分布的情况

## 3.1 操作路径

打开金蝶云APP：应用>客户管理>客户分布地图

## 3.2 功能介绍

（1）支持筛选附近距离：500M、1km、5km、30km、不限（默认筛选附件5km的客户，如果模糊查询客户，则距离自动换成不限）

（2）支持的其他筛选条件：客户类别、部门、业务员、客户标签（专业版/旗舰版还支持按客户分层筛选）

（3）字段说明

- 客户总数：该账号用户权限内可见的客户总数
- 新增客户数：该账号用户权限内新增的客户数
- 潜客总数：该账号用户权限内可见的潜客总数

备注：客户总数、新增客户数、潜客总数不随筛选条件而变化

 ![](https://vip.kingdee.com/download/0100314b4fa72c3249228ad59f60984f25ed.png)

附近客户：该账号用户权限内可见的附近距离筛选条件内的客户总数

附近潜客：该账号用户权限内可见的附近距离筛选条件内的潜客总数

备注：附近客户、附近潜客随附近筛选条件而变化
