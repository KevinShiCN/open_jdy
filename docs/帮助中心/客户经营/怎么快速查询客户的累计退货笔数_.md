# 怎么快速查询客户的累计退货笔数?

【问题描述】
怎么快速查询客户的累计退货笔数?

【解释说明】

依次点击【客户经营】-【客户分析表 】,进入客户分析界面![](https://vip.kingdee.com/download/01099c6faae0bf2a4421a2c275ca956e63f7.png)

找到【累计退货单数】列,即可实现快速查询不同客户的累计退货笔数![](https://vip.kingdee.com/download/0109b8918b174a7e4c279c8a08664059fb2d.png)
