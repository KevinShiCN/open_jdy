# 跟进汇总表如何使用?

【问题描述】

跟进汇总表如何使用?

![](https://vip.kingdee.com/download/01091a899c52776f4722b5172b2d3a4a62be.png)

【解释说明】

跟进汇总表用来查询潜客及正式客户的跟进汇总记录

汇总依据包括“业务员+跟进类型”、“客户+跟进类型”、“潜客+跟进类型”

![](https://vip.kingdee.com/download/01097fc50752928e45c881ebe7d76e8bbb42.png)
汇总依据同,查询条件有所不同,汇总依据、跟进日期、跟进类型、跟进状态是固定支持的条件

![](https://vip.kingdee.com/download/0109c88de836902b44b69b1bad5f7a38279b.png)
支持引出、列显示字段设置

![](https://vip.kingdee.com/download/010959223014e9ee4eec8c5c76bdd690899e.png)
