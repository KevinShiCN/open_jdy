# 客户分层管理中逾期跟进天数,取数来源?

【问题描述】

客户分层管理中逾期跟进天数,取数来源?![](https://vip.kingdee.com/download/01098c9ed18ae3614186b6b4229a7e7030ed.png)
【解释说明】

取APP创建的所有未完成的跟进任务(拜访任务/联系任务)中**下次拜访时间和下次联系时间中的最早的时间距离现在的天数**(包含历史的下次时间,但不包含查询当天)
