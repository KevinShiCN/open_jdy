# 跟进日报

> 产品：金蝶云·星辰 | 分类：客户经营 | 类型：快速入门, 功能介绍
> 作者：移动生产白建凡 | 阅读：2828 | 有用：7
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/240478693750289664)

# **1 应用场景**

业务员外勤拜访作业，主管可通过跟进日报查看业务员拜访情况，实时了解业务员跟进客户的情况，以此达到管控的效果，提高开单量

# **2 操作路径**

移动端客户管理>跟进日报

# **3 功能介绍**

对业务员：可以查看自己不同日期的跟进数据

对主管：可查看部门下所有业务员的跟进记录
