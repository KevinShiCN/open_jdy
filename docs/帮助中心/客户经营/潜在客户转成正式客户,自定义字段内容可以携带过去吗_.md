# 潜在客户转成正式客户,自定义字段内容可以携带过去吗?

> 作者：Schatz | 阅读：70 | 有用：0

【问题描述】

潜在客户转成正式客户,自定义字段内容可以携带过去吗?

【解释说明】

在为潜在客户新增自定义字段时,若将该字段的适用单据范围设置为包括“客户”,

![](https://vip.kingdee.com/download/01096d2d2912765f44eebfd5475dc264e558.png)

当潜在客户转化为正式客户后,这些自定义字段及其数据将被自动携带至正式客户的记录中。

![](https://vip.kingdee.com/download/01093b2ad76eb67b4944bae660a104c032c2.png)
