# 跟进任务保存提示:联系人地址-省、地址-市为空,请添加

> 作者：云社区用户8x80607 | 阅读：145 | 有用：0

【问题描述】

跟进任务保存提示:联系人地址-省、地址-市为空,请添加

【提示原因】

因为任务的跟进类型为客户拜访,客户拜访必须录入客户详细地址,能提供导航。

![](https://vip.kingdee.com/download/01097c4253bda3ee496782355759a1d4c35e.png)【解决方案】

1、基础资料-客户信息,首要联系人地址:行政区划(国家/省/市/区)及详细地址维护完成,再去添加客户拜访任务

![](https://vip.kingdee.com/download/010924616c0c7db54840a2d13c3fe42aacca.png)
2、维护完成,重新添加拜访任务效果如下
