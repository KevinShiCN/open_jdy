# 跟进记录和跟进任务有什么区别?

> 产品：金蝶云·星辰 | 分类：客户经营 | 类型：常见问题

【问题描述】

跟进记录和跟进任务有什么区别?

【解释说明】

**跟进记录:**销售员跟进拜访后形成记录,方便销售经理查看员工的外出拜访情况以及各门店的现状。

**跟进任务**:销售员可制定周期性计划(按规划拜访路线、或批量选择客户建立自己一天/周期性的拜访),定期执行后在系统内填写跟进记录进行存档,计划预警能及时提醒销售员完成跟进任务,保障每一个客户都能及时跟进,防止客户流失

**两者的关系** 在实际业务流程中,两者常常形成闭环:

首先,创建一个**跟进任务**,计划在某个时间点联系客户。

任务执行后,填写**跟进记录**,汇报本次联系的具体情况和成果

根据本次跟进的结果,可能会再创建新的**跟进任务**,安排下一步计划
