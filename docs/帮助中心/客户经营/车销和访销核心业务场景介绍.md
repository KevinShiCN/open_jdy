# 车销和访销核心业务场景介绍

**一、车销**

车销是带着货跑客户，业务员和司机开车带着货，到客户现场进行现场销售打印收款的销售方式。车销优势是因为车上随时备货， 客户不用等货，退换货也方便。

** **

比如快销品行业中，车销业务是常见的一种业务方式。客户距离公司较远，相对分散。

** **

**车销功能可以解决****6****大难题：**

①提货难，业务员需要手工提交次日要的货品；

②开单难，手写开单效率低，动作慢，易出错；

③查库难，车上货品多，库存统计不正确，经常对不上账；

④还货难，手抄还货清单，效率低，易出错；

⑤交账难，交款交账与财务对账时间长，效率低，账实不符时责任互相推诿；

⑥决策难，不能随时掌握哪些客户和商品挣钱或赔钱，不利于次日销售决策。



**核心业务场景：批发企业的车销**

销售员可依据客户销售订单快速提货装车。车销过程中，可现场销售、出库、打印、收款、订单上报等业务处理。

**二、访销**

**访销巡店**：挖掘高价值客户，提升销售额。功能亮点：

- 销售员通过定期巡店访销，可维护下游经销商或代理商；
- 访销过程中，可上报订单、产品销量、库存情况，可进行现场销售出库等
