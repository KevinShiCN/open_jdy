# 订货商城-客户基础资料里设置了联系人地址，但是客户下单的时候带不出对应联系人地址，需要手动新增，如何解决？

## 1 问题描述

订货商城-客户基础资料里设置了联系人地址，但是客户下单的时候带不出对应联系人地址，需要手动新增，如何解决？

![](https://vip.kingdee.com/download/0109668eba15d6af4a6e90f45bf147695c7b.png)

![](https://vip.kingdee.com/download/01093a22ba0db5834c17a31dc57791f74f02.png)

## 2 适用版本

金蝶云·星辰v3.1标准版

## 3 原因解析

客户首次登录订货商城小程序会获取首要联系人地址，之后若客户自行新增收货地址，则不会再读取首要联系人的地址，也不会反写到对应的客户资料当中，同时要确保客户资料下的行政区划符合系统格式。
