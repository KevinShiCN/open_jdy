# 潜在客户如何删除和找回?

> 产品：金蝶云·星辰 | 分类：客户经营 | 类型：常见问题
> 作者：Schatz | 阅读：156 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/634198505761644544)

【问题描述】

潜在客户如何删除和找回?

【解释说明】

在“潜客列表”和“潜客公海”列表选中潜客,点击删除,填写删除理由即可删除。

潜在客户没有产生交易,可随时删除,不影响业务数据,删除之后可重新添加。

![](https://vip.kingdee.com/download/01093cb2f2fbbdc84408ba7ad2cefa1f1cdf.png)

删除的潜客将被移至回收站,后续可在回收站界面进行恢复。如以下动图所示:

![](https://vip.kingdee.com/download/0109d9e77ff1124b4ccc8f6c8b2aaf0a9e5f.gif)

若在回收站中删除了潜在客户就是彻底删除了潜额资料,后续需手动重新添加潜客信息

![](https://vip.kingdee.com/download/010954faa25d3b5c439e96f0a50e059662c3.png)

【注意事项】

在回收站中进行恢复操作时,被删除的潜在客户只能恢复到“潜客公海”,需要重新分配到潜在客户列表

![](https://vip.kingdee.com/download/010903b4ea1079c143eaa77141ab149a412d.png)
![](https://vip.kingdee.com/download/0109cb67c7605c454109a37cf76dc4c8b497.png)
