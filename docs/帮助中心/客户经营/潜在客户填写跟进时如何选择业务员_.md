# 潜在客户填写跟进时如何选择业务员?

> 产品：金蝶云·星辰 | 分类：客户经营 | 类型：常见问题
> 作者：shaoling | 阅读：98 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/725889609846195456)

【问题描述】

潜在客户填写跟进时如何选择业务员?

【解释说明】

潜在客户跟进记录填写时,系统会自动获取登录用户关联的职员信息并填入业务员栏位。若未获取到,或需更换跟进业务员,可点击该栏位重新选择。

业务员不支持多选,若有多个业务员共同跟进,建议选择其中一人作为操作人。
