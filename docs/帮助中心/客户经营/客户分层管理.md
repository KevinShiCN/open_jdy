# 客户分层管理

> 产品：金蝶云·星辰 | 分类：客户经营 | 类型：功能介绍, 快速入门, 精华汇总
> 作者：毛豆豆_x | 阅读：508 | 有用：0

# **1 简介  **

## **1.1 功能介绍**

根据客户的价值、行为及需求等维度，将客户划分为不同层级，便于快速定位客户，优化资源配置。支持设置分层规则执行周期、自定义分层规则等。

## **1.2 适用版本**

金蝶云·星辰专业版、旗舰版

## **1.3 系统路径**

客户经营>客户管理>客户分层管理

# **2 操作指南**

## 2.1 对客户设置分层规则

进入客户分层管理，点击页面右上角<设置分层规则>进入配置页面

- 支持按月/按周设置分层规则执行周期，设置完成后每月/每周系统将在设定的日期当04:00-06:00自动执行分层规则
- 详细分层规则设置：近期销售金额时间，销售金额或占比排名

![](https://vip.kingdee.com/download/0109109fb390955f46e8969ea22eedf4a100.png)

功能操作说明：

| 操作按钮 | 功能描述 |
| --- | --- |
| 保存 | 仅保存当前配置的分层规则，不触发执行（需等待自动执行时间或手动触发） |
| 保存并执行 | 保存当前规则配置后，立即触发分层计算，无需等待自动执行周期 |
| 自动执行分层规则 | 保存当前规则配置，系统将按设定的周期时间（每月 / 每周指定日期 04:00-06:00）自动执行 |

***特别说明：**规则执行后，若系统检测到客户分层结果有变动，将在<客户分层管理>列表左侧弹出确认提示。**需手动确认后，才会正式生效客户分层的实际变动**（具体操作可参考下文 “2.2 客户分层变动” 相关说明）

## 2.2 客户分层变动

支持按设置的分层规则将变动的客户转入最新所属分层或维持现有分层

![](https://vip-admin.kingdee.com/download/01099fb7c0334fef414595159e8222fb759a.png)

## 2.3 客户分层列表管理

- 搜索：通过客户编码/客户名称/联系人进行模糊搜索或通过设置过滤条件搜索
- 修改分层：选择对应客户修改分层
- 写跟进：选择对应客户填写跟进记录
- 数据查看：查看分层规则下各层级客户相关数据可点击“上次销售额”数据联查销售出库单查看销售出库详情
- 可点击“近30天销售额”、“本月销售额”、“上月销售额”数据联查销售汇总表查看详情
