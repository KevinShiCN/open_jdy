# 哪里可以看连带率?

【问题描述】

哪里可以看连带率?

【解释说明】

**目前星辰系统中没有连带率相关的分析表可以查看**

连带率是销售过程中一个非常重要的判断依据,它是店铺销售过程中的一个重要指标,反映了顾客平均单次消费的产品件数。具体来说,连带率是销售总件数除以销售总笔数的比值,也可以理解为平均一笔单成交的销售数量。连带率越高,店铺的客单价也就越高,它是影响店铺客单价的重要组成部分。

连带率的计算公式为:连带率 = 销售总数量 / 销售笔数。例如,如果一个店铺当日成交了100笔订单,订单数量是150件,那么连带率就是150/100 = 1.5。

连带率的高低可以反映店铺的销售效率和顾客的购买行为。如果连带率低于1.3,则可能意味着店铺的连带销售存在问题,需要关注和改进。
