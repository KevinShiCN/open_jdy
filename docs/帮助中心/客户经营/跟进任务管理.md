# 跟进任务管理

# 1 功能介绍

跟进任务，适用于维护客户关系时，建立线上联系和线下拜访的跟进任务，帮助业务员追踪后续的行动计划和承诺，确保及时回复客户的需求和问题。从而增加销售机会，提升客户满意度，加强销售团队的协作和管理

# 2 应用场景

**场景一：销售拜访后的跟进**

当业务员与客户进行销售拜访后，需要及时记录拜访内容和讨论结果。建立跟进任务可以帮助业务员追踪后续的行动计划和承诺，确保及时回复客户的需求和问题。

**场景二：客户问题解决**

客户可能在拜访过程中提出一些问题或需求，需要业务员及时解决。建立跟进任务可以确保业务员及时跟进并与相关团队合作解决问题，增强客户满意度。

**场景三：订单跟踪**

当业务员成功获取客户订单时，建立跟进任务可以帮助业务员跟踪订单的执行和交付情况。

**场景四：跟进市场活动和促销**

如果业务员参与了一些市场活动或促销活动，建立跟进任务可以帮助他们记录活动结果和客户反馈，为后续的市场策略和销售决策提供参考。

**场景五：客户关怀和保持联系**

建立跟进任务可以帮助业务员定期与客户保持联系，发送问候或关怀信息，增进客户关系，维护客户忠诚度。

**场景六：协作**

通过建立跟进任务，业务员可以与销售团队内部分享拜访和跟进情况，促进信息共享和团队合作。

# 3 操作步骤

## 3.1 系统路径

路径一：APP客户管理>跟进任务（线上联系和线下拜访任务）

路径二：APP访销>巡店任务（仅线下拜访任务）

路径三：APP客户列表>右上角…（线上联系和线下拜访任务）

路径四：APP客户详情页>跟进任务（线上联系和线下拜访任务）

路径五：电脑端客户经营>跟进任务

1） 新增跟进任务，可以建立线上联系任务或线下拜访任务，也可以通过线路和客户列表批量选择客户建立任务，如图

![](https://vip.kingdee.com/download/0109a868afbdb6b84bd6be65e71f473b05c5.png)

2）  选择客户。可以从线路选或客户列表选，支持多选客户批量建立任务

![](https://vip.kingdee.com/download/010982c4ab5f476b4893ba518d63becce775.png)

3）任务保存后可查看每日任务。首页待办事项也会提醒当日需跟进的任务，支持将未来的任务转入今日跟进

![](https://vip.kingdee.com/download/01094e4bc3299df64fa0b9cc1b00fea910be.png)

     备注：权限将会在3.0版本改为仅受数据授权控制
