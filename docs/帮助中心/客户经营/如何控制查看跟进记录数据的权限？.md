# 如何控制查看跟进记录数据的权限？

> 产品：金蝶云·星辰 | 分类：客户经营 | 类型：常见问题, 进阶技巧
> 作者：移动生产白建凡 | 阅读：1143 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/358646777555191040)

### 问题描述

如何控制查看跟进记录数据权限？

### 解决方案

①正式客户的跟进记录受基础资料的客户、职员，制单人的权限控制。需同时满足可以查看相关客户、职员、制单人，才能看到对应的跟进记录。

②潜客的跟进记录受基础资料的职员、制单人的权限控制。需同时满足可以查看相关职员、制单人，才能看到对应的跟进记录。

### 操作步骤

PC端设置权限路径如下

• 打开系统设置并>基础设置>用户管理

![上传图片](https://vip.kingdee.com/download/01008a763d4bd5604838a1e342371ff4cf6e.png)

• 勾选需要控制授权的职员，点击<授权>

![上传图片](https://vip.kingdee.com/download/01000b189bd33be0464eb9ed185c97ada8f9.png)

• 选择“数据授权”左侧栏可按客户、职员、制单人进行数据权限控制。如下图所示，授权客户名称等于“客户A”，即只能看到客户A的跟进记录

![上传图片](https://vip.kingdee.com/download/0100890431b764db49f4b102602cd7cb3ff2.png)
