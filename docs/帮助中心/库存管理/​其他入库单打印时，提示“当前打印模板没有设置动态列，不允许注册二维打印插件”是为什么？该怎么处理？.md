# ​其他入库单打印时，提示“当前打印模板没有设置动态列，不允许注册二维打印插件”是为什么？该怎么处理？

【问题概述】

![](https://vip.kingdee.com/download/01008be0423442ba4a658a651c3d3db92e9b.png)

【解决方案】

只有辅助属性商品打印才会使用二维打印，使用二维打印时，必须在打印模板中有设置动态列且插件注册了二维打印插件。

一般出现上述情况是因为用户复制了系统预置的二维打印模板进行修改，所以模板中默认携带了二维打印插件。

分2种情况解决：

- 如果确实要使用二维打印，需对辅助属性组的列设置“类型”为“动态行”：

![](https://vip.kingdee.com/download/01091f13132c06bd40e18792d705ec9fe842.png)

- 如果不需要使用二维打印，则需要删除打印模板中的二维插件，保存即可：

![](https://vip.kingdee.com/download/01095c7badb684994a6494e50395a82c7c41.png)

【适用版本】
星辰2.0所有版本
