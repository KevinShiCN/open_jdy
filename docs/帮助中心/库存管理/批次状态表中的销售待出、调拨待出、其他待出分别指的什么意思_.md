# 批次状态表中的销售待出、调拨待出、其他待出分别指的什么意思?

【问题描述】

批次状态表中的销售待出、调拨待出、其他待出分别指的什么意思?

![](https://vip.kingdee.com/download/010962837ad06b204a3ca64e4c5390f9c0e5.png)

【解释说明】

在“系统设置-系统参数-进销存参数”中,只有设置了审核时更新库存,才能显示以下字段的数据:

销售待出:指该批次在未审核的销售出库单中的数量。

调拨待出:指该批次在未审核的调拨出库单中的数量。

其他待出:指该批次在未审核的其他出库单和盘亏单中的数量。
