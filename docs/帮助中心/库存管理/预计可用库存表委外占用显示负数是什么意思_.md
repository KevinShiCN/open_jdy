# 预计可用库存表委外占用显示负数是什么意思?

【问题描述】

预计可用库存表委外占用显示负数是什么意思?

【解释说明】

“委外占用” 是指已下达委外加工单但尚未发料所占用的库存量,正常情况下该数值不应为负数。若出现负数,建议在库存查询表中校对库存操作后再查询。
