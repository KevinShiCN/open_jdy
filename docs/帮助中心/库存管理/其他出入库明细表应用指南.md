# 其他出入库明细表应用指南

【问题描述】

其他出入库明细表应用指南

![](https://vip.kingdee.com/download/01094d4500b6c4ad4105bc23322a05f01988.png)
【业务场景】

1.部分客户存在除本身的销售出库和采购入库业务外,还有一些跟外部相关的其他出入库业务(例如批发行业,可能自己库存不够,需要临时跟其他同行借货),此时需要与采购销售类的出入库业务做一个区分,同时更好管理这种借还业务。

2.通过其他出入库明细表,可实现在同一张报表展示其他出库(借出)、其他入库(归还)的业务,从而快速地查询其他出入库单据。

【温馨提示】

**新增其他入库单或者其他出库单后,不是一定需要做其他出库单或者其他入库单,其他出入库单可以起到调整结存库存的作用。**
