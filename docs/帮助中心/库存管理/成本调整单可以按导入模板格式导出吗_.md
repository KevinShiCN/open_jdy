# 成本调整单可以按导入模板格式导出吗?

【问题描述】

成本调整单可以按导入模板格式导出吗?

【解决方案】

**成本调整单不可以按导入模板导出。**

当前引出是按列表引出,在成本调整单列表勾选需要引出的单据,直接点右上角的引出即可。
