# 为什么库龄分析表勾选显示批次和保质期,和不勾选显示的数量不一致?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：进销存熊诗琴 | 阅读：57 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/718638674057004800)

【问题描述】

为什么库龄分析表勾选显示批次和保质期,和不勾选显示的数量不一致?

#

【原因解析】

出现数量不一致的情况可能是在负库存仓库中出库了未入库的批次保质期商品导致的。具体情况如下:

步骤一:先入库批次001(10个) → 出库2个 → 剩余8个(正常计算库龄);

步骤二:又出库批次002(2个),但批次002从未入库过,没有入库信息,无法计算库龄;

![](https://vip.kingdee.com/download/01095afd4a6bbcb54042b575673e81805e7a.png)

步骤三:此时再查询库龄分析表,**不勾选显示批次保质期时**,只看总数,不算批次维度,库存就只剩6个;**勾选批次保质期时**,就会分开算,批次001的是8个,批次002是-2个,因为没做过入库,无法计算库龄,所以B这条记录不会显示,库龄分析表就会显示数量为8。

![](https://vip.kingdee.com/download/0109b52225e323844b6082bc53a2fc07f162.png)![](https://vip.kingdee.com/download/010917f586cdd9a04ea4b89cd730c9e0f651.png)

【解决方案】

当在库龄分析表勾选显示批次和保质期和不勾选显示的数量不一致时,请先找到数量有差异批次商品,需要到批次跟踪表查询该商品是否有批次做了负库存出库,然后补录对应批次的入库信息。

【适用版本】

星辰3.1及以上所有版本
