# “结存出现0成本单价(结存金额为0,数量不为0),可通过成本调整类单据进行金额调整”是什么意思?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：云社区用户6w624200 | 阅读：2285 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/553019118782130944)

【问题描述】

“结存出现0成本单价(结存金额为0,数量不为0),可通过成本调整类单据进行金额调整”是什么意思?

【业务场景】

![](https://vip.kingdee.com/download/01092b540cdc49434caba71fc9120c8ad02f.png)

在做【库存管理】-【出库成本核算】时,有提示“结存出现0成本单价(结存金额为0,数量不为0),可通过成本调整类单据进行金额调整”,这个一般是有数量但是没有金额导致的,通常发生这个的情况是负库存出库,没有设置成本取数来源的,可以设置下取数来源即可。

![](https://vip.kingdee.com/download/010904e1bb58e4014ffb82986d2e111b5c1b.png)

可以按【系统参数】-【存货核算参数】-【成本来源】-【无源单销售退货单取数来源】里,把这些都勾下,如果之前有对应单据就可以取到数;如果都没有单据,那就在商品资料里点进去有个参考成本维护下,到时候做出库成本核算就可以带出来数据。![](https://vip.kingdee.com/download/0109639f43eaf5ec4c5ab1ae06fcf65116b1.png)

或者还有一种方式,就是在出库前做一笔成本调整单把金额补进去,那就出库会有成本了。
