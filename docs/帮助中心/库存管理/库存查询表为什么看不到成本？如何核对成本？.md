# 库存查询表为什么看不到成本？如何核对成本？

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：快速入门, 常见问题
> 作者：进销存赵嘉丽 | 阅读：3600 | 有用：16
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/42634631360086016)

【问题概述】

库存查询表为什么看不到成本？如何核对成本

【解决方案】

库存查询表查询成本的方法：

| 版本 | 操作方法 | 产品页面 |
| --- | --- | --- |
| 星辰2.0 | 先点击计算成本（确定查询时已做过出库核算），然后在列设置开启显示成本字段 |  |
| 星辰3.1以上版本 | 先点击显示成本（确定查询时已做过出库核算），然后列设置开启显示成本字段 |  |

【字段说明】

商品的成本根据实际出入库成本来计算

成本=期初成本+入库成本-出库成本

单位成本=成本/即时库存

期初成本：为账套启用时录入，如果未录入或录入错误可能会导致出库成本核算错误，最终导致成本计算异常；

入库成本：为入库类单据（如采购入库单等）审核后产生，如果入库金额未录入或者录入错误则可能会导致出库成本核算错误，最终导致成本计算异常；

出库成本：是通过出库成本核算后自动计算得来（具体核算方式见系统参数——存货核算参数），如果期初或者入库成本录入不规范，则会导致出库成本异常最终导致成本异常。

综上所述，如果发现成本有异常，可先从以上三个环节排查，比如期初或入库类单据是否只录入了数量，价格信息未录入或者录入错误，以及是否进行了出库成本核算（核算是否有异常，是否处理好了异常），这些通过商品收发明细表或收发汇总表可以查出，如下图。找到原因后进行调整和修改，然后重新出库核算后再查看。

![](https://vip.kingdee.com/download/0100bb798e3e035f418fa4d1d1761dbfa4fb.png)

通过收发明细表和收发汇总表查成本注意事项：

1、如果成本核算方式启用分仓核算，查看收发明细表请按商品+仓库维度进行查询，查看收发汇总表时汇总依据按“商品+仓库”进销存查询；

2、收发明细表和收发汇总表会统计未审核的单据，但此类型单据不参与成本计算，需要确认审核或过滤；

3、如果是子账号查询，请检查子账号是否有设置数据授权（比如仓库、商品等）。
