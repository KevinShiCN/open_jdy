# 功能解析：结存成本异常查询![](https://vipfront.s3.cn-north-1.amazonaws.com.cn/emotion/define/86.gif)![](https://vipfront.s3.cn-north-1.amazonaws.com.cn/emotion/define/86.gif)

1、应用价值：

用于业务期末结账前，查询统计库存商品是否存在数量为0成本不为0的结存成本，查询到后支持选择后批量生成入库成本调整单。

2、适用场景：

当存在暂估、退换货等业务时，可能会导致商品期末数量已为0，但进行出库成本核算后，还是有结存成本。每次都需要手工当入成本调整单，不能统一进行查询和处理。

应用流程：

1）【库存管理】-增加【结存成本异常查询】，当前支持查询【数量为零，成本不为零】及【数量为正，成本为负】的商品结存数据及成本

![上传图片](https://vip.kingdee.com/download/0100b1dc92b903ff49209470c60875c1d1d1.png)

2）支持录入调整后成本，系统会根据结存成本-调整后成本计算需调整金额，选择对应的分录行后，通过【生成调整单】功能按钮，可以生成对应商品的入库成本调整单
