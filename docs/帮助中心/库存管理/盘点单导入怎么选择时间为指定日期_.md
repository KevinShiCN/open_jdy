# 盘点单导入怎么选择时间为指定日期?

【问题描述】

盘点单导入怎么选择时间为指定日期?

【解决方案】

第一步:下载盘点单导入模板。

先在【库存管理】-【盘点单】查询,跳转到盘点单列表

![](https://vip.kingdee.com/download/0109953a5df6cd34436ca5f685494a79594f.png)
然后下载盘点单引入模板,和引入示例填写。
![](https://vip.kingdee.com/download/010904b5f5688751488ba04877d621231cae.png)

第二步:盘点单导入模板,**盘点日期录入具体指定日期即可。如下图所示:**

![](https://vip.kingdee.com/download/0109521f3fe75ea744ddb828e8b5993c6933.png)
