# 拣验汇总表如何查询/引出/打印?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：14 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/783887659495581696)

【问题描述】

拣验汇总表如何查询/引出/打印?

【业务场景】

仓管员依据拣验货单去拣货/验货,需按月统计拣货员拣验的商品数量信息,按总数量给拣货员计算提成。

【解决方案】

库存管理的“拣验汇总表”,支持按拣货员维度汇总拣货/验货的已拣验数量。

**1、查询**

依次点击【库存管理】-【拣验汇总表】,在过滤条件中设置筛选条件,勾选“显示商品明细”后按拣货员+商品维度汇总显示数据。勾选“显示小计”后按拣货员小计已拣验数量和已拣验基本数量。

![](https://vip.kingdee.com/download/01093502889ad09945edb077b0b1841efa4b.png)

**2、引出**

点击右上角“引出”即可导出为Excel,拣验汇总表中显示的内容都可以引出,若某一列数据不需要引出可以点击右上角“列设置”隐藏对应的列后再引出。

![](https://vip.kingdee.com/download/01096ec0da9575db4f45bac8515fd59ecfaa.png)

**3、打印**

点击右上角“打印”功能即可进行打印,打印之前可以点击“打印”下拉框的“打印设置”选择打印方式、打印模版及调整打印行数等。

![](https://vip.kingdee.com/download/01095c8aaf16a515497eb9f6135d0328a560.png)
