# 成本调整单--采购单价从3000降到2800，如何在成本核算中体现出来？

# 关于成本调整单的小知识![](https://vipstatic.obs.cn-north-4.myhuaweicloud.com/statics/emotion/define/86.gif)

 1、业务场景：
企业业务发生过程中，由于价格波动和业务处理，已确认的成本可能需要进行调整。例如：某手机销售商3月
份从华为厂家采购700台华为某款手机单价3000元，然后销售到7月份时，厂家给予这家公司做保价政策，起始就相
当于厂家给的返利，每台手机反200元，相当于成本从3000降到了2800元。

 2、制单指南：
单击【库存管理】→【成本调整单】，进入“成本调整单”页面，如下图所示：

![](https://vip.kingdee.com/download/01092851149886ff48129467d238175aefa5.png)

![](https://vip.kingdee.com/download/0109808f61c0f3b7408889579fe2b94a1dd8.png)

 1）成本调整单支持两种业务类型：出库调整和入库调整。

2）选择出库调整类型时，还需要再次选择出库类型，包括销售出库和其他出库

3）成本调整单支持根据商品批号和辅助属性进行成本调整，直接录入需要调整的总金额，保存即可。

4）选择入库调整类型时，如果同时录入了供应商，则单据审核后同时影响供应商应付账款，在应收应付付款
类型单据中可以选择该单据进行付款或核销。
