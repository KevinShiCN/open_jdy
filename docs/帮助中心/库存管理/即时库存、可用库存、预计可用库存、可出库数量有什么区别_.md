# 即时库存、可用库存、预计可用库存、可出库数量有什么区别?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：4145 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/568964240183307776)

- **【问题描述】**
- 即时库存、可用库存、预计可用库存、可出库数量有什么区别?
- **【解释说明】**
- **即时库存:**指的是某一商品在当前时间点的实时库存数量,即仓库有实物存在的真实库存。
- 若需具体查看出入库记录,更多可以点击:[如何使用商品收发明细表查询商品的出入库明细及成本数据?](https://vip.kingdee.com/link/s/Z1g1l)
- **可用库存: **等于仓库真实库存+入库在途-即将出库的或预留出库的(可自行设置公式,可在系统参数-进销存参数-库存参数中设置。
- 更多可点击:[可用库存是如何计算的?](https://vip.kingdee.com/link/s/MUjNM)
- **预计可用库存**:主要适用于批次/保质期/序列号商品、或普通商品销售订单和采购订单上没有录仓库、仓位时特有的库存。
- **可出库数量:**可出库数量根据【系统参数】-【进销存参数】中的【负库存参数】选择进行计算
- 1、当选择按即时库存控制负库存时,可出库数量=即时库存-预留量
- 2、当选择按按可用库存控制负库存时,可出库数量=可用库存
