# 序列号状态表如何查询?

> 作者：Simone邓 | 阅读：256 | 有用：2

【问题描述】

序列号状态表如何查询?

【 功能介绍】

在【库存管理】--【序列号状态表】中查询。

启用了序列号管理的商品,一般都是需要重点关注的商品(如单价比较高),序列号状态表是用于快速查询序列号商品是否在库或者被占用,用于跟踪序列号商品的在库状态的报表,便于用户追踪序列号商品流程过程、价格。

更多序列号状态表详情请查看:[序列号状态表](https://vip.kingdee.com/link/s/ln5as)
