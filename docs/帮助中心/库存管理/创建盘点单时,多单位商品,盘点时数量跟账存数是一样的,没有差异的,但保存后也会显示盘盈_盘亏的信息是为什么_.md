# 创建盘点单时,多单位商品,盘点时数量跟账存数是一样的,没有差异的,但保存后也会显示盘盈/盘亏的信息是为什么?

> 作者：海鸥乔纳森 | 阅读：344 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352850368466432)

【业务场景】

创建盘点单时,多单位商品,盘点时数量跟账存数是一样的,没有差异的,但保存后也会显示盘盈/盘亏的信息是为什么?

![](https://vip.kingdee.com/download/010967eb4ad646c549988ec9c7795010bb82.png)

![](https://vip.kingdee.com/download/0109bf512796e56e490a918c135c488f3a7f.png)

【原因分析】

这是因为辅助盘点数被清空了没有填写,多单位商品,如果辅助账存数不为0,盘点时也要填写“辅助盘点数”,否则相当于辅助单位对应的数量是有盘亏或者盘盈。

![](https://vip.kingdee.com/download/01099801d3d8591f4ba3af32aa390185b5b9.png)
