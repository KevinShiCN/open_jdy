# 结账好几个，到底用哪个

结账好几个，到底用哪个

      公司因业务需要，星辰的模块基本都买全了。但月底结账，小黄一看系统好几个地方都有结账，不知道选哪个，两眼一抹黑。好在小蓝之前用过星辰，于是他很热情地跟小黄娓娓道来：小黄，先别急，你看：

      首先，各个地方的结账与菜单对应关系：

1、“库存管理--存货核算-业务期末结账”这里的结账，对应菜单有：销售管理、采购管理、库存管理、生产管理、委外管理、应收应付。

![](https://vip.kingdee.com/download/01000a60cd2d4ddf43f5838c13fd42b67eab.png)2、“财务管理-期末处理-结账”这里的结账，对应菜单有：账务处理、财务报表、工资管理。

![](https://vip.kingdee.com/download/0100172a77a557a34cf3a6655b669ebe3c67.png)

3、“出纳管理-期末处理-出纳结账”这里的结账，对应菜单有：出纳管理。

![](https://vip.kingdee.com/download/01009650ab22972f4e12bcf899238ad448fe.png)

4、“资产管理-期末处理-结账”这里的结账，对应菜单有：资产管理。

![](https://vip.kingdee.com/download/0100aa47badfe3624143ba16f2a9f47d8868.png)

这下你知道各个地方的结账是干嘛用的了吧，小黄赶紧点点头回应。

      其次，各个地方的结账虽然是独立的，但是也可以设置关联关系：

1、业务结账与财务结账的关联关系设置：

系统设置-系统参数-存货核算参数-参数设置，可以控制：

1）业务结账必须早于财务结账和财务结账前检查未生成凭证的业务单据。

![](https://vip.kingdee.com/download/01004e95c0265a4a4064b269b5d3cc7874ac.png)

2）系统设置-系统参数-财务参数，可以控制：不允许修改和删除业务系统生成凭证。

![](https://vip.kingdee.com/download/01002c22b1884f46450b99f571ef0b5e14ab.png)

2、 出纳结账与财务结账关联关系设置：

系统设置-系统参数-出纳参数，可以控制：

1） 勾选“与总账结账”，则出纳模块不能单独结账。总账结账以后，出纳模块会自动跟着一起结账。如果点出纳结账会提示：

![](https://vip.kingdee.com/download/010049620ce137f74e4eb7c263b4e30a1b7b.png)

2） 不勾选“与总账结账”，则出纳跟财务模块单独结账，相互独立。

![](https://vip.kingdee.com/download/010022e6b746e6504aaa9e7474bee648bccc.png)

3、 资产结账与财务结账关联关系设置：

系统设置-系统参数-资产参数，可以控制：

1） 勾选“与总账结账”，则资产模块不能单独结账。总账结账以后，资产模块会自动跟着结账。如果点资产结账会提示：

![](https://vip.kingdee.com/download/0100715c8efeb1994784b59930bf9d7ddbaf.png)

2）不勾选“与总账结账”，则资产模块跟财务模块单独结账，相互独立。

![](https://vip.kingdee.com/download/01002545eaca4ba94bc7adf8c274c5b5a67c.png)

      备注：如果资产与总账勾选连用，没有勾选“与总账一起结账”，总账结账的时候会校验资产模块要先结账，才能总账结账。

![](https://vip.kingdee.com/download/0100864821980c3541499a50b7bad70876e9.png)

![](https://vip.kingdee.com/download/01001c1a1e9f190744a2b2e5b92a0ff5b6bc.png)
