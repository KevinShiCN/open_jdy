# 库存查询表如何默认按照商品编码排序?

> 作者：胡大大 | 阅读：32 | 有用：1

【问题描述】

库存查询表如何默认按照商品编码排序?

【操作步骤】

1.点击【库存管理】-【库存查询表】

![](https://vip.kingdee.com/download/0109eaee22fae81e4e29a2a7ae616763dbee.png)
2.点击<列设置>选择需要排序的字段设置排序依据即可

![](https://vip.kingdee.com/download/0109a22a00704e754fe18eeda2aa4f8321d4.png)
