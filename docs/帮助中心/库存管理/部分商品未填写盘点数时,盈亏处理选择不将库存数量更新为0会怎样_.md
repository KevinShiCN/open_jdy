# 部分商品未填写盘点数时,盈亏处理选择不将库存数量更新为0会怎样?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：257 | 有用：5
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/129339770097650305)

【问题描述】

部分商品未填写盘点数时,盈亏处理选择不将库存数量更新为0会怎样?

【概述】

此时会认为这部分商品不参与盈亏处理,即等同于将这部分商品从盘点单中删除后再进行盈亏处理。

![](https://vip.kingdee.com/download/01099fd78b63192743a4bacb6bd4e5264126.png)

说明:盘点数为空表示商品未盘点,需要补录盘点数;

盘点数为0表示商品已盘点且盘点数为0;

更新:将盘点数更新为0;

**不更新:将盘点数为空的商品行从单据制除**;

取消:回到单据详情界面,可继续编辑.
