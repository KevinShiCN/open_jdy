# 审核盘点单时没有录入盘点数的商品是否会默认为0?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：ylim | 阅读：187 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/718268547549485312)

【问题描述】

审核盘点单时没有录入盘点数的商品是否会默认为0?

【解释说明】

盘点商品比较多,有的商品没库存或库存数和账存数一样就没有录入盘点数,在审核盘点单时系统会提示:存在盘点数为空的商品,是否将其盘点数自动更新成0

①盘点数为空表示商品未盘点,需要补录盘点数;

②盘点数为0表示商品已盘点且盘点数为0。

③更新:将盘点数更新为0;

④不更新:将盘点数为空的商品行从单据删除;

**更新或不更新覆盖单据上所有盘点数为空的商品,不支持指定个别商品,若不同商品需不同的结果,建议手工录入盘点数。**

【操作步骤】

盘点单审核时,若有盘点数为空的商品,系统提示“存在盘点数为空的商品,是否将其盘点数自动更新成0”,根据实际库存情况点击<不更新>或<更新>;

![](https://vip.kingdee.com/download/0109ecc1c2c3faad4deda7dc9fdcf75690bc.png)

若盘点数为空的商品有的需更新为0有的库存不变需从盘点单上删除,建议点击<取消>按钮返回编辑,把盘点数不变的商品删除或手工录入盘点数,剩余实际库存为0的可审核时批量更新为0。
