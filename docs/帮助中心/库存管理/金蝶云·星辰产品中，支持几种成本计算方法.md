# 金蝶云·星辰产品中，支持几种成本计算方法

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题, 快速入门
> 作者：Zyl | 阅读：4002 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/65132018715266816)

问题描述：

金蝶云·星辰产品中，支持几种成本计算方法

解决方案：

支持移动平均、先进先出、加权平均三种，同时支持按批号核算成本和按辅助属性核算成本。
