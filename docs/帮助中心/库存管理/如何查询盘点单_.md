# 如何查询盘点单?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：160 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/628760595726172928)

【问题描述】

如何查询盘点单?

【解决方案】

在盘点单列表中可以查询历史的盘点记录;如果需要查询某一个商品的历史盘点记录,可以通过导出盘点单后搜索。

依次点击【库存管理】-【盘点单/查询】打开盘点单列表界面,在筛选条件设置中可以点击<展开过滤>选择查询范围;点击单据编号可以打开盘点单查看详细盘点记录。

![](https://vip.kingdee.com/download/010956a143366c0a48e79488e5e37f615002.png)

点击盘点单编号,可以打开盘点单明细界面,查询盘点数、盘盈数、盘亏数等详细情况。

![](https://vip.kingdee.com/download/0109f87e7bd6c37b429f81aef8029400afa1.png)

若想查询某一个商品的历史盘点记录,选择盘点记录后再点击右上角【引出】-【按引入格式引出】,在导出的Excel表格中搜索对应的商品盘点记录。

![](https://vip.kingdee.com/download/01090bdbcebc1a7e437dba735419627100b5.png)
