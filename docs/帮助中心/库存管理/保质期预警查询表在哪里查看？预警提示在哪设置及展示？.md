# 保质期预警查询表在哪里查看？预警提示在哪设置及展示？

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：功能介绍, 快速入门
> 作者：进销存熊诗琴 | 阅读：2663 | 有用：8
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/75955749658408448)

问题描述：

解决方案：

1、查询路径：【仓库管理】—【预警报表】-【保质期预警查询表】

![](https://vip.kingdee.com/download/01005f57dfa519ab444eae7a4d090eb10c41.png)

2.打开保质期预警查询表，可以看到批次、保质期，预警天数，剩余到期天数（天）等信息。支持按照批次，商品，商品类别，标签，条码，规格型号、剩余到期天数筛选，同时支持打印，引出等；

![](https://vip.kingdee.com/download/0100161dfed16f3748aca443ec8178b4b925.png)

3.首页预警提醒设置，位于保质期预警查询表界面右上角：

![](https://vip.kingdee.com/download/0100296993c21a6446aeaf9ce11360f147d3.png)

如果勾选，则当有商品到期预警期内或者已经超期，**首页右上角的【预警信息】**会有对应的提醒，数字代表具体预警的商品种数（如下图）。不勾选，则不会提醒。

点击“发现预警期内商品”或“发现过期商品”，则会跳到保质期预警查询表，展示保质期预警商品。

![](https://vip.kingdee.com/download/0100a2e82525615f4c3099955388fa042541.png)
