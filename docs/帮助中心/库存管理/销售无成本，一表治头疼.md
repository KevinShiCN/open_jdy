# 销售无成本，一表治头疼

> 作者：咚咚 | 阅读：2283 | 有用：3

销售无成本，一表治头疼

![上传图片](https://vip.kingdee.com/download/0100bc81d9b392c047aea59b78eff7e3175e.png)

      月底了，老板在APP上看销售毛利，不看不知道，一看吓一跳。毛利咋这么高？于是立马电话小涂说：小涂你看下公司这个月的销售毛利，我们没挣这么多吧？你半个小时以后来跟我汇报下情况。

      小涂暗想：完了完了，该不会哪里没做好，要挨训了。但他很快调整状态，安慰自己：莫慌，淡定，先解决问题，毕竟只有半小时。小涂赶紧坐回电脑前，打开星辰系统，查看了本月的销售利润表。销售收入能对上，销售成本一眼看过去好像都没有。那就按以往两大步的方法来排查吧。那究竟他用的是哪两大步呢，一起来看看吧！

      首先按照第一步，他先检查了业务模块的当前期间和出库核算结果查询，当前期间是本月，出库核算结果查询里面没有本月的核算记录，那问题应该出在本月没做出库成本核算。于是他直接做了出库成本核算。算完以后，很快销售利润表的大部分销售类单据都有了成本，小涂松了一口气，心想：还好还好，问题不是很大。

![](https://vip.kingdee.com/download/0100df10e892ae6e4e759b24b8b643a53730.png)

![](https://vip.kingdee.com/download/0100df1174ede2a840f1bd0608d6c6ad77c7.png)

      再按照第二步，小涂在销售利润表“查看类型”里选上“查看商品明细”，查询以后，先过滤出成本是0的商品。

![](https://vip.kingdee.com/download/0100b39a3aea798d41b39ace44667c6aa4a6.png)接着点开“查看成本计算报告”，勾选“仅显示异常成本计算报告”，依次查看相应0成本商品的“成本计算表”。

![](https://vip.kingdee.com/download/010039600feb57a442d2a529ccf2b28085a1.png)

![](https://vip.kingdee.com/download/010011aab51e522549aa9c7be554e69d3ffb.png)

      他看了这些异常的成本计算表以后，总结原因大概就以下三种情形：

      第一种：入库没有成本，销售出库单没有成本。

![](https://vip.kingdee.com/download/01006560d179b3fd47f09118bbbb3a0eddf1.png)

![](https://vip.kingdee.com/download/0100fc99f206975e4df7b4c40ccc9bf3b74d.png)

如果是这种情形，则需要补录入库单的入库成本，成本重算以后销售出库才会有成本。

      第二种：负库存出库，销售出库单没有成本。

![](https://vip.kingdee.com/download/0100128c971e18314f9d97ca3ad5e8f9370b.png)

![](https://vip.kingdee.com/download/0100347688f23d1f4dd1a3ab7511d2dde82b.png)

如果是这种情形有两种方法可以处理：

方法1：允许负库存出库，则需要在存货核算参数里设置负库存出库单的取数来源。当四种成本来源都有值或者部分有值，重算以后会按优先级顺序依次取优先级高的成本来源。如果四种成本来源都没有值，则成本还是0；

![](https://vip.kingdee.com/download/010038d0fb7affed4688909fec966091c9c3.png)

方法2：不允许负库存出库，则要先补入库单或者将以后时间的入库单调整到负库存出库单的单据时间之前或者同一天，保证先入库在出库。

      第三种：无源单的销售退货，销售退货单没有成本。

![](https://vip.kingdee.com/download/01000a324ad67d464abc84fef372c49f33a3.png)

![](https://vip.kingdee.com/download/01008b7c2d47ef37476d9031364a3ac9c099.png)

如果是这种情形，类似负库存出库处理，则要在存货核算参数里设置“无源单的销售退货单”的取数来源。当成本来源有值时，无源单的销售退货单重算成本以后可以取成本；当成本来源没有值，则成本还是0。

![](https://vip.kingdee.com/download/0100df16d79390be45b8b819a1bb9c0c4f74.png)

      小涂对照这些成本异常的的商品的“成本计算表”，一顿操作猛如虎，很快销售利润表就都有成本了。小涂看了下时间，离半小时还有5分钟。他加快脚步来到老板办公室。他跟老板汇报说：老板，星辰的出库类单据都要先做出库成本核算才会有成本，您看毛利时，我们还没来得及做出库成本核算，现在做了就都有成本了。老板说了句：月底了收尾的工作要抓紧，别拖拖拉拉，要不然手机上看毛利都不准。小涂连忙点头回应：是是是。暗自庆幸：总算躲过一劫了。

从老板办公室出来时，他脑海里蹦出了另外一种情形，销售类单据也会取不到成本。

      第四种：销售出库单生成结转成本的凭证后，又修改了销售出库单，成本重算以后销售出库单也会取不到成本。

![](https://vip.kingdee.com/download/01007cd701d53ae8496880d9140e4e503147.png)

![](https://vip.kingdee.com/download/010029bcbba7ce604409801e9e629992dcaf.png)

![](https://vip.kingdee.com/download/01002b1251394f30489babfab5d1cfba0590.png)

如果是这种情形，则需要先删除结转成本的凭证，在重算成本即可。

      小涂心里嘀咕着：这四种情形我们自己知道就好，可别让老板知道了，不然会被说工作没做好。还好有“成本计算表”，此表在手，我才会在核对成本时如鱼得水，不会太头疼。
