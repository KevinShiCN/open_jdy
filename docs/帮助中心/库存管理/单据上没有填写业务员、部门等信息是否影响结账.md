# 单据上没有填写业务员、部门等信息是否影响结账

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：shaoling | 阅读：115 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/617524573239783424)

【问题描述】

单据上没有填写业务员、部门等信息是否影响结账

【解释说明】

单据上业务员、部门等表头信息是可以选填项,公司需要按部门和业务查询统计出入库数量则开单时需要选择对应的数据,后续相关报表可按部门和业务员的维度统计查询,反之不填写也可以正常开单,不会影响结账。

比如销售汇总表可以按业务和部门统计销售
