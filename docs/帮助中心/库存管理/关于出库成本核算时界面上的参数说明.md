# 出库成本核算的参数说明

 1、出库成本核算是指对企业存货价值（即成本）的计量。

 2、出库成本核算前，请先根据企业管理需求，进行存货核算参数设置。【见系统参数】

 3、单击【库存管理】→【出库成本核算】，进入出库成本核算界面，如下图所示：

![](https://vip.kingdee.com/download/01090ceaf4a4ce6941d18984e366b4472b08.png)



 1）核算本期所有商品：取业务系统当前期间期初余额和单据进行出库成本核算。

 2）核算指定截至日期范围商品：根据指定的单据日期范围，取期初余额和单据进行出库成本核算，与核算本期所有商品互斥。

 3）核算指定商品类别：仅取指定的商品类别中的商品进行出库成本核算。

 4）核算指定商品：支持多选，取指定商品进行出库成本核算。

 5）核算所有仓库：取所有仓库进行出库成本核算。

 6）核算指定仓库：支持多选，取指定仓库进行出库成本核算。

5、设置完成核算范围，单击“开始核算”，系统进行出库成本核算，核算完成后，提供成本计算报告。如下图
所示：
