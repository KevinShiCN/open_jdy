# 自制入库核算和出库成本核算有什么区别?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：ZXYzxc | 阅读：105 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/766829630078256896)

【问题描述】

自制入库核算和出库成本核算有什么区别?

【解释说明】

**自制入库核算:**核算无源单、已审核且未生成凭证的产品入库单和产品退库单对应的所有产品信息,可对其录入库成本

业务场景:当企业未使用生产管理完整功能,仅通过库存管理手工录入生产领料和产品入库进行简单的生成管理时,可以通过自制入库核算,手工填写入库成本后,通过核算将入库成本核算填写到产品入库和产品退库单。

![](https://vip.kingdee.com/download/0109fe2f1cb4c37a4c8a8d98fecf86f82714.png)

**出库成本核算:**核算出库类单据的成本数据,涉及到的单据类型可到【库存管理】-【商品收发明细表】中“单据类型”字段进行查看。

出库成本核算是指根据设置的成本核算方法、期初结存及入库成本计算所核算期间的出库单的成本,核算后出库单上的商品将计算出出库的成本,并在库存报表上结存显示商品的剩余成本。

![](https://vip.kingdee.com/download/01091e0d99709a7540538d0c8f1492b0bc76.png)
二者之间不冲突,可同时使用,自制入库核算录入后会影响后续出库成本的核算
