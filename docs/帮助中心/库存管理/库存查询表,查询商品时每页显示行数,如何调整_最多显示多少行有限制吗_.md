# 库存查询表,查询商品时每页显示行数,如何调整?最多显示多少行有限制吗?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：634 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391350565546083072)

【问题描述】

 库存查询表,查询商品时每页显示行数,如何调整?最多显示多少行有限制吗?

如在库存查询表查询商品时,每页仅显示20行,如何调整?

如图:在库存查询表的界面,在选择商品的界面,为什么一页的数据只展示20条商品信息?

![](https://vip.kingdee.com/download/0109f02d5f07468a48a8aa3a19f095386b19.png)

【解释说明】

选择商品显示行数主要是由于在商品列表界面,设置了每一页只显示20条商品信息

【解决方案】

1、首先,点击[基础资料]--[商品信息]界面,设置好每一页显示的商品条数,设置好之后,需要关闭商品列表页签,相当于是保存当前设置,再去库存查询表界面查询筛选商品,商品一页会展示更多的商品信息,如图:

![](https://vip.kingdee.com/download/0109afc457f61529451c984a9d3c982bb0a2.png)

![](https://vip.kingdee.com/download/010943d996b3def84d9da0620b77ed7d48af.png)

【温馨提示】

库存查询表页面可设置 **50 条 / 页、100 条 / 页、500 条 / 页、1000 条 / 页** 的展示规格,每页最多显示 1000 条数据;若数据总量超过单页上限,则会自动分页展示。

![](https://vip.kingdee.com/download/010900f05a0de3db46b3addc77c901412269.png)
