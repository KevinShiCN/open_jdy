# 库存是保存更新,入库单没审核的情况下出库了,库存和成本会错吗?

> 作者：云社区用户6w624200 | 阅读：146 | 有用：0

【问题描述】

库存是保存更新,入库单没审核的情况下出库了,库存和成本会错吗?

【解释说明】

**1、库存计算**

保存更新库存的情况下,保存入库单和出库时已经更新了库存了,出库后再审核入库单不会再次更新库存,所以库存数量不会错。

**2、成本计算**

成本会计算不对的,单据要审核才能参与成本核算,建议先把入库单审核了,并检查入库单单据日期和出库单单据日期是否是先入后出,然后再核算出库成本。
