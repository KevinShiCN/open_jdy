# 库存管理

整体介绍库存管理中的仓存单据、存货核算功能、仓存报表的查询。

| 步骤 | 具体说明 | 学习链接 |
| --- | --- | --- |
| 1.仓库单据 |  | —— |
| 1.1其他出入库单 |  | —— |
| 1.1.1其他入库 | 处理其他非采购类型的入库单据，比如获赠物料的入库、委托加工等不参与采购管理的入库类业务 | 点击学习 |
| 1.1.2其他出库 | 处理其他非销售类型的出库单据，比如销售附赠物料、委托加工等不参与销售管理的出库类业务 | 点击学习 |
| 1.2调拨业务 | 区分移仓业务和调拨业务的差异 | 点击学习 |
| 1.2.1移仓单 | 移仓单用于处理不需要跟踪在途情况的调拨业务，如地址位置相邻仓库直接的商品转移   | 点击学习 |
| 1.2.2调拨出库 | 由商品出库的仓库发起，用于处理商品在仓库间的转移业务 | 点击学习 |
| 1.2.3调拨入库 | 由商品的入库方发起。当调拨的商品经过运输到达收货仓库时，由收货仓库的仓管员清点商品并填写调拨入库单，确认收货明细 | 点击学习 |
| 1.2.4调拨差异处理表 | 处理调拨过程中可能存在忘发、错发、损坏等导致调拨出库的商品数量与调拨入库的商品数量不一致的问题   | 点击学习 |
| 1.3盘点业务 |  | —— |
| 1.3.1盘点单 | 盘点是指定日期或实时对库存商品实际数量进行清点，将结果与系统数据进行对比，调整并统一系统数据的业务 | 点击学习 |
| 1.3.2盘盈单 | 商品实际库存大于系统中的账存数，可以直接录入盘盈单来处理实际库存和账存数的差异   | 点击学习 |
| 1.3.3盘亏单 | 商品实际库存小于系统中的账存数，可以直接录入盘亏来处理实际库存和账存数的差异 | 点击学习 |
| 1.4组装拆卸业务 |  | —— |
| 1.4.1组装单 | 组装单用于记录将多个子件或组件（如A和B）组装成一个新的成品（如C）的过程， 采购原材料或配件后，将多种原材料或配件通过加工或组装的方式，变为一种新的产品销售，可用组装单处理   | 点击学习 |
| 1.4.2拆卸单 | 拆卸单允许用户将一个成品拆解为多个子件，这通常用于退货、维修、产品升级或物料重新分配等业务场景 | 点击学习 |
| 1.4.3组装拆卸模板 | 组件和子件有稳定的配比关系，可以用来复用的，则可将对应的组装单和拆卸单保存为模板，保存为模板后，后续在开单时，可调用模板，帮助快速开单 | 点击学习 |
| 1.5简单生产 | 企业是工贸企业，简单生产主装业务，只购买进销存+财务管理，需要实现生产的出入库需求，可通过：生产领料单，生产退料单，产品入库，产品退库四类单据进行变通处理。适用于专业版、旗舰版，标准版不支持这四类单据 | —— |
| 1.5.1生产领料单 | 若未开通生产模块，直接录入生产领料单理解为领用原材料出库。 | 点击学习 |
| 1.5.2生产退料单 | 在领料后发现物料质量不合格,需要退回给仓库并要求重新提供合格可用的材料。 | 点击学习 |
| 1.5.3产品入库 | 完成生产，产成品入库 | 点击学习 |
| 1.5.4产品退库 | 若产成品不合格，可通过产品退库，退回生产仓库 | 点击学习 |
| 1.5.5自制入库成本核算 | 企业未使用生产管理完整功能，仅通过库存管理手工录入生产领料和产品入库进行简单的生产管理时，可以通过自制入库核算，手工填写入库成本后，通过核算将入库成本填写到产品入库和产品退库单。 说明：该功能专业版及旗舰版均支持，标准版不支持 | 点击学习 |
| 2.存货核算 | 存货核算是系统中非常重要的一部分，做了出库成本核算，系统中的出库单据才会核算成本信息 | 点击学习 |
| 3.仓库常用报表 |  | —— |
| 3.1库存查询表 | 库存查询表是根据当前的过滤条件，展示筛选出的商品库存情况，包括即时库存和可用库存   | 点击学习 |
| 3.2库存横向查询表 | 在查询库存的时候按照横向展示仓库的样式看到仓库的库存情况，在有多个仓库的情况下，这样的展示会比较简单直观。 | 点击学习 |
| 3.3预计可用库存表 | 根据当前过滤条件，展示筛选出的商品库存情况，包括即时库存、可用库存、待入库存和待出库存 | 点击学习 |
| 3.4商品收发明细表 | 根据当前过滤条件，按单据展示筛选出的商品的出入库业务明细情况，是进销存最常用的报表之一 | 点击学习 |
| 3.5商品收发汇总表 | 根据当前过滤条件，展示筛选出的商品的出入库数据统计，是进销存最常用的报表之一 | 点击学习 |
| 3.6调拨汇总表 | 总仓调拨给各个门店后，需要统计调拨汇总情况；对于异价调拨场景，门店与总部之间调拨产生的成本及调出毛利情况 | 点击学习 |
| 3.7批次跟踪表 | 由于批次商品在管理上的特殊性，批次商品在业务流程中往往需要单独进行跟进。跟踪批次商品的全流程，包括入库（采购、盘盈、调拨、其他入库等）、出库（销售、盘亏、调拨、其他出库等）   | 点击学习 |
| 3.8批次状态表 | 用于展示批次商品的库存状态，同时支持查询保质期商品的库存状态，起到保质期预警的作用。可查看：(1)批次对应的保质期时间(2)商品各批次的剩余库存数量 | 点击学习 |
| 3.9序列号跟踪表 | 序列号商品一般都是价值比较高，需要重点跟踪出入库情况的商品，序列号跟踪表可以用于查询序列号商品的一个出入库详细情况，详细记录每一个序列号的出库和入库详情   | 点击学习 |
| 3.10序列号状态表 | 序列号状态表是用于快速查询序列号商品是否在库或者被占用，用于跟踪序列号商品的在库状态的报表，便于用户追踪序列号商品流程过程、价格 | 点击学习 |
| 3.11其他出入库明细表 | 对进行其他出入库业务的商品进行汇总，方便统计一段时间内其他出入库业务数据；支持统计成本，展示商品资料自定义字段   | 点击学习 |
| 3.12生产领料明细表 | 生产领料明细表是根据当前的过滤条件，展示筛选出的生产领料及退料的情况。适用于专业版、旗舰版 | 点击学习 |
| 3.13生产领料汇总表 | 支持多维度展示生产领料汇总情况。  适用于专业版、旗舰版 | 点击学习 |
| 3.14产品入库明细表 | 产品入库明细表展示了不同仓库、部门、物料等的产品入库及退库明细的情况。适用于专业版、旗舰版 | 点击学习 |
| 3.15产品入库汇总表 | 产品入库汇总表支持多维度展示产品入库汇总情况。适用于专业版、旗舰版 | 点击学习 |
| 4.仓库分析 |  | —— |
| 4.1库存预警查询表 | 对于商品资料中设置了最低库存、最高库存、安全库存的商品，可通过《库存预警查询表》来追踪目前商品的库存水平，并针对所设置的预警参数给出相应的采购建议。针对超储量的商品，及时调整相应的销售策略，提高库存周转率，加速库存消耗。   | 点击学习 |
| 4.2保质期预警查询表 | 对于启用了保质期管理的商品，可通过《保质期预警查询表》追踪商品的剩余到期天数（天），预警天数等信息，同时可以通过设置“登录时首页弹窗提醒”来实现每天的保质期预警通报，更好的掌握商品临期状况，以便及时调整相应的销售策略，提高临期商品库存周转率，加速库存消耗，避免商品出现过期等问题   | 点击学习 |
| 4.3库龄分析表 | 支持及时查询库存商品在库时长，以便合理安排库存计划，避免库存积压，提高库存周转率。星辰的库龄分析表，支持展示库存商品在库时长，可自定义不同库龄分组区间，可按商品，按仓库不同维度查看库龄数据 | 点击学习 |
