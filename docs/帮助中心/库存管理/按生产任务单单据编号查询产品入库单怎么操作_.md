# 按生产任务单单据编号查询产品入库单怎么操作?

> 作者：倒也不扶墙 | 阅读：9 | 有用：0

【问题描述】

按生产任务单单据编号查询产品入库单怎么操作?

【操作说明】

可以通过产品入库单列表按以下两种方式进行筛选:

方式一:

1、产品入库单列表列设置显示【源单单号】列字段

![](https://vip.kingdee.com/download/0109e3c3323f24724366a9dc52976c068697.png)

2、源单单号处可通过小漏斗,按生产任务单号进行过滤![](https://vip.kingdee.com/download/0109f50c91c3902041ec83971548c5450fe7.png)

方式二:

新增过滤方案,可按源单单号作为筛选条件进行过滤

![](https://vip.kingdee.com/download/01090d173dd626dd490a965667e51140b3dd.png)
