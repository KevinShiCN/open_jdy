# 拣验工作台可以录入商品的辅助数量吗？

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题, 进阶技巧, 新特性
> 作者：移动生产白建凡 | 阅读：162 | 有用：0

# **1 问题描述**

- 拣验货工作台只能录入基本单位数量吗？能不能和拣验货单一样显示辅助单位，录入辅助单位数量呢？
- 为什么销售订单上录入的是辅助单位数量，去拣验工作台拣验时显示的是基本单位？

# 2 适用版本

星辰3.0以上所有版本（需购买拣验货）

# **3 原因解析**

因为工作台会对同一单据上的相同商品、不同单据上的相同商品的数量进行合并，多单位商品必须换算成基本单位后再去合并数量

电脑端可以通过编辑“本次拣验整件散包”字段解决

![上传图片](https://vip.kingdee.com/download/0100650c30e3c0da4f5494498d1cfa78c439.png)

移动端可以通过查看拣验工作台待拣页签的“已拣”数量、已拣页签的“已拣整件散包数”来解决。扫码拣验货时，也支持通过录入整件散包数量来进行拣验

![上传图片](https://vip.kingdee.com/download/01009fa00d29863242bc8b2944bfd6fec4f1.png)
