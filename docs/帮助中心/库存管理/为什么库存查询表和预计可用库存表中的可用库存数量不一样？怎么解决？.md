# 为什么库存查询表和预计可用库存表中的可用库存数量不一样？怎么解决？

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题, 快速入门
> 作者：Nanh | 阅读：2180 | 有用：15
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/22010)

# 【问题概述】

为什么库存查询表和预计可用库存表中的可用库存数量不一样？

# 【原因说明】

以下应用场景，库存查询表统计不到：

1、采购订单/销售订单中，没有填写“仓库”字段的所有商品的库存；

2、采购订单/销售订单中，填写的“仓库”启用了仓位，但没有填写“仓位”字段的所有商品的库存；

3、调拨出库单中，调入仓库信息填写的“仓库”启用了仓位，但没有填写“仓位”字段的所有商品的库存；

4、采购订单/销售订单中，没有填写“批次”的批次商品；

5、采购订单/销售订单中，没有填写“保质期”的保质期商品；

6、采购订单/销售订单中，没有填写“序列号”的序列号商品；

如果存在以上场景，就可能出现库存查询表和预计可用库存表中的可用库存数量不一样的情况。

针对批次、保质期、序列号、辅助属性商品订单不包含商品批次，保质期、序列号明细信息、辅助属性信息，可用库存不会扣减此部分商品的订单数量，以上场景若需要查看请到预计可用库存表查询。

# 【解决方案】

如果可用库存公式中勾选了订单相关的选项（采购在途量、、销售占用量），只有录入以上信息才能影响到可用库存。

# 【适用版本】

星辰3.1以上所有版本
