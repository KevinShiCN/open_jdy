# 预计可用库存表

## 1、简介

### 1.1 功能介绍

预计可用库存表是根据当前的过滤条件，展示筛选出的商品库存情况，包括即时库存、预计可用库存、待入库存和待出库存。

### 1.2 适用版本

金蝶云·星辰所有版本

### 1.3 系统路径

库存管理-仓存报表-预计可用库存表

### 1.4 字段/按钮/参数说明

[https://vip.kingdee.com/link/s/lSRAs](https://vip.kingdee.com/link/s/lSRAs)

## 2、报表查询说明

单击库存管理-仓存报表-预计可用库存表，进入预计可用库存表页面，选择商品，报表查询参数，查询时间，详情如下：

### 2.1 商品选择

商品资料条件筛选：根据表头的筛选条件得到需要统计的商品信息，如果没有设置筛选条件则等同于选择基础资料中的所有商品；

### 2.2 参数控制

①显示零库存商品：勾选时，将展示即时库存为0的商品数据

②显示禁用商品：勾选时，将展示状态为“禁用”的商品

③显示辅助属性：勾选时，将按辅助属性维度展示商品数据

④显示仓位：勾选时，将按仓位维度展示商品数据

⑤显示批次和保质期：勾选时，将按批次和保质期维度展示商品数据

⑥仅显示即时和可用库存：勾选时，表体库存库存数据仅显示即时库存和预计可用库存两个字段

⑦显示小计：即按商品维度或仓库维度进行小计，报表同时显示仓库+商品或商品的明细数据与汇总数据：勾选后报表不支持排序；

⑧仅显示小计：按商品维度或仓库维度进行小计后，仅显示小计行，不显示明细行：勾选后报表不支持排序；

⑨按商品统计：以商品为维度汇总统计每个商品在所有仓库下的预计可用库存；

⑩按仓库统计：以仓库+商品为维度汇总统计每个仓库下的不同商品的预计可用库存。

### 2.3 工具栏

![](https://vip.kingdee.com/download/01097bb7a7919814491081ac9eb37516956a.png)

- 引出：可以把库存查询结果按Excel格式引出，
- 打印：支持打印及预览。
- 列设置：用户可以自定义报表中的列显示以及顺序。
- 刷新：即刷新当前页面的结果。
- 组合排序（勾选“显示小计”后报表不支持排序）：可以对查询结果中的指定字段列按升序/降序排序。

## 3、名词解释：

预计可用库存：理论上可以使用的库存数量，根据系统参数的公式计算得出，可在系统参数-进销存参数-库存参数中设置

即时库存量：查询时间点的系统中在库库存数量，即仓库真实存在的库存

预计入库量：即会增加预计可用库存的数量，具体的统计范围及单据定义可见下图勾选框后的文字说明

预计出库量：即会减少预计可用库存的数量，具体的统计范围及单据定义可见下图勾选框后的文字说明

**预留量**：未审核发货单、未审核配货单、零售单、第三方占用
