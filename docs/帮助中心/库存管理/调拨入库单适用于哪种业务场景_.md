# 调拨入库单适用于哪种业务场景?

【问题描述】

调拨入库单适用于哪种业务场景?

【解释说明】

企业有多个仓库,可能会需要从一个仓库调拨商品到另一个仓库。在调出仓库调出库存的时候,如果做的是调拨出库单,那调入仓库收到货之后,就必须做调拨入库单。

调拨入库单新增无法手工选择商品,必须选择调拨出库单为源单带出。

调拨入库了调拨流程就完成了。
