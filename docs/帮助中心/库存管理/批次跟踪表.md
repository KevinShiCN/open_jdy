# 批次跟踪表

## 1、简介

### 1.1功能介绍

批次跟踪表：由于批次商品在管理上的特殊性，批次商品在业务流程中往往需要单独进行跟进。跟踪批次商品的全流程，包括入库（采购、盘盈、调拨、其他入库等）、出库（销售、盘亏、调拨、其他出库等）。

## 2、报表查询说明

### 2.1 商品选择

（1）商品资料条件筛选：如商品、商品类别、商品品牌等。根据表头的筛选条件得到需要统计的商品信息，如果没有设置筛选条件则等同于选择基础资料中的所有商品；

（2）支持按商品行备注查询

（3）支持按仓库/仓库分类/仓位查询

（4）批次筛选：支持按批次筛选，查询具体批次的商品信息

### 2.2 单据选择

（1）审核状态：可根据审核状态来筛选表体中需要参与统计的单据

（2）单据类型：可选择具体的单据类型来控制表体中需要参与统计的单据

（3）单据日期：可选择时间范围来筛选需要参与统计的单据

（4）单据关联的业务员、部门、制单人、审核人、最后修改人等

### 2.3 公共功能

支持引出、打印、列设置、组合排序

支持刷新、重置筛选条件、列设置、引出、打印等功能，列数据中的商品编码、批次、库存相关字段、日期相关字段可以直接点击表体进行升序或者降序排序。
