# 盘点单上可以添加/删除商品吗?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：云社区用户5j6u671 | 阅读：377 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/611726457332457216)

【问题描述】

盘点单上可以添加/删除商品吗?

【解释说明】

未审核的盘点单支持添加和删除商品行信息,若是系统中不存在的商品,支持在选择商品选项框界面点击左下角<新增>按钮快速新增商品。

【操作指引】

没有审核的盘点单可以添加和删除商品,鼠标移到盘点单最后一个商品左侧的操作栏会显示”-“和”+“号,点击”+“号增加行再选择商品添加,点击”-“号则是删除这一行商品

![](https://vip.kingdee.com/download/0109b0c2641724ad48a394d948e65282fd12.png)
如果误删除了商品后,可以点击“+”添加商品重新新增即可
如果已审核但是没有进行盈亏处理,可以反审核后操作

注意:只能在盘点单最后一个商品后面添加,不能在中间插入添加。
![](https://vip.kingdee.com/download/01098547fa5439ed461399f1eda9ea4d717a.png)
若是系统中不存在的商品,可以在选择商品选项框界面新增商品

![](https://vip.kingdee.com/download/010937f3a80ad6d44a3f83e62702afda5964.png)

如果下推了盘盈盘亏单的盘点单则不能添加商品,一定要添加的话,只能删除盘盈盘亏单,修改盘点单添加商品并保存审核以后再做盈亏处理。

![](https://vip.kingdee.com/download/01099ae0c7b9bcee4daa86f5397d03fc538f.png)
