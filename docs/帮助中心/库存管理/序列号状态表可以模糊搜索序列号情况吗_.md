# 序列号状态表可以模糊搜索序列号情况吗?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：Schatz | 阅读：121 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/624410094385690624)

【问题描述】

序列号状态表可以模糊搜索序列号情况吗?

【解释说明】

在序列号状态表界面,若直接于左上角设置序列号筛选条件,则支持模糊搜索功能;

![](https://vip.kingdee.com/download/0109951c4fbb1d894708a36bc2c207705722.png)

而通过点击进入批量查询并输入序列号信息进行查询时,则不支持模糊搜索。

![](https://vip.kingdee.com/download/010906bb5eebe15c4e0fbac1f538253e239b.png)

此外,还可利用表头处的序列号漏斗图标来设定模糊查询条件。

![](https://vip.kingdee.com/download/0109d3448615f0eb4bed974ab1474646f2a2.png)
