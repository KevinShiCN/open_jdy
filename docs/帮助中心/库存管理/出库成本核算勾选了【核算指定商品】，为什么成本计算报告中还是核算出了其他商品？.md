# 出库成本核算勾选了【核算指定商品】，为什么成本计算报告中还是核算出了其他商品？

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题, 进阶技巧
> 作者：Daisy_ | 阅读：1284 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/262612423008189952)

问题描述：

出库成本核算勾选了【核算指定商品】，为什么成本计算报告中还是核算出了其他商品？

解决方案：

![](https://vip.kingdee.com/download/01001ec3c509a3464cf18b3ae04dcae84cce.png)

要检查是否有组装拆卸业务或调拨业务。

1.一个商品有组装拆卸业务，在做成本核算时就要包含和它相关联的所有商品一起计算；

2.调拨业务是同商品不同仓库，所以做成本核算时包含同商品的所有仓库一起计算。
