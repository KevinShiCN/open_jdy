# 1. 简介

## 1.1. 功能介绍

商品实际库存小于系统中的账存数，可以直接录入盘亏来处理实际库存和账存数的差异。

![](https://vip.kingdee.com/download/0100a9a8e69efdd040eda7e161c5579d9719.png)

## 1.2. 适用版本

金蝶云·星辰所有版本

## 1.3. 应用场景

- **通过完整的盘点流程生成**：盘点单完成后，实际库存数小于系统账存数，在单据详情中，通过操作盈亏处理或通过盘点单列表的盈亏处理来创建盘亏单，系统会自动生成盘亏单（盘点数比账存数少）和盘盈单（盘点数比账存数多），实现账实一致的目的。
- **手工新增**：当盘点的商品数量较少时，可以直接得出盘点结果，若商品实际库存小于系统中的账存数，可以直接录入盘亏单来处理实际库存和账存数的差异，而不需要在系统中操作完整的盘点流程。

## 1.4. 系统路径

库存管理>仓存单据>盘亏单

# 2. 主要操作流程

## 2.1 单据录制

1) 单击库存管理-仓存单据-盘亏单，即可进入盘亏单操作界面

2) 输入本次盘亏的仓库，商品，数量等信息，点击保存；

3) 字段如下：

**表头：**

单据编号：可根据编码规则自动生成单据编号，由编码规则指定是否允许编辑

单据日期：默认为系统当天日期，可编辑

**表体：**

商品编码：手工录入或扫码录入

仓库：盘亏商品需要出库的仓库

单位：商品单位，默认显示商品资料中的库存单位，可编辑

盘亏数：盘亏数量

## 2.2 **出库成本计算**

盘亏单的最终结果是减少库存，属于出库类型，因此出库的商品成本需要通过【库存管理>存货核算>出库成本核算】来计算，计算完毕后，重新打开单据，出库成本就会反算到单据上。

操作流程：

1）录入盘盈单，审核。

2）打开库存管理>存货核算>出库成本核算，选择好核算的时间反馈，点击“开始核算”

3）核算完毕后，点击“完成”

4）重新在盘亏单列表打开单据，可以看到出库成本已经更新到单据上了。

字段取值说明：**单位成本=出库成本/盘亏数**
