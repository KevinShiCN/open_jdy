# 组装单费用已计入成本还需要形成支出单吗?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：Schatz | 阅读：99 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/659566765176605440)

【问题描述】

组装单有组装费用,已经分摊到成本,这一笔组装费用是否需要生成支出单?

【解释说明】

企业组装过程中会产生一些费用,直接在组装单上录入组装费用可以分摊到成本。一般这一笔费用会存在三种情况。第一种:需要支付,并且在做组装单之前没有做付款的单据;第二种:需要支付,但是在做组装单之前已经做了付款的单据;第三种:无需支付费用。

不同的情况,在组装单做费用的时候需要有不同的操作,第一种会需要组装费用生成其他支出单挂往来,以便后续支付费用;第二和第三种无需生成其他支出单。

在星辰软件中,组装费用是否要生成其他支出单可以通过费用录入时候的“是否待支出”按钮来控制。

![](https://vip.kingdee.com/download/0109236240cabd6446b681514b7babd72226.png)

【操作流程】

1、需要生成其他支出单挂往来:录入组装费用的时候,开启“是否待支出按钮”,

![](https://vip.kingdee.com/download/0109ea6267e96bf649399d5604a7b4a68270.png)注意:开启是否待支出,供应商必须录入,否则无法保存费用;

![](https://vip.kingdee.com/download/01094fca39ad9b8b401ab34b24e2ea229d86.png)等组装单审核之后,这一笔费用会直接生成其他支出单;

![](https://vip.kingdee.com/download/0109af543cbf32e54a2d8080def048c60c2a.png)
生成的其他支出单是未付款的状态,审核之后,会挂在供应商的应付上,需要后期付款核销。

![](https://vip.kingdee.com/download/01094d16c74ae93b4c7eb17a0717d08b443b.png)

2、无需生成其他支出单挂往来:录入组装费用的时候,不要开启“是否待支出”按钮;

![](https://vip.kingdee.com/download/010994f685b192894dd2a44283393227076f.png)
注意:是否待支出按钮未开启的状态下,供应商是非必录的,是否录入不影响。
等组装单审核之后,这一笔费用不会生成其他支出单,不影响往来数据。

![](https://vip.kingdee.com/download/0109489e603ed7284db6b7af58dd46976944.png)
