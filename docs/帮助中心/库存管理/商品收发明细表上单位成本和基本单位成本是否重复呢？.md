# 常见问题：商品收发明细表上单位成本和基本单位成本是否重复，有啥区别？![](https://vipfront.s3.cn-north-1.amazonaws.com.cn/emotion/define/64.gif)

![](https://vipfront.s3.cn-north-1.amazonaws.com.cn/emotion/define/93.gif)解析：（1）基本单位成本是商品在计算过程中累计核算得来的，一般与出库成本核算上的成本保持一致

                  （2）单位成本是反算得来的，并不是做单后，单据上的金额。

![](https://vipfront.s3.cn-north-1.amazonaws.com.cn/emotion/define/86.gif)区别：如下图

基本单位成本15.00248是成本核算得来的，而15.00200是由成本本位币除以数量反算得来的，至于成本本位币的金额则是由基本单位成本*数量后四舍五入保留2位小数得来的。
