# 其他入库单可以和什么单据关联?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：_Leo | 阅读：2188 | 有用：21
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/443577920892627200)

【问题描述】

 其他入库单可以和什么单据关联?

【解释说明】

其他入库单主要用于处理**购货业务以外**的商品入库,例如盘盈、借入、产成品入库等。在右上角可通过**选单**(销售订单、销售出库单、其他出库单、生产领料单)带出商品等信息快速开单;同时支持**下推生成其他出库单。**

【操作流程】

1、在其他入库单界面选单带出商品信息快速开单,支持选择销售订单、销售出库单、其他出库单、材料报废单

![](https://vip.kingdee.com/download/0109431737473b1441c1901bf43924eaa2a7.png)

2、其他入库单可以作为其他出库单的源单

![](https://vip.kingdee.com/download/01094832799eea0043ef92fa8fe3682edfce.png)
![](https://vip.kingdee.com/download/01097c51a4d205ce402fb2054e3b437bf84e.png)

【温馨提示】

**其他入库单可复制生成生产领料单,复制生成的单据跟源单没有关联关系。**

**其他入库单不支持关联生成日记账。**
