# 盘点单的状态显示盘亏处理中是什么意思?

【问题描述】
盘点单的状态显示盈亏处理中是什么意思?![](https://vip.kingdee.com/download/010974dab2776d114b8095cd3b22f5d54320.png)

【解释说明】

盈亏处理中表示所关联的盘盈单或盘亏单未审核,需要点击盘亏/盘盈单进入审核。

点开盘亏单/盘盈单 单号,进入点击审核
