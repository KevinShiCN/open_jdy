# 如何修改/删除分量盘点单?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：899 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/471481782467113728)

【业务场景】

多人盘点时已生成的分量盘点单可以修改,或者删除吗?怎么操作?

【解决方案】

分量盘点单支持修改或删除,依次点击【库存管理】-【盘点单】-【查询】打开盘点记录中点击“分量盘点单”。

![](https://vip.kingdee.com/download/010919cc0cae3b3444ddbc6793bd6984d112.png)

1、修改:选择需要修改的分量盘点单(未审核)双击打开即可修改。

![](https://vip.kingdee.com/download/010903b06e7de25e4f848063fc4ddb4d3928.png)

2、删除:选择需要删除的分量盘点单(未审核)点击“删除”即可。

![](https://vip.kingdee.com/download/01094e78fb0a235049289194cfbb062efb3c.png)

【温馨提示】

1、已审核的分量盘点单需要反审核后才能修改/删除,选择需要修改/删除的分量盘点单点击反审核,反审核成功后即可修改/删除。

![](https://vip.kingdee.com/download/010919289a91954d4551bcf4e9fd6b88394b.png)

2、已进行汇总的分量盘点单不能反审核,若需要反审核则需要先取消汇总后才能反审核修改/删除:打开对应的盘点单,点击取消汇总分量盘点,取消成功后即可反审核分量盘点进行修改/删除。

![](https://vip.kingdee.com/download/0109d667bdc8db0b45cea2ac27f2685308ad.png)
