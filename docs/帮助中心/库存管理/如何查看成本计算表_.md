# 如何查看成本计算表?

【问题描述】

如何查看成本计算表?

![](https://vip.kingdee.com/download/0109467b9e1b43e2497b835ca2aaaa867472.png)[视频观看](https://vip.kingdee.com/school/detail/737634674137963008?productLineId=35&lang=zh-CN&cosmic_type=video)

【解释说明】

系统完成出库成本核算任务后,将自动生成针对每个商品的成本计算表,详细记录所采用的存货计价方式、参与核算的相关业务单据,以及商品成本的计算步骤和最终结果,据此确保商品成本数据的准确性。

【操作流程】

1、如何查询成本计算表

①完成出库成本核算后,系统自动生成【成本计算表】,点击“查看成本计算报告”,即可查询到每一个商品的【成本计算表】。

![](https://vip.kingdee.com/download/010902ce5cc3d59f46bf8c0b3bd1af269b02.png)

②在【库存管理】-【出库核算结果查询】模块中,可查询到历史的核算报告。

![](https://vip.kingdee.com/download/0109973a705a2e5d4a05938d0c8d231830af.png)

点击【成本计算报告】即可查看参与本次核算的所有商品的成本计算表。如图:(**注意成本核算结果,仅保留最近10次执行结果详情,如需保留记录后续查询,建议每次核算后[引出核算结果](https://vip.kingdee.com/link/s/ZOA6p)**)

![](https://vip.kingdee.com/download/0109ed827111eee649af9053a7b66c7bda0d.png)

①基本操作

②成本计算表介绍

![](https://vip.kingdee.com/download/010957bf3c7615ae4224adad8139db6838f9.png)
入库列:入库类单据成本显示,采购入库单、其他入库单、盘盈单等入库类单据的成本是直接手工录入的,销售退货、组装单、移仓调拨单等是出库成本核算后计算出的

出库列:出库类单据成本显示,根据计价方法、入库列成本、结存列结存成本计算

结存列:期初余额为上期结存的数量及成本,中间多行结存合计均是按日期顺序显示,上一日期的结存+后续同一日期的入库-同一日期的出库=所计算日期的结存

如上图7月26日结存=7月23日结存+7月26日入库-7月26日出库

若发现结存不对,需检查是否期初结存或入库不对
