# 库存横向查询表

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：快速入门, 功能介绍
> 作者：吊梢眉 | 阅读：907 | 有用：2

## 1、简介

### 1.1 功能介绍

作为用户，想要在查询库存的时候按照横向展示仓库的样式看到仓库的库存情况，在有多个仓库的情况下，这样的展示会比较简单直观。【库存查询表】是按照仓库竖向展示的，【库存横向查询表】是按照仓库横向展示的，提供了不同的查询展示方式，满足不同客户的需求。

![](https://vip.kingdee.com/download/0100a34ef3a545ad43d581aa3460074f27e9.png)

### 1.2 适用版本

金蝶云·星辰所有版本

库存管理-仓存报表-库存横向查询表

## 2. 报表查询说明

1) 库存横向查询表最多支持展示20个仓库，当仓库数量大于20时需选择仓库类别或仓库进行查询。

2) 库存横向查询表的库存和库存查询表均一致，唯一最大的差异主要是在报表的显示上，横向查询表的仓库是横向展示的。
