# 单据未审核可以在报表中体现吗?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：酸梅汤啦啦 | 阅读：21 | 有用：0

【问题描述】

单据未审核可以在报表中体现吗?

【解释说明】

部分报表可体现出未审核的数据。若报表过滤条件中包含审核状态选择,即该报表可查询出未审核单据数据。

例如:商品收发明细表过滤条件可以选择审核状态“全部”、“未审核”、“已审核”,并且列设置可放开显示“审核状态”字段,可查询到单据的审核状态。
