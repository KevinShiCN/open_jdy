# 金蝶云星辰零售门店POS盘点场景问题Q&A及解决办法

问题描述：

金蝶云星辰零售门店POS盘点场景问题Q&A及解决办法

解决方案：

1、Q：在POS新增了盘点单，为什么在App上看不到？

A：点击App上的【同步】按钮同步一下数据。

2、Q：在POS新增了盘点单，点了审核后，变成了审核中状态，为啥审核按钮置灰不能再操作了？

A：盘点单审核后，可能会立即成功，变成审核成功状态；也可能会有一个审核过程，就会变成审核中状态，此时无法再操作审核，只能等待审核成功或失败。

3、Q：在POS新增盘点，不合并盘点单数据，传不到后台是什么原因？

A：盘点任务传输策略改成了异步传输，不是实时传输数据，可能会等待一点时间。

4、Q：多人盘点，为什么不能提交盘点任务？

A：先检查是否合并盘点数据，多人盘点需合并盘点数据才能提交盘点任务。提交审核后，在POS点击【云同步】，加快数据同步到后台。

5、Q：POS盘点如果需要扫一次码后手动填入数量，要怎么操作？

A：点击【扫码盘点】，把“自动模式”取消即可，手机端POS操作一致。

6、Q：POS盘点是否支持手持无线扫码枪移动盘点，扫码后自动上传商品盘点数量？

A：方案1—用手机POS扫码盘点（优点：便捷；缺点：目前扫码获取商品速度较慢，优化中；需购买POS数）；方案2—用PDA扫码盘点（优点：便捷；缺点：需要购买PDA设备，价格较昂贵；需购买POS数）；方案3—用无线扫码枪扫码盘点（优点：便捷，无需另外购买POS数；缺点：限10米范围内操作）

7、Q：POS盘点的下载上传是什么意思？

A：上传是指把原有盘点数据上传至后台；下载是指从后台下载数据到POS上，一般是在POS机更换后数据未同步时使用的，不是下载Excel或是上传盘点表使用的。
