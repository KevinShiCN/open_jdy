# 移仓-库存管理-金蝶云·星辰

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题, 快速入门
> 作者： | 阅读：4810 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/6164)

V2.0版本增加了投标管理应用，投标管理是招标管理对应的供应商端在线协同应用，包括投标报名，招标邀请，我的投标三个部分，实现了内部招标管理平台和供应商投标管理平台的无缝链接。基于投标中心的全业务集中式办理，使投标过程可追溯，业务处理更便捷。主要包括以下内容：

1、投标报名：查看所有招标公告信息，在截止报名时间前可进行报名处理。

![](https://vip.kingdee.com/download?fileName=0100e67865460d2644af884c0d651c773635.png)

2、招标邀请：查看处理邀请函，可接收邀请，也可拒绝邀请。

![](https://vip.kingdee.com/download?fileName=01005d37a2c645674303aa3a4cba4a71d02c.png)

3、我的投标：查看招标的基本信息，报名信息，下载招标文件，在线答疑，在线投标，澄清，查看中标结果

![](https://vip.kingdee.com/download?fileName=0100e5dab7dfa8704bedbff1d4eada449287.png)

招标管理应用中增加了在线投标相应的业务处理功能，具体如下

- 招标立项:发布招标公告到供应商门户功能

![](https://vip.kingdee.com/download?fileName=01005f891393db924b82a425555d30f8b35e.png)

   2.供方入围：公开招标时处理报名供应商入围业务，以及邀请招标时发送邀请函功能

![](https://vip.kingdee.com/download?fileName=0100ad1a035e224447a3927a942394a7edba.png)

   3.发标：点确认发标后发送标书到供应商门户

   4.开标:查看供应商回标信息以及开标后自动显示供应商投标数据功能

![](https://vip.kingdee.com/download?fileName=0100299fda077f354c6f8a64c0d0edb22941.png)

   5.评标：评标专家或者经办人对投标信息进行质疑，供应商对此进行澄清

![](https://vip.kingdee.com/download?fileName=010034e318332fed4b3d8bf1d48b77c20720.png)

   6.定标：发送中标公告，感谢信，中标通知书至供应商门户

![](https://vip.kingdee.com/download?fileName=0100ce4afc536e144e879840d39243f32c15.png)

   7.消息推送功能：系统支持分别在邀请函发布，入围审批，标书发标，确认开标，定标发送感谢信和中标通知时自动发送消息给对应供应商![](https://vip.kingdee.com/download?fileName=0100bb2c7e3e637e4ecd8fdb6c6194f7175c.png)
