# 组装拆卸模板“按子件采购”应用场景是什么？如何操作？

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：应用案例, 快速入门
> 作者：进销存赵嘉丽 | 阅读：1530 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/152731293094928640)

# 应用背景

组装拆卸模板，如何操作“按子件采购”？如何生成采购申请单或采购订单？

# 应用方案

当组装成品时，不确定组装所需的原材料库存是否足够，此时可以在组装拆卸模版中通过“子件采购”计算所需子件数量，缺料数量，并即时采购。

# 应用流程

1、在组装拆卸模板列表中，选择已审核的组装模板，点击“子件采购”。

![上传图片](https://vip.kingdee.com/download/010075fa0331a590414c8fc146d79624f16a.png)

2、输入组件数量。

可手工输入需采购的组件数量

![上传图片](https://vip.kingdee.com/download/010068870f51165449ff9c1ad13abec9acec.png)

若是根据销售订单采购，则在第一步中，选中模板后，点击“选择销售订单”，在弹出的销售订单界面，勾选对应的销售订单，并点击确定，将自动计算所需组件数量，无需手动录入数量。

![上传图片](https://vip.kingdee.com/download/0100a16966a11a56441fb043903161d54855.png)

![上传图片](https://vip.kingdee.com/download/01001186401e224f4b3e9a26996ba4308f3b.png)

3、计算子件数量。

录入完组件数量后，单击“计算子件数量”，系统会根据选择的模版组件子件数量关系，分别计算出不同组件需要多少子件。

4、子件数量汇总。

子件数量计算完后，确认好子件仓库，单击“子件数量汇总”。因为不同的组件可能需要用到相同的子件，需要将相同的子件合并，才是真实需要采购的数量。

![上传图片](https://vip.kingdee.com/download/010051a442b0b000434a8062dd730f31e6b2.png)

得出最终待采购数量。

![上传图片](https://vip.kingdee.com/download/0100ebda7a4cf7be47c3a3c80a6d8211e6b3.png)

5、下推生成采购申请单（标准版不支持）或采购订单。

缺料数量=子件数量-可用库存，当库存足够时，缺料数量和待采购数量为0，不需要采购，也可以手工输入待采购数量生成采购订单。

选中对应的子件分录，录入供应商、采购业务员后，单击“生成采购采购申请单”或“生成采购订单”，即可生成对应的单据。

单击单据编号可以跳转到单据详情进行编辑，点击“完成”，退出子件采购。

# 适用版本

星辰3.1以上所有版本
