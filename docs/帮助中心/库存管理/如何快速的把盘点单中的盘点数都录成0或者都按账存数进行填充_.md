# 如何快速的把盘点单中的盘点数都录成0或者都按账存数进行填充?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：670 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352847398152448)

【问题描述】

如何快速的把盘点单中的盘点数都录成0或者都按账存数进行填充?

【解决方案】

可以通过盘点单上“盘点数”的批量填充按钮来实现。

![](https://vip.kingdee.com/download/010927ed174506b14885b3c573317a6406ac.png)【温馨提示】

1、若每个商品的盘点数不一样,无法使用批量填充功能时建议通过导入Excel的方式批量引入,具体操作可参考:[如何导入盘点单?](https://vip.kingdee.com/knowledge/471481773591733248?productLineId=35&isKnowledge=2&lang=zh-CN)
