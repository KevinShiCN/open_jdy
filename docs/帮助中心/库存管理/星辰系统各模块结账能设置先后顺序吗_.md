# 星辰系统各模块结账能设置先后顺序吗?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：ylim | 阅读：469 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/569326669706629888)

【问题描述】

星辰系统各模块结账能设置先后顺序吗?

【解释说明】

**财务模块和进销存模块**结账先后顺序取决于参数设置,

在【系统设置】-【系统参数】-【存货核算参数】-【其他设置】可以选择:

**业务系统单独结账,且总账无关联:**财务模块和进销存模块分别单独结账,没有先后顺序

**业务系统单独结账,且必须早于总账结账:**则总账结账前业务需要要先结账

建议是先进销存再财务,先锁定进销存的数据,确定财务的没问题了再结财务锁定财务的数据,防止两边不一致;

![](https://vip.kingdee.com/download/0109ba0db19389074824aef735b05233be1c.png)

**进销存模块、出纳模块、资产模块**之间结账也没有先后顺序之分;

**财务模块和出纳模块**如果参数勾选了【与总账一起结账】,则无需结账出纳,直接结账财务模块即可,财务模块结账了,出纳就一起结了;不勾选的情况下没有先后顺序

![](https://vip.kingdee.com/download/0109de7ca43f54c7454f84cfb6a866f843d7.png)
**财务模块和资产模块**如果勾选了【与总账一起结账】的参数,则无需单独结账,直接结账财务模块即可

![](https://vip.kingdee.com/download/010936d30d30a87647b180420d4321ba9c2d.png)
如果勾选了【与总账连用】,但是没有勾选【与总账一起结账】,需要先结账资产,再结账财务;如果都没有勾选,则不分先后顺序。

![](https://vip.kingdee.com/download/010915f31143a8f8428c9538e5cc9633d35a.png)
