# 如何导出盘点单上的商品明细?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：2894 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/395742744813038848)

【业务场景】

如何导出盘点单上的商品明细?

【操作步骤】

首先,在【库存管理】-【盘点单】界面,点击创建盘点任务,如图:

![](https://vip.kingdee.com/download/0109040a08f803f749f39779ab07e20d768a.png)

点击创建盘点任务,并勾选显示商品清单,如图:![](https://vip.kingdee.com/download/01099ed05675d3bc40658d3d59c467bb24d3.png)
根据需要选择需要盘点的仓库信息,以及其他筛选条件,并勾选“显示商品清单”

![](https://vip.kingdee.com/download/0109850305d02e7842cdab812ff433f76e58.png)然后,到盘点列表上勾选对应盘点单,选择按引入方式

![](https://vip.kingdee.com/download/010950a89c51e57346d5b16460ab91b82066.png)
引入模版可自定义设置哪些字段需要导出:

点击盘点单列表右上角的【引入】按钮,在模版管理里面新增或者复制模版编辑,设置完成可按新模版导出和导入。

![](https://vip.kingdee.com/download/0109d33b7a7e92124728b887349a51376c80.png)

![](https://vip.kingdee.com/download/010970e4d24221334a089cb45ebff07e2d38.png)

![](https://vip.kingdee.com/download/01092265af3fee734301bc0c77fead091732.png)
