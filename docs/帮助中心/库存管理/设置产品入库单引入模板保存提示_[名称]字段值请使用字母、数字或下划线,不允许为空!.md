# 设置产品入库单引入模板保存提示:[名称]字段值请使用字母、数字或下划线,不允许为空!

> 作者：杨桂平 | 阅读：116 | 有用：0

【问题描述】

设置产品入库单引入模板保存提示:【名称】字段值请使用字母、数字或下划线,不允许为空!

【解释说明】

需要设置好模板的名称字段(字母、数字或下划线),不能为空。录入保存即可。
