# 组装单上选择销售订单后,销售订单全流程查不到组装单,组装单全流程也查不到销售订单,这是为什么呢?

> 作者：winnie_wen | 阅读：813 | 有用：1

【问题描述】

组装单上选择销售订单后,销售订单全流程查不到组装单,组装单全流程也查不到销售订单,是什么原因?

![](https://vip.kingdee.com/download/0109b2838cadad5f406091415659c33fc432.png)

![](https://vip.kingdee.com/download/01095c0e50c35fbd4d189b82402a575be57b.png)![](https://vip.kingdee.com/download/0109d56506e4e4364261a01e4ee960c7e894.png)

【解释说明】

因为组装单上选销售订单只是用于计算子件,实际并不产生关联关系,全流程跟踪的依据是单据间要有关联关系,而星辰的关联关系是指修改本单后,对上游单据有反写和有影响的单据关系,但是组装单和销售订单不存在这样的关系,只是作为一个计算的参考依据,因此全流程跟踪里不会显示。

类似的场景还有销售出库单上生成的费用相关的其他支出单,销售出库单跟其他支出单之间也没有关联关系,删除销售出库单也不会影响这笔其他支出单。
