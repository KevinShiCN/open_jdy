# 核算单据查询分录行有限制吗？为什么有些单据未能显示？

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：快速入门, 常见问题
> 作者：Daisy_ | 阅读：554 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/262592649465873408)

问题描述：

通过核算单据查询，发现一些单据汇总查不到，但是单独搜索单据编号确查到，这是什么原因导致的？

解决方案：

目前星辰系统限制最多只能显示3000条分录，超过3000条是无法显示或者导出的，所以会出现汇总超过3000条无法显示，但是单独查询可以正常搜索到的情况。
