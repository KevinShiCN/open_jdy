# 移动平均法下做成本调整单对成本有什么影响?

移动平均法下做成本调整单对成本有什么影响?

【解释说明】

#### 移动平均法下做成本调整单会影响商品的结存成本以及后期的出库成本。

1、成本调整单业务类型为“出库调整”时

成本调整单中业务类型选择为**出库调整**,调整金额录入**正数**时会**减少**该商品的结存成本。

如:商品A商品截止到1月15号数量100,单位成本10.5,总成本为1050;1月23日的出库成本为10.5;
![](https://vip.kingdee.com/download/010962623a8649a340b0a23b6ce57bb2e30f.png)
做成本调整单“出库调整”50,那么总成本更新为1000,单位成本更新为10【(1050-50)/100】,重新核算成本后23号单据的出库单位成本更新为10。

![](https://vip.kingdee.com/download/0109de958a97856446ea931ed66a2c72276d.png)

2、成本调整单业务类型为“入库调整”时

成本调整单中业务类型选择为**入库调整**,调整金额录入**正数**时会**增加**该商品的结存成本。

沿用上述案例:商品A商品截止到1月15号数量100,单位成本10.5,总成本为1050;1月23日的出库成本为10.5;若将成本调整单改为“入库调整”,调整金额录入为150,那么总成本更新为1200,单位成本更新为12【(1050+150)/100】,重新核算成本后23号单据的出库单位成本更新为12。
