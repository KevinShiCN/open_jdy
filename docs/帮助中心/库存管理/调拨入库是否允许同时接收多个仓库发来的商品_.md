# 调拨入库是否允许同时接收多个仓库发来的商品?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：Simone邓 | 阅读：189 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/542872297988499712)

【问题描述】

调拨 单是否允许同时接收多个仓库发来的商品?

【解释说明】

允许。

一张调拨出库单允许同一调出仓库向不同调入仓库发货;一张调拨入库单允许接收不同的调出仓库发过来的商品进行入库,即可以批量选择发给本仓库的调拨出库单。

举例1:A仓库存分别给B仓库和C仓库调拨一批商品,在一张调拨单上选择不同的调出仓库,B仓库和C仓库分别在调拨入库单列表入库即可。

![](https://vip.kingdee.com/download/01098ca34658b0964a7481f727cf23f14cab.png)

举例2:如分别从A仓库和B仓库调拨到C仓,C仓库在调拨出库单列表选择两张调拨单入库合并调拨入库即可。
