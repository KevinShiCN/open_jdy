# 库存预警查询表会显示零库存商品吗

【问题描述】

库存预警查询表会显示零库存商品吗

【解释说明】

**发生过业务的商品,目前库存为零库存**的,库存预警查询表里会有对应的记录。

1、如下图所示,两个商品SP00096与SP00097库存预警信息一致,安全库存数量5,最低库存数量1,最高库存数量50![](https://vip.kingdee.com/download/0109417d351c30224b8293068defba35ce2c.png)
2、SP00096商品有发生过业务,采购数量100并销售出库100![](https://vip.kingdee.com/download/01091106c5af6f624697ab70e75e2e1e3e25.png)
3、此时库存查询查看,两个商品即时库存为0![](https://vip.kingdee.com/download/01090837344a7d70481c8935a5768ef80a24.png)
4、查看库存预警表,只有SP00096商品,发生过业务且库存为0商品会提示预警信息。效果如下图所示
