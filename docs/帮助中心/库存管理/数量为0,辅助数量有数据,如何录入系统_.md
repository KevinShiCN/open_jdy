# 数量为0,辅助数量有数据,如何录入系统?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：126 | 有用：0

【问题描述】

数量为0,辅助数量有数据,如何录入系统?

【解释说明】

在正常业务开单过程中(包括采购、销售、其他出库、其他入库等),商品数量为必填项,不允许将数量录入0,辅助数量还有数据的单据保存。

如存在数量为0,辅助数量还有数据的情况,建议可以通过盘点方式变通处理。

**此情况仅支持浮动换算的商品,普通固定多单位商品不支持。**

【操作指引】

菜单栏点击【库存管理】>【盘点】

![](https://vip.kingdee.com/download/0109cf6a3785796948c2909466c4d495dafb.png)

可以按需求创建盘点单,详细可以点击:[库存盘点如何操作?](https://vip.kingdee.com/link/s/Zcmip)

在盘点单界面,支持手工修改盘点数和辅助盘点数,可以将盘点数录入0,辅助盘点录入对应的数字

![](https://vip.kingdee.com/download/01098e069688f71f4358a6c2435ee3cf70ea.png)
进行盈亏处理,盘盈数这里为空,辅助盘盈数显示有数据
