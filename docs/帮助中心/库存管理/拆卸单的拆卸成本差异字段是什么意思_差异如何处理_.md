# 拆卸单的拆卸成本差异字段是什么意思?差异如何处理?

【问题描述】

拆卸单的拆卸成本差异字段是什么意思?差异如何处理?

【解释说明】

该字段用于记录: 子件的成本合计与组件的成本的差异。

例如:当系统参数-拆卸成本的取数规则按参数中指定的成本来源取值,或者是手工录入子件成本,可能会造成子件与组件成本存在差异。一般建议使用根据比例分摊

拆卸单子件成本如何计算,详细可以点击:[拆卸单子件成本如何计算?](https://vip.kingdee.com/link/s/ZzKsH)

拆卸单列表可以点击列设置显示<拆卸差异成本>字段,**出现拆卸成本差异,无需特殊处理,仅供查看!**
