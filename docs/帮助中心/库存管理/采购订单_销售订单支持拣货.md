# 采购订单/销售订单支持拣货

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：整体介绍, 精华汇总, 进阶技巧
> 作者：Freya_peng | 阅读：971 | 有用：4

# 1 适用版本

标准版、专业版、旗舰版

拣货验货模块为收费模块，2000/年

# 2 应用场景

## 2.1 一单一拣

销售员下销售订单，订单审核，仓管员根据订单去仓库拣货，按照订单的商品顺序来拣货，拣好货之后放用在备货区，确认无问题后生成出库单，出库单审核后给到司机装车配送。

## 2.2 汇总拣货

业务分成商超、小门店，针对小门店，一单可能就几百块钱，门店不需要按单配货，内勤在电脑上将这些门店已审核未拣货的的销售订单下推汇总的拣货单，然后仓管去仓库拣货，拣货完后把货全部放在备货区，完成拣货，然后这个拣货单关联的订单上都更新拣货状态。 然后仓管再自己分拣，分拣后给司机装车出货

# 3 操作步骤

下文以销售订单为例，采购订单同理

## 3.1 拣验入口

### 3.1.1 拣验货单选源单去拣验

库存管理>仓存拣验货>拣验货单，新建拣验货单，选源单后保存拣验货单

![](https://vip.kingdee.com/download/0100b2a4e4b36fbf463c8300d41bf0d04916.png)

选择源单后，可分配拣货员，确认拣货仓库

![](https://vip.kingdee.com/download/0100fb052c48ce5b4124b98a6840217aa90c.png)

点击“去拣验”跳转至拣验工作台进行拣货

![](https://vip.kingdee.com/download/0100198e7e5656bc48159405d374bcce8215.png)

### 3.1.2 单据下推拣验

在销售订单/采购订单的列表或详情页，点击“生成拣验货单”跳转至拣验货单详情页确认和编辑相关待拣信息，也可以点击“去拣验”按钮直接跳转到拣验货工作台去拣验

![](https://vip.kingdee.com/download/01098d9e01f31fe845b4984c46710933bc27.png)

### 3.1.3 拣验工作台拣验

点击拣验工作台，支持输入拣验货单单号和源单单号拣货。输入源单编号的前提条件是源单已经下推了待拣验的拣验货单

![](https://vip.kingdee.com/download/0100cb43f766ad514e19824890d83fba279a.png)

## 3.2 进行拣货

输入条码拣货

![](https://vip.kingdee.com/download/01006a12d8dd291c4523a1f43359515c68bb.png)

支持的码：商品条形码、SKU条形码、批次复合码、序列号码

![](https://vip.kingdee.com/download/0100ee0937ac062f4924bc2aa7818d43c8fa.png)

拣验工作台操作说明

- 可在表头选择仓库，后续扫码商品都从改仓库出货
- 可手动开启赠品按钮，将赠品按照单独的批次出货
- 可切换超出、部分页签查看当前已拣商品的超出和不足情况

货物全部拣完后，点击页面右上角“拣验完成”按钮，跳转至拣验货单详情，并支持在拣验货单详情页一键下推出库

![](https://vip.kingdee.com/download/010084cce7dd7fab4a96bebfc74131bc2062.png)

## 3.3 可拣验数量

- 订单行可下推拣货数量=订单数量-已执行数量-行已拣验未执行数量
- 同一个订单只能同时存在一个未拣验或者拣验中的拣验货单，避免重复下推，拣完之后再继续下推，如需修改数量去修改拣验货单的待拣数量
- 订单-预留单-拣验货单，如果预留单预留数量为0，则会导致下推到拣验货单待拣验数量为0

## 3.4 权限控制

- 拣验工作台：通过库存管理-拣验货单-修改的功能权限控制
- 销售/采购订单列表/详情页、拣验货单列表/详情页的去拣验按钮：通过库存管理-拣验货单-新增+修改的功能权限控制，缺一不可
- 拣验货单新增/修改/删除/拣验完成/反审核：通过库存管理-拣验货单-新增/修改/查询/审核/反审核的功能权限控制
- 点击“拣验完成”会按照是否允许部分拣验和超额出库的参数校验是否可以拣验完成，拣验完成即审核

##  3.5 其他说明

- 拣验货单支持全流程跟踪，在拣验货单详情界面支持关联单据信息联查，支持上查源单（销售订单/采购订单），支持下查单据（销售出库/采购入库）
- 拣验货单支持单据列设置
- 拣验货单支持辅助属性打印，显示的是待拣数量
- 拣验货单支持单据云打印、字段配置，单据操作控制，自定义字段、消息设置等
