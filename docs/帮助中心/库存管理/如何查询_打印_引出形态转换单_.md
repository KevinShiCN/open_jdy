# 如何查询/打印/引出形态转换单?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：ylim | 阅读：26 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/797268862554139392)

【问题描述】

如何查询/打印/引出形态转换单?

【适用版本】

星辰3.0以上所有版本

【操作指引】

形态转换单打印、引出为2025年12月新功能,如需体验,请填写:**[问卷](https://survey.kingdee.com/vm/Pk2Pfhz.aspx)** 申请 。以下对查询、打印、引出分别说明

**1、查询形态转换单**

点击【库存管理】>【仓存单据】>【形态转换单l查询】

![](https://vip.kingdee.com/download/0109337c618d17184da5ab147dc5d245a1f7.png)
默认显示本月的形态转换单,点击本月的“x”删掉条件后可查询所有形态转换单,也可点击<展开过滤>显示出更多过滤条件

![](https://vip.kingdee.com/download/0109f805888f59664d68850be95a3d97e345.png)
查询范围支持选择“转换前商品分录”和“转换后商品分录”,选择“转换后商品分录”可将转换前和转换后商品都显示出来

![](https://vip.kingdee.com/download/0109b20476abe4fb4bd18b6505e498957244.png)
支持按“单据日期”“审核状态”“业务类型”“单据来源”查询

![](https://vip.kingdee.com/download/0109534bb309865b419d99659088576680b3.png)
点击做左上角搜索框可以按单据编号、商品编码、商品名称等搜索,也可点击<列表搜索设置>设置搜索条件

![](https://vip.kingdee.com/download/0109eae73e8150ca4230a78230883d97d75a.png)
也可鼠标放对应列字段名称上点击小漏斗对字段进行过滤搜索

![](https://vip.kingdee.com/download/01096596b2d73fa640a69b6af007325868a8.png)
点击常用过滤条件下的<新增方案>可以设置常用过滤方案,具体可参考:[凭证、单据、报表如何设置常用的查询方案?](https://vip.kingdee.com/link/s/ZXs9L)

![](https://vip.kingdee.com/download/01099dd454d6d398419eb976d9535cf472a7.png)
**2、打印形态转换单**

查询后可勾选要打印的单据,点击“打印”按钮下的<打印设置>,选择打印模板,点击<确定并打印>

![](https://vip.kingdee.com/download/010916ecbfaf8b064d80b7bf1c5812f7566d.png)
**3、引出形态转换单**

查询后也可将形态转换单引出,支持直接按列表引出、按引入模板引出、引出分析数据,三种引出方式详见:[单据如何导出?三种引出方式有什么区别?](https://vip.kingdee.com/link/s/ZXiME)

![](https://vip.kingdee.com/download/0109d69fe4f7411c4581a437bdebe4e4496a.png)
也可以按打印模板引出表格,勾选单据后,点击“打印”下的<导出为Excel>

![](https://vip.kingdee.com/download/01093d014b3d927443f1850b9daeaa03d7be.png)
