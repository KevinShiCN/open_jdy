# 如何打印盘盈单/盘亏单?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：ZOELI | 阅读：64 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/693264358067031040)

【问题描述】

如何打印盘盈单/盘亏单?

【操作流程】

点击【库存管理】>【盘盈单】/【盘亏单】查询,打开盘盈单/盘亏单记录。

![](https://vip.kingdee.com/download/0109f23034183c334dcea6ecf8d32c214af2.png)

点击打印下的<打印设置>,设置打印模板,点击<确定>(首次打印或者需要换模板时需设置)

![](https://vip.kingdee.com/download/01094d2a250e650043d4826db6eedbd5e5d7.png)

![](https://vip.kingdee.com/download/010933759ef916584e73b9ccc5237b9f22c4.png)

勾选要打印的盘盈单/盘亏单,点击【打印】按钮打印单据。

![](https://vip.kingdee.com/download/0109ee0cfd7b1f7f4da6914d1ce09493d992.png)
