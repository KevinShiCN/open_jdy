# 只需要调整库存数量金额不变如何处理?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：嘿唬哈 | 阅读：88 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/771906013254378496)

【问题描述】

只需要调整库存数量金额不变如何处理?

【概述】

只调整数量不调整金额,可以做其他出入库单,直接填写数量,成本为0保存即可。

【相关截图】

如果需增加库存,则录入其他入库单,只录数量

![](https://vip.kingdee.com/download/01096a96bf4bc1974279aaf0bfa8debc59fb.png)

如需减少库存,则录入其他出库单;

![](https://vip.kingdee.com/download/0109bbe711a1159848689c5e4fc2ba4f04f4.png)
注:如果出库成本核算后其他出库单上有成本,可以通过核算单据查询将成本清空为0后保存,保存后不需再做出库成本核算。

![](https://vip.kingdee.com/download/0109668b893775dd418ea02149944445553e.png)
