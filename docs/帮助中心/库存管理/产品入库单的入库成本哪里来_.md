# 产品入库单的入库成本哪里来?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：winnie_wen | 阅读：3380 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/569326626337729792)

【问题描述】

产品入库单的入库成本哪里来?

【解释说明】

1、若未开通生产管理模块,可以直接在【库存管理】-【产品入库单】进入

![](https://vip.kingdee.com/download/010958700d9714974324b80db29df5e95d11.png)

①无源单的产品入库单,单位成本和入库成本允许手工录入,入库成本=数量*单位成本

![](https://vip.kingdee.com/download/0109df004ca5070642bab85efd69b713d962.png)

②若是选源单,按销售订单生成产品入库单,入库成本无法手工输,可通过【库存管理】-【存货核算】的自制入库核算功能进行处理。

![](https://vip.kingdee.com/download/0109c15529abca4742b5bfcf18796d10ae38.png)

具体操作步骤可参考:[自制入库核算](https://vip.kingdee.com/link/s/lRDa0)

2、若已开通生产管理模块,除了可以按步骤1,通过库存管理进入,还可以在【生产管理】-【产品入库单】进入

- ![](https://vip.kingdee.com/download/010911fcac1c812043c6960195458167b9ec.png)
- ①若是无源单的产品入库单,单位成本和入库成本允许手工录入,入库成本=数量*单位成本。
- ②若是选源单-销售订单生成的产品入库单,入库成本无法手工输,可通过【库存管理】-【存货核算】的自制入库核算功能进行处理。
- 以上步骤可以参考【若未开通生产管理模块】操作步骤。
- ②若是通过生产任务单生成的产品入库单,单位成本和入库成本不允许手工录入,需要通过生产成本核算得到,入库成本=直接材料成本+费用。
- ![](https://vip.kingdee.com/download/0109a3f5e058517b45a390c4aff2f80fb9f6.png)费用为费用归集后分配核算的人工费、折旧费等
- 欢迎查看:[生产产品入库费用归集分配说明](https://vip.kingdee.com/knowledge/359821304159577344?productLineId=35&isKnowledge=2&lang=zh-CN)![](https://vip.kingdee.com/download/010997c450be1fc3491ab4c0f78e6367a084.png)
- 可以根据业务情况选择录入产品入库成本。
- 有源单部分入库核算成本:
- 1、有源单生产任务单部分入库按完工与在制品归集设置的约当系数来核算入库成本
- 2、当期源单生成任务单关闭的部分入库,会把涉及该生产任务单的成本一起计入当期入库核算带出
- 3、之前已部分入库当期未有入库的生产任务单关闭如有期初在制品成本,这部分成本会丢失,因以前期间入库的成本已核算无法修改,当期也没有入库,关闭的生产任务单以后期间也无法核算到
