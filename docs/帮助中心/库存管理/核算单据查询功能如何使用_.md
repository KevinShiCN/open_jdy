# 核算单据查询功能如何使用?

【问题描述】

核算单据查询功能如何使用?

![](https://vip.kingdee.com/download/0109a14cb515d2b148aca3c95afdc07f1f9d.png)[视频观看](https://vip.kingdee.com/school/detail/743895581528772864?productLineId=35&lang=zh-CN&cosmic_type=video)

【业务场景】

因业务处理中的各种特殊情况而无法计算成本产生的0成本单据,或者需要手动录入出库成本,例如:

1)企业收到使用软件前的退货,除该笔退货外,再无其他业务发生;

2)企业月底进行盘点,盘盈了部份商品,无法查明原因;

3)其他入库业务,无法提供精确的成本等

为保证业务能正常在软件中运行,系统提供无成本单据查询及处理相关功能。

【操作流程】

依次点击【库存管理】-【核算单据查询】

![](https://vip.kingdee.com/download/010913e088be2d8e4f08bd180a5805578e79.png)

进入核算单据查询界面,如下图所示:

![](https://vip.kingdee.com/download/01093e417f84e03341b289a2746e6c245717.png)

本单据成本录入方式有以下两种:

1、支持手工在成本和单位成本处录入对应成本数据:录入成本后点击右上角保存即可;

![](https://vip.kingdee.com/download/01094cbbec40e6e74eb4a88e3124a50730d2.png)

2、通过点击右上角<更新>按钮,可以选择的更新方式有“最近一次出库成本”、“最近一次入库成本”、“参考成本”、“期初加权平均价”,选择其中一种更新方式进行成本更新;

商品参考成本设置,可点击:[商品的参考成本在哪里设置?](https://vip.kingdee.com/link/s/ZUYwl)

![](https://vip.kingdee.com/download/0109da40217259be4f31be6ae69e58157cc4.png)

3、若需单独查看某个单据类型,可在单据类型处勾选过滤

![](https://vip.kingdee.com/download/0109234f63acd6574d40b9d1195eb2479b4d.png)
4、可以选择需要的数据,点击<引出>,通过引出的方式导出,填写完数据后,点击<引入>,引入完成后,确认数据无误,点击<保存>,即可批量进行更新成本数据

![](https://vip.kingdee.com/download/01092c4273616fcd4459b721588b2cd5c8e7.png)

【温馨提示】

**1.目前星辰系统限制最多只能显示3000条分录,超过3000条是无法显示或者导出的,所以会出现汇总超过3000条无法显示,但是单独查询可以正常搜索到的情况,建议将核算期间缩小后再进行调整,直到可以显示3000条以内的数据;**

**2.销售出库,销售退货,生产领料等类型的单据都可以在核算单据查询模块中手动录入成本;**

**3.手动录入完成本后,建议将手动录入成本的单据生成凭证,锁定单据成本,或者直接结账(没有买财务模块或者不使用财务模块的,推荐直接结账)。否则如果填写完成本后,重新进行出库成本核算,系统会将手动录入的出库类单据的成本自动清零,并按照成本核算方式重新计算。**
