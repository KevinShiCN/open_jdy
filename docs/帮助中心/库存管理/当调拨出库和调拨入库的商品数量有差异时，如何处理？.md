# 当调拨出库和调拨入库的商品数量有差异时，如何处理？

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题, 快速入门
> 作者：赵光芬 | 阅读：6108 | 有用：24
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/62856348362684160)

精斗云支持异地调拨业务，由调出方建立《调拨出库》单发出商品，调入方建立《调拨入库》单接收商品入库，在这个过程中，存在以下几种情况：

1、 调出方发出的商品与调入方接收的商品完全一致；

2、 调入方多收；

3、  调入方少收；

4、  商品在运输过程中损坏。

对于第1种情况，不需要做额外的处理，对于第2、3、4种情况，为了保证系统中的账面库存与实际库存相符，调出方的库存与调入方的库存在调出、调入的数据流转中相互匹配，需要在系统中进行调拨差异处理，我们可以在《调拨差异处理表》中查询调拨业务中产生的差异数据，并根据实际业务进行差异处理。

![](https://vip.kingdee.com/download/0100161cc9beb977484aa17d59c361c68b8d.png)

**场景一，调入方多收。**

比如，A仓库（门店）向B仓库（门店）调出商品100件，B仓库收到了101件，多收了1件时，解决方案：

多收的1件商品，需要双方沟通确认，是否需要将商品还回调出方。

1）若商品不需要还回A仓库，那么，建议B仓库在调拨入库单中录入实际到货的101件商品，形成调拨差异，A仓库通过调拨差异处理表，生成其他出库单，将多发的1件商品出库。

2）若商品需要还回A仓库，那么，建议B仓库在调拨入库单中录入实际到货的101件商品，形成调拨差异，B仓库通过调拨差异处理表，生成其他出库单，将多收的1件商品出库，发还给A仓库，A仓库收到货后，通过其他入库单，将商品入库。

3）若商品需要还回A仓库，调拨差异也可以在系统外处理，B仓库在调拨入库单中录入接收的100件商品，多的1件商品不作入库，系统外处理退回。

**场景二，调入方少收。**

比如，A仓库（门店）向B仓库（门店）调出商品100件，B仓库收到了99件，少收了1件时，解决方案：

少收的1件商品，需要双方沟通确认，是在路上丢失或者报废，还是A仓库少发了这1件商品。

1）若是A仓库少发了1件商品，A仓库会补发，那么，建议B仓库按照实际和99件商品录入调拨入库单，并等待下次补发的1件商品到货后，再录入一张调拨入库单完成本次调拨。

2）若是A仓库少发了1件商品，且A仓库不会补发，那么，建议A仓库反审核原调拨出库单，修改数量为实际发货的99件，审核后，B仓库再录入调拨入库单，录入实际到货的99件商品，完成入库。

     也可以A仓库不修改原调拨出库单，B仓库录入实际到货的99件商品后，形成调拨差异，在《调拨差异处理表》中，将A仓库少发的一件商品生成《其他入库单》重新入库回A仓库。

3）若双方确认这件商品丢失，那么，需要判断是谁的责任，若是由A仓库负责处理，那么，建议A仓库反审核原调拨出库单，修改数量为实际发货的99件，审核后，B仓库再录入调拨入库单，录入实际到货的99件商品，完成入库；若是由B仓库负责处理，那么，建议B仓库在调拨入库单中录入数量100件，再做其他出库单，将丢失的1件商品进行出库。

**场景三，商品在运输过程中损坏。**

A仓库（门店）向B仓库（门店）调出商品100件，B仓库收到了100件，但其中1件商品损坏时，解决方案：

双方需要确认损坏的商品由哪方承担。

1）若是由A仓库承担且损坏的商品不需要B仓库寄回，那么，建议B仓库在调拨入库单中录入实际可用的商品99件，形成调拨差异，A仓库通过调拨差异处理表，生成其他入库单，将1件损坏商品重新入库，再做一张其他出库单，将这件损坏商品报废出库。

2）若是由A仓库承担且损坏的商品需要B仓库寄回，那么，建议B仓库在调拨入库单中录入实际到货的商品100件，再通过调拨出库单或其他出库单将其中的1件损坏商品寄回A仓库，A仓库入库后，再做报废处理。

3）若是由B仓库承担损坏的商品，那么，建议B仓库在调拨入库单中录入实际到货的商品100件，然后，做一张其他出库单，将其中的一件损坏商品做报废处理。
