# 查询“库存查询表”和“商品收发明细表”“商品收发汇总表”时，库存余额为什么不相同？

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题, 新特性, 进阶技巧
> 作者：财税云肖梅梅 | 阅读：1313 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/369842404850942720)

常见问题：查询“库存查询表”和“商品收发明细表”“商品收发汇总表”时，库存余额为什么不相同？

![](https://vipstatic.obs.cn-north-4.myhuaweicloud.com/statics/emotion/define/93.gif)解析： 系统参数设置和报表查询条件都会影响到报表的查询结果，当发现不同的报表库存余额不同时，可以检查以下几个方面：

（1）检查【系统设置】→【系统参数】→【进销存参数】中，选择的库存更新方式是“保存时更新库存”还是
“审核时更新库存”。若选择为“保存时更新库存”，这样库存类单据保存后即使不审核，也会增加或减少库存，
“库存查询表”是包含了未审核的库存类单据的数据。

（2）“商品收发明细表”和“商品收发汇总表”的查询条件中，默认查询“全部”单据，因此查询出来的库存余额不包含未审核单据；可以修改审核状态为“全部”后查询，数据就会与库存查询表一致。
