# 启用浮动单位的商品,查询库存显示基本单位还有库存,辅助单位库存为0,如何清空商品的基本单位库存?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：513 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391353546975325184)

【问题描述】

启用浮动单位的商品,查询库存显示基本单位还有库存,辅助单位库存为0,如何清空商品的基本单位库存?

![](https://vip.kingdee.com/download/0109238e75a90b694b5296a13123af48047d.png)

【原因分析】

当基本单位库存还有数量,辅助单位库存为0,做【其他出库单】,系统会提示:

请录入第[1]行商品[浮动测试]的辅助数量
第[1]行商品[浮动测试]浮动基本单位或者数量为空!

主要由于其他出库单设置的逻辑是,其他出库单辅助单位的库存不能为0。

【解决方案】

当辅助单位已经为0,需要清理浮动单位的基本库存信息时候,可以直接做【盘亏单】
