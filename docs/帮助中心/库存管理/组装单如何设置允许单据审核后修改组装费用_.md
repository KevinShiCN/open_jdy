# 组装单如何设置允许单据审核后修改组装费用?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：winnie_wen | 阅读：257 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/584115627502310656)

【问题描述】

组装单如何设置允许单据审核后修改组装费用?

【解决方案】

点击【系统设置】-【系统参数】

![](https://vip.kingdee.com/download/01091fab1bf52abc4b60a116a9e783e35a39.png)
【进销存参数】-【组装单参数】中勾选“允许修改组装费用(已审核组装单)”保存
