# 其他出库其他入库和调拨出库调拨入库区别?

【问题描述】

其他出库其他入库和调拨出库调拨入库区别?

【解释说明】

其他出库一般是借出、报损等场景需要将仓库里面的商品库存减少,会引起总库存的减少。

其他入库单是处理其他非采购类型的入库单据,比如获赠物料的入库、委托加工等不参与采购管理的入库类业务管理。其他入库单也是财务人员据以记账、核算成本的重要原始凭证,其他入库确认后,成本取数由其他入库单取价优先级控制或手工录入。

![](https://vip.kingdee.com/download/01097509fd6e52064f2da1659d45e4806f21.png)

调拨出库是把商品从一个仓库调到另外一个仓库,调拨出库后可以通过调拨出库生成调拨入库,只是仓库间的库存增加或者减少,总库存是不变的。
![](https://vip.kingdee.com/download/0109089c2ab6c67444638d1f621d85803b33.png)
如果调拨出库而未入库或者未全部入库就会形成在途

![](https://vip.kingdee.com/download/01091b8505d7d2d94b689154bfa6d2db92a6.png)
