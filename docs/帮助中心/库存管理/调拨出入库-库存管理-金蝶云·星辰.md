# 调拨出入库-库存管理-金蝶云·星辰

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：功能介绍, 快速入门
> 作者： | 阅读：9204 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/6165)

问题描述：

应收票据勾选了期初不能退票，提示“票据没有完全引用不能进行退票，如果是期初应收票据可以通过收款退款单进行退票！”

解决方案：

期初应收票据的退票可以新增收款退款单，结算方式选“票据退票”，再关联上该张应收票据流水号之后即可退票。![](https://vip.kingdee.com/download?fileName=0100adce2e828d5345b099584e1bc8b06fd6.png)
