# 怎么设置赠品不参与成本核算?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：Simone邓 | 阅读：3145 | 有用：8
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/462060032213785088)

【问题描述】

怎么设置赠品不参与成本核算?

【 业务场景】

例如跟供应商A采购了10个商品B,供应商送货送了15个,其中5个是免费的赠品,赠品单价为0会导致平均成本降低,销售这5个赠品的时候不希望取到平均成本,而是跟入库一样取零成本。

【解决方案】

系统支持设置 “不核算仓”(如赠品仓):新增仓库时选择 “不核算成本仓” 核算类型,单价为 0 的赠品单独入库至该仓,从不核算仓出库时不会参与核算成本。

【基础资料】-【仓库】,新增仓库时仓库类型选择“不核算成本仓”
