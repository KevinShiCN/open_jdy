# 基础问题

### 1、库存预警统一库存管理和分仓库存管理有什么区别?

答：统一库存管理：所有仓库的统一库存数量进行预警,当某个仓库的库存数量达到预警值时不会进行预警,只有所有仓库的库存总数达到预警值才进行预警。

分仓库存管理：按仓库分别进行预警,仓库的库存到预警值进行预警。

相关知识：[https://vip.kingdee.com/link/s/ZOezR](https://vip.kingdee.com/link/s/ZOezR)

### 2、“统一管理库存”支持按指定的仓库进行预警吗？例如总共三个仓库，只需要按其中两个仓库的总库存设置预警。

答：不支持，系统仅支持两种预警方式。

方式一：统一管理库存。意味着所有仓库的库存是加起来一起判断和预警的。不能排除掉任意一个仓库。

方式二：分仓管理库存。意味着所有仓库的库存是分别各自判断和预警的。

### 3、如何在商品引入模板里设置仓库的管理方式？如何引入商品的最低库存数量、安全库存数量、最高库存数量？

答：在商品引入模板中，找到“仓库管理方式”字段，并开启引入。

商品使用“统一管理库存”方式时，要开启引入模板【单据头】里的库存字段并填入相关的库存数量。

商品使用“分仓管理库存”方式时，要开启引入模板【仓库设置分录】里的库存字段并填入相关的库存数量。

相关知识：[https://vip.kingdee.com/link/s/ZyMwX](https://vip.kingdee.com/link/s/ZyMwX)

### 4、库存预警查询表支持显示商品自定义字段吗？

答：目前不支持。

# 字段说明

### 1、 库存预警查询表的“仓库名称”为什么有的有值，有的为空？

答：可以把鼠标放到“仓库名称”右侧的小问号上，查看说明：只有在基础资料>商品信息>库存预警中，开启了“分仓管理库存”，仓库才会有值。

相关知识：[https://vip.kingdee.com/link/s/ZrQMW](https://vip.kingdee.com/link/s/ZrQMW)

![](https://vip.kingdee.com/download/010013c213fec51c44d2afe0278aabef4eb9.png)

### 2、 库存预警查询表生成采购申请单，是否可以不填供应商和采购员？

答：由于采购申请单中“申请人”是必录的，所以库存预警查询表生成采购申请单的时候，采购员是必录的，供应商可以根据需要设置。采购员批量填写，可以维护第一行采购员后，选中第一行的采购员信息，按F6，批量往下填充。

### 3、 库存预警查询表中的单据备注、商品行备注的应用场景是什么？

![](https://vip.kingdee.com/download/01003ba8e8616f88425b85dd0a62db2d2945.png)

答：库存预警查询表可以勾选商品后，点击右上角的“生成采购订单”、“生成采购申请单”，因此如果这里填写了单据备注、商品行备注，后面可以快速携带到单据上。

相关知识：[https://vip.kingdee.com/link/s/Zr2eQ](https://vip.kingdee.com/link/s/Zr2eQ)

### 4、库存预警查询表中预计交货日期是怎么计算的？

答：预计交货日期=当天日期+备货天数

备货天数取自商品>其他信息。如果没有看到，可以在基本信息右侧的列设置中找到并开启该字段。

![](https://vip.kingdee.com/download/0100d71106e43be8437893152a4a29233f00.png)

相关知识：[采购申请的交货日期是怎么计算的?](https://vip.kingdee.com/link/s/ZrTUB)

# 取值问题

### 1、 库存预警查询表中的预警基准、建议采购量参考基准是什么？是否可以修改?

答：预警的参考基准支持：可用库存、即时库存、预计可用库存。

采购的参考基准支持：最低库存、最高库存、安全库存。

以上两个基准均可在系统设置>预警设置>仓存预警中，进行设置。

相关知识： [https://vip.kingdee.com/link/s/ZrTNg](https://vip.kingdee.com/link/s/ZrTNg)

### 2、库存预警查询生成采购订单的数量是怎么取值的?

答：是根据库存预警设置中设置的库存预警查询表预警基准及库存预警查询表中的建议采购量基准计算的。

相关知识：[https://vip.kingdee.com/link/s/Zr2e5](https://vip.kingdee.com/link/s/Zr2e5)

# 数据不一致问题

### 1、 为什么首页右侧的预警卡片点击跳转库存预警查询表后，查询条数跟预警卡片显示的商品数量不相等？

![](https://vip.kingdee.com/download/0100b0055653eac44c368688b0f6fe608563.png)

![](https://vip.kingdee.com/download/0100e8bba376321a494a82cbef14f0687dbd.png)

答：首页预警卡片的商品数量是去重的，而库存预警查询表是展示商品+仓库维度的明细，当一个商品在多个仓库都有库存的时候，就会显示多行，此时两边的数量就会不一致，这是正常的。

### 2、 商品设置了“分仓管理库存”，为什么管理员查看库存预警查询表的商品数量跟子账号查看的不一样？

![](https://vip.kingdee.com/download/0100e1cb9ac23e5b420f8d75fc4d43ddf00c.png)

管理员看到的：有两个仓库

![](https://vip.kingdee.com/download/0100703b18e8ba994ec084527fbf31c892e7.png)

子账号看到的：只能看到深圳仓

![](https://vip.kingdee.com/download/01005f583c9355794e038062b6d652919243.png)

答：库存预警查询表是受数据授权中的“仓库”权限管控的，如果子账号的数据授权设置了“仓库”的权限（承接上图，子账号只能查看深圳仓）。

![](https://vip.kingdee.com/download/0100483143e1a4694ec297c6134fe5c9bd3b.png)

那么子账号只能查看有权限的仓库，这样看到的就不是全部仓库的数据，就会出现跟管理员查看的数据不一致的情况。

# 设置不生效问题

### 1、 商品设置了统一仓库预警，目前库存为0，为什么首页右侧的预警卡片没有统计到这部分商品？

答：对于无发生额的商品如需参与库存预警，需要打开系统参数>进销存参数>库存预警与异常检查，勾选库存参数“商品无期初和业务时提醒”，然后保存后，重新刷新首页即可。

### 2、商品设置了“统一管理库存”，且子账号设置了部分仓库权限，在库存预警查询表中查看不到对应的商品数据，是为什么？

![](https://vip.kingdee.com/download/0100637f049a5c124134be3a6f69687f5eb0.png)

![](https://vip.kingdee.com/download/0100e32c7c1e391f4b79b389d98c9da11b2a.png)

管理员看到的：正常

![](https://vip.kingdee.com/download/0100d8dce4b16c87463db86c3fd1682c5a65.png)

子账号看到的：没有任何数据

![](https://vip.kingdee.com/download/0100f0bb7c3d0e4444a39149fd303d80bcc9.png)

答：商品设置了“统一管理库存”后，库存预警表中计算待采购数量时，是需要扣减该商品在所有仓库的库存的，所以需要有所有仓库权限才能查看该商品的预警。如果子账号没有所有仓库的权限，就会导致无法计算出完整的待采购数量，故此时查询不出来结果。

>
解决方案：若只有其中一个仓库的权限，请确认商品的采购流程，是只负责有权限的仓库采购，还是其他仓库都需要一起集中采购。若只采购有权限的仓库，建议使用智能补货功能；若需一起采购，建议开通所有仓库的权限。

### 3、 为什么设置了商品的库存预警数量，但是库存预警查询表中不显示数据？

答：需要检查实际的商品库存是多少，如果当前库存数量高于当前设置的最低库存、安全库存，则不满足预警的条件。

例如当前库存是28000，最低库存是10000，安全库存是25000。28000均大于了这两个数值，所以不会在预警数据中显示。

>
**综上所述：**如果设置了库存预警不生效，用户可以从下面四个方面来检查。

**1.商品未发生过交易**，**库存预警表无法统计到**；

**2.商品现有库存未低于库存预警的值**；

**3.没有查询对应商品的权限**；

**4.筛选条件的范围不对。**
