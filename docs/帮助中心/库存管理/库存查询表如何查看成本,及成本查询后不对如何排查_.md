# 库存查询表如何查看成本,及成本查询后不对如何排查?

**【问题描述】**

库存查询表如何查看成本,及成本查询后不对如何排查?

![](https://vip.kingdee.com/download/0109d338834f2e454078b177b70909d43bef.png)[视频观看](https://vip.kingdee.com/school/detail/737736117792793344?productLineId=35&lang=zh-CN&cosmic_type=video)

【解决方案】

**一、库存查询表查询成本**

**建议在查询成本前,先进行[出库成本核算](https://vip.kingdee.com/article/391350577223119872?productLineId=35&isKnowledge=2&lang=zh-CN),保证数据准确**

1.点击进入库存查询表界面,在过滤条件中勾选“显示成本”,点击<查询>按钮

(库存查询表“显示成本”的条件默认不勾选,若需库存查询表显示出商品的成本信息,需手动勾选此条件后查询。若关闭后重新打开库存查询表,需重新勾选此条件。)

![](https://vip.kingdee.com/download/0109b68925119b7240a7bead059534a63c19.png)

2.,点击右上角的<列设置>,筛选列名,可以搜索‘成本’,系统会将成本的相关字段筛选出来,可以设置将成本相关字段设置显示,点击确定

![](https://vip.kingdee.com/download/010971103ebc164d42f8aa0a3cd32807780c.png)3.在库存查询表的界面可以查询到对应的成本数据了

![](https://vip.kingdee.com/download/01094e25a7fc136d465bac69ea2596624bd5.png)

**二、库存查询表查询成本异常**

库存查询表的商品成本根据实际出入库成本来计算,可以通过查询商品收发明细表或者商品收发汇总表进行查询商品成本数量变动过程。

**若发现商品行成本异常,可以双击商品行,直接穿透到商品收发明细表中**

![](https://vip.kingdee.com/download/01090415764d90dc435d9a5f4b6ecf2238f4.png)

**成本=期初成本+入库成本-出库成本;   单位成本=成本/即时库存**

**期初成本**为账套启用时录入,如果未录入或录入错误可能会导致出库成本核算错误,最终导致成本计算异常;

**入库成本**为入库类单据(如采购入库单等)审核后产生,如果入库金额未录入或者录入错误则可能会导致出库成本核算错误,最终导致成本计算异常;

**出库成本**是通过出库成本核算后自动计算得来(具体核算方式见系统参数——存货核算参数),如果期初或者入库成本录入不规范,则会导致出库成本异常最终导致成本异常。

**综上所述:**如果发现成本有异常,可先从以上三个环节排查。

比较常见的原因有:

1.期初或入库类单据是否只录入了数量,价格信息未录入或者录入错误;

2.是否进行了出库成本核算(核算是否有异常,是否处理好了异常),这些通过商品收发明细表、商品收发汇总表可以查出。

![](https://vip.kingdee.com/download/010927d3354ad9f34cf5af189bed76842891.png)

出库成本不对,详细可以点击:[出库成本不对怎么检查?](https://vip.kingdee.com/link/s/ZzY7v)

【温馨提示】**
**

**通过收发明细表和收发汇总表查成本注意事项:**

1.如果成本核算方式**启用分仓核算**,查看收发明细表请按商品+仓库维度进行查询,查看收发汇总表时汇总依据按“商品+仓库”进行查询;

如果是**总仓核算方式**,则不需选择仓库,按照商品查询成本即可

2.收发明细表和收发汇总表会统计**未审核的单据,**但此类型单据不参与成本计算,需要确认审核或过滤;

3.如果是子账号查询,请检查子账号是否有设置数据授权(比如仓库、商品等)
