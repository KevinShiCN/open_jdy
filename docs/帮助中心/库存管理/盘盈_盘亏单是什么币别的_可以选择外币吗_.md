# 盘盈/盘亏单是什么币别的?可以选择外币吗?

【问题描述】

盘盈/盘亏单是什么币别的?可以选择外币吗?

【解释说明】

**盘盈盘亏单是对库存结存的盘点,对应的成本金额是本位币,无法设置其他币别。**

一般来说涉及外币的是往来业务,若是采购或销售类的入库,需按采购入库单或销售出库单录入,采购销售单据可以按外币录入。
