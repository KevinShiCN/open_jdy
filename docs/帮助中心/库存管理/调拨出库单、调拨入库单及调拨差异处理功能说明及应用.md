# 1、简介

## 1.1应用场景

- 若商品从A仓调拨到B仓，调出仓立即扣减库存，调入仓则会产生在途，需要确认收货后才更新库存，同时处理收货差异（报废、丢失、错发）。
- 对于连锁门店来说，各个门店需独立核算，总部配货给门店，需要进行加价（相当于总部批发给各个门店进行销售），此时则采用异价调拨的方式。例如：

场景1：总仓库管理员小王收到一号仓库管理员关于商品苹果X的要货申请，由于和一号仓库在不同的城市，因此，总仓库管理员需要先做一个调拨出库单，等到一号仓库管理员收到货入库后调拨才真正的完成。

场景2：一号仓管员接到总仓库调拨来的货品苹果，入库时，通过选择总仓库管理员做的调拨出库单，然后核对数量，即可快速录单入库。

场景3：一号仓管员在收到总仓库管理员调来的苹果X时候发现少了15台，只有485台，于是在调拨入库单将入库数量改为485台后审核入库。然后，对15台未收到货的商品通过调拨差异处理表进行处理，只需要在调拨差异处理单上将未入库的15台手机通过其他入库单处理即可。

## 1.2功能介绍

调拨出/入库单是用来处理仓库间的调拨业务，用于仓库在不同地方之间的调拨。

- **调拨出库**

有同价调拨和异价调拨两种类型，可以根据实际情况进行选择，异价调拨多用于仓库不在同一个地方的情况，需要将运费等计算到成本内。

- **调拨入库**

通过选择已有的调拨出库单入库，调拨入库的调拨类型和调拨出库的调拨类型是保持一致，不可修改的。

- **调拨差异处理**

由于调拨过程中可能存在忘发、错发、损坏等情况，调拨出库和调拨入库的商品数量会出现不一致的情况，此时会产生调拨差异。调拨差异处理表通过列表的方式，按单据展示调拨过程中出现的差异详情，并引导对差异进行处理。

通过其他入库进行处理后原调拨仓库的数量会相应增加，通过其他出库处理后原调拨仓库数量会相应减少。

## 1.4功能特性

1）调拨入库单的表体只能通过选调拨择出库单带出信息，不可新增行；

答：由于调拨入库是被动接收的操作，是记录商品调拨发出后的到货情况，调拨入库单只提供【选源单】的功能，由调拨出库单生成，不允许新增商品行。

2）调拨入库单的表头的业务类型只能根据选择的出库单自动填入，不可编辑；

3）异价调拨支持设置价格取数优先级，在基础资料>价格取数优先级>异价调拨单取价中设置。

**相关知识**： [调拨单取价整体介绍](https://vip.kingdee.com/link/s/lGADV)

4）支持录入调拨费用并分摊到成本上

**相关知识**： [调拨费用](https://vip.kingdee.com/link/s/MDklB)

# 2、主要操作流程

## 2.1调拨出库

操作路径：【库存管理】→【仓存单据】→【调拨出库】

必录字段：

- 表头：单据编号、单据日期、业务类型（默认为同价调拨）
- 表体调出仓库、调出仓位（若调出仓库启用了仓位管理，则必须填写调出仓位）调入仓库、商品编码、单位、数量

## 2.2调拨入库

操作路径：【库存管理】→【仓存单据】→【调拨入库】

调拨出库单审核后，可点击单据里的【入库】按钮，生成调拨入库单进行入库；也可以在调拨出库单列表中，选择多张调拨出库单进行批量入库；还可以在调拨入库单中通过【选源单】的方式进行入库。

必录字段

- 表头：单据编号、单据日期、业务类型（根据选择的出库单自动填入，不可编辑）
- 表体*调拨入库单的表体通过选择出库单带出信息，不可新增行，允许编辑的字段有：数量、商品行备注

## 2.3调拨差异处理

操作路径：【库存管理】→【仓存单据】→【调拨差异处理表】

允许多次入库，若入库数量与出库数量不相等，请通过《调拨差异处理表》进行处理，通过生成其他出库单、其他入库单调整调出仓库和调入仓库的库存，避免商品无端丢失、账实不符。
