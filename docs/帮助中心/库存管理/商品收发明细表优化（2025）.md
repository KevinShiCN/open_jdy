# 商品收发明细表优化（2025）

# **1 业务场景**

商品收发明细表筛选条件优化；报表增加字段，如期末库存增加辅助数量相关字段

# **2 适用版本**

星辰3.0以上所有版本

# **3 功能介绍**

## **3.1 筛选条件优化**

1、批次增加批量查询功能，支持精准查找：

![](https://vip.kingdee.com/download/0100b1c53ef9cc9544b4bc931dd6fae31632.png)

点击“批量查询”后，可在弹窗中批量录入需要查找的批次号，此处是根据批次号进行精准查找（横线上直接输入批次时，还是模糊查找）：

![](https://vip.kingdee.com/download/01000bcba8de27de46709bc58b53331f2aff.png)

另外，从库存查询表、商品收发汇总表中查询数据后，双击跳转到商品收发明细表时，若双击的数据中包含批次维度，则跳转到商品收发明细表后，会携带批次的筛选条件，并根据批次进行精准搜索。

2、增加“辅助单位”筛选条件，默认为空，可下拉选择库存单位、采购单位、销售单位、报表辅助单位：

![](https://vip.kingdee.com/download/0100703dafcbcadf4e88803fed305ad73540.png)

## **3.2 报表增加字段**

1、增加产地、注册证号、生产许可证、制单人字段显示：携带单据中的信息，期初不显示

2、增加业务类型字段显示：

- 其他出入库单、调拨出入库单：取“业务类型”字段
- 生产领料单：取“领料类型”字段
- 委外发料单：取“发料类型”字段
- 销售采购类单据：取“业务模式”字段

3、期末库存增加辅助数量

- 筛选条件“辅助单位”为空时，报表中显示单据中的辅助单位、辅助数量，期末库存不显示“辅助数量”字段
- 筛选条件“辅助单位”有值时，报表中按所选的商品辅助单位进行显示，并重新计算每张单据对应该单位时的辅助数量，再汇总计算出期末库存的“辅助数量”

>
注意：因商品收发明细表中，需要先根据换算率计算出单据中的辅助数量，再加减计算出期末库存的“辅助数量”，而商品收发汇总表、库存查询表是先加减汇总出单据中的合计值，再根据换算率计算出辅助数量，计算顺序不一样，所以结果值可能会存在尾差的偏差。

（1）辅助数量计算逻辑：

辅助单位=基本单位时，辅助数量=基本数量；否则：

1、固定多单位商品：辅助数量= 基本单位数量 /商品中辅助单位与基本单位之间的换算率；

2、浮动多单位商品：

辅助单位=浮动基本单位时，辅助数量=单据中的浮动基本数量；

辅助单位≠浮动基本单位时，辅助数量=单据中的浮动基本数量 / 商品中辅助单位与浮动基本单位之间的换算率；

小数位取对应单位资料中的小数位。

- 字段授权增加字段：单据备注、业务员、商品描述、商品行备注
