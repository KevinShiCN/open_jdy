# 系统哪些类型的商品不会参与成本核算?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题

【问题描述】

系统哪些类型的商品不会参与成本核算?

【解释说明】

系统商品类型分为普通、服务、套装三类,其中服务类和套装类商品**不参与**成本核算,成本核算**仅支持**普通商品类型。
