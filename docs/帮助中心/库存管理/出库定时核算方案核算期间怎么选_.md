# 出库定时核算方案核算期间怎么选?

> 作者：云社区用户8x80607 | 阅读：75 | 有用：0

【问题描述】

出库定时核算方案核算期间怎么选?

【解释说明】

库存管理-存货出库定时核算,新增方案。**目前支持两种核算期间:核算本期所有商品、核算近三期所有商品(从未结账当期+2期计算)**![](https://vip.kingdee.com/download/010960af3e99dced489ebc812d6307405429.png)
可根据业务需要选择期间核算成本,如如需要核算范围大,可选择近三期所有商品。
