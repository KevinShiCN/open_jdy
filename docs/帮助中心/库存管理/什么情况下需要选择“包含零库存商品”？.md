# 什么情况下需要选择“包含零库存商品”？

> 作者：赵光芬 | 阅读：342 | 有用：5
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/23465)

问题描述：

解决方案：

当“按商品清单盘点”时，盘点单会提取有库存记录的商品，此时，若用户需要核实账存数为0的商品数量是否正确，需要勾选此参数，盘点单才会将这部份库存为0的商品显示在盘点单中，防止漏盘。
