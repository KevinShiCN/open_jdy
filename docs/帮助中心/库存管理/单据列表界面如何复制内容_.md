# 单据列表界面如何复制内容?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：徐某某A | 阅读：156 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/651851350920937472)

【问题描述】

怎么在单据列表复制单据上面的字段内容?

【操作步骤】

打开需要复制的单据的单据列表,右键需要复制的字段,点击复制功能,即可复制单个字段,然后再到需要粘贴的位置粘贴下即可。

![](https://vip.kingdee.com/download/010961620b39ce364e2f8e47f463d8998577.png)
鼠标左键拖动选择出需要复制的部分的数据,然后右键可以批量复制单元格,然后到其他表格内粘贴即可把选中的部分数据粘贴过去。

![](https://vip.kingdee.com/download/01090371471e21154d9792c5fe4e6c744c85.png)
