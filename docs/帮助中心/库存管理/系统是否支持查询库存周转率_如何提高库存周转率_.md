# 系统是否支持查询库存周转率?如何提高库存周转率?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：happy忆霖 | 阅读：396 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/725889552484520960)

【问题描述】

系统是否支持查询库存周转率?如何提高库存周转率?

【业务场景】

希望能在系统中查询库存商品的周转率,哪里可以进行查询,以及如何可以提高库存周转率?

【解释说明】

**1.目前系统暂不支持查询库存商品周转率。**

**建议可以通过导出销售汇总表和库存查询表的方式,手动统计计算库存商品周转率。**

**若需要了解如何导出销售汇总表,请点击:[如何导出销售汇总表?](https://vip.kingdee.com/knowledge/585552552549367552?productLineId=35&isKnowledge=2&lang=zh-CN)**

**若需要了解如何查询指定日期的库存查询表,请点击:[库存查询表/预计可用库存表/库存横向查询表可以按指定日期查询](https://vip.kingdee.com/knowledge/534899798734637824?productLineId=35&isKnowledge=2&lang=zh-CN)**

**2.库存周转率是在某一时间段内库存货物周转的次数,它是反映库存周转快慢程度的指标,周转率越大表明销售情况越好,其计算公式为:**

**库存周转率 = 时间段天数 / 库存周转天数,其中库存周转天数 = 时间段天数×(1/2)×(期初库存数量 + 期末库存数量) / 时间段销售量,即库存周转率=时间段销售量/[(1/2)×(期初库存数量 + 期末库存数量)]**

**根据上述公式可以得出,若希望提高库存周转率,可以采取以下措施实现:**

**方案一:减少库存资金占用:减少每月库存的资金占用量,合理规划采购和生产计划,避免过多的库存积压,根据市场需求和销售预测来确定库存水平;可以通过系统中的库存预警查询表查询最高库存、最低库存,结合库存的预警情况进行合理的采购,避免资金占用过多。**

**若希望了解如何使用库存预警表,请点击:[库存预警查询表如何使用?](https://vip.kingdee.com/knowledge/636731344680299008?productLineId=35&isKnowledge=2&lang=zh-CN)**

**若希望了解如何设置商品的最低、最高、安全库存,请点击:[最低库存、最高库存、安全库存如何设置](https://vip.kingdee.com/knowledge/571138785564399104?productLineId=35&isKnowledge=2&lang=zh-CN)**

![](https://vip.kingdee.com/download/0109ceeae73898e04e99b2f2574bb16668ea.png)

**方案二:增加商品销售总额:建立独立、专业、高效的销售队伍和营销方案等,扩大销售额的提升。**
