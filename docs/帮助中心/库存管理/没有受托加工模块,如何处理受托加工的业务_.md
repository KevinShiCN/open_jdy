# 没有受托加工模块,如何处理受托加工的业务?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：云社区用户0531183 | 阅读：316 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/722980992847184384)

【问题描述】

没有受托加工模块,如何处理受托加工的业务?

【解释说明】

若没有开通受托加工模块,可参考以下步骤变通处理受托业务。

1.仓库设置

新建一个独立受托仓库用于处理受托业务。

![](https://vip.kingdee.com/download/010977119d6ff6e54ee893aa55b0d6840ac6.png)

2. 接收客供料

1. 客户提供的材料需以零成本入库,通过其他入库单处理受托材料入库。单据类型可设为"受托材料入库"。[其他出入库单的业务类型如何设置?](https://vip.kingdee.com/knowledge/129339770114427397?productLineId=35&isKnowledge=2&lang=zh-CN)****

其他入库单操作路径:库存管理 → 其他入库

![](https://vip.kingdee.com/download/0109fb9c17becb1c443d899da3dfe505ce37.png)

新增单据商品单价设置为0 → 选"受托材料入库"类型 → 存入受托仓
![](https://vip.kingdee.com/download/010992f0a511a44942b2b0092e6d5fb80365.png)
3.生产任务处理

正常使用生产任务单进行排产,领料环节指定受托仓出库。加工完成品入库按正常产成品流程操作即可。

4. 受托产品完工出库

出库时使用其他出库单,单据类型可设为“受托产品出库”。

其他出库单操作路径:库存管理 → 其他出库

![](https://vip.kingdee.com/download/0109716550c9848345edb2c927703ee53c3a.png)
