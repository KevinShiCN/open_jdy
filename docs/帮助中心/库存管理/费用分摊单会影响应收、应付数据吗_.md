# 费用分摊单会影响应收、应付数据吗?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：259 | 有用：0

【问题描述】

费用分摊单会影响应收、应付数据吗?

【解释说明】

**费用分摊单不会影响供应商的应付和客户的应收数据,不需要进行收款/付款操作。**

**费用分摊单的应用价值是:**

企业在采购、销售、组装过程中会产生一些费用,与经营业务不是同时确认的,比如采购销售的运费,可能在与物流公司月结的时候才能确认金额;组装时产生的人工费等,在月底核算工资的时候才知道。当出现这种情况时,可以通过“费用分摊单”来将费用与经营业务关联起来,从而对每笔业务产生的成本费用更精确核算。

费用确认后可以做其他支出单,其他支出单录入供应商,不录账户和金额可以影响供应商的应付账款。

如果再通过费用分摊单把费用分摊到对应的单据上即可。
