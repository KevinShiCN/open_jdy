# 其他出库

## 1、简介

### 1.1功能介绍

其他出库单是处理非销售类型的出库单据，比如销售附赠物料、委托加工等不参与销售管理的出库类业务。

### 1.2适用版本

金蝶云·星辰所有版本

### 1.3应用场景

其他出库单是处理非销售类型的出库单据，比如销售附赠物料、委托加工等不参与销售管理的出库类业务。其他出库单也是财务人员据以记账、核算成本的重要原始凭证，该单据的商品成本通过成本核算得出。

### 1.4系统路径

库存管理-仓存单据-其他出库

## 2、 主要操作流程

单击【仓存管理】→【仓存单据】→【其他出库】，进入其他出库新增界面，可直接新增其他出库单。

必录字段包括——

1）表头：

单据编号：可根据编码规则自动生成单据编号，由编码规则指定是否允许编辑

单据日期：默认为系统当天日期，可编辑

业务类型：默认为其他出库

2）表体

商品编码：

仓库：商品“出”的仓库，若仓库启用了仓位管理，需要增加仓位的录入项

单位：商品单位，默认显示商品资料中的库存单位，可编辑

数量：出库数量

## 3、常见问题：

### 1) 其他出库单的出库类型是否可以增加？

可以。进入菜单【基础资料】—>【辅助资料】，在“其他出库类型”树目录下增加资料后，《其他出库单》中的“业务类型”可引用此辅助资料。

### 2) 其他出库单的“出库单位成本”从何而来？

其他出库单的“出库单位成本”是由【库存管理】—>【出库成本核算】计算后返填回单据的。

### 3) 商品返修出库怎么处理？

建议处理的方式：首先，在【基础资料】—>【辅助资料】中，增加“其他出库类型”为“返修出库”，然后，在新增《其他出库单》时，业务类型选择“返修出库”，并进行相应的单据维护，录入客户、联系信息、商品等。

系统提供了【选源单】的功能，可以将其他入库单中的商品引用到其他出库单中，提高用户的录单效率。以数码3C行业为例，手机返修后，需要发回给客户，通过其他入库单记录返回公司维修的商品，修好后再发出给客户，选源单功能可以实现快速录单。

### 4) 商品借出怎么处理？

建议处理的方式：首先，在【基础资料】—>【辅助资料】中，增加“其他出库类型”为“借出”，然后，在新增《其他出库单》时，业务类型选择“借出”，并进行相应的单据维护。
