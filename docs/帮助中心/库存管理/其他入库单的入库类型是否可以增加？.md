# 其他入库单的入库类型是否可以增加？

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题, 快速入门
> 作者：赵光芬 | 阅读：792 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/23427)

问题描述：

解决方案：

可以。进入菜单【基础资料】—>【辅助资料】，在“其他入库类型”树目录下增加资料后，《其他入库单》中的“业务类型”可引用此辅助资料。
