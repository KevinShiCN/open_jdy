# 库存管理反结账提示:业务系统的当前期间**年第*期需要大于财务系统的当前期间**年第*期,业务系统才可反结账

【问题描述】

库存管理反结账提示:业务系统的当前期间**年第*期需要大于财务系统的当前期间**年第*期,业务系统才可反结账

【业务场景】

业务做反结账时提示:业务系统的当前期间2024年第3期需要大于财务系统的当前期间2024年第3期,业务系统才可反结账。是为什么?

![](https://vip.kingdee.com/download/0109b9422415c0254838b33f3dfa61c17180.png)【解释说明】

打开【系统设置】-【系统参数】-【存货核算参数】,当勾选了参数“业务系统单独结账,且必须早于总账结账”时,系统会校验业务期间必须大于财务期间,若业务期间等于财务期间,反结账就会提示;如需反结账修改业务数据需要先反结账财务期间。
