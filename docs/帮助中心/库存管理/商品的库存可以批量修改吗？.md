# 商品的库存可以批量修改吗？

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题, 快速入门
> 作者：杨桂平 | 阅读：1753 | 有用：5
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/357536910660400896)

**【问题描述】**

需要调整商品的库存，是否可以批量修改？

**【解决方案】**

商品库存不能批量修改，建议通过盘点，可批量调整商品的库存。在【库存管理】-【盘点单】中进行盘点。
