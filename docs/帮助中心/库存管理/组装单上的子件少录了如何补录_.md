# 组装单上的子件少录了如何补录?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：云社区用户ySCF4404 | 阅读：6 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/792558742763454976)

【问题描述】

组装单上的子件少录了如何补录?

【概述】

如果组装单上的子件少录了需要补录,解决方案需分单据日期在以前期间还是本期来处理;若需反审核修改,不管是已结账期间还是未结账期间,单据若已生成凭证均需先[删除对应凭证](https://vip.kingdee.com/link/s/Z4SpO)再操作反审核修改。

【解决方案】

**1,组装单在本期的情况下:**

本期的组装单少录了子件,可在本期将单据反审核,使单据处于未审核的状态进行修改

![](https://vip.kingdee.com/download/01091c13b41ae5664f73bf0c9762285622f4.png)

反审核在子件处点击添加,新增物料,录入仓库及数量,然后保存审核即可

![](https://vip.kingdee.com/download/01098107d5fbb0b546c8a457fc7478d61377.png)![](https://vip.kingdee.com/download/0109f767a00001204b69b013ea3043645c47.png)

**2,组装单在以前期间的情况下:**

以前期间的组装单需要补录子件的话,无法反审核修改以前期间的组装单,需[反结账](https://vip.kingdee.com/link/s/ZcXnx)后再操作步骤1

![](https://vip.kingdee.com/download/01098854c061ce874c1488d01f34f069847e.png)

如果不想反结账补录单据的话,需要变通方案实现:

先将需要补录的子件做其他出库单,视作子件的补料

![](https://vip.kingdee.com/download/0109e96462ca01974311901f10b9fdcd1070.png)

此时可以核算子件的出库成本,再将子件成本通过[成本调整单](https://vip.kingdee.com/link/s/Zmm2i)加入到组件的成本中即可

![](https://vip.kingdee.com/download/0109f4906d8297034ffcb108ef2f54db5160.png)
