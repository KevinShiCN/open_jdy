# 存货核算流程

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：快速入门, 功能介绍
> 作者：Cat_Chow | 阅读：6058 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/634808044433285120)

存货核算包括成本的调整、成本分摊，最终出库成本核算，即对企业存货价值（即成本）的计量。

![](https://vip.kingdee.com/download/010936491a1208734200b6c809868492d797.png)

| 步骤 | 具体说明 | 学习链接 |
| --- | --- | --- |
| 1.成本分摊及调整 |  | —— |
| 1.1费用分摊单 | 企业在采购、销售、组装、拆卸过程中会产生一些费用，有的费用与经营业务不是同时确认的，比如：采购销售的运费，可能在与物流公司月结的时候才能确认金额；组装/拆卸时产生的人工费等，在月底核算工资的时候才知道。当出现这种情况时，可以通过“费用分摊单”来将费用与经营业务关联起来，从而对每笔业务产生的成本费用更精确核算。 | 点击学习 |
| 1.2成本调整单 | 企业业务发生过程中，由于价格波动和业务处理，已确认的成本可能需要进行调整。支持两种业务类型：出库调整和入库调整。 | 点击学习 |
| 1.4自制入库核算 | 企业未使用生产管理完整功能，仅通过库存管理手工录入生产领料和产品入库进行简单的生产管理时，可以通过自制入库核算，手工填写入库成本后，通过核算将入库成本填写到产品入库和产品退库单。   | 点击学习 |
| 2.成本核算 |  | —— |
| 2.1注意事项 | (1)已业务结账期间的单据不再参与成本核算。(2)销售出库单、销售退货单等出库类的单据，已生成结转发出商品凭证后，通过成本核算将无法再反写单据成本，如需更新，需要删除结转成本凭证再进行成本核算。建议出库类单据请先进行出库成本核算再生成凭证。  (3)未审核的单据不参与成本核算：①若有未审核的采购入库单，可能会导致无法核算出库成本。②若有未审核的销售出库单，可能会导致销售出库取不到出库成本。 | 点击学习 |
| 2.2开始成本核算 | 支持按期间核算成本；支持指定核算至某天的成本；支持核算以后期间的成本。 | 点击学习 |
| 3.核算结果 |  | —— |
| 3.1出库核算结果查询 | 系统完成出库成本核算任务后,将自动生成针对每个商品的成本计算表,详细记录所采用的存货计价方式、参与核算的相关业务单据,以及商品成本的计算步骤和最终结果,据此确保商品成本数据的准确性。 | 点击学习 |
| 3.2核算单据查询 | 查询出库成本核算后，无成本单据，并单独进行成本处理。成本核算完成界面，点击“无成本单据查询”即可即跳至【核算单据查询】页面。 | 点击学习 |
| 3.3结存成本异常查询 | 用于业务期末结账前,查询统计库存商品是否存在数量为0成本不为0的结存成本,查询到后支持选择后批量生成入库成本调整单。 | 点击学习 |
| 3.4成本异常处理 | 成本异常的几种情况及处理方案 | 点击学习 |
| 4.存货出库定时核算 | 设置定时核算方案进行存货出库成本核算 | 点击学习 |
| 5.业务关账 | 财务人员在结账前,需要做成本核算,若在核算的过程中,单据不断变化,会导致核算结果不准确,因此需要在结账前,控制所有的库存单据不能新增、修改、删除。系统提供“业务关账”功能,控制影响库存的单据不能操作。 | 点击学习 |
| 6.业务结账 | 成本没问题后，即可进行业务结账 | 点击学习 |
