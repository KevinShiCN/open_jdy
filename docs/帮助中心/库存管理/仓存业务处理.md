| 6.2.4、调拨出库：     由商品出库的仓库发起，用于处理商品在仓库间的转移业务。业务场景如下：     1）需要跟踪在途状态（如，需要物流运输）的调拨业务；     2）发货、收货需要双方分别确认，且需要处理收货差异的调拨业务。     金蝶云·星辰支持同价调拨和异价调拨两种业务类型：     a同价调拨：指商品调拨前后价格不变的调拨方式。     b异价调拨：用于处理调拨过程中有价格变动的调拨方式，需要录入调出单价。 | 点击学习 |  |  |

| 盘点业务 | 6.2.10、盘点单：     盘点是指定日期或实时对库存商品实际数量进行清点，将结果与系统数据进行对比，调整并统一系统数据的业务。     支持以下几种盘点方式：       1）按空白单据盘点：盘点时将商品名称、盘点数量记录在纸上或使用盘点机盘点，盘点结束后直接将数据录入或导入到空白的盘点单中，与系统数据对比得到盈亏结果。适用于商品较少，摆放随意、无规则的仓库。     2）按商品清单盘点：选择商品创建盘点清单，打印后按清单进行盘点，盘点完成后将数据录入或导入系统，与系统数据对比得到盈亏结果。商品数量多，且码放整齐有规律的仓库。       3）按商品总量盘点：对于批次和保质期商品，允许不按批次和保质期进行粗略盘点，初步仅统计数量差异。若有盈亏数据产生，再选择按批次和保质期重新盘点或者在盈亏单中手工录入批次和保质期。       4）指定日期盘点：当用户去仓库实际清点库存的日期不是当前日期，而是前几天时，盘点日期就要设置为清点库存的那个日期，这时盘点单提取的账存数就是指定日期那个时点的即时库存。 | 点击学习 |  |

| 组装拆卸业务 | 6.2.16、组装单：       1）采购原材料或配件后，将多种原材料或配件通过加工或组装的方式，变为一种新的产品销售，可用组装单处理；比如购买电脑配件，组装成一台完整电脑。     2）多种原材料或配件，经过加工后，可同时生产出不同的几种商品；比如建材行业，相同的几种板材，切割和安装方式的不同，一次可生产出不同的家具。     3）原材料和成品的配比关系基本不变，可设置组装模版，每次只需修改成品数量，即可自动计算原材料用量。     4）仓库中存储的都是原材料，接到客户销售订单时才开始生产。可根据销售订单数量计算所需的材料数量，领料生产。     6）仓库中没有存储原材料，接到客户销售订单时才开始采购。可根据销售订单数量计算所需的材料数量，下达采购订单。 | 点击学习 |  |
| 6.2.17、拆卸单：     1）采购某种产品后，将其拆成零件销售，可用拆卸单处理；比如回收旧电脑，销售配件。     2）多种原材料或配件，经过加工后，可同时生产出不同的几种商品；比如石材加工行业，将原石切割成不同的形状。     3）成品和配件的配比关系基本不变，可设置拆卸模版，每次只需修改成品数量，即可自动计算可拆成配件多少。 | 点击学习 |  |  |
