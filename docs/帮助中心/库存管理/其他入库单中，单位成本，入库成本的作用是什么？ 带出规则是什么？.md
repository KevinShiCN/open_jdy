# 其他入库单中，单位成本，入库成本的作用是什么？ 带出规则是什么？

【问题概述】

其他入库单中，入库单位成本，入库成本的作用是什么？ 带出规则是什么？

【原因说明】

同采购等入库类单据一样，商品入库需要有入库成本，从而保证最终出库成本核算准确性。

其他入库单入库成本携带规则可以根据基础资料>价格取数优先级带出，也可以手工修改。

单位成本=入库成本/数量

【适用版本】

星辰2.0所有版本
