# 销售出库单的单价为0是否影响成本核算吗?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：若有灵兮山之阿 | 阅读：401 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/562803568990597376)

【问题描述】

销售出库单的单价为0是否影响成本核算吗?

【情景说明】

实际做销售出库单时,有赠品出库情况下,想知道是否影响出库成本核算。

【解释说明】

出库类单据的出库成本是根据之前入库的成本结合存货计价方法计算得出,与销售出库单售价无关,因此销售单单据为0,也不影响成本核算。
