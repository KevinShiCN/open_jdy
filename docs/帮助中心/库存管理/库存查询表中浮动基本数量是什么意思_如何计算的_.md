# 库存查询表中浮动基本数量是什么意思?如何计算的?

【问题描述】

库存查询表中浮动基本数量是什么意思?如何计算的?

【解释说明】

浮动基本数量与浮动换算有关。当企业商品在采购、销售等出入库环节使用不同计量单位,且各个计量单位之间的换算关系不是固定比例时,就会在多单位商品场景下启用浮动换算。

**浮动基本数量是指多单位商品启用浮动换算后,以设置的浮动换算单位为基准核算的库存数量**

例如:例如:假设某商品的基本单位是“瓶”,辅助单位是“箱”,且设置为浮动换算关系,此时浮动基本单位为“箱”

![](https://vip.kingdee.com/download/010979ace871ef9d4dc8be2c63946b7315be.png)

在做入库单据时,若录入12 瓶跟对应1箱
![](https://vip.kingdee.com/download/010944b050497b464e6ab645337a5d86311f.png)
则库存查询表中该商品的浮动基本单位为“箱”,浮动基本单位数量会增加1箱
