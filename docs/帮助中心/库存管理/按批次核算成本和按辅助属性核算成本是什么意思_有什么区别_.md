# 按批次核算成本和按辅助属性核算成本是什么意思?有什么区别?

> 产品：金蝶云·星辰 | 分类：库存管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：257 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/715370606295689216)

【问题描述】

按批次核算成本和按辅助属性核算成本是什么意思?有什么区别?

【解释说明】

**若您想辅助属性、批次保质期、序列号适合哪种业务场景,请查看知识:[系统的辅助属性、批次、序列号如何使用?](https://vip.kingdee.com/link/s/ZjuJW)**

**①商品或参数设为 “按批次 / 辅助属性核算成本” 时,已启用批次 / 辅助属性管理的商品,将按对应明细计算成本。不同批次 / 辅助属性商品入库成本不同的,出库成本需分明细核算,勾选对应参数即可。**

启用批次管理的商品:如 3 月入库批次单价 50 元、5 月入库批次单价 45 元,需分批次核算成本,勾选 “按批次核算成本” 参数。启用辅助属性管理的商品:如灰 / 黑色车采购价相同,珍珠白车因工艺成本更高,需按颜色核算成本,勾选 “按辅助属性核算成本” 参数。

![](https://vip.kingdee.com/download/010913b97b7e0c7a46059953a6e529e433c3.png)

![](https://vip.kingdee.com/download/01099e1e0ecada134ed0826023f759d3bda0.png)

**②若同一批次,需要体现不同成本,可以将商品同时启用批次管理和辅助属性管理,成本核算维度参数同时勾选“按批次核算成本”+“按辅助属性核算成本”**

![](https://vip.kingdee.com/download/0109c061578cdb5c4ba48bb24fb220cb45ed.png)

在【商品信息】>【商品特性】中启用”批次管理“和”辅助属性管理“。

![](https://vip.kingdee.com/download/01093ec84ade17564542a4a7763e6c1c5467.png)

在"辅助属性管理"下面选择辅助属性类别,例如选择类别"颜色",再点击下方颜色右边的<+>按钮选择对应的辅助属性明细,再点击<生成>,最后点击<保存>即可。

![](https://vip.kingdee.com/download/010960e9f69be5944a27b4dc0cabe6119a07.png)

这样在新增入库类单据时就可以实现相同商品同一批次不同辅助属性成本不一致,下面以其他入库单为例。
