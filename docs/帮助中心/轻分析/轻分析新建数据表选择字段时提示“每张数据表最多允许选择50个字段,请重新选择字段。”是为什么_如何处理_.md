# 轻分析新建数据表选择字段时提示“每张数据表最多允许选择50个字段,请重新选择字段。”是为什么?如何处理?

【问题概述】

~~~~轻分析新建数据表选择字段时提示“每张数据表最多允许选择50个字段,请重新选择字段。”是为什么?如何处理?

![](https://vip.kingdee.com/download/0109cad9bce26b824aaeba0f5e5fa9d85eba.png)
【原因说明】

选择字段页面之前是默认全选,而实际用户在创建时,有很多字段用不到,而全选查询会有很大的性能问题,导致使用卡顿,影响体验。因此基于性能及用户体验的考虑,现在产品侧是控制了最多选择50个字段。

【解决方案】

建议客户根据实际需求,需要用到哪些字段就勾选那些字段。不要点击“全选”。

【适用版本】

星辰3.1以上所有版本
