# 轻分析之如何按订单查看出库金额，成本和毛利？

> 产品：金蝶云·星辰 | 分类：轻分析 | 类型：快速入门, 应用案例
> 作者：吊梢眉 | 阅读：644 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/669966657707597056)

# 【问题描述】

如何按订单为维度查看出库金额，成本和毛利？

# **一、业务场景**

公司所有的销售业务都按订单来执行，不存在或极少存在没有订单直接出库的情况。销售员按订单来跟进自身业务；领导层通常也只关注销售订单，不会关注销售出库单。

系统只能在按销售出库单，销售退货单维度（专业版）查看实际出库成本，实际出库金额，并计算毛利，毛利率。

# **二、应用流程**

下面就以星辰专业版为例讲解如何按订单为维度查看出库金额，成本和毛利。

## 1）**数据准备阶段**

选取销售订单，销售出库单，销售退货单三张表。

>
注意：由于销售出库单和销售退货单在数据库是储存在一张表里面的，因此销售退货单的数据基本都是按正数储存。且选择销售出库单会将销售退货单的数据一并抽取出来。

最好在数据准备阶段就先用“单据类型_名称=销售出库单和”单据类型_名称=销售退货单”过滤一遍源数据，即可获得日常理解层面的销售退货单和销售出库单。

![](https://vip.kingdee.com/download/01007f113cac1be5430885a17fb894f2bb81.png)

## 2）**建立三表的关联关系**

用销售订单的商品分录.pk和销售出库单商品分录的源单分录内码关联，关联关系是一对多。销售出库单和销售退货单同理。

>
注意：如果销售出库单不是由订单下推生成（或选源单生成），那就无法通过这种方式关联。

![](https://vip.kingdee.com/download/0100099b33ef3fc84d6ebb945ed77cddcfdb.png)

**注意：**由于销售订单，出库单，退货单都有商品分录表，因此系统为了以作区别，

销售订单的商品分录子表叫：商品分录

销售出库单的商品分录子表叫：商品分录_1

销售退货单的商品分录子表叫：商品分录_2

## 3）**数据分析阶段**

**第一步，**添加计算字段的方式，先选中表名称，再点击右上角的下拉按钮，选择“创建计算字段”：

![](https://vip.kingdee.com/download/0100fcc316c6084c4e2495fb674233935680.png)

在销售订单维度增加三个计算字段：订单实际出货金额，订单实际成本，订单毛利率。

**第二步**，编辑字段计算公式。

| 公式表达式 | 示例 |
| --- | --- |
| 订单实际出货金额=SUM([商品分录_1.实际不含税金额本位币])-SUM([商品分录_2.实际不含税金额本位币]) |  |
| 订单实际成本=SUM([商品分录_1.出库成本本位币])-SUM([商品分录_2.出库成本本位币]) |  |
| 订单毛利率=([销售订单.实际出库金额]-[销售订单.实际出库成本])/[销售订单.实际出库金额] |  |

>
特别说明：如果您的表格顺序和示例中一致，就可以直接复制上面的公式，如果不是，请注意更改取值。

**第三步**，根据需要，将字段拖拽到分析区域，**生成分析表格**。

支持按不同的维度汇总查看数据，行字段可以设置小计，比如按部门小计，按客户小计等；

>
注意：按销售订单查看数据时，存在部分分录行没有出库或部分出库的情况，则出库金额和成本都会为空，可以提前将这部分数据剔除掉。

**最后**：对比起轻分析查出来的数据和系统中销售利润表的数据，发现一致。

![](https://vip.kingdee.com/download/010071a19541493d4c68b1db683d2a627024.png)

以上示例是在星辰专业版做的，如果是星辰旗舰版，注意使用销售发票单获取最准确的收入，单票勾稽后即可拿到成本。底层逻辑相似，可自行测试。
