# 轻分析做数据分析时,如何设置数值字段有金额为0时显示为空?

> 产品：金蝶云·星辰 | 分类：轻分析 | 类型：常见问题
> 作者：Simone邓 | 阅读：582 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/426908106019545856)

【问题描述】

轻分析做数据分析时,如何设置数值字段有金额为0时显示为空?

【业务场景】

统计客户的销售额及其对应的客户承担费用金额数时,客户承担费用字段有部分金额为0,如何设置显示为空

![](https://vip.kingdee.com/download/01094e5582f8deef4f0cacb812e11969dc87.png)

【解释说明】

数值字段的数据格式不支持直接设置,可创建相关字段,使用IF函数,设置客户承担费用为0时=null,否则等于本身数值,如以下公式:

IF([销售出库单.客户承担费用]=0,null,[销售出库单.客户承担费用])

【操作流程】

1、点击左上角字段区任意字段,点击下拉符号-【创建计算字段】

![](https://vip.kingdee.com/download/0109953e2a04005546e18e3b1ad5f6ce0b64.png)

2、打开创建计算字段界面,输入函数公式,确定
![](https://vip.kingdee.com/download/0109752174c2ec404b678f7421afa2df4739.png)

3、回到数据分析界面,将新建的计算字段拖拽到左下角数值区域即可

![](https://vip.kingdee.com/download/01091cd3b7bbb9cb436a9e50a831fc9bebbd.png)
