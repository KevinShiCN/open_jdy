# 仪表板功能说明

【问题描述】

仪表板功能说明

【解释说明】

概述:仪表板支持对数据斗方、网页、文字及组合卡片等组件进行综合布局,并可定义组件数据更新频率;支持将仪表板发布到应用菜单、轻分析中心和移动轻应用并授权给指定用户或角色;支持大屏展现。让用户可以在同一屏幕上集中展现、比较和监视一组特定的数据内容。同时,仪表板还提供筛选、钻取、再分析等交互操作。

仪表板设计器分为六个区域:工具栏、组件工具箱、大纲、设计区、属性区和状态栏,如下图所示:

![](https://vip.kingdee.com/download/0109aa95e505b9f744608cd1fdb0bb3de659.png)

**工具栏**

是用户在仪表板设计过程中对数据视图进行各项操作的一个功能组,工具栏的内容包括:保存、撤消、反撤销、预览、发布到和退出。

**组件工具箱**

罗列出仪表板支持的组件,包括数据斗方、数据分析等数据可视化内容,文字、网页等通用内容,下拉列表、日期时间等筛选器内容,以及组合卡片。

**大纲区域**

罗列出仪表板设计区域已使用的组件名称,默认有“页面”根目录。选中“页面”节点后可以对整个仪表板的属性进行设置。

在大纲区域,选中仪表板中的组件,点击组件右上方的图标,可进行重命名、剪切、删除操作。选中仪表板中的组件,点击大纲区域的上移、下移图标,可对该组件的顺序进行调整。

![](https://vip.kingdee.com/download/0109bf100caaadf74d8481beee227a7ce9d1.png)

**属性区**

(1)可以设计页面的大小,系统提供了几种选择方案,也可以自定义输入宽度和高度值。

(2)可以根据需要上传仪表板的背景图片。

(3)仪表板的外观风格支持淡雅白、深邃蓝以及自定义。

(4)还可以选择预览或发布后的展示模式,轻分析目前支持4种展示模式:充满、适应(保持横纵比)、横向充满(保持横纵比)、不缩放(保持横纵比)。
