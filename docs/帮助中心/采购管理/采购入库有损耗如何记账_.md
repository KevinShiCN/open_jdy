# 采购入库有损耗如何记账?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：502 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/633107417839766016)

【问题描述】

采购入库有损耗如何记账?

【解决方案】

举例:比如采购商品数量100个,单价1元,已录入一张已审核采购入库单,在运输过程中出现了损耗,损耗数量10个。

**业务场景一:损耗供应商承担,且无需补货**

采购订单下推采购入库单,数量和总金额都是90。

![](https://vip.kingdee.com/download/0109f78eb85a661349d59b70807787b9fe4e.png)
采购订单显示部分入库,后续无需再入库,采购订单手动关闭即可。

![](https://vip.kingdee.com/download/01091a369e43bb454009ac10ff88d5d4a248.png)

**业务场景二:损耗供应商承担,且需要再次补货**

采购订单分批入库,先入库90,等供应商再次补货在采购订单重新下推补货的10个即可。

采购订单分批入库参考:**[采购订单如何部分入库?](https://vip.kingdee.com/link/s/Z2aRp)**

**业务场景三:损耗自己承担,无需补货.**

采购订单下推入库数量90,总的应付金额100不变。

![](https://vip.kingdee.com/download/0109308e14c29b3445d6bed61274265abe34.png)
采购订单显示部分入库,后续无需再入库,采购订单手动关闭即可。

![](https://vip.kingdee.com/download/01097f53839abb7c411082a5cbbe0d458253.png)
