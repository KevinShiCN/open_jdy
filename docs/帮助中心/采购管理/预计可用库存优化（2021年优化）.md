# 预计可用库存优化（2021年优化）

# 1 简介

## 1.1 功能发布时间

2021/10/12

## 1.**2 优化背景**

实现智能补货、库存预警查询表、预计可用库存表在系统参数及查询条件一样的情况下，预计可用库存数量一致，可指导客户做采购决策。

## **1.3 应用场景**

客户的销售及采购参考的可用库存依据是不一样的，需要分别控制。如销售下单时，可用库存需要扣减掉销售占用的库存，而在采购时预计可用库存需要扣减掉采购在途和销售占用，以便能准确采购，提升库存周转率，避免库存多采购或是少采购的情况。

## 1.4 适用版本

星辰所有版本

# **2 功能特性**

## **2.1 ****系统参数**

进销存参数>仓存参数下有预计可用库存参数

![上传图片](https://vip.kingdee.com/download/0100c726f1ec4728414a985ce29e88bb052c.png)

预计可用库存可以对**仓库为空的销售订单** 或 **已审核的销售订单**同时进行扣减（即自定义预计出库量的计算范围），用户可结合实际业务场景来设置。

## **2.2 智能补货**

智能补货中的减项支持选择“预计可用库存”

![上传图片](https://vip.kingdee.com/download/01006c7bc21168dd4f8589c32d9e0f276e03.png)

“设置预计可用库存公式”取自系统参数中预计可用库存弹框设置，支持手工修改。

>
注意：智能补货这里修改了公式项，不会影响系统参数中的公式设置。

![上传图片](https://vip.kingdee.com/download/0100781353effb374384ac2efbc50674fb72.png)

## **2.3 库存预警查询表**

1） 表头增加“预计可用库存”字段

![上传图片](https://vip.kingdee.com/download/01000fed4a5e96894f80b9dc5e98a053de89.png)

2）预计可用库存是按照“系统设置>系统参数>进销存参数”的预计可用库存公式来计算的。

## **2.4 预计可用库存表**

可以查看公式中每一项的明细，预计可用库存是按照系统设置>系统参数>进销存参数”的预计可用库存公式来计算的。

![上传图片](https://vip.kingdee.com/download/010090d70f00b98a41599aa524ec33fc0971.png)

# **3 应用流程**

## **3.1 应用场景**

如果您的销售及采购所参考的可用库存依据是不一样的，销售时是根据可用库存判断，而采购补货或者库存预警时，需要剔除对应的已审核的销售订单，以实现更精准的补货。

## 3.2 操作流程

1）在“系统设置>系统参数>进销存参数”的预计可用库存参数中进行设置公式

![上传图片](https://vip.kingdee.com/download/0100558cc3cc0eb84681ba89d3a4c42458e6.png)

2）设置好后，智能补货、预计可用库存报表以及库存预警查询表都会按照设置好的预计可用库存参数来进行采购量的计算，同时保证了库存预警下推的采购补货及智能补货的采购数量一致，且不受可用库存参数的影响，同时满足销售开单及采购补货2种不同场景的决策依据。
