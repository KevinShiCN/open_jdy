# 报表的筛选条件，不用的字段支持自定义显示/隐藏吗？

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题, 快速入门
> 作者：进销存赵嘉丽 | 阅读：1600 | 有用：7
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/154892344947276544)

# 【问题概述】

报表的筛选条件，不用的字段支持自定义显示/隐藏吗？

# 【解决方案】

支持。当用户在报表查询时，仅仅需要其中的某些条件进行查询，而当前报表筛选条件都是默认放开的，操作较为复杂，为方便用户快速查询，支持对报表筛选字段进行显示/隐藏设置，并支持排序。

应用流程：点击报表”筛选设置“，选择字段，不需要显示的，关闭“显示列”，最后单击“确定”即可。
