# 税票登记所选的结算客户名称与销售出库单一致，直接在销售出库单列表可进行税票登记，但在销售税票登记选择源单选不到这笔出库单是为什么？

# **1 问题描述**

销售出库单的未开税票价税合计不为0，税票登记时所选的结算客户名称与销售出库单一致，且可直接在销售出库单列表进行税票登记，但在销售税票登记中的“选择源单”弹窗中选不到这笔出库单，这是为什么？

# 2 适用版本

星辰3.0及以上所有版本

# **3 原因解析**

出现以上现象通常是因为存在重名并且有禁用的结算客户。税票登记选源单只能显示“启用”的结算客户，如果出库单上录的是同名且禁用的结算客户，虽然客户名称相同，但是结算客户编码不同，就会导致选源单里找不到对应的结算客户。

# **4 解决方案**

当存在同名的结算客户时，建议手工登记税票时，选择结算客户，要用“客户编码”来搜。
