# 手工录入采购退货单和通过采购入库单生成退货单有什么区别?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：1325 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/129339770114427511)

【问题描述】

手工录入采购退货单和通过采购入库单生成退货单有什么区别?

【名词解释】

采购退货:当之前采购的货物已入库,由于某种原因(如发现质量问题等)需退回给供应商,就需要录入一笔采购退货单。

通过采购入库单生成采购退货单:会反写采购入库单上的“退货数量”字段,即可以在采购入库单中看到退货了多少。若是采购入库单是由采购订单生成的,还会反写订单的入库数量及入库状态。例如当采购订单全部入库后,采购入库单又全部退货,则采购订单的未入库数量为0,入库状态为未入库。

手工录入的采购退货单:对采购订单和采购入库单没有影响,仅仅扣减库存数量。

【操作流程】

1、通过采购入库单生成采购退货单:当已入库的采购单发生退货时,找到对应采购入库单点击“退货”生成采购退货单并保存审核

![](https://vip.kingdee.com/download/010949172947a9384ad5941c59ce1d9096ab.png)

2、查询采购入库单时“退货数量”会反写采购退货单的数据

![](https://vip.kingdee.com/download/0109558fb7f18a104eb49e03997de74822e7.png)

3、如果采购入库单是通过采购订单下推,并且退货单上有开启退货重发按钮,查询关联的采购订单库存状态:由已入库变成未入库,若再次入库可继续生成采购入库单

![](https://vip.kingdee.com/download/010981ea80e8ceb149fe8cb38368fefd3422.png)

4、手工录入的采购退货单时对采购订单和采购入库单没有影响,单据不存在关联关系,仅仅扣减库存数量。
