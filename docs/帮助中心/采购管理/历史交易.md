# 历史交易

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：功能介绍, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：993 | 有用：8
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/474295648242627584)

## 功能介绍

单据上选中单据其中一条分录（要先选中商品分录），点击表体上“历史交易”即可查看到该商品相关的出入库流水，还支持按客户/供应商进行过滤。方便业务员开单时快速查看，作为开单环节的辅助参考。

适用版本

星辰3.1以上所有版本

注：本文提及的所有功能截图均是星辰3.1以上版本，部分特性是在星辰2.0没有的，如需体验，请联系售后客服升级至星辰3.1

## 操作流程

进销存单据上，选中单据其中一条分录（要先选中商品分录），点击表体上“历史交易”即可查看

## 功能特性

- **页签的查看需要有对应报表的功能授权的“查询”权，否则会无法查看，请管理员提前在【用户管理-用户授权-功能授权】中设置好：**

| 页签名称 | 读取对应报表的“查询权” |
| --- | --- |
| 采购记录 | 采购明细表 |
| 销售记录、委托代销记录 | 销售明细表 |
| 库存数量 | 库存查询表 |

- **支持查看采购记录、销售记录、委托代销记录和库存数量**

![上传图片](https://vip.kingdee.com/download/01003876796561ff4c8698e7815825be102b.png)

- **可以选中只看当前客户/供应商的明细，或者查看未审核的单据、显示退货单据**

****

![上传图片](https://vip.kingdee.com/download/01006e9946aed542438e861de35f9bbb1e98.png)

- **支持查询多个字段，灵活满足不同客户需求**

![上传图片](https://vip.kingdee.com/download/010058c6b239a78641be9001d35959555e93.png)

- 库存数量里支持按仓位维度查询、显示零库存查询

![上传图片](https://vip-admin.kingdee.com/download/01008a4ae60d15d74a82beaf2e06fd6a24c0.png)

- **历史交易页面最多展示20条信息**，因为这里本身就是一个快捷查看的入口，不是报表，当明细内容超过20行时，如果需要查看更多相关的历史交易记录，请点击右上角的【查看更多】来进入到具体的报表查看。
