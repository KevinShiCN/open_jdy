# 金蝶云星辰，做采购入库单的时候怎么使用快速粘贴序列号信息，表格里面录入了4个序列号，但是粘贴到系统只识别到一个怎么处理？

金蝶云星辰，做采购入库单的时候怎么使用快速粘贴序列号信息，表格里面录入了4个序列号，但是粘贴到系统只识别到一个怎么处理？怎么录入才可以识别为准确的序列号信息？

![image.png](https://vip.kingdee.com/download/010971e9ce476d9849c885f9444cdd47364e.png)

![image.png](https://vip.kingdee.com/download/01098aeffaee0ca745b59ed33925b8c3cfd7.png)

解决方案：实操之前，注意查看系统的快速粘贴说明：序列号之间要以分号隔开才能识别准确，附截图如下：

![image.png](https://vip.kingdee.com/download/010900788502fac044b99da6c2b615b20b7b.png)

修改excel表格里面的序列号分隔符号为分号，粘贴到系统即可识别为准确的数量，如下图：

![image.png](https://vip.kingdee.com/download/0109db16a10cf7ff419eb13bf1266bc20426.png)

![image.png](https://vip.kingdee.com/download/010967f5d158a1ac41628aa924c86d1623c0.png)
