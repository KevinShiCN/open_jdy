# 采购管理

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：快速入门, 整体介绍, 精华汇总
> 作者：_Leo | 阅读：16965 | 有用：10
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/634426040931950592)

采购业务是企业供应链活动中重要的一环。采购业务主要包括采购申请、发起采购需求；制定采购合同，与供应商按约定好的商品、价格、数量、支付方式及订金比例等，对于不做库存的商品或者库存不足的商品，还会根据订单需求进行补货；商品到货后进行[入库](https://www.kingdee.com/product/stellar_inventory.html?utm_source=shequ)，同时完成后续的付款动作。

采购业务还需处理因产品质量不合格或者需求变更，需要退货给供应商的采购退货场景。完成发货及退货后，接到供应商的发票，再完成发票的确认和登记，这部分属于单票分离的场景，又或者发票未到但需要先预估或核算财务成本或者发票与实际单据金额有差异，需要进行调整，这部分属于单票合一的场景。

| 1.采购单据 |  | —— |
| 1.1采购申请 | 企业设置专门的采购部门，负责库存商品及内部办公用品的采购。当其他部门有采购需求时，提交“采购申请单”给采购部门。说明：该单据专业版及旗舰版均支持，标准版不支持。 | 点击学习 |
| 1.2采购订单 | 当采购员根据企业的采购需求，向供应商发出采购要求后，供应商可能需要一段时间后才能发货或者发货后途中运输几天才能到达仓库，此时需要录制采购订单，当实际到货后，将采购订单转为采购入库单，增加仓库的库存。说明：可通过【系统参数-进销存参数】控制采购订单是否参与库存余额统计。 | 点击学习 |
| 采购订单支持批量导入 | 点击学习 |  |
| 1.3采购入库 | 用于采购订单下达后，货物到仓，需要把采购订单下推成采购入库，填写到库商品的信息，完成库存的更新及影响应付款。或者无需下采购订单，直接增加库存及应付的入库场景。 | 点击学习 |
| 采购入库单支持批量导入 | 点击学习 |  |
| 1.4采购退货 | 主要有两种采购退货方式：(1)直接新增采购退货单，(2)按源采购入库单下推生成采购退货单。说明：不同的采购退货方式，退货成本取数方式有所不同。 | 点击学习 |
| 1.5以销定购 | 企业为了实现零库存或避免库存积压，事先不采购商品放入仓库，当接收到客户订单后，根据采购订单的需求量进行采购。在以销定购中，可以根据销售订单的相关数量，计算待采购量后生成采购申请单/采购订单，并能直观看到采购的商品是否已入库，便于及时出库。 | 点击学习 |
| 1.6智能补货 | 企业为了避免出现断货的情况，需要提前补充库存。不同行业、不同企业对于备货量的需求是不一样的。本系统提供补货方案的设置，企业可以根据自己的需要，设置不同的公式来计算建议采购量。 | 点击学习 |
| 1.7暂估调整 | 采购入库时，没有收到采购发票，因此不能确定供应商开票金额是否与入库单一致。当收到发票后，需要根据实际情况进行调整。说明：该功能仅适用于专业版。 | 点击学习 |
| 1.8采购税票登记表 | 手工录入采购单据的税票信息,也可以将采购入库单(标准版/专业版)/采购发票(旗舰版)与税务那边开具的进项发票关联起来。 | 点击学习 |

| 2.4采购订单跟踪表 | 实时跟踪一段时期内所有采购订单的详细信息，可选择全部供应商、商品和入库状态进行统计，也可挑选部分供应商、商品、入库状态进行统计。 | 点击学习 |
| 2.5采购付款一览表  | 标准版/专业版:统计一个时间范围内的每笔采购入库单的实际付款或退款状态。旗舰版:统计一个时间范围内的每笔采购发票的实际付款或退款状态。 | 点击学习 |
| 2.6采购日报 | 通过采购日报，可以查询指定日期或者月份的采购概况。例如：采购订单总数、金额、采购出库单总数、金额、采购退货单总数、金额。 | 点击学习 |

| 2.8采购税票汇总表 | 统计一段时期内,采购税票登记的汇总情况,以开票日期、税票类型、开票名称、开票税号、税票号码、税票代码这6个方面做汇总依据,完全一致,才会汇总显示。 | 点击学习 |
| 3.受托代销 | 用于接收第三方委托给本公司的商品，并由本公司进行代理销售，代销完成后，所得收入需与委托方进行计算的场景。常见于品牌方委托代理商（代理商使用星辰）的销售的场景。说明：该功能专业版及旗舰版均支持，标准版不支持。 | —— |
