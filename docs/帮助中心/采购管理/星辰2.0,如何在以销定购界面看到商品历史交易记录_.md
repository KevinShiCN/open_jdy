# 星辰2.0,如何在以销定购界面看到商品历史交易记录?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：姜宁8 | 阅读：229 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/538807433045451264)

【问题描述】

金蝶云星辰如何在以销定购的页面查询到商品的历史交易记录?

【操作方式】

进入以销定购界面,选中任意一行的商品,然后按F4可以自动弹出历史交易记录信息,鼠标点击空白则可关闭历史交易页签。
(注意:历史交易页签展示的数据内容跟单据的数据权限控制一致)
