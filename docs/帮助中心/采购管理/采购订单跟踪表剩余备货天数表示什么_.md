# 采购订单跟踪表剩余备货天数表示什么?

> 作者：云社区用户8x80607 | 阅读：228 | 有用：0

【问题描述】

采购订单跟踪表剩余备货天数表示什么?

【解释说明】

剩余备货天数:表示在**查询****当天订单距离收货日期还有多少天**。若剩余天数为负数,表示已经超过预计的交货日期。![](https://vip.kingdee.com/download/010984c67afcb72943a48cf01849224c98fe.png)
