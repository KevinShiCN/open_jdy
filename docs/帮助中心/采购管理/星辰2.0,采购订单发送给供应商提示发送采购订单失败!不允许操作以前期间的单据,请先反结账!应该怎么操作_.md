# 星辰2.0,采购订单发送给供应商提示发送采购订单失败!不允许操作以前期间的单据,请先反结账!应该怎么操作?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：730 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352148593929984)

【问题描述】

采购订单发送给供应商提示发送采购订单失败!不允许操作以前期间的单据,请先反结账!

![](https://vip.kingdee.com/download/010938d29583453548b29abe239aaec56deb.png)

[解释说明]

发送方单据日期的当前期间已结账不影响,但是接收方需要查询期间,已结账的话,就会发送失败。

[解决方案]

接收方操作反结账。
