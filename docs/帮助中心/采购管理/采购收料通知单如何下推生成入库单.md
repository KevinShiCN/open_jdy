# 采购收料通知单如何下推生成入库单

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：shaoling | 阅读：366 | 有用：1

【问题描述】

采购收料通知单如何下推生成入库单

![](https://vip.kingdee.com/download/0109207f4a0d424640c18aed602436f406fb.jpg)**[视频观看](https://vip.kingdee.com/school/detail/718861054226458880?productLineId=35&lang=zh-CN&cosmic_type=video)**

【适用版本】

金蝶云星辰旗舰版

【解释说明】

采购的收料通知单可通过收料通知单列表点击 <入库> 或 < 合并入库 > 按钮生成采购入库单,或者在收料通知单页面点击 < 入库 > 按钮生成采购入库单

【操作流程】

从【采购管理】>【收料通知单】>【查询】进入收料通知单列表页面

![](https://vip.kingdee.com/download/010914a7c178f2e540fa88d14951c6382c05.png)

在收料通知单列表页面,选择对应单据,点击<入库>或<合并入库>按钮

![](https://vip.kingdee.com/download/01099eab278184aa4ac984b9e7d9ccc9262d.png)

下推生成采购入库单后,点击<保存>或<审核>按钮

![](https://vip.kingdee.com/download/010984638b78f42f46f9a5fb96f0de560fae.png)

保存或审核完成采购入库单即可
![](https://vip.kingdee.com/download/01096f77fa2189694fb19bfae73f345deab5.png)

或者在收料通知单列表页面,点击单据编号进入收料通知单页面
![](https://vip.kingdee.com/download/01095cf880f4411d487eb68cc8f54e6ea852.png)

收料通知单页面,点击 < 入库 > 按钮
![](https://vip.kingdee.com/download/01098a21a0737c114ce2830ff19f437d0cf2.png)

下推生成采购入库单,点击<保存>或<审核>按钮

![](https://vip.kingdee.com/download/01098ee91695bf4547f88e3e4ef100101b9d.png)

【温馨提示】

若直接新增采购入库单,也可通过点击【选源单】按钮,选择【收料通知单】生成

![](https://vip.kingdee.com/download/0109a60a6d3ef7cc4609b5b2a8a8a80f60b0.png)
