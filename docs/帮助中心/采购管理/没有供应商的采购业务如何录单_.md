# 没有供应商的采购业务如何录单?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：酸梅汤啦啦 | 阅读：98 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/694716809823320064)

【问题描述】

没有供应商的采购业务如何录单?

【解释说明】

**采购入库单中“供应商”字段为必填项,无法设置为非必填项。**

当出现零星采购无法确认供应商的业务场景,可通过创建一个通用供应商专门用于无法确认供应商的场景。

【操作流程】

“基础资料”-“供应商”进入供应商列表,然后点右上角“新增”进入新增供应商界面。

![](https://vip.kingdee.com/download/0109ca1eb8c3475547bdba3cda57f225ceaf.png)
![](https://vip.kingdee.com/download/0109fd26945df1dc44dc8ff41fe276dbe76c.png)
输入供应商编码以及名称,名称以输入“零星采购供应商”为例,可自行修改,便于跟其他供应商进行区分即可。

![](https://vip.kingdee.com/download/01097e1d905db12b4b898083a62f4fe8e12b.png)
开单时选择到创建的通用供应商,正常录入商品即可。

![](https://vip.kingdee.com/download/01099d987da6a40c46b9ac35daeab47d0263.png)
