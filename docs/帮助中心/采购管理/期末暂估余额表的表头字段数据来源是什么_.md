# 期末暂估余额表的表头字段数据来源是什么?

【问题描述】
【采购管理】-【期末暂估余额表】中的表头字段数据来源是什么?

![](https://vip.kingdee.com/download/01095851e4ebaea54107b60a2da255e2b26d.png)
![](https://vip.kingdee.com/download/010933b3e014afd3436e8b8b7b198a11183e.png)

【适用版本】

金蝶云·星辰旗舰版

【取数说明】

1.期初暂估余额:上一期的期末暂估余额,当从系统启用第一期开始查询时,期初余额来自暂估入库期初单据。

2.本期暂估:数量取值本期发生的采购入库单减采购退货单的同期未开票数量,暂估成本根据源单据的发票类型,取值同期未开票金额本位币或同期未开票价税合计本位币。

3.本期冲回:取值本期采购发票审核后,自动生成的入库冲回单减退货冲回单的数量及入库成本本位币;或者没有生成采购发票,而是通过采购对等核销冲回的以前期间的暂估成本。

4.期末暂估余额:期初暂估余额+本期暂估-本期冲回
