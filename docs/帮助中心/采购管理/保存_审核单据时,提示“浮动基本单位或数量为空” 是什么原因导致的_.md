# 保存/审核单据时,提示“浮动基本单位或数量为空” 是什么原因导致的?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：若有灵兮山之阿 | 阅读：167 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/686019386561550848)

【问题描述】

保存/审核单据时提示“浮动基本单位或数量为空” 或提示“请录入商品的辅助单位”如何处理?

![](https://vip.kingdee.com/download/010963d27f9d5955419d8714e1d38b356075.png)

【解释说明】

若商品启用了多计量单位时(特别是浮动换算的商品),必须录入辅助单位及辅助数量,否则系统就会进行提示。未启用多计量单位的商品则无需录入。

![](https://vip.kingdee.com/download/0109ff1462cc133b4610ac86eaad79d382cc.png)

【温馨提示】

1、商品启用了浮动换算时录入单据可以手工修改辅助数量和数量,未启用浮动换算的商品修改数量/辅助数量时会反算辅助数量/数量,浮动换算的具体设置及使用可参考:[浮动换算是什么意思,如何设置?](https://vip.kingdee.com/knowledge/472206608613614336?productLineId=35&isKnowledge=2&lang=zh-CN)

2、多计量单位商品如何设置可参考:[如何新增多单位商品?](https://vip.kingdee.com/knowledge/568239379257329920?productLineId=35&isKnowledge=2&lang=zh-CN)
