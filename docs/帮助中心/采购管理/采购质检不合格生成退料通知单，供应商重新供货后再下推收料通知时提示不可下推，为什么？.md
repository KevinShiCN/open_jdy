# 采购质检不合格生成退料通知单，供应商重新供货后再下推收料通知时提示不可下推，为什么？

# **1 问题描述**

采购质检不合格生成退料通知单，供应商重新供货后再下推收料通知时提示不可下推，为什么？是不支持供应商重新供货质检入库吗？

# 2 适用版本

星辰3.0以上旗舰版

# **3 原因解析**

出现这个提示是因为采购订单已经完全下推收料通知了，采购订单此时为关闭状态。如需继续收料质检，建议生成的退料通知单开启“退货重发”功能按钮

![上传图片](https://vip.kingdee.com/download/01005a319bd2cebf44e5a8256efc73de0fce.png)

开启后，采购订单的关闭状态为“未关闭”，此时可以正常下推收料通知单，重新质检入库
