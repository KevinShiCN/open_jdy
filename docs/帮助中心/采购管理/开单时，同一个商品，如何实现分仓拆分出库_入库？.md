# 开单时，同一个商品，如何实现分仓拆分出库/入库？

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：应用案例, 快速入门
> 作者：进销存赵嘉丽 | 阅读：1406 | 有用：12
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/206430521201358080)

# 【问题概述】

采购入库或销售出库时，同一个商品，如果涉及多个仓库，如何快速拆分成按仓库的多行分录入库或出库？而不是手动一行行去新增分录再选仓库？

# 【解决方案】

## 方法一：按库存开单

未录商品时，可以使用单据上的“按库存开单”功能

![上传图片](https://vip.kingdee.com/download/0100ca2ed8db19054547a02ac99f6f359973.png)

快速将同一个商品下多个仓库的明细反填到单据上。

![上传图片](https://vip.kingdee.com/download/01003c296502598247f8975e63fdc8a04948.png)

详细功能说明请查看：[按库存开单](https://vip.kingdee.com/link/s/Z1QJZ)

## 方法二：使用单据分录的库存弹窗

如果已经录了商品未录仓库，且商品行不多，可以直接点击任意的库存字段（即时库存、可用库存）打开库存弹窗，然后选择多个仓库后，系统会自动根据仓库拆分成多行分录。可见GIF动图操作演示。

![上传图片](https://vip.kingdee.com/download/0100792833d66a8b4fbdab544fd2a6d22bbf.gif)

采购类单据也是同理。

>
注意：采购入库单上即时库存/可用库存弹窗里显示的仓库，是商品已经发生过入库业务（包含发生过业务但目前零库存的）仓库，如果商品是首次入库，可用库存弹窗里不会显示仓库，必须手动录入。

## 方法三：智能选仓

如果存在以下场景，推荐使用智能选仓。

场景一：企业有多地仓库，比如上海仓、广州仓，上海仓负责向华东地区的客户发货，广州仓负责向华南地区的客户发货。通过智能选仓功能，在销售订单、销售出库单中选择客户自动携带出收货地址后，系统根据收货地址自动匹配不同的仓库进行出库。

场景二：企业有多地仓库，且每个仓库按仓位管理，每个商品有固定存放的仓位，通过智能选仓，在匹配到发货仓库后，系统可以计算有库存的仓位进行自动匹配，减少人工选择，提高发货效率。

详细功能说明请查看：[智能选仓](https://vip.kingdee.com/link/s/ZMlKd)

# 【适用版本】

星辰3.1以上所有版本（智能选仓不适用于标准版）
