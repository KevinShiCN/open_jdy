# 如何新增采购订单?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：2457 | 有用：14
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/129339770097650425)

【问题描述】

如何新增采购订单?

【业务场景】

当采购下单与收到商品入库不是同时进行时,建议使用采购订单。当采购员根据企业的采购需求,向供应商发出采购要求后,供应商可能需要一段时间后才能发货或者发货后途中运输几天才能到达仓库,此时需要录制采购订单,当实际到货后,将采购订单转为采购入库单,增加仓库的库存。**采购订单不影响实际的库存,只是记录预期的交易。**

不同的供应商、部门、业务员等情况分别独立开单,互不关联。

【操作流程】

采购订单可以通过手工新增、Excel导入、其他单据复制生成、采购申请单关联生成、或者通过智能补货/以销定购/库存预警等功能自动生成等。

**1、手工新增**

依次点击【采购管理】-【采购订单】打开采购订单新增界面,支持手工录入商品信息,也可以通过“按BOM生成”计算并带出BOM单中的子件信息。

![](https://vip.kingdee.com/download/010985f1e6c647d44c7ebdb2dc9501a45ab7.png)

**1.1 手工录入商品信息**

表头基本信息中录入单据日期、供应商、业务员、部门等,在表体商品信息中选择商品,并录入数量、单价、税率等,金额信息中可以录入预付订金、预计采购费用等信息,录入完成后保存/审核单据即可(标记有*是必填项)。
![](https://vip.kingdee.com/download/0109196dc501e10145c6a2f34b1522dfbb48.png)
**1.2 通过BOM生成**

若有使用生产管理系统,并且生产管理中维护了BOM单,则录入采购订单时可以点击“按BOM生成”,打开BOM关联生成界面选择商品并输入数量后点击“生成”即可将BOM单对应的子件信息带到采购订单上,具体操作可参考:[如何按bom生成采购订单?](https://vip.kingdee.com/knowledge/647240664840971776?productLineId=35&isKnowledge=2&lang=zh-CN)

![](https://vip.kingdee.com/download/01095faaf2ecab1a45fbbdac237ebc9339eb.png)

**2、Excel导入**

采购订单列表界面点击右上角【引入】即可通过Excel表格批量导入采购订单,具体操作可参考:[如何批量导入采购订单?](https://vip.kingdee.com/knowledge/542872318003718400?productLineId=35&isKnowledge=2&lang=zh-CN)

![](https://vip.kingdee.com/download/01095e408196d3a54b5eb99493182ed5bc0b.png)

**3、采购申请单下推生成**

新增采购订单时先选择供应商,再点击“选源单”关联采购申请单,或者在采购申请单列表界面下推生成采购订单,具体操作可参考:[采购申请单如何下推采购订单采购入库单?](https://vip.kingdee.com/knowledge/586721563777310208?productLineId=35&isKnowledge=2&lang=zh-CN)

![](https://vip.kingdee.com/download/010929a53587262749a4b200525c27715e1f.png)

**4、其他单据复制生成**

采购订单可以通过其他单据复制生成,如销售订单可以复制为采购订单,复制单据具体操作可参考:~~~~[复制单据如何操作?](https://vip.kingdee.com/knowledge/443940362780912640?productLineId=35&isKnowledge=2&lang=zh-CN)

![](https://vip.kingdee.com/download/01090f2ffdee4e0541f78b6ec39215e339c7.png)

**5、通过智能补货/以销定购/库存预警等功能自动生成**

**①智能补货**:在智能补货界面计算出需要补货的商品及数量后可以直接生成采购订请单,具体操作可参考:[智能补货怎么使用](https://vip.kingdee.com/knowledge/561354029708643072?productLineId=35&isKnowledge=2&lang=zh-CN)

![](https://vip.kingdee.com/download/0109bf8ef87e11ac4f33820a7ec79c431414.png)

**②以销定购:**计算以销定购的数量后,点击“生成采购订单”即可,具体操作可参考:[以销定购的使用方法?](https://vip.kingdee.com/knowledge/534899773938240512?productLineId=35&isKnowledge=2&lang=zh-CN)

![](https://vip.kingdee.com/download/01093c1c5f3ee41541bc83c4f0f3a947f52c.png)

**③库存预警查询表:**在库存管理的库存预警查询表中可以根据计算的建议采购量生成采购订单,具体使用可参考:[库存预警查询表如何使用?](https://vip.kingdee.com/knowledge/636731344680299008?productLineId=35&isKnowledge=2&lang=zh-CN)

![](https://vip.kingdee.com/download/010977d2eea006f64727b0b3d0cde1d04892.png)
