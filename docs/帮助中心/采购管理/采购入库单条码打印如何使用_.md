# 采购入库单条码打印如何使用?

【问题描述】

采购入库单条码打印如何使用?

【解决方案】

在单据列表/单据详情里面,打印下拉选择条码打印设置。

![](https://vip.kingdee.com/download/0109d1b2cd92150d4e458ee2fe6390a51f06.png)
当前系统有预设的打印模板,可选择打印即可。
![](https://vip.kingdee.com/download/0109c1ee99e7c24f4db3921ccf608fa922ed.png)
系统自带打印模板,打印预览效果如图所示:

![](https://vip.kingdee.com/download/010948935fcc46824b30b4fdd3dbe1dcca49.png)
