# 采购入库单支持下推生产领料单

# **1 简介  **

## **1.1 功能介绍**

此功能主要适用于材料采购入库后直接领用的情况，可建立采购入库单到生产领料单直接关联关系。

## **1.2 适用版本**

星辰专业版/旗舰版

## **1.3 按钮说明**

在采购入库单列表、详情新增【领料】按钮

# **2 操作指南**

1、在采购入库单单据列表选择需要领料的单据，点击【领料】按钮，则按单生成生产领料单；点击【合并领料】，则按所选的分录合并生成生产领料单。

- 生产领料单的单据选项支持设置采购入库单下推生成生产领料单时的字段携带项，可设置携带单据日期、部门、业务员；

![上传图片](https://vip.kingdee.com/download/0100792b4b4a2a184b2394a1a41724ca309a.png)

2、采购入库单生成生产领料单时，携带“数量-退货数量-已领料数量”到生产领料单的数量字段，保存生产领料单，反写领料数量到采购入库单的行已领料数量字段，不允许超量领料。

![上传图片](https://vip.kingdee.com/download/010078bf467596fc4dac9c017adeddfd12c4.png)
