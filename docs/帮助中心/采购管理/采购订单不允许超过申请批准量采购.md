# 采购订单不允许超过申请批准量采购

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：功能介绍, 新特性, 快速入门
> 作者：进销存赵嘉丽 | 阅读：696 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/474209709185171968)

# 功能介绍

通过参数控制采购申请单下推采购订单采购时，数量能否超过申请单的批准数量

# **适用版本**

星辰3.1以上的专业版、旗舰版

# **操作流程**

1）打开“系统参数>进销存参数>采购参数”，勾选“允许订购数量大于申请批准数量”，然后“保存”。

![上传图片](https://vip.kingdee.com/download/01001628cb24308940e2a93bceed5c4a885a.png)

2）假设申请单数量为200，单击“采购”，下推采购订单，

![上传图片](https://vip.kingdee.com/download/010010a9fe052e604ec9a29138027b9553b9.png)

将订单上的数量修改成大于200，然后“保存”，可以保存成功。

![上传图片](https://vip.kingdee.com/download/0100deb41e363f534978a8bf2e6f7a7d8900.png)

3）若不勾选“允许订购数量大于申请批准数量”，在采购申请单下推采购订单时，会提示“不允许超额采购”。

![上传图片](https://vip.kingdee.com/download/010063b816186216445794f5d6a74b4e7263.png)
