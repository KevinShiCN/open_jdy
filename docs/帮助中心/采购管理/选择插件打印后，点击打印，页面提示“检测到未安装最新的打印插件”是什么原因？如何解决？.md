# 选择插件打印后，点击打印，页面提示“检测到未安装最新的打印插件”是什么原因？如何解决？

# 问题描述

选择插件打印后，点击打印，页面提示“检测到未安装最新的打印插件”是什么原因？如何解决？

![上传图片](https://vip.kingdee.com/download/0100e90a45f031f74731bc81d463002912a4.png)

# 适用插件版本

星辰4.0及以上插件

# 解决方案

如果出现了以上的情况，请按以下步骤排查和操作（点击图片均可放大查看）。也可以点击右侧目录可以快速定位到文章对应位置。

| 步骤 | 检查项 | 具体说明（详细操作流程） |
| --- | --- | --- |
| 1 | 先检查是否已安装了插件如已安装，请跳到步骤2。 | 注：Windows XP系统、Mac系统不支持插件打印。如果未安装，需在“打印设置”界面下载插件（下图红箭头），并注意检查是否为最新版本。插件是否为最新版本同样可以在插件页面查询。打开【金蝶云·星辰打印】，在“关于”中可以看到插件版本号以及是否是最新的说明。如果不是最新，请到打印设置中点击“打印插件”下载并安装即可。 |
| 2 | 谷歌浏览器/Edge浏览器“本地网络访问权限”检测如果使用的不是chrome浏览器、Edge浏览器，请查看下一步骤。 | 1）打开您使用的浏览器，新增一个页签，输入：chrome://settings/content/siteDetails?site=https%3A%2F%2Ftf.jdy.com%2F2）打开后，找到“本地网络访问权限”，设置为“允许”，如下图：3）打开星辰产品，按F5刷新浏览器。4）重新进入单据页面，单击【打印】测试是否可以打印。如果这一步依然有问题，再按下面的步骤继续排查。 |
| 3 | DNS排查 | 如前面步骤都没问题，依然还是提示检测不到插件，先确认用户电脑的DNS解析是否成功，请点击下方帖子进行查看和操作：DNS解析失败引起的检测插件安装失败解决办法 |
| 4 | 重启插件服务 | 1）打开插件【金蝶云·星辰打印】菜单的【帮助】，在下拉列表中选择“重启服务”，即重新启动插件。2）打开星辰产品，按F5刷新浏览器的网页。3）重新进入单据页面，单击【打印】测试是否可以打印。如恢复正常打印，请继续往下按步骤3操作，避免再次出现检测不到插件的情况出现。 |
| 5 | 检查电脑的“服务”是否为“自动”如没有问题，跳到下一步骤检查。 | 这一步的操作是为了确保插件是电脑开机后自动运行的，不需要每次手动启用。具体的检查和设置方法见下方图文说明：1）单击桌面菜单栏的【开始】，搜索“服务”，打开2）在【服务】中，找到插件，检查启动类型3）在右键单击插件的【属性】中，将启动类型设置为“自动”，完成。 |
| 6 | 检查杀毒软件“开机项”是否禁用了插件如没有问题，跳到下一步骤检查。 | 如果电脑安装了杀毒软件，则需要检查是否是电脑开机加速项/服务项禁用了插件，如果是，则需将插件设置成启用状态，并忽略此项检查即可。下面以“XX管家”为例，说明如何设置开机项不要禁用插件：1）检查【启动项管理】中的“服务项”，检查插件服务“kpjdyprtsvc”的状态是否启用2）在“电脑加速”中，选择忽略插件服务，保持启用状态。注：具体如何设置杀毒软件的开机项，由于不同软件的设置路径不一样，这里不提供统一说明，请通过网络搜索来查询设置方法。 |
| 7 | 检查电脑是否使用了VPN | 如果以上步骤均正确设置，依然无法检测到插件，那么有可能是用户的浏览器使用了代理，即电脑使用了VPN的话，则也会调用不到插件服务，需要在VPN中设置白名单，过滤插件服务。具体如何设置代理服务的白名单，需要根据用户电脑所使用的VPN来设置，无法提供统一的解决办法，请通过网络搜索/AI咨询来查询设置方法。 |

**温馨提示：如果以上所有步骤都排查过了，还是提示检测不到插件，请联系星辰售后在线客服提交工单，需要研发远程分析原因。**
