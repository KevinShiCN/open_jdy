# 导入采购或销售单失败提示单价为0

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：云社区用户0531183 | 阅读：119 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/741098931677094400)

【问题描述】

导入采购或销售单失败提示单价为0

【解释说明】

若系统参数设置了采购类单据或销售类单据单价未0时不允许开单。引入单据时会检查商品单价是否为0。[采购或销售类单据单价是否可以为0?当单价为0时怎么设置是否提醒?](https://vip.kingdee.com/knowledge/555193411750174720?productLineId=35&isKnowledge=2&lang=zh-CN)

![](https://vip.kingdee.com/download/010900c4c5401c184fc5891ddbbe02953d0a.png)
![](https://vip.kingdee.com/download/01097c74106c8ccd46e3ab742f80cf6c9618.png)

【解决方案】

以下以采购入库单为例,销售出库单操作类似。

1. 检查模板上是否确实没有录入单价,如果模板中录入的是含税单价,可以修改引入参数为根据数量和含税单价计算再引入。

![](https://vip.kingdee.com/download/010948a50954dfeb4e618a9a771cbe996a88.png)![](https://vip.kingdee.com/download/0109e2e8369d849943d0beb422be91f97305.png)

2. 若单价和含税单价都没有录入,需填写好后再引入。若模板上没有相关字段,可修改模板后重新下载,并在模板上填写好单价或含税单价。

![](https://vip.kingdee.com/download/01091733970cc00a45929ededbcc1bae566e.png)
3. 若商品单价确实为0,可在模板中添加是否赠品字段,并且在填写数据时,【是否赠品】选择【是】。![](https://vip.kingdee.com/download/0109df4bbed329ab433f8c0412ace2a0df4b.png)
![](https://vip.kingdee.com/download/01091184c7fca05e4778922fa830a1e26298.png)
4. 也可以设置系统参数在新增单据时单价为0【不控制】或【提示,允许继续操作】,则引入单据时单价为0也可以成功引入。

![](https://vip.kingdee.com/download/0109bb84bf8445dc4a97b769d648607a13a6.png)
