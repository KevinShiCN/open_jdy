# 以销定购专题优化（2024年）

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：功能介绍, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：1322 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/593882535755328768)

## 1 简介

### 1.1 功能介绍

根据收集和调用的客户需求，对以销定购进行专题优化，以丰富产品功能，满足更多场景。

**
**

### 1.2 适用版本

金蝶云·星辰3.1所有版本

### 1.3 系统路径

采购管理-采购单据-以销定购

![](https://vip.kingdee.com/download/0109d2f82d0880f5499da181001e11a5c8bb.png)

### 1.4 迭代特性

- “采购生成设置”按钮名称更新为“单据生成设置”，并增加勾选项
- 以销定购页面优化（增加字段、批量编辑单价、供应商批量填充优化、限价提示）
- 多单位商品的关联单据的数据一致性处理
- 销售订单增加采购入库状态筛选

## 2 特性详情

### 2.1 单据生成设置优化

1）原“采购生成设置”按钮名称更新为“单据生成设置”

更名前：

![](https://vip.kingdee.com/download/01003a53773e45524238b415977e1b4e4179.png)

更名后：

![](https://vip.kingdee.com/download/0100f013301734154ebf82b9196af1ece403.png)

2）单据生成设置（即原来“采购生成设置”）增加勾选项

- 采购单据中，支持携带销售订单中的单据日期、换算率及换算公式、预计交货日期
- 支持以销定购页面默认显示最近成交供应商、销售订单中的业务员

![](https://vip.kingdee.com/download/0100203d6f2b1f1a457da1644e858a936009.png)

*携带“换算率及换算公式”重点说明：
    仅适用于商品的采购单位和销售订单中的开单单位一致的浮动换算商品，勾选后，换算率不同的商品行不合并。由于存在部分采购的场景，所以换算率无法直接携带，当单位精度较高时，携带换算率后，采购单据中的换算率和销售订单中的换算率可能有偏差。

### 2.2 以销定购页面优化

#### 1）以销定购页面增加显示字段

供应商商品编码（录入供应商后支持编辑）、供应商商品名称、供应商规格型号、销售预计交货日期、销售订单表体自定义字段、商品自定义字段

#### 2）采购单价支持批量编辑：

![](https://vip.kingdee.com/download/0100c6340ef151ce46ad8cf1c1259691324d.png)

#### 3）支持批量填充为最近成交供应商：

![](https://vip.kingdee.com/download/01007b234206e7e244fe8b6e36e121aa1583.png)

#### 4）支持按行提醒限价：

![](https://vip.kingdee.com/download/010071a0933202a14d5eb26bc80e24f7a7d6.png)

### 2.3 浮动多单位商品的关联单据数据处理

以销定购，关联销售订单生成的采购订单、采购申请单中，浮动换算商品的辅助数量、辅助单价、辅助含税单价放开编辑，编辑后反写关联字段，同手工新增采购订单时的逻辑保持一致。

### 2.4 销售订单增加采购入库状态筛选

#### 1）销售订单列表，增加采购入库状态、行采购入库状态字段。方便以销定购的订单做完采购入库后，快速找到销售订单，下推发货/出库：

![](https://vip.kingdee.com/download/0100632ac2c682c04c9baf2328cf5011cb72.png)

其中，行采购入库状态说明：

| 空 | 该分录行尚未通过以销定购生成采购申请单或采购订单 |
| --- | --- |
| 未入库 | 该分录行已通过以销定购生成采购申请单或采购订单，但采购单尚未入库 |
| 部分入库 | 该分录行已通过以销定购生成采购申请单或采购订单，且采购单已部分入库 |
| 全部入库 | 该分录行已通过以销定购生成采购申请单或采购订单，且采购单已全部入库 |

采购状态说明：

| 空 | 该销售订单尚未通过以销定购生成采购申请单或采购订单 |
| --- | --- |
| 未入库 | 该销售订单已通过以销定购生成采购申请单或采购订单（部分采购或全部采购），但采购单尚未入库 |
| 部分入库 | 该销售订单已通过以销定购生成采购申请单或采购订单，且采购单已部分入库 |
| 全部入库 | 该销售订单已通过以销定购生成采购申请单或采购订单（全部下推采购），且关联的采购单已全部入库 |

#### 2）以销定购页面，增加采购入库状态、行采购入库状态筛选：

**操作说明：取消勾选“待采购为零不显示”后，可筛选出销售订单中已经采购入库的商品行、已经全部采购入库的销售订单，再下推出库/合并出库。**
