# **1 功能介绍**

![](https://vip.kingdee.com/download/010950b9716779014e6ea5e598cb992420a9.png)

整件散包其实是**多单位商品**的应用，适用于批发零售行业中同时存在按整箱/整件以及按拆零散装销售和发货的业务场景；按整件散包下单，也可以方便仓管人员拣货。

例如：在食品行业，客户下单食用油13桶，假设1件=4桶，仓库拣货就可以按3件1桶来拣货，从而提高拣货效率。

# 2 适用版本

星辰3.1以上所有版本

# 3 功能特性

## 3.1 适用单据

- **整件散包**：适用于所有进销存的销售、采购、库存类单据
- **整件数、散包数**：仅适用于销售订单、销售出库单

## **3.2 名词解释**

- **整件数3**：在同时启用四个单位时的最大单位。
- **整件数2**：在同时启用四个单位时的第二大单位，或者同时启用三个单位时的最大单位。
- **整件数1**：在同时启用四个单位时的第三大单位，或者同时启用三个单位时的第二大单位，或同时启用两个单位时的最大单位。
- **散包数**：多单位中的最小单位。
- **整件散包**：即整件数与散包数的汇总（是一个拼接的字段，**不支持运算**）。
- **辅助单位**：除商品基本单位以外的单位（见下图说明）。

![](https://vip-admin.kingdee.com/download/0100d4ef74a776bf4bafa9276124640d8599.png)

# 4 实操应用与特别说明

## 4.1 已开启“浮动换算”的单位，不参与整件散包的换算

- **示例1：**

以下商品，“件”为最大单位，并开启“浮动换算”：

![](https://vip.kingdee.com/download/01000c58adb667b04b8791374924098e29f8.png)

录入单据时，“件”不参与整件散包的换算，而是去除“件”之后的最大单位“箱”来进行整件散包的换算。

![](https://vip.kingdee.com/download/0100a4592b827a644915827615509ded52b6.png)

## 4.2 与浮动单位进行换算的单位，不参与整件散包的换算

- **示例2：**

以下商品，“件”为最大单位，并开启“浮动换算”；“箱”单位与“件”单位进行换算：

![](https://vip.kingdee.com/download/0100817fee6228034a12befa524e8754bb47.png)

录入单据时，“件”“箱”这两个单位都不参与整件散包的换算，而是由“包”进行整件散包的换算。

![](https://vip.kingdee.com/download/01007932627edc0d4734b68eb3eed15d601f.png)

## 4.3 若开启了“浮动换算”，且其他多单位都与浮动单位进行换算，则以商品的“基本单位”参与整件散包的换算

- **示例3：**

以下商品，“件”单位开启“浮动换算”；“箱”“包”单位都与“件”单位进行换算：

![](https://vip.kingdee.com/download/01005eeab5864a5a4dcdaf8bcbc648cc3b2a.png)

录入单据时，“件”“箱”“包”这三个单位都不参与整件散包的换算，而是以商品的基本单位（即计量单位）“个”进行整件散包的换算。

![](https://vip.kingdee.com/download/010071a9ffb8e08849129de6501055f75fa9.png)

## 4.4 当最小单位非基本单位、且基本单位允许小数位时的换算规则

- 整件散包的换算是按照单位换算从大到小依次计算整件数3、整件数2、整件数1和散包数；当最小单位不是基本单位且基本单位允许小数位时，整件散包数只会计算到基本单位，不再往下计算更小单位的整件数/散包数。

- **示例4**：

以下商品，“斤”为基本单位，且开启两位小数位，1包=0.5斤，从大到小的单位依次是：件，盒，**斤（基本单位）**，包。也就是“包”是最小的单位：

![](https://vip.kingdee.com/download/010097e7c107949c4bccb0145bab9fd20e1f.png)

计算整件散包时，件、盒的整件数计算完后，剩余的数量均计算到“斤”的整件数中，不再往下计算“包”的散包数。

![](https://vip.kingdee.com/download/01000ff9e172ba36414886d61ad613b76047.png)

- **示例5：**

以下商品，“斤”为基本单位，且开启两位小数位，1包=0.5斤、1盒=0.1斤，从大到小的单位依次是：件、**斤（基本单位）**、包、盒。也就是“盒”是最小的单位：

![](https://vip.kingdee.com/download/0100074475abca644ab9ad9b488d8cd80e8b.png)

计算整件散包时，“件”的整件数计算完后，剩余的数量均计算到“斤”的整件数中，不再往下计算“包”“盒”的整件数/散包数。

![](https://vip.kingdee.com/download/01005b7df7ad0c1e42ee96d3ffd7dc84904c.png)
