# 以销定购多单位商品,销售订单录入的是基本单位,为什么以销定购采购单位是辅助单位

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：833 | 有用：2

【问题描述】

金蝶云星辰,以销定购多单位商品,销售订单录入的是基本单位,为什么以销定购采购单位是辅助单位

【业务场景】

销售订单上的单位是基本单位,但是到以销定购界面生成采购订单时,带出的是辅助单位,每次都要手动修改回基本单位

![](https://vip.kingdee.com/download/01097ddb6ebf6d0f4b2f9390d6c4bfd9543e.png)

![](https://vip.kingdee.com/download/01098f335ec331f34cee911a505dc0068b41.png)

【解决方案】

到【基础资料】-【商品信息】-找到对应商品,点击修改商品的采购单位即可
