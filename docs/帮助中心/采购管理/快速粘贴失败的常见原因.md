# 快速粘贴失败的常见原因

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：6871 | 有用：22

## 1 原因定位

由于快速粘贴在系统后台解析是有一定规则的，一般解析失败都是因为粘贴的内容中含有了特殊的格式与字符，导致解析识别成多个字段了，造成解析失败。

## 2 失败原因及解决办法

把错误的格式修改正确的，再重新复制到星辰单据单据体的[商品编码]字段进行快速粘贴**Ctrl+V**操作后即可重新解析。

**【****重点！****】常见的识别失败的字符和格式及解决办法（图片或动图可点击放大查看）**：

| 序号 | 常见原因 | 错误的数据源示例 | 粘贴到星辰提示或效果 | 解决办法 |
| --- | --- | --- | --- | --- |
| 1 | 价格或数量字段中有逗号（，） |  | 粘贴失败，直接粘贴不进去 | 把价格或数量字段中逗号都去掉即可假设复制的数据很多，为了提高效率，建议使用excel本身自带的替换功能（快捷键Ctrl+F）来批量替换即可。 |
| 2 | 字段前后中有空格 |  | 粘贴失败，或者粘贴的值不对 | 把“单价”中空格去掉即可假设复制的数据很多，为了提高效率，建议使用excel本身自带的替换功能（快捷键Ctrl+F）来批量替换即可。注意：如果字段本身中就含有空格，则不支持快速粘贴功能，因为代码里会用到空格作为区分粘贴字段名称和具体内容一个分隔符，如果有空格的话，会造成数据切割有问题。 |
| 3 | 商品资料被禁用了 |  | 会出现操作确认弹窗，提示“第X、X、X、X行数据无法识别， 请检查..... | 快速粘贴只能识别状态为“可用”的商品，如果商品“禁用”了，是无法识别的，需要先把商品启用。 |
| 4 | 字段格式不对，例如单价格式是“会计专用” |  | 会出现操作确认弹窗，提示“第X、X、X、X行数据无法识别， 请检查商品编码 或 条形码 或 客户/供应商商品编码是否正确， 如果为辅助属性商品，请检查SKU编码是否正确或是否被禁用， 如果为序列号商品，请检查序列号是否正确， 如果为服务或套装商品，请检查商品类型是否正确” | 把“单价”格式改成“常规或数字”即可 |
| 4 | excel中有隐藏的空格，但是直接看没有问题(没有格式问题，没有符号问题，没有空格）说明：一般这种现象会出现在粘贴的数据是从第三方系统导出的，然后导出的时候源数据格式中有换行符或者隐藏的空格的场景。需要我们使用excel中的公式来解决这类问题。见本文第三点“TRIM(）函数使用说明” |  | 粘贴了四行数据，但是带出了3行，同时提示检查数据 | 步骤1：先把excel里的内容粘贴到记事本里，通过光标确认是否有空格。步骤2：如果有，回到原excel文件，添加新的空白列，然后录入TRIM(）函数来清除空格。步骤3：把清除完空格的数据重新复制到星辰单据体的“商品编码”单元格中操作Ctrl+V即可重新解析。 |

## 3 TRIM(）函数

### 3.1 使用说明

Excel中的TRIM函数可用于去除字符串开头和结尾的空格。

基本语法：**TRIM(text)**，这里的text参数是你想要去除空格的字符串。

`示例：=TRIM（“ hello kingdee ”）`

`将返回  hello kingdee`

### 3.2 应用流程

针对上面第2点中序号4的情况【**excel中有隐藏的空格**】，可以按下面方式处理：

1）先录入=TRIM(）函数，然后括号中间选择要清除空格的单元格，例如：要清除A2里的空格，就选择A2：

![](https://vip-admin.kingdee.com/download/0109bc317f20fbd94cf3a2c614f4b72584e7.png)

2）设置好公式后，鼠标放到右下角，出现实心十字符号后，直接双击，可以把这一列的数据都批量处理了。其他列也参考此方式调整即可：

![上传图片](https://vip-admin.kingdee.com/download/01002e020a2bd83e4c87995eb2a6ee822a11.gif)

3）这样处理完后，重复处理过的列数据再操作Ctrl+V就成功了：

![上传图片](https://vip-admin.kingdee.com/download/01009dbc2c450d944e73bf1ceba16955b4a5.gif)
