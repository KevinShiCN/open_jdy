# 账套备份与恢复功能说明

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：快速入门, 常见问题
> 作者：进销存赵嘉丽 | 阅读：7129 | 有用：51
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/49513528100979200)

# 1 简介

## 1.1功能说明

同一个租户下，如果包含多个子账套，系统提供备份与恢复功能。服务管理员可以在不同子账套直接进行备份及跨账套的恢复。

## 1.2 适用版本

星辰所有版本

## 1.2 适用版本

系统设置>基础设置>账套备份与恢复

# 2 功能特性

- 账套备份恢复仅账套管理员可操作
- 仅支持同在同一租户的账套内进行备份恢复，同一租户指的是，工作台的服务ID（见下图红框）必须相同，只有这样，子账套之间才能相互备份恢复![上传图片](https://vip.kingdee.com/download/010066325c85c9714a16a890dfa5f1b10199.png)
- 如果服务ID不一样，这种恢复属于跨账套恢复，属于收费服务，具体收费标准建议您咨询代理商或者对应省区的渠道

# 3 操作流程

## 3.1 备份

1）账套管理员登录账号，然后进入到产品，打开系统设置>基础设置>账套备份与恢复
2）点击“开始备份”>“确定”，就会对当前账套进行备份，假设当前账套是子账套A,

![上传图片](https://vip.kingdee.com/download/0100cd2deeb1af2947999b472cfaef260a47.png)

>
注意，由于备份会占用服务器的资源，因此避免大批量客户同时并发备份造成性能瓶颈，系统限制了每个账套每天最多只能备份3次，另外为了提高备份的速度，建议您在业务平峰期再操作，或者下班前进行备份。

![上传图片](https://vip.kingdee.com/download/010071f3769c372348f5a2b20c362ce54557.png)

3）系统进入备份进度条页面，此时当前进度条的关闭页面并不影响备份，备份是在后台进行的

因为备份会影响当前账套的使用，故操作前请确保账套里的其他账户已经退出系统!

>
注意，因为后台备份也是有队列的，好比坐车排队，需要前面的人先上车，后面的人才能接着上，如果队列中目前有校对的备份任务，那么备份的时间也会比较长，需要等待，因此有时如果需要很久才能备份完，通常是因为前面还有很多备份任务在等待。为了提高备份的速度，建议您在业务平峰期再操作，或者下班前进行备份。

4）备份完成，点击“立即登录”

5）进入产品后，打开系统设置>基础设置>账套备份与恢复，会看到刚刚备份成功的的一条备份记录。

假设是尾号3527这条

![上传图片](https://vip.kingdee.com/download/0100de9be1dddf6642aba58b24c2f5db2ac4.png)

## 3.2 恢复

1）管理员点击右上角的账套名称，选择需要备份的子账套B

2）打开系统设置>基础设置>账套备份与恢复

3）点击“查看所有账套备份”，我们就会看到子账套A尾号3527那条备份记录

![上传图片](https://vip.kingdee.com/download/0100cb0a81aaf1b8418188e1a96ef67b7e3d.png)

4）选中它，点击“恢复”即可，恢复完成后重新登录即可

![上传图片](https://vip.kingdee.com/download/01000e276d7259fc47a1be77390ac70226c5.png)

>
注意：

- 因为恢复和备份一样，会影响当前账套的使用，故操作前请确保账套里的其他账户已经退出系统!
- 一旦恢复，当前系统账套数据将永久删除，建议提前备份当前账套数据，以免数据丢失
