# 单据上的税率取数逻辑是什么？如果不想启用税率怎么设置？

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题, 快速入门
> 作者：进销存赵嘉丽 | 阅读：1794 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/49165)

【问题描述】

单据上的税率取数逻辑是什么？如果不想启用税率怎么设置？

【原因说明】

1、单据上的税率取值说明：

| 单据类型 | 取值说明 |
| --- | --- |
| 销售类单据 | 税率优先取商品信息上的销项税率，其次取客户资料里的增值税税率，最后取系统参数上的默认增值税税率。详情可查看：销售类单据税率取值说明 |
| 采购类单据 | 可以自己设置优先级，设置路径：系统参数>基础资料参数>税率参数。设置完成后，保存后对新增的单据生效。 |

2、如果不想启用税率，可以将系统参数>基础资料参数>税率参数里的“默认增值税税率”设置为0，同时在新增商品、客户/供应商时也不要设置税率即可。

【适用版本】

星辰3.1以上所有版本
