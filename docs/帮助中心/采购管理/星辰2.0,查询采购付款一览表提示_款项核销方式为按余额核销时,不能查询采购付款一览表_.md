# 星辰2.0,查询采购付款一览表提示:款项核销方式为按余额核销时,不能查询采购付款一览表?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：502 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/129339770080872976)

【问题描述】

查询采购付款一览表提示:款项核销方式为按余额核销时,不能查询采购付款一览表?

【故障现象】

查询采购付款一览表提示:款项核销方式为按余额核销时,不能查询采购付款一览表?

![](https://vip.kingdee.com/download/0109d04a44cc218a4378957503829b1879cb.png)

[故障原因]

当核销方式为“按余额核销”,不能查询采购付款一览表。

[解决方案]

若需查询采购付款一览表,可在[系统设置]-[系统参数]-[资金参数]款项核销方式进行修改。

![](https://vip.kingdee.com/download/010936ac17b9c7ef4c728cd157bf7a75aac9.png)
