# 采购退货后不退款如何处理?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：418 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/701236241647401472)

【问题描述】

采购退货后不退款如何处理?

【解决方案】

录入采购退货单后,单据会产生应退账款,会减少供应商的应付账款余额。若款项供应商不需要退回款项,**可以用于冲减下一次的购货款**,或者由于某种原因达成一致**只退货不退款并且要求冲平退款金额;**也可以直接通过其他出库单记录库存减少**。**不同的业务场景解决方案如下:

1、冲减下一次的购货款

采购退货单会产生应退款项,减少应付款,若后期再次采购时,可以在付款单或者核销单中同时选择采购入库单和采购退货单冲销。

1.1 在【应收应付】-【付款单】中做付款单选择源单时可以同时采购入库单和采购退货单。

![](https://vip.kingdee.com/download/010954908abe1e894f67924c692a42df27e1.png)

1.2 手动核销

若再次采购暂时不用付款时可以通过【应收应付】-【手动核销】新增核销单,核销类型选择【退货冲购货】选择采购入库单和采购退货单核销完成即可。

![](https://vip.kingdee.com/download/01095f3d324bdd89469daf246e3821e929a0.png)

2、只退货不退款,直接冲平退款金额

如果无需退款,可以录入付款单,应退金额直接录入优惠金额中。点击【应收应付】-【付款单】,源单信息选择采购退货单,付款金额不填写,把应退金额全部录入至**“优惠金额”**处。

![](https://vip.kingdee.com/download/01095ba92cd8d23a417e85b66cd9670023e2.png)
3、不记录往来直接减少库存

若不需要记录是往来只需要冲减库存,可以通过其他出库单录入;或者录入采购退货单时不要录入单据金额。

![](https://vip.kingdee.com/download/0109d51a7c3b802443ffab1a3dc196da1b05.png)
