# 金蝶云·星辰，条码标签打印中条码标签打印与单据列表中的条码标签打印有什么区别？

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：快速入门, 常见问题
> 作者：小微微 | 阅读：1806 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/165472828220359168)

**问题描述**：金蝶云·星辰，条码标签打印中条码标签打印与单据列表中的条码标签打印有什么区别？

**问题解答：**

条码标签打印在星辰标准版、专业版、旗舰版中均支持，但单据条码标签打印仅支持专业版、旗舰版。

条码标签打印中打印条码标签与单据列表中的条码打印本质上都是打印商品标签，区别在于：

- 条码标签打印中选单仅支持采购入库单；而单据已经支持商品期初、采购订单、采购入库单、销售订单、销售出库单等多种单据进行条码打印；
- 条码标签打印中只能对采购入库单或者商品上有【条形码】信息的商品进行打印商品标签；而单据列表中条码打印可以对单据上所有商品都打印标签，若商品没有条码只是在打印标签时对应的条码信息打印不出来而已。
- 条码标签打印中只能打印商品资料上的相关信息，不能打印单据上才有的信息，比如供应商、仓库仓位、批次保质期等信息；而单据列表中条码打印支持打印供应商、仓库仓位、批次保质期等单据信息。

简言之，条码标签打印适用于打印商品条码以及商品资料标签；而单据列表的条码打印适用于打印商品条码、商品资料标签以及单据上商品库存标签。
