# 受托入库供应商余额表中的数量怎么计算的?

【问题描述】

受托入库供应商余额表中的数量怎么计算的?

【解释说明】

受托入库供应商余额表中数量=已审核受托入库单的基本数量-已审核受托结算单的结算数量。
