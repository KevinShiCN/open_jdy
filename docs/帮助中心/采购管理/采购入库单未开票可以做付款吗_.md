# 采购入库单未开票可以做付款吗?

【问题描述】

采购入库单未开票可以做付款吗?

【解释说明】

**可以付款**。采购入库单能否付款取决于其“付款状态”。若付款状态显示为“全部付款”,则系统会显示下图所示报错信息。
