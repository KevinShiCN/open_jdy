# 金蝶云星辰，采购订单列表的上一步处理人名字中带有电话号码，但是用户管理里的名字是没有带手机号的，是什么问题呢？

金蝶云星辰，采购订单列表的上一步处理人名字中带有电话号码，但是用户管理里的名字是没有带手机号的，是什么问题呢？附截图：

![image.png](https://vip.kingdee.com/download/010930960f23228d4278b0943da0cec1cf83.png)

![image.png](https://vip.kingdee.com/download/010994871b6a153143f3b8c8c2954493fc93.png)

原因：显示登录名是因为数据库里这个租户有重名的；

解决方案：建议修改姓名，改了就不会显示， 或者名字后面加个点号.
