# 保存单据提示:单据[bp24122-02]:第[22]行的不含税金额计算异常,请将数量修改为其他值再改回原值,手动触发重新计算。

【问题描述】

保存单据提示:单据[BP24122-02]:第[22]行的不含税金额计算异常,请将数量修改为其他值再改回原值,手动触发重新计算。

【解释说明】

新增单据保存提示:单据[的价税合计计算异常,请将数量修改为其他值再改回原值,手动触发重新计算。

原因:单据的价税合计金额与数量*单价金额差异过大,系统金额校验不通过,需要将数量修改为其他值再改回原值,手动触发重新计算。
