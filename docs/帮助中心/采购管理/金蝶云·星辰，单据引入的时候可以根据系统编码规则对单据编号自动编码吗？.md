# 金蝶云·星辰，单据引入的时候可以根据系统编码规则对单据编号自动编码吗？

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题, 快速入门
> 作者：小微微 | 阅读：1553 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/140026467542728448)

问题描述：

解决方案：

金蝶云·星辰在单据引入的时候可以根据系统编码规则对单据编号自动编码，只需要在引入模板中不引入单据编号即可。如下图：

![](https://vip.kingdee.com/download/0100a9e9dd8602ab4659843de369ebfd097c.png)

注意：

- 模板中无单据编号列，其他列根据需要填写。模板中有单据编号列但是不填写，在引入时会要求填写单据编号。
- 如果一张单据多商品分录，只需要填写一个表头信息。
