# 方案查询中的“包括”与“包含”有什么区别？如何使用？

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：快速入门, 常见问题
> 作者：进销存赵嘉丽 | 阅读：2242 | 有用：8

# 【问题概述】

方案查询中的“包括”与“包含”有什么区别？如何使用？

![上传图片](https://vip.kingdee.com/download/0100aefcc1d230a640e8ba0b3fcb5ae12939.png)

# 【名词解释】

- 包含：是like，是模糊匹配，即查找含有关键字的所有字段，不需要一一对应。
- 包括：是in（），是完全匹配，即查找所有完全相同的字段，需要一一对应，支持录入多个字段同时查询。

# 【应用说明】

## **示例1：模糊匹配与完全匹配**

假设有客户名字叫“金蝶商行”，在客户资料列表，客户名称的表头漏斗里，用“包含”过滤“金蝶”可以模糊匹配出来，同时还会查询到含有“金蝶”的其他字段。

![上传图片](https://vip.kingdee.com/download/0100d8aa7db3d03741b29c2bfb7c8a979504.png)

但是用“包括”来查“金蝶”则查不出来，因为要求必须完全匹配，如果客户列表里没有一个跟“金蝶”完全一致的值，就会显示暂无数据，表示匹配不到值。

![上传图片](https://vip.kingdee.com/download/010013bc8c67bc6040b6b2bfba247ea6b7ef.png)

## **示例2：多个字段同时查询（必须精准查询）**

**应用场景：如果需要查询多个字段，且字段非常明确（属于精准查询），可以用“包括”查询。如果不满足精准查询，则无法使用。**

>
注意，多个字段的搜索，只能用“包括”条件，不支持用“包含”条件。

**操作方法：**

在“包括”条件下，可以在右侧直接手工输入要查询的字段，并用英文逗号将字段分开

![上传图片](https://vip.kingdee.com/download/01000cecf94698e64a18b12141bd989cf0c4.png)

如果字段太多，也可以直接单击右侧的录入图标，在弹窗中录入即可（用英文逗号将字段隔开）

![上传图片](https://vip.kingdee.com/download/0100f39a1209bda54fdfb8826fc8b349523b.png)

输入后效果

![上传图片](https://vip.kingdee.com/download/0100bbebfa1547da40329b357f9c98d48366.png)

单击“确定”后的查询结果就是精准查找。

![上传图片](https://vip.kingdee.com/download/0100b4d156dafeae484eb51dc9c840d98198.png)
