# 序列号如何引出

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：782 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/371692758064804352)

【问题概述】

星辰的序列号如何引出？

【解决方案】

# • **单张单据引出**

单据保存或审核后，如果分录里有序列号商品，在菜单栏【更多】按钮里会显示“导出SN”的按钮，点击导出SN，则将单据内序列号导出：

![上传图片](https://vip.kingdee.com/download/0100bc2425dcc7d54f50bc7c7255dde680b8.png)

导出后的Excel文件如下图：

![上传图片](https://vip.kingdee.com/download/01006dbcb76adccc439a914db2e949100cdf.png)

>
注：导出的Excel文件为系统预置模板，不支持自定义设置及修改！！！

# • **单据列表引出（多张单据）**

使用系统预置的序列号引入模板

![上传图片](https://vip.kingdee.com/download/01002e3602f2ae324223b4862bc84bb416c0.png)

引出效果：

![上传图片](https://vip.kingdee.com/download/0100b364ee4be2b24963b55bd46bb316c1ec.png)

如果是自定义的模板，则在引入模板中开启“序列号组合”，按引入模板引出时就会有对应的信息

![上传图片](https://vip.kingdee.com/download/0100c18e9c29fa974921b6e81c3c312f9ce7.png)

【适用版本】

星辰3.1以上所有版本
