# 采购订单的状态是未关闭、未入库,但是下推入库之后有几行商品带不出来

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：进销存熊诗琴 | 阅读：33 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/751608778021166592)

【问题描述】

采购订单的状态是未关闭、未入库,但是下推入库单总有一行商品无法带出;订单全流程跟踪查了也没有其他的关联单据。

#

![](https://vip.kingdee.com/download/010975a87bd72b594e05af4de23364069cfd.png)

【适用版本】

金蝶云·星辰3.1及以上所有版本

【解决方案】

1)订单列表的“关闭状态”字段指的是整单的关闭状态,若下推入库单时部分商品行丢失,需要在列设置里开启“行关闭状态”来检查每一商品行,被手动关闭的明细行无法下推,需将其启用后下推才能正常带出。

![](https://vip.kingdee.com/download/01094a1a00aae0a642b4bd4de6e755c0a04f.png)

2)关闭和启用单行商品行,可以在列设置-选项中开启【同一单据拆分显示】

![](https://vip.kingdee.com/download/0109b2526ac581c54c2b817d4a832cab8215.png)
