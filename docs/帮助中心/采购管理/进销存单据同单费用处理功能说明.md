# 进销存单据同单费用处理功能说明

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：功能介绍, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：1337 | 有用：6

# 适用单据

进销存云所有含“分摊费用”按钮的**出入库单据**。

# 适用版本

星辰3.1以上所有版本

# 功能点

## 1、金额为0或者开启赠品的商品行也可以参与分摊

当单据分摊规则选择**按“数量”分摊**时，金额为0或者开启赠品的商品行，也可以参与分摊

![上传图片](https://vip.kingdee.com/download/0100c7d0ab7b7e45480980825db930ac721b.png)

## 2、删除优化

删除业务单据时，会提示是否要同步删除已经生成的费用单，不用再单独到其他支出单列表查找对应单据，关联单据操作更方便。

![上传图片](https://vip.kingdee.com/download/01001207b6aa6964496d946524ad8cb68a7c.png)

注意，如果其他支出单已审核，操作“同时删除”时，会提示该支出单已经审核

![上传图片](https://vip.kingdee.com/download/0100ad38731021244ca59c2dcc16dd4bcaf8.png)

点击查看单据，可以看到费用单据的详情。用户可以自行决定是否要删除还是保留，系统允许删除业务单据，而保留该其他支出单。

![上传图片](https://vip.kingdee.com/download/01009382796f35bb4fb8a14e78e6c49ebc6d.png)

# 3、费用明细支持录入部门、业务员

## **3.1 单据头部门、业务员支出携带到费用明细**

如果单据头录入了部门、业务员，则费用弹窗选了供应商后，系统自动携带单据头的部门、业务员到费用弹窗，也支持手动修改

![上传图片](https://vip.kingdee.com/download/0100c3f2a17b5ffd41199b607b11e1b8ef8a.png)

## **3.2 费用明细的部门、业务员支持携带到其他支出单上**

业务单据审核后，支持携带费用弹窗上的部门、业务员到生成的其他支出单上。

## **3.3 单据体自定义字段支持携带到其他支出单单据头上**

如果单据头自定义字段勾选了适用其他支出单，那么在单据关联时，单据头的自定义字段将自动携带到其他支出单上
