# 快速粘贴使用说明

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：快速入门, 功能介绍
> 作者：进销存赵嘉丽 | 阅读：14738 | 有用：36
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/111537926130676224)

## 1 简介

### 1.1功能介绍

支持批量复制本地表格的数据，并在单据体的[商品编码]字段进行快速粘贴Ctrl+V操作，不用通过引入模板进行导入单据，实现快速开单，有效地提升工作质量和效率。

例如：在接收到客户的订单或是供应商的送货需求的表格时，能直接复制粘贴表格内的数据到系统里

### 1.2 适用版本

星辰所有版本

### 1.3 功能特性

• 支持销售类和采购类单据；

• 支持粘贴的字段：条形码、SKU编码、商品编码、商品名称、客户商品编码、单位、数量、仓库、仓位、批次序列号、生产日期等

• 最大复制粘贴行数1000行，最小复制粘贴5个单元格（如粘贴1列时最少复制5行）

• 识别优先级：条形码>SKU编码>商品编码>客户/供应商商品编码>单位、数量、仓库、仓位

• 列字段顺序：商品识别字段列放在前面，包括：条形码、SKU编码、商品编码、客户/供应商编码、序列号。其他字段需放在商品识别字段列后面，才能粘贴

## 2 应用流程

### 2.1 使用方法

请复制Excel中包含列字段名在内的数据，在系统单据体中的[商品编码]处进行Ctrl+V粘贴。

### 2.2 步骤说明

（1）在excel中，选中数据（注意一定要把列名也给一起选中了，也就是下图中蓝色的一行）右键复制，或者直接使用键盘Ctrl+C

![](https://vip.kingdee.com/download/0100c87874e90d4c470b8e864f31fbb830bd.png)

（2） 新增单据，把光标放到单据体“商品编码”内（只能是这个字段，不能是商品名称）

![](https://vip.kingdee.com/download/01009ec4604c1b5e475db98d528237deca18.png)

（3）使用键盘Ctrl+V粘贴，稍等几秒钟即可成功带出复制的信息。

![](https://vip.kingdee.com/download/0100f1951d3317714d3ca1f3bc166ae1397c.png)

## 3 注意事项（精华）

| 序号 | 说明 | 产品说明 |
| --- | --- | --- |
| 1 | 快速粘贴的操作只能在单据体“商品编码”内进行，在其他字段粘贴都不会生效。如果没有看到这个字段，请查看列设置中是否隐藏。  |  |
| 2 | 快速粘贴时一定要连同列字段名一起复制，否则会失败，无法解析成功 | 错误示例：不带列字段名正确示例：带列字段名 |
| 3 | 一次复制多行数据时，最大粘贴行数：1000行商品信息。超过会有右边的提示 |  |
| 4 | 仅复制1列时，最少复制5行数据（列字段名也算一行，所至少还要带上4行数据） | 错误示例：复制4行正确示例：复制5行 |
| 5 | 仅复制1行时，最少复制3列 | 错误示例：复制2列正确示例：复制3列 |
| 6 | 粘贴序列号商品方式1：直接录数量，粘贴后，序列号要自己录入方式2：按序列号明细导入，序列号请用分号“；”隔开，例如下图注意：分号不区分中文符号的差异，都可以。 | 方式1：录数量方式粘贴方式2：录序列号方式粘贴 |
| 7 | 复制粘贴客户商品编码、供应商商品编码时，请先选择单据头的对应的客户或供应商信息 | 销售类采购类 |
