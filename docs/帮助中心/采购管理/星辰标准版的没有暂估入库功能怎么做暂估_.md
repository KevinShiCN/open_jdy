# 星辰标准版的没有暂估入库功能怎么做暂估?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：shaoling | 阅读：2637 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/597955203295348224)

【问题描述】

星辰标准版的没有暂估入库功能怎么做暂估?

【业务场景】
采购入库时,没有收到采购发票,因此不能确定供应商开票金额是否与入库单一致。当收到发票后,需要根据实际情况进行调整

若使用的其他版本实际有暂估入库功能,但看不到此功能菜单,可点击查看:[为什么有的功能在左侧模块菜单里看不到?](https://vip.kingdee.com/link/s/Z6Emx)

【解决方案】

星辰标准版没有暂估入库的功能,可通过采购入库发票来了,根据发票重新做采购入库单,原采购入库单源单退货操作实现

【操作步骤】

①采购存货发票尚未收到,现存货按暂估价格18元录入采购入库单(进行暂估入库);

![](https://vip.kingdee.com/download/0109c2300d576bfa4f3b924341f2d71af875.png)

②月底发票来了,发票金额为20元,按发票金额重新录入采购入库单

![](https://vip.kingdee.com/download/01095c1f5b35597e498fa38bad946a0589bf.png)

③然后找到原暂估入库的采购入库单进行退货操作,生成采购退货单,冲销原入库的成本金额及应付账款

![](https://vip.kingdee.com/download/01091b0ac4a5384f4240b4ab5668df7368db.png)
【相关知识】

收到发票,发票金额与单据金额不一致,且系统未结账、单据未生成凭证、单据已审核且不想反审核修改单据金额时可通过“金额变更”调整单据的金额,详细可以点击:[销售/采购金额变更功能](https://vip.kingdee.com/link/s/ZqNqP)
