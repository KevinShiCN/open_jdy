# 采购订单状态说明

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：快速入门, 功能介绍
> 作者：进销存赵嘉丽 | 阅读：2995 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/267055229941172992)

# **应用场景及价值**

原采购订单下推成采购入库单后，订单就会变为已入库状态，而实际上库存可能并没有增加。增加了订单执行状态，能更准确的反映订单的处理情况。同时增加订单入库核销状态，已入库核销金额，退货数量，退货金额，关闭数量，关闭金额等信息的查询。

# **字段说明**

• **执行状态**：采购订单生成采购出库单；订单分为整单的执行状态和行执行状态，分别代表订单整单是否生成采购入库单，某一行商品是否生成采购入库单。

• **入库状态**：采购订单生成的采购入库单，是否增加了库存，根据系统参数，分为保存时更新库存和审核时更新库存；商品行中可查看本行的已执行未入库数量和已执行已入库数量。

• **订金状态与入库核销状态的区别**： 在采购订单中录入了订金，并生成的预付款单审核后，订金状态变为“已付”，而已付的订金不代表订单付款状态为“已付款”。入库核销状态是由该采购订单生成的采购入库单的付款金额反写而来。若本张订单的订金，没有用于抵扣订单生成的采购入库单，则这笔订金不计入本张订单的已入库核销金额中。

• **关闭状态**：当订单全部执行时，订单自动变为关闭状态。当订单未执行完，剩下的部分不需要再执行，可以将订单手工关闭。系统会记录手工关闭数量和手工关闭金额。

• **退货数量**：订单生成的采购入库单，按源单退货后，在订单上可查看关联的退货数量和退货金额。

# **应用流程**

1、打开采购订单列表，整单模式下，可以看到整单相关的字段

![上传图片](https://vip.kingdee.com/download/01002d53127378cb47fbadb702c9e52bb981.png)

2、关闭整单按钮，打开可以看到行相关的字段

![上传图片](https://vip.kingdee.com/download/0100f8346bd9fbe34785a278c3ac05066ac3.png)

3、单据的保存、审核、下推入库、退货后，都会反写对应的字段。
