# 采购汇总表怎么按照开票状态区分查询?

> 作者：云社区用户55os622 | 阅读：99 | 有用：1

【问题描述】

采购汇总表怎么按照开票状态区分查询?

【适用版本】

金蝶云星辰·旗舰版

【操作指引】

依次点击【采购管理】--【采购汇总表】,进入采购汇总表查询界面![](https://vip.kingdee.com/download/0109b1bab11c3c1a4ff59c39020891f4fac7.png)

过滤条件中找到并选择【开票状态】后点击查询即可
