# 采购入库单、其他入库单、产品入库有什么区别?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：2938 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/129339770114427406)

【问题描述】

采购入库单、其他入库单、产品入库有什么区别?

【解释说明】

**采购入库**:就是从外采购验收入库的意思,是从供应商处采购进来,从而引起库存的增加,同时采购入库后会产生与供应商的资金往来,会收到采购发票抵扣增值税等等;

新增采购入库单,详细可以点击:[如何录入采购入库单?](https://vip.kingdee.com/link/s/ZFli5)

**其他入库单:**如盘盈入库、借入等,因企业内部原因,而引起的库存增加,其他入库单不需要支付资金,也不会有税金发生。

其他入库单使用,详细可以点击:[其他入库单,如何使用?](https://vip.kingdee.com/link/s/Zcq75)

**产品入库单:**车间在完成生产任务后,需要将完工产品入到仓库中,仓库人员需要进行验收入库,并通过产品入库单记录入库的数量及仓库信息,便于管理库存。

新增产品入库单,详细可以点击:[如何录入生产产品入库单?](https://vip.kingdee.com/link/s/ZaQPz)

【温馨提示】

未开通【生产管理】模块,其他入库单与产品入库单的入库成本需自行核算录入(其他入库单可在【基础资料】-【价格取数优先级】设置取其他入库单取价规则);如开通【生产管理】模块,产成品入库成本会在生产领料加工后,进行生产成本核算后,计算出来。
