# 商品的采购单价可以在开单的时候自动带出吗？

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：快速入门, 常见问题
> 作者：进销存熊诗琴 | 阅读：424 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/7696)

# 问题描述：

商品的采购单价可以在开单的时候自动带出吗？

商品的采购价格大部分时间都是不变的，如何快速录单？

# 解决方案：

步骤一：需要先在商品资料中填写“采购单价”；

![上传图片](https://vip.kingdee.com/download/0100ae26e0d4f0ed49e2885653bac0e7c601.png)

步骤二：需要明确该价格是含税价还是不含税价，在《系统设置——系统参数——商品参数》中指定是否启用商品价格含税；若启用，则将该价格带入采购入库单的“含税单价”字段，若不启用，则带入采购入库单的“单价”字段。

![上传图片](https://vip.kingdee.com/download/010009ed470f55c645bcac0aed7860196554.png)
