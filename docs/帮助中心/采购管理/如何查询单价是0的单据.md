【问题描述】

如何查询单价是0的单据

【操作步骤】

1:进入对应单据的列表;

![](https://vip.kingdee.com/download/0109a8f8af6621864ce18953f2c5b58c1e57.png)

2:找到单价的字段,点击字段右边的漏斗标志![](https://vip.kingdee.com/download/01092ee8405bd81f4f4784762a68fe0669c7.png)

3:选择等于0的条件确定即可显示单价为0的单据![](https://vip.kingdee.com/download/01096f1eeaa3bb4e490981211bcec31f685e.png)
