# 金蝶云星辰，批次商品采购订单入库，实际已经全部入库了的，但是采购订单显示部分入库？

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：功能介绍, 精华汇总, 进阶技巧
> 作者：小橙子和小柠檬 | 阅读：1120 | 有用：9
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/538748433952225024)

金蝶云星辰系统在做采购订单的时候启用了批次管理的商品行数量是5个，如果一个数量是一个批次，那采购入库单选择源单就需要选择5次吗？如果采购入库单上直接新增行的话，这样又会导致采购订单上显示部分入库，实际又是已经全部入库了的，有没有什么更好的解决方案？

解决方案：可以使用采购入库单上的拆分行入库；

具体步骤：

1、新增采购订单，选择批次商品录入对应商品行数量保存审核采购订单；

2、新增采购入库单，选择供应商信息后，选择源单，选择到对应采购订单；

3、采购入库单上点击拆分行，选择分摊拆分，将选中行拆分为采购订单数量的对应行数，比如订单是5行，这里就可以拆分为5行；选中行下方插入；然后采购入库单上补充对应商品行的批次信息，保存审核采购入库单；

4、回到采购订单列表查看这张采购订单的入库状态就更新为全部入库状态；

备注：销售订单批次商品出库，同采购订单批次入库一样的操作逻辑操作即可；
