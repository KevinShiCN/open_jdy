# 进销存部分采购/销售/库存类报表支持添加公式字段、支持排序依据

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：功能介绍, 新特性, 快速入门
> 作者：进销存熊诗琴 | 阅读：1126 | 有用：4

# 1 简介

## 1.1 功能介绍

采购/销售类部分报表支持添加公式字段，满足更多场景。同时列设置中的排序支持记住上一次设置的结果，避免重复设置，提高查询效率。

## 1.2 适用版本

星辰3.1以上专业版/旗舰版

## 1.3 适用单据

| 类型 | 单据名称 |
| --- | --- |
| 销售类 | 销售汇总表——2025/05/29发布销售明细表——2025/05/29发布销售订单汇总表——2025/06/26发布销售发票汇总表（仅旗舰版）——2025/06/26发布 |
| 采购类 | 采购汇总表——2025/05/29发布采购明细表——2025/05/29发布采购订单汇总表——2025/06/26发布采购发票汇总表（仅旗舰版）——2025/06/26发布 |
| 仓存类 | 库存查询表——2025/09/29发布 |

# **2 功能特性**

## 2.1 报表公式字段设置方式

- **公式字段设置入口：**有权限的用户，在列设置弹窗中，可以看到“添加公式字段”的按钮，点击该按钮即可为报表添加公式字段（账套级），每个报表可以添加12个公式字段。

![](https://vip.kingdee.com/download/010976c0ac3936e343569b98db56b431cdfb.png)

- **公式设置弹窗：**

![](https://vip.kingdee.com/download/010902f43da679bf4c79b9e0603b34f7ebec.png)

数字函数的应用说明：

①四舍五入：实现对数值进行四舍五入，比如：ROUND(1.24,1)，返回1.2;ROUND(1.25,1)返回1.3；

②向上舍入：实现对数值向上舍入，比如ROUNDUP(1.24, 1)，返回1.3；

③向下舍入：实现对数值向下舍入，比如ROUNDDOWN(1.29, 1)，返回1.2；

④取绝对值：实现返回表达式的绝对值，比如ABS(-5)，返回5；

⑤取整：实现返回数值的整数部分，比如INT(1.9)，返回1。

- **公式设置完成后：**

![](https://vip.kingdee.com/download/0109d13f46e7d4f347aa91aba57a717b2978.png)

## 2.2 报表排序规则

- **设置入口：**用户在列设置中，可以设置当前筛选条件下对应的排序规则，最多支持选择5个字段，设置完成后，会按用户记住上一次设置的规则。

![](https://vip.kingdee.com/download/0100d05e8776d4d041fe83296c5af4684fcd.png)

>
1）汇总表中，不同的汇总依据对应列设置的字段不同，因此排序规则也需单独设置；

2）如存在多个排序字段，则排序字段的优先级与列设置的优先级一致。
