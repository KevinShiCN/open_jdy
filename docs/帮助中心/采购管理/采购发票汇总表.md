# 采购发票汇总表

## 1、简介

### 1.1功能介绍

统计一段时期内商品的采购发票及采购退货发票的详细情况，可选择全部供应商、商品、仓库进行统计，也可挑选部分供应商、商品、仓库进行统计。

![](https://vip.kingdee.com/download/0100520c65eb4f014f62a630d55193b50974.png)

### 1.2适用版本

金蝶云·星辰旗舰版

### 1.3系统路径

采购管理-采购单据-采购发票汇总表

## 2、报表查询说明

1）按不同的汇总依据查询时，报表有不同的展现形式，当选择汇总依据为结算供应商，商品类别时，还可以通过勾选“显示商品明细”来实现结算供应商+商品，商品类别+商品的多维度的汇总统计。

2）当统计到商品维度时，若系统启用了辅助属性管理，可以通过勾选“显示辅助属性”来统计到商品的属性维度。

3）当业务类型选择为“全部”时，数据来源于采购入库发票和采购退货发票，相同字段的数据由采购入库发票数据汇总减去采购退货发票数据汇总得出；

4）供应商、商品、仓库如果空表示查找所有供应商、商品、仓库，可分别打开各选择框进行勾选；审核状态为空，表示查询已审核单据和未审核单据。

5）类别级次汇总：当汇总依据为商品类别时，可以选择汇总到第几级明细类别。

![](https://vip.kingdee.com/download/01001965a298331449f38988fc88445d7073.png)

6）在筛选设置中，将不常用的查询条件隐藏

![](https://vip.kingdee.com/download/0100e7f421bc8ad84958a0f5f31f419524eb.png)

7）可将常用的查询条件保存为默认方案

8）报表支持引出、打印、列设置、组合排序
