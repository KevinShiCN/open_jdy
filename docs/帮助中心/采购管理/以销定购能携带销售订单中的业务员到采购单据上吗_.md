# 以销定购能携带销售订单中的业务员到采购单据上吗?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：新用户000 | 阅读：214 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/612451034186891264)

【问题描述】

销售订单中设置了业务员,以销订购生成采购单据,可以携带销售订单中的业务员到采购单据上吗?

【操作步骤】

以销订购界面,点击右上角的“单据生成设置”,在以销定购页面中以下字段显示默认值中勾选“销售订单中的业务员”;

![](https://vip.kingdee.com/download/01092c2daef3e6aa40108a9500c57300d3b2.png)
勾选之后,在查询单据,采购业务员可以按销售订单上维护的业务员带出;

![](https://vip.kingdee.com/download/0109ef8848c163344b4f8a8f40af24f8dfd7.png)
注意:业务员设置带出默认值,也还是可以手工修改。
然后点击生成采购单据,采购单据就会带出业务员的信息。

![](https://vip.kingdee.com/download/010986b4bdcaea7e471d83e8344bc591eb2d.png)
