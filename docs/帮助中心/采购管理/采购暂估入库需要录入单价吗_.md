# 采购暂估入库需要录入单价吗?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：shaoling | 阅读：116 | 有用：0

【问题描述】

采购暂估入库需要录入单价吗?

若您想要了解暂估入库操作问题,请点击:[暂估入库如何操作?](https://vip.kingdee.com/knowledge/391352395873684992?productLineId=35&isKnowledge=2&lang=zh-CN)

【适用版本】

金蝶云星辰专业版

【解释说明】

采购入库单开启暂估入库后,可根据需要录入暂估单价。若仅需跟踪库存数量(如部分贸易型企业),可先不录入单价,待发票到票后补录,一般建议录入单价(可录入参考单价或平均单价)

若不录入单价,后续采购入库单核算成本时会提示入库成本为零。

若需生成暂估凭证需录入单价,否则无法生成采购暂估凭证
