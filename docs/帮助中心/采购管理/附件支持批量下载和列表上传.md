# 附件支持批量下载和列表上传

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：功能介绍, 新特性, 快速入门
> 作者：进销存熊诗琴 | 阅读：601 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/716005396254821888)

## 1 简介  

### 1.1 功能介绍

定制化产品，在订单上上传了本单商品的产品结构图，设计部门，生产部门需要批量将近期要设计，需要批量下载生产的产品结构图。

### 1.2 适用版本

星辰3.1以上所有版本

## **2 功能特性**

### 2.1 单据列表支持批量下载

- 以销售订单为例，销售订单列表增加【下载附件】的按钮；
- 选中多张订单后再点击【下载附件】，会将所选订单中的附件批量下载下来，生成一个压缩包。

![](https://vip.kingdee.com/download/0109fe0ad47479bd4c2b8823b961ffbae5e1.png)

| 销售类 | 销售报价单、销售订单、销售出库单、销售退货单、销售发票（旗舰版）、发货通知单（旗舰版） |
| --- | --- |
| 委托代销 | 委托代销单、委托代销退货单、委托代销结算单、委托结算退货单 |
| 采购类 | 采购申请单，采购订单，采购入库单，采购退货单、采购发票（旗舰版）、收料通知单（旗舰版） |
| 其他类 | 其他入库单，其他出库单，移仓单，调拨出库单，调拨入库单，盘点单，分量盘点单，组装拆卸单模板，组装单，拆卸单 |
