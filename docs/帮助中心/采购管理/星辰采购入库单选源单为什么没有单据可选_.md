# 星辰采购入库单选源单为什么没有单据可选?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：脸脸1 | 阅读：138 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/604841048476705280)

【问题描述】

星辰采购入库单选源单为什么没有单据可选?

【概述】

采购入库单选不到源单可能有以下原因,可对账逐一排查:
**星辰标准版 / 专业版:**未录入采购订单,或已录入的采购订单处于 “未审核”“已关闭” 状态;
**星辰旗舰版:**未录入采购订单 / 采购发票,或已录入的采购订单 / 采购发票处于 “未审核” 状态,或采购订单已关闭、采购发票已完成入库操作。
