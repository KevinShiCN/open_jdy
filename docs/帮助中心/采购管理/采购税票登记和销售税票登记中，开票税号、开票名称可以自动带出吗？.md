# 采购税票登记和销售税票登记中，开票税号、开票名称可以自动带出吗？

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：快速入门, 精华汇总, 常见问题
> 作者：进销存赵嘉丽 | 阅读：65 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/802223491159758080)

# **1 问题描述**

采购税票登记和销售税票登记中，已经有了结算供应商/客户，对应的开票名称就应该默认等于结算供应商/客户，但每次都要填写开票名称很麻烦，有什么办法可以自动带出吗？

![上传图片](https://vip.kingdee.com/download/0100477d1e7ba0e6483587e660d2b6765333.png)

# 2 适用版本

星辰3.0及以上所有版本

# **3 解决方案**

可以。提前在客户/供应商资料中维护好开票税号、开票名称等字段，维护采购税票登记或者销售税票登记时就能自动带出来。
