# 采购退货单,退货成本如何取数?

采购退货单,退货成本如何取数?

【解释说明】

采购退货成本有两种取数方式:

1、按源采购单生成采购退货单,取源采购单入库成本做为出库成本。

如图,源采购入库单位成本120元,按源单生成采购退货,出库单位成本也是120元

![](https://vip.kingdee.com/download/010985d2ac5eae734f83823e19b6c19382d8.png)

![](https://vip.kingdee.com/download/01091042c90cef3d45c8aee5dcb886f966c5.png)

注意:按源采购单生成退货单,即手工修改采购退货的单价也不影响退货成本取数,只影响退货往来金额。

2、直接新增采购退货单,退货成本取单据上录入的退货单价为出库成本。

如图,采购退货单价:89元;成本核算后,出库单价成本也是89元。

![](https://vip.kingdee.com/download/010979f32d5523064d98a74fd7206ceb4c38.png)

注意:采购退货的仓库允许负出库,也是直接取采购退货单上录入的单价,并不是按系统参数负出库规则取数。

**若采购退货单核算不到成本的原因有以下几方面:**

1、无源单新增采购退货单,单据上价格为0,需要补录退货单价才能核算到退货成本

2、采购退货有源单,源单成本为0

采购退货单列表上点击源单单据编号打开对应采购入库单,查看采购入库单的入库成本,若源单就没有成本那么按源单退货取不到成本也是正常的,无需处理

![](https://vip.kingdee.com/download/0109e9bf5ced51294348b8dff6a8461216ed.png)

3、采购退货有源单,核算成本前采购退货单生成了凭证会导致退货单核算不到成本

采购退货单列表上开启显示“凭证字号”字段,查看对应单据是否有生成凭证(包含暂存凭证),若有凭证,需点击对应凭证字号或暂存凭证打开删除,删除成功后再重新核算成本即可。

![](https://vip.kingdee.com/download/010980573659e87b49588787e87797fcf633.png)
