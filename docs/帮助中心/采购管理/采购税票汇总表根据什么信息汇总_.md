# 采购税票汇总表根据什么信息汇总?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：云社区用户8x80607 | 阅读：97 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/622960280661850112)

【问题描述】

采购税票汇总表根据什么信息汇总?

【操作步骤】

**采购税票登记的汇总情况,以开票日期、税票类型、开票名称、开票税号、税票号码、税票代码这6个方面做汇总依据,完全一致,才会汇总显示**。

税票登记的时候,一张发票分多次登记到了多个单据;在采购税票汇总表界面,会把同一张发票的数据汇总显示。
