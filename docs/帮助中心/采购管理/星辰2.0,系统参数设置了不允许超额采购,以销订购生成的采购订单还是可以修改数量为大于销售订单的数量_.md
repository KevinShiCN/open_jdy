# 星辰2.0,系统参数设置了不允许超额采购,以销订购生成的采购订单还是可以修改数量为大于销售订单的数量?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：姜宁8 | 阅读：482 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391351987163999232)

【问题描述】

星辰2.0,系统参数设置了不允许超额采购,以销订购生成的采购订单还是可以修改数量为大于销售订单的数量?

![](https://vip.kingdee.com/download/010984cb049bbf3747498695c26b50fc0552.png)         【故障原因】

这个参数是控制在以销订购界面生成的采购订单的数量,但是生成的采购订单后面还是可以去修改数量的,不受该系统参数的控制

【解决方案】

如果是要控制生成的采购订单数量不能修改为大于销售订单的数量,建议是通过字段授权,控制子账号不允许修改采购订单“商品编码”、‘数量’,才能控制采购员不超额采购。
