# 采购类单据选供应商价格资料开单

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：快速入门, 应用案例, 新特性

# 1 应用场景

商贸等行业存在销售链路：品牌商→代理商→客户，品牌商只管控代理商的拿货价，代理商以销定购，品牌商向代理商供货的价格与代理商销售的终端客户所在地区相关，即同一供应商（品牌商）、同一商品，代理商存在多个采购价格方案，做采购单时需要选择供应商价格资料。

# 2 适用版本

星辰3.0以上所有版本

# 3 功能特性

## 3.1 适用单据

采购申请单、采购订单、采购入库单、采购退货单、采购发票（旗舰版）

## 3.2 功能介绍

### 3.2.1 选供应商价格资料开单操作

- 步骤1：采购类单据详情，点击“ 供应商价格查询“

![](https://vip.kingdee.com/download/01008d649dd3f7fe41aca5d6e86594f946d6.png)

- 步骤 2：弹窗形式打开 “供应商价格资料查询表”

①按供应商、按商品、按价格类型查询价格资料，

②勾选价格资料

③确认开单，将价格资料的信息如商品、单位、采购价格、折扣等携带到单据中

![](https://vip.kingdee.com/download/0100e303cb14f60a4749b96535874ac0fe60.png)

说明：

1、查询供应商价格资料的默认值：

①查看类型：“按供应商”

②供应商：默认取单据已录入的供应商

![](https://vip.kingdee.com/download/0100b0282e6f56844405877407e0c92f3b24.png)

③币别：仅账套开启“按币别维护供应商价格资料”参数，才有币别的查询条件，且默认取单据币别

（参数位置：系统参数>进销存参数，按币别维护供应商价格资料）

![](https://vip.kingdee.com/download/0100d46e8af19f0f4d4bb52984144c71b8bf.png)

④状态：执行中

2、供应商价格资料的“供应商”携带到单据：

①采购订单/入库单/退货单/发票：单据**未选择**供应商，将携带成功的第一行价格资料相关的“供应商”携带到单据中

![](https://vip.kingdee.com/download/0100dd32b0bcff944ab4a645f3bea4f79af9.png)

②采购申请单：无论单据基本信息是否选择“建议供应商”，都支持将价格资料相关的“供应商”携带到采购申请单商品分录的“建议供应商”，注意：若按不同供应商采购，建议使用按相同商品分录.建议供应商合并采购

![](https://vip.kingdee.com/download/0100e09d272f5de74ad2919913f60e82259a.png)

3、不核算成本仓相关

若商品行的仓库对应为”不核算成本仓“，携带的价格资料的价格会被清空。

### 3.2.2 携带供应商价格资料失败的场景

| 序号 | 失败原因分类 | 失败原因明细分类 | 具体场景 |
| --- | --- | --- | --- |
| 1 | 价格方案相关 | 价格资料类型 | 价格类型为“供应商+商品类别”和“供应商+商品品牌”的价格资料，无具体商品 |
| 2 | 价格资料执行状态 | 非执行中的价格资料 |  |
| 3 | 价格资料审核状态 | 开启价格资料审核，未审核的价格资料 |  |
| 4 | 选价格资料冲突 | 供应商冲突 | 采购订单/采购入库/采购退货单/采购发票单据，①选择供应商与单据已录入的供应商不同的价格资料；②选择多条供应商不同的价格资料，其他行与所选第一行供应商不同的价格资料。 |
| 5 | 币别冲突 | 开启按币别维护供应商价格资料，选择币别与单据币别不同的价格资料 |  |
| 6 | 商品相关 | 商品状态 | ①已禁用商品的价格资料②商品开启审核，选择未审核商品的价格资料 |
| 7 | 商品类型 | ①选择服务商品的价格资料②选择套装商品的价格资料 |  |
| 8 | 商品特性 | 选择未开启“可采购“的商品的价格资料 |  |

- 失败详情将弹窗提示：

![](https://vip.kingdee.com/download/0100059d66d8085a453599ad1a3cb5e547a4.png)

### 3.2.3 仅查询供应商价格资料

- 以下场景，**单据不允许添加商品行**，故查询供应商价格资料，无”确定“、”取消“按钮：

| 序号 | 场景分类 | 具体场景 |
| --- | --- | --- |
| 1 | 单据的状态 | ①单据已审核（含金额变更，不含订单变更）②单据已作废③单据已关闭 |
| 2 | 单据的审核流程 | 启用审批流的单据，审批过程中 |
| 3 | 单据的业务流程设置 | 业务流程设置单据必选源单 |
| 4 | 上游单据的业务模式 | 直运的销售订单下推的采购订单/入库单直运的销售退货单生成的采购退货单 |
