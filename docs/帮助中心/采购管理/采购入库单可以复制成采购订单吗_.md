# 采购入库单可以复制成采购订单吗?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：嘿唬哈 | 阅读：196 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/604116132789931264)

【业务场景】

采购入库单可以复制成采购订单吗?

【解释说明】

**目前暂不支持。**

采购入库单目前支持复制的单据类型有:采购入库单、销售订单、销售出库单、委托代销单、调拨出库单、移仓单、其他出库单、生产领料单。
![](https://vip.kingdee.com/download/01095f9b7dd1219a4053b039078884e92b13.png)
变通处理方案:

可以将[采购入库单引出为Excel](https://vip.kingdee.com/article/633107224029327360?productLineId=35&isKnowledge=2&lang=zh-CN),把数据粘贴到采购订单的引入模板中然后[导入采购订单](https://vip.kingdee.com/article/542872318003718400?productLineId=35&isKnowledge=2&lang=zh-CN)。
