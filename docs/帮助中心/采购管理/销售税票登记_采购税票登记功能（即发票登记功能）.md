# 销售税票登记/采购税票登记功能（即发票登记功能）

## 1 功能介绍

### **1.1 应用场景**

针对开出发票/收到发票中经常反馈的场景进行了专题优化，涵盖以下场景：

• 场景一：出库后又发生了退货，实际开出的发票金额是扣减退货后的金额（出库与退货合并登记发票、合并匹配税务发票）

• 场景二：商品分录是赠品或者0金额的行无法登记发票信息，导致单据一直显示部分开票

• 场景三：手工登记的发票想要直接修改、批量删除

• 场景四：一张发票多次登记，无法按发票查询已开票信息

### **1.2 适用版本**

星辰3.1所有版本

## **2 ****专业版/标准版的应用**

发票登记优化销售和采购业务都支持，下面举例以专业版的销售开票为例进行讲解（标准版也一样）：

### 2.1 手工登记

支持出库、退货合并登记发票；支持不同客户但发票抬头一样的单据合并登记。以登记销项发票为例。

- 入口一：在销售出库单列表，选中单据行后，点击【销售税票登记】

![上传图片](https://vip.kingdee.com/download/01002c6024187e7a4040b8c13215a00d867a.png)

- 入口二：在销售出库单列表点击【销售税票登记】-【手工录入】，进入销售税票登记界面后填写发票信息，点击“选择单据”补充选择其他单据；

![上传图片](https://vip.kingdee.com/download/0100e1ddb0d1d8064582a25bfd6daa26584d.png)

- 入口三：在销售管理-开票登记表（2024/6/27后更名为“销售税票登记表”）点击新增，选择客户、币别后进入税票登记页面，填写发票信息、选择单据之后进行登记开票。具体流程见下方。

第一步，点击销售税票登记表（2024/6/27后更名，更名前为“开票登记表”）

![](https://vip.kingdee.com/download/01098acca9a40e484338b7b2b5d8f77e8f89.png)

第二步，在销售税票登记列表，点击“新增”

![](https://vip.kingdee.com/download/010933d882d8c10144f8bda7a9557ec0dad0.png)

第三步，选择客户、币别

![](https://vip.kingdee.com/download/0109e0d7cbe2aacd4c6fb7fde2f191678e26.png)

### 关于含有赠品行的单据的税票状态特别说明

“税票状态”（原“开票状态”）属于单据头字段，**该字段是由单据中的非赠品行的****“税票状态”（原“开票状态”）****决定的**，如果单据中的商品行都是赠品行，则分录字段“行税票状态”（原商品开票状态）都是全部是“已开票”，整单的表头字段税票状态（原“开票状态”）也是“已开票”。

## 3 **旗舰版的应用**

### **3.1 应用场景**

星辰旗舰版中，【采购发票】【销售发票】单据，是用于确认应收应付的往来单据，并不是真实的税票。企业对账完成确定采购销售金额后，即可录制【采购发票】【销售发票】单据。税局开出的税票，可能是将多张发票单据合并开具，也有可能是一张发票单据分多次开票。此功能用于将税票信息登记到发票单据中，以便查看发票单据的实际开票情况。

### **3.2 功能特性**

• 旗舰版中，税票登记功能在发票单据列表（销售管理-销售发票列表、采购管理-采购发票列表），采购入库和销售出库单列表不提供税票登记功能

• 专业版/标准版中，税票登记功能在采购入库和销售出库单列表

• 可从税务系统中下载税票后，导入到星辰系统中，用于与业务单据进行匹配。
