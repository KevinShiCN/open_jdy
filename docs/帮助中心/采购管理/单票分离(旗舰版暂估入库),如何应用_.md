# 单票分离(旗舰版暂估入库),如何应用?

【问题描述】

单票分离(旗舰版暂估入库),如何应用?

【相关视频】

[金蝶云·星辰旗舰版功能特性—单票分离&第三方结算介绍](https://vip.kingdee.com/school/458939089681648128?productLineId=35&lang=zh-CN&cosmic_type=video)

【业务场景】

1、采购入库业务:在收到货时录入采购入库单增加仓库库存,收到采购发票后,录入发票,由发票确认应付款;

2、销售出库业务:在发货时录制销售出库单扣减仓库库存,开具销售发票后录入发票,由发票确认收入以及应收款。

【适用版本】

**金蝶云星辰旗舰版**

【解释说明】

**一、采购入库业务**

采购暂估调整方式分“单到冲回”和“差额调整”两种,默认为“单到冲回”****

(1)若在当月收到采购发票,会更新采购入库单的入库成本;没有开票的部分计入暂估成本;

(2)若在次月收到发票,系统会自动生成两张标记为“系统冲回单”的采购入库单。

![](https://vip.kingdee.com/download/01093f4fc2ec585544519d652b1a483990c3.png)
当参数选择“差额调整”时,将生成暂估调整单,具体说明见:[旗舰版如何选择采购暂估调整方式?](javascript:void(0);)

**tips:1.其中业务类型为“入库冲回”的一张用于将原采购入库单的暂估成本冲减;业务类型为“入库”的一张按采购发票单上的的实际开票金额作为正确的入库成本;**

**2.不同的发票类型(专票和普票),对成本的计算方式不同。**

![](https://vip.kingdee.com/download/0109951a4cea9d984fee8f7bae180cbf346c.png)

**二、销售出库业务**

先销售出库,同期或跨期开发票,由销售出库单生成销售发票,销售出库单的成本会带入销售发票。

![](https://vip.kingdee.com/download/010953d5a3815b674ca3985a15bd1f7f0f73.png)

【操作步骤】

**一、采购入库业务**

实际入库时,在【采购管理】-【采购入库】中新增采购入库单,等实际到票后,在【采购管理】-【采购入库查询】中选择需要生成发票的入库单,点击【生成发票】即可;

![](https://vip.kingdee.com/download/010945d79541bd4247f58f92f5bf34e05256.png)**
**

**二、销售出库业务**

实际出库时,在【销售管理】-【销售出库】中新增销售出库单,等实际到票后,在【销售管理】-【销售出库查询】中选择需要生成发票的出库单点击【生成发票】即可。

![](https://vip.kingdee.com/download/01097a27b34ed85840c4bbc499e26995caaa.png)
