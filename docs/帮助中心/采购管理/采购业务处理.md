| 4.5、以销订购 | 企业为了实现零库存或避免库存积压，事先不采购商品放入仓库，当接收到客户订单后，根据销售订单的需求量进行采购；目前，有很多企业甚至会将销售订单发给供应商，由供应商直接发货给客户。在以销定购中，可以根据销售订单的相关数量，计算待采购量后生成采购订单，并能直观看到采购的商品是否已入库，便于及时出库。 | 点击学习 |  |
