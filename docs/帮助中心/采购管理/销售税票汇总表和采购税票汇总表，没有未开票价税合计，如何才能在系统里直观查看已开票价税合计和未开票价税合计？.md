# 销售税票汇总表和采购税票汇总表，没有未开票价税合计，如何才能在系统里直观查看已开票价税合计和未开票价税合计？

> 作者：进销存赵嘉丽 | 阅读：25 | 有用：1

# **1 问题描述**

销售税票汇总表和采购税票汇总表中没有“未开票价税合计”字段，如何才能在系统里直观地查看已开票价税合计和未开票价税合计？

# 2 适用版本

星辰3.0及以上所有版本

# **3 解决方案**

销售税票登记后，相关状态和金额会反写到销售出库单列表（标准版/专业版）或销售发票列表（旗舰版）中，可在销售出库单列表（标准版/专业版）或销售发票列表（旗舰版）查看税票状态、已开税票价税合计、未开税票价税合计等字段。采购类单据也是同理。
