# 采购/销售发票数量能否大于采购入库单/采购订单或销售出库单/销售订单的数量,进行超额开票?

> 作者：云社区用户55os622 | 阅读：450 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/660648738657749760)

【问题描述】

采购/销售发票数量能否大于采购入库单/采购订单或销售出库单/销售订单的数量,进行超额开票?

【适用版本】

**金蝶云星辰旗舰版**

【解释说明】

下面以采购业务举例说明,销售业务同理。

采购发票的数量不允许大于采购入库单数量,但是可以大于采购订单的数量,具体由系统参数“允许入库数量及开票数量大于订单数量”控制。

销售业务由参数“允许出库数量及开票数量大于订单数量”控制。

**1、采购发票数量不允许大于采购入库单数量**

为确保了单据与发票勾稽准确性和一致性,当采购发票选择采购入库单为源单 或 由采购入库单下推生成时,发票数量不允许超过源单数量(金额可以超过,金额本身是由发票确定,因此允许不一致);若采购发票数量大于采购入库单时系统会提示“[采购发票 (CGFP-20251215-00001),商品分录第1行] 反写 [采购入库单 (CGRK-20251210-00062),商品分录第1行]: 不允许超额开票!”

![](https://vip.kingdee.com/download/010907e9bf4f0a77427f8020a26c685f2ba4.png)

如发票数量有超出部分,需要手动新增录入采购发票,等对应超额数量部分商品出库后,再生成采购入库单。确保单据与发票的数量一致。

**2、采购发票数量是否允许大于采购订单数量**

依次点击【系统设置】-【系统参数】-【进销存参数】在采购参数中检查“允许入库数量及开票数量大于订单数量”是否勾选。

![](https://vip.kingdee.com/download/01091b164012dbe24946854e1466f865d6cd.png)

**2.1 勾选参数**

勾选该参数表示采购发票数量是否允许大于采购订单数量,由采购订单关联生成采购发票时修改发票数量大于订单数量,发票可以正常保存。

![](https://vip.kingdee.com/download/0109389a0e40e5f14475ada73255f6879fa9.png)

**2.2 不勾选参数**

若不勾选该参数,则采购发票数量不允许大于采购订单数量,否则系统会提示“[采购发票 (CGFP-20251215-00002),商品分录第1行] 反写 [采购订单 (CGDD-20251016-00001),商品分录第1行]: 不允许超额开票”

![](https://vip.kingdee.com/download/0109c70f3c8c63ef4013b668b97f933a6865.png)
