# 负库存检查的应用场景

# **适用版本**

星辰3.1以上所有版本

# 1 非浮动单位商品（单一单位商品或固定换算率多单位商品）产生负库存

## 1.1 控制逻辑

如普通单位商品，系统库存为100，销售数量为101。

![上传图片](https://vip.kingdee.com/download/010076472a0e59c34b1f93a46222c3831f39.png)

- 若库存更新的方式为“保存时更新”，当出现负库存，系统会在单据保存时出现提示并无法保存
- 若库存更新的方式为“审核时更新”，当出现负库存，单据依然可以正常保存，但在审核时系统会出现提示并无法审核成功

## **1.2 解决办法**

出现负库存提示后，检查若实际实物有库存，可通过新增盘点单盘点，调整为正确的库存信息后再进行出库操作，若实际库存确实不够，需要再通过采购入库或者其他入库增加库存后再进行出库。

# 2 浮动多单位商品产生负库存

## **2.1 **控制逻辑——**库存预警与异常检查**

![上传图片](https://vip.kingdee.com/download/01001fd66e75805f4a7282d2680f3f59f734.png)

浮动单位商品做单据时不仅会检查负库存，还会优先根据库存控制策略进行校验，关于库存预警与异常检查的说明，详见知识：

[当您在保存/审核单据时，出现负库存提示弹窗，该如何处理呢？](https://vip.kingdee.com/link/s/Zfx4J)

>
单据检查判断优先级：库存异常检查优先级高于仓库负库存检查（浮动单位库存异常会优先于负库存检查；简单来说，仓库负库存参数控制基本单位的检查，如果是浮动单位负库存，则要看库存异常检查参数是否已控制）

下面通过同一例子来分别演示库存控制策略实现效果。

示例：若库存更新的方式为“审核时更新”，当前浮动多单位商品基本单位可用库存为620，辅助单位库存为79。

![上传图片](https://vip.kingdee.com/download/01007caa57d9b3e4492ba8ead43fac57075e.png)

## **2.2 严格控制——****若系统参数中选择了****严格控制****，则单据保存或审核不成功**

### **2.2.1 ****基本单位与浮动单位在扣减库存后一个为正一个为负**

当新增销售出库单，录入基本单位数量为600，辅助数量为80，审核单据时会提示“单据XXX商品XXX在仓库XXXX库存扣减后，“可用库存、即时库存”的“基本库存和浮动基本库存”存在异常：一个为正一个为负。”

![上传图片](https://vip.kingdee.com/download/0100fd7a39ea3f914b0b90cd04256e9dd545.png)

### **2.2.2 ****基本单位与浮动单位在扣减库存后一个为0，一不为0**

当新增销售出库单，录入基本单位数量为620，辅助数量为78，审核单据时会提示“单据XXX商品XXX在仓库XXXX库存扣减后，“可用库存、即时库存”的“基本库存和浮动基本库存”存在异常：一个为零一个不为零。”

![上传图片](https://vip.kingdee.com/download/01008718df0227974a9ba45a0d9a9a447957.png)

### **2.2.3 解决办法**

出现负库存提示后，若实际实物有库存，可通过新增盘点单盘点，调整为正确的库存信息后再进行出库操作；若实际库存确实不足，需通过采购入库增加库存后再进行出库。

## 2.3 不予控制——若参数选择不予控制，做单据的时候可以正常保存或审核

### **2.3.1 ****基本单位与浮动单位在扣减库存后一个为正一个为负**

当新增销售出库单，录入基本单位数量为600，辅助数量为80，单据可以审核成功

![上传图片](https://vip.kingdee.com/download/0100229b8a7473554c29ae537b9ad05aa27b.png)

到库存查询表中，可以看到基本单位库存为正数，但辅助单位库存已经为负数

![上传图片](https://vip.kingdee.com/download/0100c1b196c304024b9d8ad68d501c5318f0.png)

### **2.3.2 ****基本单位与浮动单位在扣减库存后一个为0，一个不为0**

当新增销售出库单，录入基本单位数量为620，辅助数量为78，单据可以审核成功

![上传图片](https://vip.kingdee.com/download/010022e336830e3b4fe2b06429fb82a7a01d.png)

到库存查询表中，可以看到基本单位库存为零，但辅助单位库存不为零

![上传图片](https://vip.kingdee.com/download/0100043f9d36f4ce4c65af8a2ec22f9596c2.png)

### 2.3.3 建议

在库存控制策略为“不控制”的时候，如果不希望库存查询表中出现任何负库存，或不希望基本单位库存出完了，但是辅助单位库存还有，需要结合企业管理库存需要，或是修改系统参数控制，或是通过盘点单将商品的基本单位或浮动单位库存调整后，以保证单据库存更新后值为大于0后再进行出库类单据的保存和审核。
