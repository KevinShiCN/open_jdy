# 采购订单汇总表如何使用?

> 作者：winnie_wen | 阅读：1268 | 有用：1

【问题描述】

采购订单汇总表如何使用?

【解释说明】

统计一段时期内商品**采购订单**的总体情况,汇总依据可选择“商品,供应商,商品类别,仓库”。

![](https://vip.kingdee.com/download/010927a9a22cf0b14cb4a09e0a7aaa3d095a.png)

按不同的汇总依据查询时,报表有不同的展现形式,当选择汇总依据为供应商,商品类别,仓库时,还可以通过勾选“显示商品明细”来实现供应商+商品,商品类别+商品,仓库+商品的多维度的汇总统计。

当统计到商品维度时,若系统启用了辅助属性管理,可以通过勾选“显示辅助属性”来统计到商品的属性维度。

查询条件及界面:

 ①供应商、商品、仓库如果空表示查找所有供应商、商品、仓库,可分别打开各选择框进行勾选;审核状态为空,表示查询已审核单据和未审核单据。

 ②日期范围可以任意调整;默认显示本月的第一天到当天;

![](https://vip.kingdee.com/download/01096829d439facc44628de63af200119d55.png)

可以进行引出;支持PDF打印和插件打印;可以进行列设置显示隐藏;可以进行组合排序设置,设置排序字段和排序方式;双击商品行可以跳转到采购订单跟踪表界面查看订单的入库状态和其他信息。

![](https://vip.kingdee.com/download/010969930cd62e174f819ce50f55341be6d1.png)
