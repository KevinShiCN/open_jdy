# 收料通知单/退料通知单的业务类型支持编辑吗?

> 作者：胡大大 | 阅读：18 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/723705126078350592)

【问题描述】

收料通知单/退料通知单的业务类型支持编辑吗?

【说明】

系统的收料通知单/退料通知单中的业务类型字段仅支持采购和委外,不支持自定义业务类型
