# **2 字段说明**

## 2.1 修改税率计算依据

**适用场景：**税率发生变化时，由用户自定义重算金额的计算规则

**适用单据：**进销存业务单据，如订单、出入库单等

**功能含义：**当税率发生变化时，可选择应付金额不变重算不含税金额，还是不含税金额不变重算应付金额

## 2.2 费用分摊规则

| 分摊规则 | 涉及单据 |
| --- | --- |
| 商品价税合计 | 采购入库，销售出库，销售退货 |
| 总净重总毛重总体积 | 采购入库，销售出库，销售退货，委托代销结算，异价调拨出库，异价调拨入库 |
| 数量辅助数量基本数量 | 采购入库，销售出库，销售退货，委托代销结算，异价调拨出库，异价调拨入库，组装单，拆卸单 |
| 调出金额 | 异价调拨出库 |
| 调入金额 | 异价调拨入库 |

用户可以按需选择。当本次销售的多种商品，计量单位差异较大，比如有的商品单位是件，有的商品单位是斤，此时不适合选择“数量”的分摊方式。

## 2.3 币别分摊规则

**适用场景：**如外贸公司一笔采购或销售业务经常会产生国内国外的费用，在分摊时，用原币或本位币分摊可能会造成商品行原币和本位币的费用分摊合计与费用弹窗的合计不一致，导致生成凭证不平，因此系统提供两种分摊方式

**规则说明：**

- 按原币分摊，单据保存时检查商品行费用分摊合计与总费用原币一致；商品行费用本位币合计与总费用本位币合计可能不一致
- 按本位币分摊，单据保存时检查商品行费用分摊本位币合计与总费用本位币一致；商品行费用原币合计与总费用原币合计可能不一致

**使用建议：**

- 当费用币别与本单业务币别一致时，建议选择**按原币分摊**
- 当费用币别与本单业务币别不一致时，建议选择**按本位币分摊**
