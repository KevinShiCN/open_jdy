# 移动端采购管理整体介绍

1 产品概述

### 1.1 产品介绍

采购管理是金蝶云·星辰的核心功能之一，通过采购申请、采购订单、采购入库、采购退货、供应商价格等功能，对采购全过程进行有效的控制和跟踪。可以帮助企业的采购人员完成供应商评估、采购订单管理、采购成本控制、采购合同执行、等一系列采购管理事务，并通过实时数据分析，让采购相关的人员及时了解采购过程中每个环节的准确情况和数据信息，更好地管理采购业务，优化供应链效率。

### 1.2 适用版本

星辰3.0以上标准版/专业版/旗舰版/所有版本

### 1.3 业务场景

1）支持直接采购业务场景：

- 适用于企业直接从供应商处采购原材料、商品等，包括采购订单的创建、审批、跟踪和入库处理。

2）支持计划采购业务场景：

- 适用于企业根据生产计划或销售预测进行的批量采购，包括采购计划的制定、采购订单的批量处理和供应商的交货期管理。

3）支持采购退货业务场景：

- 适用于企业在收到不符合质量标准或多余库存时，向供应商发起的退货流程，包括退货申请、审批、退货处理和退款跟踪。

4）支持供应商协作业务场景：

- 适用于企业与供应商之间的信息共享和协作，包括企业互联

5）支持采购成本分析业务场景：

- 适用于企业对采购成本进行监控和分析，包括成本比较、预算控制和成本优化建议。

通过这些业务场景的支持，金蝶云·星辰的采购管理功能能够满足不同企业在采购过程中的多样化需求，帮助企业提高采购效率，降低成本，增强供应链的竞争力。
