# 订单状态何其多，关联关系慢慢说

订单状态何其多，关联关系慢慢说

![上传图片](https://vip.kingdee.com/download/0100377713f548a3418c9277198ea9dcf09b.png)

      小敏作为刚入职的采购，看到采购订单上的各种状态，真是眼花缭乱。于是她向公司的前辈老李请教。老李也丝毫不吝啬，将其所学倾其所有告诉了小敏，那小敏都学到了什么，我们一起来看看吧！

1、 订单执行状态

1） 采购订单全部下推采购入库单“保存”后，订单的“执行状态”和“行执行状态”都是“全部执行”。

![](https://vip.kingdee.com/download/01000869605d3e104779bb69116d7f943235.png)

![](https://vip.kingdee.com/download/0100042cd046426746a4837579e06bbca5f3.png)

![](https://vip.kingdee.com/download/01003ddd78286d7640a79b3de06521851379.png)

2） 采购订单部分下推采购入库单“保存”后，采购订单的“执行状态”是“部分执行”，“行执行状态”按商品行入库数量显示“全部执行”、“未执行”、“部分执行”。

![](https://vip.kingdee.com/download/0100331044cffb984a77b9a4eeec9b4bd3ec.png)

![](https://vip.kingdee.com/download/0100dd5d688e57984917b85b63c1d381683b.png)

![](https://vip.kingdee.com/download/01009644ce4c44bc45bb9852db42c3f8482f.png)

2、 订单入库状态

1） 当库存更新方式为“保存时更新”。

![](https://vip.kingdee.com/download/0100fc6f25421df44449b793840b4f2e22b5.png)

i）采购订单全部下推采购入库单“保存”后，订单的“入库状态”和“行入库状态”均是“全部入库”。

![](https://vip.kingdee.com/download/01007bdd0812cf434cefad16ec0baca79d44.png)

![](https://vip.kingdee.com/download/0100171b6ed24e9a47aa9d282ddb3bd36c3a.png)

ii）采购订单部分下推的采购入库单“保存”后，采购订单的“入库状态”是“部分入库”，“行入库状态”按商品行入库数量展示“全部入库”、“未入库”、“部分入库”。

![](https://vip.kingdee.com/download/010055f3667d773b4ced98a8763d4e4893b4.png)

![](https://vip.kingdee.com/download/0100d0455a18df6247488cdf9bcbb976f1fd.png)

![](https://vip.kingdee.com/download/0100edf72aed60814174b9853d5d258aa00e.png)

2） 当参数库存更新方式选“审核后更新”。

![](https://vip.kingdee.com/download/010025fc261553ad4e66bb99ad2fe9581543.png)

i) 采购订单全部下推采购入库单“保存”后，采购订单的“入库状态”和“行入库状态”均显示“未入库”。

![](https://vip.kingdee.com/download/01000463abdbd94e4365970fc18cfde0490d.png)

![](https://vip.kingdee.com/download/01000afae0486be04a3ba70ef7a7b41d6605.png)

ii) 全部下推的采购入库单“审核”后，采购订单的“入库状态”和“行入库状态”均显示“全部入库”。

![](https://vip.kingdee.com/download/0100b6ab097d41db48e49b00fd98a03806dc.png)

![](https://vip.kingdee.com/download/01001a5426e9825f49c9863d3f3a518da833.png)

iii) 采购订单部分下推采购入库单“保存”后，“入库状态”和“行入库状态”均显示“未入库”。

![](https://vip.kingdee.com/download/010092de77512b914bfdaaf1b513db9ce95e.png)

![](https://vip.kingdee.com/download/0100697f32a624584680a074d4eaf980e011.png)

iv) 部分下推的采购入库单“审核”后，“入库状态”显示“部分入库”，行入库状态按商品行实际商品入库数量显示“未入库”、“全部入库”、“部分入库”。

![](https://vip.kingdee.com/download/010066757dbbe9ae45378a2a8ec175e14ad3.png)

![](https://vip.kingdee.com/download/0100b7ba8094c7374032b413a2affe6266c0.png)

3、 订单关闭状态

1） 自动关闭

i） 采购订单全部下推采购入库单“保存”后，采购订单“关闭状态”和“行关闭状态”均显示“已关闭”。自动关闭的订单不能在启用。

![](https://vip.kingdee.com/download/0100d7f746d32f64481b9944f529108a4c24.png)

![](https://vip.kingdee.com/download/01007e21f500c790405daf8fe39bdd227c3c.png)

ii) 采购订单部分下推采购入库单“保存”后，采购订单“关闭状态”会显示“未关闭”，“行关闭状态”会根据商品实际下推数量显示“未关闭”和“已关闭”。

![](https://vip.kingdee.com/download/010023e17cde659a4f048e4bc6042f0a6980.png)

![](https://vip.kingdee.com/download/010000c92c04137c41d5ae401e62fb3df43e.png)

2） 手动关闭

当采购订单没有全部下推但又不用继续下推，一般会选择手动关闭。手动关闭的订单可以再次启用并下推。

i）采购订单所有行手动关闭。采购订单的“关闭状态”和“行关闭状态”均显示“手动关闭”。

![](https://vip.kingdee.com/download/0100ef807c16d57f4b2b96b99a8b39e296bf.png)

ii） 采购订单部分行手动关闭

先列设置，拆分行显示，在勾选需要关闭的行，关闭即可。采购订单的“关闭状态”是“未关闭”，“行关闭状态”会显示“手动关闭”和“未关闭”。

![](https://vip.kingdee.com/download/01005c008d8cb5f74f99b65c108a9ec26a79.png)

![](https://vip.kingdee.com/download/0100809e761fe9b9420da3fec237be0d7438.png)

4、 订单核销状态

1）采购订单未下推之前，订单上核销字段显示。

![](https://vip.kingdee.com/download/01007ce8b928135b4890a4dd4b7f2d0a4a24.png)

2）采购订单下推采购入库单审核，未付款。“入库核销状态”和“行核销状态”均显示“未付款”。

![](https://vip.kingdee.com/download/01002e7db2aaff2c4814b810b3579bba5d32.png)

3） 采购入库单通过付款单付款。

i) 全部付款，付款单“保存”后，订单上的“入库核销状态”，“行核销状态”均显示“全部付款”。

![](https://vip.kingdee.com/download/01002638d8b05be84c9384f7bc080d083bdc.png)

![](https://vip.kingdee.com/download/0100a3b8da0edee0477da2114f311fd1bf4e.png)

ii) 部分付款，付款单“保存”后，订单上的“入库核销状态”显示“部分核销”，“行核销状态”按付款金额分摊到商品行显示“全部付款”、“未付款”。

![](https://vip.kingdee.com/download/01006e838b7ce67446a4879eaafff5471e46.png)

![](https://vip.kingdee.com/download/0100ddcb8f1a8fc542d8bb8a59d6c5571b27.png)

5、 本次定金状态，累计预付状态，预付审核状态

假设：采购订单成交金额10000。

i) 预付定金6000，生成预付款单“未审核”，“本次订金状态”和“累计预付状态”均显示“未付”，“预付核销状态”显示“已预付未审核”

![](https://vip.kingdee.com/download/0100a0734cd5cb3f4b7e827bd102b921d6ad.png)

![](https://vip.kingdee.com/download/010022129d06aaff4a8fafc6ae4046744720.png)

ii）预付6000的预付款单“审核”。“本次订金状态”显示“已付”，“累计预付状态”显示“部分已付”，“预付核销状态”显示“已预付部分审核”

![](https://vip.kingdee.com/download/010069e818f7185f4177acab597c62360f46.png)

iii）预付款单选源单，预付剩下的4000，预付款单“未审核”，“累计预付状态”显示“部分已付”，“预付核销状态”显示“已预付部分审核”。

![](https://vip.kingdee.com/download/01009d2891ef95f64ac8a3f5095fdf0930f2.png)

![](https://vip.kingdee.com/download/0100aadc1e6f215440c7a17332105c76db30.png)iv）预付4000的预付款单“审核”。“累计预付状态”显示“全部已付”，“预付审核状态”显示“已预付已审核”。

![](https://vip.kingdee.com/download/010008795a37e4f646f497f95f4e5b54e387.png)

![](https://vip.kingdee.com/download/01003e50f9f38f6c4abbaab66a4fbb4aa2e8.png)

通过小李的指导，小敏对订单字段不在迷茫了。
