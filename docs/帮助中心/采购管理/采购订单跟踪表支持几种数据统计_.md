# 采购订单跟踪表支持几种数据统计?

【问题描述】

采购订单跟踪表支持几种数据统计?

【操作步骤】

采购管理-采购订单跟踪表。目前支持按单据统计和按商品统计两种方式。

1、按单据统计指**按单据日期排列,一张单据的多行商品排在一起**,用于跟踪每张订单的入库状态;如下图效果所示

![](https://vip.kingdee.com/download/0109f73d227197a44e10b907b769605513f1.png)

2、按商品统计指将**不同订单中的相同商品排在一起**,用于跟踪每种商品的入库状态。![](https://vip.kingdee.com/download/01099f31332fadc44f2abcea0342a0683c3e.png)
