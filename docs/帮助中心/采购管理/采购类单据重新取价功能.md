# 采购类单据重新取价功能

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：功能介绍, 进阶技巧, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：2867 | 有用：5
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/139058116314103552)

# **1 应用场景及价值**

当商品的采购价格变化后，未执行完成的采购单据需要按照新的定价更新价格，以更好地反映实际交易金额。例如在采购订单后，在下推入库单前，价格发生了变化，需要对下推的入库单进行重新取价。

# 2 适用版本

星辰3.1以上所有版本

# 3 功能特性

采购类单据包括采购订单、收料通知单、采购入库单、退料通知单、采购退货单、采购发票

# **4 应用流程**

1）首先要在系统设置>系统参数>进销存参数>采购参数中开启允许重新取价的功能，勾选“已审核的采购类单据允许重新取价”

>
注意：重新取价在以下情况中是不允许的。

标准版/专业版：采购订单已关闭、已入库，已开票；采购入库单已生成凭证、已付款、已开票、有退货；采购退货单已生成凭证、已退款，已开票，则不允许重新取价

旗舰版：采购订单已关闭、已入库，已开票；收料通知单已有入库、有退料；采购入库单已生成凭证、已付款、已开票、有退货；采购退货单已生成凭证、已退款，已开票；采购发票已付款、已生成凭证、已生成红字发票、已对账，则不允许重新取价

2）打开采购类的单据列表，选中要改价的单据，点击更多>重新取价即可重新取价。重新取价的依据来自基础资料>价格取数优先级>采购类单据取价中的优先级顺序

![上传图片](https://vip.kingdee.com/download/0100dcbd05ee1e7743da9a7798eb0c039d97.png)

单据列表中可以操作批量重新取价。

>
说明：批量操作也是有单据上限的，单次批量操作不得超过100张单（合计不超过1000行分录），单据详情适合单张单操作，但是商品分录也不能超过1000行。

# **5 特别说明**

采购订单下推生成采购入库单时是携带源单上的价格，修改采购入库单上的数量也不重新取价，但是点击“重新取价”后，不管是不是源单下推，都会更新当前单据的价格。
