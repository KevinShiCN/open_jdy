# 插件打印直接点击“打印”时，总是出现“将打印输出另存为”的页面，该怎么处理？

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：2698 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/363024275483010304)

**【问题概述】**

打印方式选择“插件打印”，直接点击“打印”时，总是出现“将打印输出另存为”的页面，该怎么处理？

![上传图片](https://vip.kingdee.com/download/0100a429dc8c6190402d876f7c08b32db076.png)

![上传图片](https://vip.kingdee.com/download/0100a0d9b99f2db04d03bd5b078364e68d31.png)

**【原因说明】**

这是因为插件打印预览中的“目标打印机”没有选择真实的打印机，而是选择了虚拟打印机“Microsoft Print to PDF”。虚拟打印机的主要作用就是把打印文件另存为本地的PDF，所以点击“打印”时，就等同于把PDF另存为，因此系统才会弹出“另存为”的窗口。

![上传图片](https://vip.kingdee.com/download/0100e9e572d4bad844399b42bb5db71be462.png)

**【解决方案】**

打开“打印>打印预览”，在“目标打印机”下拉列表中选择真实的打印机，然后点击“打印”，下次就不会再弹出“另存为”的窗口了。

![上传图片](https://vip.kingdee.com/download/01006c3b43af8a954bffacb10838ea843d79.png)

**【适用版本】**

星辰3.1以上所有版本

星辰4.0以上打印插件
