#  金蝶云·星辰，支持只能修改/删除自己录入的单据吗？支持反审核与审核人不能为同一人吗？支持审核人与制单人不能为同一人吗？

> 作者：小微微 | 阅读：4014 | 有用：25

问题描述：

金蝶云·星辰，支持只能修改/删除自己录入的单据吗？支持反审核与审核人不能为同一人吗？支持审核人与制单人不能为同一人吗？

解决方案：

支持。金蝶云·星辰支持对进销存、资金单据设置单据操作控制

## 应用场景

适用于需要做严格的岗位操作权限隔离的场景，例如只允许子账号修改/删除自己录入的单据，反审核与审核人不能为同一人，制单人与审核人不能为同一人等等。

## 功能特性

支持设置：

- 是否“审核人与制单人可为同一人”；
- 是否“反审核人与审核人可为同一人”；
- 是否“只能修改/删除本人录入的单据”；
- 控制单据审核前是否允许修改单据编号

## 适用版本

星辰2.0专业版及星辰3.0以上所有版本

## 应用流程

1、打开【系统设置-个性化设置-单据操作控制】（如果子账号需要编辑这个菜单，需要管理员先做功能授权）

2、管理员或者有权限的账号针对具体单据设置好单据操作控制之后，**任何账号（含管理员）**在具体操作审核、反审核、修改、删除单据时会判断是否符合单据操作控制。

3、单据操作控制开启后的页面提示效果：

| 控制项 | 关闭后提示效果（点击图片可放大查看） |
| --- | --- |
| 审核人与制单人可为同一人 |  |
| 控制项 | 开启后提示效果（点击图片可放大查看） |
| 反审核人与审核人只能为同一人 |  |
| 只能修改/删除本人录入的单据 |  |

## 应用说明

由于单据操作控制是校验制单人、审核人等字段（如下图），**并不是根据用户权限来判断的**。也就是说，如果制单人不是管理员账号，是子账号，那么当开启了单据操作控制后，管理员也是会受到操作限制的。

解决办法：如果管理员需要操作非本人操作的单据，只能先把单据操作控制给关闭了再操作。
