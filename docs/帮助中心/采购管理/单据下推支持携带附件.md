# 单据下推支持携带附件

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：快速入门, 功能介绍, 新特性
> 作者：进销存熊诗琴 | 阅读：784 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/715901848904958720)

## 1 简介  

### 1.1 功能介绍

销售订单上，将和客户签订的合同作为附件上传到系统，下推销售出库单时将附件一并携带到出库单上，避免重复上传操作或者因为遗忘忘记上传。

### 1.2 适用版本

星辰3.1以上所有版本

## **2 功能特性**

### 2.1 单据下推支持携带附件

- 以销售订单为例，如果想要将业务员在订单上上传的附件携带到销售出库单上，则在单据详情的附件栏右侧勾选参数【携带附件到下游单据】：

![](https://vip.kingdee.com/download/0100bc6300f3a92f472fa292be367b1af786.png)

- 勾选之后，在订单上上传的附件，就会自动携带到出库单上：

![](https://vip.kingdee.com/download/010003c6c9f7aa2c41b090a271f6398e6418.gif)

### 2.2 支持携带附件的单据

| 单据类型 | 源单-添加参数的单据 | 目标单-获取附件的单据 |
| --- | --- | --- |
| 采购类 | 采购申请 | 采购订单 |
| 采购订单 | 收料通知单，采购入库 |  |
| 收料通知单 | 采购入库 |  |
| 采购入库 | 采购发票 |  |
| 销售类 | 销售报价 | 销售订单 |
| 销售订单 | 发货通知单，销售出库单，委托代销单 |  |
| 发货通知单 | 销售出库 |  |
| 销售出库 | 销售发票 |  |

### 2.3 特别说明

- 该参数是按用户维度的，比如小花和小金都是销售员，那么她们可以根据自己的实际情况决定是否要勾选本参数；
- 如果是合并下推的场景：比如销售订单001和订单002合并生成销售出库单，则001和002的附件都会携带到出库单上；
- 如在系统参数中，设置了单据最多上传的附件数，则从源单携带的附件不控制是否超额，但是再新增附件，就会按携带源单的附件数+本单上传的附件数之和控制；比如：用户设置的附件个数上限为5个，当销售订单001和订单002分别上传了3个附件（合计6个），若将这两张订单合并下推生成销售出库单，系统将所有附件（共6个）携带至出库单。此时由于出库单上的附件总数已超过上限，所以控制不允许再新增附件。
