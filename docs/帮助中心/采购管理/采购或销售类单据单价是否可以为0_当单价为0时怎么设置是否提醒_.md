# 采购或销售类单据单价是否可以为0?当单价为0时怎么设置是否提醒?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：ylim | 阅读：1020 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/555193411750174720)

【问题描述】

采购/销售类单据单价是否可以为0?当单价为0时怎么设置是否提醒?

【业务场景】

在采购业务中,会有赠品0成本入库的情况,同时也为了防止录错价格,当单价为0时希望系统能够提醒。若入库不填单价表示该商品没有成本,日后出库也不会核算到成本。

销售业务中为了防止销售开单人员忘记录入单价,当销售的单价是0时希望能提醒或者控制不允许开单。若出库不填单价表示对应的商品不产生收入。

【解释说明】

**采购类、销售类单据单价可以为0,且不影响结账;**

**可以通过设置进销存参数,控制采购类销售类单据单价为0时提示或者不能保存审核,后续若不想提示了也可以选择不提示。**

**tips:若单据上商品开启“赠品”按钮,不受此参数影响,可正常保存审核。**

【操作指引】

点击【系统设置】>【系统参数】

![](https://vip.kingdee.com/download/0109b4521975098843eba6f1ffe90f232c1b.png)
**1、采购类单据参数控制**

在【进销存参数】>【采购参数】中,“若您在新增采购类单据时采购单价为0, 您希望”这一项设置;

1)当采购类单据单价为0时希望系统提醒,但是可以继续开单选择“提示,允许继续操作”

2)当采购类单据单价为0时希望系统提醒且不允许继续开单,选择“提示,不允许继续操作”

3)若希望不提醒,则在参数中修改为“不控制”

设置好之后点击右上角<保存>按钮保存参数

![](https://vip.kingdee.com/download/0109d9da829629e147c6b4c119bda0e6de80.png)
可以选择被控制的采购类单据有:采购订单、采购入库、采购退货

![](https://vip.kingdee.com/download/0109bfce61918ddb4631a71b365d69a64a00.png)
**2、销售类单据参数控制**

在【进销存参数】>【销售参数】中,“若您在新增销售类单据时销售单价为0, 您希望”这一项设置;

1)当销售类单据单价为0时希望系统提醒,但是可以继续开单选择“提示,允许继续操作”

2)当销售类单据单价为0时希望系统提醒且不允许继续开单,选择“提示,不允许继续操作”

3)若希望不提醒,则在参数中修改为“不控制”

设置好之后点击右上角<保存>按钮保存参数

![](https://vip.kingdee.com/download/0109e4afdfdf327941ddb399641d462ead0c.png)

可以控制提示的单据,不同版本产品对应的单据类型不同,具体可自行在账套参数中查看

![](https://vip.kingdee.com/download/010928d7bce342834b00b0214a299ea6fc6a.png)
