# 星辰序列号数量录入及一次批量生成的最大限制是多少？

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：快速入门, 精华汇总, 功能介绍
> 作者：进销存赵嘉丽 | 阅读：724 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/371690735386257152)

- 录单数量：

**进销存一张单据最多支持录入2W个序列号。**

例如：单据001里，A商品（1w个序列号）+B商品（5000个序列号）+B商品（5000个序列号）,或者A商品（共2w个序列号）。

![](https://vip.kingdee.com/download/0109fe613f953e73441b872a89f677d768b8.png)

- # SN弹窗的批量生成： 一次最多生成**1000个**序列号，当超过1000个序列号时，需要分多次点击“添加”，添加完毕再点击“确定”即可录入到单据分录上。

![](https://vip.kingdee.com/download/0109e62f600f2e3f498f8fee3b523574eca8.png)

**温馨提示：这里的单次创建序列号的个数是考虑了性能做的限制，暂时不会放开上限，如您的商品单次入库超过了1000个序列号，建议分多次添加即可。**
