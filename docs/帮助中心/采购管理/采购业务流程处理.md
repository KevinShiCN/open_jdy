# 采购业务流程处理

【业务场景】

采购业务处理的流程有哪些?

【适用版本】

金蝶云星辰专业版、金蝶云星辰标准版

【解决方案】

公司需要对外采购如办公用品、成品、原材料、设备等,生产的采购订单执行、收货及付款的全流程,可在采购管理提交**采购申请**(**专业版**)、**采购订单**,或者做**采购入库单**直接购买入库

登录系统后在首页界面点击右下角的【业务流程图】打开业务流程图,可查看采购流程的整体介绍,采购申请单与采购订单均为非必录单据,可根据公司业务需求选择。

1、采购申请单(非必录):[如何录入采购申请单?](https://vip.kingdee.com/link/s/ZLjKt)

2、采购订单(非必录):[如何新增采购订单](https://vip.kingdee.com/link/s/ZLjKI)

3、采购入库单:[采购入库单如何录入?](https://vip.kingdee.com/link/s/ZLj6e)
