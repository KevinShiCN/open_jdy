# 批次保质期应用-采购单据

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：快速入门, 功能介绍

1、 采购入库，可自动生成批次

在系统设置>系统参数>基础资料参数>商品参数中，勾选“入库批次自动生成”后，则在录入采购入库单时，输入商品信息后，系统会根据批次编码规则，自动生成批次，可编辑：

![](https://vip.kingdee.com/download/01005b8db2053c4f40bd9451f9ce7f7b7bdd.png)

点击批次图标![](https://vip.kingdee.com/download/01005be288484571442ba6a99de97afe9fbb.png)，可批量选择批次（一次最多只能选择20个批次），录入同批次的入库数据，也可在弹窗内点击“按规则填充批次”，则会根据批次编码规则及填充规则，自动生成新的批次号：

![](https://vip.kingdee.com/download/01004116a3b94640480eaa7767c2467e12d9.png)

2、保质期商品应用：到期日计算、重新计算保质期

（1）对于保质期商品，采购入库时需要录入生产日期，录入后，系统会自动根据商品信息中的保质期单位、保质期，计算出到期日：

![](https://vip.kingdee.com/download/0100ff3383f29cdf408999083d25e4788bf7.png)

生产日期可使用键盘的“F6”快捷键批量向下填充。

使用方法：先在日期控件选择一个日期，假设是2022-11-01，然后将鼠标光标点击到单元格外任意的空白处（目的是确认选择的日期），然后再将鼠标光标定位到日期控件里，直接点击键盘的F6键，即可批量向下填充（详见下面动图演示。）

![](https://vip.kingdee.com/download/0100884667de27084826b440d00a18b5bb14.GIF)

（2） 其中“到期日”的计算方法有两种，在商品信息中选择计算方式：

![](https://vip.kingdee.com/download/0100b47f6b86c6e74e1780d9084eee2d41f2.png)

根据实际需要选择对应的计算方法即可

（3） 录入生产日期后，到期日也可手动编辑，可在商品信息中选择修改到期日时是否重新计算保质期：

![](https://vip.kingdee.com/download/0100fd320add1009410686a2f1861c4f850a.png)

注意：选择“重新计算到期日后，对不同保质期单位的处理逻辑不一样：

①“保质期单位=天”时，会根据生产日期、到期日更新“保质期”

②“保质期单位=月或者年”时，会根据到期日、保质期更新“生产日期”

③详细说明可参考商品信息中“修改到期日”的Tips说明：

![](https://vip.kingdee.com/download/0100cb9119f1d3634862818736fe363a4db5.png)
