# 设置了商品价格为什么开单带出来的价格不对？

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题, 快速入门
> 作者：进销存赵嘉丽 | 阅读：1773 | 有用：15
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/113693272018868992)

# 【问题概述】

设置了商品价格为什么做单带出来的价格不对？

# 【原因说明】

开单时带出价格主要由维护的价格资料以及价格取数优先级决定。

# 【解决方案】

1）打开基础资料>价格设置>价格取数优先级，检查设置项，如果都没有启用，需要启用，确认好取数来源及其优先级

![上传图片](https://vip.kingdee.com/download/01007b7dd82fa717443c8eaa855a6269159c.png)

2）根据优先级检查对应的价格资料设置是否有做维护。

最近价字段：取值已审核的业务单据，可以在商品价格资料列表中查看。

价格资料类：取自商品/客户/供应商价格资料

具体的价格策略设置方法及应用场景可以查看以下知识。

采购类：[采购价格策略整体介绍](https://vip.kingdee.com/article/61561416544752640)

销售类：[销售价格策略整体介绍](https://vip.kingdee.com/article/66466587460309248)

3）确保第一二点都设置好后，再重新开单验证即可。

# 【适用版本】

星辰3.1以上所有版本
