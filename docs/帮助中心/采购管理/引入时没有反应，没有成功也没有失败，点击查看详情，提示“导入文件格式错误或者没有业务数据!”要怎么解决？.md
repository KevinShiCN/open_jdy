# 引入时没有反应，没有成功也没有失败，点击查看详情，提示“导入文件格式错误或者没有业务数据!”要怎么解决？

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：2416 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/338329680330109184)

**【问题描述】**

引入时没有反应，没有成功也没有失败，点击查看详情，提示“导入文件格式错误或者没有业务数据!”是为什么？要怎么解决？

![上传图片](https://vip.kingdee.com/download/0100fc8df4bd48594d18a450f92b040d79c0.png)

**【原因说明】**

系统对引入文件是有格式校验的，系统正常的引入模板的前三行必须要有，如果您在维护数据时，不小心删除模板的前三行任意一行，导致有缺失的行，就会出现上述的问题。

![](https://vip.kingdee.com/download/010062d5012ad10a451b97978644afcd798b.png)

**【解决办法】**

手动增加一行空白行，保证导入文件的前三行一定要有，这样就可以正常引入了。

![](https://vip.kingdee.com/download/0100534804d53ee8483cac4ef2a7c5dfddbd.png)

**【适用版本】**

星辰3.1以上所有版本
