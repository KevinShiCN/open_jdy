# 多个订单合并入库/出库后,采购入库单/销售出库单部分付款/收款后,订单的入库已付款金额或出库已收款金额怎么反写?

【问题描述】

多个订单合并入库/出库后,采购入库单/销售出库单部分付款/收款后,订单的入库已付款金额或出库已收款金额怎么反写?

【解释说明】

多个采购订单合并入库,采购入库单部分付款,订单的入库已付款金额采用“自上而下”分配逻辑,优先分配至上游订单,直至付款金额核销完成。多个销售订单订单合并出库同理。

“自上而下”是指在采购入库单中某商品位于商品信息中的第一行分录,则付款金额优先分配至该商品采购订单中,若付款金额没有核销完,则会递延分配至第二行分录商品的采购订单中,以此类推。
