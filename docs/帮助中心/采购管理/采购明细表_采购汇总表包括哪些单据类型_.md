# 采购明细表/采购汇总表包括哪些单据类型?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：shaoling | 阅读：102 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/713921296006122752)

【问题描述】

采购明细表/采购汇总表包括哪些单据类型?

【解释说明】

采购明细表 / 采购汇总表包含的单据类型,在金蝶云星辰标准版、专业版、旗舰版等系统版本中有所不同。

**1、标准版中**,采购明细表 / 采购汇总表包含的单据类型有:**采购入库单、采购退货单**

![](https://vip.kingdee.com/download/01093d19c7bf35a84dd3a9887b3e65c6e1cd.png)
**2、专业版中**,采购明细表 / 采购汇总表包含的单据类型有:采购入库单、采购退货单、暂估调整单

![](https://vip.kingdee.com/download/0109450b0cc37a0047c0bf6baadb8519ef48.png)
**3、旗舰版中**,采购明细表 / 采购汇总表包含的单据类型有:采购入库单、采购退货单、暂估入库期初单
![](https://vip.kingdee.com/download/0109ff4b154590164a61ae1cfd2ea56fb529.png)

【注意事项】

采购明细表与采购汇总表的数据均不包含采购订单、采购申请单
