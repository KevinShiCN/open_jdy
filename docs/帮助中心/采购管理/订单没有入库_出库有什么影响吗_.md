# 订单没有入库/出库有什么影响吗?

【问题描述】

订单没有入库/出库有什么影响吗?

【解释说明】

1、未入库的采购订单无法生成应付账款记录;未出库的销售订单同样无法生成应收账款记录。

2、未出入库的订单是否影响库存,由系统参数设置决定。

3、未出入库的订单不影响业务结账,可以结账后再按实际业务做出入库。
