# 查询相关汇总报表或单据时，已经勾选了显示列，但是查询结果却看不到对应字段，是什么原因？

> 作者：进销存赵嘉丽 | 阅读：1391 | 有用：10

# 【问题概述】

查询相关汇总报表或单据时，已经勾选了显示列，但是查询结果却看不到对应字段，是什么原因？

# 【解决方案】

当出现上述问题时，按以下步骤来逐步检查：

1）检查报表或者单据的列设置是否开启“显示列”

2）如果已开启但仍不显示，可点击“恢复默认”按钮

![上传图片](https://vip.kingdee.com/download/0100b6b000e7e6804f49bdb8484248ca67a1.png)

3）检查列宽是否显示，点击鼠标右键选择“全部列宽自适应”

![上传图片](https://vip.kingdee.com/download/01006cb78120588c489fa0bfed92f833be0c.png)

4）清除浏览器缓存后重新登录或者更换其他浏览器。

![](https://vip.kingdee.com/download/0100bfd6cf4607b847dda6384b250151b032.png)

>
说明：不同浏览器的功能界面不一样（上图仅供参考），请打开您所用浏览器的设置菜单，找到清除缓存的按钮（一般在隐私中）进行清除，完成后重新登录系统。如前面1、2步均已检查，第3步清除缓存后依然不显示列，请更换其他浏览器（如谷歌chorme）并重新登录。

# 【适用版本】

星辰3.1以上所有版本
