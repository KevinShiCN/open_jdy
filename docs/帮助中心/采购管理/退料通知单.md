# 退料通知单

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：功能介绍, 新特性, 快速入门

1 简介

### 1.1 功能介绍

- 当企业采购的原材料、委外加工回来的产品在检验环节发现质量不符合要求时，需要进行退料处理。

1.2 适用版本

- 星辰·旗舰版

### 1.3 系统路径

- 采购管理-采购单据-退料通知单
- 委外管理-委外单据-退料通知单

## 2 操作指南

1）收料通知单列表，选择判退数量大于0的数据，下推生成退料通知

![](https://vip.kingdee.com/download/0109d4d19d3752ee4ca8ae752e4ece582e04.png)

2）退料通知单中确定本次退料数量等信息，确定退给供应商之后是否要求供应商重新送货，如果需要重新送货，则需要开启表头的“退货重发”参数。

## 3 常见问题

1）对于免检商品，收料后发现质量问题，是否能够从收料通知单中下推退料通知单处理？

不能。免检商品收料之后发现质量问题，需要从收料通知单下推入库单，再从入库单下推退货单，退回给供应商。

2）退料通知单保存之后，没有审核，为什么不能修改“退货重发”？

因为保存的时候需要按“退货重发”是否开启反写源订单（采购订单、委外加工单）的未执行数量，保存成功之后已按保存时的数量进行反写更新，不再允许修改。如果允许修改的话，前后两次保存时录入的数量有差异时，会造成反写回源单的相关数量出现错误。
