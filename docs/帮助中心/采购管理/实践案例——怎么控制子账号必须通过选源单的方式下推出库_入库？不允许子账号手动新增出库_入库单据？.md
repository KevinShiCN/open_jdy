# 实践案例——怎么控制子账号必须通过选源单的方式下推出库/入库？不允许子账号手动新增出库/入库单据？

> 作者：进销存赵嘉丽 | 阅读：1113 | 有用：3

# 1 问题概述

怎么控制子账号必须通过选源单的方式下推出库/入库？不允许子账号手动新增出库/入库单据？

# 2 解决方案

标准版：可以通过对子账号单据的“商品编码”设置字段授权来控制。

专业版/旗舰版：通过 [业务流程设置](https://vip.kingdee.com/link/s/ZbuON)

# 3 应用流程

## 3.1 标准版

下面通过销售出库单为例讲解如何设置：

1） 保证子账号有销售订单和销售出库的功能权限

注意，如果子账号要做下推出库的，一定要勾选“查询”和“新增”（这样才能直接从订单下推出库），并建议开启“修改”权限（即保存单据后还能修改单据内容并且保存），如果只给“新增”，不给“修改”，那么一旦子账号下推并保存单据后，则无法修改内容。

其他权限的控制则取决于用户企业自身的管理要求即可。

![](https://vip.kingdee.com/download/0100f7bee652659748a09a7d54cdb20fe169.png)

2） 打开字段授权-销售出库，点击“选择字段”，勾选“商品分录.商品编码”，点击“确定”

![](https://vip.kingdee.com/download/010031067e163eb347329bd54edf51582787.png)

3） 开启“禁止编辑”，保存。

![](https://vip.kingdee.com/download/0100f328f6efbfcc4d178d5f8f1d15632e57.png)

4） 子账号新增销售出库单，不能手动添加商品编码，只能通过：

- 选源单的方式来添加商品

![](https://vip.kingdee.com/download/0100415feb91a1e44219929b7bc306f5c79c.gif)

- 销售订单列表“出库”来生成出库单，带出商品

![](https://vip.kingdee.com/download/010075944624b8e8442abb2a48ac0ae73552.gif)

通过以上2种方式，从而达到了必须关联源单下推出库的目的。

注：采购或者其他通过源单下推的单据的设置方式同理。
