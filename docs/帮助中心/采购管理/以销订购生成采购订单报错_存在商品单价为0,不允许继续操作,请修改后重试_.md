# 以销订购生成采购订单报错:存在商品单价为0,不允许继续操作,请修改后重试?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：456 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/538807298810887424)

【问题描述】

在以销订购生成采购订单提示报错:“存在商品单价为0,不允许继续操作,请修改后重试”

![](https://vip.kingdee.com/download/010988157b147b844783a3de20e730be3701.png)

【故障原因】

如果在相关订单商品信息没有设置默认采购价,以销订购生成的采购订单时是没有商品单价的,所以当【系统参数】-【进销存参数】-【采购参数】勾选了“采购单价金额为0时:提示,不允许继续操作”,会出现相关提示。

![](https://vip.kingdee.com/download/01095e89bdfba9534dedb9eda0bfda1bd365.png)

【解决方案】

(1)修改参数设置,更改为:[提示,允许操作]或者[不控制];保存之后,重新去生成相关采购订单。![](https://vip.kingdee.com/download/0109430ebf0b16f5483eba9a9f443382965a.png)

(2)维护相关商品信息里的采购价

![](https://vip.kingdee.com/download/0109ad59271de90d4de88e047b7f8a40256e.png)

(3)直接在【以销定购】界面,录入采购价,再生成采购订单

![](https://vip.kingdee.com/download/01095a2d808f91f2447b80286bd02e0d641b.png)
