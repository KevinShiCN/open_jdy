# 暂估调整提示本次开票数量不能超过暂估未开票数量,如何处理?

> 产品：金蝶云·星辰 | 分类：采购管理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：17 | 有用：0

【问题描述】

暂估调整提示本次开票数量不能超过暂估未开票数量,如何处理?

【解释说明】

暂估调整业务的核心适用场景为**采购总数量准确、暂估金额与实际结算金额存在差异**的情形。系统支持按数量分批录入暂估调整单,限制了总的开票数量不能超过暂估开票数量。

【操作指引】

如采购暂估的数量为总数量10,暂估调整录入本次开票数量为12,生成暂估调整单是则系统会出现提示:本次开票数量不能超过暂估未开票数量

![](https://vip.kingdee.com/download/0109c6b5c69013834923b3927ed2f759cad0.png)
若实际入库数量大于原暂估入库单的记录数量,建议按原暂估数量完成暂估入账操作;超出部分需单独补录采购入库单,该部分无需执行暂估流程,可直接手工录入对应发票信息完成结算。
