# 旗舰版,单票自动勾稽失败可能有哪些原因?

【解释说明】
**适用版本:金蝶云星辰旗舰版**

单票自动勾稽的必要条件是发票和单据来源于相同的订单,且参与本次勾稽的商品相同,同时受到自动勾稽规则的控制。 因此自动勾稽失败可能有以下原因:

① 发票与单据不是来源于同一订单(采购发票/采购入库单需要来源于同一采购订单; 销售发票与销售出库单需要来源于同一销售订单)

② 发票和单据上的商品不同

③ 检查自动勾稽规则,是否设置了批次相同、辅助属性相同的勾稽条件。若勾选了批次或者辅助属性相同,则采购发票和采购入库单上需要录入相同的批次或辅助属性才能自动勾稽。![](https://vip.kingdee.com/download/01099ec54a1df1f84bd79565ed97f2ada29d.png)
