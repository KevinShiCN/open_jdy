# 商品开单时，下拉列表的字段显示不完整，或者单据列表/单据体明明列设置开启了显示，但是就是看不到字段，这是为什么呢？

> 作者：进销存赵嘉丽 | 阅读：828 | 有用：3

【问题概述】

商品开单时，下拉列表的字段显示不完整，或者单据列表/单据体明明列设置开启了显示，但是就是看不到字段，这是为什么呢？

![上传图片](https://vip.kingdee.com/download/0100975959279bef40619942811d8438dabc.png)

![上传图片](https://vip.kingdee.com/download/0100d140b4e8559c4432a05129f084a0571a.png)

【原因说明】

一般出现上述现象，是因为列宽显示太窄了，人为不小心拖拽给隐藏了。

【解决方案】

如果开单页面中，商品编码的下拉列表显示异常，将鼠标放置在下拉列表的列头，鼠标右击“重置列宽”后即可。

![上传图片](https://vip.kingdee.com/download/01000703392aef1a47f7bbfb50963c2d6481.gif)

- 如果是单据列表/单据体，则将鼠标放置在列表的空白处，鼠标右击“全部列宽自适应”后即可。

![上传图片](https://vip.kingdee.com/download/010060b294256ae84e2e87f8b522e9f140f8.gif)

【适用版本】

星辰3.1以上所有版本
