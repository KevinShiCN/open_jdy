# 门店商城产生的配送费如何查询?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：happy忆霖 | 阅读：158 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/650144246493689344)

【问题描述】

门店商城产生的配送费如何查询?

【业务场景】

小程序商城中下单的运费如何进行查询?

【解决方案】

在零售商城后台设置好运费信息后,在小程序端进行下单的时候,系统会自动计算出需要的运费信息,下单支付即可

![](https://vip.kingdee.com/download/010986769895f4e341d8bb13c488e96ca8a2.png)

在小程序端,用户也可以看到下单的详情信息,包含的支付的运费信息

![](https://vip.kingdee.com/download/01092e8fa5ac3db44bff8556153548db8966.png)

下单支付后的运费,可以在门店商城-商城订单中,在商城订单列表里面可以查询对应单据信息,可以点击运费进行查询此订单的运费情况的

![](https://vip.kingdee.com/download/01096c02d31c24884319aa10b8d32bddc0ee.png)
