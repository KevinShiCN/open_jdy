# 订货商城必须关联微信服务号吗?

【问题描述】

订货商城必须关联微信服务号吗?

【解释说明】

**订货商城不是必须关联微信服务号。**

订货商城可以通过小程序登录或者H5链接进入,微信服务号不是必须的。

如果有微信服务号,可以添加自定义菜单,跳转小程序或者跳转H5链接,也可以开通微信服务号支付。
