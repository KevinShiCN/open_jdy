# 金蝶云·星辰，小程序还是H5商城好？要不要开通公众号？

问题描述：

解决方案：

1、小程序和H5商城的区别？

（1）版本及审核

小程序更接近app，具有版本特性，当商城发布新功能时，商城端不能立即使用，需要重新发布更新小程序版本，提交微信审核通过上线需要1~7个工作日，且根据售卖内容有资格要求。

H5商城没有版本概念，当新功能上线可立即享用，商城不需微信审核即可上线。

（2）生态功能

微信小程序可以使用其生态功能，例如：客服及直播。

使用H5商城无法使用微信生态功能。

（3）商城入口

相对于H5商城，小程序的入口更加丰富且可以生成桌面快捷方式。

微信客户端如何找到小程序[https://kf.qq.com/faq/170109bEvqUn170109AVzUFf.html](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fkf.qq.com%2Ffaq%2F170109bEvqUn170109AVzUFf.html)

![](https://vip.kingdee.com/download/0109c2ae48aec6454551b142cf23cef96d56.png)

（4）小程序平台报送涉税信息

微信反馈：根据税务部门的要求，小程序平台会报送小程序内产生的交易数据。

只要使用了微信支付，不管是哪个软件厂家产生的交易数据，都会被报送。

目前从官方公开资料看，国家针对所有非银行支付机构，单笔金额超过5万的重点监控;目前经营者的收款银行账户、个人身份证、营业执照法人等各类信息，都要记录在国家系统中。

若您的业务属于自营类型，对这个情况有顾虑建议使用H5商城，目前暂时没有在公开统计口径中。

具体详见：[https://mp.weixin.qq.com/cgi-bin/announce?token=730001117&action=getannouncement&key=11762413408sLJEE&version=1&lang=zh_CN&platform=2](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fmp.weixin.qq.com%2Fcgi-bin%2Fannounce%3Ftoken%3D730001117%26amp%3Baction%3Dgetannouncement%26amp%3Bkey%3D11762413408sLJEE%26amp%3Bversion%3D1%26amp%3Blang%3Dzh_CN%26amp%3Bplatform%3D2)

2、要不要申请公众号？

申请公众号的服务号，服务号可菜单关联H5商城链接，也可以免认证申请10个关联小程序（第一年免小程序认证费用），每个月可以推送四条图文信息。

**总结：**

**若所售卖的商品范围所要求资格都具备，建议申请小程序并关注更新公告及时更新小程序版本。**

**若未能达标资格要求建议使用H5商城。实际请视具体情况而定。**

注意：公众号/小程序认证主体需要与支付主体保持一致。
