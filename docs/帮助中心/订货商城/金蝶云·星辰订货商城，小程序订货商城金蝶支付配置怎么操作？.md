# 金蝶云·星辰订货商城，小程序订货商城金蝶支付配置怎么操作？

我开通了第三方支付配置，支付渠道是金蝶支付，请问我在订货商城应该如何配置？

### 金蝶支付开通及订货商城支付配置

### 一、说明

本手册主要介绍如何开通订货商城用金蝶支付。目标阅读人群：产品、运营、客服、客户。



### 二、金蝶支付进件

第一步：准备好商户资料（营业执照、法人身份证、银行卡（一类卡）、门头照、2张环境照、门店信息表）

第二步：点击链接填写进件资料：[https://docs.qq.com/form/page/DS2Z3cFdqbExXWmlK](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fdocs.qq.com%2Fform%2Fpage%2FDS2Z3cFdqbExXWmlK)[](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fdocs.qq.com%2Fform%2Fpage%2FDS1FnUFBCa0ZFbWxR%23%2Ffill)

第三步：进件成功后金蝶运营人员会根据【填表人的电话或微信】跟客户联系，向客户提供【商户号】、【加密密钥】、【商户私钥】信息，客户保存信息后去【云星辰订货商城】完成配置。（客户可以直接联系金蝶支付运营人员:林伟强15899790447，杨皓皓15919712330）



第四步：客户全部配置完成后，会收到金蝶支付发送成功短信，短信内容包含【金蝶支付商户后台网址、用户名、密码】。客户可以登录金蝶支付商户后台查看收款信息和收款对账。

## ** **

### 三、获取小程序appid

在微信小程序后台，地址：https://mp.weixin.qq.com，菜单-开发者中心，复制并保存下小程序APPID。

将小程序Appid提交给金蝶支付运营人员，报备成功后才能使用小程序微信支付。

 ![](https://vip.kingdee.com/download/01093115be2a048e46c68c9f7d96ac3dabdb.png)

### 四、订货商城后台配置支付参数

登录星辰系统选择【订货商城】菜单下【商城支付配置】

![](https://vip.kingdee.com/download/0109369e917e5d5c4633af2d43661f6c45aa.png)

第三步：点击支付商户号设置【编辑】

 ![](https://vip.kingdee.com/download/0109804a5435562e412dacc8638ce682fbfb.png)

第五步：选择金蝶支付，将获取的【商户号】、【加密密钥】、【商户私钥】信息进行登记 。

按照金蝶支付参数填入点击保存即可，注意不要复制到空格。

![](https://vip.kingdee.com/download/0109627fea11d8a14d988bb683d697077511.png)
