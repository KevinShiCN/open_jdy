# 订货商城新增了运费模板，怎么下单没有计算运费？

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题, 进阶技巧
> 作者：商贸韩佩奇 | 阅读：1355 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/285054067565657344)

问题描述：

解决方案：

1、检查发货方式是否为物流发货，【订货商城】-【商城系统设置】-【运维设置】中，发货设置勾选物流发货，若不勾选物流发货，商品无法选择运费模板且下单时不显示运费。

![](https://vip.kingdee.com/download/0100cfa56e1288c1423dbbf86f30392c0f9a.png)

2、检查运费付款方式是否为寄付，只有选择寄付才能按照运费模板计算运费

![](https://vip.kingdee.com/download/0100a1a842e09a4a4490ab052027d3ee89e8.png)

3、检查商品详情中运费模板是否配置，

（1）单个商品设置运费模板：在商品详情中选择运费模板，多单位商品可以按单位设置不同的运费模板

（2）多个商品批量设置运费模板：订货商品列表中勾选商品后批量修改，勾选运费模板，选择对应的模板即可批量设置商品运费模板

![](https://vip.kingdee.com/download/0100c25fc5148abf4eb8a1708cd676195bed.png)
