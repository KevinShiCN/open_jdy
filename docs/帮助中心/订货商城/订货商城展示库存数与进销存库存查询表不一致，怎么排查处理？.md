# 订货商城展示库存数与进销存库存查询表不一致，怎么排查处理？

> 作者：Lizzy | 阅读：226 | 有用：0

1 问题描述

订货商城展示库存数与进销存库存查询表不一致，怎么排查处理？

![](https://vip.kingdee.com/download/0109b9b1a46488ae4c63981970610dea9a7c.png)

![](https://vip.kingdee.com/download/0109188eff98a2924bf39934af397fb6e255.png)

## 2 原因解析

订货商城展示的是进销存预计可用库存-待审核的商城订单占用，库存查询表为即时库存

## 3 解决方案

库存管理-预计可用库存表，查询预计可用库存表进行比较。

![](https://vip-admin.kingdee.com/download/0109c7fd0ad0f44c4273a7c7f4bce5945095.png)

若预计可用库存表还是与订货商城显示库存数还是不一致，请检查以下几项：

（1）检查参数配置是否扣减了已审核的销售订单。特别是商品为批次商品，建议开启销售订单自动生成预留单，设置预计可用库存公式勾选“已审核的销售订单”，如下图所示，开启后再查看预计可用库存表。

![](https://vip.kingdee.com/download/0109762a856b5c204a6086b4c53c21412f86.png)

![](https://vip.kingdee.com/download/01095f28a718543f4da59bfefc982c70eb6a.png)

（2）检查商品编码是不是同一个，防止相同命名

（3）是否设置了仓库可见范围：订货商城-商城系统设置-商城设置，商城库存可见范围设置。

检查开启的仓库有哪些，商城显示库存为开启仓库的库存合计数；

检查是否设置了适用客户分类，若为空，则所有客户都可见，若设置了客户分类，请检查登录手机号的绑定客户，其客户分类是否在命中条件。

（4）检查有没有未审核的商城订单占用
