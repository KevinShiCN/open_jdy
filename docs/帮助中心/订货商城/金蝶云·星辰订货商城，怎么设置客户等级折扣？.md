# 金蝶云·星辰订货商城，怎么设置客户等级折扣？

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题, 快速入门
> 作者：Lizzy | 阅读：1200 | 有用：4

问题描述：

解决方案：

若商城管理-系统设置-交易设置中支持三种交易方式：先付款后发货/先发货后收款/按客户等级设置是否允许先发货后付款。

选择前两种，所有客户将按照所选的进行交易。若选择【按客户等级设置是否允许先发货后付款】需要对客户等级进行设置。
