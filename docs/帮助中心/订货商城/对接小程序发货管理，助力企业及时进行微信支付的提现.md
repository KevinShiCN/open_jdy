# 对接小程序发货管理，助力企业及时进行微信支付的提现

**1、简介  **

**1.1功能介绍**

通过微信支付和微信小程序开放的接口，在客户完成支付或发货后通过系统告知微信官方订单已发货，触发微信通知客户订单已发货，客户确认收货后即可提现资金，若客户一直不点确认收货，微信也会根据规则自动确认收货。

**1.2适用版本**

- 金蝶云星辰3.1版本
- 金蝶云·星辰专业版/旗舰版

**1.3应用场景**

在线支付对接的是微信官方直连通道，且小程序启动了小程序发货管理监管的客户需接入该功能

**2、主要操作流程**

1）第一步,先设置小程序发货对接的参数，参数1“推送发货状态的时间”和参数2“发货的方式”，默认选项支付后的次日凌晨和同城配送

![](https://vip.kingdee.com/download/0109b6f452fb31df498f98bb05f09efe0309.png)

2）客户下单并发生在线支付，支付单会同步生成小程序发货信息管理的待推送发货状态的信息，此时可手动点击发货进行人工推送状态，也可能等待系统按设置的时间进行自动推送发货状态，点击更新状态会实时同步微信小程序中该支付单的发货状态

![](https://vip.kingdee.com/download/0109e5fa42071f62412d960915b0bba6a05d.png)

3）第三步,如果发现资金发现有不能体现的状况，或是发现发货状态推送异常，可以点击微信支付单号进入详情查询推送状态的日志

![](https://vip.kingdee.com/download/01093740c34f7a394ff8b70fe9290f4a8575.png)
