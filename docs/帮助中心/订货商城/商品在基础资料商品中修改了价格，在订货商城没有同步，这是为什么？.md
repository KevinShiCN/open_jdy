# 商品在基础资料商品中修改了价格，在订货商城没有同步，这是为什么？

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题, 快速入门
> 作者：Lizzy | 阅读：626 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/90733798740146688)

问题描述：

解决方案：

订货商品价格策略中除数量报价独立外，其他报价方式是直接取值基础资料-商品价格资料中的价格设置。

如果您选择数量报价方式，是是独立的，需要另外设置价格报价。
