# 微信支付配置“支付授权目录”的时候,提示“公众号支付支付域名不符合规范要求,请检查修改后重试”,如何处理?

> 作者：姜宁8 | 阅读：423 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352741132086016)

【问题描述】

微信支付配置“支付授权目录”的时候,提示“公众号支付支付域名不符合规范要求,请检查修改后重试”,如何处理?

![](https://vip.kingdee.com/download/01091b0f48ac2ab34bcb8c156e4de809f7c5.png)

【解决方案】

微信配置“支付授权目录”时,需要使用:“sc.jdy.com/”,域名后面需要加“/”。

【温馨提示】

配置公众号支付时,使用的是【微信公众平台】>【服务号】,并非公众号。

![](https://vip.kingdee.com/download/0109fdd1993c4dea464b90f15eecf336c522.png)
