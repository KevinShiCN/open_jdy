# 订货商城如何实现库存为0时显示售罄或者商品自动下架?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：Schatz | 阅读：250 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/639992689256991232)

【问题描述】

订货商城如何实现库存为0时显示售罄或者商品自动下架?

【业务场景】

订货商城上架商品后,有些商品如果被订购完了没有库存了,希望在订货商城中能自动不显示。

【操作步骤】

1、订货商城目前不支持将库存设置为0时自动下架商品,仅允许设置当库存不足时禁止下单。这一设置可以在“订货商城-商城系统配置-商城设置”中进行调整。

![](https://vip.kingdee.com/download/0109e4c11267d45744fba8a88530a734cd45.png)

这样,在商城下单的时候,库存为0的商品无法加购。

![](https://vip.kingdee.com/download/0109289814a708b64b7e8fe9df678e0081bd.png)

2、建议可以在订货商城-商城页面设置分类列表,在商品列表展示内容勾选库存参数。

![](https://vip.kingdee.com/download/0109149ec4ce704444bdb26b8fb9aab55770.png)

这样,在商城下单时,若商品库存为0, 商品会显示“售罄”。

![](https://vip.kingdee.com/download/0109e679a2a1496746afa2d4112938b6c756.png)
3、若是库存为0的商品不想显示在商城,也可手工下架,等补库存后重新上架

![](https://vip.kingdee.com/download/0109f1c3d4ed7561460c8a9d68a3df4e0d97.png)
