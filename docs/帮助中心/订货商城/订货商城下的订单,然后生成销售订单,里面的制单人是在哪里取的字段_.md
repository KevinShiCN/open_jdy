# 订货商城下的订单,然后生成销售订单,里面的制单人是在哪里取的字段?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：若有灵兮山之阿 | 阅读：212 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/525115340540238848)

【问题描述】

订货商城下的订单,然后生成销售订单,里面的制单人是在哪里取的字段?
【业务场景】
订货商城小程序下单后,确认会生成销售订单,销售订单的制单人需要知道来源
【解释说明】
有以下3种情况:
1、商城订单自动审核成功生成销售订单的,制单人是商城链接分享出去的人或者小程序绑定的发布人
2、商城订单自动审核失败,由系统补偿任务补偿审核成功生成销售订单的,制单人是系统管理员
3、商城订单人工审核生成销售订单的,制单人是商城订单审核人
