# 订货商城,如何筛选没有商品图片的商品?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：lykk | 阅读：84 | 有用：0

【问题描述】

订货商城,如何筛选没有商品图片的商品?

【解决方案】

路径:【订货商城】-【商品列表】

![](https://vip.kingdee.com/download/01092b62026f22cf4b00bf3710616b9a2ca9.png)

勾选“图片筛选”,可下拉选择“有图”、“无图”,则会过滤出有图片或无图片的商品;

![](https://vip.kingdee.com/download/01098faf8951a7424b37b3fcf9de0be8b39f.png)
