# 推流直播操作指引

一、准备工作

1、获取推流地址

在小程序后台MP（[/](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fobsproject.com%2F)下载安装在电脑上

![](https://vip.kingdee.com/download/0100818b738bd43041b69010b0a1b149586c.png)

2） 【推流】推流地址配置

打开obs进行进行推流地址配置

![](https://vip.kingdee.com/download/01009a4c54accab94789928bf99f27267fac.png)

• 在设置中输入我们刚在第一步在小程序MP端拿到的推流地址

例如：在MP端复制到的推流地址如下rtmp://wxalivepush.weixin.qq.com/live/wx40f8626ddf43d361_2796?txSecret=4b886b8127b82cde297c7ff6d76915a4&txTime=5efba7d0

• 服务器部分填

rtmp://wxalivepush.weixin.qq.com/live/

• 串流密钥

复制推流地址上 live/后面的内容：wx40f8626ddf43d361_2796?txSecret=4b886b8127b82cde297c7ff6d76915a4&txTime=5efba7d0

![](https://vip.kingdee.com/download/0100fbd6aa6480154e098026c7af5fe35ca6.png)

3） 其他参数配置

*参数配置表

研究得到两种常用分辨率下的推流参数的推荐值（也可根据需要配置其他比例）：

• 720P：1800Kbps-推荐

•  1080P：3000Kbps-清晰度更佳（但对网络要求高，如果网络条件达不到（100兆以上），则会出现卡顿的情况，如果直播过程出现明显卡端可及时调整到720P的配置）

| OBS推荐配置 | 【视频】配置 | 【输出】配置 | 【音频】配置 |  |  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 基础（画布）分辨率 | 输出（缩放）分辨率 | 帧率 | 视频比特率 | 编码器 | 音频比特率 | 采样率 | 声道 |  |  |  |
| 横屏流 | 竖屏流 | 横屏流 | 竖屏流 |  |  |  |  |  |  |  |
| 720P（推荐） | 1280x720 | 720*1280 | 1280x720 | 720*1280 | 25 | 1800Kpbs | x264 | 160 | 44.1KHz | 立体声 |
| 1080P | 1920*1080 | 1080*1920 | 1920*1080 | 1080*1920 | 3000Kpbs |  |  |  |  |  |

下面按720P举例如何配置

a、【视频】配置

分辨率：这个设置是决定最后流的画面尺寸（横竖屏等），obs该项配置可以选择，也可以自定义填写。观众端会根据流的画面比例自动适配横竖屏

• 720P（竖屏720*1280；横屏1280*720）

![](https://vip.kingdee.com/download/01009ac72af78e0f4d159581d35f742580ab.png)

b、【输出】配置

• 输出模式：高级

• 视频比特率

1800Kbps左右（分辨率配置：720P类型，720*1280）

      20000-3000kbps左右（分辨率配置：1080P类型，1080*1920）

• 关键帧间隔：3

• cpu使用预设：superfast

• 配置: high

![](https://vip.kingdee.com/download/010070707ea205564c4e8f79e5cfb433678a.png)

c、【音频】配置

• 采样率：44.1KHz

• 声道：立体声

• 其他配置项：按实际需要进行配置

![](https://vip.kingdee.com/download/010088a0a29185f24b08983c2c3606dfd038.png)

d、采集来源配置

如果要用PC电脑做为直播录制设备，可以在【来源】处选择

• 视频捕捉设备

• 音频捕捉设备

4）启动推流

设置好可以调整整个窗口的占比，确认好效果后，先在obs点击推流

![](https://vip.kingdee.com/download/0100a8211c26d28540e098d4d81bb1e2e281.png)

*直播中展示PPT

如果同时需要一边录制一边播放PPT，可以在【来源】处选择选择窗口捕捉设备

• 窗口捕捉设备

![](https://vip.kingdee.com/download/010082b615e6ec5d4e018161bfe14f9bdfc9.png)

*推横屏流，设置竖屏背景

注意：该方式其实是设置了视频流为竖屏流的形式，则观众端无横屏切换按钮

1）横屏流在竖屏的直播间会有未被占满的黑色屏幕，如果想添加背景，可以在obs上把分辨率设置为竖屏的比例（720*1280或1080*1920）![](https://vip.kingdee.com/download/0100c26c8620b04247ff9622a6561fd98a07.png)

 2）再在【来源】处选择选择【图像】导入，即可导入需要设置在直播间的背景图

![](https://vip.kingdee.com/download/0100a8d2eb82f32a41798e482fc27cae9990.png)![](https://vip.kingdee.com/download/010088b2defc660d40c49f7781015ac4fab9.png)

如果要用专业摄像机做为开播设备等更多能力，可以参考第三方推流软件的说明

二、开播

1、回到小程序Mp后台

以上准备工作完毕后，可以回到MP小程序后台，直播-控制台

![](https://vip.kingdee.com/download/0100c72767272a844d628e685bd19228d01b.png)

2、开播

1） 准备开播

进入到控制台，检查推流的状态是否正常，正常的情况下即可开始直播啦，点击【开始直播】则正式开播

a、检查推流是否正常

![](https://vip.kingdee.com/download/0100c675947fd20647b3976ed22db20c5b14.png)

b、点击【开始直播】

![](https://vip.kingdee.com/download/010021ed6c11d7a14934bbccfff682044a10.png)

2） 观众端表现

• 分辨率为竖屏流（720*1280或1080*1920），竖屏直播间无横屏切换按钮

• 分辨率为横屏流（1280*720或1920*1080），竖屏直播间有横屏切换按钮

显示的比例和视频区域等可以在第三方软件上自由调整

![](https://vip.kingdee.com/download/0100bcaa2b3828a54652a8c26b446cd99b8c.png)![](https://vip.kingdee.com/download/010020a214e402574ba6978629e24b727bd0.png)![](https://vip.kingdee.com/download/0100047ae33d9d904fa0870aef937b52c5d2.png)

3） 结束直播

优先在控制台点击【结束直播】，再关闭obs等第三方推流软件

![](https://vip.kingdee.com/download/01006e456793599f433cb17c3c945f96d3d0.png)![](https://vip.kingdee.com/download/0100d7f967262e6a4bb88b71db2acebf93ca.png)
