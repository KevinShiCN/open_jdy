# 金蝶云·星辰订货商城小程序发布，提示【代码审核失败】，如何操作？

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题, 进阶技巧
> 作者：Lizzy | 阅读：998 | 有用：5
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/55960035960383488)

问题描述：

![](https://vip.kingdee.com/download/010078cce5d30a7b459c8ed7e5ea75b509ac.png)

解决方案：

到微信平台登录小程序账号，检查以下信息：

1、【设置】-【基本设置】，检查小程序信息的是否完整。特别留意检查小程序名称、头像及认证信息。![](https://vip.kingdee.com/download/010033c1b1ec356941d686e3814c487d1f45.png)

2、【设置】-【第三方设置】检查授权信息是否完整，需要授予下图的权限：

![](https://vip.kingdee.com/download/0100a0ea8427bd4e40ccbabb6855f730e6f5.png)

授权不完整如要点击停止授权按钮，重新到金蝶云·星辰-订货商城-小程序微信设置，点击【授权发布小程序发布】，扫二维码授权，勾选如上权限。
