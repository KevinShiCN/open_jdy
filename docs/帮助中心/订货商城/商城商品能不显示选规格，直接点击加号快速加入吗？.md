# 商城商品能不显示选规格，直接点击加号快速加入吗？

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：进阶技巧, 常见问题
> 作者：商贸韩佩奇 | 阅读：376 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/285063931897635584)

问题描述：

![](https://vip.kingdee.com/download/0100d91a977e5b504846807180a0866424cd.png)

解决方案：

【订货商城】-【商城页面设置】列表展示内容勾选库存，且商品为单单位单规格商品

注意：库存展示设置中不要选择商城不展示库存，如果选择商品列表设置中没有库存字段，无法勾选
