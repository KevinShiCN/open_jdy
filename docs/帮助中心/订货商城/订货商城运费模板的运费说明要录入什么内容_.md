# 订货商城运费模板的运费说明要录入什么内容?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：云社区用户55os622 | 阅读：92 | 有用：0

【问题描述】
订货商城运费模板的运费说明要录入什么内容? 为什么是必录的?![](https://vip.kingdee.com/download/01097e10326107664a30a1b0e370974234cd.png)

【解释说明】
运费说明作为必填项将展示在商品详情页,帮助客户清晰了解运费的计算方式和标准,减少因运费不透明产生的疑虑和纠纷。提升购物体验和订单转化率。

因此,商家应尽量使用简短清晰的文字说明运费收取规则,帮助用户理解。例如:全国范围内,前10双运费30元,超过10双,每1双多收取运费2元
