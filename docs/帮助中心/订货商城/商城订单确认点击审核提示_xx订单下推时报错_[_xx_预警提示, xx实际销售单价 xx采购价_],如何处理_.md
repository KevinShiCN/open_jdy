# 商城订单确认点击审核提示:xx订单下推时报错:["xx:预警提示, xx实际销售单价 xx采购价"],如何处理?

【问题描述】
商城订单确认点击审核提示:xx订单下推时报错:["xx:预警提示, xx实际销售单价 xx采购价"],如何处理?![](https://vip.kingdee.com/download/0109adeeaf7601b244a4b283be09ac529442.png)

【解释说明】
出现该提示的原因是设置了采购销售限价,而订单上的价格不满足限价方案。 可一次点击【基础资料】-【采购销售限价】![](https://vip.kingdee.com/download/0109bc9edfe5fded4af4bfc49f2e12012449.png)

查看设置的销售限价详情,同时查询看下订单里面的价格是否低于限价。如果确定需要审核订单,则可暂时将限价方案禁用,审核完订单后再重新启用该限价方案![](https://vip.kingdee.com/download/0109f4083f80c3784517bbe58670d1b9e568.png)
