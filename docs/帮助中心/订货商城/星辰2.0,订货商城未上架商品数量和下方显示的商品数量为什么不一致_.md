# 星辰2.0,订货商城未上架商品数量和下方显示的商品数量为什么不一致?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：姜宁8 | 阅读：260 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352751835720704)

【问题描述】

订货商城未上架商品数量和下方显示的商品数量为什么不一致?

如图,订货商城--订货商品列表--未上架商品,显示总共339条:![](https://vip.kingdee.com/download/010991ae091e12204d879c52d72f30a7c874.png)【解释说明】

禁用商品后会在表头中的未上架商品显示数量,但是不会在商品信息中显示,解除禁用商品后才会保持一致。![](https://vip.kingdee.com/download/010932195c7192f7453abb330b7542977a33.png)
