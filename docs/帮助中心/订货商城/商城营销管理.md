# 商城营销管理

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：快速入门, 整体介绍
> 作者：杨桂平 | 阅读：373 | 有用：5

**业务场景**

借助商城营销设置，能轻松开展多样化的营销活动。无论是刺激批量化采购的批发促销，还是吸引用户消费的优惠券发放，亦或是提升用户忠诚度的积分管理，都可精准实现。同时，还支持在线推广，助力商城拓展客源，提升业务量。

**解决方案**

| 步骤 | 说明 | 学习链接 |
| --- | --- | --- |
| 批发促销 | 设定特定商品开展促销活动，可规定促销时间与范围。灵活调控销售策略，有效清理库存，提升商品流转速度。 | 点击学习 |
| 优惠券 | 发放各类优惠券，如代金券、折扣券等。刺激用户消费欲望，提高客单价，增强用户对商城的粘性和购买频次。 | 点击学习 |
| 积分管理 | 用户消费或参与活动获积分，积分可兑换礼品或抵扣现金。增加用户忠诚度，促进用户持续消费，完善用户运营体系。 | 点击学习 |
| 商城在线推广 | 通过网络渠道对商城进行宣传推广，吸引潜在用户访问。扩大商城知名度和影响力，提升流量与转化率，促进业务增长。 | 点击学习 |
| 公众号消息服务 | 微信绑定账套接收公众号消息，可以在微信接收商城用户下单及付款的提示。 | 点击学习 |
