# 订货商城积分过期时间设置怎么设置?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：ZXYzxc | 阅读：51 | 有用：0

【问题描述】

订货商城积分过期时间设置怎么设置?

【操作步骤】

【订货商城】-【积分】-【积分设置】,在积分设置的积分过期设置中,启用【开启积分过期时间设置(不开启的情况下,积分永久有效)】,在积分过期时长设置对应的时长天数
