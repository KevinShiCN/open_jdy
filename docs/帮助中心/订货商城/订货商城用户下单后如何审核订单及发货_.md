# 订货商城用户下单后如何审核订单及发货?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：杨桂平 | 阅读：108 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/782412015519655936)

【问题描述】

订货商城用户下单后如何审核订单及发货?

【解决方案】

商城用户下单后,商家在订货商城后台可查询到相关订单,并且需要商家在订货商城后台确认订单交易再发货出库。

依次点击【订货商城】-【商城订单确认】打开订单列表界面,可查询到“待确认订单”、“已确认订单”、“已取消订单”数据。

![](https://vip.kingdee.com/download/0109a146e4d3bbcd478b8c6a0bf66867e631.png)

**1、查看订单详情**

在商城订单确认列表中点击操作功能下的【详情】可查询到订单的详细数据,如:订单信息、结算信息、商品清单、收款记录等。

![](https://vip.kingdee.com/download/0109a39ea81d96ae43ccaa8b42577b839777.png)

**2、订单确认(审核)**

对待确认的订单进行订单确认(即订单审核),订单审核支持单张审核,也支持批量审核。订单审核后即可生成销售订单。

**2.1 单张审核**

选择需要审核的订单,点击操作功能中的【审核】按钮,系统会弹出审核详情界面,订单确认无误后点击【审核通过】即可。

![](https://vip.kingdee.com/download/0109576712c41bbc4dfcb2d0a3fcd7ae8a01.png)

**2.2 订单批量审核**

在待确认订单中选择需要审核的订单,点击【批量审核】即可。

![](https://vip.kingdee.com/download/0109152a4313b35a43879a33916be1931a43.png)

订单审核通过后订单会显示在“已确认订单”列表中,订单确认状态显示为【审核通过】,并且会自动生成了对应的销售订单和预收款单(商城订单已付款才会生成预付款单)。而商城的客户可以在“我的”-“进行中”查询订单,订单状态更新为“已付定金(已付支付的订单)”、“未出库”、“货款未核销”。

![](https://vip.kingdee.com/download/01091d590e319a6f47008747bba07124721b.png)

**3、发货出库**

商城订单审核后会生成销售订单,商家通过【销售管理】-【销售订单-查询】中将销售订单生成销售出库单即可完成出库(详细操作可参考[如何按销售订单生成销售出库单?](https://vip.kingdee.com/knowledge/391351743222458624?productLineId=35&isKnowledge=2&lang=zh-CN)),商城客户订单状态时会更新为“全部出库”(部分出库时会显示部分出库)。
![](https://vip.kingdee.com/download/01098c12a83380b44e689dc750478e3f2a93.png)
