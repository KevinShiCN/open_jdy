# 订货商城中已确认订单生成的销售订单误删后能否重新生成?

> 作者：Schatz | 阅读：92 | 有用：0

【问题描述】

订货商城中已确认订单生成的销售订单误删后能否重新生成?

【解释说明】

订货商城中已确认订单生成的销售订单被删除之后,无法重新生成,只能重新下单。
