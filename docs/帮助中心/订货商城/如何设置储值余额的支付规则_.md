# 如何设置储值余额的支付规则?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：云社区用户8x80607 | 阅读：276 | 有用：0

【问题描述】

如何设置储值余额的支付规则?

【操作步骤】

**订货商城-储值-储值设置**,可选择:

1)商城订单付款时,若有储值余额,是否默认优先用余额支付;

2)储值支付可抵扣金额参数;

3)使用储值支付时,若本次可抵扣金额不足应付款,是否允许使用储值支付;

4)使用储值余额时的消费扣减规则
