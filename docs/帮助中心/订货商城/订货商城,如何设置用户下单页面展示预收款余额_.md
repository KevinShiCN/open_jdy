# 订货商城,如何设置用户下单页面展示预收款余额?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：云社区用户55os622 | 阅读：83 | 有用：0

【问题描述】
订货商城,如何设置用户下单页面展示预收款余额?

【业务场景】
若下单人已经是正式客户,则支持在支付界面配置显示该客户在系统中的预收款余额,帮助用户更好地了解自己的账户情况,同时若展现的预收款余额较低,可促进用户充值。

【解释说明】
点击【订货商城】-【商城页面】,进入页面设置界面![](https://vip.kingdee.com/download/0109fd261e83bdbe407d8b74a711017c382b.png)
 在【其他页面】-【确认订单页】,右侧勾选<支付页面展示预收款余额>
![](https://vip.kingdee.com/download/010900c22c277e834920b4831a5b89a715b2.png)
