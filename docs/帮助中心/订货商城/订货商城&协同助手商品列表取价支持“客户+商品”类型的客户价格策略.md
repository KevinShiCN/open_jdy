# **1 简介  **

## **1.1 功能介绍**

商城取价支持“客户+商品”的客户价格资料，主要用用于不同的客户，商品价格各不相同，按客户维护商品价格资料。

## **1.2 适用版本**

星辰3.0以上标准版/专业版/旗舰版，协同助手

星辰3.0以上专业版/旗舰版，订货商城
