# 金蝶云·星辰，订货商城小程序直播如何将商品库导入及直播间推送商品？

1、问题描述

- 小程序直播商品库如何导入
- 直播间如何推送商品

2、场景介绍

商家使用小程序直播功能，在讲解商品后希望能推送商品，促进大家订购。

3、解决方案

## 3.1 商品库导入

有两种导入方式，可以根据需要选择：

（1）方案一：直接在小程序直播后台进行导入。操作方式如下：

[https://mp.weixin.qq.com/](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fmp.weixin.qq.com%2F)登录小程序，左侧菜单点击直播进入直播后台，点击商品库-未入库商品，点击右上角添加商品。按照要求上传商品信息，商品路径可以在星辰-订货商城-商品推广，复制获取。

提交商品后，需经微信官方审核通过后，才会进入到已入库商品列表。

![](https://vip.kingdee.com/download/0109bba9f3773af84c60a229c3313c5214ca.png)

（2）方法二：在订货商城将商品添加到商品库

路径：订货商城后台-小程序直播-直播商品库

点击设置，对商品名称默认填充、价格形式默认填充、价格默认填充和价格单位默认填充进行填充规则设置。

点击添加商品，选择商品并按照微信方要求，重新上传对应格式的图片（300像素 * 300像素，图片大小不得超过1M，更容易通过微信审核），商品名称需14字以内以便通过审核。

提交后等待微信审核商品结果。

### 3.2 商品导入直播间

[https://mp.weixin.qq.com/](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fmp.weixin.qq.com%2F)登录小程序，左侧菜单点击直播进入直播后台，点击直播间-创建直播间，创建后点击控制台。

下拉到互动，可以导入已审核的商品，直播开始则推送按钮取消置灰，点击后直播间可以看到商品弹出链接。点击可以进入订货商城中购买。
