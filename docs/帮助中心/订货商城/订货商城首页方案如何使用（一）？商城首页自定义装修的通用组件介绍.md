# 订货商城首页方案如何使用（一）？商城首页自定义装修的通用组件介绍

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：快速入门, 功能介绍, 新特性
> 作者：Lizzy | 阅读：4322 | 有用：8

订货商城首页装修，除了提供固定模板供商家设置，也支持设置装修方案自定义添加组件。

一、商城模板设置，首页固定模板

登录PC端星辰，订货商城-商城模板设置-商城首页，在固定模板进行选择商品链接及上传图片

具体操作可以点击下面链接查看

图文教学：[https://vip.kingdee.com/link/s/MkaQS](https://vip.kingdee.com/questions/129339652304816811/answers/129339687906069024?productLineId=35&isKnowledge=2)

二、商城首页方案，自定义添加组件

登录PC端星辰，订货商城-商城首页方案，右上角点击新建方案，并设置方案参数。后将方案发布商城端。

1、填充方案名称，选择模板，点击确定，进入方案编辑页。

订货商城根据高频行业及偏好，提供了5种免费的主题颜色和行业，可选择模板快速编辑。

![](https://vip.kingdee.com/download/010974690fe966c047a4baae01a67b746800.png)

此处，我们选择空白模板作为演示效果：

（1）页面设置的背景颜色，可以根据喜欢的主题色进行设置。颜色框点击有预设的颜色卡片支持选择，若要进行微调可以直接输入颜色值。

![image.png](https://vip.kingdee.com/download/010963f3005830774022bb3b014b291494c7.png)

（2）搜索组件，左侧的组件点击搜索组件，编辑对应样式。

![image.png](https://vip.kingdee.com/download/0109a3338ea95bec4c5a8d261292742f3766.png)

效果如图：

![image.png](https://vip.kingdee.com/download/01092b4ae109c1e74660bc6c9e3905c3b3dc.png)

（3）图片广告组件，支持5中图片样式：图片轮播、一行一个、大图横向滚动、小图横向滑动、导航横向滑动，可以根据图片的位置及样式进行设置。

上传的图片将根据样式的固定宽度进行填充，为了美观建议上传图片的宽高是一致的。

![image.png](https://vip.kingdee.com/download/0109b81ab69f0f5d40e9bc54886fc4a139e2.png)

图片倒角：用于控制图片是否需要圆角处理

填充样式：处理图片填充的方式

左右边距：图片组件跟页面根据需要设置左右间隙

图片边距：除图片轮播样式外，图片和图片之间支持设置间距

![image.png](https://vip.kingdee.com/download/010929606b8e0efa42ab9a4a39b0305573f1.png)

（4）图文导航：支持设置图文导航样式或者纯文字样式。

![image.png](https://vip.kingdee.com/download/0109b3a94dd8bb9f42f29534fab6b73769b4.png)

根据上图参数配置并上传图片，效果图如下：

![image.png](https://vip.kingdee.com/download/0109fedbc5e1bf544a4d82f83b32a9292804.png)

（5）标题文本：设置标题名称及描述内容，支持查看更多链接跳转

![image.png](https://vip.kingdee.com/download/0109c4aebcd4a1cd4aa79be6fbf0308763f4.png)

（6）商品：商品组件支持大图模式、一行两个、一行三个、列表模式和一大两小，5种样式选择。根据需要选择样式及链接商品。

![image.png](https://vip.kingdee.com/download/01093a24a2a2f7e947868562b485d652e862.png)

跟标题文本组件结合使用，效果如下：

![image.png](https://vip.kingdee.com/download/01094938a62f90c44b1fa9c23616d946cbf8.png)

（7）头条组件：头条组件可以放置一些公告通知或者公司介绍等文章。

![image.png](https://vip.kingdee.com/download/0109bb3bcaf908b7462e8f999e84c05ced11.png)

效果图如下：

![image.png](https://vip.kingdee.com/download/01090e5dc0f31605494eab06d2e2220b0017.png)

（8）辅助分割：组件和组件之间，高度间隔的调节可以使用插入辅助分割线，一般建议使用背景颜色，具体可以根据喜好调整。

![image.png](https://vip.kingdee.com/download/010912e8e6d0cdce44129f52ccd8f0853be9.png)

效果图如下：

![image.png](https://vip.kingdee.com/download/0109704e87a30a3042f394417b14df0d283b.png)

2、设置好方案，点击右上角保存模板，点击手机预览，则可以扫码查看H5商城效果，点击预览则使用浏览器模拟手机页展示设置效果。

3、确认好效果，可以点击保存并发布，或者列表勾选方案点击发布。刷新进入订货商城可以看到设置效果。
