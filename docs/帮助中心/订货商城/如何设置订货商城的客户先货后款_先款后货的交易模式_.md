# 如何设置订货商城的客户先货后款/先款后货的交易模式?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：杨桂平 | 阅读：1088 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/129339770097650352)

【问题描述】

如何设置订货商城的客户先货后款/先款后货的交易模式?

【操作流程】

订货商城客户先货后款/先款后货的交易模式由商城设置参数“交易设置”控制。

登录系统,点击【订货商城】-【商城系统配置】-【交易设置】

![](https://vip.kingdee.com/download/01095d2a45625d5344e2b024e6c1fd5ff759.png)

**“交易模式”种类包括以下三类:**

**1、勾选“先付款后发货”:该模式代表下游订货客户都需要先付款才会发货**

**2、勾选“先发货后收款”:模式代表下游订货客户都是先确认订单发货后收款**

**3、勾选“按客户等级设置是否允许先发货后付款”:该模式代表若下游订货客户有一些是先付款再发货,有一些是先发货再付款,则可以按客户等级设置不同客户的交易模式**

![](https://vip.kingdee.com/download/010963edffb3942b423ca5f2da466e2c93e5.png)

【温馨提示】

勾选“按客户等级设置是否允许先发货后付款”参数,由此客户对应的客户等级判断是“先款后货”还是“先货后款”:

1、登录系统,点击【订货商城】-【客户等级管理】

![](https://vip.kingdee.com/download/0109d7f22b4b73fb47b19f1e9afa4c2eb9cd.png)

点击修改可以修改客户等级的交易方式

![](https://vip.kingdee.com/download/0109b3f6e4c3605f4873ab49f9abe4be4ac3.png)

2、点击【订货商城】-【客户列表】,勾选需要修改等级的客户点<批量修改>按钮,勾选<价格等级>可修改客户价格等级

![](https://vip.kingdee.com/download/0109975c32d9864944acac6a4cc68518d2f9.png)
