# 金蝶云·星辰订货商城到货异常处理功能介绍

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：功能介绍, 快速入门
> 作者：商贸韩佩奇 | 阅读：984 | 有用：9
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/194957739750608128)

功能介绍：

客户可以在商城端进行确认到货的操作,若到货有疑问可以填写异常信息并上传图片,进行到货异常申请的提交,商家可在后台查看并进行处理,帮助商家提升异常订单处理效率。

操作步骤：

在销售订单进行出库操作后，订货商城我的包裹能查到此次发货的信息

![](https://www.yunzhijia.com/docrest/file/downloadfile/60d0ae89a90edd0001f60102?big)   ![](https://www.yunzhijia.com/docrest/file/downloadfile/60d0ae89cc9a8200017ce246?big)

若客户收到货物，查看货物的数量及品质，如果没有问题，可以直接点击确认收货

![](https://www.yunzhijia.com/docrest/file/downloadfile/60d0ae8991196e00010bfeef?big)

若收到的货物有漏发或破损的情况，可以进销到货异常的反馈，点击到货异常，如下图，选取异常的商品，点击下一步

![](https://www.yunzhijia.com/docrest/file/downloadfile/60d0ae8acc9a8200017ce247?big)

确认每种商品反馈异常的原因，可填写备注信息和拍照上传证据

![](https://www.yunzhijia.com/docrest/file/downloadfile/60d0ae8a89ab530001fc5e7a?big)  ![](https://www.yunzhijia.com/docrest/file/downloadfile/60d0ae8a89ab530001fc5e7b?big)

反馈异常后，或确认到货后可查看包裹的到货状态

![](https://www.yunzhijia.com/docrest/file/downloadfile/60d0ae8b89ab530001fc5e7c?big)  ![](https://www.yunzhijia.com/docrest/file/downloadfile/60d0ae8b91196e00010bfef1?big)

商家可以在后台进行到货异常的处理，操作路径：【订货商城】>【到货异常处理】

商家查看到异常情况后，线下沟通处理好这个异常后，点击异常已处理，这个包裹即变更为已完成状态

![](https://vip.kingdee.com/download/0109c8afcf97ac5441f2b8aaafa6e9a885b7.png)
