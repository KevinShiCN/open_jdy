# 订货商城中,游客、散客、潜在客户、正式客户的区别

【问题描述】

订货商城中,游客、散客、潜在客户、正式客户的区别

【名词解释】

游客:在订货商城浏览未登录的用户,不用输入手机号登陆,列表界面不显示游客

散客:在订货商城登录了还未成为正式客户的用户,也没有点击申请成为经销商,申请成为加盟伙伴

潜在客户:散客点击申请成为经销商,申请成为加盟伙伴,但是网页端还没有点击转为正式客户

正式客户:进销存的客户管理位置录入的客户,或者在商城申请网页端点击转为正式客户的潜在客户
