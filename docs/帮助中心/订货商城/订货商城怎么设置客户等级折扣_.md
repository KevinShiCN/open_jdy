# 订货商城怎么设置客户等级折扣?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题

【问题描述】

订货商城怎么设置客户等级折扣?

【解决方案】

若【订货商城】-【商城系统配置】-【交易设置】中的设置是否允许先发货后付款。

选择前两种,所有客户将按照所选的进行交易。

若选择【按客户等级设置是否允许先发货后付款】需要对客户等级进行设置。
