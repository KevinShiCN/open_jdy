# 金蝶云·星辰,订货商城小程序直播怎么配置?

【解决方案】

1、订货商城接入的是微信小程序直播功能;![](https://vip.kingdee.com/download/01092791a374d65243768befc0986fc98004.png)2、开通微信小程序直播的前提是在微信那边有一个微信小程序,且小程序上发生过一笔在线支付的交易;

【接入步骤】

步骤1:申请一个微信小程序(PS:如果没有的话);

步骤2:星辰订货后台-微信小程序设置,授权发布商城到申请的微信小程序上(PS:操作指引[https://vip.kingdee.com/questions/152369/answers/182386](https://vip.kingdee.com/questions/152369/answers/182386));

步骤3:申请接入通联支付(PS:操作指引[https://vip.kingdee.com/questions/68702724870588416/answers/68702726095259648](https://vip.kingdee.com/questions/68702724870588416/answers/68702726095259648));

步骤4:完成一笔在线支付的交易;

步骤5:在微信小程序后台开通小程序直播功能(按照微信后台界面的流程操作);

步骤6:开通好直播功能后,微信小程序后台取消第三方授权,重新授权发布小程序(PS:操作指引[https://vip.kingdee.com/questions/152369/answers/182386](https://vip.kingdee.com/questions/152369/answers/182386));![](https://vip.kingdee.com/download/01099541b0ecbd1b42f4b1bcf6befc12fafe.png)步骤7:微信小程序后台设置好主播,开通直播;

步骤8:在微信小程序中有正在直播的内容时,订货商城的首页会出现直播入口;
