# 订货商城的订单如何查询是谁下的单?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：shaoling | 阅读：110 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/710296175382700544)

【问题描述】

如何查询订货商城订单是谁下的单?

【业务场景】

订货商城订单有些是员工代客下单,有些是客户自己下单的,想要查看订货商城订单具体是由谁下单操作的

【操作流程】

从【订货商城】>【商城订单确认】进入商城订单确认列表页面

![](https://vip.kingdee.com/download/01097af5a51695b1433aae2e44241fe948c1.png)
在商城订单确认列表页面,点击订单编号或者点击<详情>按钮,进入商城订单页面

![](https://vip.kingdee.com/download/01097faf84ffb578415d99c71f911d4c4eb2.png)
在商城订单页面,鼠标下滑到“订单操作日志”,即可查询到什么时间,哪个用户操作的订单保存

![](https://vip.kingdee.com/download/01098e7a6dfe124f41adb66ede63c225b7f9.png)
