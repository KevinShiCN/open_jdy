# 客户如何在订货商城端使用积分?

【问题描述】
客户如何在订货商城端使用积分?

【解释说明】
后台配置好积分抵扣规则后,商城客户下单时,系统将自动通过客户信息及下单商品信息匹配到对应的积分抵扣规则,计算积分本次可抵扣金额。显示可抵扣积分,以及积分对应抵扣的金额,参考界面如下:
