# 金蝶云·星辰，订货商城，微信小程序设置扫码授权时提示“微信小程序未检测微信授权成功信息，请重新扫码授权”，应该如何操作？

> 作者：Lizzy | 阅读：1636 | 有用：8
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/172299322758077952)

问题描述：

金蝶云·星辰，订货商城，微信小程序设置扫码授权时提示“微信小程序未检测微信授权成功信息，请重新扫码授权”，应该如何操作？

![](https://vip.kingdee.com/download/01005fef7916e7d544ebaf3c402a855cea87.png)

解决方案：

出现该问题说明您的小程序授权异常，需要解绑权限重新授权，请进行以下步骤：

1、登录微信公众平台（https://mp.weixin.qq.com/），进入小程序，到设置-第三方设置，对精斗云腾飞订货开发平台停止授权。若发现未授权请跳过这一步。

![](https://vip.kingdee.com/download/0100e39a2e4b341a4bc6a87fb799181b5232.png)

解绑授权

若有授权其他第三方的，也需要停止授权。小程序的第三方开发仅支持一个，或者另外申请小程序，在授权。

![](https://vip.kingdee.com/download/01002317923d49964fa8969e627f0bff83b6.png)

2、清除浏览器缓存，重新打开授权二维码，用微信小程序的超级管理员扫码授权。ctrl+H可以快捷清除浏览器缓存。

![](https://vip.kingdee.com/download/0100ef39b979e5cf49fc8592a899efcf602c.png)

清除浏览器缓存

![](https://vip.kingdee.com/download/0100a07f80f2842e464a81d1850f99f18a15.png)

查看小程序管理员的微信是哪个

![](https://vip.kingdee.com/download/0100c2d6a32d463e483da415b4267fb936c4.png)

重新扫码授权

3、微信上操守授予所有权限，当手机上微信提示授权成功时，回到网页端点击下一步。
