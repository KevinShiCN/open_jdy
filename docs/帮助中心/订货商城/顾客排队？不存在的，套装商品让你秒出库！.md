# 顾客排队？不存在的，套装商品让你秒出库！

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：知识软文, 进阶技巧
> 作者：Simone邓 | 阅读：1242 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/323099838437141248)

“怎么搞的呀，我们都排队了三十分钟了，怎么才收银两个顾客！？”后面的顾客抱怨喊道。
       原来小敏刚开了一家护肤品店，由于刚开业，购买护肤品都会赠送一支防晒霜，来购货的客户非常多，小敏忙得大汗淋漓，但购货的队伍一直没有移动。
       一直在柜台拿货给顾客的小美，看到这个情况，急急忙忙跑过去，问明原因，小敏说：“顾客购买的每一套护肤品，包含水、乳、霜，洗面奶、又有赠品，做销售单据得一个一个商品去选择，赠品还要单独备注，实在忙不过来了呀，姐。”
       小美说道：不怕，这个可以做套装商品出售，只要选择一下套装商品，其他的明细商品就直接带出了，很简单，我教你：首先，【基础资料】-【商品信息】新增商品，商品类型选择套装
![image.png](https://vip.kingdee.com/download/01007aded8705f4b48f5b9c4c46db2640b6a.png)
![image.png](https://vip.kingdee.com/download/0100f70e6b0ad35a4a36a85dcedecc37a7f8.png)
需要注意的是套装商品信息中，其中一个明细商品为赠品，要将【是否赠品】打开，如明细商品需要进行替换，需维护可替换的商品，如下图：
![image.png](https://vip.kingdee.com/download/0100c9593b37d75141f1981eaa86fcf97da8.png)
未打开【是否赠品】的明细商品，需录入明细商品的价格
![image.png](https://vip.kingdee.com/download/0100a198075c22794e8687621b40d8b26988.png)
录入好之后进行保存套装商品，做销售单据时，选择套装商品，自动带出套装明细商品，无需再手动选择单个商品做销售单。
![image.png](https://vip.kingdee.com/download/0100a2d92e4fd720489ebd42338c2842a488.png)

![image.png](https://vip.kingdee.com/download/0100c17322a80035406a9942d622a74649e0.png)

开单支持查看套装商品历史销售出库记录，如果出库套装明细无库存，需要更换其他商品，可点击替换商品，选择其他商品替换出库。

需要注意套装商品的套装信息中维护对应商品的替换商品才能替换，未启用【允许替换】的商品是不允许替换的。

小敏惊讶的说道：这个真的太棒了！后面的顾客不担心排队了！
