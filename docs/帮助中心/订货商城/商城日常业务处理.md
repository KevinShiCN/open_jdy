# 商城日常业务处理

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：应用案例, 快速入门
> 作者：杨桂平 | 阅读：620 | 有用：5
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/754366665944658688)

**业务场景**

商城日常业务包括用户下单、后台订单确认及发货、用户确认收货等流程完成交易闭环。 若用户对订单存疑，可便捷申请取消订单、退货或退款；同时，商城具备灵活管控能力，能针对客户设置商品屏蔽与限购。此外，充分考量用户需求，提供公司配送、物流发货、门店自提多种发货方式，并依此精细设置运费管理与自提点，全方位优化购物体验。

**解决方案**

| 步骤 | 说明 | 学习链接 |
| --- | --- | --- |
| 商城订单处理 | 对用户在商城下达的订单进行统一处理，包括商城客户下单、商城订单确认、发货、收款、核销，确保订单能高效准确地进入后续处理环节。 | 点击学习 |
| 订单收款及退款 | 处理订单的收款事务，支持多种支付方式；同时，按规则和用户需求及时处理退款，保障交易资金的安全与流转。 | 点击学习 |
| 订单发货及退货 | 依据订单信息进行商品发货安排，跟踪物流；对于退货订单，协调处理商品收回、检验及退款等事宜，维护用户权益。 | 点击学习 |
| 商城经营屏蔽 | 可针对特定商品、用户设置经营屏蔽规则，灵活控制商城业务范围，避免不必要的交易风险和运营问题。 | 点击学习 |
| 商品限购 | 设定商品的购买数量限制，可用于促销活动管控、库存保护等，满足多样化的营销和运营需求，平衡供需关系。如一些特价和降价的产品，商品限购可以有效防止其他商业伙伴以及同行业恶意抢购。 | 点击学习 |
| 商城运费管理 | 制定和管理商城商品的运费规则，根据地区、重量、订单金额等因素计算运费，合理控制物流成本，优化用户购物体验。 | 点击学习 |
| 客户自提业务 | 管理客户自提业务，包括自提点的设置、信息维护和调度等，为用户提供便捷的自提服务选择，提升服务灵活性。 | 点击学习 |
| 储值业务 | 储值业务是商家推出的便捷消费服务，用户提前将资金存入账户，消费时可直接扣款。储值赠送规则还能设置赠送金额，既帮用户省钱，又能提升消费频次。 | 点击学习 |
