# 金蝶云·星辰，订货商城，范围内指定商品选赠的促销活动介绍。

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：功能介绍, 新特性, 进阶技巧
> 作者：Lizzy | 阅读：1060 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/324491708857611776)

问题描述：

怎么设置商品参与促销活动-满足促销条件后可以在指定范围内选择赠品？

解决方案：

（更新时间2022年4月21日，需更新小程序版本v2.0.8以后的版本）

单击【订货商城】-【批发促销】，范围内商品促销新增【指定商品选赠】活动，设置活动名称、活动时间、指定商品范围、优惠规则和适用对象。![](https://vip.kingdee.com/download/0109e27f2711102e4083bcae88cada16a31a.png)

优惠规则可以配置满金额或者满数量。赠送商品设置赠送数量和范围，赠品商品分录配置零售价以便商城用户参考。

![](https://vip.kingdee.com/download/0109347286eaa8ee49249b10e3776ba1a325.png)

使用场景如下：商城配置促销活动，购买星辰系列产品满200.00元，赠品可在三种凭证账簿中选择一件。且设置允许倍增，即满400元，可选两件赠品。
