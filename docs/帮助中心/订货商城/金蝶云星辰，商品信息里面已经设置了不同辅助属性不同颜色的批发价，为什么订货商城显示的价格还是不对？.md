# 金蝶云星辰，商品信息里面已经设置了不同辅助属性不同颜色的批发价，为什么订货商城显示的价格还是不对？

> 作者：小橙子和小柠檬 | 阅读：142 | 有用：0

金蝶云星辰，商品信息里面已经设置了不同辅助属性不同颜色的批发价，为什么订货商城显示的价格还是不对？

![image.png](https://vip.kingdee.com/download/01094b193f9ce7a047aa93d3ef52dc5ecf5b.png)

![image.png](https://vip.kingdee.com/download/0109ef1804607e914574997127926a073d13.png)

解决方案：进入【订货商城】-【商品列表】进入商品详情，设置商品销售报价这里为【不同规格设置不同价格】就可以了，附图如下：

![image.png](https://vip.kingdee.com/download/0109a4a4dbc839c74480a7df4e208b81f0a2.png)

![image.png](https://vip.kingdee.com/download/010914ca95482313403fab9caf9d166ee311.png)
