# 金蝶云·星辰订货商城，怎么设置客户信息？

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：功能介绍, 快速入门
> 作者：Lizzy | 阅读：4159 | 有用：12
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/68760963586799616)

问题描述：

解决方案：

订货商城客户分为三种状态游客、散客及正式批发客户。

1、游客

（1）游客指在订货商城浏览未登录的用户。

（2）订货商城系统配置可以控制游客是否能浏览商城及查看价格。![](https://vip.kingdee.com/download/010903302206a0ab4638ac89f20bea83b012.png)

2、散客

散客指在订货商城登录了还未成为正式客户的用户，散客可以申请经销商成为潜在客户。

3、正式客户

（1）正式客户指在客户列表中存在客户信息，手机号已经作为客户下的联系人，会根据设置的价格策略取价。

（2）散客转为正式客户有两种方式：

A、手机端通过订货商城申请

订货商城登录时，识别到手机号未在客户列表联系人中，会跳转经销商申请界面，提交申请信息。散客用户也可以在我的-申请经销商填写申请信息，或查看申请状态。

![](https://vip.kingdee.com/download/01002f227b3221f440968af303e5c0d5806e.png)

B. 商家审核

商家在后台潜在客户列表可以看到该用户申请信息，点击转为正式客户，保存客户信息。手机用户重新进入订货商城就会以正式客户身份查看商城。

2) 商家直接添加正式客户信息

商家在后台订货商城-客户，新增客户信息，添加联系人手机号，手机号登录商城即为正式客户。商城用户登录将直接读取基础资料客户信息。

注意：

订货商城同步不允许重复手机号在多个或同个客户下。

若系统中已经有客户信息，申请时会提示已经存在该公司，需要买家将手机号添加到对应客户下的联系人。![](https://vip.kingdee.com/download/0109c576b3b346cb4409a1a745830c1377f9.png)

4、商城登录限制

单击【订货商城】-【商城登录限制】支持按单用户（手机号）/单客户/客户分类/客户标签禁止客户登录商城。
