# 订货商城小程序或者h5链接下单后微信/支付宝支付提示“支付失败,sub_mch_id与sub_appid不匹配:invalid_reque”,应该怎么检查配置?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：姜宁8 | 阅读：1034 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352740142001152)

【问题描述】

订货商城小程序或者h5链接下单后微信/支付宝支付提示“支付失败,sub_mch_id与sub_appid不匹配:invalid_reque”,应该怎么检查配置?

![](https://vip.kingdee.com/download/0109eaac97fbb5a24002bbcc5d78f61594fc.png)

【解决方案】

检查在通联后台配置APPID和域名配置是否正确。

![](https://vip.kingdee.com/download/010921f54ede74d943269ddf16a5341239e4.png)

登录通联后台点击【产品中心】>【微信支付参数配置】,将服务号或者小程序的APPID填入支付APPID配置。
![](https://vip.kingdee.com/download/0109380674f6835b45d39eba04a9e94eec00.png)
