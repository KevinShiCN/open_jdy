# 微信公众号添加自定义菜单,配置跳转网页的时候报错:发布失败,请输入http://或https://开头的公众号相关链接,如何处理?

> 作者：姜宁8 | 阅读：926 | 有用：0

【问题描述】

微信公众号添加自定义菜单,配置跳转网页的时候报错:发布失败,请输入http://或https://开头的公众号相关链接,如何处理?

![](https://vip.kingdee.com/download/0109f947a826bd6f49df91bec664dff27c01.png)

【解决方案】

**建议进入【微信公众平台】>【服务号】,在后台检查微信服务号是否有做企业认证。**

【操作步骤】

点击[https://mp.weixin.qq.com/](https://mp.weixin.qq.com/),登录【微信公众平台】>【用服务号的管理员微信账号登录】
![](https://vip.kingdee.com/download/010907ebfc0d252f48e2bb723ded9e58651c.png)
在服务号设置的界面,点击【设置与开发】>【公众号设置】>【微信认证】,按照微信方的步骤进行认证即可

![](https://vip.kingdee.com/download/010997af6f91af254371a1b8ae6503819f32.png)
