# 金蝶云·星辰 订货商城微信小程序扫码授权一直提示未授权成功，请重新扫码授权？

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题, 进阶技巧
> 作者：商贸韩佩奇 | 阅读：529 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/376027298501931008)

****

**1、问题描述：订货商城微信小程序扫码授权一直提示未授权成功，请重新扫码授权？**

![](https://vip.kingdee.com/download/0109c85cdd37cf3245288877fc21dfc8e9f2.png)

2、解决办法：订货商城需要小程序的独立授权，如果有绑定其他第三方平台，需要先解除其他第三方平台的授权，再重新扫码授权。（解除路径：微信公众平台小程序管理员扫码登录：设置—第三方设置）

![](https://vip.kingdee.com/download/010917473c3b82084000b069b570b3298544.png)
