# 订货商城退货如何处理？

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：进阶技巧, 常见问题
> 作者：杨桂平 | 阅读：1665 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/357453128146155264)

**【问题描述】**

商城客户在商城中对订单已进行确认到货操作后才发现商品损坏或质量问题等需要退货，那么订货商城中如何处理退货业务？

【**解决方案】**

**1、客户申请售后**

客户收到货后需要申请退货，可以在订货商城中申请售后，客户在【我的】-【我的包裹】，对异常订单点击【申请售后】，再点击【申请退货】，勾选需要退货的商品点击【下一步】，选择填写退货数量、退货原因、退货详细描述、并上传图片，填写完成后点击提交。

![](https://vip.kingdee.com/download/01093ad59811555e461d9d9daeb28c427ae7.png)

**2、商家处理退货申请**

客户提交退货申请后，系统将生成未审核的销售退货申请单。商家再根据实际进行处理，若商家同意退货后，审核此退货申请单，再生成销售退货单即可。
