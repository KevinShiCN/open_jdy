# ​金蝶云·星辰订货商城，商城下单/商城订单审核填写单据的自定义字段

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：快速入门, 常见问题
> 作者：Lizzy | 阅读：741 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/322375692120296704)

【系统设置】-【单据新增字段】新增表体且类型为“文本型”和“辅助资料型”的自定义字段。支持在商城下单及后台订单审核页面设置是否展示和必填。

![](https://vip.kingdee.com/download/010943177043bea8402e9065592202c52406.png)

单据字段新增

![](https://vip.kingdee.com/download/0109c5f7383d9c924cc5b52c778a11a6d692.png)

商城下单支持的自定义字段

开启商城订单确认显示时，商城订单列表-待确认订单，操作列【审核】按钮，弹窗将展示对应字段。开启商城端显示及必填时，商城端下单才会展示该字段或检查必填。

![](https://vip.kingdee.com/download/010974787d3990eb4c82a3d9631751ac1b62.png)
