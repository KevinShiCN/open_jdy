# 金蝶云·星辰，订货商城  商城端用户是否能接收商城的消息通知

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：进阶技巧, 功能介绍, 新特性
> 作者：Lizzy | 阅读：1203 | 有用：7
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/372747690733842944)

功能介绍：在商城端界面，正式客户扫码关注“金蝶心链”公众号后，接收商城订单审核和发货的通知。

适用版本：星辰·专业版

应用场景：商城端的用户，希望下单后可以接收商城订单商家已审核的信息或者商家已操作发货的信息，以便知道商城订单的处理进度。

应用流程：商城端用户登录后，在我的-消息推送，扫码关注公众号产生账套绑定关系后，商城用户可以配置是否需要收到商城订单审核和发货提醒。
