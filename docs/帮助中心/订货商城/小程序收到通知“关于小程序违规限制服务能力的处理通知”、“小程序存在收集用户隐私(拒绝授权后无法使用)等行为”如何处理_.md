# 小程序收到通知“关于小程序违规限制服务能力的处理通知”、“小程序存在收集用户隐私(拒绝授权后无法使用)等行为”如何处理?

> 作者：lykk | 阅读：1525 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/649777393925764096)

【问题描述】

小程序收到通知“关于小程序违规限制服务能力的处理通知”、“小程序存在收集用户隐私(拒绝授权后无法使用)等行为”如何处理?

【业务场景】

小程序收到通知如下,应该如何处理?

场景一:

“关于小程序违规限制服务能力的处理通知

你好,经平台审核,此小程序 存在问题:用户拒绝授权手机号权限后无法使用小程库已封禁服务的“手机号授权”能力

处理时间:2024年05月13日违反规范:《微信小程序平台运营规范》15.用户隐私和数据规范-15.1数据收集和存储(点击此处)违规内容:请在站内信中查看违规详情”

场景二:

“你好,经核实,小程序存在收集用户隐私(拒绝授权后无法使用)等行为,请自觉遵守国家相关法律法规,不从事违法或违反《微信小程序平台运营规范》及相关规则的行为,谢谢。相关行为规范和修改指引可参考:[https://developers.weixin.qq.com/community/develop/doc/0008ce8a908108c5d4fee910856c09](https://developers.weixin.qq.com/community/develop/doc/0008ce8a908108c5d4fee910856c09)”

【解释说明】

由于微信对自营商城获取隐私信息管控严格,要求获取手机号之前需要将提供商城可浏览功能,若要作为内部系统使用需要达到以下三个要求:

1、使用账号+密码登录,账号可以是手机号,但密码不能是短信验证码

2、输入手机号+验证码登录,但非内部(特定)人员点击发送验证码后,不支持下发验证码成功,需提示:该号码无权限登录访问等文字说明

3、手机号一键授权登录,需自行开发接口,非内部(特定)人员使用手机号登陆时,无法登录成功,页面有文字说明:该号码无权限登录访问

【解决方案】

1、游客浏览及散客下单参数设置调整

微信小程序该规则将不支持游客不能浏览商城+散客下单的模式,到商城系统配置-商城设置修改商城浏览模式设置和商城散客模式设置修改参数

若商城是内部经销商下单使用,不对外开放,需要同时取消勾选“客户未登录可先浏览商城”和“开启散客模式”。

若商城支持散客下单,请必须开启“客户未登录可先浏览商城”。

![](https://vip.kingdee.com/download/0109b45daa61371a416d99beab8f067b9a9a.png)

2、商城若不对外开放使用,需要取消关闭微信一键登录功能

![](https://vip.kingdee.com/download/0109b722eeb2c41d4a5892acbe239eefc814.png)

3.小程序版本需要更新至3.2.17版本之后,最新版小程序已根据微信要求进行优化

如果您的小程序最新小程序,请到小程序后台解绑第三方开发,再重新授权。

4.在小程序后台进行申诉。

打开小程序公众平台,扫码登录小程序后台。右上角消息找到对应通知,打开详情点击申诉。

![](https://vip.kingdee.com/download/01096fa332820b9f4f6e9a588e95702fa4b7.png)

![](https://vip.kingdee.com/download/010999b8653be4f64aedb3979b5ee3c493a6.png)

申诉填写文本可如下:

小程序商城为公司内部供应链下单,商品信息及价格为商业敏感信息,通过手机号识别内部供应链用户身份。已经按照《指引》整改,烦请获取最新的小程序版本检查。

对应整改及截图详见下图:

1、图1:打开商城说明不针对游客开放

2、图2:输入手机号+验证码登录,发送前已授权隐私协议,且非内部人员点击发送验证码提示无权限

3、图3:允许取消注册及隐私协议
