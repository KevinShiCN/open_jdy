# 订货商城储值消费暂扣是什么意思?

【问题描述】

订货商城储值消费暂扣是什么意思?

【解释说明】

商城储值消费暂扣:是指商城订单下单时使用储值支付,此时储值支付状态显示为 “消费暂扣”,待销售出库单完成出库后,系统将自动触发生成核销单。

![](https://vip.kingdee.com/download/0109759d9e06dff74a95b0e9f3a9665623c6.png)

若需要改为现付或使用其他预收款抵扣,可取消暂扣(请谨慎操作取消暂扣,客户储值余额将增加,后续可以消费使用。若取消暂扣后,[本次消费暂扣金额】等于0,将不再触发储值消费核销。)
