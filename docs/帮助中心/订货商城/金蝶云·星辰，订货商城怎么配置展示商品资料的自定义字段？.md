# 金蝶云·星辰，订货商城怎么配置展示商品资料的自定义字段？

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：进阶技巧, 常见问题, 新特性
> 作者：Lizzy | 阅读：1005 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/342689122903036672)

**商城**支持自定义字段搜索及信息展示

功能介绍：支持配置自定义字段（文本类型）的搜索方式，并展示在商品列表及商品详情中

适用版本：星辰·专业版

应用场景：商家希望商城能展示在基础资料-商品配置的自定义字段，并能使用搜索功能快速查找到该商品。例如，化学用品有国际通用的CAS编码及化学式，配置后能使用CAS编码或化学式搜索，并展示在商品列表和商品详情中。

应用流程：

1、 在基础资料新增字段-商品中新增文本类型的自定义字段。

![](https://vip.kingdee.com/download/010097fc38dcad5c4de38c13d10810b22992.png)

基础资料新增字段

2、在基础资料-商品详情中维护自定义内容和信息。

![](https://vip.kingdee.com/download/010082a3c45566a740cdb20f4d24d0e901eb.png)

商品详情

3、订货商城-商城模板设置，分类列表中的商品列表展示内容、商品详情页中的商品详情页展示，配置对应的展示内容。其他页面-搜素页配置更多搜索方式
