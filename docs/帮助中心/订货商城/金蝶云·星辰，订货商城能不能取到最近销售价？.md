# 金蝶云·星辰，订货商城能不能取到最近销售价？

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：功能介绍, 新特性, 进阶技巧
> 作者：Lizzy | 阅读：627 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/388700046252337408)

当商家由于商品过多且价格体系紊乱，或者在议价中属于弱势方，在逐渐规范维护价格体系过程中，往往需要订货商城展示登录客户最近销售价来辅助报价。

    订货商城商品取价支持获取最近销售价。

    开启最近销售价，在基础资料-价格优先级-开启最近销售价。再到订货商城-商城系统设置，开启“优先取进销存中的客户最近销售价”。

    开启后，商城经按照登录手机号对应的正式客户身份，按照价格优先级“客户价格策略>最近销售价>客户等级价”，获取该客户的最近销售价。
