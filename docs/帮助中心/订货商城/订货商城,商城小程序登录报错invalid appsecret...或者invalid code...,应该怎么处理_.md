# 订货商城,商城小程序登录报错invalid appsecret...或者invalid code...,应该怎么处理?

【业务场景】

金蝶云·星辰订货商城,商城小程序登录报错invalid appsecret...或者invalid code...,应该怎么处理?

![](https://vip.kingdee.com/download/0109872822f7e94043beb9ca61134136b38e.png)

【解决方案】

商城小程序登录报错invalid appsecret或者invalid code...,是商城配置小程序的appid及秘钥填写错误,请重新检查秘钥并正确填写。

invalid appsecret指的是错误的小程序秘钥;invalid code指的是错误的小程序appid

请进行以下操作:

1、解绑小程序

![](https://vip.kingdee.com/download/010996b40e16345547df86a573be45c49d04.png)

2、获取密钥:
    打开微信公众平台网址[https://mp.weixin.qq.com/](https://mp.weixin.qq.com/),登录账号,开发-开发者设置-开发者ID,获取微信APPID和微信APP密钥。其中AppSecret的查看需要点击该行后面的“重置”或者“生成”链接,查看后要记得把AppSecret在其他位置记录下来。

![](https://vip.kingdee.com/download/010985cf089f34364926a8b93e6279512802.png)

3、精斗云订货商城,微信小程序设置。
