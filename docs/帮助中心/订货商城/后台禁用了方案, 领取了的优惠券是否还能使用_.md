# 后台禁用了方案, 领取了的优惠券是否还能使用?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：云社区用户8x80607 | 阅读：156 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/626946767384493824)

【问题描述】

后台禁用了方案, 领取了的优惠券是否还能使用?

【操作步骤】

**后台方案作废,所有未使用优惠券将不能使用,优惠券状态为作废**

1、如下图所示,小程序有一张【国庆显示活动】代金券未使用

![](https://vip.kingdee.com/download/010958ba5b3740284d5b9616fdf05c302bd3.png)
2、订货商城-优惠券,操作禁用该方案![](https://vip.kingdee.com/download/01098ec0556fc34247f8960c04134bcf12bd.png)
3、禁用成功,优惠券状态已失效,发券状态为结束发放

![](https://vip.kingdee.com/download/010961ceab8419594e67b9c430ce8d652659.png)
4、小程序查看该优惠券【已失效】,不能继续使用。

![](https://vip.kingdee.com/download/0109fdb117aa3b2e47f9b866a984d3959626.png)
5、如需继续使用优惠券,重新启用优惠券方案即可。
![](https://vip.kingdee.com/download/01097ecbd7a949a24888bec90dd47357b023.png)
6、此时之前领用的优惠券状态为【未使用】

![](https://vip.kingdee.com/download/01092ff0b81762ed4fa4bd93e10308b88484.png)
需注意:重新启用方案,方案需未过期优惠券才可恢复未使用状态
