# 金蝶云·星辰，订货商城设置了促销活动，为什么商城端下单/购物车没有实现促销优惠？

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：进阶技巧, 常见问题
> 作者：Lizzy | 阅读：920 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/279561606294298880)

问题描述：

解决方案：

一、首先，先查看促销的设置类型，只有指定商品满件数赠类型会显示在购物车，其他活动类型在下单界面根据提交的商品计算出最优促销方式。

![](https://vip.kingdee.com/download/010033e3035cc80d445cb4eadac247b91005.png)

指定商品满件数赠的活动，在商品详情及购物车可以显示活动，加入购物车的商品满足活动条件时显示对应赠品。如图：

 ![](https://vip.kingdee.com/download/0100f7b59ee359a24f09993e95895397d2d7.png)![](https://vip.kingdee.com/download/0100a4f7603ac10a46d892b91c157fc3c47d.png)

其他促销活动类型，商城用户选购参与活动的商品合计金额满足条件，则在提交订单界面显示对应促销立减金额或赠送。

![](https://vip.kingdee.com/download/010090b8d8f387044005befce381a1485e91.png)   ![](https://vip.kingdee.com/download/0100a10ce2d1d7b74ea8ae19b6331ef0d0cb.png)

二、若在商品加购时，没有看到商品的促销活动信息，请检查是否在活动范围内，包括登陆手机号所属的客户是否在活动的客户范围，选择的商品的单位/规格是否在活动商品范围内。

例如常见问题：商品设置了促销活动，加入大单位包，没有参与促销。这是因为促销活动的商品范围是有设置单位的，默认携带销售订单，其他单位需要另外选择。如图，在商城中查看商品，有一个单位带了促销活动规则标签，两位一个单位没有。

![](https://vip.kingdee.com/download/0100ceeca1695fd34d8c9b29acb5b2b3c2f0.png)      ![](https://vip.kingdee.com/download/01007f795d2d012944329fc4950d2395fd40.png)

查看活动方案发现，多单位商品仅设置了一个单位，需要将活动禁用，修改方案增加其他单位商品。

![](https://vip.kingdee.com/download/010007633ce2582e47de9606ec2b0440982d.png)
