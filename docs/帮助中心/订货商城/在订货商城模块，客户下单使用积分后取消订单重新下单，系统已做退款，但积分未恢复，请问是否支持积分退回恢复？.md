# 在订货商城模块，客户下单使用积分后取消订单重新下单，系统已做退款，但积分未恢复，请问是否支持积分退回恢复？

> 作者：Weixinxin | 阅读：79 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/689515918682113536)

## 1 问题描述

在订货商城模块，客户下单使用积分后取消订单重新下单，系统已做退款，但积分未恢复，请问是否支持积分退回恢复？

## 2 适用版本

星辰3.0以上版本

## 3 解决方案

步骤1：目前星辰订货商城不支持自动退回积分，可以通过在【客户积分列表】中选择相应客户，调整积分-增加积分，并维护相应原因即可。

![](https://vip.kingdee.com/download/0109cd820692bf40433f8c2f638af9f08bf5.png)

![](https://vip.kingdee.com/download/010917331a18285c4bc5808ee6de2782a03b.png)

步骤2：后续可以到【订货商城】>【积分】>【积分流水】中查看相关积分的备注信息。![](https://vip.kingdee.com/download/0109795944fb550c4794b24b683d5ab64e2a.png)

更多积分相关知识请查看：[订货商城积分功能介绍](https://vip.kingdee.com/link/s/Zhqmf)
