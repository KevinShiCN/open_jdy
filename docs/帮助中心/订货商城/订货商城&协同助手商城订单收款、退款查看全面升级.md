# **1 简介  **

## **1.1 功能介绍**

**商家审核端（PC）：**

- 订单列表**新增独立显示**“商城退款状态”和“在线交易状态”，支持多维度筛选。
- 界面**清晰展示**所有明细金额，如线下/在线收款金额、储值消费、线下/在线退款金额等。
- 订单详情页新增 **“在线交易记录”**，可查看支付、退款的详细流水。

**商城用户端（小程序）：**

- 付款状态简化合并为“未付款”、“已付款”、“部分付款”等，一目了然。
- 新增订单列表，支持查看“全部订单”和快速筛选“待付款订单”。

## **1.2 适用版本**

星辰3.0以上标准版/专业版/旗舰版，协同助手

星辰3.0以上专业版/旗舰版，含订货商城模块

## **1.3 系统路径**

协同助手-协同客户-商城订单

订货商城-订货管理-商城订单确认

## **1.4 字段/按钮说明**

1、电脑端新增字段“确认前收款状态”、“在线交易状态”、“商城线下收款金额”、“商城在线收款金额”、“储值消费金额”、“商城收款金额”、“商城退款状态”、“线下退款金额”、“在线退款金额”、“储值取消消费暂扣”、“商城退款金额”、“在线交易记录”。

确认前收款状态、在线交易状态和退款状态支持列表小漏斗快速筛选、常用筛选过滤和新增方案条件过滤。

为避免误解，确认前收款状态在已确认商城订单列表屏蔽，仅保留商城收退款金额字段。

2、商城端订单列表显示商城付款状态

原进行中订单“已付定金”、“货款未核销”简化状态值“无需付款”（订单金额为0），“部分付款”、“已付款”、“未付款”、“部分退款”、“已退款”，便于商城用户理解。

状态值说明：

商城订单审核前和审核后，订单收款状态存在取值差异，具体如下

（1）商城订单未审核

| 状态 | 取值说明 |
| --- | --- |
| 无需收款 | 商城订单总金额（含运费）=0 |
| 未收款 | 商城收款金额=0 或订单类型为市价成交 |
| 部分收款 | 商城订单总金额（含运费）>商城收款金额 > 0 |
| 已收款 | 商城订单总金额（含运费）≤商城收款金额 |

（2）商城订单审核：直接读取销售订单关联记录

| 状态 | 取值说明 |
| --- | --- |
| 无需收款 | 销售订单成交金额=价税合计+预计客户承担费用=0 |
| 未收款 | 销售订单已确认货款+剩余未抵扣定金+储值未抵扣金额=0，且订单总金额>0 |
| 部分收款 | 05、商城端新增全部订单列表和待付款订单列表。
