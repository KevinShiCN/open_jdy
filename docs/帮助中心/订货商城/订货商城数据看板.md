# 订货商城数据看板

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：进阶技巧, 精华汇总, 功能介绍
> 作者：Lizzy | 阅读：310 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/557898903974986752)

订货商城数据看板可在金蝶云app首页添加并查看

- 金蝶云app添加数据看板首页设置-添加商城看板。![](https://vip.kingdee.com/download/01098512750d8fcd463a95c55738731f0644.png)    ![](https://vip.kingdee.com/download/0109676802faa32a447fbe11abf51c6db3c0.png)点击齿轮可设置可见数据

      ![](https://vip.kingdee.com/download/0109c8d18e5ab7cb4816878609ebd1091b45.png)

  2.返回首页查看商城看板数据

- 商城订单数量：客户经由商城自主下单的订单数量汇总（不含移动端代客下单、不含已取消的商城订单、不含待确认订单） 商城浏览人次：有访问过商城页面的人次数（统计期间内同一个用户的多次访问仅算一次）
- 商城转化率：   商城转化率=商城订单数量 / 商城浏览人次
- 商城活跃客户：在期间内，触发过任意一个条件即为活跃客户：1.下过单；2.支付过货款；3.浏览商城停留时长超过3分钟
- 商城新增客户：从未登录过商城的老客户登录了商城、新注册的用户
- 商城本月留存率：商城留存率=上月新增的客户本月浏览过商城的客户数 / 上月新增的客户数
- 商城销售金额：客户经由商城自主下单的订单金额汇总（不含移动端代客下单、不含已取消的商城订单、不含待确认订单）
- 商城支付金额：客户经由商城自主下单的订单金额中已支付部分的金额汇总（含线下付款已确认的部分）
