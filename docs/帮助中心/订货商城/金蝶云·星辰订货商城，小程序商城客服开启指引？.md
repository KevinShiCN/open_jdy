# 金蝶云·星辰订货商城，小程序商城客服开启指引？

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题, 快速入门
> 作者：Lizzy | 阅读：7611 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/109656925289390336)

问题描述：

![](https://vip.kingdee.com/download/0100517d6715c4be45939a036b6509290c6a.png)

解决方案：

步骤一：小程序绑定客服

微信小程序自带客服功能，小程序管理员需在小程序后台完成客服人员的绑定。目前小程序支持绑定不多于100个客服人员。

小程序管理员需在小程序后台完成客服人员的绑定。目前小程序支持绑定不多于100个客服人员。

![](https://vip.kingdee.com/download/01091e99ba16e02148a586cf806fea9ab5cf.png)![](https://vip.kingdee.com/download/01098fa03d7725df470ab27677aa52af8503.png)

步骤二：商城开启小程序商城客服按钮，商城端会显示[客服消息按钮](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fdevelopers.weixin.qq.com%2Fminiprogram%2Fdev%2Fcomponent%2Fcontact-button.html)，点击按钮在商城内唤起客服会话页面与商城客服人员对话。

![](https://vip.kingdee.com/download/0100517d6715c4be45939a036b6509290c6a.png)

![](https://vip.kingdee.com/download/010097a36435e32f42f8a145d228d5dd40a3.png)

#### 客服人员绑定后的客服帐号，可以登录网页端客服 或 移动端小程序客服 进行客服沟通。

详细查看转载微信官方文档-小程序-客服功能使用指南
