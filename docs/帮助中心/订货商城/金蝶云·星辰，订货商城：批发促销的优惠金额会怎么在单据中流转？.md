# 金蝶云·星辰，订货商城：批发促销的优惠金额会怎么在单据中流转？

问题描述：

解决方案：

以高级促销-满金额打折为例，提交订单时，促销计算会展示促销立减对应优惠金额。

![](https://vip.kingdee.com/download/010972feff85148e491da6624124c841cf1b.png)

促销活动：高级促销-满金额折

![](https://vip.kingdee.com/download/0109f9db6635bc794223a4df1543211d3449.png)    ![](https://vip.kingdee.com/download/0109ca1be4c67cf34590ac40e3e7d4a51569.png)

商城端：促销活动立减

下单后，商城订单的商品分录会记录促销商品优惠的对应金额。商城订单审核生成销售订单后会将订单调价金额和促销优惠金额和优惠券优惠金额合计记录到销售订单折扣额中。销售订单下推销售出库单跟随销售订单展示对应金额。
