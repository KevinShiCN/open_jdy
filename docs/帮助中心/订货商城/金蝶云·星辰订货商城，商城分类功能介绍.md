# 金蝶云·星辰订货商城，商城分类功能介绍

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：功能介绍, 快速入门
> 作者：Lizzy | 阅读：5504 | 有用：8
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/23826)

操作步骤：

 【订货商城】>【商品管理】>【商城分类】，进入商城商品分类列表。

一、商城分类导入

当商城商品分类列表为空时支持一键导入已经基础资料中的商品类别。导入仅支持到三级类别。![](https://vip.kingdee.com/download/0109a4c043e117f74783ad5b235a7631842e.png)

导入基础资料商品类别

二、商城分类新增

商城商品分类仅支持新增到三级分类。

(一) 新增一级分类，单击右上角新增商品分类。![](https://vip.kingdee.com/download/0109483f00b0c43144cd968f271e77a63870.png)

新增一级分类

(二)新增二级分类，一级分类下单击【新增子类】。![](https://vip.kingdee.com/download/0109dfed3d7d735248789ef16505f828d9d0.png)

新增二级分类

(三)新增三级分类，二级分类下单击【新增子类】。![](https://vip.kingdee.com/download/0109dc05732b210347abb4703c0639880b14.png)

新增三级分类

三、商品分类修改

(一)顺序修改，点击上移下移进行位置变更

(二)点击编辑，对分类名称和图片修改

四、商城分类商城展示

商城分类展示样式支持四种模板，可在【订货商城】>【商城页面设置】中分类列表选择喜欢的样式。![](https://vip.kingdee.com/download/0109eea90470b8ae437f88051fb6f970a6c2.png)
