# 订货商城中如何上架套装商品？

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题, 快速入门
> 作者：商贸韩佩奇 | 阅读：1434 | 有用：9
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/195137445862454016)

问题描述：

解决方案：

1、基础资料-商品管理中，添加一个套装类型的商品，设置好套装内容

![](https://vip.kingdee.com/download/010031443648bc8d48c3a707798f4b0b53e2.png)

2、订货商城上架该商品，注意：套装商品不能用进销存的可用库存进行管控，所以套装商品的虚拟库存必须录入，用虚拟库存管控套装的库存

![](https://vip.kingdee.com/download/010003c25732ab954ecdadba13caff04f22d.png)

3、在商城展示套装商品

![](https://www.yunzhijia.com/docrest/file/downloadfile/60d15827f4d5bc00016f94a7?big)
