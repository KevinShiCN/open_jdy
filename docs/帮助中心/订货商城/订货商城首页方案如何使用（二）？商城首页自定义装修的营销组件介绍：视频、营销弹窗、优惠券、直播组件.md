# 订货商城首页方案如何使用（二）？商城首页自定义装修的营销组件介绍：视频、营销弹窗、优惠券、直播组件

**1、简介  **

**1.1功能介绍**

介绍订货商城首页装修的营销组件：视频组件、营销弹窗组件、优惠券组件。

通用组件详见文章介绍[《订货商城首页方案如何使用？商城首页自定义装修的通用组件介绍》https://vip.kingdee.com/link/s/lFppg](https://vip.kingdee.com/link/s/lFppg)

**1.2视频组件**

左侧的组件点击视频组件，右侧上传视频文件，支持MP4格式，视频大小50M以内。

![](https://vip.kingdee.com/download/01095d70121ec3f844dc89dcf04651e6b711.png)

**
**

**1.3优惠券组件**

（1）首页领取入口

右侧选择优惠券方案后，开启首页入口。编辑显示模型、显示样式、文本颜色、焦点颜色、背景颜色。

**![](https://vip.kingdee.com/download/0109bb4c55cd7ff74fcdb55dcf9aa969a526.png)**

优惠券首页入口设置效果：

**![](https://vip.kingdee.com/download/0109e7fd69d9d61248379738dadae063cf63.png)**

（2）弹窗领取入口

右侧选择优惠券方案后，开启领券弹窗。弹窗样式可使用系统默认样式或自定义图片。

**![](https://vip-admin.kingdee.com/download/01096db95af2ccb84b8a8f09b0cefcd95f05.png)**

商城端领券弹窗系统默认效果及自定义图效果：

![](https://vip.kingdee.com/download/0109527b1441f9464cf7b96842189d267b37.png)   ![](https://vip.kingdee.com/download/010966965f93079240a2ba746bd4e2c51564.png)

**1.4**营销弹窗组件

营销弹窗支持公告文本/自定义图片弹窗，左侧设置弹窗按钮颜色及链接页面。

![](https://vip.kingdee.com/download/010971be703ea4344f0e89d8f4f1cfb9f8e6.png)

弹窗自定义图制作示意图

弹窗广告图片可以制作为规则的卡片或不规则的氛围型，建议尺寸: 600 * 800像素，为防止按钮遮挡，可以预留下高度136像素；若为氛围型，建议使用png图片。

![](https://vip-admin.kingdee.com/download/0109009d8850fb2f4aeea6de62f72ea8c3f0.png)

1.5直播组件

微信小程序的直播间开启详见文章

[《金蝶云·星辰，订货商城小程序直播使用指引。》https://vip.kingdee.com/link/s/lFpmH](https://vip.kingdee.com/link/s/lFpmH)

左侧的组件点击直播组件，调整直播入口组件位置，可编辑可见客户范围。
