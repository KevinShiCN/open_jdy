# 星辰2.0订货商城中设置商品促销方案启用了允许倍增,但是客户下单时为什么没有按倍数减少?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：杨桂平 | 阅读：374 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391351727670799360)

【问题描述】

订货商城中设置商品促销方案启用了**允许倍增**,但是客户下单时为什么没有按倍数减少?

例如:设置商品促销方案为所有商品满金额减:设置两条规则①满100元减2元,②满250元减5元;并且启用了允许倍增,客户下单购买200元的商品,为什么只减少了2元,而不是4元?

![](https://vip.kingdee.com/download/01097ae268e364344b5098ded33f3b80ae96.png)

【解决方案】

允许倍增仅适用于最后一条规则,例如:设置两条规则①满100元减2元,②满250元减5元;则金额满250元减5元支持倍增。

当客户购买参与活动的商品金额120元则减2元,金额200元还是减2元,不能参与满100元减2元的倍增规则;当购买商品的金额为250元则减5元,购买商品金额520元则减10元,参与满250元减5元倍增。
