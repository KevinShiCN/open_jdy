# 订货商城如何设置在确认订单界面隐藏商品行价格?

> 作者：lykk | 阅读：128 | 有用：0

【问题描述】

订货商城如何设置在确认订单界面隐藏商品行价格?

【解决方案】

路径:【订货商城】-【商城页面设置】-【其他页面】-【确认订单页】

1、将隐藏商品行价格进行勾选;

![](https://vip.kingdee.com/download/0109b734b3bd43ba457b99b7046e6d94a848.png)

2、勾选后,订货商城确认订单界面,商品行不显示金额

![](https://vip.kingdee.com/download/01099e272d1e080f4b42947c84e7f3516250.png)
