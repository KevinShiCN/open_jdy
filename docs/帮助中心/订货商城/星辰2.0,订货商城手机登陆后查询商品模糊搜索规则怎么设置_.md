# 星辰2.0,订货商城手机登陆后查询商品模糊搜索规则怎么设置?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：姜宁8 | 阅读：431 | 有用：0

【问题描述】

![](https://vip.kingdee.com/download/01098aa8c0a5e28946b08090c362886b44f2.png)【

【解决方案】

搜索条件支持:综合搜索(后面几种方式综合)商品名称搜索、助记码搜索、商品编码搜索、规格型号搜索

【APP端操作可以点击截图位置综合后面的小三角下拉,默认设置点击网页{订货商城}——{商城系统配置}——{其他设置}——选择默认搜索条件】
