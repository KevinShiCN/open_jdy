# 金蝶云·星辰，订货商城，怎么设置单品限时特价？

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题, 快速入门
> 作者：Lizzy | 阅读：1331 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/283618069174607104)

问题描述：

解决方案：

在基础资料-客户价格资料，设置客户价格策略。设置好策略后，到【订货商城】-【促销管理】，新增【单品限时特价】活动，将客户价格策略添加到商城活动列表。

![](https://vip.kingdee.com/download/0100fca2705c802e407abbef4ca6e36bff9a.png)

新增客户价格策略

![](https://vip.kingdee.com/download/0100c19e3420a5c34de196b8660de8acbb32.png)

新增单品限时特价

![](https://vip.kingdee.com/download/01002bc61e5e53754d53a9763e94965cff00.png)

将客户价格策略添加到商城活动

订货商城对应客户将按照客户价格资料规则取价，商城商品详情将展示特价样式。商品图片下方展示优惠力度最大的商品，即对应价格减零售价最大值。
