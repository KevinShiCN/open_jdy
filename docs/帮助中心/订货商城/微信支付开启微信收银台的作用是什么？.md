# 微信支付开启微信收银台的作用是什么？

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：进阶技巧, 功能介绍, 新特性
> 作者：Lizzy | 阅读：858 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/593480202882001152)

在线支付通道支持金蝶支付小程序收银台和通联小程序收银台，不再受微信直连相关政策限制。

根据2022年12月25号更新的「交易类小程序运营规范!，适用于符合如下一种或多种情形的小程序:

1、小程序内提供珠宝玉石、3C数码、盲盒、服饰内衣、海淘、美妆、酒类、家用电器、玩具、箱包皮具、鞋靴、运动户外等商品在线销售及配送服务;

2、小程序的账号主体为近一年内新成立的企业或个体户主体;

3、小程序的账号管理员、运营者等角色，与其它高风险小程序存在关联;

4、小程序内经营预售商品。
如符合以上情况的，需在用户主动/系统自动确认收货后(在用户点击“确认收货”、或商家录入发货信息后达到系统自动确认收货的时间周期)，进行资金结算不支持绑定机构类商户。

命中以上4种情况的小程序支付，只能用微信直联通道0.6%费率或B2b门店助手通道。

开通以后可以享受支付费率最高0.38%。具体开通及政策请咨询通联支付和金蝶支付。

配置路径：

（1）基础资料-支付渠道，新增自建小程序的支付渠道时，开启勾选收银台。

![](https://vip.kingdee.com/download/010986cd7240444e4c0b943c321a64e7d4dc.png)

若您的版本未有基础资料“支付渠道”菜单，可到订货商城-商城支付配置，微信支付开启微信收银台；

![](https://vip.kingdee.com/download/010995aa0030aa024f49905c0c9f0d47da2d.png)

（2）小程序需升级3.3.8以后版本（查看路径：订货商城-微信小程序设置）；

（3）联系第三方支付厂家开通微信小程序收银台。
