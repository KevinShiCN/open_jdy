# 小程序审核提示：小程序功能不符合规则：小程序内涉及收集“摄像头/位置信息/麦克风”相关接口组件，请通过接口完善【用户隐私保护指引】后再重新提审，如何操作？

## 1 问题描述

小程序审核提示：小程序功能不符合规则：小程序内涉及收集“摄像头/位置信息/麦克风”相关接口组件，请通过接口完善【用户隐私保护指引】后再重新提审，如何操作？

![](https://vip.kingdee.com/download/0109919a40f869f240ef827990f347d1a568.png)

## 2 适用版本

金蝶云·星辰v3.1标准版/专业版/旗舰版/所有版本

## 3 原因解析

出现上述情况，是因为众多小程序同时补交小程序隐私协议，微信方会出现接口繁忙情况提交失败，导致小程序审核失败。

## 4 解决方案

订货商城-微信小程序设置，重新扫码授权发布小程序，系统会自动提交用户隐私协议，提交后等待微信审核通过即可。
