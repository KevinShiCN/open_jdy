# 小程序商城审核通过后，微信搜索不到，为什么？

## 已发布小程序搜不到的常见原因和解决办法

小程序已发布上线，为什么使用微信搜一搜却搜不到？其实造成这种情况的原因有很多，如何解决还是要根据实际情况而定。下面为常见原因和解决办法。

1、收录时效

根据微信平台规则，在小程序审核发布后，需要1-2天左右才能被收录，也就是我们所说的被搜索到。

解决办法：如果发布时间尚短，建议先保持观察。

2、违规或被举报

比如简介中包含了敏感词，小程序内容涉及违规，或者部分关键词被多次举报等情况，会导致被系统屏蔽、关闭看关键词/敏感词的搜索、被微信平台封禁等等。

解决办法：建议大家先登录微信公众平台，自查小程序站内信，查看是否有相关通知，若有，根据官方要求在站内信中申诉即可。

**3、小程序后台错误设置**

在小程序后台设置了不允许被搜索或暂停服务，导致无法用户无法在微信内通过名称搜索到小程序账号。这两项设置都是需要手动关闭才会变成不允许被搜索以及暂停服务，有可能会被管理员误操作关闭。

解决办法：通过登录小程序后台—>设置—>基本设置—>隐私设置/暂停服务设置中可查看。

![](https://vip.kingdee.com/download/0109885f551cd122422aa31caae219cb4fb1.png)

4、排名靠后

实际上已收录并可以搜索到，只是排名过于靠后，曝光力不够明显，导致用户无法有效抓取到，被误认为未收录。

解决办法：这种情况建议商家要重视并做好小程序的搜一搜优化，保持定期的内容更新，避免因更新不及时出现死链影响排序结果。努力提高自己的小程序排名，就能获得更多用户流量。
