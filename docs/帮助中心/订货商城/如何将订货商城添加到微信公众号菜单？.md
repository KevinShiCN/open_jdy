# 如何将订货商城添加到微信公众号菜单？

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题, 进阶技巧
> 作者：Lizzy | 阅读：9180 | 有用：10
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/24000)

问题描述：

解决方案：

第一步：商家注册公众号账号

1、登录 [https://mp.weixin.qq.com,](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fmp.weixin.qq.com%2F)点击右上角立即注册

![](https://vip.kingdee.com/download/0100dee342ba3e6b466dade2d0bf4324dc53.png)

2、选择【服务号】

![](https://vip.kingdee.com/download/0100372c27dffcc047178021269faa48ff96.png)

3、按照微信要求填写信息完成注册

![](https://vip.kingdee.com/download/0100ed564f45e25340e688650068ed523c9a.png)

第二步：注册成功后，进入服务号后台-左侧菜单-自定义菜单

![](https://vip.kingdee.com/download/0100aa0dcc42ac74498895b10b7604c23a10.png)

第三步：将商城绑定到微信公众号

根据商户需求，添加一个或多个菜单，每个菜单还可以添加子菜单给菜单命名，选择内容为【跳转网页】，到订货商城后台-商城预览复制商城链接填入。

![](https://vip.kingdee.com/download/01008864a0a273b14db1ba5c8e4a15e5068d.png)

![](https://vip.kingdee.com/download/01007b5be3c562714be9be1f26ca163cf217.png)

客户关注您的公众号后，可以从公众号进入您的店铺浏览商品，并完成下单。

若公众号菜单提示如下图时，请先进行公众号认证。

![](https://vip.kingdee.com/download/010061db535378b74e4a95fb6770cdc56ae3.png)
