# 金蝶云·星辰 订货商城客户下单提示：“订单保存失败：您身份信息有变化，请重新登录！”

> 作者：商贸韩佩奇 | 阅读：210 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/611852405050949632)

****

1 问题描述

订货商城客户下单，金蝶云·星辰 订货商城客户下单提示：“订单保存失败：您身份信息有变化，请重新登录！”

![](https://vip.kingdee.com/download/0109b3314208b09e46f6afd07678eb2ba010.png)

## 2 适用版本

金蝶云·星辰v3.1标准版/专业版/旗舰版/所有版本

## 3 原因解析

这种情况通常是，禁用客户导致的，需要在基础资料-客户中搜索登陆手机号是否禁用。（需注意，有可能一个手机号设置到两个客户名下，禁用了一个，因为默认筛选条件为“可用”检查时需要将筛选条件设置为禁用或不限）

## 4 解决方案

客户列表：查看禁用客户中，当前客户是否被禁用（1、正式客户可根据手机号查询对应客户是否被禁用；2、商城散客需要启用系统内置的订货散客，禁用系统默认的订货散客，订货商城散客将无法下单，也无法生成销售订单）
