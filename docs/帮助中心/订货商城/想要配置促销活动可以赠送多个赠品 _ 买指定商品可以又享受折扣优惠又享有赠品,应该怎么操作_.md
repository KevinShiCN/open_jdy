# 想要配置促销活动可以赠送多个赠品  /  买指定商品可以又享受折扣优惠又享有赠品,应该怎么操作?

> 作者：姜宁8 | 阅读：632 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352831410309376)

【问题描述】

想要配置促销活动可以赠送多个赠品/买指定商品可以又享受折扣优惠又享有赠品,应该怎么操作?

【解决方案】

高级促销支持配置优惠规则多种类型组合。

(1)方案和方案间不支持叠加,效果相同于新建多个促销活动,适合相同活动时间内快速建立方案;

(2)一个方案内支持配置多个优惠规则。

- 若配置了满件数和满金额的优惠条件,在都满足条件可享优惠叠加。

例如:购买指定范围内商品满2000元打折9.8,满50件则享价值200元的赠品,同时满足这两个条件可以享有该活动。

- 若配置了同种优惠类型且开启了倍增,则倍增条件以最后阶梯计算倍增。

例如:设置了满200元减18元,满400元减40元,开启了倍增后。下单指定商品金额区间在[200,400)可减18元,[400,800)可减40元,下一个阶梯为[800,1200)可减80,以此类推。

(3)同个优惠规则可以设置多种方式的优惠,例如满足设置条件后可享有赠品及满减优惠金额。![](https://vip.kingdee.com/download/01090d088f2bad9b4ff296cd628f8482c43b.png)
