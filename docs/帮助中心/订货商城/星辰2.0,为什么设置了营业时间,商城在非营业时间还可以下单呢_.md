# 星辰2.0,为什么设置了营业时间,商城在非营业时间还可以下单呢?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：姜宁8 | 阅读：272 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352630184380672)

【问题描述】

为什么订货商城设置了营业时间,商城在非营业时间还可以下单呢?

【解决方案】

开启下单时间校验且设置营业时间,则只有在营业时间才能下单成功,其它时间不能下单。检查一下客户的设置是否有勾选下单时间校验的参数![](https://vip.kingdee.com/download/01096fe6b2469efc4a44bd50cec9af0dd5cc.png)
