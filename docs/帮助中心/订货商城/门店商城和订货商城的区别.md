# 门店商城和订货商城的区别

【问题描述】

门店商城和订货商城的区别

【解释说明】

零售商城和订货商城同为金蝶云星辰的功能模块,互相独立,可分别购买使用。

零售商城: 针对零售行业,帮助零售商家在线上开店,为个人消费者提供线上购物平台。零售商城可以满足更多与会员、营销相关的功能场景。零售的小程序是针对有自己门店的商家,且小程序服务的是门店周边可到店的消费者。同时零售商城可以与零售 POS 结合使用,更好地实现线上商城与线下门店的融合。  而对于零售行业客户,特别是已经有线下门店的客户,推荐使用零售商城。

![](https://vip.kingdee.com/download/0109e5a6eb549b2b4b71b00b99625b50e592.png)

订货商城:是针对有批发业务的商家,商城是商家的下游经销商或者门店订货系统;

![](https://vip.kingdee.com/download/0109f592c3541a594814866ab45603ca9ea9.png)

订货是B2B商城,零售是B2C商城,由于交易对象不同,产生的交易交互行为,客户管理,价格管理,库存管理,营销管理,都有行为差异。两个应用是为两种交易场景产生的应用。具体的功能也会相应产生差异。
