# 如何设置商品库存不足时不允许商城客户下单?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：姜宁8 | 阅读：553 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352577922634240)

【问题描述】

如何设置商品库存不足时不允许商城客户下单?

【解决方案】

在【订货商城】-【商城系统配置】-【商城设置】-【库存展示设置】中选择【展示基于进销存的商城可用库存】参数,并且选择【库存不足无法下单】;![](https://vip.kingdee.com/download/0109028c8e6e834b456692f033171da3ed46.png)
当商品库存不足时, 商城里商品会显示“无货”,并且没有加购商品的按钮。
![](https://vip.kingdee.com/download/01095cc2aa7a0ddf4e40821b255ceae9902b.png)
