# 订货商城用户管理添加手机号提示当手机号已被用作用户管理中的联系方式时,该账户无法启用子公司管理功能是什么意思?

> 作者：红笺小字 | 阅读：58 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/779162822663680768)

【问题描述】

订货商城用户管理添加手机号提示当手机号已被用作用户管理中的联系方式时,该账户无法启用子公司管理功能是什么意思?

【解释说明】

【订货商城】-【商城用户管理】里面设置订货-客户启用子公司管理(订货角色为老板才可以设置启用子公司管理,其他角色按钮无法启用)

![](https://vip.kingdee.com/download/01098fd5972350394d8a8d07ebf88a706769.png)

保存提示:当手机号已被用作用户管理中的联系方式时,该账户无法启用子公司管理功能。

![](https://vip.kingdee.com/download/01093b8e036cb8604c38bc6ee350b99c6237.png)

这是因为对应设置的手机号是系统用户管理里面某一个账号绑定的手机号,系统用户不支持开启子公司管理。

![](https://vip.kingdee.com/download/010968f7ade9e5334f3eaf8fb353ce03c944.png)
