# 订货商城商品限购,如何删除/禁用?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：Cat_Chow | 阅读：116 | 有用：0

【业务场景】

订货商城商品限购,如何删除/禁用?

【操作步骤】

1、打开【订货商城】-【商品限购】

![](https://vip.kingdee.com/download/010927bce7b9a3624f7ba91b7d7aed869c26.png)

2、打开商品限购方案列表

![](https://vip.kingdee.com/download/0109582b6a8b40a6493584fcf8bbdf7ef24d.png)

①禁用:若限购方案不需要了可以直接禁用,在商品限购方案列表中选择需要禁用的方案点击【禁用】即可,禁用方案后商品客户可以正常购买商品。

②启用:已禁用的方案支持重新启用。

③删除:若限购方案不需要了可以点击【删除】;**注意删除后的方案不能再恢复**,可以重新新增。
