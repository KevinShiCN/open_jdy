# 商品分等级设置价格，如何在订货商城应用？订货商城客户价格资料“仓库+商品”类型功能介绍

# **1 简介  **

## **1.1 功能介绍**

订货商城支持取价“仓库+商品”类型的客户价格资料，主要应用于商品通过仓库进行商品分等级管理管理，并设置不同的商品等级价格不同价格，或商品价格按区域进行管理，不同区域不同价格的场景。

## **1.2 适用版本**

星辰3.0以上专业版/旗舰版，含订货商城模块

# **2 操作指南**

1）第一步,设置商城库存展示方式为“展示基于进销存的商城可用库存”

路径：订货商城>商城管理>商城系统配置>库存展示设置

![上传图片](https://vip.kingdee.com/download/01004d88ef6a5393489d895d869b3a0f5480.png)

2）第二步,开启商城发货仓为“单选”

路径：订货商城>商城管理>商城页面设置>商品列表筛选项

![上传图片](https://vip.kingdee.com/download/010023e05bab6f3b4fb18a42207ed0a589db.png)

3）第三步,创建“仓库+商品”类型的客户价格资料

路径：基础资料>价格设置>客户价格资料，新增方案选择类型“仓库+商品”类型。

![上传图片](https://vip.kingdee.com/download/0100cf8ca9b09b444b4a9d5967eca520ae6a.png)

![上传图片](https://vip.kingdee.com/download/01002dc8dccf7f2f4827811b8fe16ac97c1a.png)

# **3 商城端效果**

（1）发货仓库：客户可见仓库列表中，库存数量>0的仓库列表

![上传图片](https://vip.kingdee.com/download/0100797e7e334c9c4131afa012faf8bc0595.png)

（2）单选后仓库，当商品存在有效的“仓库+商品”客户价格资料，将获取该仓价格。

![上传图片](https://vip.kingdee.com/download/010053e1f8ee7bee461cabbe5c2a390fb969.png)   ![上传图片](https://vip.kingdee.com/download/01008b69ec5e969b40cfa09fcde69da3c7d3.png)

（3）筛选仓库后加入购物车，将携带指定仓库提交订单。

![上传图片](https://vip.kingdee.com/download/01003053604902224e47bc41015e19d70381.png)  ![上传图片](https://vip.kingdee.com/download/01005c8a7e5bec0146b29110beec3dcb8fc4.png) ![上传图片](https://vip.kingdee.com/download/0100ecef50f8247041d2bf2df18e732b885c.png)

（4）商城订单审核及下推到销售订单将填充指定仓库。

![上传图片](https://vip.kingdee.com/download/01002965e9c107114fe3a4de30a3451b4e49.png)

# **4 其他说明**

（1）商城首页组件支持链接“仓库+商品”的价格资料。

①单品限时特价添加客户价格资料

![上传图片](https://vip.kingdee.com/download/01003f2292b396f447cb8c62ff7afe1741ea.png)

![上传图片](https://vip.kingdee.com/download/010011402df2a7b04f8ba0f010eab81d5da5.png)

②商城首页方案添加组件，链接客户价格策略，并设置对应仓库，保存并发布  。

支持链接客户价格策略的组件：图片广告、图文导航、标题文本、促销组件，下面以促销组件为例。

![上传图片](https://vip.kingdee.com/download/01003995266ffe564b1a8ddd4e5bfe6e2a62.png)

（2）套装商品由于涉及子件，不支持在商品详情及购物车直接切换仓库，需要通过商品列表筛选仓库后再加入；

（3）当按照步骤1-3完成商城“仓库+商品”客户价格资料，同时开启商品详情展示发货仓，商品详情支持查看商品所在仓库的基本单位即时库存数量和对应价格，购物车切换仓库支持查看对应单位的即时库存数量。

路径：订货商城>商城管理>商城页面设置>商品详情页，商品详情页展示开启“发货仓”。

![上传图片](https://vip.kingdee.com/download/010048cb951ec7f640c9baa8f8dd480b2351.png)

![上传图片](https://vip.kingdee.com/download/0100f959c546e98847f599ffe0cd3e41fe2a.png)  ![上传图片](https://vip.kingdee.com/download/0100884cfd5600f64df39412af1e97d7f91e.png)

（4）若加购过程中，指定仓库库存数量为0，选购切换仓库隐藏库存为0的仓库，并自动切换到“自动分配”仓库。

（5）若加购过程中，商品列表或商品详情对已加购商品切换仓库，将更新购物商品指定仓库，提交订单以最后一次更新提交。

（6）未审核的商城订单，若订单已经指定仓库，由于涉及价格，不持支编辑订单修改，若需要编辑，请取消订单通过再来一单将商品加入购物车，重新下单。
