# 订货商城订单下推时生成销售订单,多单位商品的辅助单位会带出哪个?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：云社区用户55os622 | 阅读：121 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/662099615134557952)

【问题描述】
订货商城订单下推时生成销售订单,多单位商品的辅助单位会带出哪个?

【解释说明】
可通过参数进行配置,依次点击【订货商城】-【商城系统配置】![](https://vip.kingdee.com/download/010901eabda1c5f94898887608741b2ddc29.png)

点击到【交易设置】-【订单下推时生成销售订单的多单位商品的辅助单位填写】,可选择设置取:与购买商品单位保持不同 或 与商品的报表辅助单位保持一致
![](https://vip.kingdee.com/download/0109efba200c15824cfbb47d87fe3af305cf.png)

【与购买商品单位保持不同】:查看订单时方便换算查看不同单位的数量
【与商品的报表辅助单位保持一致】:方便统一单位查看单据与报表的数据(浮动单位商品除外)
