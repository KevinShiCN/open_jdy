# 订货商城的运费配置内的自定义是做什么用的，是否需要配置？

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题, 快速入门
> 作者：Weixinxin | 阅读：63 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/754400704936225280)

# **问题**：

订货商城的运费配置内的自定义是做什么用的，需要配置什么才能用？

![上传图片](https://vip.kingdee.com/download/0100b3f591675f7748bebde72aab56cb7ebd.png)

# **解答**：

订货商城运费配置中的自定义，指的是自定义运费付款方式， 例如可以设置为垫付等。除了寄付方式，系统会进行运费计算和收取外，像到付、月结以及自定义的付款方式，系统仅作标记 ，实际的运费收取需要商家自行留意并操作。
