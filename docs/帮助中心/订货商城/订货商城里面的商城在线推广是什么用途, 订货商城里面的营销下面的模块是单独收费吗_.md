# 订货商城里面的商城在线推广是什么用途, 订货商城里面的营销下面的模块是单独收费吗?

> 作者：谭偲娉 | 阅读：323 | 有用：5
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/490688386873893888)

【问题描述】

订货商城里面的商城在线推广是什么用途, 订货商城里面的营销下面的模块是单独收费吗?

【解释说明】

1、商城在线推广主要是为了宣传、推广商城内的商品而进行的操作,同时还可以在商城维护文章列表入口,展示公司信息,品牌等信息。

2、订货商城里面的营销下面的模块不用单独收费,需要的话可以直接使用。

点击【订货商城】-【营销】下面对应的功能模块即可设置营销相关方案,如图:
