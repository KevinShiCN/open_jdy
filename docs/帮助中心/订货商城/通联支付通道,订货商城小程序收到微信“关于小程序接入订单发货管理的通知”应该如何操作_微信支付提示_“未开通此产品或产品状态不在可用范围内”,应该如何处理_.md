# 通联支付通道,订货商城小程序收到微信“关于小程序接入订单发货管理的通知”应该如何操作?微信支付提示:“未开通此产品或产品状态不在可用范围内”,应该如何处理?

【问题描述】

通联支付通道,订货商城小程序收到微信“关于小程序接入订单发货管理的通知”应该如何操作?

![](https://vip.kingdee.com/download/010905aea7caf85c406589899574b97b9f74.png)

微信支付提示:“未开通此产品或产品状态不在可用范围内”,应该如何处理?

![](https://vip.kingdee.com/download/01095bf94f2ea297479b99710709809e71f4.png)

【适用版本】

含订货商城模块,使用小程序微信支付,且支付通道为通联支付。

【解释说明】

1)小程序收到“关于小程序接入订单发货管理的通知”,则说明您的交易收到微信小程序的官方管控,特定类型的小程序需要在平台完成发货信息录入及确认收货流程后方可进行资金结算。

2)发起微信支付提示:“未开通此产品或产品状态不在可用范围内”,您的商户号开通异常,可能需要开通通联支付的【微信订单预消费】,需联系通联方确认商户号情况。

【解决方案】

1)请**联系通联方**(帮您开通商户的工作人员)确认商户号是否收到管控,需要**开通新的交易类型【微信订单预消费】的交易类型。**

2)确认通联方已开通【微信订单预消费】的交易类型后,到订货商城-商城系统配置-其他设置,开启“小程序发货信息自动录入”。

**【若没看到该功能,请联系客服升级3.1版本,并通知技术人员】**

![](https://vip.kingdee.com/download/0109fcf7ee127ceb4df3abbc4057ffe59e14.png)

3)开启后,商城将在支付时调用通联支付交易类型为【微信订单预消费】

4)小程序发货信息,系统将自动按照参数配置上传发货信息,详见:**[小程序订货商城需要去小程序后台操作发货操作,能不能系统自动批量标记发货?](https://vip.kingdee.com/knowledge/582971119951287552?productLineId=35&isKnowledge=2&lang=zh-CN)**
