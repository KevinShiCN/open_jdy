# 订货商城或协同助手的商品列表的价格单位想要显示为指定单位，怎么设置？

# **1 问题描述**

订货商城或协同助手的商品列表的价格单位想要显示为指定单位，怎么设置？

# 2 适用版本

星辰3.0以上标准版/专业版/旗舰版/所有版本的协同助手；

星辰3.0以上专业版/旗舰版的订货商城

# **3 原因解析**

当您的商品是多单位商品，客户浏览商品列表查看价格是最小单位的价格，您希望展示为常用的销售单位的价格。我们可以通过销售单位设置和参数设置达到效果。

# **4 解决方案**

1）第一步，设置参数【订货商城】>【商城管理】>【商城系统配置】，参数”商城列表取价设置“切换为”销售单位中的最低价“

![上传图片](https://vip.kingdee.com/download/0100af130c0ed4db44aba2e4be1bb350c944.png)

2）第二步，【基础资料】>【商品资料】>【商品信息】，设置商品的销售单位为指定的单位

![上传图片](https://vip.kingdee.com/download/01006e42f2883ad545fe88340e23d08e846a.png)

3）设置后效果
