# 金蝶云·星辰订货商城，小程序发布微信审核失败常见原因？

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：快速入门, 常见问题
> 作者：Lizzy | 阅读：3929 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/108984119752518144)

问题描述：

解决方案：

订货商城小程序上架需要经过微信平台审核，当审核失败时，可以到到微信公众平台-登录小程序后台查看详细原因。

微信公众平台链接：https://mp.weixin.qq.com/

![](https://vip.kingdee.com/download/01095676191a13d04d49b586d6fea0c870f7.png)

下面常见的审核失败原因：

1、提示：“小程序功能不符合规则：小程序账号登录功能暂未符合规范要求，请在用户了解体验小程序功能后，再用户进行账号登录。请整改后在重新提交审核。”

解决办法：开启游客可浏览商城，重新提交审核，待审核通过后再关闭。

2、提示：“小程序服务提供内容涉及电商模式（如：含购物车或涉及订单按钮，含立即购买流程等），属于个人小程序，建议选择企业主体小程序。”

解决办法：前往公小程序平台进行微信认证，再提交重新审核，微信认证入口：登录小程序 -> 设置 - >微信认证详情。

参考腾讯文档链接：小程序注册主体验证方式[https://kf.qq.com/faq/170109QvMNRB170109nYnYFr.html](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fkf.qq.com%2Ffaq%2F170109QvMNRB170109nYnYFr.html)

3、提示：服务类目与页面内容不一致，或核心功能未在账户中设置中登记服务类目。

解决办法：前往公小程序平台修改或添加服务类目，再提交重新审核，可以查看其他已上线的类似小程序所属类目。修改入口：微信认证入口：登录小程序 -> 设置 - >服务类目

4、提示：“小程序打开页面空白，无具体运营内容”

解决办法：到订货商城后台上架商品并配置商城首页，再提交重新审核。

5、提示：特殊类目（如金融业、驾校预约、医疗等）

解决办法：在公众号平台选择对应类目后，补充资质证明材料。

6、提示：商城登录报错，截图报错含：“invalid appsecret...”

解决办法：检查订货商城后台Appid和秘钥是否正确。参考链接：https://vip.kingdee.com/questions/60724971630621184/answers/60724972452705280
