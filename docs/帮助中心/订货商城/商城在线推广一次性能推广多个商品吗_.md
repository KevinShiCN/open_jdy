# 商城在线推广一次性能推广多个商品吗?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：云社区用户55os622 | 阅读：97 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/625134449897889792)

【问题描述】

商城在线推广一次性能推广多个商品吗?

【解释说明】

商城在线支持批量推广多个商品

操作路径:【订货商城】-【商城在线推广】列表,勾选多个商品,点击<推广链接>

![](https://vip.kingdee.com/download/010913f0266ade41496ca84ed8f89db4a02d.png)
编辑推广方案之后复制方案+链接或者只复制链接
![](https://vip.kingdee.com/download/010967a9c626d0f3462196af47f2f3d92c91.png)
打开推广链接可展示多个商品
