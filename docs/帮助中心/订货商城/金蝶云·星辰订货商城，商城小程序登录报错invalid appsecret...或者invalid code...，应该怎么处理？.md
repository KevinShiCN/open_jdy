# 金蝶云·星辰订货商城，商城小程序登录报错invalid appsecret...或者invalid code...，应该怎么处理？

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题, 进阶技巧
> 作者：Lizzy | 阅读：1961 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/60724972670659840)

问题描述：

![](https://vip.kingdee.com/download/01002752ca0b7f974369b24769438235a95c.png)

解决方案：

商城小程序登录报错invalid appsecret或者invalid code...，是商城配置小程序的appid及秘钥填写错误，请重新检查秘钥并正确填写。

invalid appsecret指的是错误的小程序秘钥；invalid code指的是错误的小程序appid

请进行以下操作：

1、解绑小程序

![](https://vip.kingdee.com/download/0100fdd5a09eb8b54555908065194735662a.png)

2、获取密钥：

    打开微信公众平台网址[https://mp.weixin.qq.com/](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fmp.weixin.qq.com%2F)，登录账号，开发-开发者设置-开发者ID，获取微信APPID和微信APP密钥。其中AppSecret的查看需要点击该行后面的“重置”或者“生成”链接，查看后要记得把AppSecret在其他位置记录下来。

![](https://vip.kingdee.com/download/010047402455034e4d8094bdee2df85b5549.png)

3、到精斗云订货商城后台，小程序发布，对应填写。

![](https://vip.kingdee.com/download/01009fbc97195d49407486b5e9b21695f2bb.png)
