# 订货商城,如何设置已优惠过的商品不积分?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：云社区用户55os622 | 阅读：97 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/663083022337750528)

【问题描述】
订货商城,如何设置已优惠过的商品不积分?

【业务场景】
例如,当商品已经通过打折、满减等促销活动参与过优惠时,商家可以选择不额外赠送积分,从而避免过度促销导致利润率下降。

【解释说明】
依次点击【订货商城】-【积分规则】![](https://vip.kingdee.com/download/0109986ffbf8c0f549199212c125a11dddae.png)
点击进入具体需要调整的积分规则![](https://vip.kingdee.com/download/0109eacfe531797b46218a98bc4aa22b6f85.png)
在【不参与积分场景】处,勾选<已参与促销的商品不积分>,<已用优惠券的商品不积分>,即可完成设置

![](https://vip.kingdee.com/download/0109dba2475ce7ab4b44b5e755bca3816068.png)
