# 小程序商城是否支持指定收款账号?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题

【问题描述】

小程序商城是否支持指定收款账号?

【解释说明】

目前星辰系统暂**不支持**指定小程序商城收款账号。小程序商城通过支付宝、微信或第三方支付渠道对接,收款账号由微信、支付宝或者第三方支付平台中绑定的账户决定。
