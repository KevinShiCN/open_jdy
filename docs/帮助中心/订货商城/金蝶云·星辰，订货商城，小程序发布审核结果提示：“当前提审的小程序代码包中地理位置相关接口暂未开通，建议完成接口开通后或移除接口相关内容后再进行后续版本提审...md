# 金蝶云·星辰，订货商城，小程序发布审核结果提示：“当前提审的小程序代码包中地理位置相关接口暂未开通，建议完成接口开通后或移除接口相关内容后再进行后续版本提审...”，应该怎么操作？

> 作者：Lizzy | 阅读：1504 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/311910224363630592)

问题描述：

![](https://vip.kingdee.com/download/01091ac57b0b5527429faa9a42b782603da8.png)

解决方案：

由于微信的接口能力变更，由于订货商城的下单地址填写为方便客户定位获取地址有用到微信提供的地址能力：[wx.getLocation](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fdevelopers.weixin.qq.com%2Fminiprogram%2Fdev%2Fapi%2Flocation%2Fwx.getLocation.html)和[wx.chooseLocation](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fdevelopers.weixin.qq.com%2Fminiprogram%2Fdev%2Fapi%2Flocation%2Fwx.chooseLocation.html)。因此需要对应填写相关申请。操作路径如下：

打开小程序后台登录网址[https://mp.weixin.qq.com/](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fmp.weixin.qq.com%2F)，管理员扫码登录小程序，点击开发-开发管理-接口设置。

![](https://vip.kingdee.com/download/0109e5bc530681e04a7f91848ea28a8532ca.png)

一、点击【打开地图选择位置】开通按钮，填写原因：“因当前业务需要，为快速获取地址信息并填写到收货地址中，需要地址相关信息”；下载本文附件视频（填写收货地址提交订单）并解压后在申请页面上传视频，填写后提交。待微信审核通过。

![](https://vip.kingdee.com/download/010911cb8f3088ab48e291beedb730035cfb.png)

![](https://vip.kingdee.com/download/0109c57e032a6faa44e2814e50bc26b11ec5.png)

1、打开辅助网页链接，登录自营商城；

2、使用测试账号：手机号xxxxxxxxxxx，密码xxxxxx，登录商城；

3、点击下方菜单我的 - 收货地址-新增地址-点击地图选择，获取定位反写收货地址信息。

（4）视频请下载本文附件压缩包中的视频（填写收货地址），上传后提交。待微信审核通过。

二、重新发布小程序

**接口都通过了需要做什么？**

回到星辰-订货商城-小程序发布配置，重新提交小程序发布，小程序版本需在v2.0.12以上。
