# 订货商城优惠券

1 简介

### 1.1 功能介绍

优惠券功能主要应用于商品优惠的场景，可以帮助商家提升复购率与客单价。

目前优惠券分为两种类型：

（1）代金券：买家可凭券享抵扣现金服务。如：无门槛减10元、满100元减20元 ；

（2）折扣券：买家可凭券享消费折扣服务。如：无门槛打9折、满100元打9折 ；

1.2 系统路径

订货商城-营销-优惠券

## 2 操作指南

（1）选择方案：代金券/折扣券

![](https://vip.kingdee.com/download/0109040c7a69ac1149c0b91659aee0a65e07.png)

（2）按提示填写优惠券方案![](https://vip.kingdee.com/download/0109525b88a85dae4fa994949c5e5257d120.png)![](https://vip.kingdee.com/download/010966a90e03edfe4b86b84e4758a4284f76.png)

（3）保存优惠券方案后，在优惠券列表进行优惠券发放，选择一个优惠券方案，点击发放

![](https://vip.kingdee.com/download/0109589ff9b5a3ed46eb94f081b60d81d655.png)

（4）选择发放的用户和优惠券发放数量

![](https://vip.kingdee.com/download/01093e96222def4c477f93472066452a140a.png)

（5）发放成功后，在订货商城下单时可使用优惠券

![](https://vip.kingdee.com/download/010907b94f6e0060444a8db355088a75c935.png)
