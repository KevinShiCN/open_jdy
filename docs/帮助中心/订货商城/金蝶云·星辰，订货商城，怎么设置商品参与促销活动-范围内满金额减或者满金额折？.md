# 金蝶云·星辰，订货商城，怎么设置商品参与促销活动-范围内满金额减或者满金额折？

# **一、范围内满金额减**

单击【订货商城】-【批发促销】，新增【满金额减】活动，设置活动名称、活动时间、指定商品范围、优惠规则和适用对象。![](https://vip.kingdee.com/download/01095c19fc8650ba4addbd0f21add360e1c6.png)

![](https://vip.kingdee.com/download/01099c4f6ec81a12405fbd490193ea0b24b7.png)

优惠规则支持设置多规则和倍增，倍增仅适用于最后一条规则。

例如，设置多规则满100元减2元，满250元减5元。满250元减5元支持倍增。

当参与活动金额120元则减2元；金额200元则减2元，不能参与100元减2元的倍增；金额250元则减5元；金额520元则减10元，参与满250元减5元倍增。

![](https://vip.kingdee.com/download/010038a32687978b4ce5a90d5b4c1fdba49d.png)

满金额减优惠规则设置

商城用户选购参与活动的商品合计金额满足条件，则在提交订单界面显示对应促销立减金额。

![](https://vip.kingdee.com/download/01009ee0cbdfc5cb476db59682380443fd5e.png) ![](https://vip.kingdee.com/download/010067ff2d09bd19446c912152e20e2a3854.png) ![](https://vip.kingdee.com/download/0100dfb0a26a2b0d46b2a30368a44dc6eac1.png)

## 二、范围内满金额折

满金额折指商城用户选购范围内的商品满金额以上，提交订单是参与打折。满金额减活动设置和商场参与活动与满金额减活动相同，可参见上一节。满金额减活动不能设置规则倍增。
