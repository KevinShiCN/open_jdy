# 订货商城优惠券领用记录如何引出?

【问题描述】

订货商城优惠券领用记录如何引出?

【操作步骤】

1)第一次引出需要新增模板。如下图所示![](https://vip.kingdee.com/download/0109bf1ae77cb78546ad8e7553169a23a17b.png)

2)根据业务需要,把需要的字段【是否允许引入引出】打开显示,可选择引出属性,名称/编码。选择完成,维护模板编码,名称。点击保存![](https://vip.kingdee.com/download/01099de8930829d042cb81ae05e30c91e8ff.png)

3)引出-选择模板-确定![](https://vip.kingdee.com/download/01096032bbeb7f654654a2dd196e10d97835.png)

4)引出效果如下图所示![](https://vip.kingdee.com/download/0109fa36c77f330c44f5bb75d673ff86ae72.png)
