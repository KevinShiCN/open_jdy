# 订货商城微信小程序审核,提示:小程序上架需先完成备案?

【问题描述】

订货商城微信小程序审核,提示:小程序上架需先完成备案?

![](https://vip.kingdee.com/download/01098a9ffad9ff3249f181136629645c0a1a.png)

【解释说明】

**微信小程序备案说明:**

微信小程序已于2023年9月1日上线备案系统:

1. 若微信小程序未上架,自2023年9月1日起,微信小程序须完成备案后才可上架;

2.若微信小程序已上架,请于2024年3月31日前完成备案,逾期未完成备案,平台将按照备案相关规定于2024年4月1日起进行清退处理。

3. 备案系统将于9月1日上线。

4. 备案流程:备案信息填写-平台初审-工信部短信核验-通管局审核-备案成功。

5.各省通信管理局将在1-20个工作日内(具体以实际审核时间为准)完成审核,审核结果以站内信、模板消息、短信等形式通知。

**
**

【解决方案】

请各商城小程序及时完成小程序备案,避免影响商城运营:

微信官方小程序备案操作指引,详细点击:[小程序备案操作指引](https://developers.weixin.qq.com/miniprogram/product/record_guidelines.html)
