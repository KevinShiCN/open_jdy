# 星辰2.0,商城商品详情显示[产地]如何取消?商城商品列表展示内容里,并没有勾选[产地]项,且是商品信息里有录入[产地]则显示,没有录入不显示?

> 产品：金蝶云·星辰 | 分类：订货商城 | 类型：常见问题
> 作者：姜宁8 | 阅读：214 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352762690579200)

【问题描述】

城商品详情显示[产地]如何取消?

【业务场景】

商城商品列表展示内容里,并没有勾选【产地】项,如图所示:![](https://vip.kingdee.com/download/010996902b35efb54af29455e5e2e465fe4c.png)

但是,商城中还是显示 “产地”
![](https://vip.kingdee.com/download/0109cda859df3d74469ba345d11e70aed561.png)
【解决方案】

这是商品详情展示,并不是商品列表展示,在  商城模版设置-商品信息  取消即可。
![](https://vip.kingdee.com/download/0109c6789a2f70f94f8db54b7741d61002c6.png)
