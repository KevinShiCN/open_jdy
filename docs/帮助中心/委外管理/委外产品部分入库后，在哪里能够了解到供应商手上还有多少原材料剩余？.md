# 委外产品部分入库后，在哪里能够了解到供应商手上还有多少原材料剩余？

> 产品：金蝶云·星辰 | 分类：委外管理 | 类型：常见问题, 进阶技巧
> 作者：Lisa798 | 阅读：989 | 有用：7
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/485135917913494272)

****

1、问题描述

- 月度对账时，怎么了解委外加工的产品入库了多少，多少没入库？
- 没入库的部分，材料数量有多少，这部分材料成本高不高？

2、场景介绍

委外加工业务中，财务对账时很关注发出去的材料，加工回来了多少产品？还有多少材料在供应商手中，材料成本有多少。

3、解决方案

1）打开《委外加工单跟踪表》，勾选显示材料。

2）点击列设置，勾选显示“入库消耗量、供应商剩余量、供应商剩余材料成本、已发数量、未发数量、发料差异率”字段。

3）点击查询，即可查看每一张委外加工单的成品入库情况、材料发料情况及剩余未入库的成品数量、在供应商手工的剩余材料的数量及参考成本。
