# 委外加工单如何导出材料明细?

> 产品：金蝶云·星辰 | 分类：委外管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：207 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/657750898151830784)

【业务场景】

委外管理中的委外加工单如何引出材料明细?

【解决方案】

委外加工单的三种引出方式均可以引出材料明细:按列表引出、引出数据(按引入模版)、引出分析数据。

![](https://vip.kingdee.com/download/0109828e8f0083a7442d97bfc235a017efef.png)

1、按列表引出

选择按列表引出前需要显示出材料明细,打开委外加工单列表界面,点击左上角展开过滤【查询范围】选择“基本信息 / 商品分录 / 材料分录”,然后点击右上角【按列表引出】。

![](https://vip.kingdee.com/download/0109e9f1b4fb63b4494a8709c71382ce1849.png)

2、引出数据(按引入模版)

这种引出方式需要选择引入模版,选择系统预置的模版或者自定义的模版引出即可。

![](https://vip.kingdee.com/download/0109f6ef512f6161472cb7b43216adba771d.png)
3、引出分析数据

第一次使用委外加工的引出分析数据时是需要新增引出模版,打开新增模版界面需要将材料分录的相关字段是否引出启用。

![](https://vip.kingdee.com/download/01098d90ebe679f84cbdbcd89f3114951bf7.png)
设置好引出模版后直接引出即可。

![](https://vip.kingdee.com/download/0109ab5a65f940024e15a2508c64d0869e1a.png)
