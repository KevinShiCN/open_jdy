1 产品概述

### 1.1 产品介绍

以委外加工单为核心，实现委外加工业务全流程的管理，含委外发料/退料、委外产品入库/退库、委外加工费处理等流程，并提供委外成本核算功能。

### 1.2 适用版本

星辰3.0以上专业版/旗舰版

4  功能清单

（主要的功能清单，可链接到具体的用户手册内容）

| 委外管理 | 二级菜单 | 三级菜单 | 功能说明 |
| --- | --- | --- | --- |
| 生产数据 | BOM维护 | 维护BOM信息，即物料清单，也叫产品结构或配方 |  |
| BOM批量修改 | 可批量对BOM单中的子项物料进行修改、替换等操作 |  |  |
| BOM正向查询 | 查询产成品或半成品的BOM结构，了解产成品在生产时用到了哪些半成品，以及半成品用到了哪些原材料 |  |  |
| BOM子件反查 | 查询某个配件在哪些半成品或成品的BOM单中用到 |  |  |
| BOM成本查询 | 根据BOM组成情况，展示产品参考成本，为销售人员报价提供参考 |  |  |
| 替代方案 | 设定物料替代的替代策略、采购策略、替代优先级等基础数据 |  |  |
| 委外单据 | 委外加工单 | 明确加工的产品及材料、数量以及加工的费用 |  |
| 委外发料单 | 记录发往供应商的材料信息 |  |  |
| 委外退料单 | 供应商在加工过程中发现材料质量不合格等原因，需要退回给企业 |  |  |
| 收料通知单（旗舰版） | 供应商发货指令通知仓库准备收料；通知质检部门准备质检的指令 |  |  |
| 质检单（旗舰版） | 记录检验结果 |  |  |
| 委外产品入库单 | 记录入库的数量及仓库信息 |  |  |
| 退料通知单（旗舰版） | 委外加工回来的产品在检验环节发现质量不符合要求时，进行退料处理 |  |  |
| 委外产品退货单 | 记录退回的产品及数量、以及对应出库的仓库等信息 |  |  |
| 委外加工费用 | 记录与供应商确认后的加工的费用 |  |  |
| 委外加工费（退货） | 确认应退加工费的产品、数量及费用 |  |  |
| 委外报表 | 委外产品明细表 | 统计一段时间内委外产品入库及退货的业务数据 |  |
| 委外材料明细表 | 统计一段时间内委外材料发料及退料的业务数据 |  |  |
| 委外加工单跟踪表 | 统计一段时间内发生的委外加工业务数据，支持查看加工单的各个状态和到期预警数据 |  |  |
| 委外加工费用明细表 | 统计一段时间内委外加工产生的费用数据 |  |  |
| 委外加工执行明细表 | 展示每张委外加工单的执行情况，可了解一段时间内委外加工单的入库数据、成本数据、加工费用及其核销数据 |  |  |
| 委外加工执行汇总表 | 按汇总依据展示委外加工单的执行情况，支持按商品汇总、按供应商汇总、按商品类别汇总 |  |  |
| 销售订单全程跟踪表 | 可了解销售订单目前生产加工的进度，包含原材料的采购进度，半成品的生产/委外进度，产成品的生产/委外进度 |  |  |
| 质量分析表（旗舰版） | 统计质检结果，有助于分析材料和成品合格率和不合格率 |  |  |
| 委外入库核算 | 委外成本核算 | 核算委外产品入库成本 |  |
| 委外成本核算结果查询 | 可查询最近10次的委外成本核算结果 |  |  |
| 委外成本构成明细表 | 展示各委外产品入库单的成本构成情况 |  |  |

注：备注了（旗舰版）字样的为仅旗舰版支持的单据。
