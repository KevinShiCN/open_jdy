# 委外产品入库单的入库成本哪里来？ 

> 产品：金蝶云·星辰 | 分类：委外管理 | 类型：常见问题, 快速入门
> 作者：Esperanza | 阅读：514 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/336180067502275584)

入库成本基本等式：入库成本本位币=材料费本位币+不含税加工费本位币。若通过委外
加工费用审核后更新了不含税加工费本位币，则入库成本本位币也会对应更新。当进行
委外入库核算时，会同时更新材料费本位币和入库成本本位币。
