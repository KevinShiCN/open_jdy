# 为什么委外加工单列表勾选移仓相关字段为什么没有生效？

> 产品：金蝶云·星辰 | 分类：委外管理 | 类型：快速入门, 精华汇总, 常见问题
> 作者：Weixinxin | 阅读：24 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/789082155094832896)

# 问题描述：

在委外加工单列表勾选显示移仓相关字段，为什么在页面上仍无法看到相关字段？

![上传图片](https://vip.kingdee.com/download/0100b59184bbca0a408da1427673b57f93b8.png)

# 解决方案：

原因是移仓相关字段是和材料相关联的，需要在过滤条件中**选择到【材料分录】，**即可在列表中查看移仓相关字段。

![上传图片](https://vip.kingdee.com/download/0100ce24c172ff4e4340ba9885afe9a6b149.png)
