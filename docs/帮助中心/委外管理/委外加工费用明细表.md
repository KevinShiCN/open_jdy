# 委外加工费用明细表

> 产品：金蝶云·星辰 | 分类：委外管理 | 类型：功能介绍, 新特性, 快速入门

**1、简介  **

**1.1功能介绍**

委外加工费用明细表是根据当前的过滤条件，展示筛选出的委外加工应付的费用情况，方便统计一段时间内委外加工产生的费用数据，支持查看不含税加工费、含税加工费、税额和已付款费用，待付款费用等信息。

**1.2适用版本**

金蝶云·星辰旗舰版/专业版

**1.3应用场景及价值**

委外加工费用明细表是根据当前的过滤条件，展示筛选出的委外加工应付的费用情况，方便统计一段时间内委外加工产生的费用数据，支持查看不含税加工费、含税加工费、税额和已付款费用，待付款费用等信息。

**2、取数说明：**

（1）数据来源：委外加工费用和委外加工费（退货），委外加工费（退货）的数量及费用相关字段在报表中显示为负数；

（2）筛选条件：

①单据日期：默认当月的第一天到当前日期，展示该时间段内发生的数据

②单据类型：默认为全部，可以仅查询委外材料出库或者委外材料退回的数据

③包括未审核单据：默认不勾选，不勾选时，将仅展示已审核的数据

④其他查询条件：支持按供应商、商品编码、商品类别、单据编码查询数据
