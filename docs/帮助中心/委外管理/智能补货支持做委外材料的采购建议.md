# 智能补货支持做委外材料的采购建议

> 产品：金蝶云·星辰 | 分类：委外管理 | 类型：功能介绍, 新特性, 快速入门
> 作者：Esperanza | 阅读：1342 | 有用：7
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/448526738171958784)

功能介绍：智能补货选单支持选择委外加工单，可根据设置的补货公式自动计算委外加工单材料的采购建议。

适用版本：星辰专业版

应用场景：可通过智能补货自动生成委外加工单材料的采购建议，实现委外加工所需材料的智能补货。

系统路径：采购管理-采购单据-智能补货

![上传图片](https://vip.kingdee.com/download/0100e4543606d84449d9ac8ff026892fdb78.png)

应用流程：

智能补货方案设置时，选单支持选择委外加工单，根据设置的补货公式，系统自动计算委外加工单材料的采购量，可生成材料的采购订单或采购申请单**。**

**操作步骤：**

①选单，选择委外加工单：

![上传图片](https://vip.kingdee.com/download/0100d85059d1aa2c4a8eb4909efedf5a7060.png)

②设置补货计算公式：

注意：补货计算公式的”减项：现有库存“必须选择【预计可用库存】，并且预计可用库存的公式里须勾选【委外占用量：已审核委外加工单】的选项，否则无法计算委外加工单的材料采购需求。

![上传图片](https://vip.kingdee.com/download/0100674935983a0a4eb4a742814aad95db40.png)

③可保存方案，下次可以在智能补货列表选择已保存的方案进行计算。

![上传图片](https://vip.kingdee.com/download/01004770f4f3cdeb4e7399b87b75273ecec1.png)

![](https://vip.kingdee.com/download/0109da3a30b417d04be593d55e6424a6844d.png)

![上传图片](https://vip.kingdee.com/download/0100181a172cc1624ed8881fe3bfd88424e1.png)

④点击开始计算：

![上传图片](https://vip.kingdee.com/download/010089269b256eb24b53a65c4e3674dbaf34.png)

⑤系统自动计算出委外加工单的材料的采购建议，【本次采购量】可根据实际需要进行调整：

![](https://vip.kingdee.com/download/0109e9ff5e5b71dd4ffabfddbca7d5cb1c33.png)

⑥勾选对应材料，点击生成采购订单或采购申请单，可生成对应的采购单据：

![](https://vip.kingdee.com/download/0109fa533256420f4dfbaa24a2ed06feb244.png)

![](https://vip.kingdee.com/download/01095a56c833808b4aa782d47fc3fa91a322.png)
