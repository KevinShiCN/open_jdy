# 委外退料单

> 产品：金蝶云·星辰 | 分类：委外管理 | 类型：功能介绍, 新特性, 快速入门
> 作者：Esperanza | 阅读：1172 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/359829284326253824)

**1、简介  **

**1.1功能介绍**

供应商在加工过程中发现材料质量不合格，需要退回给企业并要求重新提供合格可用的材料。或者加工完成后，有剩余材料退回给企业。可以用委外退料单处理。

**1.2适用版本**

金蝶云·星辰专业版

**1.3应用场景**

供应商在加工过程中发现材料质量不合格，需要退回给企业并要求重新提供合格可用的材料。或者加工完成后，有剩余材料退回给企业。仓管员接收到供应商退回的材料后，可以通过委外发料单列表的“退料”按钮生成委外退料单，将退回的材料入库。也可以在【委外管理】→【委外单据】中新增【委外退料单】，【选源单】选到对应的发料单生成委外退料单。

2、常见问题

1）委外退料单必须关联委外发料单生成？

是的，委外退料单仅支持通过选源单→委外发料单进行退料，退料数量不可超过发料的数量。

2）委外退料单的“出库成本”从何而来？

委外退料单需要区分录入时间节点，如果委外退料时，原委外发料单已通过委外入库成本核算计算出对应的成本，则关联生成的委外退料单会自动携带委外发料单成本。如果录入委外退料单时，委外发料单未核算过成本，则在进行委外入库成本核算时，会同时计算委外发料及关联的委外退料单成本。

    3、录单指南：

![](https://vip.kingdee.com/download/01009fa3bfceb6fa41bd8080fa8d61819513.png)

单击【委外管理】→【委外单据】→【委外退料单】

必录字段包括——

1）表头：

单据编号：可根据编码规则自动生成单据编号，由编码规则指定是否允许编辑

单据日期：默认为系统当天日期，可编辑

供应商：支持手工录入

2）表体

物料编码：通过“选源单”返回

仓库：物料“发出”的仓库，若仓库启用了仓位管理，需要增加仓位的录入项

单位：物料单位，默认显示商品资料中的库存单位，可编辑

数量：退回数量

4、委外退料单列表

![](https://vip.kingdee.com/download/010093689052097b4817bb9d9fee042feca6.png)

用于展示所有委外退料单记录。

表头及按钮操作——

1、搜索：包含四种方式

1）快速搜索：在列表上方的搜索框中输入单据编号快速搜索；

2）常用条件搜索：支持按 ①单据日期②审核状态 筛选委外退料单；

3）方案搜索：通过配置多个查询条件进行搜索

4）漏斗搜索：将鼠标放在列表的列标题上，出现，可以实现对列表数据的过滤

2、新增：新增委外退料单，跳转到委外退料单录入界面

3、刷新：刷新列表表体内容

5、引出：将列表中的委外退料单引出到EXCEL，分为按列表引出、引出分析数据；

选择具体的委外退料单（支持多选）后，根据弹出的操作按钮，支持：

1、审核：对未审核的委外退料单进行审核

2、删除：对未审核的委外退料单进行删除

3、打印/打印模板/打印设置

4、全流程跟踪：可查看当前单据所关联的上、下游单据

5、修改单据标签：对选中的单据，修改单据标签的内容或清空单据标签内容。
