# 委外加工费（退货）可以为0吗？ 

> 产品：金蝶云·星辰 | 分类：委外管理 | 类型：常见问题, 快速入门
> 作者：Esperanza | 阅读：420 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/336188880624008192)

允许存在部分分录金额为0，委外加工费（退货）整单的总金额不能为0。
