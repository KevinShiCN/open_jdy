# 委外管理支持折扣

# **1 简介  **

## **1.1 功能介绍**

业务场景：在委外加工业务中，当加工数量达到较高水平时，供应商通常会提供相应的价格折扣。该类折扣可能根据不同产品的特性、成本结构及市场策略而有所差异，即不同产品可能适用不同的折扣比率。此外，供应商也可能依据整笔订单的合计金额制定折扣方案。

功能简介：委外管理支持折扣，即在委外加工单、委外产品入库单、委外加工费整个业务流程中均支持维护分录折扣和整单折扣。

## **1.2 适用版本**

星辰3.0以上专业版/旗舰版

# **2 操作指南**

1）当签订委外加工合同时已经确认了商品折扣或整单折扣，①可以在委外加工单中维护商品分录的折扣率或者折扣额，系统自动计算折后单价和金额；②在表头录入整单折扣率或整单折扣额，系统根据【分录中“整单折前含税加工费”/各分录“整单折前含税加工费”合计】作为权重对表头的整单折扣分配额进行分配。

![上传图片](https://vip.kingdee.com/download/0100b623c6ca7b0849079eb858b5933a9682.png)

2）若在入库环节或开票环节才确认折扣，也可在委外产品入库单/委外加工费中录入相关折扣字段， 计算规则与委外加工单一致

![上传图片](https://vip.kingdee.com/download/01006147bf1361ab4ebdb574b0869f3850d4.png)

3）影响折扣计算的参数

①系统参数-采购/销售/委外类单据打折依据

**按金额打折:**

先算出“折前不含税加工费”＝数量×原加工费单价，再以折前不含税加工费×折扣率得到“折后不含税加工费”。

此时界面显示的折后加工费单价精度与实际存储的折后加工费单价精度不一致，界面展示的折后加工费单价仅为参考。

**按单价打折:**

先算出“折后加工费单价”＝原加工费单价×折扣率，再算“折后不含税加工费”＝数量×折后加工费单价。界面显示的折后加工费单价即为计算所用的精确值。

两种方式的折后不含是加工费可能略有差异。

![上传图片](https://vip.kingdee.com/download/01009489580e429d49e98df1c614a888e5cf.png)

②单据选项：录入折后加工费单价时，加工费单价不变；录入折后单价时根据折扣率反算单价

**录入折后加工费单价时，加工费单价不变：**

录入折后加工费单价、折后含税加工费单价时，加工费单价不变，根据折前加工费单价倒算折扣率，并重算折扣额

**录入折后加工费单价时根据折扣率反算加工费单价：**

录入折后加工费单价、折后含税加工费单价时，折扣率不变，根据折扣率倒算折前加工费单价，并重算折扣额

![上传图片](https://vip.kingdee.com/download/0100dcb362dc7b7f4a1699034000d8ae786a.png)

4）分录的折扣率/折扣额支持引入，表头的整单折扣率和整单折扣额不支持引入（根据表体引入的整单折扣分配额倒算表头的整单折扣额和整单折扣率）

5）移动端委外加工单、委外产品入库单、委外产品退货单支持显示新增的折扣字段，控制与PC端一致。
