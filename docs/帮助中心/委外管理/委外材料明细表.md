# 委外材料明细表

**1、简介  **

**1.1功能介绍**

委外材料明细表是根据当前的过滤条件，展示筛选出的委外材料的发料及退料的情况，方便统计一段时间内委外材料出库的业务数据，支持查看材料出库数量、出库成本、超发数量及超发比例等。

**1.2适用版本**

金蝶云·星辰旗舰版/专业版

**1.3应用场景及价值**

委外材料明细表是根据当前的过滤条件，展示筛选出的委外材料的发料及退料的情况，方便统计一段时间内委外材料出库的业务数据，支持查看材料出库数量、出库成本、超发数量及超发比例等。

**2、****取数说明：**

（1）数据来源：委外发料单和委外退料单，委外退料单的数量及成本相关字段在报表中显示为负数；

（2）筛选条件：

①单据日期：默认当月的第一天到当前日期，展示该时间段内发生的数据

②单据类型：默认为全部，可以仅查询委外材料出库或者委外材料退回的数据

③包括未审核单据：默认不勾选，不勾选时，将仅展示已审核的数据

④其他查询条件：支持按供应商、商品编码、商品类别、仓库及退货原因查询数据
