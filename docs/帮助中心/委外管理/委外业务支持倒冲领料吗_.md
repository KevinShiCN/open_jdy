# 委外业务支持倒冲领料吗?

> 产品：金蝶云·星辰 | 分类：委外管理 | 类型：常见问题
> 作者：云社区用户55os622 | 阅读：240 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/651852309420389888)

【问题描述】
委外业务支持倒冲领料吗?

【解释说明】
支持,委外业务倒冲领料功能适用于企业因自身产能或技术等原因无法满足生产需求,需要委托供应商进行加工,在委外产品入库之后再按入库数量及标准用量倒算材料数量进行倒扣库存。
点击【委外管理】-【委外加工单】![](https://vip.kingdee.com/download/0109e52fc31a73e440a3b46114c3c2058b0e.png)

录入商品\材料信息后,将对应倒冲领料材料的【发料方式】选择为【倒冲领料】
![](https://vip.kingdee.com/download/01098196c427a051477bbdd0bac45de6c69e.png)
如果委外加工单上存在发料方式为倒冲领料的材料,则对应的委外入库单审核时,会自动生成倒冲的委外发料单。![](https://vip.kingdee.com/download/010998a15ae6a2d44b7693997c8d203c1b1b.png)
