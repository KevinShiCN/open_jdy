# 委外加工单部分领料全部入库，委外加工单未关闭，委外产品入库单显示的材料成本和关联的委外发料的成本不一致？

# **1 问题描述**

- 当期委外加工单已完成发料，当期全部入库，但是委外产品入库单上的材料成本和实际关联的委外发料成本不一致？
- 委外补料的成本没有计入到当期委外产品入库单上，是什么原因

# 2 适用版本

星辰3.0及以上专业版/旗舰版

# **3 排查思路**

查看委外加工单是否关闭：

- 若委外加工单已关闭：委外产品入库单的材料成本和关联的委外发料单的成本应一致。如不一致，通过委外发料单的源单单号字段筛选对应的委外加工单单号，汇总发料合计是否一致，正常是一致的
- 若委外加工单未关闭：委外成本算按委外加工单的标准用量进行定额核算，不包含补料成本，只有当加工单关闭时，系统才会将这部分委外补料成本一并计入产品材料成本。如后续无需继续发料生产，可手工关闭委外加工单，再进行委外成本核算即可
