# 委外加工费退货如何核销,可以和委外加工费冲抵吗?

> 产品：金蝶云·星辰 | 分类：委外管理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：108 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/724066620238431488)

【问题描述】

委外加工费退货如何核销,可以和委外加工费冲抵吗?

【解释说明】

当期入库后退回的产品需要确认扣减应付加工费,通过录入委外加工费(退货)确认应退加工费的产品、数量及费用。

委外加工费退货为应退款项,可以直接冲减委外加工费,减少应付款,可通过付款单选源单的方式抵扣应付款或者通过核销单在应付单据中冲减应付金额

【操作指引】

委外加工费退货通过选择委外产品退货单源单生成,会直接冲减供应商应付款余额

![](https://vip.kingdee.com/download/01092faa84718b7247009843bcda087c3472.png)
查询应付账款明细表,会直接冲减供应商应付账款余额

![](https://vip.kingdee.com/download/010928b59f540c344952bd09eeb61e54c6d9.png)
付款单界面,可以选择委外加工费和委外加工退费一起付款核销

![](https://vip.kingdee.com/download/01096a1921233ffe494dbb3513021ec38428.png)
委外加工费(退货)是应退款项,冲减企业对供应商的应付款项,可通过核销单在应付单据中冲减应付金额

![](https://vip.kingdee.com/download/0109001384eb1f974a468a212298cf66601c.png)
