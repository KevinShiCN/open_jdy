# 委外产品入库后发生返修的情况,如何下委外加工单?

【问题描述】

委外产品入库后发生返修的情况,如何下委外加工单?

【操作步骤】

新增委外加工单,业务类型选择【返修】即可,后续的流程与普通委外加工单一致。

返修类型的委外加工单,即领用产品本身继续进行维修或再加工,所以允许产品和材料的商品编码一致,并支持手工添加返修需要补发的材料。
