# 委外管理,如何查询某一段交货日期内的销售订单加工情况?

> 产品：金蝶云·星辰 | 分类：委外管理 | 类型：常见问题
> 作者：ylim | 阅读：83 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/621873509198919168)

【问题描述】

委外管理,如何查询某一段交货日期内的销售订单加工情况?

【解决方案】

点击【委外管理】-【销售订单全程跟踪表】

![](https://vip.kingdee.com/download/01090b84eaab86ee41d6a6420ef2d9a707a4.png)
在表头过滤条件中筛选订单日期、设置预计交货日期,设置后点击【查询】按钮

![](https://vip.kingdee.com/download/01092a566e56e461412c963d1712b8308fe8.png)可以查询到对应交货日期内的销售订单,并可查看其加工情况。

![](https://vip.kingdee.com/download/01091cb92f9bcaaa49f4a39dd447c167dc09.png)
