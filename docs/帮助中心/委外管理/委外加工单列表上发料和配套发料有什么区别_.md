# 委外加工单列表上发料和配套发料有什么区别?

【问题描述】

委外加工单列表上发料和配套发料有什么区别?

![](https://vip.kingdee.com/download/0109712cdc37a6e7433dbeb3f34e97b83725.png)

【解释说明】

**发料:**适用于不需要分批发料加工、材料库存足够一次发完,或者材料品种少计算量小,可以自行快速计算好要发料的数量,下推委外发料单后手工修改发料数。

**配套发料:**一键自动计算待发料套数、库存可配套数量,帮助仓库实现快速、准确发料。

配套发料的业务场景

①一张工单,生产数量300,由于产能等原因,需要分批生产、分批领料,如每次只生产30个,就一次领生产30个产品需要的材料,手动计算每一种材料要发多少数量,工作量大且易出错。
②一张工单需要发20种材料,生产数量20,但本次库存数只够发15套材料,需要系统自动计算15套的每一种材料应该发多少数量,希望自动按库存数量进行发料。
