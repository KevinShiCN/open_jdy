# 委外成本核算显示:本期成本合计为0,数量不为0,商品成本单价设置为0,请检查单据是否正确

> 产品：金蝶云·星辰 | 分类：委外管理 | 类型：常见问题
> 作者：向钱kan | 阅读：94 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/734215080585442560)

【问题描述】

委外成本核算显示:本期成本合计为0,数量不为0,商品成本单价设置为0,请检查单据是否正确

【解决方案】

根据提示的意思,是由于关联委外加工单的委外发料单没有核算出出库成本,需要进行出库成本核算查看委外发料单的材料的成本计算表情况,排查成本核算不出来的原因,进行调整之后,再去进行委外成本核算。

如图:

![](https://vip.kingdee.com/download/0109c8fcb84f3c6a46a18266b222fd798c94.png)

复制委外加工单去单据列表过滤出对应单据,全流程跟踪,联查到委外发料单,找到对应的物料。

![](https://vip.kingdee.com/download/0109cf17928cd74a4ed080095d9e096c853f.png)

出库成本核算,找到对应商品的成本计算表,查看系统计算情况。

![](https://vip.kingdee.com/download/01092f5cff63fc7f450398f993cde14b07a7.png)
如图,成本计算表显示,由于是其他入库单没有入库成本,导致委外发料单无法核算出出库成本。可以去维护一下其他入库单的入库成本,再去进行委外成本核算即可。

![](https://vip.kingdee.com/download/010990413a38e04740b9b162beb9ba6e4f88.png)
