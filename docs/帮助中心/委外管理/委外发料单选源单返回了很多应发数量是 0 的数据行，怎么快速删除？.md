# 委外发料单选源单返回了很多应发数量是 0 的数据行，怎么快速删除？

> 产品：金蝶云·星辰 | 分类：委外管理 | 类型：常见问题, 快速入门
> 作者：Esperanza | 阅读：383 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/336175970556723712)

通过应发数量字段的 小漏斗进行过滤，输入小于 1 的条件，过滤出
所有应发数量为 0 的数据行，再使用左上角的全选框，选中所有过滤出的数据行，点击
“批量删除”即可，然后清除掉应发数量中小漏斗的过滤条件，就可以继续对其他物料
进行发料。
