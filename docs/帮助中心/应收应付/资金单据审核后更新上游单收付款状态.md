# 资金单据审核后更新上游单收付款状态

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：进阶技巧, 功能介绍, 新特性
> 作者：鸟人 | 阅读：26 | 有用：0

**灰度发布时间：2025/08/2**

# **1 简介  **

## **1.1 功能介绍**

保存时更新适用于业务流程简单、收付款单录入错误率低，需要实时掌握出入库单收付款动态，快速响应业务的场景；

审核时更新适用于业务流程规范、需严格管控收付款准确性；收付款单需多级审核，避免录入错误等场景

## **1.2 适用版本**

星辰3.0以上标准版/专业版/旗舰版/所有版本

## **1.3 系统路径**

系统设置-基础设置-系统参数-应收应付参数

## **1.4 功能说明**

![上传图片](https://vip.kingdee.com/download/0100ef5c2054c171431e967b6ee5b8d17959.png)

**1、增加系统参数支持选择按保存或审核后更新**

保存时更新

应收应付单保存时更新上游源单收付款状态及已收付款金额；

审核时更新

应收应付单审核时更新上游源单的收付款状态及已收付款审核金额；

**2、系统参数支持切换**

支持由保存时更新切换到审核时更新；

支持由审核时更新切换到保存时更新

参数切换的前置条件为不存在未审核的资金单据（校验的单据包括：收款单，收款退款单，预收单，预收退款单，付款单，付款退款单，预付单，预付退款单，核销单、同单抵扣预收预付的采购和销售单据（标准版专业版：采购入库单、销售出库单 ，旗舰版：采购发票、销售发票；同单收付款的单据）

**3、****更新逻辑说明**

以专业版销售出库单为例，收款信息更新规则的处理逻辑如下：

1）当勾选“保存时更新”

系统处理逻辑与当前保持一致：

- 收款单 / 核销单保存时，实时更新销售出库单原有字段（“已收款金额”“未收款金额”“已核销金额”“未核销金额”）；
- 收款状态根据 “已收款金额” 与 “应收金额” 的对比结果自动判断。

2）当勾选“审核时更新”

系统新增字段并区分更新时机：

- 新增字段：表头增加 “已收款审核金额”“已收款审核金额本位币”；表体增加 “已核销审核金额”“已核销审核金额本位币”。
- 更新规则：收款单 / 核销单保存时：仅更新原有字段（“已收款金额”“未收款金额”“已核销金额”“未核销金额”）；收款单 / 核销单审核时：同步更新上述新增字段；收款状态根据 “已收款审核金额” 与 “应收金额” 的对比结果自动判断。
