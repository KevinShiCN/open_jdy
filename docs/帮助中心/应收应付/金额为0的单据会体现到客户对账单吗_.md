# 金额为0的单据会体现到客户对账单吗?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：Schatz | 阅读：182 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/645066399991180544)

【问题描述】

金额为0的单据会体现到客户对账单吗?

【解释说明】

在客户对账单中,即使金额为0的单据也会被显示出来。

![](https://vip.kingdee.com/download/0109842fec0f052d4b76bab33f32c2c45879.png)

需要显示的话,查询条件中不要勾选“不显示已完全核销的单据”。0金额单据默认就是已全部核销的,勾选这个条件就不会显示0金额单据了。

![](https://vip.kingdee.com/download/010932fddf72965641f7a5ef25e979d1eb5e.png)
