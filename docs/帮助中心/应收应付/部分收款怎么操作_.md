# 部分收款怎么操作?

部分收款怎么操作?

【解决方案】

**场景一:出库时未收款,出库后收到一部分款**

销售出库单列表上勾选对应出库单点击<收款>按钮

![](https://vip.kingdee.com/download/0109f9328eb34cf84d43a159912ccbbbb917.png)

收到部分货款收款单上表体的收款金额和下表体的本次核算金额填写一致,只核销本次收到款的金额,源单未核销完的金额下次收到款了再做收款单选择源单核销即可。

![](https://vip.kingdee.com/download/010905eac059386e415d8c7c63ed58efc600.png)

部分收款后,销售出库单列表收款状态字段会显示“部分收款”

![](https://vip.kingdee.com/download/01093d106f0e11df464ebb53f49fba1c64d2.png)
**场景二:出库时现收了一部分,剩余尾款后续再收**

做收款单时在收款单下方的金额信息中录入现收收到的款,保存审核单据,剩余未收的部分会形成应收

![](https://vip.kingdee.com/download/0109f8bd4625eef64eb7a5c822345f1194f3.png)
收到尾款时再对单据进行收款,出库单列表上找到对应单据,勾选点击【收款】按钮

![](https://vip.kingdee.com/download/0109e3ace0633dd2412ab7a2e61b7dd7b6a9.png)

或者按场景一中步骤在收款单中选源单对剩余款项进行收款,收款信息中填写收到的金额、银行信息,保存审核单据。

![](https://vip.kingdee.com/download/01090be5f9923f7f41f5a137c98a713e06b4.png)
