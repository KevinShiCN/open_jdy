# 进行预收退款时，系统提示“当前客户无可核销余额”是什么原因？

> 作者：财税云肖梅梅 | 阅读：569 | 有用：3

****

常见问题：进行预收退款时，系统提示“当前客户无可核销余额”是什么原因？ ![](https://vipstatic.obs.cn-north-4.myhuaweicloud.com/statics/emotion/define/64.gif)

![](https://vipstatic.obs.cn-north-4.myhuaweicloud.com/statics/emotion/define/93.gif)答案：预收退款要求有预收客户款项，且该预收款未被核销。提示“当前客户无可核销余额”可以通过预收款单列表，
查询是否有该客户预收款，并通过【全流程跟踪】查询对应预收款单是否有被核销。
