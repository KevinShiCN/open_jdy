# 应收账款汇总表按照业务员查询,是否支持仅按照业务员汇总,不展示客户明细?

【问题描述】

应收账款汇总表按照业务员查询,是否支持仅按照业务员汇总,不展示客户明细?

【业务场景】

希望应收账款汇总表按照业务员查询,仅按照业务员汇总就行,不需要单独展示客户明细。

若您想要了解应收账款汇总表按业务员查询问题,请点击:[查看应收账款汇总表如何显示业务员?](https://vip.kingdee.com/knowledge/581285393320775168?productLineId=35&isKnowledge=2&lang=zh-CN)

![](https://vip.kingdee.com/download/01094da5d58a38234427874fb17979bce22c.png)
【解释说明】

应收账款汇总表按业务员查询的格式为系统默认,无法设置不展示客户明细。

若希望仅按业务员汇总显示,建议通过应收账款汇总表的汇总依据自定义功能(如自定义汇总维度设置仅按业务员汇总)实现。

若您想要了解自定义汇总依据问题,请点击:[应收账款汇总表能新增汇总依据吗?](https://vip.kingdee.com/knowledge/627309559161530112?productLineId=35&isKnowledge=2&lang=zh-CN)
