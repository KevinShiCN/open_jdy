# **1 简介  **

## **1.1 功能介绍**

      分期收款功能主要应用于设备销售、大型项目服务项目、销售商品与安装服务一并销售且客户需要分期付款的场景，可解决客户因一次性支付全款压力过大而导致的订单流失或交易难以达成的问题，从而帮助用户降低销售门槛、促进大额交易成交，将应收账款分、账龄计算与预警精细化，有效管理回款现金流。

## **1.2 适用版本**

星辰3.1 以上 旗舰版

## **1.3 操作指引**

## **1.3.1 系统参数设置**

操作路径：系统设置-参数设置-应收应付参数

![上传图片](https://vip.kingdee.com/download/0100afff3ae0b14e47f4b70aed65e33922b9.png)

勾选【启用分期收款】参数，默认不勾选，勾选后可对销售发票、其他收入单进行维护分期收款计划

## **1.3.2 基础数据维护**

操作路径：基础数据-业务资料-收付款条件，打开功能，如下图：

![上传图片](https://vip.kingdee.com/download/010002ba87e62ceb4e328a647371a7a383b1.png)

点击【新增】按钮，界面跳转到新增页面，如下图：

![上传图片](https://vip.kingdee.com/download/010046754c39515c4a64a632528893d7192c.png)

需维护收款条件的名称、分期金额比例、对应的结算期限；

注意：保存后，默认为启用，已经在单据中被选择占用后的收付款条件基础数据不能删除；

## **1.3.3 维护分期收款计划**

操作路径：销售管理-销售单据-销售发票或应收应付-应收单据-其他收入

1）在维护销售发票/其他收入单时，可选择收款条件字段，下图是销售发票维护界面：

![上传图片](https://vip.kingdee.com/download/010095d548e6342a4e95bec773cc35d23569.png)

2）、选择收款条件后，单据中呈现收款计划表体，分期收款计划会根据单据的今日及单据日期分别计算，应收日期、应收比例是由收款条件携带过来、应收金额即为成交金额或本次应收金额*应收比例;

举例说明：a)维护的选择收款条件基础数据，如下表：

| 应收比例 | 结算期限 |
| --- | --- |
| 40% | 结算期限：30天 |
| 50% | 结算期限：60天 |
| 10% | 结算期限：180天 |

b)在维护销售发票页面，选择该收款条件，分期收款计划如下：

| 单据日期 | 2025-7-1 | 成交金额（单位：元） | 10000.00 |
| --- | --- | --- | --- |
| 收款计划 |  |  |  |
| 应收比例 | 计算说明 | 应收日期 | 应收金额 |
| 40% | 单据日期+30，金额为40%*10000 | 2025-7-31 | 4000.00 |
| 50% | 单据日期+60，金额为50%*10000 | 2025-8-30 | 5000.00 |
| 10% | 单据日期+180，金额为10%*10000 | 2025-12-28 | 1000.00 |

## **1.3.4 分期收款核销**

针对分期收款的销售发票、其他收入单，在销售发票或其他收入单做同单收款、收款单收款、自动核销、手工核销后，系统会自动根据已核销金额，按照分期收款的应收日期依次往后核销分期收款计划；

承1.3.3的例子：

c)收款及核销分期收款计划：新增收款单选该发票作为源单，收款金额为5000元，按应收日期核销从前往后核销，核销结果如下图：

| 单据日期 | 2025-7-1 | 成交金额（单位：元） | 10000.00 |
| --- | --- | --- | --- |
| 收款计划 |  |  |  |
| 应收比例 | 应收日期 | 应收金额 | 核销金额 |
| 40% | 2025-7-31 | 4000.00 | 4000.00 |
| 50% | 2025-8-30 | 5000.00 | 1000.00 |
| 10% | 2025-12-28 | 1000.00 |  |

## **1.3.5 应收账款预警及账龄分析统计**

对于存在分期收款计划的销售发票/其他收入单在账龄分析表与应收账款预警表中，是按收款计划的分录行计算账龄及预警；

承1.3.4 的例子：

d)应收账款预警表，查询条件及结果如下表：

| 应收账款预警表： | 截止日期 | 11月28日 |  |  |  |
| --- | --- | --- | --- | --- | --- |
| 单据日期 | 应收金额 | 已核销金额 | 未核销金额 | 结算日期 | 超期天数 |
| 7月1日 | 5000.00 | 1000.00 | 4000.00 | 8月30日 | 88 |

e)应收账款的账龄分析表，查询条件及结果如下表：

| 账龄分析表： | 截止日期 | 12月31日 | 余额类型：最终余额，账龄计算依据：截止日期 |
| --- | --- | --- | --- |
| 总金额 | 超期未收总金额 | 1-30天 | 90天以上 |
| 5000.00 | 5000.00 | 1000.00 | 4000.00 |

| 账龄分析表： | 截止日期 | 11月28日 | 余额类型：时点余额，账龄计算依据：截止日期 |
| --- | --- | --- | --- |
| 总金额 | 超期未收总金额 | 未超期金额 | 61-90天 |
| 5000.00 | 5000.00 | 1000.00 | 4000.00 |
