# 金蝶云星辰，对账单期初余额为0时不要打印空白行

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题, 进阶技巧
> 作者：鸟人 | 阅读：516 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/497471536584862464)

****

**1、问题描述**

客户对账单打印，客户不在表体，期初余额为0，会打印空白行

2、场景介绍

客户对账单打印模板，数据行值都为空或0时，不打印空白行

3、解决方案

找到一个期初余额为0的示例，进行打印，观察数值型字段是否有打印出0值的，如有，则在打印模板中选中此字段，在控件属性中勾选【零值时显示为空】参数并保存，再次打印即可发现不再打印空白行了
