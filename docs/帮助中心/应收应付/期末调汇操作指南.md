## 1 简介

1.1 功能介绍

        期末调汇功能针对企业全球化经营中的多币种核算痛点，基于汇率变动，实现应收/应付账款重估、外币资金账户价值校准、汇兑损益计量、会计凭证处理、日记账及关联报表的一体化闭环管理，确保跨境业务数据在财务系统中的真实性与时效性。

1.2 适用版本

星辰3.1以上专业版/旗舰版

1.3 系统路径及操作指南

系统路径：应收应付-期末处理-期末调汇

![](https://vip.kingdee.com/download/0109ab4c7fce90f64fc1bfddebd358b22334.png)

1.3.1 新增期末调汇

路径：应收应付-期末处理-期末调汇，打开菜单界面如下

![](https://vip.kingdee.com/download/01097a228321244a4859a171a558ecd0db38.png)

1.3.2 调汇前检查

点击调汇前检查，系统会开展调汇前检查，检查完成后，存在异常会提示，如以下图：

![](https://vip.kingdee.com/download/01090dac26bfa48c4cb58f5da9c3d132a1a2.png)

可勾选【仅查询存在未审核单据】，检查结果列表中，仅显示有存在未审核单据的数据行，点击【关联操作】列，可跳转到对应的单据列表，处理相关单据，后点击【期末调汇】，也可点击【重新检查】，系统将重新进行调汇前检查；

1.3.3 期末调汇

![](https://vip.kingdee.com/download/0109a31907d77c8847e0bc2465b5dd336082.png)

1）勾选需要调汇的类型，如应收调汇、应付调汇、账户余额调汇，默认是勾选全部类型，也可在【指定结算客户、指定结算供应商、指定账户】分别选择需要调汇的客户、供应商、账户等；

2）输入本次调汇的汇率；如需了解银行汇率情况，可点【查询中行外汇价】跳到【中国银行外汇牌价】页面查看汇率

3）计算汇兑损益，点击【开始计算】，系统根据输入的汇率进行汇兑损益计算，计算完成后，跳转到以下页面：

![](https://vip.kingdee.com/download/0109bd5517f9ede54f0fa44d6ed211bcdffa.png)

可查看到应收调汇、应付调汇、账户调汇的明细情况；

tips:

点击【自动更新本次汇率到币别基础资料】，将本次调整汇汇率的值更新到基础数据【币别】对应的汇率字段中；

1.3.4 生成调汇单

点击【生成调汇单】，生成后，页面跳转到以下页面：，点击【调汇单查询页面】跳转到调汇查单查询页面

![](https://vip.kingdee.com/download/0109c0573cf823a24222bbb36453b89ea92f.png)

1.3.5 调汇单查询

路径：应收应付-期末处理-调汇单查询

在此页面，可查看应收调汇、应付调汇、账户余额调汇单，也可对调汇单审核、对账、删除、打印等操作；

![](https://vip.kingdee.com/download/01092c9f8a9491fa4a2fb366b55d67704e03.png)

1.3.6 生成凭证

如需要针对调汇单生成凭证，需在【账务处理-智能核算中心-生成凭证-参数设置】中的【单据范围设置】中勾选调汇单（应收）、调汇单（应付）、调汇单（账户余额）；

![](https://vip.kingdee.com/download/0109aeb34cf8b87047ddb72d1991c8a5c13a.png)

即可在【账务处理-智能核算中心-生成凭证】的【应收应付-应收调汇单/应付调汇单/账户余额调汇单】中分别勾选对应调汇单生成凭证，如下图：

![](https://vip.kingdee.com/download/0109c6d8bd093f6647e2a52a9c74caf4bb7d.png)

1.3.7 日记账处理

1）参数管理：在【系统设置-基础设置-系统参数-出纳参数-出纳管理参数】中登账方式选择【单据登账与手工登账】；

2）如设置登账方式包含了单据登账，在【出纳管理-现金银行-业务单据生成日记账】中可选择账户余额调汇单生成日记账；

![](https://vip-admin.kingdee.com/download/010937d8aea595f64fc58727f91c1e8409a0.png)

3）如勾选开启【业务单据审核后自动登账】，账户余额调汇单审核通过会自动登记日记账

1.3.8 关联报表处理

1）报表【应收应付-资金分析-企业收支明细表/账户收支明细表/账户余额表]等三个报表统计账户调汇单，报表【销售管理-经营分析-企业经营利润表】将统计调汇单；

2）客户对账单：【应收应付-智能往来-客户对账单】，可勾选【包含应收调汇】，即可筛选到对应的调汇单进行对账；

3）供应商对账单：【应收应付-智能往来-供应商对账单】，可勾选【包含应付调汇】，即可筛选到对应的调汇单；
