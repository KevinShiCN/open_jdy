# 在账户余额表中显示账户状态为禁用的账户余额如何操作？

## 1 问题描述

在账户余额表中显示账户状态为禁用的账户余额如何操作？

## 2 解释说明

点击【应收应付-资金分析-账户余额表】

![](https://vip.kingdee.com/download/01093d9046509a6145a4b9ccf084793a9400.png)

在筛选条件中，勾选【显示禁用账户】，界面可查询统计并显示到账户状态为【禁用】的账户余额

![](https://vip.kingdee.com/download/0109136af5c749f44c8b8085eaf36eb0dca4.png)
