# 星辰2.0版本客户对账单打印时表头或者表尾有一行空白行是什么原因?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：BBAABABA | 阅读：619 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/458798254784938752)

【业务场景】

星辰2.0版本客户对账单打印时表头或者表尾有一行空白行是什么原因?

【解决方案】

**1、表头显示空白行**

![](https://vip.kingdee.com/download/0109acf027a022d14161b000b4e9aee3407e.png)
**原因:**系统预设模板中第一行默认取期初余额行的信息,不支持修改取消第一行空白显示

查询界面期初余额显示情况,如图:

![](https://vip.kingdee.com/download/01092a854809b5c846a3996178d7b0360e55.png)
**2、表尾显示空白行**

![](https://vip.kingdee.com/download/0109f4a83afdf2fc406881eb8ded43a535c0.png)

**原因:**这一行空白行对应的打印模板里的合计行,到模板中删除合计行即可(建议复制一个打印模板后再删除,因为合计行删了就无法找回来了)。

![](https://vip.kingdee.com/download/0109842a79cfe54f468cb9644a82b52c6c80.png)

删除合计行后打印预览效果如下:

![](https://vip.kingdee.com/download/0109ab3758c40585479a8934525eadbbd80b.png)
