# 资金转账单

 1、【业务场景 】

企业各账户之间，经常会发生资金转账业务，对于只使用进销存相关功能的情况下，可以通过【资金转账单】 进行相关业务处理

 2、【应用指南】

单击【资金管理】-【资金转账单】，录入转出转入账户信息和金额，如果有转账手续费，需手动录入转出手续 费；若有跨币别转账，差异部分会自动统计汇兑损益。如下图所示：
