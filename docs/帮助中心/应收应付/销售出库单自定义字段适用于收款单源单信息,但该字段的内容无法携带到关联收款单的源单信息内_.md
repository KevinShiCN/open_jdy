# 销售出库单自定义字段适用于收款单源单信息,但该字段的内容无法携带到关联收款单的源单信息内?

> 作者：大家好才是真的好 | 阅读：42 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/703773247657105408)

【问题描述】

销售出库单自定义字段适用于收款单源单信息,但该字段的内容无法携带到关联收款单的源单信息内?

【解释说明】

自定义字段定义在销售出库单的商品明细分录行,当核销方式为“按单核销“,下推收款仅下推表头信息到收款单,不会下推商品明细到收款单,因此无法携带。

如收款单需携带商品明细分录行自定义字段,需应收应付参数的款项核销方式为“按分录核销”才支持携带。

【操作步骤】

1、当核销方式为“按单核销”。销售出库单自定义字段“携带”设置适用于“收款单(源单信息)”,出库单自定义字段的内容无法携带到对应收款单上。

2、当核销方式为“按分录核销”。销售出库单自定义字段“携带”设置适用于“收款单(源单信息)”,出库单自定义的内容可携带到对应收款单上。
