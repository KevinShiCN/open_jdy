# 员工欠款(预支备用金)如何在系统中录入单据?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题

【问题描述】

员工欠款(预支备用金)如何在系统中录入单据?

【相关视频】

#### ![](https://vip.kingdee.com/download/0109c2d69983b90244afad6e12203dad9650.jpg)[观看视频](https://vip.kingdee.com/school/detail/704618109717526016?get_from=school-detail-id&productLineId=35&lang=zh-CN&cosmic_type=video)

【适用版本】

金蝶云星辰标准版、金蝶云星辰专业版、金蝶云星辰旗舰版

【业务场景】

部分员工可能存在预先透支工资,先向公司进行借款,后期员工还款或者用工资冲抵借款。

【解决方案】

系统没有专门员工欠款的单据。可通过以下方法变通处理:

**方案一:星辰专业版/旗舰版**,可通过其他往来单处理,详情参考:[其他往来单如何使用?](https://vip.kingdee.com/link/s/ZOcmY)

**方案二:星辰标准版/专业版/旗舰版**,均可通过其他收入单、其他支出单处理

1、先在【基础资料】-【客户】里面把员工作为客户增加一个客户资料,并且选择【保存并新增供应商】,同时生成一个关联的供应商资料

![](https://vip.kingdee.com/download/0109412d13641baf409e9fa329c61be910f6.png)
2、给员工借款时在【应收应付】模块做一个其他支出单![](https://vip.kingdee.com/download/0109399023660e3048dbad75f5b49ed6ac73.png)

增加一个员工借款的支出类别

![](https://vip.kingdee.com/download/010922d07ca7f19b41099699dd9fb9bb6a32.png)
![](https://vip.kingdee.com/download/0109c398395090e34a81a2ffbc0b23e50f54.png)

供应商选择借款的员工供应商资料,填好应付信息和付款信息将钱借给员工

![](https://vip.kingdee.com/download/01096732b10205304ba5970280258fcf2018.png)

3、借钱给员工之后需要针对该员工记录一笔其他应收账款,以便后续查账还款

在【应收应付】做一张其他收入单

![](https://vip.kingdee.com/download/0109d4cdbb01796740f5bc717b611cb4c3a4.png)
增加一个员工欠款的收入类别

![](https://vip.kingdee.com/download/01095ced96e945e14ea8bbd91ae8fea11e7c.png)

![](https://vip.kingdee.com/download/01098a36139a90704bf9917805aae1ab8f0b.png)
客户信息选择借款员工的客户名称,只填应收信息,不填收款信息,表示应收未收,后期员工还款时再收款核销

![](https://vip.kingdee.com/download/0109a7fdff831154411f9e984cade660fc6c.png)

4、实际收款的时候,做收款单,客户选借款的员工,选源单可以选到未收款的其他收入单

![](https://vip.kingdee.com/download/01090bdc28589cc8487985b8e87f3aabff9d.png)

![](https://vip.kingdee.com/download/010969db3714d96741b88a240eb60c6971ee.png)
![](https://vip.kingdee.com/download/0109a2f1d18b5f33480d8517981b7d58de07.png)
5、如果员工要求用工资抵借款时,可以先做一张其他支出单,只填应付信息工资金额,不填付款信息,表示应付员工工资还未支付

![](https://vip.kingdee.com/download/0109c758055903634df2ae51304574c19233.png)
6、然后在【应收应付】-【手动核销】模块中,按照以下步骤进行操作:

- 1)、核销类型选择【应收冲应付】
- 2)、结算客户:选择借款员工、结算供应商:同样选择该借款员工
- 3)、应收单据源单:选取挂账的其他收入单(即需收回的员工借款) 应付单据源单:选取挂账的其他支出单(即需支付给员工的工资)

- 4)、在本次核销金额栏位中,准确录入本次需要进行抵扣核销的具体金额

通过以上操作,即可完成员工借款与应付工资的相互冲抵处理

![](https://vip.kingdee.com/download/01090abe7027b8684ac28d20327a62217d37.png)
