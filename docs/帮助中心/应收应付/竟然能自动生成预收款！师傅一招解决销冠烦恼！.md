# 竟然能自动生成预收款！师傅一招解决销冠烦恼！

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：知识软文, 进阶技巧
> 作者：Simone邓 | 阅读：1157 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/323107006318127872)

最近小方销售业绩不错，上个月拿到公司的销冠，师傅小思想表扬一下小方，到工位却看到愁眉苦脸的小方，问道：怎么了，我们的销冠还有解决不了的事情吗？

       小方说，客户购买家具，购买之前都需要先付订金，每一次我都要将客户的付款给财务看，财务确认收款了，我才能通知仓库发货。业务少还好，现在业务多了，每次都要去找收款单对应的订单，繁琐又麻烦，头都大了！

        小思笑着对小方说：系统有一个好用的功能，在系统参数——进销存参数中，勾选“销售订单审核时，自动生成预收款”，设置参数后，录入销售订单，在订单中填写预收订金的收款账户，收款方式，订金金额，在审核销售订单后，填写的订金会自动生成一张预收款单，将这张预收款单审核，仓库在销售订单列表中，会看到订金状态为“已收”，就可安排发货了。

        师傅，我的好师傅，快教教我！

        谁让我是你师傅，教你一次，说着在小方电脑演示起来：
 首先，在【系统参数】—【进销存参数中】勾选“销售订单审核时，自动生成预收款”；设置参数：

![image.png](https://vip.kingdee.com/download/010085083b0680394a51bdd4b2b9db8ac5f2.png)

![image.png](https://vip.kingdee.com/download/0100e201d5e7bc274132846cb8c7c435afa9.png)

业务员录入销售订单，在订单中填写【预收订金】的收款账户，收款方式，订金金额。

![image.png](https://vip.kingdee.com/download/01001fa05f3ec1234c15a6000409cabab2fd.png)

在审核【销售订单】后，填写的订金会自动生成一张【预收款单】

![image.png](https://vip.kingdee.com/download/0100f830c2408fdb4c8986f59cc059896521.png)

财务将这张预收款单审核，仓库在销售订单列表中，会看到【订金状态】为“已收”，即可安排发货。

![image.png](https://vip.kingdee.com/download/010083d4d5f6e41e4219a8fe38e31021bfcc.png)

![image.png](https://vip.kingdee.com/download/01008deb5f8053b742cfbddff4f911ad2b83.png)

需要注意的是如果订单列表不显示【本次订金状态】，点击列设置，将本次订金状态字段放出即可

![image.png](https://vip.kingdee.com/download/010043dd093ebb26493eaab2110b72956d2f.png)

小方激动的说道：师傅，不愧是我师傅，真棒！
