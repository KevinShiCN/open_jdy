# 如何修改对账函模版?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：ylim | 阅读：482 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/621511548782866176)

【问题描述】

如何修改对账函模版?

【解释说明】

支持修改5个用户字段、删除或添加单据类型、对账函模板名称、对账类型;修改后不影响已生成的往来对账函

【解决方案】

点击【应收应付】-【对账函模版】

![](https://vip.kingdee.com/download/01094bde8ab0ffe44e9e9798034d59664c9a.png)
点击要修改的对账函模板编码

![](https://vip.kingdee.com/download/010958bb5c2295d54997b22ed030f7acb630.png)
**修改用户字段**

点击要修改字段的按钮,选择后确定

![](https://vip.kingdee.com/download/01092821273ea3194313bf2434b3c5e59404.png)
若要修改标题行,点击标题行的按钮,修改编辑后确定

![](https://vip.kingdee.com/download/0109beeceb96f66749e78b64a3feee364842.png)
注意,若修改标题行取值来源,修改后对应列已设置的取值将被清空,需重新设置

如图,标题行取值来源从原来的单据头改成了单据体,对应列下设置的单据取值被清空了。
![](https://vip.kingdee.com/download/0109a27764032c7b41b8a7fa969cb5f71d4c.png)
**删除/添加单据类型**

鼠标放行上会出现+和- 需要增加行就点击+,需要减少行就点击-,

![](https://vip.kingdee.com/download/01099e0e2a0ca9904ce2b1a745aa3e03baee.png)
同一个单据类型只能存在一行,选择重复了系统就会提示:此单据类型已存在,请重新选择!

![](https://vip.kingdee.com/download/01097a6efa14799e4b748d8880c9cde6630f.png)
**修改对账函模板名称、对账类型**(若修改对账类型,用户字段如需自定义设置则需重新设置)

![](https://vip.kingdee.com/download/0109a4cde250566f46ae9f62d0a35b197580.png)
修改后保存,并回到对账函模版列表上,勾选已修改设置好的对账函模版,点击【生成对账函】

![](https://vip.kingdee.com/download/0109d59dfc7dc425446683bade7b2bde77a2.png)
