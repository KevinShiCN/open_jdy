# 采购订单/其他支出单点击生成付款申请单时报错:请检查单据总可申请金额是否大于0?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：Schatz | 阅读：25 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/784233764418807808)

【问题描述】

采购订单/其他支出单点击生成付款申请单时报错:请检查单据总可申请金额是否大于0?

![](https://vip.kingdee.com/download/0109dac8b059175a486e9fc27a6115128595.png)

【适用版本】

金蝶云星辰3.1旗舰版

【解释说明】

采购订单单据总可申请金额=采购订单成交金额-累计预付-已申请金额。

其他支出单可申请金额=未付款金额(应付金额-已付款金额)。

如果没有剩余的可申请金额,则无法继续生成付款申请单。
