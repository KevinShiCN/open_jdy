# 关于几种常见的核销类型的名词解释

1、【核销类型】
 往来核销就是指应收账款的核销和应付账款的核销，常见的核销类型包括：预收冲应收、预付冲应付、应收冲应付、应收转应收、应付转应付、退货冲购货、退货冲销货。
2、【名词解释】

1）预收冲应收、预付冲应付
用于处理先预收（付）了款项，未使用“定向抵扣”相关功能时，在对应往来交易实际发生后，通过核销单进
行往来核销处理。

2）应收冲应付：用于处理往来单位之间相互有业务往来时，直接将应收与应付款项进行抵销。

3）应收转应收：用于两个客户之间的应收账款转销

4）应付转应付：用于两个供应商之间的应付账款转销

5）退货冲销货：用于同一客户，发生了销售退货业务后，将退货与原来的出库往来进行冲销

6）退货冲购货：用于同一供应商，发生了采购退货业务后，将退货与原来的入库往来进行冲销
