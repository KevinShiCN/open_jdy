# 预收预付业务场景概述

问题描述：

预收预付业务场景概述

解决方案：

预收：就是在交易没有发生之前，预先收了客户的款项。（通俗讲就是押金）

预付：就是在交易没有发生之前，预先支付给供应商的钱。（通俗讲就是订金）
