# 星辰2.0收款单选择源单窗口界面“同时携带销售出库单关联的销售退货单到单据界面”按钮如何使用?

> 作者：海鸥乔纳森 | 阅读：300 | 有用：0

【问题描述】

收款单选择源单窗口界面“同时携带销售出库单关联的销售退货单到单据界面”按钮如何使用?

![](https://vip.kingdee.com/download/0109dd19d4e79e5b4f37a7fe7273a7d8fbd1.png)

【解释说明】

按钮开启后,选择对应销售单,会将关联该销售单对应的退货单一起选择到收款单源单界面进行核销,避免出现销售退货单漏选的情况

![](https://vip.kingdee.com/download/0109f04dbd831b40434f87c5007f5e7210ab.png)

【温馨提示】

1、启用该按钮需要款项核销方式为【按单据核销】,如果参数选择【按分录核销】和【按余额核销】,收款单选择源单界面不会出现这个按钮选项。

2、如果销售退货单是单独手工新增,不是通过源单生成,也无法将销售退货单携带到源单信息中
