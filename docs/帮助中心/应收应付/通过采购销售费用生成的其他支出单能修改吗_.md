# 通过采购销售费用生成的其他支出单能修改吗?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：shaoling | 阅读：109 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/718268486262290944)

【问题描述】

通过采购销售费用生成的其他支出单能修改吗?

【解释说明】

通过采购销售费用生成的其他支出单,单据日期、结算日期及涉及付款信息的字段可以修改;而其他字段(如供应商、部门、应付信息里的字段)均根据采购或销售单录入的数据自动带出,带出后无法修改。

若需修改,需先删除其他支出单,再通过采购入库单 / 销售出库单重新录入修改(例如:删除其他支出单后,在采购入库单中修改费用部门等信息,重新生成其他支出单)

![](https://vip.kingdee.com/download/0109ea2ef61e6d8b42b4b61426510fd97e99.png)

单据编号不允许直接手工修改,若单据编号的规则是以单据日期为依据,且编码依据变化时重新编码,则修改单据日期时单据编号会跟随变动
