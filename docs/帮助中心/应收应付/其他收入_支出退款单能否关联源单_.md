# 其他收入/支出退款单能否关联源单?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：shaoling | 阅读：95 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/724066620557198592)

【问题描述】

其他收入/支出退款单能否关联源单?

【解释说明】

其他收入 / 支出退款单暂不支持关联源单处理。

该类单据与其他收入 / 支出单相互独立,无法通过其他收入 / 支出单下推生成,即二者无直接关联关系。其他收入 / 支出退款需手工录入应退信息及退款信息
