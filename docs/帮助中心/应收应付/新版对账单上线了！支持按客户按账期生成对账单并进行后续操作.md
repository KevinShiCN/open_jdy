# 新版对账单上线了！支持按客户按账期生成对账单并进行后续操作

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：快速入门, 功能介绍, 新特性
> 作者：王二的低级趣味 | 阅读：1738 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/668155296488905984)

### 1.1 功能简介

我们对客户对账单、供应商对账单做了流程驱动型的优化，以客户对账单为例，支持按客户按账期等条件筛选出需要对账的单据生成对账单并保存在对账单列表里，支持对生成的对账单进行整单对账确认、录备注和结算日期等，并发起收款（暂未上线）等操作；增加历史对账单列表，能查往期对账单。从而能够实现对账流程线上化，方便在线上进行全流程的对账、收款等操作，减少线下对账的工作量，提升对账效率，促进应收账款回收。

### 1.2 适用版本

星辰3.1以上标准版/专业版/旗舰版

### 1.3 系统路径及操作说明
1.3.1 菜单调整说明：

（1）【待收款清单】、【合并收款记录】菜单移到【应收单据】下；

（2）【客户对账单】、【供应商对账单】、【客商匹配对账单】菜单移到【智能往来】下；

（3）【往来对账函】改名为【历史对账单】，放在对账函模板之上

1.3.2 客户对账单新功能说明

菜单路径：[应收应付](https://www.kingdee.com/product/small?utm_source=shequ)-智能往来-客户对账单。

![](https://vip.kingdee.com/download/01093f2403cf47e74c12929ea8eaa8ecbbe4.png)

![](https://vip.kingdee.com/download/0109c7449e060bf04df68b7fa6fc47089721.png)

进入客户对账单新界面，能看到新手引导部分，可参考引导进行使用：

第一步：选择对账类型，按应收对账还是按应收及收款对账，切换下方页签即可选择。

其中，如仅关注应收账款，包括所有未结清的应收单据以及这些单据的明细，选【应收对账】（对应原对账单的非简易对账单）；

如同时关注应收账款和收款情况，除了包含应收单据的明细外，还包括收款单据的明细，需要选【应收及收款对账】（对应原简易对账单）。

第二步：选择日期范围和客户等筛选条件，点击“查询”，筛选出对应的单据。

第三步：选择本次需要对账的单据，点击【生成对账单】按钮，进行对账确认，可选择对账期间、对账日期范围，点击【确认生成】按钮后，即可生成对账单。单个客户生成对账单后会直接跳转到对账单详情页，多个客户批量生成对账单会给出查看历史对账单按钮，点击即可跳转到历史对账单列表查看。注意，只有已审核并且未生成过对账单的单据才能勾选生成对账单。

![](https://vip.kingdee.com/download/01097b8bf5336ef4476eae97592172222885.png)

![](https://vip.kingdee.com/download/01092368a522dd594c6583f0797a28236454.png)

第四步：查看历史对账单并进行操作，支持在对账单详情页进行对账确认、收款等操作，也支持录入备注、结算日期等，这些操作也支持在列表进行。

1、备注：对账单随时可以录入/更新备注，编辑完之后需要点击保存才能更新；

2、对账确认：跟客户对账完成后，可点击【对账确认】，确认则该对账单下的所有单据会更新为已对账，取消确认则单据变为确认之前的状态（之前是已对账就是已对账，之前是未对账就是未对账）；

![](https://vip.kingdee.com/download/010917fcff2bed7b44e3b6c52daa7e3bfd7a.png)

3、对账单对账确认后可以录入结算日期，并选择是否要更新对账单源单的结算日期，在对账单详情页录结算日期后需要保存才会更新。

![](https://vip.kingdee.com/download/010945797a0e9266481d8f89eda60a459eb0.png)

![](https://vip.kingdee.com/download/01096616f3b2c56047c68a79a3ceb9e1747a.png)

4、 历史对账单

菜单路径：应收应付-智能往来-历史对账单，如下图，包含了客户对账单历史、供应商对账单历史和对账函历史，切换页签即可分别查看。以客户对账单列表为例，支持对历史对账单进行对账确认、录入结算日期、删除等，后续还会支持发起收款。

![](https://vip.kingdee.com/download/010900e2af6186f34519903d45b8421ff319.png)

其他功能说明：

1、客户对账单支持对查询出的单据直接标记对账/取消对账

![](https://vip.kingdee.com/download/0109eadd14fd9873457aa0fc8cab88357094.png)

2、支持编辑已生成、未对账确认的对账单，可以增删单据，在对账单里重新查询，勾选需要增加的单据点击【添加到历史对账单】，选择添加到哪张对账单即可；从对账单里删除单据需要在对账单详情页操作，勾选需要删除的源单点击【移除】按钮即可。

![](https://vip.kingdee.com/download/0109ca07505e9d2747e7887696d0c2c6564f.png)

![](https://vip.kingdee.com/download/0109cc32c83470b74aefa7616b1f9615602d.png)
