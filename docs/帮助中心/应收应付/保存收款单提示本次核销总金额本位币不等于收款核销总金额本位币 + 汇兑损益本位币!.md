# 保存收款单提示本次核销总金额本位币不等于收款核销总金额本位币 + 汇兑损益本位币!

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：shaoling | 阅读：471 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/619699120600656640)

【问题描述】

保存收款单提示本次核销总金额本位币不等于收款核销总金额本位币 + 汇兑损益本位币!

【故障原因】

当收款单是跨币别核销时,收款信息【收款金额本位币】+【汇兑损益本位币】跟源单信息的【本次核销金额本位币】不一致时就会提示:**本次核销总金额本位币不等于收款核销总金额本位币 + 汇兑损益本位币!**

![](https://vip.kingdee.com/download/010968ddcce37b0b433ab8caef8b6b8e3794.png)

【解决方案】

点击【汇兑损益本位币】旁边的小钱袋![](https://vip.kingdee.com/download/010997027330208c4d84981afe4bbad64376.png)自动平衡,把总的【收款金额本位币】+【汇兑损益本位币】跟【本次核销金额本位币】调整为一致再保存即可
