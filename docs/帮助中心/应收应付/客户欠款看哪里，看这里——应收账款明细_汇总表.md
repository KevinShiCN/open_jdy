# 客户欠款看哪里，看这里——应收账款明细/汇总表

## 客户欠款看哪里，看这里

               ——应收账款明细/汇总表

![上传图片](https://vip.kingdee.com/download/01006133e58b7a55416f96dee32265b836ec.png)

      小蓝是公司的业务员，月底了，他想看下手上几个大客户的实际欠款情况。随即他点开了应收账款明细表，不看不知道，一看吓一跳！这报表也忒长了，哪个字段才表示欠款呢？然后他赶紧去请教财务小黄，小黄不紧不慢，慢条斯理的跟他讲起了如何看客户欠款。小黄说：小蓝你看，报表一共有三大块：预收，销售应收，其他应收。客户有没有欠款也就看这三大块了。

      首先，我们来看下甲客户的预收情况：这里的“预收余额=期初预收余额+预收金额-预收核销金额”，带入数值看“1493=500+1020-27” ，验证没有问题对吧。

![](https://vip.kingdee.com/download/010036f8af0be8a0489f99d786a7b43442cb.png)

      其次，我们来看下甲客户的销售应收情况：这里的“销售应收余额=期初销售应收余额+销售应收金额-销售款核销金额 ”，带入数值看“110=100+80-70”，验证也没有问题。

![](https://vip.kingdee.com/download/0100278f281321d74e2fa71246e0c9d76729.png)

      再次，同理甲客户的其他应收情况：这里的“其他应收余额=期初其他应收余额+其他应收金额-其他应收核销金额”，带入数值看“50=50+0-0” ，也是对的。

![](https://vip.kingdee.com/download/0100218fe3c3bef5474a838dd9863dbcf5ea.png)

      最后，你最关心的客户欠款就是看这个“余额”，这里的“余额=销售应收余额+其他应收余额-预收余额 ”，带入数值看：“ -1333=110+50-1493” ，也没有问题对吧。

![](https://vip.kingdee.com/download/0100e4145598e85f461aa8d3f6abd386bb85.png)

      或者余额也可以这样看：“余额=期初余额+本期应收发生额-本期预收发生额 =期初余额+（应收合计金额-应收核销金额）-（预收金额-预收核销金额）”带入数值看“-1333=-350+（80-70）-（1020-27）”，也是可以的。

![](https://vip.kingdee.com/download/0100b9cc784227d749e49488f1ed1878b488.png)

      小蓝看之后惊呆了，原来数字之间这么奇妙，加加减减以后，欠款多少就一目了然了。系统还是很强大的！谢谢你小黄，又教了我个大招。

      补充1：应收账款汇总表的“余额”和应收账款明细表“余额”的计算是相同的。

![](https://vip.kingdee.com/download/010028d6962af5a24f5abbe40c9a92e2dd45.png)

期初余额=期初销售应收+期初其他应收-期初预收。

余额=销售应收余额+其他应收余额-预收余额

=期初余额+本期应收发生额-本期预收发生额

=期初余额+（应收合计金额-应收核销金额）-（预收金额-预收核销金额）

      补充2：本期实收金额含：收款单、收款退款单、预收单、预收退款单、其他收入单，其他收入退款单，核销单（应收转应收中的转出金额）这几类单据上的金额，不含销售单的同单收款的金额。
