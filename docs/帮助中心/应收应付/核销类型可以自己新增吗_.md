# 核销类型可以自己新增吗?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题

【问题描述】

核销类型可以自己新增吗?

【解释说明】

核销单的核销类型是固定的,不支持手动新增。

当前有,预收冲应收,预付冲应付,应收冲应付,应收转应收,应付转应付,退货冲销货,退货冲购货。
