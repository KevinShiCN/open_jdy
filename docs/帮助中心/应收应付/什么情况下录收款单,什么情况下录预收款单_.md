# 什么情况下录收款单,什么情况下录预收款单?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：岸边把酒 | 阅读：492 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/584184614072303360)

【问题描述】

系统里有收款单和预收款单两种不同类型的收款单据,应该在什么情况下录入收款单,在什么情况下录入预收款单呢?

【解释说明】

1、在先货后款的情况,货已经发出后再收到的货款,在收到款项时,做收款单。同时需要选择相对应的源单进行核销。

![](https://vip.kingdee.com/download/01091eafa54ec65243958a681771285251ab.png)
![](https://vip.kingdee.com/download/01099e037e3487c743b68f2ca8eb383332ac.png)

![](https://vip.kingdee.com/download/010917492c0c69744e258f22994df9767b66.png)

![](https://vip.kingdee.com/download/0109ce4166b7629e486b9dd242d1591683b7.png)

2、在客户先付款,暂未发货的情况下做预收款单 。预收款单可以直接录入,也可以通过销售订单录入定金,然后下推生成预收款单。
