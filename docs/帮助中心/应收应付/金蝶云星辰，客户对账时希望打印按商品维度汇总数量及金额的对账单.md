# 金蝶云星辰，客户对账时希望打印按商品维度汇总数量及金额的对账单

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题, 进阶技巧
> 作者：鸟人 | 阅读：370 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/497443130510027520)

****

**1、问题描述**

客户对账单打印希望按商品维度汇总商品数量及价税合计

2、场景介绍

客户对账单打印模板配置商品维度汇总表格

3、解决方案

1）第一步，打开客户对账单打印模板，如使用的是预置模板，需复制模板

![](https://vip.kingdee.com/download/0109fdd689bcbbe340279be800d677627696.png)

2）第二步，新增数据表格，并选择数据源

![](https://vip.kingdee.com/download/0109e67f607c02564b1dabe2f462fecceff9.png)

3）第三步，选中数据行，选择合并依据字段为商品编码或商品id，合并汇总字段为商品基本数量及价税合计；

合并依据字段表示用哪个字段进行合并，合并汇总字段表示哪些字段需要进行汇总计算

![](https://vip.kingdee.com/download/010931bda460f44e4fbf8a151cba58a7cd70.png)
