# 账户余额表如何查询统计到未审核单据？

## 1 问题描述

账户余额表如何查询统计到未审核单据？

## 2 解释说明

点击【应收应付-资金分析-账户余额表】

![](https://vip.kingdee.com/download/010973caa7dbaeda48b5aa84c47405930e44.png)

在筛选条件中，勾选【包含未审核】，即可统计查询包含未审核单据的数据。

![](https://vip.kingdee.com/download/0109e731119933464df688b16ce09806a574.png)
