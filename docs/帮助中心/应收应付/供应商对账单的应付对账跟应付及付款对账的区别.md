# 供应商对账单的应付对账跟应付及付款对账的区别

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：shaoling | 阅读：194 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/716096772187421184)

【问题描述】

供应商对账单的应付对账跟应付及付款对账的区别

【解释说明】

**应付对账:**仅关注应付账款,包括所有未结清的应付单据以及这些单据的明细。

常用于日常的应付款项梳理,或向供应商发送对账单时,仅需确认企业的欠款金额。

**应付及付款对账:**同时关注应付账款和付款情况,除了包含应付单据的明细外,还包括付款单据的明细。

常用于核对应付账款的核销状态,例如检查某笔应付款是否已付款、是否存在多付 / 少付、是否有款项未匹配到对应应付单据等。
