# 总账与业务对账中，总账和账户收支对账时，账户收支没有包含对应日记账的账户收支数据，导致对账不平怎么办？

> 作者：王二的低级趣味 | 阅读：559 | 有用：2

## 1 问题描述

客户反馈，总账与业务对账中，总账和账户收支对账时，账户收支没有包含对应日记账的账户收支数据，导致对账不平。

## 2 **原因解析**

【客户业务场景*】银行账户会发工资，会付一些公司的费用，还会付税款等，这些都是没有在账户收支明细表中体现的。工资是直接从总账发的，在出纳管理里面，日记账里面会做工资的支出，然后再转到总账里面工资的凭证。没有通过应收应付的单据进行体现，所以总账和账户收支对账的时候，没有把使用日记账登记账户的收支这部分算进来，导致对账不平。

## 3 解决方案

把税费、工资等直接通过日记账生成凭证的支出做成其他支出单，即可在应收应付的账户收支明细里体现出来，就能对得上了。
