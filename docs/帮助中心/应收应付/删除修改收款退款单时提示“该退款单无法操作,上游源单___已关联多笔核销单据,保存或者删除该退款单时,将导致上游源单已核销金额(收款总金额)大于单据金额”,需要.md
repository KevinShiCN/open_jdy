# 删除修改收款退款单时提示“该退款单无法操作,上游源单***已关联多笔核销单据,保存或者删除该退款单时,将导致上游源单已核销金额(收款总金额)大于单据金额”,需要如何处理

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：755 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/398279474787215872)

【故障现象】

收款退款单修改删除时提示该退款单无法操作,上游源单***已关联多笔核销单据,保存或者删除该退款单时,将导致上游源单已核销金额(收款总金额)大于单据金额。

![](https://vip.kingdee.com/download/01093e19b38db9df46529f60596428056876.png)

【解释说明】

由于原收款单退款后,又重新对同一销售出库单进行了收款,此时若删除原收款单对应的退款单,会导致该销售出库单的累计收款金额超过单据金额。因此,系统会提示无法删除收款单退款单或修改退款金额。

【解决方案】

1、修改再次收款单的收款金额,使原收款单和再次收款的合计等于或者小于单据应收金额

2、把再次收款的收款单删除,再对原收款单的退款单进行修改删除

可通过全流程跟踪查询销售出库单关联的收款单据
