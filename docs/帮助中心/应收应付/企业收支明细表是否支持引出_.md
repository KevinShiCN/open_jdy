# 企业收支明细表是否支持引出?

> 作者：云社区用户55os622 | 阅读：151 | 有用：0

【问题描述】
企业收支明细表是否支持引出?

【解释说明】

支持引出,依次点击【应收应付】-【企业收支明细表】,进入查询界面![](https://vip.kingdee.com/download/01096442c53c91644a1dab8fa406cae53dc5.png)

点击右上角【引出】,即可将数据引出保存至本地
