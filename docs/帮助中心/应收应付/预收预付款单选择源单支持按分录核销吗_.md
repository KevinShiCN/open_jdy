# 预收预付款单选择源单支持按分录核销吗?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：shaoling | 阅读：104 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/721537505207643648)

【问题描述】

预收预付款单选择源单支持按分录核销吗?

【解释说明】

预收预付款单选择源单时不支持按分录核销,目前仅支持按单据整体选择核销。

如图所示,参数已设置按分录核销

![](https://vip.kingdee.com/download/010953161e989ea24aaf8cad446d889e4fc3.png)

选源单界面不支持开启显示商品分录的字段信息

![](https://vip.kingdee.com/download/0109d16e60608e6647b0a92ccee7f13fc890.png)
且源单列字段上没有商品相关字段

![](https://vip.kingdee.com/download/010912360487dfff49a682289a8d31904d54.png)

若订单的预收 / 预付金额与实际收付款金额不一致,可在预收 / 预付款单选源单后修改本次预收 / 预付总金额操作

![](https://vip.kingdee.com/download/0109fece6b1d9d4e4b9da0e9eb4e3de75c38.png)
