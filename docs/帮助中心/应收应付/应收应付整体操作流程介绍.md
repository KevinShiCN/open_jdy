# 应收应付整体操作流程介绍

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：快速入门, 整体介绍, 精华汇总
> 作者：王二的低级趣味 | 阅读：1466 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/660200740417484032)

金蝶云星辰致力于打造业[财](https://www.kingdee.com/product/stellar_finance.html?utm_source=shequ)一体的应用系统，通过应收应付模块去处理企业的账户管理和对内、对外的往来结算业务，涵盖往来资金的全面追踪，包括预收预付货款登记、应收及应付账款的管理，信用管理等。通过高效录入相关单据，系统自动生成详尽的[应收应付](https://www.kingdee.com/product/small?utm_source=shequ)明细表、汇总表、往来对账单、账龄分析表以及资金分析报表等关键报告，确保资金流动的透明度和准确性。

![](https://vip.kingdee.com/download/010960ae0426d46043e9a92caee1ebdf77c5.png)

| 步骤 | 具体说明 | 学习链接 |
| --- | --- | --- |
| 1.应收单据 |  |    —— |
| 1.1收款单 | 用于记录公司收到的款项 |   点击学习 |
| 1.2收款退款 | 用于记录由于某些原因，卖方需要将已经收到的款项退还给买方 |   点击学习 |
| 1.3预收款 | 在商品或服务实际交付之前，客户预先支付给卖方的款项 |   点击学习 |
| 1.4预收退款 | 用于记录由于某些原因，卖方需要将客户预先支付给卖方的款项退还给买方 |   点击学习 |
| 1.5其他收入 | 用于记录正常经营活动之外获得的收入，这些收入不直接来源于企业的主要业务 |   点击学习 |
| 1.6其他收入退款 | 用于记录由于某些原因，需要退还非主营业务取得的这部分收入 |   点击学习 |
| 2.应付单据 |  |   —— |
| 2.1付款单 | 用于记录企业对外支付款项的详细信息 |   点击学习 |
| 2.2付款退款 | 在交易过程中，由于某些原因，买方要求将已经支付的款项退还 |   点击学习 |
| 2.3预付款 | 企业在购买商品或接受服务前，向供应商或服务提供者支付的款项 |   点击学习 |
| 2.4预付退款 | 企业在交易过程中预先支付了款项，但后来由于合同取消、服务未完成、商品退回或其他原因需要将部分或全部款项退还给付款方 |   点击学习  |
| 2.5其他支出 | 企业主营业务成本和运营费用之外的其他费用，可能包括罚款、捐赠、非常规损失等。 |   点击学习 |
| 2.6其他支出退款 | 企业在支付了某些非经常性或非主营业务相关的费用后，由于特定原因收到的退款 |   点击学习 |
| 2.7资金转账 | 企业内部账户之间进行的资金流动，包括但不限于银行账户之间的转账、电子支付等 |   点击学习 |
| 3.智能往来 |  |   —— |
| 3.1自动核销方案 | 方便用户快速定位未核销单据;可提供定时方案,方便用户快速核销实际工作场景中的常见单据;包含更具体的核销规则,以满足用户多样化的核销需求。 |   点击学习 |
| 3.2手动核销 | 用于处理企业之间相互有业务往来时,直接将应收与应付款项进行抵销 |   点击学习 |
| 3.3待收款清单 | 待收款清单可以多个客户,多个不同类型的单据一起收款,达到批量收款的目的   |   点击学习 |
| 3.4合并收款记录 | 合并收款记录是指通过待收款清单增加收款,不同客户一起合并收款生成收款单的记录,在合并收款记录中能够看到哪些收款单是通过合并收款生成的,共收了多少钱,币别是什么 |   点击学习 |
| 3.5对账函模版 | 往来对账涵处理的业务场景与客户供应商对账单一致,但支持根据业务需要进行自段的设置与编辑,系统提供预设模板与字段 |   点击学习 |
| 3.6往来对账函 | 往来对账涵处理的业务场景与客户供应商对账单一致,但支持根据业务需要进行自段的设置与编辑,系统提供预设模板与字段 |   点击学习 |
| 3.7现金银行流水 | 企业日常经营发生资金收付业务，出纳需将收付款信息登记生成现金银行日记账流水 |   点击学习 |
| 3.8智能收款 | 银行交易流水和应收单据自动匹配,快速完成收款   |   点击学习 |
| 3.9智能付款 | 银行交易流水和应付单据自动匹配,快速完成付款   |   点击学习   |
| 4.往来报表 |  |   —— |
| 4.1应收账款明细表 | 应收账款明细表用于查询应收客户款项明细发生额及余额   |   点击学习 |
| 4.2应付账款明细表 |  应付账款明细表用于查询应付供应商款项明细发生额及余额   |   点击学习 |
| 4.3应收账款汇总表 |  应收账款汇总表用于按不同的汇总维度,查询应收客户款项及余额   |    点击学习 |
| 4.4应付账款汇总表 | 应付账款汇总表用于按不同的汇总维度,查询应付供应商款项及余额 |   点击学习 |
| 4.5应收账款预警表 | 用于查询客户供应商应收应付款超期及信用超额情况 适用于专业版、旗舰版 |   点击学习 |
| 4.6应付账款预警表 | 用于查询客户供应商应收应付款超期及信用超额情况适用于专业版、旗舰版 |   点击学习 |
| 4.7客户对账单 | 客户对账单是查询客户往来余额和进行对账的重要工具   |   点击学习  |
| 4.8供应商对账单 | 供应商对账单用于查询供应商往来余额，提供对账依据   |   点击学习 |
| 4.9客商匹配对账单 | 客商匹配对账单用于统计既是客户又是供应商的单位往来余额,便于用户对账 |   点击学习 |
| 5.资金分析 |  |   —— |
| 5.1企业收支明细表 | 企业收支明细表用于查询账套所有收支业务明细,除应收应付相关单据数据外,还包括采购、销售同单收付款明细 |   点击学习 |
| 5.2信用管理 | 直观地展示信用管控结果,并且能够一站式完成信用管理的设置与启用 适用于专业版、旗舰版 |   点击学习 |
| 5.3账龄分析表 | 用于查询企业往来业务账龄,可根据企业管理需要设置账龄分组 适用于专业版、旗舰版 |   点击学习 |
| 5.4其他收支明细表 | 应收应付中的其他收支明细表统计了收入和支出类别的收支明细 |   点击学习 |
| 5.5账户收支明细表 | 查询企业每个账户的资金流入流出明细 |   点击学习 |
| 5.6账户余额表 | 用于查询进销存资金收付款单对应的账户收支合计与余额 |   点击学习 |
