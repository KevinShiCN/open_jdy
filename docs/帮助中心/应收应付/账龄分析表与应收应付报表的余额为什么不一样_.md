# 账龄分析表与应收应付报表的余额为什么不一样?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：嘿唬哈 | 阅读：124 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/733127217278108416)

【问题描述】

账龄分析表与应收应付报表的余额为什么不一样?

【概述】

1、账龄分析表中应收 / 应付余额的计算核心规则如下:

以**业务发生时间(即单据日期)** 为统计基准,对所有未结清的应收 / 应付账款余额进行分类汇总;

余额统计**不涉及收付款核销的时间范围限制**—— 无论收付款核销操作发生在何时,只要对应账款已完成核销,相关数据将自动排除在账龄分析表的统计范围之外,仅保留未核销的账款余额。

2、预收/预付账款、定金、订金都**不计入**账龄分析表

3、部分收款/付款的单据,账龄分析表会对未收/未付部分的数据进行统计

**故:可能存在预收预付的差额**

#
**账龄分析表:**不统计预付单据金额;其应收 / 应付余额统计不涉及收付款核销时间范围 —— 无论核销操作发生在何时,已完成收付款核销的账款数据均不会纳入统计,仅保留未核销余额。
**应付账款明细表:**会统计预付单据金额,但仅包含付款单日期在明细表过滤时间范围内的相关数据;若付款单日期超出该过滤时间范围,则对应数据不会被统计。
综上,因两者在 “预付单据统计规则” 和 “付款单日期过滤要求” 上存在差异,可能导致应付账款明细表与账龄分析表的统计结果不一致。
