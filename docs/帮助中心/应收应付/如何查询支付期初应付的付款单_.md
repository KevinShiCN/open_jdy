# 如何查询支付期初应付的付款单?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：Mosh555555 | 阅读：83 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/713921167006167296)

【问题描述】

如何查询支付期初应付的付款单?

【操作步骤】

方法一:

【应收应付】>【付款单】,点击“查询”进入付款单列表

![](https://vip.kingdee.com/download/010987fb5cdaf92e410d831677a12a072aa2.png)
进入付款单列表后,<展开过滤>,选择查询范围为“基本信息/源单信息”,可以查看付款单的源单类型

![](https://vip.kingdee.com/download/010964b5c7ed953d4324a8d1409c1f5fe143.png)
过滤源单类型,显示“供应商应付初始数据”的付款单,就是支付期初应付的付款单。

![](https://vip.kingdee.com/download/0109522ac30d2517406594ac6e954a344b7c.png)
方法二:

付款单列表全选单据,“按引入模版引出”

![](https://vip.kingdee.com/download/0109b64db16268ba4f44b06bfafa84c38553.png)
选择系统预置模版,<确认>,下载文件
![](https://vip.kingdee.com/download/0109698b322c38d2448790d778ffb574b254.png)
在下载的Excel文件中过滤筛选源单信息

![](https://vip.kingdee.com/download/010951d0f741b04345f7890bbec719ef9bf6.png)
方法三:

进入【供应商初始数据】

![](https://vip.kingdee.com/download/0109ec2a9e02fb304d7eab056897fa5483c1.png)
勾选对应的单据,<全流程跟踪>

![](https://vip.kingdee.com/download/01094e55c63e28af4bf48a3cb28d8c06241e.png)
可以联查期初应付的付款单

![](https://vip.kingdee.com/download/01094ef142a26fe24c8d9ec7dd4283cbbe59.png)
