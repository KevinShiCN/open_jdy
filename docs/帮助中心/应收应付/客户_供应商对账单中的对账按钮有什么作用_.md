# 客户/供应商对账单中的对账按钮有什么作用?

> 作者：Schatz | 阅读：91 | 有用：0

【问题描述】

客户/供应商对账单中的对账按钮有什么作用?

【解释说明】

客户/供应商对账单中的“对账”按钮,适用于无需生成对账单的单据直接标记对账完成的业务场景。操作时,只需选择对应单据并点击“对账”,该单据即被标记为“已对账”。后续通过“对账状态”筛选未对账单据时,这类无需生成对账单或无需对账的单据就不再显示。
