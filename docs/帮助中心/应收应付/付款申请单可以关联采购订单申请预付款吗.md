# 付款申请单可以关联采购订单申请预付款吗

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：97 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/709933508831059968)

【问题描述】

付款申请单可以关联采购订单申请预付款吗?

【解释说明】

**当前系统支持付款申请单关联已审核的采购订单申请预付款。**

【适用版本】

金蝶云星辰3.1旗舰版

【解决方案】

在付款申请单界面,将业务类型选择为“预付款”,即可选择已审核的采购订单录入申请预付款,如下图所示:
