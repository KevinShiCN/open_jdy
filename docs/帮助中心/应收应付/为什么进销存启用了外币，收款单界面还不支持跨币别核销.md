# 为什么进销存启用了外币，收款单界面还不支持跨币别核销

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题, 快速入门
> 作者：Zyl | 阅读：1320 | 有用：9
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/198442060994071808)

问题描述：

为什么进销存启用了外币，收款单界面还不支持跨币别核销

![](https://vip.kingdee.com/download/010078516ae8ed144b638565c685dd1caf9b.png)

解决方案：

系统参数-资金参数-基础参数-款项核销方式选择”按单据核销“、”按分录核销“，资金单据才支持跨币别核销。选择”按余额核销“时，资金单据不支持跨币别核销。

![](https://vip.kingdee.com/download/010001b89d97342245cab14ec3ac208a0679.png)
