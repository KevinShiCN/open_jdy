# 其他往来单上的结算方向有什么作用?

> 作者：RJS | 阅读：33 | 有用：1

【问题描述】

其他往来单上的结算方向有什么作用?

【说明】

1、“结算方向”决定了这笔其他往来单的“结算金额”显示在账户收支明细表收入列还是支出列

如图,当“结算方向”为【付款】时,显示在账户收支明细表的支出金额列‘;当“结算方向”为【收款】时,显示在账户收支明细表的收入金额列
