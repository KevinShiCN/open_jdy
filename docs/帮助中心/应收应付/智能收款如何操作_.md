# 智能收款如何操作?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：倒也不扶墙 | 阅读：3408 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/571863346949750016)

【问题描述】

智能收款如何操作?

【应用场景】

智能收款是指根据银行实际到账的流水(日记账)智能生成预收款单或者收款单。

【操作说明】

依次点击【应收应付】-【智能往来】-【智能收款】,在智能收款界面先设置收款规则,然后新增/引入银行流水,再设置匹配往来账户即可生成收款单/预收款单。
![](https://vip.kingdee.com/download/0109ced81aa1d1e74f30bfbdfea0bf43a33a.png)
**1、设置智能收款规则**

点击右上角“智能收款规则”设置生产收款单据规则、日期、状态以及账户匹配。
![](https://vip.kingdee.com/download/0109bbb73d0e396d4269bac70ecc2f48c213.png)
**1.1 生成收款单据规则**(三种规则选择其中一种):银行流程生成预收款单还是收款单

**①银行流水自动生成预收款单:**指银行流水智能收款时全部生成预收款单。

**②银行流水与应收单据待收款金额相同时生成收款单:**指当银行流水金额与往来账户匹配的结算客户的应收单据有相同金额时,智能收款会直接生成收款单;金额不同时,银行流水则不能生成收款单。

**③银行流水与应收单据应收款金额相同时生成收款单,金额不同时生成预收款单:**指当银行流水金额与往来账户匹配的结算客户的应收单据有相同金额时,智能收款会直接生成收款单;金额不同时则直接生成预收款单。

④若生成收款单据规则选择“银行流水与应收单据待收款金额相同时生成收款单”或“银行流水与应收单据应收款金额相同时生成收款单,金额不同时生成预收款单”时需要再选择“相同资金单据核销顺序”:

![](https://vip.kingdee.com/download/0109950059cf20574628802aed8869f85802.png)

**a、收款时按单据结算日期先后顺序:**银行流水智能收款时,按结算客户应收单据上的结算日期顺序生成先后核销顺序的收款单。

**b、收款时按单据类型 + 结算日期先后顺序:**银行流水智能收款时,先按结算客户应收单据的单据类型顺序,再按结算日期排序。需要等上个单据类型所有结算日期都收款完再收下个单据类型的款。

**1.2 收款单据日期取值**(两种规则选择其中一种):银行流水生成收款单/预收款单时,单据的日期取值规则。

**①取银行结算日期:**指日记账中的业务日期,选择此参数时生成的预收款/收款单的单据日期自动带出日记账的业务日期。

**②取系统日期:**选择此参数时生成的预收款/收款单的单据日期自动带出系统时间。

![](https://vip.kingdee.com/download/0109b32fd4ed6d154d6984c4c1d42adac559.png)
**1.3 收款单据状态**(两种规则选择其中一种):银行流水生成收款单/预收款单时单据状态

**①已保存状态:**银行流水生成的收款单/预收款单是保存的状态,后续可以通过收款单/预收款单列表界面审核单据。

**②已审核状态:**银行流水生成的收款单/预收款单是已审核的状态。

**1.4 往来账户匹配设置**

若**勾选“手工生成收款单据时,自动保存并更新往来账户与结算客户/供应商的匹配关系”**,当日记账往来账户的名称与客户名称不相同时,通过日记账手动生成收款单据时手动选择结算客户,保存单据后匹配往来账户里面往来账户与结算客户就会自动匹配上。

![](https://vip.kingdee.com/download/010903c4b2bafd3e457abcd07b7b5a293062.png)

**2、导入银行交易流水**

点击右上角“流水引入”即可导入日记账数据,流水引入支持流水引入、同步交易明细、模板引入多种方式,通过智能收款界面或日记账界面都可以操作引入,智能收款这里引入的银行流水会同步到日记账中,同理日记账中新增/引入的银行流水也会同步到智能收款界面。

![](https://vip.kingdee.com/download/010975af5d4ca6f14b4fb3f057b238211b0d.png)

2.1 手工新增:在【出纳管理】-【日记账】中手工新增的日记账会自动同步到智能收款界面。

2.2 流水引入:下载的银行流水可以直接导入为日记账,具体操作可参考[如何导入日记账?](https://vip.kingdee.com/knowledge/391351962014723072?get_from=article-id&lang=zh-CN&productLineId=35&isKnowledge=2)

2.3 同步交易明细:若启用银企互联功能,可以直接下载银行流水,具体操作可参考[银企互联功能如何使用?收费吗?](https://vip.kingdee.com/knowledge/568964245635623168?get_from=article-id&lang=zh-CN&productLineId=35&isKnowledge=2)

2.4模板引入:下载引入模板,在模板中录入日记账后再导入系统,具体操作可参考[如何导入日记账?](https://vip.kingdee.com/knowledge/391351962014723072?get_from=article-id&lang=zh-CN&productLineId=35&isKnowledge=2)

**3、匹配往来账户**

**3.1 往来账户匹配规则设置**

点击右上角“往来账户匹配表”打开后可以设置日记账中往来账户和星辰的结算客户匹配关系。

![](https://vip.kingdee.com/download/01099b21791fbbe241d391d7fa88e1f4f6ab.png)

**3.2 匹配往来账户**

匹配关系设置好之后,选择对应的银行流水点击左上角的“匹配往来账户”即可匹配成功。

![](https://vip.kingdee.com/download/010906e9994ad54a4e9c9e9a4124050d86a2.png)

**4、智能收款**

前面三步都操作完成后点击右上角“智能收款”功能,系统会根据匹配的情况生成智能收款确认单,勾选对应的记录后点击“确认收款”或“全部确认收款”即可生成对应的收款单据。

![](https://vip.kingdee.com/download/0109a99d35fe866849628929ad1b3cf80cea.png)
