# 账龄分析表支持查询预收/预付数据

# **1 简介  **

## **1.1 功能介绍**

该功能主要用于协助企业掌握应收与预收、应付与预付轧差后的实际债权债务规模及账龄分布，为资金计划制定提供依据；方便企业对预付资金进行管控，分析垫资情况，重点关注长账龄的预付款，以及时评估是否存在履约以及资金风险。

## **1.2 适用版本**

星辰3.0以上专业版/旗舰版

## **1.3 系统路径**

应收应付-资金分析-账龄分析表

1）勾选“包含预收/预付款”

以 “客户 / 供应商” 为维度，将应收与预收、应付与预付按净额合并，以净额为基数计算账龄；

2）勾选“仅显示预收/预付款”

屏蔽应收 / 应付数据，仅取预收 / 预付款明细，按款项收取 / 支付日期至统计基准日的间隔计算账龄，遵循 “先进先出法” 划分账龄区间。

3）勾选“显示本位币”

所有金额字段拆分为双表头显示，分别显示“原币”和“本位币”金额，包括账龄区间的金额也会区分原币和本位币显示；本位币金额按业务发生时汇率折算的本位币进行合计汇总;
