# 销售订单号可以在预收款单上显示吗?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：云社区用户8x80607 | 阅读：101 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/628760664378722304)

【问题描述】

销售订单号可以在预收款单上显示吗?

【操作步骤】

1、应收应付-预收款单列表,**筛选条件查询范围选择【基本信息 / 源单信息】**,即可查询源单销售订单单据编码![](https://vip.kingdee.com/download/0109c57ba17600ca4c10a06ff95ef6f07380.png)
2、若查询范围选的是【基本信息 / 收款信息】时,只能查看预收款单由销售订单生成的源销售订单号![](https://vip.kingdee.com/download/010966a94ebbd66548ffa88f161c384be06a.png)
