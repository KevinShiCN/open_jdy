# 其他收入单、其他收入退款单如何填写优惠?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：胡大大 | 阅读：84 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/691089393003457024)

【问题描述】

其他收入单、其他收入退款单如何填写优惠?

【解决方案】

1.点击【应收应付】-【其他收入】

![](https://vip.kingdee.com/download/0109bfb4edee8b404c259f5ec9f45f33ee82.png)

2.在其他收入库单新增界面将优惠金额字段填写到<优惠金额>字段

![](https://vip.kingdee.com/download/0109b3fe729271b94fc18b85f0ac11370d7d.png)

同理,其他收入退款发生的优惠也可以填写到此字段

![](https://vip.kingdee.com/download/010907afb544a7fe4dc8aa56545ae5435f6e.png)
【温馨提示】

手续费+优惠金额+本单收款金额合计=收款核销总金额
![](https://vip.kingdee.com/download/0109254c39603a6a4d27bd4387e3f6913949.png)
