# 预收转应付、预付转应收业务如何处理?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题

【问题描述】

预收转应付、预付转应收业务如何处理?

![](https://vip.kingdee.com/download/0109e8fc80ba50af49db950d8763ff5f0b07.png)**[视频观看](https://vip.kingdee.com/school/detail/723200317331780864?productLineId=35&lang=zh-CN&cosmic_type=video)**

【业务场景】

1、预收账款转为应付账款

预收账款转为应付账款通常发生在企业预收了客户的款项,但后续业务发生变化,导致预收款需要转为其他形式的负债。

比如预收了客户的货款,企业无法按约定交付商品或服务,双方协商终止合同,企业需要退回预收款,但由于资金暂时无法退回,想将其转为应付账款(对客户的负债)。

2、预付账款转为应收账款

预付账款转为应收账款通常指企业预付了供应商的款项,但是后续业务发生变化导致预付款需要转为其他形式的资产。

比如企业预付供应商货款后,供应商未能按约交付货物或服务,经协商决定终止交易并收回预付款项,但由于资金原因暂时无法收回,想将其转为应收账款。

【解决方案】

在进销存云系统中可以通过应收应付系统的其他收入/其他支出单调整往来数据,再将单据生成凭证即可;若只使用了财务云系统则直接在账务处理中录入转账凭证即可。

1、预收账款转为应付账款

先通过应收应付系统录入其他收入单(应收)冲减预收,再录入其他支出单(应付)增加应付款。

1.1 依次点击【应收应付】-【其他收入】界面,新增其他收入单(注意不要录入收款信息会生成应收)。

![](https://vip.kingdee.com/download/010922f8cf8104ef4e59911fa5315ec22da1.png)

1.2 在【应收应付】-【手动核销】中选择预收冲应收,将预收款单和其他收入单(应收)核销了。

![](https://vip.kingdee.com/download/01096e4e7d63a0164eda95ecb5220a7353b2.png)

1.3 在【应收应付】-【其他支出单】中新增其他支出单,注意不要填写付款信息会生成应付。若后期把款项退回了再录入付款单关联其他支出单即可。

![](https://vip.kingdee.com/download/01092eae4ea1619d444c9984541e73129be6.png)

2、预付账款转为应收账款

先通过应收应付系统录入其他支出单(应付)冲减预付,再录入其他收入单(应收)增加应收款。

2.1 依次点击【应收应付】-【其他支出单】界面,新增其他支出单(注意不要录入付款信息会生成应付)。

![](https://vip.kingdee.com/download/010945484a34106d4f74a42adfcfb7828914.png)

2.2 在【应收应付】-【手动核销】中选择预付冲应付,将预付款单和其他支出单(应付)核销了。

![](https://vip.kingdee.com/download/010947cb638bd16a4fb984bc756e182bb397.png)

2.3 在【应收应付】-【其他收入单】中新增其他收入单,注意不要填写收款信息会生成应收。若后期收回了款项再录入收款单关联其他收入单即可。

![](https://vip.kingdee.com/download/0109e77ee3c4274e4d02b3987777a4d37ae9.png)
