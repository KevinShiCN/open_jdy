# 金蝶云星辰，应收应付单据审批流程中支持审批人修改单据字段

1 简介

### 1.1 功能介绍

功能适用于应收应付单据启用了审批流，在流程中需要修改某些字段信息的场景。比如付款单时业务员发起，出纳支付后审批，支付后才能确认付款账户，此时出纳需要在审批流程中修改账号信息。

### 1.2 适用版本

星辰3.0以上标准版/专业版/旗舰版/所有版本

### 1.3 系统路径

系统设置>工作流>流程设计

## 2 操作指南

1）应收应付单据支持审批流程中设置修改付款账户，结算方式，单据日期，备注等信息。支持配置的单据和字段如下：

| 可编辑单据 | 单据可编辑字段 |
| --- | --- |
| 付款单 | 单据日期，单据备注，付款信息.付款账户，付款信息.付款方式，付款信息.手续费，付款信息.交易号/票据号，付款信息.付款备注，附件 |
| 预付款单 | 单据日期，单据备注，付款信息.付款账户，付款信息.付款方式，付款信息.手续费，付款信息.交易号/票据号，付款信息.付款备注，附件 |
| 其他支出单 | 单据日期，单据备注，付款信息.付款账户，付款信息.付款方式，付款信息.手续费，付款信息.交易号/票据号，付款信息.付款备注，附件 |
| 收款退款单 | 单据日期，单据备注，退款信息.退款账户，退款信息.退款方式，退款信息.手续费，退款信息.交易号/票据号，退款信息.退款备注，附件 |
| 预付退款单 | 单据日期，单据备注，退款信息.退款账户，退款信息.退款方式，退款信息.手续费，退款信息.交易号/票据号，退款信息.退款备注，附件 |
| 其他收入退款单 | 单据日期，单据备注，退款信息.退款账户，退款信息.退款方式，退款信息.手续费，退款信息.交易号/票据号，退款信息.退款备注，附件 |

2）配置步骤：

以付款单为例，打开流程设计器，选择需要配置的审批节点，在基本信息中开启“单据是否可修改”，并在PC可修改的字段中选择需要修改字段，进行开启即可。

3）配置后效果：配置后，该节点的审批人点击单据进行审批时，可对相应字段进行修改并保存。
