# 赊账业务如何开单?

【问题描述】

赊账业务如何开单?

【解释说明】

赊账:买卖双方在交易发生时暂不进行货款收付,将应收/应付金额计入往来账款,按约定延期结算。

赊账业务仅影响财务往来账务处理,**库存仍按实际出入库情况实时更新**,与账款结算周期无关。即:

**销售赊销**:商品出库后,库存减少,但货款暂未收回(挂应收账款)

**采购赊购**:商品入库后,库存增加,但货款暂未支付(挂应付账款)

由于无法直接进行收款操作,因此在开具业务单据时,不填写收款账户便可生成应收/应付记录。
