# 客户/供应商对账单查询条件里面为什么没有简易对账功能了?

> 作者：shaoling | 阅读：107 | 有用：0

【问题描述】

客户/供应商对账单查询条件里面为什么没有简易对账功能了?

【解释说明】

客户 / 供应商对账单更新后,应收及收款对账/应付及付款对账功能,对应之前简易对账的操作页面(即原对账单勾选简易对账后显示的数据页面)

![](https://vip.kingdee.com/download/010960b964fdd1c049168dd78d4fc7ce6735.png)

对账单全面升级后,支持按客户或供应商和账期生成对账单,生成后可查询历史对账单,进行对账单确认、收款等操作。
