# 返利、坏账、押金、调账、员工借款、报销之单据录入

返利、坏账、押金、调账、员工借款、报销之单据录入

![上传图片](https://vip.kingdee.com/download/01004ef32f0b65864663a07d4061cf029010.png)

      小李最近入职了一家新公司，公司里面都是老会计说了算。这不看小李是个新人，就想给他立好规矩，要求他业务这边做单要和财务那边每一笔都要对上。小李不禁打了个寒颤，这椅子还没坐热，就要开始忙活配合老会计的工作了。好在小花是个热心的同事，她看不惯老会计欺负新人，就跟小李讲了一些常见资金类单据的录入。我们一起来看看小花都传授了哪些经验：

1、 返利

1） 客户返利

i） 如果客户返利直接返款，则做其他收入退款单，直接账户支付。

![](https://vip.kingdee.com/download/0100eb8fb540e2e241518ce3dfd1499a278d.png)

ii） 如果客户返利不返款，用于冲抵下次货款，则做其他收入退款单，不退款。收款时可以和销售出库单一起冲抵货款。

![](https://vip.kingdee.com/download/0100b570549088a048eba627c7193173722d.png)

![](https://vip.kingdee.com/download/01008ca4e24fd595462cae566b6e39489abd.png)

2） 供应商返利

i） 如果供应商返利直接返款，则做其他支出退款单，直接账户支付。

![](https://vip.kingdee.com/download/0100a4c2272232874f70bd2b5f1cd1da0d7e.png)

ii） 如果供应商返利不返款，用于冲抵下次货款，则做其他支出退款单，不退款。付款时可以和采购入库单一起冲抵货款。

![](https://vip.kingdee.com/download/0100504760e6e9b84c8d92b9edd16df0fb4c.png)

![](https://vip.kingdee.com/download/01002d03dbc3217341529488962f690e3132.png)

2、 客户坏账

常见因债务纠纷或者货物破损等原因导致坏账不能收回。可以做收款单选源单，收款金额是0，金额填写在优惠金额处，表示此单实际没有收款，但是在系统里面核销了。

![](https://vip.kingdee.com/download/010007f2394d6501416484c3dfcba15395c4.png)

3、 押金

1） 客户押金

i） 收客户押金时，做其他收入单。

![](https://vip.kingdee.com/download/01007bf29860362f43bc8d321acd1c33e277.png)

ii） 退客户押金时，做其他收入退款单。

![](https://vip.kingdee.com/download/0100dc1682d224e8436eb5e4030aeb5fadcd.png)

2） 供应商押金

i）付供应商押金时，做其他支出单。

![](https://vip.kingdee.com/download/010056dc6a49bca04ecca048f087c06c537c.png)

i） 供应商退押金时，做其他支出退款单。

![](https://vip.kingdee.com/download/0100d263771a995a41a392d44f18b79272b4.png)

4、调账

1） 应收调账

之因前销售开单金额与实际结算的金额不一致，需要调整应收金额。若要影响销售应收，只能做销售出库或者退货，但是库存会变化。所以此处调账不用销售类单据。

i） 应收调增用其他收入单，通过影响其他应收余额，进而影响余额，但不影响库存。

![](https://vip.kingdee.com/download/01007166885f3a904cd3abbaa823d8bc0139.png)

此处余额从110调增至120。

![](https://vip.kingdee.com/download/010058d65562391a45e2a6cd0f90765903c9.png)

ii） 应收调减用其他收入退款单。

![](https://vip.kingdee.com/download/0100e3d32f8e1e3f4c0bbda968d4ae89569a.png)

此处余额从410调减至310。

![](https://vip.kingdee.com/download/0100840a6605efd44bd3af6af4551cc08543.png)

2） 应付调账

同理应付调账不用采购类单据。

i） 应付调增用其他支出单。通过影响其他应付余额，进而影响余额，但不影响库存。

![](https://vip.kingdee.com/download/010084a7c2d6de6c4f7ba4ddd252928ce3a0.png)

此处余额从0调增至100。

![](https://vip.kingdee.com/download/01009157574e8b424f6ca194266dabf77202.png)

ii） 应付调减用其他支出退款单。

![](https://vip.kingdee.com/download/0100ad30ca93ba3f4532b44591a93cb7ed2c.png)

此处余额从100调减至20。

![](https://vip.kingdee.com/download/010074834608804d435d86bbee743a983e60.png)

5、员工借款

1）员工借款，做其他支出单，账户直接付款，供应商不必录。

![](https://vip.kingdee.com/download/0100145243f121804d0698cc2b59e70c6dcf.png)

2）员工还款，做其他支出退款单，账户直接退款，供应商也是不必录。

![](https://vip.kingdee.com/download/010063191aa5d036416cb20cae3580ad7984.png)

6） 员工垫付报销

1） 员工垫付，将员工做成一个账户，单据录入的时候选员工账户，员工账户赤字。

![](https://vip.kingdee.com/download/0100d99ebcb64ec8454fb6d6861c32b41d20.png)

2） 员工报销，做资金转账单，将公司账户转给员工账户。员工账户进账，员工账户余额是0 。

![](https://vip.kingdee.com/download/010004129be9bc2847f9b45d9cf51f9de19f.png)

![](https://vip.kingdee.com/download/01008bd5d2d8483944b9a1e72e9bdf7c103a.png)

通过小花的热心帮助，让小李感受到了人间温暖，这让他有了配合老会计工作的信心。
