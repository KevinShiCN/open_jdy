# 智能收款生成收款单据的单据日期怎么设置?

【问题描述】
智能收款生成收款单据的单据日期怎么设置?

【解释说明】

依次点击【应收应付】-【智能收款】![](https://vip.kingdee.com/download/01093fd7af3d3e8f4c2192d106d9b5f3d971.png)

点击右上角【智能收款规则】,进入规则配置界面

![](https://vip.kingdee.com/download/0109dc56443beaed483ab6bf7b2d1059e7cd.png)

可设置单据日期的取值规则:取银行结算日期 或 取系统日期
