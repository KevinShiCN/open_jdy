# 如何查询/打印/导出客商匹配对账单?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：云社区用户253x0504 | 阅读：2011 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/463871722752454912)

【问题描述】

如何查询/打印/导出客商匹配对账单?

【业务场景】

客商匹配对账单用于统计既是客户又是供应商的单位往来余额,便于用户对账。客户和供应商需要先关联后,才能体现在客商匹配对账单中。

【操作步骤】

**1、客户和供应商设置关联关系**

用户可以选择在新增客户的同时新增同名供应商,也可以选择在已有客户中关联已有供应商,具体操作可参考:[既是客户也是供应商在系统中如何维护基础资料?](https://vip.kingdee.com/knowledge/391353658795469568?productLineId=35&isKnowledge=2&lang=zh-CN)

系统目前仅支持一个客户关联一个供应商,已经关联的供应商不支持再次被其他客户关联。为了避免重复设置,客商关联设置仅支持在客户侧进行设置。

![](https://vip.kingdee.com/download/010939a4fd3f3f004abab4ca15b83bedd98f.png)

**2、查询客商匹配对账单**

客商匹配对账单界面客商匹配对账单支持按照以客户为主体或者以供应商为主体双向对账(余额相等,正负号不同)。并支持按照特定筛选条件进行对账。比如部门、业务员、结算日期和单据类型。

客商匹配对账单支持多样化筛选过滤条件,比如“包括未审核”“显示单据明细”“往来余额为 0 不显示”“仅显示小计行”“不显示小计行”等

可点击【应收应付】>【客商匹配对账单】设置条件查询
![](https://vip.kingdee.com/download/01096d9fdbb7b38c4fcf84e7ec81363aae07.png)
查询结果可点击右上角<引出>按钮导出到电脑本地

![](https://vip.kingdee.com/download/0109c4a01d56daf24209bf59d06fedf2b7e4.png)

若需打印,可点击打印按钮下的<打印设置>,选择打印模板后点击<确定并打印>

![](https://vip.kingdee.com/download/0109cd0c51c102db42e2a5407a8fec4a7c70.png)
