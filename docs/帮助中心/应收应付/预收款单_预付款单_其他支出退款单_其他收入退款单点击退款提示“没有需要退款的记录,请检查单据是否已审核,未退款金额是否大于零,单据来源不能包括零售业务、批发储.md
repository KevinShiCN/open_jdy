# 预收款单/预付款单/其他支出退款单/其他收入退款单点击退款提示“没有需要退款的记录,请检查单据是否已审核,未退款金额是否大于零,单据来源不能包括零售业务、批发储值!”

> 作者：幽乐6 | 阅读：119 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/721537555270551808)

【问题描述】

预收款单/预付款单/其他支出退款单/其他收入退款单点击退款提示“没有需要退款的记录,请检查单据是否已审核,未退款金额是否大于零,单据来源不能包括零售业务、批发储值!”

【解决方案】

下面以**预收款单**为例截图指导说明

预收款单点击退款提示“没有需要退款的记录,请检查单据是否已审核,“退款”是否大于0,或单据来源是否包括零售业务和批发储值。”

![](https://vip.kingdee.com/download/01090d291b4af7b64d03a4498f4b640a3806.png)
首先看单据是否已审核,确保单据已审核状态,退款金额大于0

![](https://vip.kingdee.com/download/01097726d0edc7c644618266d2b71d42f1a0.png)
其次是**预收款**单的情况下,我们需要确保**单据来源**是非零售和批发储值

![](https://vip.kingdee.com/download/01097db47ff4ff184c6db61432748f5eb46e.png)
确保满足以上条件之后,这样单据可以正常操作点击退款成功,如图:

![](https://vip.kingdee.com/download/0109d8f5e6a2572f438abbd937069af10761.png)
另外针对**预收款**单是来源于批发储值的退款请参考:[订货商城,如何进行储值退款?](https://vip.kingdee.com/link/s/Z752f)

针对**预收款**单是来源于零售充值的退款请参考:[会员充值后可以退冲吗?](https://vip.kingdee.com/link/s/ZpJBE)
