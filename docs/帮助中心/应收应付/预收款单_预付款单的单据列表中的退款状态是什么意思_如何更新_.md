# 预收款单/预付款单的单据列表中的退款状态是什么意思?如何更新?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：向钱kan | 阅读：217 | 有用：0

【问题描述】

预收款单/预付款单的单据列表中的退款状态是什么意思?如何更新?

【解释说明】

预收款单 / 预付款单列表中的**退款状态**,用于反映单据的退款情况。将预收款单 / 预付款单下推生成退款单并保存后,系统将自动更新退款状态、已退款总金额及未退款总金额。
