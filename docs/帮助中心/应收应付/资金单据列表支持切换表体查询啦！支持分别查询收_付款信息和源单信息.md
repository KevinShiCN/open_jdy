# 资金单据列表支持切换表体查询啦！支持分别查询收/付款信息和源单信息

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：快速入门, 功能介绍, 新特性

灰度发布时间：2024年9月

注意：本文所提及的功能都还在星辰3.1灰度，如您已经升级到星辰3.1，需要拉3.1灰度的话，请联系客服处理

## 1 简介

### 1.1 功能介绍

这个功能主要应用于在资金单据列表查看其源单信息和收/付款信息，之前只能查看收/付款信息，查不到源单信息。本次补全了这种场景，可以解决客户想在资金单据列表查看源单信息的需求。

### 1.2 适用版本

星辰3.0以上标准版/专业版/旗舰版/所有版本

### 1.3 系统路径

本功能对以下圈起来的资金单据列表都生效。

## 2 操作指南

![](https://vip.kingdee.com/download/010978f8ee21bef349df92dbbd096659bd2d.png)

以收款单为例，收款单列表可以切换查询收款信息/源单消息，但是不能同时查询上下表体。注意，如果想看这两个表体信息，就不能开启【整单】按钮，开了整单就只能看到表头字段。

后续我们会支持对收款消息、源单消息下的分组字段做列设置去控制显隐，此需求已在开发中。
