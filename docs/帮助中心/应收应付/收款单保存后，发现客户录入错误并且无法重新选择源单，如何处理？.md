# 收款单保存后，发现客户录入错误并且无法重新选择源单，如何处理？

> 作者：财税云肖梅梅 | 阅读：587 | 有用：2

常见问题：收款单保存后，发现客户录入错误并且无法重新选择源单，如何处理？![](https://vipstatic.obs.cn-north-4.myhuaweicloud.com/statics/emotion/define/64.gif)

![](https://vipstatic.obs.cn-north-4.myhuaweicloud.com/statics/emotion/define/93.gif)答案：收款单保存后，不允许再重新选择源单。可以选择录入错误的收款单，复制，重新录入对的客户和选择源单。
再将原来录入错误的收款单删除即可。
