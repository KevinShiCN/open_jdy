# 其他支出退款单如何核销?可以与哪些单据核销?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：ZXYzxc | 阅读：308 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/680948815213591552)

【问题描述】

其他支出退款单如何核销?可以与哪些单据核销?

【解决方法】

其他支出退款单上可以直接退款核销;未退款的其他支出退款单可以通过付款退款单及手动核销单核销,支持与其他支出单、预付款单、采购类单据、其他收入单、其他收入退款单、销售类单据等各种往来单据核销。

**1、其他支出退款单直接退款核销(现付)**

在【应收应付】-【其他支出退款单】录入其他支出退款单时直接录入“退款信息”,单据审核后即已经完成了核销,不会影响往来款,不会参与后续的核销抵扣。

![](https://vip.kingdee.com/download/01097a27e14f1b86407f9f24b37d8d03d673.png)

**2、通过付款退款单及手动核销单核销**

做其他支出退款单时不录入“退款信息”会产生应退款账款,先记录需要退回的费用,后期收到退回的费用时通过付款退款单核销,或者与其他的往来单据进行抵扣(手动核销单)。

![](https://vip.kingdee.com/download/0109ea007bf0062c4ecdb46ee59f1a909301.png)

**2.1 做付款退款单完成退款核销**

收到这笔退款时,通过【应收应付】-【付款退款单】完成收款,付款退款单的源单信息选择“其他支出退款单”。

做付款退款单时可以选择多个不同业务类型的单据一起退款核销,如:付款单、采购退货单(旗舰版的采购退货发票)、委外加工费用(退货)。
![](https://vip.kingdee.com/download/0109ffd21f3b17b4483eac42e8e400471946.png)
【温馨提示】

若这次只是收到部分退款,则注意修改源单信息中的本次核销金额(退款信息中退款金额+优惠金额=源单信息的本次核销金额),下次再支付时可以再次选择这个其他支出退款单。

![](https://vip.kingdee.com/download/0109247e863803e5411a8739a3ebb0270695.png)
**2.2 做付款单时与其他应付单据抵扣核销**
其他支出退款单可以与其他的应付单据抵扣核销,如:与其他支出单、采购入库单(旗舰版的采购发票)、委外加工费用等一起抵扣,在做付款单时可以同时选择其他支出单和其他支出退款单,付款单录入实际付款金额即可(若刚好全部抵扣则不用录入付款金额)

![](https://vip.kingdee.com/download/0109cefc13fce1d1445bb7f48eacc2608c8f.png)

**2.3 录入手动核销单抵扣**

未付款的其他支出退款单还可以与其他往来单据一起抵扣,核销在【应收应付】-【手动核销】中新增手动核销单。

如:核销类型为“预付冲应付”是,可以同时选择其他支出退款单与其他应付的业务单据一起抵扣核销;

![](https://vip.kingdee.com/download/0109a207f481637b41e88069bd98863f5f1a.png)

核销类型选择“应收冲应付”时,可以选择其他支出退款单与其他的应收及应付往来单据一起抵扣核销。

![](https://vip.kingdee.com/download/01098276ada8b4454027a442a20ac827d324.png)
核销类型选择“应付转应付”时可以将这笔应退款项转给其他供应商。

![](https://vip.kingdee.com/download/0109a67c0a4779ca4e8fa43ee9351647da46.png)

【温馨提示】

其他支出单不支持自动抵扣预付款,可做手动核销,核销类型选择预付冲应付。(可参考上方**2.3**的操作步骤)
