【问题描述】

业务单据中的收/付款方式有什么用?和收/付款账户有什么区别?能否设置选账户默认带出结算方式?

【解释说明】

①业务单据中的**收 / 付款方式**,指收款人接收款项的具体途径,例如银行转账、支付宝支付、微信支付、现金支付等。该字段为非必填项,其可选值来源于基础资料的结算方式界面,且结算方式支持自定义新增、修改与删除。

②业务单据中的**收 / 付款账户**,指收款人指定的收款账户,包含现金、微信账户、银行账户、支付宝账户等类型。需注意,单据中一旦录入收付款金额,收 / 付款账户即为必填项。

③系统支持单独设置默认结算方式,但暂不支持配置 “选择账户后自动带出对应默认结算方式” 的联动规则。

默认结算方式设置,可点击:[如何设置/取消默认结算方式?](https://vip.kingdee.com/knowledge/594331738181271040?productLineId=35&isKnowledge=2&lang=zh-CN)
