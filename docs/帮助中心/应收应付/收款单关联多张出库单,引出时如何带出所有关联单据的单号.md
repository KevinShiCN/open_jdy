# 收款单关联多张出库单,引出时如何带出所有关联单据的单号

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：财税李炳全 | 阅读：134 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/568601846546270464)

**【****问题描述****】**

收款单列表导出时可以带出源单的单据编号,目前系统如果一张收款单有多张出库单的,导出的excel中没有数据的源单单号,如何显示完整单号?

![](https://vip.kingdee.com/download/0109385f2ec8e69d4daabedf0557309a52d6.png)

**【****解决方案****】**

此需求可以变通解决,在收款单列表点击按引入模板引出即可。

![](https://vip.kingdee.com/download/0109ccf7f001c3f841c189be66c42bb0cadd.png)
