# 自动核销执行日志如何应用?

> 作者：ylim | 阅读：258 | 有用：0

【问题描述】

自动核销执行日志如何应用?

![](https://vip.kingdee.com/download/0109c58ba66686944aa8b465da6811e97740.png)
【解释说明】

自动核销执行日志是指设置的自定核销方案执行的日志记录,在这里可以看到生成了几张核销单、未核销的单据有哪些,核销类型是什么,以及对应执行的方案名称、方案编号。

![](https://vip.kingdee.com/download/0109f54fc3170e324b05a3d9deaeea98d90d.png)
自动核销执行日志显示的“创建人”为本次执行该方案的用户。

![](https://vip.kingdee.com/download/01099be48fe5afda4ee0b9a90f62cc33f55c.png)

![](https://vip.kingdee.com/download/01099db136f44aef4599903f172110309ff0.png)【温馨提示】

自动核销执行日志默认保留一个月,超过一个月后,最多保留20条。
