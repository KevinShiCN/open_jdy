# 应收账款明细表和账户收支明细表怎么对上？

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题, 进阶技巧
> 作者：Daisy_ | 阅读：885 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/262238188481945856)

问题描述：

应收账款明细表和账户收支明细表，收入的总金额怎么对不上？

解决方案：

1、如果需要看部门的收款，可以查看【账户收支明细表】的【净收入】字段（收入-支出）

2、如果要拿应收账款明细表和账户收支明细表去对收款数据，请务必保证两个报表的筛选条件一致（如是否包含零售结存单），并按以下公式对：

   应收账款明细表: 【收款合计金额】+【预收金额】 - 【优惠金额】

   账户收支明细表： 【收入】-【支出】

【收款金额合计】+【预收款金额】合计 - 【优惠金额】合计 = 【收入】-【支出】
