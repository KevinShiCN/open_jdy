# 账龄分析表的应收余额需要和科目余额的应收款余额保持一致吗?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：若有灵兮山之阿 | 阅读：98 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/726603587140261120)

【问题描述】

账龄分析表的应收余额需要和科目余额的应收款余额保持一致吗?
【情景描述】
账龄分析表的应收余额和科目余额的应收款余额查询时不一致
【解释说明】
不一定一致

**1、应收应付中的账龄分析表和应收账款科目余额不一致**

①应收应付中的账龄分析表不是按查询日期筛选的,截止日期是指单据日期,并不是截止收款日期,如果查询截止日期后面有收款业务发生也会统计进去。

比如下图,客户是2024年11月才收款核销,账龄分析表查询截止到2024年10月31,不会显示该客户有余额了
![](https://vip.kingdee.com/download/01092e1577ab99664463a64d8a8296e641b7.png)![](https://vip.kingdee.com/download/0109ddb96546a66e4460a8e24a9ad001b2dd.png)
②账龄分析表数据来源是单据,科目余额表数据来源是凭证

当有的单据没有生成凭证、或业务中没有录入相关单据但是有录凭证,均会导致科目余额表和账龄分析表余额不一致

③账龄分析表仅显示应收余额,若财务上预收使用的应收科目负数表示,会导致两个数据不一致

**2、账务处理中的账务账龄分析表和应收账款科目余额不一致**

①筛选条件不一致

如账龄分析表中设置的查询某和具体客户的,但是科目余额表中只显示了一级科目

![](https://vip.kingdee.com/download/0109a248966795774488ad8934f9d5999a18.png)
![](https://vip.kingdee.com/download/01093e081cddcd0e4aee992c7f37f8af90c4.png)
②科目余额表的期间是按凭证日期,账龄分析表查询选择的按业务日期计算

![](https://vip.kingdee.com/download/01096491343b3d8c4916850cc292af5ef4b0.png)
③账龄分析表上只能查询截止日期,当截止这个日期的在后期有凭证冲抵后,也会被更新

![](https://vip.kingdee.com/download/010931b2994582494ba9b52fe5b87fbb4e6f.png)
④当设置的是明细科目时,仅部分应收明细科目设置了往来业务核算,而科目余额表上只查看一级科目,会导致两边所看到的数据不一致

![](https://vip.kingdee.com/download/0109382f2d2e5ecf42cc82c2793c80993491.png)
