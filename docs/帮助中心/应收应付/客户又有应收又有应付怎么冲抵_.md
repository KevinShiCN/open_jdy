# 客户又有应收又有应付怎么冲抵?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：达拉崩吧 | 阅读：2203 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/562803579795524096)

【问题描述】

客户又有应收又有应付怎么冲抵?

【适用版本】

金蝶云星辰标准版,专业版,旗舰版

【业务场景】

即是客户又是供应商,产生的货款不需要支付结算,应收和应付直接抵扣的业务场景。

【解决方案】

星辰系统已有功能解决该业务场景,手动核销,可以支持选择业务类型“应收冲应付”。

【操作步骤】

点击【应收应付】>【智能往来】>【手动核销】

![](https://vip.kingdee.com/download/0109b4bf8fd494e642f39534a8c26f47d505.png)

新增核销单,选择业务类型“应收冲应付”,选择需要抵扣的双方,选源单后,注意上下单据**本次核销金额保持一致**,即可核销抵扣。

![](https://vip.kingdee.com/download/0109f901fb7d31b3483e91037b182f8d3da9.png)
若存在上下表体本次核销金额不一致,但是需要全部都冲掉不留余额,可以通过优惠金额来冲平

![](https://vip.kingdee.com/download/010955cfbdd2d5cb4eedb7be521113b377ee.png)
