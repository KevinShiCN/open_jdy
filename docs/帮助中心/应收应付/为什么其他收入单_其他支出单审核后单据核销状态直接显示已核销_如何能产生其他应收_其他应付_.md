# 为什么其他收入单/其他支出单审核后单据核销状态直接显示已核销?如何能产生其他应收/其他应付?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：ylim | 阅读：28 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/766105052821019904)

【问题描述】

为什么其他收入单/其他支出单审核后单据核销状态直接显示已核销?如何能产生其他应收/其他应付?

【解释说明】

当其他支出单或其他收入单在做单时,直接在单据上录入了付款或收款,单据审核后其核销状态会显示为已核销

![](https://vip.kingdee.com/download/01092ee5e44dd8fb4b61bc5a960a6918e543.png)![](https://vip.kingdee.com/download/01097d79f08a68994d23840a126b506cef14.png)
若需其他支出单产生其他应付、其他收入单产生其他应收,可在录单时不录付款信息和收款信息

具体参考知识:[其他支出单上的付款信息是必录的吗?](https://vip.kingdee.com/link/s/Z01DM)

[其他收入单上的客户和收款信息是必填项吗?](https://vip.kingdee.com/link/s/Z01Do)

若已在单据上录入,可反审核单据删除收款信息或付款信息录入行
