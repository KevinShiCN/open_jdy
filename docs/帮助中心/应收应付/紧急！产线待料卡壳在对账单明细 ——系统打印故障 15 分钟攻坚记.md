# **紧急！产线待料卡壳对账单明细——系统打印故障15分钟攻坚记**

![上传图片](https://vip.kingdee.com/download/0100153751b78cf948738390c0ead12fca3c.jpg)

“上午10点，系统支持部接到紧急电话，客服人员小夏紧急报告称：“李工，A车间生产线因缺乏材料而面临全面停滞。”采购部门需要根据对账单明细进行对账并付款，但金蝶系统在打印对账单时未能显示商品明细。若情况持续半小时以上，生产线将不得不停止运作。

系统支持工程师小李迅速赶往采购部，并在途中确定了问题的核心：对账单明细打印问题可能由对账单明细显示错误或打印模板设置不当导致数据缺失。

采购经理老张焦急不安地盯着电脑屏幕，对账单打印预览界面的供应商、金额、日期清晰可见，唯独商品名称、数量栏一片空白。小李一边安抚老张，一边说：“没事，这边调整一下就可以。”让我们看看小李是怎么排查的。

首先，访问【应收应付】-【历史对账单】-【供应商对账单历史列表】，点击相应的对账单编号，打开【供应商对账单详情】界面，并检查是否显示了商品信息。

![上传图片](https://vip.kingdee.com/download/01001598dea62f464e3fb354ef5deea37201.png)

![上传图片](https://vip.kingdee.com/download/01008602a96711dc4ba2a320616ab5576631.png)

![上传图片](https://vip.kingdee.com/download/010004633727cf75423f9bacf125490148ee.png)

若发现供应商对账单详情中没有显示商品信息，需要按照以下步骤删除该供应商对账单：1. 打开对账单详情界面；2. 点击“删除”按钮；3. 确认删除操作。

![上传图片](https://vip.kingdee.com/download/010097e3ef1dcce6429baf50020cb4124fe3.png)

再到【应收应付】-【供应商对账单】中，勾选参数“按商品明细对账”。

![上传图片](https://vip.kingdee.com/download/0100becccdb5095a4b6795aebe4fe779b522.png)

![上传图片](https://vip.kingdee.com/download/0100244f3b8270a1452788d5bf2f6eb3a311.png)

选择单据，点击“生成对账单”，此时单据中就会显示对应的商品信息。

![上传图片](https://vip.kingdee.com/download/0100ae1651e512bd4342a2e095f2ebb10308.png)

之后，在【系统设置】-【打印模板】中找到供应商对账单详情模板，并查看其中商品信息的设置情况。![上传图片](https://vip.kingdee.com/download/010041f190db4e634fe7a6a8e7ecb5c595d6.png)

![上传图片](https://vip.kingdee.com/download/0100d867c66ad02648f4b25c01291c39dd11.png)

![上传图片](https://vip.kingdee.com/download/01005913dc578b994b0fa6fa390f731c202a.png)

此时，如果打印模板未设置商品信息内容，需选中一列，单击鼠标右键，再点击“右侧/左侧插入”以新增一列。

![上传图片](https://vip.kingdee.com/download/0100f9bfd42f73a84027b8381a12e6e843f0.png)

设置好标题后，点击数据行表格，再点击“内容”，绑定到对应的商品编码、商品名称、规格型号等字段。

![上传图片](https://vip.kingdee.com/download/010001f2e23c0b664dbe85657f32c98e7c32.png)

打印模板设置完成后，保存模板。

![上传图片](https://vip.kingdee.com/download/0100a8c331f8f4b0431db9ae3404dba84a64.png)

最后，到【供应商对账单历史列表】界面，依次点击【打印】-【打印设置】-【选择模板】，选择新增的模板后点击【确定】。

![上传图片](https://vip.kingdee.com/download/010016d61988d0bf4c7ea1020f385856dc48.png)

![上传图片](https://vip.kingdee.com/download/01008df20ef7e6ed447c8abe89c0435fbf9a.png)

随后选择对应单据打印，即可在对账单中显示对应的商品信息。

![上传图片](https://vip.kingdee.com/download/010042236542ccc54125b1356a74be8af322.png)

![上传图片](https://vip.kingdee.com/download/0100d31f4a4c67394995948ef11eae84b1ce.png)

从开始排查问题到解决，仅用了15分钟。在此期间，老张迅速安排打印对账单，财务部门也快速完成了款项发放。供应商的物料在半小时后准时到达工厂，生产线随即恢复了运转。

在下午的例会上，项目经理对小李的工作给予高度评价，认为他快速准确地解决了问题，不仅保障了生产运营的连续性，还增强了用户对系统的信任。
