# 其他收入单如何录入/修改/删除发票信息?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：ZXYzxc | 阅读：382 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/660648770719495424)

【问题描述】

其他收入单如何录入/修改/删除发票信息?

【操作步骤】

**1.其他收入单录入发票信息**

点击【应收应付】>【其他收入】>【查询】

![](https://vip.kingdee.com/download/01094797faf6c78d4b7c94b281a85be8e348.png)

其他收入单列表选中其他收入单,点击【录入发票信息】,在开票信息界面进行录入数据保存;

![](https://vip.kingdee.com/download/0109bb0e0a70136d44b69196d24afd63f53c.png)

**2.其他收入单删除发票信息**

其他收入单列表点击“已开票金额”字段下的金额,在开票记录界面点击<减号>,删除发票信息;

**tips:若单据列表中没有看到“****开票金额****”字段,可以在列设置中将该字段显示,详情请点击:[如何设置单据/报表字段的显示或隐藏?](https://vip.kingdee.com/knowledge/547945651133299200?productLineId=35&isKnowledge=2&lang=zh-CN)**

![](https://vip.kingdee.com/download/0109396c0da9d7c0489fb590192f6ce5e0ac.png)
**3.其他收入单修改发票信息**

其他收入单的发票信息不能直接修改,需要将发票信息进行删除,再重新录入发票信息;

![](https://vip.kingdee.com/download/0109ef1dd111b8dc42c0bb97376578bb8adf.png)
![](https://vip.kingdee.com/download/0109b7d98593f9b243dbb08113b6a6e1e5ba.png)
