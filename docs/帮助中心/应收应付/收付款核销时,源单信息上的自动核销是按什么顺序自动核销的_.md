# 收付款核销时,源单信息上的自动核销是按什么顺序自动核销的?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：咚咚 | 阅读：702 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/616075393644293120)

【问题描述】

收付款核销时,源单信息上的自动核销时按什么顺序自动核销的?

【解释说明】

下面只举例收款单的场景,付款单类似 。

场景一、【系统设置】-【系统参数】-【应收应付参数】,按单据核销。

![](https://vip.kingdee.com/download/0109a76a9cc7d063414e9421e5e84b3743b3.png)

收款单源单信息会显示自动核销,但是需要先选择源单,填写收款金额,在点自动核销。按手动选择源单顺序,源单未核销金额依次分配到本次核销单金额里。

![](https://vip.kingdee.com/download/010962e0b2f9715247cbb04c8d5afb2a52f2.png)
备注:按分录核销类似。

场景二:【系统设置】-【系统参数】-【应收应付参数】,选择按余额核销。

收款单源单信息没有选源单,只有按余额核销。填写收款金额以后,点自动核销,源单会自动出来,不用手动选。

1、按“付款时按单据结算日期先后顺序”,按源单的结算日期顺序带出源单。

![](https://vip.kingdee.com/download/01099a83ec5e00314bdab83e97d12cd5b6f5.png)

![](https://vip.kingdee.com/download/01090f0e28ad8c4e48dcb34c9841d2e713bd.png)

2、按“收付款时按单据类型 + 结算日期先后顺序”,先按单据类型,在按结算日期顺序带出源单。

![](https://vip.kingdee.com/download/0109e087d816150645fb9d8536f0d483b0de.png)
![](https://vip.kingdee.com/download/01097c6d5b72c5834b4494284bd207826265.png)
