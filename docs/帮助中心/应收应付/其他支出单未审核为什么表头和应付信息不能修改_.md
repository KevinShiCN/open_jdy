# 其他支出单未审核为什么表头和应付信息不能修改?

> 产品：金蝶云·星辰 | 分类：应收应付 | 类型：常见问题
> 作者：lykk | 阅读：97 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/642895886325657344)

【问题描述】

其他支出单未审核为什么表头和应付信息不能修改?

【解决方案】

通过采购、销售、库存类单据下推的其他支出单,不允许在其他支出单修改表头和应付信息;

如:采购费用(采购入库单、采购退货单)、销售费用(销售出库单、销售退货单)、委托代销费用(委托代销结算单)、调拨费用(调拨出库单、调拨入库单)、组装费用(组装单)、拆卸费用(拆卸单)
