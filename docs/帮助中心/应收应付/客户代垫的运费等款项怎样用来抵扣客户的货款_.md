# 客户代垫的运费等款项怎样用来抵扣客户的货款?

【问题描述】

金蝶云星辰,客户代垫的运费等款项怎样用来抵扣客户的货款?

【解释说明】

可以通过录入其他收入退款单挂账的方式,形成一个负数的应收,达到抵扣客户应收货款的效果。

1、依次点击应收应付--其他收入退款。
