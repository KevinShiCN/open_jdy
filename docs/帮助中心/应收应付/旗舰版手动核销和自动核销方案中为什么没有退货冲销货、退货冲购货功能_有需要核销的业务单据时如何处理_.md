# 旗舰版手动核销和自动核销方案中为什么没有退货冲销货、退货冲购货功能?有需要核销的业务单据时如何处理?

【问题描述】

旗舰版手动核销和自动核销方案中为什么没有退货冲销货、退货冲购货功能?有需要核销的业务单据时如何处理?

【概述】

因为星辰旗舰版是通过销售发票和采购发票来产生应收款和应付款,所以不需要通过销售退货和采购退货来进行核销。

若有退货业务需要冲抵,可通过收付款单选源单,选择销售发票或采购发票来冲抵,采购发票的业务类型有入库和退货,销售发票的业务类型有出库和退货

【操作步骤】

以采购的退货冲购货业务为例

付款单选采购发票的源单,分别选择业务类型为入库和退货的单据

![](https://vip.kingdee.com/download/0109be4a4e7de8bd448ead5557aaf4dddccb.png)
可以看到源单金额为一正一负,可进行冲抵
