# 销售业务如何分期收款?

【问题描述】

销售业务如何分期收款?
【操作流程】

业务场景:发完货之后客户分多少付款给企业

当收到客户货款时在应收应付模版做收款单,本次收到多少款源单的本次核销金额就核销多少,剩余未收完的后续收到完了再做收款单核销即可。
