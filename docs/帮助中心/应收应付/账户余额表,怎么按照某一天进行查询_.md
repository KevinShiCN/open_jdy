# 账户余额表,怎么按照某一天进行查询?

> 作者：云社区用户55os622 | 阅读：237 | 有用：0

【问题描述】
账户余额表,怎么按照某一天进行查询?

【解释说明】

依次点击【应收应付】-【账户余额表】![](https://vip.kingdee.com/download/01091f9a0eabbd9447afa55ec3041db3db5f.png)

左上角切换到【日期】,即可选择具体的日期范围进行查询
