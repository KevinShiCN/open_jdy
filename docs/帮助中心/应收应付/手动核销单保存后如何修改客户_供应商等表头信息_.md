# 手动核销单保存后如何修改客户/供应商等表头信息?

> 作者：云社区用户3364330 | 阅读：49 | 有用：1

【问题描述】

手动核销单保存后如何修改客户/供应商等表头信息?

【解释说明】

手动核销单一旦保存,就不可修改表头的核销类型、结算客户/结算供应商、币别等字段。如单据错误,只能将单据删除再重新做。
