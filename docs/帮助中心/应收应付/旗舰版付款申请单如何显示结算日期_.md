# 旗舰版付款申请单如何显示结算日期?

【问题描述】

旗舰版付款申请单如何显示结算日期?

若您想要了解付款申请单使用问题,请点击:[付款申请单如何使用?](https://vip.kingdee.com/knowledge/614625765342326272?channel_level=%E9%87%91%E8%9D%B6%E4%BA%91%E7%A4%BE%E5%8C%BA%7C%E6%90%9C%E7%B4%A2%7C%E7%BB%BC%E5%90%88&productLineId=35&isKnowledge=2&lang=zh-CN)

【适用版本】

【解释说明】

付款申请单只有源单信息里有“源单结算日期”的字段,若需显示可在列设置里设置显示

![](https://vip.kingdee.com/download/01095c26e1c997bf4c6e9c94110ef9f723c2.png)
列设置设置显示后,在付款申请单源单信息就会显示该字段
