# 如何维护零售商品的商品类别?

> 产品：金蝶云·星辰 | 分类：零售资料 | 类型：常见问题

【问题描述】

如何维护零售商品的商品类别?

【操作步骤】

零售商品的商品类别是同步商品资料的商品类别。可在【基础资料】-【商品信息】,新增商品时维护。维护好商品类别点击保存。具体新增商品类别的知识可点击:[商品如何进行分类管理(如原材料/成品)?](https://vip.kingdee.com/knowledge/534899794523831040?smartSearchResId=626881440311164672&productLineId=35&isKnowledge=2&lang=zh-CN)
