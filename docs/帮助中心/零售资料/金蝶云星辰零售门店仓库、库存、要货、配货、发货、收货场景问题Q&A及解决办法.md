# 金蝶云星辰零售门店仓库、库存、要货、配货、发货、收货场景问题Q&A及解决办法

问题描述：

金蝶云星辰零售门店仓库、库存、要货、配货、发货、收货场景问题Q&A及解决办法

解决方案：

1、Q：不同的门店可以共用一个仓库吗？

A：不可以，一个仓库只能对应一个门店。

2、Q：门店支持异价调拨吗？

A：店间调拨暂时不支持异价调拨。门店向总部要货，总部有异价调拨策略。

3、Q：POS进行门店要货时，调拨类型都是默认异价，但是我的商品全都是同价调拨，有没有什么方法可以快捷设置？

A：前往后台【门店管理-价格-门店配送价】进行门店要货配送策略设置。若你的商品全部都是同价调拨，则创建同价策略即可；若你的商品有部分是同价策略，有部分是异价策略，则先创建一个同价策略，再创建一个异价策略，将那些异价的商品放到异价策略中即可。以上两种方式，均需要在POS端点击云同步，这样在进行门店要货时，就会根据后台配置的同价/异价策略展示了。

4、Q：门店可以直接对供应商采购的商品进行收货吗？

A：进入POS【店务】，选择【门店采购收货】可以对门店采购的商品进行收货操作。

5、即时库存与可用库存是什么意思，有什么区别？

A： 即时库存是当前商品在某个仓库的全部库存，即时库存=可用库存+占用库存（占用库存包括调拨出入库/采购入库/销售出库等各种在途库存+占用库存）；可用库存是当前商品在某个仓库的实时库存，即当前可售库存，不包括占用库存。两者的本质区别在于是否包括占用库存。

6、Q：门店的仓位出库是根据商品资料里的默认仓位进行出库吗？

A：是的。商品资料可以设置默认仓库和默认仓位，若该商品在默认仓库只有一个仓位，则从该默认仓位出库；若该商品在默认仓库有多个仓位，则按库存数量多的仓位先出库；若每个仓位的库存数量一样，则随机从任意仓位出库。
