# 金蝶云·星辰 - 门店收款方式操作指南

> 产品：金蝶云·星辰 | 分类：零售资料 | 类型：功能介绍, 快速入门
> 作者：YoujunYUE | 阅读：3013 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/158155499920147968)

**一. 功能简介**

维护各门店的结算方式与账户的绑定关系、支付类型、自助收银是否显示。

![](https://vip.kingdee.com/download/0100562b0fec6c2544a7afa7fbc903e16b22.png)

其中包含两类设置：

（1）通用收款设置：所有门店统一使用相关的收款方式；

（2）门店收款设置：如果门店之间存在收款方式差异或排序差异，可独立设置。

![](https://vip.kingdee.com/download/0100faf6cdcd4de74233a60eff62f2ed008c.png)

*在进行操作，请先确保以下设置已完成：

- 结算方式和账户管理已完成基础设置（操作路径：金蝶云·星辰后台-【基础资料】-【财务资料】-【结算方式】/【账户管理】![](https://vip.kingdee.com/download/0100909d6117bf244b4cb60210b26521bf1e.png)

**二. 门店收款方式操作步骤**

第一步：在门店收款方式页面，将鼠标悬停在某一条现有的门店收款，点击【+】按钮；

![](https://vip.kingdee.com/download/01006d189ed978264e09856264e1a0bedaf6.png)

第二步：选择需添加的结算方式、账户、币别、支付类型以及顺序号（当存在多种结算方式时）；

![](https://vip.kingdee.com/download/01005bacef41b3f44548ae6f0212c7dd6ff0.png)

第三步：当所有信息都录入完成后，点击右上角【保存】按钮

![](https://vip.kingdee.com/download/0100feb21b62c4194f6e9a1cd43060281415.png)

- 对应账户：选择正确的对应账户，务必慎重选择，避免后续资余额记录有差异；
- 在线支付方式：系统内置6种在线支付方式，仅在开通在线支付的情况下需要选择，如果没有开通支付，此列信息务必为空，否则开单会校验支付通道。不维护则作为系统记账；
- 支付类型：现在的支付类型包括9种，扫码类选择“在线扫码”，刷卡类选择“银行卡刷卡”，会员卡方式选择“会员储值卡”，现金收款选择“现金”，其他收款方式选择“其他”；
- 是否自助：对于使用自助收银机的情况，需要开启是否自助，自助收银界面根据自助标志显示对应的扫码及会员卡收款方式；
- 新增的收款方式默认开启。
