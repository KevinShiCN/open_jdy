# 金蝶云 · 星辰 零售第三方挂账操作指引

> 产品：金蝶云·星辰 | 分类：零售资料 | 类型：快速入门, 功能介绍
> 作者：商贸韩佩奇 | 阅读：3478 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/242334006509844224)

应用场景：商家专柜入驻商场，商场待收银，门店开单记录出库商品并计为商城欠款。商场根据返款时间跟商家结算，商家在星辰系统中做收款单核销欠款的零售单及零售结存单。
