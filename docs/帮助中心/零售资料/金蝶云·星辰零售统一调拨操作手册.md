# 金蝶云·星辰零售统一调拨操作手册

> 产品：金蝶云·星辰 | 分类：零售资料 | 类型：功能介绍, 快速入门
> 作者：商贸韩佩奇 | 阅读：1684 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/267016874171281152)

1． **功能介绍**

统一调拨适用于门店货品需要大调的业务场景，支持在一张统一调拨单上完成多个门店之间的货品调换的决策和调拨指令，各门店在POS端进行发货和收货即可。为了帮助总部做出更好的调拨决策以提高库存周转率，统一调拨单上提供调入调出门店商品的库存、销量、最近销售日期以及畅滞销的信息来辅助决策。

2． **功能流程**

![](https://vip.kingdee.com/download/01004bb9e69dff4d4550ab065dc05421022f.png)

3． **操作指引**

1. 在金蝶云·星辰后台-【门店管理】-【要货配货】下，进入【统一调拨单】

![上传图片](https://vip.kingdee.com/download/01006d8b5be894be474387dc1ff2a549c979.png)

2. 进入统一调拨单列表，点击右上角【新增】按钮，新增一张统一调拨单，录入需要进行调拨的门店信息和商品信息，并点击【保存】，进行【审核】。

![上传图片](https://vip.kingdee.com/download/010027662e9daea54c10b5a90611ebb84172.png)

注：

（1） 销量字段的数据是一个月内的总销量

（2） 门店和商品的信息支持引入

3. 审核成功之后，总部可以在后台看到各门店的调拨情况。进入统一调拨单列表，点击右上角的【统一调拨发货单】

![](https://vip.kingdee.com/download/010030206e0cfba64fb9bbcc79c351e0d581.png)

![](https://vip.kingdee.com/download/0100f5c5e9081c3949088ae6367107202a52.png)

4. 当统一调拨单审核后，调出门店的POS会收到调出的指令。以第二条分录为例，零售2店会收到调出商品可口可乐5件的指令（在POS的配送台-调拨发货处查看）

![](https://vip.kingdee.com/download/010091e040b1ffd849e8b655080a41f33d9f.png)

![](https://vip.kingdee.com/download/0100f3e18944eb644456ad0baa2dd3ae07b0.png)

5. 调出门店进行发货后，可以在统一调拨发货单看到具体的发货情况

![](https://vip.kingdee.com/download/0100ce17bc83200646bea91133f904a4ba33.png)
