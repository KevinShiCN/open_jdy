# 金蝶云·星辰 - 零售多账套初始化说明

> 产品：金蝶云·星辰 | 分类：零售资料 | 类型：整体介绍, 进阶技巧
> 作者：YoujunYUE | 阅读：1629 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/157062736189712128)

当购买的一个服务里有多个账套，且子模块包含零售云和门店收银时，则每个账套都会包含零售模块。由于门店收银POS数的购买是独立的，所以需要汇总所有账套的门店收银POS的需求数，购买后到收银机管理进行各账套POS数授权。如果后续新增POS数也需要对账套进行分配。

**操作指引：**

（1）打开菜单：【门店管理】—【设置】—【收银机列表】

![](https://vip.kingdee.com/download/010048b9b84ea79049aea1f6ebc5872ed7b9.png)

（2）点击【分配收银机】按钮

![](https://vip.kingdee.com/download/0100085099f599784b8187bbb49719a81b0d.png)

（3）对各账套进行收银机数分配，已授权数+已分配数不允许超出购买数。

![](https://vip.kingdee.com/download/0100d34ca2cc1d3a4d2fb093b50482a138be.png)
