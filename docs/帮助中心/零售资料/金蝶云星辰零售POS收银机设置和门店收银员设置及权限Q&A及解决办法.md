# 金蝶云星辰零售POS收银机设置和门店收银员设置及权限Q&A及解决办法

问题描述：

金蝶云星辰零售POS收银机设置和门店收银员设置及权限Q&A及解决办法

解决方案：

1、Q：客户只有一家门店，购买了一个POS，可以支持同时登录多台设备吗？

A：不支持。一个POS数只能登录一台设备。能否登录多台设备与收银员账号无关，只与POS数量有关。只买了一个POS数，就只能在一台设备登录，买了100个POS，就可以在100台设备上登录。

2、Q：登录POS时总是提示POS未授权，在后台收银机列表又找不到这个设备，是什么问题呢？

A：勾选【未授权】，将未授权的收银机过滤出来，进行授权即可。

3、Q：登录POS时提示POS端尚未编号，是什么问题呢？

A：检查一下是否正确选择门店登录，登录成功后系统会自动分配一个POS收银机编号。

4、Q：收银机的站点数是按每个账套分开计算，还是所有账套总数计算的？

A：按所有账套总数计算，购买后再分配到各账套。如果是分开多个服务，则分别计算。

5、Q：购买5个POS站点，我可以每个账套都在5台收银机吗？

A：不可以，5个POS站点要先进行分配。

6、Q：新购买的账套，后台收银机列表怎么没有数据，要怎么操作才能显示出来？

A：下载POS安装包后，使用收银员账号登录POS，会自动授权并在后台收银机列表显示。授权后若取消授权，则此收银员账号再次登录会提示“未授权”，此时前往后台收银机列表重新授权即可。

7、Q：门店不想给收银员改价和折扣权限怎么设置

A：到后台门店管理——设置——收银员管理设置权限，其他收银员操作权限都在这里设置
