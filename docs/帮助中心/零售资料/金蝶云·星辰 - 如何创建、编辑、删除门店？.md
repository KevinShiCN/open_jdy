# 金蝶云·星辰 - 如何创建、编辑、删除门店？

> 产品：金蝶云·星辰 | 分类：零售资料 | 类型：功能介绍, 快速入门
> 作者：YoujunYUE | 阅读：1643 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/157077480845451008)

**一. 如何创建门店？**

（1）进入金蝶云·星辰后台，点击【门店管理】-【资料】-【门店资料】

![](https://vip.kingdee.com/download/010091f05d0175ad4e87b118c0dfec4fe2e8.png)

（2）门店资料列表页，点击右上角【新增】按钮，进入门店资料新增页面

![](https://vip.kingdee.com/download/0100da24363111f94166bdea2fbb86f5e3f5.png)

（3）录入门店基本信息后，点击【保存】即可。门店创建成功后即为启用状态。

![](https://vip.kingdee.com/download/0100cffbc2c441c04944b5668bf3e8212644.png)

**补充说明：**

（1）门店经营方式支持三种：自营、加盟、联营，仅作为门店经营方式标识，暂不支持更多业务流程

（2）门店需要指定一个对应的仓库，门店产生零售单据后，该仓库不能修改。

（3）支持设置门店标签，可同时贴多个。门店标签可在【基础资料】-【商品资料】-【标签中心】进行配置。

![](https://vip.kingdee.com/download/010010f52b48b93040c9b482fa369c0f0558.png)

**二. 如何编辑门店？**

(1)门店资料列表页，点击待编辑门店的编码，页面跳转至该门店的详情页，录入修改信息后保存即可。

![](https://vip.kingdee.com/download/0100ef271926b8f649508730210937409120.png)

**三.如何删除门店？**

(1)门店资料列表页，勾选待删除门店的复选框，支持多选，点击【删除】即可。

![](https://vip.kingdee.com/download/0100b86cf4fb0de34fd2a436a46e59505db2.png)
