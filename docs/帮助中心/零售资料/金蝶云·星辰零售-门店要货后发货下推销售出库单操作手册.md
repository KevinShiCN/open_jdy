# 金蝶云·星辰零售-门店要货后发货下推销售出库单操作手册

> 产品：金蝶云·星辰 | 分类：零售资料 | 类型：快速入门, 功能介绍
> 作者：商贸韩佩奇 | 阅读：3427 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/267017125661578496)

摘要：

1. 后台设置加盟门店，务必要维护商品到这些门店的配送价，下推销售出库单时销售价格等于配送价；

2. 门店POS发起要货；

3. 后台审核并下推销售出库单；

4. 门店通过POS的采购收货或其他入库，引入单据进行收货。只有经营方式为加盟的门店，POS才会有引入发货单的按钮。

一、功能亮点：

在加盟模式下，门店要货与总部间的异价调拨是买卖关系。当总部收到要货单后，进行配货发货。发货确认后，自动在进销存后台生成并下推一张销售出库单，生成凭证可进行总部与门店之间的资金结算。（已实现）

下推销售出库单时，关联客户账户赊销信用余额，查询信用余额是否充足；当充足情况才下推销售出库单，余额不足则提醒还款或增加额度。（后期规划功能）

三、操作指引：

（1）后台设置

1.设置加盟门店：在【门店资料】里，将经营方式设置为“加盟”，维护好仓库和部门，在关联客户中选择对应门店的客户。一个门店可以关联一个客户,一个客户可以关联多个门店；
关联客户可修改，修改后所有涉及到门店关联客户字段的单据以最新的为准；

![](https://vip.kingdee.com/download/01002032b47c6248451e955c5772df109bc9.png)，

![](https://vip.kingdee.com/download/01007cc4a49f563c47db96fe1a4d673ac99e.png)

2. 设置商品门店配送价：在【门店配送价】把需要调拨的商品维护好配送价，选择“异价”调拨，填写参考配送价。下推销售出库单的时候价格取的是配送价。

![](https://vip.kingdee.com/download/0100ab0428855bbb4aa39ee44d70532a0037.png)

![](https://vip.kingdee.com/download/01001c8894d1faf942238a385930b4ea8a16.png)

（2）门店POS发起要货

![](https://vip.kingdee.com/download/0100867b831aaeff4e26831ba8cd9e3fb80a.png)

![](https://vip.kingdee.com/download/01002531fb4559434b62bd03ad7c88dc56b4.png)

（3）后台审核并下推销售出库单

1. 后台【要货单】查看POS提交的要货单

![](https://vip.kingdee.com/download/01006dd99e8d41ad4b1d9cfdd27fd3e47e22.png)

2. 在配货单处，新增配货单，并引入要货单，维护配货数量并审核

![](https://vip.kingdee.com/download/0100e5365cbf787b446c9d6a5e2f0ed086e8.png)

3.审核配货单及对应的发货单，加盟门店的发货单会自动下推成为销售出库单

![](https://vip.kingdee.com/download/010081868f963ba049f097e2ee35d237e061.png)

4.在【销售管理】查看自动生成并审核的【销售出库单】是否存在，若不存在，可以反审核发货单再审核，重新下推。
