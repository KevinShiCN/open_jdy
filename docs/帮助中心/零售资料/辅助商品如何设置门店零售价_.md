# 辅助商品如何设置门店零售价?

> 产品：金蝶云·星辰 | 分类：零售资料 | 类型：常见问题
> 作者：sunwendy | 阅读：122 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/566065246998201344)

【业务场景】辅助商品如何设置门店零售价?

【解决方案】如果商品的某一个或多个辅助属性价格与主商品价格不一致,可选择所需要的辅助属性把商品添加进来设置商品的属性价格。

如:商品芍药不同颜色价格不一样,勾选“芍药”再点击【添加辅助属性商品】,选择对应的属性点击确定后,即可设置芍药不同颜色的价格。
