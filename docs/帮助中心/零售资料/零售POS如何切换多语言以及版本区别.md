# 零售POS如何切换多语言以及版本区别

> 产品：金蝶云·星辰 | 分类：零售资料 | 类型：快速入门, 精华汇总, 功能介绍

## **1 简介  **

### **1.1 功能介绍**

POS支持切换简体中文,English,繁体中文三个语言，以满足收银员本地化作业的需求

### **1.2 适用版本**

星辰3.0以上标准版/专业版/旗舰版/所有版本

## **2 版本对比**

| 应用场景 | 多语言版（需加购星辰多语言服务） | 简体中文版 |
| --- | --- | --- |
| 支持版本 | 标准版、专业版、旗舰版 | 标准版、专业版、旗舰版 |
| 语言 | 简体中文、繁体中文、英文 | 简体中文 |
| POS开单店务 | 全量支持 | 全量支持 |
| POS打印及设备 | 需适配本地硬件 | 全量支持 |
| 收银开单秤重设备 | 需适配本地硬件，建议win收银机 | 全量支持 |
| 在线支付 | 不支持 | 金蝶支付、通联支付、银联商务、小程序直连、生态开放接口 |
| 轻餐管理 | 不支持扫码点餐，仅支持POS点餐 | 全量支持 |
| 门店商城小程序 | 不支持 | 全量支持 |
| 微信会员卡 | 不支持 | 全量支持 |
| 多币别 | 跟随本位币 | 跟随本位币 |
| 多时区 | 支持 | 不支持 |
