**1.3 银联商务支付（仅支持门店POS扫微信/支付宝/云闪付支付）**

为满足商户需求，金蝶云·星辰目前仅对接银联商务支付接口（通过银联商务深圳分公司接入），暂未与银联商务支付有官方商务合作。

由于银联商务支付跨省使用（深圳市以外的所有省市）需要银联商务深圳分公司授权，因此需要**开户当地（商户号开户）银联商务分公司找银联商务深圳分公司做授权**，次授权金蝶无法干预。

授权过程需要用到金蝶云星辰appid，将以下信息发给银联商务作为授权使用

星辰appid：8a81c1bf6d4afcb6016f16bd8f0f08c8

**1.5 外部对接支付（仅支持门店POS扫微信/支付宝/云闪付支付）**

外部对接支付是指金蝶云星辰零售云将支付通道接口开放，**第三方开发者可自行对接**非金蝶支付和通联支付以外的**其他聚合支付通道及银行支付通道**，实现使用微信/支付宝/云闪付支付。开发对接完自行部署服务器，自行做数据接口维护。

该方案核心优势是，当商户经营流水较大时，可与银行建立深度合作关系，银行将给与**商户费率补贴**，甚至是**0费率**；金蝶服务商（伙伴）也可与当地银行建立**共同拓客**的合作关系，深度经营当地零售客户。

具体的开发对接文档，在购买金蝶云星辰产品后，请通过金蝶售后技术支持获取。如您是金蝶服务商（伙伴），也可通过各分公司渠道经理和云之家【星辰伙伴学习讨论群】咨询获得。



**1.6 一卡通支付（支持门店POS扫码一码通二维码/刷脸/IC卡支付+门店商城一卡通余额支付）**

一卡通是指种集多种功能于一体的智能储值卡，主要应用于企业、园区和学校等场景消费。

星辰一卡通支付是指金蝶云星辰零售云将支付通道接口开放，第**三方开发者可自行对接**任何一家**一卡通品牌**，实现企业、园区和学校的零售店，使用金蝶云星辰POS以后，也能支持到使用原有的一卡通储值系统进行消费结算。开发对接完自行部署服务器，自行做数据接口维护。

该方案核心优势是，当企业、园区和学校消费场景较复杂时，需要食堂、停车、零售店等多个业务都接入一套储值系统，方便企业财务管理。金蝶云星辰开放性的接口，可接灵活自主接入任何一家一卡通品牌，极大**满足商户的多平台集成需求**。金蝶服务商（伙伴）也可各大一卡通品牌建立**共同拓客**的合作关系，深度经营**企业、园区和学校**等场景的零售客户。

具体的开发对接文档，在购买金蝶云星辰产品后，请通过金蝶售后技术支持获取。如您是金蝶服务商（伙伴），也可通过各分公司渠道经理和云之家【星辰伙伴学习讨论群】咨询获得。
