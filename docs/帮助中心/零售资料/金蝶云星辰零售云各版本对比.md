# 金蝶云星辰零售云各版本对比

> 产品：金蝶云·星辰 | 分类：零售资料 | 类型：应用案例, 新特性, 快速入门

标准版：主要适合单店商户使用，不支持配送中心，不支持门店要货调拨场景，无业绩统计，无门店商城。按门店数量销售，最多不超过5门店，不支持购买移动POS

专业版：拥有全部功能，支持多结算客户及供应商，与旗舰版无差异

旗舰版：拥有全部功能，支持多结算客户及供应商，与专业版无差异
