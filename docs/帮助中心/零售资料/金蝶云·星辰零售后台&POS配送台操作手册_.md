### 二、 **操作说明**

1、**后台**

第1步：进入后台【门店管理-资料-商品配送区域】，创建商品与配送区域的关联关系。

![](https://vip.kingdee.com/download/0100e383532ba7df4014981a310fa16932c6.png)![](https://vip.kingdee.com/download/010032d122adb5f84db4b354aabd3ef0f30a.png)



创建商品与配送区域的关联关系时：

1. 创建配送区域名称

2. 是否启用配送台：某个区域只有未启用配送台时，POS才需要配置该区域的打印机，否则将调配送台本身连接的打印机自动打印

3. 是否打印即配送：勾选打印即配送时，发货单打印之后即自动将状态置为发货完成（仅对店内配送生效）

4. 选择适用门店：选择该配送区域适用的门店

5. 选择适用商品：选择该配送区域适用的商品





第2步：进入POS【设置-其他参数设置】，开启“自提配送”

![](https://vip.kingdee.com/download/0100c2f6010d7b1d43229ac2501c26b9d68f.png)



第3步：进入POS【设置-打印设置】，找到“区域打印机设置”，设置好区域打印机；同时找到“销售开单业务”，设置好发货单默认打印方式和多联单打印模板。

![](https://vip.kingdee.com/download/010025a1599c6c924a8fb5b1d34c55ac2b69.png) ![](https://vip.kingdee.com/download/01001a32766957eb47f6b7c94b4cbcff4bfd.png)





第4步：进入POS【配送台】，开启主配送台

 ![](https://vip.kingdee.com/download/01097176bafde5524d5b874b4521d21b918e.png)



第5步：进入POS【零售开单】，选择商品开单，结算页开启“自提配送”选择自提或店内配送或配送到家，并填写提货人或收货人信息，最后完成结算。

 ![](https://vip.kingdee.com/download/0100d3ffae90b8e8451fbc0d47c2d667d7f9.png)

第6步：进入POS【配送台】，查看发货单并对发货单做拣货、自提/配送操作。（发货单包括门店自提、店内配送、配送到家三种）。

 ![](https://vip.kingdee.com/download/0100258e9f1f2270469d9fd2426b1499cbbc.png)

1. 左上角数据看板支持查看今日订单数与订单商品数（包括门店自提、店内配送与配送到家），点击可进入详情查看订单明细；

2. 右侧单据搜索框支持搜索单据编号或扫描提货码检索发货单；

待拣货：展示所有待拣货状态的发货单，根据区域分开展示（以下发货单展示在“其他”区域：a未启用配送台的区域的发货单；b未分配配送台的配送区域的发货单；c未与任何配送区域关联的商品）

3. 拣货中：展示所有拣货中状态的发货单；

4. 待配送：展示所有待自提或待配送状态的发货单。



第7步：进入POS【配送台】，点击左上角数据看板进入详情查看所有发货单列表，或者点击单个发货单进入处理。下面以“门店自提”举例说明，店内配送/配送到家类似，不赘述。

 ![](https://vip.kingdee.com/download/0100487cca2d14724f7aac914ac7eedff32c.png)

![](https://vip.kingdee.com/download/0100fc626cbcbfa94dbf8ca52a2a0b9a96e1.png) ![](https://vip.kingdee.com/download/0100c8ac79986b65412399af785eaa893e03.png)

![](https://vip.kingdee.com/download/0100b8ecbe4bfbeb48a8998dd2522a59743b.png)

![](https://vip.kingdee.com/download/0100bbb7e719e2b044c79ccc9eb7d7dd14b5.png)



1. 默认展示所有发货单数据，支持点击右上角“过滤”筛选不同时间的发货单；

2. 点击单个发货单可以查看单据详情；

3. 商品条码/序列号搜索框：支持搜索或扫描商品条码/序列号维护条码/序列号；

4. 暂存：维护条码/序列号、发货数量后可以暂存单据；

5. 处理：点击可弹出弹层对发货单做拣货、提货或配送处理；

6. 打印：支持打印小票/多联单；

7. 自提单完成自提时需要维护自提码，配送到家单完成配送时需要维护配送人与订单编号。



关于自动打印小票：以下四种场景均会自动打印小票

1. 启用配送台的打印机（与配送台连接的打印机自动打印）；

2. 已启用主配送台，未启用配送台的其他区域打印机（主配送台拉取发货单，并自动调区域打印机打印）；

3. 已启用主配送台，未分配配送台的其他配送区域的区域打印机（主配送台拉取发货单，并自动调区域打印机打印）；

4. 未与任何配送区域关联的商品，推给主配送台，由主配送台调区域打印机自动打印。



以上，就是配送台目前的基本内容，后续迭代会持续更新，敬请期待。
