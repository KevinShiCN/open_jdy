# pos端如何设置/取消扫码开单的时候,将“ean13“条码去除首位为0的字符?

> 产品：金蝶云·星辰 | 分类：零售资料 | 类型：常见问题
> 作者：RJS | 阅读：11 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/786760898848199680)

【问题描述】

pos端如何设置/取消扫码开单的时候,将“EAN13“条码去除首位为0的字符?

【解决方案】

登录POS端,依次点击【设置】-【其他参数设置】,开启"EAN13"条码去除首位为零的字符 参数

![](https://vip.kingdee.com/download/01090ed2e0648ed14e13848c1b6e14f79d6c.png)
备注:该参数分别在电脑端pos和移动端pos独立设置,移动端POS操作路径基本一致,具体可参考下列截图

![](https://vip.kingdee.com/download/010937c379f2da264c3dbd27e3402d79aead.png)
