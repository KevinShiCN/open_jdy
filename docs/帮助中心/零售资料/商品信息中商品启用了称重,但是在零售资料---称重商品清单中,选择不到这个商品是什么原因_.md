# 商品信息中商品启用了称重,但是在零售资料---称重商品清单中,选择不到这个商品是什么原因?

> 产品：金蝶云·星辰 | 分类：零售资料 | 类型：常见问题
> 作者：杨桂平 | 阅读：59 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/749795896065239296)

【问题描述】

商品信息中商品启用了称重,但是在零售资料---称重商品清单中,选择不到这个商品是什么原因?

【解决方案】

称重商品单位至少一个有条形码就可以在称重商品清单里面选择到该商品,在商品资料中添加一下称重商品的条形码即可
