# 零售模块如何新增促销活动?

> 产品：金蝶云·星辰 | 分类：零售资料 | 类型：常见问题
> 作者：财税李炳全 | 阅读：639 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/470032265901171456)

【问题描述】

零售模块如何新增促销活动?

【操作步骤】

**一、新增方案**

点击【门店营销】-【零售促销】

![](https://vip.kingdee.com/download/0109dd567b312626494e9335f7611caad872.png)

点击右上角【新增】按钮
![](https://vip.kingdee.com/download/010986801ec17ece4cf9969b8fca5338b7ec.png)
选中要设置的活动类型,比如:指定商品特价
![](https://vip.kingdee.com/download/0109f574dd7021964c7395768462bd93c182.png)
录入活动名称、开始结束时间等,新增商品,录入特价金额,如果需要设置限量也可以录入限量的数量

![](https://vip.kingdee.com/download/01098a05c8ae018148ba8baea7b6d88e559b.png)

可以设置适用对象:适用渠道、适用门店、适用会员
![](https://vip.kingdee.com/download/0109570fa0ad3acc436c88238c133704fb7b.png)若设置限量,还可以设置启用限量的门店及会员

![](https://vip.kingdee.com/download/0109785310391a84410fbe863eba509fb600.png)
活动设置好之后点击右上角【保存】按钮来保存活动方案

![](https://vip.kingdee.com/download/0109754ef1fe413c43d2a584dd69410827d1.png)

**二、pos应用**

在pos端开单选到对应商品时,当满足适用条件,系统将执行促销方案
![](https://vip.kingdee.com/download/01093658e6ba12f8439fafb7f440d081bf7c.png)
点击【选择促销方案】

![](https://vip.kingdee.com/download/0109eec32542c9d5413fb6a6182785bec73b.png)
可以看到执行的是哪个方案,若有多个方案,也可以自行选择;当需要取消方案执行时,可以选择“不使用促销”

![](https://vip.kingdee.com/download/01098edacef8cf6c4056927c7c107daffef1.png)
