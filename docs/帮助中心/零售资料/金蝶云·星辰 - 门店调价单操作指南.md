# 金蝶云·星辰 - 门店调价单操作指南

> 产品：金蝶云·星辰 | 分类：零售资料 | 类型：快速入门, 功能介绍
> 作者：YoujunYUE | 阅读：1486 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/157551499223542016)

**一. 功能简介**

门店调价单功能可以基于不同的价格维度，对单个或多个门店的商品的价格进行调整，商家在此模块里也可以看到所有的调价记录。

**二. 操作指南**

第一步：进入金蝶云·星辰后台，点击【门店资料】-【价格】-【门店调价单】

![](https://vip.kingdee.com/download/0100b6a880751eb244749319954ab3b7ceda.png)

第二步：点击右上角【新增】按钮

![](https://vip.kingdee.com/download/0100c1798ff377cd467a919fd17dfc05b775.png)

第三步：录入门店调价单的基本信息以及需要调价的商品信息

![](https://vip.kingdee.com/download/0100e449f2254f7c479b9dca81f7bb87ff78.png)

第四步：调价单信息输入完成后，点击右上角【保存】按钮

![](https://vip.kingdee.com/download/0100cfda5dc7dbcd4513a0ecbb8a42f8501c.png)

第五步：新增门店调价单保存后，门店调价单在列表显示生效状态为“未生效”，单据状态为“待审核”。

![](https://vip.kingdee.com/download/0100d6b8bf2b39fe42089753915f222000f3.png)

在单据状态为“待审核”的调价单中，选择需要审核的调价单，点击“审核”按钮，调价单生效状态变更为“生效中”，单据状态变更为“已审核”，门店POS优先获取最后修改的生效中的调价单显示商品价格。

![](https://vip.kingdee.com/download/010071fedb59370745a9b7e07987df916248.png)
