# 微信小程序申请与认证操作指南

> 产品：金蝶云·星辰 | 分类：零售资料 | 类型：快速入门, 功能介绍, 新特性
> 作者：商贸韩佩奇 | 阅读：18342 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/359013549794315776)

**一、前言**

本文档详细介绍了微信小程序账号注册开通、填写小程序信息、小程序微信认证、完善小程序信息的流程。

**二、流程指引**

1. 微信小程序账号注册开通

2. 填写小程序信息

3. 小程序微信认证

4. 完善小程序信息

**三、操作说明**

1. **微信小程序账号注册开通**

• 方式一：单独申请小程序（路径：打开微信公众平台 - 注册 - 选择「小程序」按步骤申请小程序并完成[点击链接查看](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fkf.qq.com%2Ffaq%2F170109iQBJ3Q170109JbQfiu.html)。）

• 方式二：若你已经注册并认证了公众号，可复用公众号主体资质申请小程序（路径：打开微信公众平台 - 登录公众号 - 广告与服务 - 小程序管理 - 添加小程序 - 选择「快速注册并认证小程序」按步骤申请小程序，详情请[微信认证](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fdevelopers.weixin.qq.com%2Fminiprogram%2Fproduct%2Frenzheng.html)，以获取相应的小程序权限，认证成功才表示真正的注册完成。下图是已认证的状态，未认证时会提示需要认证。*必须步骤。需要完成以下操作并提供资料：

| 名称 | 说明 |
| --- | --- |
| 签署《微信公众平台认证服务协议》 | 签署《微信公众平台认证服务协议》，勾选「同意」，点击「下一步」 |
| 认证主体类型 | 选择认证主体类型 |
| 认证材料 | 1. 企业名称2. 组织机构代码 / 统一社会信用代码3. 工商执照注册号 |
| 填写发票信息 | 1. 发票类型2. 纳税识别号3. 企业注册地址4. 企业电话5. 企业的开户银行 |
| 支付微信认证费用 | 支付 300 元认证费用，目前仅支持微信支付 |

4. **完善小程序信息**

![](https://vip.kingdee.com/download/0100b8e47d882b6f496d8b8f49b01761d276.png)![](https://vip.kingdee.com/download/010086827dc76ed7468588601ce6f55d529b.png)

![](https://vip.kingdee.com/download/0100c1aefa57ee6c407bb7cc33d22179ea79.png)

注册成功后，登录小程序后台，点击《首页》查看小程序发布流程，找到「小程序信息」，点击【完善信息】进入小程序信息填写页面，完善小程序名称、简称、头像、介绍、服务类目等基本信息后，保存即可。下图是已完善基本信息的状态，未完善信息时会提示需要完善信息。*必须步骤。需要完成以下操作并提供资料：

| 名称 | 说明 |
| --- | --- |
| 小程序名称 | 按照小程序要求和你个人实际情况，填写小程序名称 |
| 小程序简称 | 填写小程序简称，会体现在微信小程序搜索栏 |
| 小程序头像 | 按照小程序要求和你个人实际情况，上传小程序头像 |
| 介绍 | 按照你的小程序应用场景，填写介绍内容 |
| 服务类目 | 按照你的小程序经营的行业，务必准确选择服务类目，包括主营类目与一级类目• *注意：￮ 主营类目有且只有 1 个，选择即生效，不可删除￮ 主营类目每年有 5 次修改机会，修改即生效，请谨慎修改￮ 完成主营类目设置后才可提交代码审核￮ 主营类目选择错误可能影响代码审核结果，请正确选择点击这里查看官方说明 |
| 主营类目：是小程序内实际运营的主要服务类型，可选主营类目基于已通过的服务类目生成，开发者需结合小程序主要运营服务内容，正确选择主营类目。2022 年 6 月 8 日后，未设置主营类目的帐号将无法提交代码审核。 |  |
