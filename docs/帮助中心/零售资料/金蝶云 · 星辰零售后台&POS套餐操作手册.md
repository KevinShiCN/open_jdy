二、功能概述

商家可以将多种商品包装在一起作为一个组合套餐进行打包销售，以低于单品单独售卖的价格销售，通过让利一部分给顾客，刺激顾客购买，提升销售；也能帮助商家处理积压商品，消耗库存。优惠套餐就可以解决这个业务问题。

四、操作说明

1、 后台

1.%2 创建套餐

![](https://vip.kingdee.com/download/010098153f74110944a8944964c2237ca57a.png)

![](https://vip.kingdee.com/download/010063d232e97fc54229856757f6d7f2fbc5.png)

![](https://vip.kingdee.com/download/0100716be6a2227548938f8d0fbc061be691.png)

第1步：登录后台，前往【营销管理-促销-套餐】，点击【新增】创建套餐：

• 维护套餐名称和套餐价

• 添加商品组，选择“固定搭配”或“随心搭配”

• 填写商品组名称

• 选择套餐子件商品

• 若是随心搭配，可以设置可选商品种类、套餐加价、默认选中、必选、最多可选数量等

• 创建套餐后，系统会在商品资料自动创建一个相同名称的对应套餐的服务商品

1.2 对套餐进行管理

![](https://vip.kingdee.com/download/0100c90d1c92099c4dcd863da4a46e45e648.png)

第2步：前往【营销管理-促销-套餐】，可以查看套餐列表，并对套餐进行管理：

• 可以按照不同条件或维度搜索筛选套餐

• 点击套餐编号可以进入编辑（注：已产生零售单的套餐不支持编辑）

• 可以选择套餐做启用/禁用操作

• 可以选择套餐做删除操作（注：已产生零售单的套餐不支持删除）

• 可以选择套餐做关联门店上架/下架操作（注：也可以到【门店管理-资料-商品上下架】做上下架操作，两边状态会相互同步）

1.3 对套餐做商品上下架操作

![](https://vip.kingdee.com/download/01000da8abb2271e46dcb8ded495733cfb63.png)

第3步：也可以前往【门店管理-资料-商品上下架】对套餐做上下架操作，两边状态会相互同步。这一步非必须，这里是为了说明可以在两个地方做套餐上下架操作。

前往【门店管理-零售单-零售单/零售结存单】，可以查看零售单（套餐）及成本数据。具体表现为：

• 零售单详情可以查看套餐商品，点击套餐编号可以查看套餐明细商品（即套餐子件）

• 结存后在零售单点击【刷新成本】后，可以在零售单或零售结存单查看套餐成本数据

• 提交零售单按套餐提交，零售单结存根据套餐子件结存，而不是套餐

前往【销售管理-销售报表-销售明细表/销售汇总表/销售排行表】，可以查看套餐子件商品销售数据。具体表现为：

• 因为进销存的销售数据是根据结存单统计，而结存单是根据套餐子件结存，所以在销售明细表/销售汇总表/销售排行表查看销售数据时只能统计套餐子件商品的数据，无法统计套餐商品销售数据

• 因进销存统计批发和零售数据，批零一体会有差异，会导致套餐子件商品销售统计数据略有差异，建议以零售销售报表为准

2、 POS

2.1 套餐开单

![](https://vip.kingdee.com/download/0100b182351df0d14d3082bd637040701838.png)

![](https://vip.kingdee.com/download/0100920813ccf64e4b09be28caac5e375663.png)

第4步：进入POS，前往【零售开单】，点击下方【套餐商品】，选择顾客想要购买的套餐商品开单。具体表现为：

• 固定搭配无需选择套餐子件，直接加购即可；随心搭配需要选择顾客想要购买的套餐子件和数量

• 加购后，套餐及套餐子件均会展示在左侧列表

2.2 套餐开单记录

![](https://vip.kingdee.com/download/0100fbde4f952eb249d68c360e66fdee6425.png)

![](https://vip.kingdee.com/download/0100f1436e0107dd45018d76bbdfb4405613.png)

![](https://vip.kingdee.com/download/0100e5d923bcef454c398821577d84fb3379.png)

第5步：开单成功后，打开【开单记录】，可以查看零售单（套餐）记录，具体表现为：

• 下单后，在开单记录中可以查看套餐及套餐子件

• 套餐只支持源单退货，且只允许退套餐，不允许单独退套餐子件
