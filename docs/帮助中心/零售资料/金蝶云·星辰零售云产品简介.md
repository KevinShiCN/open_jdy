# 金蝶云·星辰零售云产品简介

> 产品：金蝶云·星辰 | 分类：零售资料 | 类型：快速入门, 整体介绍, 精华汇总

### 一、产品介绍

金蝶云·星辰零售是金蝶集团面向小微商贸企业推出的一款商业信息化、门店数字化、业财一体化和线上线下一体化的零售管理Saas软件 ，包含进销存管理、门店管理、会员管理、门店营销、门店POS收银和门店商城小程序等模块。支持自助收银、离线扫码支付、智能补货、会员画像、精准营销、AI赋能、一键生成财务凭证、全面支持数电票等业务场景，助力商贸企业向科技零售转型，为终端消费者带来全新的购物体验。

### 二、适用行业及场景

主要应用角色包含企业老板、财务、库管、运营、店长、收银员、团购团长、配送员等。

主要应用场所为市场、临街商铺、写字楼、园区、学校、医院、景区、展会中心、商场等。

主要支持的经营模式包含直营、加盟、联营、代销、合伙人及社区团购模式。

主要适用细分行业包含生鲜水果、百货便利、景区文旅、卤味熟食、烘焙汉堡、酒水批零、3C智能、玩具运动、轻奢饰品、家居建材、文玩手信、休闲餐饮。

### 三、关键术语

交班

收银员在每个班次收银工作完毕需进行一次交班操作，系统自动统计该收银员的收款及商品销售情况，支持打印交班记录（含经营收入统计及商品明细）。交班记录后台存储，并生成收银员缴款流水，交班数据便于财务对账。

零售结存单

零售单后台创建完成自动触发零售结存单进行库存扣减，如果是会员充值结存时生成预收款单，退充值生成预收退款单等。如果商品库存不足无法立即结存，待库存入库后会定时触发再次结存，如果其他原因无法结存，在POS的开单记录或者后台的《结存任务》可以看结存异常描述。建议门店每天核查开单记录确认零售单是否正常出库，如有异常请进行排查处理。

结存任务

零售单通过结存任务触发库存扣减生成结存单，如果门店管理参数不选择零售单合并结存，则零售单后台创建成功就会触发结存，如果库存扣减条件满足便立即生成结存单，如果库存不足，且不允许负库存出库，会先执行库存占用。有库存新增时自动触发一次结存，系统也会定时触发结存。

如果由于某些异常导致零售单超过30天未完成结存出库，则需要手工触发超过30天的结存任务。

### 四、功能概述

门店管理

支持设置门店收款方式、支付通道、收银员权限、商品上下架、门店价格资料等信息；支持门店要货配货；支持零售出库，零售出库汇总、零单明细查询、门店收银日报、门店销售汇总表查询等。

会员管理

支持会员信息，会员卡信息，会员积分规则及充值赠送规则设置。支持查看会员储值积分账户，充值记录，储值流水，积分流水等信息；支持对接微信会员。

营销策略

13大场景促销、高级自定义促销、微信公众号营销、优惠券、礼品卡券等。

POS开单

收银员权限控制、改价、打折、挂单取单、备注、整单折扣、抹零、称重、交班等。

POS要货

支持按照上架商品一键生成要货单，

支持按模板要货

支持指定单位要货，支持浮动单位。

支持手工再要货建议数量上修改

支持根据要货策略要货-目前支持最大库存，最小库存，安全库存。

要货单金额限量

支持按门店设置每一个要货单的最小起订量和最大起订金额。

要货商品限量

支持按门店设置要货单中指定商品的最小起订量和最大起订量以及要货单位

手机POS要货

支持在手机POS上要货

支持边扫码边要货

下推单据

支持批量下推配货单

支持批量下推发货单

支持批量下推其他出库单，支持组装商品按子件下推。

支持批量下推采购订单（规划中）

要货汇总表

根据时分秒按照门店和商品维度汇总要货数据。

配货

配货模板配货、按库存配货、多单合并配货、分批配货

下推发货单

-   支持按门店维度自动拆成门店维度的发货单

发货（后台）

配送中心对确认好配货决策的单据进行发货操作，发货单以门店维度生成。

收货

一键收货，产生收货差异，包括店间要货及向总部要货

店间要货

可向有库存的门店发起要货申请，支持门店之间平价调拨和改价调拨

店间调拨出库

根据库存情况出库将其他门店的要货申请自动生成调拨出库单出库

盘点

多种盘点方式，多人盘点和快速盘点。在收银机POS和手机POS都可盘点

寄存

消费者可将购买的商品寄存门店

会员批量充值

会员期初的时，历史储值余额需要通过POS批量充值完成

实体礼品卡激活

IC礼品卡需要在门店激活后才可售卖，在此批量激活

打印模板设置

各类标签模板、单据打印模板设置

零售与进销存对接

库存占用、库存扣减、库存相关报表统计、成本核算、储值预收款

零售资金对接

应收账款明细统计、充值生成预收款单、会员卡支付抵扣预收款、零售结存单生成凭证等业务的处理。
