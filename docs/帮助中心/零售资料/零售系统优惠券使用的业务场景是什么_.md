# 零售系统优惠券使用的业务场景是什么?

> 产品：金蝶云·星辰 | 分类：零售资料 | 类型：常见问题
> 作者：财税李炳全 | 阅读：870 | 有用：0

【问题描述】

零售系统优惠券使用的业务场景是什么?

【解释说明】

优惠券功能,不但可以帮助商家维护老客户粉丝、提升复购率与客单价,还可以进行品牌传播,拉新粉丝;

目前优惠券分为三种类型:

(1)代金券:买家可凭券享抵扣现金服务。如:无门槛减10元、满100元减20元 ;

(2)折扣券:买家可凭券享消费折扣服务。如:无门槛打9折、满100元打9折 ;

(3)抵扣券:买家可凭券享兑换指定商品服务。如:3元啤酒抵扣券、0元面包抵扣券;

目前优惠券支持三种形式:

优惠券非实体券,是通过微信二维码分享领取的,非常适合当下互联网时代的新营销,店主可以做好优惠券后,将领券二维码打印出来或展示在副屏上,供来店顾客扫码领取,也可以分享到朋友圈或微信群里,更大范围的传播门店优惠活动

(1)电子优惠券解决方案,商家通过二维码[电子优惠券系统](javascript:void(0);)向消费者手机发送二维码电子优惠券,届时消费者只需调出手机中收到的二维码电子优惠券出示扫描,验证通过即可享受优惠等功能,整个发放领取过程无纸化低碳环保、高效便捷、安全新颖、省时省力。

简单方便快捷对商家来说:可随时制作不同价格及类型的电子券,快捷方便,节省时间及人力等成本;对客户来说:客户手机即可接收电子优惠券,消费时只需出示手机里的电子券即可。

安全二维码电子优惠券具有安全、易识别、多重加密功能,相对纸[质票](javascript:void(0);)具有更高级别的安全防伪功能。

(2)支持纸质优惠券解决方案,客户可以导出券号进行纸质优惠券的制作;

(3)支付付费优惠券解决方案,客户可以付费购买面值高额的优惠券;
