# 金蝶云星辰零售支付设置、支付通道及门店收款方式Q&A及解决办法

> 产品：金蝶云·星辰 | 分类：零售资料 | 类型：常见问题, 进阶技巧
> 作者：商贸韩佩奇 | 阅读：6700 | 有用：12

问题描述：

金蝶云星辰零售支付设置、支付通道及门店收款方式Q&A及解决办法

解决方案：

1、Q：零售目前对接了哪些支付通道？可以以个人申请吗？

A：金蝶支付、通联支付、银联商务、和微信官网直连（可找金蝶支付代开通或者从微信公众平台自助开通）。以上支付通道均不支持个人申请，支持个体工商户与企业资资质申请（必须要有营业执照）。

如有其他支付通道，如银行、第三方通道对接的需求，可自行开发对接，星辰提供标准接口文档，可通过客服渠道获取

2、Q：星辰支付通道支持实时到账吗？

A： 门店收银是实时到账的。客户提现到自己的银行账户，是T+1到银行卡

3、Q：客户有三家门店，不同门店的收款方式不同怎么设置？

A：前往后台【门店管理-资料-门店收款方式】设置。如果所有门店收款方式都不同，可以点击【门店收款设置】逐个门店进行设置；如果大部分门店收款方式相同，只是个别门店收款方式不同，则可以点击【通用收款设置】先设置通用的，再点击【门店收款设置】进行例外设置。

4、Q：后台支付通道支持一个门店配置多个支付通道吗？

A：不支持。一个门店只能配置一个支付通道，配置时没有限制，启用时会做限制。

5、Q：客户之前配置了支付通道，可以正常使用。现在又新开了一个门店，需要怎么配置支付通道呢？

A：前往【门店管理-资料-支付通道配置】选择之前已经配置的支付通道点击编辑，把新增的门店添加进来即可；或者新增一个支付通道，把原来已配置的支付通道信息填进来，配置一下即可。

6、Q：客户之前配置了支付通道，可以正常使用。现在又新开了一个门店，按照新的配置文档重新获取秘钥，会影响之前门店的支付通道使用吗？

A：不影响。

7、Q：客户配置支付方式的时候，支付方式选择“银行转账”，支付类型选择“银行卡刷卡”，用于记账使用，为什么POS结算界面不显示这个支付方式？

A：客户选择银行转账若仅用于记账使用， 支付类型要选择“其他”， 不能选择“银行卡刷卡”，否则POS结算界面不会显示支付方式。 最初没严谨控制，后来做好了限制。

8、Q：现在哪些支付通道可以对接POS刷卡？

A：银联商务和通联。

9、Q：后台设置了门店收款方式，为什么POS结算界面有些支付方式不显示，是什么原因？

A：按照以下方式排查。1）前往后台【门店管理-资料-门店收款方式】检查后台支付方式是否配置正确，按如下方式设置：

| 结算方式 | 支付类型 | 在线支付方式 |
| --- | --- | --- |
| 微信支付 | 在线扫码 | 微信 |
| 支付宝支付 | 在线扫码 | 支付宝 |
| 银联支付 | 在线扫码 | 银联二维码(云闪付、银行掌银) |
| 会员卡支付 | 会员储值卡 |  |
| 现金支付 | 现金 |  |
| 赊账 | 赊账 |  |

2）设置完成后，登录POS，点击【云同步】，进入结算界面，检查后台设置的支付方式是否全部展示；3）会员卡支付只有选择会员开单才会展示；4）会员充值业务没有会员卡支付。；5）会员卡余额不够时会提示。

10、Q：后台门店收款方式设置了“赊账”，为什么POS结算界面不显示，是什么原因？

A：按照以下步骤排查：1）前往后台【门店管理-设置-参数设置-会员营销参数】找到“赊账设置”，开启赊账，并设置授信额度；2）前往后台【会员管理-赊账-还款方案】创建还款方案；3）前往后台【会员管理-会员资料-会员信息】给会员授信并到【会员信息】确认会员是否有可用额度；4）前往后台【门店管理-资料-门店收款方式】门店收款方式设置“赊账”（注：在线支付方式不要填写）；5）POS开单时需选择会员。

11、Q：后台不同门店设置了不同的收款方式，门店收的款是放在门店自己的账户吗，POS收款一般多久可以到指定账户？

A：这个是根据门店设置的支付通道来的。看门店配置的支付通道是哪个，门店收款就是在该支付通道所对应的支付服务商账户。比如门店A配置的支付通道是通联，则收款放在通联的账户。POS收款一般 T+1 到账。
