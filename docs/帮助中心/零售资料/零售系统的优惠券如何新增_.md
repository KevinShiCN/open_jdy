# 零售系统的优惠券如何新增?

> 产品：金蝶云·星辰 | 分类：零售资料 | 类型：常见问题
> 作者：财税李炳全 | 阅读：290 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/470032274709209856)

【问题描述】

零售系统的优惠券如何新增?

【操作步骤】

1、优惠券列表

点【后台】-【门店营销】-【优惠券】,此处可以看到某个优惠券方案的优惠券数量、领取数量、优惠券状态等信息,还可快速分享优惠券,可控制优惠券的发券状态,开始发券和暂停发券;

![](https://vip.kingdee.com/download/010944023d604d1c429e85092bb58409ca3c.png)

**2、新增优惠券:代金券、折扣券、抵扣券、多次核销代金券、多次核销折扣券、随机代金券。******

优惠券类型说明如下:

(1)代金券:买家可凭券享抵扣现金服务,如:无门槛减10元 / 满100元减20元

(2)折扣券:买家可凭券享消费折扣服务,如:无门槛打9折 / 满100元打9折

(3)抵扣券:买家可凭券享兑换指定商品服务,如:3元啤酒抵扣券 / 10元面包抵扣券

多次核销代金券:买家可凭券分多次,享抵扣一定总额度现金服务,如:抵现总额度为1000的优惠劵,可多次使用

(4)多次核销折扣券:买家可凭券分多次,享抵扣一定总额度现金或消费一定总金额均打折服务,如:一张满100元打5折的折扣劵,总折扣金额800 / 一张满100元打5折的折扣劵,总消费额度1800

(5)随机代金券:买家获取金额随机的抵扣现金服务的优惠券,如:一个买家领取了一张满100元减10元,另外一个买家领取了一张满100元减20元
