# 如何给子账号设置盘点的权限?

> 作者：云社区用户5j6u671 | 阅读：247 | 有用：0

【问题描述】

如何给子账号设置盘点的权限?

【解决方案】】

系统设置-用户管理-选择子账号-授权。

![](https://vip.kingdee.com/download/0109e0341e8f67d6438b98129945f955ad55.png)

功能授权,选中盘点单,勾选对应权限保存即可。
