# 操作单据审核时，提示“当前单据操作人与制单人相同，不允许审核”该怎么处理？

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：快速入门, 常见问题
> 作者：进销存赵嘉丽 | 阅读：1469 | 有用：1

# 【问题概述】

操作单据审核时，提示“当前单据操作人与制单人相同，不允许审核”该怎么处理？

# 【原因说明】

一般出现这个提示是因为账套中设置了单据操作控制，不允许制单人和审核人为同一人导致的。需要让拥有“单据操作控制”功能权限的子账户或当前账套的管理员来进行设置。

操作路径：打开系统设置>个性化设置>单据操作控制，找到对应模块，再对各模块下的具体单据进行设置，设置完毕后点击‘保存’即可生效。

# 【解决方案】

下面以收款单为例，讲解一下具体的场景应用及操作方法。

场景：假设业务员A操作收款单时，保存单据后点击审核，弹出提示“当前单据操作人与制单人相同，不允许审核”。

解决办法：

1）业务员A需要找账套管理员确认一下单据控制的权限，如确认单据操作人与制单人可以为同一人，则让账套管理员前往[系统设置>个性化设置>单据操作控制]菜单，找到收款单。

2）打开“审核人与制单人可为同一人”，“反审核与审核可为同一人”按钮，并单击“保存”。

![上传图片](https://vip.kingdee.com/download/01002047ffe6b313474b802f688f3f1c6263.png)

3）重新回到收款单界面，点击“审核”，出现“审核成功”，完成。

# 【适用版本】

星辰3.1以上所有版本
