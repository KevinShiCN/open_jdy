# 2026年进销存打印功能更新清单

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：功能介绍, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：267 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/805008826407214336)

## **2026-01-29 **[查看详情](https://vip.kingdee.com/link/s/ZwIOZ)

- 【所有版本】打印设计器内现有控件增加多个提示语及超链接查询，方便用户了解功能介绍，自主答疑。

| 功能按钮 | Tip 文案 |
| --- | --- |
| 文件名 | 可自定义打印文件名称，生成 PDF 或 Excel 打印文件时，将按规则命名，查看详情。 |
| 纸张大小 | 即纸张的宽度、高度规格。复写纸的规格说明，查看详情。 |
| 语言 | 切换语言后可直接维护对应语言文本，系统不自动翻译（需手动维护）。未购买多语言包时，仅“简体中文”支持打印，查看详情。 |
| 多语言布局控制 | 控件不同语言的模板样式是否保持一致，查看详情。 |
| 分页拼接打印 | 模板有多个 “分页” 时，可设置下一页是否接上一页空白处打印（有使用条件，与纵向节纸功能用法不同），查看详情。 |
| 打印所在页 | 即设置控件打印位置。顺序打印：控件按顺序动态适配，跟随上一页控件位置打印。所有页面：即每页打印，控件位置不受其他控件影响。仅第一页：即只在第一页打印，通常适用于表头控件。最后一页：即只在最后一页打印，通常适用于表尾控件，查看详情。 |

- 【所有版本】列表打印时支持将超链接字段（如物流单号）正常打印出来
- 【所有版本】AI打印助手性能优化、体验优化
