# 精斗云v3/v5中结转余额升级,提示“库存为0,结存成本为***””库存为-2,成本为***“怎么处理?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：winnie_wen | 阅读：306 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352545543158784)

【解释说明】

  结转余额升级是将原账套中的余额结转到星辰做为期初,商品期初数据是不允许期初库存为负数的,所以若V3V5中商品库存为负,请调平库存后再进行升级。另外若存在商品库存已经为0,但是还有结存成本的情况,请在V3V5中将成本调平后再进行升级。
