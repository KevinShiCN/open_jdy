# 数据授权里面仓库和调入仓库的区别

【问题描述】

数据授权里面仓库和调入仓库的区别

【解释说明】

**仓库权限:**设置仓库权限后,子账号查询仓库列表只能看到有权限的仓库,并且只能查询到有权限仓库的出入库单数据,没有授权的仓库做的出入库单则查询不到单据或者无法打开单据明细查询。

**调入仓库权限**:仅管控录调拨出库单、移仓单时,“调入仓库”能选到哪些仓库
