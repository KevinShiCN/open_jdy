# 打印设计器-数据源整体介绍

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：快速入门, 精华汇总, 功能介绍
> 作者：进销存赵嘉丽 | 阅读：1509 | 有用：7
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/454593747838166784)

## 1 含义

数据源，就是打印的信息来源，只要是单据或者报表上看得见的字段，理论上都会在数据源中支持，只要打印模板上绑定了对应的数据源，则打印便能显示。

## 2 适用版本

金蝶云·星辰 V3.1以上

## 3 类型及应用场景

星辰3.1支持三种不同类型的数据源，对应的应用场景可见下表的说明：

| 数据源类型 | 应用场景 |
| --- | --- |
| 主数据源 | 打印模板中常常需要用到数据源中的字段，通过不同的字段值组合为一个业务模板。比如在设计销售订单模板时，常常使用到单据编号、单据日期、商品编码等，都需要从数据源中获取。 |
| 关联数据源 | 关联数据源常用于主数据源中内容还不满足需求时，可以引入其他单据（引入的单据与主单据需要有关联性）。如：在设计采购订单打印模板时，需要关联预付款单，并添加预付款单单号 |
| 自定义数据源 | 当原单据的数据不足以满足打印模板的个性化打印时，可以自定义数据源便于在打印是满足个性化场景。 |
