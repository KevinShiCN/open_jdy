# **1 简介  **

## **1.1 功能介绍**

邀请成员功能，目前最多可邀请1位成员参与协作，可对成员设置任意1种角色和权限。

- 销售员：仅看个人数据，无采购/采购付款/对账等权限，不可看销售毛利。
- 管理员：公司所有数据可见，所有功能可用无限制，可看毛利。

## **1.2 适用版本**

金蝶云星辰·AI进销存

# **2 操作指南**

![上传图片](https://vip.kingdee.com/download/010087bd523d1088447f99f14511f5409064.png)

第一步：点击首页的头像进入管理页面

第二步：点击企业信息的“编辑”按钮进入企业管理页面

第三步：点击【成员账号/权限】进入成员管理页面

第四步：点击【邀请】，发送邀请函后，成员注册登录金蝶云星辰APP即可看到邀请函。
