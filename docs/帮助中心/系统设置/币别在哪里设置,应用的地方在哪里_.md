# 币别在哪里设置,应用的地方在哪里?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：进销存赵嘉丽 | 阅读：1535 | 有用：5
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/584115613996291584)

【问题描述】

币别在哪里设置,应用的地方在哪里?

【业务场景】

日常业务涉及到多种币别,除本位币外,可在币别中新增多种其他币别,以便满足日常业务、财务需求

【操作指引】

星辰的币别资料在【基础资料】>【财务资料】>【币别】中设置

![](https://vip.kingdee.com/download/0109d4f8e1e730294620afa4954650a6d172.png)
点击右上角<新增>即可新增币别

![](https://vip.kingdee.com/download/0109b0e8dd29d8ab4731923540f46dbabc6a.png)
币别的编辑及名称均不允许重复,若币别原本禁用了可重新启用,不可再次新增一样的币别

![](https://vip.kingdee.com/download/0109b2deffa386244655b8ffaf344deee16b.png)
在币别界面可操作“启用”“禁用”“删除”“复制”,已使用的币别不允许删除,可以禁用

![](https://vip.kingdee.com/download/010995490622fb7e48f0a559c03da6d036b6.png)
点击币别编码可打开币别的编辑界面进行修改,如修改汇率、小数位;需注意,小数位只能改大,不能改小,谨慎操作。

![](https://vip.kingdee.com/download/0109523916c610f4452b9aa8c5df3519dd41.png)

【应用场景】

1、基础资料(客户、供应商)中的默认币别

2、涉及外币账户收付款的,可在账户管理中新增或者设置已有账户的币别

![](https://vip.kingdee.com/download/0109874c9633832441ab8e7af79a12b46d7f.png)

3、单据开单选币别,如果开单需要选择外币,需要在系统参数-进销存参数中开启“启用外币”

4、科目中启用外币使用:[如何设置外币科目?如何进行期末调汇?](https://vip.kingdee.com/link/s/ZQpaE)

5、打印币别符号及金额大写

【打印金额大写应用说明】

![](https://vip.kingdee.com/download/01095e4e228f2cbc468098735a88469e3b21.png)

针对打印模板设置“金额大写”时,须确保币别资料里的**币别编码**规范符合国际通行的法定货币编码,**如不规范,就很有可能会导致即使打印模板都设置了,但是打印时中文大写不能生效。**

例如:人民币,币别编码设置了CN,或者美元,币别编码设置了US,这种都是不合规范的设置。**
**

![](https://vip.kingdee.com/download/01096a878ddb772b4c8fa29e72d18ccc95a5.png)

**为了确保金额大写可以正常生效,请务必保证星辰里的币别编码按如下规范设置:**

| 币别名称 | 币别编码 | 特别说明 |
| --- | --- | --- |
| 人民币 | CNY | 系统同时兼容了币别编码“RMB” |
| 美元 | USD |  |
| 港币 | HKD |  |
