# 打印基础——纸张大小及打印机设置

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：应用案例, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：9568 | 有用：5
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/697125312777345024)

# **1 业务场景**

在进行打印前，需要设置打印机纸张规格并设计套打模板。

- 例如使用针式打印机，需要设置多联纸/复写纸的纸张规格
- 例如使用标签打印机，需要设置标签纸的纸张规格

# **2 解决方案**

由于激光打印机通常使用标准的A4或A5纸张，用户无需另行设置，因此本文重点介绍针式打印机和标签打印机的纸张设置。

# **3 适用版本**

星辰3.1及以上所有版本

# **4 应用流程**

1）测量打印纸张尺寸

2）设置星辰打印模板尺寸

3）检查和设置打印机首选项纸张尺寸

4）打印测试及调整

# **5 操作步骤说明**

## 5.1 **测量纸张尺寸**

### **5.1.1 针式打印机专用的复写纸/多联纸（一等分/二等分/三等分）**

如果您不确定纸张的类型是什么，可以先用尺寸量一下，记住，无论是A4纸还是带孔的针式打印机复写纸，我们所说的纸张的宽和高都是**包括了实际不打印的边距位置**的，如下图：

![](https://vip.kingdee.com/download/010098731e6609cd4197b6a34138b3c2d8e3.png)

![](https://vip.kingdee.com/download/0100103f4b8437d94e75b48adbcf79f2bf59.png)

### **5.1.2 条码标签纸**

标签纸和复写纸不一样，我们说的宽，一般是指不干胶纸本身的尺寸，并不包含左右两侧的留白位置的，高则一般等于纸张的实际高度。例如下图是一个宽59mm，高40mm的标签。一般购买的时候，外包装上也会注明尺寸。

![](https://vip.kingdee.com/download/0109b92d006b2bc34b3f9a3647252fd97bb6.png)

![](https://vip.kingdee.com/download/010984272797a604476fa592e97bd161dd09.png)

## 5.2 配置**星辰打印模板的“**纸张设置”

第一步完成后，我们就需要到星辰的打印模板里设置对应的纸张参数，打开打印模板，可以看到模板尺寸设置基本都是在右侧属性栏里。

![](https://vip.kingdee.com/download/01090887693bc6d540aba034c0632c60f9eb.png)

星辰打印参数说明

| 参数名称 | 含义 |
| --- | --- |
| 纸张/纸张大小 | 系统预置的纸张规格：打印基础——星辰预置纸张规格说明（纸张大小）自定义：如果您需要的纸张规格不在里面，直接选“自定义”，并录入纸张的宽度和高度。 |
| 纸张方向 | 默认纵向（星辰3.1以上设计器里，“纵向”是一个竖着的矩形），关于纵向和横向的说明，详见下文的“延展说明”。 |
| 宽度和高度 | 把【5.1 测量纸张尺寸】测量的纸张的宽和高填到对应位置即可 |
| 边距 | 因为纸张的宽和高是包含了边距部分的，实际打印的区域可能会更小些，用户可以根据机器的实际放纸位置来微调边距，这一步在星辰的打印模板里设置。支持手工拖拽调整页边距。操作方法：鼠标放到画布最外层的虚线边框上，出现双向箭头即可拖动，手工拖动的同时，会同步自动更新页边距的数值。该功能10/31发布到灰度环境，功能还在灰度环境，如需体验，请联系在线技术支持。 |

## **5.3 检查和设置用户电脑所连接的“**打印机首选项”

**1）找到打印机首选项设置入口**

在电脑“开始”菜单里搜索“控制面板>硬件和声音>查看设备和打印机，找到默认打印机，选中打印机右键，查找“首选项”点击打开：

![](https://vip.kingdee.com/download/01098ea7edcbf75b46df8fb85309fbe0d786.png)

然后参考下一步的说明进行检查和设置，设置完成后点“应用>确定”即可。

**3）关于如何添加自定义纸张**

如打印机首选项里的纸张尺寸没有符合条件的，需自行新增自定义纸张规格，具体设置方法见下表说明。

| 系统 | 设置方法 | 截图说明（数字代表操作顺序） |
| --- | --- | --- |
| win7 | 找到“设备及打印机”，选中打印机—打印机服务器属性—创建新表单（设置好表单名称）—填入实际测量的宽高—保存表单---确定 |  |
| win10win11 | 找到“打印机和扫描仪”，打开打印机服务器属性—创建新纸张规格（设置好纸张名称）—填入实际测量的宽高—保存规格---确定 |  |

>
补充说明：

1、其他系统如果未找到需要自行上网搜索，关键词：XX系统打印机如何设置纸张大小。或者直接咨询您所使用的的打印机厂商售后。

2、常见问题：添加了自定义纸张后，在打印机首选项里还是选不到，一般有2种可能：

1）打印机驱动没装对，建议根据打印机型号到打印机官网下载最新的驱动

2）打印机不支持设置自定义纸张或者设置的自定义纸张超出了打印机本身可以支持的范围

## **5.4 打印****测试及调整**

按以上步骤都设置好后，就是打印调试了，用户可以根据实际打印的位置来微调打印模板，直到符合您的打印预期。

# 6 拓展知识——纸张方向的说明

在打印机设置中，**纸张方向**决定了打印内容在纸张上的排列方式，分为**纵向（Portrait）**和 **横向（Landscape）**两种模式。详见知识：[打印基础——纸张方向的说明](https://vip.kingdee.com/link/s/ZYD38)
