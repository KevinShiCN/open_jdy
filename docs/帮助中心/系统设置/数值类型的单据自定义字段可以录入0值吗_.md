# 数值类型的单据自定义字段可以录入0值吗?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：12 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/748348612681115904)

【问题描述】

数值类型的单据自定义字段可以录入0值吗?

【解决方案】

数值类型的单据自定义字段**勾选“****必录****”**才可以录入0值。

新增自定义字段勾选“必录”。

![](https://vip.kingdee.com/download/01091f8660bd99b347d2a885f011f3fe98fc.png)
然后就可以录入0值了。

![](https://vip.kingdee.com/download/0109e7155d8893194e5da276e0c978232adc.png)

数值类型的单据自定义字段不勾选“必录”的情况下不可以录入0值。
