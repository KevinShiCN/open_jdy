# 打印实践案例——单据上打印客户/供应商资料中联系人中的“座机”字段

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：功能介绍, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：642 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/551129333994269696)

## 1 应用场景

对于批发商贸的用户，企业往往是有使用座机的，因为对接的业务人员流动性大，频繁的信息变更容易带来的不便，打印座机可以保证信息相对稳定，方便联系。

## 2 适用版本

星辰3.1以上所有版本

## 3 功能特性

• 特性1：打印模板的数据源中增加了座机字段

• 特性2：单据上不显示座机字段，但是只要客户/供应商资料中的联系人分录中有维护对应的座机值，且打印模板有设置座机字段，打印时，系统会根据单据上的地址字段（见特性3）自动匹配

• 特性3：目前支持的单据范围

| 单据类型 | 支持单据 | 单据上用于匹配打印座机的地址字段 | 打印模板中数据源对应的“座机”字段 |
| --- | --- | --- | --- |
| 销售类 | 销售类 销售报价、销售订单、销售出库、销售退货申请 | 客户收货地址 | 客户座机 |
| 销售换货申请、销售退货 | 客户联系信息 | 客户座机 |  |
| 销售发票（旗舰版） | 结算供应商地址 | 客户座机 |  |
| 采购类 | 采购申请、采购订单、采购入库 | 供应商发货地址 | 供应商座机 |
| 采购退货 | 供应商联系信息 | 供应商座机 |  |
| 采购发票（旗舰版） | 结算供应商地址 | 供应商座机 |  |

## 4 操作流程

接下来，我们以销售出库单为例，具体演示如何设计座机打印：

1）首先，复制一个系统的预置模板来进行编辑

![](https://vip.kingdee.com/download/01000362e926f19d497a8d6f3c63ad11061d.png)

2）打开复制后的模板，点击收货地址后面的公式进行编辑：

![](https://vip.kingdee.com/download/0100753f58ac17e8400d8ab684fd3c5c5500.png)

![](https://vip.kingdee.com/download/01006330024408b84ed88139fcacc3d79ee0.png)

3）假设我们需要添加在这个地址上，添加客户联系人里的联系人、手机、座机

![](https://vip.kingdee.com/download/010040ac6c20f8db41608e32100423eb3e24.png)

4）那么我们在公式弹窗里找到对应的字段，

![](https://vip.kingdee.com/download/0100291037822fa54ded8a5b8441cb4c7ca6.png)

![](https://vip.kingdee.com/download/0100713d8d37afd64dd0901ac59ada6ef6c5.png)

点击后，添加到公式里，添加完成后如下图

![](https://vip.kingdee.com/download/01002b8ae508e226462192fb6a6c4d00439f.png)

点击“公式校验”，出现“校验成功”后，点击“确定”，然后“保存”模板即可。

![](https://vip.kingdee.com/download/0100d0e95c2c09af42c2b64c0bd1ba461eac.png)

5）打印效果

单据详情（详情上不会显示座机）：

![](https://vip.kingdee.com/download/01000c97fe3e555a4769a8865176ac87e415.png)

打印预览：

![](https://vip.kingdee.com/download/0100b10b706d4a684474b6a775dc26a15a43.png)
