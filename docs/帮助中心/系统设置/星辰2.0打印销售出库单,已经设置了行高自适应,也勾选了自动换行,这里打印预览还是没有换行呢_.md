# 星辰2.0打印销售出库单,已经设置了行高自适应,也勾选了自动换行,这里打印预览还是没有换行呢?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：457 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391353481627839488)

【问题描述】 :

打印销售出库单,已经设置了行高自适应,也勾选了自动换行,这里打印预览还是没有换行呢?

![](https://vip.kingdee.com/download/01097cb03fffee2a4509b81c3358391eb3ea.png)

![](https://vip.kingdee.com/download/01097267ea96ed7d4198aaf6253906417764.png)

【解决方案】:

当单据的打印模板中的数据表格的数据行开启了行高自适应时,打印设置里不要设置固定行数量,否则会有冲突。

把打印设置中的【每张打印分录数】设置为0即可自动适应行高
