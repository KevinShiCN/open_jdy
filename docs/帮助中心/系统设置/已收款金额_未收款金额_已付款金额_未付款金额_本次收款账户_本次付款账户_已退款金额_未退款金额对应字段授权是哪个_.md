# 已收款金额/未收款金额/已付款金额/未付款金额/本次收款账户/本次付款账户/已退款金额/未退款金额对应字段授权是哪个?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：进销存赵嘉丽 | 阅读：278 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/590712125199393792)

【问题描述】

标准版/专业版的销售出库单列表上的已收款金额、未收款金额、本次收款账户,采购入库单列表上的已付款金额、未付款金额、本次付款账户,销售/采购退货单上的已退款金额/未退款金额在字段授权里对应是哪个字段?

【解决方案】

| 单据类型 | 列表字段 | 字段授权对应的字段 |
| --- | --- | --- |
| 销售出库单 | 已收款金额 | 累计收款 |
| 销售出库单 | 未收款金额 | 本单未收 |
| 销售出库单 | 本次收款账户 | 收取货款(多账户收款时,也是通过这个字段控制的) |
| 采购入库单 | 已付款金额 | 累计付款 |
| 采购入库单 | 未付款金额 | 本单未付 |
| 采购入库单 | 本次付款账户 | 支付货款(多账户付款时,也是通过这个字段控制的) |
| 销售退货单采购退货单 | 已退款金额 | 累计退款 |
| 销售退货单采购退货单 | 未退款金额 | 本单未退 |

【适用版本】

星辰3.1标准版/专业版
