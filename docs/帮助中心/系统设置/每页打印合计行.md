# 每页打印合计行

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：功能介绍, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：388 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/714869509726932480)

## 1.  简介

通常情况下，打印模板中的合计行，如果设置了控件属性中对应的合计类型，则会根据合计类型来控制合计行的显示：

| 合计类型 | 合计行的打印效果 |
| --- | --- |
| 不合计 | 只在最后一页打印 |
| 累计求和 | 只在最后一页打印 |
| 每页求和 | 每页打印 |
| 分组合计 | 跟着分组走，显示在每个分组最后一行数据行的下方 |
针对合计类型为“不合计/累计求和”合计行的内容，如果我们需要每页打印，那么可以设置数据表格控件属性中的参数“每页打印合计行”来实现。
## 2. 适用版本

星辰3.1以上所有版本

## 3. 功能介绍

打开打印模板后,选中数据表格,勾选参数“每页打印合计行”后保存模板即可。![](https://vip.kingdee.com/download/0109c7e6827eccd845308e1ccce634a5217e.png)
