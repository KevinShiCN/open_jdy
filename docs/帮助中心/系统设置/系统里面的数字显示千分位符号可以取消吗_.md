# 系统里面的数字显示千分位符号可以取消吗?

【问题描述】

系统里面的数字显示千分位符号可以取消吗?

【解释说明】

单据、凭证、账簿、报表等的数字格式是默认的,**不支持**修改或者取消千分位符号。
