# 系统支持哪些成本计价方法?支持新增吗?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题

【问题描述】

系统支持哪些成本计价方法?支持新增吗?

【解释说明】

系统成本计价方法目前支持“移动平均法”、“先进先出法”、”加权平均法”三种计价方法。暂不支持自定义添加成本计算方法,建议点击产品右上角“产品共创平台”提交您的需求建议。

1、移动平均法计算原理:所有的商品入库后,出库单位成本取平均值;适用于商品购入**价****格波动小**的场景!出库单位成本 = 结存成本 / 结存数量

2、先进先出法计算原理:先购入的商品先领用或发出的原则进行计价;适用于商品购入**价格波动大**的场景!出库单位成本 = 先入库的存货单位成本

3、加权平均法计算原理:所有商品入库总成本/仓库总数量;出库单位成本=(期初成本+核算期间内入库成本)/(期初数量+核算期间内入库数量)

温馨提示:选择存货计价方法时,企业应根据自身的实际情况和需求,选择最适合的计价方法。同时,企业应关注相关会计准则的变化,确保所选方法符合法规要求‌
