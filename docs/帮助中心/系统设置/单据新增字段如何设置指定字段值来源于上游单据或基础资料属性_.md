# 单据新增字段如何设置指定字段值来源于上游单据或基础资料属性?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：Simone邓 | 阅读：61 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/799443904809275392)

【问题描述】

单据新增字段如何设置指定字段值来源于上游单据或基础资料属性?

【适用版本】

星辰3.0及以上**专业版**/**旗舰版**

【业务场景】

场景一:在销售订单中新增了自定义公式字段,希望可以携带到销售出库单中。

场景二:希望销售出库单中展示销售订单的预计交货日期,以便比较实际交货日期与预计交货日期,查看是否按时交货。

场景三:在商品资料中新增了自定义字段,用于维护商品的默认包装要求,并展示在销售出库单中;实际发货时,包装可能临时变更,需要手工修改。

**注意:该功能仅专业版支持,且目前处于灰度测试阶段。如需提前体验,请复制工作台的服务 ID 到售后在线服务界面发送给人工客服,申请加入灰度测试。**

【解释说明】

在【系统设置 】> 【单据新增字段】中,字段类型选择“文本”“数值”“日期”“引用辅助资料”或“引用基础资料”时,可设置字段值来源。

- **1、字段值来源于“上游单据”时**,例如设置销售出库单表体的自定义字段来源于销售订单的“预计交货日期”,设置后,当销售订单下推销售出库单或在销售出库单中选择源单为销售订单时,销售出库单中的该字段将携带销售订单的“预计交货日期”,用户可修改;若要禁止修改,需在用户管理中设置该字段的字段授权。
- **2、字段值来源于“基础资料”时**,比如设置销售出库单的表体自定义字段来源于商品资料中的包装要求,设置后,新增销售出库单时,在单据中选择商品后,该字段将携带商品资料中维护的包装要求,用户可修改,若要控制不可修改,需要到用户管理中设置该字段的字段授权。
- **3、文本型字段只能来源于文本字段,数值型字段只能来源于数值字段,日期型字段只能来源于日期字段。**
- **4、类型为“引用辅助资料”或“引用基础资料”的字段,需在上游单据或基础资料中存在相同类型的字段方可携带。**
