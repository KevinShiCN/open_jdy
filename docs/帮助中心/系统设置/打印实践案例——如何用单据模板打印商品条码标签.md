# 打印实践案例——如何用单据模板打印商品条码标签

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：快速入门, 精华汇总, 应用案例
> 作者：进销存赵嘉丽 | 阅读：1917 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/543093092241492480)

## 1 应用场景

在打印商品条码标签时，因为能取的字段有限，不能很好满足打印的场景，通过打印模板里的分录表格，可以实现：

• 标签打印取到单据分录上的信息，诸如数量、批次、生产日期

• 标签打印取到单据分录上的自定义字段，从而解决星辰2.0单据条码打印模板取不到单据体自定义字段的问题（星辰3.1以上的版本的条形码已经支持取自定义字段了）

• 购买了星辰标准版的客户，没有单据条码打印的功能，也可以通过这个方式来变通实现单据的条码标签打印

## 2 适用版本

星辰3.1以上所有版本

## 3 应用流程概述

1）新增空白的打印模板

2）确定纸张规格，即标签大小及边距

3）添加“分录表格”控件，选择数据源

4）调整分录表格位置，铺满可打印区域

5）绑定条形码控件、打印的名称及要打印的字段

6）结合用户实际需求，做其他个性化的设置（列宽、边距、字体类型、字号等等），根据预览效果进行微调，完成

## 4 实操演示

接下来，我们以销售出库单为例，具体演示如何通过单据打印模板设置出库单的商品标签。

1） 在【系统设置-打印模板】中，新增空白的打印模板，选择单据类型、命名好模板名称，单击创建。

## 5 设计注意事项

使用分录表格打印标签时，一定要注意一些的事项：

| 应用场景 | 参考示例 | 设计注意事项 | 正确的打印模板 | 错误示范 |
| --- | --- | --- | --- | --- |
| 标签需要有标题。如：公司名称 |  | • 标题放到分录表格上方的文本控件• 文本控件一定要设置“所有页面”• 模板上下方不能留白太多 |  |  |
| 标签上无标题，全部是商品分录的字段 |  | • 必须把分录的数据全部放到单元格里• 模板上下方不能留白太多。否则会出现第二行商品打印在第一页的下方，这样就不能很好实现商品标签的单页打印了。 |  |  |
