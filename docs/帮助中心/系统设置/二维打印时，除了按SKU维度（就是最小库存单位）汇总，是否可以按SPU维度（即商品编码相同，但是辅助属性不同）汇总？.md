# 二维打印时，除了按SKU维度（就是最小库存单位）汇总，是否可以按SPU维度（即商品编码相同，但是辅助属性不同）汇总？

> 作者：进销存赵嘉丽 | 阅读：666 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/363016143449611776)

【问题概述】

二维打印时，除了按SKU维度（就是最小库存单位）汇总，是否可以按SPU维度（即商品编码相同，但是辅助属性不同）汇总？

【解决方案】

星辰2.0可以支持。应用流程：

在打印模板中，找到合计行，把“数量”字段单元格类型设置成 “统计”，统计类型选“求和”，统计范围选"本页合计"。

![](https://vip.kingdee.com/download/01000f8495b5349a4a899503165a1a137f94.png)

打印预览效果：

![](https://vip.kingdee.com/download/0100aacc471462984fb693b7b6082f5a1410.png)

如果不选择“本页合计”，则打印时只会按照SKU维护汇总。

![](https://vip.kingdee.com/download/0100d1f824efca9f41898d7162dad19f2b12.png)
