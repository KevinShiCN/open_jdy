# 任务中心的待办任务,已办任务,在办申请,已办申请分别是什么意思?

> 作者：杨桂平 | 阅读：1148 | 有用：0

【问题描述】

任务中心的待办任务,已办任务,在办申请,已办申请分别是什么意思?

【解释说明】

任务中心主要是查看我提交的任务的进度和需要我处理的任务。工作流设置,详细可以点击:[工作流(单据审批)如何设置流程,如何设置条件审批?](https://vip.kingdee.com/link/s/ZGcMB)

【操作指引】

任务中心分为以下四种任务状态:

![](https://vip.kingdee.com/download/0109c200afebdcd1455f85564ce2d895d706.png)

1、待办任务:审批节点到了自己这里的单据

2、已办任务:自己已经审批过的单据

3、在办申请:自己提交的正在等待审批完成的单据

4、已办申请:自己提交的审批完成的单据

若需查询某段时间的,可点击<展开过滤>,在“创建时间”条件上设置时间条件查询

![](https://vip.kingdee.com/download/010979b2be37bb984bfeb8fd9d8fd73236c8.png)
当流程中任务有更新,任务中心还没刷新,可点<刷新>按钮更新

![](https://vip.kingdee.com/download/010978a35d3bc16a4dcdbf97d33446f79088.png)

注:即使单据后来删除了,只要有提交过,也能在任务中心看到记录。任务中心的记录不能删除或修改。

![](https://vip.kingdee.com/download/010969199f4c250f4d4289c1ab97f0e2dd57.png)
