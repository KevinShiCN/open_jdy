# 金蝶排障：合同留白的 “隐形克星”

![上传图片](https://vip.kingdee.com/download/01000c2a4fe2b80843e2972619424f4a4297.jpg)

“小王，这份采购合同怎么回事？首页内容过少，留下大量空白，客户还以为我们敷衍了事呢！” 采购部张姐拿着打印纸，急匆匆地闯进了 ERP 支持组的办公室。

小王赶紧接过合同，只见首页刚显示完商品表头和商品分录信息，下方就空出了大半页，关键的合作条款全被挤到了第二页。这个问题显然是由于打印格式设置不当引起的，根据经验，这类问题通常与表格尾部区域的配置错误有关。“张姐别急，我马上帮你调整！” 话音未落，小王便立刻着手操作起来。让我们看看，小王是怎么操作的呢：

首先，到【系统设置】-【打印模板】-【采购订单】找到对应的采购订单合同打印模板，点击打印模板的编码，跳转到模板修改界面；

![上传图片](https://vip.kingdee.com/download/0100b8529ffb10a24fec85e6fe337806376f.png)

![上传图片](https://vip.kingdee.com/download/01007be4bfeb81ec4f4c91d64852f96c4e8f.png)

在打印模板编辑界面，合同条款被错误地放置在了表尾区域，这导致了首页内容过少，留下大量空白；

![上传图片](https://vip.kingdee.com/download/010052f319376a814703a3ceabfb21f0da7e.png)

之后，为确保条款不丢失，需要将合同条款移动至一个新的页签，并调整页面布局;

![上传图片](https://vip.kingdee.com/download/01009e77753553044038a50c9ae719e2d449.gif)

在第一页中，选中数据表格，取消表尾区域设置；

![上传图片](https://vip.kingdee.com/download/010032b78f97d45f44e5b740117c4f6f12fa.png)

接下来，切换到第二页，重新复制条款，粘贴回第一页并调整位置，同时将条款的【打印所在页】设置为‘按顺序打印’。

![上传图片](https://vip.kingdee.com/download/0100c18a208c80044ec5ba9dea3e8931ec10.png)

最后，删除第二页，保存模板。

![上传图片](https://vip.kingdee.com/download/0100fc54862ef2e44ae393cfa5eb3c23f5b8.png)

打印合同效果：

![上传图片](https://vip.kingdee.com/download/0100232d330a4c5e424a8f27fef1d779f741.png)

调整完毕，小王点击 “打印预览”。这一次，首页标题下方紧跟合作双方信息、采购标的、数量条款，内容充实且排列有序，没有任何多余空白，后续页面的条款也过渡自然。

他重新打印出合同，递给张姐。张姐仔细检查了合同页面，满意地点头说："张姐仔细检查了修正后的合同页面，满意地点头说：‘这次的格式非常清晰，内容安排合理，看起来专业多了。"
