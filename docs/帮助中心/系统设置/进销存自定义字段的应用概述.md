# 进销存自定义字段的应用概述

> 作者：进销存赵嘉丽 | 阅读：2773 | 有用：19

问题描述：

1、自定义字段含义是什么，是用于什么地方的？有什么作用？

2、单据自定义与基础资料自定义有什么区别？如何应用？

解决方案：

**一、自定义字段：**

- 含义：是由用户自行添加的字段，可以进行相关属性的设置、内容编辑及删除。
- 应用场景：不同行业有不同的差异化字段需求，当系统内置的字段不能满足实际需求时，可以提供一定的权限，允许完全由用户自定义、个性化设置一些新的字段。
