# 为什么自定义字段名称修改之后仍然显示原名称?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：进销存熊诗琴 | 阅读：509 | 有用：3

# 【问题描述】

为什么自定义字段修改之后仍然显示原名称?

单据新增字段的名称无法修改吗?为什么修改之后不会同步至单据列表?

![](https://vip.kingdee.com/download/0109b9fbfb9c78d445f983a556b46c1cea3c.png)

![](https://vip.kingdee.com/download/01093bc7537b3c0b456fa407560e3e1326b1.png)
![](https://vip.kingdee.com/download/01092b014e2a979346f18923ef31a5d80936.png)

# 【原因说明】

1、单据新增字段里的自定义字段是可以修改名称的,文本框只要未置灰就是可编辑修改的。

![](https://vip.kingdee.com/download/01092b325b7a228d4dc9ba87fc0d5e6729ce.png)

2、在单据新增字段里修改完自定义字段的名称之后如果未同步至单据及单据列表,需要进入相关单据的列设置中**恢复默认列设置**再开启对应字段的显示,就会展示修改后的名称。

![](https://vip.kingdee.com/download/0109a66caa560d7745fb9eeb424214b6b336.png)

![](https://vip.kingdee.com/download/0109d135f795000d46e9904b48c901a16e9e.png)
![](https://vip.kingdee.com/download/010922b9450282db4cff875ef21a50ddd2b4.png)
