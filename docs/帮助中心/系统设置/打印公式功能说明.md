# 打印公式功能说明

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：快速入门, 精华汇总, 功能介绍

# 1 业务场景

当一个控件绑定一个字段值不满足场景时，需要组合多个字段组合或通过函数获取数据源，或者需要对数值型字段做一些运算处理时，可以通过公式来解决。也可以解决以下场景：

- 场景一：星辰的销售退货单不支持录入负数，但是可以通过打印模板的公式来满足单据打印的负数显示问题
- 场景二：需要把多个字段拼接成一个字段打印，例如地址、或者批次保质期等信息
- 场景三：需要手工录入一部分文本，再拼接系统的字段，输出一个完整的字段
- 场景四：需要对打印单据的数据做一些加工处理，例如用if函数做判断、取整、截取等等

# 2 适用版本

星辰3.1以上所有版本

# **3 功能介绍**

## 3.1 公式编辑器功能

公式编辑器是一个灵活强大的工具，它允许用户在标准场景无法满足时，自定义组合文本、字段和进行函数运算

文本、二维码、数据表格、分录表格、布局网格等控件均支持设置公式，控件可绑定**单个字段**和**公式**，公式在此处的用法为：**不同字段组合**、**文字拼接**、**函数**，最终按照公式表达式生成控件的值。

>
注意：公式的结果不影响数据库的结果，只是影响打印的展示效果而已。

控件添加公式的具体操作方法：

- 文本：选中控件后，点击控件属性>数据源>内容，然后在“选择数据源”弹窗中切换到公式页签即可
- 数据表格/分录表格/布局网格：选中控件后，单击其中需要录入公式的单元格，点击控件属性>数据源>内容中，然后在“选择数据源”弹窗中切换到公式页签即可
- 二维码也是同理

## 3.2 字段类型

在公式编辑页面，从左侧数据源中，可以通过直接查看字段前的图标来判断字段的类型

![](https://vip.kingdee.com/download/01093a64325146304a699816bd86123bd4b3.png)

字段类型说明：

| 字段类型 | 图标 | 字段颜色 | 参考字段 | 应用注意 |
| --- | --- | --- | --- | --- |
| 数值 |  | 绿色 | 数量、保质期 | 可以参与数学运算；不能用空字符串判断（如果数值字段为空，要用"字段=0"表示） |
| 文本 |  | 蓝色 | 商品编码、商品名称、批次 | 不能参与数学运算 |
| 单价/金额 |  | 绿色 | 单价、税率、价税合计、折扣 | 跟数值的区别是，可以设置货币相关的属性、金额大写等等 |
| 日期 |  | 浅米色 | 单据日期、生产日期 | 日期类函数只能使用该类型 |
| 图片 |  | 蓝色 | 商品图片 |  |

## 3.3 公式支持函数

打印公式弹窗中预置了非常多的函数类型，同时也支持直接录入回车键换行，可满足多种应用场景。

![](https://vip-admin.kingdee.com/download/0109c9d678d11df042dfbe30397fd62e12c1.png)

详见知识：[打印模板公式及换行功能使用说明](https://vip.kingdee.com/link/s/Zcrgl)

>
注意：函数只能使用模板中预置的，不支持手工录入的函数，手工输入的，即使校验成功，实际打印预览也是不生效的

## 3.4 公式校验

为了避免基础的用法出错问题，系统会先校验语法是否正确。点击“公式校验”和“确定”时校验表达式是否合法。

![](https://vip.kingdee.com/download/010077c5990c8e1a42ddb668839d787ca7ce.GIF)

**当存在多个语法错误或以下错误时，则校验不通过**：

| 错误描述 | 示例说明 |
| --- | --- |
| 符号不是英文半角符号，而是中文全角符号 | 双引号要使用"" ，不能使用“” |
| 手动输入文本时，文本左右两侧是否使用了英文双引号标识 | 应该是"金蝶"，不能输入“金蝶”或   金蝶 ——双引号使用不对或者没有双引号 |
| 存在不合法函数嵌套使用 |  |
| 存在多余的符号 | 函数（1,）就是错误的，中间多了个逗号, |
|  |  |

**即使校验通过，但是实际是不生效的场景：**

| 描述 | 示例说明 |
| --- | --- |
| 错误的使用 | 例如：以下SUM函数的使用都是不支持的：SUM(字段1+字段2+...) ——不支持SUM函数中对多个字段进行数学加法运算SUM(IF()）                   ——不支持SUM函数中嵌套IF函数SUM(字段1*字段2*...） ——不支持SUM函数中对多个字段进行数学乘法运算 |
| 函数只能使用模板中预置的，不支持手工录入的函数，手工输入的，即使校验成功，实际打印预览也是不生效的 |  |
| 拼接的结果如果是作为函数的值输出时，当存在手工录入文本与系统字段拼接的情况时，两个字段之间要用&连接，例如下图中的“生产日期：&日期” |  |

# 4 公式的应用

## 4.1 字段组合/拼接

- 拼接字段如果不是作为函数的值输出时，**字段的拼接不用添加符号**，直接选中即可，例如：

![](https://vip-admin.kingdee.com/download/0109ce756fe2168f4cc98f51c096771e66f7.png)

**场景案例**：打印销售订单中的客户+联系电话+省市区地址

**操作方法**：先增加一个文本控件，点击公式，把要组合的字段都找到并选上，保存后就可以实现拼接的效果。

![](https://vip-admin.kingdee.com/download/0100a52b3a18475e42dfa025135b5a71de78.GIF)

- 拼接的结果如果是**作为函数的值输出**时，当存在手工录入文本与系统字段拼接的情况时，两个字段之间要用&连接，例如下图中的“生产日期”是手工录入的，而“日期”又是系统的字段。

![](https://vip.kingdee.com/download/0109505855dbf60f4e13b0411f298adee43f.png)

## 4.3 函数

通过系统预置的函数，可对打印单据的数据做一些加工处理，例如用if函数做判断，小数的取整、字符串的截取等等。处理后，只影响打印的结果，不会影响单据上的结果

- **场景案例1：计算销售出库单中的平均客单价，用含税单价来计算。**

**操作方法：**添加文本后，点击公式，添加平均函数avg( )，然后找到“含税单价”，填入公式中，并设置好字段的类型、精度，保存即可。

![](https://vip-admin.kingdee.com/download/01007680b75473054b139532e11921597044.GIF)

- **场景案例2****：输出商品编码中，从左边第一位开始数，前6位的编码**

**操作方法：**可以使用左截取函数LEFT（str,len）操作，输入字段（即语法中的str）后和截取长度数值（即语法中的len）即可。

**![](https://vip.kingdee.com/download/01099a2ee3ee46a845d0b136e21dc5ebaf6d.png)**
