功能说明
常用于发票、凭证打印模板中以填格来展示大写金额或客户编码。

# 应用流程

1、 选中打印模板中数据表格中的单元格，在控件属性-业务中开启填格，设置填格数，可自动定义是否显示分隔线

![](https://vip.kingdee.com/download/01003c74f1aafb2c4b84b2801e0c411777e1.png)

2、 设置填格单元格的对齐方式

![](https://vip.kingdee.com/download/0100444802a8715b4d55bf150d9d00b6272c.png)

如果是要显示中文数字大写，则需要设置右对齐。
