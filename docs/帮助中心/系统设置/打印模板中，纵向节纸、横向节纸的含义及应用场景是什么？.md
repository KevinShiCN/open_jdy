# 打印模板中，纵向节纸、横向节纸的含义及应用场景是什么？

【问题概述】

打印模板中，纵向节纸、横向节纸的含义及应用场景是什么？

【名词解释】

节纸，顾名思义就是节约纸张。即当一页纸没有打完时，不另起一页，直接从本页空白处接着打印新的一张单据/条码，从而起到节约纸张的作用。

【应用说明】

在打印模板中，点击单据的空白处，在右侧的“业务”中，可以看到有纵向节纸和横向节纸的参数，开启则启用，不开启则禁用。

![](https://vip-admin.kingdee.com/download/0100878ec7f6e2a749aaa8762eba8b7ffd81.png)
【应用场景】

- **纵向节纸：一般用于纵向出纸的单据的打印。**

示例：如下图，单据纸张是A4纸，但单据没有打完一张A4纸时，而下方空白可以接着打印另一张单，则显示如下：

![](https://vip.kingdee.com/download/0100827e825c8e2049ff97053bdd4855eaba.png)

如果不开启纵向节纸，则每页只打印一张单：

![](https://vip.kingdee.com/download/0100b00bd4c4ef4a40a69bdfedd979ffa355.png)

- **横向节纸：一般用于横向出纸的标签打印、或者多排标签打印场景**

![](https://vip.kingdee.com/download/010019eff3ac27b84cd6a579d659faed1eb4.png)
