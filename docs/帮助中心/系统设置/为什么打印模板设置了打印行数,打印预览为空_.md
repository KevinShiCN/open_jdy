# 为什么打印模板设置了打印行数,打印预览为空?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：进销存熊诗琴 | 阅读：628 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/634919622281374208)

# 【问题描述】

打印模板在打印设置中输入打印行数,打印预览就是空的,打印行数设置为0,打印预览就是正常的?

![](https://vip.kingdee.com/download/0109955713187a4645fc84987b3dadc2d05a.png)
![](https://vip.kingdee.com/download/0109c611f682b8fe455ca4e3718da287e03c.png)

# 【原因排查】

            1、首先需要检查打印模版中的数据行中的数据源是否有值,如果数据源为空或者不存在的话,打印预览就会为空,需要重新设置数据源;如果数据源是正常的,则继续往下一步检查;

![](https://vip.kingdee.com/download/01097d7bbbd3cb2f4b488c3e6059256a343e.png)

2、我们可以看到打印模版中标题行和数据行的行高很窄,且刚好单元格的“内容超长时”选择了“行高自适应”;

![](https://vip.kingdee.com/download/0109fb4592ee327b441f9a64648af5205c98.png)

因为在**固定行打印的场景下,行高是固定的,内容不支持自适应**,如果行高无法展示完整的文本时,文本就无法正常显示,这个就是第二个原因。

# 【解决方案】

针对上述问题,有两种解决方案:

**方法1:调整数据表格的行高。如果行高是允许修改的,那么建议把数据表格的行高改大,预留足够的空间以保证可以完整打印文本**:

![](https://vip.kingdee.com/download/01095823b94ebcb64d64b6fe7ea0ff6c6948.png)

调整行高之后每行打印分录数设置为0或者其他值都是可以正常打印的。

![](https://vip.kingdee.com/download/0109ee7f5fea31a447fdb442af6a80b3114b.png)
![](https://vip.kingdee.com/download/010960e3b3313b554556b2884bda54e157eb.png)

**方法2:如果行高是固定的,不允许调整,那么只能修改单元格的“内容超长时”,选择“截断”**:

![](https://vip.kingdee.com/download/0109ac6aeaa5dd6c4104be49bfba35966d86.png)

**设置后,商品行就能打印,但是预览会是截断的效果,也就是无法展示完整的文本:**

![](https://vip.kingdee.com/download/0109751fd1662910444d8ae7356668729e42.png)

具体使用何种方法,请您结合实际业务需求进行选择。

# 【适用版本】

适用星辰3.1以上所有版本
