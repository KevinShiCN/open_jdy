# 预览有多页只需要打印某一页怎么操作

【问题描述】

预览有多页只需要打印某一页怎么操作

【解释说明】
如果打印预览有多页,可以通过选择打印范围,选择需要打印的某一页。
![](https://vip.kingdee.com/download/01090132bb948eb54c718b3246abfeab51af.png)
进入选择打印机界面 ,可以选择需要 打印的页面
