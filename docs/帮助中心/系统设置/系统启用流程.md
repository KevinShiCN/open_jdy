# 系统启用流程

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：快速入门, 整体介绍
> 作者：Cat_Chow | 阅读：14474 | 有用：12
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/622791405533847040)

服务启用后，有三种创建账套方式：直接新建全新账套、导入第三方财务软件数据、试用账套数据导入正式版本；企业可根据实际情况选择创建账套的方式。

![](https://vip.kingdee.com/download/0109d61132a1d4274724997472f6de90d8e7.png)

| 步骤 | 具体说明 | 学习链接 |
| --- | --- | --- |
| 1.直接新建账套 |  | —— |
| 1.1前期资料准备 | (1)创建账套前，确认启用期间，以便整理财务初始数据录入系统，保证数据的连续性，如确实来不及整理，后续结账前把期初数据补充完整。(2)注意会计科目的设计，如辅助核算设置、数量金额核算、外币核算等，是否需要调整，可提前规划。(3)业务资料分类及编码设计等，便于后续不同维度统计分析数据。 | —— |
| 1.2创建账套 |  | —— |
| 1.2.1启用期间设置 | 系统支持按自然年度设置会计期间，也支持自定义会计期间。系统起始期间一旦在建账时选择后，则在系统的【公共参数】体现，且不允许修改。系统起始期间会限制财务参数、进销存参数等的启用范围，最早只能选择前5年，最晚只能选择后15年。如系统起始期间为2024年1期，则财务参数最早只能选择2019年1期，最后只能选择2038年12期。 | 点击学习 |
| (1)自然年度期间自公历1月1日起至12月31日止为一个会计年度，分为12个期间，每个期间的1  日为期间开始日。自定义期间：例如部分国家会采用4月制的,从每年4月1日起至下年3月31日止为一个会计年度。 | 点击学习 |  |
| (2)自定义会计期间企业可以根据自己的需要自定义区间,比如中国香港一般每年的4月1日至次年的3月31日作为一个会计年度,这种期间在星辰的设置里就属于自定义期间。在创建账套界面的【期间设置】里进行设置   | 点击学习 |  |
| 1.2.2会计制度选择 | 目前系统支持7种会计制度：小企业会计准则(2013年颁)、新企业会计准则(2006领)、新企业会计准则(未执行新金融和新收入准则)、新企业会计准则(执行新收入新金融新租赁准则)、民间非营利组织会计制度、农民专业合作社会计制度和基层工会会计制度注意：系统启用后，若存在以下情况，会计制度不支持修改：①已录入财务初始数据，②已录入现金流量初始数据，③已录入资产卡片，④出纳日记账选择了对方科目，⑤已录入凭证。 | 点击学习 |
| 1.2.3本位币 | 支持外币做为本位币，且系统预设有多种币别，若列表中没有对应的币别，可以先选择人民币启用，启用成功后，再在系统里修改币别名字。注意：系统启用后，本位币不支持修改 | 点击学习 |
| 2.第三方导入 | 支持将其他版本的账套数据，导入星辰，如精斗云产品、金蝶KIS、用友等 | —— |
| 2.1导入方案介绍 | 导入方案四种：从精斗云3.0版本导入、从金蝶KIS云商贸版导入、从第三方产品导入、从金蝶KIS云专业版导入。 | 点击学习 |
| 3.试用账套数据导入 | 试用账套可在工作台启用正式服务时,选择迁移数据到正式账套中 | —— |
| 3.1注意事项 | 支持标准版升级专业版、同一版本内增减模块迁移试用数据。注意：试用专业版，购买标准版时，无法迁移试用数据 | 点击学习 |
| 3.2操作步骤 | 情况一：企业管理员通过试用账套内顶部提示，点击购买；或企业管理员在工作台，点击转付费。情况二：金蝶工作人员帮企业进行代客下单。 | 点击学习 |
| 4.账套首页介绍 | 创建账套成功后，系统自动进入账套首页。首页为常用功能的快捷入口以及进销存/财务的一些关键统计数据展示。 | 点击学习 |
| 5.公共参数设置 | 涉及系统信息、企业信息、敏感数据设置等 | 点击学习 |
