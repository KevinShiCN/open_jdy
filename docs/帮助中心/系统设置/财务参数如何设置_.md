# 财务参数如何设置?

【问题描述】

财务参数如何设置?

【解释说明】

财务参数包括基础参数和财务参数,可以对财税系统的业务处理规则和处理惯例进行统一的设置和管理。

若需修改不同账套参数,需切换账套分别设置

【操作指引】

1、打开【系统设置】-【系统参数】

![](https://vip.kingdee.com/download/01093b5d412092054da0a92acf676997493b.png)

2、点击【财务参数】,界面如图所示:

![](https://vip.kingdee.com/download/010987a76b0428bb44ddb9540a5d41e36041.png)

3、功能说明如下:

**基础参数**

①启用期间:即建账时设置的系统启用期中,刚启用的账套,如果没有录入任何数据,可以修改启用期间,录入数据后则不允许再修改。

②当前期间:当前财务所在期间,仅显示,不允许修改,由系统自动判断显示

③会计制度:即建账时选择的会计制度,未录入科目初始数据和凭证前允许切换,但切换后,需要重新设置科目信息。

**注意:系统启用后,若存在以下情况,会计制度不支持修改:**

①已录入财务初始数据;②已录入现金流量初始数据;③已录入资产卡片;④出纳日记账选择了对方科目;⑤已录入凭证。
