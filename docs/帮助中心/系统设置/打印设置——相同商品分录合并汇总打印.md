# 打印设置——相同商品分录合并汇总打印

# 1 简介

## 1.1 业务场景

对于进销存业务单据，如采购入库单、销售出库单等，当单据存在相同分录时，可以在打印设置里勾选对应的汇总参数，从而满足客户汇总打印的需求。减少冗余分录信息，提高出入库拣货、核对的效率。

![上传图片](https://vip.kingdee.com/download/0100447e586cfeb947f5be55004f56a22f82.png)

## 1.2 适用单据

进销存单据

## **1.3 适用版本**

星辰3.1以上所有版本

## **1.4 **汇总依据

①同一商品，在仓库、仓位、是否赠品、单位、换算率、商品行备注、组件子件类型均相同时汇总，否则不汇总。

②同一商品在辅助属性、批次、保质期均相同时汇总，否则不汇总。

③同一商品，注册证号，产地，生产许可证相同则汇总，不相同则不汇总

④同一商品，单价，实际含税单价，税率，折扣，零售价相同则汇总打印，不相同则不汇总

⑤同一商品，在采购入库单和销售出库单中，取第一行商品分录的成本作为入库成本和出库成本。

⑥序列号商品不支持合并汇总打印，自定义字段也不支持

# 2 操作指南

## 2.1 应用流程

在单据的打印设置处勾选“相同商品分录合并汇总打印”，则打印时相同商品分录将根据判断条件汇总打印。

>
注意：若需要根据以上汇总依据进行汇总打印，则打印模板中必须设置对应的字段。

举例：同一商品在单据中录入了两行分录，但单价不同；当模板未设置单价而仅打印金额时，系统打印时不会判断单价是否相同，会直接合并为一行并汇总金额打印。

# **3 与打印模板中“合并汇总字段”的区别和应用场景？**

## **3.1 **合并汇总字段的设置入口

在打印模板中，选中数据表格的数据行后，可在【数据行设置>合并汇总字段】中设置需合并的字段。

![](https://vip-admin.kingdee.com/download/010016da99ca65d241cbb5f300eb23ccdd4e.png)

## 3.2 差异与应用场景

打印模板中的【合并汇总依据】与打印设置中的【相同商品分录合并汇总打印】存在冲突，且适用场景不同，**二者一般不可同时使用。**

如果打印设置中的合并功能勾选后即可**满足**您的使用场景，则直接使用该参数即可。

如果打印设置里自带的合并功能勾选后，**无法满足**您的适用场景，例如要按**源单单号**维度合并，那么建议不勾选，直接在打印模板里设置。
