# **1 简介  **

## **1.1 功能介绍**

分页——打印分页设计常用于打印的内容较多时，在一个画布展示不全，希望可以多个页面设计打印模板，保证内容的完整性。例如合同打印。

## **1.2 适用版本**

星辰3.0以上所有版本

## **1.3 系统路径**

打印设计器中的“打印设置”

# 3 **功能特性**

- **分页上限：系统最多支持增加20个分页**
- **分页打印的范围**

打印的分页多少取决于模板纸张尺寸，分页不能超过纸张可打印的范围。意思是，假设纸张最大就是A4纸，那一个分页的打印内容就是A4纸，多出的部分就放在下一个分页，不能说把第二页的内容强制放到第一页内，这样的话，打印机也读不到这部分的内容。

- **分页拼接打印**：增加分页后，用户可以自定义选择分页是否拼接。拼接的效果就是下一页的内容接着上一页的尾部继续打，保持连贯，不拼接则是每个分页都另起一页打印。

>
注意，使用该参数要求上一张分页必须铺满。也就是页面基本都塞满了，没有过多的留白，此时下一张分页才会实现拼接，**如果上一张分页底部还剩余较多的空白，那么系统就会认为上一页没有打印完，此时就不会实现拼接。**
