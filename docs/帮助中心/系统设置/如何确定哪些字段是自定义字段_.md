# 如何确定哪些字段是自定义字段?

【问题描述】

在系统设置中增加了自定义字段之后,在单据中如何快速找出这个字段为自定义字段,如何查看字段标识?

【解释说明】

可以在单据或者基础资料引入引出模板中看到字段标识;备注中有标识的即为手工新增的自定义字段。
