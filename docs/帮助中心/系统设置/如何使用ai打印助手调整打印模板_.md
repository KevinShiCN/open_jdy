# 如何使用ai打印助手调整打印模板?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：Simone邓 | 阅读：67 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/799443879207907072)

【问题描述】

如何使用AI打印助手调整打印模板?

【适用版本】

星辰3.0以上所有版本

【解释说明】

在进销存业务单据的打印模板上增加AI打印助手入口,可使用AI协助调整打印模板。

**注意:该功能仅专业版支持,且目前处于灰度测试阶段。如需提前体验,请复制工作台的服务 ID 到售后在线服务界面发送给人工客服,申请加入灰度测试。**

【操作步骤】

【系统设置】-【打印模板】,点击模板编码打开模板编辑界面

![](https://vip.kingdee.com/download/010963651d69e48e410c8aee53da2fefdfff.png)
右上方【AI打印助手】,点击后会调出AI打印助手,收起设计器控件、控件属性、打印设置,仅保留模板及AI打印助手

![](https://vip.kingdee.com/download/0109e9d96a1352644014a0f46da429dffe03.png)

![](https://vip.kingdee.com/download/010982218816fac640839e34b04421cc9444.png)
可以在AI助手界面点击预置的指令执行

![](https://vip.kingdee.com/download/0109b8c76e460226499ca17f37e04279e050.png)
支持发送文字指令,AI自动识别、执行调整模板

![](https://vip.kingdee.com/download/010975554d42994c44d58d097c2b763e043a.png)
