# 其他入库单打印时,提示“当前打印模板没有设置动态列,不允许注册二维打印插件”是为什么?该怎么处理?

> 作者：海鸥乔纳森 | 阅读：339 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391353488439100160)

【

问题描述 】:

![](https://vip.kingdee.com/download/0109b12c559849ee4d71b68e3004465672a7.png)

【解决方案】:

只有辅助属性商品打印才会使用二维打印,使用二维打印时,必须在打印模板中有设置动态列且插件注册了二维打印插件。

一般出现上述情况是因为用户复制了系统预置的二维打印模板进行修改,所以模板中默认携带了二维打印插件。

分2种情况解决:

1、如果确实**要使用**二维打印,需对辅助属性组的列设置“类型”为“动态行”:

![](https://vip.kingdee.com/download/0109a7b5030c9bbf4c6d875a92299b9d2e46.png)

2、如果**不需要使用**二维打印,则需要删除打印模板中的二维插件,保存即可:

![](https://vip.kingdee.com/download/0109027c74cfa2484734b4c6c3c12493f339.png)
