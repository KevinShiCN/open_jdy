# 整行跨页时换至下一页/打印时整行换至下一页功能说明

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：快速入门, 精华汇总, 功能介绍
> 作者：进销存赵嘉丽 | 阅读：1315 | 有用：0

## 1.  简介

    通常情况下，打印模板中含有表格、网格时遇到内容过多换页时，表格一行数据将会被分开在两页中展示。但某些场景下，用户希望一行数据在分页时，能够整体换至下一页打印，保证一行数据的完整性，不被拆分，以便保证数据在一页中是完整的。本文将以采购申请单为例，介绍整行换页的功能。

## 2. 适用版本

星辰3.1以上所有版本

## 3. 功能介绍

• 操作

整行换页功能由开关“整行跨页时换至下一页”控制，选中数据表格后，可以在右侧的控件属性里看到。

当行出现内容过多从而导致跨页打印时：

开启开关，则整行记录将会换至下一页打印；关闭开关，则整行记录将被拆分，打印在两页中。

• 支持控件

    数据表格、分录表格、网格、纵向子数据表格

• 注意事项

    1. 整行换页不支持纵向合并单元格的场景（所有列均纵向合并，使得合并后最终多行展示为一行时，除外）；

    2. 整行换页不支持横向子数据表格；

    3. 如果某行记录中内容过多，当前页打印后剩余内容超出页中可打印区域的一半，则此时整行换页失效；

    4. 设置了【每页打印标题行】时，标题行不支持整行换页。
