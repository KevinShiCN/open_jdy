# 启用了企业互联发送单据,为什么操作日志上提个单据的发送会有多条记录呢

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：ylim | 阅读：160 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/588895987960264192)

【问题描述】

启用了企业互联发送单据,为什么操作日志上提个单据的发送会有多条记录呢?如下图:

![](https://vip.kingdee.com/download/01097119ecc81b76419598123faf53b70c53.png)【解释说明】

关于企业互联发送操作日志显示的问题,每点一次都会出现两条操作日志 不管失败与否,都会记录一条“编号XX,发送成功”,用于记录用户点了这个按钮;实际上是否发送成功要看第二条操作日志。
