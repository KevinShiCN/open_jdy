# 编码规则中的流水号起始值如何设置?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题

【问题描述】

编码规则中的流水号起始值如何设置?

【概述】

如编码规则中的流水号起始值设置为1:

举例:

1、若日期设置为流水号依据,则日期不同时流水号重新编号。

某编码规则为“常量DD-日期yyyymmdd-流水号”,今日日期为2025-04-01,流水号从001开始编号,则01日的单据编号从DD20250401001、DD20250401002、…、DD20250401099这样编号,02日的单据编号从DD20250402001开始编号。

![](https://vip.kingdee.com/download/01099a013bde20054b67b532f145bb891ab5.png)

![](https://vip.kingdee.com/download/0109bf5913d1b75443908bf9dc78c06b6d37.png)
![](https://vip.kingdee.com/download/010936175293a02f46f5aa0da855fd0d4f56.png)
2、若日期未设置为流水号依据,且没有设置其他的流水号依据,则02日的单据编号从01日最大的流水号DD20250401099接着编号,即从DD20250402100开始编号。
