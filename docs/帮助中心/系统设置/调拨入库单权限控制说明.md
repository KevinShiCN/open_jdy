# 调拨入库单权限控制说明

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：快速入门, 功能介绍, 新特性
> 作者：进销存赵嘉丽 | 阅读：744 | 有用：5

# 1 简介

## 1.1 功能说明

调拨入库单/移仓单会对调入仓库有仓库权限管控，只有当子账号数据授权中“仓库”中拥有调拨出库单上的“调入仓库”的仓库权限时，才能操作调入。只有同时拥有“调入仓库”和“仓库”的数据授权，子账号才能操作下推入库，从而避免了非本仓库的人误操作了入库的情况。

## 1.2 适用版本

星辰3.1以上所有版本

## 1.3 适用单据

库存管理>仓存单据>调拨入库单、移仓单

## 1.4 功能发布时间

发布时间：2022/11/25

# **2 应用说明**

假设对子账号张三设置了拥有仓库“深圳仓”权限

![上传图片](https://vip.kingdee.com/download/0100cf28134fed314923901880b163ec63a6.png)

以及调入仓库“广州仓”的权限

![上传图片](https://vip.kingdee.com/download/01008018e718972443b48454c941200b23ed.png)

下面以调拨出库单和调拨入库单为例说明，移仓单也是同理，本文不再赘述。

现在有调拨出库单如下：（调入仓库=广州仓）

![上传图片](https://vip.kingdee.com/download/0100651b3ad573e6459d9b8b0411f86ce36f.png)

此时，张三对下面这张调拨出库单做下推入库，会提示“调入仓库：广州仓，无仓库权限，请联系管理员开通仓库权限”，

![上传图片](https://vip.kingdee.com/download/0100bd7601629d424c4cad018f2dc185e85f.png)

原因就在于，目前做调拨入库时，系统会同时判断当前子账号是否拥有了“仓库”及“调入仓库”的数据权限，不能只满足其中一个条件。

解决方案：

就是在子账号【数据授权>仓库】里点击“添加条件”，增加一条“仓库名称=广州仓”的条件，一定要记得把逻辑关系改成“或者”

![上传图片](https://vip.kingdee.com/download/0100316da87839ee4768be59328447580d0c.png)

此时子账号再做下推入库就可以正常生成调拨入库单了。

![上传图片](https://vip.kingdee.com/download/01008cd14f3afd0846fa838fa67f15d49f8a.png)

# 3 特别说明

如果数据授权中的“仓库”不做任何设置（如下图），则表示子账号拥有所有仓库的权限。

![上传图片](https://vip.kingdee.com/download/0100fc21c0aa9d2940d599188d6d5e09d4dc.png)

此时只要拥有“调入仓库”的权限即可下推调拨出库单。

![上传图片](https://vip.kingdee.com/download/01002346b35361794b9c8882d94492fdd5a0.png)
