# 图片(打印控件）

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：快速入门, 精华汇总, 功能介绍
> 作者：进销存赵嘉丽 | 阅读：807 | 有用：11
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/454307506907384832)

# 1 简介

常常用于打印模板中上传固定的图片（如上传企业logo）或者绑定单据上的商品图片字段。

# **2 适用版本**

金蝶云星辰 V3.1以上

# **3 功能说明**

## **3.1 工具栏功能**

图片控件拖入设计器画布后，仅支持线框设置、复制、删除和格式刷功能。

## **3.2 图片的数据来源**

| 来源 | 操作方法 | 演示 |
| --- | --- | --- |
| 直接上传图片 | 1）选中图片控件2）点击“请上传”，唤起本地上传弹框，选择本地图片后点击“打开”完成上传即可。注意：支持本地图片格式为jpg、jpeg、png、bmp，不支持gif文件。 |  |
| 选择图片字段绑定 | 1）选中图片控件2）点击“选择字段”，选择图片字段即可。一般选择“商品编码.商品主图”即可。 | 绑定后的效果 |

## 3.3 图片控件在不同位置的取值逻辑

### **3.3.1 表格外图片控件绑字段**

通过字段形式取图片的话，在表格以外的区域只支持绑定**单据头字段**，若绑定**单据体字段**（如：商品分录.商品图片），默认只取一行分录。

## **3.3 图片显示方式**

• 原始大小：按照图片的宽高展示原始图片大小，原始大小超出控件大小部分截断处理，这时需要自行调整控件大小的宽高。

• 按比例缩放：根据图片的宽高大小的等比缩放展示。
