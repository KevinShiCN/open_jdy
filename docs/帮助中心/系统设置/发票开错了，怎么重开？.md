# 发票开错了，怎么重开？

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题, 快速入门
> 作者：杨桂平 | 阅读：928 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/375690267267608576)

| 电子发票（普通发票）红冲   登录官网：www.jdy.com，进入【我的工作台】-【我的发票】处，找到需要重开的订单，点击操作处点击【重新申请】按钮可申请红冲发票。   注：电子发票提交退票申请后，我们将在3个工作日内完成审核，通过后您可以在【我的发票】 处再次重新申请开票 |
| --- |
| 增值税专用发票（纸质）退回  需将原错误发票寄回，寄回地址：深圳市南山区科技南十二路金蝶软件园A栋四楼，杨捷锴收，电话：0755-86073145（注：寄回运费需由用户自行承担，不收到付件） 同时发邮件至客服信箱cs@jdy.com，提供以下信息：订单号：发票退回邮寄快递公司及物流单号：换开原因：正确开票信息公司名称：统一的社会信用代码（税号）：开户行：银行账号：公司地址：公司电话备注：（需求时填写）收件人信息收件人姓名：寄送地址：联系电话：邮编： |

说明：我们会在工作日内24小时回复邮件，烦请留意我们的邮件回复。
