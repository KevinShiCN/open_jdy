# 可用库存、预计可用库存参数，没有生产待领料量（未审核生产领料单）、生产待入量（未审核产品入库单）这2个选项，为什么？

# **1 问题描述**

可用库存、预计可用库存参数，没有生产待领料量（未审核生产领料单）、生产待入量（未审核产品入库单）这2个选项，为什么？

# 2 适用版本

星辰3.0及以上专业版/旗舰版

# **3 原因解析**

出现这个情况是因为库存更新方式是保存时更新，只有审核时更新才会有这个参数

![上传图片](https://vip.kingdee.com/download/01002e5242e46a0d488892b9d2c417807077.png)

库存更新方式改为审核时更新后，可用库存、预计可用库存参数中增加的生产单据：

预计入库量：生产待入量（未审核产品入库单）

预计出库量：生产待领料量（未审核生产领料单）
