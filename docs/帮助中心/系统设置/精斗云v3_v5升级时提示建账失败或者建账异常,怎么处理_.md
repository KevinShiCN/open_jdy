# 精斗云v3/v5升级时提示建账失败或者建账异常,怎么处理?

> 作者：winnie_wen | 阅读：235 | 有用：0

【问题描述】

  升级时提示建账失败或者建账异常

![](https://vip.kingdee.com/download/0109464bf7cebd574c18867e7b37ba6f77f0.png)

![](https://vip.kingdee.com/download/0109c5e12afc084e4704b872a0b70861ba76.png)

【解释说明】

  这种一般是升级失败后没有重新初始化账套再次升级导致的。可先手工建账,然后重新初始化账套后再进行升级。

1、先不要升级,正常创建一个账套:[如何启用服务?如何新增账套?](https://vip.kingdee.com/link/s/lC0LV)

2、创建账套之后再做重新初始化

![](https://vip.kingdee.com/download/01098dba1a972e0c4197babdb1fb3ee48209.png)
选择全部数据初始化

![](https://vip.kingdee.com/download/0109db15b04c1f6549ebbc77f8788f892586.png)
重新回到升级界面操作升级
