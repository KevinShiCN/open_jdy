# 批次自动填充规则应用

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：应用案例, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：607 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/700363638896160000)

# **1 业务场景**

- 启用了保质期管理，希望按照近效期的先出
- 启用了批次管理，且批号是有流水号顺序的，希望可以按批次顺序来出库
- 同时启用了批次+保质期的商品，希望在出库时，优先考虑保质期，即优先发出到期日更近的，其次再考虑批次
-

# **2 解决方案**

针对上述场景，可以通过星辰的批次自动填充规则来实现。

# **3 适用版本**

星辰3.0以上所有版本

# **4 应用流程**

打开系统设置<系统参数<进销存参数<仓存参数，选择出库时自动填充规则。

下面以旗舰版为例进行说明。

## 4.1 订单批次为空的情况

假设有销售订单，批次为空：

![](https://vip.kingdee.com/download/010972883a3a8e2b44fb927af199cf299fef.png)

且存在批次保质期商品库存如下：

![](https://vip.kingdee.com/download/01093a0e43388ded430fac72fb94427966a4.png)

那么不同的出库自动填充规则下，效果各不相同：

| 出库批次填充规则 | 下推销售出库单效果 |
| --- | --- |
| 手工选择 | 如果开启了“手工选择”，销售订单下推销售出库单时，批次为空，需要用户自己选择 |
| 批号顺序出库 | 如果开启了“批号顺序出库”，销售订单下推销售出库单时，系统会优先带出批号中流水号更小，同时结合订单数量和库存数量来自动带出如果把销售订单数量改成20，那么订单下推时，就会带自动出2个相同的批次，数量分别是10 |
| 近效期先出 | 如果开启了“近效期先出”，销售订单下推销售出库单时，系统会优先带出所有批次中，到期日最近的，换句话说，最早到期的，同时结合订单数量和库存数量来自动带出。 |

>
注意：对于多单位商品，当销售订单录入的是大单位，且下推出库时，存在某个批次的数量*换算率不等于基本数量，则该商品行不自动拆分，无法自动带出批次。

示例：

1箱=10kg，箱小数位为0，kg小数位为2。

销售订单录入了仓库及数量1箱，下推出库时，此时库存还剩1kg，换算出来的箱是0.1箱，由于箱小数位为0，四舍五入就是0箱，所以出库单上数量为空，也就是所谓【数量*换算率不等于基本数量】，因此无法带出对应的批次。

解决办法：需要对销售订单的开单单位进行修改，改成小单位。

## 4.2 订单批次不为空的情况

注意：订单支持录入批次、保质期的功能已经发布至星辰3.1了，详情查看：[批次专题优化：订单支持批次保质期](https://vip.kingdee.com/link/s/ZhpX0)

假设有销售订单，批次不为空：

![](https://vip.kingdee.com/download/0109a9803c61d7b843d69a84c24525b437e0.png)

那么销售订单下推销售出库单时，不管系统的自动填充规则选的是什么，都会自动携带上游订单上的批次。同时也允许用户手工修改批次。

![](https://vip.kingdee.com/download/01098787a41691c04c66946f5550212a0144.png)

## 4.3 直接新增销售出库单

如果您的业务是不做销售订单，直接录销售出库单，并且希望自动填充批次的话，前提条件一定是要填写分录上的仓库，如果不填写则不会自动带出。

| 出库时自动填充规则 | 出库单效果 | 说明 |
| --- | --- | --- |
| 手工选择 |  | 自动带出的是按照优先级：仓库、仓位、到期日、批次携带的。也就是近效期优先级会高于批次。 |
| 批号顺序出库 | 仓库为空的情况下，不会自动带出：仓库不为空的情况，才会自动带出，且只带出符合该规则的最近一次批次的库存，如果该批次库存小于订单数量，也不会再自动带出其他批次： | 如果开单数量大于库存数量，这种情况下建议分录先不要填写仓库，而是点击批次弹窗，过滤好仓库仓位等条件，同时使用“批次”表头的快捷排序升序，然后再手工录入“开单数量”注意：目前此弹窗中，自动分配的规则是到期日优先级高于批次的，因此如果您的系统参数是“批号顺序出库”，请不要使用自动分配，需要手工录入数量。 |
| 近效期先出 | 优先带出到期日最近的一条批次记录，如果该批次库存小于订单数量，也不会再自动带出其他批次： | 如果开单数量大于库存数量，这种情况下建议分录先不要填写仓库，而是点击批次弹窗，使用“总开单数量”并且自动分配来填写，这样就会按一定的规则来将数量分配完。具体规则说明可以看批次弹窗中的说明。 |
