# 如何查询多账套销售汇总表?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：Simone邓 | 阅读：32 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/805242064522356992)

【问题描述】

如何查询多账套销售汇总表?

【适用版本】

星辰3.0以上专业版/旗舰版

**注意:该功能仅专业版支持,且目前处于灰度测试阶段。如需提前体验,请复制工作台的服务 ID 到售后在线服务界面发送给人工客服,申请加入灰度测试。**

【业务场景】

随着业务规模的扩大,小微企业逐渐发展成为多个独立核算实体公司,如多个销售公司或供产销分离的多个公司。多个销售公司可能经营相同的业务,也可能按地区、品类等划分经营,公司之间存在调拨等销售协同;产供销分离的多个公司由上游公司统一生产或采购后向下游公司供货,下游公司或者上下游多公司分别对外销售。为了掌握企业整体经营情况,管理者需要统一分析各公司的销售情况,但各公司的业务数据分散在多个账套,给业务的全局洞察带来不便。

此次新增“多账套销售汇总表”,支持在同一服务下购买多个账套后,在一个账套按客户/业务员/部门/仓库/商品/品牌等多个维度,自由选择维度组合,汇总多个账套的销售数据,并显示数据来源的账套。

**特殊**情况下**,当多个销售公司经营相同的业务,**且**对应的多个账套的基础资料相同时,支持按客户名称/业务员名称/部门名称/仓库名称/商品名称/商品品牌名称等维度,将不同账套的销售数据合计。**

**【操作步骤】**

**1、**需要**服务管理员**或者**账套管理,**在系统参数勾选启用合并报表

![](https://vip.kingdee.com/download/0109577e502d62404d048b38f8828309619d.png)

2、【系统设置】-【组织机构设置】设置母公司账套

![](https://vip.kingdee.com/download/01092293d1aafe134b929d7607cbf857339e.png)
点击账套名称将销售汇总按钮开启

![](https://vip.kingdee.com/download/0109b2800d6eb07046f7b43fe55b01bc812b.png)
3、销售管理--销售报表--多账套销售汇总表![](https://vip.kingdee.com/download/010901f8e4d76e8d4b999db7b6fa020ea8cb.png)
可以选择多个账套汇总查询

![](https://vip.kingdee.com/download/0109d7e164dde5c446c48658cf8157fbd603.png)

可选择不同的汇总依据查询,并且可自定义增加不同维度的汇总依据

![](https://vip.kingdee.com/download/0109fa52fb425f854bd28de31845bc5e72fb.png)

![](https://vip.kingdee.com/download/01091186e580cc8644b082d241b7950451c9.png)
**4、计算毛利**

报表默认不显示销售单据的出库成本、毛利,勾选计算毛利参数后显示。

①销售成本:参与销售汇总的账套在各账套分别出库成本核算后,在多账套销售汇总表中分别取各单据上的成本再进行汇总。

②销售收入:

专业版——销售收入=销售出库单的实际不含税金额;

旗舰版——销售收入取决于选项参数

当销售收入取“销售出库单实际不含税金额”,销售收入参照销售出库单取;

当销售收入取“数量*开票单价”,销售收入参照销售发票取,即销售收入=销售数量(来源:基本单位的销售数量)*开票单价(来源:已开票金额/已开票基本数量)。
