# 进销存参数——第三方结算

# 1 简介

## 1.1 应用场景

甲乙双方发送业务往来后，应收应付是发生在甲与第三方丙的场景。例如：

• 在采购业务中，本公司向A公司采购，应付账款记入B公司；B就是相对于本公司而言的结算的第三方。

• 在销售业务中，本公司销售给C公司，应收账款记入D公司。D就是相对于本公司而言的结算的第三方。

## 1.2 适用版本

星辰3.2以上所有版本

### 1.3 设置路径

系统设置-进销存参数-全局参数

>
**注意：启用参数后不允许取消，请谨慎操作。**

# 2 应用流程

### 2.1 新增与维护

结算客户/供应商是从客户/供应商资料里获取的，如果客户/供应商资料还没有维护，不支持新增/引入的时候直接填写结算客户，系统会提示基础数据不存在。

### **2.2 基础资料及开单应用**

• 客户/供应商资料可维护“结算客户/供应商”

• 销售/采购类单据可选择“结算客户/供应商”
