# 打印实践案例——如何打印单据上的客户收货地址/供应商发货地址及联系人及手机信息？如何打印客户/供应商资料中的联系人信息到单据上？

> 作者：进销存赵嘉丽 | 阅读：1919 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/551132102285376768)

## **1 应用场景**

需要打印单据上的客户收货地址/供应商发货地址和联系人信息，但是不知道打印模板改怎么设计

## **2 适用版本**

星辰3.1以上所有版本

## 3 特别说明

### 3.1 客户/供应商资料打印取值说明

1）针对基础资料中的单据头信息，打印时可以直接取到。例如：客户资料中的备注、结算期限等

- **客户单据头信息**

**![上传图片](https://vip.kingdee.com/download/010011390db6b84e4c9782fe89b12e965551.png)**

- **供应商单据头信息**

![上传图片](https://vip.kingdee.com/download/01009d4cec5c73a343e9878b55f30071bfef.png)

2）针对基础资料单据体字段，则是取决于单据上存储的值，打印无法直接获取，例如：联系人、手机等。

- **单据体信息**

![上传图片](https://vip.kingdee.com/download/0100e82d04406cd146719a1430a8f4b56c6e.png)

3）单据打印不能直接取到客户/供应商资料中的联系人信息，而是通过单据上存储的**客户收货地址/供应商发货地址**来打印的，如果这个两个地址字段在单据上没值，是不会从基础资料中直接获取的，所以在确认打印是否生效时，请务必先检查单据上的**客户收货地址/供应商发货地址**的值，而不是去检查基础资料上的值！

| 单据类型 | 打印时，客户/供应商资料中的联系人信息字段取值来源 |
| --- | --- |
| 销售类单据 |  |
| 采购类单据 |  |

### 3.2 销售和采购类单据上的发货地址、收货地址打印取值说明

- 打印时也是需要保证单据上该字段有值，如果没值，也是不会打印的
- 取值来源**只跟单据头的仓库对应的基础资料里的地址有关**，跟客户/供应商资料中的地址**没有任何关系**

| 单据类型 | 字段 | 携带逻辑 | 仓库地址的维护路径 |
| --- | --- | --- | --- |
| 销售类单据 | 发货地址 | 点击发货地址弹窗后，选择仓库后带出仓库资料中的地址，也支持手工新增 |  |
| 采购类单据 | 收货地址 | 填写单据头仓库后，自动携带仓库资料中的地址，也支持手工新增 |  |

### 3.3 打印是所见即所得&后来更新的不影响之前录入的

如果单据上没有这个值，意味着没存库，那么打印就是空的。特别是先开单，后维护的信息，系统是不会同步更新到单据上的，如果需要带出基础资料最新的值，建议重新录入对应的字段，触发更新

## **4 操作流程**

接下来，我们以销售出库单为例，具体演示如何设计地址打印：

## **方法一：直接复制（也是最快捷的）**

1）直接打开系统的预置模板

![](https://vip.kingdee.com/download/01008514b3c8210d45ad9ddc589caf9aa883.png)

2）然后复制预置模板中对应的地址字段即可，在销售出库单预置模板中，对应的则是“收货地址”后面的公式字段：

![](https://vip.kingdee.com/download/0100dd47009a555e4501b0e81233bc5e5053.png)

系统的预设公式如下：拼接了客户联系人分录里的 “行政区划”

![](https://vip.kingdee.com/download/0100ef1b7252b1304e729ba41bb029e9d530.png)

对应的基础资料位置：

![](https://vip.kingdee.com/download/0100bece11f41677444dab431cb082e298ad.png)

直接用ctrl+C复制公式字段，

![](https://vip.kingdee.com/download/0100bc52866bb7464b1b95a13bd6fc462034.png)

然后打开一个新的模板，直接ctrl+V粘贴即可

![](https://vip.kingdee.com/download/0100b283a8c36d894e498ed05c08d7550605.png)

上面方法是最不容易出错的，建议不熟练的用户可以用此方法。

### **方法二：自定义设计**

1）如果系统提供的预置模板不是您需要的，可以复制一个系统的预置模板，然后再编辑

![](https://vip.kingdee.com/download/01004f2e8e5565994b98a92ae449d7c5978e.png)

2）打开复制后的模板，点击收货地址后面的公式进行编辑：

![](https://vip.kingdee.com/download/01008964abda08de4d51aa0252272a3c9934.png)

![](https://vip.kingdee.com/download/0100e1a0dd6867f349cd843078e0c4fc0198.png)

3）假设我们需要添加在这个地址上，添加客户联系人里的联系人、手机

![](https://vip.kingdee.com/download/01002083efc4a747459693db2b466b478c64.png)

4）那么我们在公式弹窗里可以看到系统预置的地址字段，如果有不需要的字段（如：客户国家.名称），也可以删除，然后在数据源中找到要添加的字段，

![](https://vip.kingdee.com/download/01008ce249659691499b80142e96ac5c37ed.png)

点击后可以直接添加到公式里，添加完成后如下图。点击“公式校验”，出现“校验成功”后，点击“确定”，然后保存模板即可。

![](https://vip.kingdee.com/download/0100744b872b7bb74fb6a39b39974625c309.png)
