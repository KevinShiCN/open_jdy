# 打印实践案例——金额及大小写设置及应用

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：快速入门, 精华汇总, 应用案例
> 作者：进销存赵嘉丽 | 阅读：1428 | 有用：3

## 1 应用场景
在打单时，需要根据实际业务需求，对金额的显示做一些个性化的处理，例如显示币别符号、显示币别代码、或者转化成中文大写和英文大写。

## 2 适用版本

星辰3.1以上所有版本

## 3 应用流程

### 应用一：如实打印数字中文或英文大写

| 示例（以金额900.00为例） | 打印设置 | 预览效果（点击图片可放大查看） |
| --- | --- | --- |
| 数字中文大写预期结果：玖佰点零零 |  |  |
| 2 | 如果是美元等外币，需要先选择币别类型才会生效，不选择“币别类型”，只选择英文大写，则如实打印阿拉伯数字 |  |

### 应用二：币别代码、币别符号和金额大写的综合应用

币别代码即货币的英文大写字母代码，币别符号则是货币对应的符号，如下图：

|  | 人民币 | 欧元 | 美金 | 泰铢 |
| --- | --- | --- | --- | --- |
| 币别代码 | CNY | EUR | USD | THB |
| 币别符号 | ￥ | € | $ | ฿ |

下面以币别类型 为“人民币”，金额900元为例说明：

1）单击要设置的文本或双击选中数据表格中要设置的单元格，格式选择“金额”；

2）具体设置：

| 使用场景 | 设置说明 | 打印模板设置 | 预览效果（点击图片可放大查看） |
| --- | --- | --- | --- |
| 打印币别代码+币别符号+金额例如：CNY￥900 | 需勾选“币别符号”、“币别代码”。“金额大写”选“不使用” |  |  |
| 打印币别符号+金额的中文大写例如：人民币玖佰元整 | 需勾选“金额大写”及“币别符号” |  |  |
| 只打印金额的中文大写例如：玖佰元整 | 只需勾选“金额大写”，其他不用勾 |  |  |
| 注：1、币别类型 为“从单据中获取币别”、“港币”、“美元”设置同理。2、选择“金额大写”时，只勾选“显示币别代码”，是不会显示“人民币”三个字的，“人民币”三个字一定是跟着“显示币别符号”结合使用的。 |  |  |  |
