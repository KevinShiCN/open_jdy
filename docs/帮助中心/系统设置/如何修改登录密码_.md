# 如何修改登录密码?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题, 快速入门
> 作者：杨桂平 | 阅读：2271 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/375690727466233344)

[操作流程]

1、[操作流程]

1,[操作步骤]登陆精斗云官网:www.jdy.com ,点击右上角头像,下拉选择[账号设置],进入后可看到[修改密码]按钮,输入原密码、新密码,即可修改密码。

![](https://vip.kingdee.com/download/0109ce7f1909f93845c295ecce8de1fd5098.png)

2、点击[修改密码]。

![](https://vip.kingdee.com/download/01096b2cbcc2e25b49069577bf7b86abcb87.png)

![](https://vip.kingdee.com/download/010999d55860de12468fabfdd82efd3fa280.png)

[温馨提示]

1、登录用户名不可修改,个人信息以及密码可进行修改

2、若忘记登录密码,点击[系统密码忘了,如何找回密码?](https://vip.kingdee.com/knowledge/specialDetail/375651802161025536?category=375652076635585280&id=360910517075715840&productLineId=6#)
