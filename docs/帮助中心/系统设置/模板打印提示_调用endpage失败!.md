# 模板打印提示:调用endpage失败!

【问题描述】

模板打印提示:调用EndPage失败,如何处理?

![](https://vip.kingdee.com/download/010967bcf378da3445dbadd8084be4c6481b.png)

【解释说明】

出现上述提示,可以通过一下思路处理:
1、建议用谷歌浏览器,或者清理浏览器缓存,重新登录进行打印

[如何清理浏览器缓存?](https://vip.kingdee.com/link/s/ZMdtO)

2、检查电脑C盘的空间是否不足,清理空间再打印

![](https://vip.kingdee.com/download/0109440ccbbef7264dcca86bb695cd79d9b7.png)

3、重新安装打印插件。

4、如果使用的是小微客户端,可以尝试切换浏览器打印排查,若浏览器可正常打印建议重新安装客户端试一下

如果以上步骤都操作还有问题,可以通过产品右上角点击售后在线服务获取帮助。
