# 工作流打印——可以不打印提交环节或者只打印同意的审批结果吗？可以倒序打印审批结果吗？

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：功能介绍, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：1009 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/492778804784553984)

**【问题概述】**

工作流打印可以不打印提交环节或者只打印同意的审批结果吗？可以倒序打印审批结果吗？

**【解决方案】**

可以，开启了工作流的单据，在打印设计器中通过导航栏的“工作流设置”可以实现。

![上传图片](https://vip.kingdee.com/download/0100e380d8b593c844a1aa2faede9cef333b.png)

# **1 适用版本**

星辰3.1以上的专业版和旗舰版

# **2 功能介绍**

## **2.1审批路线**

**1、****所有审批记录：**

打印单据的所有审批记录，包括同意、驳回的记录。

**2、所有审批同意的记录：**

仅包括审批同意的记录

**3、当前节点之前最新审批同意的记录：**

当单据被多次驳回到提交节点的时候，勾选开关仅打印最后一次提交并且审批通过的记录，不打印之前被驳回过的审批记录。

## **2.2 审批线路排序方式**

**1、默认为“空”时打印排序如下**：

①选“所有审批记录”时，打印记录为顺序

②选“所有审批同意记录”时，打印记录为顺序

③选“当前节点之前最新审批同意记录”时，打印记录为倒序、

**2、顺序：所有审批记录均按顺序打印**

**3、倒序：所有审批记录均按倒序打印**

## **2.3 打印时包含人工节点**

将打印人工节点的记录，注意：子流程 的第一个人工节点不会被打印出来。

**人工节点=提交节点，勾选了意味着审批过程的“提交”节点不会打印。**

## **2.4 打印时包含影像上传节点（****星辰不涉及这个功能，可以忽略****）**

默认在审批路线中不展示，勾选则展示影像上传的节点。

## **2.5 打印时包含子流程节点（****星辰不涉及这个功能，可以忽略****）**

默认在审批路线中审批记录仅显示主流程记录，未显示子流程。勾选后则可展示子流程审批记录。

# **3 应用流程**

1）需要打印工作流的单据必须有启用工作流，启用后，对应单据的打印模板在创建数据表格时才会有工作流的数据源

![](https://vip.kingdee.com/download/0100241a8396554945979cc20a56c5aaf879.png)

2） 按需要打印工作流的信息设置好对应的数据表格（如下图）

![](https://vip.kingdee.com/download/010011ed0802b254419a8581f3280e0564c1.png)

3）在打印模板的导航栏里有“工作流设置”

![](https://vip.kingdee.com/download/010091d916390ba6492a969fc54905a9b56f.png)

4）根据实际打印的需求进行设置后，保存模板后即可打印。
