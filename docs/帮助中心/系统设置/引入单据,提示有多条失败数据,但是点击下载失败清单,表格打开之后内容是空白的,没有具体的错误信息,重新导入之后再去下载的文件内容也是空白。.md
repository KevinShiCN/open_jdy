# 引入单据,提示有多条失败数据,但是点击下载失败清单,表格打开之后内容是空白的,没有具体的错误信息,重新导入之后再去下载的文件内容也是空白。

> 作者：进销存熊诗琴 | 阅读：206 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/749434309647562240)

【问题描述】

1)引入单据,提示有多条失败数据,但是点击下载失败清单,表格打开之后内容是空白的,没有具体的错误信息,重新导入之后再去下载的文件内容也是空白。

2)引入单据或者基础资料,提示引入失败,但是失败文件里没有提示具体的错误原因,只有“系统预置模版”/“用户自定义模版”这几个字。

![](https://vip.kingdee.com/download/0109af6910100a88441dbdea22bfbb46829d.png)
![](https://vip.kingdee.com/download/01096ef360c97c8d4750a8cdf571f9bb25f6.png)

【适用版本】

金蝶云·星辰3.1及以上所有版本

【操作指引】

方法一:用Office打开下载的文件,能够查看到具体的报错内容

![](https://vip.kingdee.com/download/0109f685a1ea19e54d418cdd022f36eb7b0a.png)

- 方案二:如果是用WPS打开文件,没有显示出具体的报错信息,
- 1)请在工作表标签栏的任意一个**可见的工作表标签**上**单击鼠标右键;**
- 2)在弹出的菜单中,选择 **“取消隐藏”**。这时会弹出一个对话框,列表里会显示出所有被隐藏的工作表名称;

- 3)选中您想要显示的工作表名称(一般是第一个sheet),然后点击 **“确定”** 按钮

![](https://vip.kingdee.com/download/0109986109d99b884056a2d8bdf2c0a6a54a.gif)
