# 星辰3.1新旧模板使用说明

## 1 新旧模板的说明

### 问题1：什么是新模板旧模板？

答：通过2.0升级到3.1的账套，原来2.0的模板就是旧模板，只能通过升级携带到星辰3.1，无法在3.1内新增。

如果您的账套是新购的，则不存在旧模板的问题，直接在星辰3.1内新增的，则是新模板。

### 问题2： 怎么知道是新模板还是旧模板？

答：在打印模板列表里，通过“模板类型”可以区分。

![上传图片](https://vip.kingdee.com/download/01000386108a374d448cbaf4b97b1cce16a8.png)两种模板有什么差异？见下表说明：

| 模板类型 | 版本功能差异 |  |
| --- | --- | --- |
| 星辰2.0版本 | 星辰3.1版本 |  |
| 2.0旧模板 | 新增、修改、复制、另存为、下载、引入 | 只能打印、删除、禁用。不支持修改、复制、另存为、下载、引入、导出为excel |
| 3.1新模板 | 不支持使用 | 支持所有3.1的打印功能，支持导出为excel |

## 2 星辰3.1新打印模板

各位用户请注意，新版打印上线后不影响旧版的打印模板，支持继续在3.1内使用，但是不能再编辑修改。新版打印模板为各用户提供一种新的设计模板的方式，功能丰富且体验极佳。其目的给大家降低打印设计门槛，提升设计效率并提供更多打印相关能力。

## 3 星辰2.0旧模板相关问题

### 问题1： 升级后，2.0的旧模板可以直接打印吗？

答：**支持。**从星辰2.0升级到星辰3.1的租户，原来2.0的旧模板支持在3.1内直接打印，如旧模板内含有新模板不支持的功能（如组合字段），请您往下看第4点，需要升级到3.1新模板才能编辑修改。

### 问题2：升级后，2.0的旧模板还能修改、下载、导入吗？

答：**都不可以。**由于底层代码差异非常大，原来的旧模板都**不再支持编码、下载和导入**。未来将下线旧版打印模板（不提供创建旧版模板的入口，且功能将不更新），若您有未满足的模板需求、新的打印场景，请用新版打印进行设计，以免影响您的使用。

### 问题3：升级后，2.0的旧模板还能使用导出excel、合并打印等3.1的新功能吗？

答：都不可以。由于底层代码差异非常大，原来的旧模板都**不兼容所有3.1的新功能，**如需使用，请将您的旧模板升级到新模板中。

## 4 旧打印模板升级方法说明

如果是通过升级上来的账套，2.0旧模板可以直接在3.1系统里面升级为新模板，具体操作见下文说明。

1）先在星辰3.1打印模板列表右上角找到“模板升级”按钮

![](https://vip.kingdee.com/download/0100236088097f7940db963afad97705e4db.png)

2）找到要升级的旧模板，然后点击“下一步”，开始处理

![](https://vip.kingdee.com/download/01000100d6b4f37d463dbaeddff0086a9658.png)

3）当升级进度条完成后，会显示处理差异结果

![](https://vip.kingdee.com/download/01001d3cbdc9882141b598be5399feb17a2e.png)

4）当提示“无差异，可升级”，选中后继续点击“下一步”即可升级完成

![](https://vip.kingdee.com/download/0100ec6141a4f34241c8905820fa47d16586.png)

5）点击去查看可以在打印模板列表看到“新模板”，模板编码和模板名称都会跟旧模板有区别，分别多了个“_NEW”和“_新”的后缀

![](https://vip.kingdee.com/download/010096ebacc2a92c4fb09c93b2b6ee9cffd5.png)

6）使用新模板直接打印即可，也可以点击模板编码后对模板进行修改操作。
