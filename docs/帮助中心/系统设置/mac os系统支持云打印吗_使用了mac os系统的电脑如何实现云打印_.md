# mac os系统支持云打印吗?使用了mac os系统的电脑如何实现云打印?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：381 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/710658670018782464)

【问题概述】

mac OS系统支持云打印吗?使用了mac OS系统的电脑如何实现云打印?

【答案解析】

由于星辰的插件不支持mac OS系统,故不能直接使用云打印

【变通方案】

如果您的公司同时存在使用mac OS系统与Windows系统的电脑,那么可以这样来变通处理:

1、用Windows系统的电脑连接打印机;

2、在Windows系统的电脑上安装打印插件(在打印设置中直接下载即可)并且配置好云打印的打印机;

3、安装完成后,这样使用mac OS系统的电脑就可以发送打印任务到Windows系统的电脑,从而完成打印。

注意:如果您所使用的打印机所连接的电脑设备全部都是mac系统,那么暂时没有办法处理电脑端的云打印。

【适用版本】

星辰3.1以上所有版本
