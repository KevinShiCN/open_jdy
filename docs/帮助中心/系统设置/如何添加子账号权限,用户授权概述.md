# 如何添加子账号权限,用户授权概述

【问题描述】

 如何添加子账号权限,用户授权概述

【解释说明】

①购买软件之后管理员可以根据公司内部每个员工岗位分工情况给子账号分配对应的权限。**管理员用户不需要进行授权操作,默认有所有权限**。

②如果需要授予其他账套的权限,请先切换账套后,再启用对应用户并进行授权。

③该操作仅支持在**网页端操作**,手机端暂时不支持。

④管理员需要授权子账号哪些权限,由公司内部自行决定,系统支持**灵活配置**,无强制统一要求。

⑤若系统设置里没有用户管理的功能,可先确认是否登录的管理员账号,如是管理员账号则需检查是否把用户管理菜单隐藏了

![](https://vip.kingdee.com/download/0109053b07b0a4f449738f52f9a4a83189d9.png)

【操作指引】

**一、管理员账号登录**

管理员账号(服务管理员或者账套管理员)登录系统后依次点击【系统设置】-【用户管理】,如图:

![](https://vip.kingdee.com/download/010978cce30968fd4b559dc8e130c296715e.png)

然后,在用户管理界面,勾选需要授权的用户,点击【授权】按钮,可对用户进行权限设置,如图:

![](https://vip.kingdee.com/download/01090693ae33e5704c5db0acb73fb512f31c.png)点击【功能授权】,勾选对应功能权限,点击保存即可完成功能授权,如图:
![](https://vip.kingdee.com/download/0109e9213ffa317e4bffb43f6f6498140418.png)

**二、授权**

授权分为“功能授权”、“字段授权“、”数据授权“

![](https://vip.kingdee.com/download/01098b0b74db435447da8e71fe8ed621de60.png)

**功能授权**:是否有操作某些功能的权限,比如:销售出库单新增、修改、查询等,需要有权限就勾选对应项。

![](https://vip.kingdee.com/download/0109ee95a9b648f1484dacf60299802d9217.png)

**字段授权**:对字段设置是否“禁止编辑”、“禁止查看”,可以限制操作员查看某些敏感字段的权限,例如采购价、参考成本、销售价等等。

![](https://vip.kingdee.com/download/01090dc6b2cd82b341db818f6bf8886f53ba.png)

**数据授权**:可以对基础资料进行数据授权设置,条件可以选择到分类、编码、名称等。例如A用户只能查看某一类别的供应商,可以按下图方式设置。

![](https://vip.kingdee.com/download/0109babd42b648ae4560a198ecac130cd5c0.png)

**三、设置管理员**

若需要将用户设置为管理员,选择对应用户,单击<设为管理员>,可以将用户设置为账套管理员,账套管理员有账套所有功能权限,且可以对普通用户进行授权操作。

![](https://vip.kingdee.com/download/0109cc2150a60b2b41888f026ffe043e3c65.png)
