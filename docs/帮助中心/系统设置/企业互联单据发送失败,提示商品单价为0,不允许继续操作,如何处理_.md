# 企业互联单据发送失败,提示商品单价为0,不允许继续操作,如何处理?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：599 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/398279485222773504)

【问题描述】

企业互联单据发送失败,提示商品单价为0,不允许继续操作,如何处理?

如下图:

![](https://vip.kingdee.com/download/010902213b05977f45beb59a7150cf3fa0d3.png)![](https://vip.kingdee.com/download/010908d9fefb181141a1b3848f6ad492ab77.png)

【解决方案】

1、单据中录入单价后再发送

2、若是采购订单发送失败,通知销售方,到【系统设置】-【系统参数】-【进销存参数】-【销售参数】,“若您在新增销售类单据时销售单价为0, 您希望”选择【不控制】

![](https://vip.kingdee.com/download/010965c44e5fd9874c4db39598e327ea80cf.png)
若是销售出单发送失败,通知采购方在【系统设置】-【系统参数】-【进销存参数】-【采购参数】中“若您在新增采购类单据时采购单价为0,您希望”选择【不控制】

![](https://vip.kingdee.com/download/010953e22d6cf51241588631525c5ca1c61c.png)

【温馨提示】

参数只能设置【不控制】,选择【提示、允许继续操作】也是会发送失败
