# **授权不生效**

## **1、做了用户授权，但是子账号还是能看到他权限以外的信息，授权不生效，是什么原因？**

回答：请务必先检查是否有对该子账号同时做了**用户组授权**，当用户授权和用户组授权同时启用时，二者的授权是叠加的。

下面举例说明：

假设子账号“张三”在用户组“销售组”里，而“销售组”数据授权了只能看仓库A，同时单独对张三数据授权了能看仓库B，那么这个时候他的授权就会叠加，仓库A、B他全部都能看到。

如果想实现“张三”既在“销售组”里又只能查看仓库A，那么只对销售组做数据授权仓库A即可，不必单独对张三做仓库的数据授权。

## **2、为什么授权子账号后子账号看不到该功能的入口？**

**解决办法：**检查一下子账号的页面的【自定义导航】设置是否隐藏了（没有勾选）该功能。

![上传图片](https://vip.kingdee.com/download/01007a986a3c1629483da9073fa3368ea9c4.png)

## **3、用户没有做单独的用户授权的，但做了用户组授权，去该用户组下单独查看子账户的权限，为什么都是空白的？**

回答：用户组授权的权限项，选中子账户点击"授权"是看不到**用户组**的权限的。只能通过先点击左侧分组，再点击右上角的"用户组授权"查看**该分组**的权限。

# **字段授权**

## **1、不允许子账号查看商品基础资料中的配送价、批发价、零售价等价格相关字段，要在哪里设置？**

回答：对子账号单个用户做【字段授权】，找到商品后，添加要授权的字段，选中后保存即可。

![](https://vip.kingdee.com/download/010020b2761767a34b54b52d944407f8abba.png)

## **2、单据中（例如采购入库单、销售出库单）禁止子账号查看商品“单价”中的任何成本（如下图），要在哪里授权？**

![上传图片](https://vip.kingdee.com/download/0100cd8a7eded8c842bab4e1a2592778d441.png)

回答：进销存单据中“单价、含税单价”下拉框中的**价格权限**都在【字段授权>基础资料>商品信息>选择字段】中进行设置。

![上传图片](https://vip.kingdee.com/download/0100256e7488e8ab4530a25b8930326d28f8.png)

# **数据授权**

## **1、可以设置单据只可以看到业务员为自己的单据吗？**

回答：可以，做制单人的数据授权。参考知识：[单据内的审核人和制单人的姓名是取哪里的姓名？如何修改？  按制单人数据授权时，制单人账号是录入什么账号？](https://vip.kingdee.com/link/s/MUjtX)

## **2、同一个用户名 ，有2类商品，一类需要不让看成本，另一类可以看成本，请问如何设置？**

回答：目前只能要么按商品分类授权查看范围（数据授权），要么授权商品的成本查看权限（字段授权），像这种从商品类型去限制成本的查看**目前不支持**，因为属于两类不同的授权。

## **3、供应商授权后可以只看到自己的供应商了，但是在商品信息中看默认供应商还可以看到别的供应商，这里有字段控制吗？**

回答：供应商授权后，还需在数据授权-商品信息里做“默认供应商”的字段授权设置。注意“供应商名称”与“默认供应商”是2个不同的字段，需要在数据授权中分别做字段的控制。

![上传图片](https://vip.kingdee.com/download/0100fb89daf0c91a45ac87da320c3e5a8fdb.png)

## **4、客户想看所有的商品信息，但是当“默认供应商”非子账号授权的“供应商名称”的字段显示为空，可以实现吗？**

回答：目前**不支持**。一旦做了数据授权，就只能展示授权字段对应的信息，不能交叉过滤。

举例：假设对A公司的子账户张三**数据授权**了字段“供应商名称”、“默认供应商”均为“金蝶”，那么张三在商品信息列表只能看到“默认供应商”为“金蝶”的所有商品，看不了其他商品信息。

## **6、数据授权的“条件”设置可增加“不包括”吗？**

回答：可以，参考知识：[https://vip.kingdee.com/link/s/MUjyA](https://vip.kingdee.com/link/s/MUjyA)

## 7、数据授权里，使用“不包括”设置了多个条件后，为啥不生效，还是可以看到数据？

![上传图片](https://vip.kingdee.com/download/01003c2679db610545c0a6e496e8c89b6ac7.png)

回答：原因在于使用了“或者”的逻辑。使用“不包括”条件时，逻辑**必须使用“并且”才能生效。**

![上传图片](https://vip.kingdee.com/download/01006bc51af2e97c470691e9f2ae8bc1236f.png)
