# 星辰2.0,打印模板中,设置绑定字段时,提示“请先设置该字段对应的数据源”是什么原因?如何解决?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：杨桂平 | 阅读：195 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/537436465878505984)

【故障现象】

打印模板中,选择绑定字段时,字段列表为空且提示““请先设置该字段对应的数据源”(如下图)

![](https://vip.kingdee.com/download/0109aab9a40d350b4ca59e25482dd401372d.png)

【故障原因】

出现上面的提示,是因为没有设置数据源造成的。

【解决方案】

第一步,在打印模板中,先点击左侧的大纲,找到“数据表格”后,设置右侧控件属性的数据源

第二步,点击“···”,选择要绑定的数据源,单击“确认”,设置完成。

![](https://vip.kingdee.com/download/0109e4d2567f965040198c6ec2b7a404d114.png)

完成以上2步后即可正常设置绑定字段。
