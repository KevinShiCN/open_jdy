# 打印设置——二维打印支持横向创建空白列

# 1 简介

## 1.1 功能介绍

当业务单据需要二维打印，但横向属性又较少时，打印出来纸张横向不能充满，打印效果不美观，通过设置空白的动态列，使得页面打印可以更饱满，灵活满足不同客户的打印场景。

![上传图片](https://vip.kingdee.com/download/0100bb0b9f09460f47cda0f9eee7f2b8263d.png)

## **1.2 适用版本**

星辰3.1以上所有版本

## **2 应用流程**

在打印-打印设置中设置横向显示列数，系统自动创建二维空白列，使打印内容饱满，布局均匀
