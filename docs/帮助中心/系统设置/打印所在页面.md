# 打印所在页面

# **1 简介  **

## **1.1 功能介绍**

- 应用于控制通用控件（即：图片、二维码、条形码、文本）的打印位置。
- 应用于数据表格的表尾区的打印位置。
- 打印位置包含：顺序打印（即默认的常规打印）、所有页面（即每页打印），只在第一页打印，只在最后一页打印

## **1.2 适用版本**

星辰3.1以所有版本

## **1.3 系统路径**

打印模板列表，打开具体的打印模板，选中通用任意控件后，在右侧控件属性中可以查看该参数。

# **2 通用控件的“打印所在页面”**

## **2.1 参数含义及应用场景**

| 类型 | 含义 | 应用场景 |
| --- | --- | --- |
| 顺序打印 | 控件按顺序动态适配，跟随上一控件位置打印。系统默认的打印方式。假设打印模板中包含了表头、表格、表尾三部分。表头字段比较长，设置了行高自适应，那么表格也会受到表头文本的影响，而动态向下移动。而表格又是是跟随单据上分录动态变化的， 因此打印的时候，表尾会等表格打印完所有的分录再打印。总的来说，每个控件的打印位置都跟随着上一个控件的变化而变化，不是固定的，这就是顺序打印。 | 单据内容不是固定，有时候多，有时候少，如实打印即可。打印模板新增的文本、图片、条码、二维码控件一般都是默认顺序打印。 |
| 所有页面 | 每一页都打印。打印位置是固定的，不受其他控件的影响。 | 一般适用于需要每一页都出现的内容或单据分录打印行数是固定的，如单据标题、公司logo、页码或者公章之类的图片、二维码等。 |
| 仅第一页 | 只在第一页打印。 | 一般适用于如单据标题、表头的字段、图片、条码等。 |
| 最后一页 | 只在最后一页打印。 | 一般适用于表尾的字段、图片。如公司公章。 |

# **3 数据表格表尾区的“打印所在页面”**

## 3.1 应用场景

单据中表尾的控件（如文本、图片、二维码等等）需要实现每页打印，而且打印单据内容超过1页。

## 3.2 应用流程

需要结合打印设置中的“每张打印分录数”（相关知识： [进销存单据打印设置-基础设置](https://vip.kingdee.com/link/s/ZpGMj)）一起使用。
