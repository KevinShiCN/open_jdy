# 启用了工作流的销售订单，提交时提示“单据XXXXX: 订金信息，收款账户、收款金额没有全部录入”是为什么？需要怎么处理？

【问题概述】

- 启用了工作流的销售订单，提交时提示“单据XXXXX: 订金信息，收款账户、收款金额没有全部录入”是为什么？需要怎么处理？
- 启用了工作流的采购订单，提交时提示“单据XXXXX: 订金信息，付款账户、收款金额没有全部录入”是为什么？需要怎么处理？

![上传图片](https://vip.kingdee.com/download/0100611d216e75e24eb7b48541ce2cbee500.png)

【原因说明】

当订单启用了工作流后，提交时，如果不填“收款账户/付款账户”容易有工作流异常，因此在审核时校验了“收款账户/付款账户”的必录性。

![上传图片](https://vip.kingdee.com/download/0100b114557e2a4343af97ec8e67c8d830ef.png)

【解决方案】

以采购订单为例：

- 下单时，**不需要订金**，那么订金相关的三个字段都不要填写，这样可以提交成功。

![上传图片](https://vip.kingdee.com/download/01000c4fd71ea6704eea84de8dab43aa41dd.png)

- 下单时，**需要订金**，那么“付款账户”一定要录入才能提交，而其他2个字段可录可不录，用户可以自己决定。

![上传图片](https://vip.kingdee.com/download/0100e103734576614cc493994d13a6a0e448.png)

【适用版本】

星辰3.1以上专业版/旗舰版
