# 打印设置——序列号拆分打印功能

# 1 简介

## 1.1 应用场景

客户打印出入库单据时，需要把序列号明细打印出来，方便拣货或核对。

## **1.2 适用版本**

星辰3.1以上所有版本

## **1.3 **适用单据

适用于包含序列号字段的进销存出入库单据，其中：采购申请单、销售订单、采购订单的不支持该功能。财务资金类单据的打印设置界面也不会显示该功能。

# **2 应用流程**

1） 打印模板中设置字段

需要先在单据的数据表格中绑定字段：在数据表格的数据行中选中单元格，然后在右侧的数据源中选择“序列号组合”即可。

![](https://vip.kingdee.com/download/010002af71556d004283a4a44c74547cdd7b.png)

2）在进销存出入库单据的打印设置中勾选“序列号拆分行打印”参数，默认不勾选；勾选后，序列号将按行展示并打印。
