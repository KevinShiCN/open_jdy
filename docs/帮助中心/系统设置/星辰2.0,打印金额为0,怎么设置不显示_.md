# 星辰2.0,打印金额为0,怎么设置不显示?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：杨桂平 | 阅读：363 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/537436499902541824)

【问题描述】

打印金额为0,怎么设置不显示?

【解决方案】

打开模板,选择对应的数值字段,点开样式的格式,选择数值,零值显示为选择“空”即可![](https://vip.kingdee.com/download/01090a454ab82ee94101bce4089163c4ed2e.png)比如下方的单据中数值为0

![](https://vip.kingdee.com/download/0109f13c8b95913046988fa756d079d9a02b.png)

打印预览的时候,显示为空值

![](https://vip.kingdee.com/download/01098b2d246cbe894bfe95f8023e95537c5a.png)
