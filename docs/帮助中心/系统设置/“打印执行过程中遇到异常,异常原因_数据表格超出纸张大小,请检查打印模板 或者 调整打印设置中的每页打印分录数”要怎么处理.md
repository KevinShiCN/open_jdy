# “打印执行过程中遇到异常,异常原因:数据表格超出纸张大小,请检查打印模板 或者 调整打印设置中的每页打印分录数”要怎么处理

> 作者：进销存赵嘉丽 | 阅读：2061 | 有用：122

**【问题描述】**

点击打印时,提示“打印执行过程中遇到异常,异常原因:数据表格超出纸张大小,请检查打印模板 或者 调整打印设置中的每页打印分录数”要怎么处理?

![](https://vip.kingdee.com/download/01094b7e266348cf4a428ca1688fdcc41f88.png)

**【原因说明】**

在打印设置中设置了每页打印分录数,且表头或表尾设置了每页打印(即打印所在页选了“所有页面”),而这个固定的分录数加上表头表尾的内容,**已经超过了一页纸所能打印的范围了。**

我们以销售出库单为例,当出现下面两种条件同时设置时,就可以复现这个问题:

条件1:

![](https://vip.kingdee.com/download/01090e5af5846565490f990876afcae03f24.png)

条件2:

![](https://vip.kingdee.com/download/0109fb25779f7fc24b348349aa69cd3a4017.png)
按以上的条件设置,打印时就会提示“打印执行过程中遇到异常,异常原因:数据表格超出纸张大小,请检查打印模板 或者 调整打印设置中的每页打印分录数”。

**【解决办法】**

以上面例子的模板为例,如果需要每页打印表头表尾时,实际只能打印8行分录,如下图:

![](https://vip.kingdee.com/download/0109e21068df9ba740358a88559deddcd8a9.png)
如果打印11行,那么表尾肯定不够位置了,所以系统才抛出提示“数据表格超出纸张大小”

**【解决方案】**

| 方案 | 打印设置或者模板设置 | 截图说明 |
| --- | --- | --- |
| 方案一 | 把【打印设置】中每页打印分录数改小。如果不能改小,就看方案二和三 |  |
| 方案二 | 修改数据行的行高 |  |
| 方案三 | 修改表头表尾的行间距 |  |
