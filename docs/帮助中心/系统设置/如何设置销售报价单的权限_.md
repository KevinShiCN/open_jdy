# 如何设置销售报价单的权限?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：ylim | 阅读：377 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/576936715076696832)

【问题描述】

如何设置销售报价单的权限?

【解决方案】

**管理员账号**登录进入账套后,点击【系统设置】-【用户管理】

![](https://vip.kingdee.com/download/01098783f86bf45c4df09e22c67b9f17c6ea.png)
勾选要授权的用户,点击【授权】

![](https://vip.kingdee.com/download/0109511963f367e64342b7a47a6537c3cdbe.png)
功能授权下点击【销售管理】,销售报价单的功能权限需要给到哪个权限勾选哪个权限,不勾选的功能就是没有权限

![](https://vip.kingdee.com/download/0109b9f8a3780ae548c1ab4584770bc26331.png)

比如勾选了查询和新增的权限,就是只能查询历史单据和新增单据,新增保存后不能修改、审核、反审核、删除等操作(没被勾选的功能都没有权限)
![](https://vip.kingdee.com/download/0109df2b5e1544c14951a0082ee55f97c8d2.png)
