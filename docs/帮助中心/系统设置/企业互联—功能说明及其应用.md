# 企业互联—功能说明及其应用

## 1、简介

### 1.1功能介绍

企业互联可以实现，企业上下游互联，实现人、数据、流程、和事物的联通，从而提高生产力，并推动企业可持续的差异化竞争。加入企业B是某品牌的总代，企业A是企业B的一级经销商，企业A需要向企业B采购一批商品，在其软件系统中下达采购订单，可直接发送到企业B的系统中，形成企业B的销售订单。

### 1.2适用版本

星辰所有版本适用

### 1.3应用场景

1)对于企业B来说 ，帮助企业快速经营，实时了解市场需求，快速决策，快速备货/发货，提高经营效率；

2)对于企业A来说，帮助企业快速要货，节省到货周期，把握市场先机；

3)企业互联，实现人、数据、流程、和事物的联通，从而提高生产力，并推动企业可持续的差异化竞争

### 1.4系统路径

系统设置>企业互联

### 1.5字段/按钮说明

**密钥：**就是账套id，用于系统识别账套唯一标识

**采购方**：下采购订单，并完成商品入库的公司

**销售方**：下销售订单，并完成商品出货的公司

**互联客户**：就是本公司销售的下游公司

**互联供应商**：就是本公司采购的上游公司

### 1.5特别说明

• 企业互联的设置是基于账套ID维度来设置，同一个租户下不同账套之间也支持配置企业互联；

• 企业互联是可以上下游互为买卖双方的，那么就需要在各自的账套里同时设置互联客户、互联供应商。

### 2.2详细操作流程

1）找到对接密钥，将本企业的对接密钥发给对方企业。需要企业A和企业B双方都操作。

![](https://vip.kingdee.com/download/0100f50d8df9e4284865a334185e5b7070bf.png)

2）以上方企业B为例，企业B在【企业互联—互联客户】页签新增客户对应关系，将客户发过来的对接密钥与客户资料关联，如下图：

![](https://vip.kingdee.com/download/0100b57b6751d6274fbfa4095e8882be8585.png)

3）企业A在【企业互联—互联供应商】页签新增供应商对应关系，将客户发过来的对接密钥与供应商资料关联，至此,企业A与企业B完成了互联,可以进行日常业务往来。

![](https://vip.kingdee.com/download/0100f5d4b00f099845d48f7527efd8966f2a.png)

4）在日常业务往来过程中，需要确定企业A和企业B之间的商品对应关系，通过【基础资料】—【外部商品库】-【客户】来实现，由企业B配置。

![](https://vip.kingdee.com/download/01002e93736aeb474ac383f188cecd72bbd3.png)

5）企业A需要向企业B采购一批商品，在本企业的管理系统中创建“采购订单”，审核后，点击【发送给供应商】。操作方:企业A

![](https://vip.kingdee.com/download/0100727c9f4be20c439b8dd0e188f12bc82e.png)

当采购方每天按需采购，形成多张采购订单，下午或者晚上统一发送给供货方，批量发送要货需求。支持在采购订单列表批量选单后 【发送给供应商】。

![](https://vip.kingdee.com/download/0100fa29b7bc840e43b29eec0cb31dde8989.png)

6）企业B接收到对方发送的订单，在本企业的账套中查看“销售订单”列表，可以查看到保存状态的销售订单，然后可以处理后续的发货业务。操作方:企业B

![](https://vip.kingdee.com/download/0100754963acfb24463c8638cc08c9391fb7.png)

7）企业B接收到对方发送的订单，在销售订单中下推销售出库单并审核后，企业B可以在单据详情或单据列表中点击《发送给客户》，实现将《销售出库单》发送到对方的《采购入库单》中，生成企业A保存状态的《采购入库单》。

操作方:企业B

![](https://vip.kingdee.com/download/0100ff8d39ed17184047bbf01c1ca801185a.png)

“发送给客户”此功能需企业B在《销售出库单》中打开此功能的权限项。

![](https://vip.kingdee.com/download/010012b32f7833bc43d78cef895e06bf57ec.png)
