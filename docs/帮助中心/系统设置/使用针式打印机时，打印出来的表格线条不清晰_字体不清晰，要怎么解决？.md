# 使用针式打印机时，打印出来的表格线条不清晰/字体不清晰，要怎么解决？

**【问题概述】**

使用针式打印机时，打印出来的表格线条不清晰/字体不清晰，要怎么解决？

**【解决方案】**

首先检查打印预览页面是否正常；如果正常，则可以排除打印模板的问题，通常是硬件问题。例如出现上述图片情况，通常是跟打印机的打印速度及打印质量有关：

**名词解释：**

- **打印质量**：即打印的分辨率，数字越大，越清晰，一般建议使用360*180dot，具体参数以各打印机实际支持的参数为准（可查看打印机属性中的高级项）

举例：某针式打印机的打印质量设置路径：

![](https://vip.kingdee.com/download/01001a6b272edc664465ac39f48570dea18c.png)

- **打印速度**：一般而言，当需要打印清晰度高的线条或文字时，建议使用**正常速度或高速**；如使用**超高速，**可能会导致打印头温度过高，造成打印效果不佳。

![](https://vip.kingdee.com/download/01004065eb82f80f45faac90bd9e2d250f13.png)

如出现上述问题，建议用户将打印机首选项-高级项中的打印质量调到最高，并将打印速度调到正常速度后重新打印即可。由于不同打印机首选项的设置各不相同，请根据实际打印机的参数进行设置。

>
注意：有的打印机的【打印速度】设置并不在【高级】设置里面，而是在打印机机身上的实体按钮。

例如，某款打印机的“高密”（如下图）相当于咱们高级打印中的“标准速度”，如果选择“高速”时打印字体很模糊，建议切换成“高密”。

![](https://vip.kingdee.com/download/0109f7fb5f8ba8c349caa9f14556b2358a9b.png)

>
**温馨提示**：因市面打印机类型很多，星辰无法列举所有打印机的情况。如果您在高级设置中找不到打印速度的调整选项，请检查打印机机身上的按钮；如果仍不清楚，建议直接**咨询打印机厂家的售后，**以获得帮助和指引。

**【适用版本】**

星辰3.1以上所有版本
