# 智能打印（按条件指定打印模板）

# **1 简介 **

## **1.1 应用场景**

在给客户打印销售订单合同时，不同的客户有不同的模板格式要求，希望能根据不同的客户自动匹配不同的打印模板。

## **1.2 功能特性**

- 销售订单，销售出库单等销售类单据的打印设置弹窗中，增加按条件设置打印模板的页签，支持设置不同条件下对应的打印模板
- 采购订单，采购入库单等采购类单据的打印设置弹窗中，增加按条件设置打印模板的页签，支持设置不同条件下对应的打印模板
- 其他出入库单，组装拆卸单等库存类单据的打印设置弹窗中，增加按条件设置打印模板的页签，支持设置不同条件下对应的打印模板

## **1.3 适用版本**

星辰3.1以上所有版本

# **2 操作指南**

以销售订单为例：

1）在打印设置弹窗中，设置不同条件下的打印模板；

![上传图片](https://vip.kingdee.com/download/0100e7625ad0951c4cc09974e3f0b3833ee6.jpg)

2）点击过滤条件按钮，在打印条件弹窗中，设置打印条件；

![上传图片](https://vip.kingdee.com/download/0100138e4ad0298b45ed8ebf453deb956438.jpg)

3）将条件与对应的模板设置完成，注意必须设置“否则”类型的打印模板；

![上传图片](https://vip.kingdee.com/download/0100823bd8b67e454380832453644729d6a1.jpg)

4）设置完成后，即可打印；

![上传图片](https://vip.kingdee.com/download/01008abe59dadd754e35a05eb4d1163bf432.jpg)

# 3 注意事项

- 按条件设置打印模板功能中，条件仅支持选择**单据头**的字段，不支持单据体字段、不支持单据表头新增字段，即表头自定义字段；
- 如单据001同时命中了多个打印模板，则无法继续打印：可以调整条件后重新打印，也可以关闭条件，以基础设置中的打印模板为准来打印；
- 按条件设置打印模板功能是按用户维度生效的，也就是同一个账套里，用户A设置的条件不影响用户B，用户之间相互独立不会互相影响。
