# 如何邀请用户?如何启用或禁用员工?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：杨桂平 | 阅读：13987 | 有用：13
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/129339770114427402)

【问题描述】

如何邀请用户?如何启用或禁用员工?

【解释说明】

**有新的人员需要设置账号使用系统时,可以通过用户管理新增发送邀请,仅支持邀请为企业内用户,不支持设置为外部协作者(邀请为外部协作者为历史功能,现已不支持,仅支持邀请为公司员工,使用时两者功能没有区别);**

**新增用户的登录名不支持自定义修改设置;**

**邀请用户需要登录服务管理员或者账套管理员账号进行操作;**

**邀请员工仅支持在PC端进行操作,手机端app暂不支持邀请员工。**

[用户邀请与授权](https://vip.kingdee.com/school/528543198667408640?productLineId=35&lang=zh-CN&cosmic_type=video)

【操作流程】

**1、邀请账号**

首先,登录系统后依次点击【系统设置】-【用户管理】,如图:

![](https://vip.kingdee.com/download/01095ca7627855064f268c77f401f6fbe57d.png)然后,在【用户管理】页签,单击【新增用户】按钮,打开新增用户窗口,录入手机号,回车,如果已在官网注册,则会显示对应的信息,输入姓名等确定发送邀请,**(如果未在官网注册账号,则系统会提示录入姓名、初始密码(密码应设置为大写字母+小写字母+数字+字符),单击[邀请],系统同时增加用户并在官网注册对应用户信息。)**

**注意:如果只是邀请授权移动端,邀请信息确认界面“星辰产品授权”请选【星辰移动端】**

![](https://vip.kingdee.com/download/0109fee1ab74911845c28c74867871131e24.png)
用户状态会显示邀请中
![](https://vip.kingdee.com/download/0109dfd5e732e95240a59b350903973ebce3.png)
被邀请的账号登录后会看到邀请提示,某公司要求接入,点击【接受】按钮即可确认接受要求加入公司

![](https://vip.kingdee.com/download/0109cf2f19d3dfda466189b814c9926470a6.png)

![](https://vip.kingdee.com/download/010977ca1c72fc4248ef9dcd080fbb76add9.png)
**注意:邀请函有效期是7天,超过期限无法收到邀请函,需删除重新邀请;若是在工作台上个邀请的,可以让企业管理员在部门与员工——邀请失效人员中,重新发送邀请**

![](https://vip.kingdee.com/download/0109bf071160c3584659ae8058c161698118.png)

在用户管理中可以看到用户状态变成【启用】,根据需要进行授权即可

![](https://vip.kingdee.com/download/0109bbbdea3bb5ef47a2a3b7992201373920.png)
子账号重新登录可以看到已加入到新公司并且能看到被授权的账套

**tips:若需要了解如何对员工进行账套授权,请点击:[如何给子账号授权和取消某个账套的权限?](https://vip.kingdee.com/knowledge/607377457150257408?productLineId=35&isKnowledge=2&lang=zh-CN)**![](https://vip.kingdee.com/download/0109c9fbf917b88c4057b1a73dc9056ddaff.png)

**2、直接同步工作台用户**

若是管理员已在工作台成功添加/邀请同事,管理员可直接进入星辰服务,在【设置】-【用户管理】,点击右上角【同步工作台用户】即可。

具体操作步骤可参考:[如何把工作台用户同步到账套的用户管理中](javascript:void(0);)

**3、禁用/删除离职的员工账号**

可在【系统设置】-【用户管理】处,选中需要禁用的员工账号,点击<禁用>

![](https://vip.kingdee.com/download/0109cd0538c30d00405ba4571ee596c0063f.png)
也可以勾选点<删除>,删除后账套里不再显示这个账号,如需重新授权或者使用该删除的账号,需重新进行邀请
![](https://vip.kingdee.com/download/0109499321254da64d7fb89b94cde3cce461.png)

**4、重新启用已被禁用的员工**

可把用户状态过虑条件改为禁用显示出对应用户后,勾选该用户点击启用。

![](https://vip.kingdee.com/download/0109f04a58dd5e6a4e809cd1a9dbef818260.png)
