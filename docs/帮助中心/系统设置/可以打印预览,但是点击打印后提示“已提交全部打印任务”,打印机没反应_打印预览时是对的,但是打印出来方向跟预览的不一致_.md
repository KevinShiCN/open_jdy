# 可以打印预览,但是点击打印后提示“已提交全部打印任务”,打印机没反应?打印预览时是对的,但是打印出来方向跟预览的不一致?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：杨桂平 | 阅读：1545 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/534899750651463680)

【问题描述】

PDF打印/插件打印,可以正常预览,点击打印,显示“已提交全部打印任务”后,打印机不出纸会是什么原因?

![](https://vip.kingdee.com/download/0109d010d69c22244132badaad16c4bd98df.png)

打印预览时是对的,但是打印出来时缩放的尺寸或者方向跟预览不对是为什么?

【解释说明】

打印提示“已提交全部打印任务”,表明打印任务已经生成成功了,而这个时候打印机依然没出纸,通常都是由于默认打印机的打印首选项中的纸张尺寸与打印模板的尺寸不一致导致的。

**而打印预览时是对的,但是打印出来时缩放的尺寸或者方向跟预览不对也通常是因为同样的理由。**

【解决方案]

重新调整打印模板的尺寸或者是打印机首选项的纸张尺寸,保证二者尺寸一致即可。整体解决思路概述如下:

  ![](https://vip.kingdee.com/download/01091dc4e1fd846f43e0be89a71e3eb842d3.png)

下文将以Windows10系统为例讲解如何设置。其他系统设置见下方说明,本文不做论述。

• Windows7系统:请参考相关知识:

套打打印基础-打印机及纸张规格大小设置

[https://vip.kingdee.com/link/s/MUWJE](https://vip.kingdee.com/link/s/MUWJE)

• Windows XP系统:星辰不建议使用XP系统。

Windows10系统操作步骤:

1、 找到电脑的[打印机和扫描仪]并打开,找到电脑的[默认]打印机

![](https://vip.kingdee.com/download/0109c006b33e518046d5b14f9ed702f10ed4.png)

  注:如果没有看到[默认]二字,检查一下[让Windows管理默认打印机],去掉勾选即可。

![](https://vip.kingdee.com/download/0109aac817a57d4c4f81852dafdff0e81947.png)

如果需要切换默认打印机,则找到设备后,点击[管理],设置默认值即可:

![](https://vip.kingdee.com/download/01099b627126572746eb909ff860b1f27006.png)![](https://vip.kingdee.com/download/01094afa96a4077c4c9a818778508ef1885b.png)

2、 验证电脑默认打印机[打印测试页],先确保打印机可以正常运作!点击打印测试页后,打印机可以正常出纸即为正常。

如打印测试页无法出纸,请先检查打印机驱动的连接,或者删除打印机后重新添加驱动后再测试,如果重新添加驱动后依然不出纸,此问题属于设备问题,请联系打印机厂家处理。

![](https://vip.kingdee.com/download/01091c6c063a45454a18b8a77c179ff72f61.png)

3、 检查默认打印机的[打印首选项]内“纸张摘要”的纸张尺寸是否与星辰打印模板中的尺寸一致:

**[标准纸张]**

假设纸张是标准A4纸,则打印机首选项里显示应该是A4纸,如下图:

![](https://vip.kingdee.com/download/010971c18da3ab054eb2b0cc55b699bbbb7a.png)

星辰打印模板的纸张[样式]尺寸也应该是A4纸:

![](https://vip.kingdee.com/download/010927516d920f9c4cf185df24af116bc6c1.png)

[自定义纸张]

如果是自定义的纸张尺寸,假设是210*140mm,纸张名字“自定义订单模板”,那么首先打印模板中的“样式”尺寸要选择“自定义”,宽度和高度分别设置240mm,140mm,具体设置如下图:

![](https://vip.kingdee.com/download/0109f8aa1c5836d14265b3877f03a3e77366.png)

同时在计算机的[打印机和扫描仪]-[打印服务器属性]里,勾选“创建新的自定义纸张规格”,输入好尺寸、纸张规格名称,单击“保存规格”、“应用”即可新增成功。

![](https://vip.kingdee.com/download/01090c1b88fccb244409875c87f476a0d39b.png)

最后就是到[打印首选项]里找到刚刚新增的纸张尺寸,单击确定-应用-确定即可(下图序号为操作顺序)

![](https://vip.kingdee.com/download/010995d7c34bf1974114b552da77100eaadd.png)

都设置好后,就可以正常进行打印了。

如果是PDF打印,打印预览效果如下图:

![](https://vip.kingdee.com/download/010966f11c9b10cc418594fdbbdd5b5c256d.png)

这里要注意,当模板打印机是默认打印机、打印纸尺寸也跟打印模板尺寸一致时,才可以正常打印出纸。

如果是插件打印,打印预览效果如下图:

![](https://vip.kingdee.com/download/0109bc9f3086d01d4bf3902f8eb2e016ae9f.png)

  插件打印预览界面只选确认好打印机是默认打印机即可。
