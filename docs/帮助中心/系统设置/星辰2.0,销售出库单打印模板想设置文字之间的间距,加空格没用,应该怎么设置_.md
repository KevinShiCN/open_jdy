# 星辰2.0,销售出库单打印模板想设置文字之间的间距,加空格没用,应该怎么设置?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：杨桂平 | 阅读：156 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/537436499231490560)

【问题描述】

销售出库单打印模板想设置文字之间的间距,加空格没用,应该怎么设置?

【解决方案】

在打印模板里点击选中字段,点击右上角的业务,启用填格,先把文本框拉到对应宽度,然后填入对应的填格数,保存完成设置即可。
