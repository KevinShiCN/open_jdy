## 3 原因解析

- ### 字体转化差异 星辰3.1从发版之初,为了解决针式打印机场景下,宋体打印不清晰的问题,后台强制做了字体样式转化,宋体打印效果是按微软雅黑生效的,之前是不支持打印宋体的。由于8月中旬后，已经支持了打印字体管理,用户可以自己上传字体,也为了解决非针式打印机的客户需要如实打印宋体的需求,在8月的系统更新后,放开了限制,也就是不再对宋体做强行转化。因此，8月系统更新后,凡是通过新增,复制,另存为,引入模板的方式来创建的模板,都会按照字体本身的样式打印。换句话说:如果原来的打印模板字体就是宋体,依然保留了强制转化,也就是默认显示微软雅黑的效果; 而复制后的模板,则是默认按宋体正常样式显示。具体原因及解决方案可查看：[使用针对打印机,原来模板打印都是正常的,复制原来的打印模板后,打印字体和边框线/表格线都不清晰了?如何解决?](https://vip.kingdee.com/link/s/lsvHu)

[](https://vip.kingdee.com/link/s/lsvHu)

- ### 插件渲染模式的差异 星辰存在三种版本的插件，随着产品优化和技术迭代，插件的绘图方式（也叫渲染模式）也在逐步变化，因此也导致的打印效果的差异，详见下表的说明：

| 插件版本 | 插件版本说明 | 渲染模式及效果说明 | 插件预览页面 | 插件实际打印效果 |
| --- | --- | --- | --- | --- |
| 3.0 |  | 3.0插件使用的是转成图片后，在打印纸张上绘制两次图像。所以字体会更黑更粗。 |  |  |
| 4.0 |  | 插件预览页面的【渲染模式】支持自定义切换：其中：高清渲染 -- 是从PDF文件获取的线条文本进行打印，对针式打印机的效果最好。最接近打印预览的效果。增强图像渲染 --  是PDF文件转换成图片重叠后进行打印，目的是加粗打印的效果，效果等同于原来3.0插件打印效果。 | 高清渲染模式：增强图像高染模式： | 高清渲染模式：增强图像高染模式：（与上一行3.0打印效果一致）对比上面2种效果，当字体笔画多的时候，可以看到这种差异会更明显： |
| 5.0 |  | json渲染，也就是就是调用本地接口绘制文字的。插件预览页面的【渲染模式】默认“高清渲染”，不支持切换，最接近打印预览效果。 |  |  |

## 4 解决方案

综上所述，如您的使用的打印模板，字体是宋体，且之前用的是3.0的插件，那么当您的模板重新复制后，并且安装了最新的插件的话，就会出现打印效果的差异。其实并不是bug，而是历史插件与现在新插件的渲染模式有差异，而用户习惯了之前的效果，导致对比认为现在的打印效果有误。

如果您暂时不希望使用最新插件，希望使用3.0或者4.0插件，那么本文也提供历史的插件安装包供用户下载（压缩包解压后，双击即可安装成功）：

3.0插件 JDYPDFPrint.exe.zip

4.0插件 JDYPDFPrint.exe.zip

鉴于星辰插件版本的持续迭代，后续老版本的插件不会再维护，为了提供最新的功能和最优的用户体验，建议大家使用最新的插件（直接在打印设置里下载插件即最新的）进行打印，同时为了尽可能保证打印效果的一致性，建议您在打印前，对您的打印模板做以下调整：

1）如果您使用的是针式打印机，打印模板建议字体选择“微软雅黑”

![](https://vip.kingdee.com/download/010984a4357132f24bfd88e68ab48e201c9e.png)

2）如果对字体粗细不满意，建议字体加粗

![](https://vip.kingdee.com/download/0109659be45d328a4da0ab1a22646bc226b9.png)

3）针对字体和表格颜色，请务必选择色值最深的纯黑，#000000

![](https://vip.kingdee.com/download/0109031abf3576ee45b0a3db0cb20a14e7de.png)

表格线的颜色：

![](https://vip.kingdee.com/download/0109fb2d5632d3fa409493583432cf8b7e57.png)

边框线如果太浅，建议设置更粗的边框线

![](https://vip.kingdee.com/download/0109315e3a735f7f42ff817547602c3657eb.png)
