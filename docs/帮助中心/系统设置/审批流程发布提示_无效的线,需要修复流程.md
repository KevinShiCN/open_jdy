# 审批流程发布提示:无效的线,需要修复流程

【问题描述】

审批流程发布提示:无效的线,需要修复流程

![](https://vip.kingdee.com/download/0109e4f216a2ba824efea304468e4aaf11ec.png)

【解释说明】

通常是由于手动修改了流程中的线,比如拖拽线位置或者直接复制线粘贴出来的线,都会导致系统无法识别到,认为是无效的线。

![](https://vip.kingdee.com/download/0109759e6af8bea94b21bdc9a61d04ca4d06.png)

【解决方案】

可通过删除无效线重新添加一条有效线处理

①选中无效线点击键盘上的删除键删除

![](https://vip.kingdee.com/download/01099cdc65ddd77c4645875e817609ddb34a.png)
②选中上一个节点,点击显示出的箭头即可自动生成一条有效节点连接线,生成后可移动连接的节点看连接线是否会跟随节点的移动而移动,若会移动表示线条连接有效,保存发布即可。

![](https://vip.kingdee.com/download/01097cf3fec547e04b878ae1e16528fd38d6.gif)
