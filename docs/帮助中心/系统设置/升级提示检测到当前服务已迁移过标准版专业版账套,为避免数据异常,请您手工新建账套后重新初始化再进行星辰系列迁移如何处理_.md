# 升级提示检测到当前服务已迁移过标准版专业版账套,为避免数据异常,请您手工新建账套后重新初始化再进行星辰系列迁移如何处理?

> 作者：Siwennn | 阅读：101 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/647965128549348608)

【问题描述】

升级提示检测到当前服务已迁移过标准版专业版账套,为避免数据异常,请您手工新建账套后重新初始化再进行星辰系列迁移如何处理?

![](https://vip.kingdee.com/download/010980525249d83c4c3899adcb7af12d5afc.png)

【解释说明】

多个账套数据迁移时,为了避免数据混乱,需要先建账再初始化后进行迁移。

创建账套需要**服务管理员账号**,点击右上方账套下拉即可创建账套。

![](https://vip.kingdee.com/download/0109413ca91e1d13469cb01efb4d38e0e135.png)

点击**“系统设置”-“重新初始化”**,即可选择初始化的范围。

![](https://vip.kingdee.com/download/010947eebda503f34e60af4876a95d0a3781.png)
