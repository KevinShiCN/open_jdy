# 打印次数如何设置？如何统计？预览会算入打印次数中吗？

# 1 简介

## 1.1 应用场景

打印次数主要记录打印成功的次数，一般插件打印成功或者PDF打印预览成功都会更新打印次数，每次计入一次。

## 1.2 适用版本

星辰3.1以上所有版本

# 2 功能应用

## **2.1 打印模板如何设置打印次数**

下面以销售订单打印模板为例进行说明。

打开打印模板，在左侧数据源搜索“打印”，找到“打印次数”，单击即可增加到模板中，然后单击右上角的“保存”即可。如果是要添加到数据表格的单元格中，也是一样的操作逻辑。

![上传图片](https://vip.kingdee.com/download/01009aed584718e44048b20710371a3ed087.png)

## **2.2 打印次数如何统计**

打印次数按“打印次数+1”显示在打印出来的纸上，**即显示的是含本次打印在内的累计次数。**

例：第一次打印，打印次数是1，第二次打印，打印次数就是2。

>
注意：如果使用了【系统设置>打印控制】的“控制打印次数”功能，系统会强制使用插件打印，打印设置>打印方式的PDF打印将不能选择，按钮会置灰。

打印设置>打印方式的按钮置灰效果：

![上传图片](https://vip.kingdee.com/download/010075c1ad3041ca439c91e3b9b8b1a46764.png)

## **2.3 打印预览影响打印次数吗**

星辰支持2种打印方式，且打印预览对打印次数统计逻辑是不一样的：

**PDF打印：**单击“打印”或“打印预览”，都算打印了一次。

**插件打印：**单击“打印预览”不计入打印次数**，显示的只是打印效果，只有单击“打印”才会算一次。**

>
注意：使用插件打印时，打印次数如果已经达到打印控制的上限（例如只允许打印1次，并且已经打印了1次），那么此时是不允许点击打印预览的。

之所以不让打印预览，是因为打印控制的校验是在点击打印预览的时候判断的，如果打印次数已经达到上限，就无法预览（因为预览的弹窗里有“打印”的按钮，这里控制不到用户的打印操作，因此在进入预览页面前先做了判断和过滤），因此，只要打印次数小于打印控制的次数，就可以继续打印预览，否则不允许打印预览。
