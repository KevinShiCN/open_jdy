# 金蝶云星辰，打印销售出库单，日期为什么显示为：2023-00-24，单据日期打印显示异常怎么处理？

金蝶云星辰，打印销售出库单，日期为什么显示为：2023-00-24，单据日期打印显示异常怎么处理？

![image.png](https://vip.kingdee.com/download/0109566ea2d3fb5447d6a9c1f1aeea0a86f7.png)

解决方案：星辰3.1版本的，打印模板文本的“格式”设置“日期”后，还需要设置对应的日期“类型”，设置后才能正常显示。

![image.png](https://vip.kingdee.com/download/010986152f99eef542dfb80af1fb6ab34b78.png)

假设设置的是YYYY-MM-DD的格式：

![image.png](https://vip.kingdee.com/download/010963f73dee4e6945e48e14d0ce4a91f155.png)

打印预览就正常了：

![image.png](https://vip.kingdee.com/download/0109f43a6fca57be4794b0ca026b5df7738a.png)
