# 详细解析

当分页2已经铺满了，要使用“分页拼接打印”时，实现下一页能拼接到上一页底部空白还需要满足：**上一页的剩余可打印区域（也就是纸张底部的空白区域）能容纳下一页的文本控件。**

# 解决方案

如果分页2的内容很多，又想使用分页拼接打印，建议把分页2的单个文本控件拆成多个单独的文本控件，这样就能实现拼接打印了，不要把分页2 的内容都放到一个大的文本控件里。
