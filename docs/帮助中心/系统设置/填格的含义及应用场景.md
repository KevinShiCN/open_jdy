# **适用版本**

星辰2.0所有版本

# 名词解释

填格：适用于数据表格的单元格、文本等等控件，目的是把文本按指定的数量拆分后填入到格子内。

# 应用场景

金额大写，或者财务凭证的样式，或者业务表格设计对齐的美观，可结合客户实际使用场景来设计。

# **应用流程及效果**

在打印模板中，选中数据表格单元格或文本后，在“业务”中设置，可以自定义格子数（填数字）以及是否显示分隔线。
