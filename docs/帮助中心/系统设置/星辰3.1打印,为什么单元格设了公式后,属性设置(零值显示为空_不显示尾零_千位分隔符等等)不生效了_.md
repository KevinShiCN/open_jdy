# 星辰3.1打印,为什么单元格设了公式后,属性设置(零值显示为空/不显示尾零/千位分隔符等等)不生效了?

**【问题描述】**

星辰3.1打印,为什么单元格设了公式后,属性设置(零值时显示为空/不显示尾零/千位分隔符等等)不生效了?

![](https://vip.kingdee.com/download/0109128958d8833e40e3a495098a1a138a83.png)

【原因说明】

使用公式的功能时,存在2种不同的设置方式:

| 设置方式 | 逻辑说明 | 单据详情 | 打印模板设置 | 打印预览 |
| --- | --- | --- | --- | --- |
| 方式一 | 公式输出的结果是数值时,控件属性可以生效 |  |  |  |
| 方式二 | 公式输出的结果是文本时(也就是拼接了文本字段 ),控件属性不能生效 |  |  |  |

【解决方案】

如果要使用控件属性中针对数值数值型生效属性,如:显示币别符号/显示币别代码/零值时显示为空/不显示尾零/千位分隔符等等,**不能使用包含文本字段的公式。除非分开两个字段设(如下图)**

![](https://vip.kingdee.com/download/0109d9a7012b5729413582bee76d7aeb741a.png)

星辰3.1以上所有版本
