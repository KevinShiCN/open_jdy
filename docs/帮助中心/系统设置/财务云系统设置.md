# 财务云系统设置

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：快速入门, 整体介绍
> 作者：Cat_Chow | 阅读：11983 | 有用：15
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/627277825493484032)

账套创建成功后，服务管理员首先进行系统设置，包括系统参数配置、用户权限分配、打印设置等。

![](https://vip.kingdee.com/download/010923a7a7d15920473a9bfcd4d52f8c57ac.png)

| 步骤 | 具体说明 | 学习链接 |
| --- | --- | --- |
| 1.基础设置 |  | —— |
| 1.1系统参数 | 公共参数、财务参数、资产管理、出纳管理、进销存云等模块会有各自启用参数设置。 | 点击学习 |
| 1.2用户管理 | (1)账套用户管理，如新增用户、同步工作台用户、用户禁用/删除、用户分组管理、管理员设置等。 | 点击学习 |
| (2)账套权限管理：单独授权、用户组授权、复制权限；其中权限分为：功能授权、字段授权、数据授权。 | 点击学习 |  |
| 1.3操作日志 | 用于记录用户使用系统的操作行为日志记录。通过操作日志，可以查询到系统每个操作员使用软件的记录 | 点击学习 |
| 1.4重新初始化 | 企业在使用系统之后，可以根据需要对系统进行重新初始化。重新初始化有三种方式可以选择：(1)单据: 清空所有单据,保留基础资料、期初、设置;(2)单据+期初: 清空所有单据、期初,保留基础资料、设置;(3)全部数据: 清空所有单据、期初、基础资料、设置、并重建账套。注意：重新初始化操作不可逆，建议操作前先备份。 | 点击学习 |
| 1.5网络控制 | 系统为最大限度的保证网络并发控制和数据一致性而设的网络控制。在用户出现异常中断、死机等原因可能造成单据锁定，会导致其他用户不能执行互斥功能。另外，如果是浏览器关闭或是关机，产品是捕捉不到退出信息的无法解锁。所以如果关闭浏览器或者直接关机时正打开着单据，即使这个人关机了，别人再操作这个单据仍会提示有人编辑着。 | 点击学习 |
| 1.6帐套备份与恢复 | 系统独具有云备份功能,只要不主动删除数据,数据一般不会丢失,也无需定期备份数据。同时，系统也提供手动备份功能，支持管理员手动备份当前账套数据，也支持恢复备份数据至原账套下。 | 点击学习 |
| 2.个性化设置 |  | —— |
| 2.1编码规则 | 自定义基础资料、资产卡片、出纳单据等的编码规则 | 点击学习 |
| 2.2打印模板 | 系统预置多种格式的凭证打印模版，也支持自定义编辑打印模版 | 点击学习 |
| 2.3引入引出模板 | 引入引出模板是系统所有支持引入引出功能的引入引出模板列表。这里可以直接选择单据新增引入/引出模板，然后到具体单据列表中使用。 | 点击学习 |
| 2.4单据字段配置 | 目前财务云支持对出纳管理的日记账和序时簿单据的部门、业务员、结算方式、备注字段进行设置是否为必录项 | 点击学习 |
| 2.5多帐套设置 | 用于同一公司中多个账套间设置匹配关系。业务场景：企业有 A、B、C 三个账套，A 账套为主账套，需要在 A 账套中汇总显示 B、C 账套的数据进行分析。则需要先在多公司设置中，在 A 账套，通过账套关联设置，将 B、C 账套设置为关联。 | 点击学习 |
