# 单据/凭证内的审核人和制单人的姓名是取哪里的姓名,如何修改?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：shaoling | 阅读：4433 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/566427472359390464)

【问题描述】

单据/凭证内的审核人和制单人的姓名是取哪里的姓名,如何修改?

【业务场景】

制单人或审核人显示不对,想要修改制单人,在哪里操作修改

【解释说明】

**系统中单据或凭证的制单人、审核人,默认提取当前实时操作的账号信息,不支持手动自定义指定,后续修改数据也不会更新制单人,修改后重新审核会更新审核人。**

若需将某张单据或凭证的制单人 / 审核人修改为系统现有账号用户,需先删除该单据 / 凭证,使用目标账号重新录入;或先反审核该单据 / 凭证,再使用目标账号重新完成审核操作。

若是用户变了,建议更换使用的账号,更换使用账号不影响历史已有单据制单人审核人的姓名:[如何更换账套使用人员?](https://vip.kingdee.com/link/s/ZcYLt)

若是账号录入的姓名信息错了导致单据或凭证上显示的姓名有误,可以通过用户管理修改姓名,修改后这个账号做的单或审核的单上的制单人或审核人姓名都跟着变成修改后的姓名;操作步骤见下方解决方案。

【解决方案】

单据/凭证内的审核人和制单人的姓名为各个账号设置的姓名

![](https://vip.kingdee.com/download/0109bacdc80ec23c4e68aee5e11eb9dcb298.png)

如单据里现在的【制单人】显示为【测试99998888】

![](https://vip.kingdee.com/download/0109b07973284c5b4aa5a64690de4e67da8f.png)

如果要修改制单人就需要**管理员账号**登录进入账套点击【系统设置】-【用户管理】

![](https://vip.kingdee.com/download/010937f52bf5eb3d4881be0fb4931165bec9.png)

点击需要修改姓名的账号的登录名到修改用户界面,在姓名栏修改,修改后点击【确定】按钮来保存。

![](https://vip.kingdee.com/download/0109782903b5531f415f8d6748c368cef5f7.png)

![](https://vip.kingdee.com/download/01091f6fa4076f92402e99819ba70a2ee836.png)

【用户管理】里的【姓名】修改后,这个账号用户所做的单据【制单人】都会更新成修改后的姓名显示,包括之前该用户所有的所有单据也是会更新

![](https://vip.kingdee.com/download/01098076697235fa45d7ab8a6bee9c3aa949.png)
