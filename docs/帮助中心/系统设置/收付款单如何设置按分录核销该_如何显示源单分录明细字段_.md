# 收付款单如何设置按分录核销该?如何显示源单分录明细字段?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：姜宁8 | 阅读：1670 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352033032087808)

【问题描述】

收付款单如何设置按分录核销该?如何显示源单分录明细字段?

【解释说明】

收付款按分录核销,是进销存软件中针对 “应收 / 应付账款” 的精细化对账匹配功能,即当单据包含多行商品分录时,支持单独针对某一行商品分录进行收付款操作。

【操作流程】

1、在【系统设置】-【系统参数】-【应收应付参数】选择“按分录核销”

![](https://vip.kingdee.com/download/0109350051496ed547af95266be000b5ff6c.png)

2、选择 “按分录核销” 模式(以收款单为例):首先选择【客户】,点击单据右上角的【选源单】按钮,系统将弹出源单选择窗口,窗口内列设置商品明细字段可开启显示,然后按需勾选本次需核销的具体单据分录即可。如下图所示:

![](https://vip.kingdee.com/download/010961afb57d5e154f4bb3ed8f1536b470b7.png)

![](https://vip.kingdee.com/download/01099ca3933093fd43aea93c80ac1277bd30.png)
![](https://vip.kingdee.com/download/01093f24909500b945ee92c020ebf8569502.png)
参数未选择按分录核销时,选源单列设置里面的分录明细字段为灰色,无法开启显示

![](https://vip.kingdee.com/download/010928ce7f051b554eb7b16f645270647f2e.png)

3、然后手动录入对应分录的本次收款金额,完成后点击【保存 / 审核】即可完成核销,**或者**先录入总收款金额,再点击【自动核销】按钮,系统将按源单分录的显示顺序,自动把总金额分摊至各选中分录,并填列对应的 “本次核销金额”

![](https://vip.kingdee.com/download/01094539ee17c13e41c589e5028e534c7c51.png)
