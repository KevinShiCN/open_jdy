# 星辰2.0,客户没做数据授权和字段授权,但是在录入单据的时候只能选择一个仓库是为什么?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：杨桂平 | 阅读：201 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/537799043410465536)

【问题描述】

客户没做数据授权和字段授权,但是在录入单据的时候只能选择一个仓库是为什么?

【解释说明】

检查子账号是否设置了业务员,且对应仓库是否设置了对应的业务员作为为仓管,若同时满足以上两个条件的,子账号只能看到对应业务员为仓管的仓库;若需要子账号看到所有仓库,取消设置仓库的仓管员即可。
