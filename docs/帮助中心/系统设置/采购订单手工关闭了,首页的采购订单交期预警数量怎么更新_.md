# 采购订单手工关闭了,首页的采购订单交期预警数量怎么更新?

> 作者：_Leo | 阅读：370 | 有用：1

#### 【问题描述】

在采购订单中,订单已手动关闭了,想要首页的交期预警同步更新。

【操作流程】

点击首页右上角,的“刷新”按钮,即可更新首页显示的信息。
