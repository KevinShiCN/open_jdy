# 星辰2.0,打印模板的字体使用了“黑体”,金额设置了显示币别格式,但打印预览是怎么都显示不出来,为什么?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：杨桂平 | 阅读：265 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/534899718808114688)

【问题描述】

星辰2.0,打印模板中字体使用了“黑体”,金额设置了显示币别格式,但打印预览是怎么都显示不出来。

![](https://vip.kingdee.com/download/0109b1d6ea3b52ce4726892ce4ce92603c4f.png)

![](https://vip.kingdee.com/download/0109e94791820ec94b479f0c0632e7f8963d.png)

【解决方案】

目前星辰打印模板的“黑体”字体不支持显示币别符号￥、£ ,建议如需使用该两种币别符号时,字体建议使用微软雅黑和宋体。其他币别符号$、€、US$则不受此影响。
