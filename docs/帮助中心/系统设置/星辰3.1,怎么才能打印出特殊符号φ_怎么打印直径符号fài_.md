# 星辰3.1,怎么才能打印出特殊符号φ?怎么打印直径符号fài?

**【问题描述】**

星辰3.1,怎么才能打印出特殊符号φ?怎么打印直径符号fài?

**【解决办法】**

**1、单据上怎么录入?**

针对特殊符号Φ的添加方式,先按住ALT键,再按数字42677,就能在单据中录入这个符号

**2、打印模板怎么才能正常打印显示?**

对含有这个字体的字段设置**黑体**即可打印显示
