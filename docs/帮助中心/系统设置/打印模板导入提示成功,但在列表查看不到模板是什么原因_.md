# 打印模板导入提示成功,但在列表查看不到模板是什么原因?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：ZXYzxc | 阅读：24 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/780962312693430528)

【问题描述】

打印模板导入提示成功,但在列表查看不到模板是什么原因?

【解释说明】

导入打印模板后,页面显示 “导入成功”,但系统内未查询到该模板。

![](https://vip.kingdee.com/download/0109cacf7a320f8c487bb4bdb2bb0f135438.png)

需要检查【引入】-【设置】中的引入方式

![](https://vip.kingdee.com/download/0109e4fb863e0253412d90d2c85e9d10d4fe.png)引入方式为 【更新引入】时,表示若系统中已存在 ID 相同的模板,将直接更新该模板,不会新增一个模板。

![](https://vip.kingdee.com/download/0109ce3713e6e3684dfd8c852f5f089ba0ac.png)
若需在模板 ID 相同的情况下仍能新增模板,可将引入方式设置为 【新增引入】,系统会自动生成新编码并完成模板新增。

![](https://vip.kingdee.com/download/0109d60ab41c2bd0462d89a649ee5f98cb4a.png)
如果出现上述情况,可以切换为新增引入重新引入一次。
