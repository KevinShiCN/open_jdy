# 打印预览正常,但是条码打印出来出现锯齿/波浪纹是什么原因?

【问题描述】

打印预览正常,但是条码打印出来出现锯齿/波浪纹是什么原因?

【解释说明】

如果打印预览是正常的,按以下顺序检查和调整:

①检查打印出来的竖线、边框等是否有波浪纹,如果有,需重装插件或者重启打印机;

②如果打印方式为插件打印,一般出现波浪纹是因为图片有失真,用插件打印条码会有一定的失真,建议改用PDF打印。
