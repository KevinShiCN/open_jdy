# 进销存列表打印

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：功能介绍, 快速入门, 精华汇总

## **1 简介**

### **1.1 功能介绍**

可以实现列表的打印即所得，也就是列表的数据如实打印。

### **1.2 适用版本**

星辰3.1以上所有版本

### **1.3 系统路径**

进销存的单据列表<打印<列表打印

![](https://vip.kingdee.com/download/0109faa2c8e5033d49678ac9214f9d3f8212.png)

## **2 功能特性**

- 列表打印不支持插件打印，只能使用PDF打印
- 页面设置参数

|  | 具体参数 |
| --- | --- |
| 页面 | 纸张类型：支持A4、B5纸张方向：纵向/横向页面缩放：可自定义缩放比例；也可调整成一页宽 |
| 页边距 | 支持设置上下左右边距，单位都是毫米 |
| 工作表 | 打印标题：支持自定义增加标题。只能居中字体大小列表行高：支持默认行高/自定义固定行高，可以勾选换行 |

- 目前支持的单据：移仓单、调拨出库单、调拨入库单、盘点单、盘盈单、盘亏单、采购申请单、采购订单、采购入库单、采购退货单、采购发票、销售订单、销售报价单、发货通知单、销售退货单、销售出库单、销售发票、销售退货申请、销售换货申请、委托代销单、委托代销结算、委托代销退货、委托结算退货、其他入库单、其他出库单、组装拆卸模板、拆卸单、组装单

## **3 操作指南**

1）过滤好列表要打印的数据后，勾选要打印的行，点击“列表打印”。

注意，如果不勾选数据行，也不选择“选择全部”，则默认打印当前页的所有行，假设需要打印的内容有多页，则默认只打印第一页，需要切换到下一页点击“列表打印”才会打印第二页。所以如果列表需要打印的数据很多，都需要打印，建议点击列表左下角的“选择全部”。

![](https://vip.kingdee.com/download/0109529a923d47ad4ecaaa4267eea3e85862.png)

2）设置好页面、页边距、工作表等参数，点击确定，即可进入PDF打印预览页面

3）确认无误后，点击打印即可。

4）打印效果验证。

假设把销售订单列表打印成A4纸的效果，并且需要把页面调整为一页宽。
