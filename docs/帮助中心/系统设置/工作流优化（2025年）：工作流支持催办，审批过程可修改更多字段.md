# 工作流优化（2025年）：工作流支持催办，审批过程可修改更多字段

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：快速入门, 功能介绍, 新特性
> 作者：进销存熊诗琴 | 阅读：1070 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/682910933407880448)

## 1 简介

### 1.1 功能介绍

- 启用工作流后，希望通过系统发送消息提醒审批人及时完成审批任务。
- 在审批过程中，若审批人发现单据中的某些信息（如自定义字段、单价、金额等）存在错误，但又不想驳回据单让提交人修改后重新提交审批流程，希望能够直接在审批环节中进行修改。

### 1.2 适用版本

星辰3.1以上专业版/旗舰版

## 2 功能特性

### 2.1 工作流支持催办

（1）提交人可以在任务中心的在办申请界面，选择需要催办的任务，点击催办：

![](https://vip.kingdee.com/download/0109023e491bc35f4f38b703ce59a2c38476.png)

（2）催办完成之后，审批人在消息中心就会收到消息了：

![](https://vip.kingdee.com/download/010939d0ed36479e4e299365736e5a23f219.png)

注意：如审批人=提交人，则不允许催办；一次仅允许催办一张单据。

### 2.2 审批过程中，支持编辑更多的字段

| 单据名称 | 支持的字段 |
| --- | --- |
| 采购申请单 | 自定义字段，批准数量，预计到货日期 |
| 采购订单采购入库单采购发票 | ● 自定义字段，交货日期（仅采购订单）● 单价，含税单价，不含税金额，价税合计● 预付订金（仅采购订单），支付货款（剔除采购订单和旗舰版的采购入库单）● 付款方式，付款账户（剔除旗舰版的采购入库单） |
| 采购退货单 | 自定义字段，收取退款，退款方式，退款账户 |
| 销售报价单 | 自定义字段，数量，实际报价，含税单价，价税合计 |
| 销售订单发货通知单销售出库单销售发票 | ● 自定义字段，预计交货日期（仅销售订单）● 单价，含税单价，不含税金额，价税合计● 预收订金（仅销售订单），收取货款（剔除销售订单和旗舰版的销售出库单）● 收款方式，收款账户（剔除旗舰版的销售出库单） |
| 销售退货单 | 自定义字段，支付退款，退款方式，退款账户 |
| 委托代销单委托代销结算单 | 自定义字段，单价，含税单价，不含税金额，价税合计 |
| 销售退货申请销售换货申请委托代销退货委托结算退货其他入库其他出库盘点单分量盘点单盘盈单盘亏单移仓单调拨出库单调拨入库单组装单拆卸单 | 自定义字段 |

注意：带出已有基础资料属性和公式类型的自定义字段不支持在审批过程中编辑。如果需要更新公式型自定义字段的值，请查阅该知识：[自定义字段优化（2024年）：通过公式计算的数值型自定义字段支持打印/引出/过滤/合计，支持嵌套公式](https://vip.kingdee.com/link/s/ZzKwu)
