# 打印设计器-控件整体介绍

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：快速入门, 精华汇总, 常见问题
> 作者：进销存赵嘉丽 | 阅读：1389 | 有用：20
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/454247118023250688)

## 1 简介

在设计打印模板时，需要用到文本、表格、图片、二维码、页眉页脚等等相关的内容，而承载这些内容的工具就是我们要介绍的“控件”。



## 2 适用版本

金蝶云星辰 V3.1以上



## 3 功能说明

星辰3.1提供通用控件和容器控件2种：

| 控件类型 | 控件名称 | 用途 | 相关预览(点击图片可以放大查看） | 相关知识链接 |
| --- | --- | --- | --- | --- |
| 通用控件 | 文本 | 1. 输入固定的文本内容，例如单据的名称、表头表尾的固定文字信息等等2. 绑定数据源字段后作为活动文本取值在做订单模板、合同模板作用较多。 |  | https://vip.kingdee.com/link/s/lWcv0 |
| 图片 | 绑定固定图片、商品图片等，可用于显示和打印企业logo、水印、公章、商品图片等等 |  | https://vip.kingdee.com/link/s/lWcvF |  |
| 二维码 | 显示商品详细信息、单据的信息、或者企业的相关信息等等 |  | https://vip.kingdee.com/link/s/lWcvB |  |
| 条形码 | 用于打印商品标签 |  | https://vip.kingdee.com/link/s/lWcvO |  |
| 富文本 | 富文本控件常用于业务单据中使用富文本编辑器，编辑内容需带有文本格式或图片。富文本编辑器的内容可多样化，如支持文字样式、格式、图片、表格等。因此若需要单据打印模板中支持富文本的场景，需要借用打印设计器中“富文本控件”，支持此类场景。若业务单据中不存在富文本内容时，则无需拖入富文本控件使用。 |  | https://vip.kingdee.com/link/s/lWcyY |  |
| 容器控件 | 数据表格 | 用于打印单据分录的内容，属于正常的纵向明细打印，更通用 |  | https://vip.kingdee.com/link/s/lpYBE |
| 分录表格 | 可以自由设计布局的数据表格，更灵活，适用于一些非商贸行业的场景 |  | https://vip.kingdee.com/link/s/lSC5T |  |
| 布局网格 | 类似于excel的表格，可以做一些整体的布局 |  | https://vip.kingdee.com/link/s/lWcKU |  |
| 页眉 | 设置打印模板的页眉 |  | https://vip.kingdee.com/link/s/lWcKN |  |
| 页脚 | 设置打印模板的页脚 |  |  |  |

关于上述的每一种控件及其属性的详细说明，可以点击对应的链接进行查看。
