# 如何设置子账号商品价格资料的查看/编辑权限?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：BBAABABA | 阅读：616 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/589620507385963520)

【问题描述】

如何设置子账号商品价格资料的查看/编辑权限?

![](https://vip.kingdee.com/download/0109ae10c8a736b947d9a6f528d84d9ddab2.png)

【业务场景】

在价格设置中,**不同的角色或部门可能需要不同的查看或编辑权限**。例如,采购部门可能需要查看和编辑采购价,以确保进货成本的准确性;销售部门则需要查看和编辑销售价和会员价,以根据市场情况灵活调整售价;财务部门则需要查看所有价格项,以进行成本核算和利润分析。

因此,如何合理设置用户商品信息里价格设置的查看或编辑权限,成为了一个需要仔细考虑的问题。这既需要保证各部门能够高效地完成自己的工作,又需要确保价格信息的保密性和安全性,防止敏感信息泄露或被篡改。

【解释说明】

通过字段授权功能,可以禁止用户查看或者禁止用户编辑相关字段。**网页端设置权限后,手机端同步生效。**

【解决方案】

管理员账号登录,菜单栏点击【系统设置】>【用户管理】

![](https://vip.kingdee.com/download/010942da83aab85d4a528cafab28689c74c3.png)

勾选对用的用户,点击<授权>,进入到“字段授权”

![](https://vip.kingdee.com/download/01092c16b8562944419191a06ada930b6c9c.png)

字段授权界面,选择“商品信息”,点击右上角<选择字段>,可搜索“价格分录”相关字段,选择需要设置权限的字段后有两种选择

①禁止查看(禁止查看默认禁止编辑)

②禁止编辑(可查看,但不允许编辑)

![](https://vip.kingdee.com/download/0109f35fe05212e2489cb54c00313ecb343e.png)

选择字段时可搜索价格分录相关的字段,如下图所示:

![](https://vip.kingdee.com/download/010941e439c8f9f14fdeb6b5ec1fff1aca64.png)
