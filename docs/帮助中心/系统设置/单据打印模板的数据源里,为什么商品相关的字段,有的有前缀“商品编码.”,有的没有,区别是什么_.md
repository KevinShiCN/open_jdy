# 单据打印模板的数据源里,为什么商品相关的字段,有的有前缀“商品编码.”,有的没有,区别是什么?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：进销存赵嘉丽 | 阅读：107 | 有用：0

【问题描述】

单据打印模板的数据源里,为什么商品相关的字段,有的有前缀“商品编码.”,有的没有,区别是什么?

例如:“商品编码.条形码”与“条形码”有什么区别?

![](https://vip.kingdee.com/download/0109c749ae88cdc64567b8c6c2ff3e59d030.png)

【问题解析】

**1、****商品相关的字段:**

单据的打印模板里,字段里带“商品编码.”前缀的,表示是从商品信息资料里取的,不是从单据上取的,例如商品中的:条形码、产地、注册证号、生产许可证;而字段里没有带“商品编码.”前缀的,则表示字段是从单据分录上取的。对应关系如下:

| 字段来源 | 产品页面 | 打印数据源对应的字段 |
| --- | --- | --- |
| 商品信息>基本信息 |  | 商品编码.条形码 |
| 单据体(商品分录)的“条形码” |  | 条形码 |
| 商品信息>产地 |  | 商品编码.产地 |
| 单据体(商品分录)的“产地” |  | 产地 |

其他商品字段也是同理的,例如产地、注册证号。

基于以上逻辑,当我们开单时,如果存在商品信息上字段跟开单时不一样(例如一品多码的商品),**我们在设计单据的打印模板时,数据表格的商品分录的字段就要取没有带“商品编码.”的,这样才能确保所见即所得**:
