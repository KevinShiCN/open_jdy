# 如何查询系统的当前期间及启用期间是什么时候?如何确定是否结账/反结账成功?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：杨桂平 | 阅读：883 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/642167202929580800)

【问题描述】

如何查询系统的当前期间及启用期间是什么时候?如何确定是否结账/反结账成功?

【解释说明】

**当前期间**:是指当前对应模块所在的期间,此参数仅显示,不允许手工修改,由系统自动判断显示,在对应模块操作期末结账或反结账后会自动更新;

**启用期间**:表示对应模块开始使用的时间,系统可以根据启用期间可以录入开始期间和以后期间的数据,**启用期间以前的数据作为初始数据录入系统**;

可以通过系统参数中查询各个业务模块的当前期间及启用期间,**若需要修改或删除启用期间的数据,则需要将对应模块反结账到启用期间才可以进行修改,即当前期间需要等于启用期间才可以修改。**

【温馨提示】

**若结账后,右上角的期间未变化,仍然显示当前期间,考虑是否由于结账错了模块导致的,比如财务期间结账需要在财务模块结;进销存结账,需要在库存管理模块结;详情请参考:[在系统结账/反结账如何操作?](https://vip.kingdee.com/knowledge/538807544060483072?productLineId=35&isKnowledge=2&lang=zh-CN)**

【操作指引】

依次点击【系统设置】-【系统参数】查询。

![](https://vip.kingdee.com/download/01096934c9e0d1f74b25b99e6821a9d1c342.png)

1、系统当前期间

若当前期间为需要结转到的最新期间,则表示结账/反结账成功。

1.1  点击【财务参数】查询账务处理当前期间:
![](https://vip.kingdee.com/download/010913f55a1af8ab4d8cb9cd38f8b430f6ba.png)
**需注意,财务期末结账不是只在期末智能结转中结转,结转后还需到【账务处理】>【期末结账】中操作结账**

![](https://vip.kingdee.com/download/0109ca3a07605e1e4f779f3742fe7b4d3583.png)

1.2  点击【进销存参数】查询进销存云系统的当前期间:

![](https://vip.kingdee.com/download/0109b6ac816a95c74062b4c490a9b3cf17c1.png)

1.3  点击【资产参数】查看固定资产系统的当前期间:

![](https://vip.kingdee.com/download/0109c430868f127147a880afc2e080931040.png)

1.4  点击【出纳参数】查看出纳管理系统的当前期间:

![](https://vip.kingdee.com/download/010952fda070abe44a47880ef0afca4286ad.png)2、启用期间

2.1  点击公共参数查询系统的起始期间。

![](https://vip.kingdee.com/download/0109ba686f56d7b34496873a09036237c8d8.png)
2.2  点击各个模块的系统参数可以查询各个模块的启用期间

![](https://vip.kingdee.com/download/0109dc8e1193b50341c3a292320f241164dc.png)
