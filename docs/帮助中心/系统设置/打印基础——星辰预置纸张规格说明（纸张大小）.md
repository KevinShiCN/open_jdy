# **1 简介**

## 1.1 **纸张设置**

![上传图片](https://vip.kingdee.com/download/0100c847c3f355cc42718e110328c306644c.png)

**纸张大小**：系统提供了预置纸张（详见本文“1.3 预置纸张大小”）及自定义规格。

**宽、高：**单位是厘米（cm），最大上限为1500mm。

**纸张方向**：![](https://vip.kingdee.com/download/0100343935cb7b97414a9254bb8a8d4e7a09.png)对应即纵向/横向，默认纵向 。

## **1.2 页边距**

- **通过录入数值直接调整**

只能设置≥0的整数，当设置小数时不生效，假设录入0.2mm，只能录0或者1（如下图提示）。

![](https://vip.kingdee.com/download/010004ad45fe85cd4f4fbda62129024389d6.png)

- **通过手工拖拽页边距直接调整**

操作方法：将鼠标移到画布最外层虚线边框，出现双向箭头时拖动，拖动时页边距数值会自动同步更新。

![上传图片](https://vip.kingdee.com/download/01001a41ab05d56040198f37eccef0899e29.gif)

>
说明：该功能10/31发布到灰度环境，如需体验，请联系在线技术支持。

## 1.3 预置纸张大小

为了减少用户设置纸张的工作量，降低设置出错的概率，星辰预置了一些常用的业务纸张大小规格以供用户直接选择。

| 纸张名称 | 英制尺寸（英寸） | 国际单位尺寸 | 默认方向 | 纸张样式 |
| --- | --- | --- | --- | --- |
| A4 | / | 宽210*高297mm | 纵向 |  |
| 一等分 | 复写9.5*11英寸 | 宽241*高280mm | 纵向 |  |
| 二等分 | 复写9.5*5.5英寸 | 宽241*高140mm | 纵向 |  |
| 三等分 | 复写9.5*3 2/3英寸 | 宽241*高93mm | 纵向 |  |
| A3 | / | 宽297*高420mm | 纵向 |  |
| A5 | / | 宽148*高210mm | 纵向 |  |
| B4 | / | 宽257*364高mm | 纵向 |  |
| B5 | / | 宽182*257mm | 纵向 |  |

>
注意：

1、“一等分/二等分/三等分”属于新增的纸张规格，该功能10/31发布到灰度环境，如需体验，请联系在线技术支持。

2、关于等分纸，市面上的针式打印机首选项中一般都会预置上述三种复写纸规格，名称前缀通常带有“复写”。如果没有找到，可能是驱动未正确安装，建议您直接咨询打印机售后。
