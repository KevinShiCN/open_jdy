# 自定义字段的小数位数怎么设置?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：杨桂平 | 阅读：417 | 有用：0

【问题描述】

自定义字段的小数位数怎么设置?

【解释说明】

自定义字段可以对数值类型的字段设置小数位,以下操作以单据自定义字段为例,基础资料自定义字段同理。
【操作指引】

点击【系统设置】>【单据新增字段】

![](https://vip.kingdee.com/download/01090a205e3bf18f4334bdf172b0693649b5.png)
新增自定义字段时选择“类型”为数值,即可设置小数位

![](https://vip.kingdee.com/download/010982d9f5ae12f540ada44db4c4c960c8f4.png)
已设置好的数值类型字段也可以点击字段前的编辑按钮

![](https://vip.kingdee.com/download/010901c4c9e5653449e9bea922910f7ebbc0.png)
对小数位进行设置,设置后点击<确定>(**只能改大,不能改小**,修改后影响历史数据)

![](https://vip.kingdee.com/download/010936983f43c0d043fcb79409fbe28e58f8.png)
