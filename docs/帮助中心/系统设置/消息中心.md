# 消息中心

## 1. 功能介绍

金蝶云星辰的消息中心是为不同岗位的用户提供各类操作支持和业务提示,实现业务流程的畅通和协同,提升企业运营效率。

**功能包含：**后台设置、后台消息中心列表、web端消息通知、app消息通知、钉钉消息通知、企微消息通知。业务涵盖：单据消息提醒（增删审核）、经营消息、预警通知等。

![](https://vip.kingdee.com/download/01093ef6805b8ada4db691e921f971b59e0a.png)

### 2. **应用场景**

1) 单**据消息推送人、内容自定义**

单据消息通知的时候，可以根据用户的业务要求自定义消息内容。

比如：用户要看单据编码、单据金额、制单人、部门、客户名称等信息，或者业务员只需要接收销售单据的信息，采购员只需要接收采购相关的消息，管理员只想看需要看收付款的信息等，都可以通过后台进行设置。

2) **重要业务事件通知，提升业务协同**

支持单据的待审核、单据已审核、单据删除的消息提醒。

比如：可以在销售订单审核提醒仓管员拣货；仓管对销售出库单审核后通知配送员和司机进行配送；配送完成后，提醒业务员收款。或者提醒业务员有拜访任务和联系任务需要处理。

3) **经营报告定时推送管理员**

可以设置业务员管理员或者老板，推送经营日报、周报、月报；或者对销售推送销售部门销售数据；对仓管推送仓库的出入库和库存数据；对财务推送资金数据等。

4) **业务数据监控预警**

库存（最低、安全、最高）的通知，单据交期预期通知、应收应付预警通知、客户流失预警通知等。

**5) 订单、账套消息通知客户（规划中）**

报价单、销售出库、发货等通知到客户的微信；或者定期借款的客户，能在指定的时间自动给客户发送对账单。

### 3. **本次发布内容**

发布时间：2023-8-25
版本：3.1版本，包含标准版、专业版、旗舰版（2.0版本不支持，需要联系经销商升级系统版本）

1) **后台设置**

可以设置开启、关闭消息

可以设置单据类消息的通知内容

可以设置推送的渠道（如app、电脑端，钉钉专版可以推送钉钉，企微专版可以推送到企微中）

2) **单据消息**

单据消息：保存、审核、删除提醒。

推送内容：支持自定义

销售类单据：全量支持，旗舰版的销售发发票开发中（2024年1月30日）

采购单据：全量支持，旗舰版的销采购发票开发中（2024年1月30日）

生产单据：全量支持

资金单据：全量支持

订货、零售单据、客户经营：后期迭代

3) 预警消息

推送类型：库存预警、客户流失预警、应收预警、应付预警

自定义设置：时间、推送对象

4) **经营报告消息**

面向角色：管理员

消息类型：经营消息日报、周报、月报

自定义设置：时间、推送对象



### 4. **功能演示**

3.1. **电脑端：消息设置**

1) **消息设置：**系统设置中，打开消息设置，子账号需要管理员授权功能权限

![](https://vip.kingdee.com/download/0109b7a95f8ec00c486c9912e368238a33d1.png)

2) **消息设置列表**：默认开启【待审核消息】，推送‘有审核权限’的用户（跟发布之前推送的消息逻辑一致）

可以选择多种单据，批量开启。

![](https://vip.kingdee.com/download/01093f75df3549794ed39a7d3da736b33bd9.png)

3) **消息设置：**点击列表，可以自定义消息的推送

**推送对象：**保存时消息可以设置为有审核消息的人，也可以设置为部门负责人（部门负责人在职员表中设置，并且需要再用户管理把职员关联到用户），或者自定义用户。

**推送方式：**电脑端是必须要要开启的，推送到头像，移动端的消息可以设置关闭。

如果是钉钉专版可以关闭推送到钉钉，企微的可以关闭推送到企微。

![](https://vip.kingdee.com/download/01099714368895e2468089f3dd2ba9e020d9.png)

4) **消息内容：**点击内容的‘编辑’，可以自定义消息的内容。

**标题：**点击后可以编辑标题。

**内容：**鼠标点击右边内容可以直接编辑，然后双击左边的字段，可以设置为消息的推送内容中。内容中最多可以设置200字。大括号表示获取单据的字段值。

**预览：**设置后，鼠标点击空白处，可以预览推送的内容。

**恢复默认：**点击后，可以恢复到初始值。

![](https://vip.kingdee.com/download/0109633732a9709a47e4a4bd849aef062e1d.png)

3.2. **电脑端：消息中心**

消息中心列表，最多查询一个月的消息。默认会显示未读的消息；点击消息单据可以直接跳转到单据详情中查看详细内容。点击单据后单据状态变成已读状态。

![](https://vip.kingdee.com/download/0109ea7bad803f244ddaa21310bfa4a6201b.png)

3.3. **电脑端：消息通知**

电脑端登录后，如果有新的消息，则头像的数字更新，点击头像后，展示未读的消息（注意：新的消息需要点击【刷新】后才能看到）

点击消息后，消息变成已读状态，如果是单据消息则进入到单据详情；再次刷新后，已读的消息则不再显示（可以到消息中心中查看已读才能看到历史的消息）

点击更多，进入到【消息列表】。

![](https://vip.kingdee.com/download/0109457f79240a5a44459611d5d676bbc579.png)
