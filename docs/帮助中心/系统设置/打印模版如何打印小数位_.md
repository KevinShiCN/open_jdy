# 打印模版如何打印小数位?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：云社区用户5Uv15796 | 阅读：1125 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/534899801100359168)

【业务场景】

打印单据如何设置数量/单价/金额的小数位?

【解释说明】

设置打印模板时可以设置数量、单价、金额字段的小数位。

若是在开单界面设置数量、单价、金额小数位,详细可以点击:[怎么设置数量、单价、金额的小数位数?](https://vip.kingdee.com/link/s/ZlNYH)

【操作步骤】

依次点击【系统设置】>【打印模板】进入打印模板列表界面,

![](https://vip.kingdee.com/download/0109b5d27fc4811647ada59aa6a38ff199d0.png)

在左侧选择对应单据类型,再打开对应的打印模板([系统预设模板不支持修改,建议复制系统预置模板后修改](https://vip.kingdee.com/article/639992744571342080?productLineId=35&isKnowledge=2&lang=zh-CN)),以采购入库单为例:

![](https://vip.kingdee.com/download/0109c356b5c913e946cc808712e5cc912e64.png)

打开打印模板设置界面,双击选中要设置的单元格,右侧选择打印控件中,找到小数位数即可调整打印的小数位。

如:设置为“0”则表示保留整数,“1”表示保留一位小数,依次类推,设置完成后保存模版,再打印即可。
![](https://vip.kingdee.com/download/01090ac6d905cfcf405d9923ccecd67f93a0.png)
【注意事项】

以上只适用于云星辰3.1版本
