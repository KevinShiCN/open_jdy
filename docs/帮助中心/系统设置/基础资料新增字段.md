# 基础资料新增字段

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：快速入门, 精华汇总, 功能介绍
> 作者：进销存赵嘉丽 | 阅读：920 | 有用：8
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/639482227612420096)

## 1 简介

### 1.1 功能介绍

通过对基础资料进行新增自定义字段，满足企业业务需求。适用基础资料有供应商、客户、商品、会员，潜在客户。

![](https://vip.kingdee.com/download/0109e84ae813b15545ad9c369df19c8cef05.png)

### 1.2 适用版本

金蝶云·星辰3.1以上所有版本

### 1.3 系统路径

系统设置>个性化设置>基础资料新增字段

### 1.4 功能特性

- 表头最多可自定义20个字段，表体目前不支持。
- 字段类型支持：文本、数值、日期或引用辅助资料。

| 字段类型 | 属性 | 其他 |
| --- | --- | --- |
| 文本 | 长度上限1000 |  |
| 数值 | 小数位0~10位 | 只能设置纯数值型，不支持设置公式 |
| 日期 | 支持2种类型：年月日年月日时分秒 | 支持设置默认取系统日期 |
| 引用辅助资料 | 支持从基础资料>业务资料>辅助资料中直接引用值 |  |

- 字段支持设置必录

## 2 操作指南

### 2.1 创建基础资料自定义字段

下面以商品自定义字段引用辅助资料的使用举例说明：

1）首先建立名为“商品保存方式”的辅助资料：

![](https://vip.kingdee.com/download/010006df4c8834d9494188bcdd54464d9336.png)

2）在基础资料新增字段—商品里，新建名为“存放方式”的基础资料自定义字段，类型为“引用辅助资料”，值来源为“商品保存方式”（即上一步设置的自定义辅助资料）

![](https://vip.kingdee.com/download/0100a5c3919bca8747468e60840ddb28a8e9.png)

3）到商品列表，新增商品“冰激凌”，选择存放方式为“冷藏”，完成。

![](https://vip.kingdee.com/download/010008bf9318448f47cbae254527e6a9ab17.png)

说明：筛选查询如同单据新增字典，即在资料列表列和方案查询-高级查询进行筛选查询。

### 2.2 基础资料自定义字段在单据中的应用

应用流程：先建立商品的自定义字段，再建立单据的自定义字段，选择“带出已有基础资料属性”，可选择商品的自定义字段。

下面还是以商品为例。

比如，商品A在企业内部管理中的条码是9136**55，在超市中使用条码966**746来管理，企业在给超市送货时，送货单上需要打印超市条码，方便超市收货。那么，我们就可以通过下面的方式来实现：

1）在基础资料自定义中，建立商品的自定义字段“超市条码”

![](https://vip.kingdee.com/download/0100810fd33a1f6149d8ae030299bd2c736a.png)

2）在商品资料中，录入商品A的“超市条码”：

![](https://vip.kingdee.com/download/0100b4f62f7cbfa544cfb8353e8c97df858d.png)

3）在单据新增字段中，选择表头，建立销售出库单的自定义字段，选择“带出已有基础资料属性”，选择值来源为“商品—超市条码”。

![](https://vip.kingdee.com/download/010069e03fa79cee46cba93557d2fc99c8b3.png)

4）在新增销售出库单时，可以看到单据中已显示“超市条码”，并可以打印设置时，将此字段的内容打印出来。

![](https://vip.kingdee.com/download/0100d883cbc07c6e42eaa89fb0864be1e160.png)
