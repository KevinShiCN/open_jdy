# 如何设置自定义打印模板？

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：快速入门, 常见问题
> 作者：进销存赵嘉丽 | 阅读：3529 | 有用：8
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/139423908427583232)

**【问题概述】**

如果不想用系统预设模板打印单据，是否支持自定义打印模板？

**【解决方案】**

支持。应用流程如下。

1）在打印模板界面左侧功能分组中，找到要自定义模板的单据类型，勾选系统预设模板进行“复制”，再进行修改，或直接点击“新增”，或直接“引入”已有的本地模板

![上传图片](https://vip.kingdee.com/download/010019750f0da867447eb489adf06f790038.png)

2）点击“复制”或者“新增”后，会进入自定义模板编辑界面，编辑完成后可以预览效果，完成后保存即可。

![上传图片](https://vip.kingdee.com/download/01008b19c218a6614098ab8eb1ff7c911577.png)

3）如果是通过“引入”方式添加的模板，点击“模板编码”，则可以进入打印设计器页面，然后进行自定义编辑及调整，完成后保存即可。

![上传图片](https://vip.kingdee.com/download/0100be6ef66f8baf4669aa59ef3b8c8c6cb7.png)

4）在单据界面或者单据列表，点击"打印>打印设置"

5）在“选择模板”中选择自定义模板，点击“确定并打印”，则可按该模板进行打印

![上传图片](https://vip.kingdee.com/download/0100be7baca1f48c4bad8fe9dff4a435c769.png)

**【适用版本】**

星辰3.1以上所有版本
