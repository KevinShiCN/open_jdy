# 企业互联、外部商品库、商品共享是什么关系?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题

【问题描述】
企业互联、外部商品库、商品共享是什么关系?

【解释说明】
概括来说:外部商品库、商品共享(商品发送)是【企业互联】大功能下细分的功能;企业互联是载体,外部商品库和商品共享是依托于这个载体做的信息流的互通。

![](https://vip.kingdee.com/download/010900ce8ff8e0064afd89d798ba811ab23b.png)

**外部商品库:**建立本企业与外部合作伙伴(如客户、供应商或其他系统)之间的商品编码和名称的对应关系,以便在业务单据中准确展示对方所需的商品信息,支持跨系统的单据传递和对接;涵盖了原企业互联所对应的“供应商商品编码”、“客户商品编码”两个功能,并在原功能基础上扩展了新功能以及单据引用逻辑上的变更。

![](https://vip.kingdee.com/download/01095e28a183b79142839d27edbbdb8ad69f.png)

**商品共享(商品发送):**将商品资料从A账套发送到B账套,实现B账套的商品资料快速创建,并在A账套中的《外部商品库》中自动创建客户商品编码匹配关系,为企业互联的基础数据搭建提供了更便捷的途径。
