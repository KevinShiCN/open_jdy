# “分页拼接打印”与“纵向节纸”有什么区别？

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：236 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/778994863822861824)

# 问题描述

“分页拼接打印”与“纵向节纸”有什么区别？

# 详细解析

## **一、使用场景不同**

- **分页拼接**

打印一般适用于是模板存在多个分页（参考下图红框）的情况，常见于合同打印，即一张单打印内容超过一页A4纸的业务。**其主要目的更多在于保持不同页面的连贯及美观**。

![上传图片](https://vip.kingdee.com/download/0100b89ebfdf34be48acb0d62f4958e91575.png)

- **纵向节纸**

一般适用于多张不同单据的拼接，通常应用于多单打印时，上一单没有打满，下一单接着上一单的空白接着大，**其主要目的更多在于节省纸张**。

## **二、适用要求不同**

| 参数名称 | 要求 |
| --- | --- |
| 分页拼接打印 | 要求上一张分页必须铺满。也就是页面基本都塞满了，没有过多的留白，此时下一张分页才会实现拼接，如果上一张分页底部还剩余较多的空白，那么系统就会认为上一页没有打印完，此时就不会实现拼接。分页2的控件可以完整打印在分页1中 |
| 纵向节纸 | 会判断上一页分页的剩余高度是否可以容纳下一页分页的全部内容。换句话说，要求下一张分页的全部内容可以完整打印在上一张的空白处。 |

## **三、打印效果演示**

下面通过具体例子进行说明。

- **分页拼接打印**

| 场景 | 打印模板的设计条件 | 打印模板效果 | 打印预览 |
| --- | --- | --- | --- |
| 一 | 1、勾选了“分页拼接打印”。2、分页1底部存在较多空白，没有铺满。3、分页2存在部分信息，其高度小于分页底部剩余的空白， |  | 分页2没有拼接到分页1的底部 |
| 二 | 1、勾选了“分页拼接打印”。2、分页1底部基本铺满。3、分页2存在部分信息，其高度小于分页底部剩余的空白。 |  | 分页2拼接到了分页1 的底部 |

>
结论：场景一和二的条件1、3是一样的，唯一不同是条件2。侧面说明了，上一张分页必须铺满了，下一张分页才会实现拼接。

- **纵向节纸**

会判断上一页分页的剩余高度是否可以容纳下一页分页的全部内容。换句话说，要求下一张分页的全部内容可以完整打印在上一张的空白处。这里包含2个情况，一种是分页内只包含单个控件的，另一种是分页内包含多种控件，下面通过2个例子进行更好的说明。

| 场景 | 打印模板的设计条件 | 打印模板效果 | 打印预览 |
| --- | --- | --- | --- |
| 一 | 1、勾选了“纵向节纸”。2、分页1底部存在较多空白，没有铺满。3、分页2只有1个控件。其高度小于分页1底部剩余的空白。 |  | 分页2拼接到分页1上，打印成了一张单。 |
| 二 | 1、勾选了“纵向节纸”。2、分页1底部存在较多空白，没有铺满。3、分页2只有1个控件，其高度大于分页1底部剩余的空白。 |  | 分页2没有拼接到分页1上，打印成了两张单。 |
| 三 | 1、勾选了“纵向节纸”。2、分页1底部存在较多空白，没有铺满。3、分页2有多个控件，全部控件的累计高度大于分页1底部剩余的空白。 |  | 分页2没有拼接到分页1上，打印成了两张单。 |

>
结论：场景一二三的条件1、2是一样的，唯一不同是条件3。侧面说明了，纵向节纸是否生效，是取决于下一页分页的全部内容是否可以一次性打印在上一页分页的底部。不管是单个控件或多个控件，能否打印在上一分页，需要判断整体高度是否小于上一页分页的底部剩余高度。
