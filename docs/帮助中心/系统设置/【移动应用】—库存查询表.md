# 【移动应用】—库存查询表

## 1. **简介**

### 1.1 **功能介绍**

移动端库存查询功能，包含可用库存查询表、预计可用库存查询，支持表格样式和卡片样式的切换。能查询商品的可用库存、即时库存、预计可用库存、库存成本、保质期、批次、辅助属性、仓位等

- 即时库存：仓库真实存在的库存
- 可用库存：仓库真实库存+入库在途-即将出库的或预留出库
- 预计可用库存：主要适用于批次/保质期/序列号商品、或普通商品销售订单和采购订单上没有录仓库、仓位时特有的库存

系统设置>进销存参数中，可设置可用库存、预计可用库存的计算方式

### 1.2 **适用版本**

星辰3.1及以上版本

### 1.3 **场景介绍**

**手机端查询不同商品的场景**

- 查询商品库存（即时库存、可用库存、预计可用库存）
- 查询商品序列号
- 查询商品不同批次库存
- 查询商品不同保质期的库存
- 查询商品在不同仓库的库存
- 查询商品在不同仓位的库存
- 查询商品不同辅助属性的库存
- 查询商品的基本单位和辅助单位的库存

**手机端查询表不同角色的使用场景**

- 支持表格和卡片样式的切换
- 支持卡片底部的合计项设置
- 支持扫商品条形码快速搜索

### 1.4 操作路径

应用>仓库管理>库存查询表

## 2. **操作步骤+效果呈现**

### **2.1 列表查询**

进入默认显示可用库存查询表，点击界面左上角“可用库存”可选择切换为预计可用库存查询表

![上传图片](https://vip.kingdee.com/download/0100b6736b7deaa04fd0a53871b9972b96b5.png)

点击界面右上角的设置按钮，可以切换【表格样式】、【卡片样式】

- 卡片（下图左）：适合仓管查看商品的库存、商品数合计。可点击跳转查看详细信息
- 表格（下图右）：适合业务员查询商品库存、价格、成本、批次、保质期等信息；支持左右拖动查看更多的信息，底部支持合计项查看

![上传图片](https://vip.kingdee.com/download/01009151f1680dc84374b11136b10bbfb481.png)

- 表格模式可以设置显示表格中显示的字段

![上传图片](https://vip.kingdee.com/download/0100fefca063c2c0460db9c60b94d8b663c9.png)

- 卡片模式可以设置底部的合计项

![上传图片](https://vip.kingdee.com/download/01000e9272df511a48d68643fda251208438.png)

### **2.2 筛选/成本核算**

如果需要更精准的查询商品，可以通过【筛选】功能查找

多选查询条件：商品类别、商品标签、商品品牌、仓库分类、仓库、仓位、规格型号、辅助属性

参数设置条件：显示辅助属性、显示仓位、显示批次和保质期

库存时间：默认即时库存，可以切换指定时间查询

其他条件：显示零库存商品、显示已禁用商品、仅显示无发生额商品（无出入库记录）

计算成本：可手动进行出库成本核算。支持选择核算时间范围、核算仓库。完成核算后即更新最新的成本

![上传图片](https://vip.kingdee.com/download/0100c74c11eff6db4b05ae83679f3746a623.png)

### **2.3 列表联查**

- 查询到商品后可以点击商品进入联查到【仓库】，可以看到每个仓库的商品数量
- 如果开启了仓位、批次、保质期、辅助属性，则点击仓库能查到详细库存

![上传图片](https://vip.kingdee.com/download/01000cb004caabc64115bbfd394ec27d4a5e.png)

- 如果商品开启了序列号，可以查看详细的序列号

![上传图片](https://vip.kingdee.com/download/0100374db000cd3a4c7db5dcc6dc199dd174.png)
