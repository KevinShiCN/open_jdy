# 库存功能优化（2025年）

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：功能介绍, 新特性, 快速入门
> 作者：进销存赵嘉丽 | 阅读：681 | 有用：0

# **1 简介**

## **1.1 功能介绍**

- 销售订单保存时即可校验负库存，库存不足时不允许保存单据
- 未审核的采购订单也可增加可用库存量
- 下销售订单时不管控库存，下推发货通知单时需要自动预留，库存不足时不允许保存发货通知单
- 审核时更新库存，支持在单据保存、提交时提醒库存不足
- 仓库允许负库存时，支持提醒库存不足

## **1.2 功能特性**

- 可用库存公式、预计可用库存公式中，增加：“未审核销售订单”、“未审核发货通知单”、“未审核采购订单”、“未审核收料通知单”的勾选项，未审核单据可计入可用库存、预计可用库存中。
- 可用库存公式中，“未审核销售订单”、“未审核发货通知单”支持自动预留，其中可设置销售订单不参与可用库存计算，发货通知保存时即自动预留
- 新增“单据负库存预警”参数，可自行配置需要负库存预警的单据、仓库允许负库存时是否需要预警

## **1.3 适用版本**

星辰3.0以上所有版本

# **2 ****详细功能介绍**

## 2.1 可用库存公式、预计可用库存公式中，增加未审核单据

### 2.1.1 系统参数优化

| 公式类型 | 增加勾选项 | 产品示意图 |
| --- | --- | --- |
| 可用库存公式 | 未审采购在途量（未审核采购订单）未审收料待入量（未审核收料通知单）未审销售占用量（未审核销售订单）未审发货通知占用量（未审核发货通知单） |  |
| 预计可用库存公式 | 未审采购在途量（未审核采购订单）未审收料待入量（未审核收料通知单）未审销售占用量（未审核销售订单）未审发货通知占用量（未审核发货通知单）注意：包含仓库为空的单据 |  |

### 2.1.2 库存公式优化说明

①同一张单据（以销售订单为例），勾选了未审核销售订单，则已审核销售订单必须勾选，可以只勾选已审核订单，但不能只勾选未审核销售订单，勾选交互上进行了控制

②未审核单据勾选后，则单据保存时就会计算库存，未审采购订单、未审收料通知单会增加可用库存量/预计可用库存量，删除时会校验负库存，库存不足时则不允许删除；未审销售订单、未审发货通知单会扣减可用库存量/预计可用库存量，库存不足时不允许保存

③针对批次、保质期、序列号、辅助属性商品订单不包含商品批次，保质期、序列号明细信息、辅助属性信息，或订单中未录入仓库仓位时，可用库存不会扣减此部分商品的订单数量，保存时也不会校验负库存，如有需要可以开启自动预留

## 2.2 自动预留参数优化

1）可用库存公式中勾选未审核销售订单后，可以在销售订单保存时就自动预留，占用住库存，库存不足时不允许保存，针对特性商品也可自动分配批次、保质期、仓位后来判断库存是否足够、是否能保存成功

![](https://vip.kingdee.com/download/0100c319ac61954a4fca9252023bd03a8f02.png)

若销售订单保存时不想占用库存，审核时再占用库存，则可以不勾选未审核销售订单，只对已审核销售订单进行自动预留：

![](https://vip.kingdee.com/download/010000278b45d71f4c24957e29f87c54627a.png)

2）当销售订单不需要计入可用库存，只在发货通知单中扣减库存时，则可开启对发货他通知单的单据预留

![](https://vip.kingdee.com/download/010092908b21ab264ab19d80c7c1ec2c560e.png)

此时就可支持业务场景：不管有没有库存，都可以做销售订单，下推发货通知单后，就需要预留好足够的库存，以便仓库拣货发货时库存是充足的

## 2.3 单据负库存预警

### 2.3.1 新增“单据负库存预警”参数

![](https://vip.kingdee.com/download/010007aae38846524242a7a90abc58817d99.png)

开启后可设置单据范围，选择哪些单据需要负库存预警：

![](https://vip.kingdee.com/download/01006fa0cd808675403885fb28007ab1adea.png)

勾选单据后，则单据保存、提交等操作节点就可以进行负库存预警，当前库存减去当前单据中的数量后，若出现负库存，则会进行预警提示，仓库允许负库存且出现库存为负时，也会进行预警提示。

### 2.3.2 仓库允许负库存时不提醒

若仓库允许负库存时，不需要进行负库存预警，则可以勾选参数（此参数对PC和移动都生效）：

![](https://vip.kingdee.com/download/01004418f31e3ee8424f8901bd4a2f1e8ca5.png)

### 2.3.3 特别说明

①对于不允许负库存时的库存校验，**若当前操作节点本身会控制负库存，则不会进行预警**。

例如保存时更新库存，销售出库单保存时本身就会扣减库存，库存不足时不允许保存，则此时会直接提示负库存控制的弹窗，不会预警了（**此时弹窗中不会有“继续”的按钮，单据无法继续往下执行**）：

![](https://vip.kingdee.com/download/010019c195eb2ff94bd390276048755e3f46.png)

②对于启用审批流的单据，且审核时更新库存，由于进入到审批流中后，单据无法调取业务功能，所以审批流中间及最后一步的审批环节，**无法进行负库存预警**，目前只能支持在单据保存、提交时预警

③销售订单、发货通知单开启自动预留后，**也不支持负库存预警**
