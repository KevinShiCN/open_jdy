# 星辰2.0，点击打印提示“汇总打印目前只支持商品维度，请检查对应的打印模板”是什么意思呢？

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：快速入门, 精华汇总, 常见问题
> 作者：进销存赵嘉丽 | 阅读：964 | 有用：1

问题描述：

![](https://vip.kingdee.com/download/010061e46b907802421191154ff1abcceaf4.png)

![](https://vip.kingdee.com/download/010020674a47c33a4e99876c9c621bfc4224.png)

解决方案：

答：使用汇总打印的前提是打印模板的数据表格中中必须含有商品编码维度的相关字段（如：商品编码、商品名称、规格型号等等），如果没有，那么系统就取不到汇总的依据，因此会抛出上述的提示。因此，如果需要使用打印设置中合并汇总打印的功能，必现打印模板中的数据表格绑定对应的商品编码维度字段字段。

举例：在打印模板的表格中绑定商品名称后，可以正常打印了。
