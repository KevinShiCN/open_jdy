# 企业互联,能接收客户的退货吗?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：ylim | 阅读：75 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/621873469587549952)

【问题描述】

企业互联,能接收客户的退货吗?

【应用场景】

做了客户企业互联,如果客户(采购方)发生退货情况,希望客户在做采购退货单时能发送退货单过来到本企业中生成销售退货。

【解决方案】

点击【系统设置】-【互联客户】

![](https://vip.kingdee.com/download/01093f6714f667c842f9a3f24732d60dc44c.png)
在设置互联对接时,勾选“接收该客户的采购退货单”选项,若是已有设置记录,也可以点击客户编码进行修改勾选,勾选后保存即可。

![](https://vip.kingdee.com/download/01099af17d02a82e459790f795d076957970.png)
