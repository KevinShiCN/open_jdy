# 单据新增字段中的适用单据显示灰色是什么原因?

【问题描述】

单据新增字段中的适用单据显示灰色是什么原因?

【解释说明】

当自定义字段启用设置公式时,适用单据就为灰色了,无法编辑,设置公式不支持跨单据适用
