# 如何把商品类别设为必填项?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：BBAABABA | 阅读：642 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/459160614045094400)

【业务场景】

想要把商品信息里的商品分类设为必录项,应该如何设置?

【解决方案】

通过【单据字段配置】的功能,可以设置基础资料中某些字段是必录项,或者为非必录项。操作步骤如下:

①【系统设置】-【单据字段配置】-【基础资料】-【商品信息】;

②把【商品类别】的必录按钮启用;

③最后,点击右上角【保存】即可。![](https://vip.kingdee.com/download/010984ae5bac5d384f34886b01944e3c7d08.png)
