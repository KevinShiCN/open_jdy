# 出纳参数

# 1 简介

## 1.1 操作路径

系统设置>系统参数>出纳参数

## 1.2 适用版本

星辰3.1以上专业版、旗舰版

# 2 参数含义及应用场景

## **2.1 出纳管理参数**

1）启用期间：即建账时设置的系统启用期间，刚启用的账套，如果没有录入任何数据，可以修改启用期间，录入数据如果需要再修改，系统会给出对应提示，根据更改到的期间进行业务数据的判断。

2）当前期间：当前出纳所在期间，仅显示，不允许修改，由系统自动判断显示

3）登账方式：系统提供两种登账方式：

①凭证登账与手工登账：手工录入日记账或凭证保存/审核/复核后自动生成日记账；

②单据登账与手工登账：手工录入日记账和业务单据生成日记账列表操作登账

## **2.2 银企互联配置**

为了提升企业资金管理效率，金蝶云星辰上线了银企互联功能，申请银企互联后，可直接在星辰查询企业最新的交易动态及余额，无需登录银行系统。

目前对接的银行有：交通银行、招商银行、平安银行、广发银行、民生银行、浦发银行，星辰正在快马加鞭陆续对接更多的银行，后续将为大家提供全方位的银企互联服务，请大家持续关注哦~~
