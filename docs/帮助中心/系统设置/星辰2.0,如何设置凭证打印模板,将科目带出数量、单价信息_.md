# 星辰2.0,如何设置凭证打印模板,将科目带出数量、单价信息?

【问题描述】

星辰2.0,如何设置凭证打印模板,将科目带出数量、单价信息?

【解决方案】

1、在打印设置的界面,选中分录单元格,分别绑定数量和单价字段

![](https://vip.kingdee.com/download/0109a62c0296159d4b798bbf2ea48d51a6ae.png)

   ![](https://vip.kingdee.com/download/0109dc21d5fe508a4371803bb635ebfa1da5.png)

2、注:打印出来如果没有启用数量金额的科目打印出来带有0,要设置显示为空值,需要在格式界面,将零值显示为空注:打印出来如果没有启用数量金额的科目打印出来带有0,要设置显示为空值,需要在格式界面,将零值显示为空
