# 打印模板另存为有什么用？

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：快速入门, 常见问题
> 作者：小微微 | 阅读：1621 | 有用：13
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/110336721371603200)

**问题描述：**打印模板另存为有什么用？

**应用场景：**

对于已经设计好的模板，若样式差异不大，通过“另存为”可以将A单据类型的模板另存为B单据类型的模板，如可将销售出库单的模板另存为销售退货单、销售订单的模板，可将采购入库单的模板另存为采购退货单、采购订单的模板，也可以支持不同单据类型的模板相互另存为。

![](https://vip.kingdee.com/download/0100150c45848613416a8691f123d9498ba9.png)

**操作方法：**

点击系统设置-打印模板，在打印模板列表界面，选择具体模板点击“另存为”，选择要另存为的单据类型，录入模板编码、模板名称，点击保存即可。

**打印模板间支持互相另存为，不限定单据类型，如采购类单据打印模板另存为销售类单据打印模板、仓库类单据打印模板甚至资金类单据打印模板；**
