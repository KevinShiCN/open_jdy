# 金蝶云星辰：企业微信的聊天工具栏【侧边栏】设置

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：精华汇总, 进阶技巧, 功能介绍
> 作者：移动生产白建凡 | 阅读：3745 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/361232949381305600)

场景说明：

- l 能跟客户沟通的同时，快速查询客户的单据、账单等资料信息。
- l 跟客户沟通确认快速开业务单据。
- l 客户、供应商对账。

![](https://vip-admin.kingdee.com/download/0109086e62edd79c438f9bd1086c74b4f4da.png)

#### 1、**后台设置：管理员可以进入**

电脑端，企业微信中，进入管理后台；

客户于上下游--聊天工具--聊天工具栏工具管理

![](https://vip.kingdee.com/download/0109e13f81dfc3f24f7b8388529fc49cc365.png)

#### 2、**新增工具方法**

- 手机端，打开【我的客户】的聊天页面
- 底部的工具栏，拖到最后的【添加】
- 选择【金蝶云星辰】，进入某个页面后，点击底部【添加到工具栏】

![](https://vip.kingdee.com/download/010954751b153acf49d1b65acda683251db3.png)

#### 4、**功能演示**

**电脑端**

- l 在企微的聊天列表中移除原来的两天的客户；
- l 通讯录>我的客户>客户列表，聊天；
-  **聊天沟通的时候可以处理业务单据：**
- **根据个人的要求，调整工具**

![](https://vip.kingdee.com/download/010929d603d0739b4bf99fd3d55f68ebd927.png)

**手机端**

- l 客户列表中，跟客户聊天。
- l 聊天中，底部的工具栏，可以快捷进入功能。
- l 进入【添加】可以设置底部的菜单工具；调整顺序。
