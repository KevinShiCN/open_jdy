# 如何给子账号设置打印单据/条码/报表的权限?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：ylim | 阅读：492 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/571500951718920704)

【问题描述】

如何给子账号设置打印单据/条码/报表的权限?

【解决方案】

点击【系统设置】>【用户管理】

![](https://vip.kingdee.com/download/0109b26b8b0e0b624d3ab8351bd5c9bee434.png)
勾选需要设置授权的用户,点击【授权】按钮

![](https://vip.kingdee.com/download/010980c63e0d205840c0af42d1703828f995.png)
表单类,比如销售类单据,除了必须勾选查询权限外,还需要勾选“打印”的权限,设置后保存(保存成功后子账号重新登录即可)

![](https://vip.kingdee.com/download/0109d2625a44a535438bac11043b3f2dbdf4.png)
打印条码权限,在【基础资料】下找到“条码标签打印”勾选后保存

![](https://vip.kingdee.com/download/01090f2617e6084741fe9ea93cad9d3c274d.png)
