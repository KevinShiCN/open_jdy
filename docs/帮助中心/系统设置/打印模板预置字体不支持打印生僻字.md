# 打印模板预置字体不支持打印生僻字

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：快速入门, 精华汇总, 常见问题
> 作者：进销存赵嘉丽 | 阅读：29 | 有用：0

# **1 问题描述**

“𬊤、鿏、𰻞”这三个字符在打印模板中可以手动输入，单据上也能显示，但是用系统打印模板中预置的所有汉字字体打印时，均无法显示出来，这是为什么？

![上传图片](https://vip.kingdee.com/download/010084bde454c8a84c7c975d93468731182a.png)

# 2 适用版本

星辰3.0及以上所有版本

# **3 原因解析**

首先，打印时能否显示对应的内容，取决于打印字体是否可以支持该字体。

其次，这三个字符属于 Unicode 中的基本区生僻字或者高扩展区的字形，而系统中的打印模板预置汉字字体，只能覆盖到基本扩展区常用字。基本区生僻字、高扩展区字符主要见于古籍、方言、地名、姓氏、考古文献等场景，日常办公极少用到，也是普通字体不覆盖的核心原因。

# **4 解决方案**

若遇到其他同样的录单可以录入，但是打印时无法显示的字符，可以先试试切换打印模板内预置的所有字体，点击“预览”看一下效果。

![上传图片](https://vip.kingdee.com/download/01009d08f2ea41c445c59c870ad278200bf8.png)

若都试过无法显示，要实现正常打印，需选用**覆盖对应 Unicode 区块的完整字形字体**，换句话说，需要下载能支持打印该字符的字体（需要用户自行在第三方渠道购买或搜索，星辰无法提供购买渠道），再通过[打印字体管理](https://vip.kingdee.com/link/s/ZtMgI)导入。
