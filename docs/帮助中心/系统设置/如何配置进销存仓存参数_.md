# 如何配置进销存仓存参数?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：Cat_Chow | 阅读：391 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/634198479488880640)

【业务场景】

如何配置进销存仓存参数?

【解释说明】

单击【系统设置】→【系统参数】→【进销存参数】,即可以对进销存系统的业务处理规则和处理惯例进行统一的设置和管理。进销存参数主要分为四部分:全局参数、采购参数、销售参数和仓存参数。

![](https://vip.kingdee.com/download/010944644f26622045f9b369a58e586b9128.png)

其中仓存参数主要包括以下几点:

**1、通用**

1.1【批次商品在出库时批次自动填充规则,系统提供三个选项】:手工选择、批号顺序出库、近效期先出。此参数设置用于当商品启用了批次或批次+保质期管理时,商品出库批次是否自动填充。控制规则如下:

①手工选择:出库时,需要手工录入或选择批次相关信息;

②批号顺序出库:出库时,按批号顺序自动取批号,如果商品同时启用的保质期管理,则同时会取对应批号的保质期信息;

③近效期先出(同时启用批次和保质期商品):出库时,按保质期到期先后顺序取批号和保质期信息;注意此参数既适用于同时启用批次和保质期商品,也适用于只启用保质期管理的商品。

1.2 【设置库存查询表是否受库存数据权限控制】

①仅可查看仓库权限的库存:选择此项时,若用户做了仓库的数据授权,则在查看报表时看到的数据仅是有权限仓库的数据

②可查看所有仓库的库存:选择此项时,用户即使做了仓库的数据授权,查询报表数据也不受影响,能查询到所有仓库的数据

1.3【库存更新的方式】

①保存时更新:单据保存即更新系统的即时库存数量,如果不允许负库存出库,则在出现负库存时,单据无法保存,是单据审核前就扣减库存的管控方式。

②审核时更新:单据审核时才更新系统的即时库存数量。如果不允许负库存出库,则在出现负库存时,单据无法审核。

**2、负库存控制**

2.1 可设置订单类单据负库存控制方式为不控制或按可用库存控制(订单包括:采购订单、销售订单等非出入库类单据)

2.2 出入库类单据负库存控制方式

①当订单类单据负库存控制方式选择不控制时,出入库类单据负库存控制方式仅能按即时库存控制

②当订单类单据负库存控制方式选择可用库存时,出入库类单据负库存控制方式可选择按**可用库存**或**即时库存与可用库存双重控制**(单据保存/审核时,即时库存、可用库存都不允许出现负库存)

**3、可用库存参数**:该参数用于设置预计可用库存表中可用库存和单据录入时可用库存字段取数计算规则。企业根据实际业务需要勾选对应参数设置即可

**4、预计可用库存参数**:设置该参数可以对仓库为空的销售订单 或 已审核的销售订单同时进行扣减(即自定义预计出库量的计算范围),用户可结合实际业务场景来设置。一般适用于销售及采购所参考的可用库存依据是不一样的场景,如销售时是根据可用库存判断,而采购补货或者库存预警时,需要剔除对应的已审核的销售订单,以实现更精准的补货。

相关设置可参考:**[如何设置可用库存、预计可用库存公式](https://vip.kingdee.com/link/s/ZyQZ7)**

**5、库存预警与异常检测:**库存预警与异常检查是指两部分,库存预警检查与库存异常检查

1)可以设置商品无期初和业务时是否进行预警提醒。

2)可以设置浮动多单位商品若出库类单据扣减库存时,要不要检查基本单位与浮动单位库存是否异常的校验控制,具体是针对以下两点的校验检查。

①若基本单位库存量与默认浮动单位库存量一个为正一个为负时,可选择不控制、严格控制。
若选择【不控制】,最终浮动计量的商品库存可能会存在基本单位库存量、默认浮动单位库存量一个为正,一个为负,即使仓库设置了不允许负库存,也会出现某个单位库存量为负。
若选择【严格控制】,则在上述情况下系统会严格控制,不允许出库。需要调整本单出库数量、辅助数量或者其他调整库存的操作使最终库存不会出现上述情况为止才可以出库。

②若基本单位库存量与默认浮动单位库存量一个为零一个不为零时,可选择控制、严格控制。

若选择【不控制】,最终浮动计量的商品库存可能会存在基本单位库存量、默认浮动单位库存量一个为零,一个不为零,这时需要进行盘点才可以将基本单位库存量、默认浮动单位库存量都盘成零。

若选择【严格控制】,则在上述情况下系统会严格控制,不允许出库。需要调整本单出库数量、辅助数量或其他调整库存的操作使最终库存全部为零或全部为正才可以正常出库。

**6、组装单**

①允许修改组装费用(已审核组装单):与“允许修改采购费用(已审核采购类单据)”类似,控制是否允许在组装单已审核的情况修改组装费用。

②多组件业务,组件成本分摊比例计算依据:控制组装单上多个组件时,子件的成本+组装费用以怎样的比例分摊到多个组件上。提供按商品资料上的参考成本、数量两种方式进行分摊。

**7、拆卸单**

①允许修改拆卸费用(已审核拆卸单):与“允许修改组装费用(已审核组装单)”类似,控制是否允许在拆卸单已审核的情况修改拆卸费用。

②子件成本分摊比例计算依据:控制拆卸单上多个子件时,组件的成本+拆卸费用以怎样的比例分摊到多个子件上。提供按商品资料上的参考成本、数量两种方式进行分摊。

**8、调拨单**

①允许修改拆调拨费用(已审核调拨出库单/调拨入库单) :控制是否允许在调拨单已审核的情况修改调拨费用。

②调拨费用分摊比例计算依据:提供按商品资料上的参考成本、数量两种方式进行分摊
