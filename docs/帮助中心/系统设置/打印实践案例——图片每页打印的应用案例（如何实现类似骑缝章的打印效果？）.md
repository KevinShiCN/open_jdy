# 打印实践案例——图片每页打印的应用案例（如何实现类似骑缝章的打印效果？）

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：精华汇总, 进阶技巧, 应用案例
> 作者：进销存赵嘉丽 | 阅读：227 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/755829989353925376)

# **1 业务场景**

打印单据合同时，由于业务需要，除了最后一款的落款需要盖章以外，还需要每页固定位置都显示印章，以实现类似骑缝章的效果。

# **2 解决方案**

星辰不支持直接打印像骑缝章的效果，但是可以实现每页固定位置都显示印章，同时只在最后一页显示落款印章的双印章打印效果。



# **3 适用版本**

星辰3.1以上所有版本



# **4 涉及特性**

• 需提前准备好背景色是透明的印章照片，格式是PNG

• 添加背景图片及打印背景图片功能

• 表尾落款的布局网格和图片控件打印所在页设置为“顺序打印”



# **5 应用讲解**

下面通过采购订单为例子进行应用说明及注意事项。

>

第一步，打开一个A4纸大小的打印模板，依次添加好对应的表头、数据表格及表尾合同信息，

这部分的设计可以参考知识：打印实践案例——[合同打印（超长合同打印）/分页打印](https://vip.kingdee.com/link/s/ZptvD)

第二步，落款设计

首先，针对合同落款处的设计，需要使用“布局网格”来承载甲方乙方的信息，

![](https://vip.kingdee.com/download/0100b882f867801e4a8689ea8ef93a90938a.png)

布局网格设置好后，再添加图片控件，上传印章后，把图片放置在网格上对应的位置，并且图片的打印所在页设置为“顺序打印”

![上传图片](https://vip.kingdee.com/download/0100bba1a82ba63b4a0c894a95633b61cc96.png)

关于布局网格和图片的具体设计，可以参考知识：[打印实践案例——打印模板图片/合同公章设置常见问题及设计方案](https://vip.kingdee.com/link/s/Zl6XW)，本文就不再赘述。

第三步，添加空白分页

点击画布下方“分页1”右侧的加号，即可添加

![](https://vip.kingdee.com/download/01000a29e2f6b89d4a50bdab5cfd81598c10.png)

第四步，创建含有固定印章的背景图

在分页2中，添加图片，上传背景是透明的印章

![](https://vip.kingdee.com/download/01005bde18360ef54c5d83913a0b52c9354b.png)

点击预览，显示比例为100%，浏览器设置成全屏显示，然后利用截图工具（需要用户上网搜索对应的工具）把分页2按A4纸的完整尺寸截图保存到本地，

![](https://vip.kingdee.com/download/0100ca58f72fae44441b86eb6c25a731881d.png)

保存到本地后的效果

>
注意：黑色边框是为了让大家看得出是A4纸，实际截图时不要显示任何多余的边框线

![](https://vip.kingdee.com/download/010065cdf0ea8c6943a59cded6dac9c3f124.png)

第五步，添加背景图片

回到分页1，点击页面空白的位置，点击“添加背景图片”，上传第四步的截图，并勾选“打印背景图片”

![](https://vip.kingdee.com/download/01003fa79d37f58c4417942bf12013511357.png)

完成后，直接点击打印预览，查看是否符合预期。

![](https://vip.kingdee.com/download/01003a90cc44cd79400d8e05799ec9a48b48.png)

如果位置不合适，那么就删除背景图后，对分页2的图片位置进行调整，直至符合预期，再重新保存背景图。如果位置没问题，回到打印模板中，删除分页2，并保存模板。

第六步：验证数据表格内容超过一页的打印效果，确认实际分页打印时的效果。

可以看到，当数据超过一页时，可以实现每页显示印章，同时最后一页落款需要显示印章的效果。

![](https://vip.kingdee.com/download/0100902d1100f1ae4fd4a8a2fdd85a0f30db.png)

# 6 注意事项

1）本文例举的是分页只有1个的合同打印情况，当单据的设计内容超过1个分页时，也是相同的设置方法，因为分页1中设置的“打印背景图片”是适用于所有的分页的

2）不支持每个分页的背景图片单独设置，是按模板维度全局生效的

3）落款处的图片，打印所在页页面注意只能选择“顺序打印”，不要使用“最后一页”，因为“最后一页”会导致图片在最后一页的位置是固定的，那么当时落款的数据打印完后，该图片就没法跟随布局网格动态移动，有可能会导致印章没盖在网格上
