# 其他支出单打印审核人等字段总是跳到上面应如何处理?

> 作者：杨桂平 | 阅读：134 | 有用：0

【问题描述】

其他支出单打印审核人等字段总是跳到上面应如何处理?

如下图所示:

![](https://vip.kingdee.com/download/0109cf225f3bc18443b4920736ffe4b19486.png)

【解决方案】

模板里设置表尾区域,表尾字段在表尾区域里设置

删除已有的制单人字段

![](https://vip.kingdee.com/download/01099b8c17062d3e4ee488f1a7ab3e9d9347.png)

选中数据表,勾选表尾区域

![](https://vip.kingdee.com/download/01097e9ac24a515549b9a9d6e6d587d8d59b.png)
选中表尾区域后再添加取值,保存模板

![](https://vip.kingdee.com/download/010958f757306b024978820328c29812fc1b.png)
预览效果。
