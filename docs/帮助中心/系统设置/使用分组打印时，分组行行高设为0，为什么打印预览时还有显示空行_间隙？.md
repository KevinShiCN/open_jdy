# 使用分组打印时，分组行行高设为0，为什么打印预览时还有显示空行/间隙？

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：70 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/786317301069878272)

# 【问题概述】

使用分组打印时，数据表格中插入了分组行，并且分组行行高设置为0，但是为什么打印预览时还是有显示空行/间隙？

![](https://vip.kingdee.com/download/0100ed0202b3abf54cec905659fc893e335e.png)

![](https://vip.kingdee.com/download/0100bfd5347465204dde8a958bac0c465a2e.png)

# 【原因说明】

当分组行行高为0且数据表格“行高自适应”开启了，在目前系统逻辑下，系统会自动撑高行高，导致出现空行。

![](https://vip.kingdee.com/download/010072cf6fc14bbe4ee8915b4f7d43b5b189.png)

# 【解决方案】

1、 选中数据表格，关闭“行高自适应”，保存模板。

![](https://vip.kingdee.com/download/01003a6ddec29b6b415d93b6aacd2bcba85d.png)

2、如果数据表格中的内容需要实现行高自适应，就要对表格的单元格设置单独的行高自适应格式，选中单元格，再右侧控件属性“且内容超出时”中选择“行高自适应”，如果不涉及则可以忽略此点。

![](https://vip.kingdee.com/download/0100e7bc0d45c62b4c9dae5bd492589c730d.png)

这样修改后，再次预览就不会有空行/间隙了。

![](https://vip.kingdee.com/download/010096c919275b074fcd9e0a26c2fb348364.png)

后续星辰会反馈此逻辑给到苍穹平台进行优化。

星辰3.1以上所有版本
