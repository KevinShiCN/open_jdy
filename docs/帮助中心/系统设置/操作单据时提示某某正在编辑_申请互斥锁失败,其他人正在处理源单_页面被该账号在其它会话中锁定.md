# 操作单据时提示某某正在编辑/申请互斥锁失败,其他人正在处理源单/页面被该账号在其它会话中锁定

> 作者：ylim | 阅读：2906 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/580923107494356736)

【问题描述】

操作单据提示某某正在编辑/申请互斥锁失败,其他人正在处理源单/页面被该账号在其它会话中锁定

【解释说明】

系统为最大限度的保证网络并发控制和数据的一致性而触发的操作锁定。

- 用户出现异常中断、死机等原因可能造成单据锁定,会导致其他用户不能执行互斥功能,也属于网络控制的范围。
- 网络控制的目的是保证一个用户的工作不会对另一个用户的工作产生不合理的影响。例:数据的重复录入,造成数据不准确。
- 【操作步骤】
- 先确认是否有账号打开了这个单据在操作,**确定没有账号在操作对应单据的情况下,**
- 用有权限的账号在账套里点击【系统设置】-【网络控制】
- ![](https://vip.kingdee.com/download/010938e1b1359f9546a6939e40665e53727d.png)勾选控制记录,点击【删除】按钮,删除控制记录后即可刷新继续操作单据。![](https://vip.kingdee.com/download/010938ecf3c066854226b3d47b311aa4e590.png)

手机端也可以进行强制踢出,详细应用及解决方案请点击:[网络控制/强制踢出](https://vip.kingdee.com/link/s/l1uMz) 查看

【温馨提示】

如果发现系统设置没有【网络控制】的功能可以让管理员操作,或者让管理员给子账号授权该功能权限

在系统设置-用户管理界面,勾选子账户点击授权,在功能权限中,勾选网络控制的权限即可。
