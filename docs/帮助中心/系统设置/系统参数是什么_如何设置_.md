# 系统参数是什么?如何设置?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：功能介绍, 快速入门, 精华汇总

【问题描述】

系统参数是什么?如何设置?

【解释说明】

系统参数用于对账套整体进行控制,根据所购买的模块不同,系统参数中所显示的可控制参数不同,在设置时需根据自身业务需求进行设置。

**系统参数修改后即可生效,仅支持在电脑端设置**

【操作指引】

菜单栏点击【系统设置】>【系统参数】即可进入。

![](https://vip.kingdee.com/download/0109135c848772dc4a14a10bb03ba1de0551.png)

**注意,部分参数是要购买了模块才会显示的:如生产参数、委外参数、税务参数,具体参数显示请以您购买的模块及实际账套显示为准。**

【操作流程】

单击【系统设置】-【系统参数】,如下图所示,用户根据实际业务场景进行设置,设置后点击右上角的保存即可生效。

![](https://vip.kingdee.com/download/01098994a243edff4dc29a2810b2a95395b6.png)
