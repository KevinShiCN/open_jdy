# 星辰2.0，打印模板的字体使用了“黑体”，金额设置了显示币别格式，但打印预览是怎么都显示不出来，为什么？

解决方案：

答：目前星辰打印模板的“黑体”不支持显示币别符号￥、£ ，建议如需使用该两种币别符号时，字体建议使用微软雅黑和宋体。其他币别符号$、€、US$则不受此影响。
