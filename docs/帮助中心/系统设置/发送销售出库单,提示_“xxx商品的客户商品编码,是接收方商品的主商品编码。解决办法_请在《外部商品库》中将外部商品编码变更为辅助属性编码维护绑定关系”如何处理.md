# 发送销售出库单,提示:“xxx商品的客户商品编码,是接收方商品的主商品编码。解决办法:请在《外部商品库》中将外部商品编码变更为辅助属性编码维护绑定关系”如何处理?

【问题描述】
发送销售出库单,提示:“xxx商品的客户商品编码,是接收方商品的主商品编码。解决办法:请在《外部商品库》中将外部商品编码变更为辅助属性编码维护绑定关系”如何处理?
![](https://vip.kingdee.com/download/0109c62bcab490f149d3a46f8ed8bd8ea8a6.png)

【解释说明】
出现该提示是因为对方商品启用了辅助属性管理,需要将对方商品各辅助属性的的【SKU编码】维护到【外部商品编码】中,如果只维护商品编码,而辅助属性组合有很多,则会出现一对多的关系,系统无法准确识别,所以必现靠SKU编码进行唯一识别。

依次点击【基础资料】-【外部商品库】![](https://vip.kingdee.com/download/0109cc93f5a5c7cc481388d9edc1fcac9e8d.png)
找到对应的商品,并将对方商品各辅助属性的SKU码录入到【外部商品编码】中
![](https://vip.kingdee.com/download/01094cad8883c7f14d6190621d498ad2e3c7.png)
