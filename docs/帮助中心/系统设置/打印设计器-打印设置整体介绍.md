# 1. 业务场景

可以对纸张、打印位置和方式进行一些个性化的属性配置，使之更贴合实际业务场景，灵活满足不同的打印需求。

# 2. 适用版本

星辰3.1以上所有版本

# **3. 功能介绍**

## 3.1 基础属性

- **模板编码**：编码必须由小写字母、数字或下划线组成
- **模板名称**：可以自定义
- **文件名**：支持打印设计器中对打印文件自定义名称，引出PDF文件后，就会按该规则命名

具体应用流程详见知识：** **[打印文件名自定义命名功能](https://vip.kingdee.com/link/s/ZVbDy)

## 3.2 纸张设置

**纸张大小**：系统提供了预置纸张及自定义规格。详见知识：[打印基础——星辰预置纸张规格说明（纸张大小）](https://vip.kingdee.com/link/s/ZVAgv)

**纸张方向**：即纵向/横向，默认纵向 。

**页边距**：只能设置≥0的整数，当设置小数时不生效，假设录入0.2mm，只能录0或者1（如下图提示）。

## 3.7 文本控件中“且内容超长时”

**功能介绍**

当文本的内容不固定，如采购订单中备注字段，输入的字段的内容不确定多少，为了不影响其他控件的展示，若文本内容超过了控件大小，存在几种处理方式：截断、行高自适应、自动缩小。

**功能特性**

- 使用范围：文本控件及数据表格的单元格均支持
- 规则及应用场景

| 规则 | 应用场景 | 效果预览 |
| --- | --- | --- |
| 截断 | 当文本内容超出文本宽度时，宽高空间不够时截断展示，不改变宽高。 |  |
| 行高自适应 | 当文本内容超出文本控件宽度时，按照内容自适应换行，且行高自适应。不会影响或覆盖其他控件。 |  |
| 自动缩小 | 当文本内容超出文本控件宽度时，宽高空间不够时缩小内容展示，不改变宽高。注意：1）当数据表格开启了“行高自适应”时，数据表格单元格的“内容超长时-自动缩小”会失效，数据行会优先行高自适应，“自动缩小”不再生效；2）缩小比例与内容长度有关，内容越长，缩小比例越大。 |  |
| 溢出 | 当文本内容超出文本控件宽度时，且“文本换行规则”选择不换行时，将内容展示到控件外侧，不改变行高。 |  |

- 互斥说明

控件属性中的“文本换行规则”与“且内容超长时”设置存在逻辑互斥时，互斥的选项会置灰显示，表示不可选择。减少用户歧义，改善用户体验。

| 文本换行规则 | “且内容超长时”不可选 | 置灰效果（表示有冲突，不允许设置） |
| --- | --- | --- |
| 不换行 | 行高自适应 |  |
| 超出控件宽度自动换行按文字类型换行 | 溢出 |  |

## 3.8 打印公式功能

**业务场景**

当一个控件绑定一个字段值不满足场景时，我们需支持控件数据源由多个字段组合或通过函数获取。或者需要对数值型字段做一些运算处理时，可以通过公式来解决。

场景二：需要把多个字段拼接成一个字段打印，例如地址、或者把批次保质期等等

场景一：星辰的销售退货单不支持录入负数，但是可以通过打印模板的公式来满足单据打印的负数显示问题

**功能介绍**

## 3.9不打印

**业务场景**

通过在打印模板中放入图片作为模板背景，作为辅助参考线，或者预留位置，但是实际不打印图片出来达到效果。

一般适用于如财务的凭证模板设计、或者批发商品行业的价格标签或者一般有自己的表单样式的打印场景，要求打印时严格按照现有的纸张或者标签样式和位置来打印。

**功能介绍**
