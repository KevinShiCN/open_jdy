# 国际化支持多时区

# **1 简介  **

## **1.1 功能介绍**

多时区功能主要应用于需要针对在国外的客户设置系统切换的时区，或者跨国企业需要不同用户看到不同时区的场景。可以解决用户在不同时区做单及查询相关数据时，需人工转换时区的问题。从而帮助用户实现实现跨区数据的实时共享，实现全球协作的零时差办公。

## **1.2 适用版本**

星辰3.0以上所有版本

## **1.3 系统路径**

用户头像>个人设置>国际化设置

![上传图片](https://vip.kingdee.com/download/010022ee4fdd914d4cf7999c37594d39c812.png)

## **1.4 功能特性**

- 系统时区设置：用户可根据所在地或业务需求选择对应时区，系统自动换算显示时间
- 区域格式设置：支持自定义日期、时间格式，贴合个人使用习惯
- 消息推送时区设置：支持对预警消息、经营消息配置推送时区，配置后对该账套所有用户生效

# **2 操作指南**

## 2.1 系统时区和区域格式设置

### 2.1.1 时区设置

点击“国际化设置”，系统默认的时区为“东八区”，默认的区域格式为“中文（中国）”，当前北京时间为“2025-09-29 10:15”。点击时区右侧的“...”进入时区选择页面选择时区

![上传图片](https://vip.kingdee.com/download/0100beb6bd8d35104f62b21fd310aaf68bb6.png)

选择需要切换的时区后点击“确定”。如图示，将时区改为“（GMT-03：00）巴西利亚，里约热内卢”

![上传图片](https://vip.kingdee.com/download/0100b7bca0478994447a9804a576aa2d898a.png)

此时金蝶云星辰系统时间应比北京时间晚 8+3=11 小时，也就是2025-09-28 23:16”。查看系统时间变为“2025-09-28  23:16”

![上传图片](https://vip.kingdee.com/download/010017b658b895ca484bb65e140fe0311e6e.png)

### 2.1.2 区域格式设置

设置当前系统内时间、日期的显示格式。点击“区域格式”右侧的“...”进入区域格式选择页面

![上传图片](https://vip.kingdee.com/download/010016b500f9bfa547f19f57b4cd13de40b3.png)

选择需要的格式后点击“确定”。如图所示，选择“葡萄牙语（巴西)”

![上传图片](https://vip.kingdee.com/download/01000cf63c5611b0452f91c440416e2e9c18.png)

日期即由“2025-09-28 ”变为“28/09/2025”

![上传图片](https://vip.kingdee.com/download/01001871fa42c464434aaef6629224c6f8f3.png)

若系统预置的区域格式不满足您的需求，可以开启页面右上角的“自定义”按钮，自定义日期和时间格式

![上传图片](https://vip.kingdee.com/download/0100b7a2ed651a324796b1aa67e5d032346a.png)

点击击日期页签下“日期格式”、时间页签下“时间格式”右上角的“？”图标，可查看格式中各符号的含义说明：

日期：d,dd=日 m=月 y=年 E=星期

时间：h=小时 m=分钟 s=秒 a=上午或者下午 h/H=12/24 hh,mm,ss=小于10的数值显示零 h,m,s=小于10的数值不显示零

| dd/MM/yyyy |  |
| dd-MM-yyyy |  |
| M/d/yyyy |  |
| yyyy/M/d |  |
| yyyy/MM/dd |  |
| yyyy-M-d |  |

### 2.1.3 制单、查数据

（1）制单时

- “单据日期”“单据编号”等和日期相关的字段，都取当前用户时区下的日期
- “制单时间”、“审核时间”、“最后修改时间”、“变更时间”等和时间有关的字段，都取当前用户时区下的日期和时间

（2）查数据时

- “年月日”为非敏感时区，无论在哪个时区查看，均不做日期转换。也即：巴西利亚当地时间2025-09-28做的数据，北京时间查看相关日期字段也是2025-09-28
- “年月日+时分秒”为敏感时区，在不同时区查看都会做转换。也即：巴西利亚当地时间“2025-09-28 23:40”做的数据，北京时间查看制单数据为“2025-09-29 10:40”

   见下表图示说明：

|  | 巴西利亚时间录入 | 北京时间查看 |
| --- | --- | --- |
| 年月日 | 单据日期：2025-09-28单据编号：XSDD-20250928-00001 | 单据日期：2025-09-28，不做日期转换单据编号：XSDD-20250928-00001，不转换 |
| 年月日+时分秒 | 制单时间：2025-09-28 23:40 | 制单时间：2025-09-29 10:40，时间做转换 |

## 2.2 消息中心多时区设置

消息推送时区与系统时区相互独立，建议由管理员统一配置，配置后对该账号所有用户生效。

进入“系统设置>消息和预警>消息设置”，支持对预警消息、经营消息配置推送时区。点击消息对应的编码进入消息设置页面。默认时区为东八区，可根据需要调整为其他时区

![上传图片](https://vip.kingdee.com/download/0100684ad27f685d4a959a42191555f9cede.png)

默示例设置为东八区10:08。那么所有用户都会在北京时间10:08收到消息，只是不同时区用户的各自当地时间不同
