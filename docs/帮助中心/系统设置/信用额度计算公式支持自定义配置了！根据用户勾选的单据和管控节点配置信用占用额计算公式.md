# 信用额度计算公式支持自定义配置了！根据用户勾选的单据和管控节点配置信用占用额计算公式

1 简介

### 1.1 功能介绍

本次优化主要满足客户自己决定怎么计算信用额度的需求，因为不同客户对于信用额度占用的计算需求不同，之前的额度计算公式是写死的，不够灵活，所以本次优化成根据用户设置来配置计算公式。

### 1.2 适用版本

星辰3.1专业版/旗舰版

### 1.3 系统路径

系统设置-预警设置-销售信用额度预警/采购信用额度预警

### **1.4 功能说明**

根据用户勾选结果来调整信用占用额的计算公式，单据勾选则计入信用额度，不勾选则不计入；勾选保存时提醒，则计算未审核和已审核未执行的金额，勾选审核时提醒，则只计算已审核未执行的金额，不计算未审核的单据。

以销售订单为例，若勾选【保存时提醒】，则未审核销售订单和已审核未执行销售订单都占用信用额度；若勾选【审核时提醒】，则只有已审核未执行销售订单占用信用额度，未审核销售订单不占用。若不勾选销售订单，则不计算销售订单的额度占用。

此外，增加了【预收占用】参数，用户可以设置【未审核预收款单不计入信用额度】。

信用额度的具体占用情况以及可用额度情况可以去应收账款预警表查看。
