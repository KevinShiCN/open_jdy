# 用户权限 / 用户组权限是否可以批量下载、导入？

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题, 快速入门, 精华汇总
> 作者：移动生产白建凡 | 阅读：115 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/784060260138412032)

# **1 问题描述**

用户权限 / 用户组权限是否可以批量下载、导入？

# 2 适用版本

星辰3.0以上所有版本

# **3 功能说明**

**（1）下载权限：**用户权限、用户组权限仅支持单个下载，暂不支持批量下载

**（2）跨服务跨账套的权限复制：**

- 一个用户组权限可以单次导入到一个目标用户组
- 一个用户权限可以单次导入到多个目标用户

**（3）同服务不同账套的权限复制：**

- 一个用户组权限可以单次复制到多个目标用户组
- 一个用户权限可以单次复制到多个目标用户

若需批量导入用户组权限，可先将用户组权限导入对应目标账号，再通过 用户组复制权限功能实现批量导入

# 4 特别说明

用户权限、用户组权限支持批量引出（查询权限后，可将当前权限结果按列表以Excel文件进行引出，所见即所得

*注：“引出” 与 “下载” 是不同功能 —— 引出用于权限查看（参见：*[内置用户组及权限查询](https://vip.kingdee.com/link/s/ZAd1s)*），**下载用于跨账套（同服务 / 不同服务）的权限导入（参见：*[跨账套复制权限](https://vip.kingdee.com/link/s/ZAyp2)*）*
