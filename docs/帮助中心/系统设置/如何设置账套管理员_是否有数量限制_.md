# 如何设置账套管理员?是否有数量限制?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：1422 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/466408350218166528)

【问题描述】

如何设置账套管理员?是否有数量限制?

【解释说明】

需要登录**[服务管理员](https://vip.kingdee.com/article/606290444715876096?productLineId=35&isKnowledge=2&lang=zh-CN)**账号登录系统后给其他账号设置账套管理员权限,**账套管理员可以设置其他子账号的功能权限,但暂时不支持设置其他用户为账套管理员。**

账套管理员支持设置多个,**没有数量限制**,账套管理员拥有账套所有的功能模块权限,无需再授权。

账套管理员在用户管理界面可以邀请子账号并给子账号进行授权。

**该操作仅支持在网页端操作,手机端暂时不支持。**

【操作指引】

服务管理员登录后依次点击【系统设置】>【用户管理】,

![](https://vip.kingdee.com/download/01093d71e1d124d240aa89adf8dbe76e2f85.png)

选择需要授权的用户点击“设为管理员”即可

![](https://vip.kingdee.com/download/0109002cc827346f426e808f956d49f3e233.png)

**账套管理员登录**后,可以进入用户管理界面对子账号进行授权

![](https://vip.kingdee.com/download/0109bc06efc2c68a4b17b0f68d47b2541f15.png)

账套管理员无法取消其他账套管理员的权限,取消会出现系统提示:登录用户不是服务管理员,不能取消管理员!

![](https://vip.kingdee.com/download/010920662c26319e4a95aebb82030b36a50b.png)

**【注意事项】**

服务管理员具有唯一性,账套管理员支持设置多个;服务管理员权限高于账套管理员权限

**服务管理员**:支持设置其他子账号为账套管理员和取消管理员权限,支持设置其他子账号权限

**账套管理员**:不支持设置其他子账号为账套管理员和取消管理员权限,仅**支持设置其他子账号权限**

【温馨提示】

如需取消管理员账号,可参考查看:[如何取消账套管理员?](https://vip.kingdee.com/knowledge/568239416787928320?productLineId=35&isKnowledge=2&lang=zh-CN)
