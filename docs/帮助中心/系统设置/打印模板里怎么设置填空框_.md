# 打印模板里怎么设置填空框?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：快速入门, 精华汇总, 常见问题

【问题描述】

星辰打印模板里怎么设置填空框?

![](https://vip.kingdee.com/download/0109a232d100032c402d830540890dab718a.png)

【操作步骤】

1)在打印模板中,添加一个文本控件

2)对文本控件设置实线边框

3)把文本控件设成一样的宽和高,调整成正方形,完成。

**注意,无论是表格单元格内,还是表格外,都支持设置。**
