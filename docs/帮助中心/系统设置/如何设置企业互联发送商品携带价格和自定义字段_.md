# 如何设置企业互联发送商品携带价格和自定义字段?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：Simone邓 | 阅读：9 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/805242136731095552)

【问题描述】

如何设置企业互联发送商品携带价格和自定义字段?

【适用版本】

星辰3.0以上所有版本

**注意:该功能仅专业版支持,且目前处于灰度测试阶段。如需提前体验,请复制工作台的服务 ID 到售后在线服务界面发送给人工客服,申请加入灰度测试。**

【业务场景】

同一家公司的不同账套,使用企业互联发送商品和单据,需要将发送方商品价格资料、自定义字段和单据中的自定义字段发送到接收方账套。

【操作步骤】

【系统设置】-【企业互联】-【互联客户】

![](https://vip.kingdee.com/download/010971057bb6e9af490b8f982642298b840b.png)

点击客户编码打开互联客户参数界面,可配置否携带价格资料、自定义字段

**商品的价格资料:**包括采购价、最高采购价、委外价、受托加工价、参考成本、配送价、批发价、零售价、会员价、最低销售价,不包括客户等级价。

**自定义字段:**自定义字段则是根据《基础资料新增字段》中相同字段名称和字段类型匹配不同账套的自定义字段。支持文本、日期、数值、引用辅助资料的自定义字段的发送。但是,若接收方为公式型字段,不支持发送;若接收方的辅助资料分类和辅助资料名称不存在或不唯一,不支持发送。

![](https://vip.kingdee.com/download/0109ed3d6bd9bf2a46d6a168ace876280566.png)

另外,发送商品时,增加了默认仓库/仓位、生产的最小订货量/关键件/倒冲领料等、财务的进项税率/销项税率/商品开票名称等字段的携带,这些字段不受参数控制,默认携带。
