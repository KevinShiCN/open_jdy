# 修改总账或出纳启用期间,系统提示“出纳与总账一起结账,总账启用期间不允许修改”或“与总账一起结账时,不能修改启用期间”,应该如何处理?

> 作者：Schatz | 阅读：371 | 有用：0

【问题描述】

修改总账或出纳启用期间,系统提示“出纳与总账一起结账,总账启用期间不允许修改”或“与总账一起结账时,不能修改启用期间”,应该如何处理?

![](https://vip.kingdee.com/download/0109a32437ecf3ca420ebf9db284f0baa36a.png)

【解释说明】

这两个提示都是因为设置了出纳与总账一起结账,修改前需要将此参数取消勾选。

依次点击:【系统设置】-【出纳参数】-不勾选参数【与总账一起结账】

![](https://vip.kingdee.com/download/0109aae685185b7d43b396853c78950f974a.png)

如需继续使用资产和总账一起结账功能,可修改启用期间后,将此参数重新进行勾选即可。
