# 打印语言的使用方法

# **1 简介  **

## **1.1 功能介绍**

打印语言用于同一个模板在不同地区或国家需做相应转换的场景，以满足当地语言的使用习惯。

## 1.2 **应用场景**

- 如某企业当前系统语言为简体中文，在打印销售订单时，需给国外客户发送英文的打印模板样式。
- 外贸企业，在中国和泰国都有分公司，中国的员工使用的系统语言为简体中文，而在泰国的员工使用的系统语言是泰语，打印给当地客户的单据也需要显示为泰语。

## **1.3 适用版本**

星辰3.0以上所有版本

>
注意：只有购买了多语言包后，才能在打印时应用，否则设置后只能在打印模板中预览效果。

这里需要说明切换系统语言与切换模板语言的差异：

| 设置路径 | 操作路径 | 应用限制 |
| --- | --- | --- |
| 切换系统语言 （需要购买多语言包） |  | 购买了多语言包后，切换到其他语言后，即可以用于单据或商品标签打印 |
| 切换打印模板语言 （无需要购买多语言包） |  | 未购买多语言包的前提下，只能在打印模板里预览查看效果，不能应用于单据或商品标签打印 |

## **1.4 系统路径**

系统设置<打印模板列表<任意一个打印模板<辅助设置

# **2 功能介绍**

## **2.1 语言类型**

语言：自动同步系统语言，星辰目前支持简体中文、English、繁體中文、泰语、越南语、印尼语（如打印需要使用对应语言的模板，需购买多语言包）。

操作：在不同的语言环境下，支持新建、修改模板

## **2.2 打印语言说明**

当打印模板中切换语言时，支持以下设置：

- 工具栏的字体、字号、颜色、加粗等通用属性
- 支持控件的宽高设置
- 支持控件的样式/布局调整
- 支持同一个控件存储不同语言的内容

【同一个控件存储不同语言的内容】操作流程：

打开简体中文的打印模板→切换模板语言到“English”→维护字段的英文信息→维护完成，录入模板名称，保存模板→再次切换简体中文或者English，就会显示对应语言下的模板。

## **2.4 特别说明**

- **不支持系统自动翻译文本内容，需手动编辑好不同语言的文本。**

>
示例：当前系统语言是简体中文，那么语言切换到English后，原来模板中的文本内容是不会翻译成英语的，需用户自行翻译好后，再维护模板里。

- **不支持系统自动翻译字段变量，按数据库中实际存储值展示为准。**

>
示例：当前系统语言是简体中文，那么语言切换到English后，原来简体中文模板绑定的数据库字段不会自动翻译（例如：数量不会翻译成qty），如果需要左侧数据源也显示对应系统语言，需要退出打印模板，到头像中切换语言。详细查看本文的1.3说明。

- **切换语言时，需保存当前语言的模板内容，不保存将丢失已调整的内容。（详见下文操作说明）**

# **3 操作指南**

1）打印模板列表>选中具体的打印模板，点击模板空白处，鼠标往下滑，可以看到辅助设置

![](https://vip.kingdee.com/download/0109a3e10af6457b47d7b9b628d0fd2cfc42.png)

2）点击语言，默认是简体中文，假设我们要设置英文模板，下拉列表切换到English，页面会弹出保存当前页面的提示，建议切换语言前务必先保存当前语言的模板内容，否则切换后会丢失

![](https://vip.kingdee.com/download/0109c893f37ff2184c368e222d5585a0d4a4.png)

3）切换成功后就可以需要显示成英文的内容进行编辑了

例如我们可以把单据头的字段名都改成英文：

![](https://vip.kingdee.com/download/0109df50a8276b624415a44ef3d01f542ff0.png)

4）设置“多语言布局控制”

因为不同语言转化后，可能字段长度都会有所变化，尤其是中文转英文的时候，因此针对中文和英文的模板，我们可以在布局上做一些调整，同时希望两个语言下的布局是相互独立不影响的，那么我们就选择“不同语言隔离布局”即可。

![](https://vip.kingdee.com/download/0109ec2694a6e8974645b87f186e8a42d257.png)

然后可以把箭头所指的文本控件宽度改大些

![](https://vip.kingdee.com/download/0109a91b53ee0a4d456aa1170cf5928c908e.png)

调整后英文模板效果：

![](https://vip.kingdee.com/download/01098dda2e77687c4bdb9f901958ea185eb1.png)

中文模板效果：

![](https://vip.kingdee.com/download/0109b7022420ab42435db44ab2950929118f.png)

5）设置完成后，预览查看效果，如果没问题，点击保存即可。
