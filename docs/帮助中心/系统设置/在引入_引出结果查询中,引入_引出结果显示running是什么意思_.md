# 在引入/引出结果查询中,引入/引出结果显示running是什么意思?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：影子游戏 | 阅读：114 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/677681250869468416)

【问题描述】

在引入/引出结果查询中,引入/引出结果显示running是什么意思?

【解释说明】

在引入/引出结果查询中,显示“running”表示为此引入引出操作还在进行中尚未结束。需要耐心等待引入/引出完成。
