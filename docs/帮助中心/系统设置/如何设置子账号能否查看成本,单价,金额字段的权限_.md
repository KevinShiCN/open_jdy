# 如何设置子账号能否查看成本,单价,金额字段的权限?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题

【问题描述】

如何设置子账号能否查看成本,单价,金额字段的权限?

![](https://vip.kingdee.com/download/01092398451bbd8346edb712e2cfaa2c65aa.jpg)[视频观看](https://vip.kingdee.com/school/detail/704405780543387136?get_from=school-detail-id&productLineId=35&lang=zh-CN&cosmic_type=video)

【解释说明】

系统可以控制子账号是否可以查看成本、单价、金额字段的权限。子账号拥有单据的功能权限,默认允许查看对应单据字段的权限。如不允许子账号查看对应的成本金额字段,可以在权限设置界面选择对应的单据进行字段授权。

若是需要限制子账号查看单据中单价下拉显示的价格信息,可参考:**[如何设置不让子账号查看单据下拉列表里的价格信息?](https://vip.kingdee.com/link/s/Zc9YD)**

【解决方案】

以单据上字段为例,商品信息上价格资料查看权限设置同理操作。

第一步:管理员账号登录,在【系统设置】>【用户管理】界面,选择对应的账号授权

![](https://vip.kingdee.com/download/0109eadff9bbcb9b4a5ea91ac0ceba80aec4.png)

第二步:字段授权-批量授权可以选择授权的单据。

![](https://vip.kingdee.com/download/0109b9ae3d60b85e495081481148c1b948d3.png)
第三步:选择不想给子账号查看的单价/成本/金额字段字段,
![](https://vip.kingdee.com/download/01090ff1c6e810234624837c9d22fe97a035.png)
第四步:保存授权即可。

![](https://vip.kingdee.com/download/0109265db9a4146242409b96b8a602fa7ef8.png)
如允许子账号查看对应的成本、单价、金额字段,删除对应的字段授权即可。

![](https://vip.kingdee.com/download/010931eb32654bfe4aec8868ba5b952f88b9.png)
