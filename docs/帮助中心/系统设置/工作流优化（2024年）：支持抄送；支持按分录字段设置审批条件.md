# 工作流优化（2024年）：支持抄送；支持按分录字段设置审批条件

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：功能介绍, 新特性, 快速入门
> 作者：进销存熊诗琴 | 阅读：1711 | 有用：8
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/637296354338674432)

## 1 简介

### 1.1 应用场景

单据在审批过程中，或审批完成后，需要抄送给其他非审批人员知晓；比如“采购订单”审批通过后，需要抄送给财务人员，提醒财务准备给供应商打款。

### 1.2 功能特性

- 工作流支持启用抄送功能：支持手工抄送和自动抄送
- 单据挂起时，支持设置发送消息通知给单据提交人，让提交人及时知晓，采取行动
- 支持按单据中商品分录行上的字段设置审批流（如：商品类别）

### 1.3 适用版本

星辰3.1以上专业版/旗舰版

## 2 功能详情

### 2.1 工作流支持抄送

（1）手工抄送

在流程设计界面，选中要启用抄送功能的审批节点，开启抄送功能：

![](https://vip.kingdee.com/download/0109893ba1a6bf1e45bf89169899daf6422f.png)设置完成后，在单据审批弹窗中，即可选择本次抄送消息的消息内容与接收人；

![](https://vip-admin.kingdee.com/download/010093f5e4203c4b4c88abbb13065f915306.png)

在任务中心的待办任务和已办任务列表也可以执行抄送功能（不支持单次选择多张单据执行抄送）

![](https://vip-admin.kingdee.com/download/0100b1571dcdd64e4597a598d8e5a46aa4dd.png)

抄送完成，接收人将接收到消息通知。

（2）自动抄送

第一步，在流程设计界面，选中要启用抄送功能的审批节点，开启抄送功能后可设置自动抄送：

![](https://vip.kingdee.com/download/0100e0a1467e55624167bbb00e44edee1df4.png)

第二步，在自动抄送条件设置界面，可设置：

- 抄送时机：只能单选，包含以下时机

| 时机节点 | 含义 |
| --- | --- |
| ①审批通过时 | 即当前审批节点通过时自动通知。注意：并不是指整个审批流程完成时才通知，如果一个审批流包含多个节点，那么每个节点审批通过都会通知 |
| ②驳回时 | 即当前审批节点被驳回时自动通知 |
| ③进入审批节点时 | 即进入当前审批节点时自动通知 |
| ④离开审批节点时 | 即离开当前审批节点时自动通知，包含了前面的①②两种场景。 |

- 抄送人：可以选提交人、业务员、部门负责人，也可以点击“选择”，从用户管理中选择。
- 抄送内容：可自定义，点击文本框内的铅笔按钮即可编辑。

设置完成后，当单据满足触发自动抄送的要求时，抄送消息接收人就会收到抄送消息。

注意：

1、启用自动抄送后，仍然可以使用手动抄送。两个功能相互独立，互不影响。

2、“自动抄送设置”中的“部门负责人”指的是单据上业务员所属部门的部门负责人，部门负责人可在“职员列表”中设置；

![上传图片](https://vip.kingdee.com/download/0100d21e234be49d4f03b5fe7868a3ac5cf7.png)

3、本次优化将工作流的消息通知转移到“消息中心>单据消息”界面，详见下图：

![](https://vip-admin.kingdee.com/download/0100a9b10135cde14bd8a1a5a3edf64230e8.png)

### 2.2 工作流挂起时通知提交人

当单据由于库存不足等原因挂起之后，希望能发送消息通知给提交人，避免单据挂起后无人跟进。

设置路径如下：“流程设计>工作流设计器>全局设置>挂起时通知流程人”
