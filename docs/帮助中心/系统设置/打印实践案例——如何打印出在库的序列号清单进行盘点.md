# 打印实践案例——如何打印出在库的序列号清单进行盘点

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：精华汇总, 进阶技巧, 应用案例
> 作者：小微微 | 阅读：314 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/731546326923221760)

# **1 业务场景**

商品启用序列号管理，在盘点时需要将所有在库的序列号导出到表格，并打印出来。仓库拿着打印出来的纸张到仓库盘点，比对仓库实际在库的序列号；因序列号商品的特性（每个商品都有一个序列号），导出或者打印出来的内容，需要按照商品进行数量汇总，并且将序列号罗列展示的效果。

**期望打印出的效果：**

![上传图片](https://vip.kingdee.com/download/0100f737c50ece5c48d5b38ee6d0a4bf15c3.png)

**实际遇到的问题：**

系统中盘点单按序列号盘点时，显示商品清单无法在盘点单中显示出所有在库的序列号，将盘点单导出到EXCEL后，EXCEL中也没有序列号的信息。

![上传图片](https://vip.kingdee.com/download/01000fdc5bcc9b9e4369906df81fae3d9729.png)

# **2 解决方案**

使用序列号状态表，过滤在库的序列号进行打印。

# **3 适应版本**

金蝶云星辰3.1以上所有版本

# **4 应用流程**

**1）下载下列模板导入到系统中**

打印模板：模板只有少许字段，下载导入后可自行添加字段

序列号状态表打印在库序列号清单（勿动-原文件引入）.zip

![上传图片](https://vip.kingdee.com/download/0100c8714545e6a14604916242db0e127567.png)

**2）在序列号状态表过滤在库的序列号**

![上传图片](https://vip.kingdee.com/download/01008b19bbaaa11b4a7da0d46e55cac3ac01.png)

**3）****在打印设置中选择上一步上传的打印模板**

![上传图片](https://vip.kingdee.com/download/01004cdaff02091a4012a027c640ba493f59.png)

打印效果如图：

![](https://vip.kingdee.com/download/0100429e4ddad4824ad58133198a1be6df13.png)

**4）若需要打印时汇总序列号的个数，在商品上维护一个自定义数值字段，商品上录入“1”。这一步可以帮助在打印时汇总每个商品的序列号总数，若不需要可忽略该步。**

![](https://vip.kingdee.com/download/0100c49ef0d78c8649b6acddce1e36143f21.png)
