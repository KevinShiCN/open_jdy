# 打印模板如何插入数据表格、分录表格、布局网格,有什么区别?

【问题描述】

打印模板如何插入数据表格、分录表格、布局网格,有什么区别?

【业务场景】

打印模板中的容器控件下存在数据表格、分录表格、布局网格,这三者有啥差异?

![](https://vip.kingdee.com/download/01094fd4ca8413fb4d1f908767393e59ffc6.png)

若需要了解数据表格如何使用,请点击:**[数据表格操作步骤](https://vip.kingdee.com/knowledge/454309561630020352?specialId=454242075429607424&productLineId=35&isKnowledge=2&lang=zh-CN)**

若需要了解分录表格如何使用,请点击:[分录表格操作步骤](https://vip.kingdee.com/knowledge/454314693897787392?specialId=454242075429607424&productLineId=35&isKnowledge=2&lang=zh-CN)

若需要了解布局网格如何使用,请点击:[布局网格操作步骤](https://vip.kingdee.com/knowledge/454314693897787392?specialId=454242075429607424&productLineId=35&isKnowledge=2&lang=zh-CN)

【解释说明】

**数据表格:存在标题行、数据行、合计行、分组行的概念,在进行打印时,根据数据行中设置的字段取数进行纵向延伸打印,格式比较固定,如下图所示;**

![](https://vip.kingdee.com/download/0109548b4aaa77344e168be94153fa9b7f4e.png)
**分录表格:相对于数据表格,没有标题行的概念,设置比较灵活,行与列都是网格的普通行和普通列,根据放置的文本内容或者数据字段来显示,如下图所示:**
![](https://vip.kingdee.com/download/01091ab4fed530f94bf1a8da193c541554ff.png)
**布局网格:一般用于需要打印固定文本或者设置单据中的表头或者表尾字段数据,相对于分录表格而言,无法设置分录表格的特殊属性,如下图所示:**
