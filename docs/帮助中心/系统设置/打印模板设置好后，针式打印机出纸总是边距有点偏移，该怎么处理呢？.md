# 打印模板设置好后，针式打印机出纸总是边距有点偏移，该怎么处理呢？

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：快速入门, 精华汇总, 常见问题
> 作者：进销存赵嘉丽 | 阅读：13023 | 有用：5
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/371331959403988736)

【问题概述】

打印模板设置好后，打印出来总是边距有点偏移，该怎么处理呢？

| 异常情况 | 现象 | 图片 |
| --- | --- | --- |
| 情况1：左右位置不对 | 整体偏左、偏右最左或最右的打印有残缺（表格线或文字有缺失，打印不完整） |  |
| 情况1：上下位置不对 | 整体偏上、偏下（上边距留白太少，或者太靠下） |  |

【适用版本】

星辰2.0所有版本

【排查思路与步骤】

**1） 出现异常的时间节点前后是否有重装打印机、更换打印机等操作----先确认造成异常的根源**

如果没有，继续往一步排查。

如果重装或更换过打印机，建议直接按第3）步开始操作。

**2） 本地验证打印验证**

先找一个本地的word文档，使用相同纸张尺寸打印看看是否有偏移（如下图）：

![](https://vip.kingdee.com/download/0100d4ea1a7c54394c918b83e8c2fc6258a7.png)

验证一下本地打印出来纸张左右两边（上图红箭头处）是否有偏移或者内容缺失之类的情况。这一步是先排查是否是打印机的异常，如果正常，则继续按下一步调整，如果这一步打印已经不正常，那就是打印本身走纸异常，建议按本文最后一步操作。

**3） 检查默认打印机的【打印机首选项】**

检查默认打印机的打印机首选项里的纸张尺寸是否跟实际打印纸张一致。

如果打印不是默认打印机，先设置成默认的，其次再按一下截图箭头检查纸张。

![](https://vip.kingdee.com/download/01006e4e8fea4b864c83806d2c89d4c373cd.png)

**建议打印机首选项的纸张严格按照以下的常用纸张尺寸来选择或者设置**：

| 纸张名称 | 英制尺寸 | 国际单位尺寸 |
| --- | --- | --- |
| A4纸  | / | 21.00*29.7cm |
| 一等分 | 复写纸9.5*11英寸 | 24.13*27.94cm |
| 二等分纸（11孔） | 复写纸9.5*5.5英寸 | 24.13*13.97cm |
| 三等分纸 | 复写纸9.5*3 2/3英寸 | 24.13*9.31cm |

**4） 检查星辰打印模板里的纸张尺寸**

检查星辰打印模板里的纸张尺寸跟打印机首选项的纸张是否一致，**务必保持跟首选项里的纸张尺寸一致**！

同时，打印模板先不用设边距（**上下左右都设0**即可），数据表格**距离纸张左边稍微留白2-3mm**即可

![](https://vip.kingdee.com/download/01000dabfe1b153f4611ae4be384ee78ac17.png)

**5） 调试与验证**

• **针对情况1：**

如果是上下位置不对的（标题无法完整打印的）：

![](https://vip.kingdee.com/download/010933439766e0494432bd54e9edfb22ef3e.png)

在检查完上面的设置后还是不对的，可以通过手动调整机器的出纸和退纸：

**爱普生针式打印机**：

[https://www.epson.com.cn/Apps/tech_support/search.aspx?key=%E4%BD%8D%E7%BD%AE](https://vip.kingdee.com/tolink?target=https%3A%2F%2Fwww.epson.com.cn%2FApps%2Ftech_support%2Fsearch.aspx%3Fkey%3D%25E4%25BD%258D%25E7%25BD%25AE)

**映美针式打印机**：

进入微调整模式（建议咨询映美的售后），使用‘换行/换页’和‘进纸/退纸’键装纸调整纸张头出量。

如果是其他打印机，建议您直接咨询打印机售后。

---

• **针对情况2：**

如果是左右位置不对的（如下图）：

![](https://vip.kingdee.com/download/01002f1d8584d11a41d880642f174a53271c.jpeg)

在检查完上面的设置后，先打印验证看看位置的偏差，如果整体偏右，左边留白较多，需要调整针式打印机的挡板

![](https://vip.kingdee.com/download/0100eec410661c314c2a9545ec0e3d69bb99.png)

留出合适的位置，可以试试先预留15mm，然后逐步调试到合适的位置。

![](https://vip.kingdee.com/download/0100480324af9b98432087fd6f6770683494.png)

需要结合实际打印情况微调打印模板的左边距或者模板内容，使其尽量能完整打印，如果左右位置调整已经合适，但是表格右侧总是有短缺的（如下图），

![](https://vip.kingdee.com/download/01008d1ed0cfcbf64341a67cfde1ad4c0138.jpeg)

建议将打印模板中数据表格的**列宽**微调缩小一点（参考下图），使其能完整打印。

![](https://vip.kingdee.com/download/0100cd4ef20bb7934aebade580654ad02e52.png)

调整后的效果：（完整打印出两边的边距了）

![](https://vip.kingdee.com/download/01005e014fe4e26f45c7a2cb39c7320a81ff.png)

**6） 恢复打印机出厂打印设置**

如果按以上步骤操作后还是异常或者打印机本身走纸有异常（也就是位置不对），建议您先恢复打印机出厂打印设置，再按上述流程重新调试。

| 针式打印机品牌 | 恢复出厂设置操作 | 参考图片 |
| --- | --- | --- |
| 爱普生 Epson | 关闭打印机电源，同时按下：换行/换页+进纸/退纸 +暂停(如右图红箭头)+开机（注：按住三秒钟后松手，听到滴声后说明恢复出厂设置成功）取色带： | 电源键：换行/换页+进纸/退纸 +暂停 键： |
| 映美jolimark | 关闭打印机电源，同时按住【进/退纸】键和【换行/页】键，再开启打印机即可。 | 不同型号打印机的按钮位置可能不一样，请根据您的打印机找到对应的操作按钮即可。 |
| 其他品牌 | 请直接咨询品牌商售后获得更多帮助。 |  |
