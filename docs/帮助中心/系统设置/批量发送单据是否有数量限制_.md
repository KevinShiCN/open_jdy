# 批量发送单据是否有数量限制?

【问题描述】

批量发送单据是否有数量限制?

![](https://vip.kingdee.com/download/01099923f9aa035f41d296e763b2fbcfef8c.png)

【解释说明】

企业互联批量发送单据有数量限制。批量发送单据时,限制一次最多发送100张单据。
