# 如何授权子账号添加物流信息?

【解决方案】

如何授权子账号添加物流信息

【解决方案】

物流信息主要包含物流公司和交货方式。
授权在系统设置-用户管理-功能授权,授权物流公司和辅助资料

![](https://vip.kingdee.com/download/01097a239cd3eefd4479a7302878f71d0733.png)
录单时就可以选择物流信息

![](https://vip.kingdee.com/download/01099772e9d7a4554501a1108473fdde0a4e.png)
温馨提示:

如果单据表尾没有交货方式字段,表头单据列设置开启显示交货方式。
