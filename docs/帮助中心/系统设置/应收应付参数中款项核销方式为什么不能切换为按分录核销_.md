# 应收应付参数中款项核销方式为什么不能切换为按分录核销?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：601 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/471844307117122560)

【问题描述】:

应收应付参数中款项核销方式为什么不能切换为按分录核销?

如图:

![](https://vip.kingdee.com/download/0109c886ec391b834854abbda5d8d05e719c.png)

【解释说明】

```
当【进销存参数】中的客户承担费用设置为“计入应收账款”时,此参数不可用。
```

【解决方案】

```
在【系统参数】-【进销存参数】-客户承担费用设置为【计入其他应收款】即可

```
