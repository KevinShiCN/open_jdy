# 功能授权里的新增、修改权是什么意思,该怎么设?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：进销存赵嘉丽 | 阅读：631 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/432344023089431296)

【问题描述】

功能授权里的查询、新增、修改是怎么用?应该怎么设置?

![](https://vip.kingdee.com/download/0109b3729530c2c04f61990f47fecb66ba14.png)

【功能说明】

| 权限名称 | 含义及影响 | 适用场景 | 备注 |
| --- | --- | --- | --- |
| 查询 | 仅限于查看,不能做任何编辑的操作 | 有严格的岗位权限隔离要求,例如:销售只能看采购订单,不能建单;又或者仓管只能查看销售订单,不能开单等等。 | 查询权是最基本的授权,如果不想让子账号看到对应单据或者基础资料的任何内容,那么对应的功能授权不要勾选即可。 |
| 新增 | 1)新建单据或者基础资料的权限;2)新建后保存;3)下推关联单据的权限(例如下推采购、出库、入库、换货、退货等) | 1)业务员创建客户资料或者开单2)仓管员对采购订单入库,或对销售订单进行出库等等。 | 新增是包含了首次“保存”的权限的,如果已经点过保存再次保存,那么第二次保存取决于是否有“修改”权限。 |
| 修改 | 控制修改后的保存、打印、保存并新增、删除、费用弹窗的修改等编辑单据的操作 | 单据保存后,允许修改单据内容的场景 | 1)如果不给修改权,那么只能保存,但一旦保存后不允许修改、删除;2)修改权还要判断其他功能按钮权限,例如功能授权里单独勾选了“删除”,没有勾选“修改”,那么单据或者基础资料里点击“删除”是允许的,这里就不会再判断修改权里是否有删除权,会做过滤,其他按钮也同理。 |

【应用示例】

下面以销售出库单为例,演示一下查询、新增、修改权是怎么使用的:

| 权限设置 | 操作 | 实际效果 |
| --- | --- | --- |
| 只有查询权 | 只能查看,不能做任何编辑(保存、新增、删除、审核等等) |  |
| 有查询,新增,没有修改 | 可以新增并且保存,但是保存后不能再做任何编辑的操作 |  |
| 有查询、新增、修改 | 可以新增并且保存,且保存后还能做编辑操作,至于具体修改权包含哪些权限,还要看其他操作(例如审核、反审核、删除、打印等)是否给了权限。 |  |

【关联问题】

为什么子账号有新增权,点击销售费用、客户承担费用,不能编辑?

![](https://vip.kingdee.com/download/010925e5044d96cf4c569e033b5e365f6393.png)

答:已经保存过的单据,如果子账号没有“修改”权限,那么即使有新增权,也是无法编辑的,费用弹窗里的编辑权限取决于子账号功能授权里的“修改”,只要勾上(下图红箭头)并保存权限:

![](https://vip.kingdee.com/download/01090407367c51ad49abb6db76a807c11423.png)

那么重新打开单据后,子账号就可以编辑了:

![](https://vip.kingdee.com/download/010940f905520d7b4740ad8c9668b0a349ae.png)
