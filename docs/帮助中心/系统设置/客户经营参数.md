# 客户经营参数

# **1 简介**

## **1.1 应用场景**

可以对系统库存核算和业务期末结账处理规则和处理惯例进行统一的设置和管理。存货核算参数一般由财可以对客户经营管理的业务处理规则和处理惯例进行统一的设置和管理。

## **1.2 操作路径**

系统设置>系统参数>客户经营参数

## **1.3 适用版本**

星辰3.1以上所有版本

# 2 参数含义及应用场景

## **2.1 客户参数**

1）查重规则设置：

可设定潜客/正式客户查重的字段范围：含客户名称、联系人姓名、手机、座机、微信。可设定规则范围及控制强度；

2）潜客公海规则设置：

自动划入公海：一定时间内，销售员未进行跟进，可自动划入公海；可开启保护期，划入公海后，一定时间内无法再次领取；

3）潜客数量限制：

限制每个员工拥有的潜客数，当员工拥有潜客数超过规定上限时，新增潜客无法成功。如果在设置最大潜客数量前，员工所拥有潜客数已超上限，则原有潜客不受影响，但是该员工无法再继续添加新潜客；

## **2.2 外勤参数**

1）打卡设置

- 允许打卡范围：当前定位距离客户地址定位的直线距离，可选100米、500米、1000米、3000米和不限等，点击右侧向下箭头选择。
- 定位异常时支持拍照打卡。

2）图片采集设置

- 可选相机拍照或者相机拍照和相册上传均支持。如果勾选相册内同步存储相机所拍照片，相机所拍照片会同步存储到手机相册中。
- 可选择是否开启图片水印，如果勾选开启水印，可再选择是否展示：

• 客户名称（PC端基础资料-业务资料-客户）

• 拍照人姓名（PC端系统设置-用户管理-姓名）

• 拍照时间（例：2021.01.01  15:00:00）

• 打卡地址(可选择精确显示省、市、区和详细地址)
