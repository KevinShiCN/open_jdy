# 单据新增字段支持使用if函数设置公式吗?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：lykk | 阅读：254 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/644703753806556928)

【问题描述】

单据新增字段支持使用IF函数设置公式吗?

【解决方案】

目前单据新增字段不支持使用IF函数设置公式,只能用现有的字段进行“+”、“-”、“X”、“/”的运算;
