# 使用针式打印机打印的时候，打印模板、打印预览、打印机首选项的纸张尺寸都是一致的，但是打印出来纸张，内容总是偏移（偏上/偏下、偏左/偏右），该怎么处理？

问题描述：

![](https://vip.kingdee.com/download/010907ffb9a3d4944b83a25d4463be242a94.png)

解决方案：

当出现上述问题时，首先要检查打印模板的设置，其次是检查打印预览，最后则是检查打印机设置。

**
**

**第一步**：检查打印模板的纸张尺寸、方向、边距是否都与电脑所链接的**打印机首选项**的的纸张尺寸、方向、边距保持一致，另外，当打印的内容比较多，布局比较密集时，建议边距都设成10mm，也就是边距建议留白以获得更好的打印效果；

**第二步**：通过星辰的打印预览（PDF、插件都一样，如果条码打印，建议使用PDF打印）检查模板设置的位置是否合理、打印的区域内容是否完整；

如果上述两步都正常，但是打印出来，位置总是有偏移，如上方/下方留白太多，或者左右的位置总是跟打印预览的对不上，**通常是打印机的走纸异常有关**。

打印机位置偏移的问题，建议对打印机设备进行**恢复出厂设置**，因不同品牌的打印机恢复出厂设置的操作是不一样的，星辰无法提供统一的操作指引，具体的操作建议您直接联系打印厂家官方售后支持获取帮助。

例如：爱普生针式打印机恢复出厂设置一般是：**进退纸 + 换行 / 换页 + 暂停 + 电源键（注：按住三秒钟后松手，听到滴声后说明恢复出厂设置成功）**

**恢复出厂设置后再重新使用星辰打印进行验证。**

**如果恢复出厂设置后，打印还有偏移，那么有可能是打印机设备的其他故障，建议直接咨询您所使用的打印机官方售后支持。**

补充说明：

一般如果刚换系统，例如之前是用其他软件打印，后面换了星辰，就会比较容易出现这种走纸异常的情况，因为旧软件跟打印机的参数都是调整好的，那么切换软件后，有可能原来软件的打印设置跟现在星辰的打印设置是有差异的，因此建议恢复打印机出厂设置后，重现再进行打印设置及验证。
