# **1 简介**

## **1.1 业务场景**

针对打印模板中的控件，需要设置个性化的格式或者样式，例如设置公式、设置换行规则等等，以满足不同的打印业务场景。

## **1.2 适用版本**

星辰3.1以上所有版本

# **2 功能介绍**

## **2.1 公式功能**

**业务场景**

当一个控件绑定一个字段值不满足场景时，需要组合多个字段组合或通过函数获取数据源，或者需要对数值型字段做一些运算处理时，可以通过公式来解决。也可以解决以下场景：

- 场景一：星辰的销售退货单不支持录入负数，但是可以通过打印模板的公式来满足单据打印的负数显示问题
- 场景二：需要把多个字段拼接成一个字段打印，例如地址、或者批次保质期等信息
- 场景三：需要手工录入一部分文本，再拼接系统的字段，输出一个完整的字段
- 场景四：需要对打印单据的数据做一些加工处理，例如用if函数做判断、取整、截取等等

**功能介绍**

公式编辑器是一个灵活强大的工具，它允许用户在标准场景无法满足时，自定义组合文本、字段和进行函数运算。文本、二维码、数据表格、分录表格、布局网格等控件均支持设置公式，控件可绑定**单个字段**和**公式**，公式在此处的用法为：**不同字段组合**、**文字拼接**、**函数**，最终按照公式表达式生成控件的值。

详见知识**：**[打印公式功能说明](https://vip.kingdee.com/link/s/ZmMS2)

## **2.2 文本换行规则**

**功能介绍**

适用于文字换行的多种使用场景，例如

- 采购订单中金额字段，在打印时希望金额不换行，一行显示出来；
- 包含了英文合同模板中，我们希望英文单词在换行时在一行不够打时，整个单词换行至下一行；

详细功能介绍：[文字换行规则介绍](https://vip.kingdee.com/knowledge/444598144001700352)

## **2.3 且内容超长时**

**功能介绍**

当文本内容不固定，如采购订单的备注字段，其内容长度不确定，为避免影响其他控件的展示，若文本内容超过控件大小，可采用以下处理方式：截断、行高自适应或自动缩小。

**功能特性**

- 使用范围：文本控件及数据表格的单元格均支持
- 规则及应用场景

| 规则 | 应用场景 | 效果预览 |
| --- | --- | --- |
| 截断 | 当文本内容超出文本宽度时，宽高空间不够时截断展示，不改变宽高。 |  |
| 行高自适应 | 当文本内容超出文本控件宽度时，按照内容自适应换行，且行高自适应。不会影响或覆盖其他控件。 |  |
| 自动缩小 | 当文本内容超出文本控件宽度时，宽高空间不够时缩小内容展示，不改变宽高。注意：1）当数据表格开启了“行高自适应”时，数据表格单元格的“内容超长时-自动缩小”会失效，数据行会优先行高自适应，“自动缩小”不再生效；2）缩小比例与内容长度有关，内容越长，缩小比例越大。 |  |
| 溢出 | 当文本内容超出文本控件宽度时，且“文本换行规则”选择不换行时，将内容展示到控件外侧，不改变行高。 |  |

- 互斥说明

控件属性中的“文本换行规则”与“且内容超长时”设置存在逻辑互斥时，互斥的选项会置灰显示，表示不可选择。减少用户歧义，改善用户体验。

| 文本换行规则 | “且内容超长时”不可选 | 置灰效果（表示有冲突，不允许设置） |
| --- | --- | --- |
| 不换行 | 行高自适应 |  |
| 超出控件宽度自动换行按文字类型换行 | 溢出 |  |

## **2.5 不打印**

**业务场景**

通过在打印模板中放入图片作为模板背景，作为辅助参考线，或者预留位置，但是实际不打印图片出来达到效果。

一般适用于如财务的凭证模板设计、或者批发商品行业的价格标签或者一般有自己的表单样式的打印场景，要求打印时严格按照现有的纸张或者标签样式和位置来打印。
