# 公共参数

# **1 简介**

## **1.1 操作路径**

系统设置>系统参数>公共参数

## **1.2 适用版本**

星辰3.1以上所有版本

# 2 参数含义及应用场景

## 2.1 系统信息

系统起始期间和本位币信息，即建账时录入的启用期间和本位币信息。

>
注意，这些信息建账后不能修改，要修改只能重新初始化。

• 开始日期、结束日期：用于显示系统起始期间的开始日期和结束日期。如2020年1期的开始日期为2020年1月1日，结束日期为2020年1月31日；

## 2.2 企业信息

显示建账时录入的公司信息，可以此界面维护公司的地址、电话、传真。

>
注意：当前账套的的名称也是同步这里的“公司名称”，只有服务管理员才能修改。

## 2.3 开票信息

可以设置本账套的开票信息，在进行税票登记时，可以从这里获取信息，快速录入，不用再手动填写。

## 2.4 敏感信息设置

1）界面中脱敏显示：即支持手机号、身份证、银行卡的脱敏显示及加密存储。

示例：例如手机号123456789，开启脱敏显示后，系统内显示124****89

2）后台加密存储：修改加密存储参数，系统将进行敏感字段（手机号、身份证、银行卡）的数据迁移。

>
注意：1、开启此参数后，相关字段不支持模糊搜索。2、修改此参数，系统将自动进行敏感字段数据迁移，正在迁移的相关单据暂不能操作。

## 2.5 其他信息

• 可以自定义设置单个资料/单据/凭证的附件个数

• 可以自定义设置单个附件的大小，上限是50M。图片附件最大支持20M。

• 可以自定义设置搜索分词符：支持空格分词或中英文逗号分词

说明：分词符用于分隔快速搜索框里的文字：使用空格作为分词符时，搜索：“mate 30”，会搜索出所有含“mate”和“30”的项目； 使用逗号作为分词符时，空格会作为字符参与搜索，上述搜索只会匹配含“mate 30”的项目。 本设置只对弹出框和列表的快速搜索框有效。
