# 可以打印预览，但是点击打印后不出纸的原因及解决方法。
打印预览时是对的，但是打印出来时缩放或者方向跟预览不对是为什么？

> 作者：进销存赵嘉丽 | 阅读：8126 | 有用：6

# 【问题概述】

- PDF打印，可以正常预览，点击打印，打印机不出纸会是什么原因？
- 插件打印，可以正常预览，点击打印，显示“已提交全部打印任务”后，打印机不出纸会是什么原因？

![上传图片](https://vip.kingdee.com/download/0100544604f2ee88449c8ac7059f7bb303da.png)

- 打印预览时是对的，但是打印出来时缩放的尺寸或者方向跟预览不对是为什么？

# 【原因说明】

打印提示“已提交全部打印任务”，表明打印任务已经生成成功了，而这个时候打印机依然没出纸，通常都是由于默认打印机的打印首选项中的纸张尺寸与打印模板的尺寸不一致导致的。而打印预览时是对的，但是打印出来时缩放的尺寸或者方向跟预览不对也通常是因为同样的理由。

# 【解决方案】

重新调整打印模板的尺寸或者是打印机首选项的纸张尺寸，保证二者尺寸一致即可。整体解决思路概述如下：

![](https://vip.kingdee.com/download/010066d11b44313f408fa7b881a600383eb0.png)

**下文将以****Windows10系统****为例讲解如何设置。**其他系统设置见下方说明，本文不做论述。

• Windows7系统：请参考知识：[套打打印基础-打印机及纸张规格大小设置](https://vip.kingdee.com/link/s/MUWJE)

• Windows XP系统：星辰不建议使用XP系统。

## **Windows****10系统****操作步骤**

1、 找到电脑的【打印机和扫描仪】并打开，找到电脑的【默认】打印机

![](https://vip.kingdee.com/download/010003367b36ebea49548cc3736e68b1b4a1.gif)

注：如果没有看到【默认】二字，检查一下【让Windows管理默认打印机】，去掉勾选即可。

![](https://vip.kingdee.com/download/01008c73132fd10543b5a35c5816c42ac4b3.png)

如果需要切换默认打印机，则找到设备后，点击【管理】，设置默认值即可：

![](https://vip.kingdee.com/download/01007ecc124f48584872bc3c34fd9b761be1.gif)

2、 验证电脑默认打印机【打印测试页】，先确保打印机可以正常运作！点击打印测试页后，打印机可以正常出纸即为正常。

如打印测试页无法出纸，请先检查打印机驱动的连接，或者删除打印机后重新添加驱动后再测试，如果重新添加驱动后依然不出纸，此问题属于设备问题，请联系打印机厂家处理。

![](https://vip.kingdee.com/download/01006a2b2cbf1e644b3b93ddc842efdbb7ba.png)

3、 检查默认打印机的【打印首选项】内“纸张摘要”的纸张尺寸是否与星辰打印模板中的尺寸一致：

**【标准纸张】**

假设纸张是标准A4纸，则打印机首选项里显示应该是A4纸，如下图：

![](https://vip.kingdee.com/download/0100db29745490614aca85032562f2f2a3a4.png)

星辰打印模板的纸张大小也应该是A4纸：

![上传图片](https://vip.kingdee.com/download/0100fad03e50cbf645d987117eb41f9b514d.png)

**【自定义纸张】**

如果是自定义的纸张尺寸，假设用的是复写二等分多联纸，首先打印模板中的“样式”尺寸要选择“自定义”，宽度和高度分别设置240mm，140mm，具体设置如下图：

![上传图片](https://vip.kingdee.com/download/0100585986a6db33464d8ca6904908890efe.png)

同时在计算机的【打印机和扫描仪】>【打印服务器属性】里，勾选“创建新的自定义纸张规格”，输入好尺寸、纸张规格名称，单击“保存规格”、“应用”即可新增成功。

![](https://vip.kingdee.com/download/010054b28df23ada4ac3a20776d845960f4e.png)

最后就是到【打印首选项】里找到刚刚新增的纸张尺寸，单击确定-应用-确定即可（下图序号为操作顺序）

![](https://vip.kingdee.com/download/0100581e18ec7d3946539f3e22e864f09b78.png)

4、打印预览中，选择跟模板相匹配的纸张：

无论是PDF打印还是插件打印，打印前，务必检查预览页面，确保目标打印机对应的纸张尺寸跟星辰打印模板尺寸保持一致。

- **PDF打印预览**

![上传图片](https://vip.kingdee.com/download/0100facbe481d2964ffa9e6aa08875da7445.png)

- **插件打印预览**

![上传图片](https://vip.kingdee.com/download/0100c7e195af85b84dddbdfc087bca5fcaa8.png)

都设置好后，就可以正常进行打印了。

# 【适用版本】

星辰3.1以上所有版本
