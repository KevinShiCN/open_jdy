# 出入库单据引入模板中单位.名称 # unit.name指的是什么?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：lykk | 阅读：89 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/647965164871889408)

【问题描述】

出入库单据引入模板中单位.名称 # unit.name指的是什么?

【解释说明】

unit是指单位,name是指名称,number是指编码,若引入引出模版中该列显示为# unit.name,则需要填写的为单位名称,若显示为# unit.number则需要填写对应的编码;如下图

若商品的计量单位为“公斤”,引入模板中# unit.name则填写“公斤”

![](https://vip.kingdee.com/download/0109ffbca00ab5224030b85bec5dff339841.png)

注:因单位这个字段比较特殊,名称与编码是一致的,一般在导入是都是填写# unit.name单位名称

![](https://vip.kingdee.com/download/01096b99fba233c740d7a8472e6e37ca196c.png)

如商品名称、客户名称等,这些资料区分了名称与编码,在录入模板时,若开启的是名称则需要填写对应的名称,若是编码则需要填写对应的编码,不能填错,否则会无法识别导入;

![](https://vip.kingdee.com/download/01096dddc8e440774471986eb4592684b1d7.png)

![](https://vip.kingdee.com/download/0109edff3a456bad443d954230247fe9cfc7.png)

若需要调整引入属性是名称或者是编码,可以通过引入引出模版中自行进行选择

![](https://vip.kingdee.com/download/0109eb3d50c432774011a82892ccaf526510.png)

注:若存在相同名称的信息,建议通过编码进行引入
