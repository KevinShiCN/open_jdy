### 2.2支持审批的条件---单据体字段

| 本次新增的字段 | 支持的单据 |
| --- | --- |
| 商品类别 | 采购申请单，采购订单，采购入库单，采购退货单，采购发票销售报价单，销售订单，发货通知单，销售出库单，销售发票，销售退货单，销售退货申请单，销售换货申请单委托代销单，委托代销结算单，委托代销退货单，委托结算退货单其他出入库单，移仓单，调拨出入库单，盘点单，分量盘点单，盘盈盘亏单，组装单，拆卸单 |
| 开启赠品 | 采购订单，采购入库单，采购退货单，采购发票销售报价单，销售订单，发货通知单，销售出库单，销售发票，销售退货单委托代销单，委托代销结算单，委托代销退货单，委托结算退货单 |
| 仓库（表体） | 采购申请单，采购订单，采购入库单，采购退货单，采购发票销售报价单，销售订单，发货通知单，销售出库单，销售发票，销售退货单，销售退货申请单，销售换货申请单委托代销单，委托代销结算单，委托代销退货单，委托结算退货单其他出入库单，移仓单，调拨出入库单，盘点单，分量盘点单，盘盈盘亏单，组装单，拆卸单 |
| 商品类型（如服务商品，普通商品，套装商品） | 销售报价单，销售订单，发货通知单，销售出库单，销售发票，销售退货单，销售退货申请单，销售换货申请单委托代销单，委托代销结算单，委托代销退货单，委托结算退货单其他出库单 |
| 源单类型 | 采购订单，采购入库单，采购退货单，采购发票销售订单，发货通知单，销售出库单，销售发票，销售退货单，销售退货申请单，销售换货申请单委托代销单，委托代销结算单，委托代销退货单，委托结算退货单 |
| 折扣率（%） | 采购订单，采购入库单，采购退货单，采购发票销售报价单，销售订单，发货通知单，销售出库单，销售发票，销售退货单委托代销单，委托代销结算单，委托代销退货单，委托结算退货单 |
| 含税折扣额 |  |
| 总金额本位币 | 采购订单，销售订单，发货通知单 |
| 本次应付账款本位币 | 采购入库单，采购发票单 |
| 本次应收帐款本位币 | 销售出库单，销售发票单 |
| 本次应退帐款本位币 | 销售退货单，销售退货申请单，采购退货单 |

特别说明：商品类型、表体仓库等字段均为商品分录行的字段，同一张单据多个分录行的值可能不一样，那么工作流的匹配逻辑建议查看相关知识：[工作流优化（2024年）](https://vip.kingdee.com/link/s/l87GX)
