# 多语言布局控制（打印模板支持同时维护不同语言布局效果）

# **1 简介**

## **1.1 功能介绍**

对于购买了多语言包的客户，在打印时，可以在模板中同时维护不同语言的布局效果，本文将介绍如何在不同语言环境下增删改模板，满足不同语言下模板统一布局和不同语言下模板隔离布局两种场景。

>
注意：只有购买了多语言包后，才能在打印时应用，否则设置后只能在打印模板中预览效果。

这里需要说明切换系统语言与切换模板语言的差异：

| 设置路径 | 操作路径 | 应用限制 |
| --- | --- | --- |
| 切换系统语言 （需要购买多语言包） |  | 购买了多语言，切换其他语言后，可以用于单据或者商品标签打印 |
| 切换打印模板语言 （无需要购买多语言包） |  | 在没购买多语言包的前提下，只能在打印模板里预览查看效果，不能应用于打印或者商品标签的打印 |

## **1.2 适用版本**

购买了星辰多语言包的星辰3.0以上所有版本

## **1.3 系统路径**

系统设置<打印模板列表<具体的打印模板

# **2 **功能介绍

## 2.1 特性说明

- 打印模板多语言环境下，控件的文本内容和样式在不同语言环境下是隔离的，而控件大小、位置可以设置为隔离或统一的；
- 不同语言隔离布局：不同语言下，拖动控件位置，不影响其他语言下控件位置；
- 不同语言统一布局：不同语言下，拖动控件位置，其他语言下控件位置也会同步改变。

## 2.2 操作说明

**• 不同语言隔离布局**

设置该参数后，那么在中文环境下修改布局，在英文环境下布局与中文环境不同，不同的语言隔离布局，不会相互影响。

![上传图片](https://vip.kingdee.com/download/0100aaca9c974a124d65acbb164246a13acb.gif)

**• 不同语言统一布局**

设置该参数后，如在中文环境下调整布局后，在英文环境下与中文环境下一致，布局是统一控制的，不做区分。

![上传图片](https://vip.kingdee.com/download/01005706e8cdcf99498eac557b9d355d7220.gif)

# 3 相关问题

**1）如何打印国外的字体？**

答：目前系统预置了简体中文、繁体中文和英文的模版，如果需要打印其他小语种比如泰语、印尼语需要自行上传相关语种的字体。参考知识：[打印字体管理功能](https://vip.kingdee.com/link/s/ZtMgI)

同时，未购买多语言包的账套也支持打印小语种，具体可以参考知识：** **[如何打印韩语、柬埔寨语、越南语等小语种?](https://vip.kingdee.com/link/s/luCwz)

**2）打印的模版是三个语言通用的吗？**

答：不是。必须切换到英文才能打印英文的模版，打印模版是独立的。
