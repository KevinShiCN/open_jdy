# 税务参数

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：快速入门, 功能介绍
> 作者：ylim | 阅读：3370 | 有用：14
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/627211982722237184)

税务开票、智能取票前，需先在税务参数中维护开票信息、申请开通配置开票参数、进行税务企业认证；税务参数设置完成才能在星辰系统中使用开票和取票服务。

| 步骤 | 具体说明 | 学习链接 |
| --- | --- | --- |
| 1.开票参数设置 | 设置税务参数，实现在星辰系统中通过单据开票或直接手工开票的功能 | 点击学习 |
| 2.智能取票参数设置 | 设置税务参数，实现直接链接税局系统自动取票到星辰账套中，以便给业务单据匹配发票信息或者是通过发票生成凭证。 | 点击学习 |
| 3.税务资料 |  | —— |
| 3.1开票货物及服务 | 用于开具增值税发票的开票内容,来源于以下四种方式:(1)手工录入  (2)新增发票时自动保存  (3)进销存单据销售开票时自动保存  (4)导入发票时自动保存  (5)用EXCEL文件导入 | 点击学习 |
| 3.2票据类型 | 票据类型是用来维护除“增值税发票”以外的票据类型，比如:火车票、出租车票、定额发票等。 | 点击学习 |
