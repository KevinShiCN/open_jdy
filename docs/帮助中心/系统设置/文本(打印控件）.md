# 文本(打印控件）

## 1 通用介绍

### 1.1简介

文本控件在打印中非常常见，使用频率较多。文本可输入具体的文本内容、绑定字段、大文本输入等，在做订单模板、合同模板作用较多。

### 1.2 适用版本

金蝶云星辰 V3.1以上

### 1.3 功能介绍

### 1.3.1 文本支持工具栏功能

文本控件拖入设计器画布后，支持字体、字号、颜色、背景色、加粗、斜体、下划线、文本对齐、线框设置。

### 1.3.2 文本控件的数据来源

• 用户自定义输入文本

![](https://vip.kingdee.com/download/0100702c7531e948498290784891e1da970f.GIF)

• 用户可以选择绑定字段

![](https://vip.kingdee.com/download/0100cffeee8b2b2d44f782284fd8cb8b7159.GIF)

### 1.3.3 文本的格式类型

文本控件的格式类型为5种：常规、数值、金额、日期、时间。

![](https://vip.kingdee.com/download/0100ffa497f085c74f8785547cf7b0ac0862.png)

### 2.2不同文本格式的特殊属性

除了上述提到的通用属性，不同的文本格式也有一些特殊的格式可供设置

### 2.2.1文本格式：数值

![](https://vip.kingdee.com/download/010055423e2653a74419abb96a9dc95cd69b.png)

1、小数位数：支持显示原值，或者按客户自定义的精度显示，最多可支持9位小数位。

2、负数类型：-1234.56/(1234.56)/原值，默认为-1234.56。若数值为负数，默认保持负数

3、支持负数显示红色

4、零值时显示为空：勾选后，0值显示为空。不勾选则展示0。

5、不显示尾零：小数末尾尾零不显示，如12.3400 ——> 12.34

6、千位分隔符（,）：勾选则显示，如 1234——>1,234

7、显示百分比 ：勾选则显示，如 12——>12%

### 2.2.2文本格式：金额

![](https://vip.kingdee.com/download/0100f3afd52588674a5985440f991d83bd57.png)

1、小数位数：支持显示原值，或者按客户自定义的精度显示，最多可支持9位小数位数。

2、币别类型：

| 币别类型 | 应用场景 | 币别符号 | 币别代码 | 打印模板设置 |
| --- | --- | --- | --- | --- |
| 不使用 | 金额字段不需要打印币别 | 无 | 无 | 当币别类型为“不使用”时，币别符号、币别代码会隐藏： |
| 人民币/港币/美元 | 金额字段需要打印币别 | 可勾选 | 可勾选 | 预览效果：显示币别符号：显示币别代码： |
| 从单据中获取 | 报表打印需要取单据上的币别 | 可勾选 | 可勾选 | 详细下文“1）从单据中获取币别符号”讲解说明 |

1）从单据中获取币别符号

应用场景：打印明细类的报表时，如果存在多种不同币别的单据，那么可以设置根据单据的币别来显示。

应用操作：

假设打印的是销售订单跟踪表，打印模板的币别类型选择“从单据中获取币别符号”，币别符号也勾选上：

![](https://vip.kingdee.com/download/01002b977201c541405daae0a054ef6a1ab2.png)

打印预览效果：

![](https://vip.kingdee.com/download/010000030b53bbae445093cb2c1b3e164e87.png)

3、 金额大写（将金额小写转为大写）

注意：金额大写、币别类型结合使用会有不同的打印预览效果

假设是100元（小数位为0），效果如下：

| 人民币 | 中文大写 | 开启 |  | 金额前会显示“人民币” |
| 不开启 |  | 金额前不显示“人民币 |  |  |
| 美元 | 英文大写 | 开启 |  | 金额前会显示币别英文单词DOLLARS |

### 2.2.3文本格式：日期

![](https://vip.kingdee.com/download/0100e9f04f3f5eb5456281164c5a9e7f79f4.png)

1、 类型：

• 不格式化，与单据设置日期格式保持一致。

• 当选择固定日期格式，不跟随系统格式，打印按所选格式展示。支持24种日期格式。

---

### 2.2.4文本格式：时间

![](https://vip.kingdee.com/download/0100560e803bda064ac3ad3bd745e05b60d8.png)

1、 类型：

• 不格式化，与单据设置日期格式保持一致。

• 当选择固定日期格式，不跟随系统格式，打印按所选格式展示。支持24种日期格式。
