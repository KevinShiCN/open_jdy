# 账套重新初始化后,还会保留之前的操作日志嘛?

【问题描述】

账套重新初始化后,还会保留之前的操作日志吗?

【解释说明】

账套重新初始化并不会清掉操作日志,以前的操作日志还是可以查询到。

如下图的操作日志,

![](https://vip.kingdee.com/download/0109cb2ec5e122e549e38546499241542c14.png)

操作重新初始化之后,现在查询操作日志,还是有之前的操作日志信息,并且还可以查询到刚刚重新初始化的这个操作的记录。
