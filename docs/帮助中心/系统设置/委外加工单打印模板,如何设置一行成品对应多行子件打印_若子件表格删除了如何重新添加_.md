# 委外加工单打印模板,如何设置一行成品对应多行子件打印?若子件表格删除了如何重新添加?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：cs123456 | 阅读：222 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/525115308244108800)

【问题描述】

委外加工单打印模板,如何设置一行成品对应多行子件打印?若子件表格删除了如何重新添加?

【解释说明】

1、系统预置的模板就是可以一个成品对应多行子件打印的,如果需要修改,可以点击复制模板,[复制预置模板](https://vip.kingdee.com/link/s/ZUfOM)进行打印,

预置模板预览示例

![](https://vip.kingdee.com/download/010947ca38a853324d92a5e625a83a764e59.png)

![](https://vip.kingdee.com/download/01097e8b426ae11b41589f5069dcc26e7c27.png)

2、若修改调整过程中,如果误删除里子件表格,也可重新添加

点取消表尾区域

![](https://vip.kingdee.com/download/0109cc1dadd1a3864b16adeafdf5dea344d1.png)

然后双击选中一个表格,会出现添加子数据表格按钮,此处理不可以再重新设置表尾区域了

![](https://vip.kingdee.com/download/01091a768ef91fc9467d917b21d96637089c.png)
