# 打印实践案例——打印模板图片/合同公章设置常见问题及设计方案

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：965 | 有用：3

## 1 问题描述

您是否在打印合同时，遇到过下列问题？

| 问题描述 | 图片说明 |
| --- | --- |
| 打印模板中,表尾设置了图片,但是打印时总是出现一条边框线 |  |
| 打印模板中,表尾内容是对齐的，但是打印预览时就不对齐了 |  |
| 打印模板中,表尾的落款的公章图片打印不完整 |  |

## 2 适用版本

星辰3.1以上所有版本

## 3 原因解析

以上问题其实本质都在于表尾的设置方式不对。例如用了分录表格、文本控件,然后文本跟图片有重叠,一般就很容易出现问题。

## 4 解决方案

如果打印模板是包含了分录、合同条款、落款以及公章相关的图片的话,建议最终落款的区域设计必须要使用布局网格来设置,然后图片需要单独添加控件，然后把图片覆盖在布局网格上。

| 第一种:公章在甲方,即表尾的左侧 | 第二种:公章在乙方,即表尾的右侧 |
| --- | --- |
|  |  |

下面以采购订单为例,介绍合同模板下，公章落款在左侧的设计方法,上面两种样式设置思路都是一样的,只是最后公章摆放的位置不一样而已,本文就不再赘述第二种方法。

详细操作步骤:

| 步骤 | 说明 | 打印模板说明 |
| --- | --- | --- |
| 1、完成打印模板的合同部分的设置后,在表尾空白区域添加布局网格 | 建议把甲方和乙方放在同一个布局网格里。这样的好处是,所有的字段都可以对齐，减少排版的工作量 | 假设添加一个5行*4列的布局网格 |
| 2、添加字段 | 根据实际业务场景添加字段 |  |
| 3、调整列宽、合并单元格 | 根据字段长度调整列宽，或者按需合并单元格 |  |
| 4、调整布局网格边框 | 根据用户场景,设置边框线是否可见 | 无边框效果:有边框效果: |
| 5、添加图片控件，上传图片，并调整图片的位置 | 如果是公章，建议上传背景是透明的PNG格式图片 |  |

注意：当打印超过一页内容时，在换页时,公章图片是否可以在上一页的表尾打印,其实是取决于上一页的表尾是否有足够的位置的。如果不够位置,就会打印到下一页。

例如下面图片，一共打印2页纸，因为上一页位置不够，公章就打印到第二页了。
![](https://vip.kingdee.com/download/010926739b28c30948059a96e246ed7c146e.png)

如果不希望公章跟落款的甲乙方信息相隔太远,建议图片不要有过多的留白，尽量让图片的边距线靠近公章本身。
