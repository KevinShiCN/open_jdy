# 新增引入引出模版时的模版类型有何含义?

【问题描述】

新增引入引出模版时的模版类型有何含义?

![](https://vip.kingdee.com/download/0109604e7baeedb14f318c74f20d70b93efa.png)

【解释说明】

**引入模版:**支持按照此模版引入数据,并且可以选择“按照引入模版格式引出”。

![](https://vip.kingdee.com/download/01092bc8c0ff37e142fbb5d499a60ab4b449.png)

**引出模版:**只用于“引出分析数据”。

![](https://vip.kingdee.com/download/0109208790c9221f4cedb7953e74ddaf6fa4.png)
