# 星辰2.0,分组打印,不同分组表格中间的间隔会有很大的空白,并且不同行的表格的显示的不一样,是什么情况呢?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：winnie_wen | 阅读：783 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/566427434493409536)

【问题描述】

分组打印,不同分组表格中间的间隔会有很大的空白,并且不同行的表格的显示的不一样

![](https://vip.kingdee.com/download/010908e3da48e1f84d6bb081d880e739d9ce.png)

【解决方案】

打开打印模板,检查大纲里的单据字段是否有文本字段重复,重复的字段是否正常显示在表头或表尾中

根据以上问题描述情况,检查这个模板多设置了一个[单据日期]字段,点击其中一个字段显示这个字段不在表头表尾,位置可能在数据表里面,删除该字段之后,保存模板,打印预览中分组的表格正常显示,没有空白。
