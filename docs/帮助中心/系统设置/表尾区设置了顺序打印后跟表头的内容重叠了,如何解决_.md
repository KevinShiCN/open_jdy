# 表尾区设置了顺序打印后跟表头的内容重叠了,如何解决?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：进销存赵嘉丽 | 阅读：1230 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/578748738038794240)

**【问题描述】**

表尾区设置了“顺序打印”后跟表头的内容重叠了,如何解决?

![](https://vip.kingdee.com/download/010942ff77d3154f48c390ff64b242667748.png)

**【原因说明】**

是表头设置了“所有页面”(即每页打印)导致。

![](https://vip.kingdee.com/download/01090ef52164da2341fbba96c072181298aa.png)

因表尾区由于是跟着表格下方的,当打印完商品分录后,表尾剩余的高度小于表尾区的高度时,也就是无法一次性打印完表尾的内容时,那么表尾就会自动换页到下一张纸(见下图说明),此时就会和设置了每页打印的表头产生重叠的问题。
![](https://vip.kingdee.com/download/01094c682f6dca0a49c1b51c1ae47c7b69a2.png)
**【解决办法】**

解决办法:

| 是否需要每页固定分录数 | 表头打印所在页 | 表尾区打印所在页 | 打印模板设置 | 预览效果 |
| --- | --- | --- | --- | --- |
| 需要固定行数 | 建议设置“所有页面” | 建议设置“所有页面” |  |  |
| 不需要固定行数 | 顺序打印/仅第一页 | 顺序打印 |  |  |

**【适用版本】**

星辰3.1所有版本
