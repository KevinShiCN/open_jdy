# 如何设置子账号能看到其他账号做的单?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：lykk | 阅读：469 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/577299118578604288)

【问题描述】

怎样配置子账号,让其可以查阅其他账号的订单记录?

【解决方案】

在子账号未设置数据授权的情况下,是允许查看到所有账号的单据;如需让子账号查看指定人员的单据可以通过【系统设置】-【用户管理】-【数据授权】进行操作

1、如需对“5450”用户进行授权

![](https://vip.kingdee.com/download/0109ac74d54df5f7401fbc0c7f5c8e95078c.png)

2、点击“数据授权”后,左侧选择“制单人”

![](https://vip.kingdee.com/download/010920bcaf4a72ff4bd8b53e5520ac468025.png)

3、右侧设置条件,如设置只能查看“5450”本人的单据,其他账号的都不允许查看

![](https://vip.kingdee.com/download/0109cb3003b40a7b492ebe12da14f82aba80.png)

4、选择后点击“保存”,子账号退出登录后生效

![](https://vip.kingdee.com/download/010962bb6b1186f642368b915996aba60bd4.png)

5、如还需查看其他人的单据,可以选择条件“或者”按照上述方法继续添加其他人的账号信息

![](https://vip.kingdee.com/download/0109dc446b011ed54d58a339a8ccddd811ef.png)
