# 打印单据时为什么有的字段会显示成星号**?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：进销存赵嘉丽 | 阅读：2055 | 有用：1

**【问题描述】**

打印单据时为什么有的字段会显示成星号**?

![](https://vip.kingdee.com/download/0109fcd5817f35e4453dafd7674c700f3073.png)

**【原因说明】**

有以下两种情况会导致打印时出现星号:

| 情况 | 设置路径 | 图片说明 | 打印预览效果 |
| --- | --- | --- | --- |
| 1 | 系统设置-系统参数-公共参数,开启了“界面中脱敏显示”参数 |  |  |
| 2 | 用户管理-用户授权-字段授权,对应的字段开启了“禁止查看” |  |  |

**【解决办法】**

用户可以根据实际业务需要进行设置:

1)如敏感字段(手机号、身份证号、银行账号)无需脱敏,则建议关闭参数,具体设置可参考:[(敏感数据设置)系统中手机号、身份证号、银行账号信息如何设置/取消显示星号**?](https://vip.kingdee.com/knowledge/391352043132273408?productLineId=35&isKnowledge=2&lang=zh-CN)

2)针对“禁止查看”的字段授权,建议维持现状,保持与单据查看结果一致,具体设置可参考:[如何设置子账号能否编辑或者查询某个字段?](https://vip.kingdee.com/knowledge/583097504950937344?productLineId=35&isKnowledge=2&lang=zh-CN)
