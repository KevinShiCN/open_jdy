# 税务参数（购买了税务管理才会有）

# **1 简介**

## **1.1 应用场景**

可以对税务管理的业务处理规则和处理惯例进行统一的设置和管理。

## **1.2 操作路径**

系统设置>系统参数>税务参数

## **1.3 适用版本**

星辰3.1以上所有版本

# 2 参数含义及应用场景

## 2.1 开票信息

必须提供税号提交开票申请并通过后才能进行开票参数设置，目前两家服务商提供的开票工具，开票工具一经选定不能修改，具体开票可以按照开票向导提示进行操作。

## 2.2 发票管理

1）选择取票工具（目前支持两家服务商提供的取票工具，取票工具一经选定不能修改。）

2）预警参数

可以对【增值税行业税负率】和【预警范围(超过预计行业税负率)】进行设置，系统提供行业平均税负率供参考设置（点击小问号可以查看），设置后在【税务风险报告】中会根据设置的税负率和预警范围进行预警。

## 2.3 开票参数

可以设置具体的开票参数及信息、开票限额，并设置开票预警金额和开票预警控制方式。

说明：关于开票参数设置，开通开票管理后，可点击“开票向导”进行查看。

1）开票人、复核人、收款人、开票备注填充设置：进行开票时，会取对应设置的信息显示在发票上。

2）开票限额：即税务局给企业核定的单张发票开票限制，设置后当超出开票限额时，系统会进行控制。
