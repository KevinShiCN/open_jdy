# 星辰2.0单据打印,一个单据分多页打印,打印第一页正常,但是打印第二页时会将表头信息打印到表体中?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：杨桂平 | 阅读：2739 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/129339770097650283)

【问题描述】

星辰2.0,单据打印时,若单据有很多分录,需要分几页打印,打印第一页正常,但是第二页的表体中会显示表头的内容。

![](https://vip.kingdee.com/download/010987c4dc83a5dd4cf69a6a63f5c8409075.png)

【解释说明】

1、设置打印模板时表头的字段表格边框线跑到表体的表格框内了

2、打印设置勾选了只在第一页打印表头

![](https://vip.kingdee.com/download/0109b01ac3d7f74c4b1bba086f64d9fbb6ae.png)

![](https://vip.kingdee.com/download/0109ffe3667b58eb47a1be9e890195deaaf4.png)

【解决方案】

把打印模板表头字段的表格框线上移即可。

![](https://vip.kingdee.com/download/0109e794aefab6b6470ca54691cc32f97bb0.png)

![](https://vip.kingdee.com/download/0109c4e6fe5cfde747548d732d6f87fd371d.png)
