# 星辰2.0,另存的预置模板改了原本金额类字段为仓库货其他非金额类字段后,预览不显示是什么原因?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：杨桂平 | 阅读：115 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/537436492923310592)

【问题描述】

另存的预置模板改了原本金额类字段为仓库货其他非金额类字段后,预览不显示是什么原因?

【解释说明】

金额类取值改为非金额类取值的字段后,需检查一下字段的格式是否还有相关的数值格式没有清除,格式不对会影响字段的显示,清除掉即可。
