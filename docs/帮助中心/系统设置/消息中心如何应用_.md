# 消息中心如何应用?

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题
> 作者：_Leo | 阅读：996 | 有用：0

【问题描述】

消息中心如何应用?

【功能介绍】

消息中心:是为不同岗位的用户提供各类操作支持和业务提示,实现业务流程的畅通和协同,提升企业运营效率,(目前仅星辰3.0版本已支持)

![](https://vip.kingdee.com/download/01093dd9c72e778b42a29c953344f124481c.png)

功能包含:后台设置、后台消息中心列表、web端消息通知、app消息通知、钉钉消息通知、企微消息通知。业务涵盖:单据消息提醒(增删审核)、经营消息、预警通知等。

【业务场景】

1) **消息内容自定义,推送单据开关:**单据消息通知的时候,可以根据用户的业务要求自定义消息内容,比如用户要看单据编码、单据金额、制单人、部门、客户名称等信息,或者业务员只需要接收销售单据的信息,采购员只需要接收采购相关的消息,管理员只想看需要看收付款的信息等,都可以通过后台进行设置。

2) **重要业务事件通知:**支持单据的待审核、单据已审核、单据删除的消息提醒。可以在销售订单审核提醒仓管员拣货;仓管对销售出库单审核后通知配送员和司机进行配送;配送完成后,提醒业务员收款。或者提醒业务员有拜访任务和联系任务需要处理。

3) **经营重要报告的通知:**可以设置业务员管理员或者老板,推送经营日报、周报、月报;或者对销售推送销售部门销售数据;对仓管推送仓库的出入库和库存数据;对财务推送资金数据等。

4) **业务数据监控预警:**库存(最低、安全、最高)的通知,商品临期通知、商品过期通知、新增商品通知商品滞销通知,单据交期预期通知、应收应付预警通知、客户流失预警通知等。
