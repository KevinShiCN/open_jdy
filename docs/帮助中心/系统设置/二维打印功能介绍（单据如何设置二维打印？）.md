# 二维打印功能介绍（单据如何设置二维打印？）

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：功能介绍, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：2266 | 有用：8

# 1 应用场景及价值

服装鞋帽等启用辅助属性管理的行业在开单后需要支持二维展示打印单据，以方便业务进行核对

# **2 适用版本**

星辰 3.1以上所有版本

# **3 应用条件及流程**

1）商品启用辅助属性，且辅助属性组在2组及以上，开单后需要二维打印单据

2）采购、销售类单据内置二维打印模板（**默认模板不可修改，如需修改，复制模板后修改**）

3）开单选择辅助属性并选择二维打印模板进行打印

# **4 具体操作**

## **4.1 使用系统预置模板**

打开具体需要打印的单据列表或者单据详情，在打印>打印设置>选择模板中找到二维打印预置模板，并勾选"二维打印"，点击"确定并打印"即可

## **4.2 自定义横向打印的属性组及横向展示列数**

打印设置中，还有2个参数

- 横向展示属性：默认横向展示的属性组是属性组1，用户可以在打印设置中切换横向展示的属性组。
- 横向展示列数：设置为0或小于单据上横向显示属性的个数时，默认按单据上横向显示属性的列数展示；大于单据上横向显示属性个数时，自动增加空白列，空白列的列宽同正常的横向显示属性列。

## **4.3 自定义打印模板**

### 4.3.1 **多个属性组的二维打印**

可以选择预置模板进行复制后进行个性化的调整，降低从头开始设计的工作量。要使用打印设置中二维打印效果的关键在于，打印模板的数据表格中一定要绑定“辅助属性.辅助属性名称”字段。

![上传图片](https://vip.kingdee.com/download/01003a1eb1accbaa4fe7a8965e64465e2575.png)

### 4.3.2 **单个属性组的二维打印**

# 5 注意事项

## 5.1 二维列数字的尾零显示问题

二维打印列里的数值的小数位精度（下图红框内的数字）是保持跟分录中的单位精度一致的，

![上传图片](https://vip.kingdee.com/download/01007d9bbd7b74864db196265bcf7a313d3e.png)

如果希望二维展示列不显示尾零，可以在打印模板中，对“数量”列开启“尾零不显示”即可
