# 打印基础——纸张方向的说明

> 产品：金蝶云·星辰 | 分类：系统设置 | 类型：常见问题, 快速入门, 精华汇总
> 作者：进销存赵嘉丽 | 阅读：5137 | 有用：6

1 问题描述

打印模板/打印机首选项里的纵向和横向是什么意思？

## 2 适用版本

星辰所有版本

## 3 原理解析

在打印机设置中，纸张方向决定了打印内容在纸张上的排列方式，分为纵向（Portrait）和 横向（Landscape）两种模式。

### 3.1 纵向（Portrait）

- 定义：纸张的 高度大于宽度（默认方向），内容按垂直方向排列。

- 类似手机竖屏或A4纸的常规摆放方式

- 适用场景：

- 打印文档（如Word、PDF）、信件、网页等大多数文本内容。
- 需要页面高度容纳更多行内容时。

### 3.2 横向（Landscape）

- 定义：纸张的 宽度大于高度，内容按水平方向排列。

- 类似电视横屏或表格的宽幅展示。

- 适用场景：

- 打印宽表格、图表、横向设计的图片或幻灯片（如PPT）。
- 需要页面宽度容纳更多列内容时。

### 3.3 特别说明——针式打印机

在针式打印机中，一等分纸（连续单层纸241*93mm）和二等分纸（连续双层纸241*140mm）的默认方向设置为纵向，尽管它们的宽度 < 高度，这是由以下原因决定的：

- 连续纸的进纸方式：针式打印机通常使用链式走纸（两侧带孔），纸张是纵向连续进纸的（高度方向为进纸方向）。

- 如果默认设为横向，打印时纸张需要横向移动，但针式打印机的机械结构（打印头左右移动）和走纸齿轮设计更适配纵向走纸。
- 示例：常见的针式打印机放纸都是纵向：![](https://vip.kingdee.com/download/01092e7ee4751e2c4cb08df5c2fe1a6dce34.png)

- 打印头运动方向：针式打印头的针脚是水平排列的，打印时左右移动（X轴），纸张纵向走纸（Y轴）。这种设计下，纵向排版能最大化利用打印头的运动效率。

基于以上两点，目前主流的针式打印机的打印机首选项也是“纵向”，星辰的打印模板里也保持了一致的参数。
