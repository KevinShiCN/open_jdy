# 1、业务场景

一般在做配送时，当配送的商品非常多时，单据分录内容很多，打印时需要很多张纸，但是有时只需要打印基本信息即足够（如商品编码、名称、单位、数量），因此希望一次可以在同一页纸上把左右两边都打满，提高分拣和送货核对的查看效率，也能达到节约纸张的目的，这种打印类型我们称之为左右型双表格打印。

# 2、解决方案

针对上述场景，可以通过打印模板的横向节纸和每页打印来实现：

1）左右双表格展示

2）各表体独立统计小计+本单总计分开统计

3）节约纸张（以前打2张，可能现在打1张就好，打得越多，节约越多）

# 3、操作步骤+效果呈现

接下来，我们将以销售出库单（二等分纸：241*140mm）为例，演示如何设计双表格设计。

## 1）设置纸张样式

选择自定义，方向选纵向（一般针式打印机默认都是纵向出纸），并对上下左右边距留出至少1mm的位置。

![](https://vip.kingdee.com/download/010009aed199016444a79dd5766c40ccbc4a.png)

## 2）在模板的【业务】中，开启横向节纸

注：横向节纸的意思就是当纸张右边还有空白时，不换页打印，而是将剩余的内容接着从右边横向打印，直到打完为止。一般是为了节约纸张。同理，纵向打印则是在打印下方空白的地方不换页接着打印。

![](https://vip.kingdee.com/download/0100c3fc961174154a469f1af44f8f27e6ef.png)

## 3）在纸张一半宽度的左侧区域内设计要打印的内容（放置单据名称、单据信息、表格等等）

注：之所以是纸张宽度的一半是因为要预留另一边空白可以打印双表格。

![](https://vip.kingdee.com/download/01004f399a61ae854f8b9ca83c78bf50e53b.png)

## 4）开启首行和合计行的“每页显示”

![](https://vip.kingdee.com/download/0100d0c3c746d14a40589ebac7267ddd3379.gif)

设置完后，假设一张包含了80行分录的销售出库单，那么打印预览效果如下：

![](https://vip.kingdee.com/download/0100b9e21c77d5a94fb49c17baf72b80c2c1.png)

![](https://vip.kingdee.com/download/01003a2c2a8f311044cbb3c61e3d6afc7b34.png)

![](https://vip.kingdee.com/download/0100e61ec246b2234733aff7d058aa331a15.png)

****加分技巧~！**

一般每页显示小计，有时候还需要在最后一页加一行总计来显示本单合计，那么我们可以在选择合计行后，向下插入新的一行。

注：由于是最后一页才需要显示总计，新增的行不需要开启“每页显示”：

![](https://vip.kingdee.com/download/01006ccbb727eab94243972e50e02bf590e3.gif)

# 4、设计注意事项：

## 1）模板右侧要保持空白，不要添加多余的字段；

## 2）不要在表体外添加页码：

因页码属于固定文本，无法随着表格动态移动，添加页码会无法实现双表格的效果：

如果按下图添加页码：

![](https://vip.kingdee.com/download/01002c652d99c3cb4834985418ab970a5c24.png)

打印预览：

![](https://vip.kingdee.com/download/0100f3e0837e8a8f49f3ae9da44cf851e468.png)

如果要需要添加页码，建议在表格内添加一行空白行，将页码放置到表格内：

![](https://vip.kingdee.com/download/010046effbfe0e8a4f599f3162fbca8cf7cd.gif)

但是这样则会按照每把每一个表格当成一页统计：

![](https://vip.kingdee.com/download/0100a4b12048e8e445fd83633ca7d1b70a42.png)![](https://vip.kingdee.com/download/0100a9930234f22f4f778f7b5cdda75d52a8.png)

注意！引出使用双表格时，是否需要使用页码，建议用户自行决定。
