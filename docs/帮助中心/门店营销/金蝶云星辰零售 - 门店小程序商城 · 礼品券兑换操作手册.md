四、功能概述

礼品券作为一种常见的营销工具，在商家实际业务运营过程中发挥了重要的营销价值，可以有效锁定客户，提升复购。其主要业务场景包括：

1. 员工福利：企业批量采购，作为员工福利发放。

2. 朋友礼品：亲朋好友间赠送礼品卡，满足社交需求。

3. 购买使用：客户购买优惠的礼品卡，锁定消费和优惠，提供复购率。

第 1 步：创建礼品券方案，引入券号和密码

1. 创建礼品券方案

![](https://vip.kingdee.com/download/01003d46a0b3db844774afd90403ea946683.png)

登录零售后台，前往《门店营销 - 礼品券 - 礼品券方案》，点击【新增】可以创建一个新的礼品券方案，按照原手册操作即可。这里主要有以下改动：

| 功能调整1. 指定套餐套餐：兑换商品清单支持添加套餐，可以先创建套餐，再到这里添加套餐，商家销售礼品券给用户后，用户可以在商城使用礼品券兑换套餐。2. 可兑商品数量：可以设置添加的商品全部可兑换，或任选 N 件。 |

第 2 步：管理礼品券方案

|  |  |  |
| --- | --- | --- |

创建礼品券方案后，可以对礼品券方案进行管理。按照原手册操作即可。这里主要有以下改动：

| 功能调整1. 礼品券方案编号：点击方案编号可以对礼品券方案进行编辑，可以修改以下内容：• 礼品券方案名称、有效周期、礼品券说明、免密码、适用门店。修改方案后销售的券以修改后方案为准，修改方案前销售的券以修改前方案为准• 券类型: 礼品券发放记录为空时可以修改，有礼品券发放记录后不能修改1. 礼品券方案详情：• 点击方案详情可以查看导入的礼品券详情• 点击【修改失效日期】可以多选礼品券号批量修改失效日期￮ 可以在原来的失效日期基础上统一延长 N 天，比如 180 天￮ 可以统一延长到某一个固定日期，比如从2022.08.01延长到2022.12.31￮ 可以选择是否将此礼品券方案下的全部有效礼品券都统一修改成新的失效日期 |

第 5 步：商城礼品券兑换

进入商城《我的 - 卡券》，找到礼品券，可以查看用户名下的礼品券、兑换记录、绑定礼品券、兑换商品等。

1. 绑定礼品券

|  |  |
| --- | --- |

目前只能在线下门店购买礼品券，或者 A 购买礼品券后赠送给 B。购买礼品券后可以在 POS 或商城绑定到某个用户名下。POS绑定礼品券原手册已有详细说明，这里主要介绍在商城如何绑定礼品券。

点击【绑定礼品券】，输入券号或密码后，点击【确认绑定】即可；若你不想将此礼品券永久绑定在某个用户名下，只是给他临时使用一下，可以点击【立即兑换】，本次兑换完后，其他用户仍可绑定使用。

几点说明，请注意：

| 说明1. 购买礼品券：• 目前仅支持在线下门店 POS 购买礼品券，不支持在商城购买• 礼品券是不记名的，购买者与绑定者可以不是同一个人1. 绑定礼品券：• 支持同时在线下门店 POS 绑定和商城绑定• 绑定后不可解绑，若你不想永久绑定在某人名下，请到商城操作，绑定礼品券时选择立即兑换即可• 绑定礼品券时，券号必填，密码是否必填根据创建礼品券方案时的设置控制 |
| --- |

2. 查看礼品券

|  |  |
| --- | --- |

用户绑定礼品券后，可以查看其名下的礼品券，包括可用券和不可用券。礼品券内容主要包括：券数量、券名称、券号、已兑/未兑数量说明、有效期、面值、兑换按钮、规则或不可用原因。

3. 兑换商品

|  |  |
| --- | --- |

点击券卡片的【兑换】，进入兑换商品页面，页面顶部可以查看券详情，下方展示该礼品券可以兑换的商品清单，用户选择想要且当前仍可兑换的商品，点击【确认兑换】进入确认订单页。

|  |  |
| --- | --- |

进入确认订单页，与一般的商品下单流程一样，选择门店自提或配送到家，确认订单信息后提交订单即可。使用礼品券兑换无需支付。

|  |  |  |
| --- | --- | --- |

兑换完成后，会生成礼品券兑换的订单，用户可以进入《订单》查看，关注订单状态变化。

4. 查看兑换记录

|  |  |
| --- | --- |

点击【兑换记录】可以查看礼品券兑换记录，记录每次兑换使用的礼品券号、兑换/未兑数量、兑换时间以及业务类型。

第 6 步：POS端处理商城礼品券兑换订单

![](https://vip.kingdee.com/download/0100778a7227cce44bdf988200c1bb807d3d.png)

兑换完成后，会生成礼品券兑换的订单，推送至门店POS。商家可以进入《配送台 - 商城》查看。礼品券兑换的订单会标识兑换字样，便于商家识别。订单处理流程与一般的商城订单处理一样，此处不赘述。
