# 金蝶云星辰零售营销管理、促销、优惠券、礼品卡、会员营销、短信营销、精准营销常见问题Q&A及解决办法

金蝶云星辰零售营销管理、促销、优惠券、礼品卡、会员营销、短信营销、精准营销常见问题Q&A及解决办法

解决方案：

**促销**

**1、Q：批发促销和零售促销的促销活动有重复的，有什么区别？**

A：零售促销是应用于门店零售的，直接面向C端消费者。批发促销是针对批量购买商品的，更偏向于B端卖家。

**2、Q：促销可以直接赠送金额吗？**

A：不可以，建议赠送积分，再设置积分抵现规则。

**优惠券**

**1、Q：后台授权绑定了公众号，能直接发放优惠券到会员的微信会员卡上，不用用户领取吗？**

A：支持。当次发放的优惠券下次消费使用。非指定标签的会员在微信上能看到优惠券，但不可领取。目前在公众号上有领券中心，需要主动领券。

**礼品卡**

**1、Q：礼品卡是否可通过二维码扫码领取？ 领取后是否可以直接到门店消费？**

A：不支持电子礼品卡，不能扫码领取；礼品卡不记名，只要有卡号和密码即可领取和消费。

**2、Q：礼品卡使用过部分卡号，可以作废方案，重新引入未使用过的卡号吗，比如礼品卡有100个卡号，只使用了一个，其他99个还没有使用，要更改方案，我可以把之前的方案作废后，重新设置并重新导入这99个卡号吗？**

A：不支持。礼品卡方案只要有一个卡号被卖出或使用过，就不能再更改方案。只能重新创建一个新的礼品卡方案，引入新的礼品卡号。

**3、Q：客户已经在后台设置了礼品卡并上架，POS端购买时无法显示礼品卡列表，是什么原因，怎么处理？**

A：前往后台【营销管理-礼品卡-礼品卡方案】，找到对应礼品卡方案，检查该礼品卡设置的卡类型是不是IC卡，只有IC卡才会显示礼品卡列表。

**4、Q：创建礼品卡方案时设置免密，导入礼品卡号还需要密码吗？**

A：这种情况用 Excel 表导入礼品卡号时无需再设置密码，直接导入卡号即可。

**5、Q：礼品卡支持实体卡吗，如何设置？**

A：支持实体卡。后台创建礼品卡方案时选择“IC卡”，然后前往POS【店务-实体礼品卡批量激活】完成激活操作。

**短信营销**

**1、Q：支持发送自定义短信模板给会员吗？**

A：不支持

**2、Q：支持会员消费自动发短信告诉会员消费多少，还有多少余额吗？**

A：支持，在门店营销——短信营销——提醒设置，开启对应短信模板
