三、名词说明

说明：以下内容，线上商城简称为线上，线下门店简称为线下；客户/商户/商家均指商家；用户/消费者/顾客均指消费者。

四、功能概述

商家可以在后台配置付费优惠券，上架到线上门店，消费者可以在小程序商城付费购买优惠券包。同时，支持用户分享券包链接给微信好友；也可以生成带有小程序码的券包图片保存到本地相册后分享给微信好友或朋友圈；或者将券包购买页面链接嵌入公众号文章。适用于线上促销场景。

六、操作说明

第 1 步：零售后台建券，上架到门店

1. 创建优惠券方案

![](https://vip.kingdee.com/download/01006ca5b75bba824631b829033cd12de52f.png)

登录零售后台，前往《门店营销 - 优惠券 - 优惠券方案》，点击【新增】创建优惠券方案。（*注：如需优惠券可在门店商城使用，适用渠道需勾选线上门店商城。）

2. 创建付费优惠券服务商品

![](https://vip.kingdee.com/download/01007fd94700792e40bbb98bab2227a3af0d.png)

前往《门店管理 - 资料 - 服务商品配置》，点击【新增服务商品】新增一个付费优惠券服务商品。商品类型选择服务，建议上传图片，头图将作为券包在门店商城中显示的主图，其余图片将作为详情图展示。同时，支持配置和展示商城标签。

3. 配置付费优惠券服务商品

![](https://vip.kingdee.com/download/0100ab876122e75840b0a6dd0d964e977aaf.png)

前往《门店管理 - 资料 - 服务商品配置》配置付费优惠券服务商品。点击【配置服务商品】进入配置页。服务特性选择付费优惠券，商品选择上一步创建的付费优惠券服务商品，下方优惠券选择上一步创建的优惠券，并设置发放数量、发放方式和生效方式。

4. 上架服务商品到线上门店

![](https://vip.kingdee.com/download/01008ede97834a13478da4e8ac6d5c192c11.png)

前往《门店管理 - 资料 - 商品上下架》，选择对应的付费优惠券服务商品上架到门店，上架渠道选择线上或线上+线下。

第 2 步：商城购买付费优惠券包

|  |  |
| --- | --- |

完成上述配置后，该优惠券包将在已上架的对应门店的领券中心展示，点击可查看券包详情、券的使用范围和发放信息等。

1. 购买：点击【购买】可在线购买该券包，购买成功后券包将自动发放至用户的优惠券账户，可在《我的 - 卡券 - 优惠券》查看。

2. 分享：可以将券包分享给微信好友、微信群或朋友圈。

| 3. 微信分享 |
| --- |

1. 点击【分享】可以将券包链接直接分享给微信好友；或者生成带有小程序码的券包图片保存到本地相册后分享给微信好友或发到朋友圈。

1. 点击右上角的【...】可以直接将券包链接分享到微信朋友圈。

| 2. 文章内嵌小程序 |
| --- |

登录<a href="https://mp.weixin.qq.com/" \h"="" style="">微信公众平台 ，创建公众号文章，将优惠券包页面作为素材插入文章中，用户可在公众号文章中点击券包页面链接跳转到小程序商城内购买。操作方式：点击【小程序】，输入你想跳转的小程序名称，点击【下一步】按照指引添加落地页链接，可指定页面链接。
