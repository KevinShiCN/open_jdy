# 礼品券方案功能如何使用?

> 产品：金蝶云·星辰 | 分类：门店营销 | 类型：常见问题

【问题描述】

礼品券方案功能如何使用?

【业务场景】

礼品券作为一种典型的营销工具,在商家实际业务运营过程中发挥了重要的营销价值,可以有效锁定客户,提升复购。其主要业务场景包括:

一、员工福利:企业批量采购,作为员工福利发放;

二、朋友礼品:亲朋好友间赠送礼品券,满足社交需求;

三、购买使用:客户购买优惠的礼品券,锁定消费和优惠,提高复购率。

【解释说明】

**礼品券的主要作用是通过制作、销售礼品券后,在指定的门店兑换指定商品的过程,兑换礼品券时无需额外支付费用。在系统中的主要业务流程如下:**

**一、创建礼品券方案,引入券号和密码;**

**二、创建和配置礼品券服务商品;**

**三、创建礼品券账户和结算方式,配置门店收款方式;**

**四、打印礼品券;**

**五、开启礼品券核销参数;**

**六、购买礼品券,查看礼品券发放和激活状态及零售单;**

**七、绑定礼品券;**

**八、核销礼品券,查看交易流水。**
