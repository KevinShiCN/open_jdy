# 凭证引出时,科目名称可以只显示最明细科目吗?

【问题描述】

凭证引出时,科目名称可以只显示最明细科目吗?

![](https://vip.kingdee.com/download/0109401aa02ac7d14ba4b446d76ca4d51781.png)

【解释说明】

目前系统**无法支持**按最明细科目名称导出,引出时会显示科目的全称。
