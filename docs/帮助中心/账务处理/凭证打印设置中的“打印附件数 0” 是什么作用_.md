# 凭证打印设置中的“打印附件数 0” 是什么作用?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：杨桂平 | 阅读：362 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/537798920785793536)

【问题描述】

凭证打印设置中的“打印附件数 0” 是什么作用?

![](https://vip.kingdee.com/download/01095e61525ea4724eba8e12e6482730ce3e.png)

【解释说明】

打印凭证时,凭证的附件数为0;打印设置若勾选了该参数则显示为0,

![](https://vip.kingdee.com/download/0109a2f2b184f7f349d38c571adc8b587a12.png)

若没有勾选则显示为空值;

![](https://vip.kingdee.com/download/010937e40e27286641e78354eabff2b4e246.png)

凭证的附件数不为0时,该参数对打印的凭证附件数不影响,附件数会正常打印

![](https://vip.kingdee.com/download/0109e32da08e03274b5394e6b00e17b92112.png)
