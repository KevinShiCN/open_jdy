# 生成凭证要求高，按需设置很重要

生成凭证要求高，按需设置很重要

![上传图片](https://vip.kingdee.com/download/0100fee9b55ee8dc4641bea01e23b15a1c11.png)

      财务小张做账很严谨，凭证录得井井有条。自从看到星辰有一键生成凭证的功能，他也跃跃欲试，想要解脱双手。但生成凭证的格式远远达不他的心里预期。小张针对凭证模板进行了一番研究，终于他发现了几个好用的妙招。她都发现了什么，我们一起来看看吧！

1、 凭证日期设置

假设：财务期间与业务期间有时候不一致，希望能按业务单据日期生成凭证。

1）凭证模板选择单据日期。

![](https://vip.kingdee.com/download/0100515e13cb93b640c699fd1388287e6437.png)

2）单据生成凭证选对应模板：

![](https://vip.kingdee.com/download/01007adeb625937e4dc19a2acdfbab8984cb.png)

3）单据日期是2024-06-18，所以凭证日期也是2024-06-18。

![](https://vip.kingdee.com/download/0100cc5ee2cbde7e4a57821151cd700c10a8.png)

![](https://vip.kingdee.com/download/01001eefff8a71b04b4c808c75c8f434b7f1.png)

2、 凭证分录顺序设置

假设：凭证分录借贷是乱的，希望分录能先借后贷。

![](https://vip.kingdee.com/download/0100cfa085c430224a449df6cf7c7014a147.png)

1） 凭证模板设置凭证分录顺序为“先借后贷”。

![](https://vip.kingdee.com/download/0100237ecb0581bf4d988406382cfc9e88c6.png)

2） 单据生成凭证选对应模板。

![](https://vip.kingdee.com/download/0100316bfb42ecdd4d84aa96608f6fc5d2d8.png)

3）凭证分录顺序为先借后贷。

![](https://vip.kingdee.com/download/01005f242b13480949b78c16a7fcfb25402e.png)

3、 凭证适用条件设置

1） 凭证模板表头设置适用条件。

假设：采购入库单有的有采购费用，有的没有采购费用。希望单据生成凭证时分别对应需要的凭证模板。

i） 当采购入库单没有费用，凭证模板不设置适用条件，并将凭证模板设置为默认模板。

![](https://vip.kingdee.com/download/010099d07bf1a07d46bbb2fb6d7a9757984d.png)

![](https://vip.kingdee.com/download/01004cd4a00db65f43758f31cbcd6c939c7f.png)

ii） 当采购入库单有费用，凭证模板设置适用条件。

![](https://vip.kingdee.com/download/0100295032254faf47218eb7cd207018061c.png)

iii) 单据生成凭证时会按模板一一对应。

![](https://vip.kingdee.com/download/010074bc3e5ae811499d8d7ada5252c38750.png)

注意：此处同一单据下有多个凭证模板且适用条件为空时，取默认凭证模板；当有适用条件时，适用条件优先级会高于默认模板。

2） 凭证模板表体设置适用条件

假设：收款单选源单，当源单是销售出库单时生成凭证对应“应收账款”科目；当源单是其他收入单的时候，凭证对应“其他应收款”科目。

i) 凭证模板设置分录筛选条件：按源单类型不同，分别取各自的科目，分录行取数一样。科目关联设置也需设置上对应科目。

![](https://vip.kingdee.com/download/0100c9052587a0064325b085a9e9bba68a8c.png)

![](https://vip.kingdee.com/download/0100097b5a20be1e484e9e64549de8970243.png)

ii) 收款单有不同类型单据收款。

![](https://vip.kingdee.com/download/01006a22a357c54341ada9af69c1d8c2b0f9.png)

iii) 单据生成凭证选对应模板：

![](https://vip.kingdee.com/download/01003afa634f16414d089b695c3cfe033cc4.png)

Iv) 凭证按源单类型不同，分别带出对应的科目。

![](https://vip.kingdee.com/download/0100b249ebb077494ae48b2618f0b9f4107d.png)

4、 日记账凭证模板与摘要的巧妙连用。

1） 假设：日记账摘要每月很固定，可以根据摘要库设置科目。

i) 日记账录入时选摘要库的摘要，摘要库没有的也可以新增摘要内容保存到摘要库。

![](https://vip.kingdee.com/download/0100fa639a1398ab496ba2d9de1a3a9c3845.png)

ii) 摘要库科目关联设置，设置相应科目。

![](https://vip.kingdee.com/download/0100c5208cf4e75b4c1e8fe3d0376f970132.png)

iii) 凭证模板取摘要库对应科目。

![](https://vip.kingdee.com/download/01000ee374b462a7425f8e2826b049ea690c.png)

iv）日记账生成凭证选对应模板。

![](https://vip.kingdee.com/download/0100c57fe7ff25574607804b83306466673e.png)

![](https://vip.kingdee.com/download/0100a40a88c0012e43e4b20bdd72c9340dd3.png)

v）凭证可以带出摘要库对应科目。

![](https://vip.kingdee.com/download/0100fe918682ada44063849bbc056baaf919.png)

2） 假设：日记账生成凭证摘要想要取日记账本身的摘要。

i）凭证模板摘要取单据信息-摘要。

![](https://vip.kingdee.com/download/01004af5ab011e3f4975bfd7823618356e62.png)

ii) 日记账摘要是“支付张三的货款”。

![](https://vip.kingdee.com/download/010004881507fb954e1ca643c44ea1279670.png)

iii）日记账生成凭证选对应模板。

![](https://vip.kingdee.com/download/0100ec99a3fd616f437f941660691196e937.png)

iv) 凭证摘要带出的就是日记账的摘要“支付张三的货款”。

![](https://vip.kingdee.com/download/010000e445fa99c247fcb744b86e9938b951.png)
