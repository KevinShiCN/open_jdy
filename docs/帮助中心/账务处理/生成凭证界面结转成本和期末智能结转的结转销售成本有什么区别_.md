# 生成凭证界面结转成本和期末智能结转的结转销售成本有什么区别?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：1 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/821911741117300224)

【问题描述】

生成凭证界面结转成本和期末智能结转的结转销售成本有什么区别?

【解释说明】

**一、适用场景不同**

①**生成凭证界面 — 成本结转**:适用于已购买财务 + 进销存模块且关联使用的用户,可直接根据销售出库单等业务单据自动计算并生成结转销售成本凭证,无需手工计算

具体操作可点击:[生成凭证界面怎么生成结转成本凭证?](https://vip.kingdee.com/link/s/ZU69o)

![](https://vip.kingdee.com/download/010982fe13e6312546f592bdd426b1c794bd.png)

②**期末智能结转 — 结转销售成本(系统)**:适用于仅购买财务模块、未购买进销存的用户,需设置结转销售成本的结转方式和结转比例及科目,再点击生成凭证

具体操作可点击:[期末智能结转中结转销售成本功能如何使用?](https://vip.kingdee.com/link/s/ZRXsO)

![](https://vip.kingdee.com/download/0109eb928c0de34140bd83a9b151d91a221f.png)

二、**数据来源不同**

生成凭证界面:数据来源于进销存业务单据,进销存模块进行出库成本核算后,再生成结转成本的凭证

期末智能结转:无业务单据数据,数据来源于凭证,再通过模板结转。

**三、使用注意事项**

财务 + 进销存联用场景下,**二选一即可,不可重复结转**:

已通过业务单据在生成凭证界面结转成本 → 期末智能结转无需再操作,即使该界面显示有数据,也无需处理,可直接结转损益并进行期末结账;

未使用业务单据结转 → 可通过期末智能结转模板处理。
