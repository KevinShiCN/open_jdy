# 业务单据生成凭证时提示第x行科目启用了数量辅助核算,数量为0如何处理?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：嘿唬哈 | 阅读：182 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/660649014726839040)

【问题描述】

业务单据生成凭证时提示第X行科目启用了数量辅助核算,数量为0如何处理?

![](https://vip.kingdee.com/download/0109519575043337440181233659e79a7e58.png)

【概述】

如科目启用了数量辅助核算,需检查凭证模板中是否有设置数量取数。

【操作步骤】

以其他入库单为例;

点击【账务处理】-【业务凭证模板】,点击其他入库单编号,进入其他入库单模板设置界面,点击数量字段,进入取值界面;选择【商品分录数量】即可。保存模板再次进行凭证的生成。
