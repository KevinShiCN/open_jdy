# 账务处理反结账提示:反结账检查失败:财务当前期间等于出纳的启用期间,不能“反结账”

【问题描述】

账务处理反结账提示:反结账检查失败:财务当前期间等于出纳的启用期间,不能“反结账”

【业务场景】

账务处理反结账提示:反结账失败,原因如下: 2024年2期反结账失败,原因如下:反结账检查失败:财务当前期间等于出纳的启用期间,不能“反结账”。

![](https://vip.kingdee.com/download/01090bc569d0ada846388992372fdac9f774.png)
【解释说明】

账务处理反结账提示:财务当前期间等于出纳的启用期间,不能“反结账”,这是由于出纳参数启用与“与总账一起结账”的情况下,账务当前期间等于出纳启用期间**不允许**反结账操作。

若账务需先反结账处理,建议出纳参数可先不启用“与总账一起结账”。

账务反结账处理完重新结账后,当账务跟出纳当前期间一致时,也可重新启用“与总账一起结账”

![](https://vip.kingdee.com/download/01091e424e0918de484ebbf271a2ffe06436.png)

【操作流程】

从【系统参数】>【系统参数】进入系统参数页面
![](https://vip.kingdee.com/download/0109850c92cfccf8483ea7a634df6fc6687d.png)
在系统参数页面。找到【出纳参数】取消勾选“与总账一起结账”后点击<保存>

![](https://vip.kingdee.com/download/01093be7d8c9025d45e0828a77cff488167a.png)
出纳参数保存成功
![](https://vip.kingdee.com/download/0109884a0235ae2c40efab3db2130699d261.png)
参数保存成功后再进行账务期末反结账操作即可

![](https://vip.kingdee.com/download/0109fb88f5dfca9e4f7f9c8b3cad1da4b0b2.png)
