# 一个账套怎样做多个门店(公司)的账务/业务处理

【问题描述】

一个账套怎样做多个门店(公司)的账务/业务处理

【解释说明】

一般建议一个账套做一家公司业务,避免两家业务混乱。如果一定要一个账套做两家公司的业务:

可将不同的门店或公司设为部门,单据上按部门区分

![](https://vip.kingdee.com/download/0109d826238790264c938abf2bbe9bfe9eee.png)

科目挂**[部门辅助核算](https://vip.kingdee.com/link/s/ZyoZJ)**,单据生成凭证时,**[科目上可以取到部门信息](https://vip.kingdee.com/link/s/ZyoZV)**,月底可以根据**[项目利润表](https://vip.kingdee.com/link/s/Zyolq)**出财务利润

![](https://vip.kingdee.com/download/010904f6508e1f1a48359cb053b9adbe4683.png)

2、但是查询资产负债表,现金流量表时会有局限性。此时查的数据是两个门店总的业务,如果想要单独查看单个门店业务,只能通过自定义报表设置取数。

如下图所示,以资产负债表为例,资产负债表另存为,其中一个门店资产负债表![](https://vip.kingdee.com/download/0109897eb89f56aa440ebde68b51b2b0b52d.png)
之后可根据业务需要,去编辑取数,核算项目取:部门,选择部门1![](https://vip.kingdee.com/download/01094ec25767af794837b314aed96f2da74f.png)
综上所述:虽然一个账套可以做多家公司的账,但具体操作应根据实际业务需求、会计原则以及相关的法律法规来进行,以确保合规性、透明性和准确性。
