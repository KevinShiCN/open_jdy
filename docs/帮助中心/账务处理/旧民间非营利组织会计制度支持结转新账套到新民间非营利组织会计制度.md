# 旧民间非营利组织会计制度支持结转新账套到新民间非营利组织会计制度

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：新特性, 功能介绍, 进阶技巧
> 作者：Zyl | 阅读：394 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/806153746945349888)

旧民间非营利组织会计制度账套，支持通过账套结转的功能，结转到新民间非营利组织会计制度。

**注意事项：**

1、星辰专业版/标准版才支持，旗舰版不支持账套结转功能

2、账套结转前，**需先备份账套**

**3、账套结转后，结转前的账套将会成为备份账套，结转后的账套启用期间将会是结转前的账套所在期间。结转后的账套无法直接打开看，只能恢复备份账套进行查看。**

4、结转账套，**是业务数据、财务数据一起结转**，不只单独结转财务数据。

3、2025年12期结账后，才支持新旧民非制度切换功能

4、结转说明：

1）结转科目只列出一级科目，明细科目和辅助核算自动结转到对应的新科目下。

2）旧科目转换成新科目，余额全部结转到对应的新科目。

3）期末结转净资产模板按新制度的对应，报表模板内置为新制度报表模板。

4）结转后，2026年及以后新入账的固定资产需手动新增资产类别，选折旧政策为"新增当期计提，清理当期不计提"。

5）【科目关联设置】/【资产类别】/【资产卡片】的科目、以后期间【日记账】/【凭证】/【变动单】的科目、期末结转

6）模板里面非公式设置的科目、首页【财务指标】对应科目、【基础资料】-【凭证模板】对应科目都需按转换后的新科目更新。
