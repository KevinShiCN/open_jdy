# 业务单据生成凭证,凭证中的“单位”的取数来源是什么?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：杨桂平 | 阅读：1796 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/129339770097650214)

【问题描述】

业务单据生成凭证,凭证中的“单位”的取数来源是什么?

【操作流程】

业务单据生成凭证,凭证中的“单位”的取数根据业务凭证模板、科目设置取值。

![](https://vip.kingdee.com/download/01090d8ea2f3ec1749a5951a07eec469e791.png)

1、业务单据生成凭证,凭证中的“单位”根据业务凭证模板中设置的“单位”字段获取;依次点击[账务处理]-[业务凭证模板],如采购入库单的凭证模板,“单位”字段可选择“采购入库单.商品分录.辅助单位”、“采购入库单.商品分录.单位”、“采购入库单.商品分录.基本单位”、“采购入库单.商品分录.辅助基本单位”等,模板中设置了取哪个单位,则生成的凭证带出来就是什么单位。

![](https://vip.kingdee.com/download/0109f5c664245ba9468d9885d5c73ecc4d68.png)

2、若业务凭证模板没有设置计量单位取值,并且科目**没有**挂辅助核算,则根据科目中的计量单位取值,依次点击[基础资料]-[科目],如库存商品科目,若设置了计量单位,则生成凭证带出科目中的单位;

![](https://vip.kingdee.com/download/0109a2efb9ef27e541099a0571894c58b2e4.png)

3、若业务凭证模板没有设置计量单位取值,并且科目**挂了**辅助核算,则取商品资料里对应单位。

![](https://vip.kingdee.com/download/0109c749f51199ac4b4eac0f345b83ab5568.png)

【温馨提示】

录凭证界面单位是不支持手工录入的,只能是根据科目上录入的或者是业务凭证模板取值单据上的携带到凭证上;

如果科目有挂商品辅助核算,选了商品会带出对应商品的单位

![](https://vip.kingdee.com/download/0109f0a02a732e46437e977cbb6080cc83ba.png)
