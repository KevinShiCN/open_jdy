# 常见问题：科目关联设置指的是关联哪些信息呢？![](https://vipfront.s3.cn-north-1.amazonaws.com.cn/emotion/define/64.gif)

![](https://vipfront.s3.cn-north-1.amazonaws.com.cn/emotion/define/86.gif)解析：

1）科目关联设置会列示业务基础资料中与生成凭证相关的所有基础资料信息，包括：客户、供应商、商品、 收入类别、支出类别、辅助资料、部门、银行账户及单据上会录入的费用、折扣等。

2）支持在对应明细项目上设置一一对应的科目，例如，可以在商品上设置某一商品对应的科目，如果设置了， 则生成凭证时，该商品就取对应的科目。

3）支持同一类基础资料设置统一一个对应科目，如直接在商品上方设置库存商品科目，则当商品对应的库存 商品科目为空时，取这个统一设置的对应科目。
