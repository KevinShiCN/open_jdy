# 账务处理系统结账提示:必须先进行资产管理期末结账异常处理?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：杨桂平 | 阅读：1343 | 有用：11
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/129339770080872996)

【故障现象】

账务处理系统结账提示:必须先进行资产管理期末结账 异常处理。

![](https://vip.kingdee.com/download/0109b5ebee8cbecf49a0a2b7baf67a649b00.png)![](https://vip.kingdee.com/download/01094bb86c3bb5a44989aef34b513c9c1e49.png)

【故障原因】

【系统设置】-【系统参数】资产参数设置了与总账连用,必须要资产模块先结账,总账才可以操作结账。

![](https://vip.kingdee.com/download/01098ef304c2d5014c9a8222aee4e4b05316.png)

【解决方案】

1、到资产管理系统先结账,依次点击【资产管理】-【结账】-【结账】;

![](https://vip.kingdee.com/download/010919fde26f6a0245859ba359d5908b352b.png)

2、资产管理结账完成后再到账务处理中结账。
