# 凭证是先审核再复核吗?

【问题描述】

凭证是先审核再复核吗?

【解释说明】

凭证的审核和复核没有固定的先后顺序。‌在系统中,出纳复核和凭证审核可以同时进行,没有明确的先后顺序要求‌。

1、在财务工作中,通常的流程是先审核原始单据,然后制作与复核记账凭证,最后制作会计报表‌

2、这个流程中,审核和复核是两个不同的步骤,但并没有明确规定哪个在前哪个在后。
