# 自动匹配新增的客户 / 供应商在已经生成凭证后，可以删除吗？

【问题描述】![太阳](https://cdn-vip.kingdee.com/statics/emotion/define/86.gif)![太阳](https://cdn-vip.kingdee.com/statics/emotion/define/86.gif)

自动匹配新增的客户 / 供应商在已经生成凭证后，可以删除吗？

【答案解析】![气球](https://cdn-vip.kingdee.com/statics/emotion/define/101.gif)![气球](https://cdn-vip.kingdee.com/statics/emotion/define/101.gif)

不能。

已删除的客户 / 供应商，可以在基础资料里标记为 “禁用” ，不允许删除，以防历史数据查询错误。
