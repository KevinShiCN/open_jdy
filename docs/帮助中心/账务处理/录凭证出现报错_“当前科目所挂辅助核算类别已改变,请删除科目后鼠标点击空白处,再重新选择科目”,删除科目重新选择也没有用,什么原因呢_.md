# 录凭证出现报错:“当前科目所挂辅助核算类别已改变,请删除科目后鼠标点击空白处,再重新选择科目”,删除科目重新选择也没有用,什么原因呢?

> 作者：云社区用户5j6u671 | 阅读：107 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/609551775150074112)

【问题描述】

录凭证出现报错:“当前科目所挂辅助核算类别已改变,请删除科目后鼠标点击空白处,再重新选择科目”,删除科目重新选择也没有用,什么原因呢?

![](https://vip.kingdee.com/download/01098bb629259254488ebee1e1bcadb422b9.png)

【原因分析】

先在凭证上选择了科目,又在科目中对科目的辅助核算设置进行了修改导致。

![](https://vip.kingdee.com/download/0109c5f88c5b261844089895dda363911551.png)

【解决方案】

先清空科目信息,再重新录入科目并选择对应的辅助核算。

注意:如果删除科目直接录入科目不行。

![](https://vip.kingdee.com/download/01098a9e0ea6abd3450eb5d363316d1f1a73.png)
这样辅助核算带出的明细就会是空。

![](https://vip.kingdee.com/download/0109e4967621bef74d88823a72bd8f17f41c.png)

需要删除科目以后,鼠标点击其他空白地方,科目置空以后才行。

![](https://vip.kingdee.com/download/0109fddb58a1f9cc4bbc9894a8c204f48179.png)这样辅助核算才会有明细。

![](https://vip.kingdee.com/download/01093a7b9d1b8cfe4ae8acfef0b9dd1c5943.png)
保存凭证即可。

![](https://vip.kingdee.com/download/0109d17fddd0e4304b8784aa48f63a1e1f6a.png)
