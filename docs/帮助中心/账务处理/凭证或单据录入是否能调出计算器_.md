# 凭证或单据录入是否能调出计算器?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：财税李炳全 | 阅读：422 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/568601875503904512)

【问题描述】

凭证或单据录入是否能调出计算器?

        【解决方案】

**一、凭证界面**

录入凭证时可以调出计算器使用,快捷键ctrl+F11即可调出计算器

![](https://vip.kingdee.com/download/01090d73269c74bc4247af7092ecb2a9212d.png)也可以直接点击金额栏,点击计算器图标,或者在金额栏输入数字和运算符号,可直接调出计算器,运算后点击<Enter>,会将结果填充到金额栏

![](https://vip.kingdee.com/download/0109cca116b7e39642ef9552264e9c812f4d.png)

**二、单据界面**

在录入业务单据时,输入数字+“+-*/”时,可以调出计算器,如下图在数量栏输入“10+”会调出计算器页面

![](https://vip.kingdee.com/download/0109be70546c3087404d8c0c4fc7a8a25668.png)

运算后,点击<Enter>,会将运算的结果回填到数量栏

![](https://vip.kingdee.com/download/0109d38e847a8f884e6b8f4e0969a2d0e104.png)
