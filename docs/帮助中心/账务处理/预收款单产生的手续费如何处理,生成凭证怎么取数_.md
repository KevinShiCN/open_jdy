# 预收款单产生的手续费如何处理,生成凭证怎么取数?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题

【问题描述】

预收款单产生的手续费如何处理,生成凭证怎么取数?

【解释说明】

1、预收款单产生的手续费在预收款单上直接录入手续费即可。

备注:单据审核以后不用生成其他支出单。因为收款金额+手续费=预收总金额,如果在生成其他支出单,账户金额会减少。实际账户在预收的时候,手续费部分就没有进账户。

![](https://vip.kingdee.com/download/010951c84e7fbd86412d86926d1b2ba65c77.png)
2、含有手续费的预收款单,生成凭证时手续费在预收款单凭证上直接生成。
