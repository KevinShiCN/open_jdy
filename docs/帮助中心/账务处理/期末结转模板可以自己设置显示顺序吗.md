# 期末结转模板可以自己设置显示顺序吗

> 作者：杨桂平 | 阅读：17 | 有用：0

【问题描述】

期末结转模板可以自己设置显示顺序吗

【解释说明】

新增的自定义模版会排到最前面,可以拖动排序,但是下次进入或者刷新页面还是会按照系统默认的顺序排序。
