# 采购暂估业务,如何设置凭证模板生成凭证?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：shaoling | 阅读：2559 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/683480198993559808)

【问题描述】

采购暂估业务,如何设置凭证模板生成凭证?

【解释说明】

暂估入库生成凭证分成两部分,一个是采购入库单暂估时生成凭证,一个是采购发票到了做暂估调整单,暂估调整单生成凭证

【解决方案】

第一部分:

暂估的采购入库单生成凭证按正常流程走是在【账务处理】——【生成凭证】进入生成凭证页面

![](https://vip.kingdee.com/download/01099a538049c40342998e245f74ab617903.png)

生成凭证页面左侧选择【采购管理】——【采购入库单】里生成

![](https://vip.kingdee.com/download/0109585bcd5bcbdd43ea872999f8f11cf632.png)如果生成凭证过程中没有对应业务凭证模板,则需要先设置选择好业务凭证模板,

可在【账务处理】——【业务凭证模板】进入业务凭证模板页面

![](https://vip.kingdee.com/download/0109cc4f0b4c12cb4a9280da6d339e0bae5b.png)在【采购管理】——【采购入库单】根据需要新增一个模板

![](https://vip.kingdee.com/download/0109210783b7765d43c1a5e2787f139a7a88.png)
新增模板后根据需要设置对应的信息,如暂估凭证跟正常入库凭证模板区分可通过添加“适用条件”添加模板取值条件,字段“是否暂估”,条件“等于”,值“是”

![](https://vip.kingdee.com/download/01095d62c07f0ca24436a275e09ab2f322f7.png)

一般暂估入库的凭证分录为:借:库存商品 贷:应付账款暂估科目

因此,在模版取数设置科目时,借方为“商品.库存对应科目”  贷方为模版自定义科目,可直接指定科目为“应付账款暂估”

![](https://vip.kingdee.com/download/010930b084e9f32447c7b328a906c14c4521.png)

业务凭证模板金额取值,借贷均取数为“{商品分录.入库成本}”

![](https://vip.kingdee.com/download/01095880c074189a4744b3d8dc9f24805a5a.png)
业务凭证模板设置保存成功后,生成凭证即可带出或选择对应模板

第二部分:

暂估调整单生成凭证是在【账务处理】——【生成凭证】进入生成凭证页面

![](https://vip.kingdee.com/download/01099709c921c6f640c99a08b67066be34d6.png)生成凭证页面左侧选择【采购管理】——【暂估调整单】里可选择【差额生成】模板或【单到冲回】模板生成记账凭证

![](https://vip.kingdee.com/download/0109dc105228b914405c84e001aa209cbb81.png)
注意:

1、【单到冲回】是指冲销原暂估入库凭证后按暂估调整后的发票金额生成入库凭证。选择此模板生成凭证,科目不能合并,合并生成会导致科目金额合计显示为差额。

2、暂估调整单的业务凭证模板说明:新建的账套系统已预置暂估调整单凭证模板,默认模板为“差额调整”(推荐方式)。

3、如老用户开通功能暂估功能后可通过社区下载凭证模板导出使用:[最新业务凭证模板](https://vip.kingdee.com/knowledge/specialDetail/280269406134071040?category=280270771396436224&id=280273946451373824&type=Knowledge&productLineId=35&lang=zh-CN)

社区下载对应模板后可在【账务处理】——【业务凭证模板】进入业务凭证模板页面

![](https://vip.kingdee.com/download/0109adb9761c937d4c55a9a69c030bf81270.png)在左侧的【采购管理】——【暂估调整单】右上角【导入】操作导入模板使用
