# 凭证套打，选择正确的打印纸对应的打印模板后，打印出来的效果缩小了

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题, 进阶技巧
> 作者：Zyl | 阅读：1320 | 有用：9
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/70907183525083904)

问题描述：

凭证套打，选择正确的打印纸对应的打印模板后，打印出来的效果缩小了

解决方案：

凭证打印时，打印预览界面点击打印，左边“显示更多”打开，取消勾选“缩放”后面的适合页面大小
