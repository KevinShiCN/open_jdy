# 查凭证界面如何筛选报销单生成的凭证?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：72 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/655213373541140992)

【问题描述】

查凭证界面如何筛选报销单生成的凭证?

【解决方案】

先在【账务处理】-【查凭证】在查凭证列表开启显示业务单据编号字段。

![](https://vip.kingdee.com/download/0109c5487c8ba42d49739d37b8dd3ad228fe.png)
报销单的单据编码包含BXD,通过“小漏斗”筛选功能可以筛选出报销单对应的凭证,如下图所示:

![](https://vip.kingdee.com/download/0109e6d4377ba1de436b92bf3dfabdcd2ed4.png)
