# 凭证整理提示没有需要整理的凭证是什么意思?

【问题描述】

凭证整理提示没有需要整理的凭证是什么意思?

【解释说明】

凭证整理的功能是用来处理凭证出现断号,或需要按凭证日期重排凭证编号的功能。

如果操作凭证整理时提示没有需要整理的凭证表示凭证没有断号的情况,不需要整理。
