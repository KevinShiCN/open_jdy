# 如何按辅助核算结转损益?

【问题描述】

如何按辅助核算结转损益?

【解决方案】

账务处理-期末处理-期末智能结转-点击结转损益右上角的编辑,勾选 “按辅助核算分别结转生成”

![](https://vip.kingdee.com/download/010994a3119c1171417b8db278b402daa9be.png)
此处,假设损益类科目以及本年利润都挂了部门与商品的辅助核算,参数勾选了按辅助核算按“商品”。

结转凭证效果:商品A与商品B 会分开结转成2个凭证。
