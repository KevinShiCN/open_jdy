# 星辰的凭证附件上传的大小有限制吗？

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题, 快速入门
> 作者：Daisy_ | 阅读：1496 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/243654440652125952)

问题描述：

星辰的凭证附件上传的大小有限制吗？

解决方案：

可通过【系统设置】-【系统参数】-【公共参数】中的其他信息参数，设置单个资料/单据/凭证的附件个数及单个附件的大小。

系统可设置的最大值：凭证支持的附件数不能超过200个，单个附件的大小不能超过50M。
