# 金蝶云星​辰，批量引入科目提示现金流量项目未找到，父级科目未设置，是哪里的问题呢，详情可以查看截图：

> 作者：小橙子和小柠檬 | 阅读：700 | 有用：8

金蝶云星辰，引入科目提示现金流量项目问题，但是导入的科目表实际是设置了现金流量项目的，还有父级科目也有设置，但是导入报错，是哪里的问题呢，详情可以查看截图：

![image.png](https://vip.kingdee.com/download/0109cf3b52e5f7224fa083cd2f9417762977.png)

![image.png](https://vip.kingdee.com/download/010922e8008d4ac34613953c40449390aed2.png)

解决方案：

1、导入明细科目之前，需要先在系统将明细科目设置好，再批量导入明细科目；

2、提示现金流量项目未找到，需要检查科目导入模板里面对应科目的借贷方现金流量项目和金蝶云星辰系统的现金流量表报表项目名称对应关系，如果科目导入模板里面对应科目的借贷方现金流量项目和金蝶云星辰系统的现金流量表报表项目名称不一致，就会提示以上报错问题；需要确定要以哪个为准，从而修改对应现金流量项目后即可导入成功；
