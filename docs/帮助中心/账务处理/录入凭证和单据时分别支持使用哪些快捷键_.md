# 录入凭证和单据时分别支持使用哪些快捷键?

> 作者：杨桂平 | 阅读：642 | 有用：0

【问题描述】

录入凭证和单据时分别支持使用哪些快捷键?

【业务说明】

录入凭证和业务单据时部分操作可以通过键盘上的快捷键代替完成,提高工作效率。

注意:所有的快捷键是系统默认设置的,不支持自定义设置。

**1、财务模块**

在新增录凭证界面,点击右上角的键盘图标可显示系统常用快捷键功能
