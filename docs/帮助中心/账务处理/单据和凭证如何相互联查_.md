# 单据和凭证如何相互联查?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：Schatz | 阅读：2212 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/641442154585497344)

【问题描述】

单据和凭证如何相互联查?

【业务场景】

按单据生成的凭证怎么查,能否根据凭证字号查询单据信息

【操作步骤】

**一、按凭证字号查单据**

在查询对应单据列表时,点击右上角的齿轮按钮,打开列设置界面。在此界面中,找到“凭证字号”选项并开启显示。完成设置后,返回单据列表界面,即可查看单据是否已生成凭证以及生成的具体凭证信息。

![](https://vip.kingdee.com/download/01091ca8f4f1dfc54fa189b0b709dd77ff86.png)

也可以在“常用条件过滤”通过凭证号查询单据

常用条件下拉<新增方案>

![](https://vip.kingdee.com/download/010998406cc6b0554ad390fbf51d10ade008.png)

选择凭证字号,输入凭证号查询即可查询出对应凭证号的单据

![](https://vip.kingdee.com/download/0109d2183b66232d4d8d81a1423fe1d7cd58.png)

**二、按单据查询凭证信息**

1、在单据列表或详情上查看

1)单据列表上开启显示“凭证字号”字段,查询单据后可以看凭证字号字段上是否为空,为空表示没有生成,显示暂存凭证表示生成失败,有凭证字号表示生成成功,可以点击查看

![](https://vip.kingdee.com/download/0109f68cf29025f44b8292515a0b8fe2d194.png)
2)单据详情界面查看

打开具体单据后,点击右上角的“查凭证”按钮,即可联查到该单据生成的凭证(有时可能需要点击“更多”以查找此功能)。

![](https://vip.kingdee.com/download/01099590ab7971b2422d845bf1791a93cef9.png)

如果单据已生成凭证,点击“查凭证”按钮之后,会直接跳转到相应的凭证界面,且生成多少张凭证就会打开多少个凭证页签。例如,销售出库单分别生成了收入和成本两个凭证,点击“查凭证”后,将直接打开这两个凭证的页签。

![](https://vip.kingdee.com/download/0109ca9f1b3c9de7417d83b33e796930f94b.png)

如果单据没有生成凭证,会直接提示无关联数据。

![](https://vip.kingdee.com/download/0109ebe0efefc8834965a76473a3ea3bcc43.png)

2、查凭证界面根据单据编号查询凭证

在财务处理的查凭证界面中,设置查询全部期间的凭证。然后,点击业务单据编码字段的漏斗图标,通过设置单据编码来查询对应的凭证。

![](https://vip.kingdee.com/download/0109f4c1fb61b5614f7499e756b671deb59b.png)

3、通过机制凭证条件过滤查询哪些凭证是单据生成的,如勾选过滤进销存业务单据

![](https://vip.kingdee.com/download/010933e77b722eeb49f18eec25f720154f1f.png)

4、生成凭证界面过滤单据编号查看

在财务处理的生成凭证界面中,设置单据范围查询条件时,应同时包括未生成和已生成的单据。然后,点击单据编码位置进行漏斗搜索,查看是否有对应的凭证字号信息。

![](https://vip.kingdee.com/download/01099bae1be5877d44169deb737c76d45043.png)

但请注意,对于销售出库单、销售退货单等单据,收入和成本可能是分开生成的。因此,需要分别在左侧点击到销售管理和库存管理-结转出库成本来查询。

![](https://vip.kingdee.com/download/0109dca875397522425e83ed48de4dcb2d9b.png)
