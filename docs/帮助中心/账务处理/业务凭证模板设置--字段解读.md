# 常用功能：业务凭证模板设置--字段解读![](https://vipfront.s3.cn-north-1.amazonaws.com.cn/emotion/define/19.gif)

解析：以销售出库单为例，先在右边单据列表中单击【销售管理】→【销售出库单】，再单击【新增】按钮。打开 业务凭证模板新增界面，如下图所示：

![上传图片](https://vip.kingdee.com/download/0100796efb258c5b46cd8174cc5103b1f639.png)

相关参数及设置说明如下：

![](https://vip.kingdee.com/download/0109bec401dfe94949a09163674cdafbe095.png)

注：

1）设置凭证模板信息完成后，保存即可在业务凭证模板列表看到对应模板，

2）凭证模板设置为默认模板时，在业务单据生成凭证向导时会自动取默认凭证模板，智能核算中心业务单据 生成凭证取默认凭证模板

3）禁用的凭证模板，在生成凭证时不能选择。
