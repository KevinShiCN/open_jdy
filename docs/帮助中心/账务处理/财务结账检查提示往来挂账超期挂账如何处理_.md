# 财务结账检查提示往来挂账超期挂账如何处理?

【问题描述】

财务结账检查提示往来挂账超期挂账如何处理?

【解释说明】

往来挂账这块检验规则是,首先账套从启用期间开始计算,到当前期间要超过1年时间,之后检查对应设置的科目对应的明细科目,是否本期有期末余额,如果有期末余额,就检查从当前期间开始到最近一年所有期间,是否有做过贷方发生额,如果没有做贷方数据,则警告;但是不影响结账。

比如我的账套是24年1期启用,25年1期之前都没有录入过往来,但是25年1期做了一张检查项其他应收款的检查科目【其他应收款】凭证,有余额,25年1期结账前检查就会有预警提示,实际不影响我的结账。
