# 怎么设置凭证或单据的制单人和审核人是否可为同一人?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：云社区用户6Rw2283 | 阅读：3243 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/598317990861342208)

【问题详情】

怎么设置凭证或单据的制单人和审核人是否可为同一人?

【业务场景】

制单与审核属于不同的职务,需由不同人员担任以实现相互监督,因此根据会计规范与内部控制要求,‌制单人和审核人原则上不应为同一人。

在系统中凭证可以通过系统参数“凭证审核与制单不能为同一人”进行控制;单据则根据单据操作控制中的“审核人与制单人可为同一人”参数进行控制。

【操作步骤】

1、如何设置凭证的制单人和审核人是否可以为同一人

依次点击【系统设置】-【系统参数】-【财务参数】,在财务参数中根据需要进行勾选“凭证审核与制单不能为同一人”参数。

![](https://vip.kingdee.com/download/010939943f35e11a473bb166c2eb352b4ea7.png)
1.1 勾选该参数表示凭证的审核人与制单人不能为同一个人,否则审核的时候会提示“凭证审核人与制单人不能为同一人”。
![](https://vip.kingdee.com/download/0109b15011a914ce451a940e30d02c976ca5.png)

1.2 不勾选该参数则不会进行控制,A用户新增的凭证A用户自己可以审核。

![](https://vip.kingdee.com/download/0109504016224b684138a9f2eda5e655ceeb.png)

2、如何设置单据的制单人和审核人是否可以为同一人

不同业务类型的单据是分别进行控制的,通过【系统设置】-【单据操作控制】中选择需要控制的单据设置“审核人与制单人可为同一人”参数。
![](https://vip.kingdee.com/download/0109d742ce5fbb61454f89b8c51d87074d5e.png)
2.1 关闭该参数表示单据的审核人与制单人不能为同一人,否则审核的时候会提示“当前单据操作人与制单人相同,不允许审核!”。

![](https://vip.kingdee.com/download/01099e4c8787abbd4f4d98b39ca30930cfd2.png)
2.2 开启该参数则表示单据的审核人与制单人可以为同一人,A用户新增的单据A用户自己可以审核。

![](https://vip.kingdee.com/download/010956f3309141ea46c4ab21755ef32dd755.png)
