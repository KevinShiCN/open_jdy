# 多栏账界面解析

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：功能介绍, 快速入门
> 作者：陈娇智 | 阅读：881 | 有用：5
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/23343)

问题描述：

多栏账界面解析

解决方案：

当选择的科目设置的是辅助核算时，界面如下图所示：

![上传图片](https://vip.kingdee.com/download/0100c4f5dcf0f64b4858b2cde4cd4f050dab.png)

1、多栏账相关参数说明如下：

1）只显示最明细级：显示最明细级科目的数据，上级科目不再显示。

2）只显示下级：如果科目设置了三级明细，选择的科目是一级科目时，显示二级科目明细数据。与只显示最明细级科目互斥

3）显示明细栏余额：明细栏科目会显示一行期末余额

4）无发生额且余额为0不显示：同时满足无发生额且无余额的科目都不显示出来。

5）发生额为0不显示本期合计、本年累计：没有发生额的科目不显示本期合计、本年累计两行。

2、联查凭证

在查询结果列表中，单击“凭证字号”链接，可以查看对应的凭证信息。
