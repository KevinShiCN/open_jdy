# 其他软件的凭证可以直接同步到星辰系统吗?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题

【问题描述】

其他软件的凭证可以直接同步到星辰系统吗?

【业务情景】

如金蝶云星空、精斗云云会计、账无忧等第三方产品系统中有凭证数据,需要同步到星辰中。

【解释说明】

目前**不支持**直接将第三方软件的凭证数据直接同步到星辰系统,若有凭证需要同步到星辰建议通过Excel表格导入的方式;或在创建新账套时使用第三方导入的方式升级到星辰产品中,具体操作可参考:

①[导入凭证如何操作?](https://vip.kingdee.com/knowledge/670433132361070080?productLineId=35&isKnowledge=2&lang=zh-CN)
