# 单据生成凭证后为什么凭证的汇率和单据的汇率不一样?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：杨桂平 | 阅读：49 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/787485186060672000)

【问题描述】

单据生成凭证后为什么凭证的汇率和单据的汇率不一样?
![](https://vip.kingdee.com/download/0109bba0802d7c054773a288f75f8fdafc52.png)
【解决方案】

凭证生成方式选择“汇总生成”,并且勾选了“合并科目”-“汇率不同,允许合并反算”时,不同汇率的单据汇总生成凭证后,就会根据本位币金额和原币金额反算汇率;若需要保证单据汇率和凭证汇率一样,则不要勾选“汇率不同,允许合并反算”参数,然后再重新生成凭证时汇率不同的分录就不会合并,保证单据汇率和凭证汇率一样。
