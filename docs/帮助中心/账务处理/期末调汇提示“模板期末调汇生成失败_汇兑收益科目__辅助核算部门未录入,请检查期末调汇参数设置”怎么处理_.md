# 期末调汇提示“模板期末调汇生成失败:汇兑收益科目**辅助核算部门未录入,请检查期末调汇参数设置”怎么处理?

【问题描述】

期末调汇提示“模板[期末调汇]生成失败:汇兑收益科目[**]辅助核算[部门]未录入,请检查期末调汇参数设置”怎么处理?
【情景描述】
期末调汇无法正常生成凭证,系统提示汇兑收益科目[**]辅助核算[部门]未录入,请检查期末调汇参数设置
【操作说明】
1.财务处理>期末智能结转>期末调汇
![](https://vip.kingdee.com/download/0109bd5c6f1f323742e1921ab1106900aab7.png)
2.点击期末调汇结转右上角设置
![](https://vip.kingdee.com/download/0109bbea1f7af8a54213a61d1fdf68012ec1.png)
选择部门后,再操作生成凭证

![](https://vip.kingdee.com/download/0109c39fab1996714181b3fe881800affb74.png)
