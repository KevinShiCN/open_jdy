# 汇率是否支持录入负数?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：shaoling | 阅读：98 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/713195589357520384)

【问题描述】

汇率是否支持录入负数?

【业务场景】

凭证录入时,借方红字发生,汇率需要录入小于0,但现在汇率录入负数无法保存凭证

![](https://vip.kingdee.com/download/0109bfd6af1c7d4042698e7bd44f1863cac7.png)【解释说明】

汇率**不支持**录入负数,负数汇率在现实中无意义,因为货币价值不可能为负。

若录凭证时,外币科目需体现借贷方金额为负数,将原币金额录入为负数即可。

![](https://vip.kingdee.com/download/0109e990a61378b24317a6607e439435c8ac.png)
