# 生成凭证页面显示的单据不全排查思路

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：shaoling | 阅读：537 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/693626617721246976)

【问题描述】

生成凭证页面显示的单据不全

【业务场景】

做了单据后,生成凭证页面不显示对应的单据

以销售出库单为例,如单据列表显示有20张单据,生成凭证页面显示只有5张
![](https://vip.kingdee.com/download/01094f8043d6984c42f88ad0bb4a59672aee.png)![](https://vip.kingdee.com/download/0109c7138abd9fef41efb05ee9b96bd20849.png)
【解释说明】

做了单据在生成凭证里不显示可能的原因如下:

1、单据未审核,在生成凭证页面不显示

2、单据金额为0,生成凭证页面“是否显示金额为0的单据”未选择“是”

3、生成凭证页面“单据范围”选择不全

4、生成凭证界面筛选的日期和单据日期不一致

5、参数设置方面导致显示不全问题

6、业务单据生成的日记账,日记账生成凭证页面不显示

7、对于订单类的单据,比如销售订单、采购订单、生产任务单、付款申请单等类型的单据,不会显示在生成凭证中,此类型的单据类似合同或者计划单之类,不会对库存产生实质影响,也不需要生成凭证

8、生成凭证中的资产新增里不显示新增的资产

【操作步骤】

1、检查单据是否未审核,如未审核单据需要审核后,再去生成凭证

![](https://vip.kingdee.com/download/0109e3e0096799ca4b199fb4bfb4c3912ff0.png)
2、检查生成凭证页面“是否显示金额为0的单据”是不是未选显示“是”

![](https://vip.kingdee.com/download/0109bd72e2ef4e60497b88d95e1153a2d956.png)3、检查生成凭证页面“单据范围”是不是已经全选

![](https://vip.kingdee.com/download/010924f9146a3ee8499297166b8ed729b854.png)
4、检查生成凭证页面的日期范围是否选择准确

![](https://vip.kingdee.com/download/0109a1a6805479614038abfedb444e3c396b.png)5、检查参数设置

①**单据数据过大,生成凭证页面显示不全**

可在生成凭证页面的右上角【参数设置】
![](https://vip.kingdee.com/download/0109c7778c5c8ee349d69523f2a1d62bfb0d.png)
“生成凭证时列表显示分录最大行数的”的参数,这里数值改大,最大20000,保存参数操作

![](https://vip.kingdee.com/download/01095e255579fef74df08c49deff9ed6850c.png)
若超过20000条生成凭证时建议分日期分批次生成凭证

②**超收转预收或者超付转预付**

超收转预收或者超付转预付生成的预收款单和预付款默认是不生成凭证,源单生成凭证入账即可

若超收超付生成的预收预付款单需要生成凭证,可在生成凭证页面右上角【参数设置】![](https://vip.kingdee.com/download/0109db5eba89554645ec86976c7e99ec6bf5.png)
“超收超付生成设置”参数里选择预收单或预付款单,保存参数

![](https://vip.kingdee.com/download/010982b2920bd5794ecdb55a31540ab30be9.png)

注意:源单比如收款单、销售出库单、付款单的生成凭证模板上取值要把超收的部分扣减。

③**调拨出库单、调拨入库单、移仓单、组装单、拆卸单、核销单(预收冲应收)、核销单(预付冲应付)、核销单(退货冲销货)、核销单(退货冲购货)、调汇单(应收)、调汇单(应付)、调汇单(账户余额)类单据在生成凭证页面不显示**

可在生成凭证页面右上角【参数设置】

![](https://vip.kingdee.com/download/010945b5e03f06f642df902c7cd78c57e95e.png)

“单据范围设置”里检查是不是没有勾选生成凭证

![](https://vip.kingdee.com/download/0109c8731ef7d4c54eb0b50ec7958ff7d5f8.png)
④**库存结转成本生成凭证不显示单据可查看**

[库存类单据生成凭证界面找不到单据是什么原因?](https://vip.kingdee.com/knowledge/588533211836215808?productLineId=35&isKnowledge=2&lang=zh-CN)

6、日记账生成凭证显示不全,检查是不是由业务单据生成的日记账,业务单据生成的日记账不需要(不支持)生成凭证,对应凭证已由业务单据生成凭证

可检查【系统设置】-【系统参数】![](https://vip.kingdee.com/download/0109cea3d9264c0c40678239ebf416aecb3f.png)

【出纳参数】里出纳管理参数里是不是选择的单据登账方式

![](https://vip.kingdee.com/download/01093be9224d79014b76a744ca8f6743a63f.png)
7.生成凭证中的资产新增里不显示新增的资产

需要检查新增的卡片中的入账日期和生成凭证的筛选期间是否在同一个期间,只有在同一期间卡片才会显示在生成凭证列表中

![](https://vip.kingdee.com/download/01094494b0eb02504c29b9b086474299449b.png)
![](https://vip.kingdee.com/download/01094ba1b095b0b24d4d843576d760c74080.png)
