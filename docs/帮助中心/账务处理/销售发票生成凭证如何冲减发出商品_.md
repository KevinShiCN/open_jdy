# 销售发票生成凭证如何冲减发出商品?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：lykk | 阅读：1076 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/649777703230191872)

【问题描述】

销售发票生成凭证如何冲减发出商品?

【适用版本】

金蝶云星辰旗舰版

【业务场景】

销售出库单生成凭证为发出商品,发票收到后,需要冲减发出商品结转收入、成本;

【解决方案】

1、若销售发票成本结转为“单独结转”,可修改【库存管理】-【成本结转】【销售发票】的凭证模板;

![](https://vip.kingdee.com/download/0109eba2a9f8e55f4af3a2c832fa37218317.png)

借方科目:

科目来源:商品.成本对应科目

原币、本位币取值:{商品分录.出库成本本位币}

![](https://vip.kingdee.com/download/0109d90f41755c674c7bade256b92ae8558f.png)

贷方科目:

科目来源:模版自定义科目

科目:发出商品

原币、本位币取值:{商品分录.出库成本本位币}

![](https://vip.kingdee.com/download/0109e7871498289a4c7c8a4f7c71d201098d.png)

保存模板后,则可对单据生成凭证;生成的凭证为,借:商品成本对应科目  贷:发出商品

![](https://vip.kingdee.com/download/0109ba32f21cc5de451ca0e796159a05ea1c.png)

2、若销售发票成本结转为“同时结转”,可修改【销售管理】-【销售发票】的凭证模板;

![](https://vip.kingdee.com/download/0109ad525ac082074fc993418b9b77c618be.png)

借方科目:

①客户.应收账款科目:

原币取值为:{商品分录.价税合计}-{商品分录.本次收款金额}

本位币为:{商品分录.价税合计本位币}-{商品分录.本次收款金额本位币}

②账户管理.对应科目:

原币取值为:{收款信息.收款金额}

本位币为:{收款信息.本位币金额}

③客户.预收账款科目:

原币取值为:{抵扣预收款}

本位币取值为:{抵扣预收款本位币:}

④收款抹零:模板自定义科目-财务费用手续费

原币取值为:{收款抹零}

本位币取值为:{抹零金额本位币}

⑤商品.成本对应科目

原币、本位币取值为:{商品分录.出库成本本位币}

贷方科目:

①商品.收入对应科目:

原币取值:{商品分录.实际不含税金额}

本位币取值:{商品分录.实际不含税金额本位币}

②税金科目:模版自定义科目 -应交税费进项税

原币取值:{商品分录.税额}

本位币取值:{商品分录.税额本位币}

③客户.预收账款科目

原币取值:{转预收金额}

本位币取值:{转预收金额本位币}

④冲减发出商品科目:模版自定义科目-发出商品

原币、本位币取值为:{商品分录.出库成本本位币}

![](https://vip.kingdee.com/download/01091ec3cf56fada44abbecbf249cb9f529e.png)

保存模版后,相应的单据则可进行生成凭证;

![](https://vip.kingdee.com/download/010974c8b783d0e1410aaea86c223b18238e.png)

附件提供了单独结转成本和同时结转成本的凭证模版可下载导入使用(供参考)
