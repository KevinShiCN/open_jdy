# 新增的卡片，生成凭证如何设置科目

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题, 快速入门
> 作者：Zyl | 阅读：1256 | 有用：8
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/63677582281276928)

问题描述：

新增的卡片，生成凭证如何设置科目

解决方案：

卡片新增时，卡片界面录入固定资产、累计折旧、减值准备以及折旧费用分配、减值准备对方科目；固定资产的对方科目，通过资产变动方式中设置
