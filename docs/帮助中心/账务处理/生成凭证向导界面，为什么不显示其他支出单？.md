# 生成凭证向导界面，为什么不显示其他支出单？

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：快速入门, 常见问题
> 作者：Daisy_ | 阅读：3817 | 有用：11
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/259992818134010624)

问题描述：

由采购入库/销售出库单费用生成的其他支出单，在生成凭证向导界面为什么没有显示？

解决方案：

是否通过其他支出单生成凭证，是通过参数进行设置的：

【账务处理】-【生成凭证】-【参数设置】

![](https://vip.kingdee.com/download/01009794a3835bbc4d4c8dc6bbbcd9a141a5.png)

1.采购费用分摊勾选了【采购入库单】时，采购入库单费用生成的其他支出单且分摊状态为已分摊的，由采购入库单生成凭证，未分摊的，由其他支出单生成凭证；

![](https://vip.kingdee.com/download/0100549cfe990e7b47f1bc9f96dc2404dcd2.png)

2.采购费用分摊勾选了【其他支出单】时，采购入库单费用生成的其他支出单，分摊状态为已分摊/未分摊，均由其他支出单生成凭证;

![](https://vip.kingdee.com/download/0100901caa03219040b5909997f18dc88a0c.png)

3.销售费用分摊勾选了【销售出库单】时，销售出库单费用生成的其他支出单且分摊状态为已分摊的，由销售出库单生成凭证，未分摊的，由其他支出单生成凭证；

![](https://vip.kingdee.com/download/010075b2db17af8b4b17b9f7e8fefde5beaa.png)

4.销售费用分摊勾选了【其他支出单】时，销售出库单费用生成的其他支出单，分摊状态为已分摊/未分摊，均由其他支出单生成凭证。

![](https://vip.kingdee.com/download/0100d56f56fe16f44a098f197c8a944763b0.png)
