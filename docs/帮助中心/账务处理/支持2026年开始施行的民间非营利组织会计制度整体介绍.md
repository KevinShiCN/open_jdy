# 1 产品概述

## **1.1 产品介绍**

    星辰支持2026年开始施行的民间非营利组织会计制度，通过内置科目、结转净资产模板、三大报表模板，全面适配新民间非营利组织会计制度。

**1.2 适用版本**

星辰标准版/专业版/旗舰版

# **2 特性与亮点**

2.1 新建账套支持选新民间非营利会计制度

2.2 旧民间非营利组织会计制度账套支持直接切换新制度的报表模板

2.3 星辰专业版/标准版旧民间非营利组织会计制度账套，结转新账套支持结转到新民间非营利组织会计制度。

# **3 功能清单**

1）新建账套支持选新民间非营利会计制度

2）期末结转净资产支持最新的结转对应关系

3）支持新制度对应的资产负债表、业务活动表、现金流量表。

4）旧民非制度支持直接切换新民非制度的报表模板

5）星辰专业版/标准版旧民间非营利组织会计制度账套，结转新账套支持结转到新民间非营利组织会计制度。
