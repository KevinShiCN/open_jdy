# 修改总账或资产期间，系统提示：“总账启用期间不允许改在资产启用期间之后”或“资产的启用期间应该等于总账的当前期间”，应该如何处理？

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题, 进阶技巧
> 作者：Daisy_ | 阅读：1483 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/255017720453021440)

问题描述：

需要将总账或资产的当前期间改为系统默认期间的以后期间，但是无论修改总账期间还是资产期间，都提示：“总账启用期间不允许改在资产启用期间之后”或“资产的启用期间应该等于总账的当前期间”，无法进行修改 ，应该如何处理？

解决方案：

出现无法修改启用期间是由于资产和总账一起结账了，修改前需要将此参数取消勾选。

依次点击：【系统设置】-【资产参数】-不勾选参数【与总账一起结账】

![](https://vip.kingdee.com/download/0100070ed351cfa345e1aa6e366942ec3d35.png)

如需继续使用资产和总账一起结账功能，可修改启用期间后，将此参数重新进行勾选即可。
