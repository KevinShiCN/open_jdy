# 如何修改/删除以前期间的凭证?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：云社区用户5j6u671 | 阅读：1419 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/611726489662152192)

【问题描述】

如何修改/删除以前期间的凭证?

【解决方案】

系统结账后已结账期间的凭证是不允许修改/删除的,需要反结账到对应期间才能操作,反结账前建议先备份账套。

1、备份账套

登录管理员账号点击【系统设置】-【备份与恢复】-【开始备份】完成数据备份(为了防止操作失误或者删除后又想要恢复,建议先备份)。
![](https://vip.kingdee.com/download/010927e6d2b5fb224552bda7b528a0146ab8.png)
2、财务系统反结账

点击【账务处理】-【期末结账】,点击“反结账”将财务反结账到要修改/删除凭证的期间。
![](https://vip.kingdee.com/download/010952a5e24a36634b18a1171f90f6a5fa3a.png)
3、修改/删除凭证

3.1 反审核凭证:凭证如果审核了需要先取消审核,在【账务处理】-【查凭证】打开凭证列表界面,勾选对应凭证点击审核下的【取消审核】按钮。

![](https://vip.kingdee.com/download/0109d04fae61e97148ebb692951fbe931eea.png)3.2  修改凭证:双击打开需要修改的凭证,修改成功后保存即可。

![](https://vip.kingdee.com/download/0109211261a00931495f901d69704d8ce699.png)

3.3 删除凭证:勾选需要删除的凭证,点击工具栏的“删除”功能即可。

![](https://vip.kingdee.com/download/0109dafbdb99a36a48b6b8781f4e4842c1dd.png)
【温馨提示】

1、以前年度的凭证也需要反结账到以前年度后才能修改/删除。
