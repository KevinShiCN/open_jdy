# 财务期末结账与反结账操作指南

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：功能介绍, 快速入门
> 作者：陈娇智 | 阅读：5810 | 有用：7
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/23359)

问题描述：

解决方案：

1、单击【账务处理】→【结账】，进入“期末结账”页面。如下图所示：

![上传图片](https://vip.kingdee.com/download/0100289db5487b544496aa7692067613b4c7.png)

2、在进行期末结账前，系统提供“结账前检查”功能，可以对期末结账前相关业务进行检查。单击右上角的“检查项设置”可以对检查的项目进行设置和调整。如下图所示：

![上传图片](https://vip.kingdee.com/download/010074f619483bc34c09a52fb84f15c8904e.png)

1）在检查项目设置中修改相关参数，会同步改写系统参数中对应的参数设置；

2）如果对相关的业务不关注，无须在结账前检查，可以单击对应项目后面的开关按钮![](https://vip.kingdee.com/download/0100dc7e564d438c4514ba08840f34bf8b38.png)

将对应检查项目关闭。

3）系统根据科目体系预置了对应检查项目的科目，可以根据业务需求进行调整设置。

3、单击期末结账页面的“结账前检查”，系统根据设置进行相关检查，并给出检查结果，如下图所示：

![上传图片](https://vip.kingdee.com/download/010041d49b961c2a45bea990ff983b16a4ac.png)

单击检查项目，可以进入异常处理界面，进行相关业务处理。

4、单击“结账”，进入期末结账界面，选择需要结账的期间，单击“结账”即可。如下图所示：

![上传图片](https://vip.kingdee.com/download/0100044aee6b8a714634b4926e623368d7be.png)

5、系统提供反结账功能，操作与结账一致。
