# 生成凭证参数中采购费用分摊选择的“采购入库单(含费用)”,为何费用生产的其他支出单还能单独生成凭证?

> 作者：大家好才是真的好 | 阅读：107 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/779162774545887488)

【问题描述】

生成凭证参数中采购费用分摊选择的“采购入库单(含费用)”,为何费用生产的其他支出单还能单独生成凭证?

【原因分析】

采购费用未分摊所致。

【示例】

1、【账务处理】-【生成凭证】-【参数设置】中,费用生成设置为:采购费用分摊采购入库单(含费用)。

![](https://vip.kingdee.com/download/010947f33350cbab47f090b8e7325a582da4.png)

2、采购入库单上采购费用生成其他支出单(未进行分摊),已审核。可检查采购入库单单据体采购费用分摊字段,为空则代表未分摊。

![](https://vip.kingdee.com/download/01096504581a3aaa4035bd3e827b55897145.png)

3、【账务处理】-【生成凭证】-【其他支出单】,可以单独选择这笔采购费用生成的其他支出单。

![](https://vip.kingdee.com/download/010939e737eb36ae4d888ae1b654b2d248dc.png)

4、采购入库单上采购费用生成其他支出单(已进行分摊),已审核。可检查采购入库单单据体采购费用分摊字段,为空则代表未分摊,不为空代表已分摊。

![](https://vip.kingdee.com/download/010920e57b567aac49a28f8ceb950f9a013f.png)

5、【账务处理】-【生成凭证】-【其他支出单】,未显示这笔采购费用生成的其他支出单。![](https://vip.kingdee.com/download/01094c5f9d27b79c46f4af6c69c48ea81ea1.png)
