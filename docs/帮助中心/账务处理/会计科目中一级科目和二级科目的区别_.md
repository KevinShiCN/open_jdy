# 会计科目中一级科目和二级科目的区别?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：脸脸1 | 阅读：2336 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/627671547930257664)

【问题描述】

会计科目中一级科目和二级科目的区别?

【概述】

区别主要表现在:

1. 层次不同:一级科目是最高层级,二级科目是在一级科目下的细分。

2. 反映的详细程度不同:一级科目反映的信息较为概括,二级科目反映的信息更为具体详细。

3. 设置目的不同:一级科目主要用于会计报表的编制和外部信息的提供,二级科目则更多用于内部管理需要,提供更细致的数据支持。

4. 编码不同:一级科目有统一的编码,二级科目则根据一级科目的编码进行扩展。
