# 科目余额表上的参数勾选说明

科目余额表相关参数说明如下：

1）展开所有级次：所有科目均可逐级展开到最明细级显示。

2）只显示最明细级：显示所有最明细级科目的数据，上级科目不再显示。展开所有级次和只显示最明细级两 个选项互斥。

3）余额为 0 不显示：期末余额为 0 的科目不显示。

4）无发生额且余额为 0 不显示：同时满足无发生额且无余额的科目都不显示出来。
