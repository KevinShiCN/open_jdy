# 凭证打印如何设置只在最后一页打印表尾信息(核准、过账、出纳、制单、审核)

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：财税李炳全 | 阅读：1073 | 有用：3

【问题描述】

凭证打印如何设置只在最后一页打印表尾信息(核准、过账、出纳、制单、审核)

【操作流程】

1、如一张凭证分录太多,系统会分页打印,默认状态会在每页都打印表尾字段

![](https://vip.kingdee.com/download/01091cef59eac19a4d7bacf4287e302c79ff.png)

2、如果想在最后一页打印表尾字段,可在凭证打印模板里设置表尾只在最后一页打印,步骤如下:

【系统设置】-【打印模板】

![](https://vip.kingdee.com/download/0109913c7938d7e6423d9b174cc07711a1b2.png)
【账务处理】-【录凭证】,点击要设置的模板的编码(如果原本用的是预置模板需复制出来修改,预置模板不支持修改)

![](https://vip.kingdee.com/download/01097bd16ff40b894173ac97f327ff31b97f.png)
选中表尾字段,右侧控件属性中“打印所在页面”选择“最后一页”,设置后保存模板
![](https://vip.kingdee.com/download/01096b78482ed0674beaae595a5684696a0d.png)
如果需要打印固定行,表尾需要放在表尾区域里,设置表尾区域为“顺序打印”,设置后保存模板
![](https://vip.kingdee.com/download/0109a36d9915c4774d198dd2776c60444be6.png)

![](https://vip.kingdee.com/download/0109c6d2a08d3c864066814f6d133492b43f.png)

3、即可实现最后一页打印表尾效果

![](https://vip.kingdee.com/download/0109bd16bb590ec54b88bf0a1c876868acc9.png)
