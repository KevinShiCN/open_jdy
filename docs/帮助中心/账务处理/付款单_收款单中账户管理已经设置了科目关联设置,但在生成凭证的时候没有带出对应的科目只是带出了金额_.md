# 付款单/收款单中账户管理已经设置了科目关联设置,但在生成凭证的时候没有带出对应的科目只是带出了金额?

> 作者：杨桂平 | 阅读：270 | 有用：0

【问题描述】

付款单/收款单中账户管理已经设置了科目关联设置,但在生成凭证的时候没有带出对应的科目只是带出了金额?
【解释说明】

在业务凭证模板中,账户管理科目对应的原币金额和本位币金额设置的公式,应该选取付款/收款信息中的付款/收款金额,而不是收款/付款合计,正确选择之后生成凭证可以带出科目的。
