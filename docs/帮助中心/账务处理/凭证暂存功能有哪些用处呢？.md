# 常见问题：凭证暂存功能有哪些用处呢？

答案解析:【凭证暂存】：如果凭证未录入完全，有事需要暂时离开，可以保存为暂存凭证。单击【暂存】按钮，系统会把当前未录入完成的凭证存到暂记录表中，不进行逻辑关系判断，不判断必录字段。
后续需要继续录入该凭证时，单击【暂存】→【暂存记录】，打开会计分录序时簿的页签，直接调出该凭证继续录入即可
