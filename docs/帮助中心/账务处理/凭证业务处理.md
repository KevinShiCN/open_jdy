# 凭证业务处理

> 作者：Cat_Chow | 阅读：10594 | 有用：15

| 步骤 | 具体说明 | 学习链接 |
| --- | --- | --- |
| 1.凭证录入 |  | —— |
| 1.1手工新增 | 直接手工录入凭证 | 点击学习 |
| 1.2批量引入 | 按导入模板格式编辑凭证内容，然后批量导入凭证。或是把A账套的凭证按引入模板引出，于批量导入B账套。 |  点击学习   |
| 1.3由凭证附件生成 | 可把原始凭证附件上传系统，然后由凭证附件生成凭证，产生关联关系。支持多张凭证附件合并生成一笔记账凭证。 | 点击学习 |
| 1.4智能核算中心 | (1)将业务单据中的信息转化为财务凭证。包括进销存云模块的业务单据、资产业务中的单据、出纳日记账、工资单等等。(2)在生成业务凭证之前，首先设置业务与科目关联关系，然后再引用预置的业务凭证模板，直接生成凭证。 | —— |
| 1.4.1科目关联设置 | 通过科目关联设置，实现业务基础资料与财务的关联，做为业务数据转换为财务数据的基础。 | 点击学习 |
| 1.4.2业务凭证模板 | 业务凭证模板是将业务数据转换为财务数据的重要桥梁，基于科目与基础资料关联设置，匹配对应业务单据信息，将业务单据上的信息获取并转换为总账的凭证。系统已有预置的业务凭证模板，可根据企业业务情况自定义适用业务凭证模板。 | 点击学习 |
| 1.4.3凭证生成参数介绍 | 包括合并规则设置、辅助资料自动新增设置、单据行数设置、生成凭证单据范围设置、费用生成设置等7种业务单据生成凭证时的相关参数设置。支持根据实际业务自行调整。 | 点击学习 |
| 1.4.4业务单据生成凭证 | (1)生成凭证通过向导式操作，将业务单据匹配对应凭证模板，生成总账凭证。并同时在业务单据上记录对应凭证字号。实现业财一体。(2)支持多种生成方式：按单分别生成凭证、按业务单据汇总生成凭证、所有单据汇总生成凭证。(3)支持自行筛选，批量汇总生成凭证，设置合并科目条件等等。 | 点击学习 |
| 1.4.5定时生成凭证 | 制定执行方案，系统要根据方案条件定时自动生成凭证，并自动生成执行日志查看执行情况。 | 点击学习 |
| 1.5期末智能结转生成 | 期末处理，系统预置有结转损益、计提所得税、结转未交增值税、计提附加税、期末调汇（如果有外币业务时）等期末结转业务模板；可由系统自动生成结转凭证。相关介绍及操作步骤可通过【期末处理】了解详情。 | 点击学习 |
| 2.凭证日常管理 |  | —— |
| 2.1基本操作 | 包括凭证的审核/反审核/复核/修改/删除/整理/插入/冲销/作废/打印等操作 | 点击学习 |
| 2.2凭证组合排序 | 自由选择组合字段进行凭证列表排序 | 点击学习 |
| 2.3凭证列设置 | 凭证列表的列设置可隐藏/显示列字段，及调整列字段的顺序 | 点击学习 |
| 2.4凭证附件 | 原始凭证上传至系统分类管理，并由凭证附件生成记账凭证。 | 点击学习 |
| 2.5凭证汇总表 | 对所有科目的本期发生额进行合计，生成凭证汇总表，显示科目编码、科目名称、借方金额、贷方金额，并统计凭证张数、附件张数。 | 点击学习 |
