# 凭证和单据的保存和审核有什么区别?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：Lucky卷 | 阅读：218 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/616800658532002816)

【问题描述】

凭证、单据保存和审核的区别?

【解释说明】

**保存:**是指将用户在单据录入界面输入的数据进行存储,保存后,用户可以继续修改或查看该凭证、单据,如果需要修改,可以直接修改。

**审核:**是对业务单据进行确认和批准的过程。审核通过后,单据才能生效并影响库存、财务等数据,单据审核了才能结账到下一期。审核后的凭证和单据都不能直接修改,需要反审核后才能修改。

保存状态的凭证,可以修改科目、币别、汇率、金额、字号、日期、币别

![](https://vip.kingdee.com/download/0109587011c1a8be486cbd5624f48e3a3596.png)

在录凭证界面已审核的凭证,仅供查看,里面的内容不支持修改

![](https://vip.kingdee.com/download/0109fc572c45c3d44b0fbd1c002b5baa6b2c.png)

在查凭证界面,已审核的凭证可以勾选点击“修改摘要”跳转到凭证详情界面对摘要进行修改

![](https://vip.kingdee.com/download/01094913a19b44d14f1cacfab45f10d768ee.png)

保存状态的单据,里面的内容可以直接修改,保存状态的单据不会显示在生成凭证界面

![](https://vip.kingdee.com/download/0109570174d4df2a413f876cf5f0ac9f9f74.png)![](https://vip.kingdee.com/download/01091732e37f607d4ad3b7fe1274760de04e.png)

审核状态的库存类单据,不能直接修改商品行信息、客户信息等内容,需要反审核后修改,会显示在生成凭证界面![](https://vip.kingdee.com/download/010925a5266a17d3415c8d62fd422480a3b4.png)![](https://vip.kingdee.com/download/0109130e800a7fb7402daa1fbed3bc7c0ab3.png)
