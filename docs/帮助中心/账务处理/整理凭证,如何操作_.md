# 整理凭证,如何操作?

【业务场景】

凭证断号不影响账务,也可以设置凭证断号允许结账,但是会影响凭证的连续性。

如果凭证出现断号,或需要按凭证日期重排凭证编号,均可以使用凭证整理功能。**此操作不可逆,建议先[备份账套](https://vip.kingdee.com/article/561354015481563904?productLineId=35&isKnowledge=2&lang=zh-CN)后再操作。**

**已审核、已复核、已作废的凭证均可参与整理,不受状态影响,已结账的凭证不支持整理,需要反结账之后再进行操作。**

【操作流程】

依次点击【账务处理】>【查凭证】

![](https://vip.kingdee.com/download/01094fbdbe6dcee045939c5eaec52d759977.png)

筛选出需要整理的凭证,点击上方<整理>按钮

![](https://vip.kingdee.com/download/01098419d062b19242bc984ded899b41b0bf.png)

①当删除其中部分凭证时,凭证号不会自动补齐,会出现断号,可通过【整理】功能,选择“按凭证号顺次前移补齐断号”查询,点击<开始整理>

![](https://vip.kingdee.com/download/01096549e3b268f8493d9d1156d1530369f0.png)

②当凭证序号不按日期排序,顺序错乱时,可通过【整理】功能,选择“按凭证日期重新顺次编号”查询,点击<开始整理>

![](https://vip.kingdee.com/download/010947c962c487894790afef026968ab9cad.png)

③若作废凭证不想参与整理,取消勾选“作废凭证参与凭证整理”再点击<开始整理>

![](https://vip.kingdee.com/download/0109651610e4b61548d883f533aff0aa0fcc.png)
