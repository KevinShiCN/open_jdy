# 金蝶云星辰组装单里面的组装费用有多个费用，不同的费用要生成不同的科目凭证分录行，怎么设置业务凭证模板取值？

金蝶云星辰组装单里面的组装费用有多个费用，不同的费用要生成不同的科目凭证分录行，怎么设置业务凭证模板取值？

![image.png](https://vip.kingdee.com/download/0109e0fae0e6de0b4b1eab34269d0ce47fe3.png)

解决方案：

（1）进入【账务处理】-【业务凭证模板】进入组装单业务凭证模板设置分录行取值即可，截图如下：

![image.png](https://vip.kingdee.com/download/0109a351e3088ab74f609f95db1fdc9cf560.png)

（2）进入【账务处理】-【科目关联设置】进入设置支出类别，设置对应支出类别对应的会计科目，设置组装费，人工费等支出类别对应的会计科目即可；

![image.png](https://vip.kingdee.com/download/0109d642d16d95d64dc29e71bc1b15701cef.png)
