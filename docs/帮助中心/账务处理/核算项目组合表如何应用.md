# 核算项目组合表如何应用

> 作者：倒也不扶墙 | 阅读：1417 | 有用：3

【问题描述】

核算项目组合表如何应用?

【解释说明】

核算项目组合表,支持双辅助核算+科目二维查看,设置了多辅助核算类别的科目,可在核算项目组合表界面选择科目及多个辅助核算组合查询。

查询方式如下图所示,可选择科目后,核算维度设置,分别在行和列选择不同辅助核算类别查看。
