# 如何生成凭证?如何设置业务凭证模板?

【问题描述】

 如何生成凭证?如何设置业务凭证模板?

【适用版本】

适用星辰标准版、专业版、旗舰版

【相关视频】

[业务单据生成凭证](https://vip.kingdee.com/school/528608403015402752?productLineId=35&lang=zh-CN&cosmic_type=video)

【操作步骤】

业务模块与财务模块联动生成凭证时,需预先完成业务凭证模板配置及科目关联设置;业务单据审核通过后,系统可基于已配置的模板和科目关联自动生成对应凭证,以此实现业财数据一体化管理

**1、**系统有提供最新设置的默认业务凭证模板,可直接下载导入使用:[最新业务凭证模板-持续更新中(重要!!!)](https://vip.kingdee.com/questions/280273946451373824/answers/280273946451373824?productLineId=35&isKnowledge=2&lang=zh-CN)

**tips:系统除了有预设好的凭证模板,也可以根据需要自定义对应的业务凭证模板,详细的设置流程可参考:[凭证设置流程及凭证生成常见问题讲解](https://vip.kingdee.com/school/detail/216497543365071104?productLineId=35&lang=zh-CN)**

**2、设置业务凭证模板后,需要设置基础资料和科目的匹配,详情可点击:[如何通过科目关联设置,实现业务资料与会计科目的匹配?](https://vip.kingdee.com/knowledge/391350553869429760?productLineId=35&isKnowledge=2&lang=zh-CN)**

**3、生成凭证流程**

点击【账务处理】>【生成凭证】,进入到生成凭证列表的界面

![](https://vip.kingdee.com/download/010948bf9f51279d49b0a3fad27e757bc0ac.png)

2、可以根据需要设置多种筛选条件,系统支持按照【期间范围】、生成凭证时是否需要【合并科目】、【单据范围】、【是否显示金额为0的单据】、单据表头的【客户】、【供应商】等过滤条件筛选单据

![](https://vip.kingdee.com/download/01098f3a7bca112d4f0eba8bd74bdcce3a26.png)

①生成条件设置:

【生成方式】:系统支持三种方式

a、按业务单据汇总生成——同类型单据生成一张凭证;

b、按单生成——即一张单据一张凭证;

c、所有单据汇总生成——无论什么类型单据生成一张凭证。

![](https://vip.kingdee.com/download/01091ba984f310ff4a85ad9c35e98d19fcfc.png)

【合并科目】:当科目、辅助核算、计量单位、币别等均一致时才能合并凭证分录;系统提供了6种不同的合并项目可以选择。例如“摘要不同,允许合并”参数,将合并各分录的摘要并以逗号分隔(摘要1,摘要2)。

![](https://vip.kingdee.com/download/0109f7414b39e5d1430b911b1b319c8c96be.png)

②选择生成凭证列表中的需要生成的单据类别

![](https://vip.kingdee.com/download/01098f9db8613ba640f7a94b1b1310b388c3.png)

点击右上角【列设置】,可以根据需要设置显示或者隐藏对应的字段信息,详情可点击:[生成凭证界面的列设置如何使用?](https://vip.kingdee.com/knowledge/580923096555458048?productLineId=35&isKnowledge=2&lang=zh-CN)

![](https://vip.kingdee.com/download/0109c433a0f4a8764bd58a8105df7eb84fbc.png)

③勾选单据,选择需要的凭证模板(系统会根据默认模板自动填充或者模板中设置了适用条件后,自动匹配)后,点击<生成凭证>即可,如下图所示:

![](https://vip.kingdee.com/download/01096cbfaf26cc254f948e04290c72f1270b.png)

④生成凭证后,选择需要查询凭证的单据,点击右上角【查凭证】,可以跳转至查凭证列表中
![](https://vip.kingdee.com/download/0109da1698599db74a83b2fa2125b04f760d.png)
查看对应单据生成的凭证详情

![](https://vip.kingdee.com/download/010913f396beb05c45508f043fdaf040ff64.png)

若需要查询生成凭证的操作日志,可以点击右上角的【查看日志】,进入到凭证生成日志列表中查询

![](https://vip.kingdee.com/download/0109b4d981d514df43a78935af8b0e3b2a37.png)

3、点击右上角的【参数设置】,可以设置生成凭证的参数

![](https://vip.kingdee.com/download/010985fdbf22e75a4e81ad29a3ad373bdde8.png)
例如生成凭证时,若是暂存凭证不合并等参数

![](https://vip.kingdee.com/download/01092edbd8647eba427c9e2bd42d46e201e9.png)

4、点击右上角的【业务凭证模板】、【科目关联设置】,可以对单据的凭证模板和基础资料关联的科目数据进行修改
![](https://vip.kingdee.com/download/01090aa72896b4214dfe8989737b4fa06b43.png)
【温馨提示】

**1.业务凭证模板设置取值时,仅支持取单据中已有的字段,可以将同一个单据体中的不同字段设置加减乘除的公式,但不支持设置复杂的函数公式取数,例如IF函数等**

![](https://vip.kingdee.com/download/0109f1a1cb8767ec4adf82c0a03f745a38f2.png)
**2.系统是将业务单据通过凭证模板的方式,生成对应的凭证,仅支持单据生成凭证时关联凭证,暂不支持已经手动录入了凭证后,将业务单据和凭证关联**

**3、财务模板常用账务处理模板设置,请查看知识:[如何新增凭证模板和凭证模板管理?](https://vip.kingdee.com/link/s/ZpubX)**
