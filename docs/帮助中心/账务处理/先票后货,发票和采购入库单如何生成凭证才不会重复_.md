# 先票后货,发票和采购入库单如何生成凭证才不会重复?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：529 | 有用：1

【问题描述】

先票后货,发票和采购入库单如何生成凭证才不会重复?

【解决方案】

举例说明,科目仅供参考

发票生成的凭证

借:在途物资

        应交税费-增值税(进项税)

     贷:应付账款

入库后入库单生成的凭证是:

借:库存商品

    贷:在途物资
