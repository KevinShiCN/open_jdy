# 核算项目明细表/核算项目组合表/核算项目综合表/核算项目余额表的区别是什么?

【问题描述】

核算项目明细表/核算项目组合表/核算项目综合表/核算项目余额表的区别是什么?

【业务场景】

系统中的核算项目的报表存在明细表、组合表、综合表、余额表,这几个表之间如何进行区分?

![](https://vip.kingdee.com/download/0109b4f169dfa3db40e8af2d1ad2d580cbaa.png)

**若需要了解如何查询核算项目明细账,请点击:[如何查询及导出核算项目明细账?](https://vip.kingdee.com/knowledge/491775580434748928?productLineId=35&isKnowledge=2&lang=zh-CN)**

**若需要了解如何查询核算项目组合表,请点击:[核算项目组合表如何应用?](https://vip.kingdee.com/knowledge/553019105998015232?productLineId=35&isKnowledge=2&lang=zh-CN)**

**若需要了解如何查询核算项目综合表,请点击:[如何查询核算项目综合表?](https://vip.kingdee.com/knowledge/129339770114427433?productLineId=35&isKnowledge=2&lang=zh-CN)**

【解释说明】

**核算项目明细账:主要按照辅助核算的维度,查询相关凭证的明细数据,支持按照科目进行查询。**

![](https://vip.kingdee.com/download/01095af7030ae08a44b09352d333b57029fc.png)**核算项目组合表:主要以科目为核算维度,通过行/列的方式显示包含双辅助核算的科目在对应期间下的期初余额、借方/贷方发生额、期末余额的情况。**

![](https://vip.kingdee.com/download/010964ac8a16504146799b4687038f102c58.png)**核算项目综合表:主要按照辅助核算的维度,查询辅助核算在对应期间下的期初余额、借方/贷方发生额、期末余额的情况,支持显示对应科目下关联的辅助核算信息。**

![](https://vip.kingdee.com/download/010950c0818f3a3d451ea85c575a7481d575.png)
**核算项目余额表:主要以辅助核算维度,按照月份查询科目的辅助核算在对应期间下的期初余额、借方/贷方发生额、期末余额的情况。相较于核算项目综合表,支持按照期间分项进行查询。**
![](https://vip.kingdee.com/download/01093e6bb1f83597460bb501b064e2737b6e.png)
