# 为什么 生成凭证界面显示的业务单据不全？

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题, 快速入门
> 作者：财税云肖梅梅 | 阅读：97 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/728909865627425536)

【问题描述】

为什么 生成凭证界面显示的业务单据不全？

【解析说明】

举例：以供应链或生产为主的客户，每天的业务出单量大，高达成百上千张业务单据在月底统一入账时，生成凭证界面筛选时承载过大。

为考虑性能问题，我们在【账务处理】-【生成凭证】右上角加了一个参数，如下图

![上传图片](https://vip.kingdee.com/download/0100041a9f332f0f4d2db110fc93dd32c78e.png)

单据行数设置，系统预设最大行为2000行，客户可以根据自己的业务量调整最大至20000行

**注意：如果业务单据量超过20000的，建议筛选短时间段的业务单据去生成凭证。**
