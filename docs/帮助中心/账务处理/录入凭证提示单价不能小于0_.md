# 录入凭证提示单价不能小于0?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：shaoling | 阅读：82 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/705227941906254336)

【问题描述】

录入凭证提示单价不能小于0?

【业务场景】

录凭证时,科目开启数量金额核算后,单价录入负数就会出现提示“单价不能小于0”,凭证金额是负数需要怎么录入?

![](https://vip.kingdee.com/download/0109a09bd49f0ab84df9acc68dc51c32d105.png)

【解释说明】

单价代表商品/服务的交易价格,在现实商业行为中,单价均为非负数(不存在“负价格”的交易标的)。

例外情况(如折扣、退货)需通过 **数量为负** 或 **借贷方向调整** 实现,而非直接修改单价负数。

如凭证金额需要负数,调整数量为负数,单价仍需正常显示正数,如下图所示:

![](https://vip.kingdee.com/download/010922178b20ec37457eb79305bce36f2ef5.png)
【温馨提示】

**录入凭证时,凭证中的借方或者贷方金额不支持录入0金额,因此单价也不支持设置为0。**
