# 期末处理

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：快速入门, 功能介绍
> 作者：Cat_Chow | 阅读：6349 | 有用：10
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/630442870763171072)

期末处理即**期末智能结转**，有非必要项（可由系统自动生成或是手工录入结转凭证）和必要项（必须由系统处理）。若同时开通了进销存云模块，可进行库存与总账对账处理。

![](https://vip.kingdee.com/download/0109cbceb0a1f7b94a1392ca3f90a2ae42c3.png)

| 步骤 | 具体说明 | 学习链接 |
| --- | --- | --- |
| 1.期末智能结转 | 分为非必要项和必要项。系统预置了结转损益、计提所得税、结转未交增值税、计提附加税、期末调汇（如果有外币业务时）等期末结转业务模板。同时支持自定义期末结转凭证模板。 | —— |
| 1.1非必要项 | 非必要项，由系统自动生成或是手工录入结转凭证，二选一 | —— |
| 1.1.1计提附加税 | 系统预置计提附加税公式，可直接由系统计算并生成结转凭证；或是根据实际情况调整公式。 | 点击学习 |
| 1.1.2结转未交增值税 | 月度终了，企业应当将当月应交未交或多交的增值税自“应交增值税”明细科目转入“未交增值税”明细科目。 如待转出未交增值税为0或负数，表示为没有销售业务发生，或有留抵进项税，不需进行未交增值税转出。 | 点击学习 |
| 1.1.3计提所得税 | 系统预设的计提所得税计算公式：RPT("利润表","三、利润总额（亏损总额以“-”号填列）","月报",0,0,"本年累计")*0.25-RPT("利润表","减：所得税费用","月报",0,0,"本年累计")。 | 点击学习 |
| 1.1.4期末调汇 | 科目启用外币核算且勾选【期末调汇】，则期末结转时，可自动生成期末调汇凭证。 | 点击学习 |
| 1.1.5期末结转凭证模板 | 系统提供自定义期末结转功能,可根据实际业务需要自定义期末结转模板。  支持按测算金额科目结转，或是按辅助核算结转。 | 点击学习 |
| 1.2必要项 | 必要项，必须由系统自动结转 | —— |
| 1.2.1结转损益 | (1)结转损益是指在期末,损益类科目的余额将全部结转到“本年利润”,结转后,损益科目余额为零的过程；建议优先处理完前面的结转项后，最后再进行结转损益。(2)年末(第十二期)本年利润,未分配利润也是按以下步骤自动结转,无需其他操作。(3)系统支持按年结转损益，即1-11月不结转损益，待12月年度全部一次性结转。注意： 结转损益凭证必须系统生成,不能手工录入,否则利润表无法取数。 | 点击学习 |
| 1.3重新测算 | 对结转界面已经计算过的数据进行再次计算。如自定义结转模板修改计算公式后，可通过重新测算更新结转数值。 | 点击学习 |
| 1.4生成凭证 | 可以一次性选择所有项目并分别生成结转凭证，或者逐个点击每个项目上的【生成凭证】按钮单独生成，而结转损益凭证则应在最后生成。 | 点击学习 |
| 2.库存与总账对账 | 开通进销存云，并且进销存云已核算成本， 在账务云生成业务凭证，可通过库存与总账对账功能，进行匹配核对。 | 点击学习 |
