# 录凭证时金额如何正负切换?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：云社区用户xBO70916 | 阅读：1000 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/562803591355073536)

【问题描述】

录凭证时金额如何正负切换?

【操作说明】

点击【账务处理】>【录凭证】

![](https://vip.kingdee.com/download/010952ca316d4dc54efd900d14b3eeadde3d.png)

打开录入凭证的界面,输入科目,在录入金额的位置直接填写“-”号,金额就会红字显示,也就是负数(红字)的金额,去掉负号就是正数(蓝字)金额。

![](https://vip.kingdee.com/download/0109276aa2c29c654f4f91684be30ac73ce8.png)

![](https://vip.kingdee.com/download/01094d6e5135db604938b293b80879e78f00.png)
