# 科目启用了数量核算后录凭证可以不录入数量和单价吗?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：345 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/639268162583906816)

【问题描述】

科目启用了数量核算后录凭证可以不录入数量和单价吗?

【解决方案】

科目启用了数量核算,录凭证**可以不录入数量和单价,无法设置为必录。**

不录入数量和单据保存凭证系统会提示:“数量/单价”为空,数量*单价不等于本位币金额。

![](https://vip.kingdee.com/download/010958f0cd9b64c945dc91f9ada181440262.png)
可以忽略管系统提示,直接点“确定”,保存凭证即可。

![](https://vip.kingdee.com/download/010987ba47995b05424f85077b47fdad3e35.png)
