# 生成凭证界面采购销售发票的勾稽金额能否显示合计?

> 作者：杨桂平 | 阅读：9 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/790022330197171456)

【问题描述】

生成凭证界面采购销售发票的勾稽金额能否显示合计?

![](https://vip.kingdee.com/download/01098ff911a735af4d889b6dff67c3e689d4.png)

【解决方案】

点击右上角列设置按钮,找到勾稽金额字段,勾选显示合计即可
