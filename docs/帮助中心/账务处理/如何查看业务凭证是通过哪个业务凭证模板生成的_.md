# 如何查看业务凭证是通过哪个业务凭证模板生成的?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：Cat_Chow | 阅读：806 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/570776247677063680)

【业务场景】

如何查看业务凭证是通过哪个业务凭证模板生成的?

如图所示,业务凭证已生成,如何查询该凭证是由哪个业务凭证模板生成的。

![](https://vip.kingdee.com/download/01090a475798c5044201a858a5515b97f130.png)【解决方案】

可通过【生成凭证】右上角【查看日志】或是【查看日志】-【历史日志】打开凭证日志列表,查看生成凭证记录

![](https://vip.kingdee.com/download/0109d3f6d584552248e3ba787ccab8891ea2.png)如图所示,单据是按【销售出库-仅确认收入】模板生成的业务凭证。

![](https://vip.kingdee.com/download/010945bc1a63d1884ad78a343cf078e1597c.png)

【温馨提示】

为了保持数据的完整性,凭证生成日志不支持删除。
