# 期末结账,结账前检查结果说明及异常处理

【业务场景】

期末结账,结账前检查结果说明及异常处理

【解释说明】

在期末结账页面点击“结账前检查”,系统根据设置进行相关检查,并给出检查结果。检查项设置可参考以下步骤:[期末结账,检查项设置如何操作?](javascript:void(0);)

检查结果分为异常项目和警告项目。

异常项目:必须处理异常项目,未处理,不允许结账至下期。

警告项目:仅警告提醒,不影响结账,即未处理,也可以正常结账至下期。

3、异常处理

检查结果与检查项设置有关,可以参考检查项的设置,分别进行修改或是调整。

以上述提示为例,因检查项设置勾选【凭证必须审核才能结账】,则当期有未审核的凭证,则检查结果出现异常项目提示;必须把当期所有凭证审核后,才允许结账至下期。
