# 核算项目综合表如何查询同一辅助核算类别下的多个辅助核算?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题

【问题描述】

核算项目综合表如何查询同一辅助核算类别下的多个辅助核算?

【解释说明】

如图,同时查询辅助核算类别“客户”下不同客户名称的数据。

辅助核算类别=“客户”;

辅助核算过滤字段=按需选择“编码”或“名称”;

条件设置=“精准匹配多值”;

辅助核算=点击按钮进入相关界面多选不同的客户名称,确认查询即可。
