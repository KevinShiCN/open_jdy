# 录凭证页面,科目金额的小数位如何修改?

> 作者：云社区用户8x80607 | 阅读：203 | 有用：0

【问题描述】

录凭证页面,科目金额的小数位如何修改?![](https://vip.kingdee.com/download/0109d11a96d067144f97a3c17552c42e85da.png)

【解释说明】

录凭证页面科目金额的小数位是由**本位币金额小数位**控制,如需要调整可在【基础资料】-【币别】,修改金额小数位即可![](https://vip.kingdee.com/download/0109d884521b4ebe4c458e7d825340f8172a.png)
温馨提示:单价小数位、金额小数位一经改大无法改小,请一定谨慎修改。
