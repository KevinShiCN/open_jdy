# 生成凭证界面,参数设置功能介绍

【业务场景】

生成凭证界面,参数设置功能介绍

【解释说明】

在生成凭证前,可进行生成凭证参数设置,打开【账务处理】-【生成凭证】-【设置-参数设置】,如图:

![](https://vip.kingdee.com/download/010914629b5397f54841bcef6df48b53ccfc.png)

2、具体参数设置有:合并规则设置、自动新增设置、单据行数设置等7种,具体说明如下

![](https://vip.kingdee.com/download/0109c7c7b172eb334b9798c89d4331311a8c.png)

①合并规则设置:若生成方式选择按业务单据/所有单据汇总生成,勾选以下合并条件表示条件相同的单据将汇总生成一张凭证。

②自动新增设置:

a、发票生成凭证时自动匹配新增客户、供应商:勾选此参数,发票生成凭证时,如果存在基础资料-客户/供应商上不存在的客户/供应商,则会自动新增基础资料-客户/供应商;

b、日记账生成凭证时自动匹配新增客户、供应商:勾选此参数,日记账生成凭证时,如果存在基础资料-客户/供应商上不存在的客户/供应商,则会自动新增基础资料-客户/供应商。

c、进项发票附件自动转为凭证附件(来源于发票抵账库数据):当进项发票存在来

源于发票抵账库数据时,可设置附件是否转为生成凭证的附件。

③单据行数设置:生成凭证时列表显示分录最大行数:最大可设置到 20000 行

④单据范围设置:当存在调拨出库、调拨入库、移仓单、组装单、拆卸单、核销单(预收冲应收、预付冲应付、退货冲销货、退货冲购货)时,可设置对应的单据是否生成凭证 。

⑤费用生成设置:

a、销售费用分摊、采购费用分摊:在销售出库单、采购入库单生成凭证时,当单据上存在已分摊的销售费用、采购费用,则可设置费用是在本单生成凭证还是在生成的其他支出单上生成凭证;

b、客户承担费用:在销售出库单生成凭证时,当单据上存在客户承担费用,则可设置客户承担费用费用是在本单生成凭证还是在生成的其他支出单上生成凭证。

⑥销售成本结转设置:

a、销售出库成本结转:同时结转:在生成凭证时,通过【销售管理】-【销售同时结转:在生成凭证时,通过【销售管理】-【销售出库单】、【销售退货单】生成凭证时,确认收入的同时进行成本结转,凭证模板需要同时设置收入和成本结转相关分录 ;单独结转:在上述单据生成凭证时,只需要设置收入确认相关分录,成本结转凭证通过【库存管理】-【结转成本】单独生成;

b、委托代销成本结转设置参考销售出库成本结转。

c、零售成本结转:在生成凭证时,通过【门店管理】-【零售结存单】生成凭证时,确认收入的同时进行成本结转,凭证模板需要同时设置收入和成本结转相关分录;单独结转:在上述单据生成凭证时,只需要设置收入确认相关分录,成本结转凭证通过【库存管理】-【结转成本】单独生成。

⑦超收超付生成设置:当收款单/付款单存在超收/超付,且生成了预收/预付款单,可以设置对应的超收/超付金额是收/付款单生成或是预收/预付款单生成凭证;应收应付单据手续费:当应收应付单据存在手续费且生成了其他支出单时,可设置此部分金额是由应收应付单生成凭证,或是由其他支出单生成凭证。
