# 结账时提示有资产卡片未生成凭证,如何修改?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：shaoling | 阅读：247 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/705585547896615168)

【问题描述】

结账时提示有资产卡片未生成凭证,如何修改?

【业务场景】

资产结账前检查出现警告信息,提示“存在未生成凭证的卡片或单据”

![](https://vip.kingdee.com/download/0109214d895246394edb8083791d90c27628.png)
【解释说明】

资产结账前检查提示“存在未生成凭证的卡片或单据”,若已经手工录入凭证可忽略不处理,若还未生成凭证可点击异常提示模块进入检查结果明细页面

![](https://vip.kingdee.com/download/01098a591bf1c44e4f8da700794046e14ebc.png)或者点击<异常处理>进入检查结果明细页面

![](https://vip.kingdee.com/download/01096c59a1f445eb495a936bc05e34ee5365.png)
找到对应的关联操作点击<凭证生成向导>按钮进入生成凭证页面

![](https://vip.kingdee.com/download/010944ce3b7621374a60a2b21e77ea16f7c5.png)
在生成凭证页面找到对应的“资产新增”检查哪些卡片没有生成凭证

![](https://vip.kingdee.com/download/0109afaa4807ab0c420cbe0ff8bc6f693682.png)
选择没有生成凭证的卡片,点击右上角<生成凭证>按钮生成操作即可
![](https://vip.kingdee.com/download/0109ad57224509d642aca86dfffa9361f5b0.png)
同样在生成凭证页面找到对应的“资产清理”检查哪些卡片没有生成凭证
![](https://vip.kingdee.com/download/0109f88be106bc13413c9e57e4f0a13e988f.png)
若没有卡片未生成凭证无需处理
若有资产清理的卡片没有生成凭证,同上选择对应卡片点击右上角<生成凭证>按钮操作即可
![](https://vip.kingdee.com/download/01092597d76759044b03a76eb436b6e202a4.png)同样在生成凭证页面找到对应的“资产变动”检查哪些卡片没有生成凭证

![](https://vip.kingdee.com/download/010921db283ecf6141c4b6ec5e8e91bfd5e7.png)
选择没有生成凭证的卡片,点击右上角<生成凭证>按钮生成操作即可

![](https://vip.kingdee.com/download/0109346258f8920c427399ecf9512f9bfe78.png)
