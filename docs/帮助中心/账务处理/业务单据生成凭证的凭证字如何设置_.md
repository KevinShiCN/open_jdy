# 业务单据生成凭证的凭证字如何设置?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：BBAABABA | 阅读：605 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/539973286114143232)

【业务背景】

在生成凭证界面把业务单据生成凭证时,如何设置生成的凭证的凭证字?

![](https://vip.kingdee.com/download/0109b2ea96ea79f14899aad724397ec3a304.png)

【解决方法】

第一步:先到基础资料--财务资料--凭证字里新增好需要用到的凭证字;

![](https://vip.kingdee.com/download/01097a29ff6a51754d3795e01eb2c669f19c.png)
第二步:到账务处理--业务凭证模板--找到对应要修改的业务凭证模板,点进去修改业务凭证模板默认的凭证字。

![](https://vip.kingdee.com/download/0109a33298a89c2b430b8aeefd78e63cd81b.png)
![](https://vip.kingdee.com/download/010943521d320a694ceea6e4c3a6cf5de620.png)
![](https://vip.kingdee.com/download/0109d101b3f94c544d929d2e325ba3c4a88e.png)
【温馨提示】

生成凭证的日期也是到业务凭证模板里修改默认取值,目前支持以下几种取值:

![](https://vip.kingdee.com/download/01091921c42c19ea4f87bc5e420a4b58157a.png)

次月想生成到账套财务期间的最后一天,可以把日期选择为当前财务期间日期。
