# 账务处理反结账提示:反结账检查失败:财务的当前期间等于资产的启用期间,不能反结账

【问题描述】

账务处理反结账提示:反结账检查失败:财务的当前期间等于资产的启用期间,不能反结账

【业务场景】

账务处理反结账提示:反结账失败,原因如下: 2024年2期反结账失败,原因如下:反结账检查失败:财务当前期间等于资产的启用期间,不能“反结账”

![](https://vip.kingdee.com/download/0109ae25a035d3944ae58d0642ca327f2a16.png)

【解释说明】

账务处理反结账提示:财务当前期间等于资产的启用期间,不能“反结账”,这是由于资产参数启用“与总账一起结账”的情况下,账务当前期间等于资产启用期间**不允许**反结账操作。

若账务需先反结账处理,建议资产参数可先不启用“与总账一起结账”。

账务反结账处理完重新结账后,当账务跟资产当前期间一致时,也可重新启用“与总账一起结账”

![](https://vip.kingdee.com/download/010913f0dfa7b9104ed0a365a145825f51a1.png)

【操作流程】

从【系统参数】>【系统参数】进入系统参数页面

![](https://vip.kingdee.com/download/0109665eb0edfcc34511b2e7244e50cbb619.png)
在系统参数页面。找到【资产参数】取消勾选“与总账一起结账”后点击<保存>

![](https://vip.kingdee.com/download/01095a7de6caa63d4c2d8719d732f807cf9f.png)
资产参数保存成功

![](https://vip.kingdee.com/download/0109abca68af9db640bfbe185f68aabc0e26.png)
参数保存成功后再进行账务期末反结账操作即可

![](https://vip.kingdee.com/download/0109dead9228cd914054904f7f285ff659d0.png)
