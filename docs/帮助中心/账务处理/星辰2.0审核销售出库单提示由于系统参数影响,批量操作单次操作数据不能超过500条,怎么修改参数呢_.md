# 星辰2.0审核销售出库单提示由于系统参数影响,批量操作单次操作数据不能超过500条,怎么修改参数呢?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：云社区用户6Rw2283 | 阅读：373 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/472206621633136640)

【故障现象】

   审核销售出库单时提示由于系统参数影响,批量操作单次操作数据不能超过500条,如下图:

![](https://vip.kingdee.com/download/0109e9c68330cc534b0794f52f811bacd038.png)

 【问题原因】

    是由于在【系统设置】-【系统参数】-【进销存参数】设置了单价为零时控制导致的。如下图

![](https://vip.kingdee.com/download/010953157024e2124a7b883a355a2c8e7a4d.png)

 【解决方案】

可以取消进销存参数单价为0提醒控制,可以批量审核10000多单,如果开启了零单价提示,单次只能审核500单。如下图:

![](https://vip.kingdee.com/download/01096c3b26b6080a4bb09e8c62a917414d94.png)
