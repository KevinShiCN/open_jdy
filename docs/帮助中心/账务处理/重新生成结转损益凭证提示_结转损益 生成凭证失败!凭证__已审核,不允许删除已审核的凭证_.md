# 重新生成结转损益凭证提示:结转损益 生成凭证失败!凭证**已审核,不允许删除已审核的凭证?

> 作者：Schatz | 阅读：217 | 有用：0

【问题描述】

重新生成结转损益凭证提示:结转损益 生成凭证失败!凭证**已审核,不允许删除已审核的凭证?

![](https://vip.kingdee.com/download/010910087a6ef2974bbba6014106bbb71f70.png)

【解释说明】

重新操作生成结转损益凭证,系统会覆盖之前已生成的结转损益凭证。如果之前的凭证已经审核通过,则无法被覆盖,这会导致重新生成凭证时出现此报错。

【操作步骤】

在结转损益位置,点击打开之前的结转损益凭证,并进行取消审核操作。

![](https://vip.kingdee.com/download/0109a52518a4a86040e284270af6b9c53d65.png)

或者在“财务处理-查凭证”界面查询到本期生成的结转损益凭证,并执行反审核。

![](https://vip.kingdee.com/download/0109e93561039ab54d71bf942fb27699392e.png)

注意:结转损益可能有多张凭证,全部都需要反审核才行。

完成上述步骤后,即可重新生成结转损益凭证。

![](https://vip.kingdee.com/download/01097c52eebc1c1f4e798f05aa11f17031b0.png)
