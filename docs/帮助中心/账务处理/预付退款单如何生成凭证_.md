# 预付退款单如何生成凭证?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：云社区用户6Rw2283 | 阅读：129 | 有用：1

【问题详情】

预付退款单如何生成凭证?

【操作步骤】

一、 进入业务凭证模板编辑界面

1.点击【账务处理】-【业务凭证模板】

![](https://vip.kingdee.com/download/010923cadeafbd464426ad0b89603564497d.png)
2. 左侧单据列表点击【应收应付】-【预付款退款单】,点击系统自带的预付退款单模板,点【编号】进入模板修改界面或者点击右上角的<新增>直接新增一个空白模板。

![](https://vip.kingdee.com/download/0109bf10ae628e0c46c492e4c8c10d8be503.png)
二、对应的业务凭证模板设置

1、科目来源:借方【账户管理.对应科目】;贷方【供应商.预付账款科目】;如果有手续费和优惠金额借方还需要增加上借方【账户管理.对应科目】(金额对应手续费金额);借方【模版自定义科目】如下图:

![](https://vip.kingdee.com/download/0109f0078339328247b98325b37a4179b367.png)

如果预付账款设置了辅助核算,需要在凭证模板-核算维度处,选择对应的辅助核算对应单据字段:

![](https://vip.kingdee.com/download/0109512339e40a42405f95b08386170f37b2.png)
2、金额:

借方【账户管理.对应科目】金额取值取{退款信息.退款金额};手续费【账户管理.对应科目】金额取值{退款信息.手续费};优惠金额【模版自定义科目】金额取值取{优惠金额:};贷方【供应商.预付账款科目】金额取值取{退款总金额};

![](https://vip.kingdee.com/download/0109a93b8feca5b94b8786a7ce034413cbd9.png)

3、生成凭证示例:

如何生成凭证可参考链接[如何生成凭证?如何设置业务凭证模板?](https://vip.kingdee.com/knowledge/394293171355505152?productLineId=35&isKnowledge=2&lang=zh-CN)

【温馨提示】

可以参考预付款退款单的业务凭证模版设置预收退款单的凭证模版,但是预收退款单的手续费也会用来抵扣源单信息中的预收金额,所以录入单据时需要符合“手续费+退款金额合计+优惠金额=退款总金额”的公式。
