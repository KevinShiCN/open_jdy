# 期末调汇的金额是如何计算的?

【问题描述】

期末调汇的金额是如何计算的?

【解释说明】

若科目已启用**外币核算**且勾选**期末调汇**,当期间汇率发生变动时,需执行期末调汇操作;若期间汇率无变动,则无需进行期末调汇。

调整额 = 外币科目的原币余额 × 期末汇率 - 外币科目的本位币余额。

【操作指引】

下面以一个测试科目为例:

![](https://vip.kingdee.com/download/0109995b07c4bbb745a092ad3858ba670d26.png)

通过明细账,查询该测试科目,选择币别“美元”查看,可查看到:截止2020.12期,期末调汇填写的期末汇率是7.5,调整前期末本位币余额是1350,原币余额200

根据公式:调整额 = 外币科目的原币余额 × 期末汇率 - 外币科目的本位币余额,可计算出需调整额=原币余额200*期末汇率7.5-1350=1500-1350=150,和下图明细账生成的调汇凭证金额一致。
