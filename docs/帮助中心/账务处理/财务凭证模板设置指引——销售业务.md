# 财务凭证模板设置指引——销售业务

**一、销售订单**

销售订单可预收订金，并生成预收款单。

1、预收款单对应凭证一般为：

借：银行存款等

      贷：预收账款

2、预收款单凭证模板：

借方科目来源设置为【账户管理.对应科目】,因可能存在多账户收款，金额取值需要取收款信息表体的收款金额。

贷方科目来源设置为【客户.预收账款科目】，如果预收账款下设了客户为辅助核算，还需要在核算维度处选择客户。

**二、销售出库**

1、销售出库关联销售订单生成，可定向抵扣销售订单预收，同时可同单直接收款，也可后续对账再收款。

支付销售费用，会生成其他应付单，处理流程与采购入库中的不分摊入成本费用处理一致，不再重述。

销售费用分摊不计入销售出库成本，仅作为后续销售毛利统计分析使用。

1）销售出库可确认收入的同时结转成本。**注意：在生成凭证前，需要先进行出库成本核算**

对应的凭证一般为：

借：银行存款\应收账款等

      贷：主营业务收入

            应交税费-应交增值税（销项税）

借：主营业务成本

      贷：库存商品中

2）销售出库可只确认收入，结转成本单独生成凭证

则在【智能凭证中心】-【生成凭证】-【销售管理】-【销售出库】列表，仅生成确认收入的凭证。凭证模板设置参考上述，去掉结转成本分录即可。

![](https://vip.kingdee.com/download/0100bea3ef90ee5148a3aadac7651f3679da.png)

结转成本凭证在进行存货出库成本核算后，再通过【智能凭证中心】-【生成凭证】-【库存管理】-【成本结转】-【销售出库单】列表，仅生成结转成本凭证，凭证模板设置参考上述，取结转成本分录即可。

![](https://vip.kingdee.com/download/01008ee0875d795544f5be6d740742e96613.png)
