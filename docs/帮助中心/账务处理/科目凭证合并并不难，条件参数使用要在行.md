# 科目凭证合并并不难，条件参数使用要在行

科目凭证合并并不难，条件参数使用要在行

![上传图片](https://vip.kingdee.com/download/0100e7814fe706df4de9914e6e3c8685435f.png)

      公司最近星辰系统升级3.1版本了，小云在汇总生成凭证这块总是晕乎乎的。不知道为啥有时候能合并，有时候又是分开生成的。无奈她只能请教小雪，因为小雪用的另外一个账套，几个月前就升级了，而且她现在用的很溜。小雪很热心也很耐心地跟小云讲了三种常见的合并场景，那她跟小云都讲了些什么，我们一起来看看吧！

场景一：科目合并

1、生成凭证有“凭证号”的凭证的科目合并。

当科目没有合并时，需检查合并科目下的合并条件。

![](https://vip.kingdee.com/download/01000f833d5814bf4c67bf0147b9cdf5002f.png)

例如：

1）“凭证摘要不同，允许合并”没有勾选。

![](https://vip.kingdee.com/download/01003303b8e1f906456ca50775366a6bceb7.png)

生成凭证效果：科目相同，但是摘要不同，科目没有合并。

![](https://vip.kingdee.com/download/01009e76eb0872324184b74dba84b5017a90.png)

2）“凭证摘要不同，允许合并”勾选以后。

![](https://vip.kingdee.com/download/010032fe5eba96534861a59527c3e98a872b.png)

生成凭证效果：摘要虽不同，但相同科目合并了。

![](https://vip.kingdee.com/download/010067f9c865f9d9457f9c0f14e042bf42c9.png)

2、生成凭证是“暂存凭证”的凭证的科目合并。

暂存凭证里的相同科目没有合并，需检查：设置-参数设置，“暂存凭证不合并”是否勾选。

![](https://vip.kingdee.com/download/0100bdcfe312bafe48c9b83a6c41840e518c.png)

![](https://vip.kingdee.com/download/0100e7a3539d249545ff80f4d52664a425dc.png)

例如：

1）“暂存凭证不合并”勾选，合并科目里勾选了“摘要不同允许合并”

![](https://vip.kingdee.com/download/010030c6be8cf9cf42d5b1d580db4e193027.png)

![](https://vip.kingdee.com/download/0100bfae62fb3f8e46f58f08f3023dd39dfe.png)

生成凭证效果：虽然有相同科目，但是摘要不同也没有合并。

![](https://vip.kingdee.com/download/0100b3874cc7683e4af0a0d79b16230d96f8.png)

2）凭证生成参数里的“暂存凭证不合并”不勾选，合并科目里面勾选了“摘要不同允许合并”。

![](https://vip.kingdee.com/download/0100754df42b64504d7f8cb17563a02980de.png)

![](https://vip.kingdee.com/download/0100a583069afaa948e8ba974ba4741a0b10.png)

生成凭证效果：摘要虽不同，但相同科目合并了。

![](https://vip.kingdee.com/download/01005edf5363e6d742c18dbaa2b6268be97a.png)

注意：星辰2.0暂存凭证科目会自动合并；星辰3.1需要在“凭证生成参数”里设置“暂存凭证不合并”不勾选才可以合并科目。

场景二：按业务单据汇总生成

是指将同一种“业务类型”的单据合并生成凭证。

销售出库单是一种业务类型，可以将所有的销售出库单合并生成1张凭证。如果所有销售出库单生成不止1张凭证，需要检查“凭证生成参数”里的合并条件。

![](https://vip.kingdee.com/download/01009e1cbc7f14d24fdcac1785b6e6b164be.png)

![](https://vip.kingdee.com/download/0100b80eabac05bc41238cefc1abab0b0269.png)

例如：

1）销售出库单按业务单据汇总生成，生成了2张凭证，原因是凭证生成参数里面勾选了“按客户汇总生成”导致“按客户”分别汇总生成了凭证。

![](https://vip.kingdee.com/download/0100c72ef9e41fb648f881461d8b67178265.png)

![](https://vip.kingdee.com/download/010088ce1f064bd44c71a1690f0098c1bda9.png)

2）凭证生成参数里面“按客户汇总生成”不勾选，再按业务单据汇总生成，则是1张凭证。

![](https://vip.kingdee.com/download/0100d54e23d1353b467b890d076f2f54225d.png)

![](https://vip.kingdee.com/download/0100d46b48f5d2b8458993924608112198fa.png)

场景三：按所有业务单据汇总生成

与按业务单据汇总生成类似，区别是：按业务单据汇总生成只能是一种业务类型的单据合并生成凭证，按所有业务单据汇总生成可以将不同业务类型的单据合并生成凭证。

例如：销售单据里面同时选了不同类型的单据：销售出库单与销售退货单。

1）选择按业务单据汇总生成，两种类型的单据会按“单据类型”分开汇总生成2张凭证。

![](https://vip.kingdee.com/download/01000140f11788784dd6ad1a2ac88d6f4e88.png)

2）选择按所有业务单据汇总生成，两种类型的单据则可以合并生成1张凭证。

![](https://vip.kingdee.com/download/0100fc64ac8ed95140649ef8a68de9323f8e.png)

通过小雪的点拨后，小云不在晕乎乎，瞬间思路清晰了。
