# 关于录凭证的小知识点来啦！！！

【凭证审核与取消审核】![](https://vipstatic.obs.cn-north-4.myhuaweicloud.com/statics/emotion/define/86.gif)![](https://vipstatic.obs.cn-north-4.myhuaweicloud.com/statics/emotion/define/86.gif)

## 解析：![](https://vipstatic.obs.cn-north-4.myhuaweicloud.com/statics/emotion/define/93.gif)![](https://vipstatic.obs.cn-north-4.myhuaweicloud.com/statics/emotion/define/93.gif)

1、可对任意期间凭证进行审核与取消审核操作。

2、暂存、作废凭证不可进行审核与取消
审核操作。

3、凭证审核与系统参数设置关联

①如果在系统参数中设置为“凭证审核与制单不能为同一人”，则在凭证审核时，
系统会进行判断并控制制单与审核不能为同一人。

② 如果在系统参数中设置“凭证审核与取消审核必须为同一人”，则在凭证反审核时，系统会进行判断并控制反
审核与审核必须为同一人。
