# 凭证打印时，科目编码、名称、辅助核算编码、名称不需要全部都打印，怎么调整哪些打印，哪些不打印？

> 作者：Zyl | 阅读：2188 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/420591154338344448)

****

**1、场景介绍**

凭证打印时，科目编码、名称、辅助核算编码、名称不需要全部都打印，怎么调整哪些打印，哪些不打印？

2、解决方案

1、系统设置-打印模板-账务处理-录凭证-选择一个模板，点击“复制”，输入模板编码、模板名称

![](https://vip-admin.kingdee.com/download/0109b2f5f8772fe743dba690076e52323c64.png)

2、通过凭证模板编码，打开此模板

![](https://vip.kingdee.com/download/01090532483a34f242589875d532864f6664.png)



3、如果需要打印科目编码+科目名称+辅助核算编码+辅助核算名称，则选择科目字段，点击右边控件属性下的单元格类型为“字段”，数据源字段选择“单据体”，绑定字段选择“科目编码_科目名称_辅助核算”

![](https://vip.kingdee.com/download/01097e00765741c4425890b15b2030676251.png)

4、如果只需要打印科目编码+科目名称，不需要显示核算项目，则绑定字段中选择“科目编码_科目名称”：

![](https://vip.kingdee.com/download/010962f5d2a53e2b4401b7c17d6962b675fe.png)



5、可以设置组合字段，比如需要显示科目编码+科目名称+辅助核算名称，不需要辅助核算编码，则绑定类型需要选择“组合字段”，数据源是“单据体”，组合字段中，选择 “科目编码_科目名称”+“辅助核算名称”。

![](https://vip.kingdee.com/download/0109e7024273f9ee41d78d0abaf293a24fc2.png)

中间如果需要分隔符，则点击自定义，在双引号里面输入分隔符。

6、注意：分隔符只能在最后设置，不能中间插入，所以建议在选择了“科目编码_科目名称”后，加上分隔符，再加“辅助核算名称。
