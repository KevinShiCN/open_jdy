# 明细账表上的参数的作用是什么

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：财税李炳全 | 阅读：95 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/568601860857394944)

【问题描述】

明细账表上的参数的作用是什么?

【解释说明】

明细类账相关参数说明如下:

1)显示辅助核算:开启后,挂核算项目的科目会按核算项目展开显示,核算项目显示在对应的科目下方。

2)辅助核算明细:可选某个具体的明细辅助核算进行查询

3)只显示最明细级:科目树只显示最明细科目。

4)显示非现金银行科目对方科目:所有科目都会显示对方科目列,现金和银行类科目不受此参数影响,默认显示对方科目列

5)余额为 0 不显示:期末余额为 0 的科目不显示。

6)无发生额且余额为 0 不显示:同时满足无发生额且无余额的科目都不显示出来。

7)现金银行存款科目显示日合计:针对现金、银行存款类科目增加本日合计行,按日期汇总当天的借方金额、贷方金额、余额

8)发生额为 0 不显示本期合计、本年累计:没有发生额的科目不显示本期合计、本年累计两行。
