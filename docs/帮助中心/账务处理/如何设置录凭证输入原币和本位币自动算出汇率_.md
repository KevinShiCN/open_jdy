# 如何设置录凭证输入原币和本位币自动算出汇率?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：564 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/634557427589073920)

【问题描述】

科目开启了外币核算,如何设置录凭证时输入原币和本位币自动算出汇率?

【解决方案】

第一步:科目要开启外币。参考[科目如何设置外币核算?](https://vip.kingdee.com/school/detail/273853718859403776?productLineId=31&lang=zh-CN)

第二步:基础资料-币别,币别设置浮动汇率

![](https://vip.kingdee.com/download/0109d504fe0f962d43b3bd35e3059ec9c40a.png)

第二步:在”凭证录入“-”偏好设置“中设置反算规则为”反算汇率“即可

![](https://vip.kingdee.com/download/01097c47b784f71440b29e4d7b9000b451d9.png)
