# 固定资产卡片生成的凭证如何带出资产明细和资产类别?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：131 | 有用：3

【问题描述】

固定资产卡片生成的凭证如何带出资产明细和资产类别?

【解释说明】

录入固定资产卡片后,可将其生成新购业务的记账凭证。通常情况下,不建议对“固定资产”科目设置明细科目或辅助核算,**科目辅助核算维度也无法取到资产类别和资产明细的数据**,如需查询具体资产的明细数据,可直接查阅折旧明细表和折旧汇总表。

若需在总账中反映固定资产的类别和明细数据,则可将固定资产科目设置明细科目。如需按资产类别进行统计,可设置相应的二级科目;如需进一步体现具体资产名称,可在二级科目下增设三级明细科目。

【操作指引】

**一、固定资产科目不设置明细**

固定资产科目不设置明细科目和辅助核算,总账中只显示“固定资产”一级科目数据,如需查看资产明细,建议查看折旧明细表数据或者是折旧汇总表的数据

1.折旧明细表,会显示每一张卡片的原值,每月折旧额等相关信息

路径:【资产管理】>【折旧明细表】

![](https://vip.kingdee.com/download/0109fd2787663aca4f84896d3f5c53c95e10.png)

2.折旧汇总表,会按资产类别汇总显示折旧信息

路径:【资产管理】>【折旧汇总表】

![](https://vip.kingdee.com/download/0109ece3a94d468645448c78319328db3136.png)

**二、固定资产设置明细**

若需在总账中显示固定资产类别及资产明细,建议先按固定资产类别设置二级明细科目,再在各二级科目下新增三级明细科目,并录入具体资产名称。****

新增明细科目详细步骤可以点击: [如何新增财务科目?](https://vip.kingdee.com/link/s/Zz5iY)

在新增卡片时固定资产科目可以选到明细科目,生成凭证时则会带出明细科目

![](https://vip.kingdee.com/download/01090d02b66395a5421b963899a3ea197eb6.png)
