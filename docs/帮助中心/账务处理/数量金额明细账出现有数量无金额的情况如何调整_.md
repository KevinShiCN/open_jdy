# 数量金额明细账出现有数量无金额的情况如何调整?

【问题描述】

  数量金额明细账出现有数量无金额的情况如何调整

![](https://vip.kingdee.com/download/01099693cdecadc24a879804d92cc8fd13f4.png)

【解决方案】

  手工录入一张借贷方都是同一个科目的凭证,金额一样,贷方录入数量。金额和数量进行冲抵,使得最终余额是平的。![](https://vip.kingdee.com/download/01099ea0cb14c37e4d1290b3bd1f6bb00125.png)
