# 生成凭证时勾选了多个合并条件，但是没有合并生成一张凭证是怎么回事？

生成凭证时勾选了多个合并条件，但是没有合并生成一张凭证是怎么回事？

【答案解析】![气球](https://cdn-vip.kingdee.com/statics/emotion/define/101.gif)![气球](https://cdn-vip.kingdee.com/statics/emotion/define/101.gif)

生成凭证的条件可以根据业务场景自由勾选，不同的条件生成的凭证也不同

- 单个合并条件：勾选供应商，采购入库单汇总生成凭证时，会将不同供应商的采购入库单，分别生成凭证
- 多个合并条件：勾选供应商、单据日期，采购入库单汇总生成凭证时，会将不同供应商且单据日期相同的采购入库单，分别生成凭证![上传图片](https://vip.kingdee.com/download/0100f13716a1b52642a083d549970e4dbdb3.png)
