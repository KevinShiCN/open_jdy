# 打印凭证可以打印数量和单价吗?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：杨桂平 | 阅读：1036 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/396830120725418752)

【问题描述】

打印凭证可以打印数量和单价吗?

【业务场景】

科目启用了数量金额核算,录入凭证时也录入了对应的数量和单价,那么可以将凭证分录中的数量和单价也打印出来吗?

![](https://vip.kingdee.com/download/01099a067658890741a58bec81aea2927172.png)

【操作步骤】

凭证打印设置中可以选择系统预置的有“数量外币”字样的模板后再进行打印。

![](https://vip.kingdee.com/download/0109c6d37dea74e74c9e99c78c6e40543f98.png)

预览效果

![](https://vip.kingdee.com/download/0109446924fbd96f4771a427ab8a8b015e13.png)

【温馨提示】

若系统预置的凭证模板不满足打印需求,可以在【系统设置】-【打印模板】中复制系统预置的模板再进行调整。
