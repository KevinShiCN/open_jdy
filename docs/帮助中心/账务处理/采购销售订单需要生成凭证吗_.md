# 采购销售订单需要生成凭证吗?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题

【问题描述】

采购销售订单需要生成凭证吗?

【解释说明】

**订单只是计划单,不影响实际出库入库库存,不需要生成凭证。**需要使用已审核销售出库单/采购入库单生成凭证

【名称解释】

销售订单:是计划单,预计销售单,不会影响即时库存,可能会影响可用库存;

销售出库单:是出库单,会减少库存,可以由销售订单生成销售出库单,也可直接录销售出库单,订单为非必录单据。
