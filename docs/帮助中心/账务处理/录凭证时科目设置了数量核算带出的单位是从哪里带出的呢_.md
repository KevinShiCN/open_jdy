# 录凭证时科目设置了数量核算带出的单位是从哪里带出的呢?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：云社区用户6Rw2283 | 阅读：1206 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/472206605040067328)

【业务场景】

    科目启用了数量核算,录入凭证时单位不是想要的单位,怎么调整单位呢?

  【解决方案】

   1、科目中没有设置商品辅助核算:会根据科目资料中设置的单位显示,设置那个单位就显示那个单位,没有设置单位那么录凭证就不显示单位。

![](https://vip.kingdee.com/download/010957f894685b5945e2abafa8ef28861297.png)2、科目中设置了商品辅助核算:凭证没有选择商品会显示科目资料的单位,选择具体的商品后会带出商品资料的基本单位。

![](https://vip.kingdee.com/download/0109d1048623cf164173a0ce820cf913baa6.png)
商品资料详情,在【基础资料】-【商品信息】界面打开,如图:
![](https://vip.kingdee.com/download/0109d724398b67d249d69f2f4afa916228ae.png)

  【温馨提示】

1、科目使用后科目资料中的单位也可以修改;

2、可以在【基础资料】-【商品管理】中点击商品编码查看基本单位;

3、商品资料设置了多单位也只能带出基本单位。
