# 业务凭证模板已设置好,取数和科目匹配都正确的情况下,为什么无法带出对应的科目与金额?

> 作者：杨桂平 | 阅读：200 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/537798974791719424)

【问题描述】

业务凭证模板已设置好,取数和科目匹配都正确的情况下,为什么无法带出对应的科目与金额?

【解决方案】

 如果业务凭证模板都已经设置好,但是仍无法取数,请检查每行分录后面的条件限制,不满足条件的取值、科目不会带出来
