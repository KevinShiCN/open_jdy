# 单据金额为0时是否能生成凭证?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：1108 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/560266935770901760)

【问题描述】

单据金额为0时是否能生成凭证?

【业务场景】

业务单据只录入了数量,没有录入金额,能否生成凭证?

【解释说明】

1、财务凭证需要有借贷发生额才能生成凭证,所以当业务单据整单金额为零时该张单据无法单独生成凭证;**金额为0的单据不生成凭证也不影响结账,若[财务结账提示有单据未生成凭证](https://vip.kingdee.com/link/s/Z2Iaj),检查发现是金额为0的单据,可忽略提示继续结账。**

如图,零金额单据生成凭证金额为0无法保存。

![](https://vip.kingdee.com/download/0109a48f4e55a2f7465e80b2beaa49808b9a.png)
2、其中销售出库单除外,金额为零的销售出库单,没有产生销售收入,不需要生成确认销售收入的凭证,**但系统会正常核算销售出库成本,需要生成结转成本的凭证。**

![](https://vip.kingdee.com/download/010926a863efa59e4c6fb8b61f65d0844f70.png)
3、如果业务单据必须全部标识为生成凭证,可以将零金额单据与其他有金额的单据合并生成凭证,这样,生成的凭证可以取到有金额单据的金额,零金额的单据也会显示有凭证字号。

![](https://vip.kingdee.com/download/01093ab1d8b2223147cf852b492e7ab42613.png)
4、若是金额为0的单据需要过滤掉,只勾选金额不为0的生成凭证,可将鼠标放金额字段上,点击小漏斗选择条件“不等于”,输入0点击<确定>

![](https://vip.kingdee.com/download/010953b41bd4b3aa456996c401f30f64460e.png)
