# 期末结转里的计提附加税是如何取数的?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：Siwennn | 阅读：2201 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/614262897849294336)

【问题描述】

期末结转里的计提附加税是如何取数的?

【解释说明】

**计提附加税在期末智能结转页面,可根据需求自主选择是否生成凭证,系统支持不生成凭证的操作**

若想在系统自动生成计提附件税,可在【账务处理】>【期末智能结转】中,可直接点击生成凭证操作****

1、点击到“账务处理”模块中的“期末智能结转”中。

![](https://vip.kingdee.com/download/01091d27796b4d7c4be099bd42be881df43c.png)

2、在期末智能结转中,点击计提附加税模板的修改符号会弹出一个模板修改界面。在此界面中点击“按测算金额科目结转”右侧的“已设置”,可以查看计提附加税的计税依据。

![](https://vip.kingdee.com/download/01094b3d2b84c6a44e738ee63e40822227f4.png)
3、计税依据的取数逻辑是:本期应交的增值税和应交的消费税。

![](https://vip.kingdee.com/download/010924b6173e8b834aa6b425c33a17679945.png)

4、在模板界面中,模板中的结转比例分别是“城市维护建设税”、“教育费附加”和“地方教育费附加”的计提税率,分别为7%、3%和2%。设置完成后,点击<保存>

![](https://vip.kingdee.com/download/010964e6df75c00a438388b1d05dd28f2414.png)
5、设置完成后,点击右上角的<重新测算>,更新完成后,在计提附加税下,点击<生成凭证>即可
![](https://vip.kingdee.com/download/010954f0b47e13e04c3d8d8a0ddb9ea47a69.png)
