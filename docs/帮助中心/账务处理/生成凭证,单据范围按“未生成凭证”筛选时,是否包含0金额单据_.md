# 生成凭证,单据范围按“未生成凭证”筛选时,是否包含0金额单据?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：Cat_Chow | 阅读：345 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/532363034652838144)

【问题描述】

生成凭证,单据范围按“未生成凭证”筛选时,是否包含0金额单据?

【业务场景】

按业务单据生成凭证时,单据范围按“未生成凭证”条件筛选,是否会把0金额的单据也一起筛选出来?

【解释说明】

生成凭证时,单据范围按“未生成凭证”条件筛选,是否会把0金额单据筛选出来,取决定于【是否显示金额为0的单据】条件的选择:

如下图所示,【是否显示金额为0的单据】选择“是”,则会把0金额的销售出库单一起筛选出来;
![](https://vip.kingdee.com/download/0109e03631f6e55845a0a8d03c831965d30f.png)
【是否显示金额为0的单据】选择“否”,则不包含0金额的销售出库单。

![](https://vip.kingdee.com/download/01096fb6c37ff0444d468ce6c7fb15b75d29.png)
