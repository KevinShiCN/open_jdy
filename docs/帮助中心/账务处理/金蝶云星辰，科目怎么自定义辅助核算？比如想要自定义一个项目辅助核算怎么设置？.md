# 金蝶云星辰，科目怎么自定义辅助核算？比如想要自定义一个项目辅助核算怎么设置？

金蝶云星辰，科目怎么自定义辅助核算？比如想要自定义一个项目辅助核算怎么设置？

解决方案：

1、进入基础资料-辅助资料进入先添加一个项目辅助资料类别，之后再切换到新增的这个项目辅助资料类别下新增具体的项目信息，比如项目1，项目2，项目3；

2、进入基础资料 -科目进入选择需要启用项目辅助核算的科目，点击科目编码进入启用项目辅助核算；

3、录凭证选择对应的科目就可以选择对应的项目信息做凭证，查看账簿数据；
