# 勾选“基础资料禁用后，允许正常生成凭证”这个参数 ，是做什么呢？

勾选“基础资料禁用后，允许正常生成凭证”这个参数 ，是做什么呢？

【答案解析】![气球](https://cdn-vip.kingdee.com/statics/emotion/define/101.gif)![气球](https://cdn-vip.kingdee.com/statics/emotion/define/101.gif)

基础资料（如客户、供应商 ）已禁用，之前由禁用资料产生的业务单据数据允许正常生成凭证的。

已经发生的业务如果不入账的话，后续在查询、对账时，会增加数据核对不准确，甚至出现关联数据找不到的情况。

**注明：基础资料包括科目、所有辅助核算、凭证字**![嘻嘻](https://cdn-vip.kingdee.com/statics/emotion/define/2.gif)
