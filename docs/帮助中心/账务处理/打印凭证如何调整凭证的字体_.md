# 打印凭证如何调整凭证的字体?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：云社区用户6Rw2283 | 阅读：1082 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/562803556072483584)

【问题详情】

打印凭证如何调整凭证的字体?

【操作步骤】

1、可以在【系统设置】-【打印模版】-【财务处理】-【凭证】-【录凭证】中点击正在使用的打印模板编码,打开模板(如使用的是预置模板,需复制后修改)

![](https://vip.kingdee.com/download/0109c9a004ef96654037920ce8741d29c9f3.png)

2、在模板中选中需要修改字体的栏位,再点击上方调整字体的位置选择现有的字体,最后保存下模版,如下图:![](https://vip.kingdee.com/download/010952dbe200d48b43449c6c8f372597d3c0.png)
也可以调整字体大小、是否加粗、是否倾斜、加下划线、字体颜色等

![](https://vip.kingdee.com/download/0109d1c68626140341c6908b34256f07d2c3.png)
