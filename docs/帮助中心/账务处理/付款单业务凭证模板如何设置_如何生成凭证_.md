# 付款单业务凭证模板如何设置?如何生成凭证?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：云社区用户0531183 | 阅读：641 | 有用：1

【问题描述】

付款单业务凭证模板如何设置?如何生成凭证?

【解决方案】

系统有预置业务凭证模板,可以直接使用,也可以按企业实际业务情况调整或是自定义新的业务凭证模板。

一、 进入业务凭证模板编辑界面

1.点击【账务处理】-【业务凭证模板】

![](https://vip.kingdee.com/download/0109b30ac435a32b44be8946e87cdf046dd9.png)

2. 左侧单据列表点击【应收应付】-【付款单】,点击系统自带的付款单模板点【编号】进入模板修改界面或者直接新增一个空白模板。

![](https://vip.kingdee.com/download/0109b5ea42f344d3474b96611e9bf705ced5.png)

二、对应的业务凭证模板设置

1.科目来源:借方【供应商.应付账款科目】,贷方【账户管理.对应科目】,如果有付款优惠、手续费需设置对应的自定义借方科目,例如【财务费用-手续费】。

![](https://vip.kingdee.com/download/0109f8b5f4fbb94d4942b8bad2575a6bd66d.png)

如果应付账款设置了辅助核算,需要在凭证模板-核算维度处,选择对应的辅助核算对应单据字段:

![](https://vip.kingdee.com/download/010993dac472191d4bf49ec1cbe409a518c4.png)
2. 金额取数:

供应商应付账款科目金额取值取“{源单信息.本次核销金额}”,账户管理对应科目金额取值取“{付款信息.付款金额}+{付款信息.手续费}”。付款优惠对应科目金额取值取“-{优惠金额:}”,手续费金额取值取“{付款信息.手续费}”。

![](https://vip.kingdee.com/download/0109e6fc3527930340c886ad56f92016f1d5.png)
