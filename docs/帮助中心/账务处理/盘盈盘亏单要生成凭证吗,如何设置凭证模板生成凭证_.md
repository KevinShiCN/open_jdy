# 盘盈盘亏单要生成凭证吗,如何设置凭证模板生成凭证?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：Schatz | 阅读：1301 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/633107419970472448)

【问题描述】

盘盈盘亏单要生成凭证吗,如何设置凭证模板生成凭证?

【解释说明】

盘盈盘亏单需要生成凭证。因为盘盈单会增加库存商品/原材料的入库合计的成本金额,盘亏单会增加库存商品/原材料出库合计成本金额,从而都会影响库存商品/原材料期末的期末库存的成本金额。所以需要生成凭证,才能在财务模块反应出库存商品/原材料科目余额的增加与减少。

【操作步骤】

1、先确保盘盈盘亏单上已经有成本金额。盘盈单上可手工录入成本,盘亏单需审核之后做出库成本核算返写成本。

详细可以点击:[盘盈盘亏单如何录入成本?](https://vip.kingdee.com/link/s/Z7ZrM)

![](https://vip.kingdee.com/download/0109448cca2ad72e426fbddc42c6e19e6013.png)
![](https://vip.kingdee.com/download/010988aa496057834d1898f077344fee59cc.png)

2、设置业务凭证模板,菜单栏点击【账务处理】>【业务凭证模板】

![](https://vip.kingdee.com/download/01096b66a4a303e64d269d4d014ed1f3134a.png)

3、找到盘盈单与盘亏单的模板,打开相应模板,为盘盈单设置借方科目为商品库存对应科目,并自定义贷方科目;盘亏单,则自定义借方科目,贷方科目设置为商品库存对应科目

**盘盈单凭证模板:**

![](https://vip.kingdee.com/download/01092e47dfec53d146529b8dc49f936dbcf8.png)

①库存商品如设置了辅助核算,需要设置辅助核算维度,生成凭证带出商品明细信息

②如启用了数量核算,数量、单位、单价也需要进行取值

③贷方自定义科目可以根据公司业务自定指定

**盘亏单凭证模板:**

![](https://vip.kingdee.com/download/0109f974ae2399664951ba9bf595ba04b407.png)

①盘亏单借方科目,根据公司业务自定义设置

②库存商品如设置了辅助核算,需要设置辅助核算维度,生成凭证带出商品明细信息

③如启用了数量核算,数量、单位、单价也需要进行取值

4、根据商品性质(产成品或原材料)设定相应的库存对应科目,详细可以点击:[商品对应科目如何设置?](https://vip.kingdee.com/link/s/Z0Bhu)

路径:【账务处理】>【科目关联设置】

![](https://vip.kingdee.com/download/01091266ca67a7fa415aa045c80b35675a36.png)

5、最后,在财务处理-生成凭证界面,选取已处理好的盘盈盘亏单,直接生成相关凭证。

![](https://vip.kingdee.com/download/0109bc9ee7c5f8484f07a131924187763c73.png)
