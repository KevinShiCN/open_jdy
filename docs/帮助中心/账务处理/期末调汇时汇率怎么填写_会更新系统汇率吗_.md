# 期末调汇时汇率怎么填写?会更新系统汇率吗?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题

【问题描述】

期末调汇时汇率怎么填写?会更新系统汇率吗?

【解释说明】

期末调汇是指在会计期末(通常为每月的最后一日),对外币账户的余额按照期末汇率进行折算,以计算汇兑损益的过程。

期末调汇时需手工录入汇率进行调汇,**月末汇率**通常指月末最后一个交易日的外汇中间价 / 基准汇率,可通过权威官方渠道获取,如**中国人民银行官网、****国家外汇管理局等。**

在期末调汇录入了汇率不会影响历史凭证,会返写到币别汇率,即下次录入外币业务时会默认带出上一次调汇录入的汇率。
