# 财务与业务对账界面科目金额的取数规则是什么?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：云社区用户55os622 | 阅读：269 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/660286818390157056)

【问题描述】
财务与业务对账界面科目金额的取数规则是什么?

例如:除了库存商品外,还想要核对原材料科目,为什么对账界面没有原材料科目?

![](https://vip.kingdee.com/download/0109a1956049e00542e0b9f4f0a5c5b15c54.png)

【解释说明】

财务业务对账是**进销存业务中的商品收发明细及余额**,与**对应的科目在总账的借贷发生额及余额**进行数据核对

参与核对的科目是:**科目关联设置中商品类别、商品指定的库存商品科目**。未在科目关联设置中设置的科目将不会参与数据核对。因此,对账前需先设置好商品的**【****库存对应科****目****】,**按商品或商品类别设置均可。

依次点击【账务处理】-【科目关联设置】![](https://vip.kingdee.com/download/01097151fc84371c4c7594bd5ad11f09c192.png)

点击进入 【商品】/【商品类别】
![](https://vip.kingdee.com/download/010937ec8a6978ce4a078409464a3db4c8eb.png)
左上角可设置通用科目, 也可根据不同的商品明细设置不同的库存商品对应科目
![](https://vip.kingdee.com/download/01095314f6c906084f8b910365362d94c34f.png)
