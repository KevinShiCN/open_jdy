# 账表查询

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：功能介绍, 快速入门
> 作者：Cat_Chow | 阅读：4864 | 有用：9
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/630528007349830400)

凭证保存成功后，即可生成账薄报表。包括主账薄（明细账、科目余额表、总分类账、费用明细表和多栏账），三大主表（资产负债表、利润表和现金流量表），还有一些其他辅助账薄和报表查询。

![](https://vip.kingdee.com/download/0109595807214282478b930c180201c4fa66.png)

| 步骤 | 具体说明 | 学习链接 |
| --- | --- | --- |
| 1.账薄 | 账薄除了常用的主要账薄外，还有核算项目账薄及数量金额账薄等。不同的账薄查询条件和展示格式各有不同，所有账薄都支持打印，导出。 | —— |
| 1.1主账薄 |  | —— |
| 1.1.1明细账 | 指明细科目开设的账户，对分类账的经济内容进行明细分类核算，提供具体而详细的核算资料。系统提供明细账的查询功能，对每一科目在不同期间的余额和发生额进行统计显示。 | 点击学习 |
| 1.1.2科目余额表 | 对查询期范围内所有发生业务的科目金额进行汇总，包括此期间范围内的期初余额、本期发生额、本期累计发生额和期末余额。 | 点击学习 |
| 1.1.3总分类账 | 指按总账科目开设的账户，对总账科目的经济内容进行总括的核算，提供总括性的指标。系统提供总分类账的查询功能，对所选科目在不同期间的余额和发生额进行统计显示，并提供联查到明细账的功能。 | 点击学习 |
| 1.1.4费用明细表 | 反映企业在一定期间内发生的财务费用、管理费用、销售费用等科目及其构成情况的报表。 | 点击学习 |
| 1.1.5多栏账 | 指在一张账页上，不仅按借、贷、余三部分设立金额栏，还要按明细科目在借方或贷方设立许多金额栏，以集中反映有关明细项目的核算资料。 | 点击学习 |
| 1.2核算项目账薄 | 科目启用辅助核算设置，可按辅助核算类别或是辅助核算明细等，多维度查询。 | —— |
| 1.2.1核算项目明细账 | 对科目启用了辅助核算,并进行了相关的账务处理以后,我们可以用核算项目明细账来以辅助核算+科目的维度来查询某个辅助核算名下各个科目的发生额。   | 点击学习 |
| 1.2.2核算项目组合表 | 支持双辅助核算+科目二维查看,设置了多辅助核算类别的科目,可在核算项目组合表界面选择科目及多个辅助核算组合查询。   | 点击学习 |
| 1.2.3核算项目综合表 | 用于帮助用户进行辅助核算项目与科目组合及余额的分析 | 点击学习 |
| 1.3数量金额账薄 | 科目启用数量金额核算，录入凭证时同时录入数量及单价，即可在数量金额账薄体现及查询。 | —— |
| 1.3.1数量金额明细账 | 统计显示出数量金额核算科目在不同期间的数量、单价及发生额。数量金额明细账适用于既需要提供价值信息，又需要提供金额信息的实物资产 | 点击学习 |
| 1.3.2数量金额总账 | 对所选科目在在不同期间的数量、单价及发生额进行统计显示，并提供联查到数量金额明细账的功能 | 点击学习 |
| 1.4调汇历史信息表 | 科目启用外币核算，并由系统生成期末调汇凭证，会将调汇记录写入调汇历史信息表进行查询。如果同一期间重新生成凭证时调汇历史信息表会自动更新。 | 点击学习 |
| 2.报表 | 除了三大报表，系统支持项目利润表及自定义报表 | —— |
| 2.1资产负债表 | 系统预置报表，支持编辑报表项目及报表公式、自动重分类往来项目等操作 | 点击学习 |
| 2.2利润表 | 利润表支持月报和季报，可根据利润表数据生成利润变化趋势图表，直观看到企业收入、成本和费用等情况。 | 点击学习 |
| 2.3现金流量表 | 现金流量表支持月报和季报两种类型，可根据业务需要进行选择查询。 | 点击学习 |
| 2.4项目利润表 | 为适应企业精细核算管理需求，在损益科目设置辅助核算后（如部门、项目），支持按辅助核算出具对应利润表。 | 点击学习 |
| 2.5自定义报表 | 企业可以根据需求进行个性化报表设计，以满足企业经营管理需要。系统预置的资产负债表和利润表可另存为自定义报表，并将默认公式带到自定义报表中。支持将在EXCEL编辑好的自定义报表导入系统中使用。 | 点击学习 |
| 2.6报表不平处理方案 | 资产负债表不平的几种常见情况 | 点击学习 |
| 资产负债表与利润表勾稽关系不对   | 点击学习 |  |

| 现金流量表提示不平衡,检查方法 | 点击学习 |  |
| 3.多账套报表 | 企业有多家子公司，并且子公司都是用同一服务的金蝶云·星辰系统处理账务，如：ABC三家子公司，分别建立了3个子账套做账，月末需要看3家子公司的汇总报表时可通过多账套报表查询。  说明：该功能专业版及旗舰版均支持，标准版不支持。 | —— |
| 3.1多账套资产负债表 | 实现多套账的资产负债表同时查询 | 点击学习 |
| 3.2多账套利润表 | 实现多套账的利润表同时查询 | 点击学习 |
| 3.2多账套现金流量表 | 实现多账套的现金流量表同时查询 | 点击学习 |
| 3.3多账套科目余额表 | 实现多账套的科目余额表同时查询 | 点击学习 |
| 3.4多账套费用明细表 | 实现多账套的费用明细表同时查询 | 点击学习 |
| 4.智能查询报表 | (1)系统通用的财务账表无法满足查询需要(2)报表的行列设置有一定规律 目前支持多个维度的数据查询分析，支持自定义行列公式目前查询分析的数据源仅跟于:总账科目余额表相关的数据、法定会计报表；支持两种数据源选择【科目余额表】和【会计报表】。说明：该功能专业版及旗舰版均支持，标准版不支持。 | —— |
| 4.1智能查询方案设置 | 多元化的，自定义查询方案，直观灵活 | 点击学习 |
