# 科目余额表的本期数据与本年累计数据取值规则是什么?

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：shaoling | 阅读：556 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/709933546227126272)

【问题描述】

科目余额表的本期数据与本年累计数据取值规则是什么?

【业务场景】

科目余额余额表选择多期查询时,本期发生额数据怎么与本年累计数据显示不一致?

![](https://vip.kingdee.com/download/01095b20f49dd84d4a5eb2a52ec8d37576e5.png)

【解释说明】

科目余额表的本期发生额是指**选择期间的累计数据;**

本年累计数据:若跨年查询,取所选期间最后一年的本年累计数据,若不跨年查询,则查询数据为当年累计的数据;

【应用举例】

**一、跨年查询**

若科目余额表选择了24年1期到25年5期的数据查询:

①本期发生数据是24年1月到25年5月的累计数据,

②本年累计只统计到25年1月到5月的数据,

会出现则本期发生数据跟本年累计数据不一致的情况

![](https://vip.kingdee.com/download/0109204ce2ea95fa4fa1910ccdbdc00f3516.png)

**二、不跨年查询**

若科目余额表选择了25年1期到25年5期的数据查询:

①本期发生数据是25年1月到5月的累计数据,

②本年累计也是统计25年1月到5月的数据,

则会出现本期发生数据跟本年累计数据一致的情况
![](https://vip.kingdee.com/download/0109fe8d7e94e9384f829ed39c6fc43cd2d9.png)

若科目余额表只选择25年5期的数据查询:

①本期发生数据是25年5月份的数据,

②本年累计数据统计25年1月到5月的数据,

则会出现本期发生数据与本年累计数据不一致的情况

![](https://vip.kingdee.com/download/0109fd04077597ef4f51ab9075257177cebb.png)
