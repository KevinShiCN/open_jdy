# 如何设置子账号只能查询到自己录入的凭证

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：常见问题
> 作者：Cat_Chow | 阅读：456 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/532363038830364928)

【业务场景】

设置子账号只能查询到自己录入的凭证,其他人员录入的凭证查询不到

【操作步骤】

可以通过【权限设置】-【数据授权】-【制单人】来实现

1,管理员账号登录,在【系统设置】-【用户管理】-选择要授权的子账号,点击【授权】

![](https://vip.kingdee.com/download/0109c5f0209a7ad34ab4b5ba4640e42fa582.png)

2,在【数据授权】-【制单人】-【添加条件】,实际现数据控制

![](https://vip.kingdee.com/download/0109bab4b627371848b398e28fff7c5ebb25.png)

3,如管理员账号在查凭证里,是可以查询到本期有8张凭证

![](https://vip.kingdee.com/download/01093d9fa1ea50d946198acf696c28d6d59c.png)子账号登录后,只能看到【制单人】为自己名字的一张凭证,其他人员录入的凭证查询不到

![](https://vip.kingdee.com/download/010907dbfa3aa6fa4df0a3888561f88ac297.png)
