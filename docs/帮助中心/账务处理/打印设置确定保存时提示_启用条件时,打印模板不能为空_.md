# 打印设置确定保存时提示:启用条件时,打印模板不能为空?

> 作者：云社区用户8x80607 | 阅读：54 | 有用：1

【问题描述】

打印设置确定保存时提示:启用条件时,打印模板不能为空?![](https://vip.kingdee.com/download/0109e5f86cbfc7e447b9b9ea66221cf52a97.png)

【解释说明】

出现以上提示,是因为按条件设置中开启了“启用”按钮,若需设置打印条件,需补充完成过滤条件及打印模板,若不需要按条件设置打印,则关闭“启用”按钮
