# 科目关联设置中商品对应的收入对应科目必须设置吗?

> 作者：云社区用户u1tk1853 | 阅读：310 | 有用：0

【问题描述】

科目关联设置中商品对应的收入对应科目必须设置吗?

【解释说明】

科目关联设置里面的商品对应收入科目系统没有限制必须要录入。但是涉及到凭证的生成,一般建议设置关联科目。
