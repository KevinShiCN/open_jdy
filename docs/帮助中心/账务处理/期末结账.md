# 期末结账

> 产品：金蝶云·星辰 | 分类：账务处理 | 类型：功能介绍, 快速入门
> 作者：Cat_Chow | 阅读：3671 | 有用：7
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/632150685907437568)

结账是把一定时期(月度、季度、年度)内发生的全部经济业务登记入账的基础上,计算并记录本期发生额和期末余额。

| 1.结账前检查 | 可设置结账前检查项，提高账务的准确性 | —— |
| 1.1检查项设置 | 包括系统参数检查设置、期末结转检查设置、其他检查项设置等。 | 点击学习 |
| 1.2异常处理 | 结账前检查结果分为异常项目和警告项目 | —— |
| 1.2.1异常项目 | 红色字体标注，必须解决后，方能结账至下一期。如资产负债表不平、资产未计提折旧（资产与总账连用）等。 | 点击学习 |
| 1.2.2警告项目 | 橙色字体标注，仅警告提醒，不处理不影响结账至下期。如货币资金科目是否赤字、与资产对账是否平衡等。 | 点击学习 |
| 2.结账处理 |  | —— |
| 2.1结账 | 把一定时期（月度、季度、年度）内发生的全部经济业务登记入账的基础上，计算并记录本期发生额和期末余额。各个期间的期末结账完成状态有先后依赖关系，上一期未完成期末结账，则下一期不能进行期末结账。在某一期完成期末结账之后，系统控制不允许再发生业务操作（包括凭证的增删改），系统已结账。 | 点击学习 |
| 2.2反结账 | 系统支持跨期结账、跨期反结账。 | 点击学习 |
| 3.其他模块结账处理 |  | —— |
| 3.1资产结账 | 未启用资产与总账结账，即资产单独结账，且需提前结账，再进行总账结账 | 点击学习 |
| 3.2出纳结账 | 未启用出纳与总账结账，即出纳单独结账，且需提前结账，再进行总账结账 | 点击学习 |
| 3.3业务结账 | 开通进销存云模块，可以分别设置：①业务系统单独结账，且总账无关联；②业务系统单独结账，且必须早于总账结账。业务结账相关介绍及操作步骤可参考【业务结账处理】。 | 点击学习 |
