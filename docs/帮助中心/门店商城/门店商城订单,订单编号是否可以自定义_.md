# 门店商城订单,订单编号是否可以自定义?

> 作者：happy忆霖 | 阅读：63 | 有用：0

【问题描述】

门店商城订单,订单编号是否可以自定义?

![](https://vip.kingdee.com/download/01094f13df4245c24beaa508f0738befb940.png)

【解决方案】

目前暂时不支持自定义。

商城订单编号由系统自动生成的,暂时无法进行修改。
