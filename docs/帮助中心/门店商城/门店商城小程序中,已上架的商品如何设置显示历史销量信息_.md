# 门店商城小程序中,已上架的商品如何设置显示历史销量信息?

> 产品：金蝶云·星辰 | 分类：门店商城 | 类型：常见问题
> 作者：happy忆霖 | 阅读：199 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/625134662029083136)

【问题描述】

门店商城小程序中,已上架的商品如何设置显示历史销量信息?

【业务场景】

想要在小程序端看到商品信息的历史销量,如何进行设置?

【解决方案】

点击门店商城-商城配置,进入到商城配置详情的界面

![](https://vip.kingdee.com/download/0109777598facde04a42a625e56c2a8c0bc1.png)

点击其他参数配置,将包含历史销量这个参数进行启用,然后保存设置

![](https://vip.kingdee.com/download/010916df51c1b0634b0cbe52903c2ac44014.png)

点击门店商城-商品资料中,找到需要配置的商品信息,点击进入

![](https://vip.kingdee.com/download/01090ec25ee3724c4b738d69f3b91232f7e4.png)在商品详情的界面,点击历史销量,输入此商品历史的销售数量,保存即可

![](https://vip.kingdee.com/download/01090218cdd9e8154b418e892371b71aba66.png)

在小程序端即可看到热销+数量,进行显示了

![](https://vip.kingdee.com/download/010922fc9fc42de942b08eaa06d6d9b4d3f0.png)

PS:此处的历史销量是包含了近期的销量的数据的,也就是历史销量+实际销量=热销量,且目前系统如果历史销量+实际销量超过500+后,系统会默认显示为热销量1000+,表示商品很促销的意思

![](https://vip.kingdee.com/download/01094f49972328c54fc683995eeda696d523.png)
