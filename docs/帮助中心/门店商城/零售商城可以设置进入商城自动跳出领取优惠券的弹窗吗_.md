# 零售商城可以设置进入商城自动跳出领取优惠券的弹窗吗?

> 产品：金蝶云·星辰 | 分类：门店商城 | 类型：常见问题
> 作者：云社区用户7L8O0643 | 阅读：149 | 有用：0

【问题描述】

零售商城可以设置进入商城自动跳出领取优惠券的弹窗吗?

【概述】

目前门店商城首页方案中,商城装修暂时不支持进入商城自动跳出领取优惠券的弹窗。

【操作步骤】

1、主界面点击【门店商城】-【首页方案】点击【新增】一个方案或者打开已有方案进入商城装修。

![](https://vip.kingdee.com/download/01095e2fd1836a1f44aeb259320c094a3aee.png)
2、点击左边【优惠券】的营销组件,添加后鼠标选中显示的组件,右手边可以设置显示模型(平铺或轮播横幅)和显示样式(一行一个或一行两个)。

![](https://vip.kingdee.com/download/0109a325f19d85f04a5c8e11e6b6a4100f19.png)
