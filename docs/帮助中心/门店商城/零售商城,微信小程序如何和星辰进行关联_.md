# 零售商城,微信小程序如何和星辰进行关联?

> 产品：金蝶云·星辰 | 分类：门店商城 | 类型：常见问题
> 作者：happy忆霖 | 阅读：740 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/598680434846854912)

【问题描述】

零售商城,微信小程序如何和星辰进行关联?

【业务场景】

想要设置将微信小程序和星辰进行关联,形成对接,方便用户在小程序端进行下单。

【解决方案】

需要商家先注册微信小程序,注册步骤可以参考此链接:**[小程序注册流程](https://kf.qq.com/faq/170109iQBJ3Q170109JbQfiu.html)**

在小程序开通完成后,可以在门店商城-商城配置,小程序授权中关联已经注册小程序,点击已有小程序,立即绑定

![](https://vip.kingdee.com/download/01095c37c32c296a423bb8c0b5f58e0f45eb.png)

登录微信公众平台-小程序后台,前往《开发-开发管理-开发设置》查看App ID和App Secret

![](https://vip.kingdee.com/download/0109783abf1388c34bbc9db4843ff10fff54.png)

AppSecret:按获取途径找到「AppSecret(小程序秘钥)」,点击【重置】获取AppSecret。

App ID:按获取途径找到「AppID(小程序ID)」,获取App ID

用使用公众平台绑定的管理员个人微信号扫描,在手机端进行扫描确认

![](https://vip.kingdee.com/download/01092007eb37488e4d718ee131eb3eac2d2f.png)

使用小程序管理员微信扫码后,选择需授权绑定的小程序,进行授权确认,为保证正常使用全部功能,授权时请将所有权限统一授权给星辰,点击【确定授权】即可。确认授权后,星辰小程序界面会显示授权成功

![](https://vip.kingdee.com/download/010923d722e45095454d9668ccf3680f0780.png)

授权绑定小程序后,在【门店商城】-【商城配置】,小程序授权页面进行授权,按照下图中框出的步骤进行操作

![](https://vip.kingdee.com/download/0109716516bab54c412695f45683d2f57b7c.png)

如审核失败,请根据提示查看具体的原因,修改后再次提交审核(微信官方审核,金蝶无审核权限,也无催单权限)

审核成功后,小程序系统管理员会收到成功的短信,再回到此页面点“发布”,这样才算发布完成,扫码进入小程序进行正常使用。

![](https://vip.kingdee.com/download/010960d57f99a9ab40a69baef6a5039218e0.png)

注意:微信官方审核小程序最新代码一般1-7天,微信审核通过后用户即可在微信搜索小程序进行体验。
