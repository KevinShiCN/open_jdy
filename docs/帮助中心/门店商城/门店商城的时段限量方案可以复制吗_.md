# 门店商城的时段限量方案可以复制吗?

> 产品：金蝶云·星辰 | 分类：门店商城 | 类型：常见问题
> 作者：ylim | 阅读：74 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/622960317101694720)

【问题描述】

门店商城的时段限量方案可以复制吗?

【业务场景】

需要新增一个时段限量方案,和之前旧的方案中商品信息一样,只是时间和限量数不一样,可以通过复制功能快捷创建一个方案,修改部分内容后保存启用。

【操作指引】

点击【门店商城】-【时段限量】

![](https://vip.kingdee.com/download/01099f1420fc0757491898103a11f8235e9d.png)
勾选要复制的方案,点击【复制】按钮

![](https://vip.kingdee.com/download/01098a6e01531ae2406e8a70c9586bdd286b.png)
在复制出来的界面,要修改哪些信息修改后点击右上角【保存】、【启用】按钮

![](https://vip.kingdee.com/download/01095e5e578a55044c53bce39a56f4423a74.png)
回到时段限量列表,可以看到新复制的方案

![](https://vip.kingdee.com/download/010976674a94578048c3ad4a244febb0093a.png)
