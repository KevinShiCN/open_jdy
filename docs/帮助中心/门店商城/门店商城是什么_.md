# 门店商城是什么?

> 产品：金蝶云·星辰 | 分类：门店商城 | 类型：常见问题
> 作者：云社区用户7Z97261 | 阅读：962 | 有用：1

【问题描述】

什么是门店商城?

【默认答案】

门店把店铺搬到线上,开展线上商城业务。购买商城服务后,需要在后台进行一系列配置,维护必须的资料和参数信息。维护好之后方可正常开展线上电商业务,支持消费者线上下单。
