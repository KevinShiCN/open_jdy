# 门店商城的商品资料列表中,部分商品信息,无法看到是什么原因?

> 产品：金蝶云·星辰 | 分类：门店商城 | 类型：常见问题
> 作者：happy忆霖 | 阅读：160 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/625134657516012032)

【问题描述】

门店商城的商品资料列表中,部分商品信息,无法看到是什么原因?

【业务场景】

基础资料-商品资料中存在这个商品信息,但是在门店商城-商品资料中看不到,是什么原因

![](https://vip.kingdee.com/download/01092aa9bced96b54a07b29c93f123225122.png)

【解决方案】

点击门店商城的商品资料列表,点击展开过滤,可以显示出状态字段

![](https://vip.kingdee.com/download/01090ced2b17b7604619bef79b65f436469c.png)

在状态中勾选上可用和禁用,点击刷新

![](https://vip.kingdee.com/download/01092fc4fdbf577840bb96b26ff80ce55dc7.png)

再次搜索需要的商品即可看到

![](https://vip.kingdee.com/download/0109f5712359a5934ee8a1a214512a19cd19.png)

若需要启用商品,需要在基础资料-商品信息中,找到对应的商品进行启用

![](https://vip.kingdee.com/download/01096b5c6eb918984404ab7480e1f3a7e163.png)
