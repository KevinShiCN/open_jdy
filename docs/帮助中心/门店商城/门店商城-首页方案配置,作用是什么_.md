# 门店商城-首页方案配置,作用是什么?

> 产品：金蝶云·星辰 | 分类：门店商城 | 类型：常见问题
> 作者：happy忆霖 | 阅读：167 | 有用：0

【问题描述】

门店商城-首页方案配置,作用是什么?

![](https://vip.kingdee.com/download/010949e276dd30334c35a1367e1a5af2f7ba.png)

【解决方案】

在门店商城关联小程序后,小程序的展示界面需要呈现给客户的时候,需要进行模板设计,使用首页方案配置,可以设置不同门店下,不同的方案展示。比如可以设置优惠券、充值赠送、商品的摆放位置,以及配置需要显示的商品信息、商品展示格式、价格显示信息等。
