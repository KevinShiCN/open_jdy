# 登录门店小程序一直提示:获取不到当前门店,请尝试刷新重新进入小程序?

【问题描述】

登录门店小程序一直提示:获取不到当前门店,请尝试刷新重新进入小程序?

【解决方案】

报错现象如下图:

![](https://vip.kingdee.com/download/010936ea8f7c594e400ebd5ae3891272b874.png)
首先确保门店商城配置**小程序授权**发布

![](https://vip.kingdee.com/download/01090350d87af4c94226a1344f18c177e76a.png)
其次要在门店列表配置新增有**线上门店**

![](https://vip.kingdee.com/download/010965c90918c8b24fd697ac99e88ffe9aa9.png)
最后登录门店的时候**开启定位**获取门店即可正常登陆门店商城小程序
