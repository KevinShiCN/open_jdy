# 基础资料商品/客户价格策略开启审核功能，在订货商城的影响及应用

> 产品：金蝶云·星辰 | 分类：门店商城 | 类型：功能介绍, 新特性, 快速入门
> 作者：商贸韩佩奇 | 阅读：245 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/683249112019611904)

1.1 功能介绍

-  未开启商品审核、价格资料审核，功能照旧不受影响，对应审核状态及按钮不显示。
- 开启商品审核：

![](https://vip.kingdee.com/download/01096104afbb96d2484782161eafc742e170.png)

对订货商城影响及应用：

（1）商品上下架列表增加显示列“审核状态”，支持筛选（PC和移动端），按钮“批量审核并上架”。

![](https://vip.kingdee.com/download/0109c2eabfd0007946ed9a12facaae0bb392.png)

（2）商品详情，引用基础资料字段同基础资料，根据状态控制是否能修改，不能修改置灰。

![](https://vip.kingdee.com/download/0109032003dd1b974192a17a18941fbdd995.png)

（3）商品推广、分享（PC、移动）不显示未审核的商品

（4）促销方案选赠品不显示未审核的商品

（5）订单优惠选赠品不显示未审核的商品

（6）商城端：

- 商品列表不显示未审核的商品
- 若分享商品后续商家操作反审核商品，则打开商品详情状态为未审核显示标签为“已下架”
- 若客户将商品加入购物车后续商家操作反审核商品，则该商品会在购物车显示为失效商品

（7）金蝶云app-订货商城-商品分享和上下架应用，增加审核状态筛选

![](https://vip.kingdee.com/download/01090bdbb42dc96143d2b7bd17fe286d983a.png)

- 开启价格资料审核：商城取价过滤，不再取值未审核的客户价格策略方案。
