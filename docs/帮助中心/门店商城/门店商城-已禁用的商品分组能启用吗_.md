# 门店商城-已禁用的商品分组能启用吗?

> 产品：金蝶云·星辰 | 分类：门店商城 | 类型：常见问题
> 作者：ylim | 阅读：58 | 有用：0

【问题描述】

门店商城-已禁用的商品分组能启用吗?

【解决方案】

点击【门店商城】-【商品分组】

![](https://vip.kingdee.com/download/0109f7e14774b9ea4769aae489fb262bed71.png)
勾选已禁用的商品分组,点击【启用】按钮即可。

![](https://vip.kingdee.com/download/0109da405ffc26d1490ba0de6391af5135ca.png)
