# 门店商城的小程序如何设置显示库存?

> 产品：金蝶云·星辰 | 分类：门店商城 | 类型：常见问题
> 作者：Siwennn | 阅读：336 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/626584351819578112)

【问题描述】

门店商城的小程序如何设置显示库存?

【解释说明】

门店商城的库存统计方式:
①当前线上门店未配置自提配货点及发货仓时,库存取线上门店仓库库存
②当前线上门店已配置自提配送点时,库存取当前匹配 / 选择的自提配送点库存
③当前线上门店已配置发货仓时,命中发货仓的商品,库存取对应发货仓库存

若需调整库存显示的位置和内容,到“门店商城”-“商城配置”模块进行设置。

![](https://vip.kingdee.com/download/0109d130fec526c549658d33830d2c212668.png)

进入商城配置界面后,选择“其他参数配置”,右侧向下滑动找到“库存显示配置”进行设置即可。

![](https://vip.kingdee.com/download/0109b732e897ae134e3db52e955bd4d8a119.png)
