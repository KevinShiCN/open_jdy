# 零售门店商城下单提示“收货人、发货人的区域不能为空"，如何解决？

1 问题描述

门店商城下单提示“收货人、发货人的区域不能为空"，如何解决？

## 2 适用版本

星辰3.0以上专业版/旗舰版

## 3 原因解析

出现该提示是因为门店资料未正确设置地址导致的。我们可以通过完善门店地址来解决这类问题。

## 4 解决方案

门店资料>行政区划，精确到县即可
