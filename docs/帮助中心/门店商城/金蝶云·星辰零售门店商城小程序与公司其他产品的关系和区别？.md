# 金蝶云·星辰零售门店商城小程序与公司其他产品的关系和区别？

> 产品：金蝶云·星辰 | 分类：门店商城 | 类型：进阶技巧, 精华汇总, 常见问题
> 作者：商贸韩佩奇 | 阅读：2224 | 有用：5
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/359015959975167232)

1. **Q：门店小程序商城和零售是相互独立的模块吗？**

A：门店小程序商城与零售不是独立的，小程序商城依存于零售，属于零售必不可少的一个重要模块。客户要使用小程序商城服务，必须购买零售模块。

1. **Q：小程序商城的功能亮点有哪些？**

A：小程序商城有以下亮点：

| 亮点 | 说明 |
| --- | --- |
| 线上线下，融合互通 | 线下门店 POS 线上小程序商城一体 化，实现商品、价格、库存、会员、积 分、充值、优惠券、订单互通。 实现线上下单线下自提，以及线上下单 线下配送等多种场景。 |
| 开箱即用，轻松上手 | 使用已有配置，5 分钟为线下门店建立 线上商城。也可从零开始，快速建立独 立线上门店。 容易上手的店铺装修功能，帮助进阶客 户快速配置店铺样式和营销组件 。 |
| 多种场景，灵活配置 | 可实现 1 家线下门店对应 1 家或多家 线上门店，也可分别配置线上和线下门 店。 门店可独立收款，支持连锁经营，适用 各种业务场景。 |
| 丰富营销，助力增长 | 划线价、商品分享、会员价、优惠券、 领券中心、储值、充值赠送、积分、促销… 多种营销方式，刺激消费者增购复购， 助力零售业务增长。 |

1. **Q：小程序商城可以和订货商城对接吗，还是只能对接零售？**

A：小程序商城本身是属于零售的一部分，不存在对接问题。至于是否可以和订货商城对接，目前暂无规划。

1. **Q：门店小程序商城与酷客多有什么区别？**

A：区别如下：

| 酷客多 | 门店小程序商城 |
| --- | --- |
| 酷客多是生态合作产品，需要双方对接。两者不是同一个产品体系，很多功能与数据无法对接打通。 | 门店小程序商城是金蝶云星辰零售自研的小程序商城，不存在与任何产品进行对接打通的概念。与星辰整个产品体系属于同一套，依赖于零售，客户要使用小程序商城服务，必须购买零售模块。目前已支持商品、库存、价格、会员、积分、储值、优惠券、支付等基础能力的线上线下一体。 |

1. **Q：小程序商城与易订货有什么区别？**

A：区别如下：

| 易订货 | 门店小程序商城 |
| --- | --- |
| 易订货是生态对接产品，需要双方对接，属于 B2B 产品，主要是给 B 端客户及其员工使用的。 | 门店小程序商城是金蝶云星辰零售自研的小程序商城，不存在与任何产品进行对接打通的概念。与星辰整个产品体系属于同一套，依赖于零售。属于 B2C 产品，主要是给 B 端客户及其会员粉丝使用的。目前已支持商品、库存、价格、会员、积分、储值、优惠券、支付等基础能力的线上线下一体。 |

1. **Q：小程序商城与订货商城有功能重复吗？两者有什么区别？整体流程和订货商城是一样的吗？**

A：关系和区别如下：

|  | 关系 | 区别 |
| --- | --- | --- |
| 订货商城 | • 小程序商城和订货商城同为金蝶云星辰的功能模块，互相独立，可分别购买使用。• 一个偏 B 端商城操作流程，一个偏 C 端商城操作流程，流程大致差不多，但有细节区别，两者非竞争关系，服务于不同的客户群体，不做功能对比，请自行关注。 | 订货商城主要针对批发行业，为 B 端客户提供批发采购的线上交易平台，推荐以批发销售为主要业务场景的客户购买。 |
| 门店小程序商城 | 小程序商城主要针对零售行业，帮助零售商家在线上开店，为 C 端用户提供线上购物平台。小程序商城可以满足更多与会员、营销相关的功能场景，同时小程序商城可与零售 POS 结合使用，更好地实现线上商城与线下门店的融合。对于零售行业客户，特别是已经有线下门店的客户，推荐使用小程序商城。 |  |
