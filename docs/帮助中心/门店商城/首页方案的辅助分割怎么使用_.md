# 首页方案的辅助分割怎么使用?

> 产品：金蝶云·星辰 | 分类：门店商城 | 类型：常见问题
> 作者：云社区用户8x80607 | 阅读：164 | 有用：0

【问题描述】

首页方案的辅助分割怎么使用?

【操作步骤】

1、门店商城-首页方案,点击方案编码/新建方案进入,左边可选择组件。选中组件添加完成,如下动图所示![](https://vip.kingdee.com/download/01099b5456b518a742b4a4731c8274c592de.gif)

2、组件选择完成,点击左边辅助分割,如下图所示,左边添加了辅助分割,与下面组件分隔![](https://vip.kingdee.com/download/0109055a803dbfea4efb9059ffd8e5a830f4.png)
3、预览效果如下图所示![](https://vip.kingdee.com/download/01095ba829c311a243de9e0509a058b28bff.png)

名称解释:

辅助分割:组件和组件之间,高度间隔的调节可以使用插入辅助分割线,一般建议使用背景颜色,具体可以根据喜好调整。![](https://vip.kingdee.com/download/01095b987491f1bc41b48184b84acabb30bf.png)
