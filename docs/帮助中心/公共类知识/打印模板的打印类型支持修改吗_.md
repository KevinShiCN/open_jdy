# 打印模板的打印类型支持修改吗?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题

【问题描述】

打印模板的打印类型支持修改吗?

【解释说明】

已经设置好的打印模版的打印类型无法修改,但是新增的时候可以自己根据需要选择打印模板类型,现在主要是常规和蓝牙打印:
蓝牙打印:仅适用于移动端连接蓝牙打印机,如蓝牙小票等设备。常规:打印模板在电脑端和移动端通用,打印机无限制。
