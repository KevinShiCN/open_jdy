# 如何限制账号的登录设备、ip?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：happy忆霖 | 阅读：2850 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/539973285996602368)

【问题描述】

如何限制账号的登录设备、IP?

【故障现象】

员工目前可以在任意网络上登录。希望能限制登录操作范围。

【解决方案】

企业管理员点击到工作台-安全策略-添加服务使用控制(关于安全策略的具体介绍,可以点击:[安全策略设置](https://vip.kingdee.com/knowledge/561288387609682945?get_from=article-id&productLineId=35&isKnowledge=2&lang=zh-CN))

![](https://vip.kingdee.com/download/0109bdfbcc05931141199405f78fe60bf75e.png)
注意服务使用控制中,选择设备控制的时候,需要下载金蝶云小微客户端,方才可以控制

![](https://vip.kingdee.com/download/01095d58ae9eba0c48c9bfbe899f4fe760d5.png)
选择服务,以及需要控制的设备号,指定使用时间,确定保存即可。注意限制设备登录后,只能在设置的设备上登录,不能在网页端登录

![](https://vip.kingdee.com/download/0109a29ffc9ddac749ecae5ba86ecbafb031.png)
如果选择设备,没有可供选择的设备号,则需要在需要控制的设备上,下载小微客户端,打开金蝶云小微客户端,选择下拉菜单项中的关于

![](https://vip.kingdee.com/download/01093256f937a7be40fa9ccfbe1ec8aa187d.png)

在关于中即可查看到设备号信息

![](https://vip.kingdee.com/download/0109e5c88e09bfbb426da9d4cb4282250cf0.png)

手机端获取设备号的方式:登录金蝶云App获取设备号,我的-设置-账号与安全-登录设备管理页面

![](https://vip.kingdee.com/download/0109239250c7f6a1416f9416a066cf7e70c3.png)

若需要对网络进行控制,选择需要控制的IP,和限制使用时间即可。注意通过网络IP设置限制时,在所在的局域网中,客户端和网络端都是可以登录的

![](https://vip.kingdee.com/download/010912fc5d781d2e4133b18bd154f9da4568.png)
若需要下载小微客户端,请点击:**[小微客户端](https://vip.kingdee.com/knowledge/462309069751982592?productLineId=35&isKnowledge=2&lang=zh-CN)**
