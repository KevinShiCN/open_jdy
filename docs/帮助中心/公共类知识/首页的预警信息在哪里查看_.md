# 首页的预警信息在哪里查看?

> 作者：云社区用户55os622 | 阅读：233 | 有用：0

【问题描述】
首页的预警信息在哪里查看?

【解释说明】

当在对应预警报表中勾选了对应的:登录时首页预警提醒时,

![](https://vip.kingdee.com/download/0109e9db6845a0fa4575b42dd1da0089d94b.png)

将在星辰的首页右上角显示,点击对应项可跳转至详情界面
![](https://vip.kingdee.com/download/010938d20eacec444ead818d04e6b9f5cf42.png)
