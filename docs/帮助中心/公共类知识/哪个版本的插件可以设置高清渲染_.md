# 哪个版本的插件可以设置高清渲染?

【问题描述】

哪个版本的插件可以设置高清渲染?

【概述】

目前打印插件5.0版本及以上的版本均支持设置高清渲染。

插件打印预览界面可设置高清渲染的渲染模式。
