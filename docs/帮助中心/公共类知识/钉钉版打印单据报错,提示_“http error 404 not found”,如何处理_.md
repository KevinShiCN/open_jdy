【问题描述】

**钉钉版打印单据报错,提示:“HTTP ERROR 404 Not Found”,如何处理?**

![](https://vip.kingdee.com/download/01090bae93c0579f45129917a7c4d8302cdb.png)

【解释说明】

**因网络环境波动,导致部分用户在使用钉钉版打印单据的时候,提示报错,研发已在紧急处理中。本次故障不会给您的数据造成任何影响,给您带来的不便深表歉意,对您给予我们工作上的支持和理解深表感谢!**

【解决方案】

建议使用在钉钉上选择插件打印进行打印单据。
