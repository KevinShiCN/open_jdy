# 如何体验协同助手功能？

# **1 问题描述**

- 想要体验协同助手功能，有体验入口吗？
- 想要体验星辰发布的最新功能，有体验入口吗？

# **2 解决方案**

有，官网体验账套配置了协同助手功能。进入jdy.com网页

![上传图片](https://vip.kingdee.com/download/01007190630757844e73a72f759f8edee209.png)

1）按产品体验，选择“金蝶云·星辰”

![上传图片](https://vip.kingdee.com/download/0100862df1ce72a1445b91aa26f412d522c9.png)

2）点击“免费体验”

![上传图片](https://vip.kingdee.com/download/0100f53b96618aad472786af63b2d9c96beb.png)

3）选择需要体验的产品版本

![上传图片](https://vip.kingdee.com/download/01000092be53de9a462aa9169443ba47c196.png)

选择**金蝶云・星辰 标准版**，可体验协同助手**全部功能**；若使用专业版、旗舰版体验账套，由于开通了订货商城，**无法体验协同助手完整功能**（参数配置、下单流程会受订货商城功能优先级限制）
