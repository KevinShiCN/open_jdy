# 快捷应用应该如何设置?

> 作者：云社区用户u1tk1853 | 阅读：175 | 有用：1

【问题描述】

快捷应用应该如何设置?

常用的菜单就可以点旁边的五角星收藏。

![](https://vip.kingdee.com/download/010954cfec45bef746318091c6b324d41438.png)
收藏后,下一次进入系统,直接在快捷应用里面可以快速打开对应的菜单。
![](https://vip.kingdee.com/download/0109f49804c4a2984b0186e3fe88d1d49bc4.png)
【温馨提示】

快捷应用里面收藏的单据类型是根据收藏的先后顺序显示的,如需调整可取消收藏再重新收藏调整显示顺序。
