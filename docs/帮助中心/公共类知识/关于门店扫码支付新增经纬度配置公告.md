# 关于门店扫码支付新增经纬度配置公告

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题

【问题描述】

看到关于门店扫码支付新增经纬度配置公告,请问是什么情况?

【解释说明】

为贯彻落实“中国人民银行关于加强支付受理终端及相关业务管理的通知(银发〔2021〕259号)”,需要星辰零售云客户在【门店资料】中增加门店经纬度字段,以及在【支付通道配置】中对存量支付方案点击重新保存。为避免影响正常经营与使用,建议客户务必在2023年12月31日前完成以上操作,向相关部门提交门店具体位置,逾期将无法继续使用(微信、支付宝、云闪付)扫码支付。
