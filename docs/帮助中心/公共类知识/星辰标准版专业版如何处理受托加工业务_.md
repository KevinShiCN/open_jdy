# 星辰标准版专业版如何处理受托加工业务?

【问题描述】

星辰标准版专业版如何处理受托加工业务?

【适用版本】

金蝶云星辰标准版、专业版

【解释说明】

受托加工业务本质是由客户提供原材料,企业负责加工并收取加工费的业务模式。

星辰标准版 / 专业版无受托加工功能,变通方案如下:

1、建立虚拟仓库用于管理受托加工物料的出入库,入库通过【其他入库单】处理,出库通过【其他出库单】操作;

若您想要了解其他入库单使用问题,请点击:[其他入库单,如何使用?](https://vip.kingdee.com/knowledge/129339770114427510?channel_level=%E9%87%91%E8%9D%B6%E4%BA%91%E7%A4%BE%E5%8C%BA%7C%E6%90%9C%E7%B4%A2%7C%E5%AE%98%E6%96%B9%E7%9F%A5%E8%AF%86&productLineId=35&isKnowledge=2&lang=zh-CN)
