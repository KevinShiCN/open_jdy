# 打印如何设置打印份数?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：shaoling | 阅读：4499 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/716457210942761984)

【问题描述】

打印如何设置打印份数?

【解释说明】

打印份数通常由打印机默认设置决定。

若使用PDF打印:可在打印预览页面设置打印份数。

若使用插件打印:则直接按打印机默认设置输出。

注意:不同打印机型号、驱动可能导致设置差异,请根据实际打印机配置调整。

【操作流程】

1、若使用PDF打印,可在打印预览页面右上角点击打印机图标进入打印设置页面

![](https://vip.kingdee.com/download/0109bb30e602979d40debc2ea819c292d3dd.png)
在打印设置页面,点击<更多设置>按钮

![](https://vip.kingdee.com/download/0109b035ad3191da4b04991fd9eacf46a14f.png)
点击<使用系统对话框进行打印..(CtrI+Shift+P)>按钮

![](https://vip.kingdee.com/download/010944c603d4001446d0ba44b90917bd637d.png)

设置对应的打印份数后,点击<打印>按钮即可
![](https://vip.kingdee.com/download/0109ff829dd53bc445f9a424d35a087f8762.png)

2、若使用的是插件打印机需设置打印机默认打印份数,先找到电脑的控制面板

![](https://vip.kingdee.com/download/0109c5ea83eae12c4cd09f9d7786fc226c1c.png)
从【查看设备和打印机】进入

![](https://vip.kingdee.com/download/0109d621698a78fa404a80f83427065cf6fd.png)
选择对应的打印机右键点击<打印首选项>按钮

![](https://vip.kingdee.com/download/0109475f8fbcc66444f4bab8442fd2819ea9.png)
打印机设置默认打印份数后,点击<确定>按钮即可
![](https://vip.kingdee.com/download/01092810cb095cd44063b5c46d11a7235750.png)
