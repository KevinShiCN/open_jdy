# 启用账套时选择了模式会不会影响后期做账?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：cs123456 | 阅读：439 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/491775576810870272)

【问题描述】

启用账套时,弹出选择角色的弹窗,选择了角色,是否影响系统的权限?

【业务场景】

在启用账套时,有选择角色的弹窗,选择了角色之后,担心系统功能使用会有影响。

【解释说明】

首页选择角色的弹窗不影响系统功能的正常使用,首页角色唯一限制的地方是首页显示的内容布局,选择角色之后,也可以再次更改首页布局,设置切换成其他显示内容,操作方式如下。

【操作流程】

首先,首页想切换布局(更换角色),可以点击首页左上角的双箭头符号,选择需要的角色进行切换即可,如图:![](https://vip.kingdee.com/download/0109b4c3cb4d0671422683f2446e83adabdc.png)

然后,点击确定按钮,即可切换完成。
