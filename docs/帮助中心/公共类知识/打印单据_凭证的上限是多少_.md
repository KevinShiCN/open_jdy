# 打印单据/凭证的上限是多少?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：shaoling | 阅读：156 | 有用：0

【问题描述】

打印单据/凭证的上限是多少?

【解释说明】

打印单据或凭证时,PDF 打印最多支持1000页;**插件打印**超过1000页时仅能预览前 1000页,但可正常打印全部页面。

![](https://vip.kingdee.com/download/0109e90f6fa0f48741278c2fd7e9baadd390.png)

若打印数据较多,考虑打印时长等因素,建议可分批进行打印。
