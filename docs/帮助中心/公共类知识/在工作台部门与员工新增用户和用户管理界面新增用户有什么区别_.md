# 在工作台部门与员工新增用户和用户管理界面新增用户有什么区别?

【问题描述】

在工作台部门与员工新增用户和用户管理界面新增用户有什么区别?

【解释说明】

1. 在工作台的部门与员工中添加用户,表示将用户添加到了该企业中,但是没有添加到账套里面。如果需要给用户授权账套权限还需要把工作台用户同步到账套的用户管理中。[如何把工作台用户同步到账套的用户管理中](https://vip.kingdee.com/knowledge/586359238004152832?productLineId=35&isKnowledge=2&lang=zh-CN)

2. 如果在用户管理界面新增用户,用户会加入到账套中,同时自动同步到工作台的部门与员工中。
