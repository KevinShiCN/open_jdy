# 系统里文件引出在什么位置?如何设置引出文件下载的位置?

【问题描述】

系统里文件引出在什么位置?如何设置引出文件下载的位置?

![](https://vip.kingdee.com/download/0109fce03196290a4e61987211f59e6b0f54.png)**[视频观看](https://vip.kingdee.com/school/detail/721299516523276544?productLineId=35&lang=zh-CN&cosmic_type=video)**
【解释说明】

**情况一: 使用浏览器登录,以谷歌浏览器为例**

**1、查看文件位置**

点击浏览器右上角三个点,点击<下载内容>,可以看到已经下载的文件,点击文件名可以直接打开文件,点击文件右侧的文件夹可以打开文件的位置

![](https://vip.kingdee.com/download/0109f9bac823ffee4d26a6bd9097245e3e54.png)

**2、设置默认下载路径**
系统里的文件引出保存位置以浏览器设置的默认下载路径为准,一般浏览器都有设置文件默认下载路径的功能

① 点击浏览器右上角,打开设置;
![](https://vip.kingdee.com/download/0109832bd2259c624bc5ad98c816901dad9c.png)

② 找到左侧【下载内容】,即可看到文件引出默认位置,也可通过【更改】按钮进行修改默认引出位置。![](https://vip.kingdee.com/download/01094909c3a937b643c9b4a60e7b09ec9653.png)

**情况二: 使用小微客户端登录**

**1、查看文件位置**

点击右上角三个点的按钮,点击<下载列表>****

![](https://vip.kingdee.com/download/010955a4b046958c46b1860b7d3267ee7b0a.png)
若客户端上方没有![](https://vip.kingdee.com/download/010977a46b53a6c0436e93895cac93a24276.png)可在电脑托盘中选中小微客户端并点击鼠标右键,可调出弹窗,点击<下载列表>

![](https://vip.kingdee.com/download/0109ada3b87a3cf941b7b513a9790798495b.png)

在下载列表中点击<目录>可打开文件位置
![](https://vip.kingdee.com/download/0109775d0fd9120046a889202ad3446a7a36.png)


**2、设置默认下载路径**

进入账套后,点击客户端右上角 ┅ 图标,点击<设置>,即可看到默认下载路径

![](https://vip.kingdee.com/download/0109ea46325dc6d649849ca1f6bd8de80c0d.png)
或是在电脑托盘中选中小微客户端并点击鼠标右键,点击<设置>

![](https://vip.kingdee.com/download/01093c9ee31a7bd342c59e86406f72b02d69.png)

点击【更改目录】也可修改默认引出保存路径。
![](https://vip.kingdee.com/download/010929897109149b46fdac9841c647701457.png)
