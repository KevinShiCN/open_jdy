# 发票与合同申请指引

## 1、发票申请

工作台【发票申请】，查看已支付订单发票状态、查看已开发票信息、下载发票、申请开票等操作。
![上传图片](https://vip.kingdee.com/download/010051e47439ef2b485b8ccce6060bc17014.png)
**申请开票**：已支付未开票状态的订单，可录入发票信息申请开票；
**查看**：点击按钮直接查看已开发票内容；
**下载**：点击按钮直接下载电子发票至本地保存；
**重新申请**：发票有误，可点击【重新申请】申请重新开票 ；
**物流单号**：增值税专用发票（纸质） 点击【物流单号】按钮可查看到发票寄出的物流信息。
![上传图片](https://vip.kingdee.com/download/01007535fa148c934b0aa1e285e3acbd0f74.png)

【温馨提示】

(1)如发票要求备注其他信息，请在备注处说明；发票备注将会在发票的备注列展示，请勿填写无关信息。

(2)电票将于提交申请后48小时内发至您填写的接受邮箱，如未收到邮件，可前往【我的发票-下载】手动下载。

(3)重新申请开票后，电子发票提交退票申请后需财务完成审核，时间在3个工作日内通过，通过后可在【发票申请】 处再次重新申请开票。

(4)为响应国家环保号召，仅支持全电发票自助申请。如果需纸质的专用发票，请联系您的产品营销顾问或是参考以下链接申请：[申请增值税纸质专用发票](https://vip.kingdee.com/article/325302049380836864?get_from=article-id&productLineId=6&isKnowledge=2&lang=zh-CN)
