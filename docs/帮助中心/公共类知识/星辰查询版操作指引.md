# 【业务场景】

星辰查询版适用于用户已购买星辰标准服务产品，服务到期后因一些原因不再续租，但是希望通过付费的方式，保留并查询账套已有数据。

# 【购买方式】

进入账套后，在产品内进行购买。购买成功后，进入账套首页右上角有查询版标识。

![上传图片](https://vip.kingdee.com/download/010037394b4b737c4986b8d59cda19fc5713.png)

## · 关于新购

查询版会在标准服务到期当天，进入账套给出【产品服务已到期提醒】弹窗，引导客户续费标准服务及购买查询服务。

点击【立即续费】，则进入续费标准服务订单界面

点击【购买查询服务】，则进入查询服务购买界面，点击立即下单，提交订单并支付即可。

![上传图片](https://vip.kingdee.com/download/01007c3d2992b0c44aa0964ddc36930075d5.png)

![上传图片](https://vip.kingdee.com/download/0100ca0ecb28bb2c4b88b30a727dd419604d.png)

## · 关于续费

购买查询服务后，在查询服务期间内，可通过右上角账套名称处的下拉键头，进入【管理账套】，账套管理界面显示查询版卡片，查询服务即将到期可点击续费操作

![上传图片](https://vip.kingdee.com/download/0100b13334cf79fd45939b01bc34ae88e2b3.png)

![上传图片](https://vip.kingdee.com/download/01002f02d65983534453af8e176853e28498.png)

查询服务到期前30天，进入首页也会给出弹框提醒，引导续费标准服务或续费查询服务，您可通过弹窗入口续费查询版

![上传图片](https://vip.kingdee.com/download/01009e2c835918bd4aa3aa4f05b3d1b1d382.png)

# 【购买规格】

**账套数：**查询版账套数等于标准产品服务下已启用的账套数，不允许在产品内增删账套数，如需要减账套购买，请在云管家中减账套后再购买查询版

**有效期**：默认为1年，不允许修改，支持查询服务未到期续费

**用户数：**自动继承标准服务的用户数，不单独收费

**价格：**按账套数计费，1个账套¥200，价格=账套数x200

![上传图片](https://vip.kingdee.com/download/0100dd8a7606272e4568bf87aa9cfd69d83d.png)

# 【查询服务权限】

查询版仅支持：查询、引出、打印、账套备份与恢复、账套下载、用户权限管理。若需使用新增，编辑等更多权限，需续费标准服务。

![上传图片](https://vip.kingdee.com/download/0100c5f604d9e8b84fcbb202b6df49d467d5.png)

>
购买须知：

1.查询服务自动继承标准服务的用户数，不支持新增用户。

2.查询服务继承标准服务的权限设置，可编辑查询服务范围内的权限。

3.查询服务有效期內支持续费转为标准服务。

4.查询服务不提供售后在线服务。 5.POS收银系统、门店商城和订货商城不再提供服务。
