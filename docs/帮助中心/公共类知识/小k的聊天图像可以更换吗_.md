# 小k的聊天图像可以更换吗?

【问题描述】

小K的聊天图像可以更换吗?

![](https://vip.kingdee.com/download/01091f0ecdba950e4dbd8081225881a1f7bc.png)

【业务场景】

小k的头像不是很喜欢,希望能更换成自己喜欢的头像,是否能支持?
【解释说明】

**目前暂不支持更换小k聊天界面的头像。小k的头像根据系统后台默认的,不支持修改操作。**
