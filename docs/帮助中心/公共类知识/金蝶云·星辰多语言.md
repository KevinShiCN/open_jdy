# **1 产品概述**

## **1.1 产品介绍**

全球化业务场景中，语言壁垒与时区差异是跨团队协作的核心挑战。金蝶云星辰通过「多语言灵活切换 + 多时区自主适配」的深度融合，构建精准、高效的全球化系统支撑能力，让跨国业务协作更流畅、数据更统一。

## 1.2 适用版本

星辰3.0以上所有版本**（需购买多语言包）**

## **1.3 核心价值**

- **操作更高效**：母语操作系统 + 本地时区适配，减少语言与时间换算成本，提升跨团队操作效率
- **数据更精准**：时间字段的差异化处理规则，兼顾业务追溯性与本地查看体验，确保数据一致性
- **拓展更灵活**：支持 “同一语言覆盖多时区”“同一时区使用多语言” 等复杂场景，随企业全球化布局灵活适配

## **1.4 特别说明**

- 金蝶云·星辰标准产品全模块支持多语言**（除生态模块、税务云）**
- 零售POS支持多语言（需配置最新的POS安装包，购买多语言模块后请联系客服处理）
- 系统预置了简体中文、繁体中文和英文模版**，**如需打印越南语、泰语、印尼语及其他小语种，请自行上传相关语种字体，在对应的语言环境下维护，详情可[点击查看](https://vip.kingdee.com/link/s/luCwz)
- 已经购买了多语言包的租户，后续新上线的语言可直接使用，无需再次购买

# **2 功能介绍**

## 2.1 多语言能力

- **支持多种语言**：支持简体中文、英文、繁体中文、泰语、越南语、印尼语，形成完善的多语言体系
- **全流程语言同步切换**：从登录界面、功能菜单、业务单据到报表数据、系统提示信息，均支持所选语种的完整适配，确保操作逻辑与语言表达一致
- **灵活切换与配置**：用户可根据个人习惯随时切换语言，满足团队内多语言并行的使用需求

**更多关于多语言的功能介绍及使用说明**[点击查看](https://vip.kingdee.com/link/s/Z7XCD)

**特别说明：**金蝶云星辰本次更新后，在 “系统参数” 中增加的 “国际化参数” 功能，主要用于配置账套语言和进行多语言初始化同步操作。以下是具体介绍：

- **账套语言**：新增的印尼语、越南语、泰语默认不会显示在个人头像右上角的下拉语言切换列表中，如果需要切换印尼语、越南语、泰语使用，请在 “国际化参数” 中勾选对应语言。
- **多语言初始化同步**：该功能可以对产品内基础资料多语言字段进行初始化

## 2.2 多语时区适配

系统的语言设置与时区调整完全独立，用户可按需灵活组合（如选择 “越南语” 时手动切换至 “GMT+7”，使用 “英文” 时切换至 “GMT-5” 北美时区），核心能力如下：

**（1）时区自主选择，适配场景需求**

用户可根据所在地或业务场景手动选择对应时区，系统基于所选时区自动换算并显示时间

**（2）时间字段智能处理，数据一致性更可靠**

- **制单时**：日期相关字段（如单据日期）直接取用当前用户时区的 “年月日”；时间相关字段（如制单时间）则同步取用当前时区的 “年月日 + 时分秒”
- **查数据时**：日期相关字段（仅年月日）保持原始录入值，不做时区转换（便于跨时区追溯业务发生日期）；时间相关字段（含时分秒）自动按当前用户时区转换

**（3）跨时区数据实时共享，协作无壁垒**

无论团队分布在哪个时区，系统支持数据实时同步共享，避免因时区差异导致的业务延误
