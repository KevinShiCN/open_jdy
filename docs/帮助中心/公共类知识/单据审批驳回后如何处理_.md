# 单据审批驳回后如何处理?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题

【问题描述】

单据审批驳回后如何处理?

【业务场景】

单据被驳回后,想要重新提交审批操作

单据被误操作审批驳回后,能撤销驳回的审批吗?

【解释说明】

单据审批驳回后可能存在需修改后**重新提交**操作,可能存在不再需要该单据,**需删除单据**操作。

单据审批驳回后**不支持撤销驳回**,若该单据误操作驳回,建议可根据需要重新提交审批流程

【操作指引】

1、单据审批驳回后单据操作人员可根据驳回意见修改后,重新提交审批。
