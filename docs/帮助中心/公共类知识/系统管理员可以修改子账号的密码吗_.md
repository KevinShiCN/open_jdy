# 系统管理员可以修改子账号的密码吗?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题

【问题描述】

系统管理员可以修改子账号的密码吗?

【解释说明】

1、每个账号的密码只能由该账号自行修改,管理员无法修改子账号的密码;

2、出于密码安全考虑,账号密码不支持查询,账号本人和管理员都无法查看到账号的密码,
