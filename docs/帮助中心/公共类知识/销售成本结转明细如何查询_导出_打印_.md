# 销售成本结转明细如何查询/导出/打印?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：云社区用户6Rw2283 | 阅读：354 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/727690611016317952)

【问题详情】

销售成本结转明细如何查询/导出/打印?

【解释说明】

目前财务模块没有直接查询或导出销售成本结转明细的报表,可通过查询对应结转成本科目的明细账查看(比如主营业务成本科目),或者在生成凭证页面查询但是无法导出

业务模块可以在销售明细表中查看和导出商品明细成本。

【操作步骤】

**一、财务模块**

1、点击【账务处理】-【明细账】

![](https://vip.kingdee.com/download/0109bdd436b7288f4be38ab622bc9d013362.png)

筛选结转成本的对方科目,比如主营业务成本科目,查询明细账如下图:![](https://vip.kingdee.com/download/01096e45d80607d348c4914c923f368f8aa4.png)

2、点击【账务处理】-【生成凭证】

![](https://vip.kingdee.com/download/01097a237fd24ba244ba95f4c89e49a6f415.png)

在【单据列表】-【库存管理】-【生成凭证】页面查看如下图:

![](https://vip.kingdee.com/download/0109f56a825339b04f65b1c445ca78ed9722.png)

**二、业务模块**
点击【销售管理】-【销售明细表】

![](https://vip.kingdee.com/download/010964c616d386954b05ab76c3a1bc175889.png)
相关过滤条件设置后,勾选“计算毛利”点击<查询>按钮

![](https://vip.kingdee.com/download/01098cd97fd49ec54a968cba1aa70bccf56b.png)
即可查看成本单据明细,查询结果可点击右上角<引出>按钮导出excel文件保存

![](https://vip.kingdee.com/download/01092606108bd40f40a5844c603285def5c5.png)
**查询结果也可以进行打印**

先点开打印设置选好打印模板和其他打印参数

![](https://vip.kingdee.com/download/0109082d5d9650a347adb061caac33175118.png)

然后点击打印按钮进行打印

![](https://vip.kingdee.com/download/0109e68987dcca854788814596ff8aa9f97f.png)
