# 多语言下数据授权未生效

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题, 快速入门
> 作者：卷不动的二可 | 阅读：51 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/780858475449415168)

# **1 问题描述**

- 中文环境下给业务员做了数据授权，切换其他语言后业务员没有对应的权限

# 2 适用版本

星辰3.0以上所有版本

# **3 原因解析**

数据授权是按名称和编码精准匹配的，当其他语言环境下，对应的基础资料维护的数据和中文下不一致，或者未维护为空值，则无法匹配对应的数据授权。

# **4 解决方案**

1、使用多语言初始化同步功能，确保其他语言下该基础资料数据一致，则数据授权只需要维护一个数据即可

![上传图片](https://vip.kingdee.com/download/010063d0411b3cd14799b7e4ba2d9c0eded3.png)

2、当不同语种下基础资料维护不一致的时候，可以用【或者】授权的方式，将多种语言下该基础资料的数据都进行授权
