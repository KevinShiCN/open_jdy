# 单据或凭证的审核、制单、修改日期(时间)是如何取值的,能修改吗?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：ylim | 阅读：442 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/622598249731520512)

【问题描述】

单据或凭证的审核、制单、修改日期(时间)是如何取值的,能修改吗?

【解释说明】

凭证与单据的审核、制单、修改日期(时间),均依据实际操作的时间节点由系统自动反写,既无法手动指定具体时间,也不支持手动修改。

若确需修改日期 / 时间,可参考以下解决方案(**仅可向后调整,无法向前调整**):

**审核、修改日期 / 时间调整:**可先反审核单据,待到达目标时间后,重新完成修改与审核操作;

**制单日期 / 时间调整:**需先删除原单据,待到达目标时间后,重新新增单据。

**以审核时间为例:**审核日期时间不跟随电脑所设置的日期时间,所获取到的日期时间是后台数据库的实时日期时间。
