# 折前金额字段是什么意思?能否手工录入修改?

【问题描述】

折前金额字段是什么意思?能否手工录入修改?

【解释说明】

“折前金额”是指在**未使用任何折扣或优惠之前**的**不含税**原始金额,系统折前金额是根据**单价*数量**计算带出的,**无法手工直接录入,若需修改,可根据计算公式修改对应金额或数量**

折前金额相关的计算公式如下:

①折前金额=不含税单价*数量

②折前金额=折后金额+折扣额
