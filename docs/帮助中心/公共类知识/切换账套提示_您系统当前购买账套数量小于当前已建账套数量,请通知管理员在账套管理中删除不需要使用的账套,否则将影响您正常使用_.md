# 切换账套提示:您系统当前购买账套数量小于当前已建账套数量,请通知管理员在账套管理中删除不需要使用的账套,否则将影响您正常使用?

> 作者：咚咚 | 阅读：325 | 有用：0

【问题描述】

切换账套提示:您系统当前购买账套数量小于当前已建账套数量,请通知管理员在账套管理中删除不需要使用的账套,否则将影响您正常使用?

![](https://vip.kingdee.com/download/01093527c3b3e870442fb410cf411f428f93.png)

【解释说明】

一开始账套使用正常,中间有减少账套(如:续费减少账套)的操作,就会导致购买账套数小于之前使用的账套数。这样切换账套的时候就会有上述提示。

【解决方案】

所有账套显示状态都必须显示,右上角会有出现删除按钮,删除不用的账套。当启用账套数小于等于购买账套数时,就可以正常切换账套使用了,
