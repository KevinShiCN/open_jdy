# 单据/凭证/报表如何按打印格式导出pdf文件?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：杨桂平 | 阅读：353 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/776251472521996800)

【业务场景】

单据/凭证/报表如何按打印格式导出PDF文件?

【解决方案】

单据/凭证/报表不能直接导出为PDF文件,但是可以在打印预览界面将数据下载保存为PDF文件。若下载后找不到文件,可参考:[系统里文件引出在什么位置?如何设置引出文件下载的位置?](https://vip.kingdee.com/link/s/ZXXvN)

以凭证为列(单据和报表的操作方式一样,不再赘述):

**1、打印方式为PDF打印**

当打印设置的打印方式选择为“PDF打印”时,点击打印后将进入预览界面,此时直接点击右上角“下载”按钮,即可将文件保存为PDF格式。

![](https://vip.kingdee.com/download/0109e7cde945af3c4db1ad0008db6d174c8b.png)

**2、打印方式为插件打印**

当打印设置的打印方式选择为“插件打印”时,需要点击“打印预览”,进入到预览界面,目标打印机选择为“Microsoft Print to PDF”,再点击打印即可下载为PDF文件。

![](https://vip.kingdee.com/download/0109cf9ecff089ba4dbdad55ec82bddbae60.png)
**3、**若需将多张单据合并至同一 PDF 中,可在打印模板开启**纵向节纸**功能,在纸张排版允许的前提下,系统将自动展示多张单据,最后直接下载为 PDF 即可。

![](https://vip.kingdee.com/download/010978ecd1437bf44b1f98870524f1fdd4c1.png)

![](https://vip.kingdee.com/download/010919c9b37afa2f42118a135f26c91a10f5.png)

【温馨提示】

明细账、核算项目明细账、数量金额明细账等账簿导出PDF文件时,若需要将所有查询出来的数据导出为PDF文件,可以点击右上角“连续打印”后再下载为PDF即可;若只需要导出某一个数据,则选择某一个科目后点击“打印”即可单独下载这一个科目数据。

![](https://vip.kingdee.com/download/0109ff4e9a3b204f443ba991fc4c2a6051da.png)
