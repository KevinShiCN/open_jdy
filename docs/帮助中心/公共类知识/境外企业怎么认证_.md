# 境外企业怎么认证?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：Schatz | 阅读：300 | 有用：0

【问题描述】

境外企业怎么认证?

【解释说明】

1、境外客户认证,认证类型选择为:境外,不区分个人和企业,提交所持有的在有效期内的相应证件,如护照,驻外机构许可证,港澳通行证,商业登记证等

2、组织名称对应填写company name 后面的英文名称,其余信息与证件信息一致即可。

![](https://vip.kingdee.com/download/0109377bb81259454867a4fe1a14e9219c24.png)

认证信息提交后,3个工作日内会返回认证结果,结果将以短信和邮件的方式通知,在工作台点击【我的企业】也可看到认证状态的变化。
