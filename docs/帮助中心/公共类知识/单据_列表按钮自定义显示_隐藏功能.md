# **2 功能演示**

下面以销售出库单为例演示如何使用该功能。

## **2.1 ****单据列表**

1）打开销售出库单列表

2）点击工具栏的“更多”，找到“自定义”按钮。

3）点击“自定义”按钮，按钮名称前会出现复选框，勾选则表示要显示在外面，不勾选则默认收起到【更多】里。
