# 开单选项

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：功能介绍, 快速入门
> 作者：进销存熊诗琴 | 阅读：565 | 有用：12
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/636972947646912768)

## **1 简介  **

### **1.1 功能介绍**

通过开单选项能对单据上的部分字段支持更灵活的配置，从而满足不同的客户场景，例如反算单价、价税合计、重复商品行检查、已打印次数的显示、开单携带上一单参数等等。

![](https://vip.kingdee.com/download/010976dda75e6b714a308682104dc3334518.png)

### **1.2 适用版本**

金蝶云·星辰3.1所有版本

### **1.3 功能特性**

• 每种类型的单据都分别由单据页面的“选项”控制，不是通过系统参数控制；

• 选项的记忆维度请看下次第二点的“参数&字段说明”

• 新建账套下，选项内的参数默认不开启，开启后下次进入单据也会默认打开，无需再次设置，会记忆上一次结果；

## **2 参数&字段说明**

**（1）通用参数**

| 参数名称 | 适用场景 | 适用单据 | 记忆维度 |
| --- | --- | --- | --- |
| 修改价税合计后反算单价 | 手工录入/修改价税合计后，反算或不反算单价 | 进销存大部分销售/采购类单据。旗舰版的销售出库单在“金额计算参数”中。 | 按账套维度记忆的，同一账套内不同用户彼此互影响 |
| 修改税额重算价税合计 | 手工录入/修改税额后，反算或不反算价税合计 | 进销存大部分销售/采购类单据。旗舰版的销售出库单在“金额计算参数”中。 |  |
| 录入时检查重复商品 | WEB手工录入商品时检查重复的商品编码或SKU编码（具体应用说明参考：https://vip.kingdee.com/link/s/lnkQK） | 进销存大部分单据（不支持盘点类单据、组装拆卸类单据） |  |
| 已打印次数不显示 | 勾选则隐藏单据左上角的“已打印次数”，不勾选则不隐藏 |  | 按用户维度记忆的，同一账套内不同用户彼此互不影响 |

**（2）字段默认值**

适用于新增单据时，配置单据头的字段是否默认携带上一张单据的值：

| 字段名称 | 功能说明 | 备注 | 记忆维度 |
| --- | --- | --- | --- |
| 单据日期 | 未勾选默认为系统日期，勾选后携带上一张单据的单据日期。 | 进销存大部分单据 | 按账套维度记忆的，同一账套内不同用户彼此互影响 |
| 未勾选默认为系统日期；勾选后，申请单执行采购时携带申请单的单据日期到采购订单，多张申请单合并采购时取第一张单的值。 | 仅适用于采购订单 |  |  |
| 交货日期携带源单的预计到货日期 | 未勾选默认根据单据日期加备货天数计算；勾选后，申请单执行采购时携带采购申请单商品分录的预计到货日期到采购订单的交货日期。 | 仅适用于采购订单 |  |
| 申请人 | 未勾选时默认为空，勾选则新增单据时默认取上一张单据申请人 | 仅适用于采购申请单 | 按用户维度记忆的，同一账套内不同用户彼此互不影响 |
| 客户/供应商 | 未勾选则新增单据时默认为空，勾选则新增单据时默认取上一张单据客户/供应商 | 进销存大部分单据 |  |
| 业务员 | 未勾选则新增单据时优先携带客户/供应商中已维护的业务员，否则为空；勾选则新增单据时优先携带客户/供应商中已维护的业务员，其次按制单人取上一单的业务员 | 进销存大部分单据 |  |
| 客户收货地址 | 勾选后销售订单/销售出库单下推发票时携带订单的客户收货地址到销售发票，多张单据合并下推时取第一张单的值。勾选后，销售发票选源单时也携带源单的客户收货地址，选多张源单只取第一张单，选多次源单以最后一次为准。 | 仅适用于旗舰版的销售发票 |  |
| 结算日期 | 未勾选，在销售订单下推销售出库单时，默认根据单据日期加结算期限计算；勾选后，订单出库时携带订单的结算日期到出库单，多张订单合并出库时取第一张单的值。 | 仅适用于销售出库 |  |
| 未勾选默认系统日期；勾选后后，订单入库时携带订单的结算日期到入库单，多张订单合并入库时取第一张单的值。 | 仅适用于采购入库 |  |  |
