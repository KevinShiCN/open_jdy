# 针式打印机打印,模板和打印机纸张尺寸都设置没有问题,但是打印机无法连续打印,只能一张一张打印,并且每打印一张就需要调整纸

> 作者：杨桂平 | 阅读：1919 | 有用：1

【问题描述】

针式打印机打印,模板和打印机纸张尺寸都设置没有问题,但是打印机无法连续打印,只能一张一张打印,并且每打印一张就需要调整纸

【解决方案】

这是由于打印机进纸设置导致的问题:

如果打印放纸是从前进纸,就会出现此问题,更换为后进纸即可。
