# 如何设置单据打印时商品名称字段取设置了外部商品名称时取外部商品名称,未设置时取本商品名称?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：大家好才是真的好 | 阅读：121 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/703773246885353472)

【问题描述】

如何设置单据打印时商品名称字段取设置了外部商品名称时取外部商品名称,未设置时取本商品名称?

【解释说明】
通过对打印模板内的商品名称字段设置公式取数,建议公式为:if(客户商品编码.外部商品名称 !="",客户商品编码.外部商品名称 ,商品编码.商品名称 )

【操作步骤】

以销售出库单为例:

1、商品名称“A商品“设置的外部商品名称为“苹果”,对应客户为“00012”。

![](https://vip.kingdee.com/download/010965725c52cd5d4207adc24e4175640003.png)

2、新增销售出库单,单据上录入已设置外部商品的“A商品”和未设置外部商品的“B商品”。需要实现打印时“A商品”的名称为“苹果”,“B商品“名称不变。

![](https://vip.kingdee.com/download/010953887dbde14c4cbe985a830749f50cbe.png)

3、系统设置-系统参数-销售管理-销售出库,打开对应的模板;

![](https://vip.kingdee.com/download/01095d7f0b93657e4bcfa84afc4ab7ae8633.png)
![](https://vip.kingdee.com/download/010909e6c14cc8df47f99a1ad1e134dd5dd5.png)

4、如图所示,选择数据行内的商品名称字段,右上角的数据源设置公式取数。公式为:if(客户商品编码.外部商品名称 !="",客户商品编码.外部商品名称 ,商品编码.商品名称 )

![](https://vip.kingdee.com/download/01099bc6047b21e9443c9d8175479bd625a6.png)
![](https://vip.kingdee.com/download/0109428292b5851a45e7b35006a6ce971e03.png)
5、最终打印预览效果如图所示。“A商品”显示为“苹果”,“B商品”显示不变。

![](https://vip.kingdee.com/download/0109aea76eaec74540a39adea54aac30e02a.png)
