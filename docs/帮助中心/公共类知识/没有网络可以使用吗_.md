# 没有网络可以使用吗?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题

【问题描述】

没有网络可以使用吗?

【解释说明】

金蝶云星辰为云端产品,无需安装本地数据库,支持随时登录使用。但无论是通过浏览器还是小微客户端登录,**均需保持网络连接。**

若无网络,将无法打开网站或登录小微客户端,导致无法正常使用。

电脑无网的情况下,建议可以下载金蝶云APP在手机端操作。
