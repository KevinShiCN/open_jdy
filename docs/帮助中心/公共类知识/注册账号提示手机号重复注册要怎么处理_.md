# 注册账号提示手机号重复注册要怎么处理?

【问题描述】

注册账号提示手机号重复注册要怎么处理?

![](https://vip.kingdee.com/download/0109472006530b924d7191d0a2c3fad4f7c5.png)

【解释说明】

一个手机号只能注册一次,如果手机号已经注册可以直接在[网址](www.jdy.com)(网址:www.jdy.com)通过**手机号+验证码**进行登录。

![](https://vip.kingdee.com/download/0109d268e11e013b44b2bde875a13e8abbd9.png)
如果手机号已经注册可直接邀请加入公司,授权账套,不需要再注册

【系统设置】-【用户管理】-【新增用户】,可查看知识:[如何邀请用户?如何启用或禁用员工?](https://vip.kingdee.com/knowledge/129339770114427402)

![](https://vip.kingdee.com/download/0109054f201001a7442580ee1d27fe5fe7fd.png)

若需要找回密码的话,在登录界面中,点击**“忘记密码”**,用手机号+验证码进行密码重置即可。

![](https://vip.kingdee.com/download/0109173f8a084fb544ea86b8984d8ee38d4f.png)
