# 进销存模块业务单据支持拆分/合并单据吗?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：shaoling | 阅读：167 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/718268485591202304)

【问题描述】

进销存模块业务单据支持拆分/合并单据吗?

【解释说明】

目前进销存模块的业务单据**暂不支持**单据拆分 / 合并功能。

如需拆分单据,建议按以下步骤操作:

1、复制原单据生成新单据

2、在新单据中删除无需保留的商品项,保留需要拆分显示的内容

3、再次复制原单据,在新单据中删除无需保留的商品项,保留需要拆分显示的内容

4、确认新单据无误后,按流程作废或删除原单据

若您想要了解删除单据操作问题,请点击:[删除单据如何操作?](https://vip.kingdee.com/knowledge/571500927811689216?channel_level=%E9%87%91%E8%9D%B6%E4%BA%91%E7%A4%BE%E5%8C%BA%7C%E6%90%9C%E7%B4%A2%7C%E5%AE%98%E6%96%B9%E7%9F%A5%E8%AF%86&productLineId=35&isKnowledge=2&lang=zh-CN)

如需合并单据,建议按以下方案操作:

**方案一:若是采购销售类的业务需要合并入库/出库,建议先新增采购订单/销售订单,再选择多个订单合并入库/出库。**

若需要了解如何新增采购订单,请点击:[如何新增采购订单?](https://vip.kingdee.com/knowledge/129339770097650425?productLineId=35&isKnowledge=2&lang=zh-CN)

若需要了解如何新增销售订单,请点击:****[如何新增销售订单?](https://vip.kingdee.com/knowledge/539973281768776448?productLineId=35&isKnowledge=2&lang=zh-CN)

若需要了解如何将采购订单合并生成采购入库单,请点击:[采购订单列表上入库和合并入库有什么区别?](https://vip.kingdee.com/knowledge/634919534838607360?productLineId=35&isKnowledge=2&lang=zh-CN)

若需要了解如何将销售订单合并生成销售出库单,请点击:[多个销售订单如何合并出库?](https://vip.kingdee.com/knowledge/626946762317774592?productLineId=35&isKnowledge=2&lang=zh-CN)

**方案二:需要将系统中已经新增的单据进行合并,建议将需要合并的单据数据[导出excel表格](https://vip.kingdee.com/knowledge/534899713808318976?productLineId=35&isKnowledge=2&lang=zh-CN),在系统中新建一张对应类型的单据,通过[快速粘贴](https://vip.kingdee.com/knowledge/391351756862874368?productLineId=35&isKnowledge=2&lang=zh-CN)的方式将数据从excel表格粘贴到新建的单据中,数据复制完成后,将需要合并的单据删除即可。**

【温馨提示】

**操作方案二步骤前,建议先[备份账套](https://vip.kingdee.com/knowledge/561354015498341120?productLineId=35&isKnowledge=2&lang=zh-CN)或者确认数据都粘贴完成后,再将需要合并的数据删除。若账套未备份,****删除后的单据****不支持找回,请慎重!**
