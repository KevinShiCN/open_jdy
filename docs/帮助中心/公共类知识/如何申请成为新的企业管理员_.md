# 如何申请成为新的企业管理员?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：ylim | 阅读：1115 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/571500958883093248)

#### 【问题描述】

如何申请成为新的企业管理员?

#### 【业务场景】

情况一:因企业管理员离职且未交接等原因导致企业管理员账号丢失,导致无法继续使用企业已购买的服务

情况二:原账号无法使用手机号登录,也接收不到验证码,导致无法登陆金蝶统一云账户新增用户进行管理员移交

**以上场景需要设置新的账号成为企业管理员**。

如:A公司的企业管理员是甲,甲离职未做交接,导致系统无法登录使用,如何设置乙成为A公司新系统管理员?

【解决方案】

如您当前登录的是新管理员账号,可以点击:[新增企业管理员](https://service.jdy.com/workbench/web/index.html#/helpSelf/addCorpAdmin),直接跳转申请界面,点击右上角【新增企业管理员】按指引提交资料即可。

**注:需由新管理员本人的账号发起申请,不可他人代申请****。如乙需申请成为A公司的企业管理员,则应登录乙****的****账号发起申请。**

**
**

如果还未申请新管理员账号,请按照以下步骤操作:

【操作流程】

**1、注册新账号**

具体操作步骤:[如何注册新账号?](https://vip.kingdee.com/link/s/lu06A)

2、发起申请

登录申请的新账号,进入工作台,点击左侧【自助服务】,随后选择并点击【新增企业管理员】。

![](https://vip.kingdee.com/download/0109229018072d364ed98ae1c35e9ebc7e6f.png)

接着,点击右上角**【新增企业管理员】**按钮,,仔细阅读注意事项,然后勾选“我已知晓并同意继续申请添加企业管理员账号”,并点击【下一步】填写申请资料。

![](https://vip.kingdee.com/download/010916ce754ccaea472ca3dc87f7c529dfff.png)

**3、填写申请资料**

打开新增企业管理员的申请界面,按要求填写申请内容:

![](https://vip.kingdee.com/download/0109fbe7a21d8be140ba8cf7dd850d8b2338.png)

①当前申请人: 不同的申请人需要上传的资料不同,请根据实际情况对应操作

| 申请人类型 | 必须上传资料 | 辅助资料 |
| --- | --- | --- |
| 企业 | 必传资料:1、 申请书扫描件(加盖公章);2、 企业营业执照(复印件加盖公章);3、 工商变更登记通知书(如发生过变更需提供) | 1、 购买服务发票记录2、 购买服务付款截图3、 购买服务电子合同4、 购买服务转账记录5、 与销售人员聊天记录6、 其他 |
| 个人 | 1、 申请书扫描件(认证人签字);2、 本身身份证(电子档或复印件) |  |
| 其他组织 | 1、 申请书扫描件(加盖公章);2、 组织机构代码证或其他证件(在有效期内且年检章齐全) |  |

②企业名称:必录项,如申请成为A公司的企业管理员则企业名称填写“A公司”

③情况说明:非必录项,如不能准确填写企业名称或有其他特殊情况,可在情况说明中进行补充说明或上传补充资料。补充说明越详细越有助于快速完成审批。

④新管理员账号:新管理员账号(不可编辑,系统自动获取当前登录账号)

⑤手机验证码:点击获取验证码

**4、下载申请书**

短信检验通过后,可下载系统自动生成的申请书,进行填写,同时准备其他需要上传的资料。

![](https://vip.kingdee.com/download/0109d01a99cdd5884e1a8bcb341b3cfde4aa.png)

**5、上传资料提交审批**

资料准备完成后,点击“上传资料”,根据界面展示的必要资料和辅助资料进行上传即可。上传界面支持暂存,确认无误后点击【提交审批】。

![](https://vip.kingdee.com/download/010937d760f4599d40ff8f14dd6b5b099405.png)

**【温馨提示】**

1、上传文件格式要求:支持pdf、jpg、png格式

2、上传文件大小要求:不超过10MB

**6、审批说明**

##### 审批时间:**工作时间24小时内处理审批,如提交时间为节假日,顺延至工作日完成审批**。

审批状态说明:

① 状态为“审批中”时,用户可自主取消已提交的审批,可修改后再重新提交

② 状态为“待上传资料”时。可删除记录。,其他状态均不允许删除

③ 状态为“待补充资料”和“审批不通过”时。可点击【查看原因】查看具体原因后进行资料补充,或重新进行申请。

④ 状态为“已完成”时。可具体查看实际加入的企业信息。

【温馨提示】

① 企业管理员的作用可参考:【[系统管理员与账套管理员有什么区别](https://vip.kingdee.com/link/s/lRmjR)】

② 如已申请成功成为管理员,可参考知识:“[如何更换管理员](https://vip.kingdee.com/link/s/lRmjn)”,将旧管理员账号取消管理员身份。
