【业务场景】

购买api接口的发票如何申请?

【解决方案】

登录[金蝶云苍穹应用市场](https://appmarket.kingdee.com/ipaas),点击<买价中心>的界面,如图所示:

![](https://vip.kingdee.com/download/01093bdf82f6223641ef809ff5549f86afe7.png)

在我的订单界面找到已支付完成的订单点击“开发票”申请即可。

![](https://vip.kingdee.com/download/01090a7039c3b8614c0cbfc497bb5739ab36.png)

【温馨提示】

若操作中遇到其他问题,可点击:**[开发者社区](http://mtw.so/61gwHU)**进行提问
