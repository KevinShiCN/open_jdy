# 工作台邀请用户提示邀请失败:被邀请人已是企业成员,无需邀请?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：云社区用户8x80607 | 阅读：566 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/639267997781220096)

【问题描述】

工作台邀请用户提示邀请失败:被邀请人已是企业成员,无需邀请。

![](https://vip.kingdee.com/download/010906a016d63eab498189b0e7d48abc995e.png)【解释说明】

该提示表示当前手机号已经被该企业邀请过,且还是当前企业成员。不需要重新重复邀请。

【操作步骤】

1、登录企业管理员账号,工作台-【我的企业】-选择根级组织架构,复制被邀请手机号查看,是否在该组织下。如存在,可直接授权账套操作即可。![](https://vip.kingdee.com/download/0109cad8564d4e2e4f2699a18a0cf50d4f65.png)

2、如账套已经授权,但是子账号登录还是无法看见企业。可能是子账号进入多家企业且未切换企业导致。登录子账号,工作台-我的企业,可操作切换企业,如下图所示![](https://vip.kingdee.com/download/0109b9329dcb9b204e6da9dc30a162eb401d.png)
