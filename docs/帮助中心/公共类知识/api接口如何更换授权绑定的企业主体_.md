# api接口如何更换授权绑定的企业主体?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：ylim | 阅读：196 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/685291277566492928)

【问题描述】

API接口如何更换授权绑定的企业主体?

【业务场景】

购买的API接口已绑定授权星辰账套,但是星辰服务所属公司做了变更,对应API接口的授权也需要解绑重新授权。

【解决方案】

登录小微生态门户后在【我的应用】>【应用集成】中找到对应接口解除授权,之后重新授权要绑定的企业即可。关于如何授权,可点击:**[使用API授权服务](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=16a76687d9de11ed8e36e510300a0135)** 查看

也可以联系ISV伙伴协助处理
