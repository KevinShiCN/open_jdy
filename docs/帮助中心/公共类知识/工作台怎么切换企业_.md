# 工作台怎么切换企业?

> 作者：若有灵兮山之阿 | 阅读：878 | 有用：2

【问题描述】
工作台怎么切换企业?
【业务场景】
场景一、同一个用户被邀请到多个公司分别授权了账套,需要切换企业才能进入对应企业的账套进行操作

场景二、旧用户一个手机号注册了多个用户名,需用手机号登录后切换企业使用
【操作步骤】

1、登录时应使用手机号+密码或使用验证码登录

![](https://vip.kingdee.com/download/0109a5875c042e434a19ae4df42b51242808.png)2、工作台左上方点<击切换企业>
![](https://vip.kingdee.com/download/01094a5e4d98da3440eb96b720dade1d9504.png)
3、找到对应的企业-<进入企业>
