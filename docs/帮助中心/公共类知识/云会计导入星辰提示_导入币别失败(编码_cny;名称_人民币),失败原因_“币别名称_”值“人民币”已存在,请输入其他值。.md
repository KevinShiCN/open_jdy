# 云会计导入星辰提示:导入币别失败(编码:cny;名称:人民币),失败原因:“币别名称:”值“人民币”已存在,请输入其他值。

【问题描述】

云会计导入星辰提示:导入币别失败(编码:CNY;名称:人民币),失败原因:“币别名称:”值“人民币”已存在,请输入其他值。

【解释说明】

导入云会计,提示的错误日志,如下图所示
![](https://vip.kingdee.com/download/0109d6dd21e20b7f498f99a54a10e45dd8d2.png)
需要检查云会计中是否存在名称相同的币别

![](https://vip.kingdee.com/download/0109bb07068483da46f380383363ed6b37f2.png)
若有点击对应币别操作下方笔样图标,编辑修改为其他名称,修改保存后重新操作升级(**注意:应修改的是非本位币名称**)

![](https://vip.kingdee.com/download/0109ebed8ae2537d489695959a351e6c4687.png)
