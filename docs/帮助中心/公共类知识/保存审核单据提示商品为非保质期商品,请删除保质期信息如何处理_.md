# 保存审核单据提示商品为非保质期商品,请删除保质期信息如何处理?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：Simone邓 | 阅读：22 | 有用：1

【问题描述】

保存审核单据提示商品为非保质期商品,请删除保质期信息如何处理?

【解释说明】

出现该提示的原因是:创建商品信息时启用了**保质期管理**,新增单据选择该商品后,系统会自动带出保质期相关信息;若在单据尚未保存 / 审核时,修改该商品的基础信息并关闭了**保质期属性**,此时再去保存或审核单据,系统就会触发校验提示:**第 X 行分录,商品为非保质期商品,请删除保质期信息**。

![](https://vip.kingdee.com/download/01091a282d16d9694c33a0894896430fbbb7.png)
解决方案:单据上重新选择该商品即可
