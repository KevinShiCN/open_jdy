# 进销存录单界面如何选择多行商品进行操作?

【解释说明】

进销存录单界面,**支持**选择多行商品行。

若需选择连续商品行,可鼠标选中某行单元格后**按住鼠标左键下拉**,即可选中中间所有连续行。
![](https://vip.kingdee.com/download/0109cf62e0cda1bc412a9fea44706646740e.png)
若需选择不连续商品行,先点击某行单元格,按住 “Ctrl” 键后依次点击其他行单元格,即可完成多选。

![](https://vip.kingdee.com/download/0109aff098c440cf432f8caaac76c1b3b12d.png)
若需选择该单据全部商品行,支持 “Ctrl+A” 快捷键来选中当前页面的所有商品行。

![](https://vip.kingdee.com/download/01091a660a53c4e94cb4b05d4c8634cc0749.png)
