# 金蝶云·星辰如何切换系统语言为印尼语、泰语、越南语

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题, 快速入门, 新特性
> 作者：移动生产白建凡 | 阅读：278 | 有用：1

# **1 问题描述**

金蝶云·星辰如何切换系统语言为印尼语、泰语、越南语？

# 2 适用版本

星辰3.0以上所有版本（已购买多语言）

# **3 解决方案**

购买多语言包后，简体中文、繁体中文、英文会默认显示在个人头像右上角的下拉语言列表中

![上传图片](https://vip.kingdee.com/download/01004e8f139b29074bf3a70b3b29e1505884.png)

印尼语、越南语、泰语默认不会显示在个人头像右上角的下拉语言切换列表中，如果需要切换印尼语、越南语、泰语使用，请在 “系统参数>国际化参数” 中勾选对应语言。勾选后保存该参数，刷新页面后立即生效

![上传图片](https://vip.kingdee.com/download/010055aa789aa249427787768e7f9f80c448.png)

如图示勾选泰语后保存参数

![上传图片](https://vip.kingdee.com/download/010078bda2503dc746a1876bfcb098528fa1.png)

刷新页面后，个人头像右上角的下拉语言切换列表中即出现泰语

![上传图片](https://vip.kingdee.com/download/01002390b49a0c0b448bbb2c54ca93e7fdc1.png)
