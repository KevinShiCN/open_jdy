# 如何邀请同事加入企业

> 作者：Cat_Chow | 阅读：2256 | 有用：5

# 开通多用户的服务，企业管理员可邀请其他用户加入企业，并授予一定的权限，协同操作系统。 目前有三种邀请方式：二维码邀请、添加成员、同步企业微信员工。

## 1、邀请同事

### 1.1二维码邀请

（1）工作台【部门与员工】，点击右上角【二维码邀请】，自动生成二维码邀请图片及邀请链接；被邀请人通过微信/精斗云APP/金蝶云APP/云之家扫码注册账号，或是打开邀请链接注册账号，申请加入企业。

![上传图片](https://vip.kingdee.com/download/0100fedd9b5e5af54603a0cfceba52f63f6a.png)

![上传图片](https://vip.kingdee.com/download/010041fabeae719d40e39b0b3bbf1f20b121.png)

【温馨提示】生成的邀请二维码或邀请链接有效期为3天。

（2）被邀请人注册成功后，更新至【申请加入人员】，企业管理员【同意】后即成功加入企业；【拒绝】即删除申请申请信息。

![](https://vip.kingdee.com/download/0109e44f60c63d58457c803906aca87431f9.png)

### 1.2添加成员

（1）工作台【部门与员工】，点击右上角【添加成员】，可以批量导入员工信息，或是手工添加员工信息。

注意：操作添加成员，须用账号绑定的原手机号码+密码登录

![上传图片](https://vip.kingdee.com/download/01007b10cf0e9197430583b011d05780021a.png)

![上传图片](https://vip.kingdee.com/download/0100c1a6f9292e514efb814cb31e9d48c5a5.png)

![上传图片](https://vip.kingdee.com/download/0100c2f233ee31c84531b146b246d3bd6217.png)

### 1.3同步企业微信员工

企业微信应用中开通精斗云服务，并在企业微信的【应用管理】-对应服务【可见范围】添加企业员工信息，点击【同步企业微信员工】，系统自动把可见范围内的员工同步至【部门与员工】。

![上传图片](https://vip.kingdee.com/download/0100ed6baa61e3a54dbb9e41ac967c28eeae.png)

![上传图片](https://vip.kingdee.com/download/01004b94fbd0588d40f89069db924edf11ee.png)

## 2、接受邀请

当企业管理员通过【添加成功】邀请同事后，请通知被邀请人即时登录jdy.com接受邀请：

（1）被邀请人通过【被邀请手机号+验证码登录】方式登录[www.jdy.com接受邀请。](https://vip.kingdee.com/tolink?target=http%3A%2F%2Fwww.jdy.com%E6%8E%A5%E5%8F%97%E9%82%80%E8%AF%B7%E3%80%82)

![上传图片](https://vip.kingdee.com/download/0100ea7784df33f24a47a8409fb6c21994b9.png)

（2）被邀请人登录成功后，则会收到邀请信息，如下图所示。

![上传图片](https://vip.kingdee.com/download/01009f4517b92238410fa24e959916f44832.png)

（3）选择【接受】出现如下图所示界面，勾选“我已阅读并接受以上条款”后，再次选择【接受】即可加入企业（加入成功后需重新登录）。

![上传图片](https://vip.kingdee.com/download/01005a15137d62c141959458307c6da48620.png)

## 3、账号管理

被邀请人成功加入企业后，更新至【待分配部门】人员，企业管理员可根据公司情况分配对应的角色身份和部门。

![上传图片](https://vip.kingdee.com/download/0100df0469bcfeca4c48b94fc0f5a4d097b7.png)

![上传图片](https://vip.kingdee.com/download/0100c5395a5322254eacbab7277163424fa5.png)

## 4、登录管理

启用登录管理后，用户登录时需要管理员设置的手机号短信校验通过后才能登录成功，从而有利于管理员对登录用户进行管理，管理员能清晰的知道哪些人有进行登录使用。

(1)在【部门与员工】-【登录管理】-【启用】

![](https://vip.kingdee.com/download/0109a4aa18c1c1374b6590bff4674ca34297.png)

(2)输入收取验证码的手机号，并发送验证码确定启用【登录管理】设置

![上传图片](https://vip.kingdee.com/download/010035c9eb8810194c21b36e712e1ab6d02e.png)

(3)启用登录管理的用户，在登录或者切换企业时，会出现企业管理员短信验证的界面

![上传图片](https://vip.kingdee.com/download/01005e24dbadd0cd4d91bd00001ff1fdea0c.png)

(4)如需取消，在【部门与员工】-【登录管理】点击【停用】即可；也可以点击【更新】更改收取验证码的手机号。

![上传图片](https://vip.kingdee.com/download/0100d35866491d8e474796cf93ec13a6348c.png)
