# 由订单生成预收/预付款单,或超收/付生成生预收/预付款单可以修改单据编号吗?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：shaoling | 阅读：47 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/711384056189158912)

【问题描述】

由订单生成预收/预付款单,或超收/付生成生预收/预付款单可以修改单据编号吗?

【解释说明】

预收/预付款单修改单据编码无论是通过订单/超收自动生成还是手工录入的,都遵循统一的单据管理规则。即预收 / 预付款单的单据编码能否修改,取决于系统 “编码规则” 与 “单据操作控制” 的设置。

若“编码规则” 与 “单据操作控制”两项设置均允许修改,则可对单据编码进行编辑修改;

若“编码规则” 与 “单据操作控制”其中任一设置限制修改,或二者同时禁止,则无法对单据编码进行改动。
