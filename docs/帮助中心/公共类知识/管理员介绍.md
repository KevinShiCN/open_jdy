#

小微工作台主要区分企业管理员、普通员工和服务管理员三种角色。具体权限说明如下表所示

![](https://vip.kingdee.com/download/0109685c63e75683491e9693c6c3ce931e07.png)

在【我的企业】、【产品服务】可以查看所在企业的角色身份。

![上传图片](https://vip.kingdee.com/download/010096075fb838af45d7b83f1e9e60bdf84c.png)

## 1、企业管理员

企业管理员拥有最高权限，工作台所有管理相关的功能权限都由企业管理员完成。

本节主要介绍企业管理员与普通员工权限区别，其他共有的功能权限介绍可点击：[快速了解小微工作台](https://vip.kingdee.com/article/560899476575858944)

![上传图片](https://vip.kingdee.com/download/0100fe9c21ac58244cde8e65fd4cbb6aeb09.png)

①我的企业：企业管理员可进行企业认证，具体企业认证步骤可点击：[企业认证](https://vip.kingdee.com/article/561286143204164608)

②部门与员工：当前企业组织架构设置及员工管理，包括新增部门、邀请员工、离职员工等。

![上传图片](https://vip.kingdee.com/download/0100d100b0960d2e42719d95e1db6ec54419.png)

③安全策略：通过安全策略可以在指定的设备上、指定的ip范围内、指定时间范围使用精斗云产品。具体操作步骤请点击：[安全策略设置](https://vip.kingdee.com/article/561288387592905728)
④企业管理员即便不是服务管理员，仍是可以对服务进行基本操作，如移交、变更用户数/账套数、续费等操作。

## 2、普通员工

工作台身份为普通员工且非服务管理员，被授予的服务仅有【进入使用】基本操作；反之，未授权的服务看不到。

![上传图片](https://vip.kingdee.com/download/01003626b06ab93c41c7805a91f5bd5f3252.png)

## 3、服务管理员

每个服务都有对应的【服务管理员】，服务管理员有服务最高权限， 即拥有账套所有管理及操作功能权限，包括工作台服务常用操作，【进入使用】账套后账套内的所有操作功能权限。

![上传图片](https://vip.kingdee.com/download/010099e4598d6b4b441db8e1828b9f7a3490.png)
