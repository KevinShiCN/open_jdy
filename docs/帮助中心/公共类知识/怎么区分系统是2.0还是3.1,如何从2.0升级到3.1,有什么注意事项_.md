# 怎么区分系统是2.0还是3.1,如何从2.0升级到3.1,有什么注意事项?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：_Leo | 阅读：4786 | 有用：8

**【问题描述】 **

怎么区分系统是2.0还是3.1,如何从2.0升级到3.1,有什么注意事项?

【解释说明】

目前星辰正有2个版本产品在同时运行,星辰2.0和星辰3.1(基于苍穹5.0的底层),后续只针对3.1版本进行规划迭代更新,星辰2.0后续不再发布任何新特性,只做日常维护管理,因此会将星辰2.0用户逐步升级更新至新版。本次升级更新迭代不会产生额外的费用!

**从交互界面上可以区分使用的版本是2.0还是3.1,并且系统升级更新之后不支持再退回低版本。**

**【操作指引】**

1、如果**需要升级**,可以提供**[服务ID](https://vip.kingdee.com/knowledge/589258238974049024?productLineId=35&isKnowledge=2&lang=zh-CN)**至售后技术支持,让其反馈至研发处理;

2、如果有生态产品,电商产品,需要和对应的ISV生态厂商确认能否升级;

3、如果购买了钉钉/企业微信星辰专版暂时不能参与星辰3.1Beta,后期升级以具体通知为准。如果已经将钉钉/企业微信专版迁移回星辰2.0产品,可以正常升级星辰3.1

**【温馨提示】**

**未使用**扩展类生态产品(ISV)的客户,或**不存在**已订购扩展类生态产品但未启用扩展类生态产品的客户,可参与星辰3.1BETA

1)如果客户**购买了扩展类生态产品**不能参与星辰3.1Beta,需要等生态产品升级到新平台,以具体通知时间为准,具体请联系对应的ISV生态厂商(如云蝶/为谷等ISV厂商)

2)已经升级到3.1或在3.1上Beta的客户,如果在之前已经订购了扩展类生态产品但未启用的,在星辰3.1上还没有开放支持扩展产品之前,不支持启用扩展类生态产品

3)已经升级到3.1或在3.1上Beta的客户,在未开放对扩展类生态产品支持前,不能订购、使用扩展类生态产品

4)如果客户购买了**钉钉/企业微信星辰专版**暂时不能参与星辰3.1Beta,后期升级以具体通知为准。**如果客户已经将钉钉/企业微信专版迁移回星辰2.0产品,可以正常升级星辰3.****1**

5)若有购买api模块,升级不影响api模块的对接调用,升级时不使用即可。
