# 预计可用库存表显示负数是什么原因?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：shaoling | 阅读：220 | 有用：0

【问题描述】

预计可用库存表显示负数是什么原因?

【解释说明】

系统中预计可用库存表显示负数,可能与以下因素相关:系统参数设置预计可用库存公式的取数逻辑、业务单据处理中允许负库存的设置,或库存数据同步延迟。具体原因分析如下:

1. 系统参数设置的预计可用库存公式取数问题

①未考虑在途 / 待入库单据

场景:已创建采购订单或调拨入库单,但货物尚未实际入库,系统未将这部分 “待入库” 数量计入可用库存。

影响:若当前出库量超过实际库存 + 待入库量,预计可用库存会显示负数。
②. 存在未审核的出库单据
场景:销售订单、领料单等出库单据已创建但未审核,系统提前扣除了库存数量。
示例:实际库存为 100,未审核的出库单扣减 150,预计可用库存显示 - 50。

若您想要了解预计可用库存公式设置问题,请点击:[如何设置可用库存、预计可用库存公式](https://vip.kingdee.com/knowledge/616800689854798336?productLineId=35&isKnowledge=2&lang=zh-CN)

2. 设置允许负库存

系统设置:若在【基础资料】>【仓库】中开启了 “允许负库存”,出库量超过实际库存时,预计可用库存会直接显示负数。

若您想要了解设置是否允许负库存出库问题,请点击:[如何设置商品是否允许负库存出库?](https://vip.kingdee.com/knowledge/129339770097650297?channel_level=%E9%87%91%E8%9D%B6%E4%BA%91%E7%A4%BE%E5%8C%BA%7C%E6%90%9C%E7%B4%A2%7C%E5%AE%98%E6%96%B9%E7%9F%A5%E8%AF%86&productLineId=35&isKnowledge=2&lang=zh-CN)

若您想要了出库负库存问题,请点击:[为什么会出现负库存?](https://vip.kingdee.com/knowledge/645066602005964288?channel_level=%E9%87%91%E8%9D%B6%E4%BA%91%E7%A4%BE%E5%8C%BA%7C%E6%90%9C%E7%B4%A2%7C%E5%AE%98%E6%96%B9%E7%9F%A5%E8%AF%86&productLineId=35&isKnowledge=2&lang=zh-CN)

3. 库存数据同步延迟或异常
数据更新问题:系统未及时更新库存数据;数据同步延迟,建议可校对库存出库,操作详情可查看:[怎么校对库存?](https://vip.kingdee.com/knowledge/616075377152345856?productLineId=35&isKnowledge=2&lang=zh-CN)
