# 移动手机端用户是免费的吗

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题

【问题描述】

移动手机端用户是免费的吗
【解释说明】

金蝶云星辰的账套购买时是分为PC端(即电脑端)和移动端(即通过手机/平板安装金蝶云APP)。购买PC端的时候是默认带有移动端用户一起使用,此时,用户使用移动端是不单独收费的的,可以免费使用。

如果想只增加移动端的用户仅在移动端使用,可对移动端用户另外单独购买。如图所示;![](https://vip.kingdee.com/download/01092b1f0bd059a84beb8f92cd551a2e47c2.png)
【温馨提示】

如果要增加用户数,可以拨打金蝶售前热线400-830-0755进行购买。
