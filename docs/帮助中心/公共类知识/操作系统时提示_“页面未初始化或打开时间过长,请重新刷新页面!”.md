# 操作系统时提示:“页面未初始化或打开时间过长,请重新刷新页面!”

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：shaoling | 阅读：100 | 有用：0

【问题描述】

操作系统时提示:“页面未初始化或打开时间过长,请重新刷新页面!”

【解释说明】

查询报表时若打开时间较长,再次查询前建议先刷新浏览器并重新打开报表页面,目前暂无其他解决方案。

![](https://vip.kingdee.com/download/010935a131977eb447c59e669ab532bbacc2.png)

例如库存查询表,若超过半小时无操作,再次查询时需先刷新浏览器,再重新打开库存查询表页签。
