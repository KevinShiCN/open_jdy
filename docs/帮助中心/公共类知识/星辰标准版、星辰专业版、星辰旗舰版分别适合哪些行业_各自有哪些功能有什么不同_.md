【问题描述】

星辰标准版、星辰专业版、星辰旗舰版分别适合哪些行业?各自有哪些功能有什么不同?

【概述】

目前对行业上没有具体限制,主要根据自己企业业务需求选择使用哪个版本。

三个版本都有支持财务管理

标准版适合单纯的财务或者采购销售业务,简单零售,满足小型企业(10-50人)业财一体化管理需求;

专业版适合批发、零售、生产加工、委外加工需求的公司。

旗舰版适合采购销售单票分离,生产委外,受托加工,门店零售等需求。
