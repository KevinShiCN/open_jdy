# 什么是钉钉专版?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题

【问题描述】

什么是钉钉专版?

【解释说明】

钉钉专版就是钉钉中上架的金蝶云·星辰。金蝶云·星辰-钉钉专版,是作为金蝶和钉钉合力重点打造的一款应用,在业务能力和用户体验上做了一些特性功能。

【如何购买】

在钉钉应用中心中搜索“金蝶”购买:

钉钉工作台——应用中心——搜索金蝶——选择要购买的应用——选择合适的规格。也可以先试用再决定是否购买,提供7天免费试用。![](https://vip.kingdee.com/download/010941fc7af9e89a4ee7b38a74f5968b35f6.png)

**说明**:

1、购买钉钉专版后,可以登录使用PC版(官网[www.jdy.com](https://www.jdy.com/)):

①钉钉扫码登录;②扫码登录后—账号设置—修改密码—才能账号密码登录。

但无法使用精斗云APP(金蝶云·星辰);![](https://vip.kingdee.com/download/0109d21f608075ef40a8aeb63fe14e519624.png)

2、购买金蝶云·星辰PC版,无法使用钉钉专版。

如需要了解更多钉钉专版功能与价格,可拨打售前热线:400-830-0755、联系销售人员及金蝶售后在线服务(产品端-右上角售后在线服务)进行反馈
