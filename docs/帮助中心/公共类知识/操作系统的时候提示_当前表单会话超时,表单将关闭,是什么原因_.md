# 操作系统的时候提示:当前表单会话超时,表单将关闭,是什么原因?

【问题描述】

操作系统的时候提示:当前表单会话超时,表单将关闭,是什么原因?
![](https://vip.kingdee.com/download/0109e1347115b43847cba693ee0b713e153b.png)
【解释说明】

长时间未操作系统时,会出现提示当前表单会话超时,表单将关闭,出现该提示建议退出账号重新登录。之前页面的数据,若没有操作过保存,数据是无法找回的,需要重新录入操作。

若重新登录操作时再次提示,请提供**[服务ID](https://vip.kingdee.com/link/s/ZzcJn)**联系售后技术支持

【温馨提示】

若在操作系统过程中有需要离开或做别的工作,建议先把当前正在编辑的界面保存,防止出现数据未保存的情况。
