# 企业微信应用授权的时候提示:您的企业微信账号不是管理员,请让管理员在管理后台授权应用管理权限

【问题描述】

企业微信应用授权的时候提示:您的企业微信账号不是管理员,请让管理员在管理后台授权应用管理权限

【操作说明】

企业微信版本的星辰账套授权,需要企业微信的管理员授权可见范围。

要检查您的企业微信账号是否具备管理员权限,您可以按照以下步骤操作:

- **登录企业微信管理后台**:使用您的企业微信账号登录企业微信管理后台。

- **查看权限管理**:在管理后台界面,点击【我的企业】,然后选择【权限管理】。

- **检查管理员列表**:在【权限管理】页面,您可以看到管理员列表。如果您的账号在管理员列表中,那么您就具备管理员权限。

- **查看管理范围和权限**:如果您的账号是管理员,您还可以查看您的管理范围和具体的权限设置,包括通讯录权限、应用管理权限等。

- **手机端查看**:如果您更习惯使用手机操作,可以在企业微信手机端通过【工作台】->【管理企业】->【企业信息】查看创建人信息,或者通过【设置管理员】查看管理员列表。
