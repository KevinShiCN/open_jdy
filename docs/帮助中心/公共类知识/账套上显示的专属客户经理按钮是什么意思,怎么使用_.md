# 账套上显示的专属客户经理按钮是什么意思,怎么使用?

【问题描述】

账套上显示的专属客户经理按钮是什么意思,怎么使用?

![](https://vip.kingdee.com/download/0109e28b5353fafb4a2da076829daa4f014f.png)【解释说明】

<专属客户经理>按钮:为方便您能快速联系到该账套对应的客户经理,此功能不影响您正常使用账套

例如:有购买、续费、增加账套、减少账套、增减用户等需要协助处理事宜,可快速联系您的专属客户经理咨询。

**说明**:

①仅企业管理员及账套服务管理员支持此功能且不支持隐藏,普通用户默认不展示。

②若点击<专属客户经理>按钮未展示联系信息,说明销售系统没有维护该信息,您可拨打销售热线400-830-0755协助处理。

③此功能陆续更新中,若企业管理员和账套服务管理员未看到此按钮说明还未更新到,敬请期待~
