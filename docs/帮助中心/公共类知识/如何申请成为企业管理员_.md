# 如何申请成为企业管理员?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：快速入门, 整体介绍
> 作者：杨桂平 | 阅读：2971 | 有用：5
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/462422664775460353)

#### 【业务场景】

因企业管理员离职且未交接等原因导致企业管理员账号丢失,导致无法继续使用企业已经购买的服务,需设置新的用户成为企业管理员

如:A公司的企业管理员是甲,甲离职未做交接,导致系统无法登录使用,如何设置乙成为A公司新系统管理员?

#### 【解决方案】

 新管理员账号登录www.jdy.com 发起自助服务--新增企业管理员流程,申请成为A公司企业管理员

#### 【操作流程】

**注:需由新管理员本人的账号发起申请,不可他人代****申请。如乙需申请成为A公司的企业管理员,则应登录乙的账号发起申请。**

**1、发起申请****
**

1-1、如果您当前登录的账号即是要申请成为企业管理员的账号,请点击:[新增企业管理员](https://service.jdy.com/workbench/web/index.html#/helpSelf/addCorpAdmin)可直接跳转到新增界面

1-2、如果您当前登录的账号不是要申请成为企业管理员的账号,请按以下步骤操作:

申请人登录官网:www.jdy.com 后,点击工作台左侧的**【自助服务】**进入自助服务界面

![](https://vip.kingdee.com/download/0109e9f8725fab1844119ade9dfc935697b9.png)

在自助服务中点击【**新增企业管理员**】打开新增企业管理员界面

![](https://vip.kingdee.com/download/0109402612f6b1aa4169b4eed2a56d0a4cfd.png)
然后点击右上角**【新增企业管理员】按钮;**

![](https://vip.kingdee.com/download/010900f79296133a43698673fe666b4a931d.png)

**2、填写申请内容**

打开新增企业管理员的申请界面,按要求填写申请内容:

![](https://vip.kingdee.com/download/0109bc30de75a0314531934688f12b20f6f3.png)

![](https://vip.kingdee.com/download/01090ff11cd4e1b74dff9f9940ea015ae815.png)

①当前申请人:不同的申请人需要上传的资料不同,请根据实际情况进行选择

| 申请人类型 | 必须上传资料 | 辅助资料 |
| --- | --- | --- |
| 企业 | 必传资料:1、 申请书扫描件(加盖公章);2、 企业营业执照(复印件加盖公章);3、 工商变更登记通知书(如发生过变更需提供) | 1、 购买服务发票记录2、 购买服务付款截图3、 购买服务电子合同4、 购买服务转账记录5、 与销售人员聊天记录6、 其他 |
| 个人 | 1、 申请书扫描件(加盖公章);2、 本身身份证(电子档或复印件) |  |
| 其他组织 | 1、 申请书扫描件(加盖公章);2、 组织机构代码证或其他证件(在有效期内且年检章齐全) |  |
|  |  |  |

②企业名称:必录项,如申请成为A公司的企业管理员则企业名称填写“A公司”

③情况说明:非必录项,如不能准确填写企业名称或有其他特殊情况,可在情况说明中进行补充说明或上传补充资料。补充说明越详细越有助于快速完成审批。

④新管理员账号:新管理员账号(不可编辑,系统自动获取当前登录账号)

⑤手机验证码:点击获取验证码

**3、下载申请书**

短信校验通过后,可下载系统自动生成申请书进行填写,同时准备其他需上传的资料。

![](https://vip.kingdee.com/download/010928600797d4ac45fbb460a58f6ef0c060.png)

**4、上传资料提交审批**

资料准备完成后点击“上传资料”,根据界面展示的必要资料和辅助资料进行上传即可。上传界面支持暂存,确认无误后点击**【提交审批】**。

![](https://vip.kingdee.com/download/01096ef30a2986db49589670ea3b292e7fbb.png)

**【温馨提示】**

1、上传文件格式要求:支持pdf、jpg、png格式

2、上传文件大小要求:不超过10MB

**5、审批说明**

审批时间:工作日24小时内处理审批,如提交时间为节假日,顺延至工作日完成审批。

审批状态说明:

① 状态为“审批中”时,用户可自主取消已提交的审批,可修改后再重新提交

② 状态为“待上传资料”时。可删除记录。,其他状态均不允许删除

③ 状态为“待补充资料”和“审批不通过”时。可点击【查看原因】查看具体原因后进行资料补充,或重新进行申请。

④ 状态为“已完成”时。可具体查看实际加入的企业信息。
