# 单据/凭证的附件会占用账套内存吗?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：云社区用户13pG7392 | 阅读：157 | 有用：0

【问题描述】

单据/凭证的附件会占用账套内存吗?

【概述】

每个账套初始使用空间都为2GB

这里分为两种情况:

1.1 数据库存储:直接统计的物理文件大小,根据数据表字段的数量,这里2GB可以存储单据+基础资料数量在30万~100万之间。

1.2 文件存储:这里是1:5的规则来计算的,相当于可以存储10GB的附件、图片等文件资源。

注意:这里的“2GB”指的是上面二者之和。
