# **1 问题描述**

什么样的账套需要购买VIP部署服务？

VIP部署服务主要包含哪些服务？

# 2 适用版本

星辰3.1以上所有版本

# **3 服务说明**

### 1）什么样的账套需要购买VIP部署服务？

答：金蝶云星辰客户如果账套中的商品数（SKU）超过1万以上，或单据量超过10万张（每张单分录平均4条以上），或凭证量超过5万张（每张凭证分录平均4条以上），建议购买VIP部署服务；

### 2）VIP部署服务主要包含哪些服务？

答：购买了VIP部署服务的客户享有以下尊享权益：

1、 每账套的容量扩大到10G(非VIP客户为5G),数据存储更随心；

2、 VIP部署条件下更高的服务器性能保障，产品运行更顺畅；

3、 产品界面展示尊贵VIP客户标识；

![上传图片](https://vip.kingdee.com/download/010027612f8c83d140da827c3fa43dd07f5c.png)

4、更高的引出上限值

系统引入引出只有单次引入引出数据量的限制,没有一天内操作引入引出次数的限制,具体规则如下:

| 数据类型 | 单次引出上限 | 单次引入上限 | 额外限制 |  |
| --- | --- | --- | --- | --- |
| 普通分组 | VIP分组 | 所有分组 |  |  |
| 凭证 | 5万条 | 5万条 | 3万条 | 单张凭证分录行数≤3000 条 |
| 业务单据 | 3.5万条 | 5万条 | 3.5万条 | 单张单据单次录入≤3000 条 |
| 盘点单 | 2万条 | 2万条 | 2万条 | —— |
| 商品 | 3.5万条 | 5万条 | 3.5万条 | —— |
| 报表 | 3.5万条 | 5万条 | —— | 仅支持导出,无导入场景 |

>
说明：①假如一张订单或凭证含10行分录,那就是算10条。

②单次是指一次引入引出。由于引入和引出数据量过大会有性能问题,建议在数据量大的情况下可以分多次进行。
