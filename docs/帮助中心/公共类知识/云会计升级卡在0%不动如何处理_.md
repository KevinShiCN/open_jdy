# 云会计升级卡在0%不动如何处理?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：26 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/769728100065773568)

【问题描述】

 云会计升级星辰卡在0%不动如何处理?

![](https://vip.kingdee.com/download/01097899b7fcd24c4f3f8f3ed3dd0616c23f.png)

【解决方案】

在工作台界面确认云会计服务管理员账号。

![](https://vip.kingdee.com/download/01096bc80818920e4902848864dd6a5f7ad0.png)

用服务管理员账号登录升级且是手机号登录才可以,否则无法读取云会计数据。

![](https://vip.kingdee.com/download/01097e351441c2fc42619a769bbf819e61d8.png)
