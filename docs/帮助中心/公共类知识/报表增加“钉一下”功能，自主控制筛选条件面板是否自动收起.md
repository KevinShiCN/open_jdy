# 报表增加“钉一下”功能，自主控制筛选条件面板是否自动收起

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：快速入门, 功能介绍, 新特性
> 作者：移动生产白建凡 | 阅读：194 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/781483967760299520)

# **1 功能介绍**

原报表查询时，输入查询条件并点击“确认”后，过滤条件面板会自动收起。当用户需要频繁切换、修改查询条件时，需反复点击“展开过滤”，操作繁琐

![上传图片](https://vip.kingdee.com/download/01007f5e16a5096c4defbe9dcaf6c94c3c67.png)

本次新增“钉一下”功能，可自助控制过滤面板是否自动收起，提升报表使用便捷性

![上传图片](https://vip.kingdee.com/download/0100dacd8981a32b475d851de049946a0732.png)

# **2 适用版本**

星辰3.0以上所有版本

# **3 功能特性**

- 不启用钉一下（默认状态）：点击查询按钮后，报表过滤面板自动完全收起

![上传图片](https://vip.kingdee.com/download/01006b30fc5c6eba477e91891bbbb85c636e.gif)

- 启用钉一下：点击“查询”按钮后，报表筛选条件面板保持当前展开状态不自动收起，可手动点击“展开过滤”按钮，展开过滤面板

![上传图片](https://vip.kingdee.com/download/0100c6e411362e4940ebbe35d853187b4018.gif)
