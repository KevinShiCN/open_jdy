# 正式产品的账套数据保留与回收常见问题（如：续费、过期、备份与恢复）

2、购买财务模块和进销存模块，操作了账套备份，等到期后只续费了进销存模块，这个时候操作备份恢复，可以恢复之前那个备份文件记录吗？

# 2 详细说明

## **2.1 过期模块数据问题**

### **问题1：****现在使用了财务+进销存,假设到期后想只续费使用财务,进销存暂时不续费,进销存数据会保留吗?数据还能查看吗?后续如果进销存又续费,数据还能恢复吗?**

**答：正式服务还未到期**,但是部分模块过期的(例如购买了财务云和进销存云,进销存云过期不续费了,但是财务云还正常使用中)：只要正式账套中,有其他星辰模块在用,那么即使部分模块过期了,且该模块不续费,过期的模块不会清理,数据还是保留的,但是只能查询,不能操作。

## **2.2 备份与恢复问题**

### **问题2：**购买财务模块和进销存模块，操作了账套备份，等到期后只续费了进销存模块，这个时候操作备份恢复，可以恢复之前那个备份文件记录吗？

**答：**如果备份包里含了过期模块的数据，且过期模块没有续费，**仍然是可以恢复的**。但只会显示在服务期内的模块数据，没续费的模块的数据不会显示。

### **问题3：**标准版的账套，如果购买数据恢复服务，可以直接恢复到专业版账套吗？

**答：****支持从低版本购买数据服务后，恢复到高版本的产品中，不支持从高版本恢复到低版本。**

标准版的账套，如果购买数据恢复服务，**可以**直接恢复到专业版账套。

同理，专业版的账套，如果购买数据恢复服务，**可以**直接恢复到旗舰版账套。

## **2.3 过期服务的问题**

### **问题4-6：**整个账套过期后,数据还能查看和操作吗?数据可以保留多久?被回收了还能恢复数据吗?

**答：****若整体租户服务过期的**(即工作台的服务到期时间显示“已到期”):

![](https://vip.kingdee.com/download/0100844ac7e3293e42eb9981b1adc1600eb4.png)

**1)操作权限:**

**过期后，账套支持查询、引出。不支持新增、引入、打印、调整列设置等具体功能的操作。**

正式账套:过期后只读（包含以上提及的操作权限）,回收后不能进产品;

试用账套:过期后15天只读（包含以上提及的操作权限）,回收后不能进产品。

**2)回收规则:**

正式账套:从到期日开始算起,6个月后回收;

试用账套:从到期日开始算起,过期后15天回收

**3)数据恢复问题：**

**已回收的账套数据由于已在数据库进行物理删除，不支持恢复!**
