# 如何查询api接口的服务时间?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：shaoling | 阅读：277 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/726273533801879296)

【问题描述】

如何查询API接口的服务时间?

【解释说明】

API接口的服务时间可登录[金蝶云小微企业开发者后台](https://open.jdy.com/),进入【我的应用】,选择对应的应用点击进入API接口应用页面

![](https://vip.kingdee.com/download/01090b283ce03a9b46f4be5e0d37ea095af8.png)
在对应的API接口应用页面,选择【企业授权】查询有效期即可
![](https://vip.kingdee.com/download/0109ca68a0c416b34898ae3f2f1483bf8dbf.png)
