**2、API接口如何购买**

当完成API调试后,**需要购买API授权服务并生成授权链接,最终使用的企业通过授权链接完成授权后,企业的账套接口才可以被调用**。在开发者后台应用详情中,可购买API授权服务。

此操作建议由ISV伙伴或者企业内部IT技术人员进行。

【温馨提示】

金蝶面向全球企业应用开发者,全面开放API,助力行业发展。

一、什么是金蝶小微生态开放平台

金蝶小微生态开放平台是金蝶公司为小微企业和开发者提供的一个开放的云服务平台。它基于金蝶的云技术,旨在帮助合作伙伴和开发者快速构建、部署和运营企业级应用。这个平台提供了一系列的工具、服务和接口,以支持创新和业务扩展。

二、到小微生态开放平台能做什么?

金蝶小微生态开放平台提供了一系列的功能和服务,使得开发者、合作伙伴和企业用户能够实现多种应用场景和业务需求。以下是一些主要的功能和服务:

①API接口:提供标准化的API接口,方便开发者和合作伙伴快速集成金蝶的业务服务到自己的应用中。

②沙箱环境:提供安全的测试环境,允许开发者在不影响生产环境的情况下测试他们的应用。

③消息订阅:支持开发者订阅和接收平台消息,实现应用与金蝶平台的实时数据交互。

④企业级云原生架构:基于云原生技术,提供稳定、可扩展的企业级服务。

⑤企业内部应用开发:允许企业或委托的开发者开发仅供单个企业使用的内部应用。

⑥第三方企业应用开发:支持SaaS软件服务商开发可服务于多个企业的第三方应用,并可在金蝶应用市场出售。

⑦生态合作:与生态伙伴共建开放、多样、共赢的业财税SaaS新生态。

⑧赋能支持:提供专业赋能,帮助提升产品研发、营销推广及客户成功的核心竞争力。

⑨产品和技术能力:金蝶小微生态开放平台拥有强大的产品和技术能力,能够支持企业实现数字化转型。

通过金蝶小微生态开放平台,开发者和企业可以构建和扩展自己的业务应用,实现与金蝶云服务的无缝集成,加速企业的数字化转型进程。
