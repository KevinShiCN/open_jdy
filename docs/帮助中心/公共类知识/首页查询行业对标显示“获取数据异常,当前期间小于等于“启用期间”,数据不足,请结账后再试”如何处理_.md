# 首页查询行业对标显示“获取数据异常,当前期间小于等于“启用期间”,数据不足,请结账后再试”如何处理?

> 作者：Simone邓 | 阅读：25 | 有用：0

【问题描述】

首页查询行业对标显示“获取数据异常,当前期间小于等于“启用期间”,数据不足,请结账后再试”如何处理?

![](https://vip.kingdee.com/download/0109c51d5c747b464e7cad636dcc575a48b3.png)

【解释说明】

出现以上提示的原因是财务模块还在启用期间未结账,如果当前期间的账已做完,建议可以进行结账之后再查询行业对标数据。

【账务处理】-【期末结账】界面进行结账,[财务期末结账与反结账如何操作?](https://vip.kingdee.com/link/s/ZcxkG)
