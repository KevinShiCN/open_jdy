# 打印提示安装最新打印插件如何处理?

【问题描述】

打印提示安装最新打印插件如何处理?

**【业务场景】 **

点击打印时,系统提示“检测到未安装最新的打印插件,为保证您的正常打印,请先下载安装打印插件,并在安装成功后按F5刷新浏览器”如何处理?如图:

![](https://vip.kingdee.com/download/01099b2c2755bc6b40029463e899c1b2bbaa.png)
**【解释说明】**

**出现以上提示说明打印设置中选择的打印方式是“插件打印”,且未安装或安装的不是最新插件;若不需要使用插件打印,可修改为“pdf打印”**

若已经安装了最新版安装包,仍然提示,请点击:**[选择插件打印后,点击打印,页面提示“检测到未安装最新的打印插件”是什么原因?如何解决?](https://vip.kingdee.com/link/s/ZVDD4)**参考解决****

![](https://vip.kingdee.com/download/01095a89ccd4c9584f4db1fd9152a308ebe4.png)

若确实需要使用插件打印,请按以下步骤操作

【操作指引】

为解决已知打印问题,优化打印体验,系统统一发布了最新的打印插件,若出现此提示,一定要重装最新打印插件,才可再继续使用插件打印,否则系统检测到不是最新打印插件,会一直弹框提示。

打印设置界面下载最新版安装包安装:
![](https://vip.kingdee.com/download/0109d15c24efe11d4540a8989a320cd255d5.png)

![](https://vip.kingdee.com/download/0109869e316018db44f29b2db3e8cd2f0886.png)

**【温馨提示】**

重装打印插件前: 建议先将电脑的安全卫士关闭掉,以免被拦截;

重装打印插件后:** **在浏览器界面先按住F5刷新浏览器,再使用插件打印。
