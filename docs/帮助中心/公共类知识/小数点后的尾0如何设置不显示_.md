# 小数点后的尾0如何设置不显示?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：162 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/788572800126052352)

【问题描述】

小数点后的尾0如何设置不显示?

【解释说明】

**一、财务模块**

| 操作界面 | 尾 0 显示规则 | 设置方法 |
| --- | --- | --- |
| 凭证录入界面 | 尾 0 不支持隐藏 | 金额小数位由币别金额小数位控制,无法手动调整尾 0 不显示 |
| 凭证列表界面 | 支持隐藏尾 0 | 进入列设置,关闭数值列尾零显示按钮,详细可点击:凭证查询列表,如何设置尾零不显示? |
| 账簿查询界面 | 支持隐藏尾 0 | 1. 金额小数位由账套本位币金额小数位自动带出,进入列设置 “选项”,关闭数值列尾零显示按钮3. 数量金额明细账 / 总账可单独设置数量、单价的小数位精度,详细可点击:数量金额明细账如何设置小数位的显示位数? |
| 财务报表查询界面 | 尾 0 不支持隐藏 | 金额小数位由本位币金额小数位自动带出,报表及打印界面均无法设置隐藏尾 0 |
| 日记账界面 | 支持隐藏尾 0 | 日记账列设置界面,关闭“数值列尾零显示”按钮,详细可点击:日记账界面怎么设置/取消数值列尾零显示? |
| 工资统计表界面 | 支持隐藏尾 0 | 列设置界面,关闭“数值列尾零显示”按钮,详细可点击:工资统计表上如何设置不显示尾零? |

**二、进销存模块**

| 操作界面 | 尾 0 显示规则 | 设置方法 |
| --- | --- | --- |
| 单据录入界面 | 支持隐藏尾0 | 在单据编辑界面,点击表体右上角列设置,在选项中,关闭“数值列尾零显示”按钮 |
| 单据列表界面 | 支持隐藏尾 0 | 进入列设置 “选项”,关闭数值列尾零显示按钮,详细可点击:单据列表如何设置数值尾零不显示? |
| 报表查询界面 | 支持隐藏尾 0 | 进入报表列设置界面,关闭数值列尾零显示按钮,详细可点击:报表上如何设置尾0不显示? |
