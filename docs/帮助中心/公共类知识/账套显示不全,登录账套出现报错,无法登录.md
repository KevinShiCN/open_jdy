# 账套显示不全,登录账套出现报错,无法登录

【问题描述】

账套显示不全,登录账套出现报错,无法登录

【业务场景】

登录账套后,部分账套不显示,如有4个账套只显示出1个,或登录出现报错、无法登录

![](https://vip.kingdee.com/download/0109d1a9c57162dc4c3b8ee9e9df0711ef0e.png)![](https://vip.kingdee.com/download/010904c522db3e164f54b5e27d62bf6d563a.png)

【解释说明】

因网络环境波动,导致部分用户登录账套后,有的账套不显示,或登录出现报错、无法登录。
