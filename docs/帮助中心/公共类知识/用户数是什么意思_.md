# 用户数是什么意思?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题

【问题描述】

用户数是什么意思?

【解释说明】

用户数指的是同时使用的人数。也可以这样认为,用户数指可以同时进入到一个账套操作的人员。

例如,会计系统有人做账,有人审核,有人查账,就需要3用户;进销存系统,财务做账,销售、采购、库管进去做单据,他们使用不同的账号,拥有各自不同权限,就代表的是4用户。

【温馨提示】

账号(用户数),不能同时共享,若不同人同一时间登录一个账号,那么前面登录的,将会被T出系统。

账套服务管理员是需要算用户数的,如果只购买一用户,那么就只有账套服务管理员可以登录操作,无法再授权其他用户操作,增加用户数,需购买操作
