# 应收应付系统帐龄分析与财务处理的帐龄分析有什么不同?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：shaoling | 阅读：51 | 有用：0

【问题描述】

应收应付系统帐龄分析与财务处理的帐龄分析有什么不同?

【解释说明】

**应收应付系统账龄分析:**基于企业业务应收应付管理系统,对往来款项(应收账款、应付账款)按账龄区间(如 0-30 天、31-90 天等)进行分类统计。

若您想要了解应收应付账龄分析应用问题,请点击:[账龄分析表如何应用](https://vip.kingdee.com/knowledge/454811946219344384?channel_level=%E9%87%91%E8%9D%B6%E4%BA%91%E7%A4%BE%E5%8C%BA%7C%E6%90%9C%E7%B4%A2%7C%E5%AE%98%E6%96%B9%E7%9F%A5%E8%AF%86&productLineId=35&isKnowledge=2&lang=zh-CN)

![](https://vip.kingdee.com/download/0109c322f663a9064f3290969f3ab29e9384.png)

**财务处理的账龄分析:**基于财务科目管理中设置往来业务核算的科目(如应收账款、其他应收款、应付账款、其他应付款等),按账龄区间(如 0-30 天、31-90 天等)进行分类统计。
