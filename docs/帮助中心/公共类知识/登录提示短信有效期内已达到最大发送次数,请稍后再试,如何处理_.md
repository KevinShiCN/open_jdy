# 登录提示短信有效期内已达到最大发送次数,请稍后再试,如何处理?

【问题描述】

登录提示短信有效期内已达到最大发送次数,请稍后再试,如何处理?

【业务场景】

之前都可以正常登录的,今天登录突然出现短信发送限制了,提示短信有效期内已达到最大发送次数

![](https://vip.kingdee.com/download/0109b9013385470544cba56999857473e9ca.png)

【解释说明】

系统提示:"短信有效期内已达最大发送次数,请稍后重试 ”,这是由于已发送的**三条短信中的两个验证码尚未使用且仍在有效期内**,出现这个提示建议可以找到历史的短信信息,直接使用已发送的验证码登录即可。
