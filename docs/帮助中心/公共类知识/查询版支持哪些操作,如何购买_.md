# 查询版支持哪些操作,如何购买?

【问题描述】

查询版支持哪些操作,如何购买?

【业务场景】

星辰查询版适用于用户已购买星辰标准服务产品,服务到期后因一些原因不再续租,但是希望通过付费的方式,保留并查询账套已有数据。

【解释说明】

**1.查询版仅支持:查询、引出、打印、账套备份与恢复、用户权限管理。若需使用新增,编辑等更多权限,需续费标准服务。******

**2.查询版会在标准服务到期当天,进入账套给出【产品服务已到期提醒】弹窗,引导续费标准服务及购买查询服务。(未到期不支持购买查询服务)**

点击【立即续费】,则进入续费标准服务订单界面;

点击【购买查询服务】,则进入查询服务购买界面,点击立即下单,提交订单并支付即可。

![](https://vip.kingdee.com/download/01096867da05e8534dd5bd0375064896038d.png)

**3.购买查询版的账套数以账套内已启用的账套为准,不支持选择部分账套进行购买,除非联系销售先减账套后,再购买剩余已启用的账套数。**例如已经启用了四个账套,但是只需要用到三个,希望购买三个是不支持的,除非先联系销售减账套后,再购买剩余的三个账套的查询版。

**tips:减账套后账套数据无法恢复,建议及时下载备份账套数据或者备份账套,详情请点击:[减账套续费,需要减少的账套还可以进入吗?](https://vip.kingdee.com/knowledge/507720828167808512?productLineId=35&isKnowledge=2&lang=zh-CN)**
