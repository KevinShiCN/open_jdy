# 登录界面是空白的?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题
> 作者：杨桂平 | 阅读：474 | 有用：0

【问题描述】

登录界面是空白的?

【解决方案】

建议您换成谷歌浏览器,其他浏览器可能兼容性不太好;如果已经在使用谷歌,请按ctrl+h清除浏览器缓存数据,再按ctrl+f5刷新浏览器,再次登录系统。
