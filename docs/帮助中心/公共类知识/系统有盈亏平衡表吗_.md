# 系统有盈亏平衡表吗?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题

【问题描述】

系统有盈亏平衡表吗?

【解释说明】

系统目前**没有**盈亏平衡表功能。

如需查询进销存相关的收入、销量与成本,建议查看销售利润表;如需查询财务会计的收入与成本,建议查看利润表
