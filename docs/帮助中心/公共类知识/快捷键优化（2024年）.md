# 快捷键优化（2024年）

1 简介

### 1.1 功能介绍

快捷键优化，支持批量删除单据分录行，支持ctrl+s保存二级弹窗、支持ctrl+D批量向下填充

### 1.2 适用版本

金蝶云·星辰3.1所有版本

### 1.3 单据范围

进销存单据，具体支持范围见下文说明

### 1.4 快捷键功能说明

| 快捷键 | 功能描述 |
| --- | --- |
| Ctrl+S | 保存（在弹窗中执行“确认按钮”功能） |
| shift+delete | 删除当前行，鼠标右键可批量删除选中行tips文案：支持鼠标滑动选中多行 |
| F9 | 自动填充收退款金额，并携带默认收款账户。支持单据范围：专业版：自动填充采购入库单/退货单、销售出库单/退货单、委托结算单/结算退货单的收退款金额旗舰版：自动填充采购发票、销售发票、委托结算单/结算退货单的收退款金额 |
| CTRL+D | 复制选中行首行内容，向下填充 |
| CTRL+shift+← | 调出或关闭分屏 |
| CTRL+V | 粘贴 |

## 2 操作指南

### 2.1 批量删除单据分录行

1）鼠标在单据体分录中，点击右键，选择“删除所有行”则删除单据体中所有有序号的行

![上传图片](https://vip.kingdee.com/download/01001cfb09b4cad74b44b0be29252df3541f.gif)

2）鼠标批量选中单据分录，点击右键，选择“删除选中行”，则只删除选中的行，不删除没有选中的行

![上传图片](https://vip.kingdee.com/download/0100dc12cee580344fcdbae4f201a6653445.gif)

### 2.2 ctrl+s保存优化

- #### 功能点1：直接键盘ctrl+s，可以保存单据
- #### 功能点2：详情页二级弹窗点击ctrl+s支持携带选中的信息返填到单据

操作说明：如果单据/基础资料的详情页中，打开了任意的二级弹窗，例如商品、二维录入弹窗、单据体客户/供应商等等，那么当在二级弹窗中点击键盘ctrl+s时，会直接携带二级弹窗中选中的信息返回到单据上（如下GIF图）

### 2.4 Ctrl+D批量向下填充、Ctrl+V复制粘贴、F6批量填充优化

增加支持产品内Ctrl+D批量向下填充的字段，支持Ctrl+V复制粘贴的字段，与F6批量填充的字段

| 快捷键 | 可实现的效果 | 支持该快捷键的字段-仅列出本次新增的字段 |
| --- | --- | --- |
| F6 | 向下批量填充 | 注册证号、生产许可证、产地字段、单位、辅助单位、换算率、折扣、含税折扣额、折后单价、折后金额、整单折前价税合计、不含税金额 |
| Ctrl+V | 粘贴 | 折扣率、赠品行 |
| Ctrl+D | 按指定区域批量填充。使用方法：复制选中行首行内容，在选中区域内向下填充（具体应用流程见下文。） | 仓库、仓位、辅助单位、单价、含税单价 |

#### Ctrl+D的使用方式及效果

假设一张单有五行分录，我们选中分录行中的第二三四行的“单价”字段，然后点击Ctrl+D，那么系统就会把第二行的单价2，批量向下填充到第三四行的“单价”，但是不会填充到第五行，第五行单价还是5。

通过下面动图可以可以更直观了解其效果及应用：

![上传图片](https://vip.kingdee.com/download/0100b25b1fdf6c444b93b92fa075e7316549.gif)

#### Ctrl+D与F6的区别？

- Ctrl+D是按指定区域批量填充，F6批量填充是以选中的行向下填充，是不区分选择哪几行的，统一按列来向下生效，因此用户可以根据实际业务需要进行选择和应用。
- 如果需要向下批量填充的数据太多时，用F6可能会比较慢，更建议用Ctrl+D
