【问题描述】

售后咨询记录如何查看?

【操作步骤】

1、官网链接:www.jdy.com,登录自己的账号,点击进入使用![](https://vip.kingdee.com/download/0109e95928b5905e477ab9c37a72885caaf7.png)

2、进入账套,点击账套右上角【售后在线服务】
![](https://vip.kingdee.com/download/010981014b683fad4847b4ad8c67b8e7a827.png)
2、进入小K页面,点击中间上方【点击查看历史记录】即可查看之前咨询售后服务记录![](https://vip.kingdee.com/download/010932e8fe0f0a2c4e2ea0c565d508e4dcad.png)
