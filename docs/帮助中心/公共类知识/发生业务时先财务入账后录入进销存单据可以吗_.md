# 发生业务时先财务入账后录入进销存单据可以吗?

> 产品：金蝶云·星辰 | 分类：公共类知识 | 类型：常见问题

【问题描述】

发生业务时先财务入账后录入进销存单据可以吗?

【解释说明】

系统不限制先后顺序,根据公司业务情况进行录入财务数据或者进销存单据,后续保证两边的数据一致即可。

【注意事项】

先录财务数据后录进销存单据,不要再通过单据生成凭证,以免数据重复。
