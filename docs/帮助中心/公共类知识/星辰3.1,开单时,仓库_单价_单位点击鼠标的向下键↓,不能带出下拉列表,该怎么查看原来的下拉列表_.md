# 星辰3.1,开单时,仓库/单价/单位点击鼠标的向下键↓,不能带出下拉列表,该怎么查看原来的下拉列表?

**【问题描述】**

**星辰3.1,开单时,仓库/单价/单位点击鼠标的向下键↓,不能带出下拉列表,该怎么查看原来的下拉列表?**

![](https://vip.kingdee.com/download/0100e3d019af892443c5842ab2130cab2254.png)

**【问题说明】**

 星辰3.1对开单含有下拉列表控件的字段(如仓库/单价/单位等)的向下键↓做了改造,针对这类特殊控件的字段做了优化,兼容了2种不同的应用场景,一种是原有的调出列表功能,另一种是新加的换行的功能。具体操作往下看。

****

【适用版本】

 星辰3.1以上版本

**
**【应用场景及操作流程】

场景一:录单时需要调出下拉列表(如仓库/单价/单位等),进行选择(如下图)

![](https://vip.kingdee.com/download/010024e19445a3454571b02c46543928b9e7.png)

操作方法:鼠标光标定位到对应的单元格后,敲键盘的空格键,即可带出下拉列表,然后再按住键盘的上下箭头可以进行选择,敲回车键即可带到单元格内。

场景二:直接切换到同一列换行里的单元格进行录入(如下图)

![](https://vip.kingdee.com/download/0100361d81da69f74e92b5d09056db8b1e20.png)

操作方法:鼠标光标定位到对应的单元格后,按住键盘的下箭头↓则直接切换下一行的单元格里。

然后如果需要调出下拉列表,就敲空格键即可,操作方法见上文的场景一说明即可。
