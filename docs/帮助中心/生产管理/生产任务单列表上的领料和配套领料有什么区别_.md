# 生产任务单列表上的领料和配套领料有什么区别?

【问题描述】

生产任务单列表上的领料和配套领料有什么区别?

![](https://vip.kingdee.com/download/0109132f2cb845674cb8888841d045af1ddc.png)

【业务场景】

**领料:**适用于不需要分批生产、材料库存足够一次领完,或者材料品种少计算量小,可以自行快速计算好要领料的数量,下推领料单后手工修改领料数。

**配套领料:**系统能够提供当前库存的可领料套数和实际需要领用的套数,帮助用户快速了解当前可领取的材料,并根据用户决定领用的套数计算出每一个材料的领用数量。

①一张任务单,生产数量300,由于产能等原因,需要分批生产、分批领料,如每次只生产30个,就一次领生产30个产品需要的材料,手动计算每一种材料要发多少数量,工作量大且易出错。

②一张工单需要发20种材料,生产数量20,但本次库存数只够发15套材料,需要系统自动计算15套的每一种材料应该发多少数量,希望自动按库存数量进行发料。
