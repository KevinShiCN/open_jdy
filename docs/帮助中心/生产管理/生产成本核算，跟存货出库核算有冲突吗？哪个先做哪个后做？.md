# 生产成本核算，跟存货出库核算有冲突吗？哪个先做哪个后做？

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题, 快速入门
> 作者：Esperanza | 阅读：1448 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/354933495531475200)

生产成本核算的时候，系统会自动先调存货出库成本核算计算材料的出库成本。即如果没做出库核算，直接做生产成本核算，不影响产品入库的成本计算。
