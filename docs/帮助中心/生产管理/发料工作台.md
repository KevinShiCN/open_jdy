# **1 简介  **

## **1.1 功能介绍**

发料工作台支持集中处理发料场景，包含按未领数量、分仓领料、按库存数量、补足已领套数、配套退料等，并支持移仓，帮助客户在同一平台实现多种发料需求。

## **1.2 适用版本**

星辰旗舰版

# **2 操作指南**

**1、查询生产任务单/委外加工单**

打开生产管理/委外管理模块对应的发料工作台，查询出需要发料的生产任务单或委外加工单。

![上传图片](https://vip.kingdee.com/download/01009fb8d394deb94457bc7e0f4bde318b11.png)

**2、选择所需要的领料策略和领料方式**

**①支持5种领料策略**

- **按未领数量：**当选择“按未领数量”，点击【执行计算】后系统自动计算出工单材料未领用完的数量填充至“本次领用数量”中，并计算出工单材料未移仓的数量填充至“本次移仓数量”。
- **配套领料：**当选择“配套领料”，系统会自动计算出工单待领料套数填充至“本次领料套数”，“本次领料套数”支持手工修改，点击【执行计算】后，根据所填套数，配套计算“本次领料数量”及“本次移仓数量”。
- **按库存数量：**当选择“按库存数量”，点击【执行计算】后，系统对比库存数量及未领数量，取两者的较小值为“本次领料数量”；对比库存数量与未移仓数量，取两者的较小值为“本次移仓数量”。
- **补足已领套数：**当选择“补足已领套数”，系统自动计算所查询工单已领料的最大可生产套数，并将最大可生产套数填充至“补齐套数”，点击【执行计算】后，系统将根据补齐套数计算所需要的配套物料数量，减去已领料数量或已移仓数量，填充为“本次可生产数量”及“本次移仓数量”。
- **配套退料**（配套退料功能于2026年2月27日发布灰度环境）**：**当选择“配套退料”，系统自动计算所查询工单的可退料套数，并将可退料套数填充至“本次退料套数”，点击【执行计算】后，根据本次退料套数自动计算可退料数量，并将可退料数量填充至本次退料数量。

注：如计算出的本次领料数量或本次移仓数量为负数或为0，则显示为空。

![上传图片](https://vip.kingdee.com/download/010043deb0ab79954cbc85cabf7a1cf88463.png)

**②支持三种领料方式**

- **合并领料：**所选工单车间/供应商一致，则生成一张生产领料单/委外发料单。
- **按单领料：**每张工单生成一张生产领料单/委外发料单。
- **分仓领料：**按照所选工单材料分录维护的仓库，根据“仓库+生产车间/供应商”生成对应的生产领料单/委外发料单，如仓库相同，不考虑仓位是否相同，统一按同一仓库处理，仓库为空，单据生成一张单。
- 注意：当领料策略选择“配套退料”时，仅支持合并生成生产退料单。

![上传图片](https://vip.kingdee.com/download/0100534331eef7e24b1f9f6936d44bf24a32.png)

**③支持勾选参数【仅显示缺料的材料行】、【移仓非倒冲材料】**

- 仅显示缺料的材料行：勾选后，则计算后材料信息面板不显示未执行领料数量小于等于0的材料分录。
- 移仓非倒冲材料：勾选后则发料方式为“直接领料”和“倒冲领料”的材料都可生成移仓单。注意：当领料策略选择“配套退料”时，该参数不生效。

![上传图片](https://vip.kingdee.com/download/01002917754ba6f04d0280d46d11bef29f6a.png)

3、选择需要发料的工单，点击【执行计算】，进入到生单界面。该界面显示所计算的工单信息及其材料明细信息，勾选工单商品分录，则自动勾选材料分录，可切换至材料汇总界面，查看材料汇总数据。

- 提供参数“可领料数量为0的分录不显示”、“可移仓数量为0的分录不显示”，可过滤掉无需领料或移仓的材料分录
- 点击生成领料单/发料单，则将所选发料方式为“直接领料”的材料分录生成领料单/发料单，“本次领料数量”将携带至领料单/发料单数量。
- 点击生成移仓单，则将所选材料分录生成移仓单，“本次移仓数量”携带至移仓单数量。
- 任一材料分录生成领料单或移仓后，其分录状态即更新为已执行，不可重复生成。
- 点击引出，即可引出所选择的信息数据。

![上传图片](https://vip.kingdee.com/download/0100b3f654115b464557a87a0d2b8440533f.png)

- 当发料策略选择“配套退料”时，则可点击生成生产/委外退料单，所生成的生产/委外退料单自动关联工单对应的领料单。

![上传图片](https://vip.kingdee.com/download/01009d8c25aed0174cedb6cd5b9873565bd5.png)

**4、现有功能改造（星辰旗舰版/专业版通用）**

生产任务单/委外加工单的【复制为移仓单】按钮删除，新增【移仓】按钮，点击【移仓】按钮，则可实现生产任务单/委外加工单下推到移仓单，并支持移仓数量反写至生产任务单/委外加工单。

新增参数【生成移仓单时，所携带的材料分录发料方式为】，默认勾选“倒冲领料”，如需移仓发料方式为“直接领料”的材料，则可勾选“直接领料”。
