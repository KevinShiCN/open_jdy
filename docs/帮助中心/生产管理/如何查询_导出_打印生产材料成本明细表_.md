# 如何查询/导出/打印生产材料成本明细表?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：Simone邓 | 阅读：19 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/782411948041911296)

【问题描述】

如何查询/导出/打印生产材料成本明细表?

【业务场景】

生产材料成本明细表可以根据当前的过滤条件,查询生产任务单的单个产品在该期的材料明细情况,详细展示材料的期初在制、本期投入、本期完工、期末在制的数量及成本。

**授权说明:**该报表受数据权限控制,即若用户无某商品、部门的数据权限,则对应的产品入库和退库数据在该报表中无法查询。

**取值说明:**

**1)数据类型**

**期初在制:**取 上一期的“期末在制”的数量及成本。

**本期领料:**取 生产领料单的数量及成本。

**本期退料:**取 生产退料单的数量及成本。

**前期入库本期退库材料:**取产品退库单的分配的材料数量和成本。

**本期投入:**取 (本期领料 - 本期退料 + 前期入库本期退库材料)的数量及成本 。

**本期完工:**取生产成本核算后,该产品(含联副产品)耗用的材料数量及成本。

**期末在制:**取 (期初在制+本期投入-本期完工)的数量及成本。

**其他说明**

**材料数量**,是取材料基本单位的数量。

**本期完工取数**,是在该期间生产成本核算后才能取到数值;故只能查询截止到业务当期的数据,不支持查询以后期间的数据。

启用了联副产品的情况下,本期完工是指任务单的主产品和联副产品一起耗用的材料数量及成本。

材料分配过程会将材料按分配标准进行分配,分配过程不再按材料基本单位的精度处理,因此可能会出现该报表材料数量精度大于基本单位精度的情况。

【**操作步骤**】

**1、查询**

**【生产管理】-【生产成本核算】-【生产材料明细表】**

![](https://vip.kingdee.com/download/010953d46cf0555d426789f3d69771bac38d.png)
为保证材料成本准确性,请先进行生产成本核算。[生产成本核算](https://vip.kingdee.com/link/s/ZhouD)

操作生产成本核算后在生产材料成本明细表按需求选择**期间、生产车间、商品、生产任务单号、材料**过滤条件进行查询

![](https://vip.kingdee.com/download/0109fc49599e4ef64025a69e87faa44868fb.png)默认会**显示小计**,可在显示勾选不显示小计

![](https://vip.kingdee.com/download/0109b8e26fa7b80743b1b15125d6b9398444.png)
右上角**列设置**可开启或者隐藏相关字段

![](https://vip.kingdee.com/download/0109c65869cc7a4241a8815650a0b7e1db3c.png)
右上角的**组合排序**可设置多个字段同时进行排序

![](https://vip.kingdee.com/download/010965d07f381a454ddfa99cf45484048672.png)
**2、引出:**查询结果可点击右上角的**引出**按钮导出到本地电脑

![](https://vip.kingdee.com/download/010941274caf04d54227845e6d1d245919ef.png)
**3、打印:**右上角打印按钮下拉进行打印设置之后可以对查询结果进行打印

![](https://vip.kingdee.com/download/01092eff3c7371cb42d292dc3df643b7d056.png)
