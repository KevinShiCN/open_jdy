# 一张“卡壳”的订单：BOM备注的跨单之旅

![上传图片](https://vip.kingdee.com/download/0100f7fcd78728cb4a4a878b65a105580e7e.png)

       生产部的老周盯着电脑屏幕上的生产任务单，眉头拧成了疙瘩。这批定制机柜订单明明要求所有螺丝用防腐蚀型号，可任务单的材料清单里偏偏没写，害得车间刚装好的5台机柜全得拆了重装。“又出这种事！”老周攥着订单快步跑到计划部，找到负责排产的小林。小林点开上游的计划订单一看，BOM备注栏里明明白白写着“螺丝需为304不锈钢防腐蚀款”，可下游自动生成的生产任务单里，这行关键备注却凭空消失了。

       眼看车间还等着新任务单开工，老周正着急，计划部的小李凑了过来：“这问题我之前处理过，录单时稍作设置就能让BOM备注跟着订单走。”到底该怎么操作？我们跟着小李的步骤一起看看。

首先点击【生产管理】-【物料需求建议】，再点左下角的【选单】，选择对应的销售订单，最后点击【执行计算】计算成品和材料需求。

![上传图片](https://vip.kingdee.com/download/01007e973835bc3f4d549f9539987a834476.png)

![上传图片](https://vip.kingdee.com/download/0100978104d67f56424b9ca507af6614d5e4.png)

![上传图片](https://vip.kingdee.com/download/0100d8a0cf93c7f04b76b358e152d3ca848d.png)

计算到需求后，点击生成【按明细生成计划订单】或【按汇总生成计划订单】，生成计划订单后，点击订单的编码，进入订单内部。

![上传图片](https://vip.kingdee.com/download/01001c926bcbbf84419b97a1a92281103fc1.png)

![上传图片](https://vip.kingdee.com/download/01002b78a6907bf645009576ac4a98ceef9d.png)

![上传图片](https://vip.kingdee.com/download/0100eb558fa252d74640870bc7437eca6521.png)

计在划订单中依次点击【更多】-【选项】，勾选参数【BOM物料备注携带至下游单据的商品行备注】下对应的生产任务单；

![上传图片](https://vip.kingdee.com/download/010005a3062f0c9145928296e8b81d23693b.png)

![上传图片](https://vip.kingdee.com/download/010034ab3be7575a4e22a6e39d13f6adbcc7.png)

这样，计划订单在审核并投放到生产任务单后，BOM单里的备注就可以被携带到生产任务单中。

![上传图片](https://vip.kingdee.com/download/01003754808bdf1d40c7b07d0f3af9501447.png)

![上传图片](https://vip.kingdee.com/download/01002275e87ca6464d9ea5670f5ac1c78158.png)

注意：

参数【源单的商品行备注携带至下游单据的商品行备注】和【BOM物料备注携带至下游单据的商品行备注】互斥：若在【源单的商品行备注携带至下游单据的商品行备注】下已勾选某单据类型，则不能同时在【BOM物料备注携带至下游单据的商品行备注】中勾选该类型。

      老周盯着屏幕上清晰显示的备注，忍不住感叹：“原来还有这操作！这下备注再也不会凭空清空，关键信息总算能稳稳带过去了。”
