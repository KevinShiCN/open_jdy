# 质检单上的“结果判定”字段有什么作用?

> 作者：移动生产白建凡 | 阅读：43 | 有用：0

【问题描述】

质检单上的[结果判定]字段有什么用?

【解释说明】

结果判定合格或不合格,没有标准定义,根据自己公司标准整体算合格就可以标记为合格。这是一个整体是否合格的记录,不影响数据
