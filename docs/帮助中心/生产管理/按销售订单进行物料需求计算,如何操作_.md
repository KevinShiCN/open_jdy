# 按销售订单进行物料需求计算,如何操作?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题

【问题描述】

按销售订单进行物料需求计算,如何操作?

![](https://vip.kingdee.com/download/0109175cefcc42344869a9935900c4c2fe83.png)[视频观看](https://vip.kingdee.com/school/detail/753269292036137472?productLineId=35&lang=zh-CN&cosmic_type=video)

【操作步骤】

1、依次点击【生产管理】-【物料需求建议】

![](https://vip.kingdee.com/download/01094e11886aa97a44e98975364058643a6a.png)

2、设置计算范围及公式,然后点击【选单】按钮,选择需要计算的销售订单。

①计算范围:选择参与计算的仓库、选择不参与计算的仓库

②净需求数量=需求量-现有库存(即时库存、预计可用库存、销售占用量),其中需求量可考虑损耗;

③计划订单量=净需求/成品率;

④考虑安全库存

注意:预计可用库存的公式设置(【系统设置】-【系统参数-进销存参数-仓存参数-预计可用库存参数】)。

![](https://vip.kingdee.com/download/0109cf52423617ad4c44bc0a41c61196c262.png)3、选择需要执行需求的销售订单。

![](https://vip.kingdee.com/download/0109ccc74d002fbd45129bc61b29341da297.png)

①销售订单需满足:已审核未行关闭,且可生产可委外可采购的数量大于零。

②BOM编号:销售订单的商品如果维护了BOM,则系统自动获取已审核未禁用的最高版本BOM编号参与计算;

③ 数量:销售订单的数量扣减已经关联采购申请单/采购订单/委外加工单/生产任务单的数量,如有已出库数量,会继续扣减。即取的是可生产可委外可采购的数量。

④需求日期:默认取销售订单的预计交货日期,若预计交货日期早于当前系统日期则取系统日期。

⑤**参与过计算的销售订单,在未关闭销售订单之前,可以再次选择进行计算,计算完成后更新最后计算时间。(可能**一个销售订单会重复计算,如果不希望选到已经计算过的单据,建议在选单界面过滤“计算标记为空”的单据。)

![](https://vip.kingdee.com/download/01097c07f3fbd65e484bafd77a1eff36f817.png)

4、点击执行计算即可。

![](https://vip.kingdee.com/download/0109ce67ed06472f47a387b23baf015de10d.png)注意:

①没有BOM不影响计算的进行,影响的是无法展开BOM计算子项物料的需求。如不需展开子件,可继续计算。

**②系统不支持设置销售订单审批后自动生成计划订单,需通过物料需求建议执行计算后生成。**
