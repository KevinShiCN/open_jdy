# 现有的bom要新增物料,如何处理?

【问题描述】

现有的bom要新增物料,如何处理?

【业务场景】

由于产品设计变更、优化生产流程或引入新材料的需求,需要在现有的生产中使用最新的物料,可以通过【BOM批量修改】功能在已有的BOM基础上新增物料信息。

【解释说明】
依次点击【生产管理】-【BOM批量修改】,进入批量修改界面![](https://vip.kingdee.com/download/010970af5055a315496e872c91b4870ff037.png)
上方选择修改类型为:【新增子件】![](https://vip.kingdee.com/download/010900b7b5b1109240ada02590b9388b5ec9.png)

并选择需要添加的物料信息,点击下一步选择BOM

![](https://vip.kingdee.com/download/0109a84c960c9d994beca7555a8fda7382d4.png)

选择好适用的BOM后,下一步确认执行即可
![](https://vip.kingdee.com/download/0109a81a8329e114485099644dabbfab8cfe.png)
