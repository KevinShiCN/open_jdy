# 怎么引入bom表?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：shaoling | 阅读：1747 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/557730131339609344)

【问题描述】

怎么引入BOM表?

【业务场景】

需要录入的bom单信息比较多,想要批量导入系统中,如何操作?

**tips:引入的bom相关商品信息需要先在系统中录入好,不支持引入时,同时新增商品信息:[如何新增商品资料?](https://vip.kingdee.com/link/s/ZlgZ7)**

【解决方案】

1、下载引入模板填写资料引入

点击【生产管理】>【BOM维护】进入【BOM维护列表】页面

![](https://vip.kingdee.com/download/0109a39243aaec164774a0e5edb4a47a7247.png)

在【BOM维护列表】右上角【引入】按钮点击操作,即可到引入页面下载默认模板

![](https://vip.kingdee.com/download/0109aa77171b734f4e5bb7d6fb9f76e8fab8.png)

也可以根据需要在模板管理自定义模板,再点击下载

![](https://vip.kingdee.com/download/0109160b5d97c47f48428e9ad7efa8d4adc7.png)

下载模板后录入好BOM表数据内容,其中表格里面红色表头都是录填项,必须要录入内容;

有多个材料的BOM表头成品信息只需要填写第一行,后面的材料信息按顺序填写

![](https://vip.kingdee.com/download/0109b0f3297794e243b99e485fd39f37f4c3.png)
模版填写完成回到系统上传引入文件

![](https://vip.kingdee.com/download/0109a46770ac63aa4f89bb8bdfeaf48c98ca.png)

上传好引入的文件,点击【开始引入】即可引入BOM表,引入的数据是根据模板填写的数据更新的

![](https://vip.kingdee.com/download/010944820ffb08b4458c9546127a018186b8.png)

引入之后是未审核状态,需要审核才生效

![](https://vip.kingdee.com/download/01098ff90d0103064a1786e9b6e6ef97a448.png)

引入后系统后在星辰引入进度里可以查询引入结果,具体引入情况也可点击【查看详情】里进入查看

![](https://vip.kingdee.com/download/010947c307be1f24403c8826227591b1fac6.png)
2、可在系统手工新增一个BOM,然后按引入模板格式引出,再参考录入其他BOM资料后覆盖新增引入

按引入模板引出

![](https://vip.kingdee.com/download/010955b7ad7250cb45b3bd8e83fc7b8b03e1.png)
填写好更多BOM资料后覆盖新增引入:

![](https://vip.kingdee.com/download/0109e5ce7c67acd74c958e2a8b4e8dbd09c8.png)

**多级BOM引入方法:正常在引入模板里面分别填好成品和半成品的BOM信息,导入后会自动产生级次关系**

多级BOM模板上录入示例:
