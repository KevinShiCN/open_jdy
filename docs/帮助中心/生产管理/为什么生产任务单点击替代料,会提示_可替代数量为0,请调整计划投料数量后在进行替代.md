# 为什么生产任务单点击替代料,会提示:可替代数量为0,请调整计划投料数量后在进行替代

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：sunwendy | 阅读：430 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/513519201370281472)

【业务场景】

为什么生产任务单点击替代料,会提示:可替代数量为0,请调整计划投料数量后在进行替代

![](https://vip.kingdee.com/download/01097d771eb0d41b451eaecfbca9208ddcd6.png)

【解决方案】

如果没有维护产品的数量,则会有这个提示,需要维护产品数量后,再点击替代料。

![](https://vip.kingdee.com/download/010985e3c154438b459b94e80231a589ce13.png)
