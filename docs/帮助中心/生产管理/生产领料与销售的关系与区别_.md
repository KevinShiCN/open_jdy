# 生产领料与销售的关系与区别?

【问题描述】

生产领料与销售的关系与区别?

【业务场景】

生产领料和销售订单之间的单据有什么差异呢,如何可以将销售订单和生产领料单进行关联?

【解释说明】

生产领料单主要是在生产加工时,原材料领料出库时用到的。

销售模块的单据主要是用来签订合同后,形成销售订单,届时将商品进行销售出库,再进行收款时用到的。

**两者之间的联系:销售订单可以通过物料需求建议生成生产任务单,或者新增生产任务单选择源单销售订单,将生产任务单下推生成生产领料单和产品入库单。生产领料单不支持直接关联销售订单,只能通过生产任务单产生联系。**
