损耗率与成品率区别?

【解释说明】

**损耗率**:生产过程中物料损耗量占总投入量的比例,侧重成本控制(如原料浪费分析)

公式:损耗率 = (消耗量/标准用量 - 1)×100%

![](https://vip.kingdee.com/download/01092b0d6b91ae3d477596c7d6de5e8097cb.png)

或损耗率 = (1 - 标准用量/消耗量)×100%
![](https://vip.kingdee.com/download/0109441e4729ee13479fb096b1315ab9ae09.png)
若您想要了解BOM损耗率使用问题,请点击:[bom单损耗率如何使用?](https://vip.kingdee.com/knowledge/458798221465223680?channel_level=%E9%87%91%E8%9D%B6%E4%BA%91%E7%A4%BE%E5%8C%BA%7C%E6%90%9C%E7%B4%A2%7C%E5%AE%98%E6%96%B9%E7%9F%A5%E8%AF%86&productLineId=35&isKnowledge=2&lang=zh-CN)

**成品率**:净需求量占订单计划生产数量的比例,侧重生产效率评估(如产能利用率)

公式:成品率 = 净需求 / 计划订单量×100%
