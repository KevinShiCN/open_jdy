# 物料需求建议选单界面，如何排除已经计算过并生成计划订单的单据？

## 1 问题描述

物料需求建议选单界面，如何排除已经计算过并生成计划订单的单据？

## 2 适用版本
金蝶云·星辰v3.1专业版/旗舰版

## 3 解决方案

### 3.1 设置选单参数

操作：物料需求建议设置界面，【选单】按钮旁边增加【选单设置】按钮，勾选【仅显示未参与计算的单据】后，在选单界面系统会自动过滤掉已参与计算的单据，仅显示未完全参与计算的单据。

优点：只要单据已经完全参与过计算并生成计划订单，选单页面就会自动过滤掉这部分单据，不会出现重复生产/采购的情况

![](https://vip.kingdee.com/download/0109b7a469a47b554e7d9e5a577e5cf916da.png)

###

### 3.2 设置过滤方案

操作：选单界面，通过“计算标记”字段进行过滤。点击查看[具体操作](https://vip.kingdee.com/link/s/lntfg)

优点：只要计算过并生成计划订单，单据都会打上【Y】的【计算标记】，设置方案后下次会自动过滤掉这部分单据

缺点：可能导致多生产少采购

| 订单是否全部参与计算 | 主产品工单的计划订单是否投放 | 下次是否还能选到 | 结果 |
| --- | --- | --- | --- |
| 是 | 是 | 否 | 不会多生产也不会多采购 |
| 否 | 少生产少采购 |  |  |
| 是 | 否 | 不会多生产也不会多采购 |  |
| 否 | 少生产少采购 |  |  |

| --- | --- | --- |
| 是 | 是 | 否 |
| 否 | 是 |  |
| 是 | 否 | 是 |
| 否 |  |  |

（2）勾选该参数

不管订单是否全部参与计算，不管主产品工单的计划订单是否投放，只看销售订单上【计划数量】和【数量】字段的大小关系

| 订单是否全部参与计算 | 主产品工单的计划订单是否投放 | 【计划数量】≥【数量】 | 下次是否还能选到 |
| --- | --- | --- | --- |
| 是 | 已投放 | 是 | 否 |
| 是 | 未投放 |  |  |

**（2）只要勾选了这个参数，**下次就选不到
