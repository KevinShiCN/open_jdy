# bom子件逐级反查是如何使用的?

【问题描述】

BOM子件逐级反查是如何使用的?

【解释说明】

BOM子件反查会从下往上扫描所有级次的产品结构,列出配件的每一个父件以及父件的父件,直至最终成品,

逐级反查是指录入材料查询以后,默认层级不展开。需要手动一级级展开查看对应的父件。
