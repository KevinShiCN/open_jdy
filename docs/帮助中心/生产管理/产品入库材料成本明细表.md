# 产品入库材料成本明细表

# 1  简介

## 1.1  功能介绍

产品入库材料成本明细表是根据当前的过滤条件，展示筛选出的产品入库单/产品退库单中各项产品耗用材料的数量及成本。

## 1.2  适用版本

金蝶云·星辰专业版/旗舰版

## 1.3  应用场景

成本会计可通过产品入库材料成本明细表，精准掌握不同生产任务单、产品、会计期间等的材料耗用数量及成本明细的情况。

## 1.4 系统路径

生产管理-生产成本核算-产品入库材料成本明细表

## 1.5 取数说明

材料数量及成本，是系统按材料分配标准，通过生产成本核算计算出来的；故只能查询已参与过生产成本核算的单据的材料数量及成本。

材料分配过程会将材料按分配标准进行分配，分配过程不再按材料基本单位的精度处理，因此可能会出现该报表材料数量精度大于基本单位精度的情况。

## 1.6 权限说明

该报表受数据权限控制，即若用户无某商品、部门的数据权限，则对应的产品入库和退库数据在该报表中无法查询。
