# 生产领料差异分析表里面的入库单位用量是怎么计算的?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题

【问题描述】

生产领料差异分析表里面的入库单位用量是怎么计算的?

![](https://vip.kingdee.com/download/010943e22071782141f99f6d77f8746a3c7c.png)

【解释说明】

生产领料差异分析表里面的入库单位用量=物料已领数量/产品入库数

比如商品入库数量2,对应材料已领料数量是6,那入库单位用量=6/2=3
