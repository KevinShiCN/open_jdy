# 成本计算表界面优化

## **1  简介**

### **1.1  功能介绍**

本次对生产成本计算表和委外成本计算表的界面进行了优化，进一步提升数据查看与操作的便捷性。

### **1.2  适用版本**

金蝶云·星辰专业版/旗舰版

### **1.3  应用场景**

生产管理员/成本会计可查看成本计算表时，可折叠无关板块、集中查看核心成本项，提升成本数据查阅与分析效率。

### **1.4   系统路径**

生产管理-生产成本核算-生产成本核算结果查询-成本计算报告-成本计算表

委外管理-委外成本核算-委外成本核算结果查询-成本计算报告-成本计算表

### **1.5  优化功能点**

本次界面优化更新的主要内容包括：

1、统一报表名称，保持委外与生产模块成本计算表命名规则一致。

2、表头增加显示计算期间，可快速通过表头识别数据所属期间。

3、界面板块支持折叠/展开，可灵活聚焦关键数据区域，提升成本数据查阅与分析效率。

4、调整超长商品名称显示方式，避免因过长影响表格整体浏览。

5、优化列宽、列名与对齐方式，表格更整洁易读。

6、关键列名增加提示说明（鼠标移动至字段上，会显示tips），降低理解难度。

7、将材料成本与单位材料成本板块相邻放置，便于单位成本与总成本关联分析。
