# 提示:本期完工产品材料成本均为0,无法按完工入库材料成本分配,请尝试生产成本核算或进行手动分配!如何处理?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：_Leo | 阅读：1535 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/518592750891976704)

【问题描述】

生产成本核算异常,提示:“本期完工产品材料成本均为0,无法按完工入库材料成本分配,请尝试生产成本核算或进行手动分配!”

或是“商品成本单价为0,该产品本期未进行材料成本分配”

![](https://vip.kingdee.com/download/010930ba6b150a9246038cdafb0bcea28a0d.png)

【解释说明】

若生产成本核算提示没有成本,一般是由于未做领料,或材料没有成本所导致。

【解决方案】

首先,建议检查是否做了领料。若已领料成本还是为零,可追溯检查材料的出库是否有成本,可到【库存管理】-【出库成本核算】界面单独对材料进行成本核算,检查材料是否有入库成本,以此思路进行排查处理,只有当材料入库有成本,出库带出成本,并带入到生产入库单的成本中。如图:

①可通过本期核算的生产任务单通过【全流程查询】查询关联的领料单商品成本

![](https://vip.kingdee.com/download/0109ec5c763814c8411abd1c7f2a3a7ba493.png)

②查看材料的出库成本计算表,具体是什么原因没有出库成本;另外查询时注意思所在期间,是否存在跨期的情况。

![](https://vip.kingdee.com/download/010989322a9f76ed468eb41986fb7ccc7977.png)

③确定材料成本没问题后, 再重新回到【生产管理】进行【生产成本核算】。
