# 计划订单投放和合并投放的区别?

【问题描述】

计划订单投放和合并投放的区别?

【业务场景】

生产模块下,计划订单中显示有投放和合并投放两个功能键,这两个功能键之间的主要差异是什么呢?

若希望了解计划订单如何合并投放,请点击:[计划订单如何合并投放?](https://vip.kingdee.com/knowledge/507358269560481792?productLineId=35&isKnowledge=2&lang=zh-CN)

若希望了解已投放的计划订单如何取消,请点击:[已投放的计划订单如何取消?](https://vip.kingdee.com/knowledge/642896109144286720?productLineId=35&isKnowledge=2&lang=zh-CN)

【解释说明】

**“投放”**适用于单项或者批量投放相同的目标单据类型,暂不支持投放不同类型的目标类型单据。

**“合并投放”**适用于单项或者批量投放相同或者不同目标单据类型的单据,以及可以结合不同的目标单据类型设置不同的合并投放规则,下图是计划订单合并投放的规则设置界面。

![](https://vip.kingdee.com/download/010950763ea0035a4c0c995bbad699b19f31.png)
**合并投放相较于投放,更加灵活多样,一般实际业务场景中,建议使用合并投放比较方便。**
