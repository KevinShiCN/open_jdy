# bom成本查询支持什么展开方式?

【问题描述】

BOM成本查询支持什么展开方式?

【操作步骤】

**BOM成本查询支持两种展开方式:逐级展开与多级展开**

1、逐级展开:默认只展示父件,可点击前面三角形展开下级,效果如下图所示![](https://vip.kingdee.com/download/010914eff452a3d34998988ca5a8b77df67d.png)

2、多级展开:默认展示全部级次的信息,效果如下图所示![](https://vip.kingdee.com/download/010916772b590710441b8b76638898337559.png)
温馨提示:可根据个人习惯和业务需要选择不同的展开方式。
