# 替代方案如何应用?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：shaoling | 阅读：2051 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/713921314863359488)

【问题描述】

替代方案如何应用?

【适用版本】

金蝶云星辰专业版、旗舰版

【业务场景】

在电子元器件、五金等制造行业中,由于订单取消、工程变更、客户需求变化、预测或计划不准确等原因会造成原材料库存积压、呆滞,使企业在仓库积压大量资金。企业出于降低物料成本、或及时响应客户需求等因素考虑,都会存在使用替代物料的情况;针对不同企业、行业、产品的业务需求,可以设定相应的替代策略、采购策略、替代优先级等基础数据,在物料需求建议计算时会根据相应的设定来匹配主物料、替代料的需求供应。

【解释说明】

替代方案可在系统的生产管理或委外管理模块中使用,不管从生产管理模块还是委外管理模块进入替代方案列表,均实际上进入同一个列表,仅进入路径不同。

【操作流程】

**1、设置替代方案**

从【生产管理】>【替代方案】进入替代方案列表页面

![](https://vip.kingdee.com/download/0109f446f017033949c7835e1158ae0bcaa8.png)
或者从【委外管理】>【替代方案】进入替代方案列表页面

![](https://vip.kingdee.com/download/010979b45151c0c24437b20a20ab8f123e3a.png)
点击右上角<新增>按钮
![](https://vip.kingdee.com/download/0109f18ba35dc2454d0c9f46368b17b17e7b.png)
根据需要设置对应的替代方案
![](https://vip.kingdee.com/download/0109195da797d6034690a9d959dbae2283b1.png)

设置替代方案需关注:

(1)**唯一性**: 同一个物料可建立多个替代方案,但只有一个替代方案可被启用,当启用新的替代方案后,之前的替代方案会自动被禁用;

![](https://vip.kingdee.com/download/010901a876e4155347cc8f73f535fab94d32.png)

(2)**生效节点**:替代方案启用后方可生效;

![](https://vip.kingdee.com/download/01093dc96d24053048c5a72e2f518b06aeec.png)

(3)**替代策略**:

①混用替代: 当主料数量不足时,可用不同的替代料一起替代(例如:用B+C替代A);

②整批替代: 当主料数量不足时,只能用同一个替代料进行替代(例如:用B替代A,或者C替代A)。

![](https://vip.kingdee.com/download/01094829fe068c7f43f284419c6d103b802c.png)

(4)**采购策略**:

①采购主料: 物料需求建议计算时若主料和替代的需求均无法满足需求时生成主料的采购需求;

②采购替代料: 物料需求建议计算时若主料和替代的需求均无法满足需求时生成替代中优先级为1的物料的的采购需求。

![](https://vip.kingdee.com/download/0109552c2197cfc64220bf378e16ff433c6d.png)

(5)**适用范围**:

①全部BOM: 所有包含该主料的BOM都会适用该替代方案;

②指定BOM: 指定的BOM才对该替代方案生效。

![](https://vip.kingdee.com/download/0109bb21ca15f7e44cd588af13d730ae8a8e.png)

(6)**优先级**: 替代方案优先级不可编辑,默认取物料的排序顺序,可通过上移、下移和移到按钮调整;

![](https://vip.kingdee.com/download/01091c46b1d443f64d6faca46c42adae7ef6.png)

(7)**BOM**:若BOM中物料有对应启用状态的替代方案,则BOM中物料行显示替代方案编码;

![](https://vip.kingdee.com/download/010940e207835cb8457d867918a829172884.png)
(8)**建立条件**:替代方案仅允许最末级物料建立,半成品或成品(即有下级子件的物料)不支持建立替代方案。

![](https://vip.kingdee.com/download/0109f143b29f40654561a082c1739a813e98.png)
**2、物料需求建议里使用替代方案**

从【生产管理】>【物料需求建议】进入物料需求建议页面

![](https://vip.kingdee.com/download/0109a28a055384804efb9f921da9c17ff5c8.png)

在物料需求建议页面,净需求量计算公式里勾选“考虑替代”,选择对应的单据后,点击<执行计算>按钮
![](https://vip.kingdee.com/download/010980274789f43b4faba3a994b2c2189a26.png)若计算过程产生了替代,则替代料行显示在主料下方
![](https://vip.kingdee.com/download/010990d254963d0345d988a367a09749ec2c.png)
**3、生成计划订单和物料替代清单**

物料需求建议计算完成后点击生成计划订单

![](https://vip.kingdee.com/download/01093b793ececf134329886ababe35621227.png)

系统同时将计算结果中的替代关系生成物料替代清单,并将物料替代清单的编号写入关联的计划订单中****

![](https://vip.kingdee.com/download/01094138f4d5bae34109b22d2d4a379c8b06.png)
生成计划单和物料替代清单需关注:
(1)物料替代清单不支持新增、修改和删除,计划订单删除时关联的物料替代清单会自动删除

(2)在计划订单列表可联查物料替代清单

![](https://vip.kingdee.com/download/01098bb193b1d71449d491e8fb2a62705626.png)
**4、计划订单投放生产生产任务单/委外加工单**

(1)投放时会将物料替代清单中记录的替代料和投料数携带至生产任务单/委外加工单上

![](https://vip.kingdee.com/download/0109322d357639fd4377b410b8068f20e43f.png)

(2)可手动调整替代料及主料最终的计划投料数量

![](https://vip.kingdee.com/download/01092ac8ba447f0d4581ac8a832167edcdb5.png)

(3)主料删除时对应的替代料也会一并删除

![](https://vip.kingdee.com/download/010941f32f45af9b408a9933340ac9eae812.png)
