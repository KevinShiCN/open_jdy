# 费用归集引入内部数据时,支持从哪些地方引入?

【问题描述】
费用归集引入内部数据时,支持从哪些地方引入?
【解释说明】

费用归集是将当期发生的所有用于生产相关的费用归集起来,然后分配到生产任务单上增加商品的成本

归集费用不会对账业务和总账的数据产生影响,待归集的费用分配进商品成本了才会影响产品入库的库存成本,待产品入库之后再入总账。

支持三种数据来源:总账、生产领料单、生产退料单

![](https://vip.kingdee.com/download/0109e1bfeaa06f4b41f1ad299234b964bd66.png)

其中

1、引入内部数据-生产领料单

金额来源:本期共耗料的生产领料单出库成本、基本单位成本

支持设置增加/减少费用,可自行根据需求选择需要引入的费用类型;

2、引入内部数据-生产退料单

金额来源:本期共耗料的生产退料单出库成本、基本单位成本

3、引入内部数据-总账凭证

金额来源:所选科目的本期借/贷方发生额
