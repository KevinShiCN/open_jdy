# 材料单位是2位小数，为什么计算出的应发数量、损耗数量、标准用量是10位小数？

# **1 问题描述**

材料单位是2位小数，为什么计算出的“应发数量”、“损耗数量”、“标准用量”是10位小数？

![上传图片](https://vip.kingdee.com/download/0100bfcf3044e4af4a3aba760f355c1c69eb.png)

# 2 适用版本

星辰3.0以上专业版/旗舰版

# **3 原因解析**

“应领数量”、“损耗数量”、“标准用量”是计算过程数据，为确保结果精准，系统按最高精度10位小数存储。这些字段仅用于计算“计划投料数”，后者才是实际发料的依据，建议关注“计划投料数”

# **4 计算逻辑**

- 消耗量=标准用量/（1-损耗率），或标准用量*（1+损耗率）。取决于系统参数<生产参数<消耗量计算公式设置。如图示，本例设置为：标准用量/（1-损耗率）

![上传图片](https://vip.kingdee.com/download/010025daf79e675e431883911017a5b2372c.png)

- 标准用量=材料用量/产品产量*数量=334/600*25=13.9166666667
- 消耗量=标准用量/（1-损耗率）=13.9166666667/(1-2%)=14.2006802721
- 应领数量=消耗量+固定损耗=14.2006802721+0.01=14.2106802721

- 损耗数量=消耗量-标准用量=14.2006802721-13.9166666667=0.2840136054
