# 如何删除生产领料单?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：云社区用户6w624200 | 阅读：492 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/684572611560407808)

【问题描述】

如何删除生产领料单?

【解释说明】

1.生产任务单下推生产生产领料单没有入库的情况可以直接删除 。

2.当生产领料单有关联的生产任务单已经完工,对应的生产领料单是否可以删除,取决于【系统参数】-【生产参数】里的【若您希望控制产品入库前必须先领料,请选择控制强度及控制项-控制强度】,如果设置的【不控制】、或者是【提示,允许继续入库】,是可以直接在生产领料单列表里直接删除的,如果设置的是【提示,不允许入库】,则会提示不能删除。

![](https://vip.kingdee.com/download/01096a6f6db10e9b4bca9fb9db57b454e976.png)

举例说明:

1)如果参数设置的是【不控制】和【提示,允许继续入库】,则可以在生产领料单列表直接删除生产领料单

![](https://vip.kingdee.com/download/0109624f713bf51445c699990dd9b968f1f2.png)

生产领料单列表找到生产领料反审核删除

![](https://vip.kingdee.com/download/010902fdaa8114004cbcb0265c8cc1b89922.png)

![](https://vip.kingdee.com/download/01092d56170c4b7545cdbf6e841c6e801683.png)
2)如果勾选的参数是【提示,不允许入库】
比如这笔单子已经全部完工,

![](https://vip.kingdee.com/download/0109b84c15d47bc840128de5321d67b01652.png)

要删除对应的生产领料单的时候会提示【第1行物料“材料AAAAA”。对应产品已入库,请先退货或调整本单数量。】

![](https://vip.kingdee.com/download/0109c24d0aad3b804301b659c6cbb262b367.png)

此时需要反审核产品入库单删除之后才能删除生产领料单,

或者是在系统参数里把这个参数改成前两个不控制

![](https://vip.kingdee.com/download/01098699a5bde6f841b58e258e7bd6476081.png)

3.当生成任务单下推生成领料单后部分入库的情况,生成领料单删除也是同样取决于参数设置,如果设置【不控制】、或者是【提示,允许继续入库】,可以直接在生产领料单列表里删除;如果设置的是【提示,不允许入库】,则会提示不能删除。
