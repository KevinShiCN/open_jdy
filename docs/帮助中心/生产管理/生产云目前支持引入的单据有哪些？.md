# 生产云目前支持引入的单据有哪些？

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题, 快速入门, 精华汇总
> 作者：Esperanza | 阅读：669 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/588316563820559360)

生产云相关单据目前支持引入的情况如下表所示：

| 模块 | 单据 | 是否支持引入 | 是否支持按源单引入 |
| --- | --- | --- | --- |
| 生产管理 | 生产任务单 | √ | - |
| 生产领料单 | √ | √ |  |
| 生产退料单 | √ | √ |  |
| 产品入库单 | √ | √ |  |
| 产品退库单 | √ | √ |  |
| 委外管理 | 委外加工单 | √ | - |
| 委外发料单 | √ | √ |  |
| 委外退料单 | √ | √ |  |
| 委外产品入库单 | √ | √ |  |
| 委外产品退货单 | √ | √ |  |
| 委外加工费用 | - | - |  |
| 委外加工费（退货） | - | - |  |
| 受托加工 | 受托加工订单 | √ | - |
| 客供材料入库单 | √ | - |  |
| 客供材料退货单 | √ | - |  |
| 受托加工出库单 | √ | - |  |
| 受托加工退货单 | √ | - |  |
| 受托加工发票 | - | - |  |

特别说明：

1）有的单据只限于特定的版本，具体支持哪些单据以对应版本说明为准，没有特别说明则适用于所有版本

2）目前表中列出的所有功能支持星辰3.1以上版本，如果有您没有需要的功能，建议联系售后客服升级到星辰3.1
