1 产品概述

### 1.1 产品介绍

生产管理，与销售、计划、采购、委外、库存、质检等业务密切衔接，以销售和预测需求为驱动，以生产任务单为核心，实现从物料需求建议，到生产领料/退料/补料，产品入库/退库以及生产成本核算的全流程管理。

### 1.2 适用版本

星辰3.0以上专业版/旗舰版

4  功能清单

（主要的功能清单，可链接到具体的用户手册内容）

| 生产管理 | 二级菜单 | 三级菜单 | 功能说明 |
| --- | --- | --- | --- |
| 生产数据 | BOM维护 | 维护BOM信息，即物料清单，也叫产品结构或配方 |  |
| BOM批量修改 | 可批量对BOM单中的子项物料进行修改、替换等操作 |  |  |
| BOM正向查询 | 查询产成品或半成品的BOM结构，了解产成品在生产时用到了哪些半成品，以及半成品用到了哪些原材料 |  |  |
| BOM子件反查 | 查询某个配件在哪些半成品或成品的BOM单中用到 |  |  |
| BOM成本查询 | 根据BOM组成情况，展示产品参考成本，为销售人员报价提供参考 |  |  |
| 替代方案 | 设定物料替代的替代策略、采购策略、替代优先级等基础数据 |  |  |
| 物料需求建议 | 物料需求建议 | 为用户提供按订单产生需求的处理方法 |  |
| 计划订单 | 根据物料需求建议的计算结果生成，并可转化成采购、生产、委外的业务单据 |  |  |
| 物料替代清单 | 用于记录物料需求建议计算过程中产生的替代情况 |  |  |
| 齐套分析 | 通过齐套分析可以判断材料的已有库存中能够安排生产的产品数量，如果不能安排的，可以了解缺料情况 |  |  |
| 生产单据 | 生产任务单 | 明确生产的产品及材料、数量、计划完工时间以及所需的原材料，向生产车间下达生产任务 |  |
| 生产领料单 | 记录车间领用的材料信息 |  |  |
| 生产退料单 | 车间人员在领料后发现物料质量不合格，需要退回给仓库 |  |  |
| 质检单（旗舰版） | 记录检验结果 |  |  |
| 产品入库单 | 记录产品入库的数量及仓库信息 |  |  |
| 产品退库单 | 记录退回的产品及数量、以及对应出库的仓库等信息 |  |  |
| 生产报表 | 生产领料明细表 | 根据当前的过滤条件，展示筛选出的生产领料及退料的情况 |  |
| 生产领料汇总表 | 仓管员可通过不同的汇总依据，多维度掌握生产领料汇总情况 |  |  |
| 生产领料差异分析表 | 统计任务单实际领料情况与计划领料的差异情况 |  |  |
| 产品入库明细表 | 统计产品入库及退库明细的情况 |  |  |
| 产品入库汇总表 | 仓管员可通过不同的汇总依据，多维度掌握产品入库汇总情况 |  |  |
| 生产任务单跟踪表 | PMC可通过该表随时了解生产任务的领料和入库情况，跟进生产进度 |  |  |
| 生产任务单统计表 | 用户可通过该表以销售订单或车间或产品维度统计生产总体情况 |  |  |
| 销售订单全程跟踪表 | 可了解销售订单目前生产加工的进度，包含原材料的采购进度，半成品的生产/委外进度，产成品的生产/委外进度 |  |  |
| 质量分析表（旗舰版） | 统计质检结果，有助于分析材料和成品合格率和不合格率 |  |  |
| 生产成本核算 | 费用类型 | 指企业生产过程中发生的费用的类型：包含人工费用、水电费等，用于归集费用并进行费用的分配 |  |
| 费用归集 | 用于归集当期发生的所有用于生产相关的费用 |  |  |
| 完工与在制品归集 | 用于归集本期完工产品数量和在制品数量，为之后的费用分配及生产成本核算提供依据 |  |  |
| 费用分配 | 将费用归集中归集到的各类费用按照分配标准分配给各个任务单 |  |  |
| 生产成本核算 | 核算产品入库成本 |  |  |
| 生产成本核算结果查询 | 可查询最近10次的生产成本核算结果 |  |  |
| 生产成本构成明细表 | 展示各产品入库单的成本构成情况 |  |  |

注：备注了（旗舰版）字样的为仅旗舰版支持的单据。
