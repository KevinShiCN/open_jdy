# 委外发料单选源单返回了很多应发数量是 0 的数据行,怎么快速删除?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：winnie_wen | 阅读：147 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391353647621770752)

【问题描述】

委外发料单选源单返回了很多“应发数量”是 0 的数据行,怎么快速删除?

【解决方案】

通过“应发数量”字段的“ 小漏斗”进行过滤,输入小于 1 的条件,过滤出
所有应发数量为 0 的数据行,再使用左上角的全选框,选中所有过滤出的数据行,点击 【批量删除】即可,然后清除掉应发数量中小漏斗的过滤条件,就可以继续对其他物料进行发料。
