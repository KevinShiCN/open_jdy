# 同一张计划订单如何多次投放?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：lykk | 阅读：525 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/503009543215919872)

【问题描述】

同一张计划订单如何多次投放?

【业务场景】

1、同一张计划订单需要分批多次投放。
2、因产能不足,导致同一个产品一部分需要自制另一部分需要委外

系统路径:生产管理-物料需求建议-计划订单

![](https://vip.kingdee.com/download/0109bcb389607d2840e3a97cc09a73ab64d0.png)【解决方案】

第一步:在计划订单列表界面,勾选需要拆分的单据,点击上方的【拆分】按(注意:未审核的计划订单才支持拆分)

![](https://vip.kingdee.com/download/010951bafc47c6404f81aaf38d207e2038f8.png)

第二步:选择拆分方式,点击拆分,即会根据设置对计划订单进行拆分。
自定义拆分:将选择的单据拆分出数量**到新单据,
拆分效果:拆为2张单据,新单据的数量是输入框中的数量;原单的单据数量是拆分前-新单的数量;
例如:原单数量是10,输入3,则拆出来的新单数量是3,原单数量变为7。
![](https://vip.kingdee.com/download/0109629202b07b794d239fff088bdede9869.png)

分摊拆分:将选择的单据拆分成**张单据,
拆分效果:可拆为多张单据,按输入的数量均分拆单。
例如:原单数量是10,输入5,则拆为5张单,每张单数量都是2;
原单数量是10,输入3,则拆为3张单,数量分别是3,3,4 ;
原单数量是10.00,输入4,则拆为4张单,数量均是2.50 ;
按单位精度均分数量,余数补到最后一张单据中。
![](https://vip.kingdee.com/download/0109f08a64cac5e34fc8aa31f86f1963a292.png)

单据编号:拆出的新单,单据编号取原单据编号加后缀 -01,-02,-03,…,例如JHDD-20230926-07155,拆分成3张单据后,单据编号分别是:JHDD-20230926-07155,JHDD-20230926-07155-01。
