# 订单跑物料需求建议,考虑预计可用库存,计算出产品的毛需求为0,但材料产生了需求?

> 作者：移动生产白建凡 | 阅读：154 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/754144915105379584)

【问题描述】

订单跑物料需求建议,考虑预计可用库存,计算出产品的毛需求为0,但材料产生了需求?如图示,产品B的净需求为0,产品B的下级材料B产生了需求

![](https://vip.kingdee.com/download/0109ae17dbbfb9d84b5b803eab60691787f9.png)
![](https://vip.kingdee.com/download/0109b363e037b29f41d3ba144d89cd320d96.png)

【问题原因】

因为产品下面材料的预计可用库存是负数,跑物料需求建议时考虑了预计可用库存,所以还会针对这部分负数,单独产生来源于预计可用库存的需求
