# 费用分配单上的“完工入库材料成本”是如何取值的？

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题, 精华汇总, 进阶技巧
> 作者：蒋章龙 | 阅读：1319 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/469803554329502976)

****

【字段释义】

费用分配单上的“完工入库材料成本”，是取的生产任务单所关联的当前业务期间的产品入库单上的直接材料成本。

【查询方法】

在生产任务单列表过滤出该生产任务单，然后点击“全流程跟踪”，找到其关联的当前业务期间的产品入库单，然后查看直接材料成本字段。

【重点提醒】

产品入库单上的直接材料成本需要通过生产成本核算才能算出，也就是说，如果某费用类型的分配标准是“按完工入库材料成本”，则需要先进行生产成本核算，算出产品入库单的直接材料成本之后，才能进行费用分配。
