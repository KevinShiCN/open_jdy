# 联副产品的原型客户案例

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：应用案例, 新特性, 快速入门
> 作者：Esperanza | 阅读：278 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/697845253080051968)

1 业务场景

1）XXXX科技有限公司

原材料投入煅后焦通电后生产出负级粉（主产品）和增碳剂（副产品，可销售），煅后焦的材料成本全部分到增碳剂，电费、容器坩埚也会录入到BOM里面，对应的费用分摊给主产品负级粉。

2）肇庆市XXX食品有限公司

生产酱油时投入原材料，生产出头抽和残渣两种产品，残渣加入水、盐等继续生产出二抽和剩余残渣，继续加水生产出三抽。

3）广州XXXX餐饮管理有限公司

客户主要用生产管理鸡的屠宰过程，将原材料鸡屠宰后产出秃鸡、内脏、鸡肠、鸡架，产品为一次性全部产出，成本主要分摊在主产品秃鸡上，其他的部分或分配很少的费用，下达生产任务时可以明确产出的产品有哪些，生产周期都很短，早上屠宰，下午就拿去卖了，因为食品加工的对这方面要求很高。

4）XXXX化工股份有限公司

化工行业多种原材料一起反应生成多种产品。

5）喀什XXXX农业有限公司

通过核桃去生成成品 核桃油和核桃粉， 一顿核桃可能生成5000的核桃油和4000核桃粉。能够预估出两种成品的成本占比，按比例分摊成本。

6）XXXX卫浴有限公司

塑料生产行业，原材料为塑料，第一次融的时候产出A等级产品，第一次融时会回收剩余的一些原料（比如，回收原料为用于生产的注塑模具的十字架，也是塑料），将回收剩余的原料第二次融，产出次等品，这时候需要将次等品作为副产品进行入库。

## 2 解决方案

针对上述场景，可以通过星辰旗舰版的联副产品功能来实现。

## 3 适用版本

星辰3.0以上旗舰版

## 4 适配行业

**1）食品加工行业：**

|  | 原料 | 产品 |
| --- | --- | --- |
| 豆制品加工 | 大豆 | 千张（主）、碎皮（副） |
| 酱油酿造 | 大豆、食盐等 | 头抽（主）、残渣（副） |
| 残渣、食盐等 | 二抽（主）、剩余残渣（副） |  |
| 剩余残渣、食盐等 | 三抽 |  |
| 肉鸡屠宰 | 肉鸡 | 秃鸡（主）内脏（副）、鸡架（副）鸡肠（副） |
| 核桃加工 | 核桃仁 | 核桃粉（主）、核桃油（联） |

2）化工行业、五金行业：

|  | 原料 | 产品 |
| --- | --- | --- |
| 废旧电池处理 | 废旧电池锻后焦 | 负极粉（主）增碳剂（副） |
| 塑料加工 | 塑料原料 | A等品（主）次等品（副） |
| 金属加工 | 金属板材 | 金属条-规格1（主）金属条-规格2（副） |
