# 倒冲领料是否影响成本?

【问题描述】

倒冲领料是否影响成本?

【解释说明】

倒冲领料是产品入库后自动生成领料单的过程,相关的有产品入库商品和领料材料,成本核算后对应的产品入库单和生产领料单会被反写入库成本和材料出库成本。

倒冲领料的流程:

![](https://vip.kingdee.com/download/0109567aabc088344bfa8afe520538a03451.png)

例如有产品00284 商品成本yp,下面有两个材料【00283】倒冲领料yp,【00285】正常材料yp,材料【00283】倒冲领料yp的单位材料成本是10,【00285】正常材料yp的单位材料成本是20,在生产任务单里这两个材料的材料用量(分子)都是2

![](https://vip.kingdee.com/download/010903f674b816a34f0d9f4554d1925444bd.png)

之前入库了一个采购入库单,材料的成本就是一个10 ,一个20,

 下面是关联的生产领料单,![](https://vip.kingdee.com/download/01093658890ee6d541ff8c9c966ac7bc19bb.png)

对应产品和材料的成本:

产品入库单的成本就等于10*2+20*2=20+40=60,倒冲领料的成本也会算进去。
