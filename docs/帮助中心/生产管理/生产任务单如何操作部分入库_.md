# 生产任务单如何操作部分入库?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题

【问题描述】

生产任务单如何操作部分入库?

【解释说明】

生产任务单支持**多次分批入库**,且当月未完成入库的业务,可在后续期间继续入库。

【解决方案】

比如生产任务单产成品数量是10

![](https://vip.kingdee.com/download/010903579dfa2a574bdfada2e6c2afa8d829.png)

新增产品入库单选源生产任务单入库,修改产品入库单的数量即可![](https://vip.kingdee.com/download/01099fe86184c57a4d3d95f648a675fb9997.png)
在生产任务单列表可以看到入库状态。
