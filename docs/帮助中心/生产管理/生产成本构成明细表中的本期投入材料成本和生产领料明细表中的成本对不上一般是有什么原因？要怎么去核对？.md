# 生产成本构成明细表中的本期投入材料成本和生产领料明细表中的成本对不上一般是有什么原因？要怎么去核对？

> 作者：Esperanza | 阅读：282 | 有用：2

## 问题描述

- 生产成本构成明细表中的本期投入材料成本和生产领料明细表中的成本对不上一般是有什么原因？要怎么去核对？
- 生产成本构成明细表中的本期投入材料成本不等于本期有源单的生产领料-生产退料是什么原因？

## 原因分析

生产成本构成明细表的本期投入=本期领料成本-本期退料成本+以前期间入库在本期退库的材料成本；

生产领料明细表仅包含前面2项。

（例如6月份的入库单，在7月份退库了，那么第七期的成本构成明细表里，本期投入会体现这部分数据，但是生产领料明细表里是没有的）

## 排查思路

- 领料单是否都有源单（若存在手工新增的生产领料单，没有关联生产任务单。无源单的生产领料单的成本不会在生产成本构成明细表里面体现）
- 是否存在以前期间已入库完工的生产任务单，在本期继续领料的情况。也即领料单的源单生产任务单行完工时间小于等于本期（例如生产任务单在第6期，且在第6期已完工。第7期继续领料，那么7月的生产领料明细表里会有这部分数据。但由于生产任务单第6期已经完工了，第7期的成本构成明细表就不会有这部分数据）
