# 手工新增的生产领料单/生产退料单，单据上的物料已经开启了共耗材料，为什么费用归集的时候引入内部数据引入不了？

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：快速入门, 常见问题
> 作者：移动生产白建凡 | 阅读：1008 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/449167242718942720)

**【问题描述】：**手工新增的生产领料单/生产退料单，单据上的物料已经开启了共耗材料，为什么费用归集的时候引入内部数据引入不了？

**【问题原因】：**

- 手工新增的生产领料单/生产退料单**没有审核**
- 手工新增的生产领料单/生产退料单**没有成本**

**【解决方案】：**

- 审核手工新增的生产领料单/生产退料单
- 进入库存管理>存货核算>核算单据查询，更新生产领料单/生产退料单的成本，再去做费用归集引入内部数据
