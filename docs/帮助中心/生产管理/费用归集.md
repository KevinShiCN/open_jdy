# 1、简介

**1.1功能介绍**

费用归集用于归集当期发生的所有用于生产相关的费用。

**1.2适用版本**

金蝶云·星辰专业版

**1.3应用场景**

用于对当期发生的所有用于生产相关的费用进行归集。

# 2、主要功能说明：

1）**费用归集支持指定任务单单号**：即该笔费用仅分配给特定的任务单。

业务场景：仅发生在某张任务单的费用在费用分配时仅分配给该任务单，如该任务单发生的外包费、委外工序费等。

2）**费用归集时支持录入车间**

业务场景：该笔费用仅发生在某个车间，则在费用分配时该笔费用仅分配给该车间的任务单。

# 3、常见问题：

**1） **共耗料是如何归集的？

可以通过2种方式归集：手动归集/内部引入。

• 手动归集：直接新增费用，录入共耗料的金额。

• 内部引入：自动将本期生产领料单/退料单中分录设置了【共耗料】的出库成本进行合计引入到费用归集界面（领料出库成本为正，退料单成本为负）

**2） **费用只能手动归集吗？

目前支持手动新增和引入两种方式归集费用。

引入支持引入内部数据和外部数据。

引入内部数据包括从生产领料单引入共耗料费用和从总账凭证引入。
