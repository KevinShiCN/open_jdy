# 新增bom单选择商品后为什么不显示商品类别?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：ylim | 阅读：199 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/637094146372302336)

【问题描述】

新增BOM单选择商品后为什么不显示商品类别?

【解决方案】

**1、商品类别字段未显示**

点击bom单右上角小齿轮打开字段设置界面,开启“商品类别”字段的显示列按钮,点击【确定】

![](https://vip.kingdee.com/download/010952d01e59710349bd911b0c6fc66df991.png)
即可显示出商品类别信息

![](https://vip.kingdee.com/download/01094476478520514954863ea4daf8ccb65c.png)
**2、字段有显示出来但是没有内容**

如下图所示,有商品类别字段显示出来,但是显示为空,说明商品信息上未设置类别信息

![](https://vip.kingdee.com/download/01099a39b2e692f1496698d07072d1c7b2bd.png)
点击【基础资料】-【商品信息】

![](https://vip.kingdee.com/download/0109a301ddcab6f945c2b79902b4ba58922c.png)
勾选对应商品点击【批量修改】,勾选“商品类别”选择类别信息后点击【确定】

![](https://vip.kingdee.com/download/01091a42888a656c4e3ea3d7a1c6ba204a45.png)重新新增bo商品后即可带出商品类别信息。

![](https://vip.kingdee.com/download/0109e51301f8890f419daa1d842025ff9749.png)
