# 生产任务单/委外加工单选源单,选择销售订单后,提示:所选源单待生产/委外数量为0,请重新选择源单

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：lykk | 阅读：157 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/647240720691769600)

【问题描述】

生产任务单/委外加工单选源单,选择销售订单后,提示:所选源单待生产/委外数量为0,请重新选择源单

![](https://vip.kingdee.com/download/010925fdd042b3ec495283d339260fa53014.png)

【解释说明】

选择的销售订单可能已经销售出库、或已关联生成过委外加工单、生产任务单或采购单据,所以待生产/委外数为0。

可在销售订单列表勾选改单,点击【全流程跟踪】查看是否关联生成了上述单据。

![](https://vip.kingdee.com/download/0109975d09cb0f2a4e47aec8bd1ff852f755.png)

【解决方案】

如果不希望做生产任务单时,待生产数量为0的单据被选择到,可以通过生产任务单列表界面的【批量新增】功能生成生产任务单。该界面默认勾选了【可生产数量为零不显示】的参数。

![](https://vip.kingdee.com/download/01098dd46b55923b4133b5366aa8b1c59cfc.png)
