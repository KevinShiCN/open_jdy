# bom单/生产任务单里面的材料用量(分子)产品产量(分母)什么意思?如何录入?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：2155 | 有用：0

【问题描述】

bom单/生产任务单里面的材料用量(分子)产品产量(分母)什么意思?如何录入?

【解释说明】

该字段是成对出现的字段,体现的是生产的产品和需要的材料之间的数量比例关系,适用于化工、鞋服等行业。生产一件产品所消耗的材料用量难以计量,但是可以知道消耗一定用量的材料可以大概生产多少产品。

**注意:“材料用量”与“产品产量”均为必录项。**

例如:鞋服行业中,生产 67 双鞋子大概消耗 3 卷线,此时材料用量(分子)是 3,产品产量(分母)是 67。此时无需计算出生产一双鞋需要多少卷线,只需要录入对应的数量比例关系即可,避免了前期因为四舍五入带来的数据误差。

![](https://vip.kingdee.com/download/0109c0bb2330e6544cb08aff4acceb9f1f41.png)

例如:产品单价比较低,一个产品只有几毛钱。按一个产品来算bom单材料用量很少,误差很大,想用1000个为单位来核算材料用量。

建议在【生产管理】>【生产数据】>【bom维护】子件信息中,同时录入“产品产量”和“材料用量”,例如生产1000个产品需要3个物料,那么“产品产量”填1000,“材料用量”填3即可。这样在计算生产计划投料数时数量较为精确。
![](https://vip.kingdee.com/download/010907e7d1bed8f441a1b55bfb0f45331881.png)
而对于电子、机械等更习惯于按照生产一件产品所消耗的材料用量来维护 bom的行业,则建议将产品产量隐藏显示,仅维护材料用量即可。
