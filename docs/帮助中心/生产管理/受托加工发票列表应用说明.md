# 受托加工发票列表应用说明

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：财税李炳全 | 阅读：147 | 有用：1

【问题描述】

旗舰版,受托加工模块的【受托加工发票列表】应用说明

![](https://vip.kingdee.com/download/0109d2793e466ced4d7583b34b09483a11d6.png)

【解释说明】

用于展示所有受托加工发票记录。

表头及按钮操作——

1、搜索:包含四种方式

1)快速搜索:在列表上方的搜索框中输入单据编号快速搜索;

2)常用条件搜索:支持按 ①单据日期②审核状态 筛选受托加工发票;

3)方案搜索:通过配置多个查询条件进行搜索

4)漏斗搜索:将鼠标放在列表的列标题上,出现,可以实现对列表数据的过滤

2、新增:新增受托加工发票,跳转到受托加工发票录入界面

3、刷新:刷新列表表体内容

5、引出:将列表中的受托加工发票引出到EXCEL,分为按列表引出、引出分析数据。

选择具体的受托加工发票(支持多选)后,根据弹出的操作按钮,支持:

1、审核:对未审核的受托加工发票进行审核

2、删除:对未审核的受托加工发票进行删除

3、收款:对已审核未收款的受托加工发票进行收款

5、打印/打印模板/打印设置

6、全流程跟踪:可查看当前单据所关联的上、下游单据

7、修改单据标签:对选中的单据,修改单据标签的内容或清空单据标签内容
