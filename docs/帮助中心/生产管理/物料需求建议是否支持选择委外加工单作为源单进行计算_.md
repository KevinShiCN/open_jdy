# 物料需求建议是否支持选择委外加工单作为源单进行计算?

> 作者：云社区用户55os622 | 阅读：108 | 有用：0

【问题描述】
物料需求建议是否支持选择委外加工单作为源单进行计算?

【业务场景】

当备货成品是需要委外加工的情况下,可以按委外加工单进行物料需求计算,生成对应的计划单据。

依次点击**【**生产管理**】**-【物料需求建议】
