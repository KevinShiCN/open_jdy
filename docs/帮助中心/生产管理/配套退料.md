# **1 简介  **

## **1.1 功能介绍**

配套退料功能适用于生产任务单/委外加工单多领料或减少生产数量后需要整套退料的场景，或退回成品后，需要拆分零件整套退回的场景。

## **1.2 适用版本**

星辰专业版/旗舰版

## **1.3 系统路径**

生产管理--生产单据--生产任务单

委外管理--委外单据--委外加工单

# **2 操作指南**

1、在生产任务单/委外加工单列表选择需要配套退料的生产任务单/委外加工单，点击配套退料

![上传图片](https://vip.kingdee.com/download/01005613f165cd0d4e4d8b4befe4009a907e.png)

2、点击配套退料按钮后，弹出退料弹框，输入本次退料套数后，点击确定，即可生成生产退料单/委外退料单，注意事项如下：

- 本次退料套数默认等于可退料套数，可修改
- 可退料套数等于材料已领数量减去报废数量后，可生产/委外的最小套数
- 点击生成生产退料单时，生产退料单自动关联所选生产任务单的领料单，如果存在未审核的生产领料单或生产退料单，可能无法满足按可退料套数进行退料，委外退料单同理
- 仅支持审核状态为已审核、完工状态为未完工或已完工的生产任务单/委外加工单配套退料，即手动完工的生产任务单或手动关闭的委外加工单不支持配套退料
- 最多可选择50行商品分录同时配套退料

![上传图片](https://vip.kingdee.com/download/0100788f99921b3f4ab08de1a1ff95c4d197.png)

3、生产/委外发料工作台同步支持配套退料，详情见：[https://vip.kingdee.com/knowledge/780747529347152128](https://vip.kingdee.com/knowledge/780747529347152128)
