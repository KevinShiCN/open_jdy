# 物料需求建议中的父级是什么意思?

【问题描述】

物料需求建议中的父级是什么意思?

![](https://vip.kingdee.com/download/01097f2df63837d6443797f87e075fd64d62.png)

【解释说明】

父级是指需求单据中的产成品对应的商品。由父级的数量,计算出各子级材料所需使用的材料数量。

![](https://vip.kingdee.com/download/01090142dfbe491449fd9bc7fcf7fb778334.png)
