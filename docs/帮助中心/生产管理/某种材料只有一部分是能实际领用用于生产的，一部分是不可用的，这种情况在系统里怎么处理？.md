# 某种材料只有一部分是能实际领用用于生产的，一部分是不可用的，这种情况在系统里怎么处理？

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：快速入门, 常见问题
> 作者：Esperanza | 阅读：566 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/449895866002870784)

****

**1、问题描述**

入库的材料中只有一部分是能实际领用的，一部分是不可用的，这种情况在系统里怎么处理？

2、场景介绍

生产火锅料，材料有坛子泡椒，坛子里有固形物有水，实际领料的时候就只领这个固形物，比如入库一整坛10公斤，领料出去可能用8公斤，有2公斤水，这种情况在系统中怎么处理？

3、解决方案

建议可以考虑针对这个原料设置损耗率。

在BOM和生产任务单中，均可设置损耗率（BOM设置了，可携带至生产任务单）。
