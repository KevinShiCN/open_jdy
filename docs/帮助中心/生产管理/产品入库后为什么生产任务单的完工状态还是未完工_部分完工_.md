# 产品入库后为什么生产任务单的完工状态还是未完工/部分完工?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：shaoling | 阅读：224 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/716096843356166400)

【问题描述】

产品入库后为什么生产任务单的完工状态还是未完工/部分完工?

【解释说明】

生产任务单的完工状态根据生产参数设置判断。参数可设置为以下三种模式:

①不自动完工

②完全入库

③完全入库且完全领料

若参数设置为"**不自动完工**",则生产任务单需要手动点击完工操作。如未进行手工完工操作,任务单将始终保持"未完工"或"部分完工"状态。

若参数设置为 **"完全入库"** 或 **"完全入库且完全领料"**,则生产任务单将在满足条件后 **自动完工。**
