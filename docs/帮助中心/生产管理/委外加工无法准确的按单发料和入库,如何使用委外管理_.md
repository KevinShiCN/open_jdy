# 委外加工无法准确的按单发料和入库,如何使用委外管理?

【问题描述】

委外加工无法准确的按单发料和入库,如何使用委外管理?

【解决方案】

委外发料建议使用倒冲流程,通过移仓的方式发料给供应商,产品入库后会自动生成委外发料单。

委外倒冲发料可以点击:[委外管理的倒冲发料业务](https://vip.kingdee.com/link/s/lUrIO)进行查看。
