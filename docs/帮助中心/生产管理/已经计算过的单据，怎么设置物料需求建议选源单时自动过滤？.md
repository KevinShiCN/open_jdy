# 已经计算过的单据，怎么设置物料需求建议选源单时自动过滤？

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：新特性, 常见问题, 进阶技巧
> 作者：移动生产白建凡 | 阅读：112 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/786971458361594880)

# **1 问题描述**

- 销售订单跑出来的净需求为0，下次运算其他单据，还是能显示这张销售订单，有什么方案可以过滤掉这种已经计算过的不需要再进行计算的销售订单吗？
- 已经跑过物料需求建议的委外加工单，如何设置下次计算时不再选到，不再重复参与运算？

# 2 适用版本

星辰3.0以上专业版/旗舰版

# **3 解决方案**

- 跑过物料需求建议，并生成计划订单的单据，“计算标记”字段会标识为“Y”
- 跑过物料需求建议，但没有生成计划订单的单据，就只会记录“最后计算时间”

![上传图片](https://vip.kingdee.com/download/0100790d977aeb9949a6925a52cccb5b53ed.png)

对于参与过物料需求建议计算，但因有库存所以未生成计划订单，再次跑物料需求建议选单时，对应的源单不需要再显示的场景，物料需求建议选单计算时在“选单设置”中勾选**“仅显示计算时间为空的单据（适用所有源单）”**即可

![上传图片](https://vip.kingdee.com/download/01005343a8c0329e46fbb2c9d2217a8ffa1c.png)

![上传图片](https://vip.kingdee.com/download/01007db7db55cd72431b904ff2bec491b025.png)

对于参与过物料需求建议计算且已生成计划订单，再次跑物料需求建议选单时，对应的源单不需要再显示的场景，物料需求建议选单计算时，“选单设置”中勾选**“仅显示计算标记为空的单据（适用所有源单）”**即可

![上传图片](https://vip.kingdee.com/download/0100328182882b464bc59f4c9811667fcc20.png)
