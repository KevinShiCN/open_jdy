# 生产过程的损耗如何计入产品成本

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题, 快速入门
> 作者：移动生产白建凡 | 阅读：1073 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/444920278695803392)

【问题描述】：生产过程中的损耗如何计入产品成本？

【场景介绍】：以委外管理为例。发给委外加工商1000套产成品所需的原料，由于生产损耗只收到990件成品，导致委外加工单无法正常关闭，怎样才能将这10件的损耗都计入收到的990件成品中？

【解决方案】：进入生产任务单/委外加工单列表，勾选相应单据，点击【完工】/【关闭】，则所有领料/发料的成本都会计算到已入库的产品上。

【举例说明】：以委外管理为例。2023年4月，加工1000件商品，当期材料全部发出，入库990件

   1. 进入委外加工单列表，因为该委外加工单未全部入库/超额入库，单据不会自动关闭，关闭状态为【未关闭】。进行委外成本核算，该商品成本单价=所有委外发料成本合计/委外加工商品数量=23059.58/1000=23.05元

![](https://vip.kingdee.com/download/010962a4ee0f23f4494db64d139b9b35b8e5.png)



    2. 进入委外加工单列表，选择该委外加工单后点击【关闭】按钮手动关闭这张委外加工单，再次进行委外成本核算，该商品成本单价=所有委外发料成本合计/委外产品入库数量=23059.58/990=23.29元。也即所有发料成本都计算到了入库的990件商品上，损耗的10件商品对应的发料成本计入到了入库的990件商品中
