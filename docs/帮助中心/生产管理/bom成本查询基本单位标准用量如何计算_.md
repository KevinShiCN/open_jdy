# bom成本查询基本单位标准用量如何计算?

【问题描述】

BOM成本查询基本单位标准用量如何计算?

【操作步骤】

**BOM成本查询基本单位标准用量,是根据bom单材料单位与基本单位计算而得。**

若bom单材料用量单位与基本单位一致,则基本单位标准用量=bom单材料用量;若bom单材料用量单位与基本单位不一致,则根据商品信息多单位换算计算而得。

1、如下图所示,以bom单:BOM00017为例,**SP00112商品材料用量5,基本单位标准用量300**![](https://vip.kingdee.com/download/010949c2bd9ea48647b7be8f48d725ac5236.png)
2、生产管理-bom维护,查询bom单:BOM00017。**SP00112商品材料数量5单位套**![](https://vip.kingdee.com/download/0109facad34983d2482d99f68e3e9fa3a0dd.png)
3、基础资料-商品信息查看**单位换算率1套=60件,件为基本单位**。**bom材料用量5套,则基本单位标准用量=5*60=300**![](https://vip.kingdee.com/download/0109e72c364aadfd4df4b7db5ff841e10763.png)
