# 倒冲领料的状态下还可以做生产补料吗?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：向钱kan | 阅读：25 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/779162775133090048)

【问题描述】

倒冲领料的状态下还可以做生产补料吗?

【操作步骤】

未完工的生产任务单可以继续做生产不料,但是不能是倒冲领料生成的,需直接做生产领料单,领料类型选择“生产补料”

可选源单增加原有材料数量

也可选单添加新材料
