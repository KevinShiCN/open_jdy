# 如何设置在制品参与/不参与费用分配?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题

【问题描述】

如何设置在制品参与/不参与费用分配?

【解决方案】

在【生产管理】-【完工与在制品归集】在制品分配标准设置,费用成本分配标准选择不计算在制品成本即可,如下图所示:

![](https://vip.kingdee.com/download/01096fe7cf1758ba4093bf6f36e04a1bfdc2.png)
若需要设置在制品参与费用成本分配,则可以选择约当产量法
