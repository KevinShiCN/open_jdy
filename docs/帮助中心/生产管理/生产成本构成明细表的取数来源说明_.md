# 生产成本构成明细表的取数来源说明?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：云社区用户0531183 | 阅读：72 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/793645822877034496)

【问题描述】

生产成本构成明细表的取数来源说明?

【解释说明】

生产成本构成明细表数据来源于生产模块的生产在制品初始数据及生产任务单的期初在制、本期投入、本期完工、期末在制,包括材料费、人工费、水电费等各种费用、总成本、单位成本。

生产成本核算完成后,可查询**准确的成本构成数据**;若生产单据发生变动,可重新执行生产成本核算,**自动更新报表数据**。

![](https://vip.kingdee.com/download/01095932618cb986441590621f88d455b562.png)

1. 期初在制:生产任务单截止筛选期间的期初未完工入库的数据,等于上一期期末在制产品的成本和数量。

2. 本期投入:生产成本构成明细表的本期投入=本期领料成本-本期退料成本+以前期间入库在本期退库的材料成本;

3.本期完工:生产任务单在筛选期间下完工入库数据。本期完工数量或成本=本期任务单下推的产品入库单的数量或成本-本期产品入库单下推的退库单的数量或成本

4. 期末在制:生产任务单截止筛选期间的期末未完工入库数量和成本。期末在制= 期初在制 + 本期投入 - 本期完工。
