# 联副产品成本分配，主产品和按比例分配的联副产品，其直接材料成本、费用都是为0，为什么？

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：快速入门, 精华汇总, 常见问题
> 作者：移动生产白建凡 | 阅读：129 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/774640185341653760)

# **1 问题描述**

在联副产品成本分配功能中，主产品和按比例分配的联副产品，其直接材料成本、费用均显示为0，但生产任务单对应的产品入库单上有直接材料成本和费用，这是为什么？

# 2 适用版本

星辰3.0旗舰版

# **3 原因说明**

联副产品成本分配的核心作用是确定联副产品的成本分配标准，仅作为生产成本核算前的费用分配方案设置。该功能不会在成本核算后将分配结果反写至主产品、联副产品的直接材料成本及费用字段。
