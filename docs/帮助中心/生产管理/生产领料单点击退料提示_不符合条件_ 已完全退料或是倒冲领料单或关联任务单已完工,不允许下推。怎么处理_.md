# 生产领料单点击退料提示:不符合条件: 已完全退料或是倒冲领料单或关联任务单已完工,不允许下推。怎么处理?

【问题描述】
生产领料单点击退料提示:不符合条件: 已完全退料或是倒冲领料单或关联任务单已完工,不允许下推。怎么处理?![](https://vip.kingdee.com/download/0109cffa2a4a0d20446cacfeed31d4bf229d.png)

【解释说明】

出现该提示不允许退料有三种原因:

① 该生产领料单中的材料已经生成过生产退料单,已完全退料,无剩余可退的数量

② 该生产领料单中的材料发料方式为倒冲领料,是生产任务单生产完工后按入库数量和材料的标准用量进行倒扣的材料,不可再退料,如需退料,需先反审核产品入库单[更多倒冲领料的知识](https://vip.kingdee.com/article/442853155680563712?productLineId=35&isKnowledge=2&lang=zh-CN)

③ 关联的生产任务单已经完工, 关联的生产领料不允许再进行删除
