# 生产成本核算一般什么时候做?有次数限制吗?

【问题描述】

生产成本核算一般什么时候做?有次数限制吗?

【解释说明】

生产成本核算通常在需要查看生产类单据成本时进行操作(例如,在查询生产领料单或产品入库单成本前,需先执行生产成本核算)。一般在月末结账前,需要先完成成本核算并确认数据无误后再结账。生产成本核算的操作没有次数限制。
