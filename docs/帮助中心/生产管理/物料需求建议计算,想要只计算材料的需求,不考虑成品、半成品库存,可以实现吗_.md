# 物料需求建议计算,想要只计算材料的需求,不考虑成品、半成品库存,可以实现吗?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：精华汇总, 常见问题, 进阶技巧
> 作者：移动生产白建凡 | 阅读：116 | 有用：1

【问题描述】

物料需求建议计算,想要只考虑材料库存并计算材料的需求,不考虑成品、半成品库存,可以实现吗?

【应用场景】

成品、半成品不考虑库存,都按毛需求计算并安排生产;底层材料有库存备货,只需要考虑底层材料的库存后安排采购,然后进行生产

【解决方案】

可以实现。物料需求建议中,“库存考虑方式”选择“只考虑底层材料库存”即可

![](https://vip.kingdee.com/download/010928f6f5c656f842d68c362e15d4c2681a.png)

【数据用例】

考虑即时库存,只考虑底层材料库存,选择一张销售订单,可计算数量为100,执行计算

![](https://vip.kingdee.com/download/01096e7e2e35551144f79b3de001deef21bf.png)

便携式折叠桌对应的BOM结构,产品产量、材料用量如图
![](https://vip.kingdee.com/download/0109510a2adeef5041ebb600a548e0b447d5.png)

物料需求建议运算结果如图。底色为紫色的商品是成品、半成品,底色为黄色的商品是底层材料。可以看到:

(1)成品(便携式折叠桌)、半成品(折叠桌面板组件、折叠桌支架组件)的计划订单量=净需求=毛需求,产生生产建议

(2)而材料的计划订单量是根据上层半成品、成品的计划订单量,以及产品产量和材料用量的关系计算出毛需求,考虑即时库存后计算出来计划订单量

![](https://vip.kingdee.com/download/010912ee5f65be51469d85f2a263136547c6.png)

【运算结果】

(1)半成品、产品不考虑库存,直接按毛需求计算,产生生产建议

(2)底层材料考虑库存,扣减库存后计算出计划订单量,根据目标单据投放规则产生采购建议

![](https://vip.kingdee.com/download/0109aae98930edf24b82b568a5fa13b142b6.png)
