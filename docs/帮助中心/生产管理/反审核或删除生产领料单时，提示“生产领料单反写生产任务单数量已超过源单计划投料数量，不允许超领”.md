# 反审核或删除生产领料单时，提示“生产领料单反写生产任务单数量已超过源单计划投料数量，不允许超领”

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题, 快速入门, 精华汇总
> 作者：移动生产白建凡 | 阅读：40 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/818798434801846272)

# 1 问题描述

反审核或删除生产领料单时（受“系统参数>仓存参数”中库存更新方式影响，保存时更新将在删除生产领料单时提示，审核时更新将在反审生产领料单时提示），系统提示“生产领料单反写任务单数量已超过源单计划投料数量，不允许超领”

![上传图片](https://vip.kingdee.com/download/0100e99358d3f39c4f888526d983d1585cba.png)

# 2 适用版本

星辰3.0及以上专业版/旗舰版

# 3 原因解析

（1）历史操作中，已在 “系统设置> 系统参数 > 生产参数” 中勾选了 “允许领料数量大于计划投料数量”，并已执行超计划领料，实际领料数量超过了生产任务单的计划投料数量

（2）后续又关闭了 “允许领料数量大于计划投料数量” 参数，恢复严格管控，不允许超领

（3）此时反审核或删除历史的生产领料单，会触发生产任务单的“已领数量”反写校验。因当前“已领数量”仍大于生产任务单的“计划投料数”，与当前系统参数冲突，导致出现中断提示

![上传图片](https://vip.kingdee.com/download/01000f4aaa6d2892465b879d5f96cd0774a2.png)

# 4 解决方案

（1）进入“系统设置>系统参数>生产参数>”，重新勾选参数“允许领料数量大于计划投料数量”并保存

（2）再执行反审核或删除生产领料单操作
