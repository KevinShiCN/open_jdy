# 1  简介

## 1.1  功能介绍

生产材料成本明细表是根据当前的过滤条件，查询生产任务单的单个产品在该期的材料明细情况，详细展示材料的期初在制、本期投入、本期完工、期末在制的数量及成本。

## 1.2  适用版本

金蝶云·星辰专业版/旗舰版

## 1.3  应用场景

生产管理员/成本会计可通过生产材料成本明细表，精准掌握生产任务单中产品在某个期间的材料领用、在制和结余情况。

## 1.4 系统路径

生产管理-生产成本核算-生产材料成本明细表

## 1.5 取数说明

**1）数据类型**

期初在制：取 上一期的“期末在制”的数量及成本。

本期领料：取 生产领料单的数量及成本。

本期退料：取 生产退料单的数量及成本。

前期入库本期退库材料：取产品退库单的分配的材料数量和成本。

本期投入：取 （本期领料 - 本期退料 + 前期入库本期退库材料）的数量及成本 。

本期完工：取生产成本核算后，该产品（含联副产品）耗用的材料数量及成本。

期末在制：取 （期初在制+本期投入-本期完工）的数量及成本。

**其他说明**

材料数量，是取材料基本单位的数量。

本期完工取数，是在该期间生产成本核算后才能取到数值；故只能查询截止到业务当期的数据，不支持查询以后期间的数据。

启用了联副产品的情况下，本期完工是指任务单的主产品和联副产品一起耗用的材料数量及成本。

材料分配过程会将材料按分配标准进行分配，分配过程不再按材料基本单位的精度处理，因此可能会出现该报表材料数量精度大于基本单位精度的情况。

## 1.5权限说明

该报表受数据权限控制，即若用户无某商品、部门的数据权限，则对应的产品入库和退库数据在该报表中无法查询。
