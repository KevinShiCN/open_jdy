# 辅助属性商品的bom,是否可以设置商品的辅助属性不必录?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：云社区用户55os622 | 阅读：298 | 有用：0

【问题描述】

辅助属性商品的bom,是否可以设置商品的辅助属性不必录?

【业务场景】

同一个商品编码在不同辅助属性下的BOM是通用的,想要直接录入这个商品的通用BOM,而不是需要一个个辅助属性单独设置。

【操作流程】

****

在【系统设置】-【系统参数】-【基础资料参数】-【商品参数】-若您希望录单时辅助属性非必录,请勾选,勾选上“BOM单”并保存参数设置。

![](https://vip.kingdee.com/download/0109f8e5fcd129d2413a881a73725fbce490.png)

参数保存成功后再录入BOM单时,若产品信息和子件信息中没有选择辅助属性也可以保存单据。

![](https://vip.kingdee.com/download/01096f35428fe0794e37b22de2288c9c2b53.png)
【温馨提示】

如果没有勾选“若您希望录单时辅助属性非必录“BOM””参数,BOM单就必须录入辅助属性:

①产品信息没有录入辅助属性时系统会提示:产品中有启用了“辅助属性管理”的商品时,“辅助属性”必须录入。

②若子件信息中没有录入辅助属性时会提示:第1行,当录入的子件中有启用了“辅助属性管理”的商品时,“辅助属性”必须录入。

![](https://vip.kingdee.com/download/0109cf2299121dd542359cf6e579bccb5791.png)
