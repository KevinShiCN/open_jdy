# 计划订单的供应商、业务员如何默认带出

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：ylim | 阅读：513 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/609551802362740736)

【问题描述】

计划订单的供应商、业务员如何默认带出

【解决方案】

点击【基础资料】-【商品信息】

![](https://vip.kingdee.com/download/0109fd562fe373c644e3a8a3a5488cea92f9.png)
点击商品编码或者勾选商品点击【批量修改】,设置商品的默认供应商和采购员

![](https://vip.kingdee.com/download/0109256a8325945746f0856f182fe50938c8.png)
![](https://vip.kingdee.com/download/01098a650b48335646389c077534fb4deac8.png)

当计划订单投放的目标单据时采购申请单或者采购订单时,生成的计划订单会自动带出商品的默认供应商和采购员信息

![](https://vip.kingdee.com/download/0109f50477631db14bb49aa4a5358aedcdc3.png)
