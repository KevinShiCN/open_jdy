# 生产/委外产品部分入库，领料/发料成本如何全部摊

生产/委外产品部分入库，领料/发料成本如何全部摊

![上传图片](https://vip.kingdee.com/download/0100c5f618560407480d99a189d96efb6a21.png)

      小李所在公司是生产型企业。常常会遇到材料完全领料或者完全发料，但是产品入库时会因损耗、丢失等情况部分入库。他们的想法的是将完全领料或发料的成本全部摊到部分入库的产品上来。那要如何录单才能实现这样的需求呢？小李可是很拿手了。那他是怎样处理的，我们一起来看看吧！

1、 生产任务单全部领料，部分入库。

比方说：7月的生产任务单，商品数量是10个，材料数量是10个。7月完全领料10个，7月产品部分入库5个。

![](https://vip.kingdee.com/download/0100f0591ff2d001461f8906ba7bdbd4a59a.png)

![](https://vip.kingdee.com/download/0100f61badcc880d4456a4a6fc249e8dd44a.png)

1、生产任务单完工状态是“未完工”，手动设置约当系统是0。

1） 完工与在制品归集

路径：【生产管理】-【完工与在制品归集】，将约当系数设置为0，保存。

![](https://vip.kingdee.com/download/0100246d03f951ef4cd384974a32c3d61d26.png)

2） 费用归集

路径：【生产管理】-【费用归集】，点右上角新增

![](https://vip.kingdee.com/download/0100e028330843c24fbdbb5daa916a46e06e.png)

添加20的维修费。

![](https://vip.kingdee.com/download/01003052d6bab55b4ff0b7a68e14f64ff06c.png)

3） 费用分配：

假设费用都按照完工产量的分配标准进行分配的。

![](https://vip.kingdee.com/download/010092be20d1a4144b88982d25bd5063f4c1.png)

路径：【生产管理】-【费用分配】

![](https://vip.kingdee.com/download/01005cbd5303b32d46659919fc52c6550909.png)

先点分配，在保存。

![](https://vip.kingdee.com/download/01007decc55ac4744e84b4bbab1fdbe7406a.png)

4） 生产成本核算。

路径：【生产管理】-【生产成本核算】

![](https://vip.kingdee.com/download/0100a38dbb2015404867b9df663fcf88bcf5.png)

分别查看材料和产品的成本计算报告的成本计算表。

![](https://vip.kingdee.com/download/010018261828b1ff44b38be3aa10b2909fcf.png)

![](https://vip.kingdee.com/download/0100c26afacd92bb497180f10b5e7079836f.png)

领料成本是100，总费用是20，材料成本+费用120全部摊到了部分入库数量是5个的产品入库单上，产品入库的成本：120。

![](https://vip.kingdee.com/download/010035a014ff92f4484a8ded522f03d134ab.png)

2、 以前月份的生产任务单，生产参数的生产任务单手动完工时间选择最后入/退库时间。

1） 路径：系统设置-系统参数-生产参数

![](https://vip.kingdee.com/download/010082bf3b0d5f924c538e654de3743cc6a9.png)

生产任务单手动完工时的完工时间选择：最后入/退库时间。

![](https://vip.kingdee.com/download/010064cd5e7a67e94e0bbdea55b6935da96d.png)

2）生产任务单手动完工，行完工时间取最后一笔产品入库单的单据日期。

![](https://vip.kingdee.com/download/01000ea29a6989aa44659c7c25dddea68c27.png)

![](https://vip.kingdee.com/download/0100e4696ad40b044dadbc105088c346faf0.png)

3）手动完工的生产任务单，约当系数默认是0，而且不能修改。

![](https://vip.kingdee.com/download/010062de6d0da3e04a9db39fa32897dfccc5.png)

4）费用分配：（假设费用归集是20的维修费）

![](https://vip.kingdee.com/download/0100512f37ea81d14cd3bf77ad880ee43506.png)

5）生产成本核算：

材料成本+费用=120，全部摊到部分入库的产品入库单上，产品入库单成本：120。

![](https://vip.kingdee.com/download/0100f69585eadf0b42e6a877885ef540fa1d.png)

3、当前月份的生产任务单。生产参数的生产任务单手动完工时间选择系统时间。

比方说：8月的生产任务单，商品数量是10个，材料数量是10个。8月完全领料10个，8月产品部分入库5个。

![](https://vip.kingdee.com/download/0100ec66b9619a02438da8210039fb368bfb.png)

![](https://vip.kingdee.com/download/0100bf85d06c6132440580b4d3526d900723.png)

![](https://vip.kingdee.com/download/010014a749388d764cdc981f4f06e642afa0.png)

1） 路径：系统设置-系统参数-生产参数

![](https://vip.kingdee.com/download/01007fca3a8845e04ade81ef4aee82fe3708.png)

生产任务单手动完工时的完工时间选择：系统时间。

![](https://vip.kingdee.com/download/010040310582d58f49348e759b3cb517813e.png)

2）生产任务单手动完工，行完工时间取系统时间。

![](https://vip.kingdee.com/download/01006a4a370faffd47408ce4fb1637abfaeb.png)

3）完工与在制品归集：

![](https://vip.kingdee.com/download/0100ba1e3e8d33704ba8a46b47430c3674c1.png)

4）费用归集，产生50元的人工费。

![](https://vip.kingdee.com/download/01001383b8fbb355426b9d5036468ec0ac46.png)

5）费用分配：先分配在保存。

![](https://vip.kingdee.com/download/0100237894ce2fc5458490daae7af4ddc45d.png)

6）生产成本核算：

领料成本100，费用50。材料成本+费用=150全部摊到部分入库5个数量的产品入库单上。产品入库单成本：150。

![](https://vip.kingdee.com/download/010055e293e9f227416fa760f74d7f15e3f4.png)

2、 委外加工单全部发料，部分入库。

比方说：7月委外加工单，商品数量是10个，材料数量是10个。7月完全发料10个，7月产品部分入库5个。

![](https://vip.kingdee.com/download/0100ddf64cc1546a4cbdb808bc1fc7396aa8.png)

![](https://vip.kingdee.com/download/0100d7738f3dd4314fddb71fa8e1384fb40e.png)

1、 以前月份以前的委外加工单，委外参数的委外加工单完工时间选最后入/退库时间。

1） 路径：系统设置-设置参数-委外参数

![](https://vip.kingdee.com/download/01006bdd3650959a45b790e3d4c2396e68eb.png)

委外加工单手动关闭时间选择：最后入/退货时间。

![](https://vip.kingdee.com/download/0100902573d345c442b19a51a2c8b9a0528e.png)

2）委外加工单手动关闭，关闭时间取最后一笔入库单时间。

![](https://vip.kingdee.com/download/010044793eb5fcbe4cf18eb1b96a5c2ae047.png)

![](https://vip.kingdee.com/download/01003f6c7756efce410c8987d0dfb19ec0ea.png)

3）委外成本核算：

发料成本是21.67，不含税加工费是25，发料成本+费用=46.67全部摊到了部分入库数量是5的委外产品入库单上。委外产品入库单成本：46.67。

![](https://vip.kingdee.com/download/01001a8fd8f71f494697af1e39107a722037.png)

![](https://vip.kingdee.com/download/0100a1e6eba510e640fda3706d10b85655aa.png)

比方说8月的委外加工单，商品数量是10个，材料数量是10个。8月完全发料10个，8月委外产品入库5个。

![](https://vip.kingdee.com/download/010062a4bfdfd9f0425692576f178a88fe3b.png)
