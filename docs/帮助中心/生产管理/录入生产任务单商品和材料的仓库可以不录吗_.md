# 录入生产任务单商品和材料的仓库可以不录吗?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：113 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/641808276774066944)

【问题描述】

录入生产任务单商品和材料的仓库可以不录吗?

【解决方案】

**生产任务单商品和材料的仓库可以不录入**。

![](https://vip.kingdee.com/download/01095f9ef890116d48668a4aafdbd9eb7c62.png)
如果录生产任务单不想自动带出仓库,删除商品信息默认仓库即可。

![](https://vip.kingdee.com/download/0109846a313892a64a968470e272fbf1daf4.png)
