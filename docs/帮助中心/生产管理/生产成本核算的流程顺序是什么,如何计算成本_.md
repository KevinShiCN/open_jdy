【操作步骤】

**一、生产成本核算是计算本期产品入库单及退库单的成本,因此核算前需检查**

1)核算前需审核本期所有生产业务单据包含:检查生产任务单、产品入库单、产品退库单、生产领料单、生产退料单

2)由于生成凭证后,核算无法反写成本及材料费,所以核算前请不要对本期产品入库单、产品退货单、生产领料单、生产退料单等单据生成凭证

**二、生产成本核算的流程主要有以下几个步骤:**

第一、【费用归集】:将生产过程中产生的费用做好费用归集

![](https://vip.kingdee.com/download/01093fdf2a38c0ee4053b8e1c07ebb0de22c.png)

第二、【完工与在制品归集】:未完工的生产任务单填好约当系数保存一下

![](https://vip.kingdee.com/download/010947c48d5d55434879b72d7be8e4e6c4ea.png)

第三、【费用分配】:将归集好的费用进行分配

![](https://vip.kingdee.com/download/010982265c7e821348bc82e7295f782e0a36.png)

![](https://vip.kingdee.com/download/0109bb6d6414571548d2b16e17b62c6cb695.png)

第四、【生产成本核算】:最后进行生产成本核算

![](https://vip.kingdee.com/download/010936352f6879df44bd8c566b065c0bd345.png)
