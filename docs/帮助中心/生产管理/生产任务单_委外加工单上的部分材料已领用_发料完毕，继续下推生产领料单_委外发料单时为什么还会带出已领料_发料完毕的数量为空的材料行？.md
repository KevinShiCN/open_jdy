# 生产任务单/委外加工单上的部分材料已领用/发料完毕，继续下推生产领料单/委外发料单时为什么还会带出已领料/发料完毕的数量为空的材料行？

> 作者：移动生产白建凡 | 阅读：599 | 有用：0

【问题描述】：生产任务单/委外加工单上的部分材料已领用/发料完毕，继续下推生产领料单/委外发料单时为什么还会带出已领料/发料完毕的数量为空的材料行？

**【举例说明】：**以生产管理为例。生产任务单上的材料B2已经领用完毕，再次下推生成生产领料单时，仍然带出材料B2的数量为空的行。如下图所示：

![](https://vip.kingdee.com/download/01098738afc08fb34e159b368169b28bb94e.png)

【解决方案】：

- 在生产领料单单据界面的【更多】-【选项】中开启【选源单领料时，仅返回未完全领料的材料】
- 在委外加工单单据界面的【更多】-【选项】中开启【选源单发料时，仅返回未完全发料的材料】

以生产管理为例，勾选上述参数后，仅带出未完全领料的材料。如下图所示：

![](https://vip.kingdee.com/download/0109cfb89161570d4eea84e74e0aa9e9460a.png)
