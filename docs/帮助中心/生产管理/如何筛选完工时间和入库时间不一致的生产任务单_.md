# 如何筛选完工时间和入库时间不一致的生产任务单?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：酸梅汤啦啦 | 阅读：17 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/782053307803516928)

【问题描述】

如何筛选完工时间和入库时间不一致的生产任务单?

【解释说明】

列表字段筛选以及方案过滤均**无法直接**筛选出完工时间与入库时间不一致的生产任务单。

方案过滤查询无法选择到“行完工时间”,则不支持通过方案查询去筛选。

![](https://vip.kingdee.com/download/0109dbc07a62b545416d950f6872803a9a10.png)
可以将列字段中“入库执行时间”和“行完工时间”开启显示,按其中一个字段筛选日期后查看另一个日期不一致的

![](https://vip.kingdee.com/download/0109045fb0f865da4cb4a4ca35ea27a857e8.png)
