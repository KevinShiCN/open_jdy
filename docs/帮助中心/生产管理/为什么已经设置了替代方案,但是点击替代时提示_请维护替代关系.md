# 为什么已经设置了替代方案,但是点击替代时提示:请维护替代关系

为什么已经设置了替代方案,但是点击替代时提示:请维护替代关系

![](https://vip.kingdee.com/download/01092ff5ab6e3ed34a9d97f0f4684b05ea11.png)

【解释说明】

 有两种可能的原因
① 确认替代方案是否是启用状态,如已经禁用则无法生效,依次点击【生产管理】-【替代方案】

![](https://vip.kingdee.com/download/010958975ee77dce4113b7aefe8fcd212b0e.png)

关闭状态限制,查看方案的状态
![](https://vip.kingdee.com/download/0109e9d87a1e35d24ce48ceb39f3fa9f89e4.png)

② 确认替代方案的适用范围

点开替代方案,找到【适用范围】字段,看看是否设置了【指定BOM】,并查看适用的BOM范围。 如生产任务单中使用的BOM不在适用范围内,则无法使用替代方案

![](https://vip.kingdee.com/download/01099dcf047d5c654537875e8e394968f39c.png)
