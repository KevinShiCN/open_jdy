# 产品入库明细表如何使用?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题

【问题描述】

产品入库明细表如何使用?

【解释说明】

产品入库明细表是按所选的时间段按单据来统计过滤期间内所有的产品入库和退库数据。

产品入库单的数量和入库成本显示为正,产品退库单的数量和入库成本显示为负。****

**取数规则:**根据筛选条件中设置的单据日期范围、交货/收货部门等等条件过滤出对应的数据。

仓管员可通过产品入库明细表掌握不同仓库、部门、物料等的产品入库及退库明细的情况。

【操作指引】

菜单栏点击【生产管理】>【产品入库明细表】

![](https://vip.kingdee.com/download/010924985a6667b54d4488d9b5e3152c668d.png)
可查看入库数量以及单位成本。

![](https://vip.kingdee.com/download/0109adece6362e0949c0a805d6cbf370d655.png)
支持多种过滤查询条件。

![](https://vip.kingdee.com/download/01094d3f0df0d6694037942d66ce6136c6c8.png)

**【权限说明】**

该报表受数据权限控制,即若用户无某物料、部门、仓库、职员的数据权限,则对应的产品入库和退库数据在该报表中无法查询。

【温馨提示】

**产品入库明细表不支持按客户查询。**
