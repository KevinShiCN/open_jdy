# 审核生产领料单提示:存在材料行未关联任务单,出库成本将无法分配到产成品的材料成本中,是否继续?是什么意思?

【问题描述】
审核生产领料单提示:存在材料行未关联任务单,出库成本将无法分配到产成品的材料成本中,是否继续?是什么意思?

![](https://vip.kingdee.com/download/0109b38d8e774af44df0b3b91eff61598186.png)
【解释说明】

表明当前审核的生产领料单不是通过生产任务单领料生成的,**未关联生产任务单的领料单中材料的出库成本将不会被计入到产品的入库成本之中**。

如需要材料成本计入产品入库成本,需要删除/修改生产领料单,选择源单【生产任务单】重新生成。
