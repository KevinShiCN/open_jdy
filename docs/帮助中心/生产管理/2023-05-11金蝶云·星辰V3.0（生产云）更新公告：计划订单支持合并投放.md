## 一、新功能：（New~!![](https://vipstatic.obs.cn-north-4.myhuaweicloud.com/statics/emotion/define/86.gif)）

### 计划订单合并投放

功能介绍：在计划订单列表界面新增按钮【合并投放】、【合并规则设置】，支持按合并规则，合并投放生成采购申请单、采购订单、生产任务单、委外加工单。

适用版本：星辰专业版

应用场景及痛点：同一个材料多张计划订单，不能清晰的知道采购/生产总数，下游单据分多行显示，不便于进度跟进。

系统路径：

生产管理-物料需求建议-计划订单

![上传图片](https://vip.kingdee.com/download/010006be6afc0b8443d9bf8663f43f35e065.png)

应用流程：

①在生产管理-物料需求建议-计划订单列表，点击【合并投放】旁边的下拉框，点击【计划订单投放设置】按钮，根据需要设置合并规则。

![上传图片](https://vip.kingdee.com/download/0100c33751f092dd4aa5b47cc9d5d11d746d.png)

![上传图片](https://vip.kingdee.com/download/0100514250c6a53c46f485c6368d09a3571c.png)

②勾选多张计划订单，点击【合并投放】按钮，系统将按照设置的投放规则，在下游单据上进行合并处理。

![上传图片](https://vip.kingdee.com/download/01005a5caeb7df4d4c78a45a08b635e71ee8.png)

![上传图片](https://vip.kingdee.com/download/0100fdd2143d130544839dea2e9ef641511b.png)

**③备注说明：**

销售订单的成品与无源单的成品，默认不合并：

a、来源于销售订单的成品生成生产任务单时会反写生产数量至销售订单中。

b、而来源于负库存或安全库存的成品生成生成任务单时，因无源单不需要反写。

这两种情况同时存在时，相同商品不支持合并在一行中，默认拆分为两行分录。
