# 足缺料分析

# **1 简介  **

## **1.1 功能介绍**

足缺料分析支持按生产任务单/委外加工单分析每张单的材料库存是否充足，不充足时给出缺少的量，并支持按缺料情况申请采购。

## **1.2 应用场景**

1）生产计划制定

企业在制定生产计划时，通过足缺料分析可以准确了解每种物料的现有库存、预计可用库存等信息。根据这些数据，合理安排生产任务和跟进生产进度，避免因物料短缺导致生产延误，或者因物料积压造成资金占用和库存成本增加。

2）物料采购决策

足缺料分析能够为物料采购提供准确的依据。通过分析物料的短缺情况，企业可以确定需要采购的物料种类、数量和采购时间，避免盲目采购或采购不及时。

3）生产现场物料配送

在生产现场，足缺料分析可以帮助企业及时了解各个生产环节的物料需求情况，合理安排物料配送，确保生产线上的物料供应不断档。

4）应对紧急情况和突发事件

当企业面临紧急订单、原材料供应中断、生产设备故障等突发事件时，足缺料分析可以帮助企业迅速评估事件对物料供应和生产的影响。

## **1.3 适用版本**

星辰3.0以上专业版/旗舰版

## **1.4 系统路径**

生产管理-生产报表-足缺料分析

委外管理-委外报表-足缺料分析

# **2 操作指南**

## 2.1 选任务单及加工单分析

- 支持选择的工单需满足的条件：未审核；已审核未关闭且未全部领料。
- 说明：返回的数据是未领数量大于0的材料；未领数量小于等于0的材料行不返回。

![](https://vip.kingdee.com/download/010953ec8c023a414104ae494fbd204a7eba.png)

## 2.2 分析参数

支持考虑即时库存或预计可用库存，支持考虑安全库存，支持排除不参与计算的仓库库存。

考虑预计可用库存时，按系统参数的预计可用库存获取预计入库和预计出库进行分析。

![](https://vip.kingdee.com/download/0109478e4531af68451483c3813bad7fb391.png)

## 2.3 分析结果

**明细页签**：按工单明细显示未全部领料的材料，显示每一个材料的足缺料数量。

- 当商品的所有材料都不缺料时，工单的分析结果会显示“库存充足”，当有一个或多个材料缺料时，工单的分析结果是“材料短缺”。
- 对于足缺料数量是正数时，认为材料的库存是充足且有剩余的。对于足缺料数量是负数时材料的库存是不足的，此时会按缺料数量给出对应的建议采购数量，可以按源单生成采购订单。
- 对于足料的材料，也支持手工维护建议采购数量后进行采购。
- 按源单生成的采购订单，采购订单的源单单号是生产任务单单号或委外加工单单号，支持全流程跟踪。

![](https://vip.kingdee.com/download/01092c85fed8d6544c27a3aa7c053574b54c.png)

**汇总页签：**

按物料+辅助属性汇总明细页签的分析结果，对于重复的材料行可以查看汇总的数据，按物料+辅助属性查看足缺料数据。

- 对于足缺料数量是正数时认为材料充足，对于足缺料数量是负数时材料缺料，此时会按缺料数量给出对应的建议采购数量，可以按汇总生成采购订单。
- 汇总页签支持自动携带商品信息的默认供应商、业务员。
- 对于足料的材料，也支持手工维护建议采购数量后进行采购。
- 按汇总生成的采购订单，采购订单的源单单号为空，不支持全流程跟踪。

![](https://vip.kingdee.com/download/010924abd08363ff4c09b4b15a6c3110c607.png)

# **3 常见问题说明**

1）未审核的工单分析之后，如果按明细生成了采购订单，则控制工单不允许删除。如果是按汇总生成，因采购订单与源单没有关联关系，工单可以删除。

2）从单据类型到分析结果列，均是工单的商品信息，同一行商品多个原材料时此部分信息会重复显示，不支持合并。“分析结果”是按商品行显示，不是每个材料的分析结果，每个材料是否缺料按足缺料数量来查看，为负数时是缺料的。
