# 生产消耗量的计算公式有哪几种?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：2463 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/398279483746080256)

【问题描述】

生产消耗量的计算公式有哪几种?

```

【解释说明】

消耗量指的是除固定损耗外,生产该产品需要耗用的实际的材料用量。
有以下两种计算方式:

1.标准用量/(1-损耗率)
2.标准用量*(1+损耗率)
3.如果已知标准用量和消耗量需要计算损耗率,公式为:1-标准用量/损耗数量或者消耗量/标准用量-1
```
