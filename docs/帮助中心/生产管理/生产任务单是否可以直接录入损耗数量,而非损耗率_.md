# 生产任务单是否可以直接录入损耗数量,而非损耗率?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：Siwennn | 阅读：494 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/612451122669725184)

【问题描述】

生产任务单是否可以直接录入损耗数量,而非损耗率?

【业务场景】

不同的车间损耗不同,需要在下达生产任务单的时候录入损耗,但因为损耗率除不尽,希望可以直接填写损耗数量。

例如:标准用量是14,根据实际损耗,计划投料数应为15,损耗数量为1,但是通过调整损耗率,无法精准计算出损耗数量1,如下图所示。

![](https://vip.kingdee.com/download/010910737598e0a84b05b13d29ba64125374.png)

【解决方案】

在生产任务单通过**【固定损耗】**字段处理即可。

按举例中的应和,直接在生产任务单录入固定损耗1,即可得出计划投料数15,如下图所示。

![](https://vip.kingdee.com/download/0109f8271c4119d24a5f82b560ef46a7ae84.png)
