# 模拟报价如何使用

【问题描述】

模拟报价如何使用

【适用版本】

金蝶云星辰专业版、旗舰版

【解释说明】

模拟报价能够快速计算商品的预估价格,根据产品、数量、原材料成本、制造费用、税率等因素,为用户提供较为准确的价格参考

【业务场景】

1、在产品研发阶段,企业可以使用模拟报价来评估不同的设计方案、材料选择和生产工艺对产品成本和价格的影响。这有助于在产品开发过程中进行成本控制和优化,确保产品在满足市场需求的同时具有合理的价格竞争力。

2、当产品研发完成后,模拟报价可以为产品的最终定价提供重要参考。通过综合考虑成本、市场需求、竞争对手价格等因素,利用模拟报价生成不同的定价方案,并分析其对市场份额和利润的影响,从而确定最优的产品价格策略。

3、销售阶段,销售人员可以使用模拟报价为客户提供快速、准确的报价,帮助客户了解产品的价格范围,促进销售机会的转化。同时,也有助于销售人员在与客户谈判时更好地掌握价格底线和谈判空间。

【操作流程】

从【生产管理】>【模拟报价】进入模拟报价录入页面

![](https://vip.kingdee.com/download/0109cbab162c0e1b461f8a79112fc8624fce.png)或者从【委外管理】>【模拟报价】进入模拟报价录入页面
![](https://vip.kingdee.com/download/01099eab1d70223b4ca6a92992a4464d5b27.png)

第一步:新增模拟报价,设置取价规则。当前支持设置材料成本来源(商品参考成本、最近采购入库成本、期初平均成本)、设置委外加工费来源(商品委外价、最近一次委外价)。用户可以按实际场景调整取价优先级
![](https://vip.kingdee.com/download/0109f08f777787a54762b8b963719d12b2b5.png)

第二步:选择需要报价的商品,并确定BOM展开方式,默认是展开至底层(最多支持展开至第10层,超过10层的BOM也仅展开至第10层),可手工选择“展开至第1层”

![](https://vip.kingdee.com/download/01090adbe970eea840099d681a1e2a901390.png)若已创建销售订单或销售预测单,需进行模拟报价时,可选择源单执行模拟报价操作
![](https://vip.kingdee.com/download/0109b8615c83bb374d449446233b1995b8bf.png)选完源单返回的商品行中,单位和数量默认携带源单信息,不支持修改

![](https://vip.kingdee.com/download/0109d6f5beca440c4f6ba36f9a9591db97ae.png)第三步:选择商品后,系统按BOM展开方式及取价规则设置,获取最底层材料的材料成本和委外加工费用,再自动向上计算上一层级的材料成本和委外加工费。非底层子件,均是汇总直接子件的材料成本和委外加工费,再加上本级自身的委外加工费,得出本级总成本。
① 商品信息的材料成本等于“一级子件”的材料成本之和,非一级子件的材料成本已经累加至它的直接上级。
![](https://vip.kingdee.com/download/010981dbfeb5b8634ae29d1782c68a14f0b6.png)
②商品信息的委外加工费用等于所有子件的委外加工费用之和。
![](https://vip.kingdee.com/download/0109e81bb63c472a4ebc82c29dfaeff6836b.png)
③商品的建议报价(不含税)=【材料成本+委外加工费(子件)+委外加工费(成品)+自定义待摊费用1-5】*(1+成本上浮率)
![](https://vip.kingdee.com/download/0109de72334f39bf439ca2713e2f8a39887b.png)
第四步:保存审核单据,支持下推生成销售报价单

![](https://vip.kingdee.com/download/0109f9a20c0c8a164ff4ac48441e3355d793.png)
下推时,将模拟报价单的“不含税单价”携带至销售报价单的建议报价和实际报价,用户再按实际情况调整实际报价,即可保存审核销售报价单。
![](https://vip.kingdee.com/download/0109bf02899c8afe4c3e833ea1d0f1a80ee6.png)
