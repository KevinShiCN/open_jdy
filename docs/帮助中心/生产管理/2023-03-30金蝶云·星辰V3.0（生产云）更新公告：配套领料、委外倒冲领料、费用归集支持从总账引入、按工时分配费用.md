### 1、生产任务单及委外加工单支持配套领料/发料

功能介绍：在生产/委外的业务中，需要配套领料的时候，手动计算每一种材料的发料数量很繁琐，易出错。金蝶云·星辰在生产任务单/委外加工单列表增加【配套领料】/【配套发料】的按钮，一键自动计算待领料套数、库存可配套数量，帮助仓库实现快速、准确领料/发料。

适用版本：星辰专业版

应用场景：

①一张工单，生产数量300，由于产能等原因，需要分批生产、分批领料，如每次只生产30个，就一次领生产30个产品需要的材料，手动计算每一种材料要发多少数量，工作量大且易出错。

②一张工单需要发20种材料，生产数量20，但本次库存数只够发15套材料，需要系统自动计算15套的每一种材料应该发多少数量，希望自动按库存数量进行发料。

系统路径：

生产管理-生产单据-生产任务单查询

![上传图片](https://vip.kingdee.com/download/0100f599d2c523f24c38ac1ce333db14f573.png)

委外管理-委外单据-委外加工单查询

![上传图片](https://vip.kingdee.com/download/0100fb2927d924a74d8e84a74f6b715fec15.png)

应用流程（以生产任务单配套领料为例，委外加工单配套发料应用流程与之类似）：

①在生产任务单列表勾选需要配套领料的任务单，支持多选，限制不超过10张，点击【配套领料】按钮。

![上传图片](https://vip.kingdee.com/download/0100e08485a2630a4436b81effbee9f4f6d9.png)

②系统弹出配套领料信息界面，在此界面可直观看到各产品的【生产数量】、【待领料套数】、【库存可配套数量】等信息，可根据需要填写本次领料套数。

![上传图片](https://vip.kingdee.com/download/0100c9a5caab3f214a8e905dfd3f9c88a769.png)

【待领料套数】和【库存可配套数量】是由公式计算得出的，计算公式说明如下：

1.	任务单中所有材料的可领料的最大套数，

计算公式1 = min[ 材料1的（计划投料数-已领料数量）/计划投料数量*商品的数量，材料2的（计划投料数-已领料数量）/计划投料数量*商品的数量，材料3的（计划投料数-已领料数量）/计划投料数量*商品的数量，……]

计算公式2 = 商品的数量 – 已领套数

说明：

 a)	计算公式均取基本单位的数量参与计算。

 b)	计算结果为小数时，按商品单位精度向下取整，尾差不处理。（例如：生产1.00个商品，第一次计算的可配套数是0.4578，保留2位精度向下取整0.45，则剩余0.0078，再次计算时可配套数0.0078，向下取整0.00，不配套，此时这部分材料的领用可以使用【领料】按钮进行处理）。

2.	计划投料数-已领料数量<=0时，取0，此时最大可配套数=0。

3.	倒冲材料及不发料的材料行不影响可配套数的计算，即仅按发料方式为直接领料的所有材料计算配套数量。

数据举例：

|  | 产品 |  | 单位 | 数量 | 已领套数 |  |
| --- | --- | --- | --- | --- | --- | --- |
|  | A |  | KG | 10 |  |  |
|  | 材料 | 发料方式 | 单位 | 计划投料数量 | 已领数量 | 即时库存数量 |
|  | B | 直接领料 | 件 | 200 |  |  |
|  | C | 直接领料 | KG | 1 |  |  |
|  | D | 倒冲领料 | 包 | 50 |  |  |

1)	当所有材料全部未领料（所有材料的领料数量=0）时，可配套数= 10；

2)	当部分发料，如B的已领数量=20，C未发料，可配套数为B材料的配套数（200-20）/200*10=9；

3)	当B全部发料，C未发料，可配套数=0；

4)	倒冲领料D是否发料，不影响可配套数量的计算。

注：【本次领料套数】字段的值，默认等于待领料套数，可手工修改。若系统参数“是否允许领料数量大于计划投料数量“设置的是不允许，则控制该字段的值须小于等于待领料套数；超过时将给出提示。若设置的是允许，则按照本次配套数计算领料数量。

![上传图片](https://vip.kingdee.com/download/0100635a3edccea84da6a3320774c7054543.png)

![上传图片](https://vip.kingdee.com/download/01003f0d35c3a8454a5293ff347352befe6c.png)

③确认好本次领料套数后，点击确定。

![上传图片](https://vip.kingdee.com/download/0100cf31a54083454311ba90119bf57356dc.png)

④系统根据本次领料套数自动生成生产领料单。

![上传图片](https://vip.kingdee.com/download/010082a53cb525784c42bfa586272ec47888.png)

### 2、委外管理支持倒冲领料

功能介绍：倒冲领料功能适用于工贸企业在外发生产加工过程中，委外发料时，存在部分材料无法按单发料，委外完工后按入库数量和材料的标准用量进行倒扣的情况。

适用版本：星辰专业版

应用场景：材料因形态或包装原因，无法拆分发料。按单发料时领料数量是零散的，材料包装是整的，无法拆分，例如润滑油，通常都是罐装或桶装，生产使用时是按重量克进行计算。委外加工单计划发料数量是1千克，但仓库无法称重发放1千克，只能发1罐或一桶。经常多发，单个委外加工单使用不完时，半罐的材料又不好退回委托方的仓库。

痛点：①按单领料不符合实际，无法发料。②多发的材料会造成某个委外加工单材料成本变高，另一单因现场有余料不重新领料，造成成本偏低。

委外倒冲流程：

![上传图片](https://vip.kingdee.com/download/0100c03d216e81734bce8bdbc762bd8d2706.png)

应用流程：

①如某个材料为倒冲领料的材料，可在商品基础资料里勾选【倒冲领料】的参数，勾选后，可以设置倒冲仓库，如仓库启用了仓位，可维护仓位。

![上传图片](https://vip.kingdee.com/download/01008b75dc4b12ff423087b7ed30468cc504.png)

②BOM单和委外加工单中维护发料方式为倒冲领料。

![上传图片](https://vip.kingdee.com/download/0100ed61bf6e246f4012ba34bec1d976b4d4.png)③发料方式为”倒冲领料“的材料行，可以在委外加工单中进行移仓。（不是倒冲领料的材料行不支持进行移仓）

![上传图片](https://vip.kingdee.com/download/0100b575c95e0e004cbbb21fe4723f90632e.png)

④如果委外加工单上存在发料方式为倒冲领料的材料，则对应的委外产品入库单审核时，会自动生成倒冲的委外发料单。

![上传图片](https://vip.kingdee.com/download/01000fd353bdb81442d5bc99ae57ed59231c.png)

![上传图片](https://vip.kingdee.com/download/0100fae1a5b963a241168b0e161156c1db60.png)

自动倒冲失败时，需要手动在委外产品入库单界面点击【更多】下面的“倒冲发料”按钮，进行手动倒冲发料。

![上传图片](https://vip.kingdee.com/download/01009c49fdf8f013449194155009dd35a674.png)

⑤倒冲领料生成的委外发料单不允许反审核，不支持手动删除，委外产品入库单反审核时，系统会自动删除关联的倒冲委外发料单。

![上传图片](https://vip.kingdee.com/download/0100f429c6719bc6472fbcdc829ff68eede4.png)

### 4、费用支持按工时分配

功能介绍：支持按工时分配费用

适用版本：星辰专业版

应用场景：生产企业为了更合理的进行费用分配，分配某些费用时需要按照工时进行分配，即生产产品所用的工时越长，分配的费用越多。例如工人工资需根据工人实际生产各个产品所用的实际工时进行分配，设备折旧费应按照设备工时进行分配，才能更合理地计算成本。

系统路径：生产管理-生产成本核算-费用类型

![上传图片](https://vip.kingdee.com/download/0100178173563a014f3097ec50db4f889511.png)

应用流程：

①在商品资料-生产信息里可设置标准工时、工时单位。

![上传图片](https://vip.kingdee.com/download/0100147cd69e6d4e4213baffd4a1d6c095b1.png)

②在费用类型里，支持对各种费用的分配标准设置【按工时】分配。

![上传图片](https://vip.kingdee.com/download/0100b10db1f253154ce682fffa51a1ea0179.png)

③在费用分配界面，新增了【总标准工时】、【工时】和【工时单位】三个字段。字段的值在新增费用分配时，自动计算并刷新。

【总标准工时】=标准工时*本期完工产量。

【工时】默认等于总标准工时，可编辑修改。注意：费用分配单生成凭证后，该字段不可编辑；以前期间的费用分配单该字段不可编辑。

![上传图片](https://vip.kingdee.com/download/01000d97a3e0d0b24ca7b852cba740e7b8b9.png)

④点击分配时若有费用设置了【按工时】，则该费用按照【工时】字段的值进行分配。

适用版本：星辰专业版

①在生产任务单列表点击【批量新增】。

![上传图片](https://vip.kingdee.com/download/01007830ec8784864d99851bca4e68c31c3b.png)
