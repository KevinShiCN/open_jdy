# 生产领料差异分析表取数说明

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题

【问题描述】生产领料差异分析表取数说明

**【取数说明】******

生产领料差异分析表是按所选的时间段按单据来统计过滤期间内所有的领料数据及计算实际领料与计划领料的差额及差异率。超领则差异率为正数,少领则差异率为负数。

**【取数规则】**

根据筛选条件中设置的单据日期范围、任务单单号、完工状态等等条件过滤出对应的数据

**【权限说明】**

该报表受数据权限控制,即若用户无某物料、部门、仓库、职员的数据权限,则对应的数据在该报表中无法查询。****
