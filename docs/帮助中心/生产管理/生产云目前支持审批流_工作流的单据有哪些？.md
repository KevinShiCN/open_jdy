# 生产云目前支持审批流/工作流的单据有哪些？

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：功能介绍, 快速入门, 精华汇总
> 作者：Esperanza | 阅读：1129 | 有用：5
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/489477051007443200)

生产云相关单据支持审批流/工作流的情况如下表所示：

| 模块 | 单据 | PC端 | 移动端 | 支持审批的条件 |
| --- | --- | --- | --- | --- |
|  | BOM单 | √ | - | 制单人、BOM分类 |
| 生产管理 | 生产任务单 | √ | √ | 生产车间（部门编码、部门名称） |
| 生产领料单 | √ | √ | 领料人、领料部门（部门编码、部门名称）、仓管员、是否倒冲生成、领料用途（辅助资料编码、辅助资料名称）、领料类型 |  |
| 生产退料单 | √ | √ | 退料人、退料部门（部门编码、部门名称）、仓管员、退料原因、是否倒冲生成 |  |
| 产品入库单 | √ | √ | 交货人、交货部门（部门编码、部门名称）、仓管员 |  |
| 产品退库单 | √ | √ | 收货人、收货部门（部门编码、部门名称）、仓管员 |  |
| 委外管理 | 委外加工单 | √ | √ | 业务员、部门编码、部门名称、本次应付加工费 |
| 委外发料单 | √ | √ | 供应商分类、供应商名称、供应商编码、结算期限、业务员、部门编码、部门名称、是否倒冲生成 |  |
| 委外退料单 | √ | √ | 供应商分类、供应商名称、供应商编码、结算期限、业务员、部门编码、部门名称、是否倒冲生成 |  |
| 委外产品入库单 | √ | √ | 供应商分类、供应商名称、供应商编码、结算期限、业务员、部门编码、部门名称 |  |
| 委外产品退货单 | √ | √ | 供应商分类、供应商名称、供应商编码、结算期限、业务员、部门编码、部门名称 |  |
| 委外加工费用 | √ | - | 供应商分类、供应商名称、供应商编码、结算期限、业务员、部门编码、部门名称、本单应付加工费 |  |
| 委外加工费（退货） | √ | - | 供应商分类、供应商名称、供应商编码、结算期限、业务员、部门编码、部门名称、本单应付加工费 |  |
| 受托加工 | 受托加工订单 | √ | - | 制单人（人员名称、工作流角色、职位名称、上级组织）、本单成交金额、部门编码、部门名称、客户编码、客户名称、客户分类、结算期限、订金比率、本次订金、结算客户（客户编码、客户名称、客户分类、结算期限） |
| 客供材料入库单 | √ | - | 制单人（人员名称、工作流角色、职位名称、上级组织）、客户编码、客户名称、客户分类、结算期限 |  |
| 客供材料退货单 | √ | - | 制单人（人员名称）、客户（客户编码、客户名称、客户分类、结算期限） |  |
| 受托加工出库单 | √ | - | 制单人（人员名称、工作流角色、职位名称、上级组织）、部门编码、部门名称、客户编码、客户名称、客户分类、结算期限、结算客户（客户编码、客户名称、客户分类、结算期限） |  |
| 受托加工退货单 | √ | - | 制单人（人员名称）、部门（部门编码、部门名称）、客户（客户编码、客户名称、客户分类、结算期限） |  |
| 受托加工发票 | √ | - | 客户（客户编码、客户名称、客户分类、结算期限）、部门（部门编码、部门名称）、制单人（人员名称）、本次应收账款、收款抹零 |  |

特别说明：

1）有的单据只限于特定的版本，具体支持哪些单据以对应版本说明为准，没有特别说明则适用于所有版本

2）目前表中列出的所有功能支持星辰3.1以上版本，如果有您没有需要的功能，建议联系售后客服升级到星辰3.1
