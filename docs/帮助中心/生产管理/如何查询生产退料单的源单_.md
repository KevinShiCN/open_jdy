# 如何查询生产退料单的源单?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：嘿唬哈 | 阅读：160 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/594331742744137472)

【问题描述】

如何查询生产退料单的源单?

【操作步骤】

1、点击【生产管理】-【生产退料-查询】;

![](https://vip.kingdee.com/download/0109de2c84c75ee44bd0b6eec41b19aa8300.png)

选中需查询的生产退料单,点击【全流程跟踪】即可。

![](https://vip.kingdee.com/download/0109ef92df9593604041a06cf00550ecdbb5.png)

![](https://vip.kingdee.com/download/0109f7e32ef96cb54cd98ebd7602cd87e806.png)
2、单据列表显示出源单信息字段查询

![](https://vip.kingdee.com/download/0109c6c7b1d06efd4d95a36b07d5ba6574f1.png)

3、打开单据界面显示出源单信息字段查询

![](https://vip.kingdee.com/download/01099656bd037e7942dd9f8a7bf35f7a0544.png)
