### 【财务操作1】：进行完工与在制品归集

此步骤目的时为了确认本期的完工数量、在制品数量、约当系数、在制品分配标准信息，确认无误后点击保存，为后续的生产成本核算做准备；

注意：若本期的任务单有变化（如删除了任务单、新增了任务单、变更了任务单的产品数量等）、领料情况有变化、入库情况有变化等，都需要重新进行完工与在制品归集，否则将会影响生产成本核算的准确性。

![](https://vip.kingdee.com/download/01092d3b935904b14682bb5a66257bda01b8.png)

在制品分配标准：即在制品的成本按照什么标准进行分配。在制品的成本分为2部分：材料成本、费用成本；系统支持对两种成本类型分别设置分配标准

【材料成本不同分配标准的适用场景】

约当产量法：当企业且半成品数量较多、变化较大时，可以用约当产量法来折算本月成本。

不计算在制品成本：本期材料成本全部计入完工产品，在制品无需分配材料成本

【费用成本不同分配标准的使用场景】

约当产量法：当企业且半成品数量较多、变化较大时，可以用约当产量法来折算本月成本。

不计算在制品成本：本期费用全部计入完工产品，在制品无需分配费用

### 【财务操作2】：确认不同费用类型的分配标准

路径：生产管理-生产成本核算-费用类型

![](https://vip.kingdee.com/download/01091e0fb764f25741f19e81804cc0806134.png)

- 支持统一设置分配标准：按完工产量/按完工入库材料成本/按工时
- 支持按车间分别设置分配标准；设置了按车间设置分配标准后，则后续费用归集时也许按照车间的维度归集费用

### 【财务操作3】：费用归集

目的是为了归集本期发生的人工费和制造费用。

一般情况下人工费（车间职工工资）和制造费用（如车间水电费、机械设备折旧等）财务人员一般先做相关凭证：

| 借：生产成本-人工费 | 借：制造费用-水电费 |
| --- | --- |
| 贷：应付职工薪酬 | 贷：预付账款 |

此时建议直接通过内部引入的方式直接将凭证金额引入

### 【财务操作6】：生成凭证

- 材料成本结转为生产成本：生产领料/退料生成凭证

| 借：生产成本-原材料 |
| --- |
| 贷：原材料 |

- 制造费结转为生产成本：费用分配生成凭证（如果在前期财务人员已经手动做过该凭证，则费用分配无需再次生成凭证，否则导致生产成本过高）

| 借：生产成本-水电费 |
| --- |
| 贷：制造费用-水电费 |

- 由于在做生产成本核算前，人工费已经手动生成凭证结转为生产成本，所以此处无需再次生成凭证，否则导致生产成本过高。
- 生产成本结转为库存商品

| 借：库存商品 |
| --- |
| 贷：生产成本-原材料       生产成本-水电费 |
