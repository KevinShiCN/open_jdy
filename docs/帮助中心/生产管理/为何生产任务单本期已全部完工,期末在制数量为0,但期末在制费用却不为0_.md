# 为何生产任务单本期已全部完工,期末在制数量为0,但期末在制费用却不为0?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：Siwennn | 阅读：258 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/611726485552087040)

【问题描述】

为何生产任务单本期已全部完工,期末在制数量为0,但期末在制费用却不为0?

![](https://vip.kingdee.com/download/01095549786dd4b2461db65d868879dd0206.png)

【解释说明】

生产成本核算后,若进行了新的费用归集和分配,而未重新核算生产成本,则可能导致此问题。

在新做了费用归集和费用分配后,需要**重新进行生产成本核算**,才能将分配到任务单上的费用核算到本期完工产品上。
