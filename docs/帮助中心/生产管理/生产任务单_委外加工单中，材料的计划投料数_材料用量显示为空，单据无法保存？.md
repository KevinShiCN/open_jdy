# 生产任务单/委外加工单中，材料的计划投料数/材料用量显示为空，单据无法保存？

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：精华汇总, 常见问题, 进阶技巧
> 作者：移动生产白建凡 | 阅读：102 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/776473859486105856)

# **1 问题描述**

- 生产任务单/委外加工单中，材料的计划投料数显示为空，单据无法保存？
- 生产任务单/委外加工单中，材料的材料用量显示为空，单据无法保存？

# 2 适用版本

星辰3.0以上专业版/旗舰版

# **3 原因及解决方案**

## **3.1 计划投料数为空**

### 3.1.1 问题原因

由于生成该产品所需的材料很少，导致维护的产品产量和材料用量之间的比例很大。实际生产时，根据材料和产品的用量关系计算出材料的应领数量小于1，且小于该材料单位的小数位精度，导致该材料的计划投料数四舍五入后为0。建议扩大该材料的单位小数位，或勾选商品信息-生产信息的【投料自动取整】。

### 3.1.2 用例说明

如图示产品A的BOM信息，生产10000cm的产品A需要1cm的材料C，单位cm的精度是2位小数

![上传图片](https://vip.kingdee.com/download/010073f62682ee9c4cf889069de07038d315.png)

当生产40cm产品A时，材料C的应发数量是0.004cm，小于单位cm的精度0.01，所以计划投料数由0.004四舍五入为0.00，也就是计划投料数为0

![上传图片](https://vip.kingdee.com/download/0100c56688645ed34a3dac9a71381d43ea7f.png)

建议扩大该材料的单位小数位，或勾选商品信息>生产信息的【投料自动取整】

![上传图片](https://vip.kingdee.com/download/0100068d3f83a1c44c03a065b787c7fb48b6.png)

此时计算出的计划投料数为应发数量0.004自动向上取整为1

![上传图片](https://vip.kingdee.com/download/0100fbde5979c7d44edc98bb4d33274cb363.png)

## **3.2 材料用量为空**

### 3.2.1 问题原因

修改【计划投料数】后，系统会自动倒算【材料用量】，当结果小于该材料的单位精度时就不显示。建议在生产任务单详情页面更多<选项中勾选【修改计划投料数量不倒算标准用量】，或重新调整BOM上该材料的材料用量和产品产量，匹配实际生产需求。

### 3.2.2 用例说明

如图示生产40cm产品A需要4cm的材料C

![上传图片](https://vip.kingdee.com/download/0100ab77a5aa19ba40e5ac93b5e6603959f8.png)

此时如果修改计划投料数为0.01，计算出的材料用量为产品产量*产品的生产数量/材料的计划投料数=10/40*0.01=0.00025，远远小于材料C单位cm的精度，四舍五入后为0，所以材料用量不显示

![上传图片](https://vip.kingdee.com/download/0100efe0489912f245feae640afa4b800521.png)

建议在生产任务单详情页面更多<选项中勾选【修改计划投料数量不倒算标准用量】，或重新调整BOM上该材料的材料用量和产品产量，匹配实际生产需求![上传图片](https://vip.kingdee.com/download/0100f1f49702802a43c9b439631eee0a5ba8.png)

勾选【修改计划投料数量不倒算标准用量】，重新新增生产任务单，此时材料用量不会自动倒算，仍为1.00

![上传图片](https://vip.kingdee.com/download/0100aed8832a15ff424887c4861099a23669.png)
