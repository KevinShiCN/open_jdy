# bom单的应用场景是什么?如何创建新增?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题

【问题描述】

bom单的应用场景是什么?如何创建新增?

【解释说明】

BOM即物料清单,也叫产品结构或配方,指物料(通常是产成品或半成品、部件)的组成情况,包括该物料由哪些下级物料组成,每一下级物料的用量是多少,其对应的属性等。

企业针对需生产的物料建立BOM,可以减少录单工作量,提升工作效率。

若生产管理和委外管理均有购买,二者的bom使用方法一样,且bom单新增一个后,另一个模块下的bom单也同时新增,两个模块bom单通用。

单张BOM单上产品信息仅支持录入一个,不支持一个BOM单录入两个或多个产品信息

BOM非必须录入单据,根据实际业务需求选择即可;

BOM单中的产品信息和子件信息不可以相同。

**1、首先作为成品或者半成品需要在商品信息开启“****可自制****”/“可委外”参数。(**应用于生产管理开启“可自制”,应用于委外管理开启“可委外”**)**
