# 旗舰版当期费用金额是0的费用类型,怎么设置不显示在成本计算表里面?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：257 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/503009550631484160)

【问题描述】

金蝶云星辰旗舰版,当期费用金额是0的费用类型,怎么设置不显示在成本计算表里面。

![](https://vip.kingdee.com/download/0109bcd85f876cd7423daeb06aee672ea5dc.png)

【解决方案】

生产管理的成本计算表中目前暂不支持直接隐藏费用为零的费用类型,如果该费用类型不使用可以直接禁用,禁用后做生产成本核算就不会显示了。

依次点击【生产管理】--【费用类型】选择不需要使用的费用类型点击【禁用】即可。

![](https://vip.kingdee.com/download/01093535a7e3ef2e4788ad7401011465e565.png)
