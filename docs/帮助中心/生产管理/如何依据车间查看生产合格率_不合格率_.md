# 如何依据车间查看生产合格率/不合格率?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题

【问题描述】
如何依据车间查看生产合格率/不合格率?

【业务场景】
需要查看不同车间的合格率与不合格率,识别出生产过程中存在的质量问题,并准确定位到具体的车间或工序。这有助于企业针对性地采取改进措施,提高产品质量

【解释说明】
**适用版本:金蝶云星辰旗舰版**

依次点击【生产管理】-【质量分析表】![](https://vip.kingdee.com/download/0109da2d930b8f31457d86580813e32f554d.png)

汇总依据选择为<生产车间>
![](https://vip.kingdee.com/download/010976184ebdfe6747d498c6d67e2c83e974.png)

即可查看不同生产车间的生产合格率/不合格率
