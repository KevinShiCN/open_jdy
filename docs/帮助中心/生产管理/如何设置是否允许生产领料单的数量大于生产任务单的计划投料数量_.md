# 如何设置是否允许生产领料单的数量大于生产任务单的计划投料数量?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：杨桂平 | 阅读：680 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/398279495003898368)

【问题描述】

如何设置是否允许生产领料单的数量大于生产任务单的计划投料数量?

【业务场景】

生产任务单做好后,发现实际发料要大于计划发料数量在不修改生产任务单的情况下,需要超额发料。可以勾选“允许领料数量大于计划投料数量”参数,勾选后领料数量允许大于计划投料数量。若企业需要严格控制不允许超额领料,则不要勾选此参数。

【操作步骤】

依次点击【系统设置】-【系统参数】-【生产参数】,根据需要勾选“允许领料数量大于计划投料数量”参数,并且可以再勾选设置“按比例控制超发”。

![](https://vip.kingdee.com/download/0109eaf61a1f45184238adc4e275a707f51e.png)

**1、不勾选****“允许领料数量大于计划投料数量”****参数**

若没有勾选这个参数,当领料数量大于应领数量时无法保存单据,系统会提示“[生产领料单 (SCLL-20251223-00001),材料分录第1行] 反写 [生产任务单 (SCRW-20251223-00001),商品分录第1行,材料分录第1行]: 数量已超过源单计划投料数量,不允许超领!”****

![](https://vip.kingdee.com/download/0109571d845f100e46c5ad00605bd367bdef.png)

**2、勾选****“允许领料数量大于计划投料数量”****参数**

若系统参数勾选了该参数,则生产任务单生成生产领料单时,允许领料数量大于应领数量(即计划投料数量)。

![](https://vip.kingdee.com/download/01093dad1932856c4d03b8eee901ccd6b8c9.png)

**3、同时勾选****“允许领料数量大于计划投料数量”****和“按比例控制超发”参数**

若系统参数中同时勾选了这两个参数,则录入生产任务单的时候可以启用“控制发料上限”功能,启用后可以录入超发比例,系统自动计算发料上限(或者输入发料上限计算超发比例)。

![](https://vip.kingdee.com/download/01092112422fba364d609702bf06b4e0153b.png)

生产任务单生成生产领料单时,领料数量大于发料上限时系统会提示“[生产领料单 (SCLL-20251223-00002),材料分录第1行] 反写 [生产任务单 (SCRW-20251223-00002),商品分录第1行,材料分录第1行]: 已执行数量超过发料上限”。

![](https://vip.kingdee.com/download/010924b816cd21794081a5094632280bc220.png)

【温馨提示】

若生产任务单的商品信息中看不到“控制发料上限”、“超发比例”、“发料上限”这些字段,则需检查系统参数是否勾选了“按比例控制超发”,勾选该参数才会显示这些字段;或者点击右上角“列设置”检查是否隐藏了,显示出来即可。

![](https://vip.kingdee.com/download/0109222d6da95e3c46e7a6abd2435450ea77.png)
