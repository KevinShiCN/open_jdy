# **1 简介  **

### **1.1 功能介绍**

产品入库明细表展示了不同仓库、部门、物料等的产品入库及退库明细的情况**。**

### **1.2 适用版本**

金蝶云·星辰专业版

### **1.3 应用场景**

仓管员可通过产品入库明细表掌握不同仓库、部门、物料等的产品入库及退库明细的情况

### **1.5 取数说明**

产品入库明细表是按所选的时间段按单据来统计过滤期间内所有的产品入库和退库数据。产品入库单的数量和入库成本显示为正，产品退库单的数量和入库成本显示为负。

取数规则：根据筛选条件中设置的单据日期范围、交货/收货部门等等条件过滤出对应的数据。

### **1.6 权限说明**

该报表受数据权限控制，即若用户无某物料、部门、仓库、职员的数据权限，则对应的产品入库和退库数据在该报表中无法查询。

# 2 联查产品入库材料成本明细表

点击“直接材料成本”下的蓝色数字，可以联查[产品入库材料成本明细表](https://vip.kingdee.com/link/s/Z1tXf)，了解当前产品入库单/产品退库单中各项产品耗用材料的数量及成本
