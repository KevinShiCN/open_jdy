# 生产任务单还未保存,为什么不允许修改业务类型?

> 作者：云社区用户55os622 | 阅读：44 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/650144119758735360)

【问题描述】
生产任务单还未保存,为什么不允许修改业务类型?
![](https://vip.kingdee.com/download/01091ab5b3b50bd943ef8022fbd7984efd54.png)【解释说明】

 生产任务单选择了商品后,业务类型则不支持再修改,如需修改业务类型,需要先删除录入的商品信息,才能修改业务类型字段。![](https://vip.kingdee.com/download/0109738d5a969eb940158354d4c44829635a.png)
