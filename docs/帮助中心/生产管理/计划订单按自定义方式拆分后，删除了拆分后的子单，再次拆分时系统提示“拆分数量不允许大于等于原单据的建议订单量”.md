# 计划订单按自定义方式拆分后，删除了拆分后的子单，再次拆分时系统提示“拆分数量不允许大于等于原单据的建议订单量”

# **1 问题描述**

计划订单按自定义方式拆分后，删除了拆分后的子单，再次拆分时系统提示“拆分数量不允许大于等于原单据的建议订单量”

![上传图片](https://vip.kingdee.com/download/0100cbbcb7dccb1d4f86a9e2f9e56c4df9b9.png)

# 2 适用版本

星辰3.0及以上专业版/旗舰版

# **3 原因解析**

计划订单的拆分是针对“建议订单量”进行拆分。计划订单拆分后，原单的**建议订单量会相应减少**；但删除拆分生成的子单后，**原单的建议订单量不会自动还原**，因此无法再次按原数量拆分。

# 4 示例说明

如图示计划订单尚未拆分，建议订单量 = 计划订单量 = 100

![上传图片](https://vip.kingdee.com/download/0100ff0b4208c5c74f8fb31a9c6b7151ce28.png)

此时自定义拆分50到新的单据

![上传图片](https://vip.kingdee.com/download/01002b95ade321294819a58e76d4b2c61932.png)

拆分后，子单和原单的建议订单量各为50

![上传图片](https://vip.kingdee.com/download/01003d0e06c6e5a241f6931ff3e48c53b132.png)

删除这张子单，原单的建议订单量仍为50，不会自动恢复为100

![上传图片](https://vip.kingdee.com/download/0100d6c9d9d12be34db78d919028cb950654.png)

再次拆分时，因原单的可用建议订单量不足，系统弹出中断提示

![上传图片](https://vip.kingdee.com/download/0100cbbcb7dccb1d4f86a9e2f9e56c4df9b9.png)

# 5 解决方案

删除拆分子单后如需重新拆分：

- 先修改**原计划订单的建议订单量 = 计划订单量**，将数量还原
- 再重新执行自定义拆分操作即可
