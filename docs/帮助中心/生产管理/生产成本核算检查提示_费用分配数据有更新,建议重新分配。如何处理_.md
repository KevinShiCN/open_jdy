# 生产成本核算检查提示:费用分配数据有更新,建议重新分配。如何处理?

> 作者：xc_jdy_xue | 阅读：36 | 有用：1

【问题描述】

生产成本核算检查提示:费用分配数据有更新,建议重新分配。如何处理?

![](https://vip.kingdee.com/download/010948152eddfbb34476b76a5377fdc6e3e8.png)

【解释说明】

当检测到完工与在制品有数据变动,比如新产生了产品入库/退库等等,会出现这个提示,需要去完工与在制品归集这个功能里重新确认保存一下

![](https://vip.kingdee.com/download/01098a38ae4cc9154cd780309a5d6c8f61a9.png)

![](https://vip.kingdee.com/download/0109f9abe04f70314d8c90d09dee3a8329a4.png)

然后再到费用分配中重新进行分配

![](https://vip.kingdee.com/download/010980c99eba5b294dd19960a77a015f065e.png)
