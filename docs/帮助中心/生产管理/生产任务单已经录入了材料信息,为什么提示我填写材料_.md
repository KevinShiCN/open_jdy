# 生产任务单已经录入了材料信息,为什么提示我填写材料?

> 作者：Siwennn | 阅读：110 | 有用：0

【问题描述】

生产任务单已经录入了材料信息,为什么提示我填写材料?

![](https://vip.kingdee.com/download/01092df513c021854ec688f713e4cb55539d.png)

【解释说明】

由于生产任务单包含多行产成品信息,每行产成品都需要单独点击并录入相应的材料信息。

![](https://vip.kingdee.com/download/01095c158fa83ea846e09812c6d8e47183e5.png)
