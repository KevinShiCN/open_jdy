# 联副产品如何使用?

【问题描述】

联副产品如何使用?

![](https://vip.kingdee.com/download/0109b2bb6d86147a46299305095577d6a00c.jpg)[视频观看](https://vip.kingdee.com/school/detail/723994518977860864?productLineId=35&lang=zh-CN&cosmic_type=video)

【适用版本】

金蝶云星辰旗舰版

【业务场景】

如酿酒厂生成酒的过程中,联产的酒糟可加工为饲料,用于养殖业

屠宰场生产的牛肉,同时牛皮、牛内脏均可直接销售给不同客户(食品厂、皮革厂)

乳制品厂从牛奶中分离出奶油、奶酪、脱脂乳,奶油可进一步加工为蛋糕原料,奶酪可制成芝士片

煤化工中联产的焦炭、煤气、煤焦油,煤焦油可深加工为沥青、染料等精细化工产品

钢铁厂炼钢时产生的高炉煤气,可作为企业内部供热或发电的燃料

【解释说明】

联副产品功能主要应用于一次投料可以产出多种成品的场景,主要是在生产主产品的过程中,同时联产出其他产品。

**联副产品**是指在同一生产过程中,伴随主产品生产而附带产生价值的产品。

**1、启用联副产品管理**

在【系统设置】>【系统参数】进入系统参数设置页面

![](https://vip.kingdee.com/download/010950568b4924b448a6a73bd4f61796877c.png)
系统参数设置页面,选择【生产参数】或者【委外参数】勾选“启用联副产品管理”。

![](https://vip.kingdee.com/download/0109922c2048cebd44b9bf0e7c0814dc4f9e.png)

启用联副产品管理参数需注意

①参数设置保存启用后,需刷新页面,系统会显示联副产品的相关功能和字段

②一旦存在相关业务单据,比如BOM、生产任务单、委外加工单中录入了联副产品信息,该参数不可取消

③生产和委外的联副产品参数是共用的,只能同时启用和同时取消。

④若当期无主产品入库,仅有联副产品入库时,联副产品的成本暂不支持进行成本核算。

**2、启用联副产品管理后,相关单据操作流程**

①在BOM分为主产品信息、联副产品信息、子件信息显示

在联副产品信息中,可以维护联副产品以及对应的产出数量,成本分配方式、成本权重、单位定额成本等信息,均为非必录

**注意:联副产品需在商品特性中开启 “可自制” 或 “可委外”**

![](https://vip.kingdee.com/download/0109a9727c81ff9e49a19bebdc8c9d9b685a.png)

②在生产任务单中,会根据主产品的BOM,自动携带出联副产品以及子件材料

![](https://vip.kingdee.com/download/0109bae73ac4016048a79d036e50c5380c10.png)
若BOM中没有维护联副产品信息,在生产任务单上也可以手动添加联副产品,适用于前期并不确定会产出哪些联副产品,在实际生产时才能确定的场景。

![](https://vip.kingdee.com/download/01091d05984018b3465cb89b9003c3b386fb.png)

若已发料,则可以通过任务单变更功能来添加联副产品,操作详情可查询[生产任务单如何操作变更](https://vip.kingdee.com/knowledge/581648019137596160?channel_level=%E9%87%91%E8%9D%B6%E4%BA%91%E7%A4%BE%E5%8C%BA%7C%E6%90%9C%E7%B4%A2%7C%E5%AE%98%E6%96%B9%E7%9F%A5%E8%AF%86&productLineId=35&isKnowledge=2&lang=zh-CN)

**注意:由于材料是挂在主产品下面,所以只有主产品支持下推领料单、预留材料、配套领料、倒冲领料。联副产品不支持这些操作**

③在生产领料单中显示的领用材料信息,对应的商品信息仅会显示主产品,无法显示联副产品信息
![](https://vip.kingdee.com/download/0109f988117db7db4de0a2a298f3709c5a10.png)
④在产品入库单允许主产品和联副产品分别入库或一起入库。

**(注意:若当期无主产品入库,仅有联副产品入库时,联副产品的成本暂不支持进行成本核算****)**

![](https://vip.kingdee.com/download/0109d2cd6666683c400285a60fe62016ce5d.png)
