# 生产在制量和生产占用量的区别?

【问题描述】

生产在制量和生产占用量的区别?

【解释说明】

1、一般名词解释

‌生产在制量‌:指企业当前正在生产过程中但尚未完成的产品数量,属于在制品库存;‌

生产占用量‌:指企业为生产活动占用的资源(如原材料、设备等)数量,反映生产过程中的资源消耗情况。

两者的核心区别在于:在制量关注未完成产品的数量,占用量关注生产资源的消耗品原材料的数量。

2、星辰系统生产管理模块对应的数据来源:

生产在制量(基本):已审核生产任务单“商品信息”栏中的商品基本数量;

生产占用量(基本):已审核生产任务单“材料信息”栏中的物料基本数量
