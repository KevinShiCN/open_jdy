# 产品入库单支持下推销售出库单

# **1 简介  **

## **1.1 功能介绍**

此功能适用于产品加工完成入库后直接销售出库的场景。

## **1.2 适用版本**

星辰专业版/旗舰版

## **1.3 按钮说明**

产品入库单列表、详情增加【销售出库】按钮。

# **2 操作指南**

1、在产品入库单单据列表选择需要出库的单据，点击【销售出库】按钮，则按单生成销售出库单；点击【销售合并出库】，则按所选的产品入库单分录合并生成销售出库单。

- 销售出库单的单据选项处支持设置产品入库单下推销售出库单时的字段携带项，支持设置携带单据日期、交货部门。
- 如产品入库单存在关联的销售订单或受托加工订单，则不支持通过产品入库单直接下推销售出库单，需要通过销售订单或委外加工订单下推销售出库单。

![上传图片](https://vip.kingdee.com/download/010073a2cbe89dce480f8c3142ed3ae8461f.png)

2、产品入库单生成销售出库单时，携带“数量-退库数量-已出库数量”到销售出库单的数量字段，保存销售出库单，反写出库数量到产品入库单的行已出库数量字段，不允许超量出库。

![上传图片](https://vip.kingdee.com/download/0100a933b4bf9c314ca3b54ae414371dce83.png)
