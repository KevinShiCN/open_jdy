# 生产任务变更单如何使用?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：ylim | 阅读：24 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/816837938360612352)

【问题描述】

生产任务变更单如何使用?

【业务场景】

当原生产任务因订单调整、物料短缺、工艺优化、设备故障等原因,需要变更生产数量、交期、用料清单、生产车间等内容时,可发起此单据。单据支持[工作流](https://vip.kingdee.com/link/s/Zr0Xm)和[消息设置](https://vip.kingdee.com/link/s/Zr0XP)。

**核心价值:**
- **①规范变更流程**:支持**多级审批**,确保变更需求经过对应权限岗位审核,避免随意调整。
- **②保障信息同步**:配套**消息推送**功能,自动通知生产、仓库、采购等相关协同方,确保执行口径一致。
- **③留存追溯依据**:记录变更原因、内容、审批轨迹,为生产复盘提供可查凭证,避免车间盲目生产。
- 【适用版本】
- 星辰.旗舰版
- 【操作指引】
- 点击【生产管理】>【生产单据】>【生产任务变更单】
- ![](https://vip.kingdee.com/download/0109a21fc92e38ee48329c060875b27bd410.png)支持变更生产任务单的“商品信息”、“材料信息”、“基本信息”三部分内容![](https://vip.kingdee.com/download/01093d949bcf5e9f4001ba49b331f5a1a976.png)**1、商品信息**
- ①支持对当前期间及以后期间的生产任务单添加商品。(以前期间的生产任务单不支持添加商品)
- ②启用联副产品后,支持添加新的联副产品行。
- ③支持修改生产任务单的现有商品行(含主产品和联副产品),支持修改数量、计划开工日期和完工日期、商品行备注等信息。修改时,可对比变更前和变更后的内容,变更前内容仅查看不可编辑。
- ④支持删除商品行。
- ![](https://vip.kingdee.com/download/010949d30288f2f74b3d9c9280969d1334e7.png)**2、材料信息**
- ①支持新增子件,支持对多个任务单同时增加子件。
- ②支持修改子件,支持修改用量、发料方式、仓库等信息;修改时,可对比变更前和变更后的内容,变更前内容仅查看不可编辑。
- ③支持删除子件,仅支持删除没有下游关联单据的材料行,可以同时对多个任务单删除多个材料行。
- (变更主料和替代料的时候,需要主料和替代料一起变更。比如删除主料时,则关联的替代料会一起返回至变更单并一起进行删除。)
- ![](https://vip.kingdee.com/download/0109ee249ab5e2c44d059972683e63d16593.png)

**3、基本信息**
- ①支持对已审核未完工的生产任务单变更生产车间。
- ②支持变更源单单据备注。
- ![](https://vip.kingdee.com/download/01096862db4912f0499194cd2e7eda365e7c.png)单据录入完成后保存审核即可。**需注意:单据审核成功后便更新原有生产任务单,已审核的生产任务变更单不允许反审核及删除!**
