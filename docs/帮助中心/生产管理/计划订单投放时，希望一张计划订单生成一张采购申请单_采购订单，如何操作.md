# 计划订单投放时，希望一张计划订单生成一张采购申请单/采购订单，如何操作

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：新特性, 常见问题, 快速入门
> 作者：移动生产白建凡 | 阅读：3 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/821327127415785984)

# **1 问题描述**

计划订单投放时，希望一个商品生成一张单据，如何操作

# 2 适用版本

星辰3.0及以上专业版/旗舰版

# **3 解决方案**

计划订单投放时，在合并投放规则设置中，勾选一个商品生成一张目标单据的参数

支持的单据范围：采购申请单、采购订单、生产任务单、委外加工单
