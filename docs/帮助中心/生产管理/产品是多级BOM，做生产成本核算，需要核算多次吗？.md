# 产品是多级BOM，做生产成本核算，需要核算多次吗？

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题, 快速入门, 精华汇总
> 作者：移动生产白建凡 | 阅读：91 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/772131083248313344)

# **1 问题描述**

如果商品有5级BOM，做生产成本核算需要核算5次吗?

# 2 适用版本

星辰3.0以上专业版/旗舰版

# **3 解决方案**

不需要核算5次，核算1次就可以

 - 如果每级BOM都需要自制生产，核算1次即可

 - 如果半成品是委外生产，可以用生产成本核算中的“智能核算”功能，核算1次即可。点击查看[智能成本核算功能](https://vip.kingdee.com/link/s/lwUNg)
