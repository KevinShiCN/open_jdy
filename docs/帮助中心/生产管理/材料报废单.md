# 材料报废单

# **1 简介  **

## **1.1 功能介绍**

本次新增了材料报废单，用于记录车间/供应商处材料的报废数量及原因，以及需要重新发料的数量。便于企业①统计材料报废原因，改进生产过程；②根据报废数量控制补发数量，对材料领用进行严格控制。

材料报废的业务流程如下图所示（委外材料报废流程与生产一致）：

## **1.2 适用版本**

星辰3.0以上专业版/旗舰版

# **2 操作指南**

1）当仅需要记录报废，不关注哪个工单的材料时，可直接手动录入报废的材料、数量及报废原因；（报废单非库存单据，报废单保存/审核不影响即时库存）

①当业务类型选择生产时，记录的是车间报废的材料；当业务类型选择委外时，记录的是供应商处报废的材料；

![上传图片](https://vip.kingdee.com/download/0100c510c2446cd44ed2b12d2b142cba1559.png)

②若报废材料需要入库，则商品分录的“是否入库”选择“是”（默认为“否”）；材料报废单即可下推其他入库单，其他入库单的业务类型为“生产报废入库”或“委外报废入库”

![上传图片](https://vip.kingdee.com/download/0100d6a24cf853ec4fedb5e0e44e55ab80f7.png)

![上传图片](https://vip.kingdee.com/download/0100af74f70df96741aa9569eaacb774684b.png)

③材料报废单-单据选项，可设置材料报废入库的默认仓库；材料报废单优先携带选项中的仓库。

![上传图片](https://vip.kingdee.com/download/01001f192bb4abaf484e8cdd8658983e7c58.png)

④无源单新增时，不允许录入报废重发数量

2）当需要关联源单进行报废时，可通过选单进行报废。业务类型选择“生产”时，源单类型为“生产领料单”；业务类型选择“委外”时，源单类型为“委外发料单”；

![上传图片](https://vip.kingdee.com/download/0100209084ec60aa46aaa25e9fc5abb33d8d.png)

![上传图片](https://vip.kingdee.com/download/0100c7d124effa644e1fbc6fdac42ef46838.png)

3）若报废的材料后续需要继续发料，则选择的源单必须关联未关闭的生产任务单/委外加工单；选单后录入“报废重发数量”，审核材料报废单后，生产任务单/委外加工单的未领数量会根据报废重发数量增加，从而继续发料。

![上传图片](https://vip.kingdee.com/download/01003d26184668db4f56ab5e0bb46a44e7fd.png)

①报废重发数量不允许大于报废数量

![上传图片](https://vip.kingdee.com/download/0100513814358e5b4eb0b47bb513d0232b4e.png)

②领料单中退料数量+报废数量不能超过领料数量，即已经报废的材料不能再退料，已经退料的材料也不能再记录报废

![上传图片](https://vip.kingdee.com/download/010088fc9f006a2348e6a0ac37a217d7aa64.png)

领料单数量12个，报废了2个，则继续下推退料单时可退数量自动扣减已经报废的数量

![上传图片](https://vip.kingdee.com/download/0100684113d549ff4f1d9c5568fda3cb9929.png)

③若选择的“生产领料单”/“委外发料单”关联的工单已经关闭，则不允许录入报废重发数量

![上传图片](https://vip.kingdee.com/download/0100cd9f0c5dca0a4ca9878c93c0152b4aff.png)

④倒冲领料单不支持下推材料报废单

![上传图片](https://vip.kingdee.com/download/0100db1cfdec36c2418296741a67bb499aef.png)

4）新增材料报废统计表，支持按不同汇总依据汇总报废数量
