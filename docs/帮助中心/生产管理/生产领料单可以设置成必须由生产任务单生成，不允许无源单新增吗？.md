# 生产领料单可以设置成必须由生产任务单生成，不允许无源单新增吗？

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：快速入门, 常见问题
> 作者：嘻嘻嘻哈 | 阅读：948 | 有用：8
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/504245123769590016)

# **1 问题描述**

能否将生产领料单配置为“仅允许通过生产任务单下推生成”，禁止用户直接新增无源单的生产领料单？

# **2 解决方案**

支持该配置需求，通过以下两种方案均可实现

## **2.1 业务流程配置（推荐，源头控制）**

进入“系统设置>个性化设置>业务流程设置”页面，找到库存管理下的生产领料单，支持配置生产任务单为必选的源单类型。详见公告：[2025-12-30 金蝶AI星辰（生产云）更新公告：新增材料报废单、委外管理支持折扣](https://vip.kingdee.com/link/s/Z4sdR)

![上传图片](https://vip.kingdee.com/download/0100a9e08693475f464fa0a6ba0214593948.png)

保存配置后，生产领料单仅能通过生产任务单下推生成，无法直接新增无源单单据

## **2.2 单据字段配置（辅助控制）**

进入“系统设置>个性化设置>单据字段配置”，找到生产领料单，将【材料分录.源单单号】字段设置为必录。保存配置后，新增生产领料单时必须录入源单单号（即关联生产任务单），否则无法保存或审核单据，间接禁止无源单新增
