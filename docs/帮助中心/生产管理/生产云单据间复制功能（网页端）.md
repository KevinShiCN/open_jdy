# 生产云单据间复制功能（网页端）

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：快速入门, 精华汇总, 常见问题
> 作者：移动生产白建凡 | 阅读：365 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/588424724569177088)

## 1 应用场景

生产云不同单据间直接复制，实现快速录单

## 2 适用版本

星辰3.1版本的网页端

## 3 适用单据

金蝶云·星辰生产云目前已支持的单据间复制类型如下：

| 模块 | 原单据名称 | 支持复制为的单据名称 |
| --- | --- | --- |
| 生产管理 | 生产任务单 | BOM单 |
| 生产领料单 | 采购申请单采购订单采购入库单其它入库单其他出库单调拨出库单移仓单 |  |
| 产品入库单 | 销售订单销售出库单其他出库单调拨出库单移仓单 |  |
| 委外管理 | 委外加工单 | BOM单 |
| 委外发料单 | 采购申请单采购订单采购入库单其他入库单其他出库单调拨出库单移仓单 |  |
| 委外产品入库单 | 销售订单销售出库单其他出库单调拨出库单移仓单 |  |
| 受托加工 | 受托加工订单 | 其他入库单产品入库单 |

注：以上信息更新节点截止至2024年07月25日。
