# bom子件末级反查是如何使用的?

【问题描述】

BOM子件末级反查是如何使用的?

【解释说明】

BOM子件反查会从下往上扫描所有级次的产品结构,列出配件的每一个父件以及父件的父件,直至最终成品。

BOM子件末级反查是指按材料查询以后,只显示最末级的父件,无法再向上追溯。
