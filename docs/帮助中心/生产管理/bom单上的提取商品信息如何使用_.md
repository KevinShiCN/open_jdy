# bom单上的提取商品信息如何使用?

> 产品：金蝶云·星辰 | 分类：生产管理 | 类型：常见问题
> 作者：shaoling | 阅读：93 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/725153832958966784)

【问题描述】

星辰BOM单上的提取商品信息如何使用?

【解释说明】

BOM单上的提取商品信息是指在商品基础资料里更新默认仓库 / 仓位,以及是否倒冲领料、倒冲仓库 / 仓位等信息后,在 BOM 单列表及BOM单据界面,点击 “提取商品信息”按钮,即可将商品基础资料中更新的内容同步到BOM单上。
