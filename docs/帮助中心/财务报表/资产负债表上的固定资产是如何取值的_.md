# 资产负债表上的固定资产是如何取值的?

> 作者：红笺小字 | 阅读：297 | 有用：0

【问题描述】

资产负债表上的固定资产是如何取值的?

【取数说明】

资产负债表上的固定资产取的是固定资产科目的余额,数据来源于凭证的录入,核对数据可到科目余额表查询。报表点击编辑公式后可以查看取数科目设置。
