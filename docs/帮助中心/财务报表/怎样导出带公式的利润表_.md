# 怎样导出带公式的利润表?

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：常见问题
> 作者：云社区用户u1tk1853 | 阅读：410 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/626221889429716736)

【业务场景】

怎样导出带公式的利润表?

【解决方案】

利润表不支持直接导出带公式的表格,可以将利润表另存为自定义报表,然后导出自定义报表。

1、将利润表另存为自定义报表

打开【财务报表】--【利润表】点击右上角【另存为】输入“编码”和“名称”点击<确定>。

![](https://vip.kingdee.com/download/01093cc5fd4d35084cafa7c52ff3ee4b927d.png)

点击【确定】后会自动打开自定义报表界面,点击右上角【保存】。

![](https://vip.kingdee.com/download/0109847f1f3405dd4f9ca516dbe771f8405d.png)

2、导出自定义利润表

依次点击【财务报表】--【自定义报表】打开上一步保存的利润表,点击右上角【引出】即可。

![](https://vip.kingdee.com/download/0109ffd72f3afd9d42f38ff1c86e9eb0dd25.png)
