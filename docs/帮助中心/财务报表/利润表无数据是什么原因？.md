# 利润表无数据是什么原因？

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：常见问题, 进阶技巧
> 作者：简天 | 阅读：3400 | 有用：15
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/213604014066120960)

问题描述：

解决方案：

如果录入了会计凭证，利润表却无数据，多数情况都是因为当月的期末结转损益凭证为手工录入或第三方导入数据。解决方法：

1、删除结转损益凭证，如果以前期间的利润表也没有数据，需要反结账，删除所有期间的结转损益凭证。

2、在【账务处理】-【期末处理】-【期末结转】中生成“结转损益”凭证，再查询利润表。
