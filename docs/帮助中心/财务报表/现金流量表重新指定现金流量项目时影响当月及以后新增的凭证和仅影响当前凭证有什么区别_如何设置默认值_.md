# 现金流量表重新指定现金流量项目时影响当月及以后新增的凭证和仅影响当前凭证有什么区别?如何设置默认值?

【问题描述】

现金流量表重新指定现金流量项目时影响当月及以后新增的凭证和仅影响当前凭证有什么区别?如何设置默认值?

若您想要了解现金流量表数据不对问题,请点击:[现金流量表数据不对时怎么调整?](https://vip.kingdee.com/knowledge/391351644153588736?productLineId=35&isKnowledge=2&lang=zh-CN)

【解释说明】

**影响当月及以后新增的凭证**:修改后将影响当月已有现金银行相关凭证上该科目的现金流指定项,及后续新增现金银行类凭证科目该科目的现金流项目

**仅影响当前凭证**:修改仅针对所选凭证的现金流量项目进行调整,不会影响当月及后续期间其他凭证的流量项目指定,适用于个别凭证的修正需求

**注意:直接在录凭证界面主动指定现金流时,仅支持选择影响当前的凭证**

录凭证指定现金流操作:[录凭证时可以指定现金流吗,能否设置相关凭证必须提醒让录入?](https://vip.kingdee.com/link/s/Zq9u0)

![](https://vip.kingdee.com/download/01098cce5ba4ca464c27851ad586f830f242.png)
若需默认某个科目后续均指定到哪个现金流项目,设置时选择“**影响当月及以后新增的凭证**”
