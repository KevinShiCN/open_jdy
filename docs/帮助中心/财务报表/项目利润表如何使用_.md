# 项目利润表如何使用?

【问题描述】

项目利润表如何使用?

![](https://vip.kingdee.com/download/01097579061953f24f1b95dbd37371cb4865.jpg)[观看视频](https://vip.kingdee.com/link/s/ZJNhE)

【适用版本】

适用于星辰标准版、专业版、旗舰版

【业务场景】

项目利润表作为企业财务分析的重要工具,能够全面且系统地呈现企业各考核单元的收入状况、费用支出以及利润构成详情。需要强调的是,只有当损益类科目启用辅助核算项目的设置后,方可精准、高效地生成与之匹配的项目利润表。

项目利润表支持按部门/职员/客户/供应商等辅助核算类别查询,如项目利润表需要按“项目”出具项目利润表,可按如下步骤操作:

1、设置自定义辅助资料“项目”

依次点击【基础资料】-【业务资料】-【辅助资料】界面,点击左侧“编辑”按钮

![](https://vip.kingdee.com/download/01097075d48b7283448f83a7ee9fe4e4281c.png)

点![](https://vip.kingdee.com/download/01096a4415ab850d4fcaa0113a14ef65f9c1.png)新增项目大类
![](https://vip.kingdee.com/download/0109fae97d34b7944880a7d176b8e19633ff.png)

选中新增的项目大类,点击右上角新增按钮新增项目明细

![](https://vip.kingdee.com/download/010905337f03f66e4aada1d97a77036f26b6.png)

2、将损益类科目设置“项目”辅助核算

依次点击【基础资料】-【财务资料】-【科目】,将损益类科目设置“项目”辅助核算

![](https://vip.kingdee.com/download/0109f8a1d5dd016f44c292f913cd1fe6f10b.png)

3、录入凭证

在【账务处理】-【凭证】-【录凭证】中录入项目相关的业务凭证。

![](https://vip.kingdee.com/download/010958b398817eb24556b64a201bfa4ad140.png)

4、月末生成结转损益凭证

在【账务处理】-【期末智能结转】中结转损益。

![](https://vip.kingdee.com/download/0109b91c8ad3ce664a4f9cb184410759c641.png)

5、查看项目利润表

打开【财务报表】-【项目利润表】

![](https://vip.kingdee.com/download/01097faddc4a18dd4042b45d2e0111fcd977.png)

辅助核算类别选择“项目”点击查询即可。

![](https://vip.kingdee.com/download/010951009dc7883a4768b80227487297f518.png)

点击“会计期间”可以修改要查看的报表期间

![](https://vip.kingdee.com/download/010960b6262047b342f496c280ffec59c20f.png)

“时间类型”可以选择季报查询

![](https://vip.kingdee.com/download/0109c044e0b2e97e4470b2899f74f13df2b8.png)

【温馨提示】

**1、项目利润表能够依据客户、部门、职员、项目等多样化的辅助核算类别进行灵活查询。具体而言,损益类科目所设置的辅助核算类型,决定了项目利润表将按照何种维度进行查询与呈现。**

**2、项目利润表的公式设定取自于标准利润表,其核心数据均源自损益类科目,只不过是以项目为维度进行细致拆分与独立列示。如果标准利润表在报表公式、报表格式等方面做出调整,项目利润表也会同步更新。**
