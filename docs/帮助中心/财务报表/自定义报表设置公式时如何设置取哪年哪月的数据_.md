# 自定义报表设置公式时如何设置取哪年哪月的数据?

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：常见问题
> 作者：杨桂平 | 阅读：1079 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391351576776094720)

【问题描述】

自定义报表设置公式时如何设置取哪年哪月的数据?

【解释说明】

自定义报表设置ACCT的取数公式时需要设置会计年度和会计期间,报表查询时取值的会计年度及期间对应:

默认为0,表示查询报表时系统参数中对应的当前会计年度和期间。

也可以录入-1表示上年、-2表示上上年,正数表示具体年度
![](https://vip.kingdee.com/download/01096d7f2e4b974f4441b4292adc4c192e03.png)
会计期间也可以录入-1表示上月(上季)、-2表示上上月(上上季),正数表示具体月份、季度。

![](https://vip.kingdee.com/download/01097a7c216e8b11475baec530c0e18d1443.png)

【温馨提示】

自定义报表取科目的期末余额的时候(ACCT(Y))的时候,只能取到当前财务期间及以前期间的数据,当前财务期间以后期间的数据是取不到的,若需要取后续期间的余额,请先对财务进行结账。
