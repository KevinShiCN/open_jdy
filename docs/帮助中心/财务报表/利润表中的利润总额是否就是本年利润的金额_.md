# 利润表中的利润总额是否就是本年利润的金额?

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：常见问题
> 作者：shaoling | 阅读：77 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/716096827920919808)

【问题描述】

利润表中的利润总额是否就是本年利润的金额?

【解释说明】

利润表中的利润总额**不一定**等于"本年利润"科目的金额。通常情况下,若企业不存在所得税费用(含当期所得税与递延所得税)及其他调整事项(如以前年度损益调整等),则:**本年利润 = 利润表中的净利润 = 利润总额 - 所得税费用**

【温馨提示】

**利润总额:**利润表中反映企业本期经营活动(含营业外收支)的盈亏总额,尚未扣除所得税费用。

**所得税费用:**企业根据应纳税所得额计算的当期所得税支出,需从利润总额中扣除。
**净利润:**利润总额扣除所得税费用后的净额,即企业本期实际归属于所有者的盈利(或亏损),也是利润表最后一列的 “净利润” 项目金额。

**本年利润科目:**作为所有者权益类科目,期末需将所有损益类科目(包括所得税费用)结转至该科目,最终余额即为净利润(贷方余额)或净亏损(借方余额),与利润表 “净利润” 金额一致。
