# 新增科目映射时报错:最明细账套/不参与财务合并的账套,不支持新增科目映射,该怎么处理?

> 作者：Schatz | 阅读：16 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/794370451891480576)

【问题描述】

新增科目映射时报错:最明细账套/不参与财务合并的账套,不支持新增科目映射,该怎么处理?

![](https://vip.kingdee.com/download/01099e33b47db8ae47b4a88a9c99003a5e31.png)

【解释说明】

依次点击【系统设置】-【组织机构设置】,对以下两个关键设置项展开核查:

其一,检查是否存在其他账套将该账套设定为其上级组织的情况;

其二,确认该账套在组织架构中是否已开启“参与财务合并”功能。

![](https://vip.kingdee.com/download/0109ff9ba1d4375a411ba0d95c22e5391540.png)

需同时满足该账套为非明细账套,且已开启“参与财务合并”这两个条件,方可在该账套中新增科目映射。
