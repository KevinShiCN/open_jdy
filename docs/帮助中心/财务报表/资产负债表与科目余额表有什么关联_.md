# 资产负债表与科目余额表有什么关联?

【问题描述】

资产负债表与科目余额表有什么关联?

【解释说明】

资产负债表的公式设置中,**取数科目的“余额”**数据。

![](https://vip.kingdee.com/download/01095013ac080af74689b5bcb732c37da45a.png)

科目余额数据源自**“科目余额表”**,因此报表展示的是该表中各科目的数据。
