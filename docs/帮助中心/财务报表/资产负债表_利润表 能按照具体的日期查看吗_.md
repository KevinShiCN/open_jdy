# 资产负债表/利润表 能按照具体的日期查看吗?

【问题描述】
资产负债表/利润表 能按照具体的日期查看吗?

【解释说明】

资产负债表和利润表通常是按照一定的时间周期(如月度、季度或年度)查询,反映的是特定期间的财务状况和经营成果。无法选择中间某个具体的日期进行查询。[报表查询指南](https://vip.kingdee.com/article/585554302228446976?productLineId=35&isKnowledge=2&lang=zh-CN)
