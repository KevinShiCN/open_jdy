# 修改报表有次数限制吗?

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：常见问题
> 作者：happy忆霖 | 阅读：38 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/725889453817752832)

【问题描述】

修改报表有次数限制吗?

【业务场景】

现金流量表当月调整完成后,发现还需要调整,是否可以多次进行调整在一个月之内?

【解释说明】

**资产负债表、利润表、现金流量表、自定义报表等修改报表公式/修改报表项目皆无次数限制,可以随时进行修改。**
