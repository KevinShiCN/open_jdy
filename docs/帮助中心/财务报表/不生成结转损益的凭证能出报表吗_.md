# 不生成结转损益的凭证能出报表吗?

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：常见问题
> 作者：ylim | 阅读：553 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/557730108086387968)

【问题描述】

不生成结转损益的凭证能出报表吗?

【解释说明】

期末不结转损益,不影响报表查看,报表会自动取到科目发生额

首先在【系统设置】-【系统参数】-【财务参数】,检查<期末结账前必须结转损益>这项参数是否勾选,勾选了则必须结转损益才能结账,不勾选可不结转当期损益直接结账
