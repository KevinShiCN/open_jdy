# 查询项目利润表时可以显示合计吗?

> 作者：杨桂平 | 阅读：162 | 有用：0

【问题描述】

查询项目利润表时可以显示合计吗?

【操作流程】

打开项目利润后在筛选条件中启用“显示合计列”后点击查询即可查到合计列。
