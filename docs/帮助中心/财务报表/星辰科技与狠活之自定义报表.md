# 星辰科技与狠活之自定义报表

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：进阶技巧, 知识软文
> 作者：七戒大师兄 | 阅读：3036 | 有用：8
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/368696727660569344)

![](https://vip.kingdee.com/download/010922c13ad2c54645e7a6b741ede18ccd57.png)

      时代在进步，科技在发展，软件在为企业提供数治战斗力。

      随着东海龙王靠水吃水的水产生意越做越红火，公司对于财务数据的要求也越来越高。最近东海龙王想看一份财务数据，而系统原有的财务报表（资产负债表、利润表、现金流量表等）无法直观的查询出来。作为公司财务总监的小龙女，只好手工查询出相关数据后汇总，忙得她是焦头烂额的。就这样忙了一个月后，小龙女忽然想起了老邻居斗战圣佛上次帮她解决了复制单据的问题，想着再去向大圣取取经。

      念及行至，果断前往了花果山水帘洞。大师兄听完小龙女“特殊报表”需求的后，笑道：“这有何难，目前星辰的科技与狠活完全支持你现在说的这个需求，请看财务报表中【自定义报表】”

      1、 首先，在财务报表——自定义报表中，点击右上角的【新增】按钮，然后填写报表的编码和名称，保存后进入编辑模板的界面；

![](https://vip.kingdee.com/download/01007592c4d576614d0b840a826e0e36d710.png)

![](https://vip.kingdee.com/download/0100896745ac02f84f78bf6751ba7666135e.png)

![](https://vip.kingdee.com/download/01004725e5b4cb334a4c9ca0d8cd81cccd31.png)

      2、 制作表格样式两种方式：一是参照Excel表格的方法，直接在系统中手工编制表格的样式和文字内容；二是先到Excel表格中编辑要需要的样式，再导入系统（推荐使用方法二，文件格式支持*.xlsx格式)

![](https://vip.kingdee.com/download/0100c118e14d21894ac98861745e25963ab5.png)

      3、 制作好的Excel表格导入后，选中需要计算的单元格，点击公式栏的【Fx】，进入公式编辑

      公式列表有三种取数方式，分别为：

      ACCT总账取数公式：可以取余额表中的科目及辅助核算的余额、借贷发生额等数据

      REF表间取数公式：可以从其他自定义表的单元格取值

      RPT固定报表取数公式：可以从法定报表“资产负债表"、“利润表""现金流量表"中取项目金额值

![](https://vip.kingdee.com/download/0100515d7e1dd00342be867b3fcbb9c2b860.png)

![](https://vip.kingdee.com/download/0109b0cc10d9cb6b462eb189d8453fbedb6e.png)

      4、 我们以总账取数公式为例，分别填写好：科目、取数类型、会计年度，会计期间和取值的币别，点击确定后公式自动回填在右侧显示。依次类推，填写好所有需要计算的单元格后，模板就算编辑完成了

![](https://vip.kingdee.com/download/0109b58f36221ae44ec89c3ef3edae715227.png)

![](https://vip.kingdee.com/download/010971c39db631374827ae61362fce0de8a1.png)

![](https://vip.kingdee.com/download/01007ba7385592094c09881125a05a06a474.png)

      注意：步骤3和4为“科技”之精髓所在，可以根据图文介绍反复多看几遍，另外附上自定义报表取数函数说明一份：[https://vip.kingdee.com/link/s/MI4pz](https://vip.kingdee.com/article/136394497789099776?share_fromuid=49129992135242496&productLineId=35&isKnowledge=2)

      5、 自定义的报表编辑并保存好后，就回到报表列表勾选并启用，启用后的模板，可以再点击报表编码处进行查看

![](https://vip.kingdee.com/download/010079bec1686b344def8ccb2878d2ebe391.png)

      小龙女听完后惊呼：“这真是纯纯的科技与狠活呀，要是我早些知道，就不用自己手工去统计报表了，谢谢大师兄！”
