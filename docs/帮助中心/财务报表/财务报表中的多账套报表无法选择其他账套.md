# 财务报表中的多账套报表无法选择其他账套

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：常见问题, 快速入门
> 作者：Daisy_ | 阅读：2307 | 有用：4
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/243671601395812352)

问题描述：

为什么购买了多个账套，在使用多账套报表时只能选择当前账套，无法选择到其他账套？

解决方案：

通过【系统设置】-【多账套设置】，选择当前账套，最后设置当前账套的关联账套即可
