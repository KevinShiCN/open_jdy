# 自定义报表中如何设置if函数?

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：常见问题
> 作者：杨桂平 | 阅读：42 | 有用：1

【问题描述】自定义报表中如何设置if函数?

【解释说明】

1.进入自定义报表编辑界面,录入IF就可以看到该函数。

![](https://vip.kingdee.com/download/0109b53cccc4262a4812bb2ec47f6b36859d.png)
![](https://vip.kingdee.com/download/0109f295c6e56cfc4aafa8f16827750e5758.png)

2.选中该函数,按照提示设置条件,再设置满足条件不满足条件分别返回的值即可

![](https://vip.kingdee.com/download/01090b61c1a1a7fa44d0afbfe42b4b6afb44.png)
