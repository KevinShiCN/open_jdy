# 指定现金流量项目时，有的项目选择不到，是什么原因？

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：进阶技巧, 精华汇总, 常见问题
> 作者：简天 | 阅读：1433 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/326744147262776064)

问题描述：

解决方案：

现金流量表有这个项目，但是指定现金流量项目时，却选择不到这个项目，以下图中的“收到其他与经营活动有关的现金”为例

![](https://vip.kingdee.com/download/010917930f7f21e74dd084e228ab57558e93.png)

![](https://vip.kingdee.com/download/01092246e74aeae94b4a97cde2649d3b9628.png)

原因：是由于客户修改了系统预置的项目，把某些项目的影响项改成了”收到其他与经营活动有关的现金项目“了，要找到对应的项目把这个影响项改回来就可以了。

![](https://vip.kingdee.com/download/0109a7d11980ded145ab92b95e777f74904b.png)

切记 ：系统预置的项目内容不要随意修改，除非您需要新增、或删除项目。
