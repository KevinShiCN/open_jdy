# 为什么自定义报表设置单元格格式不生效？

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：常见问题, 进阶技巧
> 作者：Daisy_ | 阅读：1604 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/260000912217983744)

问题描述：

设置了自定义报表的单元格格式，但是没有生效

解决方案：

原自定义报表的格式未彻底清除，可通过以下方式操作：

自定义报表在编辑状态下（非查询状态），选中未生效区域，点击清除单元格图标，选择【格式】，将原有单元格格式清除，再重新设置所需要的单元格格式。
