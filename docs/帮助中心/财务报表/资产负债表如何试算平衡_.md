# 资产负债表如何试算平衡?

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：常见问题
> 作者：shaoling | 阅读：224 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/703773419086801920)

【问题描述】

资产负债表如何试算平衡?

【解释说明】

系统里没有单独的试算平衡表的,资产负债表的试算平衡是通过检查资产、负债和所有者权益三大类目的金额是否符合会计恒等式(**资产 = 负债 + 所有者权益**)来验证的。

**资产总额** = 所有资产类账户借方余额之和

**负债总额** = 所有负债类账户贷方余额之和

**所有者权益总额** = 所有者权益类账户贷方余额之和(包括当期利润或亏损)

**验证会计恒等式**检查是否满足:资产总额=负债总额+所有者权益总额

若等式成立,则资产负债表试算平衡。

若不成立,说明存在错误,需进一步排查。详情可查看:[打开资产负债表,系统提示资产负债表不平衡的常见情况](https://vip.kingdee.com/knowledge/564616536325877504?channel_level=%E9%87%91%E8%9D%B6%E4%BA%91%E7%A4%BE%E5%8C%BA%7C%E6%90%9C%E7%B4%A2%7C%E5%AE%98%E6%96%B9%E7%9F%A5%E8%AF%86&productLineId=35&isKnowledge=2&lang=zh-CN)
