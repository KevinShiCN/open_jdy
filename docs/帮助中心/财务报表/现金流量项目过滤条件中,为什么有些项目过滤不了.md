# 现金流量项目过滤条件中,为什么有些项目过滤不了

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：常见问题
> 作者：财税李炳全 | 阅读：97 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/561354040227911168)

【常见问题】

现金流量表里现金流量项目过滤条件中,有几行汇总类的项目为啥过滤不出来?![](https://vip.kingdee.com/download/0109aff2dccbfb8b4bf48a7cb15edff28f2f.png)

【解释说明】

根据系统报表设定的公式,汇总和合计类的是由其他行的金额累计得来的,因此在过滤中是不允许选择的。
