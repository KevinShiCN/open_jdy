解决方案：

辅助核算可以设置多级核算，查报表时，希望可以取上级辅助核算项目的值。比如：部门核算时，部门设置如下图

![](https://vip.kingdee.com/download/0109b5f01980e6c74f769a766161602c5cf7.png)

如果要计算市场部的月销售费用（科目编辑6601），可按以下格式设置：

=ACCT("6601|部门|@市场部","S",0,0,0,"")
