# 费用报销的流程管理列表的版本号是指什么?

> 作者：财税云肖梅梅 | 阅读：192 | 有用：0

【问题描述】

费用报销的流程管理列表的版本号是指什么?

【名词解释】

版本号是系统自动生成的一个号码,同一类单据,新增一个工作流则该工作流的版本号在之前的版本号自增
