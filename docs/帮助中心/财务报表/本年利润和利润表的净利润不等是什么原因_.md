# 本年利润和利润表的净利润不等是什么原因?

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：常见问题
> 作者：春日负暄 | 阅读：914 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/688190312677497088)

【问题描述】

  本年利润和利润表的净利润不等是什么原因?

如下图,科目余额表的本年利润科目和利润表的净利润的值不一致

![](https://vip.kingdee.com/download/0109f932d9ec22fd449b9e1366c28988e7f9.png)
![](https://vip.kingdee.com/download/010938cba2ffd4fb4649b989cf9095c1c5f7.png)

【解决方案】

1、检查是否有手工录入的“本年利润”科目的凭证

在查凭证界面,可以通过筛选机制凭证为空,快速过滤手工录入的凭证,如果存在有手工录入的凭证,会导致利润表和该科目的余额不一致

![](https://vip.kingdee.com/download/0109eb75b97cc75a4735a0209ef1a6a590d4.png)
科目余额表和明细账中会记录这张手工凭证。但是在利润表里面是没有体现的
![](https://vip.kingdee.com/download/0109e2efe2f74d9d454c9c562399cb18df5d.png)
2、是否有设置其他的结转方案,重新又结转生成了本年利润的凭证

![](https://vip.kingdee.com/download/01093f57a7cba2a548d4a442dd0513f96ec6.png)
![](https://vip.kingdee.com/download/0109bb8bf37e06fe471b8c068d87eb076bb3.png)
在查凭证的时候,会显示这张凭证是机制凭证,但是这个类型的凭证余额也不会计入到利润表中,也会导致不平

【温馨提示】

**在核对利润表的数据和科目的余额时,需要先保证利润表是平衡的**
