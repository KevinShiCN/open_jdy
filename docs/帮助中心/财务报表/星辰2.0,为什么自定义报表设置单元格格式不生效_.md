# 星辰2.0,为什么自定义报表设置单元格格式不生效?

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：常见问题
> 作者：杨桂平 | 阅读：532 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391350582759951872)

【问题描述】

为什么自定义报表设置单元格格式不生效?

【解决方案】

原自定义报表的格式未彻底清除,可通过以下方式操作:

自定义报表在编辑状态下(非查询状态),选中未生效区域,点击清除单元格图标,选择[格式],将原有单元格格式清除,再重新设置所需要的单元格格式。
