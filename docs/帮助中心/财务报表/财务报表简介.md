# 财务报表简介

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：快速入门, 整体介绍, 精华汇总
> 作者：财税云肖梅梅 | 阅读：1861 | 有用：6
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/516286405765760512)

财务报表主要由主表和附表构成，主要包括资产负债表、利润表、现金流量表，还有一些其他辅助报表查询。

1）资产负债表代表了一个企业的“实力”与“财力”，所以我们一般看一个企业行不行，首先应该看资产负债表，它可以反映某个时点企业基本的家底情况；

2）利润表反映了一个企业的“能力”，我们看企业能否利用好资源创造财富，就看利润表的盈利能力。不过，根据目前会计准则的要求，企业是按照权责发生制记账，企业利润表反映的利润未必代表企业赚到的“真金白银”，因此我们需要通过现金流量表来了解企业在盈利过程中产生现金流量的能力。

3）现金流量表我们通常比喻是企业的“血液”，反映了企业的“活力”，同时，我们也可以通过它了解企业筹资活动和投资活动的现金流转情况。
