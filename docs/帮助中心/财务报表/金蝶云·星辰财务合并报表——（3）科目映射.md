# 金蝶云·星辰财务合并报表——（3）科目映射

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：进阶技巧, 整体介绍, 新特性
> 作者：财税云肖梅梅 | 阅读：1230 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/784070866325268224)

**非明细组织机构的账套才能设置科目映射，用于建立母子公司的科目对应关系，保障下发的报表模板保留公式。**

![上传图片](https://vip.kingdee.com/download/010052335068700e47408865efa6123d6176.png)

1、当子公司账套科目体系与母公司账套科目体系一致，且未修改、删除、新增过一级科目，则不需要进行科目映射。模板下发会保留母公司的公式。

2、当子公司账套科目体系和母公司账套科目体系不一致，或修改、删除、新增过一级科目，或报表公式需要取明细科目，则需要新增母子公司的科目映射关系。

3、母公司新增科目映射时，手动录入编码、方案名称，下拉或弹框母公司需要匹配的子公司账套名。点击【自动匹配】，系统将会自动匹配母子公司账套一级科目。如需调整，或删除、新增，可新增行、删除行来操作。

![上传图片](https://vip.kingdee.com/download/0100d1020ff8d7ef4fa3aa030bf828aa6d7b.png)

4、如果已存在匹配科目，再切换匹配账套，则会清空已匹配数据，需重新进行匹配。

5、保存并审核科目映射单后，报表模板下发到匹配的子公司账套，报表公式将会按映射的科目取子公司账套的科目。

6、设置好映射关系，后续如为删除、新增、修改过科目，无需再重新修改匹配关系。
