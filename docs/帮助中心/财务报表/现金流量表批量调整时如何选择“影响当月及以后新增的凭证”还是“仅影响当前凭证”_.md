# 现金流量表批量调整时如何选择“影响当月及以后新增的凭证”还是“仅影响当前凭证”?

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：常见问题
> 作者：shaoling | 阅读：139 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/725889610265625856)

【问题描述】

现金流量表批量调整时如何选择“影响当月及以后新增的凭证”还是“仅影响当前凭证”?

【解释说明】

现金流量表批量调整时,若选择 【批量调整选中行】,则仅支持选择 “仅影响当前凭证”,不支持 “影响当月及以后新增的凭证”
![](https://vip.kingdee.com/download/0109089f3ea56ea240da938e0635cab56289.png)
若需实现“影响当月及以后新增的凭证”的调整效果,可选择【批量调整】操作

操作批量调整需注意:

1、批量修改影响所选期间的所有凭证及以后期间所有凭证,不影响之前期间凭证

2、科目选择非明细科目,将影响该科目下的所有明细科目及辅助核算。

3、批量指定后,无法撤销操作,请谨慎处理。

4. 建议备份账套后再处理。
