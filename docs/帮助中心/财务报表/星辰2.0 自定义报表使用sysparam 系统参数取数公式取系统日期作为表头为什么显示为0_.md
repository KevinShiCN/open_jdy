# 星辰2.0 自定义报表使用sysparam 系统参数取数公式取系统日期作为表头为什么显示为0?

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：常见问题
> 作者：杨桂平 | 阅读：319 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391351669017651968)

【问题描述】

自定义报表使用sysparam 系统参数取数公式取系统日期作为表头为什么显示为0?

如图:自定义报表使用SYSPARAM 系统参数取数公式取系统日期,设置公式后报表中显示为“0”。

![](https://vip.kingdee.com/download/0109030b262f53084fbfabd5b6cf9f5ad214.png)

[故障原因]

设置报表公式保存后没有点击计算。

[解决方案]

在[财务报表]-[自定义报表]中设置报表公式,添加公式后需要点击右上角[计算]才能计算出报表的数据。

![](https://vip.kingdee.com/download/01090bf6b54ead0d41969601e2bea3352ac1.png)

![](https://vip.kingdee.com/download/01095378d06339404ec19ea31e160c6e1dc9.png)
