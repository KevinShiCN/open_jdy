# 设置智能查询方案,提示数据源为“会计报表”时,必须设置“表体取数”或“取数类型”,必须设置“报表项目”请设置。如何处理?

> 作者：happy忆霖 | 阅读：64 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/722619663472740608)

【问题描述】

设置智能查询方案,提示数据源为“会计报表”时,必须设置“表体取数”或“取数类型”,必须设置“报表项目”请设置。如何处理?

【业务场景】

设置智能查询方案时,在表格设置中点击下一步的时候,提示数据源为“会计报表”时,必须设置“表体取数”或“取数类型”,必须设置“报表项目”。无法进入到下一步,是什么原因?
![](https://vip.kingdee.com/download/010963300e157cdc46279dd39caaef216b68.png)
**若希望了解如何设置智能查询方案,请点击:[智能查询如何使用](https://vip.kingdee.com/knowledge/561354035664508416?productLineId=35&isKnowledge=2&lang=zh-CN)**

【解释说明】
**在设置智能查询方案时,取数类型和表体取数是必录项目。若未录入,则报表方案无法生成,无法进入到下一步。**

【操作步骤】

在智能查询方案中,点击<设置表体取数>,根据需要设置对应的数据类型

![](https://vip.kingdee.com/download/0109aa3f5420c8154c188f5a10e5401e85cb.png)

在一级/二级/三级标题行或者列设置中,点击![](https://vip.kingdee.com/download/01096d62b4633e8f4023a40fbb15acecde13.png)进行设置标题的取数来源,设置完成后,点击<下一步>即可

![](https://vip.kingdee.com/download/0109290e6cbda66349389b1c742bcba65431.png)

【温馨提示】

**若系统提示数据源为“科目余额表”时,必须设置“表体取数”或“取数类型”,并且行列标题设置不少于一个,请设置。参考上述步骤操作即可。**
![](https://vip.kingdee.com/download/0109a370c55355ba4259bfbdc3cfe5beae8d.png)
