# 现金流量表数据为空时，如何快速设置？

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：进阶技巧, 精华汇总, 常见问题
> 作者：简天 | 阅读：3741 | 有用：5
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/340799714646313472)

其他软件升级到星辰时，有可能现金流量有数据为空，现金流量都没有指定，可参照下表，批量指定现金流量项。

1、下表科目较多，用户可能只使用到了其中一小部分科目，可以仅对已经使用的一级科目指定。

2、操作路径：【现金流量表】**，过滤初始化期间，**【查询】-【报表调整】-【批量调整】，按下表逐一设定科目及借贷方现金流量项目，全部设定完成后，点击【全部重算】即可

![](https://vip.kingdee.com/download/0109c596e5047996440c8c2278f2bdda27bd.png)
