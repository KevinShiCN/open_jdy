# 分发合并报表模板的时候为什么选择不到分发组织?

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：常见问题
> 作者：Schatz | 阅读：4 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/794370619730360832)

【问题描述】

分发合并报表模板的时候为什么选择不到分发组织?

【解释说明】

模板分发只能分给直接下级的组织,也就是【组织机构设置】中选择了这个账套为上级组织的账套。如果没有账套选择该账套为上级组织,则没有分发的操作。

如下图,在【系统设置】-【组织机构设置】界面中,假设 A 账套被设定为母公司,B 账套将 A 账套选为上级组织,C 账套将 B 账套选为上级组织。

![](https://vip.kingdee.com/download/0109d494a32612f04fff8ac4620b5ab17c11.png)

在此情况下,当在 A 账套进行合并报表模板分发操作时,只能选择 B 账套作为分发组织。

![](https://vip.kingdee.com/download/01096c415439fd5f4ba1a80fff87fc56d5fa.png)

若需将模板分发给 C 账套,则需通过 B 账套来执行分发操作。

![](https://vip.kingdee.com/download/01094d03e07b56e2458e8eb31c36c58ad21d.png)
