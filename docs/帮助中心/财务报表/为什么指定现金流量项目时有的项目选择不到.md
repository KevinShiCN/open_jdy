# 为什么指定现金流量项目时有的项目选择不到

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：常见问题
> 作者：财税云肖梅梅 | 阅读：331 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/568964220788771072)

【问题描述】

指定现金流量项目时,有的项目选择不到,是什么原因?

【解决方案】

现金流量表有这个项目,但是指定现金流量项目时,却选择不到这个项目,以下图中的“收到其他与经营活动有关的现金”为例

![](https://vip.kingdee.com/download/01098b8fe137857d4936a387c1894eff8042.png)

![](https://vip.kingdee.com/download/01092ab7de831bf0490c9e964224271ec02b.png)

**原因:是由于客户修改了系统预置的项目,把某些项目的影响项改成了”收到其他与经营活动有关的现金项目“了,要找到对应的项目把这个影响项改回来就可以了。**

![](https://vip.kingdee.com/download/01093b992472f945411697328b2ac4fd7ed5.png)

**切记 :系统预置的项目内容不要随意修改,除非您需要新增、或删除项目。**
