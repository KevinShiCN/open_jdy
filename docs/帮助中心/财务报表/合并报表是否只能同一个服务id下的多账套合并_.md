# 合并报表是否只能同一个服务id下的多账套合并?

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：常见问题
> 作者：Schatz | 阅读：43 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/794012876519336960)

【问题描述】

合并报表是否只能同一个服务ID下的多账套合并?

【解释说明】

基于【组织机构设置】中仅支持选择同一服务 ID 下的账套进行相关配置,这就从源头上排除了设置其他服务 ID 下账套的可能性。所以合并报表在数据获取时只能从同一服务 ID 下的多个账套中提取数据。
