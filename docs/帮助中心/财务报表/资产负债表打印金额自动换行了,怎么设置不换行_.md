# 资产负债表打印金额自动换行了,怎么设置不换行?

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：常见问题
> 作者：Schatz | 阅读：272 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/617524482223037696)

【问题描述】

资产负债表打印金额自动换行了,怎么设置不换行?

![](https://vip.kingdee.com/download/01098a7d67ab88e84604a5b37c29ad29de8a.png)

【解释说明】

资产负债表打印时金额自动换行,是由于金额较大,一行内无法完全显示所有内容,因此会自动换行。

【操作步骤】

若不希望出现换行,可以采用以下三种方法:

方法一:**调整字体大小**:通过缩小字体大小,单个单元格能够显示的内容量增加,从而避免内容换行。

![](https://vip.kingdee.com/download/010926363bdc57b146afa99715e7301a20bf.png)

方法二:**横向打印**:选择横向打印设置后,纸张的宽度增加,相应地单元格的列宽也会扩大,使得每个单元格能够容纳更多的内容。

![](https://vip.kingdee.com/download/0109e254d660e96048c1a1ce7f5c6691da9a.png)

方法三:**导出至Excel调整后打印**:查询到资产负债表数据后,直接导出至Excel表格,在Excel中调整好打印格式,然后再进行打印。

![](https://vip.kingdee.com/download/01096a1394efdc2346219a1776d7ad025546.png)
