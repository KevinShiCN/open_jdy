# 自定义报表如何设置金额为0时不显示?

> 产品：金蝶云·星辰 | 分类：财务报表 | 类型：常见问题
> 作者：杨桂平 | 阅读：627 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/538807122516153600)

【问题描述】

自定义报表如何设置金额为0时不显示?

![](https://vip.kingdee.com/download/0109623452a599014e8887ff67c6a7d74665.png)

【解决方案】

进入自定义报表后点击左上角的小齿轮,然后点击常规,勾选【将零显示为空白】的选项,点击【确定】按钮即可

![](https://vip.kingdee.com/download/01099f376628768d42c793371f2ad9b880ec.png)

设置后可以看到原本显示为0的都显示为空了

![](https://vip.kingdee.com/download/01092d0c46b8c05948ed897d1c8db2648066.png)
