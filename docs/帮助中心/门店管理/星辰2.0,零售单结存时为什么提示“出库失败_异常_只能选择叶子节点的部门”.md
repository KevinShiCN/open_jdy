# 星辰2.0,零售单结存时为什么提示“出库失败:异常:只能选择叶子节点的部门”

> 产品：金蝶云·星辰 | 分类：门店管理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：224 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352237479543552)

【问题描述】

如图,结存时提示的只能选择叶子节点的部门:

![](https://vip.kingdee.com/download/0109f8efa7730a0046839a2024b1e8c143a2.png)

【解释说明】

原因是门店设置到了公司名称上去了

【解决方案】

需要在门店管理-门店资料点击所属部门选择公司名称下的部门,即可正常。

![](https://vip.kingdee.com/download/010929c5e7f5c6a64fddb60624d54ce427c0.png)
