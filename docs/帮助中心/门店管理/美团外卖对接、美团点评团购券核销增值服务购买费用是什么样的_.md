# 美团外卖对接、美团点评团购券核销增值服务购买费用是什么样的?

> 产品：金蝶云·星辰 | 分类：门店管理 | 类型：常见问题
> 作者：happy忆霖 | 阅读：639 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/645066274430821888)

【问题描述】

美团外卖对接、美团点评团购券核销增值服务购买费用是什么样的?

【解决方案】

功能模块费:500 元 / 门店 / 年,免费提供 1000 次 / 门店 / 月 的基础类接口调用额度;并在购买时一次性赠送 1000 次管理类调用额度;超出费用需要自行充值。

接口调用说明及费用明细:

接口由基础类接口和管理类接口组成;

基础类接口费用为 0.15 元 / 百次(订单交易及订单备餐状态变更等过程的接口调用)

管理类接口费用为 0.3 元 / 百次(非订单交易过程的管理类接口,如商品匹配、库存同步等(暂不支持))

接口调用费价格由外卖平台制定,金蝶云星辰的收费标准,会随各大平台的收费政策调整而变化。
