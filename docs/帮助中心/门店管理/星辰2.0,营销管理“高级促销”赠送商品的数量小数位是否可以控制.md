# 星辰2.0,营销管理“高级促销”赠送商品的数量小数位是否可以控制

> 产品：金蝶云·星辰 | 分类：门店管理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：209 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352711771138304)

【解决方案】

高级促销里的商品数量小数位是默认的,不受系统参数的单位小数位控制的,请勿去修改后台的单位小数位。
