# 作废的寄存业务可以再次寄存吗?

> 产品：金蝶云·星辰 | 分类：门店管理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：97 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/625134731251957504)

【业务场景】

作废的寄存业务可以再次寄存吗?

【解释说明】

作废的寄存业务支持再次寄存,可以重新在零售开单的界面选择对应的零售单再次寄存

【操作指引】

在POS开单界面,点击【开单记录】

![](https://vip.kingdee.com/download/010990a36bf2d2564004bfa38b34e1a73ee2.png)

选择对应寄存已作废的零售单,点击右上角【寄存】,可以再次寄存

![](https://vip.kingdee.com/download/010930cdd9413a4b4940be042c37c5a2d486.png)

寄存默认带出是零售单的数量,如本次只需要寄存部分,可以修改本次寄存的数量

![](https://vip.kingdee.com/download/0109e5e2734b590b43c7b492707ccda4f2f4.png)

寄存成功后,在【店务】-【寄存】界面会新增一条寄存的记录

![](https://vip.kingdee.com/download/01096ef662142854409a87f5ce7ad3aed0fd.png)
