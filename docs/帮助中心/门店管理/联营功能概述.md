# 联营功能概述

> 产品：金蝶云·星辰 | 分类：门店管理 | 类型：常见问题
> 作者：winnie_wen | 阅读：2359 | 有用：8
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352486353283840)

【功能概述】

门店部分商品由供应商管理,门店不管理库存,提供条码服务和场地服务,商品过机销售后,定期统计销售金额扣除费用后结算给供应商。合伙人模式通过联营功能解决了商品销售的问题,降低合伙人之前的场地和库存成本,实现双方共赢。金蝶云星辰零售联营通过服务商品管理录入商品条码,上架到POS销售,定期给商品供应商做结算。
