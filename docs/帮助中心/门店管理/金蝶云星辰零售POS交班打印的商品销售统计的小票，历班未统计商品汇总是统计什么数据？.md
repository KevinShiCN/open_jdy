# 金蝶云星辰零售POS交班打印的商品销售统计的小票，历班未统计商品汇总是统计什么数据？

1 问题描述

金蝶云星辰零售POS交班打印的商品销售统计的小票，历班未统计商品汇总是统计什么数据？

## 2 适用版本

星辰3.0以上标准版/专业版/旗舰版/所有版本

## 3 字段解析
零售单据所在的班次进行交班的时候，零售单还没有同步到后台，无法统计到此零售单，当下一班登录开始后才同步此零售单就属于上一班的单据了，成为历班的单据，进而统计历班未统计商品。
