# 零售单结存,提示商品数量拆分为0,请检查单位小数位,应该如何处理?

> 作者：cs123456 | 阅读：289 | 有用：0

【问题描述】金蝶云星辰,零售单结存失败,提示商品数量拆分为0,请检查单位小数位,应该如何处理?

结存提示示例:

![](https://vip.kingdee.com/download/0109ef789780e9a148b48832c32e4cd319cc.png)

【问题原因】商品为多单位的批次商品或者保质期商品,出库时,拆分成多个批次出库,出库的批次数量四舍五入换算后数量为0 ,如示例商品,1大方=6方,零售出库1大方,拆分为2方和4方出库,2方换算成大方,四舍五入数量为0,导致结存提示商品数量拆分为0,请检查单位小数位,

库存剩余两个批次:

![](https://vip.kingdee.com/download/01090a473e4025c742d38d95f2b87114826c.png)

![](https://vip.kingdee.com/download/0109faf49795a2b24c12866a34f37830cef0.png)
