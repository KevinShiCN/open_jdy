# 通用收款设置与门店收款设置的先后顺序是怎样的?

> 作者：海鸥乔纳森 | 阅读：284 | 有用：0

【问题描述】

通用收款设置与门店收款设置的先后顺序是怎样的?

[情景1]

通用收款中设置收款方式,门店收款中未设置收款方式,POS前台默认为通用中的收款方式;

[情景2]

通用收款中及门店收款都设置完成收款方式,POS前台默认为门店收款设置中的收款方式。
