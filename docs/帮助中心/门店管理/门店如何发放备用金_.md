# 门店如何发放备用金?

> 产品：金蝶云·星辰 | 分类：门店管理 | 类型：常见问题
> 作者：lykk | 阅读：189 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/660286820554417920)

【问题描述】

门店如何发放备用金?

【解决方案】

1、在系统后台【门店管理】-【门店现金缴款单】界面,如图:

![](https://vip.kingdee.com/download/01099cc04b399f714e179386ec4c9669748c.png)

可以点击**新增**按钮,新增门店现金缴款单,缴款类型可以选择“发放备用金”,录入正数表示发放,录入负数可做冲抵。

![](https://vip.kingdee.com/download/01098f26a8aecc1446fdb281d33c648aa70b.png)

保存后,确认数据无误可操作审核

![](https://vip.kingdee.com/download/01093e9ba11b871249aba43afcfd455bd42a.png)

2、在POS中【店务】-【现金领缴记录】进行操作

![](https://vip.kingdee.com/download/01096b4296cc666842acb7da799a9394f7ae.png)

点击“备用金领取”

![](https://vip.kingdee.com/download/01092d5f32288e2548b7ab820e074e5529b3.png)

输入金额后,点击“确认领用”,则会立即生成“已审核”的领用记录

![](https://vip.kingdee.com/download/01095bedb6d447c146439c604ee9ebdccd75.png)

后台也会相应进行同步;

![](https://vip.kingdee.com/download/0109322de66fec2842f98b5574cf544cd8e7.png)
