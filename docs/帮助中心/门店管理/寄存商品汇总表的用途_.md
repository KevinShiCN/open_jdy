# 寄存商品汇总表的用途?

> 产品：金蝶云·星辰 | 分类：门店管理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：97 | 有用：1

【业务场景】

寄存商品汇总表的用途?

【解释说明】

寄存商品汇总表是用于跟踪和管理所有寄存商品的库存和相关信息的报表,可以根据需要选择汇总依据商品、商品类别、商品品牌、门店收银员信息汇总显示寄存商品对应的数据,同时寄存商品汇总表也可以选择小计,门店小计,不区分门店统计数据显示

【操作指引】

菜单栏依次点击【门店管理】-【寄存商品汇总表】

![](https://vip.kingdee.com/download/0109ab68f1cea64b465c9d8e0f436277e674.png)

支持按商品、商品类别,商品品牌、门店、收银员汇总查看寄存情况

![](https://vip.kingdee.com/download/0109eb897feb8dd8446383322c34912c695e.png)
