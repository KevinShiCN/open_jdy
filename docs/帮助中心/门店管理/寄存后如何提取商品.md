# 寄存后如何提取商品

> 产品：金蝶云·星辰 | 分类：门店管理 | 类型：常见问题
> 作者：_Leo | 阅读：531 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391353628528853760)

【操作流程】

在POS端点击【店务】-【寄存】进入寄存列表界面,选择需要提货的寄存单,点击【提货】进入提货界面。

如图: ![](https://vip.kingdee.com/download/0109450fedb679e1437c9fafd256398c57ab.png)

确认寄存信息是否准确,选择要提取的商品、录入提货数量(可以部分提货,如寄存数量是2束,现只提取1束,则提货数量可录入1),然后点击【提货】,若会员设置了密码则输入会员密码后点击【确认】提货成功;提货成功后会立即打印提货小票,供顾客确认。

![](https://vip.kingdee.com/download/0109f4e12e8742504bdfa784e93382b60021.png)

提货完成后这笔寄存单JC-20220519-00007的状态会显示为“部分提货”,打开此单可查看提货明细中会显示提货数量1、待提货数量是1。客户下次再来提货时再选择此单提货即可。

后台查看寄存流水:

在后台想要查看寄存流水,可到后台【门店管理】-【零售单】-【寄存流水】查看寄存流水。如图:

![](https://vip.kingdee.com/download/010924ac5552a1b7453fbd370e558bfdf576.png)   【温馨提示】

1. 若需要全部提取商品时可点击【全部提取】一次性提取全部商品,无需选择提货商品和录入提货数量。

2. 只有会员的零售销售单才能寄存,门店散客、预订单、会员充值等不能寄存。

3. 一张零售单同一个状态只能寄存一次。



4. 零售单结算后,不管是否结存(已结存/未结存),都可以寄存。
