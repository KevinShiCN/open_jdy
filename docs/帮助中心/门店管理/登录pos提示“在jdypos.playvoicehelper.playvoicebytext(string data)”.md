# 登录pos提示“在jdypos.playvoicehelper.playvoicebytext(string data)”

> 产品：金蝶云·星辰 | 分类：门店管理 | 类型：常见问题
> 作者：lykk | 阅读：76 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/647965266692552192)

【问题描述】

登录POS提示“在JDYPOS.PlayVoiceHelper.PlayVoiceByText(String data)”

![](https://vip.kingdee.com/download/0109d3dcc2f70ec5494fbbc2c26610c5931c.png)

【解决方案】

用户在小程序商城下单后,门店POS会进行语音播报,提示商家有新订单需要处理。这个提示是因为没有安装语音播报插件,下载语音播报插件并按照使用说明安装即可,注意区分 x32 或 x64 系统。若安装后仍然没有语音播报,尝试检查硬件是否开启声音,或者重新安装插件后再重启硬件。
