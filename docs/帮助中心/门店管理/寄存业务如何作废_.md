# 寄存业务如何作废?

> 产品：金蝶云·星辰 | 分类：门店管理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：106 | 有用：1
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/625134724624798976)

【业务场景】

寄存业务如何作废?

【解释说明】

寄存业务可以在POS端店务模块,操作作废;未提货、部分提货的寄存业务单支持作废,已经全部提货完成的单据不支持作废

【操作指引】

POS端依次点击【店务】-【寄存】

![](https://vip.kingdee.com/download/0109a9c1269eead14cbca390facda956f595.png)

提货状态为待提货,部分提货的单据,可以选择作废

![](https://vip.kingdee.com/download/010916b2b215175b4a5ea027b12a0afc4f0b.png)

全部提货的单据不允许作废,操作作废会出现如下提示

![](https://vip.kingdee.com/download/01098e184ab6b0e04f5aa524f55f0b138d84.png)
