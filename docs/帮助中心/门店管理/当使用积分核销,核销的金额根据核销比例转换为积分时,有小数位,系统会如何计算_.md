# 当使用积分核销,核销的金额根据核销比例转换为积分时,有小数位,系统会如何计算?

> 产品：金蝶云·星辰 | 分类：门店管理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：140 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/391352451757825536)

【说明】:

因积分为整数,当录入本次抵现金额后折算的积分不为整数时,默认向上取整。如下图所示,积分规则为每10分抵1元,此时,录入本次抵现金额0.01元,折算后的抵现积分应为0.1,向上取整后为1,因此,本单需要消耗的积分数量为1。
