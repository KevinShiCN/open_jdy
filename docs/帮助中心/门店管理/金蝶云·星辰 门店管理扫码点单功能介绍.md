# 金蝶云·星辰 门店管理扫码点单功能介绍

## 1 简介

### 1.1 功能介绍

商家免注册点餐小程序，支持按桌码点餐（堂食模式）和按门店码点餐（茶饮模式）

顾客进入店内后，通过扫描桌上的二维码即可进入点餐界面，选择堂食或外带。对于堂食顾客，可以选择自己喜欢的座位，然后轻松下单，无需等待服务员前来点餐。茶饮店扫描店铺门口或展示区的二维码进行点餐，下单后显示自己订单排队情况，制作完成到店取走。

扫码点餐系统可以自动接收订单，减少了人工点餐的错误和繁琐；同时系统可以实时显示订单状态，方便商家进行订单管理和制作安排，提高了制作效率。满足现代消费者对高效、便捷服务的需求，提高了顾客的满意度。

### 1.2 产品能力

●扫桌码点单，一桌一码，落座扫码后可送餐到桌

●扫门店码点单，一门店一码，扫码下单后叫号自取

●收银台按桌位手工下单，一键完订单成履约流程

●支持加点、按人数收取餐位费、打包费等行业特性

●可按门店配置展示商品类别、支付通道，适用于直营及加盟模式

●支持显示制作进度及小程序消息提醒，订单状态及时知晓

●支持会员、促销、优惠券、充值赠送等多种营销能力
