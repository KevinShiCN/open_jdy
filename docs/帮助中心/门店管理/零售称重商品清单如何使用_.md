# 零售称重商品清单如何使用?

> 产品：金蝶云·星辰 | 分类：门店管理 | 类型：常见问题
> 作者：财税李炳全 | 阅读：939 | 有用：0
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/470032291587338240)

【问题描述】

零售称重商品清单如何使用?

【解释说明】

1、基础资料商品信息里面添加称重商品,[零售称重商品创建](https://vip.kingdee.com/link/s/lDRWO)

2、称重商品清单选择商品称重商品,集中查看、管理需要传输到条码秤上的称重商品。

路径:金蝶云星辰后台 - 【零售资料】 - 【基础资料】 - 【称重商品清单】

![](https://vip.kingdee.com/download/0109760b21df8af44514990bab6dac298a68.png)

①将鼠标悬停至商品条目,左侧出现+-符号,点击可增减称重商品。也可通过页面上方的删除、新增按钮进行相关操作。

![](https://vip.kingdee.com/download/0109a28e6546f37b49d4a41758cfa0502f30.png)

②点击商品选择框,添加称重商品到称重商品清单。可以在商品选择弹窗中,通过搜索、类别筛选定位称重商品。并可进行多选操作,点击确定即可将选取的商品添加到称重商品清单。

③同时,支持通过引入/引出按钮,批量进行称重商品清单的导入导出。注意:需按照模版填写相关内容。

④可以为添加到清单中称重商品设置PLU码,PLU码可用于条码秤快速选择商品使用。注意:PLU 码与商品一一对应,不得重复。

![](https://vip.kingdee.com/download/0109b5eea09a1cd6446c9529c99ed27c489a.png)

此外,称重商品传秤清单还提供【传秤记录】与后台【手动传秤】的入口。

![](https://vip.kingdee.com/download/0109438449c801ca4417938f64e04b932f2a.png)

注意事项:

1. PLU码应当为长度为5位且大于零的整数数字(不足5位前面补零)(目前系统是使用商品条码作为PLU码(秤上的唯一码),后续会新增PLU码字段,区分条码以及PLU码);

2. 商品的价格需大于0;(部分条码秤打印价格为0的商品标签会报错或者不显示生鲜码);

3. 大华秤的PLU码范围为00001-04000,超出范围的商品无法传秤,其余秤型号PLU码范围为00001-99999;

4. 热键范围:顶尖1-224,大华1-71,容大得众1-60,迪宝1-137;具体范围要看秤的操作面板,默认值0不指定热键;

5. 称重以及计件的商品都需要添加到【称重商品清单】,目前是以单位名称进行区分单位名称是KG、G、千克、克、公斤、斤(英文字母不区分大小写)的商品分录传秤时识别为称重商品,其余单位名称的商品分录识别为计件商品;
