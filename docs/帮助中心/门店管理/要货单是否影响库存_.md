# 要货单是否影响库存?

> 作者：小朱呀 | 阅读：17 | 有用：0

【问题描述】

要货单是否影响库存?

【解释说明】

**零售要货流程**:当门店库存不足,需要补货时,收银员在POS端可发起要货申请,后台跟据要货单进行配货、发货

要货单相当于申请单,不会影响库存,需要后台配货、发货后才会影响库存
