# 如何设置门店配送价?

> 产品：金蝶云·星辰 | 分类：门店管理 | 类型：常见问题
> 作者：_Leo | 阅读：806 | 有用：1

【业务描述】

如何设置门店配送价?

【业务场景】

门店配送价主要用于门店申请要货、总部配货发货的业务(即调拨业务),根据门店业务需求配置门店配送价,门店在新增要货申请时根据方案判断此商品为同价调拨或异价调拨。

目前支持:

总部对所有门店统一配送价;

总部对不同类型的门店有不同配送价;

在不同区域有多个配送中心,由于区域管理差异,各配送中心对不同门店有不同配送价。

【操作流程】

在【零售资料】-【门店配送价】,点击右上角【新增(异价)】/【新增(同价)】可添加门店配送价。
