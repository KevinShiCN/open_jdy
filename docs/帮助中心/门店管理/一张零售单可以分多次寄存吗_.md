# 一张零售单可以分多次寄存吗?

> 产品：金蝶云·星辰 | 分类：门店管理 | 类型：常见问题
> 作者：海鸥乔纳森 | 阅读：159 | 有用：3
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/625134773362614272)

【业务场景】

一张零售单可以分多次寄存吗?

比如零售单的数量是10个,寄存了5个,还需要将剩下的5个也寄存,如何操作?

【解释说明】

一张零售单操作寄存后,不支持再次寄存,系统会提示:该零售单已进行过寄存,不能重复寄存,如寄存的数量需要修改,可以将原寄存的单据作废后重新寄存,如原先寄存的数量少了,需补寄存的数量,可以直接手工新增寄存单

![](https://vip.kingdee.com/download/0109560cf78b7a4e46ac9d40b3b7173b7e4d.png)

【操作步骤】

**一、作废寄存单,重新寄存**

菜单栏依次点击【店务】-【寄存】

![](https://vip.kingdee.com/download/0109bb1224c253494599b8c3a4d36c362580.png)

可以选择寄存有误的单据,直接点击作废,作废后零售单可以再次寄存

![](https://vip.kingdee.com/download/0109b7e619c254f0494296290b021fe927d2.png)

**二、手工新增寄存单**

路径:【店务】-【寄存】,点击新增按钮,可以手工选择商品新增寄存单

![](https://vip.kingdee.com/download/0109e9325bc2af514cf1a4651be76ab0f520.png)
