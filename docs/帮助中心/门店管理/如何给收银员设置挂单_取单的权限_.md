# 如何给收银员设置挂单/取单的权限?

> 产品：金蝶云·星辰 | 分类：门店管理 | 类型：常见问题
> 作者：Simone邓 | 阅读：403 | 有用：2
> 来源：[金蝶云社区](https://vip.kingdee.com/knowledge/684928933103512320)

【问题描述】

如何给收银员设置挂单/取单的权限?

【操作步骤】

在后台【零售资料】-【收银员管理】

![](https://vip.kingdee.com/download/0109e77052032f424f0f9fe1fb6b1cd97571.png)
【收银员管理列表】选择需要授权的收银员,点击【授予权限】

![](https://vip.kingdee.com/download/01097f9056e226914ca186634dbb83334b9d.png)

【门店收银权限】-【功能权限】-【开单权限】,可勾选或者取消【挂单/取单】的权限

![](https://vip.kingdee.com/download/0109b7395f5b7339416faa3dec8df9c9c626.png)
