# 零售pos端扫码开发票流程

【问题描述】

POS扫码开发票流程

【操作流程】

**功能配置操作: ******

**摘要:******

(1) 需购买税务云模块——销售开票

(2) 后台配置

(3) POS配置

(4) 开票查询

**后台配置******

(1) **在系统参数设置维护纳税人信息,并设置开票额度******

操作路径:【系统设置】 》【税务参数】 》【开票信息】
![](https://vip.kingdee.com/download/0109156bf7a1df464da09b04f33d526f3479.png)
(2) **开票的税收分类及项目在**【门店管理】 》【零售开票记录 】 》【开票项目配置】中设置,税率来自税收分类对应的税率。
![](https://vip.kingdee.com/download/0109979c8d95742c45468db7c51b71df28c4.png)
![](https://vip.kingdee.com/download/010970c92a92367a497fa0343fa3e75e1039.png)

![](https://vip.kingdee.com/download/0109349d08da820f446c9d36552b50365176.png)

(3)**零售参数设置中配置开票设置**

点击【零售资料】-【参数设置】

![](https://vip.kingdee.com/download/010906671bb43d874bc2a410627561350541.png)
在门店管理参数中进行开票设置

![](https://vip.kingdee.com/download/01092d7459c2bf0d4c3183071d81a81dc38f.png)

(4) **POS配置:小票模板打开二维码按钮。如果没有看到这个开关,重置模板试下******

操作路径:POS 》【设置】 》【打印设置】

![](https://vip.kingdee.com/download/0109964945432cd94fd4a0e303df0a27a539.png)

(5) **在POS开单记录里面打印零售小票,顾客可以扫描小票上的二维码开票******

操作路径:POS 》【开单记录】 》【发票】

![](https://vip.kingdee.com/download/010936824c11ea30497985fba26ab2e31f0e.png)

(6) **开票查询:顾客开票完成,可以在门店管理-开票记录或者税务模块里查询里已开发票******

路径一:【门店管理】 》【零售开票记录 】

![](https://vip.kingdee.com/download/010960cb7988eb61495b9349f0b938975e43.png)

路径二:【税务管理】 》【已开发票列表 】/【发票管理】
![](https://vip.kingdee.com/download/01095610d174f870431dabd866d1c9ef11bf.png)
