### 基本信息
- 用途说明：其他支出单新增，支持批量新增
- 请求方式：POST
- 请求地址：https://api.kingdee.com/jdyscm/otherPayment/add
### 请求参数
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)
#### params参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID

#### body参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|items|true|array|单据列表| 
|--number |false |string |编码| 
|--date|true|date|日期|
|--contactNumber|false*|string|客户、供应商编码（***付款金额小于金额合计时必填**）
|--rpAmount|true|string|付款金额
|--accNumber|true*|string|结算账户编码（***收付款金额为0时，可不填**）
|--entries|false|array|明细行| 
|----categoryName|true|string|收入、支出类别名称| 
|----amount|true|string|金额| 
|----description|false|string|备注| 

#### 请求示例
```
POST 
https://api.kingdee.com/jdyscm/otherPayment/add?access_token=xxxdbId=xxx

HTTP 1.1
```
```
{
	items: [
        {
            contactNumber: ,
            date: 2020-11-06,
            number: ,
            rpAmount: 20,
            accNumber:999999,
            entries: [
                {
                    amount: 19,
                    description: 123123,
                    categoryName: 比特币
                },
                {
                    amount: 10,
                    description: 2222,
                    categoryName: 现金
                }
            ]
        }
	]
}
```
