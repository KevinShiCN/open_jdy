### 基本信息
- 用途说明：新增收款单，支持批量新增
- 请求方式：POST
- 请求地址：https://api.kingdee.com/jdyscm/receipt/add
### 请求参数
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)
#### params参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID

#### body参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|items|true|array|单据列表| 
|--number |false |string |编码| 
|--date|true|date|日期|
|--remark|false|string|单据备注
|--contactNumber|true|string|客户编码
|--employeeNumber|false|string|收款人编码
|--discount|false|string|折扣
|-- originNo|false|string|来源单号|
|--entries|false|array|源单明细| 
|----billNo|true|string|源单编码| 
|----billType|true|string|源单类型(销货单：SALE，核销单：VERIFICA，其他收入单：QTSR)| 
|----nowCheck|true|string|本次核销金额| 
|--accounts|true|array|结算账户明细
|----accNumber|true|string|账户编码
|----wayName|true|string|结算名称
|----payment|true|string|收款金额
|----settlement|false|string|结算号
|----remark|false|string|结算备注

#### 请求示例
```
POST 
https://api.kingdee.com/jdyscm/receipt/add?access_token=xxx&amp;dbId=xxx

HTTP 1.1
```
