### 基本信息
- 用途说明：查询核销单列表
- 请求方式：POST
- 请求地址：https://api.kingdee.com/jdyscm/verifica/list
### 请求参数
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)
#### params参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID

#### body参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|filter |*false*	| object|过滤条件|
|-- number	|false	|string	|单据编码|
|-- matchCon|false|string|单据编号、备注、供应商名称关键字模糊搜索|
|-- beginDate|false|date|单据日期-起始（yyyy-MM-dd ）|
|-- endDate|false|date|单据日期-结束（yyyy-MM-dd ）|
|-- createName|false|string|创建人名称|
|-- modifyName|false|string|修改人名称|
|-- billDesc|false|string|订单备注模糊搜索|
|-- originNo|false|string|来源单号|
|--pageSize	|false	|int	|分页参数-每页记录数（默认10，最多500）|
|--page	|false	|int	|分页参数-当前页码（默认1）|
#### 请求示例
```
POST 
https://api.kingdee.com/jdyscm/verifica/list?access_token=xxx&amp;dbId=xxx

HTTP 1.1
```
```
BODY
{
	filter:
	{
		pageSize:10,
        page:1,
        matchCon:,
        beginDate:2021-11-01,
        endDate:2020-11-01,
        createName:zy,
        modifyName:zy,
        billDesc:,
	originNo:10086
	}
}
```

### []()基本信息

- 用途说明：查询核销单列表

- 请求方式：POST

- 请求地址：https://api.kingdee.com/jdyscm/verifica/list

### []()请求参数

#### []()headers参数

[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)

#### []()params参数

| 参数名称 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| access_token | true | string | 云平台访问令牌 |
| dbId | true | number | 账套ID |

#### []()body参数

| 参数名称 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| filter | false | object | 过滤条件 |
| – number | false | string | 单据编码 |
| – matchCon | false | string | 单据编号、备注、供应商名称关键字模糊搜索 |
| – beginDate | false | date | 单据日期-起始（yyyy-MM-dd ） |
| – endDate | false | date | 单据日期-结束（yyyy-MM-dd ） |
| – createName | false | string | 创建人名称 |
| – modifyName | false | string | 修改人名称 |
| – billDesc | false | string | 订单备注模糊搜索 |
| – originNo | false | string | 来源单号 |
| –pageSize | false | int | 分页参数-每页记录数（默认10，最多500） |
| –page | false | int | 分页参数-当前页码（默认1） |

#### []()请求示例

```
POST 
https://api.kingdee.com/jdyscm/verifica/list?access_token=xxx&amp;dbId=xxx

HTTP 1.1
```

```
BODY
{
	filter:
	{
		pageSize:10,
        page:1,
        matchCon:,
        beginDate:2021-11-01,
        endDate:2020-11-01,
        createName:zy,
        modifyName:zy,
        billDesc:,
	originNo:10086
	}
}
```
