### 基本信息
- 用途说明：获取核销单源单列表（支持过滤）
- 请求方式：POST
- 请求地址：https://api.kingdee.com/jdyscm/verifica/findSourceList
### 请求参数
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)
#### params参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID

#### body参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|filter |*false*	| object|过滤条件|
|-- transType	|true	|string	|获取源单类型preRec:预收款prePay:预付款invSa:应收款invPu:应付款|
|-- billNo	|false	|string	|单据编码|
|-- buId	|true	|string	|客户或供应商id|
|-- beginDate|false|date|单据日期-起始（yyyy-MM-dd ）|
|-- endDate|false|date|单据日期-结束（yyyy-MM-dd ）|
#### 请求示例
```
POST 
https://api.kingdee.com/jdyscm/verifica/findSourceList?access_token=xxx&amp;dbId=xxx

HTTP 1.1
```
```
BODY
{
	filter:
	{
		transType:invSa,
        buId:12932492569176,
        billNo:12932492569176,
        beginDate:2019-11-01,
        endDate:2022-11-01
	}
}
```
