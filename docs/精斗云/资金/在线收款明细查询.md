### []()基本信息

- 用途说明：查询在线收款明细

- 请求方式：POST

- 请求地址：https://api.kingdee.com/jdyscm/onlinePayOrder/list

### []()请求参数

#### []()headers参数

[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)

#### []()params参数

| 参数名称 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| access_token | true | string | 云平台访问令牌 |
| dbId | true | number | 账套ID |

#### []()body参数

| 参数名称 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| – createTimeBegin | true | string | 创建时间开始 |
| – createTimeEng | true | string | 创建时间结束 |
| – invsaNo | false | string | 销货单号 |
| – page | true | number | 当前页码（默认1） |
| – payType | false | array | 支付类型 |
| — items | false | number | 1：收款、2：退款、3：撤销、4：空 |
| – payWay | false | array | 支付渠道 |
| — items | false | number | 1：通联支付 |
| – receiptNo | false | string | 关联收款单号 |
| – page | true | number | 每页记录数（默认10） |
| – trxCode | false | array | 交易类型 |
| — items | false | number | 1：微信、2：QQ、3：支付宝、4：银联、5：空 |
| – trxStatus | false | array | 交易状态 |
| — items | false | string | 0000：交易成功、1001：交易不存在、3000：交易失败、2000：交易处理中 |
| – trxid | false | string | 商户单号 |
| – verificaNo | false | string | 核销单号 |
