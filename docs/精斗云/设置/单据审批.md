### 基本信息
- 用途说明：针对精斗云进销存V3系统的购货单、购货订单、销货单、销货订单、其它入库单、其它出库单、调拨单、收付款单、其他收入支出单进行批量审核或批量反审核的功能。
- 请求方式：GET
- 请求地址：https://api.kingdee.com/jdyscm/bill/check

### 请求参数
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)

#### params参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID
|billNo	|*true*	|string	|单据编号（多个单据编号要用逗号隔开）
|billType	|*true*	|string	|单据类型（pur：购货单，po：购货订单，sale：销货单，so：销货订单，oi：其它入库单，oo：其它出库单，tf：调拨单，pay：付款单，rec：收款单，exp：其他支出单，inc：其他收入单）
|state	|*true*	|number	|审核类型(1：审核，2：反审核)

#### 请求示例
```
GET 
https://api.kingdee.com/jdyscm/bill/check?access_token=1510557056dbfff1a4ab8bcf8a686679&amp;dbId=37436958539&amp;billNo=GHDD20181123001,GHDD20181108001&amp;billType=po&amp;state=2

HTTP 1.1 
```

### 返回参数
返回参数 	|字段类型 	|字段说明 
:---   |     :---     | :---
code	|string	| 返回码
msg	|string 	|结果或错误描述
data|array|审核详情
--successCount | number |成功数量
--successArr | array |成功详情
----billNo| string |单据编号
--failCount | number |失败数量
--failArr | array |失败详情
----billNo| string|单据编号
----msg| string|失败描述
----data | array |关联数据
------billNo| string |单据编号
------billType| string | 单据类型

#### 返回码
返回值 	|说明
:---   |     :---     
0	|成功
502	|内部服务器错误
40001|无API访问权限
46001|数据验证不通过
46002|无查询记录
	
其他返回值参照产品概述章节列明的通用返回值说明。

#### 返回示例
```json
{
    data: {
        successCount: 1,
        successArr: [
            {
                billNo: GHDD20181123001
            }
        ],
        failCount: 1,
        failArr: [
            {
                billNo: GHDD20181108001,
                msg: 订单 GHDD20181108001 有关联的购货单，不能对它进行反审核！,
                data: [
                    {
                        billType: PUR,
                        billNo: GH20181108001
                    }
                ]
            }
        ]
    },
    code: 0,
    msg: 成功
}
```
### 注意事项
- 批量审核的单据必须是同一种单据类型
