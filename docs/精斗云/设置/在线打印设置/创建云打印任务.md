## **创建云打印任务**

### 基本信息

- 用途说明：创建云打印任务
- 请求地址：https://api.kingdee.com/jdyscm/v2/onlinePrint/createCloudPrintTask
- 请求方式：POST

### 请求参数

#### headers参数

[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)

#### params参数

| 参数名称 | 必选  | 类型  | 说明  |
| --- | --- | --- | --- |
| access_token | *true* | string | 云平台访问令牌 |
| dbId | *true* | number | 账套ID |

#### body参数

| 参数名称 | 必选  | 类型  | 说明  |
| --- | --- | --- | --- |
| businessCode | *true* | string | 业务编码 |
| templateId | *false* | number | 打印模板id（不传时取默认模板） |
| templateSource | *false* | number | 打印模板来源（0-在线打印 1-在线专业套打  不传默认在线打印） |
| bills | *true* | array | 单据列表 |
| - billId | *true* | number | 单据id |
| - billNo | *true* | string | 单据编号 |
| printerIds | *true* | array | 打印设备id列表（暂仅支持一个设备） |

#### 请求示例

```
POST
https://api.kingdee.com/jdyscm/v2/onlinePrint/createCloudPrintTask?access_token=1510557056dbfff1a4ab8bcf8a686679&dbId=37436958539

HTTP 1.1
```

```
BODY
{
    "businessCode": "SALES",
    "templateId": 1322412412,
    "templateSource": 0,
    "bills": [
        {
            "billId": 7950959172,
            "billNo": "XS20170712001"
        }
    ],
    "printerIds": [
        129656912913
    ]
}
```

| 返回参数 | 字段类型 | 字段说明 |
| --- | --- | --- |
| code | string | 返回码 |
| msg | string | 结果或错误描述 |
| data | string | 成功时返回 “创建云打印任务成功” |

#### 响应示例

```
{
    "code": 0,
    "msg": "成功",
    "data": "创建云打印任务成功"
}
```

| 0   | 成功  |
