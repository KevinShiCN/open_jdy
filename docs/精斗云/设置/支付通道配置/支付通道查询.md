### 基本信息
- 用途说明：支付通道查询配置
- 请求方式：POST
- 请求地址：https://api.kingdee.com/jdyscm/onlinePayInfo/list

### 请求参数  
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)
#### params参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID

#### body参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|filter |*false*	| object|过滤条件
|-- types	|*false*	|array	|支付类型（1：现金、2：聚合支付、3：二维码付款）

#### 请求示例
```
POST 
https://api.kingdee.com/jdyscm/v2/onlinePayInfo/list?access_token=1510557056dbfff1a4ab8bcf8a686679&amp;dbId=37436958539

HTTP 1.1
```
```
BODY
 {
 	filter:
	 {
	 types:[1,2]
	 }
}
```

### 响应参数
返回参数 	|字段类型 	|字段说明 
:---   |     :---     | :---
code	|string	| 返回码
msg	|string 	|结果或错误描述
data|array|单据列表
--fid | number |ID
--name| string |账户标题
--payType|numer|支付渠道；1：通联支付
--payMethodName|string|结算方式名称
--payAcctName|string|结算账户名称
--isDelete|number|启用/禁用标记；0：启用、1：禁用
--type|number|支付类型；1：现金、2：聚合支付、3：二维码支付
--order|number|排序值
--privateKey|string|商户公钥
--publickey|string|商户私钥
--appid|number|商户号

#### 响应示例
```json
{
	code:0,
