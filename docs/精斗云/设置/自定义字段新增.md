### 基本信息
- 用途说明：新增保存单据自定义字段
- 请求方式：POST
- 请求地址：https://api.kingdee.com/jdyscm/udf/save

### 请求参数
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)

#### params参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID

#### body参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|code	|true	|string	|单据类型（sale：销货单（销货订单），saleback：销货退货单，pur：购货单（购货订单），purback：购货订单）
|entries |true	| jsonArray|自定义字段列表
|--index	|true	|string	|顺序号（【1~5】，1指示第一个自定义字段）
|--name	|true	|string	|名称（不能超过6个字符）
|--type	|true	|string	|类型（text：文本，radio：单选，date：日期）
|--value	|false*	|jsonArray	|单选选项值（***当type为radio时必填**）

#### 请求示例
```
POST 
https://api.kingdee.com/jdyscm/udf/save?access_token=xxx&amp;dbId=xxx

HTTP 1.1
```
```
BODY
{
    code:saleback,
    entries:[
        {
            index:1,
            name:字段1,
            type:text
        },
        {
            index:2,
            name:字日2,
            type:date
        },
        {
            index:3,
            name:字选sba,
            type:radio,
            value:[
                3,
                4
            ]
        }
    ]
}
```
