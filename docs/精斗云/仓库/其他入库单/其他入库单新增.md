#### body参数
参数名称 |  是否必选  |参数类型 |参数说明  |
 :---    | :---   |     :---     | :---
items|true|array|单据列表
-- number|true| string |编码
-- date|true|date|日期
-- supplierNumber|true|string|供应商编码
-- bizType|true|number|业务类型（1-其他入库; 2-盘盈）

---- unit|false|string|单位ID(商品存在单位时必填)
---- location|true|string|仓库
---- serNumList|false|array|序列号明细
------ serNum|false|string|序列号编码 
------ desc|false|string|序列号描述
---- qty|true|number|数量
---- price|true|number|入库单价
---- remark|false|string|备注
---- batch|false|string|批次
---- prodDate|false|date|生产日期（有批次时必填）
