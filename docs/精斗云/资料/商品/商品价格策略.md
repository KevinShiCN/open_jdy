### 基本信息

- 用途说明：获取精斗云进销存V3系统的商品价格策略（支持批量查询）
- 请求方式：POST
- 请求地址：http://api.kingdee.com/jdyscm/product/queryProductPrice

### 请求参数
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)
#### params参数

| 参数名称         | 必选   | 类型     | 说明      |
| ------------ | ---- | ------ | ------- |
| access_token | true | String | 云平台访问令牌 |
| dbId         | true | number | 账套ID    |

#### body参数

| 参数名称      | 必选    | 类型        | 说明                                       |
| --------- | ----- | --------- | ---------------------------------------- |
| filter    | true  | object    | 过滤条件                                     |
| -type     | true  | string    | 单据类型（SALE-销货类单据，PUR-购货类单据，OO-其他入库类单据）其他入库类单据总是取最近价 |
| -buId     | true  | string    | 客户/供应商id，其他入库单可以不传此id                    |
| -invs     | true  | JsonArray | 商品列表                                     |
| ---invId  | true  | string    | 商品id                                     |
| ---skuId  | false | string    | 辅助属性id                                   |
| ---unitId | false | string    | 单位id                                     |

#### 请求示例

```
POST
https://api.kingdee.com/jdyscm/product/update?access_token=xx&amp;dbId=xx

HTTP 1.1
```

 	filter:

                }
           ]，

                {
                    invId: "129112953091699",

                }
           ]
	 }
}
```

### 响应参数
响应参数 	|参数类型 	|参数说明
| ---------- | ------ | ---------------------------------------- |
| code       | string | 返回码                                      |
| msg        | string | 结果或错误描述                                  |
| items      | array  | 商品价格策略列表                                 |
| --level    | number | 客户等级（只有销货类单据才返回客户等级，0-'零售客户'，1-'批发客户'，2-'VIP客户'，3:-'折扣等级一'，4-'折扣等级二'）默认值：-1 |
| --price    | number | 商品价格（客户等级为3或4的客户，此价格为折扣前价格）              |
| --invId    | string | 商品id                                     |
| --unitId   | string | 单位id                                     |
| --discount | number | 折扣率（%）（客户等级为：折扣等级一/折扣等级二的客户才会带出折扣率）默认值：0 |
| --taxPrice | number | 含税价格（客户等级为3或4的客户，此价格为折扣前价格）              |
| --skuId    | string | 辅助属性id                                   |
| --srcPrice | number | 购货价/销售价来源（1-商品价格策略，2-最近采购价/最近售价）         |
| page       | number | 当前页码                                     |
| totalsize  | number | 当前返回总记录数                                 |
| records    | number | 总记录数                                     |
| totalPages | number | 总页数                                      |

### []()基本信息

- 用途说明：获取精斗云进销存V3系统的商品价格策略（支持批量查询）

- 请求方式：POST

- 请求地址：http://api.kingdee.com/jdyscm/product/queryProductPrice

### []()请求参数

#### []()headers参数

[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)

#### []()params参数

| 参数名称 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| access_token | true | String | 云平台访问令牌 |
| dbId | true | number | 账套ID |

#### []()body参数

| 参数名称 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| filter | true | object | 过滤条件 |
| -type | true | string | 单据类型（SALE-销货类单据，PUR-购货类单据，OO-其他入库类单据）其他入库类单据总是取最近价 |
| -buId | true | string | 客户/供应商id，其他入库单可以不传此id |
| -invs | true | JsonArray | 商品列表 |
| —invId | true | string | 商品id |
| —skuId | false | string | 辅助属性id |
| —unitId | false | string | 单位id |

### []()响应参数

| 响应参数 | 参数类型 | 参数说明 |
| --- | --- | --- |
| code | string | 返回码 |
| msg | string | 结果或错误描述 |
| items | array | 商品价格策略列表 |
| –level | number | 客户等级（只有销货类单据才返回客户等级，0-‘零售客户’，1-‘批发客户’，2-‘VIP客户’，3:-‘折扣等级一’，4-‘折扣等级二’）默认值：-1 |
| –price | number | 商品价格（客户等级为3或4的客户，此价格为折扣前价格） |
| –invId | string | 商品id |
| –unitId | string | 单位id |
| –discount | number | 折扣率（%）（客户等级为：折扣等级一/折扣等级二的客户才会带出折扣率）默认值：0 |
| –taxPrice | number | 含税价格（客户等级为3或4的客户，此价格为折扣前价格） |
| –skuId | string | 辅助属性id |
| –srcPrice | number | 购货价/销售价来源（1-商品价格策略，2-最近采购价/最近售价） |
| page | number | 当前页码 |
| totalsize | number | 当前返回总记录数 |
| records | number | 总记录数 |
| totalPages | number | 总页数 |

| --- | --- |
| 0 | 成功 |
| 500 | 内部服务器错误 |
| 40001 | 无API访问权限 |
