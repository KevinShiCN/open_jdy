### 请求参数

[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)

#### params参数

| 参数名称 | 必选  | 类型  | 说明  |
| --- | --- | --- | --- |
| access_token | *true* | string | 云平台访问令牌 |
| dbId | *true* | number | 账套ID |

#### body参数

| 参数名称 | 必选  | 类型  | 说明  |
| --- | --- | --- | --- |
| id  | true | number |     |
| routeNo | true | string | 线路编码 |
| name | true | string | 名称  |
| remark | false | string | 备注  |
| status | false | number | 状态(1:启用 0:禁用) |
| deliveryMan  | false | string | 送货员 职员ID  |
| driver | false | string | 司机 职员ID  |
| carNumber | false | string | 车牌号  |
| carType | false | string | 车辆类型  |
| carLoad | false | string | 车辆载重  |
| isDefault | false | number | 是否默认(1:是 0:否)  |

#### 请求示例

```
POST
https://api.kingdee.com/jdyscm/v2/route/update?access_token=1510557056dbfff1a4ab8bcf8a686679&dbId=37436958539

HTTP 1.1
```

```
BODY
{
    "id": 129377967792,
    "routeNo": "M0011",
    "name": "京广线3",
    "remark": "232",

}

    "code": 0,

    "data": 1
}
```
