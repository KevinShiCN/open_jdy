## **列表查询**

### 基本信息

- 用途说明：查询线路列表
- 请求地址：https://api.kingdee.com/jdyscm/v2/route/list
- 请求方式：POST

### 请求参数

#### headers参数

[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)

#### params参数

| 参数名称 | 必选  | 类型  | 说明  |
| --- | --- | --- | --- |
| access_token | *true* | string | 云平台访问令牌 |
| dbId | *true* | number | 账套ID |

#### body参数

| 参数名称 | 必选  | 类型  | 说明  |
| --- | --- | --- | --- |
| page | *false* | number | 页码  |
| rows | *false* | number | 每页行数 |
| skey | *false* | string | 搜索关键词 |
| status | *false* | number | 状态(1:启用 0:禁用) |
| sidx | *false* | string | 排序字段（routeNo 编号，name 名称，status 状态） |
| sord | *false* | string | 排序规则(asc/desc) |
| originNo| false | string | 外部id  |

#### 请求示例

```
POST
https://api.kingdee.com/jdyscm/v2/route/list?access_token=1510557056dbfff1a4ab8bcf8a686679&dbId=37436958539

HTTP 1.1
```

```
BODY
{
    "page": 1,
    "rows": 5,
    "skey": "",
    "status": 1,
    "sidx": "name",
    "sord": "asc"
}
```

| --- | --- | --- |
| code | string | 返回码 |
| msg | string | 结果或错误描述 |
| data | array |     |
| - page | number | 页码  |
| - total | number | 总页数 |
| - records | number | 总记录数 |
| - rows | array | 数据列表 |
| -- id | number |     |
| -- routeNo | string | 线路编号 |
| -- name | string | 线路名称 |
| -- remark | string | 备注  |
| -- deliveryMan | string | 送货员 职员ID  |
| -- deliveryManName | string | 送货员 名称  |
| -- driver | string | 司机 职员ID  |
| -- driverName | string | 司机 名称  |
| -- carNumber | string | 车牌号  |
| -- carType | string | 车辆类型  |
| -- carLoad | string | 车辆载重  |
| -- isDefault | number | 是否是默认线路(1:是 0:否)  |
| -- status | number | 状态(1:启用 0:禁用) |
| -- srcType | string | 来源类型(ISV,APP-IOS、APP-ANDROID、APP-H5、SCM) |
| -- userName | string | 创建人 |
| -- createDate | number | 创建时间 |
| -- modifyName | string | 修改人 |
| -- modifyTime | number | 修改时间 |
