## **删除线路**

### 基本信息

- 用途说明：删除线路
- 请求地址：https://api.kingdee.com/jdyscm/v2/route/delete
- 请求方式：POST

### 请求参数

#### headers参数

[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)

#### params参数

| 参数名称 | 必选  | 类型  | 说明  |
| --- | --- | --- | --- |
| access_token | *true* | string | 云平台访问令牌 |
| dbId | *true* | number | 账套ID |

#### body参数

| 参数名称 | 必选  | 类型  | 说明  |
| --- | --- | --- | --- |
| ids | *true* | string | 删除的id（多个用英文逗号分隔） |

#### 请求示例

```
POST
https://api.kingdee.com/jdyscm/v2/route/delete?access_token=1510557056dbfff1a4ab8bcf8a686679&dbId=37436958539

HTTP 1.1
```

```
BODY
{
    "ids": "129377967792,23213123"
}
```

| 返回参数 | 字段类型 | 字段说明 |
| --- | --- | --- |
| code | string | 返回码 |
| msg | string | 结果或错误描述 |
| data | object |     |
| - deleteSucRouteNames | string | 删除成功的线路名称 |
| - deleteFails | array | 删除失败的信息统计 |
| -- routeName | string | 线路名称 |
| -- contactCnt | number | 关联客户数(大于0表示有关联) |
| -- soCnt | number | 关联销售订单数(大于0表示有关联) |
| -- invSaCnt | number | 关联的销售单数(大于0表示有关联) |

#### 响应示例

```
{
    "code": 0,
    "msg": "成功",
    "data": {
        "deleteSucRouteNames": "京广线3,测试线路1",
        "deleteFails": [
            {
                "routeName": "测试路线",
                "contactCnt": 0,
                "soCnt": 0,
                "invSaCnt": 1
            }
        ]
    }
}
```

#### 返回码

| 返回值 | 说明  |
| --- | --- |
| 0   | 成功  |
| 500 | 内部服务器错误 |
| 40001 | 无API访问权限 |
| 46001 | 数据验证不通过 |
| 46002 | 无查询记录 |
| 48010 | 无相关权限 |

其他返回值参照产品概述章节列明的通用返回值说明。
