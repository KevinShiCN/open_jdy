[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)
#### params参数

| **参数名称** | **必选** | **类型** | **说明**       |
| ------------ | -------- | -------- | -------------- |
| access_token | *true*   | string   | 云平台访问令牌 |
| dbId         | *true*   | number   | 账套ID         |

####  body参数

| **参数名称** | **必选** | **类型** | **说明**     |
| ------------ | -------- | -------- | ------------ |
| filter       | *false*  | object   | 过滤条件     |
| –name        | *false*  | string   | 结算方式名称 |

####  请求示例

```JSON
POST 
https://api.kingdee.com/jdyscm/v2/payMethod/list?access_token=xxx&amp;dbId=xxx
 
HTTP 1.1
BODY
{
               "filter": {
                               "name":"在线支付"
               }
}
```
