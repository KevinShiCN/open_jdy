### 基本信息

- 用途说明：新增账户
- 请求方式：POST
- 请求地址：https://api.kingdee.com/jdyscm/account/add

### 请求参数
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)
#### params参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID

#### body参数
参数名称 |  必选  |类型 |说明  |
 :---    | :---   |     :---     | :---
items|true|array|列表
--number | true| string | 编号
--name | true| string | 名称
--openingBalance | false| string | 期初余额
--isDefault | false| string | 是否默认账户 （1：是，其他值：不是 默认不是）
--date | false| string | 建账日期 
--type | false| string | 账户类别（1：现金，2：银行存款）
--isApiAdd | false| boolean | api新增数据权限，为true时，在进销存页面无法直接删除该条数据，只允许通过api接口删除（默认false）

#### 请求示例
```
POST 
https://api.kingdee.com/jdyscm/account/add?access_token=xxx&amp;dbId=xxx

HTTP 1.1
```

        {
            id: ,
            number: 1007,
            name: 工商银行账户,
            openingBalance: 0,
            isDefault:1,
            date: ,
            type: 2
        }
        ]
}
```
