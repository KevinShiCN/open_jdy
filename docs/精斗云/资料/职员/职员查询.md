### 基本信息

- 用途说明：查询职员列表
- 请求方式：POST
- 请求地址：https://api.kingdee.com/jdyscm/employee/list

### 请求参数
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)
#### params参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID

#### body参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|filter |*false*	| object|过滤条件
|--number	|*false*	|string	|编码
|--name	|*false*	|string	|名称
|--isDelete|false|number|职员状态(0:只显示启用; 1:只显示禁用; 2:显示所有  默认2)
|--pageSize	|*false*	|int	|分页参数-每页记录数（默认10，500）
|--page	|*false*	|int	|分页参数-当前页码（默认1）

#### 请求示例
```
POST 
https://api.kingdee.com/jdyscm/employee/list?access_token=1510557056dbfff1a4ab8bcf8a686679&amp;dbId=37436958539

HTTP 1.1
```
```JSON
BODY
{
	filter:
	{
		number:,
		name:,
		mobile:,
		isDelete:2，
		page:1,
		pageSize:2
	}
}
```
