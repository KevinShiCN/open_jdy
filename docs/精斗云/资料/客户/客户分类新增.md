### 基本信息

- 用途说明：新增客户分类（支持批量）
- 请求方式：POST
- 请求地址：https://api.kingdee.com/jdyscm/customer/addcategory

### 请求参数
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)
#### params参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID

#### body参数
参数名称 |  必选  |类型 |说明  |
 :---    | :---   |     :---     | :---
items|true|array|列表
--name | true| string |名称
--parentId | false| number |父级Id

#### 请求示例
```
POST 
https://api.kingdee.com/jdyscm/customer/addcategory?access_token=xxx&amp;dbId=xxx

HTTP 1.1
```
```JSON
BODY

{
items: [
        {
            name: Abbbb-2,
            parentId:12926494890
        }
    ]
}

```
