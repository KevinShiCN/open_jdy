### 基本信息

- 用途说明：获取精斗云进销存V3系统的客户分类列表（支持过滤）
- 请求方式：POST
- 请求地址：https://api.kingdee.com/jdyscm/customer/listcategory

### 请求参数
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)
#### params参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID

#### body参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|filter |*false*	| object|过滤条件
|--skey|*false*|string|关键字(可填名称)
|--pageSize	|*false*	|int	|分页参数-每页记录数（默认10，500）
|--page	|*false*	|int	|分页参数-当前页码（默认1）

#### 请求示例
```
POST 
https://api.kingdee.com/jdyscm/customer/listcategory?access_token=xxx&amp;dbId=xxx

HTTP 1.1
```
```JSON
BODY
{
	filter:
	{
		skey:,
		page:1,
		pageSize:10
	}
}
```
