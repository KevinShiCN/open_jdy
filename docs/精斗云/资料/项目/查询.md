| 参数名称 | 必选 | 类型 | 说明 |

| 参数名称 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| – displayDisabled | false | boolean | 是否显示禁用项目 |
| – fuzzyQuery | false | string | 项目编码或名称 |
| –pageSize | false | int | 分页参数-每页记录数（默认10，500） |
| –page | false | int | 分页参数-当前页码（默认1） |

HTTP 1.1
