### 基本信息

- 用途说明：获取精斗云进销存V3系统的单位列表
- 请求方式：POST
- 请求地址：https://api.kingdee.com/jdyscm/unit/list

### 请求参数
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)
#### params参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID

#### body参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|filter |*true*	| object|过滤条件|
|--type	|*true*	|string	|单位类型(0: 单计量单位  1:多计量单位)
|--unitId	|*false*	|string	|单位ID
|--pageSize	|*false*	|int	|分页参数-每页记录数（默认100）
|--page	|*false*	|int	|分页参数-当前页码（默认1）
#### 请求示例
```
POST
https://api.kingdee.com/jdyscm/unit/list?access_token=1510557056dbfff1a4ab8bcf8a686679&amp;dbId=37436958539

HTTP 1.1 
```
```JSON
BODY
{
	filter:
	{
		type:1,
		page:1,
		pageSize:20,
		unitId:
	}
}
```

### []()响应参数

| 响应参数 | 参数类型 | 参数说明 |
| --- | --- | --- |
| code | string | 返回码 |
| msg | string | 结果或错误描述 |
| items | array | 库存明细 |
| –name | string | 单位名称 |
| –ID | number | 单位ID(长整型，注意避免被截断) |
| –entries | array | 多计量单位详情 |
| ----name | string | 单位名称 |
| ----ID | number | 单位ID(长整型，注意避免被截断) |
| page | number | 当前页码 |
| totalsize | number | 当前返回总记录数 |
| records | number | 总记录数 |
| totalPages | number | 总页数 |

#### []()返回码

| 返回值 | 说明 |
| --- | --- |
| 0 | 成功 |
| 502 | 内部服务器错误 |
| 40001 | 无API访问权限 |
| 46001 | 数据验证不通过 |
| 46002 | 无查询记录 |

其他返回值参照产品概述章节列明的通用返回值说明。
