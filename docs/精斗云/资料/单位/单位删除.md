### 基本信息

- 用途说明：删除单位（支持批量）
- 请求方式：POST
- 请求地址：https://api.kingdee.com/jdyscm/unit/delete

### 请求参数
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)
#### params参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID

#### body参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|type |*true*	| number|单位类型（0：单计量单位，1：多计量单位）
|items |*true*	| array|单位id列表 

#### 请求示例
```
POST 
https://api.kingdee.com/jdyscm/unit/delete?access_token=xxx&amp;dbId=xx

HTTP 1.1
```
```JSON
BODY
{
	type:0,
    items:
    [
        111,
        222
    ]
}
```

### 响应参数
响应参数 	|参数类型 	|参数说明
:---   |     :---     | :---
code	|string	| 返回码（0成功，-2失败，-3部分失败）
msg	|string 	|结果或错误描述
totalsize|number|记录数
items|array|详细结果
--code|number|返回码
--msg|string|描述

#### 返回码
返回值 	|说明
:---   |     :---     
0	|成功
500	|内部服务器错误

46002|无查询记录
	
其他返回值参照产品概述章节列明的通用返回值说明。
