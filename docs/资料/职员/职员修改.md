| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID

#### body参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
items|true|array|供应商列表
--number | true| string |编码
--name | true| string |名称
--mobile|false|string|电话

```
POST 
https://api.kingdee.com/jdyscm/employee/update?access_token=1510557056dbfff1a4ab8bcf8a686679&amp;dbId=37436958539

HTTP 1.1
```
