### 基本信息

- 用途说明：获取精斗云进销存V3系统的商品类别列表
- 请求方式：GET
- 请求地址：https://api.kingdee.com/jdyscm/productCategory/list

### 请求参数
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)
#### params参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID

#### body参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|pageSize	|*false*	|int	|分页参数-每页记录数（默认10，500）
|page	|*false*	|int	|分页参数-当前页码（默认1）
#### 请求示例
```
GET
https://api.kingdee.com/jdyscm/productCategory/list?access_token=1510557056dbfff1a4ab8bcf8a686679&amp;dbId=37436958539
HTTP 1.1

```

### 响应参数
响应参数 	|参数类型 	|参数说明
:---   |     :---     | :---
code	|string	| 返回码
msg|string 	|结果或错误描述
items|array|商品列表
--id | number |ID
--name | string |名称
--parentId |number |上级类别ID
page|number|当前页码
totalsize|number|当前返回总记录数
records|number|总记录数
totalPages|number|总页数

#### 返回码
返回值 	|说明
:---   |     :---     
0	|成功
500	|内部服务器错误
40001|无API访问权限
46001|数据验证不通过
46002|无查询记录
	
其他返回值参照产品概述章节列明的通用返回值说明。

#### 响应示例
```JSON
{
    page: 1,
    items: [
        {
            id: 1292389569470,
            parentId: 0,
            name: 半裙
        },
        {
            id: 1292389569254,
            parentId: 0,
            name: 衬衫
        }
    ],
    totalsize: 2,
    code: 0,
    records: 32,
    msg: 成功,
    totalPages: 16
}
```
### 注意事项
-  所有ID均为长整型，注意避免被截断
```
