### 基本信息

- 用途说明：根据商品条形码从“中国商品信息服务平台”获取商品的基本信息
- 请求方式：GET
- 请求地址：https://api.kingdee.com/jdyscm/v2/barcode/queryBarcodeInfo

### 请求参数
#### headers参数
[详见链接](https://open.jdy.com/#/files/api/detail?id=391b87b075fb11edb79183b17ac8ddcc&noside=true)
#### params参数
|参数名称 |  必选  |类型 |说明  |
| :---    | :---   |     :---     | :---
|access_token	|*true*	|string	|云平台访问令牌
|dbId	|*true*	|number	|账套ID
|barcode	|*true*	|string	|商品条码

#### 请求示例
```
GET
https://api.kingdee.com/jdyscm/v2/barcode/queryBarcodeInfo?access_token=1510557056dbfff1a4ab8bcf8a686679&amp;dbId=37436958539&amp;dbId=6903244950756
HTTP 1.1

```
