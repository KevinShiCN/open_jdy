职员查询
更新于2023-10-18 12:02
复制链接
基本信息
用途说明：查询职员列表
请求方式：POST
请求地址：https://api.kingdee.com/jdyscm/employee/list
请求参数
headers参数
详见链接

params参数
参数名称	必选	类型	说明
access_token	true	string	云平台访问令牌
dbId	true	number	账套ID
body参数
参数名称	必选	类型	说明
filter	false	object	过滤条件
–number	false	string	编码
–name	false	string	名称
–isDelete	false	number	职员状态(0:只显示启用; 1:只显示禁用; 2:显示所有 默认2)
–pageSize	false	int	分页参数-每页记录数（默认10，500）
–page	false	int	分页参数-当前页码（默认1）
请求示例
POST 
https://api.kingdee.com/jdyscm/employee/list?access_token=1510557056dbfff1a4ab8bcf8a686679&amp;dbId=37436958539

HTTP 1.1
BODY
{
	filter:
	{
		number:,
		name:,
		mobile:,
		isDelete:2，
		page:1,
		pageSize:2
	}
}
响应参数
响应参数	参数类型	参数说明
code	string	返回码
msg	string	结果或错误描述
items	array	客户列表
–id	number	ID
–number	string	编码
–name	string	名称
–isDelete	boolean	是否禁用
page	number	当前页码
totalsize	number	当前返回总记录数
records	number	总记录数
totalPages	number	总页数
返回码
返回值	说明
0	成功
500	内部服务器错误
40001	无API访问权限
46001	数据验证不通过
46002	无查询记录
其他返回值参照产品概述章节列明的通用返回值说明。

响应示例
{
    page: 1,
    items: [
        {
            id: 129253933694,
            number: ZY001,
            name: 职员001,
            isDelete: false
        },
        {

            number: ZY002,
            name: 职员002,
            isDelete: false
        }
    ],
    totalsize: 2,
    code: 0,
    records: 2,

    totalPages: 1
}
