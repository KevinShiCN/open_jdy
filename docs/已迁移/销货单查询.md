销货单查询
更新于2024-04-10 15:48
复制链接
基本信息
用途说明：查询销售单列表
请求方式：POST
请求地址：https://api.kingdee.com/jdyscm/delivery/list
请求参数
headers参数
详见链接

params参数
参数名称	必选	类型	说明
access_token	true	string	云平台访问令牌
dbId	true	number	账套ID
body参数
参数名称	必选	类型	说明
filter	false	object	过滤条件
– number	false	string	单据编码
–checkStatus	false	number	审核状态(0:未审核; 1:已审核; 2:所有)
–sourceOrder	false	string	源订单编码
–customerNumber	false	string	客户编码
– beginDate	false	date	单据日期-开始（yyyy-MM-dd ）
– endDate	false	date	单据日期-结束（yyyy-MM-dd ）
– updTimeBegin	false	datetime	修改时间-起始（yyyy-MM-dd HH:mm:ss）
– updTimeEnd	false	datetime	修改时间-结束（yyyy-MM-dd HH:mm:ss）
– productNumber	false	string	商品编码
– locationNumber	false	string	仓库编码
– checkName	false	string	审核人名称
– createName	false	string	创建人名称
– employeeNumber	false	string	销售人员编码
– batch	false	string	批次
– originNo	false	string	来源单号
–pageSize	false	int	分页参数-每页记录数（默认10，500）
–page	false	int	分页参数-当前页码（默认1）
请求示例
POST 
https://api.kingdee.com/jdyscm/delivery/list?access_token=1510557056dbfff1a4ab8bcf8a686679&amp;dbId=37436958539

HTTP 1.1
BODY
 {
 	filter:
	 {
	 updTimeBegin: 2017-10-08,
	 sourceOrder: NO123,
	 updTimeEnd:2018-10-20,
	 customerNumber:KH002，
	 pageSize:20,
	 page:1,
	 originNo:10086
	 }
}
