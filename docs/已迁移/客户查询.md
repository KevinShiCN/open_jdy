更新于2023-09-13 10:09
复制链接
基本信息
用途说明：获取精斗云进销存V3系统的客户列表（支持过滤）
请求方式：POST
请求地址：https://api.kingdee.com/jdyscm/customer/list
请求参数
headers参数
详见链接

params参数
参数名称	必选	类型	说明
access_token	true	string	云平台访问令牌
dbId	true	number	账套ID
body参数
参数名称	必选	类型	说明
filter	false	object	过滤条件
–number	false	string	编码
–name	false	string	名称
–contact	false	string	联系人
–skey	false	string	关键字(可填编码、名称、联系人、手机号码、座机)
–status	false	number	启用状态（0:启用; 1:禁用; 2:所有）
–updTimeBegin	false	date	修改时间-起始
–updTimeEnd	false	date	修改时间-结束
–pageSize	false	int	分页参数-每页记录数（默认10，500）
–page	false	int	分页参数-当前页码（默认1）
请求示例
POST 
https://api.kingdee.com/jdyscm/customer/list?access_token=xxx&amp;dbId=xxx

HTTP 1.1
BODY
 {
 	filter:
	 {
	 number: 0000001,
	 name: 金鼎可-谭石民,
	 contact: 王先生,
	 status: 0,
	 updTimeBegin: 2018-10-08,
	 updTimeEnd:2018-10-09,
	 pageSize:20,
	 page:1
	 }
}
响应参数
响应参数	参数类型	参数说明
code	string	返回码
msg	string	结果或错误描述
items	array	客户列表
–id	number	ID
–number	string	编码
–name	string	名称
–category	string	类别
–level	number	等级 (0: 零售客户; 1: 批发客户; 2:VIP客户; 3:折扣等级一; 4:折扣等级二)
–employeeName	string	销售人员名称
–difMoney	number	应收款余额
–taxPayerNo	string	纳税人识别码
–bank	string	开户银行
–beginDate	string	余额日期
–amount	number	期初应收款
–periodMoney	number	期初预付款
–remark	string	备注
–isDeleted	boolean	是否禁用状态
–createTime	date	创建日期
–updateTime	date	修改时间
–contacts	array	联系人列表
----name	string	姓名
----id	number	联系人Id
----isPrimary	boolean	是否首要联系人
page	number	当前页码
totalsize	number	当前返回总记录数
records	number	总记录数
totalPages	number	总页数
返回码
返回值	说明
0	成功
500	内部服务器错误
40001	无API访问权限
46001	数据验证不通过
46002	无查询记录
其他返回值参照产品概述章节列明的通用返回值说明。
