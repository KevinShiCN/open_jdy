商品查询
更新于2023-05-26 16:05
复制链接
基本信息
用途说明：获取精斗云进销存V3系统的商品列表（支持过滤）
请求方式：POST
请求地址：https://api.kingdee.com/jdyscm/product/list
请求参数
headers参数
详见链接

params参数
参数名称	必选	类型	说明
access_token	true	string	云平台访问令牌
dbId	true	number	账套ID
body参数
参数名称	必选	类型	说明
filter	false	object	过滤条件
–productId	false	string	商品id
–productNumber	false	string	编码
–numbers	false	string	多个编码（用逗号隔开，最多支持30个）
–productName	false	string	名称
–spec	false	string	规格型号
–barcode	false	string	条码
–categoryId	false	string	类别ID
–status	false	number	启用状态（0:启用; 1:禁用; 2:所有）
–updTimeBegin	false	date	修改时间-起始
–updTimeEnd	false	date	修改时间-结束
–pageSize	false	int	分页参数-每页记录数（默认10，500）
–page	false	int	分页参数-当前页码（默认1）
请求示例
POST 
https://api.kingdee.com/jdyscm/product/list?access_token=1510557056dbfff1a4ab8bcf8a686679&amp;dbId=37436958539

HTTP 1.1
BODY
 {
 	filter:
	 {
	 productNumber: 003,
	 productName: iphone7,
	 spec: 64G,
	 status: 0,
	 categoryId:
	 updTimeBegin: 2019-01-08,
	 updTimeEnd:2019-10-09,
	 pageSize:20,
	 page:1
	 }
}

返回码
返回值	说明
0	成功
500	内部服务器错误
40001	无API访问权限
46001	数据验证不通过
46002	无查询记录
其他返回值参照产品概述章节列明的通用返回值说明。
